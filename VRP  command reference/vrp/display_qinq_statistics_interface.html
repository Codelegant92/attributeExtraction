display qinq statistics interface

命令功能

display qinq statistics interface用来统计指定子接口下基于Tag的报文统计信息。

命令格式

display qinq statistics interface interface-type interface-number [ .subinterface-number ] vid vid [ ce-vid ce-vid ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：用来进行基于Tag的报文统计的接口类型和名称。

vid vid：外层的Tag值，十进制整数形式，取值范围是1～4094。

ce-vid ce-vid：内层的Tag值，十进制整数形式，取值范围是1～4094。

verbose：显示详细的基于Tag的报文统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display qinq statistics interface命令可以根据Tag的内外层标签值来查看报文的统计信息。

使用实例

# 显示GE1/0/0.1接口下终结的外层Tag为10，内层Tag为100的报文统计信息。

<Quidway> display qinq statistics interface gigabitethernet 1/0/0.1 vid 10 ce-vid 100 verbose
GigabitEthernet1/0/0.1 Tag vid=10/100
    QinQ termination tag:10/100
      Statistics on VRP:
      0 Packets received, 0 bytes
      0 Error Packets received
      0 Packets transmitted, 0 bytes
      0 Error Packets transmitted,
