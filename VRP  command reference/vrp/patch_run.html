patch run

命令功能

patch run命令用来运行补丁。

缺省情况下，在主用主控板上运行补丁。

命令格式

patch run [ all | slave | slot slot-id ]

参数说明

all：运行所有单板上的补丁。

slave：运行备用主控板上的补丁。

slot slot-id：运行指定单板上的补丁。

视图

用户视图

缺省级别

3：管理级

使用指南

执行patch run all命令，将在所有在位单板上（包括主用主控板、备用主控板和所有接口板）运行处于激活状态的补丁。

当备用主控板正在注册中且尚未注册成功时，如果进行补丁运行操作，则系统会提示：是否确认继续执行补丁操作。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 运行主控板上的补丁。

<Quidway> patch run
Info: Successed in running the patch,Patch ID:SPH001

# 运行所有补丁。

<Quidway> patch run all
running the patch，please wait a moment ...
