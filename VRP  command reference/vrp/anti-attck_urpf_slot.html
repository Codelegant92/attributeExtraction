anti-attck urpf slot

命令功能

anti-attck urpf slot命令用来使能板级URPF功能。

undo anti-attck urpf slot命令用来去使能板级URPF功能。

缺省情况下，板级URPF功能去使能。

命令格式

anti-attck urpf { loose | strict } [ allow-default-route ] [ acl acl-number ] [ slot slot-number ]

undo anti-attck urpf [ slot slot-number ]

参数说明

loose：URPF将进行松散检查。

strict：URPF将进行严格检查。

allow-default-route：允许对默认路由进行特殊处理。

acl acl-number：指定ACL规则组编号。整数形式，范围是1000～3999，其中1000～1999是基于接口的访问控制列表，2000～2999是基本访问控制列表，3000～3999是高级访问控制列表。

slot slot-number：接口板槽位号。如果不指定槽位号，缺省为所有单板使能URPF。

使用指南

只要在转发表中存在表项就通过URPF检查，不要求接口一定匹配时配置loose模式；不但要求在转发表中存在相应表项，还要求接口一定匹配才能通过URPF检查时用strict模式。

使用实例

# 在6板上使能严格URPF检查，同时允许对缺省路由进行特殊处理，并配置ACL编号为2999。

<Quidway> system-view
[Quidway] anti-attack urpf strict allow-default-route acl 2999 slot 6
