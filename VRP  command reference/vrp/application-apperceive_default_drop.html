application-apperceive default drop

命令功能

application-apperceive default drop命令用来设置在查找不到应用层联动策略时，将报文丢弃。

undo application-apperceive default drop命令用来设置在查找不到应用层联动策略时，允许报文上送。

缺省情况下，在查找不到应用层联动策略时，允许报文上送。

命令格式

application-apperceive default drop

undo application-apperceive default drop

参数说明

无

系统视图

缺省级别

2：配置级

使用指南

执行命令application-apperceive default drop后，如果一个协议既没有在接口使能，也没有全局使能，该协议类型的报文将直接被丢弃。

使用实例

# 配置应用层联动的缺省处理方式为丢弃报文。

<Quidway> system-view
[Quidway] application-apperceive default drop
