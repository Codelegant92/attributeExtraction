mpls te path explicit-path

命令功能

mpls te path explicit-path命令用来为当前隧道设置使用的显式路径。

undo mpls te path命令用来删除显式路径。

命令格式

mpls te path explicit-path path-name [ secondary ]

undo mpls te path [ secondary ]

参数说明

path-name：MPLS-TE显式路径名称，字符串形式，长度为1～31。

secondary：表示为备份隧道配置显示路径。该参数只在（用mpls te backup命令）配置CR-LSP备份时使用。

视图

Tunnel接口视图

缺省级别

2：配置级

使用指南

配置mpls te path explicit-path命令之前，必须先在系统视图下配置相应的显式路径并为显式路径指定节点，否则该命令配置不成功。

使用实例

# 创建并配置显式路径path1。

<Quidway> system-view
[Quidway] explict-path path1
[Quidway-explicit-path-path1] next hop 2.2.2.2

# 设置为Tunnel1/0/0使用显式路径path1。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol mpls te
[Quidway-Tunnel1/0/0] mpls te path explicit-path path1
