loopback（CE3接口视图）

命令功能

loopback命令用来使能CE3接口的环回功能。

undo loopback命令用来禁止环回。

缺省情况下，禁止CE3接口环回。

命令格式

loopback { local | payload | remote }

undo loopback

参数说明

local：使能CE3接口对内自环。

payload：使能CE3接口对外载荷环回。数据经过成帧器，生成载荷后再进行环回。

remote：使能CE3接口对外远端环回。数据不经过成帧器，未生成载荷即进行环回。

缺省级别

2：配置级

使用指南

环回功能主要用于检测接口或电缆本身的状况，正常工作时应禁止这些功能。

当CE3接口设置为本端环回时，CE3接口的物理状态变为Up。

当对CE3接口设置环回时，该环回也会在该CE3接口下的同步串口生效：
  • 如果同步串口的链路层协议是PPP，当设置环回时，接口的物理状态变为Up，而链路协议状态为Down。
  • 如果同步串口的链路层协议是HDLC，当设置环回时，接口的物理状态和链路协议状态都是Up。

使用实例

# 使能CE3接口1/0/0对内自环。

<Quidway> system-view
[Quidway] controller e3 1/0/0
[Quidway-E3 1/0/0] loopback local
