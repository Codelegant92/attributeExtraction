hardware-based enable

命令功能

hardware-based enable用来使能在进行Jitter测试时，采用接口板上的硬件转发引擎进行发包及打时间戳。

undo hardware-based命令用来去使能采用接口板上的硬件转发引擎进行发包及打时间戳。

缺省情况下，没有使能接口板上的硬件转发引擎进行发包及打时间戳。

命令格式

hardware-based enable

undo hardware-based

参数说明

无

视图

NQA视图

缺省级别

2：配置级

使用指南

只有Jitter测试和ICMP Jitter测试才能使用命令hardware-based enable使能接口板上的硬件转发引擎进行发包和打时间戳。

使用实例

# 使能接口板上的硬件转发引擎进行发包和打时间戳。

<Quidway> system-view
[Quidway] nqa test-instance admin jitter
[Quidway-nqa-admin-jitter] test-type jitter
[Quidway-nqa-admin-jitter] hardware-based enable
