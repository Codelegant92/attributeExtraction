display ip vpn-instance import-vt

命令功能

display ip vpn-instance import-vt命令用来显示所有具备指定入口vpn-target属性的VPN实例。

命令格式

display ip vpn-instance import-vt ivt-value

参数说明

ivt-value：指定的入口vpn-target值。

视图

所有视图

缺省级别

1：监控级

使用指南

入口vpn-target属性是确定BGP VPNv4路由是否可以添加到VPN路由表中，使用命令display ip vpn-instance import-vt可以查看具备指定入口vpn-target属性的所有VPN实例。该命令目前不支持一次查看拥有多个入口vpn-target属性值的VPN实例。

使用实例

# 查看入口vpn-target属性值为1:1的VPN实例信息。

<Quidway> display ip vpn-instance import-vt 1:1
The vpn-instances are :

  VPN-Instance Name and ID : vrf1, 2
  VPN-Instance Name and ID : vrf2, 4
  VPN-Instance Name and ID : vrf3, 5
  VPN-Instance Name and ID : vrf4, 6
  VPN-Instance Name and ID : vrf5, 7
  VPN-Instance Name and ID : vrf8, 10
