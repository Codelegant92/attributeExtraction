ipv6 address auto link-local

命令功能

ipv6 address auto link-local命令用来为接口配置自动生成的链路本地地址。

undo ipv6 address auto link-local命令用来删除自动生成的链路本地地址。

命令格式

ipv6 address auto link-local

undo ipv6 address auto link-local

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

此命令可以用来配置Ethernet及其子接口、GigabitEthernet接口及其子接口、POS接口、Serial接口、Loopback接口、Eth-Trunk接口及其子接口、IP-Trunk接口和Tunnel接口上的IPv6链路本地地址。

使用实例

# 配置GE1/0/0接口自动生成链路本地地址。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 enable
[Quidway-GigabitEthernet1/0/0] ipv6 address auto link-local
