项目

描述

Feature name

告警所属的模块名称。

Trap number

告警的数量。

Trap name

告警的名称，MPLS LSPM模块的告警包括：

  • mplsXCUp：LSP变为Up事件的告警。
  • mplsXCDown：LSP变为Down事件的告警。
  • mplsTunnelUp：隧道变为Up事件的告警。
  • mplsTunnelDown：隧道变为Down事件的告警。
  • hwMplsStaticLspUp：静态LSP变为Up事件的私有告警。
  • hwMplsStaticLspDown：静态LSP变为Down事件的私有告警。
  • hwMplsStaticCRLspUp：静态CR-LSP变为Up事件的私有告警。
  • hwMplsStaticCRLspDown：静态CR-LSP变为Down事件的私有告警。
  • mplsTunnelReoptimized：隧道重优化事件的告警。
  • mplsTunnelRerouted：隧道重路由事件的告警。
  • hwMplsTeFrrSwitch：TE FRR保护切换事件的私有告警。
  • hwMplsTeFrrResume：TE FRR故障恢复事件的私有告警。
  • hwMplsTunnelHSBSwitch：CR-LSP热备份方式下从主LSP切换到备份LSP事件的私有告警。
  • hwMplsTunnelHSBResume：CR-LSP热备份方式下从备份LSP回切到主LSP事件的私有告警。
  • hwMplsTunnelOBSwitch：CR-LSP普通备份方式下从主LSP切换到备份LSP事件的私有告警。
  • hwMplsTunnelOBResume：CR-LSP普通备份方式下从备份LSP回切到主LSP事件的私有告警。
  • hwMplsTeFrrProtAval：TE FRR旁路隧道绑定事件的私有告警。
  • hwMplsTeFrrProtNotAval：TE FRR旁路隧道去绑定事件的私有告警。
  • hwmplsTunnelUp：隧道变为Up事件的私有告警开关。
  • hwmplsTunnelDown：隧道变为Down事件的私有告警开关。
  • hwmplsTunnelChangeBW：隧道带宽发生变化事件的私有告警开关。

Status

告警的状态：

  • on：表示告警处于开启状态。
  • off：表示告警处于关闭状态。

Excessive Flag

海量告警标识符：

  • N：表示非海量告警。
  • Y：表示海量告警。
