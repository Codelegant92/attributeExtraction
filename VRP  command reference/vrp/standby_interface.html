standby interface

命令功能

standby interface命令用来为主接口指定某物理接口或Tunnel接口作为备份接口。

undo standby interface命令用来取消已指定好的备份接口。

缺省情况下，不指定任何备份接口。

命令格式

standby interface interface-type interface-number [ priority ]

undo standby interface interface-type interface-number

参数说明

interface-type：接口类型。

interface-number：接口编号。

priority：备份接口的优先级，取值范围是0～255，值越大表示优先级越高，缺省值为0。

使用指南

一个主接口可以设置多个备份接口，这些备份接口根据优先级的高低决定备份时的使用顺序，优先级高的接口优先被使用。

使用实例

# 指定接口POS1/0/0作为接口POS2/0/0的备份接口，其优先级为50。

<Quidway> system-view
[Quidway] interface pos 2/0/0
[Quidway-Pos2/0/0] standby interface pos 1/0/0 50
