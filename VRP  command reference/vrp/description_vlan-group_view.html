description（VLAN组视图）

命令功能

description命令用来配置VLAN-Group的描述信息。

undo description命令用来删除VLAN-Group的描述信息。

缺省情况下，VLAN-Group没有配置描述信息。

命令格式

description description

undo description

参数说明

description：字符串形式，支持空格，区分大小写，长度范围是1～31。

视图

VLAN组视图

缺省级别

2：配置级

使用指南

VLAN-Group视图下描述信息，通常用来描述该VLAN-Group视图下配置的作用。

使用实例

# 在VLAN-Group视图下配置描述信息，对vlan-group1的说明为for_user1。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] vlan-group 1
[Quidway-GigabitEthernet1/0/0.1-vlangroup1] description for_user1
