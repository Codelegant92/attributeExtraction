code（ATM E1）

命令功能

code命令用来设置ATM E1/T1线路编解码格式。

undo code命令用来恢复缺省设置。

缺省情况下，E1线路编解码格式为HDB3，T1线路编解码格式为B8ZS。

命令格式

code { hdb3 | ami }

code { b8zs | ami }

undo code

参数说明

ami：Alternate Mark Inversion，交替传号反转码，可以在E1或T1线路上设置。

hdb3：High Density Bipolar of Order 3，3阶高密度双极性码，只可以在E1线路上设置。

b8zs：Bipolar with 8-Zero Substitution，双极性8zero替换码，只可以在T1线路上设置。

使用指南

在设置线路编解码时，注意与对端设备保持一致。

使用实例

# 设置ATM E1 接口1/0/2的线路编解码格式为AMI。

<Quidway> system-view
[Quidway] interface atm 1/0/2
[Quidway-Atm1/0/2] code ami
