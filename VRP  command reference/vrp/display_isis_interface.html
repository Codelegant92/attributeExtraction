display isis interface

命令功能

display isis interface命令用来查看使能了IS-IS的接口信息。

命令格式

display isis interface [ [ verbose | traffic-eng ] * | tunnel ] [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

verbose：如果使用该参数，将显示接口的详细信息。

traffic-eng：显示IS-IS的流量控制信息。

tunnel：显示IS-IS的MPLS TE隧道信息。

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：显示指定IPv4 VPN的IS-IS多实例进程的接口信息。vpn-instance-name为IPv4 VPN实例的名称，区分大小写。

vpn-instance default：显示除IPv4 VPN实例外的所有IS-IS进程的接口信息，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：显示指定IPv6 VPN的IS-IS多实例进程的接口信息。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：显示除IPv6 VPN实例外的所有IS-IS进程的接口信息，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

该命令所查看的接口相关信息，包括接口名、接口IP地址、接口链路状态等多种信息。使用参数verbose除显示上述所有信息外，还可显示接口的CSNP报文广播间隔、Hello报文广播间隔、宣告邻居失效前IS-IS没有收到的Hello报文的数目等与接口相关的IS-IS参数设置情况。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
