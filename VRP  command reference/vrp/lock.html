lock

命令功能

lock命令用来锁住当前用户界面，防止未授权的用户操作该终端界面。

undo lock命令用来解除当前用户界面的锁定。

命令格式

lock

undo lock

参数说明

无

视图

用户视图

缺省级别

3：管理级

使用指南

用户界面包括CON口、AUX口以及VTY虚拟终端等用户界面。

用户输入lock命令后，系统提示输入屏保密码，并提示再次输入屏保密码，如果两次输入的密码相同，则锁定当前用户界面成功。之后如果想进入系统，必须先键入回车键，此时提示输入密码，只有输入正确的密码才可以解除锁定进入系统。

使用实例

# 用户从CON口登录，锁住当前用户界面。

<Quidway> lock
Enter Password：xxxx
Confirm Password：xxxx
Info: The terminal is locked !

# 用户稍后再想进入系统，在界面上键入回车键后出现如下提示信息。

Enter Password:
