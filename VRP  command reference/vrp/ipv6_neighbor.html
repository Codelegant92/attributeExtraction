ipv6 neighbor

命令功能

ipv6 neighbor命令用来配置邻居发现缓存中的静态邻居信息。

undo ipv6 neighbor命令用来删除静态缓存中的邻居项。

命令格式

ipv6 neighbor ipv6-address mac-address

undo ipv6 neighbor ipv6-address

参数说明

ipv6-address：指定邻居的IPv6地址。

mac-address：指定静态邻居的本地数据链路层地址（48位，格式为H-H-H）。

使用指南

在一个接口上最多可以配置300项静态邻居。

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 配置GE1/0/0接口上邻居发现缓存中的静态邻居信息。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 neighbor 5000::1 fe-e0-89
