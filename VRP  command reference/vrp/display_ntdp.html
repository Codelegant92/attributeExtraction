display ntdp

命令功能

display ntdp命令用来查看全局NTDP配置信息。

命令格式

display ntdp

参数说明

无

使用实例

# 查看全局NTDP配置信息。

<Quidway> display ntdp
 Info: Network topology discovery protocol is enabled. Hops      : 3
 Timer     : 10 min
 Hop Delay : 200 ms
 Port Delay: 20 ms
 Total time for last collection:260ms

该显示信息主要内容解释见表1。

表1 display ntdp命令输出信息描述

项目

描述

Hops

拓扑收集范围（跳数）。

Timer

定时收集拓扑的时间间隔。

Hop Delay

转发拓扑请求报文的跳数延迟时间。

Port Delay

转发拓扑请求报文的接口延迟时间。

Total time for last collection

最近一次收集拓扑时所用的时间。
