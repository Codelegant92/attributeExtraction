display interface eth-trunk

命令功能

display interface eth-trunk命令用来查看Eth-Trunk接口的状态信息。

命令格式

display interface eth-trunk [ trunk-id [ .subnumber ] ]

参数说明

trunk-id：端口聚合的Trunk ID。取值范围是0～63。

subnumber：Eth-Trunk的子接口编号，在VRP上的取值范围是1～4094。但每个Eth-Trunk主接口下最大的子接口数量为4094个。不同产品上实现的Eth-Trunk子接口取值范围不同，请以产品实现的规格为准。

使用指南

查看Eth-Trunk接口的信息时，可以查看到该接口下捆绑的成员端口的状态和权重信息。

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

表1 display interface eth-trunk命令输出信息描述

项目

描述

Eth-Trunk1 current state
捆绑接口Eth-Trunk1的当前状态。

  • UP：接口处于正常启动的状态。

  • DOWN：接口在物理上出现故障。

  • Administratively down：如果网络管理员在该接口执行shutdown命令，将显示状态为Administratively down。

Line protocol current state

接口当前的链路状态。

Last line protocol up tim

接口的协议状态最近一次变为Up的时间。

Route Port

路由式端口。

Hash arithmatic
散列算法的依据。
  • According to flow：Eth-Trunk接口根据流进行散列。
  • According to packet all：Eth-Trunk接口根据所有包进行散列。

Maximal BW

Eth-Trunk接口的最大带宽。

Current BW

Eth-Trunk接口的当前带宽。

The Maximum Transmit Unit is

接口的最大传输单元。

Internet Address is

接口的IP地址。

IP Sending Frames' Format is

接口发送的Ethernet帧的格式。

Ethernet_II为默认的帧格式。 Ethernet在接收帧时，可以识别如下几种帧格式。
  • Ethernet_II
  • Ethernet_SNAP
  • 802.2
  • 802.3

Hardware address is 00e0-fc09-9722

捆绑口的MAC地址。

Encapsulation

子接口的封装方式。该字段只有在子接口时才显示。

The number of Vlan is

子接口的VLAN数量。该字段只有在子接口时才显示。

Vlan ID

子接口上封装的最小VLAN ID。该字段只有在子接口时才显示。

Physical is ETH_TRUNK

捆绑口的类型为Eth-Trunk。

Statistics last cleared

上次清除本接口统计信息的时间。

Last 300 seconds input/output rate

这一字段显示了在前300秒内通过接口发送和接收的字节速率和报文速率。

Input/Output

接口收到或发送报文的方向。

packets

接口接收或发送总报文数。

bytes

接口接收或发送总字节数。

unicast

接口接收或发送的单播报文的数。

broadcast

接口接收或发送的广播报文的数。

multicast

接口接收或发送的组播报文的数。

errors

接口在物理层检测时发现的错误报文数目。

drops

在物理层丢弃的报文数。

unknown protocol

接收报文时，在物理层检测时发现的不能识别的协议报文数。

PortName

Eth-Trunk接口中的成员端口名称。

Status

当前成员端口的状态。

Weight

成员端口负载分担的权重值。

The Number of Ports in Trunk

Eth-Trunk接口中成员端口的数量。

The Number of UP Ports in Trunk

Eth-Trunk接口中成员端口状态是Up的端口数量。
