firewall ipv6 fragments-inspect

命令功能

firewall ipv6 fragments-inspect命令用来设置IPv6报文的分片检查功能。

undo firewall ipv6 fragments-inspect命令用来取消IPv6报文的分片检查功能。

命令格式

firewall ipv6 fragments-inspect { all | slot slot-id }

undo firewall ipv6 fragments-inspect { all | slot slot-id }

参数说明

slot-id：指定设置或取消IPv6分片报文检查功能的接口板所在的插槽号。

all：设置或取消所有接口板的IPv6分片报文检查功能。

使用指南

如果需要过滤分片报文中的上层报文信息（除了IPv6报文头信息）必须设置IPv6分片报文检查功能。

使用实例

# 使能8号接口板的IPv6分片报文检查功能。

<Quidway> system-view
[Quidway] firewall ipv6 fragments-inspect slot 8
