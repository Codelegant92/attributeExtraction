agetime

命令功能

agetime命令用来配置NQA测试例的老化时间。

undo agetime命令用来取消老化时间，恢复缺省配置。

缺省情况下，老化时间为0，表示测试例永不老化。

命令格式

agetime hh:mm:ss

undo agetime

参数说明

hh:mm:ss：生命周期长度。取值范围是，hh:0～23，mm:0～59，ss:0～59。

视图

NQA视图

缺省级别

2：配置级

使用指南

当测试例处于inactive状态时，老化时间启动。当老化时间超时后，系统自动删除该测试例。当测试例处于active状态时，老化时间被复位。

不能修改正在执行的测试例的老化时间。

使用实例

# 设置测试例user test的生命周期。

<Quidway> system-view
[Quidway] test-type icmpjitter
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] agetime 1:0:0
