display bgp vpnv4 routing-table

命令功能

display bgp vpnv4 routing-table命令用来查看BGP VPNv4的路由信息。

命令格式

display bgp vpnv4 { all | route-distinguisher route-distinguisher | vpn-instance vpn-instance-name } routing-table [ as-path-filter as-path-filter-number | cidr | different-origin-as ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | route-distinguisher route-distinguisher | vpn-instance vpn-instance-name } routing-table regular-expression as-regular-expression

display bgp vpnv4 route-distinguisher route-distinguisher routing-table ipv4-address [ mask | mask-length ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } routing-table ipv4-address [ mask [ longer-prefixes ] | mask-length [ longer-prefixes ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } routing-table peer ipv4-address { advertised-routes | received-routes [ active ] } [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | route-distinguisher route-distinguisher | vpn-instance vpn-instance-name } routing-table community-filter { { community-filter-name | basic-community-filter-number } [ whole-match ] | advanced-community-filter-number } [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | route-distinguisher route-distinguisher | vpn-instance vpn-instance-name } routing-table community [ community-number | aa:nn ] &<1-13> [ internet | no-advertise | no-export | no-export-subconfed ] * [ whole-match ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } routing-table priority [ priority ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

longer-prefixes：允许按更长的掩码匹配。

as-path-filter：用来显示匹配指定过滤器的路由。

as-path-filter-number：指定匹配的AS路径过滤器号。

cidr：显示无分类域间路由CIDR(Classless InterDomain Routing)的路由信息。

regular-expression as-regular-expression：为匹配的AS正则表达式。字符串类型，取值范围是1～80。

different-origin-as：用来显示具有相同目的地址但不同源AS号的路由。

community-filter：用来显示匹配指定的BGP团体属性过滤器的路由信息。

community-filter-name：团体属性过滤器名称。

basic-community-filter-number：指定基本团体属性过滤器号。

advanced-community-filter-number：指定高级团体属性过滤器号。

community：用来查看路由表中指定BGP团体的路由信息。

community-number：指定的团体号。

aa:nn：指定的团体号。最多可以设置13个团体号。

internet：显示带有Internet团体属性的BGP路由。

no-advertise：显示带有No-Advertise团体属性的BGP路由。

no-export：显示带有No-Export团体属性的BGP路由。

no-export-subconfed：显示带有No-Export-Subconfed团体属性的BGP路由。

whole-match：精确匹配。

peer ipv4-address：查看指定对等体的路由信息。

advertised-routes：显示发布给指定对等体的路由信息。

received-routes：显示从指定对等体收到的路由信息。

active：显示从指定对等体收到的活跃路由信息。

all：查看所有VPNv4的BGP路由信息。

vpn-instance vpn-instance-name：查看指定VPN实例的BGP路由信息。

filter-option：过滤选项。

priority priority：路由收敛优先级值。该值越大，表示优先级越高。

mask | mask-length：点分十进制掩码/掩码长度。

ipv4-address：指定要显示的对等体的IPv4地址。

route-distinguisher route-distinguisher：显示指定RD的BGP路由信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过指定不同的参数可以只查看特定的路由信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
