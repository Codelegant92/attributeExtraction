ip netstream export host

命令功能

ip netstream export host命令用来配置输出的目的地址。

undo ip netstream export host命令用来取消输出目的地址的配置。

命令格式

ip netstream export host ip-address port

undo ip netstream export host ip-address port

参数说明

ip-address：指定输出目的地址，即NSC的IP地址。点分十进制形式。

port：指定输出的UDP端口号，即NSC的UDP端口号。整数形式，取值范围是0～65535。

视图

系统视图，NetStream聚合视图

缺省级别

2：配置级

使用指南

在系统视图下配置的是原始流的输出地址，在聚合视图下配置的是聚合流的输出地址。对于聚合模式，优先采用聚合模式下配置的目的地址，若在聚合模式下没有配置输出地址，则采用系统视图下配置的输出地址。

说明：

在每一个视图下，最多可以配置两个输出地址，以实现NSC的备份。配置第三个输出地址时，系统会提示超出最大的输出地址数，从而无法进行配置。

使用实例

# 设置原始流统计信息的输出地址为19.0.0.1，端口号为3000。

<Quidway> system-view 
[Quidway] ip netstream export host 19.0.0.1 3000

# 设置聚合流统计信息的输出地址为19.0.0.2，端口号为3001。

<Quidway> system-view 
[Quidway] ip netstream aggregation as
[Quidway-aggregation-as] ip netstream export host 19.0.0.2 3001
