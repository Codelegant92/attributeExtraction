pvp limit

命令功能

pvp limit命令用来设置VP监管的参数。

undo pvp limit命令用来取消VP监管。

缺省情况下，不进行VP监管。

命令格式

pvp limit vpi peak-rate

undo pvp limit vpi

参数说明

vpi：ATM网络虚路径标识（Virtual Path Identifier，VPI）。整数形式，取值范围是0～255。

peak-rate：欲保持的正常流量。整数形式取值范围是2000Kbit/s～149760Kbit/s。缺省值是149760Kbit/s；用于ATM E1接口时，该参数的取值范围是64Kbit/s～1920Kbit/s，缺省值是1920Kbit/s；用于ATM E3接口时，该参数的取值范围是64Kbit/s～34000Kbit/s，缺省值是34000Kbit/s。

使用指南

在应用VP监管时，PVC的参数仍然有效，只有满足PVC的参数与VP监管的参数时，分组才会被发送。在计算流量时，已经包括了LLC/SNAP、MUX和NLPID封装头部，但不包括ATM信元头。

使用实例

# 设置VPI为1的VP的流量为2M。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvp limit 1 2000
