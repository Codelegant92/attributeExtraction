frame-format（CT1接口视图）

命令功能

frame-format命令用来配置CT1的帧格式。

undo frame-format命令用来恢复缺省值。

缺省情况下，CT1接口使用的扩展超帧格式（esf）。

命令格式

frame-format { sf | esf }

undo frame-format

参数说明

sf：使用超帧格式，Super Frame。

esf：使用扩展超帧格式，Extended Super Frame。

视图

CT1接口视图

缺省级别

2：配置级

使用指南

配置时需与对端保持一致。

只有当CT1接口工作在CT1工作模式下时才能进行帧格式的配置。

使用实例

# 设置CT1接口1/0/0的帧格式为超帧。

<Quidway> system-view
[Quidway] controller t1 1/0/0
[Quidway-T1 1/0/0] frame-format sf
