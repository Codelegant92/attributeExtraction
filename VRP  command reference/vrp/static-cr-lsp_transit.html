static-cr-lsp transit

命令功能

static-cr-lsp transit命令用来在转发节点配置静态CR-LSP。

undo static-cr-lsp transit命令用来在转发节点删除配置的静态CR-LSP。

命令格式

static-cr-lsp transit lsp-name incoming-interface interface-type interface-number in-label in-label { nexthop next-hop-address | outgoing-interface interface-type interface-number } out-label out-label [ bandwidth [ ct0 | ct1 | ct2 | ct3 | ct4 | ct5 | ct6 | ct17 ] bandwidth ]

undo static-cr-lsp transit lsp-name

参数说明

lsp-name：CR-LSP隧道的名字，字符串形式，区分大小写，不支持空格，长度为1～19。

next-hop-addr：下一跳地址。

interface-type：接口类型。

interface-number：接口编号。

in-label-value：入标签，取值范围是16～1023。

out-label-value：出标签的值，整数形式，取值范围是16～1023。

ct0～ct7：设置CT0～CT7的带宽值。

bandwidth：带宽值，单位为kbit/s。取值范围是0～32000000，缺省值为0。
