display nqa support-test-type

命令功能

display nqa support-test-type命令用来显示NQA支持的测试例类型。

命令格式

display nqa support-test-type

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

用户购买的License不同，支持的测试例类型就有可能不同，或者不同的产品对于NQA测试例支持的类型可能不相同。用户可以通过使用display nqa support-test-type命令，查看到当前设备上可以支持的所有测试例。

  • 如果在NQA服务器端执行，则显示服务器可以进行的测试例。
  • 如果在NQA客户端执行，则显示被检测设备支持的测试例。

使用实例

# 显示NQA支持的测试例类型。

<Quidway> display nqa support-test-type
