reset vlan statistics

命令功能

reset vlan statistics命令用来清除指定VLAN或指定以太网子接口的VLAN报文收发统计信息。

命令格式

reset vlan statistics { [ vid ] vlan-id | interface { ethernet | gigabitethernet } interface-number [.subinterface-number ] }

参数说明

vlan-id：VLAN ID，取值范围是1～4094。

interface-number：接口编号。

subinterface-number：以太网子接口的接口编号，取值范围是1～4096。

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

reset vlan statistics命令清除的是display vlan statistics显示信息的最后一部分：接口输入输出报文的统计信息。

使用实例

# 清除子接口GE1/0/0.1的VLAN报文收发统计信息。

<Quidway> reset vlan statistics interface gigabitethernet1/0/0.1
