display dhcp server database

命令功能

display dhcp server database命令用来查看DHCP数据库的存放路径和文件名，包括文件大小、保存时间等信息。

命令格式

display dhcp server database

参数说明

无

使用实例

# 查看DHCP数据库存放信息。

<Quidway> display dhcp server database
 Status: disable
 Recover from files after reboot: disable
 File saving lease items: hda1:/dhcp/lease.txt
 File saving conflict items: hda1:/dhcp/conflict.txt
 Save Interval: 300 (seconds)

该显示信息主要内容解释如下表：

表1 display dhcp server database命令输出信息描述

项目

描述

Status

是否使能数据保存到存储设备的功能，有两种情况：disable表示未使能该功能enable表示已经使能该功能

Recover from files after reboot

是否使能重启后从存储设备文件恢复数据的功能，有两种情况：disable表示未使能该功能enable表示已经使能该功能

File saving lease items

存放地址租约信息的文件的文件名及路径

File saving conflict items

存放地址冲突信息的文件的文件名及路径

Save Interval

数据的定时保存间隔，单位是秒
