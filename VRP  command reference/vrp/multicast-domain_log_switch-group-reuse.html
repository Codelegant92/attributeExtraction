multicast-domain log switch-group-reuse

命令功能

multicast-domain log switch-group-reuse命令用来打开Switch-Group重用的日志信息输出开关。

undo multicast-domain log命令用来恢复缺省配置。

缺省情况下，关闭Switch-Group重用的日志信息。

命令格式

multicast-domain log switch-group-reuse

undo multicast-domain log

参数说明

无

2：配置级

使用指南

如果该VPN 中的Share-MDT组没有配置，不允许配置本命令。

# 启动VPN实例mytest的Switch-Group重用的日志信息开关。

<Quidway> system-view
[Quidway] ip vpn-instance mytest
[Quidway-vpn-instance-mytest] multicast-domain log switch-group-reuse
