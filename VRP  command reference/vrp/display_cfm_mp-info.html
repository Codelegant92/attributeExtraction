display cfm mp-info

命令功能

display cfm mp-info命令用来查看指定接口、VLAN或VSI上的CFM管理对象的信息。

命令格式

display cfm mp-info [ interface interface-type interface-number ] [ level md-level ] [ inward | outward ] [ vlan vlanid | vsi vsi-name | no-associated-vlan ]

参数说明

interface interface-type interface-number：显示指定接口名称、接口类型和接口号。接口类型包括：Ethernet接口、GE接口、Eth-Trunk接口。

level md-level：显示指定维护域MD级别。整数形式，取值范围是0～7，默认级别是0。

inward ：MEP类型。inward型MEP发出的802.1ag协议报文通过当前MA关联VLAN内的所有接口（除MEP所在接口）向外发送，即在当前MA关联的VLAN内广播。

outward：MEP类型。outward型MEP发出的协议报文直接通过该MEP所在的接口向外发送。

vlan vlanid：显示指定VLAN信息。整数形式，取值范围是1～4094。

vsi vsi-name：显示指定VSI信息。字符串形式，不支持空格，区分大小写，取值范围是1～31。

no-associated-vlan：表示没有关联VLAN的MP。

使用指南

现如今，802.1ag协议分为draft7草案版本和standard2007标准版本，本命令只支持standard2007标准版本使用。

在同一接口上可能会配置多个MEP、生成多个MIP，但只能有一个级别的MEP和MIP。如果配置错误会导致报文泄露或者无法正确回复LBR、LTR报文。

通过使用本命令，查看CFM管理对象的信息，了解该接口的MP之间的关系，以便正确配置MEP。

使用cfm enable（系统视图）命令全局使能CFM功能后才能使用本命令。

使用实例

# 查看指定接口、MD级别、MEP类型和VLAN信息。

<Quidway> display cfm mp-info interface gigabitethernet1/0/1 level 6 inward vlan 300
MD Name            : md1
MD Name Format     : md-name
Level              : 6
MA Name            : ma1
MEP ID             : 10
VLAN ID            : 300
VSI Name           : --
Interface Name     : gigabitethernet1/0/1
CCM Send           :enable
Direction          : inward
MAC Address        : 0018-8247-b977
-----------------------------------------------------
The total number of MPs is 1.
The number of MEPs is 1. The number of MIPs is 0.
