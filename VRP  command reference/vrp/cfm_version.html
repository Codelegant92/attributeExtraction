cfm version

命令功能

cfm version命令用来在以太网CFM 802.1ag协议的draft7草案版本和standard2007标准版本之间进行切换。

缺省情况下，设备启用的是draft7草案版本。

命令格式

cfm version { draft7 | standard }

参数说明

draft7：802.1ag草案版本。

standard：802.1ag标准版本。

缺省级别

2：配置级

使用指南

为了能够同时支持标准版本和draft版本，需要进行版本切换。

切换原则是整网中的所有设备使用的版本必须保持一致，才能保证正常的检测功能。在切换过程中，会屏蔽802.1ag协议的所有功能。所有设备使用的版本切换完成以后才能执行cfm enable（系统视图）命令全局使能CFM功能。

使用实例

# 将draft7版本切换为标准版本。

<Quidway> cfm version standard
