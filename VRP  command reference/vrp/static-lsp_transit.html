static-lsp transit

命令功能

static-lsp transit命令用来为中间传送LSR配置静态LSP。

undo static-lsp transit命令用来为中间传送LSR删除一条LSP。

命令格式

static-lsp transit lsp-name incoming-interface interface-type interface-number in-label in-label { nexthop next-hop-address | outgoing-interface interface-type interface-number } out-label out-label

undo static-lsp transit lsp-name

参数说明

lsp-name：LSP名称，字符串形式，区分大小写，不支持空格，长度范围是1～19个字符。

interface-type interface-num：接口类型和编号。

next-hop-address：下一跳地址。

in-label：入标签值，取值范围16～1023。

out-label：出标签值，取值范围16～1023。

使用指南

配置完静态LSP后，可以执行display mpls static-lsp命令查看该静态LSP是否建立成功。

使用实例

# 配置本LSR作为静态LSP“bj-sh”的Transit节点，入接口为POS3/0/0，入标签为123，出标签为253。

<Quidway> system-view
[Quidway] static-lsp transit bj-sh incoming-interface pos 3/0/0 in-label 123 nexthop 202.34.114.7 out-label 253
