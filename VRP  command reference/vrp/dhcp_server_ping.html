dhcp server ping

命令功能

dhcp server ping命令用来配置DHCP服务器发送ping报文的最大数量和最长等待回应时间。

undo dhcp server ping命令用来恢复缺省值。

命令格式

dhcp server ping { packets number | timeout milliseconds }

undo dhcp server ping { packets | timeout }

参数说明

packets number：发送ping报文的最大数目。整数形式，取值范围是0～10，0表示不进行ping操作。缺省值是2。

timeout milliseconds：每个ping报文的最长等待回应时间。整数形式，取值范围是0～10000，单位是毫秒。缺省值是500毫秒。

使用指南

DHCP服务器通过发送Ping报文探测地址的使用情况，以防止重复分配IP地址导致地址冲突。

使用实例

# 设置DHCP服务器最多发送10个Ping数据包，等待时间采用缺省值500毫秒。

<Quidway> system-view
[Quidway] dhcp server ping packets 10
