radius-server retransmit

命令功能

radius-server retransmit命令用来设置重传次数。

undo radius-server retransmit命令用来恢复缺省设置。

命令格式

radius-server retransmit retry-times

undo radius-server retransmit

参数说明

retry-times：重传次数，取值范围是1～5，缺省值是3。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

路由器向RADIUS服务器发出请求报文后，如果在规定的超时时限内未得到RADIUS服务器发回的应答，需重传请求报文。重传次数超出规定最大重传次数后，认为该服务器已不能正常工作。

只有当该RADIUS模板没有用户使用时，才能改变此配置。

该命令可以与radius-server timeout命令同时使用。

使用实例

# 配置服务器重传次数为4。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server retransmit 4
