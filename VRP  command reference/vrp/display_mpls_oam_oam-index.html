display mpls oam oam-index

命令功能

display mpls oam oam-index命令用来通过OAM索引查看OAM信息。

命令格式

display mpls oam oam-index index-value [ slot slot-id ]

参数说明

index-value：OAM索引，整数形式，取值范围是1～4294967294。

slot slot-id：接口板号。指定该参数后，只显示指定接口板的OAM信息。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定参数slot-id，显示OAM的简要信息。

执行命令display mpls oam { ingress | egress } { all | tunnel interface-number } [ slot slot-id | verbose ]，可获得隧道的OAM索引值。

使用实例

# 查看OAM索引值为512的OAM简要信息。

<Quidway> display mpls oam oam-index 512
--------------------------------------------------------------------------------

No. Tunnel-name       Ttsi                  Type    Frequency  Status
--------------------------------------------------------------------------------

1   Tunnel2/0/0       1.1.1.1 : 200         CV        1 s      Start/Non-defect

项目

描述

Tunnel-name

接口名称

Ttsi

TTSI，包括LSR ID和Tunnel ID两部分。冒号左侧是LSR ID，冒号右侧是Tunnel ID

Type

OAM报文检测类型

Frequency

OAM检测频率

Status
检查状态和缺陷状态，包括：
  • Start/No-defect：检测到没有故障
  • Start/In-defect：检测到处于故障状态

描述

Tunnel-name

隧道名称

Lsp signal status

信令协议的状态

被检测的LSP的类型

Lsp ingress lsr-id

Lsp tnl-id/Lsp-id

与LSP绑定的Tunnel ID/LSP ID

Oam-Index

OAM索引

Oam select board

被检测的LSP所在的接口板

Enable-state

OAM的使能状态

Ttsi/lsr-id

TTSI中的LSR ID

Ttsi/tunnel-id

TTSI中的Tunnel ID

Type

OAM报文检测类型

Frequency

OAM检测频率

Detect state

OAM检测状态

Defect-state
缺陷状态：
  • “No-defect”表示隧道无故障
  • “dLocv”表示隧道故障

Available state

隧道是否可用

Unavailable time (s)

不可用时长，单位为秒

Share attribute

反向通道使用方式，包括共享模式（Share）和独占模式（Private）

Lsp ingress lsr-id

反向LSP信令状态

Lsp tnl-id/lsp id

反向LSP的Tunnel ID

Lsp-inLabel

反向LSP的入标签

Lsp signal status
