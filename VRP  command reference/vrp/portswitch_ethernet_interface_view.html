portswitch（Ethernet接口视图）

命令功能

portswitch命令用来将以太网接口从三层模式切换到二层模式。

undo portswitch命令用来将以太网接口从二层模式切换到三层模式。

命令格式

portswitch

undo portswitch

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

缺省情况下，路由器或三层交换机的以太网接口一般工作在三层模式，如果需要把接口加入VLAN或进行二层接口的其他配置时，就需要把接口转换为二层模式。

如果其他接口已经配置了借用ethernet接口、GE接口或VE接口的IP地址，则对应的ethernet接口、GE接口或VE接口上不能执行portswitch命令。

执行本命令时，接口下除了shutdown、undo shutdown和description，不能有其它配置。

连续执行portswitch、undo portswitch命令的最小时间间隔是30秒。

使用实例

# 将接口GE1/0/0切换为二层模式。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
