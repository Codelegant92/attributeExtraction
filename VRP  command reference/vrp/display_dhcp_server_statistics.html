display dhcp server statistics

命令功能

使用display dhcp server statistics命令用来查看DHCP服务器的统计信息。

命令格式

display dhcp server statistics

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

在DHCP服务器配置成功后，使用display dhcp server statistics命令来查看服务器的统计信息：DHCP地址池数目，自动、手工和过期绑定的地址数目，不可识别的报文数量、DHCP请求和应答的报文数目统计等。

使用实例

# 查看DHCP服务器的统计信息。

<Quidway> display dhcp server statistics

该显示信息主要内容解释如下表：

表1 display dhcp server statistics命令输出信息描述

项目

描述

Global Pool:

全局地址池的统计信息

Interface Pool:

接口地址池的统计信息

Pool Number

地址池的数目

Auto

自动绑定的IP地址数

Manual

手工绑定的IP地址数

Expire

租约超期的IP地址数

Boot Request

DHCP客户端发给DHCP服务器的消息数量

Dhcp Discover, Dhcp Request, Dhcp Decline, Dhcp Release, Dhcp Inform

收到的各种DHCP报文的统计数目

Boot Reply

DHCP服务器发给DHCP客户端的消息数量

Dhcp Offer, Dhcp Ack, Dhcp Nak

发送的各种DHCP报文的统计数目

Bad Messages

错误的报文统计

BatchBackup send msg

批量备份时主控板发送的HA消息的数目

BatchBackup recv msg

批量备份时备板接受的HA消息的数目

BatchBackup send lease

批量备份时主控板发送的Lease的数目

BatchBackup recv lease

批量备份时备板接收的Lease的数目
