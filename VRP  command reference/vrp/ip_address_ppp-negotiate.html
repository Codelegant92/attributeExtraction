ip address ppp-negotiate

命令功能

ip address ppp-negotiate命令用来使能接口的IP地址协商功能。

undo ip address ppp-negotiate命令用来禁止IP地址协商功能。

缺省情况下，接口上不使能IP地址协商功能。

命令格式

ip address ppp-negotiate

undo ip address ppp-negotiate

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

ip address ppp-negotiate命令只能应用于封装了PPP协议的接口，实现使能接口的IP地址协商功能，从而可以利用PPP的地址协商功能，由接入服务器给接口分配IP地址。

使用实例

# 在接口POS1/0/0上使能IP地址协商。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ip address ppp-negotiate
