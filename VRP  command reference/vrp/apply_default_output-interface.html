apply default output-interface

命令功能

apply default output-interface命令用来设置报文的缺省发送接口。

undo apply default output-interface命令用来取消报文缺省发送接口的设置。

命令格式

apply default output-interface interface-type1 interface-number1 [ interface-type2 interface-number2 ]

undo apply default output-interface [ interface-type1 interface-number1 ] [ interface-type2 interface-number2 ]

参数说明

interface-type1，interface-type2：指定接口类型。

interface-number1，interface-number2：指定接口编号。

说明：

出接口不能为以太接口等广播型接口。

缺省级别

2：配置级

使用指南

该命令对在路由表中未查询到路由的报文起作用。

可以配置两个发送接口以达到负载分担的目的。

使用实例

# 设置报文的缺省发送接口为POS 1/0/0。

<Quidway> system-view
[Quidway] policy-based-route policy1 permit node 10
[Quidway-policy-based-route-policy1-10] apply default output-interface pos 1/0/0
