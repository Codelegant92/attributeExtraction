tunnel remote

命令功能

tunnel remote命令用来设置安全隧道的对端地址。

undo tunnel remote命令用来删除安全隧道的对端地址。

缺省情况下，没有配置安全隧道的远端地址。

命令格式

tunnel remote ip-address

undo tunnel remote [ ip-address ]

参数说明

ip-address：安全隧道的对端地址，格式为点分十进制格式。

视图

手工方式安全策略视图

缺省级别

2：配置级

使用指南

对于manual方式的安全策略，如果已经设置了对端地址，必须先删除原先的地址才能设置新的对端地址。该配置命令只应用在手工安全策略中。

对于isakmp方式的安全策略，不需要设置本端地址，该命令无效。

安全隧道是建立在本端和对端之间，在安全隧道的两端，均需要正确设置对端地址。

使用实例

# 设置安全隧道的对端地址为10.1.1.2。

<Quidway> system-view
[Quidway] ipsec policy shanghai 10 manual
[Quidway-ipsec-policy-shanghai-10] tunnel remote 10.1.1.2
