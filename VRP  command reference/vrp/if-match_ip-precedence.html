if-match ip-precedence

命令功能

if-match ip-precedence命令用来定义IP优先级匹配规则。

undo if-match ip-precedence命令用来删除IP优先级匹配规则。

命令格式

if-match [ not ] ip-precedence ip-precedence-value&<1-8>

undo if-match [ not ] ip-precedence

参数说明

ip-precedence-value：IP优先级的值，取值范围是0～7。

not：不匹配该类。

缺省级别

2：配置级

使用指南

命令在配置后，IP优先级的值将自动按照从小到大的顺序排序。

可以配置多个IP优先级值，最多可指定8个；如果指定了多个相同的IP优先级值，系统默认为一个；多个不同的IP优先级值是或的关系。配置时将需要配置的IP优先值用同一条命令进行配置，否则最后配置的if-match ip-precedence命令会覆盖前面的配置。

使用实例

# 定义类class1的匹配规则为：匹配IP优先级值为1或6的报文。

<Quidway> system-view
[Quidway] traffic classifier class1
[Quidway-classifier-class1] if-match ip-precedence 1 6
