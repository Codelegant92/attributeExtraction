interface-status-change mac-withdraw enable

命令功能

interface-status-change mac-withdraw enable命令用来使能PE在绑定VSI的AC接口状态发生变化时向所有PEER发送LDP MAC-Withdraw消息。

undo interface-status-change mac-withdraw enable命令用来去PE在绑定VSI的AC接口状态发生变化时向所有PEER发送LDP MAC-Withdraw消息。

缺省情况下，VSI不发送LDP MAC-Withdraw消息。

命令格式

interface-status-change mac-withdraw enable

undo interface-status-change mac-withdraw enable

参数说明

无

视图

VSI-LDP视图

缺省级别

2：配置级

使用指南

该命令用在城域以太网ME（Metro Ethernet）中，使与NPE相连的PE-AGG设备在NPE侧接口状态发生变化时向所有PEER发送LDP MAC-Withdraw消息。本端发送MAC-withdraw消息会导致远端VSI清除本地MAC并重新学习MAC。

执行命令使能后，如果发生如下事件，PE向所有PEER发送LDP MAC-Withdraw消息：
  • VSI绑定的接口状态由Down变为Up
  • VSI绑定的接口状态由Up变为Down

该命令目前只在NE40E/80E/5000E产品上支持。

使用实例

# 使能PE在绑定VSI的AC接口状态发生变化时向所有PEER发送LDP MAC-Withdraw消息。

<Quidway> system-view
[Quidway] vsi v1 static
[Quidway-vsi-v1] pwsignal ldp
[Quidway-vsi-v1-ldp] interface-status-change mac-withdraw enable
