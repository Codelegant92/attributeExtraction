igmp enable

命令功能

igmp enable命令用来在接口上使能IGMP。

undo igmp enable命令用来恢复缺省配置。

缺省情况下，接口上禁止IGMP。

命令格式

igmp enable

undo igmp enable

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

只有先启动组播，才能配置此命令。

使能IGMP前在接口上配置的其他IGMP参数只有在配置了此命令后才生效。

组播CAC功能与IGMP only-link接口互斥，因此，当路由器使能了组播CAC功能后，要想使能IGMP，必须先使能PIM-SM。

说明：

IGMP Only-link接口是指接口下使能了IGMP或配置了静态组播组，但没有使能PIM-SM。

使用实例

# 在接口GE1/0/0上使能IGMP。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] igmp enable
