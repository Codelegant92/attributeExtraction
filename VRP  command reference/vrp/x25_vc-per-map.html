x25 vc-per-map

命令功能

x25 vc-per-map命令用来设置同时连接同一目的设备的虚电路数目的最大值。

undo x25 vc-per-map命令用来恢复同时连接同一目的设备的虚电路数目的缺省值。

命令格式

x25 vc-per-map count

undo x25 vc-per-map

参数说明

count：虚电路最大数目，取值范围是1～8。虚电路最大数目缺省为1。

视图

接口视图

缺省级别

2：配置级

使用指南

如果设置该参数大于1，当虚电路的发送窗口和发送队列都满了以后，系统就会创建一条新的通向同一个目的地的虚电路；如果新的虚电路无法创建，则数据报将被丢弃。

执行完该命令后，必须重新启动接口，使配置生效。

使用实例

# 将X.25接口Serial1/0/0的虚电路的最大数目设置为3。

<Quidway>system-view
[Quidway]interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 vc-per-map 3
[Quidway-Serial1/0/0] shutdown
[Quidway-Serial1/0/0] undo shutdown
