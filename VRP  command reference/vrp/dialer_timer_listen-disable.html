dialer timer listen-disable

命令功能

dialer timer listen-disable命令用来配置延迟断开备份接口的时间。

undo dialer timer listen-disable命令用来恢复延迟断开备份接口的时间为缺省值。

命令格式

dialer timer listen-disable seconds

undo dialer timer listen-disable

参数说明

seconds：延迟断开备份接口的时间。取值范围是0～65535，单位是秒。缺省值是0秒（即立刻切断备份链路）。

使用实例

# 设置接口Serial1/0/0延迟断开备份接口的时间为5秒。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] dialer timer listen-disable 5
