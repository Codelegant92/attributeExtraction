display mpls statistics interface

命令功能

display mpls statistics interface命令用来查看使能了MPLS的接口的统计信息，包括每个使能了MPLS的接口接收和发送的字节数、报文数、错误数，丢弃报文数。

命令格式

display mpls statistics interface { all | interface-type interface-number } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：所有接口。

interface-type：接口类型。

interface-number：接口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果要查看统计信息，请先使用statistics interval命令设置上报统计信息的时间间隔。缺省的时间间隔为0，即不上报统计信息。这种情况下显示出来的各项统计信息都为0。

当统计信息中的字节数（Octets）、报文数（Packets）或错误数（Errors）超过显示允许的最大值0xFFFFFFFF时，统计信息将显示为“剩余数量＋n×4294967295”，其中，“n”表示统计信息时最大数量的n倍，“4294967295”是0xFFFFFFFF的十进制形式。

例如，当显示信息为“Octets: 2954218368 + 1 * 4294967295”时，表示实际字节数为7249185663。

说明：

目前只提供出方向的LSP统计功能，不提供入方向的LSP统计。
