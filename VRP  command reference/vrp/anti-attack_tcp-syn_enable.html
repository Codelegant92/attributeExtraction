anti-attack tcp-syn enable

命令功能

anti-attack tcp-syn enable命令用来使能TCP-Syn报文攻击防范。

undo anti-attack tcp-syn enable命令用来去使能TCP-Syn报文攻击防范。

缺省情况下，使能TCP-Syn报文攻击防范。

命令格式

anti-attack tcp-syn [ slot slot-number ] enable

undo anti-attack tcp-syn [ slot slot-number ] enable

参数说明

slot slot-number：接口板槽位号。如果不指定槽位号，缺省为所有单板使能TCP-Syn报文攻击防范。

视图

系统视图

缺省级别

2：配置级

使用指南

使能TCP-Syn报文攻击防范之后，设备就会针对TCP-Syn攻击报文进行防攻击检测，如果检测到TCP-Syn攻击报文，将采用CAR对TCP-Syn攻击报文进行速率限制，保证CPU的正常工作。

使用实例

# 在系统视图下使能TCP-Syn报文防攻击命令

<Quidway> system-view
[Quidway] anti-attack tcp-syn slot 6 enable
