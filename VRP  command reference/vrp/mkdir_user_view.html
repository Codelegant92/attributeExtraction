mkdir（用户视图）

命令功能

mkdir命令用来在指定存储设备指定目录下创建目录。

命令格式

mkdir directory

参数说明

directory：目录名，字符串形式，长度范围是1～15。

视图

用户视图

缺省级别

3：管理级

使用指南

创建的目录名不能与指定目录下其它目录或文件名重名。

该命令最大支持4级目录深度，每级目录名最大长度为15个字符。

使用实例

# 创建目录dd。

<Quidway> mkdir dd
Info:Create directory cfcard:/dd......Done..
