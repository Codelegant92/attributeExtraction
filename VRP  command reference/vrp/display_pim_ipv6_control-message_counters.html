display pim ipv6 control-message counters

命令功能

display pim ipv6 control-message counters命令用来查看发送和接收的IPv6 PIM控制消息的统计数。

命令格式

display pim ipv6 control-message counters [ message-type { assert | bsr | crp | graft | graft-ack | hello | join-prune | probe | register | register-stop | state-refresh } ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display pim ipv6 control-message counters interface interface-type interface-number [ message-type { assert | bsr | graft | graft-ack | hello | join-prune | state-refresh } ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface interface-type interface-number：指定接口类型和编号。

message-type：指定PIM消息类型。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定接口，将显示所有接口的IPv6 PIM控制消息统计数。

请参见IPv4 PIM配置命令。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示接口GE1/0/0发送和接收的各种IPv6 PIM控制消息的统计信息。

<Quidway> display pim ipv6 control-message counters interface gigabitethernet 1/0/0
