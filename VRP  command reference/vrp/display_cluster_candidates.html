display cluster candidates

命令功能

display cluster candidates命令用来查看候选交换机的信息。

命令格式

display cluster candidates [ mac-address mac-address | verbose ]

参数说明

mac-address mac-address：指定候选交换机的MAC地址，以查看此候选交换机的详细信息。mac-address是用16进制表示的候选交换机的MAC地址。格式为H-H-H，每个“H”代表4位16进制数。

verbose：显示所有候选交换机的详细信息。

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

不指定可选参数时，本命令将显示所有候选交换机的简要信息。

使用实例

# 显示全部候选交换机的详细信息。

<HUAWEI_0.Quidway> display cluster candidates verbose
 Hostname  : Quidway
 MAC       : 000b-09a6-e009
 Hop       : 1
 Platform  : NE40E
 IP        : 1.1.1.2/24

该显示信息主要内容解释见表1。

表1 display cluster candidates命令输出信息描述

项目

描述

Hostname

候选交换机的主机名。

MAC

候选交换机的MAC地址。

Hop

候选交换机距离命令交换机的跳数。

候选交换机的类型。

IP
