display rmon2 hlhostcontroltable

命令功能

display rmon2 hlhostcontroltable命令用来查看主机控制表中的信息。

命令格式

display rmon2 hlhostcontroltable [ index ctrl-index ]

参数说明

ctrl-index：索引值，唯一识别主机控制表中的行数。整数形式，取值范围是1～65535。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定行索引，将显示整个表的信息。

使用实例

# 查看主机控制表中的第一行信息。

<Quidway> display rmon2 hlhostcontroltable index 1

项目

描述

index

主机控制表索引

datasource

数据源接口

droppedframe

统计接口所接收到的、但不记入相关nlHost条目的帧的总数

inserts

nlHost条目插入nlHostTable数据表的次数

Deletes

nlHost条目从nlHostTable数据表删除的次数

maxentries

nlHostTable想要的最大条目数

owner

条目的所有者

status

主机控制表条目的状态
