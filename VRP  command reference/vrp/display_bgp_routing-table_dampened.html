display bgp routing-table dampened

命令功能

display bgp routing-table dampened命令用来显示BGP衰减的路由。

命令格式

display bgp [ vpnv4 vpn-instance vpn-instance-name ] routing-table [ statistics ] dampened [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp ipv6 routing-table [ statistics ] dampened [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 vpn6-instance vpn6-instance-name routing-table [ statistics ] dampened [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpnv4：查看VPNv4的BGP路由信息。

vpn-instance vpn-instance-name：指定VPN实例名称。

ipv6：查看所有衰减的IPv6路由信息。

statistics：查看衰减路由的统计信息。

vpnv6：查看VPNv6的BGP路由信息。

vpn6-instance vpn6-instance-name：指定IPv6 VPN实例名称。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
