isis smart-hello

命令功能

isis smart-hello命令用来使能Smart-Hello功能。

undo isis smart-hello命令用来取消此功能。

缺省情况下，不使能Smart-Hello功能。

命令格式

isis smart-hello

undo isis smart-hello

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

Hello报文的处理模块分布在主控板和接口板上，使能Smart-Hello功能后，发送和接收Hello报文的处理主要由接口板来完成。

Smart-Hello功能只能在接口板上使能。

说明：
Smart-Hello功能与BFD功能不能同时配置。

使用实例

# 使能Smart-Hello功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] isis smart-hello
