ssh server timeout

命令功能

ssh server timeout命令用来设置SSH连接认证超时时间。

undo ssh server timeout命令用来恢复SSH连接认证超时时间的缺省值。

缺省情况下，SSH连接认证超时时间为60秒。

命令格式

ssh server timeout seconds

undo ssh server timeout

参数说明

seconds：指定SSH连接认证超时时间，取值范围1～120，单位为秒。

使用指南

设置的SSH连接认证超时时间在下次登录时生效。

使用实例

# 设定登录超时时间为80秒

[Quidway] ssh server timeout 80
