display current-configuration slave

命令功能

display current-configuration slave 命令用来查看备板配置信息。

命令格式

display current-configuration slave [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令只能在主控板执行。

正常情况下显示备板配置信息，如果有异常发生，例如在搜集备板配置信息时拔出备板或者进行了其他操作，系统输出等待提示，并且输出命令操作失败提示。

使用实例

# 显示备板配置信息。

<Quidway> display current-configuration slave
