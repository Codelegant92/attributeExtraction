channel-set（CT1接口视图）

命令功能

channel-set命令用来对CT1接口进行时隙捆绑。

undo channel-set命令用来取消指定的时隙捆绑。

命令格式

channel-set set-number timeslot-list slot-list [ speed { 56k | 64k } ]

undo channel-set set-number

参数说明

set-number：在CT1接口上时隙捆绑形成的接口的索引号。整数形式，取值范围是0～23。

slot-list：用于捆绑的时隙编号或时隙范围。整数形式，取值范围是0～23。指定捆绑时隙时，可以指定单个时隙，也可以指定时隙范围。使用“,”分隔多个时隙，使用“-”表示时隙范围。

speed：配置时隙捆绑的方式。选用参数56k时，捆绑方式为N×56kbit/s；选用参数64k时，捆绑方式为N×64kbit/s。

如果不指定速率，缺省采用56kbit/s。

缺省级别

2：配置级

使用指南

CT1接口有两种工作模式：

  • 当工作在通道化模式下时，CT1接口在物理上分为24个时隙，可以任意地将24个时隙分成若干组，每组时隙捆绑以后作为一个接口（channel-set）使用，其逻辑特性与同步串口相同，使用interface serial命令进入捆绑后形成的接口的视图。
  • 当工作在净通道模式下时，整个接口不再时隙化，不支持channel-set命令，而是作为一个1.544M带宽的串口使用。

使用实例

# 将CT1接口1/0/0的1、2、10-15时隙捆绑为1号channel-set。

<Quidway> system-view
[Quidway] controller t1 1/0/0
[Quidway-T1 1/0/0] channel-set 1 timeslot-list 1,2,10-15 speed 64k
