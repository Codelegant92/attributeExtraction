firewall enable

命令功能

firewall enable命令用来启用防火墙功能， undo firewall enable命令用来禁止防火墙功能。

缺省情况下，禁止防火墙功能。

命令格式

firewall enable { all | slot slot-id }

undo firewall enable

参数说明

enable：允许防火墙功能。

all：配置作用于所有接口板。

slot-id：为指定的接口板槽号。

视图

系统视图

缺省级别

2：配置级

使用指南

设备在启动防火墙功能后，再进行相关的配置，可以有效的避免网络遭受攻击。

使用实例

# 对所有接口板启用防火墙。

<Quidway> system-view
[Quidway] firewall enable all
