patch load（多框）

命令功能

patch load命令用来按照设备方式加载补丁。所有在位单板，加载补丁包中的合适补丁。

命令格式

patch load filename [ all [ active | run ] | chassis chassis-number | slave | slot slot-id ]

参数说明

filename：补丁包文件名。

all：所有单板（包括主控板和接口板）。

active：加载补丁后直接激活。

run：加载补丁后直接运行。

chassis chassis-number：转发框的框号。

slave：系统备用板。

slot slot-id：板号（输入的形式是框类型＋框号＋板号）。

使用指南

如果补丁包中没有适合某个单板类型的补丁，则不加载补丁。

使用实例

# 在所有的板上加载补丁。

<Quidway> patch load cfcard:/ NE5000EV200R005SPH001.pat
Info: Succeeded in loading the patch on the master board....
Info: Finished loading the patch.
