mpls oam egress enable

命令功能

mpls oam egress enable命令用来在LSP出节点启动指定的MPLS OAM功能。

undo mpls oam egress enable命令用来在LSP出节点停止指定的MPLS OAM功能，命令执行后，所配置的MPLS OAM参数仍保留。

命令格式

mpls oam egress enable { all | lsp-name lsp-name | lsr-id ingress-lsr-id tunnel-id tunnel-id }

undo mpls oam egress enable { all | lsp-name lsp-name | lsr-id ingress-lsr-id tunnel-id tunnel-id }

参数说明

all：启动所有以此节点为Egress的LSP的MPLS OAM功能。

lsp-name lsp-name：指定被检测LSP名称。lsp-name为字符串形式，不支持空格，长度范围是1～19。

lsr-id ingress-lsr-id：指定被检测LSP的入节点的IP地址。

tunnel-id tunnel-id：指定被检测LSP的隧道ID。tunnel-id为整数形式，取值范围取决于产品的许可证文件。

使用指南

MPLS OAM参数配置完毕后并不会自动生效，需要使用mpls oam egress enable命令启动。

如果配制了OAM自动协议扩展，则无须本命令启动OAM配置。

使用实例

# 启动名为egress-001的LSP的MPLS OAM功能。

<Quidway> system-view
[Quidway] mpls oam egress enable lsp-name egress-001
