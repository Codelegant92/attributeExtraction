http-operation

命令功能

http-operation命令用来配置NQA测试HTTP测试例的测试类型。

缺省情况下，HTTP类型的测试例的操作类型是GET。

命令格式

http-operation { get | post }

参数说明

get：指定HTTP测试执行GET操作。

post：指定HTTP测试执行POST操作。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对HTTP类型的测试例配置HTTP操作类型。

不能更改正在运行的HTTP测试例的操作类型。

使用实例

# 设置测试例user test的HTTP测试操作类型为get。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type http
[Quidway-nqa-user-test] http-operation get
