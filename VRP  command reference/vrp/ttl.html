ttl

命令功能

ttl命令用来配置NQA测试例报文中的TTL值。

undo ttl命令用来取消恢复缺省配置。

缺省情况下，TTL值为30。

命令格式

ttl number

undo ttl

参数说明

number：TTL值。取值范围是1～255。

视图

NQA视图

缺省级别

2：配置级

使用指南

不能对DHCP、DNS、Trace、LSP Trace、path Jitter、path MTU和PWE3Trace类型的测试例配置TTL值。

不能修改正在运行的测试例的此项配置。

使用实例

# 设置测试例user test的报文TTL值为10。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type icmp
[Quidway-nqa-user-test] ttl 10
