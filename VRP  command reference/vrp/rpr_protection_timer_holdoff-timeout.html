rpr protection timer holdoff-timeout

命令功能

rpr protection timer holdoff-timeout命令用来设置本地节点Span的故障延时上报时间。

undo rpr protection timer holdoff-timeout命令用来恢复缺省值。

缺省情况下，本地节点Span的故障延时上报时间为0毫秒，即本地节点Span出现故障立刻上报，不延时。

命令格式

rpr protection timer holdoff-timeout value { east | west }

undo rpr protection timer holdoff-timeout { east | west }

参数说明

value：时间值。整数形式，取值范围是0～200，单位是毫秒，缺省值的是0毫秒。该时间值必须是10的整数倍，如0，10，20等。

east：本地节点的东向连接。

west：本地节点的西向连接。

使用指南

如果网络不太稳定或者时延要求比较低，可以把时间设置的长些。

使用实例

# 配置本地节点Span的故障延时上报时间为20毫秒

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr protection timer holdoff-timeout 20 east
