display vrrp binding member-interface

命令功能

display vrrp binding member-interface命令用来显示业务接口与管理VRRP备份组的绑定关系。

命令格式

display vrrp binding admin-vrrp [ interface interface-type interface-number ] [ vrid virtual-router-id ] member-interface [ interface interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

admin-vrrp interface interface-type interface-number：配置了管理VRRP的接口类型和编号。

vrid virtual-router-id：VRRP备份组号。十进制整数形式，取值范围是1～255。

member-interface interface interface-type interface-number：业务成员接口类型和编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display vrrp binding member-interface命令时，如果不指定管理VRRP的接口名称则显示所有与管理VRRP绑定的业务成员接口信息。

使用实例

# 显示备份组号为1的管理VRRP和业务接口GE1/0/0.1的绑定信息。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ip address 10.1.1.1 24
[Quidway-GigabitEthernet1/0/0] vrrp vrid 1 virtual-ip 10.1.1.111
[Quidway-GigabitEthernet1/0/0] admin-vrrp vrid 1
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] track admin-vrrp interface gigabitethernet1/0/0 vrid 1
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] display vrrp binding admin-vrrp interface gigabitethernet 1/0/0 vrid 1 member-interface interface gigabitethernet 1/0/0.1
interface: GigabitEthernet1/0/0,  admin-vrrp vrid: 1,state:master
  member-interface number: 1
    interface: Gigabitethernet1/0/0.1, state: up

项目

描述

interface

配置了管理VRRP的接口类型和编号

admin-vrrp vrid

管理VRRP的备份组号

state
管理VRRP的状态：

  • master：主用状态

  • slave：备用状态

  • initialize：初始化状态

member-interface number

绑定的业务成员接口的数量

interface

绑定的业务成员接口的类型和编号

state
绑定的业务成员接口的状态：

  • Up

  • Down
