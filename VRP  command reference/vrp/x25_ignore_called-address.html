x25 ignore called-address

命令功能

x25 ignore called-address命令用来允许X.25在发起呼叫时省略被叫DTE的X.121地址。

undo x25 ignore called-address命令用来禁止X.25在发起呼叫时省略被叫DTE的X.121地址。

缺省情况下，禁止该功能。

命令格式

x25 ignore called-address

undo x25 ignore called-address

参数说明

无

接口视图

缺省级别

使用指南

X.25协议规定，呼叫请求分组必须携带地址码组。但是在具体网络环境中或者因为应用的需要，在某些时候，X.25的呼叫请求也可以不携带被叫DTE地址或主叫DTE地址。该命令可以方便地指定X.25在发送的呼叫请求分组中是否携带被叫DTE地址。

使用实例

# 指定X.25接口Serial1/0/0在发送呼叫请求分组时不携带被叫DTE地址。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 ignore called-address
