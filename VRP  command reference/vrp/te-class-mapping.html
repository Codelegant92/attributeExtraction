te-class-mapping

命令功能

te-class-mapping命令用来创建TE-Class映射表或进入TE-Class Mapping视图。

undo te-class-mapping命令用来删除配置的TE-Class映射表。

缺省情况下，DS-TE模式为非标准（Non-IETF）模式。

命令格式

te-class-mapping

undo te-class-mapping

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

TE-Class映射表只在标准模式（IETF模式）下生效。Non-IETF模式下也可以配置TE-Class映射表，但不生效。

配置TE-Class映射表前，需使用该命令创建TE-Class映射表。

当在系统视图下配置undo mpls命令或在MPLS视图下配置undo mpls te命令后，配置的TE-Class映射表会被删除。
