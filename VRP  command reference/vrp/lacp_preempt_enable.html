lacp preempt enable

命令功能

lacp preempt enable命令用来使能静态LACP模式下LACP优先级抢占的功能。

undo lacp preempt enable命令用来禁止静态LACP模式下LACP优先级抢占功能。

缺省情况下，优先级抢占处于禁止状态。

命令格式

lacp preempt enable

undo lacp preempt enable

参数说明

无

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

如果当前链路聚合模式不是静态LACP模式，lacp preempt enable命令不可用。要更改工作模式为静态LACP模式，请在Eth-Trunk接口视图下执行mode lacp-static命令。

在静态LACP模式下，当活动链路中出现故障链路时，系统会从备用链路中选择优先级最高的链路替代故障链路，经过一段时间被替代的故障链路恢复了正常，而且该链路的优先级又高于替代自己的链路。这种情况下，如果执行了lacp preempt enable命令，这条链路就可以回切到活动状态。

使能了优先级抢占后，系统会通过系统优先级和接口优先级来重新选择活动接口。

不使能优先级抢占，系统不会进行活动接口的选择过程。

使用实例

# 使能Eth-Trunk1优先级抢占。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] lacp preempt enable
