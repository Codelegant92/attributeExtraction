fr-class

命令功能

fr-class命令用来将帧中继class与当前帧中继虚电路或帧中继接口关联起来。

undo fr-class命令用来取消帧中继class和帧中继虚电路或帧中继接口的关联。

缺省情况下，没有帧中继class与帧中继虚电路或帧中继接口相关联。

命令格式

fr-class class-name

undo fr-class class-name

参数说明

class-name：帧中继class的名称，字符串形式，长度为1～31。

视图

帧中继接口视图、MFR接口视图、DLCI视图

缺省级别

2：配置级

使用指南

假如指定的帧中继class不存在，此命令会先创建一个帧中继class，然后将此帧中继class和当前虚电路或接口关联起来。假如指定的帧中继class存在，此命令只会将此帧中继class和当前虚电路或接口关联，不会创建新的帧中继class。

本命令的undo形式只取消指定的帧中继class和虚电路或接口的关联，并不删除实际的帧中继class。如果要删除帧中继class，请使用undo fr calss命令。

将一个帧中继class和接口关联起来之后，此接口上的所有虚电路都会继承此帧中继class的帧中继QoS参数。

使用实例

# 将名为test1的帧中继class与DLCI为200的帧中继虚电路关联起来。

<Quidway> system-view
[Quidway] interface serial 4/0/1
[Quidway-Serial4/0/1] fr dlci 200
[Quidway-fr-dlci-Serial4/0/1-200] fr-class test1
