user privilege

命令功能

user privilege命令用来配置从当前用户界面登录的用户所能访问的命令级别。

undo user privilege命令用来取消当前设置。

缺省情况下，Console口用户界面对应的默认命令访问级别是3，而其他用户界面对应的默认级别是0。

命令格式

user privilege level level

undo user privilege level

参数说明

level：指定命令级别。整数形式，取值范围是0～15。

视图

用户界面视图

缺省级别

3：管理级

使用指南

如果用户界面下配置的命令级别访问权限与用户名本身对应的操作权限冲突，以用户名本身对应的级别为准。

例如：用户001本身对应权限为能访问3级命令，而用户界面VTY 0配置的命令级别访问权限为2，则用户001从VTY0登录系统时，能访问3级及以下的命令。

使用实例

# 设置从vty 0登录的用户级别为2。

<Quidway > system-view
[Quidway] user-interface vty 0
[Quidway-ui-vty0] user privilege level 2

# 通过Telnet方式从vty 0登录进路由器后，查看用户界的详细信息。

<Quidway> display user-interface vty0
Idx  Type     Tx/Rx      Modem Privi ActualPrivi Auth  Int     
+ 34   VTY 0               -     3     3           N     -       

  +    : Current user-interface is active.
  F    : Current user-interface is active and work in async mode.
  Idx  : Absolute index of user-interface.
  Type : Type and relative index of user-interface.
  Privi: The privilege of user-interface.
  ActualPrivi: The actual privilege of user-interface.
  Auth : The authentication mode of user-interface.
      A: Authenticate use AAA.
      N: Current user-interface need not authentication.
      P: Authenticate use current UI's password.
  Int  : The physical location of UIs.
