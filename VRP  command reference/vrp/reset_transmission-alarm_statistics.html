reset transmission-alarm statistics

命令功能

reset transmission-alarm statistics命令用来清除告警统计信息。

命令格式

reset transmission-alarm statistics

参数说明

无

视图

接口视图

缺省级别

3：管理级

使用指南

支持该命令的接口类型有ATM、POS、CPOS、E1/CE1、T1/CT1、E3/CE3、T3/CT3、10GE-WAN、基于SDH和SONET的RPR接口。设备上支持的具体接口类型由产品决定。

执行reset transmission-alarm statistics命令可以清空该接口上所有告警的统计信息。

使用实例

# 清除POS1/0/0接口的告警统计信息。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] reset transmission-alarm statistics
