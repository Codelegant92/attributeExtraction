description(IPv6 VPN实例视图)

命令功能

description命令用来配置当前IPv6 VPN实例的描述信息。

undo description命令用来删除当前IPv6 VPN实例的描述信息。

命令格式

description description-information

undo description

参数说明

description-information：IPv6 VPN实例的描述信息。字符串形式，支持空格，最大长度是242个字符。

视图

IPv6 VPN实例视图

缺省级别

2：配置级

使用指南

如果多次配置description命令，最后一次配置生效。

使用实例

# 对名为vpn-red的IPv6 VPN实例配置描述信息。

<Quidway> system-view
[Quidway] ipv6 vpn6-instance vpn-red
[Quidway-vpn6-instance-vpn-red] description This VPN is used for company-red

# 删除名为vpn-red的VPN实例的描述信息。

[Quidway-vpn6-instance-vpn-red] undo description
