reset dhcp snooping bind-table

命令功能

reset dhcp snooping bind-table命令用来根据不同的参数删除DHCP Snooping绑定表中的表项。

命令格式

reset dhcp snooping bind-table { vlan vlan-id [ interface interface-type interface-number ] | static | dynamic | vsi vsi-name }

参数说明

vlan-id：VLAN ID，取值范围1～4094。

interface-type interface-number：接口类型、接口号。此参数仅在VLAN视图下可配置。

vsi vsi-name：虚拟交换实例名。字符串形式，不支持空格，长度范围是1～31。

缺省级别

3：管理级

使用指南

reset dhcp snooping bind-table 命令用来清除该DHCP Snooping绑定表的所有配置参数，执行该命令用来删除接口、VLAN以及VLAN内接口上绑定的动态和静态DHCP Snooping绑定表，即恢复到DHCP Snooping绑定表启动时的状态。

在强策略下执行该命令将导致与DHCP Snooping绑定表中IP地址对应的用户断线。

使用实例

# 删除DHCP Snooping绑定表的所有静态表项。

<Quidway> system-view
[Quidway] reset dhcp snooping bind-table static

# 删除指定VSI下DHCP Snooping绑定表中有关VSI的所有表项。

<Quidway> system-view
[Quidway] reset dhcp snooping bind-table vsi vsi1
