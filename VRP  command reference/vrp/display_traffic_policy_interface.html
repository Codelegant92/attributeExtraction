display traffic policy interface

命令功能

display traffic policy interface命令用来显示指定接口、指定ATM接口的指定PVC或所有接口与PVC上策略的配置信息和运行情况。

命令格式

display traffic policy interface [ interface-type interface-number [ inbound | outbound ] [ pvc { pvc-name [ vpi/vci ] | vpi/vci } [ user-defined ] ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type：接口类型。

interface-number：接口号。

pvc：只用于ATM接口，即可显示指定ATM接口上的指定pvc的策略配置。

pvc-name：PVC名。

vpi/vci：VPI/VCI值对。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display traffic policy interface命令可以观察到的信息包括：

  • 策略应用的方向

  • 应用的策略名

  • 策略中的类

  • 策略中为类关联的动作

对标记MPLS EXP报文数的统计值统计的是路由器对数据包的临时标记，最终是否标记在MPLS EXP域，取决于该数据包从路由器输出时是否是MPLS报文。
