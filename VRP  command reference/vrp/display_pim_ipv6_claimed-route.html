display pim ipv6 claimed-route

命令功能

display pim ipv6 claimed-route命令用来查看IPv6 PIM使用的单播路由信息。

命令格式

display pim ipv6 claimed-route [ ipv6-source-address ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv6-source-address：指定组播源的IPv6地址。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

请参见IPv4 PIM配置命令。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示到达组播源2001::2使用的单播路由。

<Quidway> display pim ipv6 claimed-route 2001::2
VPN-Instance: public net
