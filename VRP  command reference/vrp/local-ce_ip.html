local-ce ip

命令功能

local-ce ip命令用来在PE上配置与其相连的CE的以太接口的IP地址。

undo local-ce ip命令用来删除配置。

缺省情况下，PE上没有配置与其相连的CE的以太接口的IP地址。

命令格式

local-ce ip ip-address

undo local-ce ip

参数说明

ip-address：与本接口相连的CE的以太网主接口或子接口的IP地址。

视图

VE接口视图、Ethernet接口视图、Ethernet子接口视图、GE接口视图、GE子接口视图、Eth-Trunk接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

在接口绑定VLL的情况下，执行local-ce ip命令后，需要在接口执行命令shutdown和undo shutdown才能使本命令生效。

执行local-ce ip命令后，PE在向CE发送通过L2VPN隧道传输过来的IP报文时，如果没有用local-ce mac静态配置MAC地址，也没有从CE学到动态MAC地址，将使用静态配置的IP地址作为向CE发送ARP请求报文的目的IP地址。

缺省情况下，动态学到的MAC地址20分钟后会根据探测CE是否还在的结果决定是否删除该ARP表项。

使用实例

# 配置与GE1/0/0.1相连的CE的以太子接口的IP地址为197.1.1.1。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] local-ce ip 197.1.1.1
[Quidway-GigabitEthernet1/0/0.1] shutdown
[Quidway-GigabitEthernet1/0/0.1] undo shutdown
