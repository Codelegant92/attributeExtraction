hello-option holdtime（IPv4）

命令功能

hello-option holdtime命令用来配置路由器等待接收其PIM邻居发送Hello消息的超时时间。

undo hello-option holdtime命令用来恢复配置参数的缺省值。

缺省值是105秒。

命令格式

hello-option holdtime interval

undo hello-option holdtime

参数说明

interval：表示路由器等待接收其PIM邻居发送Hello消息的超时时间。整数形式，取值范围是1～65535，单位是秒。

公网实例PIM视图、VPN实例PIM视图

缺省级别

2：配置级

使用指南

该命令对于PIM-DM和PIM-SM都有效。

在超时时间内路由器未收到PIM邻居发来的Hello消息，则认为邻居失效或不可达。

该超时时间应该大于Hello消息的发送间隔。使用timer hello命令可以配置Hello消息的发送间隔。

使用实例

# 在公网实例中，配置路由器等待PIM邻居的Hello消息的超时时间为120s。

<Quidway> system-view
[Quidway] pim
[Quidway-pim] hello-option holdtime 120
