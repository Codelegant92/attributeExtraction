force-x21

命令功能

force-x21命令用来设置串口对X.21电缆的识别。

undo force-x21命令用来取消该功能。

缺省情况下，串口不能自动识别X.21电缆，对于X.21电缆，将识别为V.35类型。

命令格式

force-x21

undo force-x21

参数说明

无

视图

串行接口视图

缺省级别

2：配置级

使用指南

说明：

由其它接口板通道化成的serial接口不支持force-x21命令。

执行了命令force-x21后，如果将电缆更换为V.35类型，应使用undo force-x21命令取消对X.21电缆的识别；如果将电缆更换为V.24类型，可以不取消该功能。

使用实例

# 设置串口1/0/0能够识别X.21电缆。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] force-x21
