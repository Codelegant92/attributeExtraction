restart（NQA视图）

命令功能

restart命令将停止当前正在执行的测试例，重新启动一条NQA测试例。

命令格式

restart

参数说明

无

视图

NQA视图

缺省级别

2：配置级

使用指南

执行restart命令的效果相当于执行start now命令。

使用实例

# 重新启动测试例user test。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] restart
