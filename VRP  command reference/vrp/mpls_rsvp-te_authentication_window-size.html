mpls rsvp-te authentication window-size

命令功能

mpls rsvp-te authentication window-size命令用来配置RSVP认证消息滑窗大小。

undo mpls rsvp-te authentication window-size命令用来删除RSVP认证消息滑窗大小的配置。

命令格式

mpls rsvp-te authentication window-size window-size

undo mpls rsvp-te authentication window-size

参数说明

window-size：消息滑窗大小，整数形式，取值范围是1～64，缺省值为1。

接口视图

缺省级别

2：配置级

使用指南

在网络条件比较差，可能造成报文乱序的情况下，建议使用mpls rsvp-te authentication window-size命令配置RSVP认证消息滑动窗口。推荐将窗口大小配置为大于32，因为如果滑动窗口设置过小，则可能造成RSVP认证消息被丢弃，从而引发RSVP邻居关系终止。

执行该命令前，应先在接口启动验证功能。

执行undo mpls rsvp-te authentication window-size命令后，本地消息滑窗大小为1，所有乱序的RSVP认证消息会被丢弃。

使用实例

# 配置RSVP认证消息滑窗大小为64。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] mpls rsvp-te authentication cipher 123456789
[Quidway-GigabitEthernet1/0/0] mpls rsvp-te authentication window-size 64
