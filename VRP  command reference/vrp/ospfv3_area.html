ospfv3 area

命令功能

ospfv3 area命令用来在接口上使能OSPFv3的进程，并指定所属区域。

undo ospfv3 area命令用来将接口去使能OSPFv3。

缺省情况下，接口上不使能OSPFv3协议。

命令格式

ospfv3 process-id area area-id [ instance instance-id ]

undo ospfv3 process-id area area-id [ instance instance-id ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

area-id：区域的标识，可以是十进制整数（取值范围是0～4294967295）或IPv4地址格式。

instance instance-id：指定接口所属的实例ID。整数形式，取值范围是0～255，缺省值是0。
