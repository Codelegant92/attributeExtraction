bert（T3接口视图）

命令功能

bert命令用来启动T3的比特错码率检测功能。

undo bert命令用来停止T3的比特错码率检测。

缺省情况下，不进行比特错码率检测。

命令格式

bert { 2^11 | 2^15 | 2^20 | 2^20_151 | 2^23 } timer display interval-time

undo bert

参数说明

2^11：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为211，O.153或O.152模式。

2^15：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为215，O.151模式。

2^20：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为220，O.153模式。

2^20_151：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为220，O.151模式，QRSS＝1。（QRSS：Quasi-Random Signal Sequence）。

2^23：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为223，O.151模式。

interval-time：指定BERT的显示周期。取值范围是1～4000，单位是分钟。

使用指南

当需要检测链路质量或定位错误时，可以启动比特错码率检测功能BERT（Bit Error Rate Testing）。此功能在非通道化模式下使用。

使用实例

# 在T3接口1/0/0上启动BERT，采用伪随机序列长度215，显示周期10分钟。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] bert 2^15 timer display 10
