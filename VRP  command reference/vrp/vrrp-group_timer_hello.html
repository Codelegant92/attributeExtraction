vrrp-group timer hello

命令功能

vrrp-group timer hello time-interval 命令用来配置VRRP管理组Hello报文发送时间间隔。

undo vrrp-group timer hello命令用来恢复VRRP管理组Hello报文发送时间间隔的默认值。

命令格式

vrrp-group timer hello time-interval

undo vrrp-group timer hello

参数说明

time-interval：VRRP管理组发送Hello报文时间间隔，单位是毫秒，取值范围是200～60000，缺省值是1000。

视图

VRRP管理组视图

缺省级别

2：配置级

使用指南

同一VRRP管理组的Hello报文发送时间间隔必须相同。

使用实例

# 为VRRP管理组1配置Hello报文发送定时间隔为1000ms。

<Quidway> system-view
[Quidway] vrrp-group 1
[Quidway-vrrpgroup-1] vrrp-group timer hello 1000
