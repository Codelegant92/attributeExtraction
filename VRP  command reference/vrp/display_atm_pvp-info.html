display atm pvp-info

命令功能

display atm pvp-info命令用来显示PVP的配置信息。

命令格式

display atm pvp-info [ interface atm interface-number [ pvp vpi ] ]

参数说明

interface-number：接口编号。

vpi：指定VPI值。整数形式，取值范围是0～4294967295。

视图

所有视图

缺省级别

1：监控级

使用指南

本命令只在ATM信元透传时使用。

使用实例

# 显示设备中所有ATM接口下PVP的配置信息。

<Quidway> display atm pvp-info
VPI | STATE | PVP-NAME             | INDEX | ENCAP | PORT | INTERFACE
--------------------------------------------------------------------
10  |   UP  |                      | 0     |  AAL0 | None | Atm2/0/0.1(UP)
表1 display atm pvp-info命令输出信息描述

项目

描述

VPI

ATM子接口配置的VPI值。

STATE

当前PVP的状态。

PVP-NAME

PVP名称。

INDEX

PVP的索引号。

ENCAP

AAL的封装类型。

PROT
应用方式。

  • IP表示IPoA。

  • ETH表示PPPoEoA或IPoEoA。

  • PPP表示PPPoA。

INTERFACE

PVP所在的ATM接口。
