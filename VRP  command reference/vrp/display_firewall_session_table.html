display firewall session table

命令功能

display firewall session table命令显示会话表项的相关信息。

命令格式

display firewall session table [ vpn-instance vpn-instance-name ] [ verbose ] slot slot-id [ source { inside ip-address | global ip-address } ] [ destination { inside ip-address | global ip-address } ]

参数说明

slot-id：为指定的接口板槽号。

vpn-instance-name：VPN实例名。

ip-address：相应的IP地址。

verbose：显示表项详细信息。

使用指南

配置了NAT后，可以通过该命令查看地址的转换过程。
