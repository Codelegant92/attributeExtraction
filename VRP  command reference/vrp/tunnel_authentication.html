tunnel authentication

命令功能

tunnel authentication命令用来启用L2TP的隧道验证功能。

undo tunnel authentication命令用来取消L2TP隧道验证功能。

缺省情况下，系统对L2TP隧道进行验证。

命令格式

tunnel authentication

undo tunnel authentication

参数说明

无

视图

L2TP组视图

缺省级别

2：配置级

使用指南

为安全起见，隧道两端都需要对对方进行验证。如果为了进行网络的连通性测试或接收不知名对端发起的连接，可以不进行隧道验证。

使用实例

# 设置不验证隧道对端。

<Quidway> system-view
[Quidway] l2tp-group 1
[Quidway-l2tp1] undo tunnel authentication
