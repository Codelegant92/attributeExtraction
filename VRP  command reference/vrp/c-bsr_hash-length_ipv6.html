c-bsr hash-length（IPv6）

命令功能

c-bsr hash-length命令用来配置C-BSR的全局性哈希掩码长度。

undo c-bsr hash-length命令用来恢复该配置参数的缺省值。

缺省情况下，C-BSR的全局性哈希掩码长度是126。

命令格式

c-bsr hash-length hash-length

undo c-bsr hash-length

参数说明

hash-length：表示该C-BSR的全局性哈希掩码长度。整数形式，取值范围是0～128。

缺省级别

2：配置级

使用指南

全局性哈希掩码长度，适用于该路由器上配置的所有C-BSR。如果需要更改某接口上的C-BSR哈希掩码长度，请使用c-bsr interface-type interface-number [ hash-length [ priority ] ]命令。

c-bsr interface-type interface-number [ hash-length [ priority ] ]命令只对指定接口有效。系统优先采用c-bsr interface-type interface-number [ hash-length [ priority ] ]命令的配置。当c-bsr interface-type interface-number [ hash-length [ priority ] ]命令中没有指定哈希掩码长度时，采用c-bsr hash-length hash-length命令的配置值。

使用实例

# 在公网实例下，配置C-BSR的全局性哈希掩码长度为16。

<Quidway> system-view
[Quidway] pim-ipv6
[Quidway-pim6] c-bsr hash-length 16
