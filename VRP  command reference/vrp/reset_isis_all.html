reset isis all

命令功能

reset isis all命令用来重启IS-IS进程。

命令格式

reset isis all [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ]

参数说明

process-id：指定要清除的IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程（对于缺省的VPN实例，该参数可选）。vpn-instance-name为IPv4 VPN实例的名称，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程（对于缺省的VPN实例，该参数可选）。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

使用指南

本命令用在某些需要立即刷新LSP的情况下。

警告：

重启IS-IS进程可能导致业务中断，请谨慎使用该命令。

使用实例

# 重启IS-IS进程。

<Quidway> reset isis all
