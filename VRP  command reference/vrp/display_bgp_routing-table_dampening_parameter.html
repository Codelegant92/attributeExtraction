display bgp routing-table dampening parameter

命令功能

display bgp routing-table dampening parameter命令用来显示已配置的BGP路由衰减参数。

命令格式

display bgp routing-table dampening parameter [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 vpn-instance vpn-instance-name routing-table dampening parameter [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp ipv6 routing-table dampening parameter [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 vpn6-instance vpn6-instance-name routing-table dampening parameter [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpnv4：查看VPNv4的BGP路由衰减参数。

vpn-instance vpn-instance-name：指定VPN实例的路由衰减参数。

ipv6：查看配置的BGP4+路由衰减参数。

vpnv6：查看VPNv6的BGP路由衰减参数。

vpn6-instance vpn6-instance-name：指定VPNv6实例的路由衰减参数。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示BGP路由衰减参数。

<Quidway> display bgp routing-table dampening parameter
 Maximum Suppress Time(in second) : 3973
 Ceiling Value                    : 16000
 Reuse Value                      : 750
 HalfLife Time(in  second)        : 900
 Suppress-Limit                   : 2000
表1 display bgp routing-table dampening parameter命令输出信息描述

项目

描述

Maximum Suppress Time

最大抑制时间，单位是秒

Ceiling Value

惩罚上限值

Reuse Value

路由解除抑制状态的阈值

HalfLife Time

可达路由的半衰期，单位是秒

Suppress-Limit

路由进入抑制状态的阈值
