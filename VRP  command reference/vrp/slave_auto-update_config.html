slave auto-update config

命令功能

slave auto-update config命令用来在主用板写配置时自动地将配置数据同步到备用板保存。

undo slave auto-update config命令用来禁止此项功能。

缺省情况下，自动同步配置。

命令格式

slave auto-update config

undo slave auto-update config

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

配置slave auto-update config命令后，如果用户执行命令save保存配置，那么备板也会保存该配置。

使用实例

# 本例把保存在主板上的配置数据，同步到备用板，即备用板也保存这些配置数据。

<Quidway> system-view
[Quidway] slave auto-update config
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ip address 1.1.1.1 255.0.0.0
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] quit
<Quidway> save
