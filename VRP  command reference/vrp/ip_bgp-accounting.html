ip bgp-accounting

命令功能

ip bgp-accounting命令用来在接口上使能BGP Accounting（BGP报文统计）功能。

undo ip bgp-accounting命令用来取消该统计功能。

缺省情况下，没有使能BGP Accouting功能。

命令格式

ip bgp-accounting { inbound | outbound } * [ source ]

undo ip bgp-accounting

参数说明

inbound：在接口的入方向上进行流量统计。

outbound：在接口的出方向上进行流量统计。

source：根据源IP地址信息进行统计。若不配置该参数，则对进入该接口的报文按照目的IP地址进行统计。

缺省级别

使用指南

ip bgp-accounting命令必须先在流量的入接口上配置；如果要进一步统计出方向的流量值，还需要在对应的出接口上配置ip bgp-accounting。

仅在需要查找转发表的情况下，BGP Accounting才生效。例如，始发接口配置出方向计费，则BGP Acounting无效。

使用实例

# 在接口Pos1/0/0的入方向上使能BGP Accounting功能，并且根据源IP地址信息进行流量统计。

<Quidway> system-view
[Quidway] interface pos1/0/0
[Quidway-Pos1/0/0] ip bgp-accounting inbound source
