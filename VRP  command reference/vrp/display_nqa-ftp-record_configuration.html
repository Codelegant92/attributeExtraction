display nqa-ftp-record configuration

命令功能

display nqa-ftp-record configuration 用来显示通过FTP保存NQA测试结果的配置信息。

命令格式

display nqa-ftp-record configuration

参数说明

无

使用实例

# 显示通过FTP保存NQA测试结果的配置信息。

<Quidway> display nqa-ftp-record configuration
---------------NQA FTP SAVE RECORD CONFIGURATION---------------
   FUNCTION: ENABLE    TRAP: DISABLE
   IP-ADDRESS:11.1.1.8
   VPN-INSTANCE:
   USERNAME:ftp
   PASSWORD:ftp
   FILENAME:a
   ITEM-NUM:10010
   TIME:2
   LAST FINISHED FILENAME:
表1 display nqa-ftp-record configuration命令输出信息描述

项目

描述

FUNCTION
是否使能了通过FTP发送测试结果。

  • Enable：使能。

  • Disable：没有使能。

TRAP
通过FTP保存测试结果成功后发送Trap是否使能。

  • Enable：使能。

  • Disable：没有使能。

IP-ADDRESS

FTP服务器的地址。

VPN-INSTANCE

VPN实例名。

USERNAME

登录FTP服务器需要的命令名。

PASSWORD

登录FTP服务器需要的密码。

FILENAME

在FTP服务器端生成的文件名。

ITEM-NUM

在FTP服务器端生成的文件中保存的测试结果的条数。

TIME

在FTP服务器端保存文件的时间。

LAST FINISHED FILENAME

最近一次在FTP服务器端保存文件的名称。
