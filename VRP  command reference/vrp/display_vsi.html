display vsi

命令功能

display vsi命令用于显示VSI实例的信息。缺省参数的情况下，将显示所有VSI实例的信息。

命令格式

display vsi [ name vsi-name ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

name vsi-name：要显示的VSI实例的名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

verbose：显示VSI实例的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display vsi命令时，

  • 如果指定参数vsi-name，则显示特定VSI实例信息。

  • 如果未指定参数vsi-name，则显示所有VSI实例信息。

使用实例

# 显示名称为company1的VSI实例的信息。

<Quidway> display vsi name company1
Vsi                             Mem    PW   Mac       Encap     Mtu   Vsi
Name                            Disc   Type Learn     Type      Value State
company1                       auto   bgp  unqualify vlan      1500  up

项目

描述

VSI Name

VSI名称

VSI Index

VSI的索引值

PW Signaling

PW的信令类型。分为LDP和BGP两种方式

Member Discovery Style

成员发现方式。分为动态发现和静态发现两种方式。

PW MAC Learn Style

PW的MAC地址学习方式。分为unqualify和qualify两种方式。

Encapsulation Type

VSI的封装形式，即报文在VC上的封装形式。

MTU

最大传输单元。

Create Time

VSI的创建时间。

VSI State

VSI的状态。

BGP RD

即Route Distinguisher，在以BGP为信令的VPLS中用来标识PE上的一个VSI实例，是本端设备的标志符。此信息只有配置了Kompella方式的VPLS时，才会显示。

SiteID/Range/Offset

本端设备上VSI实例的Site ID、VSI实例中的标签范围（Site的个数）和初始Site ID的偏差值。此信息只有配置了Kompella方式的VPLS时，才会显示。

Import vpn target

来自目的VPN的入方向扩展团体属性。此信息只有配置了Kompella方式的VPLS时，才会显示。

Export vpn target

到目的VPN的出方向扩展团体属性。此信息只有配置了Kompella方式的VPLS时，才会显示。

Remote Label Block

远端标签块的起始值、标签范围和初始Site ID的偏差值。此信息只有配置了Kompella方式的VPLS时，才会显示。

Local Label Block

本端标签块的起始值、标签范围和初始Site ID的偏差值。此信息只有配置了Kompella方式的VPLS时，才会显示。

VSI ID

VSI实例的ID。此信息只有配置了Martini方式的VPLS时，才会显示。

Peer Router ID

对端设备的ID。此信息只有配置了Martini方式的VPLS时，才会显示。

VC Label

VC所使用的标签值。此信息只有配置了Martini方式的VPLS时，才会显示。

Peer Type

对端的PW类型，分为动态和静态两种。此信息只有配置了Martini方式的VPLS时，才会显示。

Session

本端与对端的会话的状态。分为Up和Down两种状态。此信息只有配置了Martini方式的VPLS时，才会显示。

Tunnel ID

Tunnel的ID。此信息只有配置了Martini方式的VPLS时，才会显示。

Interface Name

绑定了该VSI实例的接口名称。

State

与VSI绑定的AC当前状态。

Last Up Time

AC接口最后一次UP的时间。

Total Up Time

AC接口UP的总时间。

PW Information

PW相关的信息。

Peer Ip Address

对等体的IP地址。

PW State

PW的当前状态。

Local VC Label

本地分发的VC标签。

Remote VC Label

对等体分发的VC标签。

PW Type

伪线的类型。

Tunnel ID

隧道标志符。

PW Last Up Time

PW最后一次UP的时间。

PW Total Up Time

PW UP的总时间。

FIB Link-ID

PW对应的FIB Link-ID

该值非0，表示FIB表项创建成功，该值为0，表示FIB表项创建失败
