ip urpf

命令功能

ip urpf命令将在相应的接口使能URPF检查。

undo ip urpf命令在相应的接口禁止URPF检查。

缺省情况下，不使能URPF检查。

命令格式

ip urpf { loose | strict } [ allow-default-route ] [ acl acl-number ]

undo ip urpf { loose | strict } [ allow-default-route ] [ acl acl-number ]

参数说明

loose：URPF将进行松散检查。

strict：URPF将进行严格检查。

allow-default-route：允许对默认路由进行特殊处理。

acl-number：指定ACL规则组编号。整数形式，范围是2000～3999，其中2000～2999是基本访问控制列表，3000～3999是高级访问控制列表。

缺省级别

使用指南

只在转发表中存在表项就通过URPF检查，不要求接口一定匹配时配置loose模式；不但要求在转发表中存在相应表项，还要求接口一定匹配才能通过URPF检查时用strict模式。

使用实例

# 在以太口GE1/0/0上使能严格URPF检查，同时允许对缺省路由进行特殊处理，并配置ACL编号为2999。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ip urpf strict allow-default-route acl 2999
