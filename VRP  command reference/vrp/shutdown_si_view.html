shutdown（业务实例视图）

命令功能

shutdown命令用来关闭业务实例。

undo shutdown命令用来激活业务实例。

缺省情况下，业务实例处于关闭状态。

命令格式

shutdown

undo shutdown

参数说明

无

视图

业务实例视图

缺省级别

2：配置级

使用指南

在未激活业务实例之前，使用display service-instance verbose命令查看该业务实例的信息时，该实例的状态显示为“Administratively DOWN”。

如果业务实例的状态为Down，则不会转发报文，所有该业务实例的数据报文都将被丢弃。

使用实例

# 配置关闭业务实例。

<Quidway> system-view
[Quidway] service-instance instance-name mysi
[Quidway-service-instance-mysi] shutdown
