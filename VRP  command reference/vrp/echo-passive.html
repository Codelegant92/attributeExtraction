echo-passive

命令功能

echo-passive命令用来使能BFD被动ECHO功能。

undo echo-passive 命令用来取消使能BFD被动ECHO功能。

缺省情况下，不使能BFD被动ECHO功能。

命令格式

echo-passive { all | acl basic-acl-number }

undo echo-passive

参数说明

all：使能所有BFD会话的被动ECHO功能。

acl：指定ACL配置信息，通过ACL确定使能BFD会话被动ECHO功能的规则。

basic-acl-number：ACL编号。整数形式，取值范围是2000～2999。

缺省级别

2：配置级

使用指南

在使能BFD的被动回声功能之前必须全局使能BFD。

BFD回声报文在对端通过ICMP重定向的方式环回，在封装BFD回声报文IP报文中，目的地址和源地址都是本端出接口的IP地址。因此，在ACL规则中，除permit对端的源地址以外，还必须permit本端的源地址。

使用实例

# 对于符合ACL 2000（10.1.1.1是本端地址，10.1.1.2是对端地址）的BFD会话启动被动回声功能。

<Quidway>system-view
[Quidway] acl 2000
[Quidway-acl-basic-2000] rule permit source 10.1.1.1 0
[Quidway-acl-basic-2000] rule permit source 10.1.1.2 0
[Quidway-acl-basic-2000] quit
[Quidway] bfd
[Quidway-bfd] echo-passive acl 2000
