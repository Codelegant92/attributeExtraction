display ospf traffic-adjustment

命令功能

display ospf traffic-adjustment命令用来查看与流量调节（IGP shortcut及转发邻接）相关的、OSPF进程的隧道信息。

命令格式

display ospf [ process-id ] traffic-adjustment [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPF的进程号，取值范围是1～65535。如果不指定进程号，则显示所有进程的相关信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看OSPF100进程与流量调节相关的隧道信息。

<Quidway> display ospf 100 traffic-adjustment
