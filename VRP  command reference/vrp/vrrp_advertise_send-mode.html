vrrp advertise send-mode

命令功能

vrrp advertise send-mode命令用来配置super-VLAN中的VRRP通告报文发送方式。

undo vrrp advertise send-mode命令用来取消配置。

缺省情况下，super-VLAN不向自己的sub-VLAN发送VRRP通告报文。

命令格式

vrrp advertise send-mode { sub-vlan-id | all }

undo vrrp advertise send-mode

参数说明

sub-vlan-id：VRRP通告报文只发送给指定ID的sub-VLAN。VLAN ID的取值范围是1～4094。

all：VRRP通告报文以广播方式发给本super-VLAN的所有子VLAN。

使用实例

# 设置在super-VLAN 100中，VRRP通告报文以广播方式发给所有sub-VLAN。

[Quidway] vlan 100
[Quidway-vlan100] aggregate-vlan
[Quidway-vlan100] quit
[Quidway] interface vlanif 100
[Quidway-Vlanif100] vrrp vrid 1 virtual-ip 10.10.10.10
[Quidway-Vlanif100] vrrp advertise send-mode all
