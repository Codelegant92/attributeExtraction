undelete（多框）

命令功能

undelete命令用来恢复删除的文件。

命令格式

undelete [ [path] | [ drive ] ] [ filename ]

参数说明

path:文件的路径。

drive：设备名。

filename：要恢复的文件名。字符串形式，长度范围是1～64。

视图

用户视图

缺省级别

3：管理级

使用指南

恢复的文件名如果与现有的目录名重名，则执行失败；若与当前存在的文件名重名，将会提示是否覆盖。

使用实例

# 恢复转发框1中删除的文件sample.text。

<Quidway> undelete clc1/17#cfcard:/sample.txt
Undelete clc1/17#cfcard:/sample.txt?[Y/N]:y
%Undeleted file clc1/17#cfcard:/sample.txt.
