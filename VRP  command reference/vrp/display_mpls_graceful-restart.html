display mpls graceful-restart

命令功能

display mpls graceful-restart命令用来查看与MPLS相关的所有协议GR信息。

命令格式

display mpls graceful-restart [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

| ：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Protocol

协议类型。包括：

LDP、CRLDP、RSVP、BGP、L3VPN、STATIC、CRSTATIC、BGP IPV6、STATIC HA和L3VPN IPV6。

GR state

GR的状态。包括：

  • Restarting：正在进行GR。
  • Normal：未执行GR。

GR start time

GR的开始时间。

如果GR没有开始，此处显示为“-”。

GR type

GR的类型。包括：

  • System restart：系统级GR。
  • Protocol restart：协议级GR。

如果GR没有开始，此处显示为“-”。
