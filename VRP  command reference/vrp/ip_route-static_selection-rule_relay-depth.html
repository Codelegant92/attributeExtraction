ip route-static selection-rule relay-depth

命令功能

ip route-static selection-rule relay-depth命令用来控制静态路由按迭代深度进行优选。

undo ip route-static selection-rule relay-depth命令用来撤销静态路由按迭代深度进行优选的功能。

缺省情况下，静态路由不按迭代深度优选。

命令格式

ip route-static selection-rule relay-depth

undo ip route-static selection-rule relay-depth

参数说明

无

使用指南

系统中存在若干同一前缀的静态路由，其优先级相同，迭代深度不同。配置了基于迭代深度优选之后，静态路由模块会选择迭代深度较小的静态路由作为活跃路由，并下发FIB，其他路由为不活跃路由。

使用实例

# 配置静态路由按迭代深度进行优选。

<Quidway> system-view
[Quidway] ip route-static selection-rule relay-depth
