mkdir（FTP客户端视图）

命令功能

mkdir命令用来在远程FTP服务器建立目录。

命令格式

mkdir remote-directory

参数说明

remote-directory：目录名，字符串形式，长度范围是1～64。

视图

FTP客户端视图

缺省级别

3：管理级

使用指南

运行该命令后，创建完毕的目录存在于FTP服务器上。

使用实例

# 在远程FTP服务器上建立目录test。
