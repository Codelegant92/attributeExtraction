mpls rsvp-te bfd enable

命令功能

mpls rsvp-te bfd enable命令用来使能接口的BFD for RSVP能力。

undo mpls rsvp-te bfd enable命令用来恢复为缺省配置。

缺省情况下，不使能接口的BFD for RSVP能力。

命令格式

mpls rsvp-te bfd enable

undo mpls rsvp-te bfd enable

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

该命令与mpls rsvp-te bfd block互斥，两者的区别：

  • 在RSVP接口配置undo mpls rsvp-te bfd enable命令后，如果在MPLS视图下重新配置mpls rsvp-te bfd all-interfaces enable命令，则该接口仍然具备BFD for RSVP能力。

  • 在接口下配置mpls rsvp-te bfd block命令，则即使在MPLS视图下重新配置mpls rsvp-te bfd all-interfaces enable命令，该接口也不再具备BFD for TE能力。

窍门：

  • 如果要使能大部分接口的BFD for RSVP能力，则在不需要使能BFD for RSVP的少数接口上配置mpls rsvp-te bfd block命令，然后在MPLS视图下配置mpls rsvp-te bfd all-interfaces enable命令。

  • 如果只是使能少数接口的BFD for RSVP能力，则只在这些接口上配置mpls rsvp-te bfd enable命令。此种情况下，去除接口的BFD for RSVP能力可使用undo mpls rsvp-te bfd enable命令或mpls rsvp-te bfd block命令。

使用实例

# 使能接口下的BFD for RSVP能力

<Quidway> system-view
[Quidway] mpls lsr-id 1.1.1.1
[Quidway] mpls
[Quidway-mpls] mpls te
[Quidway-mpls] mpls rsvp-te
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mpls
[Quidway-GigabitEthernet1/0/0] mpls te
[Quidway-GigabitEthernet1/0/0] mpls rsvp-te
[Quidway-GigabitEthernet1/0/0] mpls rsvp-te bfd enable
