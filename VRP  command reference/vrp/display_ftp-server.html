display ftp-server

命令功能

display ftp-server命令用来显示当前FTP服务器的各项参数。

命令格式

display ftp-server

参数说明

无

视图

所有视图

缺省级别

3：管理级

使用指南

进行FTP参数设置后，可通过此命令查看设置结果。

使用实例

# 显示FTP服务器参数设置结果。

<Quidway> display ftp-server
   FTP server is running
   Max user number                5
   User count                     1
   Timeout value(in minute)       30
   Acl Number                     2345
   FTP server's source interface  LoopBack0
表1 display ftp-server命令输出信息描述

项目

描述

Ftp server is running

FTP服务器已经启动

Max user number

支持同时登录的最大用户数

User count

现在登录的用户数

Timeout value(in minute)

FTP用户的超时时间（单位为分钟）

Acl Number

FTP基本访问控制列表号（如果没有配置ACL，此项的显示就是0）

The source interface of the FTP server is LoopBack0

本端路由器的IPv4地址或者出接口信息。
