reset cfm error-packet receive statistics

命令功能

reset cfm error-packet receive statistics命令用来清除本设备接收错误CCM消息的统计信息。

命令格式

reset cfm error-packet receive statistics [ md md-name [ ma ma-name [ remote-mep mep-id mep-id ] ] ]

参数说明

md md-name：表示清除指定MD内接收错误CCM消息的统计信息。md-name表示MD的名称，字符串形式，不支持空格、“-”和“?”，与ma-name组合最长为44个字符。

ma ma-name：表示清除指定MA内接收错误CCM消息的统计信息。ma-name表示MA的名称，字符串形式，不支持空格、“-”和“?”，与md-name组合最长为44个字符。

remote-mep mep-id mep-id：表示清除本设备从指定RMEP接收到的错误CCM消息的统计信息。mep-id表示RMEP的ID，整数形式，取值范围是1～8191。

使用指南

如果需要观察从当前时间开始的某段时间内的错误CCM消息统计信息，可首先使用本命令清除历史统计信息。

使用实例

# 清除md1中ma1内接收的错误CCM消息的统计信息。

<Quidway> reset cfm error-packet receive statistics md md1 ma ma1
