smart-link flush send

命令功能

smart-link flush send vlan 用来使能1:1主备模式下主备切换时活动接口Flush报文的发送。

undo smart-link flush send用来关闭1:1主备模式下主备切换时活动接口Flush报文的发送。

命令格式

smart-link flush send vlan vlan-id

undo smart-link flush send

参数说明

vlan vlan-id ：发出SmartLink Flush报文的VLAN ID，取值范围是1～4094。

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

本命令配置的前提条件是需要该Eth-Trunk为1:1主备工作模式。如果需要更改当前Eth-Trunk工作模式，可以通过执行mode命令修改。

配置了此命令后，当主备接口切换时，会从新的主接口发送flush报文，来通知对端老化MAC地址，避免因为MAC地址不同步引起的数据传输中断。

使用实例

# 在Eth-Trunk 1上允许发送Flush报文，报文中的VLAN ID为1000。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] portswitch
[Quidway-Eth-Trunk1] smart-link flush send vlan 1000
