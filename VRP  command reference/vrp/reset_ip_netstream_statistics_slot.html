reset ip netstream statistics slot

命令功能

reset ip netstream statistics slot命令用来清除指定接口板NetStream 统计信息，同时将流缓存区中所有流老化。

命令格式

reset ip netstream statistics slot slot-id

参数说明

slot-id：指定清除NetStream状态统计信息的接口板槽号。

使用reset ip netstream statistics slot命令清除指定接口板NetStream 统计信息，同时将流缓存区中所有流老化。
