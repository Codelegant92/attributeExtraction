graceful-restart interval

命令功能

graceful-restart interval命令用来指定IS-IS协议的GR时间。

undo graceful-restart interval命令用来恢复GR时间的缺省值。

缺省情况下，GR间隔时间为300秒。

命令格式

graceful-restart interval interval-value

undo graceful-restart interval

参数说明

interval-value：GR间隔时间。整数形式，取值范围是30～1800，单位是秒。缺省值是300秒（5分钟）。
