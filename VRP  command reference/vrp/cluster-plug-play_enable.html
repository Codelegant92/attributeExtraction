cluster-plug-play enable

命令功能

cluster-plug-play enable命令用来使能集群的即插即用功能。

undo cluster-plug-play enable命令用来禁止集群的即插即用功能。

缺省情况下，命令交换机上集群的即插即用功能处于禁止状态。

命令格式

cluster-plug-play enable

undo cluster-plug-play enable

参数说明

无

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

使用实例

# 使能集群的即插即用功能。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] cluster-plug-play enable
