user-interface current

命令功能

user-interface current命令用来进入当前用户界面视图。

命令格式

user-interface current

参数说明

无

视图

系统视图

缺省级别

3：管理级

使用指南

当用户需要进入当前用户界面视图下进行配置时，无需知道当前用户界面编号，执行该命令就可以直接进入当前用户界面视图。

使用实例

# 进入当前用户视图，进行配置。

<Quidway > system-view
[Quidway] user-interface current
[Quidway-ui-vty1]
