check startup

命令功能

check startup命令用来检查各种资源文件（PAF文件、License文件、补丁包、启动软件、配置文件）是否正确。

命令格式

check startup [ crc ] [ next ]

参数说明

crc：对资源文件进行CRC校验。

next：检查下一次启动的各种资源文件。

视图

用户视图

缺省级别

2：配置级

使用指南

一般配置完下次启动的资源文件后，用此命令检查一下配置的资源文件是否完整，格式和版本是否正确。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。
