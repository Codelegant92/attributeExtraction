display ospf interface

命令功能

display ospf interface命令用来显示OSPF的接口信息。

命令格式

display ospf [ process-id ] interface [ all | interface-type interface-number ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPF进程号。整数形式，取值范围是1～65535。

all：显示所有接口的OSPF详细信息。

interface-type：接口类型，如GigabitEthernet、Loopback等。

interface-number：接口号。

verbose：显示OSPF接口的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令的输出信息有助于查看OSPF配置及运行状态，进行故障诊断并验证配置效果。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示OSPF接口信息。

<Quidway> display ospf interface
          OSPF Process 1 with Router ID 192.168.1.1
                  Interfaces

 IP Address      Type         State    Cost  Pri   DR              BDR
 192.168.1.1     PTP          P-2-P    1562  1     0.0.0.0         0.0.0.0
 Area: 0.0.0.1
 IP Address      Type         State    Cost  Pri   DR              BDR
 172.16.0.1      Broadcast    DR       1     1     172.16.0.1      0.0.0.0

# 显示OSPF指定接口的信息。

<Quidway> display ospf interface pos 1/0/0
          OSPF Process 1 with Router ID 192.168.1.1
                  Interfaces

 Cost: 1562  State: P-2-P     Type: PTP       MTU: 1500
 Timers: Hello 10 , Dead 40 , Poll  120 , Retransmit 5 , Transmit Delay 1

项目

描述

Area

接口所属的区域ID

IP address

接口IP地址（不管是否使能了流量工程）

Type

接口类型为PTP、PTMP、广播或NBMA

State

根据OSPF接口状态机确定的当前接口状态

Cost

接口开销

Pri

接口的路由器优先级（用于DR选择）

DR

接口所在网络上的DR

BDR

接口所在网络上的BDR

MTU

接口的MTU值

Interface

接口IP地址+接口号+邻居IP地址（用于PTP）

Timer Hello

Hello计时间隔

Dead

路由器失效时间

Poll

轮询时间间隔

Retransmit

LSA重传间隔

Transmit Delay

接口传输延迟

Silent interface, no hellos

禁止接口发送OSPF报文

MPLS- Traffic Engineering Link

TE链路

IGP shortcut link

是否为IGP Shortcut链路

Forwarding Adjacency Link

是否为邻接转发链路
