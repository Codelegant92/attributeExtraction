reset traffic-statistics vsi peer negotiation-vc-id

命令功能

reset traffic-statistics vsi name peer negotiation-vc-id命令用来清除指定VSI-Name中指定Martini方式VPLS PW上的公网流量统计信息。

命令格式

reset traffic-statistics vsi name vsi-name [ peer peer-address [ negotiation-vc-id vc-id ] ]

参数说明

vsi name vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31；

peer peer-addres：PW的对端IP地址。

negotiation-vc-id vc-id：PW的ID。

使用指南

在所有视图下显示指定Martini方式VPLS PW流量统计信息。Martini方式VPLS PW由用户指定的VSI-Name与指定的对端IP地址和PW ID唯一确定。

使用实例

# 在用户视图清除指定Martini方式VPLS PW流量统计信息

<Quidway> reset traffic-statistics vsi name newvsi peer 100.22.33.20 
negotiation-vc-id 2
