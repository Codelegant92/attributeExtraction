display cpu-packet statistics slot

命令功能

display cpu-packet statistics slot命令用来查看槽位中所有接口的上送到CPU的报文的统计信息。

命令格式

display cpu-packet statistics slot { all | slot-number }

参数说明

all：查看所有槽位号的统计信息。

slot-number：查看指定槽位号的统计信息。

视图

所有视图

缺省级别

1：监控级

使用指南

执行命令display cpu-packet statistics slot时，所有槽位或指定槽位上的所有接口上送到CPU的报文都会被统计。

使用实例

# 查看1号槽位上的接口上送到CPU的报文的统计信息。

<Quidway> display cpu-packet statistics slot 1
Cpu-packet Statistics Information:
--------------------------------------------------------------------------
Interface                                         Packet      Size(byte)
--------------------------------------------------------------------------
Ethernet1/0/0                                      4           580
Ethernet1/0/1                                      4           580
Ethernet1/0/2                                      4           580
--------------------------------------------------------------------------
表1 display cpu-packet statistics slot输出信息描述

Interface

经过接口上送到CPU的报文的字节数。
