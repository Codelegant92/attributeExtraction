ftp-filesize

命令功能

ftp-filesize命令用来配置NQA测试中FTP测试例上载测试的文件大小。

undo ftp-filesize命令用来恢复缺省配置。

缺省情况下，FTP测试例上载测试的文件大小为1024kbytes。

命令格式

ftp-filesize size

undo ftp-filesize

参数说明

size：FTP测试的操作文件大小。单位是kbytes，取值范围是1～10000。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对FTP上载类型的测试例配置操作文件的大小。

不能更改正在运行的测试例的操作文件大小。

采用指定文件大小的方式进行上载速度测试时，FTP客户端将自动构建指定大小的文件，文件名为“nqa-ftp-test.txt”。如果多次执行测试，后上载的“nqa-ftp-test.txt”文件将覆盖之前上载的文件。

使用实例

# 设置测试例user test的FTP操作文件大小为1M字节。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type ftp
[Quidway-nqa-user-test] ftp-filesize 1000
