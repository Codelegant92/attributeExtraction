display nat

命令功能

display nat命令显示用户对地址转换的配置。

命令格式

display nat { all | address-group | outbound | server | alg }

参数说明

all：表示显示所有的关于地址转换的配置信息及NAT ALG的配置信息。

outbound：表示显示配置的地址转换的信息。

address-group：表示显示配置的地址池的信息。

server：表示显示配置的内部服务器的信息。

alg：表示显示配置的NAT ALG的信息。

使用指南

display nat命令显示了地址池的使用信息和地址转换的情况。

用户可以根据该命令的输出信息认证地址转换的配置是否正确。

使用实例

# 显示所有的关于地址转换的信息。

<Quidway> display nat all
NAT address-group information:
      0 : from      10.110.1.1   to     10.110.1.30, reference   0 times
     10 : from      100.1.1.10   to      100.1.1.20, reference   1 times
     18 : from     10.110.27.1   to   10.110.27.100, reference   0 times
  Total   3 address-groups
NAT outbound information:
                    Gigabitethernet 4/1/0: acl(3100) --- NAT address-group(10)
Server in private network information:
      GlobalAddr   GlobalPort      InsideAddr   InsidePort     Pro    VPN   Ref
Interface:Gigabitethernet 3/1/0
    192.168.10.2        ----       100.1.1.2        ----     ---      (1)time (1)
  Total   1 NAT servers

上述信息表示：

配置了三个地址池：地址池号分别是0、10、18，其中第10号地址池被引用了一次。

在GE4/1/0配置了1个地址转换关联：ACL规则3100与地址池10关联，进行PAT方式的地址转换。

在GE3/1/0配置了1个内部服务器。http:// 192.168.10.2的服务器，内部地址是100.1.1.2。
