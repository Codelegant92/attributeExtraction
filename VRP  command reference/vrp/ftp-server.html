ftp-server

命令功能

ftp-server命令用来配置HGMP集群公用FTP服务器的IP地址。

undo ftp-server命令用来取消已配置的HGMP集群公用FTP服务器的IP地址。

缺省情况下，HGMP集群没有公用FTP服务器。

命令格式

ftp-server ip-address

undo ftp-server

参数说明

ip-address：HGMP集群公用FTP服务器的IP地址。请将其设置为合法的单播地址，否则将会提示出错信息。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

重复执行该命令来配置公用FTP服务器，新配置将替代旧配置。

在使用ftp-server命令之前，必须先配置cluster-ftp-nat enable命令，使能命令交换机的FTP NAT功能。

使用实例

# 配置HGMP集群公用FTP服务器的IP地址为1.1.1.1。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] ftp-server 1.1.1.1
