display patch-information allpatch

命令功能

display patch-information allpatch命令用来查看系统所有补丁的状态。

命令格式

display patch-information allpatch

参数说明

无

视图

所有视图

缺省级别

3：管理级

使用指南

执行该命令可查看系统中所有补丁的数目、类型、状态等信息。

使用实例

# 查看系统的所有补丁的状态。

<Quidway> display patch-information allpatch

patch area start address: 0x100000
----------The patch information of slot 2----------
patch version           :
temp patch number       : 0

active patch number     : 0
patch area length       : 0x100000
patch area start address: 0x100000

项目

描述

patch version

补丁的版本

temp patch number

临时补丁的数目

common patch number

普通补丁的数目

current patch number

当前补丁的数目

running patch number

当前正在运行的补丁的数目

active patch number

激活状态的补丁的数目

patch area length

内存补丁区的长度

patch area start address

内存补丁区的起始地址

np patch file name

NP补丁文件的名字

np patch version

NP补丁的版本

program base version

系统文件的版本

np patch number

NP补丁的数目

np patch file operational mode

NP补丁文件的操作模式

np patch file current state

NP补丁的当前状态
