user-interface

命令功能

user-interface命令用来进入单一用户界面视图或多个用户界面视图。

命令格式

user-interface [ ui-type ] first-ui-number [ last-ui-number ]

参数说明

ui-type：用户界面（User-interface）的类型名称。如果指定用户界面类型，表示使用相对编号，如果不指定用户界面类型，则表示使用绝对编号。

first-ui-number：欲配置的第一个用户界面。

last-ui-number：欲配置的最后一条用户界面，last-ui-number的取值要比first-ui-number取值大。

视图

系统视图

缺省级别

3：管理级

使用指南

使用此命令进入用户界面视图后，可以配置该用户界面的相关属性。

使用实例

# 进入user-interface vty视图，对vty 0到vty 3进行配置。

<Quidway > system-view
[Quidway] user-interface vty 0 3
[Quidway-ui-vty0-3]
