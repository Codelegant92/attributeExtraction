loopback（串行接口视图）

命令功能

loopback命令用来使能串口环回。

undo loopback命令用来禁止串口环回。

缺省情况下，禁止串口环回。

命令格式

loopback

undo loopback

参数说明

无

视图

串行接口视图

缺省级别

2：配置级

使用指南

说明：

由其它接口板通道化成的serial接口不支持loopback命令。

只有在进行某些特殊功能测试时，才将串口设为对内自环。

使用实例

# 允许串口Serial1/0/0环回。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] loopback
