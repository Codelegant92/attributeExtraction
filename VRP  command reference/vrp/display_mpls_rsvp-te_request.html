display mpls rsvp-te request

命令功能

display mpls rsvp-te request命令用来查看接口的RSVP-TE请求消息信息。

命令格式

display mpls rsvp-te request [ interface interface-type interface-number peer-ip-address ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type interface-number：接口类型和接口号。

peer-ip-address：邻居的IP地址。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定接口名，将显示所有使能了RSVP-TE的接口的请求消息信息。

使用实例

# 查看指定接口的请求消息信息。

<Quidway> display mpls rsvp-te request interface gigabitethernet 6/0/0 2.2.2.9

项目

描述

Interface

使能了RSVP-TE的接口

Tunnel Addr

隧道目的地址，格式为X.X.X.X

Ingress LSR ID

入节点的LSR ID，格式为X.X.X.X

Local LSP ID

LSP的本地标识符

Session Tunnel ID

隧道ID

Next Hop Address

下一跳地址，格式为X.X.X.X

SessionAttrib
会话属性，包括：
  • 本地保护
  • 标签记录
  • 资源预留风格

Token Bucket rate

流量参数，令牌桶速率

Token Bucket Size

流量参数，令牌桶大小

Out Interface

LSP的出接口
