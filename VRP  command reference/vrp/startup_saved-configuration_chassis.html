startup saved-configuration（多框）

命令功能

startup saved-configuration命令用来配置系统下次启动时使用的配置文件。

命令格式

startup saved-configuration configuration-file [ all | chassis chassis-number ]

参数说明

configuration-file：配置文件的文件名。字符串形式，长度范围是5～48。

all：配置所有框的主备用板。

chassis chassis-number：框号，整数形式，当框类型为线卡框时，框号取值范围1～8，例如：clc1。当框类型为中央交换框时，框号取值为1和2，例如：ccc1

使用指南

配置文件必须以“.cfg” 或“.zip”作为扩展名，而且系统启动配置文件必须存放在存储设备（CF卡）的根目录下，如果您选择缺省路径，系统配置文件将存储在中央交换框CF卡的根目录下。

使用实例

# 配置下次启动使用的配置文件。

<Quidway> startup saved-configuration vrpcfg.cfg
