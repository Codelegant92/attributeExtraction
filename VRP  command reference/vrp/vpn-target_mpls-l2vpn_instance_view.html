vpn-target（MPLS-L2VPN实例视图）

命令功能

vpn-target命令用于为L2VPN指定VPN-target属性。

undo vpn-target命令用来删除与L2VPN关联的VPN-target。

缺省情况下，系统没有为L2VPN指定VPN-Target属性值。

命令格式

vpn-target vpn-target &<1-16> [ both | export-extcommunity | import-extcommunity ]

undo vpn-target { all | vpn-target &<1-16> [ both | export-extcommunity | import-extcommunity ] }

参数说明

vpn-target：添加路由目标扩展团体属性到VPN。

  • 16位自治系统号:32位用户自定义数，例如：1:3。自治系统号的取值范围是0～65535；用户自定义数的取值范围是0～4294967295。其中，自治系统号和用户自定义数不能同时为0，即VPN Target的值不能是0:0。

  • 32位IP地址:16位用户自定义数，例如：192.168.122.15:1。IP地址的取值范围是0.0.0.0～255.255.255.255；用户自定义数的取值范围是0～65535。

import-extcommunity：可接收的VPN扩展团体。

export-extcommunity：发布的VPN扩展团体。

both：添加路由目标到当前VPN，接收的VPN扩展团体和发布的VPN扩展团体相同。

all：删除所有的VPN-Target属性。

缺省级别

2：配置级

使用指南

如果用户不指定关键字“both”、“export-extcommunity”或“import-extcommunity”，则缺省为“both”。

配置VPN target前，应该首先配置路由标识符RD（Route Distinguisher）的值。

使用实例

# 为名为VPN1的L2VPN配置VPN-target属性。

<Quidway> system-view
[Quidway] mpls l2vpn vpn1 encapsulation ppp
[Quidway-mpls-l2vpn-vpn1] vpn-target 100:1
[Quidway-mpls-l2vpn-vpn1] vpn-target 1:1 2:2 export-extcommunity
[Quidway-mpls-l2vpn-vpn1] vpn-target 1.2.3.4:11 12:12 import-extcommunity

# 删除VPN1的VPN-target属性。

<Quidway> system-view
[Quidway] mpls l2vpn vpn1 encapsulation ppp
[Quidway-mpls-l2vpn-vpn1] undo vpn-target 12:12 import-extcommunity

# 删除VPN1的所有VPN-target属性。

<Quidway> system-view
[Quidway] mpls l2vpn vpn1 encapsulation ppp
[Quidway-mpls-l2vpn-vpn1] undo vpn-target all
