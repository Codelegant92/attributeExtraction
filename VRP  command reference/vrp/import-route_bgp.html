import-route (BGP)

命令功能

import-route命令用来引入其它协议路由信息。

undo import-route命令用来取消引入其它协议的路由信息。

缺省情况下，BGP将不引入其它协议的路由。

命令格式

import-route protocol [ process-id ] [ med med | route-policy route-policy-name ] *

undo import-route protocol [ process-id ]

参数说明

protocol：指定可引入的外部路由协议。目前，对于BGP-IPv4单播地址族视图和BGP-VPN实例视图，可引入的外部路由协议包括isis、ospf、static、direct和rip；对于BGP-IPv6单播地址族视图和BGP-VPN6实例视图，可引入的外部路由协议包括isis、ospfv3、static、direct和ripng。

process-id：当引入路由协议为rip、ospf、isis、ripng或ospfv3时，必须指定进程号。

med med：指定引入路由的MED度量值。

route-policy route-policy-name：从其他路由协议引入路由时，可以使用该参数指定的Route-Policy过滤器过滤路由和修改路由属性。
