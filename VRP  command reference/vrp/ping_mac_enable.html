ping mac enable

命令功能

ping mac enable命令用来使能通用MAC Ping功能。

undo ping mac enable命令用来禁止通用MAC Ping功能。

缺省情况下，通用MAC Ping功能处于禁止状态。

命令格式

ping mac enable

undo ping mac enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

当需要使用以下功能时，需要使用本命令使能通用MAC Ping功能。

  • 进行通用MAC Ping操作。

  • 使设备对接收到的通用MAC Ping报文进行回复。

使用实例

# 使能通用MAC Ping功能。

<Quidway> system-view
[Quidway] ping mac enable
