qos lr（虚拟接口模板视图）

命令功能

qos lr命令用来启动动态QoS限速功能。

undo qos lr命令用来取消动态QoS限速功能。

缺省情况下，不启动动态QoS。

命令格式

qos lr

undo qos lr

参数说明

无

视图

虚拟接口模板视图

缺省级别

2：配置级

使用指南

只能在VT接口上应用动态QoS的限速功能。

qos lr命令只在NE16E/08E/05路由器上可用。

使用实例

# 在接口VT1上启动动态QoS。

<Quidway> system-view
[Quidway] interface Virtual-Template 1
[Quidway-Virtual-Template1] qos lr
