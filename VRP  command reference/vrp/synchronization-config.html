synchronization-config

命令功能

synchronization-config命令用来把成员交换机的配置文件保存到FTP服务器。

命令格式

synchronization-config

参数说明

无

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在成员交换机上执行。

执行该命令的成员交换机必需已经加入集群，并且命令交换机已经配置了默认的FTP服务器信息。

把成员交换机的配置文件同步到FTP服务器的目的是成员交换机更换后，可以根据原有设备的配置来恢复新设备配置，以减少人工再次配置工作量。

成员交换机的配置在FTP服务器上的存储方式是以成员交换机MAC地址命名的配置文件。

使用本命令时，成员交换机将使用命令交换机上配置的默认FTP服务器信息，此时需要保证成员交换机与默认的FTP服务器之间存在可以相互到达的路由。

说明：

命令交换机上默认的FTP服务器配置用来支持HGMP的即插即用功能。具体配置方法请参考cluster-plug-play ip命令。

使用实例

# 把成员交换机的配置文件保存到FTP服务器。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] synchronization-config
