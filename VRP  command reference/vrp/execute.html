execute

命令功能

execute命令用来执行指定的批处理文件。

命令格式

execute file-name

参数说明

file-name：批处理文件名，以“.bat”为后缀。字符串形式，取值范围是1～256。

视图

系统视图

缺省级别

2：配置级

使用指南

执行批处理文件之前，应确保当前目录为批处理文件所在目录，否则系统将找不到批处理文件。

批处理命令逐条执行批处理文件中的命令行，批处理文件中不能包含不可见字符，如果发现有不可见字符，批处理命令则退出当前执行过程，并且不进行回退操作。

批处理命令不保证每一条命令的执行，批处理命令本身不进行热备份处理，批处理文件中不限制命令的形式和内容。

批处理命令的执行过程相当于手工逐条执行的自动化过程。

使用实例

# 执行flash:/目录下的test.bat批处理文件。

<Quidway> system-view
[Quidway] execute test.bat
