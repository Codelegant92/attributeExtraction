patch deactive

命令功能

patch deactive命令用来去激活补丁。

缺省情况下，在主用主控板上去激活补丁。

命令格式

patch deactive [ all | slave | slot slot-id ]

参数说明

all：去激活所有单板上的补丁。

slave：去激活备用主控板上的补丁。

slot slot-id：去激活指定单板上的补丁。

视图

用户视图

缺省级别

3：管理级

使用指南

执行patch deactive all命令，将在所有在位单板上（包括主用主控板、备用主控板和所有接口板）去激活补丁。

只允许对处于激活状态的补丁进行去激活操作。

当备用主控板正在注册中且尚未注册成功时，如果进行补丁去激活操作，则系统会提示：是否确认继续执行补丁操作。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 去激活主控板上的补丁。

<Quidway> patch deactive
Info: Succeeded in deactivating patch ,Patch ID:SPH001

# 去激活所有的补丁。

<Quidway> patch deactive all
Deactivating the patch，please wait a moment ...
