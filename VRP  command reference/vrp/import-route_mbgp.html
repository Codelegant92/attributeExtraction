import-route（MBGP）

命令功能

import-route命令用来配置将其它协议路由信息引入到MBGP。

undo import-route命令用来取消引入其它协议的路由信息。

缺省情况下，MBGP将不引入其它协议的路由信息。

命令格式

import-route protocol [ process-id ] [ med med-value | route-policy route-policy-name ] *

undo import-route protocol [ process-id ]

参数说明

protocol：指定可引入的源路由协议。protocol可以是direct、IS-IS、OSPF、RIP、static等单播路由协议。

process-id：表示单播路由协议的进程号。当单播路由协议为IS-IS、OSPF或RIP时，需要指定process-id。

med med-value：指定引入路由的med。整数形式，取值范围是0～4294967295。

route-policy route-policy-name：从其他路由协议引入路由时，可以使用该参数指定的Route-Policy过滤器过滤路由和修改路由属性。
