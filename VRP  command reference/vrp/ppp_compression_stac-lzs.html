ppp compression stac-lzs

命令功能

ppp compression stac-lzs命令用来设置PPP协议启用Stac压缩算法。

undo ppp compression stac-lzs命令用来禁止在相应接口使用压缩。

缺省情况下，禁止使用压缩。

命令格式

ppp compression stac-lzs

undo ppp compression stac-lzs

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

在接口配置stac-lzs压缩，能通过无损的数据压缩减小数据帧的尺寸，但这项配置将增加路由器的负担，所以，当路由器负担重的时候，建议禁止该功能。

只有当链路两端都配置了stac-lzs时，该PPP链路才支持stac-lzs压缩。

使用实例

# 设置本端路由器支持stac-lzs压缩。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ppp compression stac-lzs
