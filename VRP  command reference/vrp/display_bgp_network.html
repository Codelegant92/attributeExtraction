display bgp network

命令功能

display bgp network命令用来显示BGP计划通过network命令发布的路由信息。

命令格式

display bgp network [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } network [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp ipv6 network [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name } network [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpnv4：查看VPNv4的通过network命令发布的路由信息。

all：查看所有VPNv4的通过network命令发布的VPNv4路由信息。

vpn-instance vpn-instance-name：查看通过network命令发布的指定VPN实例的路由信息。

ipv6：查看BGP发布的IPv6路由信息。

vpn6-instance vpn6-instance-name：查看指定IPv6 VPN实例发布的路由信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示BGP发布的路由信息。

<Quidway> display bgp network 
  BGP Local Router ID is 1.1.1.9
  Local AS Number is 10(Public)
  Network          Mask            Route-policy
  1.2.0.0          255.255.0.0
  3.0.0.0          255.0.0.0       Policy1
  4.4.4.0          255.255.255.0
表1 display bgp network命令输出信息描述

项目

描述

Network

本地发布的网络地址

Mask

网络地址的掩码

Route-policy

使用的路由策略

# 显示BGP VPNv4发布的路由信息。

<Quidway> display bgp vpnv4 all network
  BGP Local Router ID is 2.2.2.9
  Local AS Number is 100
  Route Distinguisher: 100:1
  (vpn1)
  Network          Mask            Route-policy
  4.4.4.4          255.255.255.255
  Route Distinguisher: 100:2
  (vpn2)
  Network          Mask            Route-policy
  5.5.5.5          255.255.255.255
表2 display bgp vpnv4 all network命令输出信息描述

项目

描述

Route Distinguisher

VPN实例的路由标识

# 显示BGP发布的IPv6路由信息。

<Quidway> display bgp ipv6 network 
  BGP Local Router ID is 5.5.5.5
  Local AS Number is 100(PublicV6)
  Network          Prefix            Route-policy
  100::            64
  200::            64

# 查看本端BGP通过network命令发布的指定VPN实例的VPNv6路由信息。

<Quidway> display bgp vpnv6 vpn6-instance vpn1 network
  BGP Local Router ID is 1.1.1.1
  Local AS Number is 100
  Route Distinguisher: 100:1
  (vpn1)
  Network          Prefix            Route-policy
  2000::           100               policy1
表3 display bgp vpnv6 network命令显示描述

项目

描述

BGP Local Router ID is

本地BGP的本地路由器ID号，注意其格式与IPv4地址一样

Local AS Number is

本地自治系统号

Route Distinguisher

VPN实例的路由标识符

Prefix

本端BGP发布的网段地址掩码
