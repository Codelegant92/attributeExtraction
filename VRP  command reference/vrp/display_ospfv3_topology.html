display ospfv3 topology

命令功能

display ospfv3 topology命令用来显示OSPFv3区域内拓扑信息。

命令格式

display ospfv3 [ process-id ] topology [ area area-id ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

area：指定区域的拓扑信息。

area-id：区域的标识，可以是十进制整数（取值范围是0～4294967295）或IPv4地址格式。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示OSPFv3区域1的拓扑信息。

<Quidway> display ospfv3 topology area 1

项目

描述

Type
SPF树节点类型

  • Rtr：Router

  • Net：Networ

ID(If-Index)

Router IDIf-Index是network类型中DR的接口ID

Bits
B比特和E比特，表示路由器的类型

  • B比特：ABR

  • E比特：ASBR

Metric

到该节点的路径开销

Next-Hop

到该节点的路径的下一跳

Interface

到该节点的路径的出接口
