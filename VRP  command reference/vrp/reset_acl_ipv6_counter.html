reset acl ipv6 counter

命令功能

reset acl ipv6 counter命令用来清除IPv6中ACL规则的匹配记录。

命令格式

reset acl ipv6 counter { acl6-number | all }

参数说明

acl6-number：指定要清除的ACL6的编号，整数形式，取值范围是1000～3999。其中1000～1999是基于接口的ACL6，2000～2999是基本ACL6，3000～3999是高级ACL6。

all：清除IPv6中所有ACL规则的匹配记录。

缺省级别

2：配置级

使用指南

使用该命令可以查看ACL的配置信息，包括规则数目、ACL步长等信息。

使用实例

# 清除编号为2100的ACL6规则匹配记录。

<Quidway> reset acl ipv6 counter 2100

# 清除IPv6的所有ACL规则匹配记录。

<Quidway> reset acl ipv6 counter all
