display ssh user-information

命令功能

display ssh user-information命令用来显示所有SSH用户的配置信息。

命令格式

display ssh user-information [ user-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

user-name：SSH用户名，字符串形式，长度范围是1～64。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行该命令，可显示SSH用户的用户名、密码，绑定的RSA公钥名，服务类型等信息。

使用实例

# 显示SSH用户client001的配置信息。

<Quidway> system-view
[Quidway] display ssh user-information client001
       User Name            : client001
       Authentication-type  : password
       User-public-key-name : -
       Sftp-directory       : -
       Service-type         : stelnet|sftp
       Authorization-cmd    : No
