telnet server enable

命令功能

telnet server enable命令用来启动Telnet服务器，允许Telnet用户登录。

undo telnet server enable命令用来关闭Telnet服务器，禁止Telnet用户登录。

缺省情况下，系统开启Telnet服务器功能。

命令格式

telnet server enable

undo telnet server enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

执行命令undo telnet server enable关闭Telnet服务器时，如果有通过Telnet登录的用户在线，该命令执行不成功。

Telnet服务器关闭后，用户只能通过SSH或者异步串行接口登录设备。

使用实例

# 启动Telnet服务器。

<Quidway> system-view
[Quidway] telnet server enable
