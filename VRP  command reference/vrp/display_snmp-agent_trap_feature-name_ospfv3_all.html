项目

描述

Feature name

特性名称

Trap number

告警数量

Trap name
告警名称

  • ospfv3VirtIfStateChange：使能系统当虚连接接口状态改变时发送告警信息

  • ospfv3NbrStateChange：使能系统当邻居状态改变时发送告警信息

  • ospfv3VirtNbrStateChange：使能系统当邻居虚连接状态改变时发送告警信息

  • ospfv3IfConfigError：使能系统当接口配置错误时发送告警信息

  • ospfv3VirtIfConfigError：使能系统当虚连接接口配置错误时发送告警信息

  • ospfv3IfRxBadPacket：使能系统当接口收到坏包时发送告警信息

  • ospfv3VirtIfRxBadPacket：使能系统当虚连接接口收到坏包时发送告警信息

  • ospfv3IfStateChange：使能系统当接口状态改变时发送告警信息

  • ospfv3RestartStatusChange：使能系统当GR Restarter状态改变时发送告警信息

  • ospfv3NbrRestartHelperStatusChange：使能系统当GR Helper状态改变时发送告警信息

  • ospfv3VirtNbrRestartHelperStatusChange：使能系统当邻居平滑重启协助状态改变时发送告警信息

Status

告警开关状态：

  • off：关闭

  • on：打开

Excessive Flag

是否海量告警：

  • Y：海量告警

  • N：非海量告警
