display logbuffer

命令功能

display logbuffer命令用来显示日志缓冲区记录的信息。

缺省情况下，不带任何参数表示显示日志缓冲区记录的所有信息。

命令格式

display logbuffer [ summary | level severity | module module-name | size value | slot slot-id ]* [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

summary：显示概要的日志缓冲区日志信息。

level：显示指定信息的级别。

severity：显示指定的信息级别，取值范围是0～7。

module：模块。

module-name：模块名称。

size：显示指定信息的条数。

value：显示指定的日志缓冲区内信息的条数，取值范围1～1024。

slot-id：显示指定的槽号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果当前日志缓冲区内的信息数少于用户指定的value，则显示实际条数的日志信息。

使用实例
