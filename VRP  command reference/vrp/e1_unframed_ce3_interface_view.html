e1 unframed（CE3接口视图）

命令功能

e1 unframed命令用来设置CE3接口的E1通道工作在净通道模式，创建一个速率为64kbit/s的同步串口。

undo e1 unframed命令用来恢复缺省设置。

缺省情况下，E1通道工作在通道化模式，需执行e1 channel-set命令进行时隙捆绑，才能形成一个或多个速率为N×64kbit/s的同步串口。

命令格式

e1 e1-number unframed

undo e1 e1-number unframed

参数说明

e1-number：E1通道的顺序号。整数形式，取值范围是1～16。

缺省级别

2：配置级

使用指南

只有当CE3接口工作在通道化模式下时，才能配置该命令。

当E1配置成净通道模式后（unframed），它将不包含帧控制信息，也不分时隙，不能进行时隙捆绑。

使用实例

# 将CE3接口1/0/0的第3个E1通道设置到净通道模式。

<Quidway> system-view
[Quidway] controller e3 1/0/0
[Quidway-E3 1/0/0] e1 3 unframed
