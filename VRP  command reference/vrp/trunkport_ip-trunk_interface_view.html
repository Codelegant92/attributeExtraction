trunkport (IP-Trunk接口视图)

命令功能

trunkport interface-type { interface-number1 [ to interface-number2 ] } &<1-16>命令用来在IP-Trunk接口视图下批量增加成员接口。

undo trunkport interface-type { interface-number1 [ to interface-number2 ] } &<1-16>命令用来在IP-Trunk接口视图下批量删除成员接口。

trunkport interfacetype interface-number命令用来在IP-Trunk接口视图下增加一个成员接口。

undo trunkport interface-type interface-number命令用来在IP-Trunk接口视图下删除一个成员接口。

命令格式

trunkport interface-type { interface-number1 [ to interface-number2 ] } &<1-16>

undo trunkport interface-type { interface-number1 [ to interface-number2 ] } &<1-16>

trunkport interface-type interface-number

undo trunkport interface-type interface-number

参数说明

interface-type：接口类型，例如POS接口。

interface-number：接口编号。

使用指南

每个IP-Trunk接口下最多可以包含16个成员接口。IP-Trunk接口中只能添加POS接口。

加入IP-Trunk的接口，链路层协议必须为HDLC。

输入端口范围时，应保证输入的端口格式正确：

  • 关键字to之后的接口号要大于to之前的接口号。
  • 要保证采用to形式输入的接口类型相同。
  • 两者之间包含的接口都存在。

执行一次trunkport命令，最多可以使用16次to形式设置接口范围。

设置成员接口范围时需注意，一次最多批量添加或删除16个成员接口。

使用实例

# 在IP-Trunk1下批量增加4个成员接口POS6/0/0～6/0/3。

<Quidway> system-view
[Quidway] interface ip-trunk 1
[Quidway-Ip-Trunk1] trunkport pos 6/0/0 to 6/0/3
