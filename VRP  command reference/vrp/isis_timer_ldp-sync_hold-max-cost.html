isis timer ldp-sync hold-max-cost

命令功能

isis timer ldp-sync hold-max-cost命令用来配置IS-IS在本地设备的LSA中通告最大metric值的时间间隔。

undo isis timer ldp-sync hold-max-cost命令用来恢复为缺省配置。

缺省情况下，IS-IS在本地设备的LSA中通告最大metric值的时间间隔是10秒。

命令格式

isis timer ldp-sync hold-max-cost value

undo isis timer ldp-sync hold-max-cost

参数说明

value：指定IS-IS在本地设备的LSA中通告最大metric值的时间间隔。整数形式，取值范围是0～65535，单位是秒。

缺省级别

2：配置级

使用指南

除Loopback和NULL0接口，其他接口均支持该命令。

使用实例

# 配置IS-IS在本地设备的LSA中通告最大metric值的时间间隔为8秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] isis enable 1
[Quidway-GigabitEthernet1/0/0] isis timer ldp-sync hold-max-cost 8
