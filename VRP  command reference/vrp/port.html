port

命令功能

port命令用来向VLAN中添加一个或一组端口。

undo port用来从VLAN中删除一个或一组端口。

命令格式

port interface-type { interface-number1 [ to interface-number2 ] } &<1-10>

undo port interface-type { interface-number1 [ to interface-number2 ] } &<1-10>

参数说明

interface-type：接口类型。

interface-number：接口序号。

视图

VLAN视图

缺省级别

2：配置级

使用指南

只有交换式接口才能加入到VLAN中。

Trunk类型的端口不能通过port命令加入VLAN。

输入端口范围时，应保证输入的端口格式正确，关键字to之后的端口号要大于to之前的端口号，并要保证采用to形式输入的端口类型相同，且两者之间包含的端口都存在。

一条port命令中，最多可以使用10次to形式输入10个端口范围。

使用port命令时，必须保证加入到VLAN的接口类型相同，FE和GE不能混合加入同一个VLAN。

使用实例

# 向VLAN3中加入端口GE 6/0/0～6/0/3。

<Quidway> system-view
[Quidway] vlan 3
[Quidway-vlan3] port gigabitethernet 6/0/0 to 6/0/3

# 从VLAN3中删除端口GE 6/0/1。

<Quidway> system-view
[Quidway] vlan 3
[Quidway-vlan3] undo port gigabitethernet 6/0/1
