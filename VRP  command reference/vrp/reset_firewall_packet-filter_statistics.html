reset firewall packet-filter statistics

命令功能

reset firewall packet-filter statistics命令用来清除防火墙的统计信息。

命令格式

reset firewall packet-filter statistics { all | interface interface-type interface-number }

参数说明

all：表示清除所有接口的接口过滤报文统计信息。

interface：表示清除某接口的接口过滤报文统计信息。

interface- type：为接口类型。

interface-number：接口编号。

用户视图

缺省级别

使用指南

清除统计信息后，以前的统计信息将无法恢复。

使用实例

# 清除接口E3/1/0的接口过滤报文统计信息。

<Quidway> reset firewall packet-filter statistics interface Gigabitethernet  3/1/0
