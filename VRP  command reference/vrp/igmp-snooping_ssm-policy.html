igmp-snooping ssm-policy

命令功能

igmp-snooping ssm-policy命令用来配置IGMP Snooping的SSM组策略。策略允许的组都将作为SSM范围内的组对待。

undo igmp-snooping ssm-policy命令用来取消SSM组策略。

缺省情况下，SSM的组范围是232.0.0.0～232.255.255.255。

命令格式

igmp-snooping ssm-policy basic-acl-number

undo igmp-snooping ssm-policy

参数说明

basic-acl-number：基本IP访问控制列表号，定义一个组播组范围。取值范围是2000～2999。

使用指南

SSM组范围可以超出232.0.0.0～232.255.255.255。SSM组范围之外的组地址一律适用于ASM。

使用igmp-snooping ssm-policy basic-acl-number命令前，必须创建参数basic-acl-number所指定的ACL，否则配置无效。

使用实例

# 配置组地址225.1.1.123作为SSM范围内的组。

<Quidway> system-view
[Quidway] acl number 2008
[Quidway-acl-basic-2008] rule permit source 225.1.1.123 0
[Quidway-acl-basic-2008] quit
[Quidway] igmp-snooping ssm-policy 2008
