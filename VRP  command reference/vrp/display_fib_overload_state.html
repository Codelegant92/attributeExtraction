display fib overload state

命令功能

display fib overload state命令用来查看接口板IPv4 FIB模块的当前工作状态以及超限处理模式。

命令格式

display fib overload state slot slot-number

参数说明

slot slot-number：需要查看IPv4 FIB模块状态信息的接口板编号。

使用指南

执行命令display fib overload state，可以查看接口板IPv4 FIB模块的当前工作状态和超限处理模式。

工作状态包括正常工作状态、超限挂起状态和超限转发状态。超限处理模式包括超限挂起和超限转发。

当接口板应用的IPv4 FIB策略路由前缀总数超过接口板的IPv4 FIB路由前缀容量时，接口板就会进入超限处理模式。

使用实例

# 查看2号接口板上IPv4 FIB模块当前的工作状态和超限处理模式。

<Quidway> display fib overload state slot 2
  Overload mode:
    Overload suspend mode.
  Overload state:
    Normal state.

项目

描述

Overload mode:
本接口板IPv4 FIB模块应用的超限处理模式。可能的处理模式有：
  • Overload suspend mode：超限挂起模式
  • Overload forward mode：超限转发模式

Overload state:
本接口IPv4 FIB模块当前的工作状态。可能的工作状态有：
  • Normal state：正常工作状态
  • Overload suspend state：超限挂起状态
  • Overload forward state：超限转发状态
