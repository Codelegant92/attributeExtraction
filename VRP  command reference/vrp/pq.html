pq

命令功能

pq命令用来将帧中继虚电路的队列类型设置为优先队列（Priority Queueing）。

undo pq命令用来将虚电路的队列类型恢复为FIFO。

缺省情况下，帧中继虚电路的队列类型为FIFO。

命令格式

pq pql list-number

undo pq

参数说明

list-number：优先队列的组号，取值范围是1～16。

缺省级别

2：配置级

使用指南

对同一个帧中继class重复使用本命令，最后一次配置将覆盖前面的配置。

使用实例

# 将优先队列的第10组应用到名为test1的帧中继class上。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] pq pql 10
