display pim ipv6 rp-info

命令功能

display pim ipv6 rp-info命令用来查看组播组对应的RP信息。

命令格式

display pim ipv6 rp-info [ ipv6-group-address ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv6-group-address：指定组播组地址。十六进制形式，取值范围是FF00::～FFFF:FFFF:FFFF:FFFF: FFFF:FFFF:FFFF:FFFF。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

RP信息包括通过BSR机制计算的RP信息和静态RP信息。如果不指定组地址，将显示所有组的RP信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

请参见IPv4 PIM配置命令。

使用实例

# 显示组播组FF0E::1对应的RP。

<Quidway> display pim ipv6 rp-info ff0e::1
 VPN-Instance: public net

     Priority: 192
     Uptime: 00:00:52
     Expires: 00:01:38
 RP mapping for this group is: 2001:AA::99 (local host)
