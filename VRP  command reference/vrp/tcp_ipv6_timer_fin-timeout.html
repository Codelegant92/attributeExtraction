tcp ipv6 timer fin-timeout

命令功能

tcp ipv6 timer fin-timeout命令用来设置TCP的FIN-Wait定时器值。

undo tcp ipv6 timer fin-timeout命令用来恢复为缺省设置。

缺省情况下，TCP的FIN-WAIT定时器值是675秒。

命令格式

tcp ipv6 timer fin-timeout timer-value

undo tcp ipv6 timer fin-timeout

参数说明

timer-value：指定TCP的FIN-WAIT定时器值。整数形式，单位是秒，取值范围是76～3600。缺省值为675秒。
