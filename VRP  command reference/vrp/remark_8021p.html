remark 8021p

命令功能

remark 8021p命令用来重新标记VLAN帧的优先级。

undo remark 8021p命令用来取消标记VLAN帧的优先级。

命令格式

remark 8021p 8021p-value

undo remark 8021p

参数说明

8021p-value：标记的VLAN优先级。整数形式，取值范围是0～7。

视图

流行为视图

缺省级别

2：配置级

使用指南

标记VLAN帧的优先级只能配置在入方向上，并且只对以太类型的接口有意义。

使用实例

# 配置标记VLAN帧的优先级值为4。

<Quidway> system-view
[Quidway] traffic behavior database
[Quidway-behavior-database] remark 8021p 4
