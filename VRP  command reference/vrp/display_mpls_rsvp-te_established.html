display mpls rsvp-te established

命令功能

display mpls rsvp-te established命令用来查看基于接口的RSVP资源预留和经过该接口的LSP基本信息。

命令格式

display mpls rsvp-te established [ interface interface-type interface-number peer-ip-address ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type interface-number：接口名称，包括接口类型和接口号。

peer-ip-address：RSVP邻居节点上与本设备相连的接口的IP地址。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
