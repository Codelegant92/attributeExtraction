display ipv6 pathmtu

命令功能

display ipv6 pathmtu命令用来查看IPv6的PMTU项信息。

命令格式

display ipv6 pathmtu [ vpn6-instance vpn-instance-name ] ipv6-address

display ipv6 pathmtu [ vpn6-instance vpn-instance-name ] { all | dynamic | static } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vpn6-instance vpn6-instance-name：IPv6 VPN实例的名称，用来查看指定IPv6 VPN实例的PMTU项。字符串形式，区分大小写，长度范围是1～31。

ipv6-address：显示匹配指定的IPv6地址的PMTU项。

all：表示显示所有PMTU项。

dynamic：表示显示所有动态PMTU项。

static：表示显示所有静态PMTU项。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当中间路由设备出接口的MTU值小于源端主机的MTU值时，通过PMTU发现机制可以确定一个可在路径上传输的最大报文的大小，即PMTU的值。

使用实例

# 查看所有的PMTU项。

<Quidway> display ipv6 pathmtu all
IPv6 Destination Address      ZoneID  PathMTU  LifeTime(M)  Type
2002::1                       0       1300     9            Dynamic
2001::1                       0       1500     -            Static
------------------------------------------------------------------
Total: 2      Dynamic: 1     Static: 1
表1 display ipv6 pathmtu命令输出信息描述

项目

描述

IPv6 Destination Address

IPv6主机地址

ZoneID

地址区域

PathMTU

IPv6地址的PMTU值

LifeTime(M)

剩余生存时间，单位是分钟，刷新精度是1分钟。对于静态PMTU项，显示为“-”。

Type

该表项的类型，可能的取值如下

  • Dynamic：动态项
  • Static：静态项（通过CLI配置）

# 查看本设备上vpn1的Path MTU信息。

<Quidway> display ipv6 pathmtu vpn6-instance vpn1 all
IPv6 Destination Address             ZoneID   PathMTU    LifeTime(M)   Type
2000::2                              1        1500       10            Dynamic
2000::1                              1        1600       -             Static
