oam-bind ingress efm interface egress mpls oam tunnel

命令功能

oam-bind ingress efm interface egress mpls oam tunnel命令用于配置EFM OAM向MPLS OAM传递故障。

undo oam-bind ingress efm interface egress mpls oam tunnel命令用于取消EFM OAM向MPLS OAM传递故障。

命令格式

oam-bind ingress efm interface interface-type interface-number egress mpls oam tunnel tunnel-number

undo oam-bind ingress efm interface interface-type interface-number egress mpls oam tunnel tunnel-number

参数说明

tunnel-number：隧道的接口编号，代表启用MPLS OAM检测功能的隧道。

interface-type interface-number：以太网接口类型和编号，取值包括：ethernet和gigabitethernet。该接口必须已经使能EFM OAM功能。

使用指南

配置EFM OAM与LSP的隧道入节点绑定，OAM管理模块通过接口向MPLS OAM传递故障。

配置该命令时，相关的MPLS隧道必需已经创建，指定的接口也已使能EFM功能。

使用实例

# 配置EFM OAM通过接口GE3/0/1向MPLS OAM传递故障。

<Quidway> system-view
[Quidway] oam-mgr
[Quidway-oam-mgr] oam-bind ingress efm interface gigabitethernet 3/0/1 egress mpls oam tunnel 1/0/0
