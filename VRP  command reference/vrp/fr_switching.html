fr switching

命令功能

fr switching命令用来允许帧中继DCE或NNI接口进行PVC交换。

undo fr switching命令用来禁止帧中继DCE或NNI接口进行PVC交换。

缺省情况下，禁止帧中继交换。

命令格式

fr switching

undo fr switching

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

配置允许帧中继DCE或NNI接口进行PVC交换之后，还需要配置进行PVC交换的路由。

使用实例

# 使能帧中继交换功能。

<Quidway> system-view
[Quidway] fr switching
