recording-mode

命令功能

recording-mode命令用来配置当前记录方案使用的记录方法。

命令格式

recording-mode hwtacacs template-name

undo recording-mode

参数说明

template-name：与记录方法关联的HWTACACS服务器模板的名称，字符串形式，长度范围是1～32。

视图

记录方案视图

缺省级别

3：管理级

使用指南

在使用recording-mode hwtacacs命令前，HWTACACS服务器模板必须已经创建完成。

使用实例

# 对于名为scheme1的记录方案，配置其使用名为test的hwtacacs模板。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] recording-scheme scheme1
[Quidway-aaa-recording-scheme1] recording-mode hwtacacs test
