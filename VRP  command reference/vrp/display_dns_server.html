display dns server

命令功能

display dns server命令用来查看当前域名服务器配置和顺序。

命令格式

display dns server [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在完成DNS的配置后，使用display dns server命令可以查看当前域名服务器的配置和顺序。

使用实例

# 查看域名服务器配置。

<Quidway> display dns server
IPv4 Dns Servers :
Domain-server        IpAddress
     1               172.16.1.1
     2               172.16.1.2

IPv6 Dns Servers :
Domain-server Ipv6Address            (Interface Name)
     1        3001::2
     2        FE80::2                 GigabitEthernet6/0/0
表1 display dns server命令输入信息描述

项目

描述

IPv4 Dns Servers

查看IPv4域名服务器的配置。

Domain-server

编号，表示域名服务器的配置顺序。

IpAddress

域名服务器的IP地址。解析的时候，首先使用第一个域名服务器172.16.1.1，如果不成功的话，则使用第二个域名服务器172.16.1.2。

IPv6 Dns Servers

查看IPv6域名服务器的配置。

Ipv6Address

IPv6域名服务器

Interface Name

接口名称（只对应IPv6链路本地地址）
