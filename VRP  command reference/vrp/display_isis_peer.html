display isis peer

命令功能

display isis peer命令用来查看IS-IS的邻居信息。

命令格式

display isis peer [ verbose ] [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

verbose：显示IS-IS邻居的详细信息。

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称，，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用参数verbose除了可以显示display isis peer的所有信息外，还将显示邻居的区域地址、处于Up状态的时长、以及直连接口的IP地址。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示IS-IS邻居信息。

<Quidway> display isis peer
                          Peer information for ISIS(1)

0000.0000.0001  GigabitEthernet1/0/0  0000.0000.0001.01  Up   24s      L2(L1L2) 64
