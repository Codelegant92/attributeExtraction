e1 set clock（CE3接口视图）

命令功能

e1 set clock命令用来设置CE3接口的E1通道的时钟模式。

undo e1 set clock命令用来恢复缺省设置。

缺省情况下，E1通道作为从时钟（slave）。

命令格式

e1 e1-number set clock { master | slave }

undo e1 e1-number set clock

参数说明

e1-number：E1通道的顺序号。整数形式，取值范围是1～16。

master：设置E1通道作为主时钟。

slave：设置E1通道作为从时钟。

视图

CE3接口视图

缺省级别

2：配置级

使用指南

只有当CE3接口工作在CE3工作模式下时，才能配置该命令。

建议对E1通道配置的时钟模式与所在的CE3接口保持一致。

使用实例

# 设置CE3接口1/0/0的3号E1通道为主时钟，使用内部时钟信号。

<Quidway> system-view
[Quidway] controller e3 1/0/0
[Quidway-E3 1/0/0] e1 3 set clock master
