display cluster-increment-result

命令功能

display cluster-increment-result命令用来查看成员交换机下发增量配置的结果。

命令格式

display cluster-increment-result

参数说明

无

视图

增量配置视图

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

当使用increment–run命令给成员交换机下发增量配置后，可以使用本命令用来查看增量下发增量配置的结果。

使用实例

# 查看成员交换机下发增量配置的结果。

<Quidway> system-view
<Quidway> cluster
[Quidway-cluster] display cluster-increment-result
The result of member switchs executing increment commands:
------------------------------------------------------------------------------
SN   Device              MacAddress     IpAddress      Result    CommandId
------------------------------------------------------------------------------
1    NE40E               0001-0001-0001 10.0.0.1       running     10
2    NE40E               0002-0002-0002 10.0.0.2       success     -
3    NE40E               0003-0003-0003 10.0.0.3       failure     20

该显示信息主要内容解释见表1。

表1 display cluster-increment-result命令输出信息描述

项目

描述

SN
成员编号。

  • “0”表示命令交换机。

  • 其它数字表示成员交换机。

Device

设备的类型。

MAC Address

设备的MAC地址。

IpAddress

设备的集群IP地址。

Result
下发增量配置的结果：

  • success

  • failure

  • running

  • error

CommandId

正在执行、执行失败或者执行错误的增量配置编号。

如果执行成功，此处显示为“-”。
