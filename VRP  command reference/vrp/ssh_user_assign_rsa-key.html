ssh user assign rsa-key

命令功能

ssh user assign rsa-key命令用来为用户分配一个已经存在的公钥。

undo ssh user assign rsa-key命令用来删除此用户和公钥之间的对应关系。

命令格式

ssh user user-name assign rsa-key key-name

undo ssh user user-name assign rsa-key

参数说明

user-name：AAA定义的有效SSH用户名，字符串形式，长度为1～64。

key-name：配置的客户端公共密钥名，字符串形式，长度为1～30。

使用指南

使用该命令为用户分配公钥时，如果用户已经被分配了公钥，以最后一次分配的公钥为准。

为用户分配公钥时，如果指定的用户的user-name不存在，则新建一个用户名为user-name的SSH用户，认证方式为所配置的认证方式。

新配置的用户公钥待下次登录时生效。

使用实例

# 为用户zhangsan分配密钥key1。

[Quidway] ssh user zhangsan assign rsa-key key1
