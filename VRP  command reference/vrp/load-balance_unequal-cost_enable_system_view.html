load-balance unequal-cost enable（系统视图）

命令功能

load-balance unequal-cost enable命令用来在系统视图下启动非等值负载分担功能。

undo load-balance unequal-cost enable命令用来在系统视图下关闭非等值负载分担功能。

缺省情况下，关闭系统视图下的非等值负载分担功能。

命令格式

load-balance unequal-cost enable

undo load-balance unequal-cost enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

用户在系统视图下可以采用此命令启动或关闭设备的非等值负载分担功能。在启动或关闭设备的非等值负载分担功能之后，设备会下发整机FIB，FIB的刷新采用慢刷机制，每24小时刷新一次，每次向所有的接口板刷新路由，保证主控板和接口板路由的一致性。

支持非等值负载分担功能的接口有Ethernet接口、GigabitEthernet接口、POS接口、ATM接口、Serial接口以及MFR接口、MP接口、Eth-Trunk接口、IP-Trunk接口、Tunnel接口。

在启动设备的非等值负载分担功能时：

  • 如果出接口支持非等值负载分担，命令display fib verbose的输出信息中会显示出接口的带宽值。
  • 如果出接口不支持非等值负载分担，命令display fib verbose的输出信息中不会显示任何接口的带宽信息。

在未启动设备的非等值负载分担时，命令display fib verbose的输出信息里，也不会显示任何接口的带宽信息。

为了防止频繁进行功能启动或关闭对设备产生严重影响，使能非等值负载分担功能和去使能非等值负载分担功能的命令间隔不得小于5分钟。

使用实例

# 使能UCMP功能。

<Quidway> system-view
[Quidway] load-balance unequal-cost enable
