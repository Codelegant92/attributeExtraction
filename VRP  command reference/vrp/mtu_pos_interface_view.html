mtu（POS接口视图）

命令功能

mtu命令用来设置POS接口最大传输单元（MTU）的大小。

undo mtu命令用来恢复其缺省值。

命令格式

mtu mtu

undo mtu

参数说明

mtu：POS接口MTU的大小。整数形式，取值范围是46～16352，单位是字节。缺省值是4470字节。

视图

POS接口视图

缺省级别

2：配置级

使用指南

注意：

当接口上运行IPv6时，如果接口的MTU值小于1280字节，可能会使IPv6在该接口上工作不正常。所以当接口上运行IPv6时，建议设置该接口的MTU值不小于1280。

使用mtu命令改变接口最大传输单元MTU后，需要先对接口执行shutdown命令，再执行undo shutdown命令将接口重启，以保证设置的MTU生效。

使用实例

# 设置POS接口1/0/0的MTU值为1492字节。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] mtu 1492
