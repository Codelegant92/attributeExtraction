display ipv6 interface

命令功能

display ipv6 interface命令用来查看接口信息。

命令格式

display ipv6 interface [ interface-type interface-number | brief ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

inerface-type：指定接口类型。

interface-number：指定接口编号。

brief：显示接口的摘要信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在执行该命令前请确保已在相应的接口下配置IPv6地址，否则查看不到接口显示信息。

说明：

  • 接口的描述信息可以通过命令display interface description查看。

  • 接口的详细运行信息和统计信息可以通过命令display interface查看。
