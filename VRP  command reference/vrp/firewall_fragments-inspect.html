firewall fragments-inspect

命令功能

firewall fragments-inspect命令用来打开分片报文检测开关，undo firewall fragments-inspect命令用来关闭分片报文检测开关。

缺省情况下，分片检测开关为关闭。

命令格式

firewall fragments-inspect { all | slot slot-id }

undo firewall fragments-inspect { all | slot slot-id }

参数说明

all：配置作用于所有接口板。

slot-id：为指定的接口板槽号。

使用指南

此命令用来实现包过滤防火墙的精确匹配功能。打开分片检测开关就可以进行分片报文的精确匹配，包过滤防火墙将记录分片报文的状态，根据高级ACL规则对三层（IP层）以外的信息进行精确匹配。

包过滤防火墙对分片报文的状态记录需要消耗一定的系统资源，如果不使用精确匹配模式，则可以关闭此功能，以提高系统运行效率，减小系统开销。

只有打开了此分片报文检测开关，才可以对分片报文进行精确匹配。

使用实例

# 打开所有接口板的分片检测开关。

<Quidway> system-view
[Quidway] firewall fragments-inspect all
