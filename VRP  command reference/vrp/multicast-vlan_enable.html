multicast-vlan enable

命令功能

multicast-vlan enable命令用来配置视图所在VLAN为组播VLAN。

undo multicast-vlan enable命令用来将视图所在VLAN恢复成普通VLAN。

命令格式

multicast-vlan enable

undo multicast-vlan enable

参数说明

无

视图

VLAN视图

缺省级别

2：配置级

使用指南

使用此命令之前，先全局使能IGMP Snooping 功能。

系统支持的最大组播VLAN个数为16个。

配置组播VLAN复制功能时需要配置这条命令。该命令在普通VLAN下执行才能生效。普通VLAN配置成组播VLAN后，就可以向它增加用户VLAN，这个VLAN收到的组播流就可以转发到它所映射的用户VLAN中去了，这样就实现了跨VLAN复制。

将组播VLAN恢复成普通VLAN要遵循如下条件：

  • 组播VLAN内不能有用户VLAN，应该删除组播VLAN内的所有用户VLAN。

  • 组播VLAN不能存在于组播保护组中，应该先删除组播所在的保护组。

  • 组播VLAN不能绑定OAM故障检测，应该先解除和OAM的绑定，删除保护组会自动解除和OAM的绑定。

使用实例

# 配置VLAN 10为组播VLAN。

<Quidway> system-view
[Quidway] vlan 10
[Quidway-vlan10] igmp-snooping enable
[Quidway-vlan10] multicast-vlan enable
