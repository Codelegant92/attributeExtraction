ntp-service multicast-client

命令功能

ntp-service multicast-client命令用来配置NTP组播客户模式。

undo ntp-service multicast-client命令用来取消NTP组播客户模式。

缺省情况下，不配置组播客户服务模式。

命令格式

ntp-service multicast-client [ ip-address ]

undo ntp-service multicast-client [ ip-address ]

参数说明

ip-address：组播IP地址，D类地址。缺省为224.0.1.1。

视图

接口视图

缺省级别

2：配置级

使用指南

指定本地设备上的一个接口来接收NTP组播消息包，本地设备运行在client（multicast-client）模式。

使用实例

# 配置在GE1/0/0接口上接收NTP组播消息包，组播消息包对应的组播组地址为224.0.1.1。

[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ntp-service multicast-client 224.0.1.1
