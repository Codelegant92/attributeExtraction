track admin-vsi

命令功能

track admin-vsi命令用来把业务VSI与指定的管理VSI进行绑定。

undo track admin-vsi命令用来删除业务VSI与管理VSI的绑定关系。

缺省情况下，没有配置业务VSI与任何管理VSI绑定。

命令格式

track admin-vsi vsi-name

undo track admin-vsi vsi-name

参数说明

vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

视图

VSI视图

缺省级别

2：配置级

使用指南

对于Kompella方式的VPLS，使用track admin-vsi命令前，需要先执行pwsignal命令和route-distinguisher命令；对于Martini方式的VPLS，使用track admin-vsi命令前，需要先执行pwsignal命令和vsi-id命令。

track admin-vsi命令只能在业务VSI中下使用，不能在管理VSI中下使用。

一个业务VSI只能绑定到一个管理VSI中。

使用实例

# 配置业务VSI(v200)和管理VSI(v100)绑定。

<Quidway> system-view
[Quidway] mpls l2vpn 
[Quidway-l2vpn] quit
[Quidway] vsi v100 static
[Quidway-vsi-v100] pwsignal ldp
[Quidway-vsi-v100-ldp] vsi-id 100
[Quidway-vsi-v100-ldp] quit
[Quidway-vsi-v100] admin-vsi
[Quidway-vsi-v100] quit
[Quidway] vsi v200 static
[Quidway-vsi-v200] pwsignal ldp
[Quidway-vsi-v200-ldp] vsi-id 200
[Quidway-vsi-v200-ldp] quit
[Quidway-vsi-v200] track admin-vsi v100
