display controller e3（CE3接口）

命令功能

display controller e3命令用来查看CE3接口的配置和状态信息。

命令格式

display controller e3 [ controller-number ]

参数说明

controller-number：接口号，包括槽号、卡号、端口号三部分。

CE3接口的编号，包括槽号、卡号、端口号三部分。如果不指定编号，将显示所有CE3接口的配置和状态信息。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

项目

描述

E3 3/0/0 current state :
CE3接口当前的物理状态：

  • UP：物理状态正常启动。

  • DOWN：物理连接不正常。

  • Administratively down：如果网络管理员在该接口执行命令，将显示状态为Administratively down。

Frame-format

CE3接口的帧格式。

code

CE3接口的线路编解码格式。

clock
时钟模式：

  • slave：从时钟模式。

  • master：主时钟模式。

national-bit 1

国际位为1。

Current mode is CE3
接口当前的工作模式：

  • CE3：接口工作在CE3模式下，表示通道化工作模式。

  • E3：接口工作在E3模式下，表示净通道工作模式。

loopback not set

没有使能CE3接口环回功能。只有在进行某些特殊功能测试时，才将CE3接口设为环回。

Alarm:ERROR:
告警信息和错误信息。可能出现的错误类型有以下几类。

  • LOS：Loss of Signal，帧同步丢失。

  • LOF：Loss of Frame，帧丢失。

  • CERR：CRC Error，循环冗余校验错。

  • LCV：Line Code Violation，线路码错误。

  • FEBE：Far End Block Error，远端块错。这种错误只有在E1通道采用CRC4的帧格式时才可能出现。

  • BPV：Bipolar Violation，线路码极性错误。

  • EXZ：Excessive Zeros，连续多0错误。

  • FrmErr：FRAME ERROR，帧错误。

  • YEL：告警等级为黄色告警。

  • AIS：Alarm Indication Signal，是指线路收到全“1”的帧格式。

E3 3/0/0 CE1 1 is

CE3接口下的CE1通道的状态。
