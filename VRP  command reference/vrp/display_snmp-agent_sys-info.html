display snmp-agent sys-info

命令功能

display snmp-agent sys-info命令用来显示当前SNMP设备的系统信息。

命令格式

display snmp-agent sys-info [ contact | location | version ] *

参数说明

contact：显示当前设备节点的联系信息。

location：显示当前设备节点的物理位置信息。

version：显示当前设备节点中运行的SNMP版本号。

使用指南

使用display snmp-agent sys-info命令用来显示当前设备的系统维护联络信息、当前设备的物理位置、SNMP使能的版本信息。

使用实例

# 显示设备系统信息。

<Quidway> display snmp-agent sys-info
   The contact person for this managed node:
           R&D Beijing, Huawei Technologies co.,Ltd.
   The physical location of this node:
           Beijing China
   SNMP version running in the system:
           SNMPv3

项目

描述

The contact person for this managed node

内容为被管理设备相关人员的标识及联系方法。可以通过此参数，将重要信息存储在路由器中，以便出现紧急问题时查询。

The physical location of this node

表示被管理设备的位置。

SNMP version running in the system

SNMP的版本信息：SNMPv1，SNMPv2c，SNMPv3。
