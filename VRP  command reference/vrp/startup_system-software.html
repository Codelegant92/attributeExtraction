startup system-software

命令功能

startup system-software命令用来配置系统下次启动时使用的系统软件文件名。

缺省情况下，配置主控板下次启动时使用的系统软件文件名。

命令格式

startup system-software system-file [ slave-board ]

参数说明

system-file：系统软件的文件名。字符串形式，格式为[ drive-name ][ file-name ]，长度范围是4～64。如果未指定drive-name，则此值为默认的设备名。

slave-board：配置备板下次启动时使用的系统软件文件名。

使用指南

配置文件必须以“.cc” 作为扩展名，而且系统启动配置文件必须存放在存储设备的根目录下。存储设备包括硬盘（Hard disk）和Flash，如果您选择缺省路径，系统配置文件将存储在Flash的根目录下。

使用实例

# 配置下次启动使用的系统文件。

<Quidway> startup system-software vrpsoft.cc
