mpls te path metric-type

命令功能

在MPLS视图下，mpls te path metric-type命令用来指定未配置度量类型的隧道选路时使用的链路度量类型。

在Tunnel接口视图下，mpls te path metric-type命令用来设置当前隧道选路时使用的链路度量类型。

undo mpls te path metric-type命令只能在Tunnel接口视图下使用，用于恢复缺省设置。

缺省情况下，隧道选路时使用链路的TE度量。

命令格式

mpls te path metric-type { igp | te }

undo mpls te path metric-type

参数说明

igp：使用IGP度量。

te：使用TE度量。

视图

MPLS视图、Tunnel接口视图

缺省级别

2：配置级

使用指南

如果在隧道接口视图下没有配置mpls te path metric-type，则使用MPLS视图下的度量类型；否则使用隧道接口视图下的度量类型。

使用实例

# 对所有未指定度量类型的TE隧道，设置其在进行路径计算时使用IGP度量。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls te path metric-type igp

# 设置为Tunnel1/0/0计算路径时使用TE度量。

[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol mpls te
[Quidway-Tunnel1/0/0] mpls te path metric-type te
