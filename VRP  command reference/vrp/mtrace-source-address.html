mtrace-source-address

命令功能

mtrace-source-address命令配置NQA mtrace测试例的组播源地址。

undo mtrace-source-address命令用来恢复缺省情况。

缺省情况下，NQA mtrace测试例未指定组播源地址。

命令格式

mtrace-source-address ipv4 ip-address

undo mtrace-source-address

参数说明

ipv4 ip-address：指定组播源地址，属于单播IPv4地址。

视图

NQA视图

缺省级别

2：配置级

使用指南

在NQA mtrace测试例中，必须指定组播源地址，且只能对mtrace类型的测试例配置组播源地址。

不能更改正在运行的测试例的组播源地址。

使用实例

# 设置测试例user test的组播源地址

[Quidway-nqa-user-test] mtrace-source-address ipv4 192.168.0.1

# 取消测试例user test的组播源地址

[Quidway-nqa-user-test] undo mtrace-source-address
