display mld ssm-mapping interface

命令功能

display mld ssm-mapping interface命令用来显示使能SSM映射的接口上的相关信息。

命令格式

display mld ssm-mapping interface [ interface-type interface-number [ group group-address ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-type interface-number：显示指定接口的SSM映射信息。

group group-address：显示与指定组关联的源地址列表。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display mld ssm-mapping interface命令时，若指定了interface-type interface-number参数则仅显示在该接口下的MLD SSM Mapping配置和运行信息。如果该接口未使能MLD SSM Mapping，或接口状态为Down，则出现提示信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示配置SSM映射的接口信息。

<Quidway> display mld ssm-mapping interface
Interface information of VPN-Instance: public net
GigabitEthernet1/0/0(FE80::1)
