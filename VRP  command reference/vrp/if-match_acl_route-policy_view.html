if-match acl（Route-Policy视图）

命令功能

if-match acl命令用来设置一个基于ACL的匹配规则。

undo if-match acl命令用来删除设置的匹配规则。

缺省情况下，没有配置基于ACL的匹配规则。

命令格式

if-match acl acl-number

undo if-match acl acl-number

参数说明

acl-number：指定基本ACL的编号。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

无

使用实例

# 设置一个基于ACL 2000的匹配规则。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] if-match acl 2000
