display l2-multicast forwarding-table

命令功能

display l2-multicast forwarding-table命令用来显示组播转发表。

命令格式

display l2-multicast forwarding-table { vlan vlan-id | vsi vsi-name } [ group-address group-address | router-group ]

参数说明

vlan vlan-id: VLAN编号，取值范围是1～4094。

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

group-address group-address：组播IP地址，取值范围：224.0.1.0～239.255.255.255。

使用指南

组播表项显示范围有以下几种：

  • 命令中加上vlan-id，显示指定的组播VLAN下的组播表项；

  • 命令中加上group-address，显示指定的组播VLAN下指定的组播组下的组播表项。

使用实例

# 显示VLAN 10的组播转发表。

<Quidway> display l2-multicast forwarding-table vlan 10
VLAN ID : 10, Forwarding Mode : MAC, Total Entry Count : 1 
Group-Address              Status    Age Index Port-cnt add del update
-----------------------------------------------------------
0100-5e01-0101:224.1.1.1  NoChg    No   2      2           0   0   0

# 显示VSI vsi0的组播转发表。

<Quidway> display l2-multicast forwarding-table vsi vsi0
VSI Name : vsi0, Forwarding Mode : MAC, Total Entry Count : 2
Group-Address             Status Age Index Port-cnt add del update
-----------------------------------------------------------
Router-port    NoChg   No   1        1      0    0   0
0100-5e01-0101:224.1.1.1 NoChg   No   2        2      0    0   0
表1 display l2-multicast forwarding-table输出信息描述

项目

描述

Group address

组播组地址。

Status
当前组状态。
  • NoChg：正常状态；
  • Add：添加；
  • Del：删除；
  • Update：更新。

Age
是否处于老化状态。
  • No：没有老化。
  • Yes：已经老化。

Index

同步索引。

Port-cnt

组播组中的端口数目。

add

处于add状态的端口数目。

del

处于del状态的端口数目。

update

处于update状态的端口数目。
