hwtacacs-server shared-key

命令功能

hwtacacs-server shared-key命令用来设置HWTACACS服务器的共享密钥。

undo hwtacacs-server shared-key命令用来删除配置。设置密钥可以提高路由器与HWTACACS服务器通信的安全性。

缺省情况下，HWTACACS服务器没有密钥。

命令格式

hwtacacs-server shared-key key-string

undo hwtacacs-server shared-key

参数说明

key-string：共享密钥，字符串形式，长度范围是1～16。

视图

HWTACACS服务器模板视图

缺省级别

2：配置级

使用指南

为了确保认证双方的身份合法性，要求路由器上的密钥与HWTACACS服务器间设定的密钥必须相同。

使用实例

# 配置HWTACACS服务器共享密钥为hello。

<Quidway> system-view
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1] hwtacacs-server shared-key hello
