vlan-type dot1q

命令功能

vlan-type dot1q命令用来设置以太网子接口上的VLAN封装类型。

undo vlan-type dot1q命令用来去除VLAN封装。

缺省情况下，以太网子接口上没有封装VLAN，也没有与以太网子接口关联的VLAN ID。

命令格式

vlan-type dot1q low-vid [ high-vid ]

undo vlan-type dot1q low-vid [ high-vid ]

参数说明

dot1q：采用802.1Q的封装格式。

low-vid：配置接口所属的VLAN ID，或在配置VLAN ID范围时标识VLAN ID的下限，取值范围是1～4094。

high-vid：在配置VLAN ID范围时标识VLAN ID的上限，取值范围是1～4094。该值必须大于low-vid。

视图

Ethernet子接口视图、GE子接口视图

缺省级别

2：配置级

使用指南

VRP上只能配置一个VLAN。

NE16E、NE08E和NE05支持在一个子接口下最多关联64个VLAN。

NE40、NE80、NE40E和NE5000E上只能配置一个VLAN。

如果子接口封装了VLAN并且绑定到VSI，必须先解除子接口和VSI的绑定，然后执行命令undo vlan-type dot1q 去除VLAN封装。

使用实例

# 设置子接口GE 2/0/0.1与编号为60的VLAN ID相关联，VLAN封装格式为dot1q。

<Quidway> system-view
[Quidway] interface gigabitethernet2/0/0.1
[Quidway-GigabitEthernet2/0/0.1] vlan-type dot1q 60
