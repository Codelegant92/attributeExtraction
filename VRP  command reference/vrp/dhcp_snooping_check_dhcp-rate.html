dhcp snooping check dhcp-rate

命令功能

dhcp snooping check dhcp-rate命令用来配置对DHCP报文上送到DHCP协议栈的检查速率。

undo dhcp snooping check dhcp-rate命令用来恢复DHCP报文上送到DHCP协议栈的检查速率的缺省值。

缺省情况下，检查DHCP报文速率为100个/秒。

命令格式

dhcp snooping check dhcp-rate rate

undo dhcp snooping check dhcp-rate

参数说明

rate：上送DHCP报文到CPU的检查速率。取值范围是1～100，单位是个/秒，缺省为100。

视图

缺省级别

2：配置级

使用指南

执行dhcp snooping check dhcp-rate rate后，在一定的时间内只允许规定数目的报文上送协议栈处理，超过速率规定的报文将被丢弃。

配置了告警功能后，如果丢弃的报文达到告警阈值，设备将向网管产生告警。

使用实例

# 设置DHCP报文的检查速率为50个/秒。

<Quidway> system-view
[Quidway] dhcp snooping check dhcp-rate 50
