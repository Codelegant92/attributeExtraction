compare master-current-configuration slave-current-configuration

命令功能

compare master-current-configuration slave-current-configuration命令用来手动比较主控板和备板的配置信息。

命令格式

compare master-current-configuration slave-current-configuration

使用实例

# 手动比较主备板配置信息是否一致。

  • 主备板配置信息一致。

<Quidway> compare master-current-configuration slave-current-configuration
System is collecting current slave configuration. Please wait a moment..................
Current configuration of the AMB and the SMB are consistent.

  • 主备板配置信息不一致。

<Quidway> compare master-current-configuration slave-current-configuration
System is collecting current slave configuration. Please wait a moment..................
Current configuration of the AMB and the SMB are inconsistent.
