rrpp domain

命令功能

rrpp domain命令用来创建RRPP域。

undo rrpp domain命令用来删除RRPP域。

命令格式

rrpp domain domain-id

undo rrpp domain domain-id

参数说明

domain-id：待创建或待删除的RRPP域的ID。整数形式，取值范围是1～8。

视图

系统视图

缺省级别

2：配置级

使用指南

删除域前必须确认此域中没有配置RRPP环，否则会导致删除失败。

使用实例

# 创建RRPP域1。

<Quidway> system-view
[Quidway] rrpp domain 1
[Quidway-rrpp-domain-region1]

# 删除RRPP域1。

<Quidway> system-view
[Quidway] undo rrpp domain 1
