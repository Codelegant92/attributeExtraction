display pim mib-notification interval

命令功能

display pim mib-notification interval用来查看发送PIM告警的时间间隔。

命令格式

display pim mib-notification interval [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在配置或修改了pim mib-notification interval发送间隔的值后，用display pim mib-notification interval命令来查看设置的间隔。

使用实例

# 查看PIM notification的设置间隔。

<Quidway> display pim mib-notification interval
Neighbor loss notification interval      : 0 s
Invalid Join-Prune notification interval : 50 s
RP-MAPPING change notification interval  : 100 s
Interface election notification interval : 2000 s
Invalid register notification interval   : 60000 s
