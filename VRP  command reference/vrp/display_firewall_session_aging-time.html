display firewall session aging-time

命令功能

display firewall session aging-time命令显示表项超时时间设定值。

命令格式

display firewall session aging-time

参数说明

无

使用指南

使用该命令可以查看各应用层协议FTP、H.323、HTTP、HWCC、MSN、QQ、RTSP、SMTP、TCP/UDP等报文的老化时间。
