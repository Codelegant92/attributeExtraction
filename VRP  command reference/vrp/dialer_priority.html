dialer priority

命令功能

dialer priority命令用来配置轮询DCC，设置物理接口在Dialer Circular Group中的优先级。

undo dialer priority命令用来恢复优先级的缺省值。

命令格式

dialer priority priority

undo dialer priority

参数说明

priority：物理接口在Dialer Circular Group中的优先级。整数形式，取值范围是1～127。缺省的优先级为1。

视图

接口视图、Dialer接口视图

缺省级别

2：配置级

使用指南

在配置该命令前，必须先执行命令dialer user设置对端用户名，或执行命令dialer enable-circular使能轮询DCC。

此命令设定一个空闲的物理接口在一个Dialer Circular Group中的使用顺序，优先使用高优先级的物理接口。

使用实例

# 设置Serial3/0/0在Dialer Circular Group0中的优先级为5。

<Quidway> system-view
[Quidway] interface serial 3/0/0
[Quidway-Serial3/0/0] dialer circular-group 0
[Quidway-Serial3/0/0] dialer priority 5
