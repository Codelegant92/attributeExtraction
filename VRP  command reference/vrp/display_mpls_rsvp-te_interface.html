display mpls rsvp-te interface

命令功能

display mpls rsvp-te interface命令用来查看接口的RSVP-TE配置信息。

命令格式

display mpls rsvp-te interface [ interface-type interface-number ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type interface-number：接口类型和接口号。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定interface-type interface-number，将显示所有使能了RSVP-TE接口的RSVP-TE信息。

使用实例

# 查看指定接口的RSVP-TE信息。

<Quidway> display mpls rsvp-te interface gigabitethernet 1/0/0
Interface: GigabitEthernet1/0/0

项目

描述

Interface

接口名称

Interface Address

接口地址

Interface state
接口状态：

  • UP

  • DOWN

Interface Index

接口的索引号

Total-BW

接口上TE可以使用的总带宽（单位：kbit/s）

Used-BW

接口上TE已经占用的带宽（单位：Kbit/s）

Hello configured
接口上是否配置了Hello特性：

  • YES：配置了Hello特性

  • NO：没有配置Hello特性

Num of Neighbors

本端的邻居个数

Srefresh feature

是否启用了摘要刷新特性

Srefresh interval

摘要刷新定时器时长（单位：秒）

Mpls Mtu

MPLS实际转发采用的MTU值

Retransmit interval

重传定时器时长（单位：微秒）

Increment value

重传增量

Authentication
是否使能了验证：

  • ENABLE：使能了验证

  • DISABLE：没有使能验证

Bfd Enabled
是否全局使能了BFD for RSVP：
  • ENABLE：全局使能了BFD for RSVP
  • DISABLE：未全局使能BFD for RSVP

Bfd Min-Tx

本地配置的BFD消息最小发送时间间隔，单位是毫秒

Bfd Min-Rx

本地配置的BFD消息最小接收时间间隔，单位是毫秒

Bfd Detect-Multi

本地BFD检测倍数
