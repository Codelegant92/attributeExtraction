dhcp snooping check dhcp-rate alarm enable

命令功能

dhcp snooping check dhcp-rate alarm enable命令用来使能DHCP报文上送到DHCP协议栈的检查速率的告警功能。

undo dhcp snooping check dhcp-rate alarm enable命令用来取消DHCP报文上送到DHCP协议栈的检查速率的告警功能。

缺省情况下，不对DHCP报文上送到DHCP协议栈的检查速率产生告警。

命令格式

dhcp snooping check dhcp-rate alarm enable

undo dhcp snooping check dhcp-rate alarm enable

参数说明

无

系统视图

缺省级别

2：配置级

使用指南

执行dhcp snooping check dhcp-rate alarm enable命令后，在一定的时间内只允许规定数目的报文上送协议栈处理，超过速率规定的报文将被丢弃。

配置了告警功能后，如果丢弃的报文达到告警阈值，设备将向网管产生告警。
