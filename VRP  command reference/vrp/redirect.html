redirect

命令功能

redirect命令用来使能异步端口重定向功能，只对AUX和TTY类型的用户界面有效。

undo redirect命令用来禁止相应端口的重定向功能。

缺省情况下，没有使能重定向功能。

命令格式

redirect

undo redirect

参数说明

无

视图

用户界面视图

缺省级别

3：管理级

使用指南

当在用户界面上执行redirect命令后，将使能该用户界面的重定向功能。

使用实例

# 使能用户界面AUX0的重定向功能。

<Quidway> system-view
[Quidway] user-interface aux0
[Quidway-ui-aux0] redirect
