info-center create-logbook

命令功能

info-center create-logbook命令用来生成二进制文件进行解析时需要的数据字典文件，生成的数据字典可以保存到设备的存储介质上，也可以发送到远端FTP服务器。

命令格式

info-center create-logbook file-name

info-center create-logbook { ftp ftp-addres user-name user-name password password [ file-name file-name | port port-number ] * }

参数说明

file-name：生成的数据字典的文件名。

ftp ftp-addres：FTP服务器地址。

user-name user-name：FTP服务器用户名。

password password：FTP服务器密码。

port port-number：FTP使用的端口号。

使用指南

要能成功生成数据字典，需要满足一下几个条件：

  • 指定路径存在并有足够的空间存储生成的数据字典；

  • FTP传输时，权限确认通过并且指定路径下没有相同文件名的文件，或有相同的文件名时选择覆盖操作。

生成的数据字典的格式是.xml文件形式的。

使用实例

# 配置生成数据字典dictionary。

<Quidway> system-view
[Quidway] info-center create-logbook file-name dictionary.xml

# 配置生成数据字典并通过FTP传送到日志主机。

<Quidway> system-view
[Quidway] info-center create-logbook ftp 10.1.2.20 user-name ftp password ftp file-name dictionary.xml port 2000
