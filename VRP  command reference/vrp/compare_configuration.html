compare configuration

命令功能

compare configuration命令用来比较当前的配置文件与存储设备中保存的配置文件内容是否一致。

命令格式

compare configuration [ line-number1 line-number2 ]

参数说明

line-number1：表示从当前配置文件的该行号开始比较，范围是0～65535。

line-number2：表示从保存配置文件的该行号开始比较，范围是0～65535。

视图

用户视图

缺省级别

2：配置级

使用指南

如果不输入参数，表示从配置文件的首行开始进行比较。这两个参数是为了在发现配置文件不同之处后，跳过该不同处继续进行比较。

compare configuration命令在比较出不同之处时，将从不同的行开始，分别对当前配置文件和保存的配置文件，显示一定数量字符的文件内容（默认显示150个字符），如果该不同之处到文件末尾不足150个字符，将显示到文件尾为止。

使用实例

# 比较配置文件

<Quidway> compare configuration
