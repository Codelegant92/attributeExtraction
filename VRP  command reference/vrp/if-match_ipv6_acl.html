if-match ipv6 acl

命令功能

if-match ipv6 acl命令用来定义IPV6 ACL匹配规则。

undo if-match ipv6 acl命令用来删除IPV6 ACL匹配规则。

命令格式

if-match [ not ] ipv6 acl access-list-number

undo if-match [ not ] ipv6 acl access-list-number

参数说明

not：不匹配该类。

access-list-number：ACL号，取值范围是2000～3999。

视图

流分类视图

缺省级别

2：配置级

使用指南

if-match ipv6 acl命令只在NE16E/08E/05路由器上可用。

使用实例

# 定义类匹配IPv6 ACL 3101。

<Quidway> system-view
[Quidway] traffic classifier class1
[Quidway-classifier-class1] if-match ipv6 acl 3101
