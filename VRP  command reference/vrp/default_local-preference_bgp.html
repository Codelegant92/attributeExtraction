default local-preference (BGP)

命令功能

default local-preference命令用来配置BGP的缺省本地优先级。

undo default local-preference命令用来恢复缺省设置。

缺省情况下，BGP本地优先级的值为100。

命令格式

default local-preference preference

undo default local-preference

参数说明

preference：设置BGP的本地优先级。整数形式，其取值范围为0～4294967295。该值越大则实际的优先级越高。

缺省级别

2：配置级

使用指南

配置不同本地优先级会影响BGP的路由选择。当一个运行BGP的路由器有多条路由到达同一目的地址时，会优先选择本地优先级最高的路由。

本地优先级属性仅在IBGP对等体之间有效，不通告给其他AS。

使用实例

# 配置BGP的缺省本地优先级为200。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family unicast 
[Quidway-bgp-af-ipv4] default local-preference 200

# 配置指定IPv6 VPN实例的BGP私网路由缺省本地优先级为200。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv6-family vpn6-instance vpna
[Quidway-bgp6-vpna] default local-preference 200

# 配置本路由器上所有BGP VPNv6路由的缺省本地优先级为100。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv6-family vpnv6
[Quidway-af-vpnv6] default local-preference 100
