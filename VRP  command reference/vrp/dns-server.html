dns-server

命令功能

dns-server命令用来配置NQA测试中DNS测试例的DNS服务器地址。

undo dns-server命令用来取消配置的DNS服务器地址。

命令格式

dns-server ipv4 ip-address

undo dns-server

参数说明

ipv4 ip-address：配置DNS服务器地址为IPv4类型。点分十进制格式。

视图

NQA视图

缺省级别

2：配置级

使用指南

只有当测试例的类型是DNS或HTTP时，才能设置DNS服务器。

不能更改正在运行的测试例的DNS服务器设置。

使用实例

# 设置测试例user test的DNS服务器地址为1.1.1.1。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type dns
[Quidway-nqa-user-test] dns-server ipv4 1.1.1.1
