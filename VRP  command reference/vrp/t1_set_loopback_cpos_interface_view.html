t1 set loopback（CPOS接口视图）

命令功能

t1 set loopback命令用来设置T1通道的环回模式。

undo t1 set loopback命令用来禁止环回。

缺省情况下，禁止任何形式的环回。

命令格式

t1 t1-number set loopback { local | payload | remote }

undo t1 t1-number set loopback

参数说明

t1-number：CPOS的T1通道号。整数形式，取值范围是1～84。

local：使能T1通道对内自环。

payload：使能T1通道对外载荷环回。

remote：使能T1通道对外远端环回。

CPOS接口视图

缺省级别

2：配置级

使用指南

环回功能通常用于进行某些特殊测试，正常工作时不要启动环回。

当对T1通道设置环回时，该环回也会在该T1通道下的同步串口生效：
  • 如果同步串口的链路层协议是PPP，当设置环回时，接口的物理状态变为Up，而链路协议状态为Down。
  • 如果同步串口的链路层协议是HDLC，当设置环回时，接口的物理状态和链路协议状态都是Up。

使用实例

# 设置T1通道1进行对外远端环回。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] t1 1 set loopback remote
