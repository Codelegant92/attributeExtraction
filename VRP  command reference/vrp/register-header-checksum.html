register-header-checksum

命令功能

register-header-checksum命令用来配置仅根据Register注册消息头信息来计算校验和。

undo register-header-checksum命令用来恢复缺省配置。

缺省情况下，根据Register注册消息全部内容来计算校验和。

命令格式

register-header-checksum

undo register-header-checksum

参数说明

无

视图

公网实例PIM视图、VPN实例PIM视图

缺省级别

2：配置级

使用指南

出于Register注册消息传递过程中信息完整性考虑，可以根据消息头或整个消息来计算校验和。

使用实例

# 在公网实例中，配置仅根据Register注册消息头信息计算校验和。
