dns server source-ip

命令功能

dns server source-ip命令用来指定本端路由设备作为DNS client进行DNS通信时的源地址。

undo dns server source-ip命令用来删除本端路由设备的源地址。

命令格式

dns server source-ip ip-address

undo dns server source-ip ip-address

参数说明

ip-address：本端路由设备的源地址。

缺省级别

使用指南

指定本端路由设备的源地址，以指定的源地址与DNS服务端通信，从而达到进行安全校验的目的。DNS服务器会将域名响应按照指定源地址的路由发送回本端路由设备上。

使用实例

# 指定本端路由设备的源地址为172.16.1.1。

<Quidway> system-view
[Quidway] dns server source-ip 172.16.1.1
