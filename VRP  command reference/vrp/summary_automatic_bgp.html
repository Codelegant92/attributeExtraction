summary automatic (BGP)

命令功能

summary automatic命令用来使能对本地引入的路由进行自动聚合。

undo summary automatic命令用来取消对本地引入的路由进行自动聚合。

缺省情况下，不对本地引入的路由进行自动聚合。

命令格式

summary automatic

undo summary automatic

参数说明

无

视图

BGP视图、BGP-IPv4单播地址族视图、BGP-VPN实例视图

缺省级别

2：配置级

使用指南

BGP的聚合分为手动聚合和自动聚合，该命令用于实现自动聚合。其优先级低于手动聚合。

配置该命令后，BGP将按照自然网段聚合路由（如10.1.1.1/24和10.2.1.1/24将聚合为A类地址10.0.0.0/8），并且BGP只向对等体发送聚合后的路由。这样可以减少路由信息的数量。

使用实例

# 对引入的路由进行自动聚合。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] summary automatic
