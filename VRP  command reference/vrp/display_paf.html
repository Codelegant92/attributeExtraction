display paf

命令功能

display paf命令用来查看系统的PAF文件信息。

命令格式

display paf { all | { resource | service } item-name }

参数说明

all：PAF文件中的所有信息。

resource：PAF文件中性能项的配置值。

service：PAF文件中属性项的配置值。

item-name：PAF文件性能项或属性项的名称，字符串形式，长度范围是1～64。

使用指南

display paf命令用来查看系统的PAF文件信息，包括PAF文件中性能项和属性项的配置值。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 查看PAF文件中性能项的配置值。

<Quidway> display paf resource PAF_LCS_IPV6_ND_SPECS_ENTRIES_NUM
PAF_LCS_IPV6_ND_SPECS_ENTRIES_NUM = 1, 4096, 8192, 2048

# 查看PAF文件中属性项的配置值。

<Quidway> display paf service PAF_LCS_MSE_SPECS_ENABLED
PAF_LCS_MSE_SPECS_ENABLED = 1, 1
