display mpls rsvp-te bfd session

命令功能

display mpls rsvp-te bfd session命令用来查看BFD for RSVP会话信息。

命令格式

display mpls rsvp-te bfd session { all | interface interface-type interface-number | peer ip-address } [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：查看所有BFD for RSVP会话信息。

interface interface-type interface-number：查看特定接口下的BFD for RSVP会话信息。

peer ip-address：查看邻居的BFD for RSVP会话信息。

verbose：查看BFD for RSVP会话的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Total Nbrs/Rsvp triggered sessions

全局或接口下BFD邻居数量/RSVP触发的BFD会话数量

Local Discriminator

BFD会话本端会话标识符，0为无效标识符

Remote Discriminator

BFD会话远端会话标识符，0为无效标识符

Local Address

BFD会话本端IP地址

Peer Address

BFD会话对端IP地址

Interface Name

BFD会话的出接口

Act-Tx Act-Rx Act-Multi
BFD会话使用的参数

  • Act-Tx：实际的BFD报文发送时间间隔。单位是毫秒

  • Act-Rx：实际允许接收BFD报文时间间隔。单位是毫秒

  • Act-Multi：实际的本地检测倍数。缺省值是3

Session-State
BFD会话的状态

  • UP：BFD会话状态为Up

  • DOWN：BFD会话状态为Down

  • NONE：没有配置BFD会话

  • ADMIN DOWN：管理员手动关闭BFD会话

  • INITIAL：初始状态
