apply ipv6 next-hop

命令功能

apply ipv6 next-hop命令用ipv6-address地址替换指定过滤策略的路由的下一跳地址。

undo apply ipv6 next-hop命令用来取消该配置。

命令格式

apply ipv6 next-hop { peer-address | ipv6-address }

undo apply ipv6 next-hop { peer-address | ipv6-address }

参数说明

ipv6-address：指定下一跳IPv6地址。

peer-address：指定对等体地址为下一跳。

Route-Policy视图

缺省级别

2：配置级

使用指南

无

使用实例

# 将下一跳地址设为3ffe:506::1。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply ipv6 next-hop 3ffe:506::1
