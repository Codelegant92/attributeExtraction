undo l2-multicast dynamic-group

命令功能

undo l2-multicast dynamic-group命令用于删除指定的动态组播组。

缺省情况下，不删除动态组播组。

命令格式

undo l2-multicast dynamic-group group-address group-address { vlan vlan-id [ interface { interface-type interface-number } ] | vsi vsi-name { interface { interface-type interface-number } | remote-peer ip-address [ negotiation-vc-id vc-id ] | remote-site remote-site-id } }

参数说明

group-address group-address：所要删除的组播组地址。采用点分十进制形式表示。

vlan vlan-id：VLAN号。整数形式，取值范围是1～4094。

interface interface-type interface-number：端口的接口类型和编号。支持的接口类型有Ethernet接口、GigabitEthernet接口、Eth-Trunk接口。

vsi vsi-name：VSI名称。字符串形式，大小写敏感，取值范围是1～31。

remote-peer ip-address：指定VPLS中远端IP地址。点分十进制形式。

negotiation-vc-id vc-id：PW信令为LDP时的对端协商VC-ID。整数形式，取值范围是1～4294967295。

remote-site remote-site-id：PW信令为BGP时的协商站点号。整数形式，取值范围是0～65534。

使用指南

此命令既可以删除某指定组播或某指定端口的组播组，也可以删除VLAN下的某个组播组。

使用实例

# 删除VLAN 1中端口GigabitEthernet1/0/0上的组播组225.1.1.1。

<Quidway> system-view
[Quidway] undo l2-multicast dynamic-group group-address 225.1.1.1 vlan 1 interface GigabitEthernet1/0/0
