interface ring-if

命令功能

interface ring-if命令用来创建一个RPR逻辑接口并进入此逻辑接口视图。

undo interface ring-if命令用来删除一个RPR逻辑口。

命令格式

interface ring-if interface-number

undo interface ring-if interface-number

参数说明

interface-number：逻辑接口编号。整数形式，取值范围是0～31。

视图

系统视图

缺省级别

2：配置级

使用指南

在RPR功能正常运行时，不要轻易对RPR逻辑接口执行restart命令或shutdown/undo shutdown命令。否则，由于重启后需要重新进行拓扑检查，将导致倒换时间延长。

说明：

只有没绑定任何RPR物理口的情况下，才可以删除RPR逻辑口。

使用实例

# 创建一个逻辑接口。

<Quidway> system-view
[Quidway] interface ring-if 1
