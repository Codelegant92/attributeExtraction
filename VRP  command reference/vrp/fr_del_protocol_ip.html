fr del protocol ip

命令功能

fr del protocol ip命令用来配置基于IP协议的DE规则列表，封装了符合指定规则的IP报文的帧中继报文的DE标志位将被置为1。

undo fr del protocol ip命令用来从DE规则列表内删除指定的DE规则。

缺省情况下，没有创建DE规则列表。

命令格式

fr del list-number protocol ip [ fragments | acl acl-number | less-than bytes | greater-than bytes | tcp ports | udp ports ]

undo fr del list-number protocol ip [ fragments | acl acl-number | less-than bytes | greater-than bytes | tcp ports | udp ports ]

参数说明

list-number：DE规则列表编号，取值范围是1～10。

protocol ip：IP协议。

fragments：所有分片的IP报文。

acl-number：符合ACL匹配条件的IP报文。acl-number的取值范围是2000～3999。

less-than：长度小于bytes的IP报文。bytes的取值范围是0～65535。

greater-than：长度大于bytes的IP报文。bytes的取值范围是0～65535。

tcp：源或目的TCP端口号为ports的IP报文。

udp：源或目的UDP端口号为ports的IP报文。

如果不使用可选参数，则表示所有IP报文。

使用指南

重复使用本命令可以为DE规则列表添加新的规则。每个DE规则列表最多可以配置100条规则。

本命令的undo形式每次只能删除一条DE规则，如果要删除一个DE规则列表，必须把列表中的所有DE规则全部删除。

使用实例

# 在DE规则列表1内添加一条规则，对所有封装IP报文的帧中继报文，将其DE标志位置为1。

<Quidway> system-view
[Quidway] fr del 1 protocol ip
