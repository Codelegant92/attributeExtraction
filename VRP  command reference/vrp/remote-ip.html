remote-ip

命令功能

remote-ip命令用来配置LDP远端对等体的IP地址。

undo remote-ip命令用来删除配置。

命令格式

remote-ip ip-address [ pwe3 ]

undo remote-ip [ pwe3 ]

参数说明

ip-address：远端对等体IPv4地址。

pwe3：禁止向指定的远端邻居分发标签。

视图

MPLS LDP远端对等体视图

缺省级别

2：配置级

使用指南

远端对等体的IP地址应该为远端LSR的LSR ID。远端对等体使用LSR ID做为传输地址来建立TCP连接。

remote-ip ip-address pwe3命令用来配置禁止向指定的远端邻居分发标签。缺省情况下，允许向指定的远端邻居分发标签。

remote-peer pwe3和remote-ip ip-address pwe3命令都用于禁止向远端邻居分发标签。前者对所有的远端邻居有效，后者只对指定的远端邻居有效。

配置LDP远端会话时，建议使用remote-peer pwe3或remote-ip ip-address pwe3命令禁止向远端邻居分发标签，否则会影响系统性能。

remote-peer pwe3和remote-ip ip-address pwe3命令不适用于LDP over TE场景。

说明：

修改或删除已经配置的远端对等体地址会导致相应的远端LDP会话被删除。

使用实例

# 配置远端对等体地址。

<Quidway> system-view
[Quidway] mpls ldp remote-peer BJI
[Quidway-mpls-ldp-remote-bji] remote-ip 3.3.3.3

# 配置禁止向远端邻居1.1.1.1/32分发标签。

<Quidway> system-view
[Quidway] mpls ldp remote-peer rtc
[Quidway-mpls-ldp-remote-rtc] remote-ip 1.1.1.1 pwe3
