display accounting-scheme

命令功能

display accounting-scheme命令用来查看计费方案的配置情况。

命令格式

display accounting-scheme [ accounting-scheme-name ]

参数说明

accounting-scheme-name：计费方案名称，字符串形式，长度范围是1～32，不区分大小写。

视图

所有视图

缺省级别

1：监控级

使用指南

如果在计费方案视图下执行此命令，或者指定了计费方案的名称，则将显示详细的计费方案配置信息；其它情况下只显示摘要信息。

使用实例

# 查看所有计费方案的摘要信息。

<Quidway> display accounting-scheme
  Accounting-scheme-name              Accounting-method
  default                             No accounting
  scheme1                             No accounting
  Total 2,2 printed

# 查看系统缺省的计费方案的详细配置信息。

<Quidway> display accounting-scheme default
  Accounting-scheme-name                : default
  Accounting-method                     : No accounting
  Realtime-accounting-switch            : Open
  Realtime-accounting-interval(min)     : 5
  Start-accounting-fail-policy          : Cut user
  Realtime-accounting-fail-policy       : Cut user
  Realtime-accounting-failure-retries   : 3
表1 display accounting-scheme命令显示信息说明

项目

描述

Accounting-scheme-name

计费方案的名称

Accounting-method

此计费方案配置的计费模式

Realtime-accounting-switch

是否启用实时计费功能

Realtime-accounting-interval(min)

实时计费间隔设置

Start-accounting-fail-policy

计费开始失败策略

Realtime-accounting-fail-policy

实时计费失败策略

Realtime-accounting-failure-retries

允许的最大实时计费失败次数
