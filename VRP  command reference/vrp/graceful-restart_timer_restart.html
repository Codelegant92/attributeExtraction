graceful-restart timer restart

命令功能

graceful-restart timer restart命令用来配置对端从发现本端重启到重新建立BGP会话的最大等待时间。

undo graceful-restart timer restart命令用来删除该配置。

命令格式

graceful-restart timer restart time

undo graceful-restart timer restart

参数说明

time：指定对端发现本端重启到重新建立BGP会话的最大等待时间，单位为秒。整数形式，取值范围是3～600秒。缺省值为150秒。

缺省级别

2：配置级

使用指南

更改重建BGP会话的最大时间将重新建立BGP对等体关系。

说明：

只有先使能了Graceful Restart特性后，才能配置该命令。

使用实例

# 配置对端重建BGP会话的最大时间为250秒。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] graceful-restart timer restart 250
