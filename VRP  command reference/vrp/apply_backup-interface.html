apply backup-interface

命令功能

apply backup-interface命令用来配置备份出接口。

undo apply backup-intreface用来删除已配置的备份出接口。

命令格式

apply backup-interface interface-type interface-number

undo apply backup-intreface

参数说明

interface-type interface-number：指定备份出接口的接口类型和接口号。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

可以在配置备份出接口之前匹配一些条件，如匹配接口等。

该命令主要用于IP FRR，需要和命令apply backup-nexthop配合使用。

使用实例

# 在名为ip_frr_rp的Route-Policy中配置备份出接口GE 1/0/0。

<Quidway> system-view
[Quidway] route-policy ip_frr_rp permit node 10
[Quidway-route-policy] apply backup-interface GigabitEthernet1/0/0
[Quidway-route-policy] apply backup-nexthop 192.168.20.2

# 在名为ip_frr_rp的Route-Policy中删除配置的备份出接口。

<Quidway> system-view
[Quidway] route-policy ip_frr_rp permit node 10
[Quidway-route-policy] undo apply backup-interface
