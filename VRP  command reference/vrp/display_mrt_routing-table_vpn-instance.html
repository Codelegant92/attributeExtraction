display mrt routing-table vpn-instance

命令功能

display mrt routing-table vpn-instance命令用来查看VPN实例MRT路由表的概要信息。

命令格式

display mrt routing-table vpn-instance vpn-instance-name [ ipv4–address [ mask | mask-length ] [ longer-match ] ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance-name：VPN实例的名称。

ipv4–address：IPv4形式的网络地址。

mask | mask-length：点分十进制掩码/掩码长度。

longer-match：仅显示与特定网络/掩码匹配的路由。

verbose：显示激活和未激活路由的详细信息。不指定该参数，则只显示激活路由的概要信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过指定参数可以只查看特定的路由信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看名为vpn1的VPN实例的所有MRT路由信息。

<Quidway> display mrt routing-table vpn-instance vpn1
