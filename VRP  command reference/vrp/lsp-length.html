lsp-length

命令功能

lsp-length命令用来配置当前IS-IS系统生成LSP报文的长度以及接收LSP报文的长度，报文长度以字节为单位。

undo lsp-length命令用来恢复缺省设置。

命令格式

lsp-length { originate | receive } max-size

undo lsp-length { originate | receive }

参数说明

originate：设置生成LSP报文的长度。

receive：设置接收LSP报文的长度。

max-size：LSP报文的最大长度。整数类型，取值范围是512～16384，单位是字节。缺省值是1497。

使用指南

设置max-size参数时请注意，生成的LSP报文（或者转发的LSP报文）的max-size必须小于等于接收的LSP报文的max-size。

使用lsp-length命令设置的max-size值必须满足以下要求。

  • 以太网接口的MTU值大于等于max-size+3。

  • P2P接口的MTU值大于等于max-size。

使用实例

# 设置当前IS-IS系统生成LSP报文最大长度为1024字节。

<Quidway> system-view
[Quidway] isis
[Quidway-isis-1] lsp-length originate 1024
