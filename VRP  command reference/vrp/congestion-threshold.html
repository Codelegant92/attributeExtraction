congestion-threshold

命令功能

congestion-threshold命令用来使能帧中继虚电路的拥塞管理功能。

undo congestion-threshold命令用来禁止此功能。

命令格式

congestion-threshold { de | ecn } queue-percentage

undo congestion-threshold { de | ecn }

参数说明

de：当拥塞发生时，丢弃DE标志位为1的帧中继报文。

ecn：当拥塞发生时，对帧中继报文的BECN和FECN标志位进行处理。

queue-percentage：网络拥塞阈值，为虚电路队列的使用率，即虚电路当前队列长度占队列总长度的百分比，取值范围是1～100。缺省值为100。

视图

FR-Class视图

缺省级别

2：配置级

使用指南

当虚电路当前队列的长度占虚电路队列总长度的百分比超过设置的拥塞阈值时，认为虚电路上发生拥塞，开始对虚电路上的报文进行拥塞处理。

使用实例

# 设置对于名为test1的帧中继class，当虚电路当前队列长度超过总长度的80％时，开始丢弃DE标志位为1的帧中继报文。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] congestion-threshold de 80
