display info-center

命令功能

display info-center命令用来显示信息中心记录的各项信息。

命令格式

display info-center

display info-center statistics [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

statistics：显示信息中心的统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Information Center

信息中心的使能状态。

Log host

日志主机的状态，包括日志主机的IP地址，所使用的通道号、通道名，语言及日志主机的级别。

Console

控制口的使用状态，包括控制口所使用的通道名，通道号。

Monitor

监视口的使用状态，包括监视口所使用的通道号、通道名。

SNMP Agent

网管代理的使用状态，包括通道号、通道名。

Log buffer

日志缓冲区的使用状态，包括使能态，最大尺寸，当前尺寸，当前信息数，通道名，通道号，已丢弃的信息数，被覆盖的信息数。

Trap buffer

告警缓冲区的使用状态，包括使能态，最大尺寸，当前尺寸，当前信息数，通道名，通道号，已丢弃的信息数，被覆盖的信息数。

Logfile

日志文件的使用状态，包括向日志文件输出信息的通道号、通道名、语言。

Information timestamp setting
日志、告警、调试信息的时间戳格式：
  • boot
  • date
  • short-date
  • format-date
  • none

Sent messages

已发送的信息数。

Received messages

已收到的信息数。
