set constant-ifindex subinterface

命令功能

set constant-ifindex subinterface命令用来设定固定索引中子接口索引的内存分配模式。

缺省情况下，子接口索引的内存分配模式为紧凑模式。

命令格式

set constant-ifindex subinterface { dense-mode | sparse-mode }

参数说明

dense-mode：设置子接口索引的内存分配模式为紧凑模式。

sparse-mode：设置子接口索引的内存分配模式为松散模式。

2：配置级

使用指南

该命令仅在接口索引固定特性使能后有效。

当用户增加子接口时，如果子接口号是连续分布的，建议采用紧凑模式；如果子接口号不连续，建议采用松散模式。

使用实例

# 设置子接口索引的内存分配模式为松散模式。

[Quidway] set constant-ifindex subinterface sparse-mode
