vrrp-group packet strict-check enable

命令功能

vrrp-group packet strict -check enable命令用来设置VRRP管理组报文严格检查方式.

undo vrrp-group packet strict -check enable命令用来删除VRRP管理组严格检查方式。

缺省情况下，检查方式为松散检查，便于支持两台备份路由器上接口板槽位号不对应的组网环境。

命令格式

vrrp-group packet strict-check enable

undo vrrp-group packet strict-check enable

参数说明

无

系统视图

缺省级别

2：配置级

使用指南

当VRRP备份组内成员的接口板槽位号不一致时，采用VRRP管理组报文松散检查方式，以保证VRRP管理组正常工作。

使用实例

# 为路由器Quidway上的所有VRRP管理组打开报文严格检查开关。

<Quidway> system-view
[Quidway] vrrp-group packet strict-check enable
