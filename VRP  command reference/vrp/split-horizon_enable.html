split-horizon enable

命令功能

split-horizon enable命令用来使能MAC Tunnel的水平分割功能。

undo split-horizon enable命令用来关闭MAC Tunnel的水平分割功能。

缺省情况下，隧道的水平分割功能处于使能状态。

命令格式

split-horizon enable

undo split-horizon enable

参数说明

无

视图

MAC Tunnel视图

缺省级别

2 ：配置级

使用指南

水平分割功能用来禁止同一台设备上的不同MAC Tunnel之间互通。

只有当两条MAC Tunnel上都使能水平分割的时候，它们之间的流量才不能互通。如果有一条MAC Tunnel未使能水平分割功能，则该隧道可以接收到其他隧道的流量。

如果MAC Tunnel配置了保护组，保护隧道的水平分割属性与工作隧道保持一致，即工作隧道的水平分割属性变化，保护隧道将同步更新。

本命令只在工作隧道上有效。

在多条隧道绑定同一个SI时，必须使能水平分割功能，确保各隧道之间的流量不会互通。而绑定不同SI的隧道之间，即使没有使能水平分割功能，各隧道之间的流量也不会互通。

说明：

PBB-TE隧道不支持在端口上配置STP/RRPP来防止网络环路。

使用实例

# 使能当前隧道的水平分割属性功能。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] split-horizon enable
