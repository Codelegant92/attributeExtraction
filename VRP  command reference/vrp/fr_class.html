fr class

命令功能

fr class命令用来创建帧中继class并进入帧中继class视图。

undo fr class命令用来删除指定的帧中继class。

缺省情况下，没有创建帧中继class。

命令格式

fr class class-name

undo fr class class-name

参数说明

class-name：class名称，最长为31个字符。

视图

系统视图

缺省级别

2：配置级

使用指南

只有将帧中继class同接口或虚电路相关联，并且使能相应接口的帧中继QoS功能，设置的帧中继class参数才会起作用。

删除帧中继class时，将释放所有接口或DLCI与该帧中继class的关联。

使用实例

# 创建名为test1的帧中继class。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1]
