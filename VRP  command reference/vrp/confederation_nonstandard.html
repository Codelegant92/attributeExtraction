confederation nonstandard

命令功能

confederation nonstandard命令用来配置与非标准的AS联盟兼容。

undo confederation nonstandard命令用来取消同非标准的AS联盟兼容。

缺省情况下，配置的联盟与RFC3065一致。

命令格式

confederation nonstandard

undo confederation nonstandard

参数说明

无

视图

BGP视图

缺省级别

2：配置级

使用指南

为了同采用非标准的设备互通，必须对联盟中所有路由器配置该命令。

使用实例

# 使本地路由器与非标准的路由器兼容。AS100包括了两个子自治系统：AS64000和AS65000。

<Quidway> system-view
[Quidway] bgp 64000
[Quidway-bgp] confederation id 100
[Quidway-bgp] confederation peer-as 65000
[Quidway-bgp] confederation nonstandard
