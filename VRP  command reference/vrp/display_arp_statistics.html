display arp statistics

命令功能

display arp statistics命令用来查看ARP表项的统计信息。

命令格式

display arp statistics { all | slot slot-id }

参数说明

all：显示整机ARP表项的统计信息。

slot slot-id：显示指定接口板的ARP表项的统计信息。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控ARP表项或ARP出现故障时，可以使用此命令查看ARP表项的统计信息。

对于集中式设备和没有接口板的分布式设备，例如NE20和NE20E，不能指定接口板槽号来查看ARP统计信息。

使用实例

# 查看ARP表项的统计信息。

<Quidway> display arp statistics all
