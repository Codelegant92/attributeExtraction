peer as-path-filter (BGP)

命令功能

peer as-path-filter命令用来为对等体/对等体组设置基于AS路径列表的BGP路由过滤策略。

undo peer as-path-filter命令用来取消已有的设置。

缺省情况下，对等体/对等体组没有配置基于AS路径列表的路由过滤策略，所有BGP路由均通过。

命令格式

peer { group-name | ipv4-address | ipv6-address } as-path-filter as-path-filter-number { import | export }

undo peer { group-name | ipv4-address | ipv6-address } as-path-filter as-path-filter-number { import | export }

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图、BGP-IPv4单播地址族视图、BGP-IPv6单播地址族视图、BGP-VPNv4地址族视图、BGP-VPNv6地址族视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP-IPv6单播地址族视图和BGP-VPN6实例视图下生效。

as-path-filter-number：AS路径过滤器号，取值范围为1～256。

import：对接收的路由进行过滤。

export：对发布的路由进行过滤。

缺省级别

2：配置级

使用指南

如果需要配置6PE，请在IPv6单播地址族视图下指定Peer为IPv4单播地址。

使用实例

# 设置对等体的AS路径过滤器。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] peer 1.1.1.2 as-path-filter 3 export
