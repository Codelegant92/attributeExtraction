display mac-address

命令功能

display mac-address命令用来查看地址表项信息。

如果不指定任何参数，将显示接口板的所有MAC地址表项。

命令格式

display mac-address [ blackhole | mac-address [ verbose ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

display mac-address dynamic [ slot-id ] [ vlan vlan-id | vsi vsi-name | source-slot source-slot-id | interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

display mac-address static [ vlan vlan-id | vsi vsi-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

mac-address：单播MAC地址。

verbose：显示基于MAC地址的详细的统计信息。

vlan-id：VLAN ID，取值范围是1～4094。

vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

blackhole：黑洞表项，不被老化。配置并保存后，系统复位或接口板热插拔表项不丢失。

static：静态表项，不被老化。配置并保存后，系统复位或接口板热插拔表项不丢失。

dynamic：动态表项，可以被老化，系统复位或接口板热插拔后丢失。

slot-id：接口板的槽位号。

source-slot source-slot-id：出接口所在的槽位号。

interface-type interface-number：出接口类型和接口编号。允许的接口类型有Ethernet接口、GE接口和Eth-Trunk接口。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过本命令可以按照指定的MAC地址查看VLAN或VSI的MAC地址表项。也可按照blackhole、static、dynamic分类查看MAC地址表项。

表1 display mac-address命令输出信息描述

项目

描述

MAC Address

MAC地址。

VLAN/VSI/SI

出接口所在的VLAN或者所属的VSI。

SI表示服务实例名。

PEVLAN

QinQ应用中，用户外层VLAN的Tag值。

CEVLAN

QinQ应用中，用户内层VLAN的Tag值。

Port

MAC地址所对应的出接口。

Type
MAC地址的类型：

  • static：静态表项，由用户配置，下发到各接口板。不被老化，系统复位或接口板热插拔后不丢失。

  • blackhole：黑洞表项，用于丢弃含有特定目的MAC地址的数据帧，由用户配置，下发到各接口板。不被老化，系统复位或接口板热插拔后不丢失。

  • dynamic：动态表项，由接口板学习存储。可以被老化，系统复位或接口板热插拔后丢失。

Aging time

老化时间。

LSP/MAC-Tunnel

MPLS的LspToken。例如：“2/-”表示入标签为2，出标签为空。此项为VSI的MAC地址表中的字段。MAC-Tunnel表示遂道名，该字段仅在NE40E、NE80E和NE5000E中显示。此项为SI的MAC地址表中的字段。

Total matching items on slot 1 displayed =

符合查看条件的显示项总数。
