dir（FTP客户端视图）

命令功能

dir命令显示目录下的所有文件或显示查询文件。

命令格式

dir [ file-name ] [ local-filename ]

参数说明

file-name：查询的文件名，字符串形式，长度范围是1～64。

local-filename：保存的本地文件名，字符串形式，长度范围是1～64。

FTP客户端视图

缺省级别

3：管理级

使用指南

执行该命令，可以查询文件内容，并且将文件内容保存在另一个文件中。
