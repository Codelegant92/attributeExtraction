cpu-packet statistics pvc enable

命令功能

cpu-packet statistics pvc enable命令用来使能基于端口和PVC的上送到CPU的报文的统计功能。

undo cpu-packet statistics pvc enable命令用来去使能基于端口和PVC的上送到CPU的报文的统计功能。

缺省情况下，去使能基于端口和PVC的上送到CPU的报文的统计功能。

命令格式

cpu-packet statistics pvc enable

undo cpu-packet statistics pvc enable

参数说明

无

视图

ATM接口视图

缺省级别

2：配置级

使用指南

在配置该命令前，必须先在系统视图下配置了命令cpu-packet statistics enable。

配置该命令后，该ATM接口上的所有PVC的上送到CPU的报文都会被统计。

目前在一台设备上，对于基于VLAN和PVC的接口，只能同时在64个接口上使能上送到CPU的报文统计功能。

使用实例

# 使能基于端口和PVC的上送到CPU的报文的统计功能。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] cpu-packet statistics pvc enable
