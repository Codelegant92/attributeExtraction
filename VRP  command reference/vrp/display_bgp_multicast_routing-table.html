display bgp multicast routing-table

命令功能

display bgp multicast routing-table命令用来查看MBGP路由表中指定网络的MBGP路由信息。

命令格式

display bgp multicast routing-table [ ip-address [ mask-length [ longer-prefixes ] | mask [ longer-prefixes ] ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ip-address：表示IPv4网络地址。点分十进制形式。

mask-length：表示IPv4网络地址的掩码长度。

mask：指定IPv4网络地址的掩码。

longer-prefixes：表示使用地址前缀最长匹配原则。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定网络地址ip-address，显示所有MBGP路由信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
