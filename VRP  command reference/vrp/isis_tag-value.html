isis tag-value

命令功能

isis tag-value命令用来配置接口的管理标记值。

undo isis tag-value命令用来取消接口的管理标记值。

缺省情况下，接口没有管理标记值。

命令格式

isis [ ipv6 ] tag-value tag [ level-1 | level-2 ]

undo isis [ ipv6 ] tag-value [ level-1 | level-2 ]

参数说明

ipv6：指定接口的IPv6管理标记值。如果不指定此参数，则默认为IPv4。

tag：指定接口的管理标记值。整数形式，取值范围是1～4294967295。

level-1：指定Level-1接口的管理标记值。

level-2：指定Level-2接口的管理标记值。

使用指南

只有当IS-IS的开销类型为wide，wide-compatible或compatible时，发布的LSP中才会携带tag值。

如果不指定Level，则同时配置Level-1和Level-2接口的管理标记值。

使用ipv6参数前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。

使用实例

# 配置接口GE1/0/0的管理标记值是77。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] isis tag-value 77
