local-ce mac

命令功能

local-ce mac命令用来在PE上配置与其相连的CE的以太接口的MAC地址。

undo local-ce mac命令用来删除配置。

缺省情况下，PE上没有配置与其相连的CE的以太接口的MAC地址。

命令格式

local-ce mac mac-address

undo local-ce mac

参数说明

mac-address：与本接口相连的CE的以太网主接口或子接口的MAC地址。

视图

VE接口视图、Ethernet接口视图、Ethernet子接口视图、GE接口视图、GE子接口视图、Eth-Trunk接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

该静态配置的MAC地址在PE向CE发送报文时使用。如果配置了静态MAC，所有通过L2VPN隧道透明传输过来的IP报文向CE发送时，均使用该MAC来封装报文。

如果是子接口，封装的VLAN ID为本接口的最小VLAN ID。所以，建议两端的VLAN ID最小值设置成一样。如果不一样，转发不通。

使用实例

# 配置与GE1/0/0相连的CE的以太网接口的MAC地址为00e0-fc20-b3a8。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] local-ce mac 00e0-fc20-b3a8
