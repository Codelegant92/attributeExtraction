local-id-type

命令功能

local-id-type命令用来指定IKE的ID类型。

undo local-id-type命令用来恢复缺省设置。

缺省情况下，ID类型为IP地址形式。

命令格式

local-id-type { ip | name }

undo local-id-type

参数说明

ip：指定IKE的ID类型为IP地址形式。

name：指定IKE的ID类型为名字形式。

缺省级别

2：配置级

使用指南

当进行IKE交换时，可以使用IP地址作为ID，也可以使用名称作为ID。缺省情况下，IKE采用的ID类型为IP地址形式。

当IKE协商模式为野蛮模式时，本地ID类型必须name；当IKE协商模式为主模式时，可以不配置本地ID，但是，本地类型不能为name。

使用实例

# 配置IKE的ID类型为名字形式。

<Quidway> system-view
[Quidway] ike peer huawei
[Quidway-ike-peer-huawei] local-id-type name
