mode user-termination（VE接口视图）

命令功能

mode user-termination命令用来配置虚拟以太网接口的工作模式为QinQ终结模式。

undo mode user-termination命令用来配置虚拟以太网接口的工作模式为非QinQ终结模式。

默认情况下，接口的工作模式为非QinQ终结模式。

命令格式

mode user-termination

undo mode user-termination

参数说明

无

视图

虚拟以太网接口视图

缺省级别

2：配置级

使用指南

只有在配置L2VPN接入多个L3VPN时，才需要配置虚拟以太网接口的工作模式为QinQ终结模式。

在将虚拟以太网接口配置成QinQ终结模式前，应该先使用命令ve-group ve-group-id l3-access将接口配置成L3VE，否则无法执行mode user-termination命令。

虚拟以太网接口工作在QinQ终结模式下时，可以创建子接口，使用子接口下的QinQ相关配置命令。

在虚拟以太网主接口上使用undo mode user-termination命令来配置接口的工作模式为非QinQ终结模式前，应先删除该主接口下的所有子接口，否则系统会显示“Error: Please delete all subinterfaces of this interface first.”，提示配置错误。

使用实例

# 配置接口VE1/0/1为QinQ终结模式。

<Quidway> system-view
[Quidway] interface virtual-ethernet 1/0/1
[Quidway-Virtual-Ethernet1/0/1] ve-group 1 l3-access
[Quidway-Virtual-Ethernet1/0/1] mode user-termination
