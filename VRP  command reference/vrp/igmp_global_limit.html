igmp global limit

命令功能

igmp global limit命令用来配置整个路由器上可以创建的所有IGMP表项的最大个数限制值。

undo igmp global limit命令用来取消整个路由器上IGMP表项总和的最大个数限制。

命令格式

igmp global limit number

undo igmp global limit

参数说明

number：指定整个路由器上可以创建的所有IGMP表项的最大个数。整数形式，取值范围是1～49152。

使用指南

系统统计IGMP表项个数的方法是：

  • 用于SSM-Mapping的每个（*，G）组成员关系统计为一个IGMP表项；

  • 对于INCLUDE模式的表项，只有（S，G）统计为一个IGMP表项，不统计（*，G）；

  • 对于EXCULDE模式的表项，每个（*，G）和（S，G）各统计为一个IGMP表项。

当统计的IGMP表项个数达到限制值后，将不再创建IGMP表项。如果确实需要加入新的组播组，建议删除一些无用的表项或者修改限制值，或者加入静态组播组/源组。

使用实例

# 配置路由器上可以创建IGMP接口表项总和的最大数量是16384。

<Quidway> system-view
[Quidway] igmp global limit 16384
