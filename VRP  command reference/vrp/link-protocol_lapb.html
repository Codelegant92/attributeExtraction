link-protocol lapb

命令功能

link-protocol lapb命令用来指定接口链路层协议为LAPB协议。

缺省情况下，LAPB工作方式为DTE，承载网络层IP协议。

命令格式

link-protocol lapb [ dte | dce ] [ ip | multi-protocol ]

参数说明

dte：指定接口工作在LAPB的DTE方式。

dce：指定接口工作在LAPB的DCE方式。

ip：指定LAPB承载的网络层协议为IP协议。

multi-protocol：指定LAPB可以承载多种网络层协议。

使用指南

虽然LAPB是X.25的二层协议，但可以作为独立的链路层协议来进行简单的数据传输。一般在两台路由器用专线直连时可以使用LAPB协议，此时一端工作在DTE方式，另一端工作在DCE方式。

使用实例

# 配置接口Serial1/0/0的链路层协议为LAPB，并工作在DCE方式。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol lapb dce
