dhcp server database

命令功能

dhcp server database命令用来使能将当前DHCP数据保存到存储设备的功能。

undo dhcp server database命令用来禁止此功能。

缺省情况下，不启用DHCP数据保存到存储设备的功能。

命令格式

dhcp server database { enable | recover | write-delay interval }

undo dhcp server database { enable | recover | write-delay }

参数说明

enable：使能DHCP数据库。

recover：从存储设备恢复数据。

write-delay interval：保存时延。整数形式，取值范围是300～86400，单位是秒。缺省值是300秒。

使用指南

如果启用此功能，将生成两个文件，名称分别为lease.txt和conflict.txt，分别保存正常的地址租借信息和地址冲突信息。

该功能与具体使用的产品有关，当产品提供存储设备时，才可以设置该功能。

使用实例

# 使能DHCP数据保存到存储设备的功能。

<Quidway> system-view
[Quidway] dhcp server database enable
