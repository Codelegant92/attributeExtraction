tunnel-dei trust instance-dei

命令功能

tunnel-dei trust instance-dei命令用来配置当前PBB-TE Tunnel隧道的DEI值为信任业务实例的DEI值。

undo tunnel-dei命令用来将当前PBB-TE Tunnel隧道的DEI值恢复为缺省值。

缺省情况下，PBB-TE Tunnel隧道的DEI值为假（0）。

命令格式

tunnel-dei trust instance-dei

undo tunnel-dei

参数说明

无

视图

MAC Tunnel视图

缺省级别

2：配置级

使用指南

PBB-TE Tunnel的DEI即B-TAG的DEI，主要应用于QoS业务。

若PBB-TE Tunnel的DEI值为信任业务实例的DEI值，则instance-dei ingress命令设置源端业务实例的DEI值即相当于设置PBB-TE Tunnel的DEI值。

如果PBB-TE Tunnel配置了保护组，对工作隧道执行tunnel-dei命令，保护隧道的配置也会同步更新。

本命令只能在工作隧道上使用。

使用实例

# 配置当前隧道的DEI为信任业务实例的DEI。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-service-instance-mytunnel] tunnel-dei trust instance-dei
