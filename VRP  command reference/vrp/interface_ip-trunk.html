interface ip-trunk

命令功能

interface ip-trunk命令用来创建一个IP-Trunk接口。

undo interface ip-trunk命令用来删除一个已存在的IP-Trunk接口。

命令格式

interface ip-trunk trunk-id

undo interface ip-trunk trunk-id

参数说明

trunk-id：IP-Trunk的接口编号，取值范围是0～127。

视图

系统视图

缺省级别

2：配置级

使用指南

接口相关的参数都需要在接口视图下配置，只有先创建了IP-Trunk接口，才可以对它进行配置。

使用实例

# 创建一个IP-Trunk接口，编号为1。

<Quidway> system-view
[Quidway] interface ip-trunk 1
[Quidway-Ip-Trunk1]
