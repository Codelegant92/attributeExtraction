display fib statistics all

命令功能

display fib statistics all命令用来查看IPv4 FIB的统计信息，主要包括接口板IPv4 FIB前缀容量及公网和VPN的IPv4 FIB路由前缀及转发表项数量。

命令格式

display fib [ slot-number ] statistics [ all ]

参数说明

slot-number：需要查看IPv4 FIB统计信息的槽号。当不指定槽号时，查看的是主控板上所有的IPv4 FIB表项信息。

使用指南

执行命令display fib statistics all时，

  • 如果指定槽号，则可以查看指定槽号的接口板上的IPv4 FIB路由前缀容量及各类IPv4 FIB策略的转发表项数量。

  • 如果不指定槽号，则可以查看主控板上所有的IPv4 FIB转发表项信息。

使用实例

# 查看2号槽的接口板上的IPv4 FIB策略的统计信息。

<Quidway> display fib 2 statistics all
IPv4 FIB Route Prefix Capacity : 320000
IPv4 FIB Total Route Prefix Count :2 ; Entry Count : 2

IPv4 FIB Public Route Prefix Count :2 ; Entry Count : 2
IPv4 FIB Vpn-instance vpna Route Prefix Count : 0; Entry Count : 0
表1 display fib statistics all输出信息描述

项目

描述

IPv4 FIB Route Prefix Capacity :

本接口板的IPv4 FIB路由前缀容量。

IPv4 FIB Total Route Prefix Count : Entry Count :

本接口板上应用的IPv4 FIB策略的路由前缀总数和转发表项总数。

IPv4 FIB Public Route Prefix Count : Entry Count :

公网转发表项的IPv4路由前缀总数和转发表项总数。

IPv4 FIB Vpn-instance vpna Route Prefix Count : Entry Count :

名为vpna的IPv4 VPN实例的路由前缀总数和转发表项总数。
