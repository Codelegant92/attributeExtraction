mask

命令功能

mask命令用来配置聚合掩码。

undo mask命令用来取消聚合掩码的设置。

命令格式

mask { source | destination } minimum mask-length

undo mask { source | destination }

参数说明

source：源地址聚合掩码。

destination：目的地址聚合掩码。

mask-length：聚合掩码长度。整数形式，取值范围是1～32。

视图

NetStream聚合视图

缺省级别

2：配置级

使用指南

source用于prefix、prefix-tos、source-prefix、source-prefix-tos模式。

destination用于perfix、prefix-tos、destination-prefix、destination-prefix-tos模式。

系统使用的掩码为FIB表的掩码与所配置掩码的最大值，在未配置聚合掩码的情况下，使用FIB表中的掩码作为聚合依据。

使用实例

# 配置source-prefix聚合的掩码长度是24。

<Quidway> system-view
[Quidway] ip netstream aggregation source-prefix
[Quidway-aggregation-srcpre] mask source minimum 24
