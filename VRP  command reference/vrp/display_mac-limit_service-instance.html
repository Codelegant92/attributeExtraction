display mac-limit service-instance

命令功能

display mac-limit service-instance命令用来查看业务实例上的MAC地址学习限制规则。

命令格式

display mac-limit [ service-instance [ instance-name service-instance-name ] ]

参数说明

service-instance service-instance-name：指定业务实例名字。长度范围是1～31个字符。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不输入业务实例的名字，查看所有业务实例上的MAC地址学习限制规则。如果输入指定的业务实例的名字，则可以查看该业务实例上的MAC地址学习限制规则。

使用实例

# 查看所有业务实例上的MAC地址学习限制规则。

<Quidway> display mac-limit service-instance
SI                                        Maximum         Action
----------------------------------------------------------------
test1                                     1000            discard
test                                      100             forward

# 查看业务实例test上的MAC地址学习限制规则。

<Quidway> display mac-limit service-instance instance-name test
SI                                        Maximum         Action
----------------------------------------------------------------
test                                      100             forward

项目

描述

SI

业务实例的名字。

Maximum

配置基于业务实例的MAC学习限制数目（如果此项没有配置，则此命令不输出信息）。

Action

基于业务实例的MAC学习超出限制规则时的动作（取值为forward，discard）。
