apply extcommunity

命令功能

apply extcommunity命令用来为BGP路由指定扩展团体属性。

undo apply extcommunity命令用来取消该功能。

缺省情况下，不设置BGP的扩展团体属性。

命令格式

apply extcommunity rt { as-number : nn | ipv4-address : nn } [ additive ]

undo apply extcommunity

参数说明

rt：指定Route Target（路由目标）扩展团体。

as-number：自治系统号。整数形式，取值范围0～65535。

ipv4-address：IPv4地址。

nn：一个整数。对于as-number，其取值范围为0～4294967295；对于ipv4-address，其取值范围为0～65535。

additive：允许给路由增加已有的团体属性。

缺省级别

2：配置级

使用指南

目前和VPNv4相关的BGP扩展团体有两种，一种是用于VPN的路由目标扩展团体，另一种是源站点扩展团体。目前，仅支持通过Route-Policy设置Route-Target扩展团体属性。如果命令中没有additive关键字，则是替换原有的扩展团体属性。

使用实例

# 将100:2添加到BGP的VPN Route-Target扩展团体属性中。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply extcommunity rt 100:2 additive
