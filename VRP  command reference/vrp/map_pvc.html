map pvc

命令功能

map pvc命令用来配置对端和本地VPI/VCI的映射。

undo map pvc命令用来删除该映射。

缺省情况下，不配置任何映射。

命令格式

map pvc vpi/vci

undo map pvc

参数说明

vpi/vci：vpi为ATM网络虚路径标识（Virtual Path Identifier），取值范围是0～255；vci为ATM网络虚通道标识（Virtual Channel Identifier），取值范围是0～2047。通常，vci取值0到31保留用于特定用途，建议用户不要使用。

说明：

参数vpi与vci不能同时为0。

使用指南

本命令只用于ATM透传。

如果对端VCI和本地不同，需要配置VCI映射。对于从本地发出的信元，VCI会被更改为对端的VCI。

使用实例

# 创建ATM子接口上的映射。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvc 1/20
[Quidway-atm-pvc-Atm1/0/0-1/20] map pvc 2/30
