display dhcp option82

命令功能

display dhcp option82命令用来显示option82的使能标记。

命令格式

display dhcp option82 { [ vlan vlan-id ] interface interface-type interface-number }

参数说明

interfaceinterface-numberinterface-type：接口类型和接口编号。

vlan-id：表示VLAN ID，取值范围1～4094。此参数仅在二层设备上可配置。

使用指南

执行display dhcp option82命令后，设备将显示VLAN和接口上使能的Option82信息。

使用实例

# 显示Ethernet4/0/0接口相关的信息。

<Quidway> display dhcp option82 interface ethernet 4/0/0
dhcp option82 enable insert interface ethernet4/0/0
dhcp option82 enable rebuild interface ethernet4/0/0
