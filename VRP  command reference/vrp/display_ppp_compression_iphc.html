display ppp compression iphc

命令功能

display ppp compression iphc命令用来查看PPP链路上对IP报文进行头压缩的统计信息。

命令格式

display ppp compression iphc { rtp | tcp } [ interface-type interface-number ]

参数说明

rtp：显示对RTP报文进行头压缩的统计信息。

tcp：显示对TCP报文进行头压缩的统计信息。

interface-type interface-number：进行压缩的链路所对应的接口类型和接口编号。

项目

描述

IPHC

IP报文头压缩。

Interface:

报文头压缩应用的接口。

Received:

接收到的报文情况统计。

Compress/Error/Discard/Total:

压缩报文数/错误报文数/丢弃报文数/报文总数。

Sent:

发送的报文统计情况。

Compress/Total

压缩报文数/报文总数。

Send/Save/Total

发送字节数/保存字节数/字节总数。

Compression Ratio(Total/Send)

压缩效率。

Connect:

连接数

Rx

作为接收方,可解压缩的数据流的最大数目。

Tx

作为发送方,可压缩的数据流的最大数目。

Long-search

查找压缩表项的次数。

Miss

查找压缩表项的失败次数。

Hit-Ratio

查找压缩表项的成功命中次数。

Five-Minute-Miss-Ratio:

按5分钟计算的未压缩报文的比率。

Max-Miss-Ratio:

最大未压缩报文比率。
