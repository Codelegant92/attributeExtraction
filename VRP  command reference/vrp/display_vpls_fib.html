display vpls fib

命令功能

display vpls fib命令用来显示VPLS FIB信息。

命令格式

display vpls fib [ vsi vsi-name | link link-id ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vsi vsi-name：用于显示指定VSI实例的FIB信息。字符串形式，不支持空格，区分大小写，取值范围是1～31。

link link-id：用于显示指定连接的FIB信息。十进制整数形式，取值范围是1～40960。

verbose：用于显示详细的FIB信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

定位VPLS故障时，可以使用display vpls fib命令查看VPLS的转发信息。执行display vpls fib命令后，可以通过在命令行中指定不同的关键字或参数来显示所需要的转发表项信息。

如果此命令不指定关键字或参数，将显示所有的转发表项信息。

使用实例

# 显示所有VPLS的转发表的信息。

<Quidway> display vpls fib
Vsi-Name                     Link-ID   Link-Type             Link-State
company1                     1          VPLS Remote Link     up

显示信息的解释请参见下表。

# 显示名称为company1的VSI实例的转发表的详细信息。

<Quidway> display vpls fib vsi company1 verbose
