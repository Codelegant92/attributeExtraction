ftp-filename

命令功能

ftp-filename命令用来配置NQA测试FTP测试例的文件名和文件路径。

undo ftp-filename命令用来恢复默认配置。

缺省情况下，没有配置文件名和文件路径。

命令格式

ftp-filename name

undo ftp-filename

参数说明

name：FTP测试的操作文件的路径和文件名。字符串形式，长度范围是1～230。其中文件名长度是1～64。

缺省级别

2：配置级

使用指南

ftp-filename命令只对FTP类型的测试有效。

不能更改正在运行的测试例的文件路径和文件名。

可以不指定文件路径，系统会在当前路径下查找该文件。

文件名不能包含字符：“~、*、/、\、'、"、，”，文件路径可以包含这些字符。

文件名中可以包含扩展名，但不能只有扩展名。如.txt。

使用实例

# 设置测试例user test的FTP操作文件。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type ftp
[Quidway-nqa-user-test] ftp-filename D:\abc\abc.txt
