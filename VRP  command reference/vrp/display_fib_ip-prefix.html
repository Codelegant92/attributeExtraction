display fib ip-prefix

命令功能

display fib ip-prefix命令用来过滤显示FIB信息。

命令格式

display fib [ vpn-instance vpn-instance-name ] ip-prefix prefix-name [ verbose ]

参数说明

vpn-instance vpn-instance-name：VPN实例的名称。字符串形式，区分大小写，长度范围是1～31。

prefix-name：指定前缀列表名。字符串形式，长度是1～169。

verbose：表示查看转发信息表的详细信息。

使用指南

根据所输入的Prefix-name名字，把通过了该过滤规则的FIB表项按照一定格式显示出来。

如果匹配的FIB表项条数为0，则提示与之匹配的FIB表项为0。

如果找不到相应的Prefix-name名字，则将所有的FIB表项信息显示出来。

项目

描述

Destination/Mask

目的地址/掩码长度

Nexthop

下一跳

Flag

当前标志，G、H、U、S、D的组合。G表示Gateway（网关路由）H表示Host（主机路由）U表示Up（可用路由）S表示Static（静态路由）D表示Dynamic（动态路由）

TimeStamp

时间戳，表示该表项存在的时间，单位是秒。

Interface

到目的地址的出接口。

TunnelID

表示转发表项索引。用于连接上行板和下行板之间的转发表项。该值不为0时，表示匹配该项的报文通过MPLS隧道转发。该值为0时，表示报文不通过MPLS隧道转发。
