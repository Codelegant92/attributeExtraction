load-balance packet

命令功能

load-balance packet命令用来配置IP主机数据和MPLS三层转发数据采用逐包负载分担。

缺省情况下，IP主机数据和MPLS三层转发数据采用逐流负载分担。

命令格式

load-balance packet [ lsp ] [ all | slot slot-id ]

参数说明

lsp：配置IP主机数据和MPLS三层转发数据只在LSP上进行逐包负载分担。如果不使用该参数，数据同时在IP路径和LSP上进行负载分担。

all：将配置应用于所有的接口板。

slot slot-id：将配置应用于指定接口板的槽号。
