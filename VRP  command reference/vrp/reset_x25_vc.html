reset x25 vc

命令功能

reset x25 vc命令用来重新建立X.25的虚电路连接。

命令格式

reset x25 vc interface interface-type interface-number [ vc-number ]

参数说明

interface interface-type interface-number：需要重新建立虚连接的接口类型和接口编号。

vc-number：虚电路号。

视图

用户视图

缺省级别

2：配置级

使用指南

重新建立虚连接会对现行业务造成中断，请慎用此命令。

使用实例

# 重新建立串口Serial 1/0/0上的所有虚连接。

<Quidway> reset x25 vc interface serial 1/0/0
