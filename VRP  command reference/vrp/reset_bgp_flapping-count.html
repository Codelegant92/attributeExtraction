reset bgp flapping-count

命令功能

reset bgp flapping-count命令用来清除指定BGP对等体的震荡次数。

命令格式

reset bgp [ vpn-instance vpn-instance-name ] ipv4-address flapping-count

reset bgp { ipv6 | vpn6-instance vpn6-instance-name } ipv6-address flapping-count

参数说明

vpn-instance vpn-instance-name：指定VPN实例的名称。

ipv4-address：指定BGP的IPv4对等体地址。

ipv6：指定清除IPv6对等体的振荡次数。

vpn6-instance vpn6-instance-name：指定IPv6 VPN实例的名称。

ipv6-address：指定BGP的IPv6对等体地址。

使用指南

查看指定时间内BGP对等体的震荡次数时，需先从计算的时间点开始使用该命令将计数清零，然后再使用display bgp peer命令查看BGP对等体的震荡次数。

使用实例

# 清除指定BGP对等体的的震荡次数。

<Quidway> reset bgp 100.116.10.2 flapping-count
