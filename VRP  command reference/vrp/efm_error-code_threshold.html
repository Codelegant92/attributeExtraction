efm error-code threshold

命令功能

efm error-code threshold命令用来配置当前接口的EFM OAM误码检测门限。

undo efm error-code threshold命令用来恢复当前接口的EFM OAM误码检测门限为缺省值。

命令格式

efm error-code threshold threshold

参数说明

threshold：指定EFM OAM误码检测门限。整数形式，取值范围是0～65535。缺省值是1。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

当EFM OAM误码检测上报功能使能后，如果在设置的观察时长内，接口检测到的误码数量达到或者超过门限值，那么设备将产生误码越限事件，并通告对端设备。

使用efm enable命令全局使能EFM OAM功能后才能使用本命令。

使用实例

# 配置接口Ethernet1/0/0的EFM OAM误码检测门限为100。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] efm error-code threshold 100
