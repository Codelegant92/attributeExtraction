efm error-frame period

命令功能

efm error-frame period命令用来配置当前接口的EFM OAM误帧检测观察时长。

undo efm error-frame period命令用来恢复当前接口的EFM OAM误帧检测观察时长为缺省值。

命令格式

efm error-frame period period

undo efm error-frame period

参数说明

period：指定EFM OAM误帧检测观察时长。整数形式，取值范围是1～60，单位是秒。缺省值是1秒。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

当EFM OAM误帧检测上报功能使能后，如果在设置的观察时长内，接口检测到的误帧数量达到或者超过门限值，那么设备将产生误帧越限事件，并通告对端设备。

使用efm enable命令全局使能EFM OAM功能后才能使用本命令。

使用实例

# 配置接口GigabitEthernet1/0/0的EFM OAM误帧检测观察时长为20秒。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] efm error-frame period 20
