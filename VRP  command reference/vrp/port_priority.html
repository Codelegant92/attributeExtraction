port priority

命令功能

port priority命令用来设置端口的优先级，优先级的数值较大的端口接收的报文将被优先转发。

缺省情况下，端口的优先级值为0。

命令格式

port priority priority-value

undo port priority

参数说明

priority-value：优先级的值，取值范围是0～7。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

如果使用三层端口，必须先执行portswitch（Ethernet接口视图）命令切换为二层端口。

使用实例

# 设置GE1/0/0的优先级值为3。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port priority 3
