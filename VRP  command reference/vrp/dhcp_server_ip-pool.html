dhcp server ip-pool

命令功能

dhcp server ip-pool命令用来创建DHCP地址池并进入DHCP地址池视图。

undo dhcp server ip-pool命令用来删除指定的地址池。

缺省情况下，系统未创建DHCP地址池。

命令格式

dhcp server ip-pool pool-name

undo dhcp server ip-pool pool-name

参数说明

pool-name：指定IP地址池名称。是地址池的唯一标识，字符串形式，长度范围是1～35。

缺省级别

2：配置级

使用指南

执行dhcp server ip-pool命令时，如果指定的地址池已经存在，则直接进入DHCP地址池视图；否则先创建该地址池，再进入DHCP地址池视图。

每个DHCP服务器可以配置多个地址池，目前支持50个非本地地址池。

使用实例

# 创建标识为0的DHCP地址池。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0]
