reset cpu-packet statistics slot

命令功能

reset cpu-packet statistics slot命令用来清除槽位中所有上送到CPU的报文的统计信息。

命令格式

reset cpu-packet statistics slot { all | slot-id }

参数说明

all：清除所有槽位的统计信息。

slot-id：清除指定槽位的统计信息。

视图

用户视图

缺省级别

3：管理级

使用指南

在某些情况下，需要统计一定时间内某接口板上送CPU的流量信息，这时必须在统计开始前清除该接口板原有的统计信息，使接口板的流量重新进行统计。

使用实例

# 清除所有上送到CPU的报文的统计功能。

<Quidway> reset cpu-packet statistics slot all
