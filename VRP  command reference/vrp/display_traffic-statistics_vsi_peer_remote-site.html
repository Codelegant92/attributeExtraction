display traffic-statistics vsi peer remote-site

命令功能

display traffic-statistics vsi peer remote-site命令用来显示指定VSI-Name中指定Kompella方式VPLS PW上的公网流量统计信息。

命令格式

display traffic-statistics vsi vsi-name [ peer peer-address [ remote-site site-id ] ]

参数说明

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

peer peer-addres：PW的对端IP地址。

remote-site site-id：远端站点值。

视图

所有视图

缺省级别

1：监控级

使用指南

在所有视图下显示指定Kompella方式VPLS PW流量统计信息。Kompella方式VPLS PW由用户指定的VSI-Name与指定的下一跳IP地址和Remote-site值唯一确定。

5分钟以内，如果出现了PW down的情况，PW down之前统计的流量不被用来计算5分钟的流量速率。

使用实例

# 在系统视图下显示指定Kompella方式VPLS PW的流量统计信息。

<Quidway> display traffic-statistics vsi-name newvsi peer 100.22.33.20 remote-site 2
