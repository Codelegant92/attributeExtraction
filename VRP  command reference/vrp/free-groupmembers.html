free-groupmembers

命令功能

free-groupmembers命令用来释放NQA测试组的所有成员测试例。

命令格式

free-groupmembers

参数说明

无

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对NQA测试组进行释放成员测试例的操作。

如果所有成员测试例被释放，那么其测试结果也丢失。

使用实例

# 释放当前测试组的所有成员测试例。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] free-groupmembers
