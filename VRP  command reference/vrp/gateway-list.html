gateway-list

命令功能

gateway-list命令用来配置DHCP客户端使用的出口网关路由设备的IP地址。

undo gateway-list命令用来删除配置。

缺省情况下，未配置出口网关路由设备。

命令格式

gateway-list ip-address &<1-8>

undo gateway-list { ip-address | all }

参数说明

ip-address：指定出口网关路由设备的IP地址。一条命令中最多可以配置8个IP地址，用空格分隔。

all：表示删除所有出口网关路由设备IP地址。

使用指南

DHCP客户端访问本网段以外的服务器或主机时，数据必须通过出口网关进行收发。

使用gateway-list命令用来配置DHCP客户端使用的出口网关路由设备的IP地址。

使用实例

# 将0号DHCP地址池对应的出口网关路由设备设为10.110.1.99。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] gateway-list 10.110.1.99
