display firewall fragment

命令功能

display firewall fragment命令显示分片表项的相关信息。

命令格式

display firewall fragment slot slot-id

参数说明

slot-id：为指定的接口板槽号。

使用指南

display firewall fragment可以查看防火墙对分片报文的监控。
