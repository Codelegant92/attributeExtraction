snmp-agent trap enable vrrp

命令功能

snmp-agent trap enable vrrp命令用来允许VRRP模块向网管发送Trap报文。

undo snmp-agent trap enable vrrp命令用来取消当前设置。

缺省情况下，VRRP模块可以发送Trap报文。

命令格式

snmp-agent trap enable vrrp

undo snmp-agent trap enable vrrp

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

snmp-agent trap enable vrrp命令必须与snmp-agent target-host命令协同使用。

snmp-agent target-host命令指定向哪些主机发送Trap信息。

为了发送Trap信息，用户必须配置至少一条snmp-agent target-host命令。

使用实例

# 允许向10.1.1.1发送所有类型的VRRP Trap报文，Trap报文格式为V3，对报文进行认证但不加密，使用团体名为super。

[Quidway] snmp-agent trap enable vrrp
[Quidway] snmp-agent target-host trap address udp-domain 10.1.1.1 params securityname super v3 authentication
