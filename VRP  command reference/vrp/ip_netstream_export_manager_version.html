ip netstream export manager version

命令功能

ip netstream export manager version version-number命令用来配置向网管输出的版本号。

缺省向网管输出增强版本1。

命令格式

ip netstream export manager version version-number

参数说明

version-number：整数形式，取值范围0～1。0代表向网管输出原始版本报文，1代表向网管输出增强版本报文。

视图

系统视图

缺省级别

2：配置级

使用指南

在全局模式下配置向网管发送报文的新旧网管版本号，控制向网管输出的模板和数据格式，兼容网管端原有的网管系统。

使用实例

# 配置输出旧网管版本。

<Quidway> system-view
[Quidway] ip netstream export manager version 0
