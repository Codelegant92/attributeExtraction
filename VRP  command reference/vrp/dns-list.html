dns-list

命令功能

dns-list命令用来配置全局DHCP地址池分配给客户端的DNS服务器的IP地址。

undo dns-list命令用来删除配置。

缺省情况下，未配置DNS服务器地址。

命令格式

dns-list ip-address &<1-8>

undo dns-list { ip-address | all }

参数说明

ip-address：指定DNS的IP地址，一条命令中最多可以配置8个IP地址，用空格分隔。

all：删除所有已配置的要分配给客户端的DNS服务器的IP地址。

使用指南

dns-list命令用来配置全局DHCP地址池分配给客户端的DNS服务器的IP地址。

在使用DNS服务之前必须已经完成DNS服务器的配置，且存在从路由设备到DNS服务器的路由。

在目前的实现中，每个DHCP地址池最多可以设置8个DNS服务器地址。

使用实例

# 为0号DHCP地址池指定IP地址为1.1.1.254的DNS服务器。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] dns-list 1.1.1.254
