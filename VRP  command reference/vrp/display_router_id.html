display router id

命令功能

display router id命令用来显示已配置的路由器ID。

命令格式

display router id [ vpn-instance vpn-instance-name ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例的名称。

使用实例

# 显示已配置的路由器ID。

<Quidway> display router id
RouterID:1.1.1.1

# 显示已配置的私网实例vpna的路由器ID。

<Quidway> display router id vpn-instance vpna
RouterID:10.1.1.1
