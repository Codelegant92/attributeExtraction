mfr fragment-size

命令功能

mfr fragment-size命令用来配置帧中继捆绑链路允许的最大分片大小。

undo mfr fragment-size命令用来恢复缺省设置。

缺省情况下，帧中继捆绑链路允许的最大分片是300字节。

命令格式

mfr fragment-size bytes

undo mfr fragment-size

参数说明

bytes：分片的大小。整数形式，取值范围是60～1500，单位是字节。

视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

帧中继接口视图下配置的分片大小的优先级高于MFR接口视图下的配置。

使用实例

# 配置多链路帧中继捆绑链路Serial1/0/0:0允许的最大分片为60字节。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr mfr 1/0/123
[Quidway-Serial1/0/0:0] mfr fragment-size 60
