stp binding process

命令功能

stp binding process命令用来使当前接口加入指定ID的MSTP进程中。

undo stp binding process命令用来使当前接口退出指定ID的MSTP进程中。

缺省情况下，接口属于MSTP进程0。

命令格式

stp binding process process-id

undo stp binding process process-id

参数说明

process process-id：MSTP进程的ID值，十进制整数形式，取值范围是1～256。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

设备正常启动后，当前接口的MSTP相关配置默认属于ID为0的MSTP进程。如果要把当前接口加入到其他的MSTP进程中，请分别使用stp process命令和stp binding process命令。

使用实例

# 把接口GE1/0/1加入ID为1的MSTP进程中。

<Quidway> system-view
[Quidway] stp process 1
[Quidway-stp-process-1] quit
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] portswitch
[Quidway-GigabitEthernet1/0/1] stp binding process 1
