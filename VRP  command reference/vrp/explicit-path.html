explicit-path

命令功能

explicit-path命令用来配置隧道的显式路径。

undo explicit-path命令用来删除配置的路径。

命令格式

explicit-path path-name

undo explicit-path path-name

参数说明

path-name：隧道的显式路径名称。字符串形式，长度为1～31。

视图

系统视图

缺省级别

2：配置级

使用指南

必须启动MPLS TE功能后才能配置隧道的显式路径。

显示路径上的节点地址不能重复，也不能形成环路。如果有环路，CSPF将检测出环路，无法成功计算出路径。

当显式路径正在使用时，不能进行如下操作：

  • 执行undo explicit-path命令删除该显式路径。

  • 对该显示路径执行next hop、add hop、delete hop或modify hop命令进行添加、删除、修改节点地址。

使用实例

# 创建一条名为path1的显式路径。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls te
[Quidway-mpls] quit
[Quidway] explicit-path path1
[Quidway-explicit-path-path1] quit
