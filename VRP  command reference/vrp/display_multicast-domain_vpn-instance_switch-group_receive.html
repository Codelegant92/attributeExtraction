display multicast-domain vpn-instance switch-group receive

命令功能

display multicast-domain vpn-instance switch-group receive命令用来查看指定VPN实例中接收到的Switch-Group信息。

命令格式

display multicast-domain vpn-instance vpn-instance-name switch-group receive brief [ | count ] [ | { begin | exclude | include } regular-expression ]

display multicast-domain vpn-instance vpn-instance-name switch-group receive [ active | group group-address | sender source-address | vpn-source-address [ mask { source-mask-length | source-mask } ] | vpn-group-address [ mask { group-mask-length | group-mask } ] ] * [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

active：显示接收的并且已经加入了Switch-MDT的Switch-Group信息。

brief：显示接收的Switch-Group的简要信息。

group group-address：显示与指定公网组播组group-address相关的Switch-Group信息。group-address表示组播组地址。点分十进制形式，取值范围是224.0.0.0～239.255.255.255。

sender source-address：显示与指定公网组播源source-address相关的Switch-Group信息。点分十进制形式。source-address表示组播源地址。点分十进制形式。

vpn-source-address：显示与指定VPN组播源vpn-source-address相关的Switch-Group信息。点分十进制形式。

mask：指定VPN组播组或源的地址掩码。

source-mask-length：表示VPN组播源地址的掩码长度。整数形式，取值范围是0～32。

source-mask：表示VPN组播源地址的掩码。点分十进制形式。

vpn-group-address：显示与指定VPN组播组vpn-group-address相关的Switch-Group信息。点分十进制形式，取值范围是224.0.1.0～239.255.255.255。

group-mask-length：表示VPN组播组地址的掩码长度。整数形式，取值范围是4～32。

group-mask：表示VPN组播组地址的掩码。点分十进制形式。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display multicast-domain vpn-instance switch-group receive命令，显示本地路由器接收到的Switch-Group信息：

  • VPN实例与MD一一对应，命令中vpn-instance-name指定了VPN实例，也就确定了MD。使用该命令仅显示指定MD的Switch-Group信息。

  • 如果指定brief关键字，则列举出所有的Switch-Group简要信息。以Switch-Group地址从小到大顺序显示。

  • 如果指定active关键字，则仅显示已经加入了Switch-MDT的Switch-Group信息。以私网（S，G）项从小到大顺序显示。

  • 如果指定group关键字，则仅显示与指定公网组播组相关的Switch-Group信息。

  • 如果指定sender关键字，则仅显示与指定公网组播源相关的Switch-Group信息。

  • 如果指定vpn-source-address，则仅显示与指定私网组播源相关的Switch-Group信息。

  • 如果指定vpn-group-address，则仅显示与指定私网组播组相关的Switch-Group信息。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
