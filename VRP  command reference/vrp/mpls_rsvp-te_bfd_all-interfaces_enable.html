mpls rsvp-te bfd all-interfaces enable

命令功能

mpls rsvp-te bfd all-interfaces enable命令用来全局使能BFD for RSVP。全局使能将使得所有没有禁止BFD for RSVP的接口具有创建BFD会话的能力。

mpls rsvp-te bfd all-interfaces enable命令用来恢复为缺省配置。

缺省情况下，不使能BFD for RSVP。

命令格式

mpls rsvp-te bfd all-interfaces enable

undo mpls rsvp-te bfd all-interfaces enable

参数说明

无
