encap-data-enable

命令功能

encap-data-enable命令用来使能在SA消息中封装组播数据报文。

undo encap-data-enable命令用来恢复缺省配置。

缺省情况下，SA消息中只包含（S，G）项，不封装组播数据报文。

命令格式

encap-data-enable

undo encap-data-enable

参数说明

无

视图

公网实例MSDP视图、VPN实例MSDP视图

缺省级别

2：配置级

使用指南

源端RP将本域内的所有活动源信息，放入多条SA消息中发布出去，每条SA消息可以同时包含多个（S，G）项。

在源端RP上配置encap-data-enable后，RP会将接收到的注册报文中封装的组播数据封装在SA消息中发送给MSDP的对端。一条SA消息中只封装一个组播数据报文。

通过在MSDP peer上配置encap-data-enable，路由器可以在域间传输带组播数据的SA消息。

使用实例

# 在SA消息中封装组播数据报文。

<Quidway> system-view
[Quidway] msdp
[Quidway-msdp] encap-data-enable
