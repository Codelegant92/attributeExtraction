dhcp enable

命令功能

dhcp enable命令用来使能DHCP服务。

undo dhcp enable命令用来禁止DHCP服务。

缺省情况下，系统已经使能DHCP服务。

命令格式

dhcp enable

undo dhcp enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

在进行DHCP配置之前，必须先使能DHCP服务。该配置对于DHCP服务器和DHCP中继都是必须的。

使用实例

# 在当前路由设备上使能DHCP服务。

<Quidway> system-view
[Quidway] dhcp enable
