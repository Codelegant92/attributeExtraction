reset ip routing-table statistics protocol

命令功能

reset ip routing-table statistics protocol命令用来清除IPv4路由表中的统计信息。

命令格式

reset ip routing-table statistics protocol [ vpn-instance vpn-instance-name ] { all | protocol }

参数说明

vpn-instance vpn-instance-name：指定VPN实例名，字符串长1～31个字符。

all：清除路由表中所有路由协议的统计信息。

protocol：清除路由表中指定路由协议的统计信息。它的取值可以是下列关键字中的一个：bgp、direct、isis、ospf、rip、unr或static。

使用指南

清除IPv4路由表中的统计信息后，以前的统计信息将无法恢复，务必仔细确认。

使用实例

# 清除IPv4路由表中所有路由协议的统计信息。

<Quidway> reset ip routing-table statistics protocol all
