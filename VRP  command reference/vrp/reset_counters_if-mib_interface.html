reset counters if-mib interface

命令功能

reset counters if-mib interface命令用来清除网管的接口流量统计信息。

命令格式

reset counters if-mib interface [ interface-type [ interface-number ] ]

参数说明

interface-type interface-number：需要清除网管的接口统计计数的接口类型和接口编号。如果不指定接口类型和接口编号，将清除所有接口的网管统计计数。

模式

用户视图

级别

2：配置级

使用指南

当需要在网管统计一定时间内某接口的流量时，需要先清除该接口在网管的原有统计信息，使网管对该接口重新进行统计。

说明：

关于如何在网管查看接口的流量统计信息，请参见相应的网管类手册。

执行命令reset counters if-mib interface后，网管的接口流量统计信息会全部清零。所以请务必仔细确认是否要执行该命令。

执行命令reset counters if-mib interface对命令display interface显示的接口流量统计信息不影响。当需要清除命令display interface查看到的接口统计信息时，可以执行命令reset counters interface。

使用实例

# 清除网管的接口POS1/0/0上的统计信息。

<Quidway> reset counters if-mib interface pos 1/0/0
