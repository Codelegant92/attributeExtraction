local-ce mac broadcast

命令功能

local-ce mac broadcast命令用来在PE上与CE相连的以太接口使能广播。

undo local-ce mac broadcast命令用来取消此功能。

缺省情况下，PE与CE相连的以太接口没有使能广播。

命令格式

local-ce mac broadcast

undo local-ce mac broadcast

参数说明

无

视图

VLAN接口视图、VE接口视图、Ethernet接口视图、Ethernet子接口视图、GE接口视图、GE子接口视图、Eth-Trunk接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

该配置在PE向CE发送报文没有本地CE的静态或动态MAC地址且没有配置与其相连的CE的以太接口的IP地址（local-ce ip）时，用于向CE发送IP报文，目的MAC为广播地址。

如果是子接口，封装的VLAN ID为本接口的最小VLAN ID。所以，建议两端的VLAN ID最小值设置成一样。如果不一样，转发不通。

使用实例

# 在以太网子接口GE1/0/0.1上使能广播。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] local-ce mac broadcast
