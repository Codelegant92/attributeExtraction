display cluster members

命令功能

display cluster members命令用来查看HGMP集群内命令交换机和成员交换机的信息。

命令格式

display cluster members [ member-number | verbose ]

参数说明

member-number：指定成员交换机的成员编号，以查看此成员交换机的详细信息。十进制整数形式，取值范围是0～255。

verbose：显示所有成员交换机的详细信息。

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

不指定可选参数时，本命令将显示HGMP集群内所有命令交换机和成员交换机的简要信息。

使用实例

# 显示所有集群成员的简要信息。

<HUAWEI_0.Quidway> display cluster members
The list of cluster member:
------------------------------------------------------------------------------
SN   Device Type              MAC Address     Status Device Name
------------------------------------------------------------------------------
0    NE40E                    0001-0001-0001  Admin  HUAWEI_0.Administrator-1
1    NE40E                    0002-0002-0002  Up     HUAWEI_1.Member-1
2    NE40E                    0003-0003-0003  Up     HUAWEI_2.Member-2
3    NE40E                    0004-0004-0004  Up     HUAWEI_3.Member-3

该显示信息主要内容解释见表1。

表1 display cluster members命令输出信息描述

项目

描述

SN
成员编号。

  • “0”表示命令交换机。

  • 其它数字表示成员交换机。

Device Type

设备的类型。

MAC Address

设备的MAC地址。

Status
设备在集群内的角色及状态。

  • Admin表示命令交换机，且处于UP状态。

  • 其它情况表示成员交换机，状态分为DOWN和UP。

UP：该设备处于正常启动状态

DOWN：该设备的物理线路没有连接好，或它的接口出现物理故障

Device Name

设备的主机名。
