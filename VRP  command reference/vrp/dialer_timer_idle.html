dialer timer idle

命令功能

dialer timer idle命令用来设定当接口的呼叫建立后，允许链路空闲的时间，即没有Interesting报文传送的时间。

undo dialer timer idle命令用来恢复缺省间隔时间。

命令格式

dialer timer idle seconds

undo dialer timer idle

参数说明

seconds：允许链路空闲的时间。取值范围是0～65535，单位是秒。缺省值是120秒。

视图

接口视图、Dialer接口视图

缺省级别

2：配置级

使用指南

在配置该命令前，必须先执行命令dialer user设置对端用户名，或执行命令dialer enable-circular使能轮询DCC。

当一条链路建立后，timer idle定时起作用。若设定的时间内没有Interesting 数据报文在此链路上传送，则DCC自动挂断链路；若timer idle设定为0，则相应的链路在建立后，无论是否有Interesting数据报文在此链路上传送，将永远不被挂断。

使用实例

# 设置接口Serial1/0/0的timer idle为50秒。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] dialer timer idle 50
