http-url

命令功能

http-url命令用来配置NQA测试HTTP测试例的相对路径名和版本信息。

undo http-url命令用来恢复缺省配置。

缺省情况下，没有配置NQA测试HTTP测试例的相对路径名和版本信息。

命令格式

http-url deststring [ verstring ]

undo http-url

参数说明

deststring：网页名，HTTP测试的网页。长度不能超过230个字符。

verstring：HTTP的版本信息。可以配置为v1.0或v1.1，缺省为HTTP1.0版本

视图

NQA视图

缺省级别

2：配置级

使用指南

执行命令http-url deststring [ verstring ]，直接输出网页名即可。如果输入的内容中包含“http：//”、域名信息，则会导致测试例失败。

本测试例支持HTTP1.0版本和HTTP1.1版本。

只能对HTTP类型的测试例配置HTTP操作的URL。

不能更改正在运行的HTTP测试例的URL。

使用实例

# 设置测试例user test的http操作url。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type http
[Quidway-nqa-user-test] http-url www.huawei.com
