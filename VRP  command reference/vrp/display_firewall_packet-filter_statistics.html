display firewall packet-filter statistics

命令功能

display firewall packet-filter statistics命令用来查看防火墙的统计信息。

命令格式

display firewall packet-filter statistics { all | interface interface-type interface-number}

参数说明

all：表示显示所有接口的接口过滤报文统计信息。

interface：表示显示某接口的接口过滤报文统计信息。

interface-type ：接口类型

interface-number：接口编号。

缺省级别

使用指南

通过display firewall packet-filter statistics命令可以查看的防火墙统计信息包括：

  • 防火墙应用的策略

  • 允许通过的报文数

  • 禁止通过的报文数

使用实例

# 显示所有接口的接口过滤报文统计信息。

<Quidway> display firewall packet-filter statistics all
  Interface: Ethernet6/0/0
  In-bound Policy: acl 3000
  From 2006-02-09 10:00:52  to 2006-02-09 10:03:20
     0 packets, 0 bytes, 0% permitted,
     0 packets, 0 bytes, 0% denied,
     0 packets, 0 bytes, 0% permitted default,
     0 packets, 0 bytes, 0% denied default,
  Totally 0 packets, 0 bytes, 0% permitted,
  Totally 0 packets, 0 bytes, 0% denied.
