t1 set clock（CPOS接口视图）

命令功能

t1 set clock命令用来设置T1通道的时钟模式。

undo t1 set clock命令用来恢复缺省设置。

缺省情况下，T1通道的时钟模式为从时钟（slave）。

命令格式

t1 t1-number set clock { master | slave }

undo t1 t1-number set clock

参数说明

t1-number：CPOS的T1通道号。整数形式，取值范围是1～84。

master：设置T1通道的时钟模式为主时钟。

slave：设置T1通道的时钟模式为从时钟。

缺省级别

2：配置级

使用指南

同一CPOS物理端口的不同T1通道的时钟模式是相互独立的。

使用实例

# 设置T1通道1使用主时钟模式。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] t1 1 set clock master
