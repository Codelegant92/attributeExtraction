display atm interface

命令功能

display atm interface命令用来查看ATM配置相关的详细信息。如果配置了子接口，该命令也将显示子接口的详细信息。

命令格式

display atm interface [ atm interface-number ]

参数说明

interface-number：接口编号。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看ATM接口2/0/0的信息。

<Quidway> display atm interface atm 2/0/0

表1 display atm interface命令输出信息描述

项目

描述

ATM interface, State

ATM接口号和状态。

Port Information:

端口信息。

Maximum VCs: 2048

ATM接口上PVC的最大数量为2048。

PVCs: 2, MAPs: 1

主接口上有2条PVC和1条MAP信息。

input pkts: 0, input bytes: 0, input pkt errors: 0

输入报文数、输入字节数和输入报文错误数都是0。

output pkts: 5915, output bytes: 141960, output pkt errors: 0

输出报文数为5915，输出字节数为141960，输出报文错误数为0。

Sub-interface Information:

子接口信息。

ATM interface Atm2/0/0.1, multi-point, State UP

子接口ATM2/0/0.1，类型为点到多点，状态为UP。
