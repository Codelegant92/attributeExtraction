ipv6 maximum load-balancing

命令功能

ipv6 maximum load-balancing命令用来配置在负载分担方式下的IS-IS IPv6等价路由的最大数量。请根据内存容量进行配置。

undo ipv6 maximum load-balancing命令用来恢复负载分担路由的缺省数量。

命令格式

ipv6 maximum load-balancing number

undo ipv6 maximum load-balancing

参数说明

number：指定在负载分担方式下IPv6等价路由的数量。整数形式，取值范围是1～6。缺省值是6。

说明：

不同的产品，不同协议，最大等价路由条数可能会不同，这个最大值可以通过购买许可证文件来调整。

使用指南

使用此命令前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。

使用实例

# 设置IS-IS进行负载分担IPv6路由的最大数为2。

<Quidway> system-view
[Quidway] isis 100
[Quidway-isis-100] ipv6 maximum load-balancing 2
