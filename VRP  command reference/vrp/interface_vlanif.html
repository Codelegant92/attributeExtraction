interface vlanif

命令功能

interface vlanif命令用来在VLAN创建后进入VLANIF接口视图。undo interface vlanif命令用来删除VLANIF接口。

命令格式

interface vlanif vlan-id

undo interface vlanif vlan-id

参数说明

vlan-id：VLAN标识，取值范围是1～4094。

使用指南

VLANIF接口的编号必须对应一个已创建的VLAN。

使用实例

# 进入VLAN2的接口视图。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] quit
[Quidway] interface vlanif 2
