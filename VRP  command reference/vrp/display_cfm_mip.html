display cfm mip

命令功能

display cfm mip命令用来查看MIP的信息，包括：

  • 所在接口名称

  • 级别

命令格式

display cfm mip [ interface interface-type interface-number | level level ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

level level：表示查看指定级别的MIP。level表示MIP的级别，整数形式，取值范围是0～7，值越大级别越高。

interface interface-type interface-number：表示查看指定接口上的MIP的信息。interface-type表示接口类型，取值包括：ethernet、eth-trunk和gigabitethernet。interface-number表示接口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如需要了解当前设备上有哪些MIP、MIP所在接口和MIP的级别时可使用本命令。

既不输入level level参数，也不输入interface interface-type interface-number参数时，表示查看本设备上所有MIP的信息。

使用实例

# 查看所有MIP。

<Quidway> display cfm mip
Interface Name                Level
-----------------------------------
GigabitEthernet2/0/4          0
Eth-Trunk10                   0
GigabitEthernet2/0/9          0
GigabitEthernet2/0/2          0

# 查看级别为7的MIP。

<Quidway> display cfm mip level 7
Interface Name                Level
-----------------------------------
GigabitEthernet2/0/2          7
Eth-Trunk5                    7
GigabitEthernet2/0/1          7

# 查看接口GigabitEthernet1/0/1上的MIP（该接口上创建了MIP）。

<Quidway> display cfm mip interface gigabitethernet1/0/1
Interface Name                Level
-----------------------------------
GigabitEthernet1/0/1          0

# 查看接口GigabitEthernet3/0/1上的MIP（该接口上没创建MIP）。

<Quidway> display cfm mip interface gigabitethernet3/0/1
Interface Name                Level
-----------------------------------
GigabitEthernet3/0/1          --
