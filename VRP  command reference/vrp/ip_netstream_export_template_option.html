ip netstream export template option

命令功能

ip netstream export template option命令用来定义Option模板刷新参数，同时根据所选参数（export-stats 或sampler）使能系统选项统计输出或接口选项统计输出功能。

undo ip netstream export template option命令用来取消Option模板刷新参数的配置。

命令格式

ip netstream export template option { export-stats | refresh-rate packet-number | sampler | timeout-rate interval }

undo ip netstream export template option { export-stats | refresh-rate | sampler | timeout-rate }

参数说明

export-stats ：系统选项统计流，流统计中包括输出流统计的总数和输出流统计报文。

refresh-rate packet-number ：表示每发packet-interval 个流统计报文，进行Option模板的刷新。整数形式，取值范围是1～600，缺省值是20。

sampler ：接口选项统计流，流统计中包括使能NetStream功能的接口索引，出方向还是入方向及对应的采样值。

timeout-rate interval ：表示每隔interval分钟，进行Option模板的刷新。整数形式，单位是分钟，取值范围是1～3600。缺省值是30分钟。

使用指南

聚合模式使用系统模式下的配置。

目前支持的Option输出包括：系统采样参数，接口采样参数，使能NetStream的接口索引及方向，NetStream槽位号，输出给NDA的报文统计。

使用实例

# 设置Option模板刷新参数。

<Quidway> system-view
[Quidway] ip netstream export template option refresh-rate 40
