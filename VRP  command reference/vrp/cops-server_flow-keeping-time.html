cops-server flow-keeping-time

命令功能

cops-server flow-keeping-time命令用于设置COPS服务器的流状态保持时间。

undo cops-server flow-keeping-time命令用于恢复缺省值。

缺省情况下，COPS服务器的流状态保持时间为300秒。

命令格式

cops-server flow-keeping-time time

undo cops-server flow-keeping-time

参数说明

time：COPS服务器的流状态保持时间。整数形式，取值范围是1～65535，单位是秒。

视图

COPS服务器组视图

缺省级别

2：配置级

使用指南

流保持时间是指客户端与COPS服务器连接断开后，连接信息的保持时间。流保持是为了防止由于网络不稳定导致连接闪断的情况。该命令使能后，在保持时间内，当客户端再次与COPS服务器建立连接时，系统能够快速恢复连接信息。

当网络状态存在不稳定因素的情况下，建议配置此选项。

使用实例

# 设置COPS服务器的流状态保持时间为200秒。

<Quidway> system-view
[Quidway] cops-server group huawei
[Quidway-cops-huawei] cops-server flow-keeping-time 200
