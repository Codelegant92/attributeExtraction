port default vlan

命令功能

port default vlan命令用来把二层端口以untagged方式加入到指定的VLAN中。

undo port default vlan命令用来把二层端口从指定VLAN中删除。

命令格式

port default vlan vlan-id

undo port default vlan

参数说明

vlan-id：IEEE802.1Q中定义的VLAN ID，取值范围是1～4094。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

对于Hybrid端口，port default vlan命令同时设置该端口的缺省VLAN。

Trunk类型的端口不能配置缺省VLAN。

使用此命令时，指定的VLAN必须已经存在。并且，如果使用三层端口，必须先执行portswitch（Ethernet接口视图）命令切换为二层端口。

使用实例

# 将GE1/0/0加入到VLAN3中。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port default vlan 3
