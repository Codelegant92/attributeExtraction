qos cql protocol mpls exp

命令功能

qos cql protocol mpls exp命令用来配置基于协议MPLS的分类规则。

undo qos cql protocol mpls exp命令用来删除相应的分类规则。

命令格式

qos cql cql-index protocol mpls exp mpls-experimental-value&<1-8> queue queue-number

undo qos cql cql-index protocol mpls exp mpls-experimental-value&<1-8>

参数说明

cql-index：定制列表的组号，取值范围是1～16。

queue-number：队列号，取值范围是1～16。

mpls-experimental-number：MPLS报文的EXP域，取值范围是0～7。

使用指南

系统是以规则被配置的顺序来匹配数据包，如果发现数据包与某个规则匹配，便结束整个查找过程。

对于同一个cql-index，本命令可以重复使用，为IP数据包建立多种分类规则。

使用实例

# 配置基于协议MPLS的分类规则，MPLS的EXP值为1。

<Quidway> system-view
[Quidway] qos cql 10 protocol mpls exp 1 queue 1
