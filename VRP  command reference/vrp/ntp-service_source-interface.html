ntp-service source-interface

命令功能

ntp-service source-interface命令用来指定本地发送NTP消息的接口。

undo ntp-service source-interface命令用来取消当前配置。

命令格式

ntp-service source-interface interface-type interface-number [ vpn-instance vpn-instance-name ]

undo ntp-service source-interface [ vpn-instance vpn-instance-name ]

参数说明

interface-type interface-number：指定发送NTP消息的接口。

vpn-instance vpn-instance-name：VPN实例名称。字符串形式，不支持空格，长度范围是1～31。

使用指南

对于单播模式，当不希望本地设备上其它接口收到应答消息时，可以指定本地发送的所有NTP消息都使用同一接口的IP地址作为源IP。

说明：

对于广播和组播方式，NTP服务是在特定的接口下进行的，源接口就是该接口。因此，ntp-service source-interface命令对广播方式和组播方式无效。

使用实例

# 指定NTP所有输出消息包的源IP地址都用接口GE1/0/0的IP地址。

[Quidway] ntp-service source-interface GigabitEthernet 1/0/0
