wfq

命令功能

wfq命令用来将帧中继虚电路的队列类型设置为加权公平队列（Weighted Fair Queueing）。

undo wfq命令用来将虚电路的队列类型恢复为FIFO。对同一个帧中继class重复使用本命令，只有最后一次配置将生效。

命令格式

wfq [ discard-threshold [ dynamic-queues ] ]

undo wfq

参数说明

discard-threshold：队列中数据报文的最大个数，超出此个数，数据报文将被丢弃。取值范围是1～1024。缺省值为64。

dynamic-queues：队列的总数。可取值为16、32、64、128、256、512、1024、2048与4096。缺省值为256。

使用指南

加权公平队列的的流量分类规则不允许人工干预，只能是系统默认的缺省流分类规则。

使用实例

# 将加权公平队列应用到名为test1的帧中继class上。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] wfq 128 512
