vlan（MAC Tunnel视图）

命令功能

vlan命令用来配置当前隧道的骨干VLAN和该VLAN的外层协议号。

undo vlan命令用来删除当前隧道的骨干VLAN。

命令格式

vlan vlan-id [ protocol protocol-id ]

undo vlan

参数说明

vlan-id：指定MAC Tunnel的骨干VLAN的ID。整数形式，取值范围是1～4094。

protocol protocol-id：指定骨干VLAN的外层协议号。取值范围是0x0600～0xFFFF，缺省值是0x88a8。

使用指南

说明：

如果要修改骨干VLAN的协议号，需要在隧道出接口下配置qinq protocol命令，否则本端会接收不到对端发送过来的业务流量。

在创建MAC Tunnel时，需要指定隧道的骨干VLAN。骨干VLAN与隧道的远端目的MAC地址B-DA共同唯一标识一条隧道。

要求用于MAC Tunnel的骨干VLAN必须已经被创建，而且加入MAC Tunnel的骨干VLAN资源池。

  • 如果已经配置MAC Tunnel使用的接口，则在配置隧道的骨干VLAN前，该接口必须已经加入该VLAN中。

  • 如果隧道已经被业务实例绑定，则不能修改或删除骨干VLAN。

  • 删除MAC Tunnel的骨干VLAN后，隧道状态将被置为Down。

如果MAC Tunnel已经加入保护组，则该隧道的骨干VLAN不可以再被修改。

使用实例

# 创建当前隧道骨干VLAN ID为100，外层协议号为0x8100。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] vlan 100 protocol 8100
