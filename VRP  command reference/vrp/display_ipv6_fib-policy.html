display ipv6 fib-policy

命令功能

display ipv6 fib-policy命令用来查看IPv6 FIB策略。

命令格式

display ipv6 fib-policy slot slot-number

参数说明

slot slot-number：需要查看IPv6 FIB策略的接口板编号。

所有视图

缺省级别

1：监控级

使用指南

执行命令display ipv6 fib-policy，可以查看接口板配置的策略，如果没有配置策略也会提示用户策略没有配置。

使用实例

# 查看2号接口板IPv6 FIB策略的应用信息。

<Quidway> display ipv6 fib-policy slot 2
  Fib-policy:
    public route
    vpn route: vpna
表1 display ipv6 fib-policy输出信息描述

项目

描述

Fib-policy:
本接口板上应用的IPv6 FIB策略。可能显示的配置信息有：

  • Have not configured FIB policy：接口板上没有配置任何IPv6 FIB策略

  • public route：接口板上配置了IPv6公网策略

  • vpn route：接口板上配置了IPv6 VPN转发表项策略

  • Have configured no-route FIB policy：接口板不接收主控板下发的任何IPv6转发表项

public route

IPv6公网转发表项。

vpn route:

IPv6 VPN转发表项。
