terminal trapping

命令功能

terminal trapping命令用来打开终端告警信息显示功能。

undo terminal trapping命令用来关闭终端显示告警信息功能。

缺省情况下，系统的配置为打开显示功能。

命令格式

terminal trapping

undo terminal trapping

参数说明

无

视图

用户视图

缺省级别

1：监控级

使用指南

使用terminal trapping命令打开终端告警信息显示功能。

使用实例

# 关闭告警信息显示功能。

<Quidway> undo terminal trapping
Info:Current terminal trapping is off
