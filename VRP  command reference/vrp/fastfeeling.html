fastfeeling

命令功能

fastfeeling命令用来使能接口的快速感知特性功能。

undo fastfeeling命令用来禁止此功能。

缺省情况下，禁止接口的快速感知功能。

命令格式

fastfeeling

undo fastfeeling

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

快速感知特性用于将物理故障迅速通告给应用程序，满足实时性要求。

使用实例

# 在接口POS1/0/0下配置快速感知功能。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] fastfeeling
