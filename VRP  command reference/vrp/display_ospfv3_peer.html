display ospfv3 peer

命令功能

display ospfv3 peer命令用来查看符合指定条件的OSPFv3邻居的信息。

命令格式

display ospfv3 [ process-id ] [ area area-id ] peer [ interface-type interface-number | neighbor-id ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

area area-id：指定特定区域的邻居，可以是十进制整数（取值范围是0～4294967295）或IPv4地址格式。

interface-type interface-number：接口类型和接口号。

neighbor-id：指定邻居的Router-ID号。

verbose：当使用该参数时，显示各区域邻居的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看Pos1/0/0接口上的OSPFv3进程1的邻居信息。

<Quidway> display ospfv3 1 peer pos 1/0/0
OSPFv3 Process (1)
OSPFv3 Area (0.0.0.0)
Neighbor ID     Pri   State           Dead Time   Interface  Instance ID
1.1.1.1           1   Full/ -         00:00:30    Pos1/0/0    0

# 查看Pos1/0/0接口上的OSPFv3进程100的邻居详细信息。

<Quidway> display ospfv3 100 peer pos 1/0/0 verbose
OSPFv3 Process (100)
 Neighbor 1.1.1.1, interface address FE80::3D43:0:8C14:1
    In the area 0.0.0.1 via interface Pos1/0/0
    DR is 0.0.0.0 BDR is 0.0.0.0
    Options is 0x000013 (-|R|-|-|E|V6)
    Dead timer due in 00:00:29
    Database Summary List 0
    Link State Request List 0
    Link State Retransmission List 0

项目

描述

Neighbor ID

邻居路由器ID

Pri

邻居路由器的优先级

State

邻居的状态（Down/Attempt/Init/2 Way/Exch Start/Exchange/Loading/Full）

Dead Time

邻居的死亡时间

Interface

与邻居相连的接口

Instance ID

邻居所属的实例ID

interface address

与邻居相连的接口的IP地址

In the area 0.0.0.1 via interface Pos1/0/0

邻居的接口POS1/0/0在区域1内

DR

DR路由器

BDR

BDR路由器

Options

邻居的Option

Dead timer due in 00:00:29

邻居距离死亡的时间

Database Summary List

邻居Database summary列表中的报文个数

Link State Request List

邻居请求列表中LSA的个数

Link State Retransmission List

邻居重传列表中LSA的个数
