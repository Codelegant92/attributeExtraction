graceful-restart（MPLS-LDP视图）

命令功能

graceful-restart命令用来使能LDP GR功能。

undo graceful-restart命令用来恢复为缺省配置。

缺省情况下，没有使能LDP GR功能。

命令格式

graceful-restart

undo graceful-restart

参数说明

无

视图

MPLS-LDP视图

缺省级别

2：配置级

使用指南

对于双主控的设备，使能LDP GR特性能够在设备出现故障时保证正常的转发，并且能够保存重启前的MPLS转发状态。

说明：

使能或禁止Graceful Restart功能都会导致所有LDP实例的会话重建。

使用实例

# 使能LDP GR功能。

<Quidway> system-view
[Quidway] mpls ldp
[Quidway-mpls-ldp] graceful-restart
Warning: All the related sessions will be deleted if the operation is performed
!Continue? (y/n)y
[Quidway-mpls-ldp] quit
