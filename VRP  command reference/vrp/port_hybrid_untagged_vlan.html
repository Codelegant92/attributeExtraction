port hybrid untagged vlan

命令功能

port hybrid untagged vlan命令用来将Hybrid端口加入到指定的已经存在的VLAN。

undo port hybrid untagged vlan命令用来将Hybrid端口从指定的VLAN中删除。

命令格式

port hybrid untagged vlan { { vlan-id1 [ to vlan-id2 ] } &<1-10> | all }

undo port hybrid untagged vlan { { vlan-id1 [ to vlan-id2 ] } &<1-10> | all }

参数说明

untagged：所指定VLAN的报文不带标签。

vlan-id1：Hybrid端口要加入的VLAN的起始值，取值范围是1～4094。

vlan-id2：Hybrid端口要加入的VLAN的结束值，取值范围是1～4094。该值必须大于vlan-id1。

all：所有VLAN。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

Hybrid端口可以属于多个VLAN。如果多次使用port hybrid untagged vlan 命令，则Hybrid端口上允许通过的VLAN是这些VLAN的集合。

如果使用三层端口，必须先执行portswitch（Ethernet接口视图）命令切换为二层端口。

使用实例

# 将Hybrid端口GE1/0/0加入到2、4、50～100 VLAN中，并且这些VLAN的报文将不带标签。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port link-type bybrid
[Quidway-GigabitEthernet1/0/0] port hybrid untagged vlan 2 4 50 to 100
