display aspf all

命令功能

display aspf all命令用来查看所有的ASPF策略和会话信息。

命令格式

display aspf all

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display aspf all命令的输出信息包括：

  • 各个具体应用层协议的阻塞情况

  • 接口上出、入方向配置的aspf策略情况

使用实例

# 下面是display aspf all命令的输出信息。
