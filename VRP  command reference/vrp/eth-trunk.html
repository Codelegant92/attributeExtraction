eth-trunk

命令功能

eth-trunk命令用来将当前以太网接口加入Eth-Trunk组。

undo eth-trunk命令用来将当前以太网接口从Eth-Trunk组中删除。

命令格式

eth-trunk trunk-id

undo eth-trunk

参数说明

trunk-id：指定Eth-Trunk接口号。取值范围是0～63。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

Eth-Trunk只能由以太网链路构成。

使用实例

# 将接口GE1/0/0加入Eth-Trunk组1。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] eth-trunk 1
