display mpls ldp peer

命令功能

display mpls ldp peer命令用来查看LDP对等体的信息。

命令格式

display mpls ldp peer [ vpn-instance vpn-instance-name ] [ verbose | peer-id ] [ | { begin | exclude | include } regular-expression ]

display mpls ldp peer [ all ] [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：显示指定VPN实例的LDP对等体信息。

verbose：显示LDP对等体的详细信息。

peer-id：显示指定LDP对等体的信息。

all：显示所有LDP对等体的信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Peer-ID

LDP标识符，格式为<LSR ID>：<标签空间>，标签空间的取值为：

  • 0：表示全局标签空间

  • 非0：表示接口标签空间

TransportAddress

LDP对等体的传输地址，用于建立TCP连接

DiscoverySource
发现LDP对等体的源端：

  • 显示接口时表示：发现本地LDP对等体的源端为本地接口

  • 显示远端对等体的配置名称时表示：发现远端LDP对等体的源端为远端对等体

项目

描述

Peer LDP ID

LDP标识符，格式为<LSR ID>：<标签空间>，标签空间的取值为：

  • 0：表示全局标签空间

  • 非0：表示接口标签空间

Peer Max PDU Length

LDP对等体的最大PDU长度

Peer Transport Address

LDP对等体的传输地址，用于建立TCP连接

Peer Loop Detection

LDP对等体的环路检测功能：

  • On：开启环路检测功能

  • Off：关闭环路检测功能

Peer Path Vector Limit

LDP对等体的最大路径向量限制值

Peer FT Flag

LDP对等体的GR FT标志（只有收到初始化消息，且初始化消息中含有FT TLV时，才能对该标志置位）：

  • On：开启LDP GR功能

  • Off：关闭LDP GR功能

Peer Keepalive Timer

LDP对等体配置的keepalive定时器的值

Recovery Timer

LDP对等体的恢复时间

只有开启了LDP GR功能，即“Peer FT Flag”的值为“On”，该字段的值才不为空

Reconnect Timer

LDP对等体的重连接时间

只有开启了LDP GR功能，即“Peer FT Flag”的值为“On”，该字段的值才不为空

Peer Type

LDP对等体的类型：

  • Local：本地LDP对等体

  • Remote：远端LDP对等体

  • Local&Remote：本地远端共存LDP对等体

Peer Label Advertisement Mode

对等体标签发布方式，VRP采用的标签分发方式为DU

Peer Discovery Source
LDP对等体的发现源：

  • 显示接口时表示：发现本地LDP对等体的源端为本地接口

  • 显示远端对等体的配置名称时表示：发现远端LDP对等体的源端为远端对等体
