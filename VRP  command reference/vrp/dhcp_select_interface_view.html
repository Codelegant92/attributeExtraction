dhcp select（接口视图）

命令功能

dhcp select命令用来设置接口对目的地址是本机的DHCP报文的处理模式。

undo dhcp select命令用来恢复缺省设置。

缺省情况下，目的地址是本机的DHCP报文将被送到内部服务器，从全局地址池中分配地址（global方式）。

命令格式

dhcp select { global | interface | relay }

undo dhcp select

参数说明

global：将DHCP报文发送到本地的DHCP服务器，由本地DHCP服务器从全局地址池分配地址。

interface：将DHCP报文发送到本地的DHCP服务器，由本地DHCP服务器从接口地址池分配地址。

relay：将DHCP报文通过中继发送到外部的DHCP服务器，由外部DHCP服务器分配地址。

缺省级别

使用指南

dhcp select命令用来设置接口对目的地址是本机的DHCP报文的处理模式，DHCP服务器和客户端都在一个子网内，直接进行DHCP协议的交互，如不在一个子网内，使用中继功能从外部的DHCP服务器分配地址，如网络的规模较小，可以通过基于接口的接口地址池来分配地址。

使用实例

# 对于目的地址是本机的DHCP报文，从内部DHCP服务器的接口地址池中分配地址。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] dhcp select interface
