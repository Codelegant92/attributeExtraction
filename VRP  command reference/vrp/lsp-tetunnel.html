lsp-tetunnel

命令功能

lsp-tetunnel命令用来配置NQA测试LSP测试例的TE隧道。

undo lsp-tetunnel命令用来取消LSP测试例的TE隧道。

命令格式

lsp-tetunnel tunnel tunnel-number

undo lsp-tetunnel

参数说明

tunnel-number：TE Tunnel的接口编号。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对LSP类型的测试例配置LSP TE隧道。

当测试类型为LSP且LSP类型为TE时，才可以设置TE隧道。

不能更改正在运行的测试例的LSP TE隧道。

使用实例

# 设置测试例user test的TE隧道。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type lspping
[Quidway-nqa-user-test] lsp-type te
[Quidway-nqa-user-test] lsp-tetunnel tunnel 1/0/1
