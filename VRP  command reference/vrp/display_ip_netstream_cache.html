display ip netstream cache

命令功能

display netstream ip cache命令用来显示指定NetStream板的NetStream缓冲区信息。

命令格式

display ip netstream cache { { all-slot | slot slot-id } [ flow-type { ip | ipv6 | mpls ip | mpls ipv6 | mpls unknown | multicast-vpn } ] [ interface interface-type interface-number ] [ bgp-nexthop bgp-nexthop ] } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all-slot：所有的接口板

slot slot-id：接口板槽号。使用此参数指定当前活跃的NetStream板。

flow-type：显示缓冲中指定类型的流统计信息。

ip ：IPv4流量信息。

ipv6 ：IPv6流量信息。

mpls-ip ：MPLS和IPv4流量信息。

mpls-ipv6:MPLS和IPv6流量信息。

mpls-unknown：未知的MPLS流量信息。

multicast-vpn：组播VPN流量信息。

interface interface-type：指定VLAN下的二层物理接口类型。

interface-number：指定VLAN下的二层物理接口号。

bgp-nexthopbgp-nexthop：bgp下一跳地址。点分十进制形式。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display ip netstream cache命令可结合NetStream流采样使能命令使用。在配置了NetStream流采样使能命令后，可以通过display ip netstream cache命令来显示指定NetStream板上指定流类型的流信息。
