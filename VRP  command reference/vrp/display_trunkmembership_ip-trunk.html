display trunkmembership ip-trunk

命令功能

display trunkmembership ip-trunk命令用来查看IP-Trunk接口的成员端口信息。

命令格式

display trunkmembership ip-trunk trunk-id [ | count ] [ | { begin | include | exclude } regular-expression ]|

参数说明

trunk-id：Trunk ID。整数形式，取值范围是0～63。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。字符串格式，支持空格，区分大小写。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看IP-Trunk成员。

<Quidway> display trunkmembership ip-trunk 3 
Trunk ID: 3
TYPE: pos
Number Of Ports in Trunk = 1
Number Of UP Ports in Trunk = 1
operate status: up
Interface Pos1/0/0, valid, selected, operate up, weight=1,
standby interface NULL

项目

描述

Trunk ID:

查看的IP-Trunk接口号。

TYPE:

IP-Trunk的类型。只能是POS。

Number Of Ports in Trunk ＝

加入IP-Trunk接口的端口总数。

Number Of UP Ports in Trunk =

加入IP-Trunk接口的端口中处于UP状态的端口总数。

operate status:

执行状态。

Interface Pos1/0/0, valid, selected, operate up, weight=1,

成员端口的状态和权重等。

standby interface NULL

没有备份端口。
