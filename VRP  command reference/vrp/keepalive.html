keepalive

命令功能

keepalive命令用来使能GRE隧道的keepalive功能。

undo keepalive命令用来取消GRE隧道的keepalive功能。

缺省情况下，没有使能GRE隧道的keepalive功能。

命令格式

keepalive [ period period ] [ retry-times retry-times ]

undo keepalive

参数说明

period：发送keepalive报文的周期。整数形式，取值范围是1～32767，单位是秒。缺省值是5秒。

retry-times：不可达计数器参数。整数形式，取值范围是1～255。缺省值是3。

Tunnel接口视图

缺省级别

2：配置级

使用指南

执行keepalive命令后，该GRE隧道的keepalive功能立即生效。执行undo keepalive后，该功能立即失效。

使能keepalive功能后，GRE隧道源端周期性的发送keepalive报文。每发送一个，不可达计数器加1。如果此计数器的值到达预先设置的参数值retry-times还没收到回送报文，就认为对端不可达。

使用实例

# 使能GRE隧道的keepalive功能，其参数使用缺省值。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] keepalive

# 使能GRE隧道的keepalive功能并指定发送周期为12，不可达计数器参数为缺省值。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] keepalive period 12

# 使能GRE隧道的keepalive功能并指定发送周期为12秒，retry-times为4。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] keepalive period 12 retry-times 4
