remote address

命令功能

remote address命令用来配置为对端用户分配IP地址的方式。

undo remote address命令用来取消当前的设置。

命令格式

remote address { ip-address | pool [ pool-number ] }

undo remote address

参数说明

ip-address：指定为对端分配的IP地址。点分十进制格式。

pool [ pool-number ]：指定为对端分配地址时使用的地址池。整数形式，取值范围是：0～99。这里的地址池是在AAA视图下配置的。如果不指定地址池编号，则缺省使用地址池0。

使用指南

在服务器端可以使用remote address命令指定一个地址池（也可以指定一个IP地址），为对端用户分配IP地址。

只有封装PPP协议的接口可以使用该命令借用其他接口的IP地址。

对端用户如果需要通过协商获得IP地址，应先使用ip address ppp-negotiate命令使能接口的IP地址协商功能。

说明：

对端用户和服务器必须是直连(包括物理直连或二层逻辑直连)的。

使用实例

# 配置接口POS1/0/0为对端分配IP地址129.102.0.1。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] remote address 129.102.0.1
