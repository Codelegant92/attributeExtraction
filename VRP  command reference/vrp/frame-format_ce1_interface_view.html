frame-format（CE1接口视图）

命令功能

frame-format命令用来设置CE1接口的帧格式。

undo frame-format命令用来恢复缺省设置。

缺省情况下，CE1接口的帧格式为no-CRC4。

命令格式

frame-format { crc4 | no-crc4 }

undo frame-format

参数说明

crc4：帧格式为CRC4。

no-crc4：帧格式为no-CRC4。

视图

CE1接口视图

缺省级别

2：配置级

使用指南

只有当CE1接口工作在CE1工作模式下时才能进行帧格式的配置。

使用实例

# 设置CE1接口1/0/0的帧格式为CRC4。

<Quidway> system-view
[Quidway] controller e1 1/0/0
[Quidway-E1 1/0/0] frame-format crc4
