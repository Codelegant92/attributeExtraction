start-script

命令功能

start-script命令用来执行指定的Modem脚本。

命令格式

start-script script-name number

参数说明

script-name：脚本集中的脚本名称，字符串形式，长度为1～50。

number：运行脚本的TTY接口编号。取值范围是0～38。不同设备上该参数的取值范围可能不同。

视图

用户视图

缺省级别

2：配置级

使用指南

这条命令提供了Modem脚本立即执行的方法，如果相应的接口上正在执行一个脚本，则本次命令不会执行，并且会报告一条错误消息。

使用实例

# 在用户界面TTY 1上执行指定Modem脚本“example”。

<Quidway> start-script example 1
