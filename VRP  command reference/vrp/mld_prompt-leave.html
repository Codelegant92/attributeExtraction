mld prompt-leave

命令功能

mld prompt-leave命令用来设置接口快速离开组。当接口接收到某组播组的离开消息时，不发送最后侦听者查询消息，立即删除该组记录。

undo mld prompt-leave命令用来恢复配置的缺省情况。

缺省情况下，当路由器接收到组播组的离开消息后，发送最后侦听者查询消息。

命令格式

mld prompt-leave [ group-policy basic-acl6-number ]

undo mld prompt-leave

参数说明

group-policy：指定组成员快速离开的组策略。

basic-acl6-number：指定IPv6基本访问控制列表号。整数形式，取值范围是2000～2999。

使用指南

该命令仅适用于MLDv1版本，不支持MLDv2。即，接口收到MLDv1的离开消息时，快速离开功能才会生效。

如果没有指定组策略，则快速离开功能应用于所有的组播组。

mld prompt-leave命令与acl命令配合使用。在ACL视图下，通过rule命令的source参数定义主机将要离开的组播组地址。

使用实例

# 使能接口GE1/0/0的快速离开功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld prompt-leave
