pvp

命令功能

pvp命令用来创建PVP，并进入PVP视图。

undo pvp命令用来删除指定的PVP。

缺省情况下，不创建任何PVP。

命令格式

pvp vpi

undo pvp vpi

参数说明

vpi：ATM网络虚路径标识VPI（Virtual Path Identifier，）。整数形式，在ATM接口上取值范围是0～255。

视图

ATM子接口视图

缺省级别

2：配置级

使用指南

本命令主要在ATM透传时使用。

创建PVP时应保证VPI的值在子接口范围内唯一。

使用实例

# 创建并进入PVP。

<Quidway> system-view
[Quidway] interface atm 1/0/0.1
[Quidway-Atm1/0/0.1] pvp 20
