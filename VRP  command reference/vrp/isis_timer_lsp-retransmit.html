isis timer lsp-retransmit

命令功能

isis timer lsp-retransmit命令用来设置点到点链路上LSP报文的重传间隔时间。

undo isis timer lsp-retransmit命令用来恢复缺省设置。

缺省情况下，点到点链路上LSP报文的重传间隔时间为5秒。

命令格式

isis timer lsp-retransmit retransmit-interval

undo isis timer lsp-retransmit

参数说明

retransmit-interval：表示LSP报文的重传间隔时间。整数形式，取值范围是1～300秒。

使用指南

只有在点到点链路上，发送的LSP报文才需要得到对端的应答，因此广播网不支持该命令。

说明：

对此参数的设置应该谨慎一些，以避免产生不必要的重传。

说明：

如果通过isis circuit-type命令将广播接口模拟为P2P接口或者通过undo isis circuit-type命令将该接口恢复为广播接口，则点链路上LSP报文的重传间隔时间恢复为缺省值。

使用实例

# 设置在接口POS1/0/0上LSP报文的重传间隔为10秒。

<Quidway> system-view
[Quidway] interface pos1/0/0
[Quidway-Pos1/0/0] isis timer lsp-retransmit 10
