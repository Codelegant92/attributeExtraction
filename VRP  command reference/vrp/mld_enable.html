mld enable

命令功能

mld enable命令用来在接口上使能MLD。

undo mld enable命令用来关闭接口上的MLD功能。

缺省情况下，接口上MLD功能是关闭的。

命令格式

mld enable

undo mld enable

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

使用此命令时，必须先使能IPv6组播路由。

使用实例

# 使能接口GE1/0/0的MLD功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld enable
