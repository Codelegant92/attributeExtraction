nqa-ftp-record vpn-instance

命令功能

nqa-ftp-record vpn-instance命令用来配置NQA测试结果通过VPN发送的实例名称。

undo nqa-ftp-record vpn-instance命令用来取消配置。

命令格式

nqa-ftp-record { vpn-instance vpn-instance }

undo nqa-ftp-record vpn-instance

参数说明

vpn-instance：设置FTP服务器所属的VPN实例。VPN实例名为字符串形式，长度范围是1～31。

视图

系统视图

缺省级别

3：管理级

使用指南

在FTP方式进行传送过程中，需要保证FTP连接的连通性，当FTP连接失败，而结果缓存已满，将挤掉最前面缓存中的结果。

使用实例

# 配置VPN实例名。

<Quidway> system-view
[Quidway] nqa-ftp-record vpn-instance v1
