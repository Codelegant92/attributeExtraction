local-user level

命令功能

local-user level命令用来设置本地用户的用户优先级。

undo local-user level命令用来取消设置。

缺省情况下，本地用户的优先级由管理模块来决定，比如Telnet、SSH。

命令格式

local-user user-name level level

undo local-user user-name level

参数说明

user-name：用户名。字符串形式，不支持通配符，长度范围是1～64。

level：用户优先级别。取值范围是0～15。

视图

AAA视图

缺省级别

3：管理级

使用指南

系统提供对命令的分级管理，即对所有命令设定一定的操作级别，只有用户的优先级等级等于或者大于命令的级别，用户才能使用该命令。

使用实例

# 设置本地用户hello@163.net的优先级为3。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] local-user hello@163.net level 3
