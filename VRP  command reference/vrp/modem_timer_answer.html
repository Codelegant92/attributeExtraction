modem timer answer

命令功能

modem timer answer命令用来设置呼入连接建立时，从提机到检测到载波的等待超时时间。

undo modem timer answer命令用来恢复缺省的等待超时时间。

缺省情况下，等待超时时间为30秒。

命令格式

modem timer answer seconds

undo modem timer answer

参数说明

seconds：超时时间。整数形式，取值范围是1～60，单位是秒。

视图

用户界面视图

缺省级别

3：管理级

使用指南

该命令只对AUX口及其它异步类型接口有效（CON口除外）。

使用实例

# 设置Modem超时等待时间为45秒。

<Quidway> system
[Quidway] user-interface vty 0
[Quidway-ui-vty0] modem timer answer 45
