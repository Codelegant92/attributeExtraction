snmp-agent notification-log enable

命令功能

snmp-agent notification-log enable命令用于使能告警日志功能。

undo snmp-agent notification-log命令用于去使能告警日志功能。

缺省情况下，去使能告警日志功能。

命令格式

snmp-agent notification-log enable

undo snmp-agent notification-log enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

只有Inform方式的告警需要记录日志缓冲区，Trap方式的告警无需记录。

满足以下两种条件之一，Inform方式的告警才记录日志：

  • 待确认的告警队列中的某告警信息重传指定次数后，仍然未收到告警确认信息。

  • 当前告警信息记录数目达到待确认告警队列能够存储的最大告警信息个数时，告警信息被丢弃。

使用实例

# 使能告警日志功能。

<Quidway> system-view
[Quidway] snmp-agent notification-log enable
