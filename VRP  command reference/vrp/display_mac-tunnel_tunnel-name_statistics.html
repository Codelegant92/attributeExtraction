display mac-tunnel tunnel-name statistics

命令功能

display mac-tunnel命令用来查看从启动流量统计开始到当前时刻的隧道流量信息：收发包数、收发字节数。

命令格式

display mac-tunnel tunnel-name tunnel-name statistics

参数说明

tunnel-name：指定需要查看的MAC Tunnel的名字。

所有视图

缺省级别

1：监控级

使用指南

该命令可以查看隧道两个方向的流量统计信息。缺省情况下，MAC Tunnel使能流量统计功能，因此在查看隧道流量统计信息之前，请使用reset mac-tunnel tunnel-name statistic命令将当前隧道的流量统计信息清零，过一段时间后再查看流量统计信息。

使用实例

# 查看隧道名称为mytunnel的MAC Tunnel流量统计信息。

<Quidway> display mac-tunnel tunnel-name mytunnel statistics
  Statistics for mac-tunnel : mytunnel
       Inbound-Packets  : 0
       Inbound-Octets   : 0
       Outbound-Packets : 0
       Outbound-Octets  : 0
