项目

描述

Feature name

模块名称。

Trap number

告警事件的类型数。

Trap name
事件类型，有以下类型：
  • CACGLBTOTAL：在配置CAC总体限制后，实例的PIM表项数量超出限制值。
  • CACGLBCHN：在配置基于节目组的CAC限制后，节目组的表项数量超出限制值。
  • CACOIFTOTAL：在出接口上配置CAC总体限制后，以该接口为出接口的PIM表项数量或带宽超出限制值。
  • CACOIFCHN：在出接口上配置基于节目组的CAC限制后，以该接口为出接口的PIM表项数量或带宽超出限制值。
  • CACGLBTOTALEXCEED：在配置CAC总体限制时，实例已有的PIM表项数量超出限制值。
  • CACGLBCHNEXCEED：在配置基于节目组的CAC限制时，节目组已有的PIM表项数量超出限制值。
  • CACOIFTOTALEXCEED：在出接口上配置CAC总体限制时，已有的以该接口为出接口的PIM表项数量或带宽超出限制值。
  • CACOIFCHNEXCEED：在出接口上配置基于节目组的CAC限制时，已有的以该接口为出接口的PIM表项数量或带宽超出限制值。

Status
告警开关状态，有以下两种：
  • on：告警开关打开。
  • off：告警开关关闭。

Excessive Flag
海量信息标识，有以下两种取值：
  • Y：表示当前类型的信息是海量信息。
  • N：表示当前类型的信息不是海量信息。
