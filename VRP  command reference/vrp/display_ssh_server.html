display ssh server

命令功能

display ssh server status命令用来显示SSH服务器的全局配置信息。

display ssh server session命令用来显示SSH服务器端的当前会话连接信息。

命令格式

display ssh server { status | session } [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

SSH相关属性配置完毕后，可运行此命令查看会话连接信息，以确认SSH连接建立成功。

使用实例

# 显示SSH服务器的全局配置信息。

 <Quidway> display ssh server status
 SSH version                        : 1.99
 SSH connection timeout             : 60 seconds
 SSH server key generating interval : 2 hours
 SSH Authentication retries         : 5 times
 SFTP server                        : Enable
 STelnet server                     : Enable
SSH server port                     : 55535

# SSH服务器端的当前会话连接信息。

<Quidway> system-view
[Quidway] display ssh server session
  Session 1:
       Conn                 : VTY 0
       Version              : 2.0
       State                : started
       Username             : client001
       Retry                : 1
       CTOS Cipher          : aes128-cbc

项目

描述

Conn

SSH会话占用的虚拟用户终端接口

Version

SSH会话连接使用的协议版本

State

SSH会话的连接状态

User-name

SSH会话连接使用的用户名

Retry

建立SSH会话连接的重试次数

CTOS Cipher

客户端到服务器的加密算法名

STOC Cipher

服务器到客户端的加密算法名

CTOS Hmac

客户端到服务器的Hmac算法名

STOC Hmac

服务器到客户端的Hmac算法名

Kex

交换算法名

Service Type

SSH用户的服务方式，包括sftp和stelnet

Authentication Type

SSH用户的认证方式，包括password、RSA、password-rsa、all
