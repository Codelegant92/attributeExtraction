display traffic-statistics vsi peer negotiation-vc-id

命令功能

display traffic-statistics vsi peer negotiation-vc-id命令用来显示指定VSI-Name中指定Martini方式VPLS PW的公网流量统计信息。

命令格式

display traffic-statistics vsi vsi-name [ peer peer-address [ negotiation-vc-id vc-id ] ]

参数说明

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

peer peer-addres：PW的对端IP地址。

negotiation-vc-id vc-id：PW的ID。

视图

所有视图

缺省级别

1：监控级

使用指南

在所有视图下显示指定Martini方式VPLS PW流量统计信息。Martini方式VPLS PW由用户指定的VSI-Name与指定的对端IP地址和PW ID唯一确定。

5分钟以内，如果出现了PW down的情况，PW down之前统计的流量不被用来计算5分钟的流量速率。

使用实例

# 在系统视图下显示指定Martini方式VPLS PW的出流量统计信息。

<Quidway> display traffic-statistics vsi-name newvsi peer 100.22.33.20 negotiation-vc-id 2
