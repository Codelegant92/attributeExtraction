peer reflect-client (BGP)

命令功能

peer reflect-client命令用来配置将本机作为路由反射器，并将对等体（组）作为路由反射器的客户。

undo peer reflect-client命令用来取消已有的配置。

缺省情况下，没有配置路由反射器及其客户。

命令格式

peer { group-name | ipv4-address | ipv6-address } reflect-client

undo peer { group-name | ipv4-address | ipv6-address } reflect-client

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图、BGP-IPv4单播地址族视图、BGP-IPv6单播地址族视图、BGP-VPNv4地址族视图、BGP-VPNv6地址族视图BGP-L2VPN地址族视图、BGP-VPLS地址族视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP-IPv6单播地址族视图下生效。

缺省级别

2：配置级

使用指南

该命令仅适用于IBGP对等体（组）之间。

在某个地址族下配置的reflect-client信息只能在该地址族有效，不能被其它地址族继承。因此建议用户在需要的特定的地址族下配置reflect-client信息。

使用实例

# 将对等体作为路由反射器的客户。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 100
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] peer 1.1.1.2 reflect-client

# 将对等体组test作为路由反射器的客户。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv6-family
[Quidway-bgp-af-ipv6] peer test reflect-client
