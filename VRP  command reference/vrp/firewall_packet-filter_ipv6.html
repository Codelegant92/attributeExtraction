firewall packet-filter ipv6

命令功能

firewall packet-filter ipv6命令用来配置接口的出或入方向上的IPv6报文过滤，并指定所使用的ACL6编号。如果报文满足匹配规则，则报文通过过滤，否则将被丢弃。

undo firewall packet-filter ipv6命令用来取消接口的出或入方向的IPv6报文过滤。

缺省匹配规则为“permit”，即如果未设置匹配规则，则允许报文通过。

命令格式

firewall packet-filter ipv6 acl6-number { inbound | outbound }

undo firewall packet-filter ipv6 { inbound | outbound }

参数说明

acl6-number：指定ACL6编号，整数形式，取值范围是1000～3999。其中2000～2999是基本ACL6，3000～3999是高级ACL6，1000～1999是基于接口的ACL6。

inbound：表示在当前接口的入方向上进行报文过滤。

outbound：表示在当前接口的出方向上进行报文过滤。

使用实例

# 配置接口GE 1/0/0 的IPv6报文过滤。设置过滤此接口输出报文的ACL6编号为2050。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] firewall packet-filter ipv6 2050 outbound
