display mpls l2vpn forwarding-info

命令功能

display mpls l2vpn forwarding-info命令用来查看MPLS L2VPN的转发信息。

命令格式

display mpls l2vpn forwarding-info [ vc-label ] interface interface-type interface-number [ | { begin | exclude | include } regular-expression ]

参数说明

vc-label：L2VPN的VC标签。十进制整数形式，取值范围是16～1048575。

interface interface-type interface-number：接口类型和接口编号。

|：使用正则表达式对输出信息进行匹配。

begin：从与正则表达式匹配的行开始输出。

exclude：只输出不包含正则表达式指定内容的信息。

include：只输出包含正则表达式指定内容的信息。

regular-expression：正则表达式，用于匹配输出信息的指定字符串。

使用指南

display mpls l2vpn forwarding-info命令可以用来查看携带指定标签的私网数据将从哪个隧道进行转发，是否使能连接性检测，该数据是否携带控制字。

使用实例

# 查看MPLS L2VPN转发信息。

<Quidway> display mpls l2vpn forwarding-info interface pos 3/0/0

项目

描述

The Main PW Forward Information :

主用PW的转发信息。

VCLABEL

该指定接口绑定的VC的标签。

TUNNELTYPE

该VC使用的公网隧道的类型，可以为LSP、GRE、CR-LSP等。

ENTRYTYPE
ENTRYTYPE表示连接的类型，它有以下值：

  • SEND：除CCC之外的其他L2VPN类型的转发表项。

  • CCC_SEND：CCC远端连接。

  • LOCAL：本地连接转发表项。

  • INVALID：无效的转发表项。

PWSTATE

用于指导设备转发的PW状态，分为Active和Inactive两种。主备PW该状态互斥，同一时刻只能有一个是Active状态。

BFDSTATE
用于指导设备转发的BFD状态。

  • 如果设备没有配置BFD，此标志位默认显示为Up。

  • 如果设备配置了BFD，且BFD会话状态是Down（不是Admin Down），此时此标志位显示为Down，转发不通。

ADMIN
是否配置了手工设置主用或者备用PW故障。

  • Up：没有手工设置主用或者备用PW故障

  • Down：手工设置主用或者备用PW故障

CTLWORD

表明本端是否使能控制字。FALSE表示未使能，TRUE表示支使能。

CC

VCCV的connection channel，连接通道，有两种方式，Control-Word和Label-alert。

CV

Connectivity Verification，连接性检测。

TUNNELID

该VC使用的公网隧道的标志号，十六进制整数。

Record(s) Found

本接口上的VC转发信息表项数目。

The Second PW Forward Information :

备用PW的转发信息。
