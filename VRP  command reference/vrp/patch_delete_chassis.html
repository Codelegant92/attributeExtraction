patch delete（多框）

命令功能

patch delete命令按照设备方式删除补丁。

命令格式

patch delete [ all | chassis chassis-number | slave | slot slot-id ]

参数说明

all：所有单板（包括主控板和接口板）。

chassis chassis-number：转发框的框号。

slave：系统备用板。

slot slot-id：板号（输入的形式是框类型＋框号＋板号）。

视图

用户视图

缺省级别

2：配置级

使用指南

指定all，则删除多框集群所有板的补丁；指定slave，则删除多框集群的系统备板的补丁；不指定任何的参数，则删除多框集群的系统主用板补丁。

使用实例

# 删除所有接口板上的补丁。

<Quidway>patch delete
