display pppoe-server session

命令功能

display pppoe-server session命令用来查看PPPoE会话状态或统计数据。

命令格式

display pppoe-server session { { all | packet }[ | count ] [ | { begin | include | exclude } regular-expression ] | statistic interface interface-type interface-number }

参数说明

all：查看所有PPPoE会话信息。

packet：查看PPPoE会话报文统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

statistic：按接口查看PPPoE会话统计信息。

interface-type interface-number：接口类型及接口编号。只能是以太类的接口，如Ethernet、GigabitEthernet。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。
