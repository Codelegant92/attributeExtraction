peer as-number

命令功能

peer as-number命令用来配置指定对等体（组）的对端AS号。

undo peer as-number命令用来删除对等体组的AS号。

命令格式

peer { group-name | ipv4-address | ipv6-address } as-number as-number

undo peer group-name as-number

undo peer { ipv4-address | ipv6-address }

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP视图和BGP-VPN6实例视图下生效。

as-number：对等体/对等体组的对端AS号。

视图

BGP视图、BGP-VPN实例视图、BGP-VPN6实例视图

缺省级别

2：配置级

使用指南

无

使用实例

# 配置IPv4对等体1.1.1.1的对端AS号为100。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.1 as-number 100

# 配置IPv6对等体1::1的对端AS号为100。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1::1 as-number 100

# 配置指定对等体组test的对端AS号为100。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer test as-number 100
