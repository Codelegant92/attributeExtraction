import-route isis level-2 into level-1

命令功能

import-route isis level-2 into level-1命令用来控制IS-IS从Level-2向Level-1进行路由渗透，该命令需配置在与外部区域相连的Level-1-2路由器上。

undo import-route isis level-2 into level-1命令用来禁止IS-IS从Level-2向Level-1进行路由渗透。

缺省情况下，Level-2区域的路由信息不渗透到Level-1区域渗透。

命令格式

import-route isis level-2 into level-1 [ filter-policy { acl-number | ip-prefix ip-prefix-name | route-policy route-policy-name } ] [ tag tag ]

undo import-route isis level-2 into level-1

参数说明

filter-policy：设置路由过滤条件。

acl-number：基本ACL编号。

ip-prefix ip-prefix-name：指定IP地址前缀名。只有与该前缀相匹配的路由才能引入。

route-policy route-policy-name：指定路由策略名称。

tag tag：为引入的路由分配管理标记值。

使用指南

通过IS-IS路由渗透功能，Level-1-2路由器可以将它所知道的其他Level-1区域路由信息和Level-2区域路由信息渗透到路由器所在的Level-1区域。

使用实例

＃ 配置IS-IS从Level-2向Level-1进行路由渗透。

<Quidway> system-view
[Quidway] isis 1 
[Quidway-isis-1] import-route isis level-2 into level-1 filter-policy 2000
