static-cr-lsp egress

命令功能

static-cr-lsp egress命令用来在出口节点配置静态CR-LSP。

undo static-cr-lsp egress命令用来在出口节点删除配置的静态CR-LSP。

命令格式

static-cr-lsp egress lsp-name incoming-interface interface-type interface-number in-label in-label [ lsrid ingress-lsr-id tunnel-id tunnel-id ]

undo static-cr-lsp egress lsp-name

参数说明

lsp-name：CR-LSP隧道的名字。字符串形式，区分大小写，不支持空格，长度范围是1～19。

incoming-interface interface-type interface-number：指定入接口的接口类型和接口编号。

in-label in-label：指定入标签。in-label的取值范围是16～1023。

lsrid ingress-lsr-id：指定入节点的LSR ID。

tunnel-id tunnel-id：指定隧道标识。tunnel-id为整数形式，取值范围取决于产品的许可证文件。
