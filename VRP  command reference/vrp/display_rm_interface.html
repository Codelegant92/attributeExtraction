display rm interface

命令功能

display rm interface命令用来显示接口的路由管理信息，包括物理接口和逻辑接口的信息。

命令格式

display rm interface [ interface-type interface-number | vpn-instance vpn-instance-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-type interface-number：接口类型和接口号。

vpn-instance vpn-instance-name：指定VPN实例名。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

该命令可以查看接口的路由管理信息，包括物理接口和逻辑接口的信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
