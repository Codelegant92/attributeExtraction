accounting start-fail

命令功能

accounting start-fail命令用来配置开始计费失败策略。

缺省情况下，如果远端计费开始失败，用户将不能上线，即采用offline方式。

命令格式

accounting start-fail { online | offline }

参数说明

online：如果远端计费开始失败，不作任何特殊处理，当作计费成功处理。

Offline：如果远端计费开始失败，将不再为用户继续提供上线服务。

使用指南

在用户上线时，如果开始计费失败，设备就可以根据此命令的配置对用户进行相应的处理。

使用实例

# 对于名为scheme1的计费方案，如果远端计费开始失败，不处理。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] accounting-scheme scheme1
[Quidway-aaa-accounting-scheme1] accounting start-fail online
