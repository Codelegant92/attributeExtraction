fr traffic-policing

命令功能

fr traffic-policing命令用来使能帧中继流量监管功能。

undo fr traffic-policing命令用来禁止帧中继流量监管功能。

命令格式

fr traffic-policing

undo fr traffic-policing

参数说明

无

视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

帧中继流量监管功能应用于路由器帧中继报文的接收接口上，并且，它只能应用在帧中继网络的DCE端。

在为入接口配置流量监管时，必须先应用fr switching命令将DCE设置为帧中继交换。

使用实例

# 使能接口Serial 2/0/0的流量监管功能。

<Quidway> system-view
[Quidway] interface serial2/0/0
[Quidway-Serial2/0/0] fr traffic-policing
