display x25 map

命令功能

display x25 map命令用来显示X.25地址映射表。

命令格式

display x25 map

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

配置X.25地址映射可以通过两条途径：专门配置（使用x25 map命令）或隐含配置（使用x25 pvc命令），而通过displayx25 map命令可以查看所有的地址映射。

使用实例

# 显示X.25地址映射表。

<Quidway> display x25 map
Interface:Serial3/0/0(protocol status is up):
ip address:202.38.162.2   X.121 address: 22
    map-type: SVC_MAP  VC-number: 0
    Facility:
      ACCEPT_REVERSE;
      BROADCAST;
      PACKET_SIZE: I 512  O 512 ;
表1 display x25 map命令输出信息描述

项目

描述

Interface:

存在地址映射表的接口类型和编号。“protocol status is”表示接口的协议状态。

ip address:

接口IP地址。

X.121 address:

接口X.121地址。

map-type:

地址映射的虚电路类型。

VC-number:

虚电路连接的总数。

Facility:

接口配置的功能。

ACCEPT_REVERSE;

接受反转。

BROADCAST;

支持广播。

PACKET_SIZE:

每个报文包的大小：接收的每个报文包的大小和发送的每个报文包的大小。
