map ppp

命令功能

map ppp命令用来在PVC视图下创建PVC上的PPPoA映射。

undo map ppp命令用来删除该映射。

缺省情况下，不配置任何映射。

命令格式

map ppp virtual-template vt-number

undo map ppp

参数说明

vt-number：PPPoA对应的虚拟模板（VT）接口编号，要求事先创建。

视图

ATM-PVC视图

缺省级别

2：配置级

使用指南

在使用本命令以前，首先应保证该VT已经创建。

使用实例

下面这个例子展示了一个完整的PPPoA配置过程。

# 首先创建一个编号为10的VT接口并配置IP地址。

<Quidway> system-view
[Quidway] interface virtual-template 10
[Quidway-Virtual-Template10] ip address 202.38.160.1 255.255.255.0
[Quidway-Virtual-Template10] quit

# 接下来创建ATM接口Atm1/0/0下的PVC 1/101。

[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvc 1/101

# 并使用刚创建好的VT接口来创建PPPoA映射。

[Quidway-atm-pvc-Atm1/0/0-1/101] map ppp virtual-template 10
