idle-timeout

命令功能

idle-timeout命令用来设置用户界面断连的超时时间，即用户没有输入命令的时间。

undo idle-timeout命令用来恢复用户界面断连的超时时间的缺省值。

缺省情况下，超时时间为10分钟。

命令格式

idle-timeout minutes [ seconds ]

undo idle-timeout

参数说明

minutes：用户界面断连的超时时间的分钟数。整数形式，取值范围是0～35791，单位是分。

seconds：用户界面断连的超时时间的秒数。整数形式，取值范围是0～59，单位是秒。

使用指南

执行命令idle-timeout后，还必须在AAA视图下再执行local-user user-name idle-cut命令来使能用户闲置切断功能，才能使得登录到该VTY界面的用户在超时时间到达后自动断开连接。设置执行命令idle-timeout 0 0即关闭VTY用户界面的超时断连功能。

如果VTY用户界面没有设置闲置断连功能，则有可能导致其他用户无法获得空闲连接。

使用实例

# 设置超时为1分钟30秒。

<Quidway> system-view
[Quidway] user-interface console 0
[Quidway-ui-console0] idle-timeout 1 30
