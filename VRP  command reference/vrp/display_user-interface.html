display user-interface

命令功能

display user-interface命令用来查看用户界面信息。

命令格式

display user-interface [ ui-type ui-number1 ] [ ui-number ] [ summary ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

ui-type：用户界面的类型名。

ui-number1：用户界面相对编号。

ui-number：用户界面绝对编号。最小值为0，最大值是系统支持的用户界面总数-1。不同设备用户界面取值范围不同。

summary：用户界面的概要信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用此命令可查看用户界面的认证方式等信息。

使用实例

# 查看绝对编号为0的用户界面的详细信息。

<Quidway> display user-interface 0
  Idx  Type     Tx/Rx      Modem Privi ActualPrivi Auth  Int
+ 0    CON 0    9600       -     3     3                 N     -
  +    : Current user-interface is active.
  F    : Current user-interface is active and work in async mode.
  Idx  : Absolute index of user-interface.
  Type : Type and relative index of user-interface.
  Privi: The privilege of user-interface.
  ActualPrivi: The actual privilege of user-interface.
  Auth : The authentication mode of user-interface.
  Int  : The physical location of UIs.
      A: Authenticate use AAA.
      N: Current user-interface need not authentication.
      P: Authenticate use current UI's password.

# 查看所有的用户界面的详细信息。

<Quidway> display user-interface

+ 260  VTY 2                  -     3     2              A     -

表1 display user-interface命令输出信息描述

项目

描述

+

当前用户接口是激活状态

F

当前用户接口是激活状态并且工作在异步模式

Idx

用户接口的绝对编号

Type

用户接口的类型和相对编号

Privi

配置的用户接口的权限

ActualPrivi

用户接口的实际权限（如果使用AAA方式验证，则实际权限的值为AAA配置信息中的本地用户优先级）

Auth

用户接口的认证模式

A

使用AAA进行验证

N

当前用户接口不需要验证

P

使用当前用户接口的密码进行验证
