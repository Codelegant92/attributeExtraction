refresh bgp multicast

命令功能

refresh bgp multicast命令用来对MBGP对等体连接进行软复位。

命令格式

refresh bgp multicast { peer-address | all | group group-name } { import | export }

参数说明

peer-address：软复位与指定地址的MBGP对等体的连接。点分十进制形式。

all：软复位所有MBGP的连接。

group-name：MBGP对等体组的名称。配置其可软复位同一对等体组内对等体间的MBGP连接。

import：触发入方向的软复位。

export：触发出方向的软复位。
