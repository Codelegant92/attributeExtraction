ip forward-broadcast

命令功能

ip forward-broadcast命令用来使能接口转发广播报文。

undo ip forward-broadcast命令用来禁止接口转发广播报文。

缺省情况下，接口不转发广播报文。

命令格式

ip forward-broadcast [ acl acl-number ]

undo ip forward-broadcast

参数说明

acl acl-number：访问控制列表号。取值范围是2000～3999，其中2000～2999是基本访问控制列表，3000～3999是高级访问控制列表。
