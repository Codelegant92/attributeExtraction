pim ipv6 holdtime join-prune

命令功能

pim ipv6 holdtime join-prune命令用来设置加入或剪枝状态的保持时间。

undo pim ipv6 holdtime join-prune命令用来恢复缺省值。

缺省情况下，加入或剪枝状态的保持时间是210秒。

命令格式

pim ipv6 holdtime join-prune interval

undo pim ipv6 holdtime join-prune

参数说明

interval：指定加入或剪枝状态的保持时间。整数形式，取值范围是1～65535，单位是秒。

使用指南

此命令与IPv6 PIM视图下的holdtime join-prune（IPv6）命令作用相同，只是视图不同。缺省情况下，如果接口上没有配置，取IPv6 PIM视图下配置的值；否则取接口视图下配置的值。

请参见IPv4 PIM配置命令中的pim holdtime join-prune。

使用实例

# 设置接口GE1/0/0上加入/剪枝状态的保持时间为280秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 holdtime join-prune 280
