display rpr ring-selection interface ring-if

命令功能

display rpr ring-selection interface ring-if命令用来查看指定接口指定板号的选环信息。

命令格式

display rpr ring-selection interface ring-if interface-number { protocol | rs-table | static } [ slot slot-number ]

参数说明

interface-number：接口编号。取值范围是0～31。

protocol：通过RPR环拓扑分析，获得的选环信息。

rs-table：实际使用的选环信息。

static：用户配置的静态选环信息。

slot-number：指定显示的板号。

使用指南

如果不指定可选参数，将显示协议所在板的板号选环信息。

使用实例

# 显示接口1板号4的选环信息。

<Quidway> display rpr ring-selection interface ring-if 1 rs-table slot 4

项目

描述

selection table of interface ring-if1

指定接口ring-if1上的RPR选环信息表。

Station total Number

节点总数。

Close Ring
RPR环网的状态：

  • Close Ring：闭环。

  • Open Ring：开环。

Ring0 reachable station number

0环包含的节点总数。

Ring1 reachable station number

1环包含的节点总数。

WrapFlag
Wrap操作标志：

  • 0：未进行Wrap操作。

  • 2：进行了Wrap操作。

Ring default select ringid

默认选还的环ID。

Westringmask

西向环的掩码。

Eastringmask

东向环的掩码。

WestL3_Mcmask

西向环的三层MAC掩码。

EastL3_Mcmask

东向环的三层MAC掩码。

Hops

到达目的节点的跳数。

Mac or IPAddr

目的节点的MAC地址或IP地址。

Ring

发送环，即从本节点到达目的节点的路径。

R0TTL

0环的TTL。

R1TTL

1环的TTL。
