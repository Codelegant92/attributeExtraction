port link-type

命令功能

port link-type命令用来设置以太网接口的链路类型.

undo port link-type命令用来恢复端口的链路类型为缺省状态，即为Hybrid端口。

缺省情况下，端口为Hybrid端口。

命令格式

port link-type { access | hybrid | trunk | dot1q-tunnel }

undo port link-type

参数说明

access：设置端口为Access端口。

hybrid：设置端口为Hybrid端口。

trunk：设置端口为Trunk端口。

dot1q-tunnel：设置端口为dot1q-tunnel端口。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

如果使用三层端口，必须先执行portswitch（Ethernet接口视图）命令切换为二层端口。

使用实例

# 将以太网接口GE1/0/0设置为Trunk端口。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port link-type trunk
