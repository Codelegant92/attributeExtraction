display vlan-group

命令功能

display vlan-group命令用来显示指定接口下vlan-group的数量和每个vlan-group视图下的具体配置。

命令格式

display vlan-group [ group-id ] interface interface-type interface-number [.subinterface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vlan-group group-id：用户组号，十进制整数形式，取值范围是1～16。

interface interface-type interface-number：配置了用户组的接口类型和接口编号。

subinterface-number：配置了用户组的子接口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当在以太网子接口下配置完vlan-group后，可以执行display vlan-group命令来显示用户组的具体配置信息。如果指定了group-id，则显示指定接口和用户组号下的相关配置信息。如果未指定group-id，则显示指定接口下的所有用户组的数量和信息。

使用实例

# 显示接口GE1/0/0上配置的所有vlan-group相关的信息。

<Quidway> display vlan-group interface gigabitethernet1/0/0
Interface: GigabitEthernet1/0/0
Vlan-group total number: 3
Interface: GigabitEthernet1/0/0.1
Vlan-group total number: 2
