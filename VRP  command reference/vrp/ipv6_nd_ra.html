ipv6 nd ra

命令功能

ipv6 nd ra命令用来设置RA（Router Advertisement）报文的发布间隔时间。

undo ipv6 nd ra命令用来将此设置恢复为缺省值。

缺省情况下，最大时间间隔是600秒，最小时间间隔是200秒。

命令格式

ipv6 nd ra { max-interval maximum-interval | min-interval minimum-interval }

undo ipv6 nd ra { max-interval | min-interval }

参数说明

max-interval maximum-interval：指定RA报文的最大发布间隔时间。整数形式，取值范围是4～1800，单位是秒。缺省值是600秒。最大时间间隔不能小于最小时间间隔。

min-interval minimum-interval：指定RA报文的最小发布间隔时间。整数形式，取值范围是3～1350，单位是秒。缺省值是200秒。

使用指南

在配置ipv6 nd ra命令前，必须先在接口视图下执行ipv6 enable命令，在接口上使能IPv6功能。如果需要使ipv6 nd ra命令生效，还需要在系统视图下执行ipv6命令，使能设备的IPv6转发功能。

RA报文发布间隔时间必须小于等于RA报文的存活时间（缺省值是1800秒，可通过ipv6 nd ra router-lifetime 命令配置）。实际的RA报文的发布间隔时间是一个在max-interval和min-interval间的随机值。

此命令支持Ethernet接口及其子接口、GigabitEthernet接口及其子接口、POS接口、Eth-Trunk接口、IP-Trunk接口以及Tunnel接口。

使用实例

# 设置RA报文的最大发布间隔时间为1000秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] ipv6 enable
[Quidway-GigabitEthernet1/0/1] ipv6 nd ra max-interval 1000

# 设置RA报文的最小发布间隔时间为300秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] ipv6 enable
[Quidway-GigabitEthernet1/0/1] ipv6 nd ra min-interval 300
