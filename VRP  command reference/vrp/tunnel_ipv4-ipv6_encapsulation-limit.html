tunnel ipv4-ipv6 encapsulation-limit

命令功能

tunnel ipv4-ipv6 encapsulation-limit命令用来指定隧道封装限制值，即本次封装后的报文可被再次进行多少层IPv6封装。

undo tunnel ipv4-ipv6 encapsulation-limit命令用来恢复隧道封装限制值为缺省值。

缺省情况下，允许内嵌隧道节点最多进行4次IPv6封装。

命令格式

tunnel ipv4-ipv6 encapsulation-limit encapsulation-limit

undo tunnel ipv4-ipv6 encapsulation-limit

参数说明

encapsulation-limit：允许内嵌隧道节点IPv6封装的最大次数。整数形式，取值范围是0～255。

视图

系统视图、Tunnel接口视图

缺省级别

2：配置级

使用指南

隧道封装限制值是IPv6隧道首部中目的选项扩展首部携带的隧道封装限制选项值。

在隧道接口视图下配置该命令则仅对当前隧道接口生效。目前，在系统视图下虽然可以配置该命令，但是配置并不生效。

在与友商设备互通时，必须配置undo tunnel ipv4-ipv6 encapsulation-limit命令，不限制IPv6封装次数。

使用实例

# 允许指定的IPv4 over IPv6隧道的内嵌隧道节点最多进行3次IPv6封装。

<Quidway> system-view
[Quidway] interface tunnel1/0/0
[Quidway-Tunnel1/0/0] tunnel ipv4-ipv6 encapsulation-limit 3

# 允许本设备所有的IPv4 over IPv6隧道的内嵌隧道节点最多进行3次IPv6封装。

<Quidway> system-view
[Quidway] tunnel ipv4-ipv6 encapsulation-limit 3
