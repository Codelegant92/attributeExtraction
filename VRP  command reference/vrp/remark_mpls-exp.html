remark mpls-exp

命令功能

remark mpls-exp命令用来重新标记报文的MPLS的EXP值。

undo remark mpls-exp命令用来配置取消标记报文的MPLS的EXP值。

命令格式

remark mpls-exp exp

undo remark mpls-exp

参数说明

exp：预设置的MPLS的EXP值，取值范围是0～7。

视图

流行为视图

缺省级别

2：配置级

使用指南

缺省情况下，在MPLS网络中标记的MPLS-EXP值在出MPLS网络后会映射到IP报文的DSCP域。

本命令为覆盖式命令，即在同一流行为视图下重复使用本命令重新标记MPLS报文的EXP优先级后，按最后一次配置生效。

使用实例

# 配置标记报文的MPLS的EXP值为0。

<Quidway> system-view
[Quidway] traffic behavior database
[Quidway-behavior-database] remark mpls-exp 0
