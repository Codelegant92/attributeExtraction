ike local-name

命令功能

ike local-name命令用来配置进行IKE协商时本机的ID。

undo ike local-name命令用来删除配置。

命令格式

ike local-name router-name

undo ike local-name

参数说明

router-name：路由器的本机ID，字符串形式，长度范围是1～15。

系统视图

缺省级别

2：配置级

使用指南

在采用野蛮模式进行IKE协商时，需要配置本地路由器ID。采用主模式时不需要。

使用实例

# 配置本路由器ID为huawei。

<Quidway> system-view
[Quidway] ike local-name huawei
