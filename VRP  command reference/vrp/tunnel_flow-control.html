tunnel flow-control

命令功能

tunnel flow-control命令用来开启L2TP通道流控功能。

undo tunnel flow-control命令用来关闭通道流控功能。

缺省情况下，关闭L2TP通道流控功能。

命令格式

tunnel flow-control

undo tunnel flow-control

参数说明

无

视图

L2TP组视图

缺省级别

2：配置级

使用指南

可以开启或关闭L2TP简单的通道流控功能，达到流控目的。开启流控功能后，L2TP的接收窗口将增大4，以避免L2TP控制报文过多导致丢包。

使用实例

# 开启通道流控。

<Quidway> system-view
[Quidway] l2tp-group 1
[Quidway-l2tp1] tunnel flow-control
