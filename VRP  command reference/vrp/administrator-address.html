administrator-address

命令功能

administrator-address命令用来在成员交换机上设定命令交换机的MAC地址和命令交换机所属HGMP集群的名称。

undo administrator-address命令用来在成员交换机上删除命令交换机的MAC地址。

命令格式

administrator-address mac-address name cluster-name

undo administrator-address

参数说明

mac-address：命令交换机的MAC地址。mac-address是用16进制表示的MAC地址，格式为H-H-H，每个“H”代表4位16进制数。

name cluster-name：指定HGMP集群名称。cluster-name表示集群名称。字符串形式，区分大小写，不支持空格，长度范围是1～8。

使用指南

本命令只能在成员交换机上执行。

在成员交换机上使用undo administrator-address命令，只会使成员交换机短暂地退出集群，而不会导致命令交换机将该成员删除。如果要从一个HGMP集群中彻底删除成员交换机，建议用户使用delete-member命令。

使用实例

# 在成员交换机上设定命令交换机的MAC地址为0001-0001-0001和命令交换机所属HGMP集群的名称为HUAWEI。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] administrator-address 0001-0001-0001 name HUAWEI
