igmp send-router-alert

命令功能

igmp send-router-alert命令用来在接口上配置发送的IGMP消息其IP报文头中包含Router-Alert预警参数。

undo igmp send-router-alert命令用来在接口上配置发送的IGMP消息其IP报文头中不包含Router-Alert预警参数。

缺省情况下，该接口发送的IGMP消息其IP报文头中包含Router-Alert预警参数。

命令格式

igmp send-router-alert

undo igmp send-router-alert

参数说明

无

接口视图

缺省级别

2：配置级

使用指南

该命令通常与igmp require-router-alert命令配合使用。

该命令和send-router-alert（IGMP视图）命令功能完全相同，差别只是作用范围不同。IGMP视图下的配置具有全局性，对所有接口有效；接口视图下的配置只对当前接口有效。系统优先采用接口视图下的配置。当接口视图下没有进行配置，才采用IGMP视图下的配置。

使用实例

# 配置GE1/0/0接口上发送IGMP消息时，其IP报文头中不包含Router-Alert预警参数。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] undo igmp send-router-alert
