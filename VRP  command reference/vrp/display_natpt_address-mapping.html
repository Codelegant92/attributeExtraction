display natpt address-mapping

命令功能

display natpt address-mapping命令用来查看NAT-PT的静态和动态映射信息。

命令格式

display natpt address-mapping [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

NATPT address mapping(v6bound view)

IPv6侧的NAT-PT地址映射信息。IPv4侧的NAT-PT地址映射信息与IPv6侧的基本相同，只需将Type字段中的DESTINATION和SOURCE互换。

IPv4 Address

映射项中的IPv4地址。

IPv6 Address

映射项中的IPv6地址。

Type

  • DESTINATION：表示该地址映射信息为报文的目的地址的映射。即从IPv6侧发往IPv4侧的IPv6报文的目的地址3001::0005，将被映射为IPv4报文的目的地址8.0.0.2。

  • SOURCE：表示该地址映射信息为报文的源地址的映射。即从IPv6侧发往IPv4侧的IPv6报文的源地址2001::0002，将被映射为IPv4报文的源地址8.0.0.5。

NATPT V6Server static mapping

从IPv4侧到目的地址的静态PAT映射

Pro

进行地址转换的报文的协议类型
