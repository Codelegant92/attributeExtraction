dhcp snooping bind-table static interface

命令功能

dhcp snooping bind-table static命令用来配置DHCP Snooping功能中的IP地址与MAC地址的静态绑定关系。

undo dhcp snooping bind-table命令用来取消DHCP Snooping功能中的IP与MAC地址的绑定表项，包括动态表项和静态表项。

命令格式

dhcp snooping bind-table static ip-address ip-address [ mac-address mac-address ] [ interface interface-type interface-number [ ce-vlan ce-vlan-id ]]

undo dhcp snooping bind-table ip-address ip-address

参数说明

ip-address：用户IP地址，点分十进制格式。

mac-address：用户MAC地址，格式为H-H-H。

interface-type：接口类型。

interface-number：接口号。

ce-vlan ce-vlan-id：用户侧虚拟局域网标识，取值范围1~4094。

使用指南

执行dhcp snooping bind-table static interface命令后，DHCP Snooping绑定表中的动态表项不需配置，使能DHCP Snooping功能即可自动生成。但静态表项需要通过命令配置。

当删除一个静态表项时，该静态表项必须在DHCP Snooping绑定表中存在，否则系统会输出错误提示信息。

使用实例

# 在VLAN100的Ethernet4/0/0上配置IP与MAC静态绑定关系。

<Quidway> system-view
[Quidway] vlan 100
[Quidway-vlan100] dhcp snooping bind-table static ip-address 10.10.10.2 mac-address 0028-0120-0327 interface ethernet 4/0/0
