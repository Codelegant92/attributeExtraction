dhcp server nbns-list（接口视图）

命令功能

dhcp server nbns-list命令用来配置当前接口的DHCP地址池为其客户端分配的NetBIOS服务器地址。

undo dhcp server nbns-list命令用来删除配置。

缺省情况下，在接口上未配置NetBIOS地址。

命令格式

dhcp server nbns-list ip-address &<1-8>

undo dhcp server nbns-list { ip-address | all }

参数说明

ip-address：指定NetBIOS服务器的IP地址。一条命令中最多可以配置8个IP地址，用空格分隔。

all：删除所有NetBIOS服务器的IP地址。

使用指南

dhcp server nbns-list命令用来配置当前接口的DHCP地址池为其客户端分配的NetBIOS服务器地址。

在目前的实现中，每个DHCP地址池最多可以配置8个NetBIOS服务器。

使用实例

# 配置GE 1/0/0的DHCP地址池为其客户端分配的NetBIOS服务器地址为10.12.1.99。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] dhcp server nbns-list 10.12.1.99
