display rip neighbor

命令功能

display rip neighbor命令用来查看RIP的邻居信息。

命令格式

display rip process-id neighbor [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIP进程号。整数形式，取值范围是1～65535。

verbose：查看RIP邻居的详细信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看进程号为1的RIP邻居信息。

<Quidway> display rip 1 neighbor
----------------------------------------------------------------
 IP Address      Interface        Type   Last-Heard-Time
----------------------------------------------------------------
 1.1.1.1         Ethernet6/2/0    RIP    0:0:7
 Number of RIP routes:1
 Number of TRIP routes:0
