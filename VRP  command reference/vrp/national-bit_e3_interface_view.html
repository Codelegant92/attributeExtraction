national-bit（E3接口视图）

命令功能

national-bit命令用来设置E3接口的国际位。

undo national bit命令用来恢复缺省设置

缺省情况下，E3接口的国际位为1。

命令格式

national-bit { 0 | 1 }

undo national-bit

参数说明

0：设置国际位为0。表示这个接口只能进行国内通信。

1：设置国际位为1。表示这个接口可以进行国际通信。

视图

E3接口视图

缺省级别

2：配置级

使用指南

建议使用缺省值。重新设置后建议重启接口，以保证设置生效。

使用实例

# 设置E3接口3/0/0的国际位为0。

<Quidway> system-view
[Quidway] controller e3 3/0/0
[Quidway-E3 3/0/0] national-bit 0
