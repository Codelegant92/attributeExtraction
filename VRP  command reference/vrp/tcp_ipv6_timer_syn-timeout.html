tcp ipv6 timer syn-timeout

命令功能

tcp ipv6 timer syn-timeout命令用来设置TCP的SYN-WAIT定时器值。

undo tcp ipv6 timer syn-timeout命令用来将此设置恢复为缺省值。

缺省情况下，TCP的SYN-WAIT定时器值是75秒。

命令格式

tcp ipv6 timer syn-timeout timer-value

undo tcp ipv6 timer syn-timeout

参数说明

timer-value：指定TCP的SYN-WAIT定时器值。整数形式，单位是秒，取值范围是2～600。缺省值是75秒。
