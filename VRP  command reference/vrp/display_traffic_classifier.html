display traffic classifier

命令功能

display traffic classifier命令用来显示路由器配置的类信息。

命令格式

display traffic classifier { system-defined | user-defined } [ classifier-name ]

参数说明

system-defined ：显示系统预定义流分类的信息。

user-defined ：显示用户定义流分类的信息。

classifier-name：指定流分类名称。字符串形式，长度范围是1～31。如果未指定流分类名称，则显示所有系统预定义类或所有用户定义类的信息。

使用指南

display traffic classifier命令可以观察到的信息包括：

  • 流分类名

  • 匹配类的规则之间的关系

  • 匹配规则

使用实例

<Quidway> display traffic classifier user-defined
User Defined Classifier Information:
 Classifier: HUAWEI
  Operator: AND
  Rule(s) : if-match ip-precedence 5
 Classifier: database
  Operator: AND
  Rule(s) : if-match acl 3131
            if-match inbound-interface GigabitEthernet10/2/0
