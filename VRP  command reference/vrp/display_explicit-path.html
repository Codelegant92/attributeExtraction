display explicit-path

命令功能

display explicit-path命令用来查看显式路径信息和正在使用该显式路径的隧道。

命令格式

display explicit-path [ path-name ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

path-name：路径名，字符串形式，不支持空格，长度范围是1～31个字符。

verbose：显示详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Path Name

路径名

Path Status

显式路径的状态，“Enabled”表示已使能

Strict

表示此地址为严格下一跳，即，此地址为入节点地址，或与上一节点所在的路由器直连

Include

表示此显式路径包含该跳

Exclude

表示此显式路径不包含该跳
