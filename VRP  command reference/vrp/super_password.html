super password

命令功能

super password命令用来设置切换低级别用户到高级别用户的密码。

undo super password命令用来取消当前设置。

命令格式

super password [level user-level ] { simple |cipher }password

undo super password [ level user-level ]

参数说明

level user-level：用户级别。整数形式，取值范围是1～15。缺省情况下，对级别3设置密码。

simple：配置明文密码，密码将以明文形式保存在配置文件中。

cipher：配置密文密码，密码将以加密的形式保存在配置文件中。

password：切换级别的密码。字符串形式。如果选择simple，则password必须是明文密码，长度范围是1～16。如果选择cipher，则password可以是24位的密文密码，如：_(TT8F ] Y\5SQ=^Q`MAF4<1!!；也可以是长度范围1～16的明文密码，如：1234567。

使用指南

不论配置的是明文密码还是密文密码，验证时必须输入明文形式的密码。

如果使用simple选项，密码将以明文形式保存在配置文件中，以低级别登录的用户可以通过查看配置文件的方式获取切换密码，造成安全问题。因此，一般情况下，应使用cipher选项，将密码加密保存。

同时，使用cipher选项设置了密码后，无法从系统取回。请妥善保管，以免遗忘或者遗失。

使用实例

# 设置以低级别登录的用户，如果要切换到level-3级别，需要输入切换密码“abcd”。

<Quidway> system-view
[Quidway] super password level 3 cipher abcd
