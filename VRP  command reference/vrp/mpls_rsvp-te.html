mpls rsvp-te

命令功能

在MPLS视图下，mpls rsvp-te命令用来全局使能RSVP-TE特性；全局使能后，在接口视图下，mpls rsvp-te命令用来使能接口的RSVP-TE特性。

undo mpls rsvp-te命令用来关闭RSVP-TE特性。在MPLS视图下，关闭RSVP-TE特性，会同时关闭所有接口上配置的RSVP-TE特性。

缺省情况下，未使能RSVP-TE特性。

命令格式

mpls rsvp-te

undo mpls rsvp-te

参数说明

无

视图

MPLS视图、接口视图

缺省级别

2：配置级

使用指南

在执行mpls rsvp-te命令之前必须先执行mpls和mpls te来使能MPLS和MPLS TE功能。

使用实例

# 使能RSVP-TE特性。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls te
[Quidway-mpls] mpls rsvp-te

# 在接口上使能RSVP-TE特性。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] mpls
[Quidway-Pos1/0/0] mpls te
[Quidway-Pos1/0/0] mpls rsvp-te
