ftp timeout

命令功能

ftp timeout命令用来设置连接空闲时间。

undo ftp timeout命令用来恢复缺省的连接空闲时间。

命令格式

ftp timeout minutes

undo ftp timeout

参数说明

minutes：连接空闲时间，单位为分钟，范围是1～35791。缺省情况下，系统的连接空闲时间为30分钟。

视图

系统视图

缺省级别

3：管理级

使用指南

用户登录到FTP服务器，与之建立了连接，若连接异常中断或用户非正常中断连接，FTP服务器是无法知道的，因而连接仍保持着。为防止这类情况的发生，使用连接空闲时间，当连接在一定时间内没有进行命令交互，FTP服务器即可认为连接已经失效，而断开连接。

使用实例

# 设置连接空闲时间为36分钟。

<Quidway> system-view
[Quidway] ftp timeout 36
