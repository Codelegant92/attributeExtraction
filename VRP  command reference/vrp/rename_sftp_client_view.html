rename（SFTP客户端视图）

命令功能

rename命令用来给SFTP服务器上的文件和文件夹重新命名。

命令格式

rename old-name new-name

参数说明

old-name：SFTP服务器上的源文件名或文件夹名，字符串形式，绝对路径名长度范围是1～64。

new-name：新文件名或文件夹名，字符串形式，绝对路径名长度范围是1～64。

视图

SFTP客户端视图

缺省级别

2：配置级

使用指南

重新命名后，老文件名或文件夹名自动失效。

使用实例

# 重新命名SSH服务器端授权目录下的文件名。

<Quidway> system-view
[Quidway] sftp 10.164.39.222
Please input the username:client001
Trying 10.164.39.222 ...
Press CTRL+K to abort
Connected to 10.164.39.222 ...
Enter password:
sftp-client> rename 222 111
Warning: Rename cfcard:/222 to cfcard:/111? [Y/N]: y
Info: Succeeded in renaming file.
