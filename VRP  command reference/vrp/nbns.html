nbns

命令功能

nbns命令用来为当前域指定NBNS名字服务器。

undo nbns命令用来删除当前域的NBNS名字服务器。

缺省情况下，域没有NBNS名字服务器。

命令格式

nbns { primary-ip | second-ip } ip-address

undo nbns { primary-ip | second-ip }

参数说明

primary-ip：设置主NBNS名字服务器。

second-ip：设置从NBNS名字服务器。

ip-address：NBNS名字服务器的IP地址，点分十进制形式。必须使用合法的单播地址。

AAA视图、AAA域视图

缺省级别

3：管理级

使用指南

改变域指定的NBNS名字服务器，用户下一次登录时生效。

使用实例

# 设置当前域使用IP地址为10.1.1.1的服务器做为NBNS名字服务器。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei] nbns primary-ip 10.1.1.1
