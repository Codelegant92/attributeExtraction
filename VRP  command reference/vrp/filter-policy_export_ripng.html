filter-policy export (RIPng)

命令功能

filter-policy export命令用来指定输出路由过滤策略，只有符合条件的路由才能被引入，并通过更新报文发布出去。

undo filter-policy export命令用来取消输出路由过滤策略。

命令格式

filter-policy { acl6-number | ipv6-prefix ipv6-prefix-name } export [ protocol [ process-id ] ]

undo filter-policy export [ protocol [ process-id ] ]

参数说明

acl6-number：基本ACL6的序号。整数形式，取值范围是2000～2999。

ipv6-prefix ipv6-prefix-name：IPv6地址前缀列表名称。字符串形式，不支持空格，区分大小写，长度范围是1～169。

protocol：引入的路由协议名称。可以是bgp、direct、isis、ripng、static或ospfv3。

process-id：指定进程号。当引入路由协议为is-is、ripng或ospfv3时，必须指定进程号。
