duplex

命令功能

duplex命令用来设置以太网接口的双工模式。

undo duplex命令用来恢复缺省设置。

缺省情况下，快速以太网接口采用自适应模式。

命令格式

duplex { full | half | negotiation }

undo duplex

参数说明

full：设置以太网接口为全双工模式。

half：设置以太网接口为半双工模式。

negotiation：设置以太网接口为自适应模式，是否支持该参数与具体产品有关。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

对以太网接口双工模式进行配置时，注意与对端设备一致。

使用实例

# 设置快速以太网接口1/0/0采用全双工模式。

<Quidway> system-view
[Quidway] interface ethernet 1/0/0
[Quidway-Ethernet1/0/0] duplex full
