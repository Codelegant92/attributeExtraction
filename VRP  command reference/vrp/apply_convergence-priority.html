apply convergence-priority

命令功能

apply convergence-priority命令用来设置VPNv4路由的收敛优先级。

undo apply convergence-priority命令用来取消对VPNv4路由的收敛优先级的设置。

缺省情况下，VPNv4路由的收敛优先级值是0。

命令格式

apply convergence-priority priority

undo apply convergence-priority

参数说明

priority：VPNv4路由的收敛优先级。取值越大，收敛优先级越高。整数形式，取值范围是0～2。缺省值是0。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

路由收敛时，具有高收敛优先级的VPNv4路由将被优先处理。

使用实例

# 设置VPNv4路由收敛优先级为2（最高）。

<Quidway> system-view
[Quidway] route-policy a permit node 10
[Quidway-route-policy] apply convergence-priority 2
