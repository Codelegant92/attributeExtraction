ftp-password

命令功能

ftp-password命令用来配置NQA测试中FTP测试例的用户密码。

undo ftp-password命令用来取消恢复缺省配置。

缺省情况下，没有配置FTP测试例的用户密码。

命令格式

ftp-password password

undo ftp-password

参数说明

password：FTP用户密码。字符串形式，长度范围是1～32。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对FTP类型的测试例配置FTP用户密码。

不能更改正在运行的测试例的用户密码。

使用实例

# 设置测试例user test的FTP测试密码为“hello”。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type ftp
[Quidway-nqa-user-test] ftp-password hello
