display vlan

命令功能

display vlan命令用来查看所有VLAN的状态，包括是否使能VLAN、是否使能MAC地址学习、是否使能广播等。

命令格式

display vlan [ vlan-id [ verbose ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vlan-id：VLAN ID，取值范围是1～4094。

verbose：显示VLAN的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

指定vlan-id将显示VLAN包含的端口，包括对报文打标签的端口（Tagged）和不对报文打标签的端口（Untagged）。

项目

描述

VLAN ID

VLAN号。

Type
VLAN类型。有三种：

  • sub：sub-VLAN

  • super：super-VLAN

  • common：普通VLAN

Status
vlan状态。

  • disable：vlan处于去使能状态（使用shutdown vlan命令可以去使能vlan）。

  • enable：vlan处于使能状态 （使用undo shutdown vlan命令可以使能vlan）。

MAC Learning
是否使能地址学习功能。

  • disable：去使能地址学习功能

  • enable：使能地址学习功能

Broadcast
是否使能广播功能。

  • disable：去使能广播功能

  • enable：使能广播功能

Untagged Port

出方向不带Tag的端口。端口配置port hybrid untagged vlan和port default vlan后，报文从此端口出去时不带Tag。

Tagged Port

出方向带Tag的端口。端口配置port trunk allow-pass，报文从此端口出去时带Tag。

Description

对VLAN的描述。
