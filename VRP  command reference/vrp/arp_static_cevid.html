arp static cevid

命令功能

arp static cevid命令用来配置带两层Tag的静态ARP映射表项。

undo arp static命令用来取消带两层Tag的ARP映射表中对应地址的静态映射项。

缺省情况下，系统ARP映射表为空，由动态ARP获取地址映射。

命令格式

arp static ip-address mac-address vid vid cevid ce-vid interface interface-type interface-number.subinterface-number

undo arp static ip-address [ vid vid cevid ce-vid interface interface-type interface-number.subinterface-number ]

参数说明

ip-address：ARP映射项的IP地址，为点分十进制格式。

mac-address：ARP映射项的以太网MAC地址，形式是H-H-H，其中H为1至4位的十六进制数。

vid vid：外层Tag的值，十进制整数形式，取值范围是1～4094。

cevid ce-vid：内层Tag的值，十进制整数形式，取值范围是1～4094。

interface interface-type interface-number：指定ARP的出接口名称。

使用指南

ARP表中除了包含IP地址、物理地址和出口地址的对应项外，还包含内外层Tag的对应值。转发报文时根据ARP表为报文增加两层Tag信息。

为QinQ终结子接口配置静态ARP表项时，指定的vid值应该与qinq termination pe-vid ce-vid命令中指定的pe-vid的值一致；指定的ce-vid值应该在qinq termination pe-vid ce-vid命令中指定的ce-vid的值范围之内。

使用实例

# 配置IP地址为1.1.1.1的用户报文的ARP表项。外层Tag为10，内层Tag为100，出接口为GE1/0/0.1。

<Quidway> system-view
[Quidway] arp static 1.1.1.1 00e0-fc01-0000 vid 10 ce-vid 100 interface gigabitethernet 1/0/0.1

# 删除IP地址为1.1.1.1的用户报文的ARP表项。外层Tag为10，内层Tag为100，出接口为GE1/0/0.1。

<Quidway> system-view
[Quidway] undo arp static 1.1.1.1 vid 10 ce-vid 100 interface gigabitethernet 1/0/0.1
