icmp-jitter-mode

命令功能

icmp-jitter-mode命令用来指定ICMP Jitter测试模式的类型。

undo icmp-jitter-mode命令用来恢复ICMP Jitter测试模式的缺省类型。

缺省情况下，ICMP Jitter测试模式的类型为icmp-timestamp。

命令格式

icmp-jitter-mode { icmp-echo | icmp-timestamp }

undo icmp-jitter-mode

参数说明

icmp-echo：指定ICMP Jitter测试模式为使用ICMP echo报文进行测试。

icmp-timestamp：指定ICMP Jitter测试模式为使用ICMP timestamp报文进行测试。

使用指南

只有在测试例类型为Icmp Jitter测试或Path Jitter测试时，才能执行命令icmp-jitter-mode配置ICMP Jitter测试模式的类型。

使用实例

# 指定ICMP Jitter测试的模式类型为使用ICMP echo报文进行测试。

<Quidway> system-view
[Quidway] nqa test-instance admin icmp-jitter
[Quidway-admin-icmp-jitter] nqa test-type icmp-jitter
[Quidway-admin-icmp-jitter] icmp-jitter-mode icmp-echo
