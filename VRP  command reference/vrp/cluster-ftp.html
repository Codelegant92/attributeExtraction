cluster-ftp

命令功能

cluster-ftp命令用来建立成员交换机与HGMP集群公用的FTP服务器之间的FTP连接。

命令格式

cluster-ftp

参数说明

无

视图

用户视图

缺省级别

2：配置级

使用指南

本命令只能在成员交换机上执行。

在执行本命令前，需要先在命令交换机上通过ftp-server命令配置本HGMP集群的公用FTP服务器的IP地址。

使用实例

# 与集群公用的FTP服务器（10.0.0.1）建立FTP连接。用户名为ftp-user，口令为ftp123。

<HUAWEI_1.Quidway> cluster-ftp
