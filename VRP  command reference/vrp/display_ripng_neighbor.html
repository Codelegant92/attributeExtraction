display ripng neighbor

命令功能

display ripng neighbor命令用来查看RIPng的邻居信息。

命令格式

display ripng process-id neighbor [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIPng进程号。整数形式，取值范围是1～65535。

verbose：打开邻居的详细内容调试信息开关。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看进程号为1的RIPng邻居信息。

<Quidway> display ripng 1 neighbor
 Neighbor : FE80::A0A:201:1 GigabitEthernet1/0/0
    Protocol : RIPNG
