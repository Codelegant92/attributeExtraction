x25 reverse-charge-accept

命令功能

x25 reverse-charge-accept用来允许本接口接受带有“反响计费请求”用户设施的呼叫。

undo x25 reverse-charge-accept用来禁止本接口接受带有“反响计费请求”用户设施的呼叫。

缺省情况下，禁止接受携带反向计费请求的呼叫。

命令格式

x25 reverse-charge-accept

undo x25 reverse-charge-accept

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

该项功能不影响任何不携带“反向计费请求”用户设施的呼叫。

若在一个X.25接口上打开“接受携带反向计费请求的呼叫”这一功能，则所有到达该X.25接口的这样的呼叫都将被接受；若通过x25 map命令中的reverse-charge-accept选项为某一条地址映射打开此功能，则只有到达该接口的针对于该地址映射的这样的呼叫才被接受，而其余的这样的呼叫（指携带了反向计费请求，但不针对该地址映射的呼叫）将被清除。

使用实例

# 接口Serial1/0/0打开接受携带反向计费请求的呼叫。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 reverse-charge-accept
