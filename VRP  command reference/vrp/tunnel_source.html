tunnel source

命令功能

tunnel source命令用来配置LAC端向LNS发起隧道建立请求时使用的隧道源接口。

undo tunnel source命令用来恢复缺省配置。

缺省情况下，LAC使用本地路由表中能够到达LNS的接口地址作为源地址向LNS发起隧道建立请求。

命令格式

tunnel source loopback interface-number

undo tunnel source

参数说明

loopback interface-number：LAC端发起隧道请求时使用的隧道源接口类型和接口编号，只能是Loopback接口。

视图

L2TP组视图

缺省级别

2：配置级

使用指南

该配置只在路由器作为LAC时有效。为了提高LAC与LNS通信的可靠性，建议用户在LAC侧配置tunnel source命令。

在指定Loopback接口作为源接口时，该Loopback接口必须已经被创建。如果指定的隧道源接口没有配置IP地址，也使用本地路由表中的地址作为源地址。

使用实例

# 配置LAC使用接口Loopback1作为隧道源接口。

<Quidway> system-view
[Quidway] interface loopback 1
[Quidway-Loopback1] quit
[Quidway] l2tp-group 1
[Quidway-l2tp-1] tunnel source loopback 1
