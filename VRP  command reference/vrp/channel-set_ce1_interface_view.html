channel-set（CE1接口视图）

命令功能

channel-set命令用来对CE1接口进行时隙捆绑。

undo channel-set命令用来取消指定的时隙。

注意：

使用undo channel-set命令时，如果不指定任何参数，则删除该CE1接口下所有的时隙。

命令格式

channel-set set-number timeslot-list slot-list

undo channel-set [ set-number ]

参数说明

set-number：CE1接口时隙捆绑形成的接口的索引号。整数形式，取值范围是0～30。

slot-list：用于捆绑的时隙编号或时隙范围。整数形式，取值范围是1～31。指定捆绑时隙时，可以指定单个时隙，也可以指定时隙范围。使用“,”分隔多个时隙，使用“-”表示时隙范围。

缺省级别

2：配置级

使用指南

只有当CE1接口工作在通道化工作模式下时，才能配置该命令。

CE1接口有两种工作模式：

  • 当工作在通道化模式下时，在物理上分为32个时隙，可以对除时隙0以外的31个时隙进行捆绑，使用该接口时，可以任意地将全部时隙分成若干组，每组时隙捆绑以后作为一个接口使用，其逻辑特性与同步串口相同。使用interface serial命令进入捆绑后形成的接口的视图。

  • 当工作在净通道模式下时，整个接口不再时隙化，不支持channel-set命令，而是作为一个2.048M带宽的串口使用。

在两种工作模式下，接口都支持PPP、帧中继、LAPB和X.25等链路层协议。

使用实例

# 将CE1接口1/0/0的1、10-15和18时隙捆绑为1号channel-set。

<Quidway> system-view
[Quidway] controller e1 1/0/0
[Quidway-E1 1/0/0] using ce1
[Quidway-E1 1/0/0] channel-set 1 timeslot-list 1,10-15,18
