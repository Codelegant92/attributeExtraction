ripng pkt-transmit

命令功能

ripng pkt-transmit命令用来在指定接口上设置RIP发送更新报文的时间间隔和每次发送报文的数量。

undo ripng pkt-transmit命令用来在接口上恢复其缺省值。

命令格式

ripng pkt-transmit { interval interval | number pkt-count } *

undo ripng pkt-transmit

参数说明

interval：报文发送的时间间隔。整数形式，取值范围是50～500，单位是毫秒。缺省值是200。

pkt-count：队列中每次发送报文的数量。整数形式，取值范围是25～100。缺省值是30。

使用指南

根据不同的网络环境在接口上配置ripng pkt-transmit { interval interval | number pkt-count } *命令，可以更精确的控制报文的发送间隔和发送的报文数，从而提升RIPng性能。

使用实例

# 在指定接口GigabitEthernet 1/0/0上配置RIPng发送报文间隔为100毫秒和发送队列报文数量为50。

<Quidway> system-view
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ripng pkt-transmit interval 100 number 50
