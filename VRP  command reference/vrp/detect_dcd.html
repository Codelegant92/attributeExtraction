detect dcd

命令功能

detect dcd命令用来使能串口的电平检测功能。

undo detect dcd命令用来禁止串口的电平检测功能。

缺省情况下，使能电平检测功能。

命令格式

detect dcd

undo detect dcd

参数说明

无

视图

串行接口视图

缺省级别

2：配置级

使用指南

说明：

由其他接口板通道化成的serial接口不支持detect dcd命令。

如果设置禁止接口检测DCD（Data Carrier Detect）信号，系统将只检测串口是否外接电缆，而自动向用户报告串口的Up和Down状态。

如果允许串口检测DCD信号，只有当该信号有效时，系统才认为串口处于Up状态，否则为Down状态。

使用实例

# 禁止串口Serial 1/0/0的DCD电平检测功能。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] undo detect dcd
