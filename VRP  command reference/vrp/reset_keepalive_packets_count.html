reset keepalive packets count

命令功能

reset keepalive packets count命令用来将GRE隧道接口发送给对端的Keepalive报文数量和Keepalive响应报文数量，以及从对端接收的Keepalive报文数量和Keepalive响应报文数量的统计信息清零。

命令格式

reset keepalive packets count

参数说明

无

视图

Tunnel接口视图

缺省级别

1：监控级

使用指南

只有将Tunnel接口的隧道类型配置为GRE后才能执行此命令。

使用实例

# 将GRE隧道接口Tunnel1/0/0的Keepalive报文统计信息清零，包括发送给对端的Keepalive报文数量和keepalive响应报文数量，以及从对端接收的Keepalive报文数量和Keepalive响应报文数量。

<Quidway1> system-view
[Quidway1] interface tunnel 1/0/0
[Quidway1-Tunnel1/0/0] tunnel-prococol gre
[Quidway1-Tunnel1/0/0] reset keepalive packets count
 Info: Reset tunnel keepalive packets count successfully.
