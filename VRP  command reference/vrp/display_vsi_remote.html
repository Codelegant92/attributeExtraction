display vsi remote

命令功能

display vsi remote命令用于显示远程的VSI实例的信息。

缺省情况下，显示所有LDP/BGP信令方式的远程VSI实例信息。

命令格式

display vsi remote ldp [ router-id ip-address ] [ pw-id pw-id ] [ | count ] [ | { begin | include | exclude } regular-expression ]

display vsi remote bgp [ nexthop nexthop-address [ export-vpn-target vpn-target ] | route-distinguisher route-distinguisher ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

ldp：用于显示LDP信令方式的各种远程VSI实例的信息。

bgp：用于显示BGP信令方式的各种远程VSI实例的信息。

route-id ip-address：只显示指定对等体的远程VSI实例信息。ip-address是对等体的IPv4地址。

pw-id pw-id：只显示指定PW的远程VSI实例信息。pw-id是虚电路号，唯一标志一条虚电路，十进制整数形式，取值范围是1～4294967295。

nexthop nexthop-address：根据标签块的下一跳地址，显示远端发布过来的标签块。

export-vpn-target vpn-target：根据标签块的下一跳地址和出方向的路由属性，显示从远端发布过来的标签块。

route-distinguisher route-distinguisher：根据路由标志符显示从远端发布过来的标签块。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display vsi remote命令后，可以通过在命令行中指定关键字来显示特定信令方式的远程VSI实例信息。

使用实例

# 显示LDP信令方式的远端VSI实例的信息。

<Quidway> display vsi remote ldp
Vsi      Peer                VC      Group   Vsi          MTU      Vsi
ID       RouterID            Label   ID      Type         Value    Index
123      3.3.3.9            17408   0       vlan         1500     1
表1 display vsi remote ldp命令输出信息描述
