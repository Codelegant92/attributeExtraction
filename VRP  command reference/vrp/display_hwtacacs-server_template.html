display hwtacacs-server template

命令功能

display hwtacacs-server template命令用来查看HWTACACS服务器的配置情况。

命令格式

display hwtacacs-server template [ template-name [ verbose ] ]

参数说明

template-name：HWTACACS服务器模板名，字符串形式，长度范围是1～32，不区分大小写。

verbose：在配置认证、授权、计费等功能后，使用此参数将显示详细的HWTACACS服务器统计信息。

项目

描述

HWTACACS-server template name

HWTACACS服务器的模板名称

Primary-authentication-server

主认证服务器IP地址、端口号及绑定的VPN实例

Primary-authorization-server

主授权服务器IP地址、端口号及绑定的VPN实例

Primary-accounting-server

主计费服务器IP地址、端口号及绑定的VPN实例

Secondary-authentication-server

次认证服务器IP地址、端口号及绑定的VPN实例

Secondary-authorization-server

次授权服务器IP地址、端口号及绑定的VPN实例

Secondary-accounting-server

次计费服务器IP地址、端口号及绑定的VPN实例

Current-authentication-server

现在运行的认证服务器IP地址、端口号及绑定的VPN实例

Current-authorization-server

现在运行的授权服务器IP地址、端口号及绑定的VPN实例

Current-accounting-server

现在运行的计费服务器IP地址、端口号及绑定的VPN实例

Source-IP-address

HWTACACS服务器的源IP地址

Shared-key

HWTACACS服务器的密钥

Quiet-interval(min)

HWTACACS主服务器恢复激活状态的时间

Response-timeout-Interval(sec)

HWTACACS服务器应答超时时间

Domain-included

HWTACACS服务器的用户名格式是否带域名

Traffic-unit

HWTACACS服务器的流量单位
