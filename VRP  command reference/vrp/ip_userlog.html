ip userlog

命令功能

ip userlog命令用来在相应接口板上启动用户日志功能。

undo ip userlog命令用来取消相应接口板上配置的用户日志功能。

缺省情况下，接口板上不启动用户日志功能。

命令格式

ip userlog nat slot slot-number [ acl acl-number ]

undo ip userlog nat slot slot-number

参数说明

nat：启动NAT日志功能。

acl-number：访问控制列表号，用来过滤掉不匹配的流，只对感兴趣的数据流进行日志记录。取值范围是2000～3999。

slot-number：启动日志功能的槽号，NAT日志是在配置了NAT的出接口所在的槽号。

使用指南

只有先使能用户日志功能，其他关于用户日志的配置才能生效。

使用实例

# 在路由器3号槽位启动NAT日志功能。

<Quidway> system-view
[Quidway] ip userlog nat slot 3
