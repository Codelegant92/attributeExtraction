display control-flap

命令功能

display control-flap命令用来查看接口的control-flap特性的当前运行状态和统计信息。用户可以根据这些统计信息调整control-flap的参数。

如果不指定接口，将显示所有接口的control-flap特性运行状态和统计信息。

命令格式

display control-flap [ interface interface-type interface-number ]

参数说明

interface interface-type interface-number：接口类型和接口编号。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

项目

描述

Control flap (OSI) status:

接口的OSI协议部分的抑制状态

suppressed表示被抑制，unsuppressed表示未被抑制

Flap (OSI) count:

接口的OSI协议部分被抑制的总次数

OSI current penalty:

接口的OSI协议部分当前惩罚值

Control flap (IP) status:

接口的IP协议部分的抑制状态

suppressed表示被抑制，unsuppressed表示未被抑制

Flap (IP) count:

接口的IP协议部分被抑制的总次数

IP current penalty:

接口的IP协议部分当前惩罚值

Control flap paramater:

接口上配置的control-flap参数

suppress

接口Damping抑制门限

reuse

接口可重用门限

decay-ok

接口Up状态半衰期

decay-ng

接口Down状态半衰期

ceiling

接口Damping最高限值
