vlan-type

命令功能

vlan-type命令用来定义Eth-Trunk子接口为VLAN成员，并指定VLAN封装方式。

undo vlan-type dot1q命令用来去除VLAN封装。

缺省情况下，子接口上没有封装VLAN，也没有与子接口关联的VLAN ID。

命令格式

vlan-type dot1q vlan-id

undo vlan-type dot1q vlan-id

参数说明

dot1q：采用802.1Q的VLAN封装方式。

vlan-id：VLAN ID。取值范围是1～4094。

视图

Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

每个Eth-Trunk子接口可配置一个VLAN ID。

使用实例

# 在Eth-Trunk子接口视图下定义VLAN ID为4094，并指定VLAN封装方式。

<Quidway> system-view
[Quidway] interface eth-trunk 3
[Quidway-Eth-Trunk3] quit
[Quidway] interface eth-trunk 3.3
[Quidway-Eth-Trunk3.3] vlan-type dot1q 4094
