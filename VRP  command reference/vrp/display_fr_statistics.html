display fr statistics

命令功能

display fr statistics命令用来查看帧中继当前收发数据的统计信息。

命令格式

display fr statistics [ interface interface-type interface-number ]

参数说明

interface-type interface-number：接口类型和接口编号，只能指定主接口，三维形式：槽号/卡号/接口号。不指定接口则显示全部帧中继接口的统计信息。

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看帧中继当前收发数据的统计信息。

<Quidway> display fr statistics interface mfr 4/0/0
Frame relay packet statistics for interface MFR4/0/0 (DTE)
  in packets = 2132202, in bytes = 25592466
  out packets = 2132208, out bytes = 25592426
  discarded in packets = 0, discarded out packets = 0
