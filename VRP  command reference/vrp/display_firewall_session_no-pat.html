display firewall session no-pat

命令功能

display firewall session no-pat命令显示一对一方式地址转换的会话表项相关信息。

命令格式

display firewall session no-pat slot slot-id

参数说明

slot-id：为指定的接口板槽号。

视图

所有视图

缺省级别

1：监控级

使用指南

配置了NAT No-pat方式后的地址转换后才可以通过这条命令查看地址转换的会话表项相关信息。

使用实例

#显示一对一方式地址转换的会话表项相关信息。

<Quidway> display firewall session no-pat slot 4
Nopat information:
NOPAT:  11.1.1.28<--->110.1.1.1  ttl:  00:00:20    left:  00:00:19
