mac-address aging-time（VSI视图）

命令功能

mac-address aging-time命令用来设置VSI中MAC地址表项老化时间。

undo mac-address aging-time命令用来恢复到缺省值。

缺省情况下，VSI中MAC地址表项老化时间为全局老化时间。

命令格式

mac-address aging-time seconds

undo mac-address aging-time

参数说明

seconds：老化时间，取值范围是0、60～1000000。单位是秒。如果老化时间被设置为0，则所有动态表项都不被老化。

视图

VSI视图

缺省级别

2：配置级

使用指南

MAC地址表的老化时间是针对动态表项，非动态表项不会老化。

如果设置为0，表示VSI中所有MAC地址表项都不被老化。

新创建的VSI默认没有配置MAC老化时间，采用全局老化时间。

必须为有效VSI才允许配置MAC老化时间，即：必须配置信令（LDP方式或者BGP方式）。并且LDP方式必须配置vsi-id，BGP方式必须配置route-distinguisher。

使用实例

# 配置LDP方式VSI中MAC地址表项老化时间为500秒。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] quit
[Quidway] vsi v100 static
[Quidway-vsi-v100] pwsignal ldp
[Quidway-vsi-v100-ldp] vsi-id 100
[Quidway-vsi-v100-ldp] quit
[Quidway-vsi-v100] mac-address aging-time 500

# 配置BGP方式VSI中MAC地址表项老化时间为600秒。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] quit
[Quidway] vsi v200 auto
[Quidway-vsi-v200] pwsignal bgp
[Quidway-vsi-v200-bgp] route-distinguisher 1:1
[Quidway-vsi-v200-bgp] quit
[Quidway-vsi-v200] mac-address aging-time 600
