display bfd ttl

命令功能

display bfd ttl命令用来查看配置的全局TTL。

命令格式

display bfd ttl [ slot slot-id ]

参数说明

slot slot-id：接口板号，多框为接口板名称。

所有视图

# 显示所有全局TTL的配置信息。

<Quidway> display bfd ttl
