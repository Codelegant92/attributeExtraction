display anti-attack statistics

命令功能

display anti-attack statistics命令用来查看指定接口板上指定类型攻击报文的统计数据。

命令格式

display anti-attack statistics [ urpf | abnormal | fragment | tcp-syn | udp-flood | icmp-flood ] [ slot slot-number ]

参数说明

slot slot-number：接口板槽位号。不指定槽位号时，缺省显示所有接口板上的攻击防范统计数据。

urpf：指定显示URPF检查的统计数据。

abnormal：指定显示畸形报文攻击防范的统计数据。

fragment：指定显示分片报文攻击防范的统计数据。

tcp-syn：指定显示TCP-Syn报文攻击防范的统计数据。

udp-flood：指定显示UDP-Flood报文攻击防范的统计数据。

icmp-flood：指定显示ICMP-Flood报文攻击防范的统计数据。

使用指南

显示的攻击类型包括URPF、Abnormal、Fragment、TCP-Syn、UDP-Flood、ICMP-Flood。

显示的内容包括，报文总数，通过的报文数和丢弃的报文数。

不指定攻击类型时，缺省显示所有攻击类型的报文统计。

使用实例

# 查看接口板Slot 6上攻击防范的统计数据。

<Quidway> system-view
[Quidway] display anti-attack statistics slot 6

项目

描述

AntiAtkType
攻击防范类型，包含以下几类：

  • URPF：URPF防攻击。

  • Abnormal：畸形报文攻击防范。

  • Fragment：分片报文攻击防范。

  • Tcp-syn：Syn-flood攻击防范。

  • Udp-flood：Udp-flood攻击防范。

  • Icmp-flood：Icmp-flood攻击防范。

TotalPacketNum

总的报文统计。

DropPacketNum

丢弃的报文数。

PassPacketNum

通过的报文数。
