apply isis

命令功能

apply isis命令用来设置引入到IS-IS协议中的路由的Level级别。

undo apply isis命令用来取消该功能。

命令格式

apply isis { level-1 | level-1-2 | level-2 }

undo apply isis

参数说明

level-1：IS-IS的Level-1路由。

level-1-2：IS-IS的Level-1路由和Level-2路由。

level-2：IS-IS的Level-2路由。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

无

使用实例

# 设置引入到IS-IS中的路由的Level级别。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply isis level-1
