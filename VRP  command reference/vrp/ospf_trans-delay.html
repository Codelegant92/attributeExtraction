ospf trans-delay

命令功能

ospf trans-delay命令用来设置接口对LSA的传输延迟时间。

undo ospf trans-delay命令用来将LSA的传输延迟时间恢复为缺省值。

缺省情况下，传输延迟时间为1秒。

命令格式

ospf trans-delay interval

undo ospf trans-delay

参数说明

interval：接口对LSA的传输延迟时间。整数形式，取值范围是1～500，单位是秒。

接口视图

缺省级别

2：配置级

使用指南

LSA在本路由器的链路状态数据库（LSDB）中会随时间老化（每秒钟加1），但在网络的传输过程中却不会，所以有必要在发送之前在LSA的老化时间上增加本命令所设置的一段时间。此配置对低速率的网络尤其重要。

OSPF协议不支持NULL接口的配置。

使用实例

# 指定接口Pos1/0/0上传送LSA的时延值为3秒。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ospf trans-delay 3
