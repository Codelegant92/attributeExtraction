rpr protection mode

命令功能

rpr protection mode命令用来配置本地节点的保护模式。

undo rpr protection mode命令用来恢复缺省值。

缺省情况下，使用wrapping保护模式。

命令格式

rpr protection mode { steering | wrapping }

undo rpr protection mode

参数说明

steering：将本节点的保护模式配置为steering模式。

wrapping：将本节点的保护模式配置为wrapping模式。

视图

RPR逻辑接口视图

缺省级别

2：配置级

使用指南

当环上的节点或链路出现异常情况时，所配置的保护模式将发挥作用以保证环上帧的传输受到的影响最小。

使用实例

# 将本地节点配置为wrapping模式。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr protection mode wrapping
