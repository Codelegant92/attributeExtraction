peer bfd enable

命令功能

peer bfd enable命令用来为对等体/组创建缺省检测参数的BFD会话。

undo peer bfd enable命令用来取消BFD功能。

命令格式

peer { group-name | ipv4-address } bfd enable

undo peer { group-name | ipv4-address } bfd enable

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

视图

BGP视图、BGP-VPN实例视图

缺省级别

2：配置级

使用指南

在使能BFD for BGP之前，需要在系统视图下先使能BFD。

BGP对等体的状态是Established时，BFD会话才能被创建。

Peer级和Group级的BFD配置遵从个体优先的原则，即Peer上的配置优先级要高于Group上的配置。如果Peer上没有配置BFD，而Peer所在的Group使能了BFD功能，则Peer会从其加入的Group中继承BFD配置。

使用实例

# 为对等体2.2.2.9配置BFD功能。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 2.2.2.9 bfd enable
