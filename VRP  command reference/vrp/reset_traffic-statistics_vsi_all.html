reset traffic-statistics vsi all

命令功能

reset traffic-statistics vsi all命令用来清除所有VPLS PW流量统计信息。

命令格式

reset traffic-statistics vsi all

参数说明

无

视图

用户视图

缺省级别

3：管理级

使用指南

用户清除所有VPLS PW流量统计信息，命令执行成功时请用户确认是否清除所有VPLS 指定PW的统计信息，输入Y确认清除，输入N取消清除操作。

使用实例

# 在用户视图清除所有VPLS PW流量统计信息

<Quidway> reset traffic-statistics vsi all 
Warning:Traffic-statistics information will be cleared! Continue? [Y/N]:Y
