if-match acl（流分类视图）

命令功能

if-match acl命令用来定义ACL匹配规则。

undo if-match acl命令用来删除ACL匹配规则。

命令格式

if-match [ not ] acl access-list-number

undo if-match [ not ] acl access-list-number

参数说明

not：不匹配该类。

access-list-number：ACL号,取值范围2000～3999。

视图

流分类视图

缺省级别

2：配置级

使用指南

在配置流分规则中匹配ACL时，需要事先定义好ACL列表。

使用实例

# 定义类匹配ACL3101。

<Quidway> system-view
[Quidway] traffic classifier class1
[Quidway-classifier-class1] if-match acl 3101
