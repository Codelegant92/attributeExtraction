pre-shared-key

命令功能

pre-shared-key命令用来配置pre-shared key认证方法的认证字。

undo pre-shared-key命令用来删除认证字。

命令格式

pre-shared-key key

undo pre-shared-key

参数说明

key：作为认证字的字符串，长度范围是1～127。

视图

IKE对等体视图

缺省级别

2：配置级

使用指南

在协商的两端需要配置相同的认证字。

如果一个策略使用pre-shared key认证方法，必须配置认证字，否则此策略不能使用。

使用实例

# 为IKE对端配置认证字为“abcde”。

<Quidway> system-view
[Quidway] ike peer huawei
[Quidway-ike-peer-huawei] pre-shared-key abcde
