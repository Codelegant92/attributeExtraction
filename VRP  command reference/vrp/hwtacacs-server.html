hwtacacs-server

命令功能

hwtacacs-server命令用来设置当前域的HWTACACS服务器模板。

undo hwtacacs-server命令用来删除为该域设置的服务器模板。

命令格式

hwtacacs-server template-name

undo hwtacacs-server

参数说明

template-name：HWTACACS服务器模板名称，字符串形式，长度范围是1～32。字符包括大写字母A～Z、小写字母a～z、数字0～9、点号“.”、短线“-”和下划线“_”。

使用指南

在用户域视图下配置hwtacacs-server命令之前，域使用的HWTACACS服务器模板必须已经存在。

使用实例

# 设置当前域使用名为huawei的HWTACACS服务器模板。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei] hwtacacs-server huawei
