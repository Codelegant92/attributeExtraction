display bootrom

命令功能

display bootrom命令用来显示串行接口或以太接口的参数信息。

命令格式

display bootrom { serial | ethernet } [ slave ]

参数说明

serial：显示串行接口的参数信息。

ethernet：显示以太接口的参数信息。

slave：显示备用主控板串行接口或以太接口的参数信息。

使用指南

display bootrom { serial | ethernet }命令可以用来查看主用主控板的接口参数信息。加上slave关键字后，可以查看备用主控板的接口参数信息。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 查看主用主控板串行接口的参数信息。

<Quidway> display bootrom serial
 Baudrate: 9600

# 查看主用主控板的以太接口的参数信息。

<Quidway> display bootrom ethernet

项目

描述

boot device

设备名

host name

提供资源文件的服务器的名字

file name

要下载的资源文件名

inet on ethernet

设备的IP地址

inet on backplane

操作系统的CPU背板的IP地址

host inet

提供资源文件的服务器的IP地址

gateway inet

网关的IP地址

user

登录服务器的用户名

ftp password

登录服务器的密码

flags

选择用FTP或TFTP方式的标志位。0表示用户以FTP方式登录；1表示用户以TFTP方式登录。

target name

将下载的资源文件重命名为该域所描述的文件名
