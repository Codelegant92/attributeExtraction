dhcp snooping bind-table autosave

命令功能

dhcp snooping bind-table autosave命令用来使能在指定路径下备份文件绑定表的功能。

undo dhcp snooping bind-table autosave命令用来取消对绑定表的自动备份功能。

缺省情况下，不使能对绑定表的自动备份功能。

命令格式

dhcp snooping bind-table autosave filename

undo dhcp snooping bind-table autosave

参数说明

filename：备份的文件路径及文件名。字符串形式，长度范围是1～51。必须同时输入当前设备支持的文件路径和文件名。

使用指南

执行dhcp snooping bind-table autosave命令后，系统将每隔60分钟在指定的备份路径下备份文件绑定表。

备份的绑定表文件的后缀名必须是.tbl。

使用实例

# 配置在硬盘下自动备份文件绑定表，文件名是backup.tbl

<Quidway> system-view
[Quidway] dhcp snooping bind-table autosave hd:/backup.tbl
