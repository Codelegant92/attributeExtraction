nqa-jitter tag-version

命令功能

nqa-jitter tag-version用来配置Jitter测试例的报文版本号。

缺省情况下，Jitter测试例的报文版本号是1。

命令格式

nqa-jitter tag-version { 1 | 2 }

参数说明

1：Jitter测试例的报文版本号1。

2：Jitter测试例的报文版本号2。

视图

系统视图

缺省级别

2：配置级

使用指南

版本1不支持单向丢包统计。

当Jitter测试的报文版本号为1时，不需要配置UDP sever端；当Jitter测试的报文版本号为2时，需要配置UDP server端，否则会导致测试例测试失败。

使用实例

# 配置Jitter测试例的报文版本号为2。

<Quidway> system-view
[Quidway] nqa-jitter tag-version 2
