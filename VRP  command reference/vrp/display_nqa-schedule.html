display nqa-schedule

命令功能

display nqa-schedule命令用来查看NQA测试例的任务调度情况。

命令格式

display nqa-schedule

参数说明

无

使用实例

# 查看当前NQA测试例的任务调度情况。

<Quidway> display nqa-schedule
   NQA Tests Max:2000                      NQA Tests Num:2
   NQA Concurrent Requests Max:1000     NQA Concurrent Requests Num:1
   NQA Jitter Concurrent  Max:5          NQA Jitter Concurrent  Num:0
   NQA icmp Concurrent  Max:50           NQA icmp Concurrent  Num:1
   Index Schedule  Time                  Type      nqa    icmp   jitter packets
       1 start     2006-8-23 19:56:54   icmp      1      1       0      0
       2 start     2006-8-23 22:0:0     jitter    2      1       1      50
       3 end       2006-8-23 22:0:4     jitter    1      1       0      0
表1 display nqa-schedule命令输出信息描述

项目

描述

NQA Tests Max:

最大NQA测试例数目

NQA Tests Num:

目前配置的NQA测试例数目

NQA Concurrent Requests Max

最大NQA并发请求数

NQA Jitter Concurrent Max:

可以同时执行的NQA Jitter测试例最大数目

NQA icmp Concurrent Max:

可以同时执行的NQA ICMP测试例最大数目

Index

测试例调度任务的索引值

Schedule
调度任务，包括：

  • start：测试例启动任务

  • end：测试例结束任务

Time

执行相应调度任务的时间

Type

测试例的测试类型

nqa

当前正在运行的NQA测试例总数

icmp

当前正在运行的ICMP类型测试例数

jitter

当前正在运行的Jitter类型测试例数

packets

当前正在运行的Jitter类型测试例每秒发包总数
