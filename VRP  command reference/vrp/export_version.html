export version

命令功能

export version命令用来设置聚合流信息的输出版本格式。

undo export version命令用来恢复聚合流信息输出版本格式的缺省值。

缺省情况下，聚合流信息的输出为版本8格式。

命令格式

export version version

undo export version

参数说明

version：指定聚合流输出的版本格式。整数形式，取值范围是8～9。缺省值是8。

使用实例

# 设置AS聚合流信息的输出为版本9格式。

<Quidway> system-view
[Quidway] ip netstream aggregation as
[Quidway-aggregation-as] export version 9
