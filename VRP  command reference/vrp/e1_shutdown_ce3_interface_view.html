e1 shutdown（CE3接口视图）

命令功能

e1 shutdown命令用来禁止CE3接口的指定E1通道。

undo e1 shutdown命令用来使能指定E1通道。

缺省情况下，E1通道是被使能的。

命令格式

e1 e1-number shutdown

undo e1 e1-number shutdown

参数说明

e1-number：E1通道的顺序号。整数形式，取值范围是1～16。

视图

CE3接口视图

缺省级别

2：配置级

使用指南

只有当CE3接口工作在CE3工作模式下时，才能配置该命令。

执行该命令可以在CE3接口下关闭某条E1通道。

禁止或使能E1通道，对于E1通道形成的串口及E1通过时隙捆绑形成的串口都有效。

对于通道化的串口，可以在串口视图下执行命令shutdown关闭串口。

使用实例

# 关闭CE3接口1/0/0的第3个E1通道

<Quidway> system-view
[Quidway] controller e3 1/0/0
[Quidway-E3 1/0/0] e1 3 shutdown
