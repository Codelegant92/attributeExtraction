c-bsr holdtime（IPv6）

命令功能

c-bsr holdtime命令用来配置C-BSR等待接收BSR发送的Bootstrap自举消息的超时时间。

undo c-bsr holdtime命令用来恢复超时时间为缺省值。

缺省情况下，C-BSR等待接收BSR发送的Bootstrap自举消息的超时时间是130秒。

命令格式

c-bsr holdtime interval

undo c-bsr holdtime

参数说明

interval：表示C-BSR等待接收BSR发送的Bootstrap自举消息的超时时间。整数形式，取值范围是1～214748364，单位是秒。“C-BSR等待接收Bootstrap自举消息的超时时间必须大于BSR发送Bootstrap自举消息的间隔时间。

缺省级别

2：配置级

使用指南

当某C-BSR竞选成为BSR后，它向网络中周期性地发送携带有holdtime超时时间参数的Bootstrap自举消息。C-BSR收到这些自举消息后，从该消息中获得超时参数并启动定时器，如果超过超时时间后C-BSR仍旧没有收到来自BSR的自举消息，则认为目前网络中的BSR失效，重新发起BSR选举。

使用实例

# 在公网实例下，在C-BSR上配置等待接收BSR发送的Bootstrap自举消息的超时时间为150秒。

<Quidway> system-view
[Quidway] pim-ipv6
[Quidway-pim6] c-bsr holdtime 150
