oam（PPP）

命令功能

oam命令用来使能PPP链路的故障检测和故障通告。

undo oam命令用来去使能PPP链路的故障检测和故障通告。

命令格式

oam { detect lcp-terminal | notify lcp-terminal } *

undo oam { detect lcp-terminal | notify lcp-terminal } *

参数说明

detect：PE设备检测本端AC链路故障的方式。

lcp-terminal：通过发送LCP的链路终止报文的方式进行故障检测和通告。

notify：把PSN侧故障通告给本端CE或者把本端AC侧故障通告给远端PE。

使用指南

PE检测到本端AC链路故障后，会把此故障发送给OAM-Manager模块。通过OAM-Manager模块把此AC链路故障通告给远端PE，从而完成端到端的FRR功能。

使用实例

# 配置PPP链路的故障检测和通告。

[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] link-protocol ppp
[Quidway-Pos1/0/0] oam detect lcp-terminal notify lcp-terminal
