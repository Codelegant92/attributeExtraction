graceful-restart (OSPFv3)

命令功能

graceful-restart命令用来使能OSPFv3进程的平滑重启功能。

undo graceful-restart命令用来关闭OSPFv3进程的平滑重启功能。

缺省情况下，不使能OSPFv3的平滑重启功能。

命令格式

graceful-restart [ period period ] [ack-time time ] [retransmit-interval interval ] [ lsa-checking-ignore] [ planned-only ]

undo graceful-restart [ period ] [ack-time ] [retransmit-interval ] [ lsa-checking-ignore ] [ planned-only ]

参数说明

period period：指定平滑重启的周期。整数形式，取值范围是1～1800，单位是秒。缺省值是120秒。

ack-time time：指定路由器在等待邻居发来Grace-LSA Ack的时长。整数形式，取值范围是1～30，单位是秒。缺省值是15秒。

retransmit-interval interval：指定重传Grace-LSA的时间间隔。整数形式，取值范围是1～5，单位是秒。缺省值是5秒。

lsa-checking-ignore：指定不执行LSA严格检查。缺省情况下，路由器对收到的各类LSA都进行严格检查。

planned-only：指定路由器只支持Planned-GR。缺省情况下，路由器支持Planned-GR和Unplanned-GR。

使用指南

ack-time是个可选参数，可以使得Restarter端在ack-time时间间隔内发现更多的邻居。

使用实例

# 使能OSPFv3进程的平滑重启功能。

<Quidway> system-view
[Quidway] ospf 1
[Quidway-ospf-1] graceful-restart
