interface { ethernet | gigabitethernet }

命令功能

interface { ethernet | gigabitethernet }命令用来创建以太网子接口。

undo interface { ethernet | gigabitethernet }命令用来删除指定的以太网子接口。

命令格式

interface { ethernet | gigabitethernet } interface-number.subinterface-number

undo interface { ethernet | gigabitethernet } interface-number.subinterface-number

参数说明

interface-number：接口编号，包括槽号/卡号/端口号。

subinterface-number：子接口编号。取值范围是1～4096。不同产品上的子接口编号范围可能不同。所有产品上，一个以太网主接口可创建的子接口数量最大为1024个。

使用指南

以太网子接口是广播类型的子接口，用于VLAN的配置。

实际使用中，一个主接口可创建的子接口数目最大为1024。

说明：

  • 当接口下配有子接口时，在主接口连续执行shutdown和undo shutdown操作的间隔应至少为15秒。

  • 当子接口配置了与L2VPN（VLL或VPLS）的绑定关系时，必须先删除子接口和L2VPN的绑定关系，才能删除子接口。

使用实例
