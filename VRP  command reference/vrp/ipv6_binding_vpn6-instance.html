ipv6 binding vpn6-instance

命令功能

ipv6 binding vpn6-instance命令用来将当前接口与IPv6 VPN实例进行绑定。

undo ipv6 binding vpn6-instance命令用来取消当前接口与IPv6 VPN实例的绑定关系。

缺省情况下，接口不与任何VPN实例相绑定，属于公网接口。

命令格式

ipv6 binding vpn6-instance vpn6-instance-name

undo ipv6 binding vpn6-instance vpn6-instance-name

参数说明

vpn6-instance-name：接口关联的IPv6 VPN实例名称。字符串形式，区分大小写，长度范围是1～31。

使用指南

配置IPv6 VPN实例后，需要将本设备上属于该IPv6 VPN的接口与该IPv6 VPN实例关联，否则该接口将属于公网接口。

在接口上配置与IPv6 VPN实例关联或取消已建立的关联都将清除该接口的IPv6地址、IPv6相关的路由协议等三层特性，如果需要应重新配置。

使用实例

# 将接口GigabitEthernet1/0/0与名为vrf1的IPv6 VPN实例关联。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 binding vpn6-instance vrf1
 All IPv6 related configurations on this interface are removed!
