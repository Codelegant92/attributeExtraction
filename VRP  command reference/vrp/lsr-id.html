lsr-id

命令功能

lsr-id命令用来配置LDP实例的LSR ID。

undo lsr-id命令用来删除LDP实例的LSR ID。

缺省情况下，没有配置LDP实例的LSR ID。

命令格式

lsr-id lsr-id

undo lsr-id

参数说明

lsr-id：LDP实例的LSR ID，点分十进制格式。

视图

MPLS-LDP-VPN实例视图、MPLS-LDP视图

缺省级别

2：配置级

使用指南

通常情况下，LDP使用缺省的MPLS的LSR ID即可，在某些使用VPN实例，例如BGP/MPLS VPN的组网方案中，如果VPN与公网地址空间重叠，则需要为LDP另外配置LSR ID，以保证TCP连接能够正常建立。

缺省情况下，LDP实例的LSR ID等于执行mpls lsr-id命令配置的MPLS的LSR ID。当LDP实例没有配置自己的LDP LSR ID时，它将使用MPLS LSR ID。

说明：

修改LSR ID的值会导致所有LDP实例的会话重建。

使用实例

# 配置LDP实例的LSR ID。

<Quidway> system-view
[Quidway] mpls ldp
[Quidway-mpls-ldp] lsr-id 2.2.2.3
Warning: All the related sessions will be deleted if the operation is performed
!Continue? (y/n)y
[Quidway-mpls-ldp] quit

# 配置VPN实例的LDP LSR ID。

<Quidway> system-view
[Quidway] mpls ldp vpn-instance vpn1
[Quidway-mpls-ldp-vpn-instance-vpn1] lsr-id 4.2.2.3
Warning: All the related sessions will be deleted if the operation is performed
!Continue? (y/n)y
[Quidway-mpls-ldp-vpn-instance-vpn1] quit
