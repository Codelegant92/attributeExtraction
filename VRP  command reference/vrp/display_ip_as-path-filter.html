display ip as-path-filter

命令功能

display ip as-path-filter命令用来显示已配置的AS路径过滤器信息。

命令格式

display ip as-path-filter [ as-path-filter-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

as-path-filter-number：指定的AS路径过滤器号。整数形式，取值范围为1～256。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示已配置AS路径过滤器。

<Quidway> display ip as-path-filter
ListID    Mode      Expression
1         permit    100,200
200       permit    500,600
