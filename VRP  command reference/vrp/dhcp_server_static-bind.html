dhcp server static-bind

命令功能

dhcp server static-bind命令用来配置当前接口的DHCP地址池中地址的静态绑定。

undo dhcp server static-bind命令用来删除配置。

缺省情况下，接口地址池中没有配置静态地址捆绑。

命令格式

dhcp server static-bind ip-address ip-address mac-address mac-address

undo dhcp server static-bind { ip-address ip-address | mac-address mac-address }

参数说明

ip-address ip-address：指定静态绑定的IP地址。必须是当前接口地址池中的合法IP地址。

mac-address mac-address：指定静态绑定的MAC地址。

使用指南

在一个接口的所有静态绑定中，IP地址和MAC地址必须是唯一的。

使用实例

# 将MAC地址为0000-e03f-0305与IP地址10.1.1.1进行静态绑定。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] dhcp server static-bind ip-address 10.1.1.1 mac-address 0000-e03f-0305
