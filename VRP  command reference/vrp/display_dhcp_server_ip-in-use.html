display dhcp server ip-in-use

命令功能

display dhcp server ip-in-use命令用来查看DHCP客户端的地址绑定信息，如硬件地址、IP地址、地址租用期限等信息。

命令格式

display dhcp server ip-in-use { all | interface [ interface-type interface-number ] | ip ip-address | pool [ pool-name ] | vlan vlan-id }

参数说明

all：表示查看所有的地址绑定信息。

interface [ interface-type interface-number ]：作用范围为指定的接口地址池，不指定接口名时显示所有接口地址池的地址绑定信息。

ip ip-address：指定IP地址，不指定该参数则显示所有地址的绑定信息。

pool [ pool-name ]：指定全局地址池。如果不指定地址池名，表示所有全局地址池。字符串形式，长度范围是1～35。

vlan vlan-id：表示范围为指定的VLAN的接口地址池，没指定参数vlan-id时指所有VLAN的接口地址池。

使用指南

如遇到客户端无法获取IP地址，查看地址池的使用情况，可以使用display dhcp server ip-in-use命令用来查看DHCP客户端的地址绑定信息。如果不指定任何可选参数，处理如下：

  • 在以太网接口视图下，范围为当前接口的地址池。

  • 在其他视图下，范围为所有的地址池。

使用实例

# 查看DHCP的地址绑定信息。

<Quidway> display dhcp server ip-in-use all
Global pool:
 IP address  Hardware address   Lease expiration      Type
 2.2.2.2     4444-4444-4444     NOT Used              Manual
Interface pool:
 IP address  Hardware address  Lease expiration          Type
 5.5.5.1     0050-ba28-930a    Jul  5 2006 13: 00:10 PM  Auto:COMMITED
