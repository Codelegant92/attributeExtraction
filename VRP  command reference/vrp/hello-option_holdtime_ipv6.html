hello-option holdtime（IPv6）

命令功能

hello-option holdtime命令用来配置路由器等待接收其PIM邻居发送Hello消息的超时时间。

undo hello-option holdtime命令用来恢复配置参数的缺省值。

缺省情况下，路由器等待接收其PIM邻居发送Hello消息的超时时间是105秒。

命令格式

hello-option holdtime interval

undo hello-option holdtime

参数说明

interval：表示路由器等待接收其PIM邻居发送Hello消息的超时时间。整数形式，取值范围是1～65535，单位是秒。

2：配置级

使用指南

如果在定时器超时后，路由器没有接收到邻居发送的Hello消息，就认为邻居无效或不可达。

此命令与接口视图下的pim ipv6 hello-option holdtime含义是相同的，只是视图不同。缺省情况下，如果接口上没有配置，取IPv6 PIM视图下配置的值；否则取接口视图下配置的值。

请参见IPv4 PIM配置命令。

使用实例

# 在公网实例中，配置路由器等待PIM邻居的Hello消息的超时时间为120秒。

<Quidway> system-view
[Quidway] pim-ipv6
[Quidway-pim6] hello-option holdtime 120
