display authorization-scheme

命令功能

display authorization-scheme命令用来查看授权方案的配置情况。

命令格式

display authorization-scheme [ authorization-scheme-name ]

参数说明

authorization-scheme-name：授权方案名称。字符串形式，长度范围是1～32，不区分大小写。

视图

所有视图

缺省级别

1：监控级

使用指南

如果在授权方案视图下执行此命令，或者指定了授权方案的名称，则将显示详细的授权方案配置信息；其他情况下只显示摘要信息。

使用实例

# 查看所有授权方案的摘要信息。

<Quidway> display authorization-scheme
  ------------------------------------------------------------
  Authorization-scheme-name          Authorization-method
  ------------------------------------------------------------
  default                            Local
  scheme1                            Local
  ------------------------------------------------------------
  Total 2,2 printed

# 查看系统缺省的授权方案的详细配置信息。

项目

描述

Authorization-scheme-name

授权方案的名称

Authorization-method
此授权方案配置的授权模式，包括：

  • 直接授权

  • 本地授权

  • HWTACACS授权

  • if-authenticated授权

  • RADIUS认证成功后授权

Authorization-cmd level

是否为为该级别的用户配置了按命令行授权

Authorization-cmd no-response-policy
按命令行授权失败时的策略：

  • Online：允许用户继续在线

  • Offline：使用户下线
