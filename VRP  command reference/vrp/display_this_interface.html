display this interface

命令功能

display this interface命令用来显示当前接口视图下的接口信息。

命令格式

display this interface [ | { begin | exclude | include } regular-expression ]

参数说明

|：使用正则表达式过滤输出的显示信息。

begin：显示以包含指定正则表达式的行开始的配置信息。

exclude：显示不包含指定正则表达式的行的配置信息。

include：显示包含指定正则表达式的行的配置信息。

regular-expression：正则表达式。字符串形式，支持空格，区分大小写。

视图

接口视图

缺省级别

2：配置级

使用指南

当在接口视图下，如果希望快速看到该接口的状态信息或报文统计信息，可以使用该命令。

display this interface命令的缺省级别虽然为监控级，但是该命令必须在接口视图下执行，因此要执行该命令，用户级别应该为配置级或更高级别。

项目

描述

Pos1/0/0 current state
出接口的物理链路状态。

  • UP：接口处于正常启动的状态。

  • DOWN：接口在物理上出现故障。

  • Administratively down：如果网络管理员在该接口执行shutdown（接口视图）命令，将显示状态为Administratively down。

Line protocol current state
链路层协议状态。

  • UP：接口的链路协议处于正常的启动状态。

  • UP(BFD status down)：与接口绑定的BFD会话状态变为down。

  • UP(Main BFD status down)：与主接口绑定的BFD会话状态down，并且联动子接口状态。只能在子接口显示该状态。

  • DOWN：接口的链路协议不正常，或者没有在此接口配置IP地址。其中damping表示该接口因为频繁震荡而被抑制。

Last line protocol up time:

接口的协议状态最近一次变为Up的时间。

Description

接口的描述。支持空格，区分大小写，最多可输入242个字符。

The Maximum Transmit Unit

接口的MTU值。

Internet Address

接口的IP地址。

Hold timer

接口的轮询时间。

Link layer protocol

链路层运行的协议。

LCP

LCP协商状态。

Physical layer is POS over STM-1

传输封装类型：SONET或SDH。

Scramble

是否使能加扰：disabled或enabled。

CRC

CRC校验字长度。

clock

时钟工作模式：slave或master。

loopback

环回功能模式：内部自环、外部环回或不使能。

Hardware

POS接口的类型。有155M、2.5G和10G等类型。

section layer:

段。B1表示进行比特间插奇偶校验后错误的报文，其检测是对整个STM-N帧信号进行传输误码检测。

line layer:

复用。

  • B2表示比特间插奇偶校验N×24位的（BIP-N×24）字节，其检测是对STM-N帧中的每一个STM-1帧的传输误码情况进行监测。
  • REI（Remote Error Indication）表示线路远端错误指示。

path layer:

通道。B3：通道BIP-8，B3字节负责监测VC4在STM-N帧中传输的误码性能，也就是监测140Mbit/s的信号在STM—N帧中传输的误码性能。监测机理与B1、B2相类似，不过B3是对VC4帧进行BIP-8校验。
