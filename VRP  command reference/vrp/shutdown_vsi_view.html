shutdown（VSI视图）

命令功能

shutdown命令用于关闭VSI实例。

undo shutdown命令用于启动VSI实例。

缺省情况下，VSI实例是处于启动状态的。

命令格式

shutdown

undo shutdown

参数说明

无

视图

VSI视图

缺省级别

2：配置级

使用指南

出于业务管理的需求，例如业务调试、业务暂停等，可以暂时禁止VSI，对VSI的功能进行增加、删除或调整。

对VSI执行shutdown操作主要影响PW连接，AC将被置为管理Down的状态，L2转发表被删除。

使用实例

# 关闭当前VSI实例。

<Quidway> system-view
[Quidway] vsi company1
[Quidway-vsi-company1] shutdown

# 启动当前VSI实例。

<Quidway> system-view
[Quidway] vsi company1
[Quidway-vsi-company1] undo shutdown
