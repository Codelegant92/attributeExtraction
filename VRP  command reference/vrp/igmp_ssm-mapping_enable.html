igmp ssm-mapping enable

命令功能

igmp ssm-mapping enable命令用来在接口上使能SSM Mapping。

undo igmp ssm-mapping enable命令用来恢复缺省配置。

缺省情况下，接口未使能SSM Mapping。

命令格式

igmp ssm-mapping enable

undo igmp ssm-mapping enable

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

只有在接口上使能SSM Mapping，配置的SSM源/组地址映射表项才能生效。

使用实例

# 在接口GE1/0/0上，使能SSM Mapping。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] igmp ssm-mapping enable
