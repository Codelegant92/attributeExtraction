项目

描述

Feature name

模块名称。

Trap number

告警事件的类型数。

Trap name
事件类型，有以下类型：
  • IFLIMIT：接口上IGMP表项数量达到限制。
  • INSLIMIT：单实例上IGMP表项数量达到限制。
  • GLBLIMIT：所有实例的总体IGMP表项数量达到限制。
  • JOINGRP：收到IGMP加入报文。
  • LEAVEGRP：收到IGMP离开报文或组定时器超时。

Status
告警开关状态，有以下两种：
  • on：告警开关打开。
  • off：告警开关关闭。

Excessive Flag
海量信息标识，有以下两种取值：
  • Y：表示当前类型的信息是海量信息。
  • N：表示当前类型的信息不是海量信息。
