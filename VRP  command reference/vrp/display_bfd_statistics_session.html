display bfd statistics session

命令功能

display bfd statistics session命令用来查看BFD会话的统计信息。

命令格式

display bfd statistics session { { all | static } [ for-ip | for-lsp | for-pw | for-te| for-vsi-pw ] [ slot slot-id ] | vsi-pw vsi vsi-name peer peer-address [ vc-id vc-id ] }

display bfd statistics session { dynamic | discriminator discr-value | peer-ip peer-ip [ vpn-instance vpn-name ] | static-auto } [ slot slot-id ]

display bfd statistics session { ldp-lsp peer-ip peer-ip [ interface interface-type interface-number ] | static-lsp lsp-name | mpls-te interface interface-type interface-number[ te-lsp ] | pw interface interface-type interface-number [ secondary ] }

参数说明

all：显示所有BFD会话统计信息。

static：显示静态BFD会话统计信息。

dynamic：显示动态BFD会话统计信息。

discriminator discr-value：显示指定本地标识符的BFD会话统计信息。

peer-ip peer-ip：显示与指定IP地址间的BFD会话统计信息。

vpn-instance vpn-instance-name：显示绑定到指定VPN实例的BFD会话统计信息。VPN实例名称是字符串形式，长度范围是1～31。

slot slot-id：显示指定接口板上的BFD会话统计信息。

for-ip：显示检测IP链路的BFD会话统计信息。

for-pw：显示检测PW链路的BFD会话统计信息。

for-lsp：显示检测LSP链路的BFD会话统计信息。

for–te：显示检测TE隧道的BFD会话统计信息。

for-vsi-pw：显示检测VSI PW的BFD会话统计信息。

vsi-name：VSI实例的名称，不区分大小写，字符串类型，长度范围是1～31。

peer peer-address：VSI PW对端的IPv4地址，点分十进制。

vc-id vc-id：虚电路的唯一标识，一般用于两端VSI ID不同但要求互通的情况。参数vc-id不能与本端其他VSI配置的VSI ID相同，也不能与本端其他negotiate-vc-id指定的VSI ID相同，即vc-id是没有被用过的虚电路号，十进制整数形式，取值范围为1～4294967295。

ldp-lsp：显示检测LDP LSP的BFD会话统计信息。

static-lsp：显示检测静态LSP的BFD会话统计信息。

lsp-name：静态LSP的名称。

mpls-te：显示检测MPLS TE的BFD会话统计信息。

te-lsp：显示检测与TE绑定的主LSP的会话统计信息。

pw：显示检测PW的BFD会话统计信息。

secondary：显示检测备PW的BFD会话统计信息。

使用指南

在标识符自协商静态会话和动态会话共用时，根据BFD会话类型（静态、动态）进行查询时，共用会话既属于静态会话，也属于动态会话。

项目

描述

Session MIndex

会话表项的索引

State

会话的当前状态

Name

配置的名称

Session Type
BFD会话类型

  • Static：静态配置建立的BFD会话。

  • Dynamic：动态触发建立的BFD会话。

  • Entire_Dynamic：完全动态触发建立的BFD会话，在LSP的宿端路由器上使能被动动态创建BFD会话功能后，将创建该会话类型。

  • Static_Auto：静态建立的标识符自协商BFD会话。

Bind Type

BFD的绑定类型

Local/Remote Discriminator

本地标识符和远端标识符

Received Packets

本端接收到的BFD控制报文数

该字段可以通过命令reset bfd statistics清空

Send Packets

本端发送的BFD控制报文数该字段可以通过命令reset bfd statistics清空

Received Bad Packets

收到的错误报文个数

Send Bad Packets

报文发送失败的个数

Down Count

会话Down的次数

ShortBreak Count

会话瞬断次数

Send Lsp Ping Count

发送的LSP Ping报文个数

Dynamic Session Delete Count

删除的动态会话个数

Create Time

会话创建的时间

Last Down Time

会话上一次Down的时间

Total Time From Last Down

会话从上一次Up到现在的运行时间

Total Time From Create

会话从创建到现在的运行时间

Total Session Number

BFD会话总数
