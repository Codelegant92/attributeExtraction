display arp-limit

命令功能

display arp-limit命令用来查询接口可以学习ARP数目的最大值。

命令格式

display arp-limit [ interface interface-type interface-number ] [ vlan vlan-id ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type：接口类型。

interface-number：接口编号。

vlan-id：VLAN ID，取值范围是1～4094。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display arp-limit命令查询接口学习的ARP数目时，二层接口和三层接口都可以显示配置的ARP学习的最大数目和主控板已经学习到的ARP数目，其余接口只能显示配置的ARP学习的最大数目。

使用实例

# 查询整机配置限制数目。

<Quidway> display arp-limit
interface               LimitNum        VlanID          LearnedNum(Mainboard)
---------------------------------------------------------------------------
 Eth-Trunk0              100             124                0
 Eth-Trunk0              100             125                0
 GigabitEthernet2/0/1    16384           0                  0
 GigabitEthernet4/0/1    100             0                  0
 GigabitEthernet4/0/2    16384           124                0
