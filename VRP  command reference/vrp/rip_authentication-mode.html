rip authentication-mode

命令功能

rip authentication-mode命令用来配置RIP-2的验证方式及验证参数。每次验证只支持一个验证字。新输入的验证字将覆盖旧验证字。

undo rip authentication-mode命令用来取消所有验证。

命令格式

rip authentication-mode { simple password | md5 { nonstandard password-key key-id | usual password-key } }

undo rip authentication-mode

参数说明

simple：使用明文验证方式。

password：明文验证关键字。字符串形式，不支持空格，区分大小写，长度范围是1～16。

md5：使用MD5密文验证方式。

nonstandard：指定MD5密文验证报文使用非标准报文格式（私有标准）。

usual：指定MD5密文验证报文使用通用报文格式（IETF标准）。

password-key：密文验证关键字，字符串形式，不支持空格，区分大小写。其中，长度为1～16的字符串对应明文；长度为24的字符串对应密文。

key-id：MD5密文验证标识符。整数形式，取值范围是1～255。

使用指南

验证字中不允许包含空格。

使用实例

# 设置通用格式的MD5认证，认证字为abc。

<Quidway> system-view
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] rip authentication-mode md5 usual abc
