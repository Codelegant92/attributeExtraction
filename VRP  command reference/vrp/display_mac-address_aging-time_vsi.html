display mac-address aging-time vsi

命令功能

display mac-address aging-time vsi命令用来查看VSI中MAC地址表项老化时间。

命令格式

display mac-address aging-time vsi [ name vsi-name ]

参数说明

name vsi-name：指定VSI的名字。字符串形式，区分大小写，长度范围是1~31。

视图

所有视图

缺省级别

1：监控级

使用指南

老化时间为0时，表示MAC地址表项不被老化。

使用实例

# 查看所有VSI中MAC地址表项老化时间。

<Quidway> display mac-address aging-time vsi
  VSI              AgingTime(sec)
  company1         200
  company2         300 
  company3         400
