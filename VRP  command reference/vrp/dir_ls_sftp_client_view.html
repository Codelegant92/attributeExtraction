dir/ls（SFTP客户端视图）

命令功能

dir / ls命令用来显示指定SFTP服务器目录下的文件列表。

命令格式

dir/ls [ -l | -a ] [ path ]

参数说明

-l：显示当前目录下的所有文件列表。

-a：显示当前目录下的所有文件名。

path：SFTP服务器上的目录名，字符串形式，绝对路径名长度范围为1～64。如果path没有指定，默认情况下显示当前目录下的文件列表。

使用指南

如果dir/ls [ path ]命令执行失败，系统将提示不能显示的错误信息。
