display bgp multicast network

命令功能

display bgp multicast network命令用来查看MBGP计划通过network命令发布的路由信息。

命令格式

display bgp multicast network [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

该命令查询的是IPv4组播地址族视图下所有的network配置，只有通过network命令后面指定，路由前缀在IP路由表中存在且活跃，才会被MBGP引入并往外发布。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示MBGP发布的网段路由信息。

<Quidway> display bgp multicast network
