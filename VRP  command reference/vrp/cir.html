cir

命令功能

cir命令用来设置帧中继虚电路的最小承诺信息速率（Minimum Committed Information Rate）。

undo cir命令用来恢复缺省值。

命令格式

cir rate-limit

undo cir

参数说明

rate-limit：最小承诺信息速率，单位是bit/s，取值范围是1000～45000000。缺省值是56000bit/s。

说明：

cir命令仅在出方向有效，即仅在流量整形时有效。

视图

FR-Class视图

缺省级别

2：配置级

使用指南

最小承诺信息速率是虚电路所能提供的最低发送速率，它保证了用户在网络拥塞时仍然可以以此速率发送数据。

当网络发生拥塞时，DCE将向DTE发送BECN标志位为1的报文。DTE接收到这个报文后，会将虚电路的发送速率由CIR逐渐调低到MinCIR；如果DTE在一定时间内没有再收到BECN标志位为1的报文，它会将虚电路的发送速率恢复为CIR。

配置时，最小承诺信息速率（MinCIR）不应大于承诺信息速率（CIR）。

使用实例

# 设置名为test1的帧中继class的最小承诺信息速率为32000bit/s。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] cir 32000
