x25 vc-range

命令功能

x25 vc-range命令用来设置X.25虚电路范围的上限及下限。

undo x25 vc-range命令用来恢复X.25虚电路范围的上限及下限的缺省值。

命令格式

x25 vc-range [ in-channel lic hic ] [ bi-channel ltc htc ] [ out-channel loc hoc ]

undo x25 vc-range

参数说明

lic hic：表示X.25虚电路单向呼入信道区间的下限及上限。取值范围是0～4095。如果hic（即单向呼入信道区间的上限）设置为0，那么必须将lic（即单向呼入信道区间的下限）也设置为0，表示禁止使用单向呼入信道区间。X.25虚电路范围中单向呼入信道区间的下限和上限缺省值均为0。

ltc htc：表示X.25虚电路双向信道区间的下限及上限。取值范围是0～4095；如果将htc（即双向信道区间的上限）设置为0，那么必须将ltc（即双向信道区间的下限）也设置为0，表示禁止使用双向信道区间。X.25虚电路双向信道区间的下限缺省值是1，上限缺省值是1024。

loc hoc：表示X.25虚电路单向呼出信道区间的下限及上限。取值范围是0～4095。如果将hoc（即单向呼出信道区间的上限）设置为0，那么必须将loc（即单向呼出信道区间的下限）也设置为0，表示禁止使用单向呼出信道区间。X.25虚电路范围中单向呼出信道区间的下限和上限缺省值均是0。

使用指南

在VRP中，X.25缺省地将单向呼入信道区间和单向呼出信道区间禁止使用，只保留双向信道区间（从1到1024）供使用；请根据接入商的要求正确地设置虚电路范围。

执行完该命令后，必须重新启动接口，使配置生效。

使用实例

# 配置接口Serial 1/0/0的链路层协议为X.25，并且启用虚电路范围中的单向呼入信道区间和双向信道区间，禁用单向呼出信道区间；执行完该命令序列后，这三个区间分别是[1，7]、[8，1024]和[0，0]。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0]link-protocol x25
[Quidway-Serial1/0/0] x25 vc-range in-channel 1 7 bi-channel 8 1024
[Quidway-Serial1/0/0] shutdown
[Quidway-Serial1/0/0] undo shutdown
