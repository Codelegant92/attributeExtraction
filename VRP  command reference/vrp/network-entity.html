network-entity

命令功能

network-entity命令用来设置IS-IS路由进程的网络实体名称NET。

undo network-entity命令用来删除一个NET。

缺省情况下，没有配置NET。

命令格式

network-entity net

undo network-entity net

参数说明

net：指定网络实体名称（Network Entity Title，NET），格式为X…X.XXXX.XXXX.XXXX.00，前面的“X…X”是区域地址，中间的12个“X”是路由器的System ID，最后的“00”是SEL。

视图

IS-IS视图

缺省级别

2：配置级

使用指南

NET也就是NSAP（Network Service Access Point）的特殊形式，IS-IS的NET为8～20字节。

NET由三部分组成：

  • 区域ID，区域ID的长度可以是变化的（1～13个字节），同一个区域内的路由器区域ID相同。

  • 本路由器的系统ID（6个字节），在整个区域和骨干区域中系统ID要求保持唯一。

  • 最后一个字节SEL，其值必须为00。

通常一台路由器配置一个NET即可，当区域需要重新划分例如合并区域或者分拆区域时，默认情况最多可配置3个区域，由此在区域更改时仍然能够保证路由的正确性和连续性。

使用实例

# 指定NET为10.0001.1010.1020.1030.00。其中系统ID是1010.1020.1030，区域ID是10.0001。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] network-entity 10.0001.1010.1020.1030.00
