ipv6 nd ra halt

命令功能

ipv6 nd ra halt命令用来抑制RA报文的发布。

undo ipv6 nd ra halt命令用来使能RA报文的发布。

缺省情况下，抑制RA报文的发布。

命令格式

ipv6 nd ra halt

undo ipv6 nd ra halt

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 抑制在GE接口上发布RA报文。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 nd ra halt
