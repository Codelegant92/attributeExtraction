group mode

命令功能

group mode命令用来配置用户工作组的模式。

undo group mode命令用来恢复用户工作组的缺省模式。

缺省情况下，用户工作组为single模式。

命令格式

group mode { single | multiple }

undo group mode

参数说明

single：此VLAN Group的使用者被当作一个用户。

multiple：此VLAN Group的使用者被当作多个用户。

视图

VLAN-Group接口视图

缺省级别

2：配置级

使用指南

当某个VLAN Group已经被使用时，则此时不能通过group mode命令来改变该VLAN Group用户工作组的模式。

使用实例

# 配置子接口GE1/0/0.1下vlan-group 1为multiple模式：

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] vlan-group 1
[Quidway-GigabitEthernet1/0/0.1-vlangroup1] group mode multiple
