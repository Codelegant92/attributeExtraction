reflector cluster-id (BGP)

命令功能

reflector cluster-id命令用来配置路由反射器的集群ID。

undo reflector cluster-id命令用来删除路由反射器的集群ID。

缺省情况下，每个路由反射器使用自己的路由器ID作为集群ID。

命令格式

reflector cluster-id cluster-id

undo reflector cluster-id

参数说明

cluster-id：路由反射器的集群ID，取值范围为1～4294967295，也可以用IPv4地址形式标识。

缺省级别

2：配置级

使用指南

在一个集群中设置多个路由反射器可提高网络的稳定性，此时需要使用该命令为所有的路由反射器配置相同的集群ID，以便标识这个集群，避免路由循环。

为了保证客户机可以学习到反射器发来的路由，集群ID不能和客户机的Router ID相同。如果相同，客户机会将收到的路由丢弃。

使用实例

# 本地路由器是集群中的路由反射器之一，设置它的集群ID为50。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] reflector cluster-id 50
