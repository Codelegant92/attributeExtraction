ip pool

命令功能

ip pool命令用来配置为PPP用户分配IP地址的本地地址池。

undo ip pool命令用来删除指定的本地地址池。

缺省情况下，系统或域没有配置本地IP地址池。

命令格式

ip pool pool-number start-address [ end-address ]

undo ip pool pool-number

参数说明

pool pool-number：指定地址池编号。整数形式，取值范围是0～99。

start-address：指定地址池的起始IP地址。

end-address：指定地址池的结束IP地址。

视图

AAA视图、AAA域视图

缺省级别

2：配置级

使用指南

ip pool命令用来在服务器端配置为客户端分配的IP地址，如在只有一个客户端的情况下，可以不配置地址池，而直接为客户端分配指定的IP地址。

说明：

  • 一个地址池中起始IP地址和结束IP地址之间的地址数目不能超过4096个。

  • 如果在定义IP地址池时不指定结束IP地址，则该地址池中只有一个IP地址，即起始IP地址。

使用实例

# 在AAA视图下配置本地IP地址池0，地址范围为129.102.0.1到129.102.0.10。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] ip pool 0 129.102.0.1 129.102.0.10

# 在域视图下配置本地IP地址池0，地址范围为129.102.0.1到129.102.0.10。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain d
[Quidway-aaa-domain-d] ip pool 0 129.102.0.1 129.102.0.10
