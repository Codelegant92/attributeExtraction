nqa-ftp-record password

命令功能

nqa-ftp-record password命令用来设置保存NQA测试结果的FTP服务器密码。

undo nqa-ftp-record password命令用来删除保存NQA测试结果的FTP服务器密码。

命令格式

nqa-ftp-record password password

undo nqa-ftp-record password

参数说明

password：保存NQA测试结果的FTP服务器密码。字符串形式，长度范围是1～32。

视图

系统视图

缺省级别

3：管理级

使用指南

配置的密码必须和nqa-ftp-record username命令配置的用户名相对应。

使用实例

# 配置登录FTP服务器的密码值为test。

<Quidway> system-view
[Quidway] nqa-ftp-record password test
