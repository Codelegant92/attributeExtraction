service cbr

命令功能

service cbr命令用来指定PVC的业务类型为确定速率（Constant Bit Rate，CBR）。

缺省情况下，PVC创建后的业务类型为UBR。

命令格式

service cbr output-pcr [ cdvt cdvt-value ]

参数说明

output-pcr：输出ATM信元的峰值速率。取值范围是64～149760，单位是kbit/s。

cdvt-value：信元时延变化容限。取值范围是0～10000，单位是μs。

视图

ATM-PVC视图

缺省级别

2：配置级

使用指南

如果将PVC的业务类型设置为CBR，不指定cdvt的值时缺省为500μs。

可以使用本命令及service ubr、service vbr-nrt和service vbr-rt设置PVC的业务类型和业务参数。新指定的PVC业务类型将会覆盖已有的业务类型。建议在创建cbr带宽时先创建大带宽的PVC，再创建小带宽的PVC；若创建不成功，可将cdvt-value的值调大，再试着创建；此情况已在命令行中给出提示，如下：

“fail to set service parameter，please adjust cdvt value”

ATM E1接口、ATM T1接口、ATM E3接口和ATM T3接口不支持业务类型CBR。

使用实例

# 创建一条名为“huawei”、VPI/VCI为1/101的PVC。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvc huawei 1/101

# 指定该PVC的业务类型为cbr，ATM信元峰值发送速率为50,000kbit/s。信元时延变化容限为1000μs。

[Quidway-atm-pvc-Atm1/0/0-1/101-huawei] service cbr 50000 cdvt 1000
