display ip vpn-instance

命令功能

display ip vpn-instance命令用来查看VPN实例信息。

命令格式

display ip vpn-instance [ verbose ] [ vpn-instance-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

verbose：显示VPN实例和关联接口的详细信息。

vpn-instance-name：VPN实例的名称。区分大小写。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

完成VPN实例的配置时，可用display ip vpn-instance命令检查VPN实例的配置情况。如果不指定可选参数，此命令将显示配置的所有VPN实例的简要信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看所有VPN实例的信息。

<Quidway> display ip vpn-instance
  Total VPN-Instances configured : 1
  VPN-Instance Name               RD                Creation Time
  vpna                            200:1             2006/03/06 11:26:06
表1 display ip vpn-instance命令输出信息描述

项目

描述

Total VPN-Instances configured

表示本端配置的VPN实例总数。

VPN-Instance Name

表示VPN实例的名称。

RD

表示该VPN实例的路由标志符。

Creation Time

显示创建该VPN实例的具体时间。

表2 display ip vpn-instance verbose命令输出信息描述

项目

描述

Total VPN-Instances configured

表示本端配置的VPN实例总数。

VPN-Instance Name and ID

表示VPN实例的名称和标识号。标识号是系统分配的。

Create date

显示创建该VPN实例的具体时间。

Up time

该VPN实例处于可用状态的持续时间。

Route Distinguisher

表示该VPN实例的路由标志符。

Export VPN Targets

出方向Route Target列表。

Import VPN Targets

入方向Route Target列表。

Label policy
标签策略：

  • label per route表示给为每一条路由分配一个标签。

  • label per instance表示为每个VPN实例分配一个标签。

Import Route Policy

应用于该VPN实例的入方向路由策略。只有在VPN实例下配置import route-policy（VPN实例视图）命令才有该项显示信息。

Export Route Policy

应用于该VPN实例的出方向路由策略。只有在VPN实例下配置export route-policy（VPN实例视图）命令才有该项显示信息。

The diffserv-mode Information is

差分服务模式，对带宽等资源进行实时分配。有uniform和pipe两种模式。

The ttl-mode Information is

TTL处理模式。有uniform和pipe两种模式。

Tunnel Policy

应用于该VPN实例的隧道策略。只有在VPN实例下配置tnl-policy（VPN实例视图）命令才有该项显示信息。

Description

VPN实例的描述信息。只有在VPN实例下配置description（VPN实例视图）命令才有该项显示信息。

Maximum Routes Limit

当前VPN实例支持的最大路由数目。只有在VPN实例下配置routing-table limit（VPN实例视图）命令才有该项显示信息。

Interfaces

表示绑定了该VPN实例的接口。

只有配置了ip binding vpn-instance命令才有该显示项。
