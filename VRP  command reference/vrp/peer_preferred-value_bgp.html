peer preferred-value (BGP)

命令功能

peer preferred-value命令用来为对等体分配首选值。

undo peer preferred-value命令用来删除分配给对等体的首选值。

缺省情况下，从其它BGP对等体学来的路由的首选值为0。

命令格式

peer { group-name | ipv4-address | ipv6-address } preferred-value value

undo peer { group-name | ipv4-address | ipv6-address } preferred-value

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图、BGP-IPv4单播地址族视图、BGP-IPv6单播地址族视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP-IPv6单播地址族视图和BGP-VPN6实例视图下生效。

value：要分配的路由首选值。整数形式，取值范围是0～65535。

视图

BGP视图、BGP-IPv4单播地址族视图、BGP-VPN实例视图、BGP-IPv6单播地址族视图、BGP-VPN6实例视图

缺省级别

2：配置级

使用指南

通过为Peer配置一个指定的首选值，所有从该Peer学来的路由都具有这个首选值属性。当到达同一地址前缀有多条路由时，优先选择首选值大的路由。

使用实例

# 为对等体设置首选值。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] peer 1.1.1.2 preferred-value 50

# 为对等体设置首选值。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv6-family unicast
[Quidway-bgp-af-ipv6] peer 1:2::3:4 preferred-value 50
