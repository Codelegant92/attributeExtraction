peer upe

命令功能

peer upe命令用来指定BGP对等体或对等体组作为HoVPN的UPE。

undo peer upe命令用来取消该配置。

命令格式

peer { group-name | ipv4-address } upe

undo peer { group-name | ipv4-address } upe

参数说明

ipv4-address：对等体的IPv4地址。

group-name：对等体组的名称。

视图

BGP-VPNv4地址族视图

缺省级别

2：配置级

使用指南

UPE是一类特殊的VPNv4对等体，它不接收普通的VPNv4路由，只接收SPE上每个相关VPN实例的一条缺省路由。SPE是普通的VPN对等体。

注意：

执行peer upe命令会导致路由器之间的BGP Peer关系中断。务必仔细确认是否必须执行该操作。

使用实例

# 指定对等体1.1.1.2作为UPE。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 100
[Quidway-bgp] ipv4-family vpnv4
[Quidway-bgp-af-vpnv4] peer 1.1.1.2 enable
[Quidway-bgp-af-vpnv4] peer 1.1.1.2 upe
