display cops-server configuration

命令功能

display cops-server configuration命令用于显示COPS服务器组配置信息，包括COPS服务器组索引、名称、客户端类型、Up/Down状态、激活状态、密钥、流状态保持时间、策略客户端编号等信息。

命令格式

display cops-server configuration [ group group-name ]

参数说明

group：显示指定的COPS服务器组配置。

group-name：COPS服务器组名。字符串形式，长度范围是1～32。必须是已存在的COPS服务器组。
