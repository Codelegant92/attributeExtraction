reset counters interface ip-trunk

命令功能

reset counters interface ip-trunk命令用来清除IP-Trunk接口的统计信息。

命令格式

reset counters interface ip-trunk [ trunk-id ]

参数说明

trunk-id：端口聚合的Trunk ID，取值范围是0～127。

视图

用户视图

缺省级别

2：配置级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

reset counters interface ip-trunk命令清除的是display interface ip-trunk显示信息的最后一部分：接口输入输出报文的统计信息。

使用实例

# 清除接口IP-Trunk1的统计信息。

<Quidway> reset counters interface ip-trunk 1
