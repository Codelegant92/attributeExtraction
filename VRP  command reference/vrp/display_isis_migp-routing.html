display isis migp-routing

命令功能

display isis migp-routing命令用来显示IS-IS的MIGP（Multicast IGP）路由信息。

命令格式

display isis [ process-id ] migp-routing [ ip-address [ mask | mask-length ] ] [ level-1 | level-2 ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。

ip-address：显示指定目的地址的路由信息。

mask：掩码，点分十进制格式。

mask-length：掩码长度。

level-1：显示Level-1的IS-IS路由。

level-2：显示Level-2的IS-IS路由。

verbose：使用此参数后，将显示路由的管理标签。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过此命令可以查看出接口为TE-Tunnel的路由的实际物理接口。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
