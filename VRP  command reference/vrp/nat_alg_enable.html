nat alg enable

命令功能

nat alg enable命令用来使能各种应用协议的NAT ALG功能。

undo nat alg enable关闭各种应用协议的NAT ALG功能。

命令格式

nat alg enable { dns | ftp | h323 | hwcc | icmp | ils | msn | netbios | pptp | qq }

undo nat alg enable { dns | ftp | h323 | hwcc | icmp | ils | msn | netbios | pptp | qq }

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

在使能某应用协议的NAT ALG功能后，该应用协议可以正常穿越NAT，否则该应用协议不能正常工作。

使用实例

# 使能FTP协议的NAT ALG功能。

<Quidway> system-view
[Quidway] nat alg enable ftp

# 关闭FTP协议的NAT ALG功能。

[Quidway] undo nat alg enable ftp
