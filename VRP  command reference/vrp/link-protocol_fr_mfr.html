link-protocol fr mfr

命令功能

link-protocol fr mfr命令用来将当前物理接口配置为一个多链路帧中继捆绑链路（bundle link），并将其捆绑到指定的MFR接口。

缺省情况下，没有多链路帧中继捆绑链路。

命令格式

link-protocol fr mfr interface-number

参数说明

interface-number：接口编号，三维形式：槽号/卡号/接口号。

视图

帧中继接口视图

缺省级别

2：配置级

使用指南

配置此命令时，指定的MFR接口必须存在。一个MFR接口最多可以捆绑16个物理接口。

如果要将物理接口从MFR接口中删除，可以使用命令link-protocol对此接口使用非Frame Relay MFR的链路层协议。

使用实例

# 将当前串口1/0/0:0配置为捆绑链路，并将其加入帧中继捆绑接口mfr1/0/123。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr mfr 1/0/123
[Quidway-Serial1/0/0:0] restart
