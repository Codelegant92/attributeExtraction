ip userlog export version

命令功能

ip userlog export version命令用来配置用户日志统计输出报文的版本号。

undo ip userlog export version命令用来恢复缺省设置。

命令格式

ip userlog nat export version version-number

undo ip userlog nat export version

参数说明

nat：NAT日志功能。

version-number：设置用户日志报文的版本，缺省值是1。

视图

系统视图

缺省级别

2：配置级

使用指南

目前系统仅支持版本1。

使用实例

# 将NAT日志报文版本号设为版本1。

<Quidway> system-view
[Quidway] ip userlog nat export version 1
