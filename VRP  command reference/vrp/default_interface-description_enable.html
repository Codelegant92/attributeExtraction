default interface-description enable

命令功能

default interface-description enable命令用来使能系统显示接口默认描述信息的功能。

undo default interface-description enable命令用来取消系统显示接口默认描述信息的功能。

缺省情况下，系统不显示接口默认描述信息。

命令格式

default interface-description enable

undo default interface-description enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

执行default interface-description enable后，通过display interface description命令查看接口描述信息时：

  • 如果用户没有配置接口描述信息，会显示接口默认描述信息。

  • 如果用户配置了接口描述信息，则会显示用户配置的接口描述信息。

执行undo default interface-description enable后，查看接口描述信息时，不再显示接口默认描述信息。

使用实例

# 使能系统显示接口默认描述信息功能。

<Quidway> system-view
[Quidway] default interface-description enable
