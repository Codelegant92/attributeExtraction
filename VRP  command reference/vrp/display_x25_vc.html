display x25 vc

命令功能

display x25 vc命令用来显示X.25虚电路。

命令格式

display x25 vc [ vcn ]

参数说明

vcn：虚电路号（Virtual circuit number）。取值范围是1～4095，如果不指定虚电路号，则显示全部虚电路。

所有视图

缺省级别

1：监控级

使用指南

SVC是在有数据传输要求时，由X.25经过呼叫临时建立的；PVC是手工配置的，不论是否有数据传输要求而总是存在的。

当路由器工作在X.25交换方式时，为了传输被交换数据，也会有虚电路被建立。这几种虚电路都可以通过使用此命令来查看信息，只是在显示信息的某些域有所区别。

表1 display x25 vc命令输出信息描述

项目

描述

Interface:

有虚电路的接口类型和接口编号。

SVC 1

虚电路的类型和虚电路号。

State:

状态。

Map:

映射的IP地址和虚电路号。

Window size:

窗口大小。

Packet Size:

报文大小。

Local PS: 5 Local PR: 5 Remote PS: 5 Remote PR: 4

本地和远端的发送和接收（PS和PR）值。

Local Busy: FALSE Reset times: 0

本地不忙，恢复次数为0。

SVC <--> Serial2/0/0 SVC 60

虚电路交换的远端接口和虚电路号。

XOT PVC <--> Serial2/0/0 PVC 1 connected

XOT连接的远端接口号和虚电路号及状态。
