display qos cbq interface

命令功能

display qos cbq interface命令用来显示指定接口基于类的队列配置信息和运行情况。

命令格式

display qos cbq interface [ interface-type interface-number [ pvc { pvc-name [ vpi/vci ] | vpi/vci } ] ]

display qos cbq interface [ interface-type interface-number [ dlci dlcinumber ] ]

参数说明

interface-type：接口类型。

interface-number：接口号。

pvc：只用于ATM接口，即可显示指定ATM接口上的指定pvc的策略配置。

pvc-name：PVC名。

vpi/vci：VPI/VCI值对。

dlci：只用于FR接口。

dlcinumber：DLCI的编号。

使用指南

display qos cbq interface命令可以观察到的信息包括：

  • EF队列信息

  • AF队列信息

  • BE队列信息

  • 带宽信息

使用实例

<Quidway> display qos cbq interface GigabitEthernet 10/2/0
Interface: GigabitEthernet10/2/0
Output queue : (Protocol queue : Size/Length/Discards) 0/1000/0
Output queue : (Class Based Queuing : Size/Discards)  0/0

  BE Queues:   0/0/256 (Active/Max active/Total)
  AF Queues:   1 (Allocated)
Bandwidth(Kbps): 74992/75000 (Available/Max reserve)
TE Qos not enabled
