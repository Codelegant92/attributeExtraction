vrrp vrid priority

命令功能

vrrp vrid priority命令用来设置路由器在备份组中的优先级。

undo vrrp vrid priority命令用来将优先级恢复为缺省值。

缺省情况下，优先级值为100。

命令格式

vrrp vrid virtual-router-id priority priority-value

undo vrrp vrid virtual-router-id priority

参数说明

vrid virtual-router-id：VRRP备份组号，备份组号范围1～255。

priority priority-value：优先级取值，取值范围1～254。

使用指南

优先级决定路由器在备份组中的地位，优先级越高，越有可能成为Master。优先级0由系统保留作为特殊用途，255由系统保留给IP地址拥有者使用。

说明：

IP地址拥有者的优先级不可配置，始终为255。

使用实例

# 设置路由器在备份组1中的优先级为150。

[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] vrrp vrid 1 priority 150
