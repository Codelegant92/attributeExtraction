copy（多框）

命令功能

copy命令用来复制文件。

命令格式

copy source-filename destination-filename [ all ]

参数说明

source-filename：源文件名。字符串形式，格式为[ path ][ file-name ],取值范围是1～64。

destination-filename：目标文件名。字符串形式，取值范围是1～64。

all：拷贝文件到所有框的主板中。

视图

用户视图

缺省级别

3：管理级

使用指南

若目标文件名与已经存在的文件名重名，将提示是否覆盖。若目标文件名是目录名，则目标文件与源文件同名。

使用实例

# 将文件sample.txt从中央交换框拷贝到转发框。

<Quidway> copy sample.txt clc1/17#cfcard:
Copy cfcard:/sample.txt to clc1/17#c:/cfcard.txt?[Y/N]:y
