ntp-service broadcast-server

命令功能

ntp-service broadcast-server命令用来配置NTP广播服务器模式。

undo ntp-service broadcast-server命令用来取消NTP广播服务器模式。

缺省情况下，不配置广播服务。

命令格式

ntp-service broadcast-server [ authentication-keyid key-id | version number ]*

undo ntp-service broadcast-server

参数说明

authentication-keyid：定义身份验证密钥。

key-id：向广播客户端发送消息时使用的密钥ID号，取值范围1～4294967295。

version number：NTP版本号，取值范围是1～3。缺省值是3。

缺省级别

2：配置级

使用指南

指定本地设备上的一个接口来发送NTP广播消息包，本地运行在服务器（broadcast-server）模式，作为广播服务器周期性地发送广播消息到广播客户端。

使用实例

# 在GE1/0/0接口上发送NTP广播消息包，用4号密钥加密，NTP版本号为3。

<Quidway> system-view
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ntp-service broadcast-server authentication-keyid 4 version 3
