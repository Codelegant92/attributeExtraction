display this ipv6 interface

命令功能

display this ipv6 interface命令用来显示当前接口的IPv6信息。

命令格式

display this ipv6 interface [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display this ipv6 interface命令的缺省级别虽然为监控级，但是该命令必须执行在接口视图下，因此要执行该命令，用户级别应该为配置级或高于配置级。
