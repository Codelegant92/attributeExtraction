display ip policy-based-route

命令功能

display ip policy-based-route命令用来查看本地或接口已使能的策略路由的策略。

命令格式

display ip policy-based-route [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

policy Name

策略路由的名称

interface

应用场所，有两种情况：

  • local表示本地策略路由，即应用于所有从本机产生的（而不是通过本机转发的）报文
  • 接口类型和标号表示接口策略路由，即对本接口收到的报文进行策略路由
