local-mt enable (OSPF)

命令功能

local-mt enable命令用来使能OSPF的本地组播拓扑特性。

undo local-mt enable命令用来取消该配置。

缺省情况下，不使能本地组播拓扑特性。

命令格式

local-mt enable

undo local-mt enable

参数说明

需要使用enable traffic-adjustment命令使能IGP Shortcut特性。

视图

OSPF视图

缺省级别

2：配置级

使用指南

无

使用实例

# 使能OSPF本地组播拓扑特性。

<Quidway> system-view
[Quidway] ospf 1
[Quidway-ospf-1] local-mt enable
