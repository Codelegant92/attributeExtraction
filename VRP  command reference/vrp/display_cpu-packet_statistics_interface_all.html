display cpu-packet statistics interface all

命令功能

display cpu-packet statistics interface all命令用来查看指定接口下所有上送到CPU的报文的统计信息。

命令格式

display cpu-packet statistics interface interface-type interface-number all

参数说明

interface-type interface-number：查看指定接口下所有上送到CPU的报文的统计信息。

视图

所有视图

缺省级别

1：监控级

使用指南

使用display cpu-packet statistics interface all命令查看指定接口下所有上送到CPU的报文统计信息之和。包括指定接口、指定接口的子接口、指定接口的VLAN或指定接口的PVC等统计信息之和。

使用实例

# 查看接口GigabitEthernet1/0/0下的所有上送到CPU的报文的统计信息。

<Quidway> display cpu-packet statistics interface gigabitethernet 1/0/0 all
GigabitEthernet1/0/0: 8 packets, 1099 bytes
表1 display cpu-packet statistics slot输出信息描述
