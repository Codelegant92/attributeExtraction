qinq termination pe-vid ce-vid

命令功能

qinq termination pe-vid ce-vid命令用来配置子接口对两层Tag报文的终结功能。

undo qinq termination pe-vid ce-vid命令用来取消子接口对两层Tag报文的终结功能。

缺省情况，子接口没有配置对两层Tag报文的终结功能。

命令格式

qinq termination pe-vid pe-vid ce-vid low-ce-vid [ to high-ce-vid ] [ vlan-group group-id ]

undo qinq termination pe-vid pe-vid ce-vid low-ce-vid [ to high-ce-vid ] [ vlan-group group-id ]

参数说明

pe-vid pe-vid：PE的VLAN ID（外层Tag的值），十进制整数形式，取值范围是1～4094。

ce-vid low-ce-vid：CE的VLAN ID（内层Tag的值）取值范围的下限，十进制整数形式，取值范围是1～4094。

high-ce-vid：CE的VLAN ID（内层Tag的值）取值范围的上限，十进制整数形式，取值范围是1～4094。

vlan-group group-id：用户组号，十进制整数形式，取值范围是1～16。

视图

Ethernet子接口视图、GE子接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

在以太网子接口上使用qinq termination pe-vid ce-vid命令前，应先在该子接口下配置control-vid vid qinq-termination命令，否则无法看到qinq termination pe-vid ce-vid命令。

当子接口需要终结接收到的带有两层Tag的用户报文时，可以在子接口上配置qinq termination pe-vid ce-vid命令。用户报文的外层Tag应该为命令中指定的PE的VLAN ID值；内层Tag应该在命令中指定的CE的VLAN ID指定的范围之内。

使用实例

# 在子接口GE1/0/0.1上配置vlan-group 1的子接口终结功能，内层Tag分别为：100～200，400～500，外层Tag为1000。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mode user-termination
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] interface gigabitethernet1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] control-vid 1 qinq-termination
[Quidway-GigabitEthernet1/0/0.1] vlan-group 1
[Quidway-GigabitEthernet1/0/0.1-vlangroup1] quit 
[Quidway-GigabitEthernet1/0/0.1] qinq termination pe-vid 1000 ce-vid 100 to 200 vlan-group 1
[Quidway-GigabitEthernet1/0/0.1] qinq termination pe-vid 1000 ce-vid 400 to 500 vlan-group 1
[Quidway-GigabitEthernet1/0/0.1] quit
