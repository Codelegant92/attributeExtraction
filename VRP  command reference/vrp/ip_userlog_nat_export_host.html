ip userlog nat export host

命令功能

ip userlog nat export host命令用来配置日志报文的目的日志服务器的IP地址和UDP端口号。

undo ip userlog nat export host命令用来取消指定的日志服务器。

命令格式

ip userlog nat export [ slot slot-number ] host ip-address udp-port

undo ip userlog nat export [ slot slot-number ] host

参数说明

nat：NAT日志功能。

slot-number：指定日志生成的接口板槽位号，每个接口板可以单独配置日志输出的用户日志服务器，起到分担用户日志服务器负载的作用，每个接口板的输出报文进行独立编号（报文头的序列号）。如果不指定slot-number，表示对所有未配置该命令的接口板有效。

ip-address：用户日志服务器IP地址。

udp-port：日志服务器监听的UDP端口号，取值范围：0～65535。缺省情况下，UDP端口号为0。

使用指南

如果以UDP报文输出用户日志，则必须配置用户日志服务器并正确指定用户日志服务器地址，否则用户日志功能不能正常工作。

目的服务器的地址必须为合法的单播地址，不能配置环路地址或组播地址，原则上应该是可以正常通信的用户日志服务器地址。为了避免与通用的UDP端口号冲突，建议使用1024以上的UDP端口号。

使用实例

# 将NAT日志输出报文目的服务器的IP地址设为169.254.1.1，UDP端口号设为2000。

<Quidway> system-view
[Quidway] ip userlog nat export host 169.254.1.1 2000
