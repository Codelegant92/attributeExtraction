x25 map

命令功能

x25 map命令用来设置局域网之间使用的网际协议的地址与X.121地址之间的地址映射。

undo x25 map命令用来删除已有的映射。

缺省情况下，系统没有任何地址映射。

命令格式

x25 map { compressedtcp protocol-address | ip ip-address }* x121-address x.121-address [ option ]

undo x25 map { compressedtcp protocol-address | ip ip-address }*

参数说明

compressedtcp：使用TCP头压缩。

ip：使用IP协议。

ip-address：对端主机的网络协议地址。

x.121-address：对端主机的X.121地址，字符串形式，长度范围是1～15 ，只有0～9数字是有效的。

option：该地址映射指定一些属性或用户设施。

使用指南

由于X.25协议能够在一个物理接口上复用多条逻辑虚电路，所以需要手工指定所有的网络地址与X.121地址的映射关系。

一旦指定一条地址映射，这条地址映射的所有内容（包括协议地址、X.121地址及其所有选项）都不能再被改变，若想改变，可以先用undo x25 map命令删除该地址映射，然后再创建一条新的地址映射。

在同一个X.25接口上，不应该存在两条或两条以上协议地址相同的地址映射。

X.25地址映射各选项意义详细解释如下：

broadcast：向该目的地发送网际协议的广播和IP协议的多播，该选项对一些路由协议（如RIP）提供了强有力的支持。

closed-user-group group-number：指定和该地址映射对应的封闭用户群号。group-number取值范围是0～9999。

encapsulation-type：设置封装方法，可选择的方法有nonstandard、ietf、multi-protocol、snap。

idle-timer minutes：指定与该地址映射相关联的虚电路的最大空闲时间。取值范围是0～255，单位是分钟。0表示空闲时间无限大。

no-callin：禁止接受针对于该地址映射的呼叫。

no-callout：禁止使用该地址映射发起呼叫。

packet-size input-packet output-packet：使用该地址映射发起呼叫时，与对端进行最大分组长度的协商，分组长度以字节为单位，这两个值必须在16到4096范围内（包括16和4096），并且必须是2的整数次幂。

reverse-charge-accept：对该地址映射发起的呼叫里如果携带了反向计费请求，只有在地址映射里配置该选项才能接受呼叫。

reverse-charge-request：使用该地址映射发起呼叫时，携带反向计费请求。

roa-list name：为该接口指定一个在系统视图下用x25 roa命令配置的ROA列表名，name为字符串形式，长度是1～19。

send-delay milliseconds：使用该地址映射发起呼叫时，携带最大传输时延请求。取值范围是0～65534，单位是毫秒。

threshold in out：使用该地址映射发起呼叫时，与对端进行吞吐量级的协商。参数in和out的值只能取75，150，300，600，1200，2400，4800，9600，19200或48000。

vc-per-map count：设置可以与该地址映射相关联的虚电路的最大数目。取值范围是1～8。

window-size input-window-size output-window-size：使用该地址映射发起呼叫时，与对端进行窗口尺寸的协商，input-window-size和output-window-size的值在1到该地址映射所在的X.25接口的模数减1之间（包括1和模数减1）。

使用实例

# 分别在X.25接口Serial1/0/0和Serial2/0/0上各设置两条地址映射，这四条地址映射具有互不相同的属性。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 map ip 202.38.160.11 x121-address 20112451 reverse-charge-request reverse-charge-accept
[Quidway-Serial1/0/0] x25 map ip 202.38.160.138 x121-address 20112450 packet-size 512 512 idle-timer 10
[Quidway-Serial1/0/0] quit
[Quidway] interface serial 2/0/0
[Quidway-Serial2/0/0] link-protocol x25
[Quidway-Serial2/0/0] x25 map ip 20.30.4.1 x121-address 25112451 window-size 4 4 broadcast
[Quidway-Serial2/0/0] x25 map ip 20.30.4.8 x121-address 25112450 no-callin
