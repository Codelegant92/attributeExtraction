timer（RRPP域视图）

命令功能

timer命令用来配置Hello定时器和Fail定时器的超时时间。

undo timer命令用来恢复Hello定时器和Fail定时器的默认值。

命令格式

timer hello-timer hello-value fail-timer fail-value

undo timer

参数说明

hello-value：Hello定时器超时时间。整数形式，取值范围是1～10，单位是秒。默认值是1秒。

fail-value：Fail定时器超时时间。整数形式，取值范围是3～30，单位是秒。默认值是3秒。fail-value的值必须大于或等于hello-value值的3倍。

缺省级别

2：配置级

使用指南

该命令只需要在主节点上配置。

Fail定时器的值必须大于等于Hello定时器的值的三倍。

使用实例

# 配置RRPP域1的Hello定时器和Fail定时器的超时时间分别为2s和7s。

<Quidway> system-view
[Quidway] rrpp domain 1
[Quidway-rrpp-domain-region1] timer hello-timer 2 fail-timer 7

# 将RRPP域1的定时器的超时时间恢复为默认值。

<Quidway> system-view
[Quidway] rrpp domain 1
[Quidway-rrpp-domain-region1] undo timer
