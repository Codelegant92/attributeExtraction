display fr fragment-info

命令功能

display fr fragment-info命令用来查看帧中继分片信息。

命令格式

display fr fragment-info [ interface interface-type interface-number ] [ dlcidlci-number ]

参数说明

interface-type：接口类型，包括MFR、POS、Serial等。

interface-number：接口编号，三维形式：槽号/卡号/接口号。

dlci dlci-number：DLCI编号，取值范围是16～1007。指定该参数将显示详细信息。

使用指南

display fr fragment-info命令可以观察到的信息包括：

  • DLCI号

  • 分片类型

  • 分片大小

  • 接收/发送/丢弃的分片报文数

使用实例

# 查看所有接口上的帧中继分片信息。

<Quidway> display fr fragment-info
interface Serial1/1/1:
dlci    type     size        in/out/drop
200  FRF12(ETE)   80           0/0/0
