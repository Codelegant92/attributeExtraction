ipv6 mtu

命令功能

ipv6 mtu命令用来设置接口上发送IPv6报文的MTU。

undo ipv6 mtu命令用来恢复MTU的缺省值。

命令格式

ipv6 mtu mtu

undo ipv6 mtu

参数说明

mtu：指定MTU的大小。整数形式，单位是字节。

说明：

在不同的接口下，MTU有不同的取值范围。

  • Ethernet接口：取值范围是1280～1500，缺省值是1500。

  • Gigabit Ethernet接口：取值范围是1280～10000，缺省值是1500。

  • 链路层协议为PPP或HDLC的POS接口：取值范围是1280～9660，缺省值是4470。

  • 链路层协议为PPP或HDLC的Serial接口：取值范围是1280～1500，缺省值是1500。

使用指南

在使用该命令更改接口的MTU值后，需要执行shutdown和undo shutdown命令重启接口使改变生效。

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 设置接口GE 1/0/0上发送IPv6报文的MTU为1280字节。

<Quidway> system-view
[Quidway] interface gigaitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 mtu 1280
