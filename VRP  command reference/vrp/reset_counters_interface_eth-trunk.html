reset counters interface（Eth-Trunk）

命令功能

reset counters interface命令用来清除捆绑口的计数器。

命令格式

reset counters interface [ eth-trunk [ trunk-id ] ]

参数说明

eth-trunk：清除Eth-Trunk的统计信息。

trunk-id：Trunk ID，取值范围是0～63。

视图

用户视图

缺省级别

2：配置级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

reset counters interface命令清除的是display interface eth-trunk显示信息的最后一部分：接口输入输出报文的统计信息。

使用实例

# 清除接口Eth-Trunk1的计数器信息。

<Quidway> reset counters interface eth-trunk 1
