display snmp-agent

命令功能

display snmp-agent命令用来查看本地或远程SNMP实体引擎ID。

SNMP引擎ID是SNMP管理性的唯一标识，它在一个管理域内唯一的标识一个SNMP实体。SNMP引擎ID是SNMP实体的重要组成部分，完成SNMP消息的消息调度、消息处理、安全验证、访问控制等功能。

命令格式

display snmp-agent { local-engineid | remote-engineid }

参数说明

local-engineid：本地SNMP实体引擎ID。

remote-engineid：远程SNMP实体引擎ID。

使用指南

在使能了SNMP agentt功能后，查看默认或设置的本地或远程SNMP实体引擎时，可以使用命令display snmp-agent。

使用实例

# 显示本地设备的引擎ID

<Quidway> display snmp-agent local-engineid
  SNMP local EngineID: 000007DB7F0000013859
