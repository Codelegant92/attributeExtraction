frame-format（ATM E3）

命令功能

frame-format命令用来设置ATM E3/T3接口的帧格式。

undo frame-format命令用来恢复缺省设置。

缺省情况下，ATM E3接口的帧格式为g751-plcp，ATM T3接口的帧格式为cbit-plcp。

命令格式

frame-format { g832-adm | g751-adm | g751-plcp }

frame-format { cbit-adm | cbit-plcp | m23-adm | m23-plcp }

undo frame-format

参数说明

g832-adm：设置ATM E3的帧格式为G.823直接成帧。

g751-adm：设置ATM E3的帧格式为G.751直接成帧。

g751-plcp：设置ATM E3的帧格式为G.751 PLCP。

cbit-adm：设置ATM T3的帧格式为C-bit直接成帧。

cbit-plcp：设置ATM T3的帧格式为C-bit PLCP。

m23-adm：设置ATM T3的帧格式为M23直接成帧。

m23-plcp：设置ATM T3的帧格式为M23 PLCP。

使用指南

注意与传输设备保持一致。

使用实例

# 设置ATM E3接口1/0/0使用的帧格式为G.832 ADM。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] frame-format g832-adm
