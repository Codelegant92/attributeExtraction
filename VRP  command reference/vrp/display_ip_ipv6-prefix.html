display ip ipv6-prefix

命令功能

display ip ipv6-prefix命令用来显示地址前缀列表。

命令格式

display ip ipv6-prefix [ ipv6-prefix-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv6-prefix-name：指定IPv6地址前缀列表的名称。字符串形式，不支持空格，长度范围是1～19。如果没有指定地址前缀列表名称，将显示所有配置的地址前缀列表。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示所有IPv6地址前缀列表的信息。

<Quidway> display ip ipv6-prefix
Prefix-list6 abc
Permitted 0
Denied 0
index:   10             permit  ::/0
index:   20             permit  ::/1              ge  1   le  128
