display cfm default md

命令功能

display cfm default md命令用来查看默认MD的相关信息。

命令格式

display cfm default md

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

现如今，802.1ag协议分为draft7草案版本和standard2007标准版本，本命令只支持standard2007标准版本使用。

使用本命令可以查看默认MD的级别、MIP生成规则、Sender ID TLV类型、关联了默认MD的VLAN列表。

使用cfm enable（系统视图）命令全局使能CFM功能后才能使用本命令。

使用实例

# 查看默认MD相关信息。

<Quidway> display cfm default md
Level   MIP Create-type   SenderID TLV-type   VLAN List
----------------------------------------------------------------------------------------
7      default            SendIdDefer         100 to 200 2049
表1 display cfm md命令输出信息描述

项目

描述

Level

默认MD级别，取值范围是0～7，级别越高，优先级越高。

MIP Create-type
MIP生成规则。取值可能是

  • none

  • default

  • explicit

SenderID TLV-type
Sender ID TLV类型，用于CCM报文的填充。可能的显示值有：

  • SendIdNone

  • SendIdChassis

  • SendIdManage

  • SendIdChassisManage

  • SendIdDefer

VLAN ID

关联了默认MD的VLAN。
