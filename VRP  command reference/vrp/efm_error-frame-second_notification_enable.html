efm error-frame-second notification enable

命令功能

efm error-frame-second notification enable命令用来使能当前接口的EFM OAM误帧秒检测上报功能。

undo efm error-frame-second notification enable命令用来禁止当前接口的EFM OAM误帧秒检测上报功能。

缺省情况下，接口的EFM OAM误帧秒检测上报功能处于禁止状态。

命令格式

efm error-frame-second notification enable

undo efm error-frame-second notification enable

参数说明

无

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

当EFM OAM误帧秒检测上报功能使能后，如果在设置的观察时长内，接口检测到的误帧秒数量达到或者超过门限值，那么设备将产生误帧秒越限事件，并通告对端设备。

使用efm enable命令全局使能EFM OAM功能后，才能成功执行本命令。

使用实例

# 使能接口GigabitEthernet1/0/0的EFM OAM误帧秒检测上报功能。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] efm enable
[Quidway-GigabitEthernet1/0/0] efm error-frame-second notification enable
