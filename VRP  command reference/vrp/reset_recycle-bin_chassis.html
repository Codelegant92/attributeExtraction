reset recycle-bin（多框）

命令功能

reset recycle-bin命令用来彻底删除回收站中的文件。

命令格式

reset recycle-bin [ [path] [ drive ] ] [ filename ]

参数说明

path：文件路径。

drive：设备名。

filename：要删除的文件名。字符串形式，长度范围是1～64。支持“*”通配符。

视图

用户视图

缺省级别

3：管理级

使用指南

用delete（多框）命令删除的文件将文件放在回收站中，如果要真正删除此文件，可以使用reset recycle-bin命令。

使用实例

# 删除转发框1回收站中的文件。

<Quidway> reset recycle-bin clc1/17#cfcard:1.zip
Squeeze clc1/17#cfcard:/1.zip ?[Y/N]:y
%Cleared file clc1/17#cfcard:/1.zip.
