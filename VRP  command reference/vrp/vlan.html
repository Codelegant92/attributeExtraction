vlan

命令功能

vlan命令用来创建VLAN并进入VLAN视图，如果VLAN已存在，直接进入该VLAN的视图。

undo vlan用来删除VLAN。

命令格式

vlan vlan-id

vlan batch { vlan-id1 [ to vlan-id2 ] } &<1-10>

undo vlan vlan-id

undo vlan batch { vlan-id1 [ to vlan-id2 ] } &<1-10>

参数说明

batch：批量创建VLAN。

vlan-id1：起始VLAN ID，取值范围是1～4094。

vlan-id2：结束VLAN ID，取值范围是1～4094，该值必须大于vlan-id1。

视图

系统视图

缺省级别

2：配置级

使用指南

可以使用vlan batch命令批量创建多个VLAN，这时不会进入VLAN视图。

使用实例

# 创建一个ID为2的VLAN，如果该VLAN已存在，则直接进入该VLAN视图。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2]

# 批量创建ID为6、7及16～20的VLAN。

<Quidway> system-view
[Quidway] vlan batch 6 7 16 to 20

# 查看已创建的VLAN。

<Quidway> display vlan
