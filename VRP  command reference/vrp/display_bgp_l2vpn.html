display bgp l2vpn

命令功能

display bgp l2vpn命令用来查看BGP中保存的Kompella L2VPN标签块信息。

命令格式

display bgp l2vpn { all | group [ group-name ] | peer [ [ peer-ip-address ] verbose ] | route-distinguisher route-distinguisher [ ce-id ce-id [ label-offset label-offset ] ] } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：显示BGP中保存的所有的Kompella L2VPN本地标签块信息。

group-name：组名称，字符串形式，不支持空格，长度范围是1～47。

peer-ip-address：显示BGP中保存的指定对等体的Kompella L2VPN标签块信息。

verbose：显示BGP中保存的Kompella L2VPN标签块的详细信息。

route-distinguisher route-distinguisher：指定的路由标识符。路由标识符格式如下：

  • 16位自治系统号:32位用户自定义数，例如：101:3。自治系统号的取值范围是0～65535；用户自定义数的取值范围是0～4294967295。其中，自治系统号和用户自定义数不能同时为0，即RD的值不能是0:0。

  • 32位IP地址:16位用户自定义数，例如：192.168.122.15:1。IP地址的取值范围是0.0.0.0～255.255.255.255；用户自定义数的取值范围是0～65535。

ce-id ce-id：VPN内CE的ID值。十进制整数形式，取值范围是0～65535。

label-offset label-offset：标签的offset值，十进制整数形式，取值范围是0～65535。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display bgp l2vpn命令可查看的标签块信息包括RD值，对应的CE-ID、标签块偏移值、标签块中起始标签的值及标签块的BGP路由属性。

如果同时指定参数route-distinguisher、ce-id和label-offset，则可查看标签块的BGP扩展团体属性、标签块中每一个标签的值及使用该标签的L2VPN连接状态，以及已对哪些BGP对等体发布了该标签块。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

BGP Local router ID :

本地BGP的本地路由器ID号，注意其格式与IPv4地址一样。

local AS number :

本地自治系统号。

nexthop:

MP-BGP的下一跳地址。

pref

本地优先级，是BGP的路由属性的一种。

as-path :

AS路径属性，记录了某条路由从本地到目的地址所要经过的所有AS编号。是BGP的路由属性的一种。

label base:

标签起始值。

label range:

标签范围。

layer-2 mtu:

二层链路的MTU值。

encap type:

二层链路的封装方式。

label

已分配的标签值。

state

已分配的标签值的状态。
