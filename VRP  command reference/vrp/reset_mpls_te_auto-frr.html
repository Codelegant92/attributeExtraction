reset mpls te auto-frr

命令功能

reset mpls te auto-frr命令用来重建某一条指定的自动生成的旁路隧道。

说明：

在使能Auto FRR特性的情况下，该命令可以重建旁路隧道。

命令格式

reset mpls te auto-frr { lsp-id ingress-lsrid tunnel-id | name bypass-tunnel-name }

参数说明

ingress-lsr-id：入节点的LSR-ID，点分十进制格式。

tunnel-id：隧道标识。整数形式。取值范围取决于产品的许可证文件。

bypass-tunnel-name：旁路隧道的名称，字符串形式，长度为1～63。
