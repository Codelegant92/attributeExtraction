dhcp server expired（接口视图）

命令功能

dhcp server expired命令用来配置当前接口DHCP地址池中IP地址的租用有效期限。

undo dhcp server expired命令用来恢复缺省设置。

缺省情况下，租用有效期限为1天。

命令格式

dhcp server expired { day day [ hour hour [ minute minute ] ] | unlimited }

undo dhcp server expired

参数说明

day day：指定租用有效期限的天数。整数形式，单位是天，取值范围是0～365。缺省值是1天。

hour hour：指定租用有效期限的小时数。整数形式，单位是小时，取值范围是0～23。缺省值是0。

minute minute：指定租用有效期限的分钟数。整数形式，单位是分钟，取值范围是0～59。缺省值是0。

unlimited：租用有效期限为无限长。

使用指南

根据不同的主机地址使用的不同，可以使用不同的租期分配方式，使用dhcp server expired命令用来配置当前接口DHCP地址池中IP地址的租用有效期限。

使用实例

# 配置GE1/0/0的接口地址池的IP地址租用有效期无限长。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] dhcp server expired unlimited
