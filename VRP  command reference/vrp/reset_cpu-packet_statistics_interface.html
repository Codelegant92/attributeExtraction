reset cpu-packet statistics interface all

命令功能

reset cpu-packet statistics interface all命令用来清除指定接口下所有上送到CPU的报文的统计信息。

命令格式

reset cpu-packet statistics interface interface-type interface-number all

参数说明

interface-type interface-number：指定接口名称和接口编号。

视图

用户视图

缺省级别

3：管理级

使用指南

在某些情况下，需要统计一定时间内某接口上送CPU的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口的流量重新进行统计。

使用实例

# 清除接口GigabitEthernet 1/0/0下所有上送到CPU的报文的统计功能。

<Quidway> reset cpu-packet statistics interface gigabitethernet 1/0/0 all
