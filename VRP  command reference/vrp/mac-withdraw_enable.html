mac-withdraw enable

命令功能

mac-withdraw enable命令用来配置在AC或者UPE故障且VSI仍UP的情况下，使能VSI删除本地的MAC地址，并通告远端所有对等体的功能。

undo mac-withdraw enable命令用来配置在AC或者UPE故障且VSI仍UP的情况下，取消VSI删除本地的MAC地址，并通告远端所有对等体的功能。

命令格式

mac-withdraw enable

undo mac-withdraw enable

参数说明

无

视图

VSI-LDP视图

缺省级别

2：配置级

使用指南

该命令需要配置在SPE上。

在AC或者UPE故障且VSI仍UP的情况下，如果UPE是双归属接入SPE，UPE需要将VPLS的流量转到另一个LSP。而对于SPE，只需要通知VSI对应的其他所有SPE清空相应VSI下的MAC表即可。倒换发生后，MAC表重新进行学习。

使用实例

# 设置名称为V100的VSI，在AC或者UPE故障且VSI仍UP的情况下，删除本地的MAC地址，并通告远端所有对等体的功能。

<Quidway> system-view
[Quidway] vsi v100 static
[Quidway-vsi-v100] pwsignal ldp
[Quidway-vsi-v100-ldp] vsi-id 100
[Quidway-vsi-v100-ldp] mac-withdraw enable
