mpls te auto-frr（接口视图）

命令功能

mpls te auto-frr命令用来在接口视图下使能Auto FRR特性。

undo mpls te auto-frr命令用来在接口视图下去使能Auto FRR特性。

缺省情况下，接口下未使能Auto FRR特性。

命令格式

mpls te auto-frr { default | link | node }

undo mpls te auto-frr

参数说明

default：缺省情况，继承MPLS视图下的Auto FRR配置。

link：只提供链路保护。

node：提供链路保护和节点保护。

视图

接口视图

缺省级别

2：配置级

使用指南

缺省情况下，全局使能Auto FRR后，该设备的所有使能MPLS TE的接口都自动配置了mpls te auto-frr default。如果要去使能部分接口的Auto FRR，需在这些接口下配置命令undo mpls te auto-frr。在接口下配置命令undo mpls te auto-frr后，不论全局是否已使能或重新使能Auto FRR，该接口都不具有Auto FRR能力。

使用实例

# 只提供接口创建链路保护的Bypass tunnel。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] mpls te auto-frr link

# 去使能接口下的Auto FRR特性。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] undo mpls te auto-frr
