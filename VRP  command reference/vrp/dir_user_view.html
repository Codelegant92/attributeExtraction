dir（用户视图）

命令功能

dir命令用来显示路由器存储设备中的指定文件或目录的信息。

缺省情况下，显示当前路径下的文件信息。

命令格式

dir [ /all ] [ filename ]

参数说明

/all：显示所有的文件（包括已删除的文件）。

filename：显示的文件或目录名，字符串形式，长度范围是1～64。

用户视图

缺省级别

3：管理级

使用指南

支持使用通配符“ * ”匹配。

使用dir /all命令显示所有文件信息，包括已经删除的文件信息，已经删除的文件的名字用“ [ ] ”括起来表示，例如 [ text ] 。这种已删除文件可以使用undelete命令恢复，若要从回收站中彻底删除该文件，使用reset recycle-bin命令。

使用实例

# 显示文件flash:/test/test.txt的信息。

<Quidway> dir flash:/test/test.txt
Directory of flash:/test/
Idx   Attr  Size(Byte)     Date      Time     FileName
   0   -rw-        80  Mar 09 2004 09:47:36   test.txt
2,540 KB total (394 KB free)

# 显示目录flash:/test/的信息。

<Quidway> dir flash:/test/
Directory of flash:/test/
Idx   Attr  Size(Byte)     Date      Time     FileName
   0   -rw-         4  Mar 01 2004 21:19:27   sample.txt
   1   -rw-        80  Mar 09 2004 09:47:36   test.txt
2,540 KB total (394 KB free)
<Quidway> dir flash:/test/t*
Directory of flash:/test/
Idx   Attr  Size(Byte)     Date      Time     FileName
   0   -rw-        80  Mar 09 2004 09:47:36   test.txt
2,540 KB total (394 KB free)
