display route-policy

命令功能

display route-policy命令用来显示配置的Route-Policy。

命令格式

display route-policy [ route-policy-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

route-policy-name：指定要显示的Route-Policy名称。字符串形式，长度范围是1～40。不指定route-policy-name时，将显示配置的所有Route-Policy。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示名为policy1的Route-Policy信息。

<Quidway> display route-policy policy1
Route-policy : policy1
  permit : 10
    Match clauses :
        if-match acl 2000
    Apply clauses :
        apply cost 100
        apply tag 100
