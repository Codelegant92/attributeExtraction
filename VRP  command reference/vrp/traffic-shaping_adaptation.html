traffic-shaping adaptation

命令功能

traffic-shaping adaptation becn命令用来使能帧中继流量整形的自适应流量调节功能。

undo traffic-shaping adaptation becn命令用来禁止此功能。

缺省情况下，应为使能帧中继流量整形的自适应流量调节功能，调节比例为25。

命令格式

traffic-shaping adaptation { becn percentage | interface-congestion number }

undo traffic-shaping adaptation { becn | interface-congestion }

参数说明

becn：对带BECN标志的报文进行流量调节。

percentage：每次调节的比例，取值范围是1～30。缺省值为25。

interface-congestion：根据接口出队列中的报文数进行流量调节。

number：接口出队列中的报文数，取值范围是1～40。

使用指南

帧中继流量整形可以使能自适应流量调节的功能，它采用两种方式确定是否降低发送速率：

  • BECN方式：当路由器发送接口接收到帧中继网络发送的BECN位为1的报文后，所有配置了帧中继流量整形的PVC将发送速率下降指定的百分比，在下降后的速率和MinCIR中取较大值，作为新的发送速率；发送速率下降后，若路由器在16个Tc时间之内没有再收到BECN位为1的报文，发送速率将上升指定的百分比。在上升后的速率和CIR中取较小值，作为新的发送速率。

  • 接口拥塞方式（interface congestion）：当发送接口的队列中的报文数达到设定值的时候，所有配置了帧中继流量整形的PVC将降低发送速率，当队列中的报文数小于设定值的时候，再提高发送速率。

使用实例

# 使能帧中继流量整形的自适应流量调节功能，对BECN位为1的报文每次调节20%。
