isis suppress-reachability

命令功能

isis suppress-reachability命令用来抑制接口地址的发布，从而可以重用接口地址。

undo isis suppress-reachability命令用来取消此功能。

如果没有指定级别，则缺省值为Level-1-2。

命令格式

isis [ ipv6 ] suppress-reachability [ level-1 | level-1-2 | level-2 ]

undo isis [ ipv6 ] suppress-reachability

参数说明

ipv6：指定抑制IPv6地址的发布。如果不指定此参数，则默认为IPv4。

level-1：只抑制Level-1接口地址发布。

level-1-2：抑制Level-1和Level-2接口地址的发布。

level-2：只抑制Level-2接口地址的发布。

使用指南

使用ipv6参数前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。

使用实例

# 抑制接口GigabitEthernet1/0/0的IPv4地址发布。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] isis suppress-reachability
