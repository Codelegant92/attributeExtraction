sftp server enable

命令功能

sftp server enable命令用来使能SSH服务器端的SFTP服务。

undo sftp server enable命令用来关闭SSH服务器端的SFTP服务。

缺省情况下，SSH服务器端的SFTP服务没有使能。

命令格式

sftp server enable

undo sftp server enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

只有SSH服务器端的SFTP服务使能以后，客户端才能以SFTP方式与SSH服务器建立连接。

使用实例

# SFTP服务使能。

<Quidway> system-view
[Quidway] sftp server enable
Info: Succeeded in starting the SFTP server.

# 关闭SFTP服务。

<Quidway> system-view
[Quidway] undo sftp server enable
Info: Succeeded in closing the SFTP server.
