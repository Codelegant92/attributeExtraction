tunnel select-seq

命令功能

tunnel select-seq命令用于配置参与负载分担的隧道的优先级顺序。

undo tunnel select-seq命令用来恢复缺省配置。

缺省情况下，只有LDP LSP或者静态LSP被选中，不进行负载分担。

命令格式

tunnel select-seq { gre | lsp | cr-lsp }* load-balance-number load-balance-number

undo tunnel select-seq

参数说明

gre：GRE隧道。

lsp：LDP LSP或者静态LSP。

cr-lsp：CR-LSP隧道。

load-balance-number：负载分担的隧道条数。取值范围由License控制，取值范围是1～6，在使用默认的License时，取值为1。

使用指南

在配置参与负载分担的隧道优先级顺序时，隧道类型越靠近关键字select-seq的，其优先级越高。并且，只有tunnel select-seq命令中列举的隧道类型可以被使用。负载分担数是指一个应用可以使用的隧道条数。在使用多个不同类型的隧道进行负载分担时，流量将优先选用优先级高的隧道。

使用实例

# 定义隧道策略为只能使用LDP LSP或者静态LSP，负载分担条数为2。

<Quidway> system-view
[Quidway] tunnel-policy l2
[Quidway-tunnel-policy-l2] tunnel select-seq lsp load-balance-number 2
