arp-miss speed-limit

命令功能

arp-miss speed-limit命令用来设置接口板对ARP Miss消息进行时间戳抑制的抑制速率。

undo arp-miss speed-limit命令用来恢复接口板对ARP Miss消息进行时间戳抑制的缺省抑制速率。

缺省情况下，时间戳抑制的功能已经使能，抑制速率为每秒500个ARP报文。

命令格式

arp-miss speed-limit source-ip maximum maximum slot slot-id

undo arp-miss speed-limit source-ip slot slot-id

参数说明

maximum：抑制速率。取值范围是0～32768，0表示不使能抑制功能。

slot-id：指定的接口板槽号。

使用指南

目前只实现对ARP Miss消息采用基于源IP地址进行抑制。

使用实例

<Quidway> system-view
[Quidway] arp-miss speed-limit source-ip maximum 50 slot 3
