slave switchover { disable | enable }（多框）

命令功能

命令slave switchover { disable | enable }用来使能或禁止强制进行主备倒换。

缺省情况下，使能强制进行主备倒换。

命令格式

slave switchover [ chassis chassis-number ] { disable | enable }

参数说明

chassis chassis-number：转发框的框号。

disable：禁止强制进行主备倒换。

enable：使能强制进行主备倒换。

视图

系统视图

缺省级别

2：配置级

使用指南

只有在使能强制进行主备倒换之后，slave switchover命令才能有效；如果禁止强制进行主备倒换，则用户无法通过配置命令强制进行主备倒换。

使用实例

# 使能转发框1主备倒换的功能。

<Quidway> system-view
[Quidway] slave switchover chassis clc1 enable
