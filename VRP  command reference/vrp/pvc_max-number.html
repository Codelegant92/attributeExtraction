pvc max-number

命令功能

pvc max-number命令用来设定ATM接口虚电路（VC）数目的最大值。

undo pvc max-number命令用来恢复其缺省值。

命令格式

pvc max-number max-number

undo pvc max-number

参数说明

max-number：可支持的最大虚电路数。整数形式，取值范围是1～2048。缺省值是256。

当基于E1线路承载ATM业务时，即用于ATM E1接口时，该参数的取值范围是1～128。缺省值是128。

当基于E3线路承载ATM业务时，即用于ATM E3接口时，该参数的取值范围是1～1024。缺省值是1024。

使用指南

本命令为ATM主接口和子接口总共可用的VC数目设定最大值。

说明：

本命令虽然同时作用于ATM主接口和子接口，但只能在ATM接口视图下使用此命令，ATM子接口视图下无此命令。

使用实例

下面两个例子都可以使ATM接口Atm1/0/0总共支持最多2048条VC。

# 指定ATM接口Atm1/0/0总共支持最多2048条VC。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvc max-number 2048

# 指定ATM接口Atm1/0/0总共支持缺省数量（即2048条）的VC。

[Quidway-Atm1/0/0] undo pvc max-number
