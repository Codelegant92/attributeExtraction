tunnel switch enable

命令功能

tunnel switch enable命令用来启用隧道交换功能。

undo tunnel switch enable命令用来关闭隧道交换功能。

缺省情况下，禁止隧道交换功能。

命令格式

tunnel switch enable

undo tunnel switch enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

当L2TP会话需要跨越一条以上的隧道时，就需要在交换节点上启用隧道交换功能。

启用隧道交换功能后，该节点将作为LNS来接受LAC或上一个节点发起的隧道建立请求，同时又作为LAC向下一个交换节点或真正的LNS发起隧道建立请求。

启动隧道交换功能之前必须先启动L2TP功能。

使用实例

# 启用隧道交换功能。

<Quidway> system-view
[Quidway] tunnel switch enable
