baudrate

命令功能

baudrate命令用来设置串口的波特率。

undo baudrate命令用来恢复缺省设置。

缺省情况下，同步串口的波特率是64kbit/s。

命令格式

baudrate baudrate

undo baudrate

参数说明

baudrate：串口的波特率。单位是bit/s。取值范围请参见下面的使用指南。

使用指南

说明：

由其它接口板通道化成的serial接口不支持baudrate命令。

同步串口支持的波特率有：

  • 1200bit/s

  • 2400bit/s

  • 4800bit/s

  • 9600bit/s

  • 19200bit/s

  • 38400bit/s

  • 56000bit/s

  • 57600bit/s

  • 64000bit/s

  • 72000bit/s

  • 115200bit/s

  • 128000bit/s

  • 384000bit/s

  • 2048000bit/s

另外同步串口对于不同的物理电气规程，所支持的波特率范围有所不同。

  • V.24 DTE/DCE：1200 bit/s～64 kbit/s

  • V.35 DTE：1200 bit/s～4.096 Mbit/s

  • V.35 DCE、X.21 DTE/DCE、EIA/TIA-449 DTE/DCE、EIA-530 DTE/DCE：1200 bit/s～2.048 Mbit/s

在设置串口波特率时，要注意串口外接电缆的电气规程等因素。在同步方式下，由DCE侧设备决定线路传输的波特率。只能在DCE侧设定线路传输的波特率。

使用实例

# 设置同步串口Serial 1/0/0的波特率为115200bit/s。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] baudrate 115200
