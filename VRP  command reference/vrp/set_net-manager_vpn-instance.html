set net-manager vpn-instance

命令功能

set net-manager vpn-instance命令用来设置网管使用的默认VPN实例。

undo set net-manager vpn-instance命令用来取消网管使用的默认VPN实例。

命令格式

set net-manager vpn-instance vpn-instance-name

undo set net-manager vpn-instance

参数说明

vpn-instance vpn-instance-name：指定网管默认的VPN实例名，字符串形式，取值范围是1～31。

视图

系统视图

缺省级别

3：管理级

使用指南

设置了网管默认的VPN实例后，用户采用FTP方式连接远程服务器时，系统将使用该VPN实例。

使用实例

# 设置网管默认的VPN实例。

<Quidway> system-view
[Quidway] set net-manager vpn-instance v1
