snmp-agent local-engineid

命令功能

snmp-agent local-engineid命令用来设置本地SNMP实体的引擎ID。

undo snmp-agent local-engineid命令用来取消当前设置。

缺省情况下，VRP采用内部算法自动生成一个设备引擎ID，包含公司的“企业号＋设备信息”。

命令格式

snmp-agent local-engineid engineid

undo snmp-agent local-engineid

参数说明

engineid：引擎ID字符串，必须为十六进制数字，长度范围是10～64。

使用指南

snmp-agent local-engineid命令用来设置本地SNMP实体的引擎ID，配置ID的算法遵从以下的原则：

  • 第1位固定取值0。

  • 前4个字节是IANA（Internet Assigned Numbers Authority）分配的企业私有设备号，16进制形式。华为公司的设备号是2011，16进制形式为0x000007DB。

  • 设备信息部分的取值由各具体产品确定，可以是IP地址、MAC地址等。

使用实例

# 配置本地设备的引擎ID为12345abcde。

[Quidway] snmp-agent local-engineid 12345abcde
