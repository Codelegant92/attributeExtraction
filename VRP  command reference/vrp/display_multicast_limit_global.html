display multicast limit global

命令功能

display multicast limit global命令用来查看实例下全局表项数量的限制值和统计值。

命令格式

display multicast [ vpn-instance vpn-instance-name ] limit [ channel channel-name ] global [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例名称。字符串形式，区分大小写。

channel channel-name：指定节目组名称。字符串形式，区分大小写。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

对于ASM模型的一个组播组G，(*,G)和所有的(S,G)计为一个表项；对于SSM模型的一个组播组G，每个(S,G)计为一个表项。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看公网实例下节目组SDTV的全局表项统计信息。

<Quidway> display multicast limit channel SDTV global
