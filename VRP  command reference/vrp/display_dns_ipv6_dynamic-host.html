display dns ipv6 dynamic-host

命令功能

display dns ipv6 dynamic-host命令用来查看IPv6动态域名缓存的内容。

命令格式

display dns ipv6 dynamic-host [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
