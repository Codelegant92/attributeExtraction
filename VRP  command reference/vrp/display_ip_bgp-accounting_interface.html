display ip bgp-accounting interface

命令功能

display ip bgp-accounting interface命令用来显示使能了BGP Accounting的接口下通过BGP路由报文的流量信息。

说明：

缺省情况下，则对所有配置该功能的接口进行inbound和outbound流量信息的显示。

命令格式

display ip bgp-accounting [ inbound | outbound ] interface [ interface-type interface-number ]

参数说明

inbound：显示在接口的入方向上根据流量索引统计的流量信息。

outbound：显示在接口的出方向上根据流量索引统计的流量信息。

interface：指定使能BGP Accounting特性的接口。

interface-type interface-number：指定接口类型和接口号。
