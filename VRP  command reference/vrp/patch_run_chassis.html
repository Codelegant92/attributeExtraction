patch run（多框）

命令功能

patch run命令用来按照设备方式运行补丁。

命令格式

patch run [ all | chassis chassis-number | slave | slot slot-id ]

参数说明

all：所有单板（包括主控板和接口板）。

chassis chassis-number：转发框的框号。

slave：系统备用板。

slot slot-id：板号（输入的形式是框类型＋框号＋板号）。

视图

用户视图

缺省级别

2：配置级

使用指南

指定all，则运行多框集群所有板的补丁；指定slave，则运行多框集群的系统备板的补丁；不指定任何的参数，则运行多框集群的系统主用板补丁。

使用实例

# 在所有板上运行已经加载的补丁。

<Quidway> patch run
