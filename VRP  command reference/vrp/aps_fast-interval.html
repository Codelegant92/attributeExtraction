aps fast-interval

命令功能

aps fast-interval命令用来配置或修改APS报文的快发时间间隔。

undo aps fast-interval命令用来恢复APS报文的快发时间间隔为缺省值。

缺省情况下，APS报文的快发时间间隔为3.3毫秒。

命令格式

aps fast-interval { 3.3, 5, 10, 15, 20, 30 }

undo aps fast-interval

参数说明

{ 3.3, 5, 10, 15, 20, 30 }：用来指定快发报文的时间间隔，单位是毫秒，缺省为3.3毫秒。

视图

MAC Tunnel视图

缺省级别

2：配置级

使用指南

本命令只有在保护组为APS型保护组，而且保护组没有使能APS协议的情况下才能配置。

通过本命令配置报文的快发时间间隔后，当APS型保护组状态变化时，就会按照此时间间隔快发3个APS报文。

使用实例

# 配置快发报文时间间隔为5ms。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] protection mac-tunnel protunnel protocol aps
[Quidway-mac-tunnel-mytunnel] aps fast-interval 5
