fragment

命令功能

fragment命令用来使能帧中继虚电路的报文分片功能。

undo fragment命令用来禁止此功能。

缺省情况下，禁止帧中继虚电路的报文分片功能。

命令格式

fragment [ fragment-size ]

undo fragment

参数说明

fragment-size：分片报文大小。单位是字节，取值范围是16～1600。缺省的分片报文大小为45字节。

视图

FR-Class视图

缺省级别

2：配置级

使用指南

VRP上实现的是FRF12中的端到端（End-to-End）的分片。分片和重组是基于PVC进行的，报文中的序号也是在PVC上增加的，多条PVC之间互不影响。

使用实例

# 在名为test1的帧中继class下配置分片大小为128字节。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] fragment 128
