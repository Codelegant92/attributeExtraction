clock（CT1接口视图）

命令功能

clock命令用来设置CT1接口的时钟模式。

undo clock命令用来恢复缺省设置。

缺省情况下，CT1接口的时钟模式为从时钟还是主时钟根据产品的不同而不同。

命令格式

clock { master | slave }

undo clock

参数说明

master：设置CT1接口作为主时钟。

slave：设置CT1接口作为从时钟。

视图

CT1接口视图

缺省级别

2：配置级

使用指南

当两台路由器的CT1接口直接相连时，应设置一端作为主时钟，使用内部时钟信号，另一端作为从时钟，使用线路提供的时钟信号。

使用实例

# 设置CT1接口1/0/0使用主时钟模式。

<Quidway> system-view
[Quidway] controller t1 1/0/0
[Quidway-T1 1/0/0] clock master
