igmp-snooping static-router-port

命令功能

igmp-snooping static-router-port命令用来配置当前子接口作为指定VSI内的静态路由端口。

undo igmp-snooping static-router-port命令用来取消当前子接口作为指定VSI内的静态路由端口的配置。

命令格式

igmp-snooping static-router-port vsi vsi-name

undo igmp-snooping static-router-port vsi vsi-name

参数说明

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

子接口视图

缺省级别

2：配置级

使用指南

子接口加入静态路由端口注意以下几点：

  • 一个子接口只能加入一个VSI中，在一个子接口上只允许配置一条该命令。

  • 如果该子接口事先没有加入此VSI中，不允许进行该配置。

  • 配置该子接口为VSI内的路由端口后，不允许再将此子接口配置为另一个VSI内的路由端口。

使用实例

# 将GigabitEthernet1/0/0.1配置为vsi0内的路由端口。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] igmp-snooping static-router-port vsi vsi0
