display virtual-ethernet ve-group

命令功能

display virtual-ethernet ve-group命令用来查看VE接口和VE-group的绑定关系。

命令格式

display virtual-ethernet ve-group [ ve-group-id | slot slot-id ]

参数说明

ve-group-id：VE-group组号。整数形式，取值范围是1～8192。

slot slot-id：接口板的板号。

视图

所有视图

缺省级别

1：监控级

使用指南

当一台设备上配置多个VE-Group时，可以使用该命令查看VE接口与VE-Group之间的对应关系。

使用实例

# 查看设备上所有VE接口与VE-Group的绑定关系。

<Quidway> display virtual-ethernet ve-group
 Ve-groupID    L2VE                        L3VE

 5               Virtual-Ethernet5/0/8   Virtual-Ethernet5/0/9
