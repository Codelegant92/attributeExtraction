display lspv configuration

命令功能

display lspv configuration命令用来查看PWE3 Tracert当前的配置信息。

命令格式

display lspv configuration

参数说明

无

使用实例

# 查看设备上PWE3 Tracert当前的配置信息。

<Quidway> display lspv configuration
lspv packet filter 2100
undo lspv mpls-lsp-ping echo enable
表1 display lspv configuration命令输出信息描述

项目

描述

lspv packet filter 2100

当前配置了对PWE3 Tracert启用发起者源地址过滤功能，过滤规则为ACL 2100。

undo lspv mpls-lsp-ping echo enable

当前配置了关闭对于MPLS PING报文的响应功能。

该命令的显示信息与具体配置有关。
