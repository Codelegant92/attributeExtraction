dir（多框）

命令功能

dir命令用来显示路由器存储设备中的指定文件或目录的信息。

缺省情况下，显示当前路径下的文件信息。

命令格式

dir [ /all ] [ drive ] [ path ] [ filename ]

参数说明

/all：显示所有的文件（包括已删除的文件）。

drive：设备名。

path：路径名。

filename：显示的文件或目录名。字符串形式，长度范围是1～64。

使用指南

支持使用通配符“ * ”匹配。

使用dir /all命令显示所有文件信息，包括已经删除的文件信息，已经删除的文件的名字用“ [ ] ”括起来表示，例如 [ text ] 。这种已删除文件可以使用undelete（多框）命令恢复，若要从回收站中彻底删除该文件，使用reset recycle-bin（多框）命令。

使用实例

# 显示转发框1中文件test.txt的信息。

<Quidway> dir clc1/17#cfcard:/test.txt
Directory of clc1/17#cfcard:/
   0   -rw-         7  Jun 24 2008 18:01:50   test.txt


2540 KB total (1959 KB free)
