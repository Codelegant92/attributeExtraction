gtsm log drop-packet

命令功能

gtsm log drop-packet命令用来打开单板的LOG信息开关，在单板GTSM丢弃报文时记录LOG信息。

undo gtsm log drop-packet命令用来关闭指定单板LOG信息的开关。

缺省情况下，在单板GTSM丢弃报文时不记录LOG信息。

命令格式

gtsm log drop-packet { slot-id | all }

undo gtsm log drop-packet { slot-id | all }

参数说明

slot-id：指定单板。

all：所有单板。
