display license resource usage

命令功能

display license resource usage命令用来显示License文件中定义的资源项的使用情况。

命令格式

display license resource usage

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

执行display license resource usage命令后，会显示License文件中定义的资源项的使用情况。

资源使用情况指资源使用百分比，即当前资源使用占License文件资源授权值的百分比。比如License文件对VPLS实例数的限制为50，当前创建的VPLS实例数为25，那资源使用百分比就是50%。

使用实例

# 显示资源使用情况。

<Quidway> display license resource usage
