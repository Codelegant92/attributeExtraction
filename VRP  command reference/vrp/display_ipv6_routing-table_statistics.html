display ipv6 routing-table statistics

命令功能

display ipv6 routing-table statistics命令用来显示综合路由统计信息。

命令格式

display ipv6 routing-table [ all-vpn6-instance ] statistics [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all-vpn6-instance：显示所有IPv6 VPN实例的综合路由统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

综合路由统计信息包括路由总数、增加的路由数、删除的路由数。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示综合路由统计信息。

<Quidway> display ipv6 routing-table statistics
Protocol   route       active      added       deleted     freed
DIRECT      1           1           1           0           0
STATIC      0           0           0           0           0
RIPng       0           0           0           0           0
OSPFv3      0           0           0           0           0
IS-IS       0           0           0           0           0
BGP         0           0           0           0           0
Total       1           1           1           0           0
表1 display ipv6 routing-table statistics命令输出信息描述

项目

描述

Protocol

路由协议。

routes

当前路由表中的路由总数。

active

路由表中激活路由的数目。

added

路由表中增加的路由数目（激活和未激活）。

deleted

路由表中删除的路由数目。

freed

路由表中释放（永久删除）的路由数目。

# 显示所有IPv6 VPN实例的综合路由统计信息。

<Quidway> display ipv6 routing-table all-vpn6-instance statistics
Protocol   route       active      added       deleted     freed
DIRECT      1           1           1           0           0
STATIC      0           0           0           0           0
RIPng       0           0           0           0           0
OSPFv3      0           0           0           0           0
IS-IS       0           0           0           0           0
BGP         0           0           0           0           0
Total       1           1           1           0           0
