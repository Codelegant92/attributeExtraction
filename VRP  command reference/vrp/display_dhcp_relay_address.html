display dhcp relay address

命令功能

display dhcp relay address命令用来查看启动中继代理功能的接口的DHCP配置情况。

命令格式

display dhcp relay address { all | interface interface-type interface-number | vlan vlan-id }

参数说明

interface interface-type interface-number：指定该接口类型和编号。

vlan vlan-id：指定VLAN编号，表示查看指定VLAN的DHCP的配置情况。

all：所有启动中继代理功能的接口。

使用指南

在配置完中继后，使用display dhcp relay address命令用来查看启动中继代理功能接口的DHCP配置情况。包括：该接口所代理的DHCP服务器地址、DHCP option与DHCP服务器地址、中继代理地址的关联配置情况。

使用实例

# 查看所有接口的DHCP配置情况。

<Quidway> display dhcp relay address all
    **  GigabitEthernet1/0/0 DHCP Relay Address  **

项目

描述

DHCP Option
DHCP Option。具体情况如下：
  • *：表示没有DHCP option与中继代理地址、DHCP服务器地址的关联，带任意选项的报文可以根据此项配置进行转发。
  • 45：表示与中继代理地址1.1.1.1关联的是option 45。
  • 60(*)：表示与中继代理地址1.1.1.1关联的是option 60，带任意option60字符串的报文都可以根据此项配置进行转发。
  • 60(a)：表示与中继代理地址2.2.2.1、DHCP服务器10.1.1.2相关联的是字符串为a的option60。
  • 60(abc)：表示与DHCP服务器1.1.1.1和10.1.1.2相关联的是字符串为abc的option60。

Relay Agent IP
中继代理地址。
  • -：表示没有配置与该DHCP option相关联的中继代理地址。

Server IP
DHCP服务器地址。
  • -：表示没有配置与该DHCP option相关联的DHCP服务器地址。
