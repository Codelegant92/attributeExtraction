pppoe-server max-sessions remote-mac

命令功能

pppoe-server max-sessions remote-mac命令用来配置一个对端MAC地址上能创建的最大PPPoE session数目。

undo pppoe-server max-sessions remote-mac命令用来恢复本参数为缺省值。

命令格式

pppoe-server max-sessions remote-mac number

undo pppoe-server max-sessions remote-mac

参数说明

number：一个对端MAC地址上能创建的最大PPPoE session数目。取值范围是1～8192。缺省值是100。
