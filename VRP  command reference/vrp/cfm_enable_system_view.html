cfm enable（系统视图）

命令功能

cfm enable命令用来全局使能设备的CFM功能。

undo cfm enable命令用来全局禁止设备的CFM功能。

缺省情况下，VRP设备的CFM功能处于全局禁止状态。

命令格式

cfm enable

undo cfm enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

使用undo cfm enable命令全局禁止设备的CFM功能时，设备自动删除所有CFM相关配置，包括MD、MA、MEP、MIP、RMEP的相关配置。

在配置和使用所有CFM功能前必须先使用本命令全局使能设备的CFM，否则其它CFM相关配置命令不可用。

如果遇到提示GTL资源不足时，用户需要释放已经使用的GTL资源或者到公司买新的资源。

使用实例

# 全局使能设备的CFM功能。

<Quidway> system-view
[Quidway] cfm enable
Info: Succeeded in enabling CFM.
