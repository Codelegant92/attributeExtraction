pim timer join-prune

命令功能

pim timer join-prune命令用来在接口上配置向上游路由器周期性发送Join/Prune消息的时间间隔。

undo pim timer join-prune命令用来恢复时间间隔为缺省值。

缺省情况下，接口向上游路由器周期性发送Join/Prune消息的时间间隔是60秒。

命令格式

pim timer join-prune interval

undo pim timer join-prune

参数说明

interval：表示发送Join/Prune消息的时间间隔。整数形式，取值范围是1～2147483647，单位是秒。

使用指南

该命令适用于PIM-SM网络。

该命令配置的时间间隔必须小于pim holdtime join-prune命令配置的时间间隔，即发送Join/Prune消息的周期必须小于Join/Prune消息的holdtime。

该命令与PIM视图下的timer join-prune（IPv4）命令作用相同，只是视图不同。缺省情况下，如果接口上没有配置，取PIM视图下配置的值；否则取接口视图下的值。

使用实例

# 在接口上配置发送Join/Prune消息的时间间隔为80秒。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] pim timer join-prune 80
