multicast ipv6 reject inbound-data

命令功能

multicast ipv6 reject inbound-data命令用来禁止该接口接收IPv6组播数据报文。

undo multicast ipv6 reject inbound-data命令用来恢复缺省配置。

缺省情况下，接口正常接收IPv6组播数据报文。

命令格式

multicast ipv6 reject inbound-data

undo multicast ipv6 reject inbound-data

参数说明

无

接口视图

缺省级别

2：配置级

使用指南

注意：

使用multicast ipv6 reject inbound-data命令后，接口忽略所有到达的IPv6组播数据报文。可能导致组播无法正常运行，请谨慎使用。

该命令不能在MTunnel接口视图下使用。

使用实例

# 禁止接口GE1/0/0接收IPv6组播数据报文。

<Quidway> system-view
[Quidway] interface gigabitEthernet1/0/0
[Quidway-GigabitEthernet1/0/0] multicast ipv6 reject inbound-data
