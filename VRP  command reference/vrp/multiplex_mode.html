multiplex mode

命令功能

multiplex mode命令用来设置AUG的复用模式。

undo multiplex mode命令用来恢复缺省设置。

缺省情况下，AUG的复用模式为au-4。

命令格式

multiplex mode { au-4 | au-3 }

undo multiplex mode

参数说明

au-4：配置AUG通过AU-4得到。

au-3：配置AUG通过AU-3得到。

视图

CPOS接口视图

缺省级别

2：配置级

使用指南

在SDH中，载荷有两种映射/复用的方案：ANSI和ETSI。

ANSI的复用方案为AU-3复用，ETSI的复用方案为AU-4复用。

当CPOS应用在SDH模式下时，可使用multiplex mode命令选择设置AUG复用到AU-4还是AU-3，如果CPOS应用在SONET模式下，则只能复用到AU-3，不能使用multiplex mode命令。

使用实例

# 在SDH模式下，设置AUG复用到AU-3。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] frame-format sdh
[Quidway-Cpos1/0/0] multiplex mode au-3
