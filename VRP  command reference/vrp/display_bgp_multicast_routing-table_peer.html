display bgp multicast routing-table peer

命令功能

display bgp multicast routing-table peer命令用来查看收发给指定组播对等体的路由信息。

命令格式

display bgp multicast routing-table peer peer-address { advertised-routes | received-routes [ active ] } [ statistics ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

peer-address：指定MBGP对等体的地址。点分十进制形式。

advertised-routes：该对等体发送给指定对等体的路由信息。

received-routes：该对等体收到的来自指定对等体的路由信息。

active：显示从指定对等体收到的活跃路由信息。

statistics：组播路由表的路由统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
