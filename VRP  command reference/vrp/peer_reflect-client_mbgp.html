peer reflect-client（MBGP）

命令功能

peer reflect-client命令用来配置将本机作为路由反射器，并将对等体（组）作为路由反射器的客户。

undo peer reflect-client命令用来取消已有的配置。

缺省情况下，自治系统内没有配置路由反射器。

命令格式

peer { group-name | peer-address } reflect-client

undo peer { group-name | peer-address } reflect-client

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

视图

BGP-IPv4组播地址族视图

缺省级别

2：配置级

使用指南

为确保MBGP对等体间的互通，通常需要在IBGP对等体间建立全连接网络。对于内部MBGP网络非常大的网络，建立此全连接网络将耗资巨大。

VRP允许用户配置路由反射器，从而能够解决此问题。设置其中一个MBGP对等体为路由反射器，其它MBGP对等体作为客户机（Client）。每个客户机都和路由反射器建立对等体关系，彼此交换路由信息。路由反射器作为MBGP路由汇聚的核心，在客户机之间传递（反射）路由信息。

使用实例

# 配置将对等体组test作为该路由反射器的客户机。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] peer test enable
[Quidway-bgp-af-multicast] peer test reflect-client
