display mpls switch-l2vc

命令功能

display mpls switch-l2vc命令用来显示指定的或者所有的交换PW信息（纯静态、纯动态或者静动混合PW交换）。

命令格式

display mpls switch-l2vc [ ip-address vc-id encapsulation { encapsulation-type | ip-interworking | ip-layer2 } | state { down | up } ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

ip-address：PW对端设备的LSR-ID。

vc-id：虚电路号。十进制整数形式，取值范围是1～4294967295。

encapsulation-type：PW的封装类型，可以为atm-1to1-vcc、atm-1to1-vpc、atm-aal5-pdu、atm-aal5-sdu、atm-nto1-vcc、atm-nto1-vpc、atm-trans-cell、ethernet、fr、hdlc、ppp、或vlan。

ip-interworking：如果进行异种介质互通的设备都是华为设备，选择参数 ip-interworking来显示交换PW的信息。

ip-layer2：如果与其他设备制造商的设备进行异种介质互通，选择参数ip-layer2来显示交换PW的信息。

state：对等体之间VC的状态信息。

down：状态为Down的VC信息。

up：状态为Up的VC信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display mpls switch-l2vc命令后，如果没有指定参数则显示所有的交换信息。

项目

描述

Switch-l2vc Type

交换类型，可以为 LDP<-->LDP、LDP-SVC或者SVC<-->SVC。

Peer ip address

交换的两个对端IP地址，分左右显示。

VC ID

交换的两个虚电路号。

VC Type

VC的封装类型。

VC State
VC的状态：

  • up表示VC建立成功；

  • down表示VC建立不成功。

In/Out Label

两侧的入/出标签。

Control Word

两端控制字的使能状态。Disable表示没有使能控制字，Enable表示使能了控制字。

VCCV Capality

两侧的VC连接检测的使能状态。Disable表示没有使能VC连接检测，Enable表示使能了VC连接检测。

Switch-l2vc tunnel info:1 tunnels for peer 3.3.3.9NO.0 TNL Type : lsp , TNL ID : 0x220001 tunnels for peer 1.1.1.9NO.0 TNL Type : lsp, TNL ID : 0x22002

两端的隧道信息。TNL Type表示隧道类型。TNL ID表示隧道ID，系统自动分配。

Create Time

该交换VC已经创建了多长时间。

Up Time

该交换VC保持UP状态多长时间。

Last Change Time

该交换VC最后一次状态变化距现在多长时间。

VC last up time :

VC最近一次UP的时刻。

VC total up time:

VC累积UP的时间（除去中间DOWN的时间）。
