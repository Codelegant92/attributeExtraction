mld timer other-querier-present

命令功能

mld timer other-querier-present命令用来设置其他MLD查询器的存活时间。

undo mld timer other-querier-present命令用来恢复其他MLD查询器的存活时间为缺省值。

缺省情况下，其他MLD查询器的存活时间的计算公式是：其他MLD查询器的存活时间 = 健壮系数 × MLD普遍查询消息发送间隔 +（1/2）× MLD查询消息的最大响应时间。当健壮系数、MLD查询消息发送间隔和MLD查询消息最大响应时间都取缺省值时，其他MLD查询器的存活时间的值为255秒。

命令格式

mld timer other-querier-present interval

undo mld timer other-querier-present

参数说明

interval：指定“其他MLD查询器的存活时间”。整数形式，取值范围是60～300，单位是秒。

使用指南

当一个网段中连接了多台组播路由器时，从中选出一台作为查询器，负责向该网段周期性发送查询消息。如果非查询器在“其他MLD查询器的存活时间”内没有接收到查询消息，就认为查询器故障，由自己来充当查询器。经过竞选后，最终由共享网段上IPv6的Link-local地址最小的路由器充当查询器。

如果接口视图和MLD视图下都配置了此命令，优先取接口视图下的值。

如果配置的“其他MLD查询器的存活时间”小于“MLD查询消息发送间隔”，将导致网络上查询器的重复改变。

使用实例

# 在接口GE1/0/0上设置MLD查询器的超时时间是200秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld timer other-querier-present 200
