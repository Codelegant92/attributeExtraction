if-match mpls-exp

命令功能

if-match mpls-exp命令用来定义匹配mpls的exp域的规则。

undo if-match mpls-exp命令用来删除匹配mpls的exp域的规则。

命令格式

if-match [ not ] mpls-exp mpls-experimental-value&<1-8>

undo if-match [ not ] mpls-exp

参数说明

mpls-experimental-value：EXP值，取值范围0～7。

视图

流分类视图

缺省级别

2：配置级

使用指南

在该命令中MPLS的EXP值可指定多个，最多可指定8个，如果指定了多个相同的值，只取一个，多个不同的值是或的关系。在同一个类下配置时将需要配置的mpls的exp域规则用同一条命令进行配置，否则只有最后一次配置的if-match mpls-exp命令会生效。命令在配置后，EXP将自动按照从小到大的顺序排序。

使用实例

# 定义类匹配exp为3或4的报文。

<Quidway> system-view
[Quidway] traffic classifier database
[Quidway-classifier-database] if-match mpls-exp 3 4
