ip netstream as-mode

命令功能

ip netstream as-mode { 16 | 32 } 用来配置NeStream采样支持的AS域模式。

undo ip netstream as-mode命令用来恢复NetStream采样支持的缺省AS域模式。

缺省情况下，NetStream采样支持16位的AS域模式。

命令格式

ip netstream as-mode { 16 | 32 }

undo ip netstream as-mode

参数说明

16：16位AS域模式。

32：32位AS域模式。

视图

系统视图

缺省级别

2：配置级

使用指南

针对网络中AS域模式的不同，在配置NetStream时，需要对AS域模式进行转换，否则NetSteam无法采集到AS域间的流量信息。

在使用32位AS域模式的网络中，网管必须要能够识别32位AS域值，否则网管端就无法识别设备发送来的域间流量信息。

使用实例

# 配置NetStream采样使用32AS域模式。

<Quidway> system-view
[Quidway] ip netstream as-mode 32
