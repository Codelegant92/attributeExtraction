x25 switch svc interface

命令功能

x25 switch svc interface命令用来增加一条转发接口为指定接口的X.25交换路由。

undo x25 switch svc interface命令用来删除指定的X.25交换路由。

缺省情况下，未配置X.25交换路由。

命令格式

x25 switch svc x.121-address [ sub-dest destination-address ] [ sub-source source-address ] interface interface-type interface-number

undo x25 switch svc x.121-address [ sub-dest destination-address ] [ sub-source source-address ] interface interface-type interface-number

参数说明

x.121-address：目的X.121地址。此参数为模式匹配字符串，长度为1～15个字符。

sub-dest destination-address：替换目的X.121地址。

sub-source source-address：替换源X.121地址。

interface interface-type interface-number：接口类型和接口编号。

使用指南

配置转发接口为指定接口的X.25交换路由后，相应的X.25的呼叫请求分组将被转发到指定接口上，从而实现X.25协议下的负载分担。

使用实例

# 增加一条目的地址为8888，转发接口为Serial1/0/0的X.25交换路由，同时将目的地址替换为9999。

<Quidway>system-view
[Quidway] x25 switch svc 8888 sub-dest 9999 interface serial1/0/0
