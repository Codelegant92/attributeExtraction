display ip routing-table limit

命令功能

display ip routing-table limit命令用来显示VPN实例的路由和前缀限制信息。

命令格式

display ip routing-table limit { all-vpn-instance | vpn-instance vpn-instance-name }

参数说明

all-vpn-instance：所有的VPN实例。

vpn-instance vpn-instance-name：指定VPN实例的名称。

表1 display ip routing-table limit命令输出信息描述

项目

描述

Limit-Object

限制的对象。包括前缀和路由两种类型。

Limit-Type

当前路由表中路由和前缀限制的方式，包括三种：

  • Simply-Alert：仅产生告警。

  • Alert-Percent：告警百分比。

  • Default：缺省情况，即无路由和前缀限制。

Upper-Limit

当前路由表中路由和前缀限制的上限值。

Warning

当前路由表中路由和前缀限制的阈值。

Current

当前路由表中路由和前缀的数量。
