ppp callback（接口视图）

命令功能

ppp callback命令用来配置本端在Callback功能中的角色。

undo ppp callback命令用来取消本端的Callback功能。

缺省情况下，拒绝回呼的协商。

命令格式

ppp callback { client | server }

undo ppp callback { client | server }

参数说明

client：本端作为回呼的客户端，发送回呼请求。

server：本端作为回呼的服务器端，接收回呼请求。

视图

接口视图

缺省级别

2：配置级

使用指南

回呼功能只在低速串口和Dialer接口上可用。

利用回呼功能可以为主叫方节省传输费用（使用被叫计费）。

使用实例

# 配置本端为Callback的客户端。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] ppp callback client
