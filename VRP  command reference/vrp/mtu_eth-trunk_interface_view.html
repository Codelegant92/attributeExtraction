mtu（Eth-Trunk接口视图）

命令功能

mtu命令用来设置Eth-Trunk接口的最大传输单元。

undo mtu命令用来恢复缺省设置。

缺省情况下，无论采用Ethernet_II帧格式还是Ethernet_SNAP帧格式，MTU均为1500字节。

命令格式

mtu mtu

undo mtu

参数说明

mtu：以太网接口最大传输单元，取值范围为46～1560，单位为字节。不同产品Eth-Trunk接口的MTU取值范围不同。

视图

Eth-Trunk接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

使用mtu命令改变Eth-Trunk接口最大传输单元MTU后，需要先对Eth-Trunk接口执行shutdown命令，再执行undo shutdown命令将接口重启，以保证设置的MTU生效。

注意：

  • 如果在Eth-Trunk接口上同时运行IPv6，当使用mtu命令设置的接口MTU值小于1280字节时，会使IPv6在该接口上工作不正常。为避免发生此类错误，当以太网接口上同时运行IPv6时，请设置接口MTU值≥1280。

  • 如果使用mtu命令改变接口MTU的值，请同时修改与本设备相连的其他设备的MTU值，确保两端设备的MTU值匹配。否则，可能导致业务中断。

使用实例

# 设置Eth-Trunk接口的最大传输单元为1400。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] mtu 1400
[Quidway-Eth-Trunk1] shutdown
[Quidway-Eth-Trunk1] undo shutdown
