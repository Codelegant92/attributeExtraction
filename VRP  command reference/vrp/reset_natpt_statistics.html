reset natpt statistics

命令功能

reset natpt statistics命令用来清除相关接口板上的NAT-PT统计信息。如果未指定槽位号，清除所有已使能NAT-PT的板上的统计信息。

命令格式

reset natpt statistics [ slot slot-id ]

参数说明

slot-id：清除分布式环境中相关接口板上的NAT-PT统计信息。
