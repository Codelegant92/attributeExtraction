ospfv3 timer hello

命令功能

ospfv3 timer hello命令用来设置接口实例上发送Hello报文的时间间隔。

undo ospfv3 timer hello命令用来恢复接口实例上该时间间隔的缺省值。

缺省情况下，P2P、Broadcast类型接口发送Hello报文的时间间隔的值为10秒；暂时还不支持P2MP、NBMA类型接口。

命令格式

ospfv3 timer hello interval [ instance instance-id ]

undo ospfv3 timer hello [ instance instance-id ]

参数说明

interval：接口发送Hello报文的时间间隔。整数形式，取值范围是1～65535，单位是秒。

instance instance-id：接口所属的实例ID。整数形式，取值范围是0～255。缺省值是0。
