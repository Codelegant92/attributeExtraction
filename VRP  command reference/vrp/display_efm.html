display efm

命令功能

display efm命令用来查看接口的EFM OAM配置信息，包括：

  • EFM OAM功能的使能状态

  • EFM OAM工作模式

  • EFM OAM协议报文的最大尺寸

  • EFM OAM误码检测功能的使能状态

  • EFM OAM误码检测观察时长

  • EFM OAM误码检测门限

  • EFM OAM误帧检测功能的使能状态

  • EFM OAM误帧检测观察时长

  • EFM OAM误帧检测门限

  • EFM OAM误帧秒检测功能的使能状态

  • EFM OAM误帧秒检测观察时长

  • EFM OAM误帧秒检测门限

  • 是否配置了EFM OAM闪断功能

  • 远端接口的MAC地址

  • 远端接口的EFM OAM功能的使能状态

  • 远端接口的EFM OAM工作模式

  • 远端接口的EFM OAM报文的最大尺寸

命令格式

display efm { all | interface interface-type interface-number }

参数说明

all：表示显示所有配置了EFM OAM功能的接口的EFM OAM配置信息。

interface interface-type interface-number：显示指定接口的EFM OAM配置信息。interface-type表示接口类型，取值包括：ethernet和gigabitethernet。interface-number表示接口编号。

使用指南

当需要查看接口的所有EFM OAM配置信息时，可使用本命令。

使用efm enable命令全局使能EFM OAM功能后才能使用本命令。

使用本命令只显示配置了EFM OAM功能的接口的EFM OAM配置信息。

使用实例

# 查看接口GigabitEthernet1/0/1的EFM OAM配置信息。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/1
[Quidway-GigabitEthernet1/0/1] efm enable
[Quidway-GigabitEthernet1/0/1] display efm interface gigabitethernet1/0/1

表1 display efm命令输出信息描述表

项目

描述

EFM Enable Flag
接口的EFM OAM功能的使能状态。

  • disable：接口的EFM OAM功能处于禁止状态。

  • enable：接口的EFM OAM功能处于使能状态。

Mode
接口的EFM OAM工作模式。

  • active：接口的EFM OAM工作模式为主动模式。

  • passive：接口的EFM OAM工作模式为被动模式。

OAMPDU MaxSize

接口的EFM OAM协议报文的最大尺寸。取值范围是64～1518，单位是字节，缺省值是128字节。

ErrCodeNotification
接口的EFM OAM误码检测功能的使能状态。

  • disable：接口的EFM OAM误码检测功能处于禁止状态。

  • enable：接口的EFM OAM误码检测功能处于使能状态。

ErrorCodePeriod

接口的EFM OAM误码检测观察时长，取值范围是1～60，单位是秒，缺省值是1秒。

ErrorCodeThreshold

接口的EFM OAM误码检测门限，取值范围是0～65535，缺省值是1。

ErrFrameNotification
接口的EFM OAM误帧检测功能的使能状态。

  • disable：接口的EFM OAM误帧检测功能处于禁止状态。

  • enable：接口的EFM OAM误帧检测功能处于使能状态。

ErrFramePeriod

接口的EFM OAM误帧检测观察时长，取值范围是1～60，单位是秒。缺省值是1秒。

ErrFrameThreshold

接口的EFM OAM误帧检测门限，取值范围是0～65535，缺省值是1。

ErrFrameSecondNotification
接口的EFM OAM误帧秒检测功能的使能状态。

  • disable：接口的EFM OAM误帧秒检测功能处于禁止状态。

  • enable：接口的EFM OAM误帧秒检测功能处于使能状态。

ErrFrameSecondPeriod

接口的EFM OAM误帧秒检测观察时长，取值范围是10～900，单位是秒，缺省值是60秒。

ErrFrameSecondThreshold

接口的EFM OAM误帧秒检测门限，取值范围是0～900，缺省值是1。

TriggerIfDown
接口检测到和对端之间的链路发生故障后是否闪断该接口。

  • disable：接口发生越限事件后不闪断该接口。

  • enable：接口发生越限事件后闪断该接口。

Remote MAC

远端接口的MAC地址。

Remote EFM Enable Flag
远端接口的EFM OAM功能的使能状态。

  • disable：接口的EFM OAM功能处于禁止状态。

  • enable：接口的EFM OAM功能处于使能状态。

Remote Mode
远端接口的EFM OAM工作模式。

  • active：接口的EFM OAM工作模式为主动模式。

  • passive：接口的EFM OAM工作模式为被动模式。

Remote MaxSize

远端接口的EFM OAM协议报文的最大尺寸。单位是字节。
