records

命令功能

records命令用来配置NQA测试的历史、结果记录最大数目。

undo records命令用来恢复缺省配置。

命令格式

records { history number | result number }

undo records { history | result }

参数说明

history number：设置历史统计最大个数。取值范围是0～1000，缺省值是50。

result number：设置结果统计最大个数。取值范围是1～10，缺省值是5。

使用指南

不能更改正在运行的测试例的此项参数。path MTU测试不支持设置records history。

使用实例

# 设置测试例user test的历史统计最大个数为1000。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] records history 1000
