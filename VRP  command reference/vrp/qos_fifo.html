qos fifo

命令功能

qos fifo queue-length命令用来设定FIFO队列的长度限制。

undo qos fifo queue-length命令用来恢复队列长度的缺省值。

命令格式

qos fifo queue-length queue-length

undo qos fifo queue-length

参数说明

queue-length：队列的长度限制。取值范围是1～1024。队列的缺省长度限制为256。

使用指南

在网络中传输的业务相对简单，对实时性、可靠性要求不高，则可以采用先进先出的队列来作为队列调度机制。

使用实例

# 下面命令把FIFO的队列长度设置为100。

<Quidway> system-view
[Quidway] interface GigabitEthernet3/0/0
[Quidway-GigabitEthernet3/0/0] qos fifo queue-length 100
