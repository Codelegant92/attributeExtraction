display pim ipv6 neighbor

命令功能

display pim ipv6 neighbor命令用来查看IPv6 PIM邻居信息。

命令格式

display pim ipv6 neighbor [ ipv6-link-local-address | interface interface-type interface-number | verbose ] * [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv6-link-local-address：指定邻居的IPv6链路本地地址。

interface interface-type interface-number：指定接口类型和编号。

verbose：显示邻居的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

VPN-Instance

PIM邻居信息所属的VPN实例

Total Number of Neighbors on this interface

接口上的PIM邻居总数

Neighbor

PIM邻居的地址

Interface

PIM邻居所在的接口

Uptime

PIM邻居已经存在的时间

Expiry time

PIM邻居还有多少时间就要超时

DR Priority

DR优先级

Generation ID

PIM邻居状态随机数

Holdtime

PIM邻居生存时间

LAN delay

传递Prune剪枝消息的延迟时间

Override interval

否决Prune剪枝的时间间隔

Neighbor tracking

是否使能邻居跟踪功能

Neighbor Secondary Address(es)

邻居IPv6地址
