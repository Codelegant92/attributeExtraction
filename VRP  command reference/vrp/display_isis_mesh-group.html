display isis mesh-group

命令功能

display isis mesh-group命令用来显示IS-IS的mesh-group配置情况。

命令格式

display isis mesh-group [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 配置路由器上运行IS-IS的pos1/0/0接口和pos2/0/0接口属于mesh-group 100。

[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] isis mesh-group 100
[Quidway-Pos1/0/0] quit
[Quidway] interface pos 2/0/0
[Quidway-Pos2/0/0] isis mesh-group 100

# 显示配置的IS-IS mesh-group的信息。

<Quidway> display isis mesh-group
               Mesh Group information for ISIS
----------------------------------------------------------------------
Interface          Status
Pos1/0/0            100
Pos2/0/0            100
