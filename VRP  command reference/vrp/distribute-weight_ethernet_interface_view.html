distribute-weight（Ethernet接口视图）

命令功能

distribute-weight命令用来配置成员端口的负载分担权重。

undo distribute-weight命令用来删除配置的端口负载分担权重。

缺省情况下，成员端口的负载分担权重为1。

命令格式

distribute-weight weight-value

undo distribute-weight

参数说明

weight-value：捆绑组成员端口的权重，取值范围是1～16。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

一个捆绑口最多允许16个负载分担项，且一个捆绑组中所有成员的权重之和不能超过16。

使用实例

# 设置Eth-Trunk成员端口GE1/0/0的负载分担权重为3。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] distribute-weight 3
