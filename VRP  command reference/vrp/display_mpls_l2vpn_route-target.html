display mpls l2vpn (Route-Target)

命令功能

该命令用来查看BGP VPN的Route-Target列表。

命令格式

display mpls l2vpn { export-route-target-list | import-route-target-list } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

export-route-target-list：BGP VPN出方向的Route-Target列表。

import-route-target-list：BGP VPN入方向的Route-Target列表。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当定位Kompella方式的L2VPN故障时，如果连接状态为Up，但CE之间转发不通，可以使用display mpls l2vpn { export-route-target-list | import-route-target-list }命令查看的VPN实例的Route-Target列表，检查两端PE的Route-Target是否匹配

  • PE上的出方向Route-Target必须与对端PE的入方向Route-Target一致。

  • PE上的入方向Route-Target必须与对端PE的出方向Route-Target一致。

使用实例

# 显示BGP VPN接收的路由目标列表。

<Quidway> display mpls l2vpn import-route-target-list
import vpn target list: 744:7  745:7  746:7  888:8

# 显示BGP VPN发送的路由目标列表。

<Quidway> display mpls l2vpn export-route-target-list
export vpn target list: 755:7  888:8
