datafill

命令功能

datafill命令用来配置NQA测试例的填充字符。

undo datafill命令用来取消NQA测试例的填充字符。

命令格式

datafill fillstring

undo datafill

参数说明

fillstring：NQA测试报文的填充字符串。长度范围是1～230。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对UDP、Jitter、ICMP、Trace、LSPping、LSPJitter、Pwe3ping、Mping、ICMP Jitter、Path Jitter类型的测试例配置填充字符。

不能修改正在执行的测试例的填充字符。

对于ICMP Jitter类型的测试例，只有ICMP echo模式下，可以配置测试报文的填充字符串。

说明：

填充时，

  • 如果测试例发送数据包的大小比配置的填充数据小，只使用填充字符串的前一部分。

  • 如果测试发送数据包的大小比配置的填充数据大，则使用填充字符串循环填充。

例如，配置填充字符串为“abcd”，如果测试数据包大小为3，则只使用“abc”填充；如果测试数据包大小为6，则使用“abcdab”填充。

使用实例

# 设置测试例user test的填充字符串为“abcd”。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] datafill abcd
