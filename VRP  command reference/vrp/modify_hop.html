modify hop

命令功能

modify hop命令用来修改显式路径中的IP地址。

缺省情况下，修改后的指定节点为include，是严格下一跳strict。

命令格式

modify hop ip-address1 ip-address2 [ exclude | include [ loose | strict ] ]

参数说明

ip-address1 ip-address2：将显式路径中的IP地址ip-address1修改为ip-address2。

exclude：显式路径中建立的LSP不能经过修改后的链路或节点。

include：显式路径中建立的LSP必须经过修改后的链路或节点。

loose：松散显式路径，即，修改后的指定节点与上一节点可以不是直连的。

strict：严格显式路径，即，修改后的指定节点与上一节点必须是直连的。

显式路径视图

缺省级别

2：配置级

使用指南

执行此命令修改显式路径中的IP地址时，不能使用该显式路径中已经存在的下一跳IP地址，且需保证本设备上没有任何Tunnel接口在使用该显式路径。

使用实例

# 将MPLS TE显式路径中的IP地址10.0.0.125修改为10.0.0.200。

<Quidway> system-view
[Quidway] explicit-path p1
[Quidway-explicit-path-p1] modify hop 10.0.0.125 10.0.0.200 exclude
