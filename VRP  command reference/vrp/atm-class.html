atm-class

命令功能

atm-class命令用来将一组参数（这些参数在ATM-Class中定义）应用于一个ATM接口或一条PVC。

undo atm-class命令用来取消指定的ATM-Class。

命令格式

atm-class atm-class-name

undo atm-class atm-class-name

参数说明

atm-class-name：ATM-Class名。字符串形式，不支持空格，区分大小写，长度为1～16。

视图

ATM接口视图、ATM-PVC视图

缺省级别

2：配置级

使用指南

如果多个ATM主接口或PVC均需要配置MAP、业务类型、封装类型、OAM，且参数相同时，可以在ATM主接口或PVC视图下调用ATM-Class。这样就不需要每个ATM主接口或PVC进行重复配置。

当需要调用某ATM-Class时，该ATM-Class必须已经存在。

使用实例

# 将一个名为“main”的ATM-Class应用于接口Atm1/0/0。

<Quidway>system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] atm-class main
