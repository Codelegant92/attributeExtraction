display dns domain

命令功能

display dns domain命令用来查看域名后缀的配置和顺序。

命令格式

display dns domain [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在完成DNS的配置后，使用display dns domain命令可以查看域名后缀的配置和顺序是否匹配正确。

使用实例

# 查看域名后缀列表。

<Quidway> display dns domain
No                    Domain-name
1                     com
2                     net
表1 display dns domain命令输出描述

项目

描述

No

编号，表示域名后缀的配置顺序。

Domain-name

显示配置的域名后缀。本例中有2个域名后缀，解析的时候，首先加入第一个域名后缀com，然后发送到各个DNS服务器上进行查找；如果没有回应，会将信息重发一次；如果仍然发送不成功，会发第三次；如果第三次还是没有得到正确的回应报文，就使用第二个后缀net进行匹配形成新的名称，然后发送到各个DNS服务器上进行查找。
