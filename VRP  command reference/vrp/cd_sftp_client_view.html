cd（SFTP客户端视图）

命令功能

cd命令用来改变SSH用户的当前工作目录。

命令格式

cd [ path ]

参数说明

path：SFTP服务器上的目录名，字符串形式，绝对路径名长度范围为1～64。

视图

SFTP客户端视图

缺省级别

2：配置级

使用指南

path在服务器上必须存在。如果该命令不带参数path，只显示SSH用户的当前工作目录。

使用实例

# 改变SSH用户的当前工作目录为directory-test。

<Quidway> system-view
[Quidway] sftp 10.164.39.222
Please input the username:client001
Trying 10.164.39.222 ...
Press CTRL+K to abort
Connected to 10.164.39.222 ...
Enter password:
sftp-client> cd directory-test
Current directory is:
cfcard:/directory-test
