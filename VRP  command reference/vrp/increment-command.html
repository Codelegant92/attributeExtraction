increment-command

命令功能

increment-command命令用来修改和增加增量配置信息。

undo increment-command命令用来删除增量配置信息。

命令格式

increment-command { command-number command-number }command-text command-text

undo increment-command { command-number command-number | all }

参数说明

command-number command-number：增量配置的编号，十进制整数形式，取值范围是1～500。

command-text command-text：增量配置的命令，字符串形式，支持空格，长度范围是0～200。

all：所有的增量配置信息。

视图

增量配置视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

使用本命令时：如果不指定增量配置的编号，新的增量配置编号为现存最大增量配置编号除以10取整后再乘以10加10。例如，当前现存最大增量配置编号是123，123除以10取整为12，如果不指定增量配置的编号，那么新的增量配置编号是12×10＋10＝130。

说明：

增量配置命令会从命令交换机的系统视图下开始执行，因此用户必须保证增量配置命令的连续性，如果相邻两条命令不在同一个视图下，那么就必须增加切换视图的增量配置命令，否则会导致增量配置命令执行失败。

增量配置命令列表编辑完成后，可以根据增量配置的编号对命令列表重新进行修改。

使用实例

# 增加增量配置信息，准备在成员交换机上创建VLAN1和VLAN5。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] increment
[Quidway-cluster-increment] increment–command command-number 10 command-text vlan 1
[Quidway-cluster-increment] increment–command command-number 20 command-text vlan 5
