mtu（ATM接口视图）

命令功能

mtu命令用来设置ATM接口最大传输单元（MTU）的大小。

undo mtu命令用来恢复其缺省值。

命令格式

mtu mtu-number

undo mtu

参数说明

mtu-number：ATM接口MTU的大小。取值范围是128～16352，单位是字节。缺省值是1500字节。

使用指南

注意：

当接口上运行IPv6时，如果接口的MTU值小于1280字节，可能会使IPv6在该接口上工作不正常。所以当接口上运行IPv6时，建议设置该接口的MTU值不小于1280。

ATM接口的MTU只影响IP层在ATM接口的组包和拆包。由于QoS队列长度的限制（如FIFO队列的缺省长度为75），MTU太小会造成分片太多，从而被QoS队列丢弃。此时，可适当增大QoS队列的长度。FIFO是PVC缺省使用的队列调度机制，可以通过PVC视图下的命令qos fifo queue-length改变其队列长度。

说明：

  • 本命令可以同时在ATM主接口和ATM子接口配置，但是互不影响。

  • 改变MTU后必须重新启动接口，保证配置生效。

使用实例

# 设置ATM接口Atm1/0/0的MTU为1492字节，并重启接口。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] mtu 1492
[Quidway-Atm1/0/0] shutdown
[Quidway-Atm1/0/0] undo shutdown
