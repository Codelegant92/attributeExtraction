mc-trunk（Eth-Trunk接口视图）

命令功能

mc-trunk命令用来将静态LACP模式的Eth-Trunk加入到指定MC-Trunk中。

undo mc-trunk命令用来删除指定MC-Trunk中的Eth-Trunk。

命令格式

mc-trunk mc-trunkid

undo mc-trunk

参数说明

mc-trunk mc-trunkid：MC-Trunk的编号。整数形式，取值范围为1～16。

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

只有静态LACP模式的Eth-Trunk可以加入MC-Trunk。一个Eth-Trunk只能加入一个MC-Trunk。如果该Eth-Trunk已经加入其他MC-Trunk时，必须退出原有MC-Trunk，才能加入新的MC-Trunk。

说明：
一个MC-Trunk最多可以加入64个静态LACP模式的Eth-Trunk。

静态LACP模式Eth-Trunk加入MC-Trunk后，该Eth-Trunk的工作模式不可修改。

两台设备上同一MC-Trunk下所加入的Eth-Trunk ID必须一致。例如，设备A上加入MC-Trunk1中的有Eth-Trunk 1和Eth-Trunk 2。那么在设备B上加入MC-Trunk1中必须是Eth-Trunk 1和Eth-Trunk 2。

使用实例

# 将Eth-Trunk 1加入ID为1的MC-Trunk。

<Quidway> system-view
[Quidway] interface Eth-Trunk 1
[Quidway-Eth-Trunk1] mc-trunk 1
