ospfv3 timer poll

命令功能

ospfv3 timer poll命令用来配置NBMA网络上发送轮询Hello报文的时间间隔。

undo ospfv3 timer poll命令用来恢复发送轮询Hello报文间隔的缺省值。

缺省情况下，时间间隔为120秒。

命令格式

ospfv3 timer poll interval [ instance instance-id ]

undo ospfv3 timer poll [ instance instance-id ]

参数说明

interval：发送轮询Hello报文的时间间隔。整数形式，取值范围是1～65535，单位是秒。

instance instance-id：指定接口所属的实例ID。整数形式，取值范围是0～255。

使用指南

在NBMA网络上，当邻居失效后，将按ospfv3 timer poll设置的轮询时间间隔定期地发送Hello报文。轮询时间间隔值至少应为Hello报文时间间隔的4倍。

OSPFv3协议不支持NULL接口的配置。

使用实例

# 配置接口POS2/0/0上发送轮询Hello报文的时间间隔为130秒。

<Quidway> system-view 
[Quidway] interface pos 2/0/0
[Quidway-Pos2/0/0] ospfv3 timer poll 130
