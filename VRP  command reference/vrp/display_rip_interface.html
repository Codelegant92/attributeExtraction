display rip interface

命令功能

display rip interface命令用来显示RIP的接口信息。

命令格式

display rip process-id interface [ interface-type interface-number ] [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIP进程号。整数形式，取值范围是1～65535。

interface-type：接口类型，如GigabitEthernet、Loopback等。

interface-number：接口号。

verbose：查看RIP接口的详细信息

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令的输出信息有助于查看RIP配置及运行状态，进行故障诊断并验证配置效果。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看GigabitEthernet1/0/0接口上RIP的信息。

<Quidway> display rip 1 interface gigabitethernet1/0/0
 --------------------------------------------------------------------------
 Interface                 IP Address      State    Protocol            MTU
 --------------------------------------------------------------------------
 GigabitEthernet 1/0/0     1.1.1.2         UP       RIPv1 Compatible    500
