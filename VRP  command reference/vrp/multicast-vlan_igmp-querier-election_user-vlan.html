multicast-vlan igmp-querier-election user-vlan

命令功能

multicast-vlan igmp-querier-election user-vlan命令用来使能用户VLAN查询器选举。

undo multicast-vlan igmp-querier-election user-vlan命令用来去使能用户VLAN查询器选举。

缺省情况下，不使能查询器选举。

命令格式

multicast-vlan igmp-querier-election user-vlan{ vlan-id1 [ to vlan-id2 ] } &<1-10>

undo multicast-vlan igmp-querier-election user-vlan{ {vlan-id1 [to vlan-id2 ] } &<1-10> | all }

参数说明

vlan-id1：表示用户VLAN的编号，要求小于vlan-id2。整数形式，取值范围是1～4094。

vlan-id2：表示用户VLAN的编号，要求大于vlan-id1。整数形式，取值范围是1～4094。

all：表示所有的用户VLAN。

视图

组播VLAN视图

缺省级别

2：配置级

使用指南

为了在跨VLAN复制时避免用户收到两份组播流，使能用户VLAN的查询器选举功能。

从组播VLAN复制到用户VLAN过程中，查询报文的源IP地址会替换成路由器地址。如果用户VLAN收到两份不同源地址的查询报文，就会启动选举功能禁止其中一个发送查询报文。

使用实例

# 使能用户VLAN 100查询器选举功能

<Quidway> system-view
[Quidway] vlan 10
[Quidway-vlan10] multicast-vlan igmp-querier-election user-vlan 100
