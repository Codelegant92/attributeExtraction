display isis route

命令功能

display isis route命令用来查看IS-IS路由信息。

命令格式

display isis route [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ ipv4 ] [ verbose | [ level-1 | level-2 ] | ip-address [ mask | mask-length ] ] * [ | count ] [ | { begin | exclude | include } regular-expression ]

display isis route [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] ipv6 [ verbose | [ level-1 | level-2 ] | ipv6-address [ prefix-length ] ] * [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称，，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

ipv4：显示IPv4的路由信息。

ip-address：指定IP地址，点分十进制格式。

mask：指定子网掩码，点分十进制格式。

mask-length：指定掩码长度。

ipv6：显示IPv6的路由信息。

ipv6-address：IPv6目的地址。

prefix-length：IPv6地址前缀长度。

level-1：显示Level-1的IS-IS路由。

level-2：显示Level-2的IS-IS路由。

verbose：显示路由的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果没有指定ipv4和ipv6，则显示IPv4和IPv6的IS-IS路由信息。

如果不指定级别，则既显示Level-1也显示Level-2的路由信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
