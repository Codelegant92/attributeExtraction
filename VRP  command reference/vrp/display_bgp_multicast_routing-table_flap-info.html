display bgp multicast routing-table flap-info

命令功能

display bgp multicast routing-table flap-info命令用来查看MBGP的振荡信息。

命令格式

display bgp multicast routing-table flap-info [ ip-address [ mask [ longer-match ] | mask-length [ longer-match ] ] | as-path-filter as-path-filter-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp multicast routing-table flap-info regular-expression as-regular-expression

参数说明

ip-address：表示IPv4网络地址。点分十进制形式。

mask：表示IPv4网络地址的掩码。

mask-length：表示IPv4网络地址的掩码长度。

longer-match：表示使用地址前缀最长匹配原则。

regular-expression as-regular-expression：显示匹配AS路径正则表达式的路由振荡统计信息。其中as-regular-expression是指定AS路径正则表达式。

as-path-filter as-path-filter-number：显示指定AS路径列表号的路由振荡统计信息。as-path-filter-number：指定匹配的AS路径列表号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
