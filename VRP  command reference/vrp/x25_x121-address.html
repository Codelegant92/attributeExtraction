x25 x121-address

命令功能

x25 x121-address命令用来为一个X.25接口配置X.121地址。

undo x25 x121-address命令用来删除X.121地址。

命令格式

x25 x121-address x.121-address

undo x25 x121-address

参数说明

x.121-address：接口的X.121地址。整数形式，格式为1～15位的字符串。

接口视图

缺省级别

2：配置级

使用指南

如果将路由器接入X.25公共分组网，则接入服务商必须为其分配一个合法的X.121地址；如果只是简单地使用两台路由器背靠背直连，则这个合法的X.121地址可以由用户随意指定；如果只是让路由器工作于交换方式，则可以不配X.121地址。

当为一个X.25接口重新配置X.121地址时，不必先删除原来的X.121地址，新的地址会自动覆盖旧的地址；但重新配置一个X.25接口后，原来配置的X.121地址会被删除，只有在重新配置后，该地址才会重新生效。

注：关于X.121地址的格式、IP地址和X.121地址的动态转换请参考《ITU-T 建议 X.121》和相关的RFC文档。

使用实例

# 配置接口Serial1/0/0的链路层协议为X.25，并配置X.121地址为20112451。

<Quidway>system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0]link-protocol x25
[Quidway-Serial1/0/0] x25 x121-address 20112451
