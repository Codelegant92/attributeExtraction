reset recycle-bin

命令功能

reset recycle-bin命令用来彻底删除当前目录下回收站中的文件。

命令格式

reset recycle-bin [ filename | devicename | slavedevicename ]

参数说明

filename：要删除的文件名。字符串形式，长度范围是1～64。支持“*”通配符。

devicename：要删除的存储设备名。可以是flash、hda1或者CF卡。

slavedevicename：要删除的备板存储设备名。可以是flash、hda1或者CF卡。

使用指南

用delete（用户视图）命令删除的文件将文件放在回收站中，如果要真正删除此文件，可以在当前目录下使用reset recycle-bin命令。

使用实例

# 删除回收站中的文件。

<Quidway> reset recycle-bin
