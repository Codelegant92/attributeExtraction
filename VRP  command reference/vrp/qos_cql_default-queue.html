qos cql default-queue

命令功能

qos cql default-queue命令用来为那些无对应规则的包指定一个缺省队列。

undo qos cql default-queue命令用来取消配置，恢复缺省值。

命令格式

qos cql cql-index default-queue queue-number

undo qos cql cql-index default-queue

参数说明

cql-index：定制列表的组号，取值范围是1～16。

queue-number：队列号，取值范围是1～16，缺省队列号为1。

使用指南

在进行流分类的时候，如果数据包不与任何规则匹配，则进入缺省队列。

使用实例

# 指定定制列表第5组的缺省队列为2。

<Quidway> system-view
[Quidway] qos cql 5 default-queue 2
