discriminator（MP-Group接口视图）

命令功能

discriminator命令用来使能终端标识符协商。

undo discriminator命令用来去使能终端标识商。

缺省情况下，使能终端标识符协商。

命令格式

discriminator

undo discriminator

参数说明

无

视图

MP-Group接口视图

缺省级别

2：配置级

使用指南

只有当两端MP Group的终端标识符配置相同时，LCP的状态才可能UP。如果配置不同，则配置undo discriminator的一端，发送参数中不携带终端标识符，但同意对端的终端标识符参数。

如果对MP Group接口执行undo discriminator命令，必须对MP Group中的子通道执行shutdown、undo shutdown后配置才能生效。

使用实例

# 在MP-Group1/0/1接口上去使能终端标识符协商。

<Quidway> system-view
[Quidway] interface mp-group 1/0/1
[Quidway-Mp-group1/0/1] undo discriminator
