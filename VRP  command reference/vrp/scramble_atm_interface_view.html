scramble（ATM接口视图）

命令功能

scramble命令用来使能ATM接口对载荷的加扰功能。

undo scramble命令用来禁止加扰功能。

缺省情况下，使能ATM接口对载荷的加扰功能。

命令格式

scramble

undo scramble

参数说明

无

视图

ATM接口视图

缺省级别

2：配置级

使用指南

scramble命令对只对载荷进行加扰和解扰，不影响信元头。

加扰可有效抑制数据中的多个“0”和多个“1”。

在配置加扰功能时，通信两端必须配置一致。

使用实例

# 禁止ATM接口对载荷的加扰功能。

<Quidway> system-view
[Quidway] interface atm 4/0/0
[Quidway-Atm4/0/0] undo scramble
