display firewall blacklist

命令功能

display firewall blacklist命令显示防火墙黑名单功能的运行状况和黑名单表项信息。

命令格式

display firewall blacklist { item [ source-address ] slot slot-id | enable }

参数说明

item：显示黑名单表项内容。

source-address：要显示的表项的IP地址。

enable：显示黑名单功能运行状况。

slot-id：为指定的接口板槽号。

使用指南

参数item [ source-address ] slot slot-id显示黑名单表项信息，如果不指定IP地址，显示当前全部黑名单表项的概要信息，如果指定了要显示的IP地址，显示特定黑名单表项的详细信息。由于各接口板上可能会动态产生黑名单，所以各接口板的黑名单不一定相同。

display firewall blacklist命令显示的信息包括：黑名单上自动、手工添加条目的信息。

使用实例

# 显示全部黑名单表项概要信息。

<Quidway> display firewall blacklist item
Firewall blacklist items : 
Current manual insert items : 2
Current automatic insert items : 0
Need aging items : 0
IP                Reason         InsertTime            AgeTime
----------------------------------------------------------------
20.202.16.5       Manual         2004/01/28 15:42:18   0
192.168.1.1       Manual         2004/01/28 15:42:01   0

# 显示特定黑名单表项详细信息。

<Quidway> display firewall blacklist item 192.168.1.1
Firewall blacklist items : 192.168.1.1
 Insert reason : Manual
 Insert time : 2003/06/11 08:04:56
 Age action : Aging
 Age time : 100 minutes

# 显示黑名单表项的运行状况。

<Quidway> display firewall blacklist enable
Blacklist is Disabled.
