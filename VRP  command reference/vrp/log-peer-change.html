log-peer-change

命令功能

log-peer-change命令用来打开IS-IS邻接状态变化的输出开关。

undo log-peer-change命令用来关闭输出开关。

缺省情况下，关闭输出开关。

命令格式

log-peer-change [ topology ]

undo log-peer-change

参数说明

topology：指定IPv6拓扑中的邻接状态变化。

视图

IS-IS视图

缺省级别

2：配置级

使用指南

在terminal monitor开关开启的情况下，当打开IS-IS邻接状态输出开关后，IS-IS邻接状态的变化会输出到配置终端上，直至邻接状态输出开关被关闭。

使用实例

# 在当前路由器上打开IS-IS邻接状态变化的输出开关。

<Quidway> system-view
[Quidway] isis
[Quidway-isis-1] log-peer-change
