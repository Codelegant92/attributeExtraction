ip netstream enable

命令功能

ip netstream enable命令用来使能统计接口下RPF检测或URPF检测后丢弃的流量。

undo ip netstream enable命令用来去使能统计接口下RPF检测或URPF检测后丢弃的流量。

缺省情况下，不使能rpf或urpf统计功能。

命令格式

ip netstream {rpf | urpf } enable [ interface interface-type interface-number ]

undo ip netstream { rpf | urpf } enable [ interface interface-type interface-number ]

参数说明

interface interface-type：指定VLAN下的二层物理接口类型。

interface-number：指定VLAN下的二层物理接口号。

rpf：RPF统计。

urpf：URPFf统计。

缺省级别

使用指南

interface-type和interface-number参数，仅在VLAN-IF接口下可配置。

在配置NetStream统计RPF或URPF检测丢弃流量之前，需要在接口下配置RPF或URPF检测功能。

使用实例

# 在Ethernet 3/0/0接口下配置RPF统计使能。

<Quidway> system-view
[Quidway] interface Ethernet 3/0/0
[Quidway-Ethernet3/0/0] ip netstream rpf enable
