msdp

命令功能

msdp命令用来进入MSDP视图。

undo msdp命令用来清除MSDP视图下的配置。

命令格式

msdp [ vpn-instance vpn-instance-name ]

undo msdp [ vpn-instance vpn-instance-name ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

系统视图

2：配置级

使用指南

无

使用实例

# 进入公网实例MSDP视图。

<Quidway> system-view
[Quidway] multicast routing-enable
[Quidway] msdp
[Quidway-msdp]

# 进入VPN实例mytest的MSDP视图。

<Quidway> system-view
[Quidway] ip vpn-instance mytest
[Quidway-vpn-mytest] route-distinguisher 100:1
[Quidway-vpn-mytest] multicast routing-enable
[Quidway-vpn-mytest] quit
[Quidway] msdp vpn-instance mytest
[Quidway-msdp-mytest]
