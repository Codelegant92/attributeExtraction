broadcast-limit link

命令功能

broadcast-limit link命令用来设置虚拟接口模板支持发送组播或广播报文的最大链路数。

undo broadcast-limit link命令用来恢复缺省设置。

命令格式

broadcast-limit link number

undo broadcast-limit link

参数说明

link number：虚拟接口模板支持发送组播或广播报文的最大链路数。取值范围是0～128。缺省值是30。

视图

虚拟接口模板视图

缺省级别

2：配置级

使用指南

当虚拟接口模板下的链路数目比较多时，组播或广播报文从每条链路上发出会影响系统性能，可以使用broadcast-limit link命令进行限制，当链路数超过限定值时就将组播或广播报文丢弃。

使用实例

# 设置虚拟接口模板1支持发送组播或广播报文的最大链路数为100。

<Quidway> system-view
[Quidway] interface virtual-template 1
[Quidway-Virtual-Template1] broadcast-limit link 100
