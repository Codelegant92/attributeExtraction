reset ip bgp-accounting

命令功能

reset ip bgp-accounting命令用来清除BGP Accounting统计信息。

命令格式

reset ip bgp-accounting [ inbound | outbound ] interface [ interface-type interface-number ]

参数说明

inbound：清除接口入方向上的流量统计信息。

outbound：清除接口出方向上的流量统计信息。

interface interface-type interface-number：接口类型和接口号。

视图

用户视图

缺省级别

2：配置级

使用指南

清除BGP Accounting统计信息后，以前的统计信息将无法恢复，务必仔细确认。

使用实例

# 清除接口POS1/0/0上的BGP Accounting统计信息。

<Quidway> reset ip bgp-accounting interface Pos 1/0/0
