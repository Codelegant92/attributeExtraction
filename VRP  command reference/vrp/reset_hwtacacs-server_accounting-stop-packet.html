reset hwtacacs-server accounting-stop-packet

命令功能

reset hwtacacs-server accounting-stop-packet命令用来清除计费停止报文的统计信息。

命令格式

reset hwtacacs-server accounting-stop-packet { all | ip ip-address }

参数说明

all：清除所有计费停止报文的统计信息。

ip ip-address：清除含指定IP地址的计费停止报文的统计信息，IP地址为点分十进制格式。

视图

用户视图

缺省级别

2：配置级

使用指南

清除统计信息后，以前的统计信息将无法恢复。

使用实例

# 清除所有计费停止报文的统计信息。

<Quidway> reset hwtacacs-server accounting-stop-packet all
