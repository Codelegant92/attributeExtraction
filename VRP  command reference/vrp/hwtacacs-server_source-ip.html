hwtacacs-server source-ip

命令功能

hwtacacs-server source-ip命令用来设置HWTACACS服务器的源IP地址。

undo hwtacacs-server source-ip命令用来删除配置。

缺省情况下，报文的源IP地址是0.0.0.0。

命令格式

hwtacacs-server source-ip ip-address

undo hwtacacs-server source-ip

参数说明

ip-address：IP地址，点分十进制形式。

视图

HWTACACS服务器模板视图

缺省级别

2：配置级

使用指南

可以指定路由器发送的所有HWTACACS报文使用同样的源IP地址。这样，HWTACACS服务器就只用一个IP地址与路由器通信。

使用实例

# 配置HWTACACS服务器源IP地址为10.1.1.1。

<Quidway> system-view
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1] hwtacacs-server source-ip 10.1.1.1
