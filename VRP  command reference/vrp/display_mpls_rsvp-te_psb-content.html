display mpls rsvp-te psb-content

命令功能

display mpls rsvp-te psb-content命令用来查看RSVP-TE PSB（Path State Block）信息。

命令格式

display mpls rsvp-te psb-content [ ingress-lsr-id tunnel-id lsp-id ] [ | { begin | exclude | include } regular-expression ]

参数说明

ingress-lsr-id：入节点的LSR ID，点分十进制格式。

tunnel-id：隧道标识。整数形式，取值范围是0～65535。

lsp-id：LSP ID，取值范围是0～65535。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

表1 display mpls rsvp-te psb content命令输出信息描述

项目

描述

Tunnel Addr

隧道目的地址，格式为 X.X.X.X

Exist time

从该LSP生成到现在的时间间隔

Tunnel ExtID

隧道的扩展ID

Session ID

RSVP会话ID

Ingress LSR ID

入节点设备ID，格式为X.X.X.X

Local LSP ID

本地的LSP ID

Previous Hop

前一跳地址，格式为X.X.X.X

Next Hop

下一跳地址，格式为 X.X.X.X

Incoming / Outgoing Interface

隧道在本设备上所经过的入接口/出接口

InLabel

入标签值

OutLabel

出标签值

Send Message ID

发送的刷新减少消息ID

Recv Message ID

接收的刷新减少消息ID

Session Attribute

RSVP会话的属性

SetupPrio

会话建立优先级

HoldPrio

会话保持优先级

SessionAttrib

RSVP会话属性，例如资源预留风格

FRR Flag
是否使能了MPLS TE快速重路由：
  • Enabled
  • Disabled

Local RRO Flag

本地RRO标识号

FAST_REROUTE Information

MPLS TE快速重路由的相关信息

ERO Information

Explicit Route Object，显式路由信息

L-Type
显式路由类型：
  • Strict
  • Loose

ERO-IPAddress

显式路由地址

ERO-Prefix Len

显式路由地址的前缀长度

RRO Information

Record Route Object，记录路由信息

RRO-C Type

记录C类路由（IPv4/标签）

RRO-IP Address

记录路由的IP地址

RRO-IPPrefixLen

记录路由前缀的长度

SenderTspec Information

发送方业务规格信息

Token bucket rate

流量参数，令牌桶速率（单位：kbit/s）

Token bucket size

流量参数，令牌桶大小

Peak data rate

流量参数：峰值数据速率（单位：kbit/s）

Minimum policed unit

流量参数，最小监管单元

Maximum packet size

流量参数，最大报文长度

CT-BandWidth Information

CT及对应的带宽信息

CT0～CT7 Bandwidth

CT0～CT7的带宽值

Path Message arrive on

消息的入接口及上一跳地址

Path Message sent to NHOP

消息的下一跳地址及出接口

Resource Reservation OK

表示完成了资源预留。只有设置了资源预留标记时才会有此项显示

LSP Statistics Infomation

LSP的统计信息

SendPacketCounter

发送的报文数

RecvPacketCounter

接收的报文数

SendPathounter

发送的Path消息数

RecvPathCounter

接收的Path消息数

SendResvCounter:

发送的Resv消息数

RecvResvCounter

接收的Resv消息数
