display ospf cumulative

命令功能

display ospf cumulative命令用来显示OSPF的统计信息。

命令格式

display ospf [ process-id ] cumulative [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：指定要显示信息的OSPF进程号。如果不指定OSPF进程号，该命令将显示所有的OSPF统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令的输出信息有助于OSPF故障诊断。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
