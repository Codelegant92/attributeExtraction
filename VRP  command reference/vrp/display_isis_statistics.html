display isis statistics

命令功能

display isis statistics命令用来查看IS-IS进程的统计信息。包括从其他IS-IS路由器学到的路由数量、从其他协议引入的路由数量以及本地产生的LSP的数量。

命令格式

display isis statistics [ level-1 | level-2 | level-1-2 ] [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display isis statistics packet [ interface interface-type interface-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display isis statistics smart-hello [ level-1 | level-2 | level-1-2 ] [ process-id | vpn-instance vpn-instance-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display isis process-id statistics [ level-1 | level-2 | level-1-2 | packet ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

packet：IS-IS中各种报文的统计信息。

interface interface-type interface-number：设定需要统计报文的来源接口。

smart-hello：Smart-Hello功能的统计信息。

level-1：IS-IS Level-1的统计信息。

level-2：IS-IS Level-2的统计信息。

level-1-2：IS-IS Level-1-2的统计信息。

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display isis statistics packet [ interface interface-type interface-number ]命令用来查看指定接口下的IS-IS报文统计信息。

display isis process-id statistics packet命令用来查看指定进程下的IS-IS报文统计信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

Total IPv4 Forwarding Routes

基于目的地址统计的IPv4路由数目。如果到达同一目的地的路由有多个下一跳，则按一条路由统计。

Total IPv6 Forwarding Routes

基于目的地址统计的IPv6路由数目。如果到达同一目的地的路由有多个下一跳，则按一条路由统计。

IPv4 Imported Routes

引入的IPv4路由信息

Static

引入的静态路由数量

Direct

引入的直连路由数量

ISIS

引入的IS-IS路由数量

BGP

引入的BGP路由数量

RIP

引入的RIP路由数量

OSPF

引入的OSPF路由数量

IPv6 Imported Routes

引入的IPv6路由信息

RIPng

引入的RIPng路由数量

LSP Source ID

生成该LSP的路由器的System-ID信息

No. of used LSPs

已使用的分片数量
