mpls te ds-te bcm

命令功能

mpls te ds-te bcm命令用来设置DS-TE的带宽约束模型。

undo mpls te ds-te bcm命令用来恢复系统缺省配置。

缺省的DS-TE带宽约束模型取决于许可证文件。

命令格式

mpls te ds-te bcm { extend-mam | mam | rdm }

undo mpls te ds-te bcm

参数说明

extend-mam：设置DS-TE的带宽模型为Extended-MAM模型。

mam：设置DS-TE的带宽模型为最大分配模型MAM（Maximum Allocation Model）。

rdm：设置DS-TE的带宽模型为俄罗斯套娃模型RDM（Russian Dolls Model）。

视图

MPLS视图

缺省级别

2：配置级

使用指南

非标准的DS-TE模式不支持Extended-MAM模型。

当非标准DS-TE模式切换为标准DS-TE模式时，带宽模型不变。标准DS-TE模式切换为非标准DS-TE模式时，切换前如果是Extended-MAM模型，则切换后，系统自动将其改为MAM模型。其他模型切换前后不发生变化。

使用实例

# 设置本设备的DS-TE的带宽约束模型为Extended-MAM模型。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls te ds-te bcm extend-mam
