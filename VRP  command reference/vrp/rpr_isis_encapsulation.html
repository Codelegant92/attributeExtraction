rpr isis encapsulation

命令功能

rpr isis encapsulation命令用来设置RPR对ISIS报文的封装格式。

undo rpr isis encapsulation命令用来恢复RPR对ISIS报文封装格式的缺省值。

缺省情况下，RPR封装ISIS报文的格式为802.2。

命令格式

rpr isis encapsulation { ethernet-2 | dot2 }

undo rpr isis encapsulation

参数说明

ethernet-2：设置RPR封装ISIS报文的格式为Ethernet-2。

dot2：设置RPR封装ISIS报文的格式为802.2。

视图

RPR逻辑接口视图

缺省级别

2：配置级

使用指南

本命令用来设置Ring-If上封装ISIS报文的类型。不同的Ring-If封装格式可以不同，与其他厂商设备互通的时候，兼容性更好。

使用实例

# 设置Ring-If接口用Ethernet-2格式封装ISIS报文。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr isis encapsulation ethernet-2
