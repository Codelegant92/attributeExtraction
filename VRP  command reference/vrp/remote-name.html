remote-name

命令功能

remote-name命令用来配置IKE Peer的名称。

undo remote-name命令用来取消对端名称的设置。

命令格式

remote-name name

undo remote-name

参数说明

name：对端名称，字符串形式，长度范围是1～15。

视图

IKE对等体视图

缺省级别

2：配置级

使用指南

该配置参数在使用于NE16E、NE08E、NE05产品时，注意如果ike peer指定引用的协商提议为rsa-encr时，配置的对端名字应该为对端rsa公钥的名字。

使用实例

# 设置IKE Peer的名称为“VRP30”。

<Quidway> system-view
[Quidway] ike peer huawei
[Quidway-ike-peer-huawei] remote-name VRP30
