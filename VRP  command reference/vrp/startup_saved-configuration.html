startup saved-configuration

命令功能

startup saved-configuration命令用来配置系统下次启动时使用的配置文件。

命令格式

startup saved-configuration configuration-file

参数说明

configuration-file：配置文件的文件名。字符串形式，格式为[ dirive-name ][ file-name ]，长度范围是5～64。如果未指定dirive-name，则此值为默认的设备名。

视图

用户视图

缺省级别

3：管理级

使用指南

配置文件必须以“.cc”作为扩展名，而且系统启动配置文件必须存放在存储设备的根目录下。存储设备包括硬盘（Hard disk）和Flash，如果您选择缺省路径，系统配置文件将存储在Flash的根目录下。

使用实例

# 配置下次启动使用的配置文件。

<Quidway> startup saved-configuration vrpcfg.cfg
