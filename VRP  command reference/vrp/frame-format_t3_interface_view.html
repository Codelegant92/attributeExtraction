frame-format（T3接口视图）

命令功能

frame-format命令用来设置T3接口的帧格式。

undo frame-format命令用来恢复缺省设置。

缺省情况下，T3接口的帧格式是c-bit。

命令格式

frame-format { c-bit | m23 }

undo frame-format

参数说明

c-bit：设置帧格式为C-bit。

m23：设置帧格式为m23。

视图

T3接口视图

缺省级别

2：配置级

使用指南

T3使用的帧格式有两种：M23和C-bit。

  • M23指的是DS-2到DS-3的复用过程。

  • C-bit又称为C-bit奇偶校验，配置为这种格式时，帧中的填充（stuffing）指示位可用于维护功能。

使用实例

# 设置T3接口1/0/0的帧格式为m23。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] frame-format m23
