hello-option neighbor-tracking（IPv6）

命令功能

hello-option neighbor-tracking命令用来使能邻居跟踪功能。

undo hello-option neighbor-tracking命令用来恢复缺省配置。

缺省情况下，未使能邻居跟踪功能。

命令格式

hello-option neighbor-tracking

undo hello-option neighbor-tracking

参数说明

无

视图

IPv6 PIM视图

缺省级别

2：配置级

使用指南

该命令仅对PIM-SM有效。

如果使能了邻居跟踪功能，上游路由器会跟踪每个下游路由器上加入的组播组信息。

说明：

共享网段中的所有路由器都使能该命令才可以实现邻居跟踪。

如果接口上没有配置该功能，取IPv6 PIM视图下的配置；否则取接口视图下的配置。

使用实例

# 在公网实例中，使能跟踪下游邻居功能。

<Quidway> system-view
[Quidway] pim-ipv6
[Quidway-pim6] hello-option neighbor-tracking
