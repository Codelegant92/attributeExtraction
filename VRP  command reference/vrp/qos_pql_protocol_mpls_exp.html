qos pql protocol mpls exp

命令功能

qos pql protocol mpls exp命令用来建立基于协议MPLS的分类规则。

undo qos pql protocol mpls exp命令用来删除相应的分类规则。

命令格式

qos pql pql-index protocol mpls exp mpls-experimental-value&<1-8> queue { top | middle | normal | bottom }

undo qos pql pql-index protocol mpls exp mpls-experimental-value&<1-8>

参数说明

pql-index：优先列表的组号，取值范围是1～16。

mpls-experimental-value：MPLS报文的EXP域，取值范围是0～7。

top、middle、normal、bottom：对应PQ的四个队列，优先级依次降低。

使用指南

系统是以规则被配置的顺序来匹配数据包，如果发现数据包与某个规则匹配，便结束整个查找过程。

对于同一个pql-index，本命令可以重复使用，为IP数据包建立多种分类规则。

使用实例

# 建立基于协议MPLS的分类规则，设置MPLS的EXP值为5。

<Quidway> system-view
[Quidway] qos pql 10 protocol mpls exp 5 queue top
