get（SFTP客户端视图）

命令功能

get命令用来从远程SFTP服务器上下载文件。并存储在本地。

缺省情况下，如果没有指定本地文件名，则认为与远程SFTP Server上文件同名。

命令格式

get remote-filename [ local-filename ]

参数说明

remote-filename：SFTP服务器上的源文件名，字符串形式，绝对路径名长度范围为1～64。

local-filename：SFTP客户端的本地文件名，字符串形式，绝对路径名长度范围为1～64。

视图

SFTP客户端视图

缺省级别

2：配置级

使用指南

如果local-filename没有指定，本地默认下载后的文件名与SFTP服务器上的文件名相同。

使用实例

# 从SFTP服务器上下载文件。

<Quidway> system-view
[Quidway] sftp 10.164.39.222
Please input the username:client001
Trying 10.164.39.222 ...
Press CTRL+K to abort
Connected to 10.164.39.222 ...
Enter password:
sftp-client> get v5r3c06b070.cc
Remote  file: flash:/ v5r3c06b070.cc --->  Local file: v5r3c06b070.cc
Downloading file successfully ended.
