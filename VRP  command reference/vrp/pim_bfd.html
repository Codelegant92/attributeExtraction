pim bfd

命令功能

pim bfd命令用来调整接口上的PIM BFD参数。

undo pim bfd命令用来恢复PIM BFD参数的缺省值。

命令格式

pim bfd { min-tx-interval tx-value | min-rx-interval rx-value | detect-multiplier multiplier-value } *

undo pim bfd { min-tx-interval | min-rx-interval | detect-multiplier } *

参数说明

min-tx-interval tx-value：表示PIM BFD消息的最小发送间隔。整数形式，单位是毫秒，取值范围是100～1000。缺省值为100毫秒。

min-rx-interval rx-value：表示PIM BFD消息的最小接收间隔。整数形式，单位是毫秒，取值范围是100～1000。缺省值为100毫秒。

说明：

tx-value和rx-value的最小值不同产品会有差异。建议用户配置这两个值大于等于100，太小可能会导致BFD会话震荡。

detect-multiplier multiplier-value：表示PIM BFD的本地检测倍数。整数形式，取值范围是3～50。缺省值为3。

使用指南

此命令只配置PIM BFD参数，而不会使能BFD功能。如果要使能BFD，建立BFD session，必须执行pim bfd enable命令。

在实际使用中，可以连续配置多个参数，也可以只配置一个参数。如果只配置部分参数，其他未配置的参数保留先前配置。

BFD参数的取值范围及缺省值在BFD模块统一定义，不同产品可能有差异。

说明：

此命令不适用于MTunnel接口。

使用实例

# 在GE 6/0/0上调整PIM BFD消息的最小发送间隔。

<Quidway> system-view
[Quidway] interface gigabitethernet 6/0/0
[Quidway-GigabitEthernet6/0/0] pim bfd min-tx-interval 100
