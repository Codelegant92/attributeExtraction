snmp-agent trap enable feature-name l2vpn trap-name

命令功能

snmp-agent trap enable feature-name l2vpn trap-name命令用来使能向网管发送L2VPN特性下的某条告警消息。

snmp-agent trap enable feature-name l2vpn trap-name命令用来取消当前设置。

缺省情况下，任何告警消息都不会向网管发送。

命令格式

snmp-agent trap enable feature-name l2vpn trap-name trap-name

undo snmp-agent trap enable feature-name l2vpn trap-name trap-name

参数说明

trap-name：告警名称。L2VPN的告警名称请参考《VRP 告警参考》。

使用指南

只有执行命令snmp-agent target-host inform或执行命令snmp-agent target-host trap配置了接收告警的网管地址后，网管才能接收到此告警信息。

使用实例
