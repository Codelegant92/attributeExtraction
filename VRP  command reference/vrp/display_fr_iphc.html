display fr iphc

命令功能

display fr iphc命令用来查看帧中继接口的IP报文头压缩信息。

命令格式

display fr iphc [ interface interface-type interface-number[.subnumber ] ]

参数说明

interface interface-type interface-number：查看指定帧中继接口的IP报文头压缩信息。

subnumber：查看指定帧中继子接口的IP报文头压缩信息。

使用指南

在帧中继主接口上配置IPHC压缩的命令是fr compression iphc，而在帧中继子接口上配置IPHC压缩的命令是fr map ip。

使用实例

# 查看IP报文头的压缩情况如下：

<Quidway> display fr iphc
Serial1/0/0 -DLCI:20
  RTP header compression information:
    Compression:
        Total packets:              63 , Packets compressed:              61
        Link searches:               0 , Search missed     :               2
        Bytes saved  :            2255 , Bytes sent        :            3163
    Decompression:
        Total packets:              94 , Packets compressed:              91
        Errors       :               0
    Compression-connections: 10 , Decompression-connections: 10

项目

描述

Serial1/0/0 -DLCI:20

可以查看到IP报文头压缩信息的接口名称和相应的虚电路号。

RTP header compression information:

RTP压缩报文头的统计信息。

Compression:

压缩的报文统计信息。

Total packets:

接收到的总报文数。

Packets compressed:

被压缩的总报文数。

Link searches:

节点在链表中的排行。值为零表示第一个节点就是对应的节点。否则向下查找时，该值依次加1。

Search missed :

压缩时，报文因没有找到相应的上下文环境而没有被压缩的报文数。

Bytes saved :

保存的字节数。

Bytes sent :

发送的字节数。

Decompression:

解压缩的报文统计信息。

Errors :

接收到的错误报文数。

Compression-connections:

允许压缩的最大连接数。

Decompression-connections:

允许解压缩的最大连接数。
