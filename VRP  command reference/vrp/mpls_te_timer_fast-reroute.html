mpls te timer fast-reroute

命令功能

mpls te timer fast-reroute命令用来设置当备份带宽可用时，设备每隔多长时间决定一次是否需要将LSP转移到一条更好的备份隧道。

缺省情况下，进行定时扫描，扫描间隔为1000毫秒（1秒）。

命令格式

mpls te timer fast-reroute [ weight ]

参数说明

weight：为决定一条LSP是否应使用新的、更好的备份隧道而进行的扫描的时间间隔权值，取值范围是0～604800。缺省情况下，扫描时间间隔权值为300。设置为0表示不进行定时扫描。

使用指南

设置了扫描时间间隔权值后，系统使用以下公式计算出扫描时间间隔t1。

t1＝weight×1000×200÷产品能创建的RSVP LSP最大数量

如果计算出的t1大于1000毫秒，则以t1的值作为实际扫描间隔；否则扫描间隔为1000毫秒。

使用实例

# 系统最多支持建立2048条RSVP LSP，设置扫描时间间隔权值为120，则每隔11718毫秒对LSP进行一次扫描，决定是否需要将它们切换到更好的备份隧道。
