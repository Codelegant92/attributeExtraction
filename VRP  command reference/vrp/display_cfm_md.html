display cfm md

命令功能

display cfm md命令用来查看MD的信息。

命令格式

display cfm md [ md-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

md md-name：MD的名称。字符串形式，不支持空格、“-”和“?”，长度范围是1～43。不输入本参数，则显示设备上已创建的所有MD的信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用本命令可以查看MD的级别、MD名称输入类型、MD内MIP生成规则、Sender ID TLV类型，以及MD内包含的MA简要信息（包括MA名称、CCM发送间隔、MA关联的VLANID、MA关联的VSI名称）。不输入md-name时，设备上会显示出创建的所有MD的相关信息。

使用实例

# 在draft7版本中，查看名称为mdcustomer的MD的级别。

<Quidway> display cfm md mdcustomer
MD-Name                                      Level
--------------------------------------------------
mdcustomer                                    0

# 在standard2007标准版本中，查看名称为mdcustomer的MD相关信息。

<Quidway> display cfm md mdcustomer

项目

描述

MD Name

MD的名称。

MD Name Format
MD名称参数输入类型:
  • no-md-name
  • dnsname-and-mdname
  • mac-address
  • md-name

Level

MD级别，范围0～7，级别越高，优先级越高。

MIP Create-type
MIP生成规则:
  • default
  • explicit
  • none

SenderID TLV-type
Sender ID TLV类型，用于CCM报文的填充。可能的显示值有：
  • SendIdNone
  • SendIdChassis
  • SendIdManage
  • SendIdChassisManage
  • SendIdDefer

MA list

MD内包含的MA信息。

MA Name

MA名称。

Interval

CCM报文的发送间隔。

Vlan ID

MA关联的VLAN ID。

VSI Name

MA关联的VSI名称。
