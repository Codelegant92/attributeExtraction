loopback（POS接口视图）

命令功能

loopback命令用来使能POS接口的环回功能。

undo loopback命令用来取消环回。

缺省情况下，禁止环回功能。

命令格式

loopback { local | remote }

undo loopback

参数说明

local：使能POS接口对内环回。

remote：使能POS接口对外环回。

视图

POS接口视图

缺省级别

2：配置级

使用指南

只有在进行某些特殊功能测试的时候，才对接口设置环回功能。

当接口设置为本端环回时，接口的物理状态变为Up，而链路协议状态为Down。

当接口设置为远端环回时，如果链路层协议是PPP，则物理状态为Up，而链路协议状态为Down。如果链路层协议是HDLC，接口的物理状态和链路协议状态都是Up。

说明：

POS接口不能同时进行本端自环和远端环回。

使用实例

# 设置POS接口对内环回。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] loopback local
