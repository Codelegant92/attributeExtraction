reset mac-tunnel tunnel-name statistics

命令功能

reset mac-tunnel statistics命令用来清除MAC Tunnel的流量统计信息。

命令格式

reset mac-tunnel tunnel-name tunnel-name statistics

参数说明

tunnel-name tunnel-name：指定MAC Tunnel的名字。长度范围是1～31个字符，首位不能是空格，名称中间不能有空格、“?”和“-”。

视图

用户视图

缺省级别

3：管理级

使用指南

为了获得某个时间段的流量统计数据，可先用此命令将隧道的流量统计数据清零，经过一段时间后，再查看隧道流量统计数据。

使用实例

# 清除隧道mytunnel的流量统计信息。

<Quidway> reset mac-tunnel tunnel-name mytunnel statistics
