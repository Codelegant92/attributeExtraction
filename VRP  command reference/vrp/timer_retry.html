timer retry

命令功能

timer retry命令用来配置连接请求重试周期。

undo timer retry命令用来恢复该配置参数缺省值。

缺省情况下，连接请求重试周期是30秒

命令格式

timer retry interval

undo timer retry

参数说明

interval：表示连接请求重试周期。整数形式，取值范围是1～60，单位是秒。

视图

公网实例MSDP视图、VPN实例MSDP视图

缺省级别

2：配置级

使用指南

MSDP Peer之间需要建立TCP连接。MSDP协议要求IP地址高的Peer在639端口监听，IP地址低的Peer发起连接。如果连接失败，一定时间后会重新发起连接。

使用实例

# 在公网实例中，配置连接请求重试周期为60秒。

<Quidway> system-view
[Quidway] msdp
[Quidway-msdp] timer retry 60
