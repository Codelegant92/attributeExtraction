display ike peer

命令功能

display ike peer命令用来查看IKE对等体的配置情况。

命令格式

display ike peer [ name peer-name ]

参数说明

peer-name：IKE对等体的名称，字符串形式，长度范围是1～15。

所有视图

缺省级别

1：监控级

使用指南

display ike peer命令能够观察到的IKE对等体配置情况包括：

  • IKE Peer的名称

  • 协商的模式

  • 身份认证字

  • IKE提议

  • Local id的形式

  • 对端的IP地址

  • IKE Peer对端的名字

  • 是否使能NAT穿越

使用实例

# 下面是配置了一特定对端的认证字后显示的例子。

<Quidway> display ike peer
---------------------------
