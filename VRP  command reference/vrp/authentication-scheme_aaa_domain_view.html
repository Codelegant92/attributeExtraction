authentication-scheme（AAA域视图）

命令功能

authentication-scheme命令用来设置当前域使用的认证方案。

undo authentication-scheme命令用来删除域的认证方案，恢复缺省设置。

缺省情况下，域使用认证方案default。

命令格式

authentication-scheme authentication-scheme-name

undo authentication-scheme

参数说明

authentication-scheme-name：认证方案名称，字符串形式，长度范围是1～32。不区分大小写，方案名需要符合Windows的命名规范，不能包括斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“"”和尖括号“<”、“>”等字符。

视图

AAA域视图

缺省级别

3：管理级

使用指南

authentication-scheme命令把域和某个特定的认证方案关联后，该域就可以用此认证方案对用户进行认证。

在使用authentication-scheme命令前，认证方案必须已经创建完成。

使用实例

# 设置当前域采用名为test的认证方案。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei] authentication-scheme test

# 删除当前域的认证方案，使用系统认证方案。

[Quidway-aaa-domain-huawei] undo authentication-scheme
