radius-server nas-port-format

命令功能

radius-server nas-port-format命令用来设置NAS端口的形式。

缺省情况下，采用新的NAS端口形式。

命令格式

radius-server nas-port-format { new | old }

参数说明

new：采用新的NAS端口形式。

old：采用旧的NAS端口形式。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

NAS端口形式影响所携带的用户的物理端口信息，提供给RADIUS服务器进行各种业务处理，如用户名和端口绑定等。该属性是华为公司内部扩展的属性，用于华为公司设备之间的互通和业务配合。

NAS端口两种形式的区别主要在于以太接入用户的物理端口：

  • 选择new时，端口形式为槽位号（8位）＋子槽位号（4位）＋端口号（8位）＋VLAN ID（12位）

  • 选择old时，端口形式为槽号（12位）＋端口号（8位）＋VLANID（12位）

对于ADSL接入用户，端口形式都是：槽位号（4位）＋子槽位号（2位）＋端口号（2位）＋VPI（8位）＋VCI（16位）。

选择NAS端口形式时，需要与使用的华为公司计费系统配合。

使用实例

# 配置RADIUS服务器模板test1的NAS端口形式为new。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server nas-port-format new
