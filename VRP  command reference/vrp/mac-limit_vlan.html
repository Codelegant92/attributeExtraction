mac-limit vlan

命令功能

mac-limit vlan命令用来配置基于接口+VLAN的MAC地址学习限制规则。

undo mac-limit vlan命令用来删除配置的MAC地址学习限制规则。

命令格式

mac-limit vlan vlan-id1 [ to vlan-id2 ] { action { discard | forward } | alarm { disable | enable } | maximum max rate interval } *

undo mac-limit vlan vlan-id1 [ to vlan-id2 ]

参数说明

vlan-id1，vlan-id2：VLAN号，取值范围是1～4094。用于指定一个VLAN，或指定某个VLAN范围。

max：该接口的指定VLAN上最多可以学习的MAC数量，取值范围是0～32768。当配置值为0时，表示不对学习数量进行限制。

interval：该接口的指定VLAN上每次进行MAC学习的时间间隔，取值范围是0～1000，单位为毫秒，当配置值为0时，表示不对学习速度进行限制。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

如果接口或VLAN上已经学习有MAC地址，请使用系统视图下的undo mac-address dynamic命令清除已经学习的MAC地址，否则会使MAC地址学习限制功能的数量限制不准确。

使用实例

# 配置以太接口GE1/0/0上VLAN1的MAC地址学习限制规则：最多允许学习100个MAC地址，每次MAC学习的时间间隔为50毫秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] mac-limit vlan 1 maximum 100 rate 50
