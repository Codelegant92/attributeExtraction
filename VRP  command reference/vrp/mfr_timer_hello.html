mfr timer hello

命令功能

mfr timer hello命令用来设置帧中继捆绑链路发送hello消息的间隔。

undo mfr timer hello命令用来恢复缺省设置。

命令格式

mfr timer hello seconds

undo mfr timer hello

参数说明

seconds：捆绑链路发送hello消息的间隔。整数形式，取值范围是1～180，单位是秒。缺省值是10秒。

缺省级别

2：配置级

使用指南

帧中继捆绑链路的两端定期向对端发送hello消息，对端收到后，回应一个hello应答消息。

使用实例

# 设置捆绑链路Serial1/0/0:0每隔15秒发送一次hello消息。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr mfr 1/0/123
[Quidway-Serial1/0/0:0] mfr timer hello 15
