display mpls oam ingress

命令功能

display mpls oam ingress命令用来在OAM源端查看LSP的MPLS OAM参数及状态信息。

命令格式

display mpls oam ingress { all | tunnel tunnel-number } [ slot slot-id | verbose ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：显示源端所有启用OAM功能的LSP的报文信息与缺陷状态。

tunnel tunnel-number：隧道接口编号。

slot slot-id：接口板号，显示指定接口板上的OAM全部信息。使用此参数可以看到OAM实例的报文统计信息。

verbose：显示全部信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定接口板号，将显示主控板上的OAM信息。

与接口板的OAM信息相比，主控板不包括OAM实例的报文统计信息，但提供本节点检测的LSP统计信息。

使用实例

# 查看源端主控板上隧道Tunnel1/0/0的OAM状态信息。

<Quidway> display mpls oam ingress tunnel 1/0/0
-------------------------------------------------------------------------
No. Tunnel-name     Ttsi              Type    Frequency  Status
-------------------------------------------------------------------------
1   Tunnel1/0/0     1.1.1.1 : 100     CV        1 s      Start/Non-defect
