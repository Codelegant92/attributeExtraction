interface（Ethernet）

命令功能

interface { ethernet | gigabitethernet }命令用来进入指定以太网接口视图。

命令格式

interface { ethernet | gigabitethernet } interface-number

参数说明

ethernet：以太网接口。

gigabitethernet：千兆以太网接口。

interface-number：接口编号。

视图

系统视图

缺省级别

2：配置级

使用指南

当需要对以太网接口进行配置时，可使用该命令进入以太网接口视图。

使用实例

# 进入以太网接口1/0/0视图。

<Quidway> system-view
[Quidway] interface ethernet 1/0/0
[Quidway-Ethernet1/0/0]

# 进入千兆以太网接口2/0/0视图。

<Quidway> system-view
[Quidway] interface gigabitethernet 2/0/0
[Quidway-GigabitEthernet2/0/0]
