reset bgp flap-info

命令功能

reset bgp flap-info命令用来清除路由的振荡统计信息。

命令格式

reset bgp [ vpn-instance vpn-instance-name ] flap-info [ regexp as-path-regexp | as-path-filter as-path-filter-number | ipv4-address [ mask | mask-length ] ]

reset bgp [ vpn-instance vpn-instance-name ] ipv4-address flap-info

reset bgp { ipv6 | vpn6-instance vpn6-instance-name } ipv6-address flap-info

reset bgp { ipv6 | vpn6-instance vpn6-instance-name } flap-info [ ipv6-address | as-path-filter as-path-filter-number | regexp as-path-regexp ]

参数说明

vpn-instance vpn-instance-name：清除指定VPN实例的振荡统计信息。

regexp as-path-regexp：复位匹配AS路径正则表达式的路由振荡统计信息。

as-path-filter as-path-filter-number：基于指定AS路径过滤器清除振荡统计信息。

ipv4-address：指定BGP的IPv4对等体网络地址。

mask | mask-length：指定网络掩码（点分十进制形式）或掩码长度。如果没有指定掩码，则按有类地址处理。

ipv6：指定复位所有IPv6对等体的振荡统计信息。

vpn6-instance vpn6-instance-name：清除指定IPv6 VPN实例的振荡统计信息。

ipv6-address：指定BGP的IPv6对等体网络地址。

使用实例

# 清除所有匹配AS路径过滤器10的路由的振荡统计信息。

<Quidway> reset bgp flap-info as-path-filter 10

# 清除所有VPNv6实例为1的路由振荡统计信息。

<Quidway> reset bgp vpn6-instance 1 flap-info
