pwsignal

命令功能

pwsignal命令用于配置VSI所采用的信令方式。

缺省情况下，系统没有配置VSI的信令方式。

命令格式

pwsignal { bgp | ldp }

参数说明

bgp：采用BGP信令方式。

ldp：采用LDP信令方式。

视图

VSI视图

缺省级别

2：配置级

使用指南

创建VSI后，建议首先配置该VSI的信令方式。

目前为止，如果VSI实例的成员发现方式配置为静态方式时，信令方式必须为LDP。如果VSI实例的成员发现方式为自动方式时，信令方式必须为BGP。

VSI实例的信令方式配置成功后，不可以更改。如果希望更改，必须先删除该VSI实例再创建一个新的VSI实例。

使用实例

# 配置当前VSI实例的信令方式为LDP。

<Quidway> system-view
[Quidway] vsi company1 static
[Quidway-vsi-company1] pwsignal ldp
[Quidway-vsi-company1-ldp]

# 配置当前VSI实例的信令方式为BGP。

<Quidway> system-view
[Quidway] vsi company2 auto
[Quidway-vsi-company2] pwsignal bgp
[Quidway-vsi-company2-bgp]
