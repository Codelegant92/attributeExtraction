snmp-agent trap enable feature-name mld

命令功能

snmp-agent trap enable feature-name mld命令用来将打开MLD相关的告警开关。

undo snmp-agent trap enable feature-name mld命令用来关闭MLD相关的告警开关。

命令格式

snmp-agent trap enable feature-name mld { non-excessive all | trap-name { joingrp | leavegrp } }

undo snmp-agent trap enable feature-name mld { [ non-excessive | excessive ] all | trap-name { joingrp | leavegrp } }

参数说明

non-excessive all：打开MLD模块所有非海量事件的告警开关。

trap-name：打开指定类型事件的告警开关。

joingrp：打开收到MLD加入报文的告警开关。

leavegrp：打开收到MLD离开报文或组定时器超时的告警开关。

non-excessive：关闭MLD模块所有非海量事件的告警开关。

excessive：关闭MLD模块所有海量事件的告警开关。

all：关闭MLD模块所有事件的告警开关。

使用指南

目前定义的MLD模块告警均为海量事件。打开某个或多个事件的告警开关时，需要指定trap-name。

使用实例

# 在系统视图下打开收到MLD加入报文的告警开关。

<Quidway> system-view
[Quidway] snmp-agent trap enable feature-name mld trap-name joingrp

# 在系统视图下关闭收到MLD加入报文的告警开关。

<Quidway> system-view
[Quidway] undo snmp-agent trap enable feature-name mld trap-name joingrp
