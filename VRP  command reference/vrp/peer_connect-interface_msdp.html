peer connect-interface（MSDP）

命令功能

peer connect-interface命令用来配置MSDP对等体。

undo peer connect-interface命令用来恢复缺省配置。

缺省情况下，未配置此MSDP对等体。

命令格式

peer peer-address connect-interface interface-type interface-number

undo peer peer-address

参数说明

peer-address：表示远端MSDP对等体的地址。点分十进制形式。

interface-type interface-number：表示接口类型和编号，本地路由器使用该接口的主地址作为源IP与远端MSDP对等体建立TCP连接。

说明：

系统不向私网发布MTI接口上的路由，因此不要以MTI接口的地址建立MSPD Peer连接。

缺省级别

2：配置级

使用指南

执行其他peer命令之前必须先执行peer connect-interface命令，否则系统提示peer不存在。

重复使用命令peer peer-address connect-interface interface-type interface-number，可以为路由器配置多个远端MSDP对等体。

  • 可以为不同的peer-address指定相同的interface-type interface-number，即为不同的远端对等体指定相同的本地接口。

  • 可以为不同的peer-address指定不同的interface-type interface-number，即为不同的远端对等体指定不同的本地接口。

使用实例

# 在公网实例中，把使用IP地址125.10.7.6的路由器配置成本地路由器的MSDP对等体，GE1/0/0为本地连接端口。
