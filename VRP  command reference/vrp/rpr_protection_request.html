rpr protection request

命令功能

rpr protection request命令用来切换本地节点的东向或者西向连接。

undo rpr protection request命令用来清除切换请求。

命令格式

rpr protection request { force-switch | manual-switch } { west | east }

undo rpr protection request { west | east }

参数说明

force-switch：强制切换，将在相应方向上产生FS（Force Switch）处理。

manual-switch：人为切换，将在相应方向上产生MS（Manual Switch）处理。

west：将本地节点的报文方向切换为西向连接。

east：将本地节点的报文方向切换为东向连接。

缺省级别

2：配置级

使用指南

这个命令用来人为的对环的拓扑结构进行干预，用来设置环的西向和东向的保护模式（Protection Mode），“切断”本地节点的东向或者西向的连接。

使用实例

# 将本地节点报文方向强制切换为西向连接。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr protection request force-switch west
