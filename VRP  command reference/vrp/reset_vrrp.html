reset vrrp

命令功能

reset vrrp命令用来清除VRRP备份组收发报文的统计信息。

命令格式

reset vrrp [ interface interface-type interface-number ] [ vrid virtual-router-id ] statistics

参数说明

interface interface-type interface-number：配置了VRRP备份组的接口类型和编号。

vrid virtual-router-id：VRRP备份组号。十进制整数形式，取值范围是1～255。

视图

用户视图

缺省级别

1：监控级

使用指南

使用reset vrrp命令时：

如果指定接口名称而不指定VRRP备份组号，则清除指定接口下的所有VRRP备份组的报文统计信息。

如果指定VRRP备份组号而不指定接口名称，则清除所有接口下的指定VRRP备份组的报文统计信息。

如果既不指定接口名称也不指定VRRP备份组号，则清除所有的VRRP备份组的报文统计信息

使用实例

# 清除接口GE1/0/0下，VRRP备份组为1的所有报文统计信息。

<Quidway> reset vrrp interface gigabitethernet 1/0/0 vrid 1 statistics

# 清除所有VRRP备份组的报文统计信息。

<Quidway> reset vrrp statistics
