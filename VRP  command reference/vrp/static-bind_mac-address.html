static-bind mac-address

命令功能

static-bind mac-address命令用来配置静态绑定的MAC地址。

undo static-bind mac-address命令用来删除静态绑定的MAC地址。

缺省情况下，未配置静态绑定的MAC地址。

命令格式

static-bind mac-address mac-address

undo static-bind mac-address

参数说明

mac-address：指定待绑定的主机MAC地址，格式是H-H-H。

视图

DHCP地址池视图

缺省级别

2：配置级

使用指南

进行静态地址的绑定时，需要使用static-bind mac-address命令并且使用此命令时必须与static-bind ip-address命令配合使用，分别配置静态绑定的MAC地址和IP地址。

使用实例

# 将MAC地址为0000-e03f-0305的PC机与IP地址10.1.1.1绑定，掩码为255.255.255.0。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] static-bind ip-address 10.1.1.1 mask 255.255.255.0
[Quidway-dhcp-0] static-bind mac-address 0000-e03f-0305
