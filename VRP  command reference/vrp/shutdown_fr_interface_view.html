shutdown（帧中继接口视图）

命令功能

在帧中继交换视图下：

shutdown命令用来禁止当前所有的交换PVC。

undo shutdown命令用来使能当前所有的交换PVC。

缺省情况下，交换PVC是使能的。

在帧中继接口视图和MFR接口视图下：

shutdown命令用来关闭当前接口。

undo shutdown命令用来使能当前接口。

缺省情况下，帧中继接口和MFR接口是使能的。

命令格式

shutdown

undo shutdown

参数说明

无

视图

帧中继交换接口视图、帧中继接口视图、MFR接口视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

当接口下配有子接口时，在主接口连续执行shutdown和undo shutdown操作的间隔应至少为15秒。

使用实例

# 禁止名为PVC1的交换PVC。

<Quidway> system-view
[Quidway] fr switch pvc1 interface serial 1/0/0:0 dlci 100 interface serial 2/0/0:0 dlci 200
[Quidway-fr-switching-pvc1] shutdown

# 关闭当前接口。

[Quidway] interface mfr 5/0/0
[Quidway -MFR5/0/0] shutdown
