apply ip-address default next-hop

命令功能

apply ip-address default next-hop命令用来设置报文的缺省下一跳。

undo apply ip-address default next-hop命令用来取消已设置的报文缺省下一跳。

命令格式

apply ip-address default next-hop ip-address1 [ ip-address2 ]

undo apply ip-address default next-hop [ ip-address1 ] [ ip-address2 ]

参数说明

ip-address1，ip-address2：指定报文的缺省下一跳IP地址。

视图

Policy-Based-Route视图

缺省级别

2：配置级

使用指南

该命令对在路由表中未查询到路由的报文起作用。命令中指定的报文缺省下一跳地址不能是本地IP地址。

可以配置两个下一跳IP地址以达到负载分担的目的。

使用实例

# 设置报文的缺省下一跳为1.1.1.1。

<Quidway> system-view
[Quidway] policy-based-route policy1 permit node 10
[Quidway-policy-based-route-policy1-10] apply ip-address default next-hop 1.1.1.1
