display bgp vpls

命令功能

display bgp vpls命令用来查看BGP路由表中的VPLS信息，可以指定参数查看符合条件的VPLS信息。

命令格式

display bgp vpls { all | route-distinguisher route-distinguisher [ site-id site-id [ label-offset label-offset ] ] } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：显示BGP-VPLS地址族的所有VPLS信息。

route-distinguisher：显示指定路由标识符的VSI信息。路由标识符格式如下：

  • 16位自治系统号:32位用户自定义数，例如：101:3。自治系统号的取值范围是0～65535；用户自定义数的取值范围是0～4294967295。其中，自治系统号和用户自定义数不能同时为0，即RD的值不能是0:0。

  • 32位IP地址:16位用户自定义数，例如：192.168.122.15:1。IP地址的取值范围是0.0.0.0～255.255.255.255；用户自定义数的取值范围是0～65535。

site-id：显示指定Site-id的VSI信息。十进制整数形式，取值范围是0～65535。

label-offset：显示指定标签偏移值的VSI信息。十进制整数形式，取值范围是0～65535。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择使用display bgp vpls peer命令和display bgp vpls group命令分别查看VPLS的BGP对等体的信息和VPLS的BGP对等体组信息。

使用实例

# # 查看BGP的路由标识符的VSI为168.1.1.2：1的VPLS信息。

<Quidway> display bgp vpls route-distinguisher 168.1.1.2:1
BGP Local Router ID : 200.1.1.1, Local AS Number : 100
Status codes : * - active, > - best
--------------------------------------------------------------------------------
Route Distinguisher: 168.1.1.2:1

--------------------------------------------------------------------------------

项目

描述

Route Distinguisher

该VPN实例的路由标志符

Site ID

远端VSI实例的Site ID值

Offset

Site ID的偏差，取值为0或1，缺省值为0。

Nexthop

下一跳IP地址

Range

VSI实例中的Site的个数范围，十进制整数形式。

LabBase

分配的标签的起始编号

Tunnel ID

隧道ID，十六进制形式，是系统自动分配的。

FromPeer

BGP对等体的地址

MHPref

多归属优先级
