display mpls te protection binding protect-tunnel

命令功能

display mpls te protection binding protect-tunnel命令用来显示Tunnel的保护关系信息。

命令格式

display mpls te protection binding protect-tunnel { tunnel-id | interface tunnel interface-number } [ | { begin | include | exclude } regular-expression ]

参数说明

tunnel-id：显示与指定Tunnel ID相关的保护关系信息，该Tunnel必须是备用Tunnel。

interface tunnel interface-number：显示与指定隧道接口相关的保护关系信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Protect-tunnel id

保护隧道(也可以称为“备用隧道”)的Tunnel-ID

Protect-tunnel name

保护隧道的名称，即Tunnel接口名称

Maximum number of bound work-tunnels

该保护隧道可以保护隧道的最大数目，该值和具体的产品有关

Currently bound work-tunnels

当前绑定的主隧道信息，包含绑定的主隧道的总数和列表
