radius-server

命令功能

radius-server命令用来设置当前域的RADIUS服务器模板。

undo radius-server命令用来删除为该域设置的服务器模板。

命令格式

radius-server template-name

undo radius-server

参数说明

template-name：RADIUS服务器模板名称，字符串形式，长度范围是1～32。字符包括大写字母A～Z、小写字母a～z、数字0～9、点号“.”、短线“-”和下划线“_”。

使用指南

配置前，域使用的RADIUS服务器模板必须已经存在。

使用实例

# 设置当前域使用名为radius-server-163的RADIUS服务器模板。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei] radius-server radius-server-163
