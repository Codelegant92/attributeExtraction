display standby flow

命令功能

display standby flow命令用来显示参与备份负载分担的主接口的流量统计数据。

命令格式

display standby flow

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

当主接口参与负载分担时，备份中心将自动检测主接口的流量，执行命令display standby flow将显示一段内的主接口的流量统计信息。

使用实例

# 先指定接口Pos1/0/0和Tunnel2/0/0作为接口Pos3/0/0的备份接口。在接口Pos3/0/0上配置备份负载分担

<uidway> system-view
[Quidway] interface pos 3/0/0
[Quidway-Pos3/0/0] standby interface pos 1/0/0 10
[Quidway-Pos3/0/0] standby interface tunnel 2/0/0 20
[Quidway-Pos3/0/0] standby threshold 80 50
[Quidway-Pos3/0/0] standby timer flow-check 100
[Quidway-Pos3/0/0] standby bandwidth 9

# 查看参与备份负载分担的主接口的流量统计数据。

[Quidway-Pos3/0/0] display standby flow
Interfacename      : Pos3/0/0
Flow-interval(s)   : 100
LastInOctets       : 868168
LastOutOctets      : 1818667
InFlow(Octets)     : 50070
OutFlow(Octets)    : 100088
BandWidth(b/s)     :9000
UsedBandWidth(b/s) : 8000
表1 display standby flow命令输出信息描述

项目

描述

Flow-interval(s)

检测主接口流量的时间间隔

LastInOctets

上一次检测时间点主接口累计的接收字节数

LastOutOctets

上一次检测时间点主接口累计的发送字节数

InFlow(Octets)

上一个时间间隔内主接口累计接收字节数

OutFlow(Octets)

上一个时间间隔内主接口累计发送字节数

BandWidth(b/s)

主接口带宽

UsedBandWidth(b/s)

上一个时间间隔内接口实际带宽
