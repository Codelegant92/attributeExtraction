domain

命令功能

domain命令用来创建一个新域，并进入用户域视图。

undo domain命令用来删除一个已经存在的域。

缺省存在一个default域，不能被删除。

命令格式

domain domain-name

undo domain domain-name

参数说明

domain-name：域名，字符串形式，长度范围是1～20。不能使用斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“"”和尖括号“<”、“>”。不区分大小写。

2：配置级

使用指南

domain命令在创建一个域时，如果同名的域已经存在，将直接进入该用户域视图。

系统支持配置255个域，加上缺省域，共支持256个域。缺省存在一个default域，不能被删除，所有没有专门指定域的用户都属于这个default域。域在初建时缺省处于激活状态。

使用实例

# 指定Huawei作为当前域，并进入用户域视图。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei]
