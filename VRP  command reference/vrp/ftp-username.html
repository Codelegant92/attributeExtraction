ftp-username

命令功能

ftp-username命令用来配置NQA测试的FTP测试例使用的用户名。

undo ftp-username命令用来恢复缺省配置。

缺省情况下，没有配置FTP测试的用户名。

命令格式

ftp-username name

undo ftp-username

参数说明

name：FTP用户名。字符串形式，长度范围是1～32。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对FTP类型的测试例配置FTP用户名。

不能更改正在运行的测试例的用户名。

使用实例

# 设置测试例user test的FTP测试用户名为“user1”。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type ftp
[Quidway-nqa-user-test] ftp-username user1
