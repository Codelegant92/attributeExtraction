display interface ethernet brief

命令功能

display interface ethernet brief命令用来显示以太网接口的简要信息，包括接口的物理状态、自协商方式、双工模式、接口速率、接口接收方向和发送方向最近一段时间的平均带宽利用率。

命令格式

display interface ethernet brief

参数说明

无

所有视图

缺省级别

1：监控级

使用指南

可以使用命令reset counter interface命令清除接口的统计信息。

项目

描述

Interface
接口名称和接口编号。所有接口按字典序显示。可显示的接口类型包括：

  • Ethernet

  • GigabitEthernet

Physical
接口的物理状态。可能的物理状态有：

  • up：表示该接口处于正常启动的状态。

  • down：表示该接口物理层出现故障。

  • *down：Administratively down，管理员在该接口执行了shutdown（接口视图）命令。

  • ^down：standby，表示该接口是备份接口。

  • (l)：loopback，表示该接口启用环回功能。

  • (b)：BFD down，表示该接口的物理层处于BFD Down状态。

Auto-Neg
接口是否使能自协商功能。可能的情况有：

  • enable：表示该接口使能了自协商功能。

  • disable：表示该接口没有使能自协商功能。

Duplex
接口的双工模式。可能的情况有：

  • full：表示该接口处于全双工模式。

  • half：表示该接口处于半双工模式。

Bandwidth
接口的带宽。可能的值有：

  • 快速以太网的速率有10Mbit/s或100Mbit/s。

  • 千兆以太网的速率有10Mbit/s、100Mbit/s或1000Mbit/s。

InUti

接口接收方向最近5分钟内的平均带宽利用率。接口接收方向最近5分钟内的平均带宽利用率=接口接收方向最近5分钟内平均速率/接口带宽。不足0.01%且有流量时显示0.01%。当接口带宽变小后（例如以太网接口通过speed命令调整接口带宽，或Eth-Trunk的成员接口状态变为Down或退出Eth-Trunk接口），由于通信流量没有及时调整，可能会出现带宽利用率为100％。

OutUti

接口发送方向最近5分钟内的平均带宽利用率。接口发送方向最近5分钟内的平均带宽利用率=接口发送方向最近5分钟内平均速率/接口带宽。不足0.01%且有流量时显示0.01%。当接口带宽变小后（例如以太网接口通过speed命令调整接口带宽，或Eth-Trunk的成员接口状态变为Down或退出Eth-Trunk接口），由于通信流量没有及时调整，可能会出现带宽利用率为100％。

Trunk

接口所属的Trunk ID。
