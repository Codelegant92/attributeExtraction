protect-switch（MAC Tunnel视图）

命令功能

protect-switch命令用来配置MAC Tunnel保护组如何执行保护倒换动作。

命令格式

protect-switch { clear | force | lock | manual }

参数说明

clear：清除本地配置的由参数force、lock、manual指定的保护倒换动作，而且系统不会进入WTR等待状态。

force：强制将业务从工作隧道切换到保护隧道。

lock：将业务锁定在工作隧道，不允许切换到保护隧道。

manual：在工作隧道和保护隧道都正常工作的情况下，将业务手动切换到保护隧道。

视图

MAC Tunnel视图

缺省级别

2：配置级

使用指南

参数的优先级顺序由高到低为：clear、lock、force、manual。带有高优先级参数的命令可以覆盖当前设备上低优先级参数的配置。带有低优先级参数的手动倒换不能够覆盖已经存在的高优先级倒换动作配置。

OAM型保护组输入命令后立即生效；APS型保护组只有当APS保护倒换协议启动，倒换命令才能生效。

使用实例

# 在工作MAC Tunnel视图下配置保护组的LOCK命令。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] protection mac-tunnel ProTunnel2 protocol aps
[Quidway-mac-tunnel-mytunnel] protect-switch lock
