apply ip-precedence（Route-Policy视图）

命令功能

apply ip precedence命令用来为路由设置QoS参数ip-precedence。

undo apply ip-precedence命令用来取消该设置。

缺省情况下，没有设置ip-precedence值。

命令格式

apply ip-precedence ip-precedence

undo apply ip-precedence

参数说明

ip-precedence：IP优先级。整数形式，取值范围是0～7。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

ip-precedence是QPPB可以设置的三种QoS参数中的一种，对应于IP报文头里的Precedence字段。与apply qos-local-id，apply behavior命令互斥。

说明：

产品NE16E/08E/05不支持apply ip precedence命令。

使用实例

# 在名为test的Route-Policy下配置ip-precedence策略。

<Quidway> system-view
[Quidway] route-policy test permit node 10
[Quidway-route-policy] apply ip-precedence 3

# 删除ip-precedence策略。

<Quidway> system-view
[Quidway] route-policy test permit node 10
[Quidway-route-policy] undo apply ip-precedence
