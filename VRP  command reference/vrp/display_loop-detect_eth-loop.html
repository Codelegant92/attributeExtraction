display loop-detect eth-loop

命令功能

display loop-detect eth-loop命令用来显示以太环路检测命令的配置结果。

命令格式

display loop-detect eth-loop [ vlan vlan-id | vsi vsi-name ]

参数说明

vlan vlan-id：用于显示指定VLAN下的以太环路检测配置信息。十进制整数形式，取值范围是1～4094。

vsi vsi-name：用于显示指定VSI实例的以太环路检测配置信息。字符串形式，不支持空格，区分大小写，取值范围是1～31。

使用指南

使用可选参数vlan-id或vsi-name可以选择只显示某一VLAN或某一VSI的以太环路检测配置信息和阻断了哪些接口或PW。

项目

描述

VLAN/VSI

使能了以太环路检测功能的VLAN的编号或VSI的名称

LTimes

配置的环回次数

D-Cycle

配置的检测周期

Cycles

配置的检测几个检测周期才执行阻断操作

Retry

配置的重试次数

Action

配置的在判断出环路后是否进行阻断以及阻断多长时间，还是只记录日志和向网管发告警不进行阻断

BPolicy

配置的阻断策略

Total Items

总共在多少VLAN和VSI内配置了以太环路检测功能

Blocked Port

被阻断的接口或PW
