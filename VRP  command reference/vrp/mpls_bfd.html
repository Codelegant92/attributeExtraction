mpls bfd

命令功能

mpls bfd命令用来设置BFD会话的相关参数。

undo mpls bfd命令用来删除BFD会话的相关参数。

命令格式

mpls bfd { min-tx-interval tx-interval | min-rx-interval rx-interval | detect-multiplier multiplier }*

undo mpls bfd { min-tx-interval | min-rx-interval | detect-multiplier }*

参数说明

min-tx-interval tx-interval：指定BFD会话最小发送时间间隔。取值范围是100～1000，单位是毫秒，缺省值是100毫秒。

min-rx-interval rx-interval：指定BFD会话最小接收时间间隔。取值范围是100～1000，单位是毫秒，缺省值是100毫秒。

detect-multiplier multiplier：指定BFD会话本地检测的倍数。整数形式，取值范围是3～50，缺省值是3。

使用指南

BFD检测参数的取值范围及缺省值在BFD模块统一定义，不同产品可能有差异。

使用实例

# 设置BFD会话的最小发送时间间隔为200毫秒。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls bfd min-tx-interval 200
