local-pw-type

命令功能

local-pw-type命令用来在NQA视图下配置本端的PW类型。

undo local-pw-type命令用来取消在NQA视图下配置的本端PW类型。

命令格式

local-pw-type { local-pw-type | ip-interworking }

undo local-pw-type

参数说明

local-pw-type：指定的本端PW类型。目前支持的封装类型有ATM-aal5、Ethernet、FR、HDLC、PPP和VLAN。缺省值为Ethernet。

ip-interworking：PW的封装类型为IP异种介质互通。

视图

NQA视图

缺省级别

2：配置级

使用指南

使用local-pw-type命令前，需要先在NQA视图下配置NQA测试例的测试类型为PWE3trace或者PWE3ping，否则无法配置local-pw-type命令。

使用实例

# 在NQA视图下配置本端的PW ID为PPP。

<Quidway> system-view
[Quidway] nqa test-instance admin pwe3
[Quidway-admin-pwe3] test-type pwe3trace
[Quidway-admin-pwe3] local-pw-type ppp
