gre checksum

命令功能

gre checksum命令用来设置对GRE隧道两端进行端到端校验。

undo gre checksum命令用来取消校验。

缺省情况下，禁止GRE隧道两端进行端到端校验。

命令格式

gre checksum

undo gre checksum

参数说明

无

视图

Tunnel接口视图

缺省级别

2：配置级

使用指南

如果本端配置了校验和，则本端将根据GRE头及payload信息计算校验和，并将包含校验和的报文发送给对端。对端对接收到的报文计算校验和，并与报文中的校验和比较，如果一致则对报文进一步处理，否则丢弃报文。

GRE隧道两端可以根据实际需要选择是否配置校验和，从而决定是否触发校验功能。

如果本端配置了校验和而对端没有配置，则本端将不对接收到的报文进行校验，但对本端发送的报文计算校验和；如果本端没有配置校验和而对端配置，则本端对从对端发来的报文进行校验和检查，但对本端发送的报文不计算校验和。

使用实例

# 路由器Quidway1的Tunnel1/0/0与Quidway2的Tunnel2/0/0之间已建立隧道，在隧道两端设置校验。

<Quidway1> system-view
[Quidway1] interface tunnel 1/0/0
[Quidway1-Tunnel1/0/0] gre checksum
 
<Quidway2> system-view
[Quidway2] interface tunnel 2/0/0
[Quidway2-Tunnel2/0/0] gre checksum
