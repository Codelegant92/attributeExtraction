reset mpls te idle-time

命令功能

reset mpls te idle-time命令用来重置RSVP-TE隧道建立LSP失败时等待重建的时间。

命令格式

reset mpls te idle-time { all | tunnel-name }

参数说明

all：设置所有隧道接口的LSP等待重建的时间。

tunnel-name：设置指定隧道接口的LSP等待重建的时间。tunnel-name为隧道接口的名称，字符串形式，不支持空格。

用户视图

缺省级别

1：监控级

使用指南

LSP建立失败后需等待一段时间后才触发重建。这段时间随连续失败次数的增加而呈指数增长，初始值是10秒，最大是320秒。reset mpls te idle-time命令用来把LSP建立失败后的等待重建的时间重置为10秒。

使用实例

# 重设所有RSVP-TE隧道建立LSP失败等待重建的时间。

<Quidway> reset mpls te idle-time all
