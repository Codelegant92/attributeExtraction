display atm pvc-group

命令功能

display atm pvc-group命令用来显示PVC-Group的信息。

命令格式

display atm pvc-group [ interface atm interface-number [ pvc { pvc-name [ vpi/vci ] | vpi/vci } ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-number：接口编号。

pvc-name：PVC名，字符串形式，长度为1～16。如果不指定PVC名且不指定VPI/VCI值对，则缺省显示指定ATM接口范围内所有PVC-Group的信息。

vpi/vci：VPI/VCI值对。详见pvc命令中的参数说明。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

缺省级别

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 显示设备中所有ATM接口下的PVC-Group的信息

<Quidway> display atm pvc-group
VPI/VCI  PVC-NAME      STATE ENCAP PROT   INTERFACE      GROUP
  1/32   pvc1          UP    SNAP  IP     Atm1/0/0(UP)   1/32
  2/32   pvc2          UP    SNAP  IP     Atm1/0/0(UP)   2/32
  3/32   pvc3          UP    SNAP  IP     Atm1/0/0(UP)   3/32

项目

描述

VPI/VCI

PVC的VPI/VCI值对。

PVC-NAME

PVC名称。

STATE

状态（Up或Down）。

ENCAP

AAL封装类型。

PROT
应用方式。

  • IP表示IPoA。

  • ETH表示PPPoEoA或IPoEoA。

  • PPP表示PPPoA。

INTERFACE

接口是ATM主接口：槽号为1，适配卡号为0，接口号为0。

GROUP

接口所在的PVC-Group是以PVC“1/32”为基础创建的。
