display controller t3（CT3接口）

命令功能

display controller t3命令用来查看CT3所有通道的配置和状态信息。

命令格式

display controller t3 [ controller-number ]

参数说明

controller-number：CT3接口的编号，包括槽号、卡号、端口号三部分。如果不指定CT3接口的编号，将显示所有CT3接口的通道状态。

使用指南

当工作在CT3模式下时，display controller t3命令将依次显示所有28个T1通道的状态信息。如果只查看某个T1通道，则应使用display controller t3 t1命令。

项目

描述

T3 1/0/0 current state :
T3接口当前的物理状态：

  • UP：物理状态正常启动。

  • DOWN：物理连接不正常。

  • Administratively down：如果管理员在该接口下执行了shutdown命令，将显示状态为Administratively down。

Cable length

电缆长度，是路由器到配线架之间的距离。可以使用cable（CT3接口视图）命令改变电缆长度。

Frame-format

T3接口的帧格式。

code

T3接口的线路编解码格式。

clock
时钟模式：

  • slave：从时钟模式。

  • master：主时钟模式。

Current mode is CT3
接口当前的工作模式：

  • CT3：接口工作在CT3模式下，表示通道化工作模式。

  • T3：接口工作在T3模式下，表示净通道工作模式。

loopback not set

没有使能CT3接口环回功能。只有在进行某些特殊功能测试时，才将CT3接口设为环回。

No back card

没有后卡。

Alarm:Error:
告警信息和错误信息。可能出现的错误类型有以下几类。

  • LOS：Loss of Signal，帧同步丢失。

  • LOF：Loss of Frame，帧丢失。

  • CERR：CRC Error，循环冗余校验错。

  • LCV：Line Code Violation，线路码错误。

  • BPV：Bipolar Violation，线路码极性错误。

  • EXZ：Excessive Zeros，连续多0错误。

  • FrmErr：FRAME ERROR，帧错误。

T3 1/0/0 CT1 1 is

T3接口的CT1通道的当前状态。

Frame-format none

表示T3接口的E1通道不可通道化。
