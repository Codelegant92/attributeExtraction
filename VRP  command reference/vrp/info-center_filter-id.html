info-center filter-id

命令功能

info-center filter-id用来配置对指定ID的日志信息进行屏蔽的功能。

undo info-center filter-id用来删除对指定ID的日志信息进行屏蔽的功能。

命令格式

info-center filter-id { { id }&<1-50> }

undo info-center filter-id { { id }&<1-50> }

参数说明

id：添加到过滤表中的日志对应的ID信息。

视图

系统视图

缺省级别

2：配置级

使用指南

当配置日志信息的屏蔽功能后，信息中心对此信息不进行发送处理，各个输出方向都不会获取到此日志信息。

在一次添加多个ID时，ID中间要用空格隔开，且每个ID都会有相应的添加结果提示信息。

过滤表中ID数超过50时，不允许添加，提示过滤表满。当过滤表满时，如继续添加新的，则需要先删除后添加。

使用实例

# 配置对ID为1098731530的日志进行过滤。

<Quidway> system-view
[Quidway] info-center filter-id 1098731530
