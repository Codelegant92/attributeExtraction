screen-length temporary

命令功能

screen-length temporary命令用来设置指定终端屏幕的临时显示行数。

undo screen-length temporary命令用来恢复至screen-length命令下的设置。

命令格式

screen-length screen-length temporary

undo screen-length temporary

参数说明

screen-length：屏幕分屏显示的行数。整数形式，取值范围是0～512。取值为0时表示关闭分屏功能。

视图

用户视图、用户界面视图

缺省级别

2：配置级

使用指南

指定temporary参数时，设置的行数只对当前活动用户界面有效，用户退出后不保存设置。

使用实例

# 设置屏幕的每屏行数为30。

<Quidway> system
[Quidway] user-interface console 0
[Quidway-ui-console0] screen-length temporary 30
Info: The configuration takes effect on the current user terminal interface only.
