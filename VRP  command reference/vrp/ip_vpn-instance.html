ip vpn-instance

命令功能

ip vpn-instance命令用来创建VPN实例，并进入VPN实例视图。

undo ip vpn-instance命令用来删除指定的VPN实例。

缺省情况下，未定义VPN实例。

命令格式

ip vpn-instance vpn-instance-name

undo ip vpn-instance vpn-instance-name

参数说明

vpn-instance-name：VPN实例的名称，字符串形式，区分大小写，长度范围是1～31。

视图

系统视图

缺省级别

2：配置级

使用指南

必须对VPN实例配置RD后，此VPN实例才可用。

使用实例

# 创建一个名为vrf1的VPN实例。

<Quidway> system-view
[Quidway] ip vpn-instance vrf1
