vrrp gratuitous-arp timeout disable

命令功能

vrrp gratuitous-arp timeout disable命令用来禁止Master发送免费ARP报文。

缺省情况下，Master每隔300秒（5分钟）发送一次免费ARP报文。

命令格式

vrrp gratuitous-arp timeout disable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

如果对于安全有严格的要求，发送免费ARP报文会导致ARP攻击。所以，需要使用此命令来禁止Master发送免费ARP报文。

使用实例

# 禁止Master发送免费ARP报文。

[Quidway] vrrp gratuitous-arp timeout disable
