loopback（ATM OC/STM）

命令功能

loopback命令用来设置ATM接口的环回功能。

undo loopback命令用来取消环回。

缺省情况下，禁止任何形式的环回。

命令格式

loopback { cell | local | remote }

undo loopback

参数说明

cell：内信元环回，主要用来检测本端物理芯片是否正常。

local：内环回，要用来检测本端业务芯片是否正常。

remote：外环回，要用来检测对端是否正常。

ATM接口视图

缺省级别

2：配置级

使用指南

只有在进行某些特殊功能测试的时候，才将接口设置为对内自环或对外环回。正常工作时，不要启用环回功能。

使用实例

# 使能ATM接口为对内自环。

<Quidway> system-view
[Quidway] interface atm 4/0/0
[Quidway-Atm4/0/0] loopback local
