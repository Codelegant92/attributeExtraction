display mac-limit

命令功能

display mac-limit命令用来查看地址学习限制规则。

命令格式

display mac-limit [ vsi vsi-name | [ interface-type interface-number ] [ vlan vlan-id ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

interface-type：接口类型，可以是Ethernet、GigabitEthernet、Eth-Trunk、Virtual-Ethernet。

interface-number：接口编号。

vlan-id：VLAN ID，取值范围是1～4094。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

MAC Limit is enabled

MAC地址学习限制规则处于使能状态。

Total MAC Limit rule count :

MAC地址学习限制的总数。

PORT

被限制MAC地址学习的端口号。

VLAN/VSI

接口所属的VLAN ID或VSI名。

Maximum

MAC地址学习的最大数量。

Rate(ms)

MAC地址的学习速率。

Action
当MAC地址达到限制后采取的动作。

  • discard：丢弃

  • forward：转发

Alarm
当MAC地址达到限制后是否进行提示。

  • enable：进行提示

  • disable：不进行提示
