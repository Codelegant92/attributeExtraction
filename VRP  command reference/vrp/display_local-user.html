display local-user

命令功能

display local-user命令用来查看本地用户的属性。

命令格式

display local-user [ domain domain-name | username user-name ]

参数说明

user-name：用户名，字符串形式，格式为“纯用户名@域名”，不区分大小写。用户名和域名中都不能包括问号“?”和“@”。长度范围是1～64。

domain-name：指定本地用户的域名。字符串形式，长度范围是1～20。

使用指南

执行该命令时需要注意以下几点。

  • 如果不使用任何可选参数，将显示所有用户的概要信息；

  • 如果使用关键字user-name，将显示指定用户的详细属性信息，包括用户状态、CAR级别和闲置切断数据等；

  • 如果使用其它关键字，将显示符合条件的用户的概要信息。

使用实例

# 查看所有本地用户的概要信息。

<Quidway> display local-user
  ----------------------------------------------------------------
  Username             State  Type         CAR Access-limit Online
  ----------------------------------------------------------------
  aaa@163              Active All          Dft    1              0
  aaa                  Active All          Dft    No             0
  ----------------------------------------------------------------
  Total 2,2 printed

# 查看指定本地用户aaa的详细信息。

<Quidway> display local-user username aaa
  --------------------------------------------------------------
  Username          : aaa
  Password          : huawei
  State             : Active
  Service-type      : All
  ACL-number        : -
  User-CAR          : -
  Idle-cut          : No
  Access-limit      : No
  Online-number     : 0
  MAC-address       : -
  User-level        : 0
  FTP-directory     : -
  Call-number       : -
  Callback-check    : Yes
  Callback-number   : -
  ------------------------------------------------------------

项目

描述

User-name

本地用户名

Password

本地用户的密码

State

本地用户的状态

Service-type

本地用户的接入类型

ACL-number

指定本地用户名接入时的过滤条件

User-CAR

本地用户的流量控制级别

Idle-cut

本地用户是否设置闲置切断

Access-limit

允许接入本地用户的最大用户数

Online-number

在线的本地用户数量

MAC-address

本地用户的MAC地址

User-level

本地用户的等级

FTP-directory

本地用户的FTP目录

Call-number

本地用户的ISDN主叫号码

Callback-check

用户的回呼认证属性

Callback-number

本地用户的回呼号码
