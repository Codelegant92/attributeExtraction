oam（HDLC）

命令功能

oam命令用来使能HDLC链路的故障检测和故障通告。

undo oam命令用来去使能HDLC链路的故障检测和故障通告。

命令格式

oam { detect hello-stop | notify hello-stop } *

undo oam { detect hello-stop | notify hello-stop } *

参数说明

detect：PE设备检测本端AC链路故障的方式。

hello-stop：用停止发送keep-alive报文(或称为Hello报文)的方式进行故障检测和通告。

notify：把PSN侧故障通告给本端CE或者把本端AC侧故障通告给远端PE。

使用指南

PE检测到本端AC链路故障后，会把此故障发送给OAM-Manager模块。通过OAM-Manager模块把此AC链路故障通告给远端PE，从而完成端到端的FRR功能。

使用实例

# 配置HDLC类型接口的OAM故障检测和通告。

[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] link-protocol hdlc
[Quidway-Pos1/0/0] oam detect hello-stop notify hello-stop
