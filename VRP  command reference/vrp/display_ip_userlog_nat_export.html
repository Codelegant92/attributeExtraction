display ip userlog nat export

命令功能

display ip userlog nat export命令用来查看指定接口板的用户日志报文输出方式的配置和运行统计的信息。

命令格式

display ip userlog nat export slot slot-number

参数说明

slot-number：接口板槽号。

视图

所有视图

缺省级别

1：监控级

使用指南

display ip userlog nat export命令可以观察到的信息包括：

  • 报文格式的版本号

  • 日志报文的目的地址

  • 日志报文的源地址

  • 输出的日志记录数与日志报文数

  • 输出失败的日志记录数与日志报文数

  • 缓存中等待输出的日志报文数

使用实例

# 查看路由器2号槽位和3号槽位的用户日志信息。

<Quidway> display ip userlog nat export slot 2
NAT：
    Version 1 export is enabled
    Exporting logs to 10.10.0.10 （30000）
    Export using source address 1.1.1.1
    10000 logs exported in 400 udp datagrams
    10 logs in 1 udp datagrams failed to output
    0 entries buffered currently
<Quidway> display ip userlog nat export slot 3
NAT：
No ip userlog export is enabled
表1 display ip userlog export显示信息解释

项目

描述

NAT

表示显示的是NAT日志信息

Version 1 export is enabled

UDP报文输出方式使能，报文格式为版本1

Exporting logs to 10.10.0.10 （30000）

日志报文的目的IP地址，以及UDP目的端口号，即用户日志服务器的IP地址，以及监听的用户日志端口号

Export using source address 1.1.1.1

配置的日志报文的源IP地址。如果没有配置，显示为0。日志报文的源IP地址为日志报文输出接口的IP地址

10000 logs exported in 400 udp datagrams

输出的日志记录数与日志报文数

10 logs in 1 udp datagrams failed to output

输出失败的日志记录数与日志报文数

0 entries buffered currently

缓存等待输出的日志表项

No ip userlog export is enabled

用户日志功能未启用，或者虽然启用了用户日志功能，但以系统日志方式输出，而不是UDP报文输出方式
