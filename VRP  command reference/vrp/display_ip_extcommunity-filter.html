display ip extcommunity-filter

命令功能

display ip extcommunity-filter命令用来显示已配置的扩展团体属性过滤器。

命令格式

display ip extcommunity-filter [ extcomm-filter-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

extcomm-filter-number：扩展团体属性过滤器号，取值范围1～199。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示扩展团体属性过滤器信息。

<Quidway> display ip extcommunity-filter
Extended Community filter Number 10
         permit rt : 100:10
