rpr ring-selection

命令功能

rpr ring-selection命令用来配置静态选环。

undo rpr ring-selection命令用来取消配置静态选环。

命令格式

rpr ring-selection { ip-address | mac-address } { west | east } [ force ]

undo rpr ring-selection { ip-address | mac-address }

参数说明

ip-address：下一跳IP地址，点分十进制格式。

mac-address：下一跳MAC地址，格式为H-H-H，H为1至4位的十六进制数。

west：本地节点的西向连接。

east：本地节点的东向连接。

force：静态配置和动态选环对应表项冲突时，该条静态配置强制替换对应表项。

缺省级别

2：配置级

使用指南

通过合理配置静态选环，可以更好的利用环上带宽，避免一个环向流量极大，而另一个环向流量较小。

除非特别设计，请用户不要配置force关键字。因为指定force的静态选环配置一旦生效，当指定环向错误，无法抵达目的路由器时，依旧按照用户指定进行转发，将导致报文丢弃或严重影响环网可用带宽。

使用实例

# 配置一条静态选环表项下一条地址为2-2-2。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr ring-selection 2-2-2 west
