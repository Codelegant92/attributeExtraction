remotehelp

命令功能

remotehelp命令用来显示FTP协议命令的帮助。

命令格式

remotehelp [ command ]

参数说明

command：FTP协议命令，取值范围1～16个字符。

视图

FTP客户端视图

缺省级别

3：管理级

使用指南

FTP协议命令有：user、pass、cwd、cdup、quit、port、pasv、type、retr、stor、dele、rmd、 mkd、pwd、list、nlst、syst、help、xcup、xcwd、xmkd、xpwd、xrmd。

使用实例

# 查看协议命令user的语法。

<Quidway> ftp 172.16.104.110
Trying ftp 172.16.104.110
Press CTRL+K to abort
Connected to ftp 172.16.104.110
220 FTP service ready.
User(ftp 172.16.104.110:(none)):huawei
331 Password required for huawei
Password:
230 User logged in.
[ftp] remotehelp user
214 Syntax: USER <sp> <username>
