display transmission-alarm threshold

命令功能

display transmission-alarm threshold命令用来查看指定接口的b3tca、sdbere和sfbere的告警门限值。如果没有指定接口，则查看所有接口的这三类告警的门限值。

命令格式

display transmission-alarm threshold [ interface-type interface-number ]

参数说明

interface-type interface-number：接口名和接口编号。

使用指南

b3tca、sdbere和sfbere告警的门限值在各个接口上可能各不相同，如果想快速查看接口上的门限值，可以执行命令display transmission-alarm threshold。若不指定接口，则查看所有接口的这三类告警的门限值。

使用实例

# 查看POS1/0/0接口的b3tca、sdbere和sfbere三类告警的门限值。

<Quidway> display transmission-alarm threshold pos 1/0/0
Interface: Pos1/0/0
    b3tca threshold: 10e-7
    sdbere threshold: 10e-9
    sfbere threshold: 10e-8

项目

描述

Interface:

查看的接口名称和接口编号。

b3tca threshold：

B3超越门限值告警指示的门限值。

sdbere threshold:

信号损耗门限告警指示的门限值。

sfbere threshold:

信号失效门限告警指示的门限值。
