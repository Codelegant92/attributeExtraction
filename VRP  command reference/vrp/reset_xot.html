reset xot

命令功能

对于SVC：reset xot命令用来主动清除一条XOT链接。

对于PVC：reset xot命令用来主动复位一条XOT链接

命令格式

reset xot local local-ip-address local-port remote remote-ip-address remote-port

参数说明

local-ip-address：XOT连接的本地IP地址。

local-port：XOT连接的本地端口号。取值范围是1～65535。

remote-ip-address：XOT连接的远端IP地址。

remote-port：XOT连接的远端设备的端口号。取值范围是1～65535。

使用指南

通过reset xot命令可以主动清除或复位XOT连接。在清除或复位XOT连接时所需要用到的端口号可以通过命令display x25 xot获得。

使用实例

# 清除或复位一条XOT链接。

<Quidway> reset xot local 10.1.1.1 1998 remote 10.1.1.2 1024
