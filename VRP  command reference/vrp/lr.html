lr

命令功能

lr命令用来启动限速。

undo lr命令用来取消限速。

缺省情况下，不启动限速。

命令格式

lr cir cir [ cbs cbs [ ebs ebs ] ]

undo lr

参数说明

cir：承诺信息速率，取值范围是8000～155000000，单位是bps。

cbs：承诺突发尺寸的值，取值范围是15000～155000000bit。cbs的取值应大于等于cir取值的1/2，当cir>30000bit/s时，缺省取值是cir的1/2，当cir<30000bit/s时，缺省取值是15000。

ebs：过度突发尺寸，取值范围是0～155000000bit，缺省为0，即只采用一个令牌桶监管。

使用指南

限速（LR，Line Rate）同GTS的区别在于GTS工作于IP层，LR工作于链路层。

lr命令只在NE16E/08E/05路由器上可用。

使用实例

# 为行为配置限速（LR），正常流量为38400bps，在第一时间可以有2倍于正常流量的突发流量通过，以后速率小于等于38400bps时正常发送，速率大于38400bps时，将进入队列缓存。
