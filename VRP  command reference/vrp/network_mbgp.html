network（MBGP）

命令功能

network命令用来配置MBGP发布的本地网络路由。

undo network命令用来删除已有配置。

缺省情况下，MBGP不发布任何本地的网络路由。

命令格式

network ip-address [ mask-length | mask ] [ route-policy route-policy-name ]

undo network ip-address [ mask-length | mask ]

参数说明

ip-address：表示MBGP发布的网络地址。点分十进制形式。

mask：表示网络地址的掩码 。

mask-length：表示网络地址的掩码长度。

route-policy route-policy-name：指定发布路由应用的路由策略。route-policy-name是路由策略名称。字符串形式，长度范围是1～40。

缺省级别

2：配置级

使用指南

network 命令用来发布精确匹配的路由。也就是说，只有路由地址前缀和掩码完全符合配置时，路由才能正确发布。如果网络掩码没有指定，此路由将被按照自然网段精确匹配。

使用实例

# 指定MBGP发布本地网段路由 10.0.0.0/16。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] network 10.0.0.0 255.255.0.0
