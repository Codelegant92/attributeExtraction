ipv6 pathmtu

命令功能

ipv6 pathmtu命令用来对指定的IPv6地址配置路径最大传输单元（PMTU）。

undo ipv6 pathmtu 命令用来删除指定IPv6地址的PMTU配置。

缺省情况下，IPv6地址的PMTU值是1500字节。

命令格式

ipv6 pathmtu ipv6-address [ [ vpn6-instance vpn6-instance-name ] path-mtu ]

undo ipv6 pathmtu ipv6-address [ vpn6-instance vpn6-instance-name ]

参数说明

ipv6-address：指定IPv6地址，格式为X:X::X:X 。它可以是单播、组播或任播地址，但不可以是链路本地地址。

vpn6-instance vpn6-instance-name：指定IPv6 VPN实例。字符串形式，长度范围是1～31，区分大小写。

path-mtu：指定路径最大传输单元。整数形式，单位是字节，取值范围是1280～10000。缺省值为1500字节。

使用指南

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 为指定的IPv6地址配置路径最大传输单元为1300字节。

<Quidway> system-view
[Quidway] ipv6 pathmtu 3001::1 1300

# 为IPv6 VPN实例的2000::1地址配置路径最大传输单元为1600字节。

[Quidway] ipv6 pathmtu 2000::1 vpn6-instance vpn6 1600
