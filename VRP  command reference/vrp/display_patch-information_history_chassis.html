display patch-information history（多框）

命令功能

display patch-information history命令用来查看补丁操作历史信息，该命令行记录补丁加载起始和终止时间；补丁激活起始和终止时间以及补丁运行的起始和终止时间。

命令格式

display patch-information history [ chassis chassis-number | slave | slot slot-id ]

参数说明

chassis chassis-number：转发框的框号。

slave：系统备用板信息。

slot slot-id：板号（输入的形式是框类型＋框号＋板号）。

视图

所有视图

缺省级别

1：监控级

使用指南

该命令行可以查询最近的几次补丁操作的记录，包括各种状态的起始时间和终止时间，支持掉电之后补丁信息不丢失。

使用实例

# 显示当前系统中补丁操作的历史信息。

<Quidway> display patch-information history
Current patch state:
---------------------------------------------------------------------------

---------------------------------------------------------------------------

项目

Type

补丁类型

  • NP：NP类型的补丁

  • deactive：去激活态

  • idle：无补丁

To

处于此状态的补丁的结束时间
