local-user password

命令功能

local-user password命令用来增加一个本地用户。

undo local-user命令用来删除一个本地用户。

命令格式

local-user user-name password { simple | cipher } password

undo local-user user-name

参数说明

user-name：用户名，字符串形式，长度范围是1～64。如果用户名中带@，则认为@前面的部分是用户名，后面部分是域名。如果没有@，则整个字符串为用户名，域为默认域。

simple：以明文形式显示口令，用display命令显示用户信息时，用户口令以不加密的形式显示。

cipher：以密文形式显示口令，用display命令显示用户信息时，用户口令以密文形式显示。

password：口令，字符串形式，不加密输入时，长度范围是1～16；加密输入时，长度范围是24。区分大小写，不能输入空格和问号等命令行专用字符。

使用指南

如果是创建新用户，建议在创建用户的同时设置口令。

选用密文还是用明文显示口令，需要看用户的需求，选用密文的安全度更高。

系统最多可配置1000个本地用户。

使用实例

# 增加一个本地用户，用户名为hello，域名为163.net。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] local-user hello@163.net password cipher hello
