snmp-agent

命令功能

snmp-agent命令用来使能SNMP Agent，指定SNMP配置信息。

undo snmp-agent命令用来禁止SNMP Agent。

缺省情况下，关闭SNMP Agent功能。

命令格式

snmp-agent

undo snmp-agent

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

可以使用snmp-agent命令来启动SNMP Agent，同时snmp-agent的任何一条配置命令也将启动SNMP Agent。

在SNMP Agent没有使能的时候配置undo命令将无效。在SNMP Agent使能的情况下，可以使用undo snmp-agent命令禁止SNMP Agent。

使用实例

# 使能SNMP Agent。

[Quidway] snmp-agent
