cluster switch-to

命令功能

cluster switch-to命令用来从命令交换机telnet到指定的成员交换机。

命令格式

cluster switch-to { member-number | mac-address mac-address }

参数说明

member-number：成员交换机的成员编号，十进制整数形式，取值范围是1～255。

mac-address mac-address：成员交换机的MAC地址。mac-address是用16进制表示的成员交换机的MAC地址。格式为H-H-H，每个“H”代表4位16进制数。

视图

用户视图

缺省级别

0：参观级

使用指南

本命令只能在命令交换机上执行。

在成员交换机的用户视图下输入quit命令，则结束telnet，返回命令交换机。

使用实例
