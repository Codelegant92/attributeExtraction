igmp-snooping static-router-port(接口视图)

命令功能

igmp-snooping static-router-port命令用来配置当前接口作为指定VLAN内的静态路由端口。

undo igmp-snooping static-router-port命令用来取消当前接口作为指定VLAN内的静态路由端口的配置。

命令格式

igmp-snooping static-router-port vlan { { vlan-id [ to vlan-id ] } &<1-10> }

undo igmp-snooping static-router-port vlan { { vlan-id [to vlan-id ] } &<1-10> }

参数说明

vlan vlan-id：VLAN编号，取值范围是1～4094。

视图

接口视图

缺省级别

2：配置级

使用指南

静态路由端口永不老化。

使用此命令前，当前接口必须已加入参数vlan-id所指定的VLAN，否则配置无效。

使用实例

# 配置GigabitEthernet2/0/1为VLAN 2内的静态路由端口。

<Quidway> system-view
[Quidway] interface gigabitethernet2/0/1
[Quidway-GigabitEthernet2/0/1] igmp-snooping static-router-port vlan 2
