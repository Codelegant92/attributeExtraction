mac-learn-style

命令功能

mac-learn-style命令用于配置VSI实例的MAC地址学习方式。

缺省情况下，MAC地址学习方式是unqualify。

目前，VRP只支持unqualify方式的MAC地址学习。

命令格式

mac-learn-style { qualify | unqualify }

参数说明

qualify：MAC地址的qualify学习方式。

unqualify：MAC地址的unqualify学习方式。

视图

VSI视图

缺省级别

2：配置级

使用指南

对于Kompella方式的VPLS，使用mac-learn-style命令前，需要先执行pwsignal命令和route-distinguisher（VSI-BGP视图）命令；对于Martini方式的VPLS，使用mac-learn-style命令前，需要先执行pwsignal命令和vsi-id命令。

执行mac-learn-style命令后，如果选择unqualify方式，则MAC地址的学习是基于VSI实例的，每一个VSI实例有一个MAC地址空间。

如果选择qualify方式，则MAC地址的学习是基于VSI实例的VLAN的，每个VLAN都有自己的MAC地址空间，不同VLAN的MAC地址空间可以相互重叠。

使用实例

# 配置当前VSI实例的MAC地址学习方式。

<Quidway> system-view
[Quidway] vsi company1 auto
[Quidway-vsi-company1] pwsignal bgp
[Quidway-vsi-company1-bgp] route-distinguisher 100:1
[Quidway-vsi-company1-bgp] quit
[Quidway-vsi-company1] mac-learn-style qualify
