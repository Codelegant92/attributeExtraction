display x25 xot

命令功能

display x25 xot命令用来显示XOT链接信息。

命令格式

display x25 xot

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

通过display x25 xot命令可以查看所有的XOT链接的详细情况：包括远端IP和端口，本地IP和端口，socket的keepalive属性设置和入出接口名。

使用实例

# 显示XOT链路信息。

<Quidway> display x25 xot
SVC 1024:  ( ESTAB )
   tcp peer ip: 10.1.1.1, peer port: 1998
   tcp local ip: 10.1.1.2, local port: 1024
   socket keepalive period: 5, keepalive tries: 3
   come interface name: Serial1/0/0-10.1.1.1-1024
   go interface name: Serial1/0/0
表1 display x25 xot命令输出信息描述

项目

描述

SVC 1024: ( ESTAB )

有一条通过SVC并且已经建立的XOT链路存在。

tcp peer ip: 10.1.1.1, peer port: 1998

对端IP是10.1.1.1，端口是1998。

tcp local ip: 10.1.1.2, local port: 1024

本地IP是10.1.1.2，端口是1024。

socket keepalive period: 5, keepalive tries: 3

socket的keepalive属性是5秒，重复次数为3次。

come interface name

入接口名是Serial1/0/0-10.1.1.1-1024(XOT接口)。

go interface name

出接口名为Serial1/0/0。
