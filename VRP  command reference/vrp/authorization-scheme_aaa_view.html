authorization-scheme（AAA视图）

命令功能

authorization-scheme命令用来创建一个授权方案，并进入授权方案视图。

undo authorization-scheme命令用来删除一个已经存在的、没有被任何域绑定的授权方案。

系统固定有一个“default”方案，不能删除，只能修改。

命令格式

authorization-scheme authorization-scheme-name

undo authorization-scheme authorization-scheme-name

参数说明

authorization-scheme-name：授权方案名称，字符串形式，长度范围是1～32。不区分大小写，方案名需要符合Windows的命名规范，不能包括斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“””和尖括号“<”、“>”等字符。

视图

AAA视图

缺省级别

3：管理级

使用指南

authorization-scheme命令在创建一个授权方案时，如果同名的授权方案已经存在，将直接进入该授权方案视图。

undo authorization-scheme只能删除一个已经存在的、没有被任何域绑定的授权方案。

系统最多可配置16个授权方案。

使用实例

# 增加一个新的授权方案，名为newscheme。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] authorization-scheme newscheme
  Create a new authorization scheme  
[Quidway-aaa-author-newscheme]
