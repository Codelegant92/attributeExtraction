description（接口视图）

命令功能

description命令用来设置接口描述信息。

undo description命令用来恢复缺省的接口描述。

缺省情况下，系统不显示接口的默认描述信息。接口默认描述信息为“HUAWEI, Quidway Series, interface-type interface-number Interface”。

命令格式

description interface-description

undo description

参数说明

interface-description：路由器接口的描述信息。字符串形式，支持空格，区分大小写，字符串长度范围是1～242。

视图

接口视图

缺省级别

2：配置级

使用指南

本命令仅用于标识某接口，并无特别的功能。使用display interface等命令可以看到设置的描述信息。描述信息把输入的第一个非空格字符作为第一个字符开始显示，最多可显示242个字符。

使用实例

# 设置指定千兆以太网接口描述为Connect to NE5000E-1 GigabitEthernet1/0/0。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] description Connect to NE5000E-1 GigabitEthernet1/0/0
