mpls l2vpn default martini

命令功能

mpls l2vpn default martini命令用来设置动态VC的信令行为是非Notification方式，也就是Martini方式。

undo mpls l2vpn default martini命令用来恢复动态VC的信令行为是Notification方式，即PWE3方式。

缺省情况下，系统的模式是PWE3。

命令格式

mpls l2vpn default martini

undo mpls l2vpn default martini

参数说明

无

使用指南

PWE3模式支持Notification消息报文协商PW状态信息。而Martini方式不支持Notification消息报文，也没有根据远端信息的自适应行为。

说明：

关于Notification消息报文的详细说明请参见《VRP 配置指南-VPN》的“PWE3配置”。

如果对端PE不支持Notification消息报文，需要使用mpls l2vpn default martini命令把系统模式切换成Martini方式。

使用实例

# 配置动态PW的信令为不支持Notification消息方式。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] mpls l2vpn default martini

# 恢复动态PW的信令为支持Notification消息方式。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] undo mpls l2vpn default martini
