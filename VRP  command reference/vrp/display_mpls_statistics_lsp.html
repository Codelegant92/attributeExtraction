display mpls statistics lsp

命令功能

display mpls statistics lsp命令用来查看LSP统计信息，包括每一条LSP的入口和出口处理的字节数、报文数、错误数，丢弃报文数。

命令格式

display mpls statistics lsp { all | lsp-index | name lsp-name } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：查看所有LSP的统计信息。

lsp-index：LSP索引号，取值范围是0～4294967295。

name lsp-name：LSP的名称，字符串形式，区分大小写，不支持空格，长度范围是1～19。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果要查看统计信息，请先使用statistics interval命令设置上报统计信息的时间间隔，缺省时间间隔为0，即不上报统计信息。这种情况下显示出来的各项统计信息都为0。

当统计信息中的字节数（Octets）、报文数（Packets）或错误数（Errors）超过显示允许的最大值0xFFFFFFFF时，统计信息将显示为“剩余数量＋n×4294967295”，其中，“4294967295”是0xFFFFFFFF的十进制形式。

例如，当显示信息为“Octets : 2954218368 + 1 * 4294967295”时，表示实际字节数为7249185663。

说明：
  • Ingress不对入方向进行统计，因此显示信息中InSegment的开始时间和结束时间都为0。
  • Egress不对出方向进行统计，因此显示信息中OutSegment的开始时间和结束时间都为0。
  • 目前只提供出方向的LSP统计功能，不提供入方向的LSP统计。
