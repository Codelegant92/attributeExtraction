display rsa local-key-pair public

命令功能

display rsa local-key-pair public命令用来查看本地密钥对中的公钥部分信息。

命令格式

display rsa local-key-pair public

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

配置路由器时，在客户端执行此命令，并将回显信息中的客户端公钥粘贴到SSH服务器端RSA公钥中去。
