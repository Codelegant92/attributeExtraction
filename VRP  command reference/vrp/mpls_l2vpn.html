mpls l2vpn

命令功能

mpls l2vpn命令用来使能L2VPN并进入L2VPN视图。只有通过此命令使能了L2VPN后，才能配置其他的L2VPN命令。

undo mpls l2vpn命令用来去使能L2VPN，并删除所有L2VPN配置。

缺省情况下，系统没有使能L2VPN功能。

命令格式

mpls l2vpn

undo mpls l2vpn

参数说明

无

系统视图

缺省级别

2：配置级

使用指南

配置过程中，P设备上不需要使能MPLS L2VPN。

mpls l2vpn命令后：

  • 如果设备上未使能L2VPN功能，则会使能L2VPN功能并进入L2VPN视图。
  • 如果设备上已使能L2VPN功能，则会进入L2VPN视图。

使用实例

# 使能L2VPN。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] quit
