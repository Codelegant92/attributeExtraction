access-vlan

命令功能

access-vlan命令用来将一个或一组sub-VLAN加入super-VLAN中。

undo access-vlan命令用来表示将一个或一组sub-VLAN从super-VLAN中删除。

命令格式

access-vlan { vlan-id1 [ to vlan-id2 ] } &<1-10>

undo access-vlan { vlan-id1 [ to vlan-id2 ] } &<1-10>

参数说明

vlan-id：sub-VLAN的ID。取值范围是1～4094。

视图

VLAN视图

缺省级别

2：配置级

使用指南

向super-VLAN中增加一个或一组sub-VLAN，可以采用区间输入方法，但要保证关键字to之后VLAN值大于前面的VLAN值，保证输入的区间没有交叉，可以输入1～10次。

使用实例

# 向super-VLAN 2中加入sub-VLAN 3～10，50，20～30。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] aggregate-vlan
[Quidway-vlan2] access-vlan 3 to 10 50 20 to 30
