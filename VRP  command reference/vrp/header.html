header

命令功能

header命令用来设置登录显示标题。

undo header命令用来关闭登录显示标题。

命令格式

header { login | shell } information text

header { login | shell } file file-name

undo header { login | shell }

参数说明

login：当用户在登录路由器过程中，激活终端连接时显示标题信息。

shell：当用户成功登录到路由器上，已经建立了会话时显示标题信息。

information text：指定标题信息和内容。字符串形式，支持空格，长度范围是1～220。

file file-name：指定标题所使用的文件名。字符串形式，长度范围是5～64。该文件名所标识的标题文件内容最大为128K字节，如果大于128K则只显示前面128K字节的内容。

使用指南

用户经过终端线登录路由器时，路由器通过设置标题属性向用户提示有关信息。终端连接被激活但用户尚未成功登录时，则系统发送登录标题login的信息内容到终端。如果用户成功登录，则系统显示shell标题。

标题内容文本以第一个英文字符作为起始和结束符。输入结束符后，系统自动退出交互过程。

如果不想进入交互过程，只要保证文本首尾有相同的英文字符，直接敲入回车即可。

设置了登录标题后，任何用户登录到系统上都可以看到该标题。

使用实例

# 配置会话建立标题。

<Quidway> system-view
[Quidway] header shell information %
info:input banner text, and quit with the character '%'.
SHELL : Hello! Welcome use NetEngine%
[Quidway] quit
<Quidway>

# 用户登录成功后会显示Shell标题。

Username:quidway
Password:******
SHELL : Hello! Welcome use NetEngine
Note: The max number of VTY users is 5, and the current number
of VTY users on line is 2.

# 指定登录标题使用的文件。

<Quidway> system-view
[Quidway] header login file flash:/header-file.txt
