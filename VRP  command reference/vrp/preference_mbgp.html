preference（MBGP）

命令功能

preference命令用来配置外部、内部、本地路由的管理优先级。

undo preference命令用来恢复各优先级的缺省值。

缺省情况下，各种路由的优先级的均为255。

命令格式

preference { external-preference internal-preference local-preference | route-policy route-policy-name }

undo preference

参数说明

external-preference：指定EBGP外部路由的管理优先级。外部路由是从其它自治系统的对等体学来的最佳路由。整数形式，取值范围是1～255。

internal-preference：指定IBGP内部路由的管理优先级。内部路由是从同一自治系统内的其它对等体学来的路由。整数形式，取值范围是1～255。

local-preference：MBGP本地路由的管理优先级。整数形式，取值范围是1～255。

route-policy route-policy-name：路由策略名称。
