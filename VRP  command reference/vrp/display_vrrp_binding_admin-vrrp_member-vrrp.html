display vrrp binding admin-vrrp member-vrrp

命令功能

display vrrp binding admin-vrrp member-vrrp命令用来显示管理VRRP备份组与成员VRRP备份组的绑定关系。

命令格式

display vrrp binding admin-vrrp [ interface interface-type1 interface-number1 ] [ vrid virtual-router-id1 ] member-vrrp [ interface interface-type2 interface-number2 ] [ vrid virtual-router-id2 ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type1 interface-number1：管理VRRP备份组所在的接口类型和接口编号。

vrid virtual-router-id1：管理VRRP备份组号。十进制整数形式，取值范围是1～255。

interface interface-type2 interface-number2：成员VRRP备份组所在的接口类型和接口编号。

vrid virtual-router-id：成员VRRP备份组号。十进制整数形式，取值范围是1～255。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display vrrp binding admin-vrrp member-vrrp命令时，如果不指定管理VRRP备份组号和成员VRRP备份组号，则显示所有管理VRRP备份组和成员VRRP备份组的绑定信息。

使用实例

# 显示所有的管理VRRP备份组和成员VRRP备份组的绑定信息。

<Quidway> display vrrp binding admin-vrrp member-vrrp
interface: GigabitEthernet1/0/0,  admin-vrrp vrid: 1, state:master
  member-vrrp number: 3
    interface: GigabitEthernet2/0/0.1, vrrp vrid: 1, state: master
    interface: GigabitEthernet2/0/0.1, vrrp vrid: 2, state: master
    interface: GigabitEthernet2/0/0.3, vrrp vrid: 3, state: master
