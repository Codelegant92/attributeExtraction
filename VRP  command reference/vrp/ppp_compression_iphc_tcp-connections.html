ppp compression iphc tcp-connections

命令功能

ppp compression iphc tcp-connections命令用来配置TCP头压缩的最大连接数。

undo ppp compression iphc tcp-connections命令用来恢复TCP头压缩的最大连接数的缺省值。

命令格式

ppp compression iphc tcp-connections number

undo ppp compression iphc tcp-connections

参数说明

number：该接口上IPHC功能的TCP头压缩的最大连接数，取值范围3～256，缺省值为16。

视图

POS接口视图、串行接口视图、虚拟接口模板视图、MP-Group接口视图、Dialer接口视图

缺省级别

2：配置级

使用指南

如果压缩端和解压缩端配置的TCP头压缩最大连接数不一致，经IPCP（IP Compression Protocol）协商后，TCP头压缩的最大连接数会取其中的最大值。

使用实例

# 配置RouterA的TCP头压缩最大连接数为100。

<RouterA> system-view
[RouterA] interface serial 2/0/0:0
[Quidway-serial2/0/0:0] link-protocal ppp
[RouterA-serial2/0/0:0] ppp compression iphc tcp-connections 100
