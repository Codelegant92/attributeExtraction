ipv6 import-route isis level-1 into level-2

命令功能

ipv6 import-route isis level-1 into level-2命令用来控制IPv6中Level-1路由向Level-2区域的发布，有利于路由管理。该命令需配置在区域边界的Level-1-2路由器上。

undo ipv6 import-route isis level-2 into level-1命令用来禁止IPv6中Level-1路由向Level-2区域的发布。

缺省情况下，不对Level-1路由向Level-2区域的发布进行控制，即所有的Level-1路由都向Level-2区域发布。

命令格式

ipv6 import-route isis level-1 into level-2 [ filter-policy { acl6-number | ipv6-prefix ipv6-prefix-name | route-policy route-policy-name } ] [ tag tag ]

undo ipv6 import-route isis level-1 into level-2

参数说明

filter-policy：设置IPv6路由过滤条件。

acl6-number：基本ACL6编号。

ipv6-prefix ipv6-prefix-name：指定IP地址前缀名。只有与该前缀相匹配的路由才能引入到Level-2区域。

route-policy route-policy-name：指定路由策略名称。

tag tag：为引入的路由分配管理标签号。

使用指南

使用此命令前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。
