ipv6 nd autoconfig managed-address-flag

命令功能

ipv6 nd autoconfig managed-address-flag命令用来设置有状态自动配置的可路由地址标志位。

undo ipv6 nd autoconfig managed-address-flag命令用来取消该标志位的设置。

命令格式

ipv6 nd autoconfig managed-address-flag

undo ipv6 nd autoconfig managed-address-flag

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

如果设置了该标志位，则主机通过有状态自动配置（例如DHCP服务器）来获得可路由的IPv6地址。如果清除了标志位，则主机进行无状态自动配置。

缺省情况下，没有设置该标志位。

此命令支持Ethernet接口及其子接口、GigabitEthernet接口及其子接口、POS接口以及Tunnel接口。

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 设置有状态自动配置的可路由地址标志位。

<Quidway> system-view
[Quidway] interface gigaitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 nd autoconfig managed-address-flag
