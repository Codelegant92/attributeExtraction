tunnel ipv4-ipv6 hop-limit

命令功能

tunnel ipv4-ipv6 hop-limit命令用来设置跳数限制值。

undo tunnel ipv4-ipv6 hop-limit命令用来使跳数限制值恢复为缺省值。

缺省情况下，跳数限制值为64。

命令格式

tunnel ipv4-ipv6 hop-limit hop-limit

undo tunnel ipv4-ipv6 hop-limit

参数说明

hop-limit：跳数限制值。整数形式，取值范围是1～255。

视图

系统视图、Tunnel接口视图

缺省级别

2：配置级

使用指南

设置隧道跳数限制值可以使IPv6隧道中出现路由循环时能快速终止报文传输。为了使IPv6隧道报文能够到达隧道的出口点，隧道跳数限制值不能过小。

在隧道接口视图下配置该命令则仅对当前隧道接口生效。目前，在系统视图下虽然可以配置该命令，但是配置并不生效。

使用实例

# 设置指定Tunnel接口的跳数限制值为100。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel ipv4-ipv6 hop-limit 100

# 设置本设备所有隧道接口的跳数限制值为100。

<Quidway> system-view
[Quidway] tunnel ipv4-ipv6 hop-limit 100
