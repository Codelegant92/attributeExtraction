set constant-ifindex max-number

命令功能

set constant-ifindex max-number命令用来设置接口索引固定最大数量。

undo set constant-ifindex max-number命令用来恢复此配置为缺省值。

缺省情况下，接口索引固定最大数量为65535。

命令格式

set constant-ifindex max-number number

undo set constant-ifindex max-number

参数说明

number：接口索引固定的数目上限值，取值范围是0～4294967295。

系统视图

缺省级别

使用指南

该命令仅在接口索引固定特性使能后有效。

使用实例

# 设置接口索引固定最大数量为10000。

<Quidway> system
[Quidway] set constant-ifindex max-number 10000
