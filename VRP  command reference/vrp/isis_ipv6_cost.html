isis ipv6 cost

命令功能

isis ipv6 cost命令用来配置IPv6拓扑中链路的开销值。只有在链路处于激活状态并且使能了IPv6拓扑的情况下，此命令才起作用。

undo isis ipv6 cost命令用来恢复IPv6开销为缺省值。

命令格式

isis ipv6 cost cost [ level-1 | level-2 ]

undo isis ipv6 cost [ level-1 | level-2 ]

参数说明

cost：指定IPv6链路的开销值，区别于IPv4的开销值。如果IS-IS的开销类型是wide或wide-compatible，则取值范围是1～16777215；否则取值为1～63。缺省值为10。

level-1：配置level-1链路的开销值。

level-2：配置level-2链路的开销值。

缺省级别

使用指南

当IPv6的拓扑是compatible模式或者标准模式时，IPv6开销与IPv4开销相同；当IPv6的拓扑是compatible enable-mt-spf模式或者ipv6模式时，IPv6开销与IPv4开销独立配置。

如果配置中没有指定Level-1或者Level-2，则给Level-1和Level-2设置同样的链路开销。

使用此命令前，必须先使能IS-IS进程的IPv6能力，拓扑类型中包含IPv6拓扑。配置过程请参见isis ipv6 enable命令。

使用实例

# 设置接口GigabitEthernet 1/0/0的IPv6链路开销值为50。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] isis ipv6 cost 50
