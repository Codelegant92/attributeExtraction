display cfm mac-tunnel status

命令功能

display cfm mac-tunnel status命令用来查看指定MAC tunnel的连通性故障处理功能的使能状态、CC检测的时间间隔。

命令格式

display cfm mac-tunnel tunnel-name status

参数说明

tunnel-name：指定MAC tunnel的名称。字符串形式，长度范围是1～31。

使用指南

当需要了解指定MAC tunnel的连通性故障处理功能的配置情况时可使用本命令。

使用实例

# 查看名称为mytunnel0的MAC tunnel的连通性故障处理功能。

<Quidway> display cfm mac-tunnel mytunnel0 status
MAC tunnel name  CFM status  CCM-interval  CCM-receive 
------------------------------------------------------
mytunnel0        enable      10ms           enable
表1 display cfm mac-tunnel status命令输出信息描述表

项目

描述

CFM status

MAC tunnel的连通性故障处理功能的使能状态。取值包括：disable：MAC tunnel的连通性故障处理功能处于禁止状态。enable：MAC tunnel的连通性故障处理功能处于使能状态。

CCM-interval

MAC tunnel的CC检测时间间隔。

CCM-receive
MAC tunnel是否接收CCM消息，取值包括：

  • disabled：不接收CCM消息。

  • enabled：接收CCM消息。
