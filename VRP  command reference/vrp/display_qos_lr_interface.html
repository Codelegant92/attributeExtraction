display qos lr interface

命令功能

display qos lr interface命令用来显示接口的LR配置情况和统计信息。

命令格式

display qos lr interface [ interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口号。

视图

所有视图

缺省级别

1：监控级

使用指南

display qos lr interface命令显示的信息包括：

  • 接口上的承诺信息速率

  • 接口上的承诺突发尺寸

  • 接口上的过度突发尺寸

  • 发送的报文数量

  • 延迟的报文数量

  • 流量整形是否活跃

如不指定接口，将显示所有接口的LR配置情况和运行统计信息。

使用实例

# 显示所有接口的LR配置情况和统计信息。

<Quidway> display qos lr interface GigabitEthernet6/0/0
Interface: GigabitEthernet6/0/0
CIR 30000000 (bps), CBS 15000000 (bit), EBS 0 (bit)
 Passed : 1254523/125452180 (Packets/Bytes)
 Delayed: 9047/904700 (Packets/Bytes)
 Active Shaping:  NO

主要字段

描述

EBS 0(Bit)

Delay: 0/0 (Packets/Bytes)
