ip-pool

命令功能

ip-pool命令用于配置HGMP集群内部通信的IP地址池，以及命令交换机用于集群内部通信的IP地址。

undo ip-pool命令用于删除HGMP集群内部通信的IP地址池。

缺省情况下，未设置HGMP集群内部通信的IP地址池。

命令格式

ip-pool administrator-ip-address { mask-length | mask }

undo ip-pool

参数说明

administrator-ip-address：指定命令交换机用于HGMP集群内部通信的IP地址。请将其设置为合法的单播地址，否则将会提示出错信息。

mask-length：指定ip-address的掩码长度。十进制整数形式，取值范围是1～30。

mask：指定ip-address的掩码。

使用指南

本命令只有在设备没有加入任何HGMP集群时有效。

改变或删除IP地址池之前需要先删除HGMP集群，否则无法执行本命令。

使用实例

# 配置集群内部通信地址池为私网IP地址池10.10.10.0/24，并且命令交换机用于集群内部通信的私网IP地址是10.10.10.1/24。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] ip-pool 10.10.10.1 255.255.255.0
