display mpls ldp interface

命令功能

display mpls ldp interface命令用来查看使能了LDP的接口信息。

命令格式

display mpls ldp interface [ vpn-instance vpn-instance-name ] [ interface-type interface-number | verbose ] [ | { begin | exclude | include } regular-expression ]

display mpls ldp interface [ all ] [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：VPN实例的名字。使用此参数查看指定VPN实例的使能了LDP的接口的配置信息。

interface-type interface-number：接口类型和编号。使用此参数查看指定接口上LDP的配置信息。

all：显示所有使能了LDP的接口信息。

| ：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

verbose：显示使能了LDP的接口详细信息。

使用指南

此命令可以用来查看使能了LDP的接口的配置信息，以及部分实时信息，比如发送和接收的Hello消息数。

使用实例

# 查看LDP接口信息。

<Quidway> display mpls ldp interface
     LDP Interface Information in Public Network
--------------------------------------------------------------------------
 IF-Name         Status       LAM   Transport-Address   Hello-Sent/Rcv
--------------------------------------------------------------------------
 Pos1/0/0        Active       DU    172.17.1.1          2495/2514
 GE2/0/0         Active       DU    172.17.1.1          1106/1094
--------------------------------------------------------------------------
 LAM: Label Advertisement Mode         IF-Name: Interface name
表1 display mpls ldp interface命令输出信息描述

项目

描述

IF-Name

使能了LDP的接口名称

Status

实体状态：激活或非激活

LAM

标签发布模式为DU：下游自主模式

Transport-Address

会话TCP连接的IP地址

Hello-Sent/Rcv

此接口发出和接收的Hello消息数目

Configured Delay Timer
