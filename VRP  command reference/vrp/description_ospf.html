description (OSPF)

命令功能

descriptin命令用来为该OSPF进程配置描述信息。

undo description命令用来删除描述信息。

命令格式

description text

undo description

参数说明

text：OSPF进程的描述信息。字符串形式，取值范围是1～80。

OSPF视图

缺省级别

使用指南

无

使用实例

# 为OSPF进程配置描述信息。

<Quidway> system-view
[Quidway] ospf 100
[Quidway-ospf-100] description this process contains 3 areas
