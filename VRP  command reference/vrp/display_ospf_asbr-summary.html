display ospf asbr-summary

命令功能

display ospf asbr-summary命令用来查看OSPF的路由聚合信息。

如果不指定IP地址和掩码，将显示所有引入路由聚合信息。

命令格式

display ospf [ process-id ] asbr-summary [ ip-address mask ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPF进程号。整数形式，取值范围是1～65535。

ip-address：匹配的IP地址，点分十进制格式。

mask：IP地址掩码，点分十进制格式。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看OSPF的所有路由聚合信息。

<Quidway> display ospf asbr-summary

项目

描述

Total Summary address count

通过asbr-summary命令聚合的路由数量

net

聚合路由的网络地址

Mask

聚合路由的网络掩码

Tag

聚合路由的标记字段

Status
聚合路由的发布状态

  • Advertise：聚合后发布

  • DoNotAdvertise：聚合后不发布

Cost

聚合路由的开销

Delay

延迟发布聚合路由的时间

The count of route

被聚合的路由数

Destination

被聚合的路由目的地址

Net mask

被聚合的路由的掩码

Proto

被聚合的路由所属的协议

Process

进程号

Type

引入的AS外部路由类型，有Type-1和Type-2

Metric

被聚合的路由的开销值
