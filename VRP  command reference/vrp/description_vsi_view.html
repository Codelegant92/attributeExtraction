description（VSI视图）

命令功能

description命令用来配置当前VSI实例的描述信息。

undo description命令用来删除当前VSI实例的描述信息。

缺省情况下，VSI实例的描述信息为空。

命令格式

description description

undo description

参数说明

description：VSI实例的描述信息。字符串形式，支持空格，区分大小写，长度范围是1～64。

视图

VSI视图

缺省级别

2：配置级

使用指南

description命令通常可以用来描述VSI实例的作用。

使用实例

# 配置当前VSI实例的描述信息。

<Quidway> system-view
[Quidway] vsi company1
[Quidway-vsi-company1] description vsi-company1
