isis enable

命令功能

isis enable命令用来使能接口的IS-IS能力并指定要关联的IS-IS进程号，命令成功执行后，接口就会和这个IS-IS进程关联起来。

undo isis enable命令用来取消该配置。

缺省情况下，接口上不使能IS-IS路由进程。

命令格式

isis enable [ process-id ]

undo isis enable

参数说明

process-id：IS-IS进程号。整数类型，取值范围是1～65535，缺省值是1。

接口视图

缺省级别

2：配置级

使用指南

为了使IS-IS协议正常运行，必需使用isis命令使能IS-IS进程，然后使用network-entity命令为路由器设置一个网络实体名称（NET），最后在各个需要运行IS-IS进程的接口上使用isis enable命令使能各个接口。只有完成这些配置之后，IS-IS协议才能真正启动。

使用实例

# 创建IS-IS路由进程1，并在接口POS1/0/0上激活这个路由进程。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] network-entity 10.0001.1010.1020.1030.00
[Quidway-isis-1] quit 
[Quidway] interface pos1/0/0
[Quidway-Pos1/0/0] isis enable 1
