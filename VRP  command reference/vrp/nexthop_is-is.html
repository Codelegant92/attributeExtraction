nexthop (IS-IS)

命令功能

nexthop命令用来设置等价路由的优先级。

undo nexthop命令用来取消路由等价路由的优先级。

命令格式

nexthop ip-address weight value

undo nexthop ip-address

参数说明

ip-address：下一跳的IP地址。

weight value：下一跳权重。value是整数形式，取值范围是1～254。缺省值是255。

视图

IS-IS视图

缺省级别

2：配置级

使用指南

等价路由不设置优先级，同时转发报文，进行负载分担。

在IS-IS根据SPF算法算出的等价路由后，再根据weight的权重值从这些等价路由中选择下一跳，值越小优先级越高。

使用实例

# 设置IS-IS中等价路由的优先级。

<Quidway> system-view
[Quidway] isis
[Quidway-isis-1] nexthop 10.0.0.3 weight 1
