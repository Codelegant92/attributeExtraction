hello-option override-interval（IPv6）

命令功能

hello-option override-interval命令用来配置Hello消息中携带的否决Prune剪枝的时间间隔。

undo hello-option override-interval命令用来恢复时间间隔为缺省值。

缺省情况下，Hello消息中携带的否决Prune剪枝的时间间隔是2500毫秒。

命令格式

hello-option override-interval interval

undo hello-option override-interval

参数说明

interval：表示Hello消息中携带的否决Prune剪枝的时间间隔。整数形式，取值范围是1～65535，单位是毫秒。缺省值是2500。

视图

IPv6 PIM视图

缺省级别

2：配置级

使用指南

请参见IPv4 PIM配置命令中的hello-option override-interval（IPv4）。

此命令与接口视图下的pim ipv6 hello-option override-interval含义是相同的，只是视图不同。缺省情况下，如果接口上没有配置，取IPv6 PIM视图下配置的值；否则取接口视图下配置的值。

使用实例

# 在公网实例中，配置Hello消息中携带的否决剪枝的时间间隔为2000毫秒。

<Quidway> system-view
[Quidway] pim-ipv6
[Quidway-pim6] hello-option override-interval 2000
