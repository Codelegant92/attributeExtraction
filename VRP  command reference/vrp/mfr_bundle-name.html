mfr bundle-name

命令功能

mfr bundle-name命令用来设置帧中继捆绑标识符（bundle identification，BID）。

undo mfr bundle-name命令用来缺省值。

缺省情况下，捆绑标识符是“mfr＋帧中继捆绑编号”，例如：MFR4/0/123。

命令格式

mfr bundle-name [ name ]

undo mfr bundle-name

参数说明

name：捆绑标识名称。字符串形式，不支持空格，区分大小写，长度为1～49。

视图

MFR接口视图

缺省级别

2：配置级

使用指南

每个多链路帧中继捆绑都有一个捆绑标识（BID）。BID只在本地有效，因此，链路两端的BID可以相同。

改变接口的捆绑标识符时，必须对该接口执行shutdown/undo shutdown命令，新的捆绑标识才会生效。

使用实例

# 设置帧中继链路捆绑标识符为bundle1。

<Quidway> system-view
[Quidway] interface MFR 1/0/123
[Quidway-MFR1/0/123] mfr bundle-name bundle1
