display igmp-snooping

命令功能

display igmp-snooping [ vlan [ vlan-id ] | vsi [ vsi-name ] ]命令用来查看VLAN/VSI上IGMP Snooping的所有运行参数，包括默认参数和非默认参数，不区分VSI是否绑定了VLAN。

display igmp-snooping [ vlan [ vlan-id ] | vsi [ vsi-name ] ] [ configuration ]命令用来查看VLAN/VSI上IGMP Snooping的配置信息，只包括非默认参数，不区分VSI是否绑定了VLAN。

命令格式

display igmp-snooping [ vlan [ vlan-id ] | vsi [ vsi-name ] ] [ configuration ]

参数说明

vlan vlan-id：显示指定VLAN的IGMP Snooping配置信息。vlan-id表示VLAN编号，取值范围是1～4094。

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

configuration：显示VLAN/VSI的IGMP Snooping非缺省的配置信息。

使用指南

使用display igmp-snooping命令时，

  • display igmp-snooping显示所有VLAN和VSI的IGMP Snooping配置信息。

  • display igmp-snooping vlan如果不配置参数vlan-id，则显示所有VLAN的IGMP Snooping配置信息。

  • display igmp-snooping vsi如果不配置参数vsi-name，则显示所有VSI的IGMP Snooping配置信息。

表1 display igmp-snooping命令输出信息描述

项目

描述

IGMP Snooping is Enable

本VLAN/VSI内的IGMP Snooping已使能。

IGMP Version is Set to default 2

本VLAN/VSI内可以处理的IGMP报文的版本为缺省值，也就是既可以处理IGMPv1的报文，也可以处理IGMPv2的报文。

IGMP Query Interval is Set to default 60

本VLAN/VSI内的IGMP通用查询消息发送时间间隔为缺省值，即60s。

IGMP Max Response Interval is Set to default 10

本VLAN/VSI内的IGMP最大响应时间为缺省值，即10s。

IGMP Robustness is Set to default 2

IGMP健壮系数为缺省值，即2。

IGMP Last Member Query Interval is Set to default 1

本VLAN/VSI内的IGMP特定组查询消息的发送时间间隔为缺省值，即1s。

IGMP Router Port Aging Interval is Set to 180s or holdtime in hello

本VLAN/VSI内的路由端口的老化时间为180s或PIM Hello报文携带的Holdtime值。

IGMP Filter Group-Policy is Set to default : Permit All

本VLAN/VSI的组播组策略为缺省值，即允许本VLAN/.VSI内的主机加入所有组播组。

IGMP Prompt Leave Disable

本VLAN/VSI内端口的快速离开功能处于未使能状态。

IGMP Router Alert is Not Required

路由器从本VLAN/VSI内接收的IGMP报文的IP头中不必带有Router Alert。

IGMP Send Router Alert Enable

路由器向本VLAN/VSI内发送的IGMP报文的IP报文头中带有Router Alert。
