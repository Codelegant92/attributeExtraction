display ip netstream export template

命令功能

display ip netstream export template命令用来显示当前发送的数据统计信息或模板统计信息。

命令格式

display ip netstream export template [ ip | ipv6 | mpls-ip | mpls-ipv6 | mpls-label aggregation | mpls-unknown | multicast-vpn ] slot slot-id

参数说明

ip：显示IPv4流模板输出的各种信息。

ipv6：显示IPv6流模板输出的各种信息

mpls-ip：显示MPLS和IPv4流模板输出的各种信息

mpls-ipv6：显示MPLS和IPv6流模板输出的各种信息

mpls-label aggregation：显示MPLS标签聚合流模板输的出各种信息。

mpls-unknown：显示未知MPLS标签流模板输出的各种信息

multicast-vpn：显示多播VPN流模板输出的各种信息。

slot slot-id：指定显示NetStream统计输出报文信息的接口板槽号。

使用指南

display ip netstream export template命令可结合配置老化时间、模板发送时间、选项模板使能和发送时间、IPv4的十种聚合使能、MPLS基于标签聚合使能等命令使用。

在配置了老化时间、模板发送时间、选项模板使能和发送时间、IPv4的十种聚合使能、MPLS基于标签聚合使能等命令后，
