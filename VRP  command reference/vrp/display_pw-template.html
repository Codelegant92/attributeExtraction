display pw-template

命令功能

display pw-template命令用来查看指定的或者所有的PW模板信息。

命令格式

display pw-template [ pw-template-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

pw-template-name：用于显示的PW模板名称。字符串形式，不支持空格，区分大小写，长度范围是1～19。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display pw-template命令后，如果没有指定参数pw-template-name，则显示所有PW模板的信息。

表1 display pw-template命令输出信息描述

项目

描述

PeerIP

PW远端IP地址。

PW Template Name

PW模板名称。

Tnl Policy Name

应用于PW的公网隧道策略名。

CtrlWord
表示该PW模板是否使能控制字：

  • Disable：没有使能控制字。

  • Enable：使能控制字。

MaxAtmCells

ATM的最大传输信元个数。

VCCV Capability

VC连接检测能力，VC连接检测有两种通道：控制字和Label Alert。

Behavior Name

该PW模板上配置的QoS行为策略名称。

Total PW

使用该PW模板的总PW个数，即静态和动态PW的个数总和。
