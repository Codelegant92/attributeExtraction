logging enable

命令功能

logging enable命令用来使能ASPF的日志功能。

undo logging enable命令用来关闭日志功能。

缺省情况下，关闭会话日志功能。

命令格式

logging enable

undo logging enable

参数说明

无

视图

ASPF策略视图

缺省级别

2：配置级

使用指南

ASPF的日志将会对维护或者故障处理形成一定的帮助。

使用实例

# 使能ASPF的日志功能。

<Quidway> system-view
[Quidway] aspf-policy 1
[Quidway-aspf-policy-1] logging enable
