remote-pw-id

命令功能

remote-pw-id命令用来在NQA视图下配置远端的PW ID或者VC ID。

undo remote-pw-id命令用来取消在NQA视图下配置的远端PW ID或者VC ID。

当VC Type为LDP时，remote-pw-id的缺省值是0，当VC Type为BGP时，remote-pw-id的缺省值是0xFFFF。

命令格式

remote-pw-id remote-pw-id

undo remote-pw-id

参数说明

remote-pw-id：指定的远端PW ID或VC ID。十进制整数形式。当VC Type为LDP时，取值范围是1～4294967295，当VC Type为BGP时，取值范围是0～249。

缺省级别

2：配置级

使用指南

使用remote-pw-id命令前，需要先在NQA视图下配置NQA测试例的测试类型为PWE3trace或者PWE3ping。

使用实例

# 在NQA视图下配置远端的PW ID为100。

<Quidway> system-view
[Quidway] nqa test-instance admin pwe3
[Quidway-admin-pwe3] test-type pwe3trace
[Quidway-admin-pwe3] remote-pw-id 100
