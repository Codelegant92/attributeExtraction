l2-multicast limit max-entry（VSI-LDP视图）

命令功能

l2-multicast limit max-entry命令用来配置PW通道内的组播组数量限制。对于不需要做数量限制的组播组，可以使用ACL的permit规则定义的范围区分出来。

undo l2-multicast limit max-entry命令用来取消PW通道内的组播组数量限制。

缺省情况下，不对任何PW通道内的组播组进行数量限制。

命令格式

l2-multicast limit max-entry count [ except acl-number ] remote-peer ip-address [ negotiation-vc-id vc-id ]

undo l2-multicast limit max-entry remote-peer ip-address [ negotiation-vc-id vc-id ]

参数说明

max-entry count：组播组限制的数量。整数形式，取值范围是1～8000。

except：排除不需要做数量限制的组播组。

acl-number：ACL号。整数形式，取值范围是2000～2999。

remote-peer ip-address：指定PW通道的远端IP地址。点分十进制形式。

negotiation-vc-id vc-id：PW信令为LDP时的对端协商站点的VC-ID。整数形式，取值范围是1～4294967295。

缺省级别

2 ：配置级

使用指南

此命令为覆盖式命令。

使用实例

# 将PW通道内的组播组数量限制配置为100。

<Quidway> system-view
[Quidway] vsi a1
[Quidway-vsi-a1] pwsignal ldp
[Quidway-vsi-a1-ldp] l2-multicast limit max-entry 100 except 2000 remote-peer 1.1.1.1 negotiation-vc-id 2
