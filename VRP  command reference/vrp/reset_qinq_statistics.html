reset qinq statistics

命令功能

reset qinq statistics用来清除指定子接口下用户组的统计信息。

命令格式

reset qinq statistics interface interface-type interface-number.subinterface-number vlan-group group-id

参数说明

interface interface-type interface-number.subinterface-number ：需要清除的用户组所在的子接口类型和编号。

vlan-group group-id：用户组号，十进制整数形式，取值范围是1～16。

视图

用户视图

缺省级别

3：管理级

使用指南

使用reset qinq statistics前请先确认好是否需要删除统计QinQ信息。删除后的QinQ统计信息不能被再次恢复。

使用实例

# 清除子接口GE1/0/0.1下用户组1的统计信息。

<Quidway> reset qinq statistics interface gigabitethernet 1/0/0.1 vlan-group 1
