mc-trunk mode

命令功能

mc-trunk mode命令用来配置Eth-Trunk在MC-Trunk中的工作模式。

undo mc-trunk mode命令用来恢复Eth-Trunk在MC-Trunk中的工作模式为缺省工作模式。

缺省情况下，Eth-Trunk在MC-Trunk中工作在自动模式。

命令格式

mc-trunk mode { auto | force-master | force-backup }

undo mc-trunk mode

参数说明

auto：自动模式。

force-master：强制为主用状态。

force-backup：强制为备用状态。

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

只能对已经加入MC-Trunk的Eth-Trunk执行本命令。当Eth-Trunk退出MC-Trunk时，该配置将自动清除。

在MC-Trunk正常工作期间，改变发送周期，会引起主备状态振荡。因此建议在改变发送周期前，先将成员Eth-Trunk的工作模式设置为强制主/备模式，待新的配置协商稳定后，再恢复自动状态。

使用实例

# 配置Eth-Trunk 1在MC-Trunk中的模式为强制主用。

<Quidway> system-view
[Quidway] interface Eth-Trunk 1
[Quidway-Eth-Trunk1] mc-trunk mode force-master
