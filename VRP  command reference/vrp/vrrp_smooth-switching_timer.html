vrrp smooth-switching timer

命令功能

vrrp smooth-switching time命令用来使能整机VRRP平滑倒换功能并配置平滑倒换期间VRRP报文中携带的时间间隔。

undo vrrp smooth-switching timer命令用来去使能整机VRRP平滑倒换功能。

缺省情况下使能整机VRRP平滑倒换功能，并且VRRP报文携带的默认时间间隔为100秒。

命令格式

vrrp smooth-switching timer timer-value

undo vrrp smooth-switching timer

参数说明

timer timer-value：VRRP平滑倒换报文中携带的时间间隔。Master设备在使能了VRRP平滑倒换功能后，主备倒换期间，以原来配置的时间间隔发送VRRP平滑倒换报文，报文中携带此值作为时间间隔，取值范围为1-255，单位为秒。默认值为100秒。

使用指南

使能整机VRRP平滑倒换功能后，当设备主备倒换时，设备上Master状态的VRRP以原来配置的时间间隔定时发送协议报文，报文中携带此命令配置的时间间隔。备份设备上Backup状态的VRRP将学习到该时间间隔。

通常情况下，建议该值不要太小，至少要大于设备原先的报文发送间隔，以避免在本设备主备倒换期间，由于系统太忙发不出协议报文引起备份设备上VRRP抢占的可能性。

配置整机VRRP平滑倒换功能之前必须先配置使能VRRP协议报文时间间隔学习功能。

使用实例

# 使能VRRP协议报文时间间隔学习功能。

<Quidway> system-view
[Quidway] vrrp smooth-switching timer 20
