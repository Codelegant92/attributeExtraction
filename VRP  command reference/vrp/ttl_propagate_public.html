ttl propagate public

命令功能

ttl propagate public命令用来使能MPLS公网报文的IP TTL复制功能。

undo ttl propagate public命令用来关闭MPLS公网报文的IP TTL复制功能。

缺省情况下，对公网报文使能TTL复制功能。

命令格式

ttl propagate public

undo ttl propagate public

参数说明

无

视图

MPLS视图

缺省级别

2：配置级

使用指南

配置ttl propagate public或undo ttl propagate public命令只影响此后新建立的LSP，如果需要对之前建立的LSP也生效，应执行reset mpls ldp命令重建LSP。

使用实例

# 使能公网报文的MPLS IP TTL复制功能。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] ttl propagate public
