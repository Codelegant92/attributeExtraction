display ospf request-queue

命令功能

display ospf request-queue命令用来显示OSPF的请求列表信息。

命令格式

display ospf [ process-id ] request-queue [ interface-type interface-number ] [ neighbor-id ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPF进程号。整数形式，取值范围是1～65535。

interface-type：接口类型。

interface-number：接口号。

neighbor-id：邻居ID。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令的输出信息有助于OSPF故障诊断。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示OSPF的请求列表信息。

<Quidway> display ospf request-queue
