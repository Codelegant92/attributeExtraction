quit（所有视图）

命令功能

quit命令用来从当前视图退回到较低级别视图，如果是用户视图，则退出系统。

命令格式

quit

参数说明

无

视图

所有视图

缺省级别

0：参观级

使用指南

视图分为三个级别，由低到高分别为：

  • 用户视图（用户级别为0）

  • 系统视图（用户级别为2）

  • 路由协议视图、接口视图、VPDN组视图等等。

执行quit命令即可从高级别的命令视图返回到低级别的命令视图，在用户视图下执行quit命令会退出登录。

使用实例

# 从接口视图返回到系统视图，再返回到用户视图。

[Quidway-GigabitEthernet1/0/0] quit
[Quidway] quit
<Quidway>
