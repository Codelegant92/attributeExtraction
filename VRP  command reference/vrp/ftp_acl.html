ftp acl

命令功能

ftp acl命令用来设置允许哪些客户端访问本FTP服务器，该配置通过为FTP服务器指定访问控制列表号来实现。

undo ftp acl命令用来取消可以访问本FTP服务器的访问控制列表号。

命令格式

ftp acl acl-number

undo ftp acl

参数说明

acl-number：基本访问控制列表编号，整数形式，取值范围2000～2999。

使用指南

当路由器作为FTP服务器时，通过在路由器上配置访问控制列表，来控制允许哪些客户端以FTP方式登录到本设备。

关于ACL的配置请参见“03 IP业务命令”。

使用实例

# 设置满足ACL 2000匹配条件的客户端可以登录FTP服务器。

<Quidway> system-view
[Quidway] ftp acl 2000
