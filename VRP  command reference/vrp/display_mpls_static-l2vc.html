display mpls static-l2vc

命令功能

display mpls static-l2vc命令用来查看路由器上配置的静态VC信息。

命令格式

display mpls static-l2vc [ vc-id | interface interface-type interface-number | state { down | up } ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vc-id：二层虚电路的ID。十进制整数形式，取值范围是1～4294967295。

interface interface-type interface-number：与CE设备相连的接口。interface-type表示接口类型；interface-number表示接口编号。

state：对等体之间VC的状态。

down：状态为down的VC信息。

up：状态为up的VC信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display mpls static-l2vc命令后，如果指定了接口名称则只显示指定接口（与CE相连的接口）上的静态VC信息。如果没有指定接口则显示所有接口的静态VC信息。

项目

描述

Total svc connections

所有已建立的SVC连接的条数以及状态为up的条数和状态为down的条数

Client Interface

用户接口

AC Status

接入线路（Attachment Circuit）的状态

VC State

VC的状态

VC ID

VC的标志符，唯一标志了一条VC

VC Type

VC的封装类型

Destination

VC对端设备的LSR ID值。

Transmit VC Label

本地VC标签

Receive VC Label

远程VC的标签

Control Word

是否使能控制字

VCCV Capability

是否使能VC的连通检测

Tunnel Policy Name

隧道策略的名称

Traffic Behavior

流量行为的名称

PW Template Name

PW模板的名称

Main or Secondary

该VC是主用VC还是备用VC。

Create time

该VC已经创建了多长时间

UP time

该VC保持UP状态多长时间

Last change time

该VC最后一次状态变化距现在多长时间

VC last up time :

VC最近一次UP的时刻。

VC total up time:

VC累积UP的时间（除去中间DOWN的时间）。
