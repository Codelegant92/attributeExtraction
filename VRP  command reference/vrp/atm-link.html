atm-link

命令功能

atm-link命令用来启动ATM P2P子接口链路的连接状态，PVC Down时把ATM P2P子接口的协议状态置Down。

undo atm-link命令用来取消该功能。

缺省情况下，启动P2P子接口链路的连接状态功能。

命令格式

atm-link check

undo atm-link check

参数说明

无

视图

ATM子接口视图

缺省级别

2：配置级

使用指南

只有当是ATM P2P子接口类型时，才能配置该命令。

ATM子接口分为P2P和P2MP两种类型，缺省情况下是P2MP类型。

使用实例

# 取消atm-link校验。

<Quidway> system-view
[Quidway] interface atm 1/0/0.1 p2p
[Quidway-Atm1/0/0.1] undo atm-link check
