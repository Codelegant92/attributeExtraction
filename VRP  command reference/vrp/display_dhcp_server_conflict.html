display dhcp server conflict

命令功能

display dhcp server conflict命令用来查看DHCP的地址冲突的统计信息，包括冲突IP地址、冲突检测类型、地址冲突发生的时间等。

命令格式

display dhcp server conflict { all | ip ip-address }

参数说明

all：表示查看所有冲突的IP地址统计信息。

ip ip-address：查看指定IP地址的冲突统计信息。

使用指南

当发生地址冲突时，可以使用display dhcp server conflict命令用来查看DHCP的地址冲突的统计信息，如果不指定任何可选参数，命令的显示情况与当前所处的视图有关：

  • 在以太网接口视图下，范围为当前接口的地址池。

  • 在其他视图下，范围为所有地址池。

使用实例

# 查看DHCP的地址冲突的统计信息。

<Quidway> display dhcp server conflict all
Address            Discover Time
10.110.1.2         Jan 11 2003 11:57: 7 PM
