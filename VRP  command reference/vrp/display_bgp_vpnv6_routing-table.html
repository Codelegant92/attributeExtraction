display bgp vpnv6 routing-table

命令功能

display bgp vpnv6 routing-table命令用来查看BGP VPNv6的路由信息。

命令格式

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name | route-distinguisher route-distinguisher } routing-table [ dest-ipv6-address [ prefix-length ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name | route-distinguisher route-distinguisher } routing-table as-path-filter as-path-filter-number [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name | route-distinguisher route-distinguisher } routing-table community [ community-number | aa:nn ] &<1-13> [ internet | no-advertise | no-export | no-export-subconfed ] * [ whole-match ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name | route-distinguisher route-distinguisher } routing-table community-filter { { community-filter-name | basic-community-filter-number } [ whole-match ] | advanced-community-filter-number } [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 {all | vpn6-instance vpn6-instance-name | route-distinguisher route-distinguisher } routing-table different-origin-as [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name | route-distinguisher route-distinguisher } routing-table regular-expression as-regular-expression

display bgp vpnv6 vpn6-instance vpn6-instance-name routing-table peer ipv6-address { advertised-routes | received-routes [ active ] } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：查看所有VPNv6的BGP路由统计信息。

vpn6-instance vpn6-instance-name：显示本端指定IPv6 VPN实例的BGP VPNv6路由信息。

route-distinguisher route-distinguisher：显示指定RD的BGP路由信息。

as-path-filter as-path-filter-number：指定匹配的AS路径过滤器号。

regular-expression as-regular-expression：为匹配的AS正则表达式。字符串类型，取值范围是1～80。

different-origin-as：用来显示具有相同目的地址但不同源AS号的路由。

community-filter：用来显示匹配指定的BGP团体属性过滤器的路由信息。

community-filter-name：团体属性过滤器名称。

basic-community-filter-number：指定基本团体属性过滤器号。

advanced-community-filter-number：指定高级团体属性过滤器号。

community：用来查看路由表中指定BGP团体的路由信息。

community-number：指定的团体号。

aa:nn：指定的团体号。最多可以设置13个团体号。

internet：显示带有Internet团体属性的BGP路由。

no-advertise：显示带有No-Advertise团体属性的BGP路由。

no-export：显示带有No-Export团体属性的BGP路由。

no-export-subconfed：显示带有No-Export-Subconfed团体属性的BGP路由。

whole-match：精确匹配。

peer ipv6-address：显示指定BGP对等体的路由信息。

advertised-routes：显示发布给指定对等体的路由信息。

received-routes：显示从指定对等体收到的路由信息。

active：显示从指定对等体收到的活跃路由信息。

ipv6-address：指定要显示的对等体的IPv6地址。

prefix-length：指定IPv6地址的前缀长度。

dest-ipv6-address：只显示指定的目的IPv6地址的路由信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过指定不同的参数可以只查看特定的路由信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

表2 display bgp vpnv6 route-distinguisher routing-table命令的描述信息

项目

描述

BGP Local Router ID

本地BGP路由器的ID号，其格式与IPv4地址一样

Local AS Number

本地自治系统号

Total routes of Route Distinguisher(1:1)

指定RD的BGP VPNv6路由总数

BGP routing table entry information of 2000::/100:

表示下面的信息是BGP的路由表项信息。

Label information (Received/Applied)

标签信息(接收的标签/发布的标签)

From

路由发布者的IP地址

Original nexthop

原始的下一跳

Ext-Community

BGP扩展团体属性

AS-path

AS路径属性，“Nil”表示属性值为空

Origin
该BGP路由的源(Origin)属性。它有以下3种类型：

  • IGP：比如用network命令注入到BGP路由表的路由，其Origin属性为IGP

  • EGP：通过EGP得到的路由信息，其Origin属性为EGP

  • Incomplete：表示路由的来源无法确定。比如BGP通过import-route命令引入的路由，其Origin属性为Incomplete

MED

MED(Multi-Exit-Discriminator)：判断流量进入AS时的最佳路由。在其它条件相同的情况下，将优先选择MED值较小者作为最佳路由。

Localpref

该BGP路由的本地优先级

pref-val

协议首选值

valid

该BGP路由为有效路由

internal

该BGP路由为内部路由

best

该BGP路由是最优路由

Pre 255

BGP路由优先级为255

Not advertised to any peer yet

该BGP路由尚未向任何对等体发布
