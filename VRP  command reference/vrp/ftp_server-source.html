ftp server-source

命令功能

ftp server-source命令用于设置FTP服务器的监听源为指定源，其中包括LoopBack源地址和LoopBack源接口。

命令格式

ftp server-source { -a source-ip-address | -i interface-type interface-num }

undo ftp server-source

参数说明

-a source-ip-address：本端路由器的LoopBack地址。

-i interface-type interface-num：本端路由器的LoopBack接口。

视图

系统视图

缺省级别

2：配置级

使用指南

配置该命令后，FTP服务器就只能允许指定IP或指定LoopBack接口的用户连接登录；配置后需要重启FTP服务，该命令才能生效。

如果未配置该命令，则默认所有用户都能登录。

使用实例

# 设置FTP服务器的监听源为LoopBack0。

<Quidway> ftp server-source -i LoopBack0
Warning: To make the server source configuration take effect, the FTP server will be restarted. Continue? [Y/N]: y
Info: Succeeded in setting the source interface of the FTP server to LoopBack0.
Info: Succeeded in starting the FTP server.
