display increment-command

命令功能

display increment-command命令用来查看增量配置信息。

命令格式

display increment-command

参数说明

无

视图

增量配置视图

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

当使用increment-command命令配置完增量配置信息后，可以使用display increment-command命令用来查看增量配置信息，以确认增量配置是否正确。

使用实例

# 查看增量配置信息。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] increment
[Quidway-cluster-increment] display increment-command
The content of increment commands:
------------------------------------------------------------------------------
SN        Content
------------------------------------------------------------------------------
10        vlan batch 10 to 20
20        ip route-static 2.0.0.0 8 10.0.0.1
