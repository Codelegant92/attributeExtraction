display igmp-snooping statistics

命令功能

display igmp-snooping statistics命令用来显示IGMP Snooping的统计信息。显示的内容包括各VLAN/VSI接收的IGMP报文和PIM Hello报文个数，和主动发送的查询报文的个数，以及所有VLAN/VSI内发生的二层事件次数统计。

命令格式

display igmp-snooping statistics { vlan [ vlan-id ] | vsi [ vsi-name ] }

参数说明

vlan vlan-id：显示指定VLAN的IGMP Snooping统计信息。vlan-id表示VLAN编号，取值范围是1～4094。

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

使用指南

当指定vlan-id 时，仅显示指定的vlan的统计信息，并且不显示VLAN内发生的二层事件次数统计。

当指定vsi-name时，仅显示指定的vsi的统计信息，并且不显示VSI内发生的二层事件次数统计。
