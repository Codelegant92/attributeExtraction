display ospf mpls-te

命令功能

display ospf mpls-te命令用来查看链路状态数据库LSDB中包含的所有TE LSA信息。

命令格式

display ospf [ process-id ] mpls-te [ area area-id ] [ self-originated ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPF进程号，取值范围是1～65535。

area-id：区域号，可以是十进制整数或IP地址格式，显示指定区域的TE信息。如果采用整数，取值范围是0～4294967295。

self-originated：显示自产生（self originated）的TE LSA信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

可以使用参数指定只查看某个进程或区域的TE LSA。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

表1 display ospf mpls-te命令输出信息描述

项目

描述

Area ID

使能了TE的OSPF区域的区域号

Lsa Type
LSA类型，取值可能为:

  • Opq-Link：表示第9类LSA，第9类LSA只能在某一个接口上扩散；

  • Opq-Area：表示第10类LSA，第10类LSA只能在某一个区域内扩散；

  • Opq-As：表示第11类LSA，第11类LSA与第5类LSA具有相同的扩散范围，可以在除了STUB、NSSA之外的整个自治系统内部扩散

Opaque Type

表示LSA的应用类型。例如，LSA应用于流量工程时，取值为1；LSA应用于OSPF Graceful Restart时，取值为3

Opaque Id

用来区分同一种应用类型的不同的LSA。Opaque Type与Opaque ID一起构成LSA报头中的链路状态ID

Advertising Router Id

生成这条LSA的节点

Lsa age

LSA的老化时间（在Opaque LSA头中携带）

Length

Opaque LSA的长度，包括LSA Header，以字节为单位

Lsa Options
LSA选项：

  • E：允许Flood AS-external-LSAs

  • MC：转发IP组播报文

  • N/P：处理Type-7 LSAs

  • DC：处理按需链路

LS Seq Number

LSA的顺序号，其他设备根据这个值可以判断哪个LSA是最新的

CheckSum

LSA除了LS age外其它各域的校验和

Link Type
链路类型：

  • 点到点（Point To Point）

  • 点到多点（Point to Multi Point）

  • 广播（MultiAccess）

Link ID
链路标识，IP地址格式：

  • 点到点类型：邻居的OSPF Router ID

  • 点到多点类型或广播类型：DR节点的接口IP地址

Local Interface Address

本地链路接口的IP地址

Remote Interface Address
对端接口的IP地址：

  • 点到点类型：使用对端IP地址

  • 点到多点类型或广播类型：使用0.0.0.0或省略

TE Metric

TE度量

Maximum Bandwidth

最大带宽

Maximum Reservable BW

最大可预留带宽

Admin Group

管理组

Global Pool

全局池

Unreserved BW [ 0 ] to [ 7 ]

全局池中8个级别的可用带宽

Sub Pool

子池（对于DS-TE LSA有意义）

Unreserved BW [ 8 ] to [ 15 ]

子池中8个级别的可用带宽

DS-TE Mode
DS-TE模式：
  • 标准的IETF模式
  • 非标准的IETF模式

Bandwidth Constraint Model
LSA的带宽约束模式：
  • RDM
  • MAM
  • Extended-MAM

Bandwidth Constraints

带宽约束（对于DS-TE LSA有意义）

BC [ 0] ～[ 7]

8种带宽约束（对于DS-TE LSA有意义）

Local Overbooking Multipliers

本地过预订倍数

LOM [ 0] LOM [ 1]

BC0或BC1的本地过预订倍数（Local Overbooking Multipliers）（对于DS-TE LSA有意义）

MPLS TE Router ID

Router ID TLV中携带的MPLS Router ID
