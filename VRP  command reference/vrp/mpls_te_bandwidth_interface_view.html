mpls te bandwidth（接口视图）

命令功能

mpls te bandwidth命令用来设置或修改链路上BC的带宽。

undo mpls te bandwidth命令用来恢复系统缺省配置。

缺省情况下，没有配置链路的BC带宽。

命令格式

mpls te bandwidth { bc0 bc0-bw-value | bc1 bc1-bw-value | bc2 bc2-bw-value | bc3 bc3-bw-value | bc4 bc4-bw-value | bc5 bc5-bw-value | bc6 bc6-bw-value | bc7 bc7-bw-value }*

undo mpls te bandwidth

参数说明

bc0 bc0-bw-value：设置BC0的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc1 bc1-bw-value：设置BC1的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc2 bc2-bw-value：设置BC2的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc3 bc3-bw-value：设置BC3的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc4 bc4-bw-value：设置BC4的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc5 bc5-bw-value：设置BC5的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc6 bc6-bw-value：设置BC6的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

bc7 bc7-bw-value：设置BC7的带宽。整数形式，取值范围是1～32000000，单位是Kbit/s。缺省值是1。

使用指南

在实际配置应用中，只要在有带宽要求的TE隧道所经链路接口的出方向配置即可。

一条命令可同时配置1个或多个BC的带宽值。配置是无序的，但不能重复配置同一BC带宽。

说明：
  • 非标准（Non-IETF）模式的DS-TE只支持BC0和BC1；标准（IETF）模式的DS-TE支持BC0～BC7。
  • 当Non-IETF模式向IETF模式切换时，BC0和BC1的带宽值保持不变；当IETF模式向Non-IETF模式切换时，除BC0和BC1保持不变外，其它BC的值被清零。
各BC的带宽最大值与链路的可预留带宽（max-reservable-bandwidth）之间的关系如下：
  • RDM模型：bc7-bw-value ≤ bc6-bw-value ≤ bc5-bw-value ≤ bc4-bw-value ≤ bc3-bw-value ≤ bc2-bw-value ≤ bc1-bw-value ≤ bc0-bw-value ≤ max-reservable-bandwidth
  • MAM模型：bc0-bw-value＋bc1-bw-value＋bc2-bw-value＋bc3-bw-value＋bc4-bw-value＋bc5-bw-value＋bc6-bw-value＋bc7-bw-value ≤ max-reservable-bandwidth
  • Extend-MAM模型：同MAM模型

如果要修改BC带宽值，可重新配置该命令，最后一次的配置会覆盖之前的配置。

不允许将BC带宽值修改成小于已经分配给CT的带宽值。例如，BC0已有10Mbit/s的带宽被分配，则BC0的带宽只能修改成大于等于10Mbit/s。

当在系统视图下配置undo mpls命令或在MPLS视图下配置undo mpls te命令后，BC带宽值会被删除。

使用实例

# 在IETF模式下，设置BC模型为MAM，链路最大可预留带宽为10000Kbit/s，链路BC0、BC2和BC3带宽值分别为1000Kbit/s、2000Kbit/s、3000Kbit/s。

<Quidway> system-view
[Quidway] mpls lsr-id 1.1.1.1
[Quidway] mpls
[Quidway-mpls] mpls te
[Quidway-mpls] mpls te ds-te mode ietf
[Quidway-mpls] mpls te ds-te bcm mam
[Quidway-mpls] quit
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mpls
[Quidway-GigabitEthernet1/0/0] mpls te
[Quidway-GigabitEthernet1/0/0] mpls te bandwidth max-reservable-bandwidth 10000
[Quidway-GigabitEthernet1/0/0] mpls te bandwidth bc0 1000 bc3 3000 bc2 2000
