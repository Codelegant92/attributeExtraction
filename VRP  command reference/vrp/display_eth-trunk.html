display eth-trunk

命令功能

display eth-trunk命令用来查看Eth-Trunk接口的配置信息。

命令格式

display eth-trunk [ trunk-id [ interface interface-type interface-number | verbose ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

trunk-id：端口聚合的Trunk ID。取值范围是0～63。

interface interface-type interface-number：Eth-Trunk中的成员接口。

verbose：显示Eth-Trunk接口的详细配置信息，包括接口流量统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

说明：

verbose关键字仅在NE40E、NE80E和NE5000E上支持。

表1 display eth-trunk命令输出信息描述

项目

描述

Local

本端Eth-Trunk接口的配置信息。

LAG ID

Eth-Trunk接口的ID。

WorkingMode

Eth-Trunk接口的工作模式。

  • NORMAL：表示手工负载分担模式。

  • STATIC：表示静态LACP模式。

Preempt Delay

优先级抢占延时。

Hash arithmetic
散列算法的依据。
  • According to flow：Eth-Trunk接口根据流进行散列。
  • According to packet all：Eth-Trunk接口根据所有包进行散列。

System Priority

系统的LACP优先级。

System ID

系统ID。

Least active-linknumber

处于Up状态的成员链路的下限阈值。

Max Active-linknumber

处于Up状态的成员链路的上限阈值。

Max bandwidth-affected-linknumber

影响二层Eth-Trunk口带宽的成员口的最大数目。

Operate status
Eth-Trunk接口的状态。

  • UP：接口处于正常启动的状态。

  • DOWN：接口在物理上出现故障。

  • Administratively down：如果网络管理员在该接口执行shutdown命令，将显示状态为Administratively down。

Number Of Up Ports in Trunk

Eth-Trunk接口中处于Up状态的成员接口数。

ActorPortName

成员接口名。

Status
本地成员接口的状态，有以下两种：

  • Selected

  • Unselected

PortType

本地成员接口的类型。1GE：表示GE接口。100M：表示Ethernet接口。

PortPri

成员接口的LACP优先级。

PortNo

成员接口在LACP中的编号。

PortKey

成员接口在LACP中的Key值。

PortState

成员接口LACP状态的state字节信息。

Weight

成员接口的权重。

Partner

对端Eth-Trunk成员接口的信息。

Flow statistic

接口的流量统计信息。

Last 300 seconds input rate Last 300 seconds output rate

显示了Eth-Trunk接口或成员接口在前300秒内发送和接收的字节速率和报文速率。

Input/Output

接口收到或发送报文的方向。

packets

接口接收或发送的总报文数。

bytes

接口接收或发送的总字节数。

drops

接口丢弃的报文数。
