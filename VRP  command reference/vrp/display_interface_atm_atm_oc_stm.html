display interface atm（ATM OC/STM）

命令功能

display interface atm命令用来查看ATM接口的配置及状态信息。

命令格式

display interface atm [ interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-number：ATM接口编号。如果不指定接口编号，将显示所有ATM接口的配置及状态信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。字符串格式，支持空格，区分大小写。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

表1 display interface atm命令输出信息描述

项目

描述

Atm2/0/0 current state :

显示接口Atm2/0/0的物理状态：

  • UP：接口处于正常启动的状态。

  • DOWN：接口在物理上出现故障。

  • Administratively down：如果网络管理员在该接口执行shutdown命令，将显示状态为Administratively down。

Line protocol current state :
显示该接口的链路协议状态：

  • UP：接口的链路协议处于正常的启动状态。

  • DOWN：接口的链路协议不正常，或者没有在此接口配置IP地址。

Last up time:

接口的协议状态最近一次变为Up的时间。

The Maximum Transmit Unit is

接口的最大传输单元（MTU），缺省值是1500字节。长度大于MTU的报文，将会被分片后再发送。如果设置了不准分片，会被丢弃。

Internet Address is

接口的IP地址和子网掩码。

AAL enabled:

ATM适配层。

Maximum VCs:

最大的VC数目。

VCs on main-interface: (Total VCs: )

当前主接口上的VC数目（当前主接口和子接口上的VC总数）。

VPs on main-interface: (Total VPs: )

当前主接口上的VP数目（当前主接口和子接口上的VP总数）。

Physical layer is

物理介质类型。例如，如果为E1链路，则会显示为ATM over E1。

Scramble enabled

使能加扰功能

frame-format

帧格式。可以使用frame-format（ATM OC/STM）命令改变帧格式。

clock

时钟模式。可以使用clock（ATM接口视图）命令改变时钟模式。

loopback not set

环回状态。

SDH alarm:

SDH告警。

section layer:

section告警。

line layer:

line告警。

path layer:

path通路。

C2(Rx):

C2标志。

SDH error:

SDH错误。

F5 end-end loopback oam sends

F5环回OAM信元发送数目。

F5 end-end loopback oam receives

F5环回OAM信元接收数目。

my oam back 0, oam forward 5248

本端发出的OAM信元被对端转发回来的数量，转发对端发送过来的OAM信元的数量。

oam send drops 0, oam forward drops 0

发送OAM信元丢弃的数量，转发的OAM信元丢弃的数量。

Statistics last cleared:

上次清除本接口统计信息的时间。

Last 300 seconds input Last 300 seconds output

这一字段显示了在前300秒内通过接口发送和接收的字节速率和报文速率。
