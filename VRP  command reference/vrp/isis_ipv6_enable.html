isis ipv6 enable

命令功能

isis ipv6 enable命令用来使能接口IS-IS的IPv6能力并指定要关联的IS-IS进程号，命令成功执行后，接口就会和这个IS-IS进程关联起来。

undo isis ipv6 enable命令用来取消该配置。

缺省情况下，接口上不使能接口IS-IS的IPv6能力。

命令格式

isis ipv6 enable [ process-id ]

undo isis ipv6 enable

参数说明

process-id：IS-IS进程号。

使用指南

为了使IS-IS协议的IPv6功能正常运行，必须先执行以下操作：

  1. 在系统视图下使用ipv6命令使能全局IPv6。

  2. 使用isis命令使能IS-IS进程，并使用network-entity命令为IS-IS设置一个网络实体名称（NET）。

  3. 在IS-IS视图下使用ipv6 enable命令使能IS-IS进程的IPv6能力。

  4. 在各个需要运行IS-IS进程的接口上使用ipv6 enable命令使能接口的IPv6能力，配置IPv6地址。

  5. 使用isis ipv6 enable命令在各接口上使能IS-IS。
说明：

所有与IPv6相关的配置命令都必须在使能了IPv6功能的情况下使用。

使用实例

# 创建IS-IS路由进程1，使能IPv6能力，并在接口POS1/0/0上激活这个进程的IPv6能力。

<Quidway> system-view
[Quidway] ipv6
[Quidway] isis 1
[Quidway-isis-1] network-entity 10.0001.1010.1020.1030.00
[Quidway-isis-1] ipv6 enable
[Quidway-isis-1] quit
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ipv6 enable
[Quidway-Pos1/0/0] ipv6 address 2002::1/64
[Quidway-Pos1/0/0] isis ipv6 enable 1
