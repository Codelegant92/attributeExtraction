efm packet max-size

命令功能

efm packet max-size命令用来配置当前接口的EFM OAM协议报文的最大尺寸。在当前接口上，超过这个尺寸的EFM OAM协议报文被作为非法报文丢弃。

undo efm packet max-size命令用来恢复当前接口的EFM OAM协议报文的最大尺寸为缺省值。

命令格式

efm packet max-size size

undo efm packet max-size

参数说明

size：指定EFM OAM协议报文的最大尺寸。整数形式，取值范围是64～1518，单位是字节。缺省值是128字节。此尺寸包括二层帧头和CRC校验字段的长度。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

使用efm enable命令全局使能EFM OAM功能后才能使用本命令。

接口的EFM OAM协议报文的最大尺寸的实际值是通过和对端设备协商确定的。协商的原则是取配置值较小的一端的配置值。

使用实例

# 配置接口GigabitEthernet1/0/0的EFM OAM协议报文的最大尺寸为120字节。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] efm packet max-size 120
