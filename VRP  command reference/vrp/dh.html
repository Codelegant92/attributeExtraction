dh

命令功能

dh命令用来配置IKE阶段1密钥协商时所使用的DH密钥交换参数。

undo dh命令用来设置Diffie-Hellman组标识为缺省值。

缺省为group1，即768-bit的Diffie-Hellman组。

命令格式

dh { group1 | group2 }

undo dh

参数说明

group1：指定阶段1密钥协商时采用768-bit的Diffie-Hellman组。

group2：指定阶段1密钥协商时采用1024-bit的Diffie-Hellman组。

视图

IKE提议视图

缺省级别

2：配置级

使用指南

在安全隧道的两端设置的Diffie-Hellman组必须是一样的，否则协商不能通过。

使用实例

# 指定IKE提议10使用768-bit的Diffie-Hellman组。

<Quidway> system-view
[Quidway] ike proposal 10
[Quidway-ike-proposal-10] dh group1
