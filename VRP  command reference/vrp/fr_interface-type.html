fr interface-type

命令功能

fr interface-type命令用来设置帧中继接口类型。

undo fr interface-type命令用来恢复缺省的帧中继接口类型。

缺省情况下，帧中继接口类型为DTE。

命令格式

fr interface-type { dce | dte | nni }

undo fr interface-type

参数说明

dte、dce和nni：帧中继接口的三种类型。

视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

在帧中继中，通信的双方分为用户侧和网络侧。用户侧称为DTE，而网络侧称为DCE。在帧中继网络中，帧中继交换机之间为NNI接口，相应接口采用NNI工作方式。如果把设备用做帧中继交换，帧中继接口应该工作在NNI方式或DCE方式。

使用实例

# 设置帧中继接口Serial1/0/0:0类型为DCE。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr
[Quidway-Serial1/0/0:0] fr interface-type dce
