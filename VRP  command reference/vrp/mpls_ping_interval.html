mpls ping interval

命令功能

mpls ping interval命令用来配置动态BFD中LSP ping定时器的时间间隔。

undo mpls ping interval命令用来恢复动态BFD中LSP ping定时器的时间间隔为缺省值。

缺省情况下，动态BFD中LSP ping定时器的时间间隔是60秒。

命令格式

mpls ping interval interval

undo mpls ping interval

参数说明

interval：指定动态BFD中LSP ping定时器的时间间隔。单位是秒。取值范围是30～600，缺省值是60秒。
