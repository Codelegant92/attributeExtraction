mpls te igp metric

命令功能

mpls te igp metric命令用来设置在IGP Shortcut和转发邻接特性的SPF计算中，MPLS TE隧道的度量值。

undo mpls te igp metric命令用来恢复缺省设置。

缺省情况下，TE隧道的度量等于其IGP度量。

命令格式

mpls te igp metric { absolute | relative } value

undo mpls te igp metric

参数说明

absolute value：绝对度量模式。value取值为正整数，取值范围是1～65535。

relative value：相对度量模式。value可以是正整数、负整数或0，取值范围是-10～10，表示相对于隧道IGP度量的差异。缺省值为0。

使用实例

# 进行SPF计算时，MPLS TE隧道的度量值为IGP的度量值减1。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol mpls te
[Quidway-Tunnel1/0/0] mpls te igp metric relative -1
[Quidway-Tunnel1/0/0] mpls te commit
