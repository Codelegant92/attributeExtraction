if-match 8021p

命令功能

if-match 8021p命令用来定义VLAN帧优先级匹配规则。

undo if-match 8021p命令用来删除VLAN帧优先级匹配规则。

命令格式

if-match [ not ] 8021p 8021.p-value &<1-8>

undo if-match [ not ] 8021p

参数说明

8021p-value：VLAN帧优先级的值，取值范围是0～7。

not：不匹配该类。

缺省级别

2：配置级

使用指南

命令在配置后，8021p优先级的值将自动按照从小到大的顺序排序。

可以配置多个8021p优先级值，最多可指定8个；如果指定了多个相同的IP优先级值，系统默认为一个；多个不同的IP优先级值是或的关系。

配置时将需要配置的8021p优先级值用同一条命令进行配置，否则最后配置的if-match 8021p命令会覆盖前面的配置。

VLAN帧优先级匹配规则只对入方向的策略有意义，并且只对以太类型的接口有意义。

使用实例

# 定义类class1的匹配规则为：匹配8021p优先级值为1或4的VLAN帧。

<Quidway> system-view
[Quidway] traffic classifier class1
[Quidway-classifier-class1] if-match 8021p 1 4
