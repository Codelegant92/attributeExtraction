tunnel timer hello

命令功能

tunnel timer hello命令用来设置隧道中Hello报文发送时间间隔。

undo tunnel timer hello命令用来恢复缺省Hello报文发送时间间隔。

缺省情况下，Hello报文每隔60秒发送一次。

命令格式

tunnel timer hello interval

undo tunnel timer hello

参数说明

interval：LAC或LNS在没有报文接收时发送Hello报文的时间间隔，整数形式，取值范围是60～1000，单位为秒。

缺省级别

2：配置级

使用指南

L2TP使用Hello报文检测隧道的连通性。LAC和LNS定时向对端发送Hello报文，若在一段时间内未收到Hello报文的应答，则重复发送Hello报文。如果重复发送报文的次数超过3次，则认为L2TP隧道已经断开，该PPP会话将被清除。

如果当前的Hello报文发送时间间隔不合适，可以使用tunnel timer hello命令来改变。LNS和LAC侧可以配置不同的Hello报文时间间隔。

使用实例

# 设置发送Hello报文的时间间隔为99秒。

<Quidway> system-view
[Quidway] l2tp-group 1
[Quidway-l2tp-1] tunnel timer hello 99
