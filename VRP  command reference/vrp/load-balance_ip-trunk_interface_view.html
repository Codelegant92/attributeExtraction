load-balance（IP-Trunk接口视图）

命令功能

load-balance命令用来配置负载分担的散列算法。

undo load-balance命令用来恢复缺省配置。

命令格式

load-balance { ip | mac | packet-all }

undo load-balance

参数说明

ip：根据源IP地址和目的IP地址进行散列。

mac：根据MAC地址进行散列。

packet-all：根据所有包进行散列。

视图

IP-Trunk接口视图

缺省级别

2：配置级

使用指南

缺省情况下，IP-Trunk接口根据ip进行散列。

使用实例

# 配置IP-Trunk 1根据IP地址进行散列。

<Quidway> system-view
[Quidway] interface ip-trunk 1
[Quidway-Ip-Trunk1] load-balance ip
