reset ipv6 fibcache

命令功能

reset ipv6 fibcache命令用来清除FIB缓存项。

命令格式

reset ipv6 fibcache { slot-id | all } [ vpn6-instance vpn6-instance-name ]

参数说明

slot-id：清除指定槽位的接口板上的FIB缓存项。仅在分布式环境中有效。

all：清除所有接口板上的FIB缓存项。仅在分布式环境中有效。

vpn6-instance vpn6-instance-name：清除指定IPv6 VPN实例的FIB缓存项。字符串形式，区分大小写，长度范围是1～31。

<Quidway> reset ipv6 fibcache all
