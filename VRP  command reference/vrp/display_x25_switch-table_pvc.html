display x25 switch-table pvc

命令功能

display x25 switch-table pvc命令用来显示X.25交换虚电路表。

命令格式

display x25 switch-table pvc

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取PVC的交换路由表。用户可以根据这些信息进行接口的故障诊断等。

使用实例

# 显示X.25交换虚电路表。

<Quidway> display x25 switch-table pvc
#1     (In: Serial1/0/0  - PVC1024   ) <--> (Out: Serial2/0/0  - PVC1   )
#2     (In: Serial2/0/0  - PVC1024   ) <--> (Out: Serial1/0/0  - PVC1   )
表1 display x25 switch-table pvc命令输出信息描述

项目

描述

#1

交换虚电路表的序号。

In:

接受报文的接口类型、编号和虚电路号。

Out:

转发报文的接口类型、编号和虚电路号。
