e3 framed（CE3接口视图）

命令功能

e3 framed（CE3接口视图）命令用来设置CE3接口的通道工作在非通道化模式。

undo e3 framed（CE3接口视图）命令用来恢复缺省配置。

缺省情况下，CE3接口通道工作在通道化模式。

命令格式

e3 framed

undo e3 framed

参数说明

无

视图

CE3接口视图

缺省级别

2：配置级

使用指南

当在CE3接口上执行命令e3 framed后，CE3接口工作在非通道化模式下。此时除0时隙外，CE3接口的所有时隙形成一个速率为32.768Mbit/s的同步串口。

当在CE3接口上执行命令using e3后，CE3接口工作在净通道化模式下。此时，CE3接口的所有时隙形成一个速率为34.368Mbit/s的同步串口。

当在CE3接口上执行命令using ce3后，CE3接口工作在通道化模式下。此时需要执行命令e1 channel-set（CE3接口视图）或e1 unframed（CE3接口视图），配置CE3接口下的E1通道，CE3接口才能形成同步串口。

使用实例

# 将CE3 1/0/0接口配置为非通道化模式。

<Quidway> system-view
[Quidway] controller e3 1/0/0
[Quidway-E3 1/0/0] e3 framed
