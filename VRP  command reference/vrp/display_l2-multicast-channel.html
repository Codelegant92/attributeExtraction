display l2-multicast-channel

命令功能

display l2-multicast-channel命令用来显示节目组信息。

命令格式

display l2-multicast-channel [vlan [vlan-id [ to vlan-id ] ] | vsi [vsi-name]| global] [channel {channel-name}]

参数说明

vlan vlan-id：VLAN号。整数形式，取值范围是1～4094。

vsi vsi-name：VSI名称。字符串形式，大小写敏感，取值范围是1～31。

global：全局配置。

channel channel-name：节目组名。字符串形式，大小写敏感，取值范围是1～31。
