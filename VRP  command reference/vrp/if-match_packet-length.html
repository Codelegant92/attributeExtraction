if-match packet-length

命令功能

if-match packet-length命令用来设置IP报文长度匹配条件。

undo if-match packet-length命令用来删除IP报文长度匹配条件的设置。

命令格式

if-match packet-length min-length max-length

undo if-match packet-length

参数说明

min-length：指定策略路由要匹配的最短IP报文长度。整数形式，单位是字节，取值范围是0～65535。

max-length：指定策略路由要匹配的最长IP报文长度。整数形式，单位是字节，取值范围是1～65535。
