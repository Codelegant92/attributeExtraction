mpls lsr-id

命令功能

mpls lsr-id命令用来配置LSR的ID。

undo mpls lsr-id命令用来删除LSR的ID。

缺省情况下，没有配置LSR ID。

命令格式

mpls lsr-id lsr-id

undo mpls lsr-id

参数说明

lsr-id：设备的LSR ID，点分十进制格式，用于标识一个LSR。

视图

系统视图

缺省级别

2：配置级

使用指南

配置LSR ID是配置其它MPLS命令的前提。LSR没有缺省的LSR ID，必须手工配置。必须使用LSR某个Loopback接口的地址做为LSR ID。

如果要修改已经配置的LSR ID，必须在系统视图下执行undo mpls命令删除所有MPLS配置。

使用实例

# 配置LSR的ID为1.1.1.1。

<Quidway> system-view
[Quidway] mpls lsr-id 1.1.1.1
