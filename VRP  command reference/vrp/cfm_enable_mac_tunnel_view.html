cfm enable（MAC Tunnel视图）

命令功能

cfm enable命令用来使能当前MAC tunnel的CFM功能。

undo cfm enable命令用来禁止当前MAC tunnel的CFM功能。

缺省情况下，MAC tunnel的CFM功能处于禁止状态。

命令格式

cfm [ ccm-interval interval ] enable

undo cfm enable

参数说明

interval：表示发送或探测CCM消息的时间间隔。字符串形式，不支持空格，取值包括：10、20、30、50、100、1000、10000，单位是ms。缺省值是10ms。

视图

MAC Tunnel视图

缺省级别

2：配置级

使用指南

当MAC tunnel的CFM功能使能后，设备开始通过该MAC tunnel发送CCM消息。

当MAC tunnel的CFM功能处于禁止状态时，设备不向该MAC tunnel对端的设备发送CCM消息，并且也不接收该MAC tunnel对端发来的CCM消息。

当需要对MAC tunnel进行连通性故障检测时，可使用本命令。

在系统视图下使用cfm enable（系统视图）命令全局使能设备的CFM功能后才能使用本命令。

MAC tunnel的目的虚拟MAC地址、骨干VLAN、接口三个属性都已经配置完整后，执行本命令才能成功。

如果当前MAC tunnel已有业务实例配置了和以太网CFM或EFM OAM联动，则不能禁止当前MAC tunnel的CFM功能。如需禁止，首先需使用undo cfm bind命令或undo efm bind service-instance命令解除该MAC tunnel的业务实例和CFM或EFM的联动关系。

使用实例

# 使能MAC tunnel MacCustomer的CFM功能。

<Quidway> system-view
[Quidway] cfm enable
[Quidway] mac-tunnel tunnel-name MacCustomer
[Quidway-mac-tunnel-MacCustomer] cfm enable
