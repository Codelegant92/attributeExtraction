display bgp routing-table flap-info

命令功能

display bgp routing-table flap-info命令用来显示BGP的振荡统计信息。

命令格式

display bgp [ vpnv4 vpn-instance vpn-instance-name ] routing-table flap-info [ regular-expression as-regular-expression ]

display bgp [ vpnv4 vpn-instance vpn-instance-name ] routing-table flap-info [ as-path-filter as-path-filter-number | network-address [ { mask | mask-length } [ longer-match ] ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp ipv6 routing-table flap-info [ regular-expression as-regular-expression ]

display bgp ipv6 routing-table flap-info [ as-path-filter as-path-filter-number | network-address [ prefix-length [ longer-match ] ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 vpn6-instance vpn6-instance-name routing-table flap-info [ regular-expression as-regular-expression ]

display bgp vpnv6 vpn6-instance vpn6-instance-name routing-table flap-info [ as-path-filter as-path-filter-number | network-address [ prefix-length [ longer-match ] ] ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpnv4 vpn-instance vpn-instance-name：显示指定VPN实例的振荡统计信息。

regular-expression as-regular-expression：显示匹配AS路径正则表达式的路由振荡统计信息。字符串类型，取值范围是1～80。

as-path-filter as-path-filter-number：指定匹配的AS路径过滤器号。

network-address：指定显示衰减信息的网络地址。

mask | mask-length：网络掩码/掩码长度。

longer-match：允许按更长的前缀长度匹配。

prefix-length：前缀长度。

vpnv6：查看VPNv6的BGP路由的振荡统计信息。

vpn6-instance vpn6-instance-name：显示指定IPv6 VPN实例BGP路由的振荡统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
