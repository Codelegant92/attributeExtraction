display snmp-agent mib-view

命令功能

display snmp-agent mib-view命令用来显示当前配置的MIB视图。

命令格式

display snmp-agent mib-view [ exclude | include | viewname view-name ]

参数说明

exclude：指定要排除显示且已设定的SNMP MIB视图属性。

include：指定要包含显示且已设定的SNMP MIB视图属性。

viewname view-name：指定要显示的视图名称，字符串形式，长度范围是1～32。

视图

所有视图

缺省级别

1：监控级

使用指南

使用display snmp-agent mib-view命令查看配置只读团体名时指定的视图，默认情况系统使用默认视图ViewDefault。

使用实例

# 显示当前配置的MIB视图。

<Quidway> display snmp-agent mib-view
   View name:ViewDefault
       MIB Subtree:internet
       Subtree mask:
       Storage-type: nonVolatile
       View Type:included
       View status:active
