tunnel ipv4-ipv6 traffic-class

命令功能

tunnel ipv4-ipv6 traffic-class命令用来设置IPv4-over-IPv6隧道的流量级别。

undo tunnel ipv4-ipv6 traffic-class命令用来使流量级别恢复为缺省值。

缺省情况下，IPv4-over-IPv6隧道的流量级别为0。

命令格式

tunnel ipv4-ipv6 traffic-class { original | class-value }

undo tunnel ipv4-ipv6 traffic-class

参数说明

original：表示IPv6隧道首部的IPv6字段的Traffic Class值拷贝原始报文中Traffic Class字段值。目前该参数不可用。

class-value：IPv4-over-IPv6隧道的流量级别值。整数形式，取值范围是0～255。

缺省级别

2：配置级

使用指南

IPv6隧道报文的流量级别是指隧道入口的隧道报文首部中的Traffic Class字段值。

在隧道接口视图下配置该命令则仅对当前隧道接口生效。目前，在系统视图下虽然可以配置该命令，但是配置并不生效。

使用实例

# 设置IPv4 over IPv6隧道的流量级别为10。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel ipv4-ipv6 traffic-class 10

# 设置本设备所有的IPv4 over IPv6隧道的流量级别都为10。

<Quidway> system-view
[Quidway] tunnel ipv4-ipv6 traffic-class 10
