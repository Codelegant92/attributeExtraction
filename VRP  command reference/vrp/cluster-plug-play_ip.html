cluster-plug-play ip

命令功能

cluster-plug-play ip命令用来配置集群默认的FTP服务器信息，包括IP地址，用户名称和密码。

undo cluster-plug-play ip命令用来删除集群默认的FTP服务器信息，包括IP地址，用户名称和密码。

缺省情况下，命令交换机上没有配置集群默认的FTP服务器信息。

命令格式

cluster-plug-play ip ftp-ip-address username username password password [ path-separator pathseparator ]

undo cluster-plug-play ip

参数说明

ip ftp-ip-address：FTP服务器的IPv4地址，点分十进制形式。

username user-name：FTP服务器的用户名称。字符串形式，区分大小写，不支持空格，长度范围是1～20。

password password：FTP服务器的用户密码。字符串形式，区分大小写，不能输入空格和问号等命令行专用字符，长度范围是1～8。

path-separator pathseparator：FTP Server路径分隔符，字符串形式，长度范围是1～4。例如：Windows系统中使用“/”，UNIX系统使用“\”。默认分隔符是“/”。

使用指南

本命令只能在命令交换机上执行。

用户密码以不加密的形式保存。

使用实例

# 在命令交换机上配置集群默认的FTP服务器地址是100.1.1.1，用户名是user1，密码是123。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] cluster-plug-play ip 100.1.1.1 username user1 password 123
