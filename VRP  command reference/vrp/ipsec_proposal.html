ipsec proposal

命令功能

ipsec proposal proposal-name命令用来创建或修改一个名字为proposal-name的IPSec提议，并进入IPSec提议视图。

undo ipsec proposal proposal-name命令用来删除一个名字为proposal-name的IPsec提议。

缺省情况下，系统没有任何提议。

命令格式

ipsec proposal proposal-name

undo ipsec proposal proposal-name

参数说明

proposal-name：指定IPSec提议的名字。命名规则为：长度范围是1～15个字符，不区分大小写。

缺省级别

2：配置级

使用指南

在配置安全策略时，需要通过引用安全提议来确定在安全隧道的两端所采用的安全协议、加密和认证算法和报文封装形式。

在使用ipsec proposal命令创建一个新的IPSec安全提议后，其缺省参数为采用ESP协议、DES加密算法、MD5认证算法、隧道模式。

使用实例

# 创建名为newprop1的提议。

<Quidway> system-view
[Quidway] ipsec proposal newprop1
