reset patch-configure

命令功能

reset patch-configure命令用来清除补丁状态和下一次启动时的补丁设置。

缺省情况下为清除当前的补丁状态。

命令格式

reset patch-configure [ next-startup ]

参数说明

next-startup：清除下一次启动后的补丁状态和补丁设置。

视图

用户视图

缺省级别

2：配置级

使用指南

若当前某单板上加载了补丁，则无法清除当前的补丁状态。

若单板上没有生效的补丁，当前的补丁设置也可以被清除。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 清除当前的补丁状态。

<Quidway> reset patch-configure
Info: Succeded in resetting the patch state

# 清除下一次启动后的补丁状态和补丁设置。

<Quidway> reset patch-configure next-startup
Info： Succeeded in resetting the next-startup patch state
