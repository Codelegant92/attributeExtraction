description（MAC Tunnel视图）

命令功能

description命令用来配置MAC Tunnel的描述信息。

undo description命令用来删除MAC Tunnel的描述信息。

缺省情况下，MAC Tunnel的描述信息为空。

命令格式

description mac-tunnel-description

undo description

参数说明

mac-tunnel-description：MAC Tunnel的描述信息。长度范围是1～80个字符，首位不能是空格。

视图

MAC Tunnel视图

缺省级别

2：配置级

使用指南

MAC Tunnel的描述信息可以方便识别和使用隧道。

如果已经配置了MAC Tunnel的描述信息，再次执行该命令后，原来的描述信息将被覆盖。

使用实例

# 配置mytunnel隧道的描述信息为“This is my tunnel for testing”。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] description This is my tunnel for testing
