reset atm

命令功能

reset atm命令用来清除ATM接口的统计信息。

命令格式

reset atm interface [ atm interface-number ]

参数说明

interface-number：ATM接口编号。

视图

用户视图

缺省级别

2：配置级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除所有ATM接口的统计信息。

<Quidway> reset atm interface
