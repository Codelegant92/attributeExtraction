enable log

命令功能

enable log命令用来使能日志信息。

undo enable log命令用来关闭日志信息。

命令格式

enable log [ config | error | state | snmp-trap ]

undo enable log [ config | error | state | snmp-trap ]

参数说明

config：配置日志信息。

error：错误日志信息。

state：状态日志信息。

snmp-trap：SNMP告警日志信息。

视图

OSPF视图

缺省级别

2：配置级

使用指南

无

使用实例

# 使能OSPF日志信息。

<Quidway> system-view
[Quidway] ospf 1
[Quidway-ospf-1] enable log
