peer filter-policy（MBGP）

命令功能

peer filter-policy命令用来为指定对等体或对等体组配置基于ACL的MBGP路由过滤策略。

undo peer filter-policy命令用来取消已有的设置。

缺省情况下，对等体或对等体组无指定的过滤策略。

命令格式

peer { group-name | peer-address } filter-policy basic-acl-number { import | export }

undo peer { group-name | peer-address } filter-policy basic-acl-number { import | export }

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

basic-acl-number：基本ACL编号。

import：对收到的路由应用路由策略。

export：对发布的路由应用路由策略。

使用实例

# 对来自对等体组test的路由应用ACL2003的路由过滤策略。

[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] peer test enable
[Quidway-bgp-af-multicast] peer test filter-policy 2003 import
