display admin-vsi binding

命令功能

display admin-vsi binding命令用来显示管理VSI和业务VSI的绑定关系。

命令格式

display admin-vsi binding [ admin-vsi vsi-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

admin-vsi vsi-name：要显示的管理VSI实例名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display admin-vsi binding命令时，如果不指定管理VSI的名称，则显示所有管理VSI和业务VSI的绑定信息。

使用实例

# 显示指定的管理VSI(v100)和业务VSI的绑定关系。

<Quidway> display admin-vsi binding admin-vsi v100
  Administrator-VSI         VSI
  V100                      V2
  V100                      V3

# 显示所有管理VSI和业务VSI的绑定关系。

<Quidway> display vsi admin-vsi
  Administrator-VSI         VSI
  V100                      V2
  V100                      V3
  V4                        V5
  V4                        V6
