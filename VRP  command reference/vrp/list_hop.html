list hop

命令功能

list hop命令用来查看MPLS TE显式路径的节点信息。

命令格式

list hop [ ip-address ]

参数说明

ip-address：指定IP地址。

显式路径视图

缺省级别

2：配置级

使用指南

无

# 查看MPLS TE显式路径的节点信息。

<Quidway> system-view
[Quidway] explicit-path path1
[Quidway-explicit-path-path1] list hop
