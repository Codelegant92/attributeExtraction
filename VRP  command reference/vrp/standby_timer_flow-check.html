standby timer flow-check

命令功能

standby timer flow-check命令用来配置检测主接口流量的时间间隔。

undo standby timer flow-check命令用来恢复流量检测时间间隔的缺省值。

命令格式

standby timer flow-check time

undo standby timer flow-check

参数说明

time：流量检测时间间隔。单位是秒，取值范围是30～600。缺省值为30秒。

视图

接口视图

缺省级别

2：配置级

使用指南

当主接口参与备份负载分担时，备份中心将自动检测主接口的流量，检测的时间间隔可通过该命令来设置。

使用实例

# 先指定接口Pos1/0/0作为接口Pos2/0/0的备份接口。并在接口Pos2/0/0上配置主接口备份带宽

<uidway> system-view
[Quidway] interface pos 2/0/0
[Quidway-Pos2/0/0] standby interface pos1/0/0 50
[Quidway-Pos2/0/0] standby bandwidth 10000
[Quidway-Pos2/0/0] standby threshold 80 50
[Quidway-Pos2/0/0] standby timer flow-check 60
