l2tp sendaccm enable

命令功能

l2tp sendaccm enable命令用来使能L2TP发送ACCM信息的功能。

undo l2tp sendaccm enable命令用来去使能此功能。

缺省情况下，使能L2TP发送ACCM信息的功能。

命令格式

l2tp sendaccm enable

undo l2tp sendaccm enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

RFC2661中提到，LNS可以通过ACCM消息，将LNS与PPP对端协商的ACCM情况通知LAC。

实际应用中，各设备制造商的LAC对ACCM要求可能不同，LNS需要根据LAC的要求配置是否发送ACCM消息。如果LAC端不支持ACCM消息，则需要在LNS端执行undo l2tp sendaccm enable命令。

使用实例

# 禁止L2TP发送ACCM消息。

<Quidway> system-view
[Quidway] undo l2tp sendaccm enable
