reset local-ce mac

命令功能

reset local-ce mac命令用来清除以太接口动态学到的本地CE的MAC地址及VLAN ID信息。

命令格式

reset local-ce mac [ interface-type interface-number ]

参数说明

interface-type interface-number：接口类型和接口编号。此接口类型仅为Ethernet接口及其子接口、GigabitEthernet接口及其子接口和Virtual-Ethernet。

视图

用户视图

缺省级别

1：监控级

使用指南

本命令应用于CE以Ethernet或VLAN类型的L2VPN异种介质互连。如果CE接口的VLAN ID变更了或者CE改用其他接口接入，可以使用reset local-ce mac命令清除MAC地址及VLAN ID信息，使PE重新学习连接CE的接口信息。

如果指定了接口名，将只清除指定接口的信息，否则清除所有接口的信息。

使用实例

# 清除以太接口的动态学到的本地ce的MAC地址及VLAN ID信息。

<Quidway> reset local-ce mac
