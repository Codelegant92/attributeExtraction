ntp-service in-interface disable

命令功能

ntp-service in-interface disable命令用于禁止接口接收NTP消息包。

undo ntp-service in-interface disable命令用于使能接口接收NTP消息包。

缺省情况下，接口可以接收NTP消息包。

命令格式

ntp-service in-interface disable

undo ntp-service in-interface disable

参数说明

无

接口视图

缺省级别

2：配置级

使用指南

ntp-service in-interface disable命令提供了一种接入控制的方法。当设备不需要同步外部时钟或为外部时钟提供参考时钟源时，可以在与外部设备相连的接口上执行本命令来禁止接口接收NTP报文。

使用实例

# 禁止GE1/0/0接口接收NTP包。

<Quidway> system-view
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ntp-service in-interface disable
