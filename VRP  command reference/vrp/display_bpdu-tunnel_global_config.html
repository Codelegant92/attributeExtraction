display bpdu-tunnel global config

命令功能

display bpdu-tunnel global config命令查看全局BPDU的配置信息。

命令格式

display bpdu-tunnel global config

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

当需要查看设备在网络中的位置以及BPDU报文的全局well-known的MAC地址被替换后的组播MAC地址，可以使用display bpdu-tunnel global config命令查看。

使用实例

# 查看全局BPDU的配置信息。

<Quidway> system-view
[Quidway] display bpdu-tunnel global config
BridgeRole       customer
GroupMac         0100-0ccd-cdd0
表1 display bpdu-tunnel global config命令输出信息描述

项目

描述

BridgeRole
设备在网络中的位置类型，共分为两种类型：

  • customer：该类型为默认类型，此类型生成的BPDU报文的Well-Know目的MAC地址为01-80-C2-00-00-00。

  • provider：此类型生成的BPDU报文的Well-Know目的MAC地址为01-80-C2-00-00-08。

GroupMac

被替换后的BPDU报文的全局well-know的组播MAC地址值。
