active region-configuration

命令功能

active region-configuration命令用来激活MST域配置。

命令格式

active region-configuration

参数说明

无

视图

MST域视图、MST进程域视图

缺省级别

2：配置级

使用指南

在配置MST域的相关参数，特别是配置VLAN映射表时，很容易引起网络拓扑振荡。为了减少网络振荡，新配置的MST域参数并不会马上生效，而是在使用本命令手动激活后，或者启用MSTP协议后才会真正生效。

执行本命令前，建议先使用check region-configuration命令查看尚未生效的域配置是否正确。证实未生效的域配置完全正确后，再执行本命令。

使用实例

# 将VLAN5映射到生成树实例2上，并手动激活该项配置。

<Quidway> system-view
[Quidway] stp region-configuration
[Quidway-mst-region] instance 2 vlan 5
[Quidway-mst-region] active region-configuration

# 在MSTP进程10中，将VLAN5映射到生成树实例2上，并手动激活该项配置。

<Quidway> system-view
[Quidway] stp process 10
[Quidway-mst-process-10] stp region-configuration
[Quidway-mst-process-10-region] instance 2 vlan 5
[Quidway-mst-process-10-region] active region-configuration
