default med (BGP)

命令功能

default med命令用来配置BGP路由的缺省MED值。

undo default med命令用来恢复缺省设置。

缺省情况下，MED的值为0。

命令格式

default med med

undo default med

参数说明

med：设置BGP路由的MED（Multi-Exit-Discriminator）值。整数形式，取值范围是1～4294967295。

缺省级别

2：配置级

使用指南

default med命令只对本路由器上用import-route命令引入的路由和BGP的聚合路由生效。

使用实例

# 配置BGP的缺省MED值为10。

<Quidway> system-view
[Quidway] bgp 1
[Quidway-bgp] ipv4-family unicast 
[Quidway-bgp-af-ipv4] default med 10
