ipv6 circuit-cost

命令功能

ipv6 circuit-cost命令用来设置IS-IS中用于SPF计算的所有IPv6接口的开销值，即IPv6全局开销值。

undo ipv6 circuit-cost命令用来删除配置的接口开销值。

缺省情况下，配置的开销值适用于Level-1-2。

命令格式

ipv6 circuit-cost cost [ level-1 | level-2 ]

undo ipv6 circuit-cost [ level-1 | level-2 ]

参数说明

cost：指定IS-IS中用于SPF计算的IPv6接口的开销值。如果IS-IS的开销类型是wide或wide-compatible，则引入路由开销值的取值范围是1～16777215；否则取值为1～63。

level-1：指定Level-1接口的开销值。

level-2：指定Level-2接口的开销值。

缺省级别

2：配置级

使用指南

接口的IPv6开销值的优先级大于全局IPv6开销值的优先级。

使用此命令前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。

使用实例

# 设置用于SPF计算的IPv6全局开销值为20。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] ipv6 circuit-cost 20
