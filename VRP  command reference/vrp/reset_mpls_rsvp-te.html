reset mpls rsvp-te

命令功能

reset mpls rsvp-te命令用来重启RSVP-TE。

命令格式

reset mpls rsvp-te

参数说明

无

用户视图

使用指南

执行reset mpls rsvp-te命令将导致CR-LSP被拆除重建，主要用于在某些特殊条件下验证RSVP-TE的工作过程。一般情况下，不需要使用此命令。

使用实例

# 重启RSVP-TE。

<Quidway> reset mpls rsvp-te
