mpls l2vpn l2vpn-name

命令功能

mpls l2vpn命令用来创建Kompella方式的VPN。

undo mpls l2vpn命令用来删除对应的VPN。

缺省情况下，系统没有创建Kompella方式的VPN。

命令格式

mpls l2vpn l2vpn-name [ encapsulation encapsulation [ control-word | no-control-word ] ]

undo mpls l2vpn l2vpn-name

参数说明

l2vpn-name：唯一标识PE上的一个L2VPN，字符串格式，不支持空格，区分大小写，长度范围是1～31。

encapsulation：同种介质互通时，需要指定L2VPN连接的封装类型。创建VPN时需要指定封装类型，否则L2VPN无法建立。目前支持的封装类型有atm-aal5-sdu、atm-trans-cell、ethernet、fr、hdlc、ppp和vlan。

control-word：使能控制字（Control Word）特性。

no-control-word：禁止控制字（Control Word）特性。

使用指南

执行mpls l2vpn命令后，缺省情况下，只有使用ATM和Frame Relay封装类型的L2VPN使能控制字特性，其他封装类型的L2VPN都禁止控制字特性。

配置VPN封装时，封装类型必须与CE接口的封装类型匹配，如果不能匹配，应选择ip-interworking封装方式。

使用实例

# 创建Kompella模式的VPN。

<Quidway> system-view
[Quidway] mpls l2vpn vpn1 encapsulation ppp
[Quidway-mpls-l2vpn-vpn1] quit

# 进入MPLS-L2VPN视图。

<Quidway> system-view
[Quidway] mpls l2vpn vpn1
[Quidway-mpls-l2vpn-vpn1] quit
