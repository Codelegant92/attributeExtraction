display igmp-snooping port-info

命令功能

display igmp-snooping port-info命令用来显示路由器上的组播端口信息，包括静态成员端口、动态成员端口。

命令格式

display igmp-snooping port-info [ vlan vlan-id | vsi vsi-name ][ group-address group-address ] [ verbose ]

参数说明

vlan vlan-id：显示指定VLAN的组播端口信息。vlan-id表示VLAN编号，取值范围是1～4094。

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

group-address ip-group-address：显示指定VLAN内与指定组播组相关的端口信息。group-address表示组播组IP地址，取值范围是224.0.1.0～239.255.255.255。

verbose：显示详细信息。

使用指南

只有路由器的IGMP Snooping使能后，执行本命令才有显示信息。

使用本命令查看VSI内组播组信息时，只能看到组播组内的PW端口和子接口；如果用户侧通过VLANIF绑定到VSI，则该VSI内的组播组端口信息无法查看到，若需要查看此信息，可以查看VLAN的端口信息。

只有当成员端口或路由端口处于Up状态时，使用本命令时才显示该端口的信息。

  • 命令display igmp-snooping port-info 用来查看所有组播端口信息。

  • 命令display igmp-snooping port-info vlan vlan-id用来查看指定VLAN内所有组播端口信息。

  • 命令display igmp-snooping port-info vsi vsi-name用来查看指定VSI内所有组播端口信息。

  • 命令display igmp-snooping port-info vlan vlan-id group-address group-address 用来查看VLAN内某指定组播组的端口信息。

  • 命令display igmp-snooping port-info vsivsi-namegroup-address group-address 用来查看VSI内某指定组播组的端口信息。

使用实例

# 显示路由器组播端口表信息

<Quidway> display igmp-snooping port-info
Group           LiveTime  Port Host PortList                        Flag
-----------------------------------------------------------------------
VSI ly, 1 Group(s)
234.0.0.1       1d:22h    1     1      PW(222.222.222.196/333)        S-R

项目

描述

Group

组播组IP地址。

LiveTime

组播组已存在时间。

Port

组播组包含的端口数目。

Host

组播组的出端口中成员端口的数目。

PortList

组播组的所有组播端口的名称。

Flag

端口的类型为：

  • S：静态成员端口

  • D：动态成员端口
