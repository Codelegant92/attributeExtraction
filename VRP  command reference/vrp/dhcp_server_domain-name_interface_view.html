dhcp server domain-name（接口视图）

命令功能

dhcp server domain-name命令用来指定DHCP服务器当前接口地址池分配给客户端的域名。

undo dhcp server domain-name命令用来删除分配的域名。

缺省情况下，未配置分给DHCP客户端的域名，域名为空。

命令格式

dhcp server domain-name domain-name

undo dhcp server domain-name

参数说明

domain-name：指定DHCP服务器分配给客户机的域名。字符串形式，不支持空格，长度范围是3～50，可以设定为包含数字、字母和下划线“_”或“.”的组合。

缺省级别

2：配置级

使用指南

在配置DHCP服务器时，使用dhcp server domain-name命令用来配置DHCP服务器当前接口的DHCP地址池分配给客户端的域名。

使用实例

# 将接口DHCP地址池的域名设为eth1_0_0.com.cn。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] dhcp server domain-name eth1_0_0.com.cn
