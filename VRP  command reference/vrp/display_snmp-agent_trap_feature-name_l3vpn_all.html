display snmp-agent trap feature-name l3vpn all

命令功能

display snmp-agent trap feature-name l3vpn all命令用来查看所有L3VPN模块告警的开启关闭状态和海量标志。

命令格式

display snmp-agent trap feature-name l3vpn all

表1 display snmp-agent trap feature-name l3vpn all命令显示信息说明

项目

描述

Module name

模块名称。

Trap number

告警事件的类型数。

Trap name

事件类型，有以下类型：

  • L3VPN_MIB_TRAP_VRF_UP：表示VRF Up。
  • L3VPN_MIB_TRAP_VRF_DOWN：表示VRF Down。
  • L3VPN_MIB_TRAP_THRESH_EXCEED：表示私网路由数目超过门限值。
  • L3VPN_MIB_TRAP_THRESH_CLEARED：表示私网路由数目回落到门限值以下。
  • L3VPN_MIB_TRAP_MID_THRESH_EXCEED：表示私网路由数量超过私网路由表的路由中限。

Status

告警开关状态，有以下两种：

  • on：告警开关打开。
  • off：告警开关关闭。

Excessive Flag

海量信息标识，有以下两种取值：

  • Y：表示当前类型的信息是海量信息。
  • N：表示当前类型的信息不是海量信息。
