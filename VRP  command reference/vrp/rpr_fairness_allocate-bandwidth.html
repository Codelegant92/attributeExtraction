rpr fairness allocate-bandwidth

命令功能

rpr fairness allocate-bandwidth命令用于配置服务类的可分配带宽。

undo rpr fairness allocate-bandwidth命令用来恢复可分配带宽的缺省值。

缺省情况下，可分配带宽是0Mbit/s。

命令格式

rpr fairness allocate-bandwidth abs-value { east | west }

undo rpr fairness allocate-bandwidth { east | west }

参数说明

east：表示本地节点的东向发送链路。

west：表示本地节点的西向发送链路。

abs-value：带宽绝对值。范围因接口类型而不同，单位Mbit/s。

缺省级别

2：配置级

使用指南

可分配带宽有以下特点：

  • 用于传输subclassA1类和classB-CIR类的流量。

  • 线路拥塞时可以保证流量，而在流量不足时可回收。

  • 流量分东西向。

使用实例

# 配置本节点的西向链路可分配带宽为2000Mbit/s。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr fairness allocate-bandwidth 2000 west
