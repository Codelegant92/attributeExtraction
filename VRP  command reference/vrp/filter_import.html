filter import

命令功能

filter import命令用来在ABR上配置对进入本区域的Summary LSA进行过滤。

undo filter import命令用来取消对进入本区域的Summary LSA的过滤。

缺省情况下，不对LSA进行过滤。

命令格式

filter { acl-number | ip-prefix ip-prefix-name | route-policy route-policy-name } import

undo filter import

参数说明

acl-number：基本访问控制列表编号。整数形式。

ip-prefix ip-prefix-name：指定地址前缀列表名称。字符串形式。

route-policy route-policy-name：指定路由策略名称。字符串形式。

使用指南

某些情况下，可能要求只发布某些满足条件的路由信息，此时，可以通过filter import命令设置过滤条件，只有通过了过滤的路由信息才能被发布。

使用实例

# 配置对进入本区域的Type-3 LSA进行过滤。

<Quidway> system-view 
[Quidway] ospf 100 
[Quidway-ospf-100] area 1
[Quidway-ospf-100-area-0.0.0.1] filter ip-prefix my-prefix-list import
