min-active-links

命令功能

min-active-links命令用来设置IMA组正常工作所需的最小可用链路数。

undo min-active-links命令用来恢复缺省设置。

缺省情况下，IMA组正常工作所需的最小链路数为1。

命令格式

min-active-links number

undo min-active-links

参数说明

number：链路数。取值范围是1～8。

视图

IMA组接口视图

缺省级别

2：配置级

使用指南

配置的最小可用链路数不能大于IMA组的总链路数。

使用实例

# 设置IMA组1/0/1正常工作所需的最小可用链路数为2。

<Quidway> system-view
[Quidway] interface ima-group 1/0/1
[Quidway-Ima-group 1/0/1] min-active-links 2
