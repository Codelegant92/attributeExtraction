pwd（SFTP客户端视图）

命令功能

pwd命令用来显示远程SFTP服务器上的工作目录。

命令格式

pwd

参数说明

无

视图

SFTP客户端视图

缺省级别

3：管理级

使用指南

如果执行pwd命令后，发现并不是正确的工作目录，可以通过运行cd命令来修改SFTP客户端在SFTP服务器上的工作目录。

使用实例

# 显示远程SFTP server上的工作目录。

<Quidway> system-view
[Quidway] sftp 10.164.39.222
Please input the username:client001
Trying 10.164.39.222 ...
Press CTRL+K to abort
Connected to 10.164.39.222 ...
Enter password:
sftp-client> pwd
flash:
sftp-client>cd flash:/zxw
Current directory is:
flash:/zxw
sftp-client> pwd
flash:/zxw
