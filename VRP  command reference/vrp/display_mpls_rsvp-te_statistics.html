display mpls rsvp-te statistics

命令功能

display mpls rsvp-te statistics命令用来查看RSVP-TE的运行统计信息。

命令格式

display mpls rsvp-te statistics { global | interface [ interface-type interface-number ] } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

global：全局RSVP-TE运行统计信息。

interface：接口上的RSVP-TE运行统计信息。

interface-type interface-number：运行RSVP-TE的接口的类型和编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
