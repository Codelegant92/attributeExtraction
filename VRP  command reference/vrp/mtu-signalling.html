mtu-signalling

命令功能

mtu-signalling命令用来使能发送标签映射消息时携带MTU TLV。

undo mtu-signalling命令用来禁止发送标签映射消息时携带MTU TLV。

缺省情况下，发送标签映射消息时携带私有的MTU TLV。

命令格式

mtu-signalling [ apply-tlv ]

undo mtu-signalling [ apply-tlv ]

参数说明

apply-tlv：发送标签映射消息时，根据RFC 3988携带MTU TLV。

视图

MPLS-LDP视图

缺省级别

2：配置级

使用指南

改变MTU信令配置将可能重新创建原始会话。基于此会话建立的LSP也将被删除重建。因此，该命令应该在接口使能LDP之前配置。

使用实例

# 使能LDP发送标签映射消息时根据RFC 3988携带MTU TLV。

<Quidway> system-view
[Quidway] mpls ldp
[Quidway-mpls-ldp] mtu-signalling apply-tlv
Warning: All the related sessions will be deleted if the operation is performed
!Continue? (y/n)y
[Quidway-mpls-ldp] quit
