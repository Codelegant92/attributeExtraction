dialer bundle-member

命令功能

dialer bundle-member命令用来在共享DCC的配置中设置一个物理接口属于哪一个Dialer bundle。

undo dialer bundle-member命令用来将该物理接口脱离该Dialer bundle。

物理接口缺省为不属于任何Dialer bundle。

命令格式

dialer bundle-member number [ priority priority ] [ max-link max-channel ] [ min-link min-channel ]

undo dialer bundle-member number

参数说明

number：Dialer bundle序号。整数形式，取值范围是1～255。

priority：物理接口在这个Dialer bundle中的优先级。整数形式，取值范围是1～255。优先级高的物理接口优先使用，本参数为可选项。priority的缺省值是1。

max-link max-channel：表示Dialer bundle能够使用该接口的最大通道数，取值范围是1～255。

min-link min-channel：表示Dialer bundle能够使用该接口的最小通道数，取值范围是1～255。

使用指南

此命令只能用于物理接口，一个物理接口可以是多个Dialer bundle的成员。

说明：

对于ISDN物理接口（BRI或PRI接口），为了支持该ISDN接口B通道能够根据Dialer接口来动态封装链路层协议，则必须先将该接口的链路层封装协议设置成PPP。

使用实例

# 设置Serial1/0/0属于Dialer bundle1、bundle2，优先级均为50。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] dialer bundle-member 1 priority 50
[Quidway-Serial1/0/0] dialer bundle-member 2 priority 50
