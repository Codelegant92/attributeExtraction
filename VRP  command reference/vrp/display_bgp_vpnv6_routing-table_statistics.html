display bgp vpnv6 routing-table statistics

命令功能

display bgp vpnv6 routing-table statistic命令用来显示BGP VPNv6的路由统计信息。

命令格式

display bgp vpnv6 { all | route-distinguisher route-distinguisher | vpn6-instance vpn6-instance-name } routing-table statistics [ as-path-filter as-path-filter-number | different-origin-as ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | route-distinguisher route-distinguisher | vpn6-instance vpn6-instance-name } routing-table statistics regular-expression as-regular-expression

display bgp vpnv6 { all | route-distinguisher route-distinguisher | vpn6-instance vpn6-instance-name } routing-table statistics community [ community-number | aa:nn ] &<1-13> [ internet | no-advertise | no-export | no-export-subconfed ] * [ whole-match ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | route-distinguisher route-distinguisher | vpn6-instance vpn6-instance-name } routing-table statistics community-filter { { community-filter-name | basic-community-filter-number } [ whole-match ] | advanced-community-filter-number } [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 vpn6-instance vpn6-instance-name routing-table peer ipv6-address { advertised-routes | received-routes [ active ] } statistics [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：查看所有VPNv6的BGP路由统计信息。

route-distinguisher route-distinguisher：显示指定RD的BGP路由信息。

vpn6-instance vpn6-instance-name：查看VPNv6实例的BGP路由统计信息。

as-path-filter as-path-filter-number：指定匹配的AS路径过滤器号。

community：用来查看路由表中指定BGP团体的路由信息。

community-number：指定的团体号。

aa:nn：指定的团体号。

internet：显示带有Internet团体属性的BGP路由。

no-advertise：显示带有No-Advertise团体属性的BGP路由。

no-export：显示带有No-Export团体属性的BGP路由。

no-export-subconfed：显示带有No-Export-Subconfed团体属性的BGP路由。

whole-match：精确匹配。

community-filter：用来显示匹配指定的BGP团体属性过滤器的路由信息。

community-filter-name：团体属性过滤器名称。

basic-community-filter-number：指定基本团体属性过滤器号。

advanced-community-filter-number：指定高级团体属性过滤器号。

different-origin-as：用来显示具有相同目的地址但不同源AS号的路由。

regular-expression as-regular-expression：为匹配的AS正则表达式。字符串类型，取值范围是1～80。

peer ipv6-address：显示指定对等体的路由信息。

advertised-routes：显示发布给指定对等体的路由信息。

received-routes：显示从指定对等体收到的路由信息。

active：显示从指定对等体收到的活跃路由信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看本端BGP VPNv6路由的统计信息。

<Quidway> display bgp vpnv6 all routing-table statistics

 Total number of routes from all PE: 2

 Total routes of vpn6-instance vpn6: 5

 Total routes of vpn6-instance vpn1: 0

# 查看本端名为vpn6的IPv6 VPN实例的路由统计信息。

<Quidway> display bgp vpnv6 vpn6-instance vpn6 routing-table statistics

 Total Number of Routes: 5

# 查看从对等体1.1.1.1接收的BGP VPNv6路由数。

<PE2> display bgp vpnv6  all routing-table peer 1.1.1.1 received-routes statistics
  Received routes total: 1

# 查看本端带有internet团体属性的指定RD的VPN-IPv6路由统计信息。

<Quidway> display bgp vpnv6 route-distinguisher 100:1 routing-table statistics community internet

 Total number of routes from all PE: 2

 Total routes of vpn6-instance vpn6: 5

# 查看本端指定团体列表的所有BGP VPNv6路由统计信息。

<Quidway> display bgp vpnv6 all routing-table statistics community-filter 1

 Total number of routes from all PE: 2

 Total routes of vpn6-instance vpn6: 5

 Total routes of vpn6-instance vpn1: 0

# 查看与指定的AS路径正则表达式匹配的所有BGP VPNv6路由统计信息。

<Quidway> display bgp vpnv6 all routing-table statistics regular-expression 65420*

 Total number of routes from all PE: 1

 Total routes of vpn6-instance vpn6: 1

 Total routes of vpn6-instance vpn1: 0

# 查看本端从私网对等体2000::1收到的BGP VPNv6路由的统计信息。

<Quidway> display bgp vpnv6 vpn6-instance vpn1 routing-table peer 2000::1 received-routes statistics
  Received routes total: 2

# 查看本端发送给对等体2000::1的vpn1的IPv6路由的统计信息。

<Quidway> display bgp vpnv6 vpn6-instance vpn1 routing-table peer 2000::1 advertised-routes statistics
  Advertised routes total: 2

  Default originated : 0
