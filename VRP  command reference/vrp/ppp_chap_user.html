ppp chap user

命令功能

ppp chap user命令用来配置本地用户名称。

undo ppp chap user命令用来取消本地用户名称。

缺省情况下，本地用户名称为空。

命令格式

ppp chap user user-name

undo ppp chap user

参数说明

user-name：为对端所配置的本地用户名称，字符串形式，长度为1～64。

视图

接口视图

缺省级别

2：配置级

使用指南

配置CHAP验证时，要将各自的user-name配置为对端的user，而且对应的password要一致。可使用undo ppp chap user命令删除配置的user-name（本地路由器名）。

使用实例

# 配置接口POS1/0/0进行CHAP验证时，本地用户名称为Quidway。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ppp chap user Quidway
