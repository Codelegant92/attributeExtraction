ndp timer hello

命令功能

ndp timer hello命令用来设置设备发送NDP报文的时间间隔。

undo ndp timer hello命令用来恢复缺省值。

缺省情况下，设备发送NDP报文的时间间隔是60秒。

命令格式

ndp timer hello interval

undo ndp timer hello

参数说明

interval：指定NDP报文的发送时间间隔。十进制整数形式，取值范围是5～254，单位是秒。

使用指南

NDP报文的发送时间间隔必须小于NDP表项的老化时间。

危险：

配置NDP表项的老化时间，请参见ndp timer aging命令。

使用实例

# 设置设备发送NDP报文的时间间隔为50秒。

<Quidway> system-view
[Quidway] ndp timer hello 50
