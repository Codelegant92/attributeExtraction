display patch-information

命令功能

display patch-information命令用来查看当前所有的补丁信息。

命令格式

display patch-information

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display patch-information命令用来查看系统的补丁信息，包括运行的补丁单元、激活的补丁单元和去激活的补丁单元。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 查看系统的补丁信息。

<Quidway> display patch-information
