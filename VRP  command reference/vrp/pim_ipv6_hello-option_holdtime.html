pim ipv6 hello-option holdtime

命令功能

pim ipv6 hello-option holdtime命令用来设置路由器等待接收邻居发送Hello消息的超时时间。

undo pim ipv6 hello-option holdtime命令用来恢复缺省值。

缺省情况下，路由器等待接收邻居发送Hello消息的超时时间是105秒。

命令格式

pim ipv6 hello-option holdtime interval

undo pim ipv6 hello-option holdtime

参数说明

interval：指定路由器等待接收邻居发送的Hello消息的超时时间。整数形式，取值范围是1～65535，单位是秒。

使用指南

此命令与IPv6 PIM视图下的hello-option holdtime（IPv6）的含义是相同的，只是视图不同。缺省情况下，如果接口上没有配置，取IPv6 PIM视图下配置的值；否则取接口视图下配置的值。

使用实例

# 设置接口GE1/0/0上PIM Hello消息的保持时间是120秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 hello-option holdtime 120
