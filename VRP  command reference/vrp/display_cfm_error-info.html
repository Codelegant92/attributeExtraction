display cfm error-info

命令功能

display cfm error-info 命令用来查看符合指定条件的配置错误信息。

命令格式

display cfm error-info [ ingterface interface-type interface-number ] [ vlan vlanid | vsi vsi-name ] [ error-type { ccm-interval-error | maid-error | mac-error | cfm-leak } ]

参数说明

interface interface-type interface-number：显示指定接口名称、接口类型和接口编号的详细信息。接口类型包括：Ethernet接口、Eth-Trunk接口和GE接口。

vlan vlanid：指定VLAN的ID。整数形式，取值范围是1～4094。

vsivsi-name：指定VSI名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

ccm-interval-error：CCM报文发送间隔时间不匹配。

maid-error：MA ID不匹配。

mac-error：源MAC地址不匹配。

cfm-leak：CCM报文泄漏。

使用指南

现如今，802.1ag协议分为draft7草案版本和standard2007标准版本，本命令只支持standard2007标准版本使用。

当网络或设备的CFM功能出现问题时，可使用本命令查看错误的配置信息。得到的错误信息可作为定位问题的依据之一。

使用cfm enable（系统视图）命令全局使能CFM功能后才能使用本命令。

使用实例

# 查看指定接口上，指定错误类型的错误信息。

<Quidway> display cfm error-info interface gigabitethernet1/0/2 error-type cfm-leak
