display fib longer

命令功能

对于该命令，按照后面选择参数的不同匹配的方式也不同。

对于display fib [ slot-id ] destination-address命令，如果目的地址在自然掩码范围内能找到相应的FIB表项，则将其子网全部显示出来，否则显示与目的地址最长匹配的FIB表项。

对于display fib [ slot-id ] destination-address destination-mask命令，该命令用来查看与目的地址和掩码精确匹配的FIB表项。

对于display fib [ slot-id ] destination-address longer命令，该命令用来查看与目的地址在自然掩码范围内的IP地址匹配的所有的目的地址FIB表项。

对于display fib [ slot-id ] destination-address destination-mask longer命令，该命令用来查看与目的地址在输入的掩码范围内的IP地址匹配的所有的目的地址FIB表项。

对于display fib [ slot-id ] destination-address1 desination-mask1 desination-address2 destination-mask2命令，该命令用来查看目的地址在输入的desination-address1 desination-mask1到desitination-address2 destination-mask2范围内的FIB表项。

如果输入了slot-id，则显示对应板上的FIB信息。

命令格式

display fib [ slot-id ] [ vpn-instance vpn-instance-name ] destination-address1 [ destination-mask1 ] [ longer ] [ verbose ]

display fib [ slot-id ] [ vpn-instance vpn-instance-name ] destination-address1 destination-mask1 destination-address2 destination-mask2 [ verbose ]

参数说明

slot-id：槽位号，根据路由设备硬件的不同而具有不同的取值范围。

vpn-instance vpn-instance-name：VPN实例的名称。字符串形式，区分大小写，长度范围是1～31。

destination-address1：为目的IP地址1。点分十进制表示。

destination-mask1：相应的子网掩码1。为点分十进制格式的掩码或整数形式的掩码长度。

destination-address2：为目的IP地址2。点分十进制表示。

destination-mask2：相应的子网掩码2。为点分十进制格式的掩码或整数形式的掩码长度。

项目

描述

Destination/Mask

目的地址/掩码长度

Nexthop

下一跳

Flag

当前标志，G、H、U、S、D的组合。G表示Gateway（网关路由）H表示Host（主机路由）U表示Up（可用路由）S表示Static（静态路由）D表示Dynamic（动态路由）

TimeStamp

时间戳，表示该表项存在的时间，单位是秒。

Interface

到目的地址的出接口。

TunnelID

表示转发表项索引。用于连接上行板和下行板之间的转发表项。该值不为0时，表示匹配该项的报文通过MPLS隧道转发。该值为0时，表示报文不通过MPLS隧道转发。
