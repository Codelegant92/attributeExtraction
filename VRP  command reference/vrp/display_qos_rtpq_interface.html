display qos rtpq interface

命令功能

display qos rtpq interface命令用来显示当前IP RTP Priority的队列信息。

命令格式

display qos rtpq interface [ interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口号。

视图

所有视图

缺省级别

1：监控级

使用指南

display qos rtpq interface命令可以显示包括当前的RTP队列长度和RTP报文的丢包数。

如果没有指定接口，则显示所有接口的RTP队列配置情况和统计信息。

使用实例

# 显示当前IP RTP Priority的队列信息:

<Quidway> display qos rtpq interface GigabitEthernet 1/0/0
Interface: GigabitEthernet1/0/0
Output queue : (RTP queueing : Size/Max/Outputs/Discards)  0/17/263/276872
