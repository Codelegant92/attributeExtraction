nqa

命令功能

nqa命令用来创建NQA测试例，并进入NQA测试例视图。

undo nqa命令用来删除NQA测试例。

命令格式

nqa test-instance admin-name test-name

undo nqa { test-instance admin-name test-name | all-test-instance }

参数说明

admin-name：指定NQA测试例的管理者。字符串形式，长度范围是1～32。

test-name：指定NQA测试例的测试例名。字符串形式，长度范围是1～32。

all-test-instance：所有NQA测试例实例。

使用指南

执行命令undo nqa test-instance admin-name test-name，不能删除正在运行的测试例。

执行命令undo nqa all-test-instance，将删除所有的NQA测试例（包括正在运行的测试例）。

使用实例
