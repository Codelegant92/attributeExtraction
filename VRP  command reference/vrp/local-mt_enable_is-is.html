local-mt enable(IS-IS)

命令功能

local-mt enable命令用来使能IS-IS的本地组播拓扑特性。

undo local-mt enable命令用来取消该配置。

缺省情况下，不使能本地组播拓扑特性。

命令格式

local-mt enable

undo local-mt enable

参数说明

无

视图

IS-IS视图

缺省级别

2：配置级

使用指南

配置此命令前，需要先将IS-IS的开销类型配置为wide，wide-compatible或compatible，使能IS-IS的TE功能。

使用实例

# 使能IS-IS本地组播拓扑特性。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] cost-style compatible
[Quidway-isis-1] traffic-eng
[Quidway-isis-1] local-mt enable
