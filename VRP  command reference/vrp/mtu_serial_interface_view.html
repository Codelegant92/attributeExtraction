mtu（串行接口视图）

命令功能

mtu命令用来设置串口的最大传输单元MTU。

undo mtu命令用来恢复MTU的缺省值。

缺省情况下，串口的MTU是1500字节。

命令格式

mtu mtu

undo mtu

参数说明

mtu：串口最大传输单元。单位是字节。不同串口的MTU取值范围不同。

串行接口视图

缺省级别

2：配置级

使用指南

注意：

当接口上运行IPv6时，如果接口的MTU值小于1280字节，可能会使IPv6在该接口上工作不正常。所以当接口上运行IPv6时，建议设置该接口的MTU值不小于1280。

使用mtu命令改变接口最大传输单元MTU后，需要先对接口执行shutdown命令，再执行undo shutdown命令将接口重启，以保证设置的MTU生效。

使用实例

# 设置串口Serial 1/0/0:0的MTU为1200。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] mtu 1200
