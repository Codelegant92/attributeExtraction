reset mld explicit-tracking

命令功能

reset mld explicit-tracking命令用来删除接口上通过MLD加入组播组的主机。

命令格式

reset mld explicit-tracking { all | interface interface-type interface-number [ host ipv6-host-address [ group ipv6-group-address [ source ipv6-source-address ] ] ] }

参数说明

all：删除所有接口上加入组播组的主机。

interface interface-type interface-number：指定接口类型和编号。

ipv6-host-address：指定主机的链路本地地址。十六进制形式，取值范围是FE80::～FEBF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。

ipv6-group-address：指定组播组地址。十六进制形式，取值范围是FF00::～FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。

ipv6-source-address：指定组播源的IPv6单播地址。

使用指南

主机的删除不影响其再次加入。

使用实例

# 删除接口GE1/0/0上通过MLD加入FF03::101组播组的主机FE80::101。

<Quidway> reset mld explicit-tracking interface gigabitethernet1/0/0 host fe80::101 group ff03::101

# 删除接口GE1/0/0上加入(2000::12, FF03::101)转发项的主机FE80::101。

<Quidway> reset mld explicit-tracking interface gigabitethernet1/0/0 host ff80::101 group ff03::101 source 2000::12
