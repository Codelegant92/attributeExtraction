ip netstream timeout inactive

命令功能

ip netstream timeout inactive命令用来配置NetStream流的不活跃老化时间，当一条流的不活跃时间（从流最后一个报文流过时间到当前的时间）超过所设置的时间时，该流将被老化。

undo ip netstream timeout inactive命令用来把不活跃老化时间设置为缺省值。

缺省情况下，NetStream流的不活跃老化时间为30秒。

命令格式

ip netstream timeout inactive inactive-interval

undo ip netstream timeout inactive

参数说明

inactive-interval：指定NetStream 流的不活跃老化时间。整数形式，单位是秒，取值范围10～600秒。缺省值是30秒。
