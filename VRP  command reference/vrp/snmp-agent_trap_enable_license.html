snmp-agent trap enable license

命令功能

snmp-agent trap enable license命令用来使能License模块的Trap告警。

undo snmp-agent trap enable license命令用来去使能License模块的Trap告警。

缺省情况下，License模块的Trap告警为去使能。

命令格式

snmp-agent trap enable license

undo snmp-agent trap enable license

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

执行snmp-agent trap enable license命令后，使能License模块的Trap告警。

可以在用户视图下执行display current-configuration | include snmp命令行，查看是否有“snmp-agent trap enable license”项。如果有则表示当前License模块的Trap告警为使能状态。

使用实例

# 使能License模块的Trap告警。

<Quidway> system-view
[Quidway] snmp-agent trap enable license
