display pim ipv6 bsr-info

命令功能

display pim ipv6 bsr-info命令用来查看自举路由器信息。

命令格式

display pim ipv6 bsr-info [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

VPN-Instance

BSR信息所属的VPN实例。

Elected AdminScoped BSR Count

当选的AdminScoped BSR个数。

Elected BSR Address

BSR的IPv6地址。

Priority

BSR优先级。

Hash mask lenth

计算RP的哈希掩码长度。

State

BSR的状态。

Uptime

BSR的运行时间。

Next BSR message scheduled at

下一个BSR消息将在多少时间后到达。只有当选BSR上维护该定时器，定时器超时则发送BSM消息。

C-RP Count

C-RP的个数。

Candidate AdminScoped BSR Count

候选的AdminScoped BSR个数。

Candidate BSR Address

候选BSR的IPv6地址。

Wait to be BSR
当前C-BSR是否有效标识，有以下两种取值：

  • 0：表示当前C-BSR有效，即参与竞选BSR；

  • 1：表示当前C-BSR无效，即不参与竞选BSR。

当路由器上配置的C-BSR超过最大规格限制时，如果继续配置C-BSR，该标识位将取1。
