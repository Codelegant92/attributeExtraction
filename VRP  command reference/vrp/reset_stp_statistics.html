reset stp statistics

命令功能

reset stp statistics命令用来清除生成树的统计信息。

使用reset stp statistics命令时，如果指定端口，则清除指定端口上的生成树统计信息；如果不指定端口，则清除所有端口上的生成树统计信息。

命令格式

reset stp [ interface { interface-type interface-number [ to interface-number ] } &<1-10> ] statistics

参数说明

interface-type：端口类型，取值包括Ethernet、Eth-Trunk、GigabitEthernet和Virtual-Ethernet。

interface-number：端口编号。

视图

用户视图

缺省级别

1：监控级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除以太网接口GE 1/0/0到以太网接口GE 2/0/0上的统计信息。

<Quidway> reset stp interface gigabitethernet 1/0/0 to 2/0/0 statistics
