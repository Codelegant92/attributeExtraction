display constant-ifindex configuration

命令功能

display constant-ifindex configuration命令用于显示固定索引特性的状态与配置信息。

命令格式

display constant-ifindex configuration

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

当需要查询固定的索引的状态是否正常，可以使用display constant-ifindex configuration命令，通过显示信息了解固定索引特性的状态与配置信息。

使用实例

# 显示固定索引特性的状态与配置信息。

<Quidway> display constant-ifindex configuration
ifindex constant : Enable
ifindex max-number : 65535
current ifindex subinterface mode : sparse-mode
next ifindex subinterface mode : sparse-mode
current ifindex allocation : no multi-chassis

next ifindex subinterface mode

current ifindex allocation

当前接口索引分配是基于单框还是多框
