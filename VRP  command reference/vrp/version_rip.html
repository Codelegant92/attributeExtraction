version（RIP）

命令功能

version命令用来指定一个全局RIP版本。

undo version命令用来恢复全局RIP版本的缺省值。

缺省情况下，RIP版本为RIP-1。

命令格式

version { 1 | 2 }

undo version

参数说明

1：指定RIP-1版本。

2：指定RIP-2版本。

视图

RIP视图

缺省级别

2：配置级

使用指南

无

使用实例

# 指定发送和接收RIP-2报文。

<Quidway> system-view
[Quidway] rip 100
[Quidway-rip-100] version 2
