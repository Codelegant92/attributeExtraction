display snmp-agent usm-user

命令功能

display snmp-agent usm-user命令用来显示SNMP用户信息。

命令格式

display snmp-agent usm-user [ engineid engineid | group group-name | username user-name ] *

参数说明

engineid engineid：显示指定引擎ID的SNMPv3的用户信息。引擎ID是字符串形式，长度范围是10～64。

group group-name：显示属于指定组的用户信息。组名是字符串形式，长度范围是1～32。

username user-name：显示指定SNMPv3用户的信息。用户名是字符串形式，不支持空格，长度范围是1～32。

使用指南

使用display snmp-agent usm-user命令用来显示命令snmp-agent usm-user指定的SNMP用户信息。SNMP用户是执行SNMP管理操作的远端用户。

说明：

display snmp-agent usm-user命令只能显示SNMPv3用户信息。

使用实例

# 显示当前所有的用户信息。

<Quidway> display snmp-agent usm-user
   User name: u1
       Engine ID: 000007DB7F00000100001106 active
