put（SFTP客户端视图）

命令功能

put命令用来上传文件到远程SFTP服务器。

缺省情况下，如果没有指定远程服务器上的文件名，则认为与本地文件同名。

命令格式

put local-filename [ remote-filename ]

参数说明

local-filename：SFTP客户端的本地源文件名，字符串形式，绝对路径名长度范围为1～64。

remote-filename：SFTP服务器上的目的文件名，字符串形式，绝对路径名长度范围为1～64。

视图

SFTP客户端视图

缺省级别

2：配置级

使用指南

如果remote-filename没有指定，SFTP服务器默认上传后的目的文件名与SFTP客户端的源文件名相同。

使用实例

# 上传文件到SFTP服务器。

<Quidway> system-view
[Quidway] sftp 10.164.39.222
Please input the username:client001
Trying 10.164.39.222 ...
Press CTRL+K to abort
Connected to 10.164.39.222 ...
Enter password:
sftp-client> put wm.cfg
local file: wm.cfg --->  Remote file: cfcard:/wm.cfg
Info: Uploading file successfully ended.
