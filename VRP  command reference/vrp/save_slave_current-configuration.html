save slave current-configuration

命令功能

save slave current-configuration destination-filename命令用来保存备板的配置信息到目标文件。

命令格式

save slave current-configuration destination-filename

参数说明

destination-filename：保存备板配置的文件名和路径。

视图

用户视图

缺省级别

2：配置级

使用指南

该命令不能和save [ configuration-file ]命令同时使用，以避免多用户同时写文件出现错误。

配置文件的后缀名必须是“.cfg”或“.zip”。

将配置信息保存到新建文件时，系统将建立指定文件名的新文件。

将配置信息保存到已有文件时，系统将覆盖原文件内容。

使用实例

# 保存备板配置信息到新建文件。

<Quidway> save slave current-configuration xxxxx.cfg
Are you sure to save the current slave configuration to cfcard:/xxxxx.cfg? [Y/N]  :y
System is collecting current slave configuration. Please wait a moment..........
........
Now saving the current slave configuration to the device
Info:Save the current slave configuration to cfcard:/xxxxx.cfg successfully.

# 保存备板配置信息到已有文件。

<Quidway> save slave current-configuration xxxxx.cfg
Are you sure to save the current slave configuration to cfcard:/xxxxx.cfg? [Y/N] :y
The file cfcard:/xxxxx.cfg exists. Overwrite it?[Y/N]:y
Deleting file cfcard:/xxxxx.cfg ...Done!
System is collecting current slave configuration. Please wait a moment..................
Now saving the current slave configuration to the device
Info:Save the current slave configuration to cfcard:/xxxxx.cfg successfully.
