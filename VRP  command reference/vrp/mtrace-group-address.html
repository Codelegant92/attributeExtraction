mtrace-group-address

命令功能

mtrace-group-address ipv4命令配置NQA mtrace追踪的组播组地址。

undo mtrace-group-address ipv4命令恢复配置的缺省情况。

缺省情况下，未配置NQA mtrace追踪的组播组地址。

命令格式

mtrace-group-address ipv4 ip-address

undo mtrace-group-address

参数说明

ipv4 ip-address：指定mtrace追踪的组播组地址，属于IPv4普通组地址。

视图

NQA视图

缺省级别

2：配置级

使用指南

按照需要追踪的路径类型，选择配置该命令。

  • 如果是RPF路径，则不能指定追踪的组地址。

  • 如果是（S，G）组播路径，则必须指定追踪的组地址。

使用实例

# 配置mtrace测试例user test的组播组地址。

[Quidway-nqa-user-test] mtrace-group-address ipv4 225.1.1.1

# 取消mtrace测试例user test的组播组地址。

[Quidway-nqa-user-test] undo mtrace-group-address
