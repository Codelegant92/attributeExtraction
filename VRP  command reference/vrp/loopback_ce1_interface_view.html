loopback（CE1接口视图）

命令功能

loopback命令用来使能CE1接口的环回功能。

undo loopback命令用来禁止环回功能。

缺省情况下，禁止CE1接口的环回功能。

命令格式

loopback { local | remote | payload }

undo loopback

参数说明

local：允许接口对内自环。

remote：允许接口对外远端环回。

payload：允许接口对外载荷环回。

视图

CE1接口视图

缺省级别

2：配置级

使用指南

只有在进行某些特殊功能测试时，才将CE1接口设为环回。

当CE1接口设置为本端环回时，CE1接口的物理状态变为Up。

当对CE1接口设置环回时，该环回也会在该CE1接口下的同步串口生效：
  • 如果同步串口的链路层协议是PPP，同步串口的物理状态变为Up，而链路协议状态为Down。
  • 如果同步串口的链路层协议是HDLC，同步串口的物理状态和链路协议状态都是Up。
说明：

如果CE1接口形成的串口封装PPP协议，当CE1接口设置远端环回后，CE1接口形成的同步串口物理状态是Up，链路层协议状态将变为Down。

使用实例

# 使能CE1接口1/0/0对内自环。

<Quidway> system-view
[Quidway] controller e1 1/0/0
[Quidway-E1 1/0/0] loopback local
