display natpt statistics

命令功能

display natpt statistics命令用来查看NAT-PT的统计信息。

命令格式

display natpt statistics [ slot slot-id ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

slot slot-id：指定接口板的插槽号，在分布式环境中有效。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

NATPT Statistics

NAT-PT的统计信息

Total Sessions

当前有效的所有会话的数目

Expired Sessions

已实效被删除的会话的数目

Hits

表示地址转换成功的次数

Misses

表示转换失败的次数

Total Fragment Sessions

表示已存在的分片会话数量

Expired Fragment Sessions

表示超时的分片会话数量

Fragment Hits

表示成功的分片会话次数

Fragment Misses

表示失败的分片会话次数

Total Address Mappings

所有地址映射的数目

Total V6Server Mappings

所有v6server映射的数目

NATPT Interfaces

使能NAT-PT的接口
