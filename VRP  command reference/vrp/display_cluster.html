display cluster

命令功能

display cluster命令用来查看本设备所属HGMP集群的信息。

命令格式

display cluster

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

本命令可以在命令交换机和成员交换机上执行。

使用实例

# 在命令交换机上显示集群信息。

<HUAWEI_0.Quidway> display cluster
Cluster name:"HUAWEI"
Role:Administrator switch

management vlan id    : 1(default vlan)
Cluster multicast MAC address : 0180-c200-000a(default)
Cluster auto-join      : disabled

Handshake timer:10 sec
Handshake hold-time:60 sec
IP pool:10.0.0.1/8
No logging host configured
No SNMP host configured
No FTP server configured
No TFTP server configured
No SFTP server configured
cluster-member ftp-timeout: 300 sec(default)
Cluster SNMP NAT capability : enabled
Cluster FTP NAT capability : disabled
There are 3 member(s) in the cluster, and 0 of them are down.

该显示信息主要内容解释见表1。

表1 display cluster命令输出信息描述（命令交换机）

项目

描述

Cluster name

本设备所属集群的集群名称。

Role
本设备在本集群中的角色。

  • Administrator Switch表示命令交换机。

  • Member Switch表示成员交换机。

management vlan id

管理VLAN的VLAN编号。缺省值是1。

Cluster multicast MAC address

集群的组播地址。缺省值是0180-C200-000A。

Cluster auto-join
是否使能了候选交换机自动加入集群的功能。

  • disabled表示禁止本功能。

  • enabled表示使能本功能。

Handshake timer

命令交换机与成员交换机之间发送握手报文的时间间隔。以秒为单位。

Handshake hold-time

角色状态的有效保留时间。

IP pool

命令交换机用于集群内通信的IP地址，以及本集群使用的IP地址池的范围。

logging host

集群公用日志主机的配置情况。

SNMP host

集群公用SNMP主机的配置情况。

FTP server

集群公用FTP服务器的配置情况。

TFTP server

集群公用TFTP服务器的配置情况。

SFTP server

集群公用SFTP服务器的配置情况。

cluster-member ftp-timeout

成员交换机FTP连接空闲时间。缺省值是300，以秒为单位。

Cluster SNMP NAT capability
是否使能集群中的设备通过NAT方式来访问SNMP主机。

  • disabled表示禁止本功能。

  • enabled表示使能本功能。

Cluster FTP NAT capability
是否使能集群中的设备通过NAT方式来访问FTP Client或者FTP Server。

  • disabled表示禁止本功能。

  • enabled表示使能本功能。

There are 3 member（s） in the cluster, and 0 of them are down.

集群中成员交换机的数量，以及在这些成员中状态为down的交换机数量。

 Handshake timer:10 sec

 Administrator device mac address:000b-09ab-5c49
 Administrator status:Up

该显示信息主要内容解释见表2。

表2 display cluster命令输出信息描述（成员交换机）

项目

描述

Member number

本设备在集群内的成员编号。

cluster autojoin

是否使能了候选交换机自动加入集群的功能。

本字段在成员交换机上没有意义。

Administrator device mac address

本集群命令交换机的MAC地址。

Administrator status

命令交换机的状态。包括DOWN和UP两个状态。
