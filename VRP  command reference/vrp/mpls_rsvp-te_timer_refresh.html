mpls rsvp-te timer refresh

命令功能

mpls rsvp-te timer refresh命令用来设置节点的RSVP刷新消息的发送时间间隔。

undo mpls rsvp-te timer refresh命令用来恢复缺省设置。

缺省情况下，刷新时间为30秒。

命令格式

mpls rsvp-te timer refresh interval

undo mpls rsvp-te timer refresh

参数说明

interval：刷新定时器时长。整数形式，取值范围是10～65535，单位是秒。

MPLS视图

缺省级别

2：配置级

使用指南

执行mpls rsvp-te timer refresh命令时，应先使能RSVP-TE特性。

说明：

如果修改刷新周期，该刷新周期的修改结果需要等到上次定时器超时以后才生效。建议不要设置超长刷新周期或反复修改刷新周期。

使用实例

# 设置每60秒发送一次RSVP刷新消息。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls te
[Quidway-mpls] mpls rsvp-te
[Quidway-mpls] mpls rsvp-te timer refresh 60
