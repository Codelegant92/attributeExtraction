ripng default-route

命令功能

ripng default-route命令用来生成一条缺省路由到RIPng路由域中。

undo ripng default-route命令用来禁止发布RIPng缺省路由和转发IPv6缺省路由。

命令格式

ripng default-route { only | originate } [ cost cost ]

undo default-route

参数说明

only：只发布IPv6缺省路由（::/0），抑制其它路由的发布。

originate：发布IPv6缺省路由（::/0），但不影响其它路由的发布。

cost cost ：指定缺省路由的开销。整数形式，取值范围是1～15。缺省值是1。

使用指南

生成的RIPng缺省路由将强制通过指定接口的路由更新报文发布出去。该路由的发布不考虑其是否已经存在于IPv6路由表中。

使用实例

# 只将缺省路由以更新报文的形式从接口发布。

[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ripng default-route only

# 将缺省路由同其它路由一起以更新报文的形式从接口发布。

[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ripng default-route originate
