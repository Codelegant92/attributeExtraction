router-id (OSPFv3)

命令功能

router-id命令用来设置运行OSPFv3协议的路由器ID号。

undo router-id命令用来删除已设置的路由器ID号。

命令格式

router-id router-id

undo router-id

参数说明

router-id：路由器ID号，点分十进制形式。

视图

OSPFv3视图

缺省级别

2：配置级

使用指南

Router ID是一台OSPFv3进程在自治系统中的唯一标识。如果用户没有指定Router ID号，则OSPFv3进程无法运行。

设置路由器ID号时，必须保证自治系统中任意两个进程的RouterID号都不相同。

说明：

通过指定不同的进程号，可以在一台路由器上运行多个OSPFv3进程，在这种情况下，必须为不同进程指定不同的Router ID。

使用实例

# 设置OSPFv3进程1的RouterID为10.1.1.3。

<Quidway> system-view
[Quidway] ospfv3 1
[Quidway-ospfv3-1] router-id 10.1.1.3
