display rmon eventlog

命令功能

display rmon eventlog命令用来查看RMON事件日志中所记录的具体信息。

命令格式

display rmon eventlog [ entry-number ]

参数说明

entry-number：事件表行索引。整数形式，取值范围是1～65535。
