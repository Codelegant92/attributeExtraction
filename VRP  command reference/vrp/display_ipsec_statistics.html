display ipsec statistics

命令功能

display ipsec statistics命令用来显示IPSec处理报文的统计信息。

命令格式

display ipsec statistics

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display ipsec statistics命令显示IPSec处理报文的统计信息包括：

  • 输入、输出的安全报文数

  • 输入、输出的安全字节数

  • 输入、输出时被丢弃的报文数

  • 被丢弃的报文的详细说明

使用实例

# 显示IPSec的报文统计信息。

<Quidway> display ipsec statistics
  the security packet statistics:
    input/output security packets: 0/0
    input/output security bytes: 0/0
    input/output dropped security packets: 0/0
    dropped security packet detail:
      no enough memory: 0
      can't find SA: 0
      queue is full: 0
      authentication is failed: 0
      wrong length: 0
      replay packet: 0
      too long packet: 0
      wrong SA: 0
