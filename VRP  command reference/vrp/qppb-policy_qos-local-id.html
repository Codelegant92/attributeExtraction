qppb-policy qos-local-id

命令功能

qppb-policy qos-local-id 命令用来配置查找FIB获得QoS Local ID的方式。

undo qppb-policy qos-local-id命令用来取消配置。

命令格式

qppb-policy qos-local-id { source [ destination ] | destination }

undo qppb-policy qos-local-id { source [ destination ] | destination }

参数说明

source：根据源IP查找FIB获得QoS Local ID。

destination：根据目的IP查找FIB获得QoS Local ID。

视图

接口视图

缺省级别

2：配置级

使用指南

destination和source两个关键字表示所发布的携带路由属性的路由方向作为流量的目的地址方向还是源地址方向。

如果流量方向是从路由策略发送方到流量策略接收方，使能QPPB功能时，使用source。

如果流量方向是从路由策略接收方到路由策略发送方，使能QPPB功能时，使用destination。

使用实例

# 在接口GigabitEthernet2/1/0上启动qppb处理，根据源IP查找FIB获得QoS Local ID。

<Quidway> system-view
[Quidway] interface GigabitEthernet2/1/0
[Quidway-GigabitEthernet2/1/0] qppb-policy qos-local-id source
