network (BGP)

命令功能

network命令用来配置BGP发布的本地网络路由，即将本地路由表中的路由以静态方式加入到BGP路由表中，并发布给对等体。

undo network命令用来删除已有配置。

缺省情况下，BGP不发布任何本地的网络路由。

命令格式

network { ipv4-address [ mask | mask-length ] | ipv6-address prefix-length } [ route-policy route-policy-name ]

undo network { ipv4-address [ mask | mask-length ] | ipv6-address prefix-length }

参数说明

ipv4-address：BGP发布的IPv4网络地址，点分十进制形式。

mask：IP地址掩码。如果没有指定掩码，则按有类地址处理。

mask-length：IP地址掩码长度。如果没有指定掩码长度，则按有类地址处理。

ipv6-address prefix-length：BGP发布的IPv6网络地址和前缀长度。前缀长度为整数形式，取值范围是0～128。

route-policy route-policy-name：发布路由应用的Route-Policy。字符串形式，长度范围是1～40。

缺省级别

2：配置级

使用指南

使用undo network命令删除已有的配置时，需要注意指定正确的掩码。

使用实例

# 指定BGP发布本地路由10.0.0.0/16。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] network 10.0.0.0 255.255.0.0

# 配置BGP4+发布路由2002::/16。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv6-family unicast
[Quidway-bgp-af-ipv6] network 2002:: 16
