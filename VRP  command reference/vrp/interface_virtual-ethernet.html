interface virtual-ethernet

命令功能

interface virtual-ethernet命令用来创建虚拟以太网接口。

undo interface virtual-ethernet命令用来删除指定的虚拟以太网接口。

命令格式

interface virtual-ethernet ve-number

undo interface virtual-ethernet ve-number

参数说明

ve-number：虚拟以太网接口的编号。编号格式为槽号/卡号/顺序号。即使接口板上的卡不在位，也可以创建基于该子卡号的VE接口。顺序号取值范围是0～1023。不同产品上最大顺序号的值可能不同。

使用指南

虚拟以太网接口主要应用在PPPoEoA和IPoEoA中。

如果虚拟以太网接口和物理接口建立了绑定关系，则需要先解除绑定关系，才能删除成功。

在配置L2VPN接入多个L3VPN时，需要创建虚拟以太网接口。

只有QinQ终结模式的虚拟以太网主接口下，才能创建虚拟以太网子接口。虚拟以太网子接口编号的取值范围是1～1024。

在将虚拟以太网接口配置成QinQ终结模式前，应该先使用命令ve-group ve-group-id l3-access将接口配置成L3VE，否则无法执行mode user-termination命令。

在虚拟以太网主接口上使用undo mode user-termination命令来配置接口的工作模式为非QinQ终结模式前，应先删除该主接口下的所有子接口，否则系统会显示“Error: Please delete all subinterfaces of this interface first.”，提示配置错误。

使用实例

# 在1号槽的接口板0上创建虚拟以太网接口1/0/12。

<Quidway> system-view
[Quidway] interface virtual-ethernet 1/0/12
[Quidway-Virtual-Ethernet1/0/12]

# 在1号槽创建虚拟以太网子接口VE1/0/1.1。

<Quidway> system-view
[Quidway] interface virtual-ethernet 1/0/1
[Quidway-Virtual-Ethernet1/0/1] ve-group 1 l3-access
[Quidway-Virtual-Ethernet1/0/1] mode user-termination
[Quidway-Virtual-Ethernet1/0/1] quit
[Quidway] interface virtual-ethernet 1/0/1.1
[Quidway-Virtual-Ethernet1/0/1.1]
