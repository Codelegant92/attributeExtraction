arp learning strict（接口视图）

命令功能

arp learning strict force-enable命令用来使能接口下的ARP严格学习功能，使只学习自己发送的ARP请求报文的应答报文。

arp learning strict force-disable命令用来去使能接口下的ARP严格学习功能。

arp learning strict trust命令用来去除接口下配置的ARP严格学习策略，采用全局下的ARP严格学习策略。

命令格式

arp learning strict { force-enable | force-disable | trust }

参数说明

无

接口视图

缺省级别

2：配置级

使用指南

ARP严格学习的匹配采用最精匹配原则，即：

  • 当全局和接口同时配置了ARP严格学习功能时，采用接口下配置的策略。

  • 当接口下没有配置ARP严格学习功能时，采用全局下配置的ARP严格学习策略。

ARP严格学习限制支持的接口类型为：以太接口及子接口、Eth-Trunk接口及子接口、QinQ终结子接口、dotlq终结子接口和VLANIF接口。

使用实例

# 使能接口下的ARP严格学习功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 2/0/3
[Quidway-GigabitEthernet2/0/3] arp learning strict force-enable
