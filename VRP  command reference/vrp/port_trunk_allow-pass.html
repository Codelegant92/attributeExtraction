port trunk allow-pass

命令功能

port trunk allow-pass命令用来配置当前端口允许带指定tag的报文通过。

undo port trunk allow-pass命令用来取消当前端口允许带指定tag的报文通过。

命令格式

port trunk allow-pass vlan { { vlan-id1 [ to vlan-id2 ] } &<1-10> | all }

undo port trunk allow-pass vlan { { vlan-id1 [ to vlan-id2 ] } &<1-10> | all }

参数说明

vlan-id1：起始VLAN ID，取值范围是1～4094。

vlan-id2：结束VLAN ID，取值范围是1～4094，该值必须大于vlan-id1。

all：所有VLAN。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

如果使用三层端口，必须先执行portswitch（Ethernet接口视图）命令切换为二层端口。

使用实例

# 设置Trunk端口GE1/0/0允许通过的VLAN为2～10，100，200。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port trunk allow-pass vlan 2 to 10 100 200
