security acl

命令功能

security acl命令用来设置安全策略引用的访问控制列表。

undo security acl命令用来取消安全策略引用的访问控制列表。

缺省情况安全策略没有指定访问控制列表。

命令格式

security acl acl-number

undo security acl

参数说明

acl-number：指定安全策略所引用的访问控制列表号，取值范围是3000～3999。

视图

安全策略视图、安全策略模板视图

缺省级别

2：配置级

使用指南

安全策略所保护的数据流由此命令指定的acl来定义。

在实施IPSec安全策略时，系统首先检查通过接口的报文是否与acl中的规则匹配，如果匹配且是被访问控制列表允许（permit）的报文，则系统对报文进行IPsec保护后再发送。如果不匹配，或者访问控制列表拒绝（deny）的报文，则直接发送报文。

使用实例

# 设置安全策略引用3101号访问控制列表。

<Quidway> system-view
[Quidway] acl number 3101
[Quidway-acl-adv-3101] rule permit tcp source 10.1.1.1 0.0.0.255 destination 10.1.1.2 0.0.0.255
[Quidway] ipsec policy beijing 100 manual
[Quidway-ipsec-policy-manual-beijing-100] security acl 3101
