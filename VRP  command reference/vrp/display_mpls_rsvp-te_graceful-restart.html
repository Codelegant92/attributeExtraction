display mpls rsvp-te graceful-restart

命令功能

display mpls rsvp-te graceful-restart命令用来查看RSVP TE GR的状态。

命令格式

display mpls rsvp-te graceful-restart [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Graceful-Restart Capability

设备的RSVP GR能力

Restart Time

设备重启时间，单位是毫秒

Recovery Time

恢复所有LSP的时间，单位是毫秒

GR Status
本地设备RSVP GR的状态：

  • Gracefully Restart Not going on：未进行GR

  • Gracefully Restart on going restarting：正在进行GR

  • Gracefully Restart on going recovery：GR在恢复中

Number of Restarting neighbors

支持节点的数量

Number of LSPs recovered

已恢复的LSP的数量

Received Gr Path message count

重启节点收到的Path消息的数量

Send Gr Path message count

支持节点发送的Path消息的数量

Received RecoveryPath message count

重启节点收到的RecoveryPath消息的数量

Send RecoveryPath message count

支持节点发送的RecoveryPath消息的数量
