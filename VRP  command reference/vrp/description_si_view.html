description（业务实例视图）

命令功能

description命令用来配置业务实例的描述信息。

undo description命令用来删除业务实例的描述信息。

缺省情况下，业务实例的描述信息为空。

命令格式

description instance-description

undo description

参数说明

instance-description：业务实例的描述信息。长度范围是1～80个字符，首位不能是空格。

视图

业务实例视图

缺省级别

2：配置级

使用指南

description命令用来配置业务实例的描述信息，便于识别与使用业务实例。

在同一业务实例视图下，如果重复执行该命令，新配置将覆盖老配置。

使用实例

# 配置业务实例mysi的描述信息为“This is my service instance”。

<Quidway> system-view
[Quidway] service-instance instance-name mysi
[Quidway-service-instance-mysi] description This is my service instance
