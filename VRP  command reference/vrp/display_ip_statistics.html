display ip statistics

命令功能

display ip statistics命令用来查看IP流量统计信息，包括接收报文（其中包括丢弃的带源路由选项报文的统计信息）、发送报文、分片、重组的统计信息。

命令格式

display ip statistics [ slot slot-id ]

参数说明

slot slot-id：表示显示指定接口板的IP流量统计信息。

所有视图

缺省级别

1：监控级

使用指南

在主控板统计丢弃的带源路由选项报文的信息，当选择slot slot-id参数查看某一接口板信息时，丢弃的报文数为0。

使用实例

# 显示IP流量统计信息。

<Quidway> display ip statistics
  Input:     sum                     0      local                     0
             bad protocol            0      bad format                0
             bad checksum            0      bad options               0
             discard srr             0      TTL exceeded              0
  Output:    forwarding              0      local                     0
             dropped                 0      no route                  0
  Fragment:  input                   0      output                    0
             dropped                 0
             fragmented              0      couldn't fragment         0
  Reassembling:sum                   0      timeouts                  0
表1 display ip statistics命令输出信息描述
