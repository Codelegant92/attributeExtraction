ls

命令功能

ls命令用来查询指定的文件。

命令格式

ls [ remote-file ] [ local-filename ]

参数说明

remote-file：查询的远程文件，字符串形式，长度范围是1～64。

local-filename：保存的本地文件名，字符串形式，长度范围是1～64。

视图

FTP客户端视图

缺省级别

3：管理级

使用指南

不带任何参数时将显示所有的文件。
