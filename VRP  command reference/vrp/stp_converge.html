stp converge

命令功能

stp converge命令用来配置STP收敛方式。

undo stp converge命令用来恢复STP收敛方式为缺省值。

缺省情况下，STP收敛方式为fast。

命令格式

stp converge { fast | normal }

undo stp converge

参数说明

fast：直接删除MAC表项和ARP表项。

normal：快速老化MAC表项和ARP表项。

使用指南

当某个生成树实例的拓扑发生改变时，和它建立映射关系的VLAN的转发路径也将发生变化。此时，交换机的MAC地址表和ARP表中与这些VLAN相关的表项也需要更新。STP的收敛方式不同，交换机对MAC表项和ARP表项的处理方式不同：

  • 如果STP的收敛方式配置为fast，交换机将MAC地址表和ARP表中的以上表项直接删除。

  • 如果将STP的收敛方式配置为normal，交换机将MAC地址表和ARP表中以上表项的剩余存活时间置为0，对这些表项进行老化处理。如果配置的ARP老化探测次数大于零，则ARP对这些表项进行老化探测。

使用实例

# 设定以太网交换机STP收敛方式为normal。

<Quidway> system-view
[Quidway] stp converge normal
