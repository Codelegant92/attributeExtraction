display mpls stale-interface

命令功能

display mpls stale-interface命令用来查看设备上stale状态的MPLS接口信息。

命令格式

display mpls stale-interface [ interface-index ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-index：指定stale接口的索引号。十六进制整数形式，取值范围是0～FFFFFFFF。

verbose：使用此参数查看详细信息。例如是否配置了FRR，如果使能了FRR，还将显示绑定隧道的信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

被TE FRR保护的主隧道出接口在如下几种情况下将变为stale接口：

  • 该接口所在的接口板被拔出；

  • 如果是逻辑接口，该接口被删除；

  • 主隧道处于FRR In-Used状态时该接口被去使能MPLS。

可使用display mpls stale-interface命令查看这些stale接口的信息。

stale接口的接口索引号可通过执行不带参数的display mpls stale-interface命令查看。

使用实例

# 查看所有stale状态的MPLS接口信息。

<Quidway> display mpls stale-interface
Stale-Interface       Status    TE Attr   LSP Count  CRLSP Count Effective MTU
0x18000106            Up        En        0          2           -

项目

描述

Stale-Interface

处于stale状态的MPLS接口索引

Status

Stale接口的状态为Up

TE Attr
接口MPLS TE的使能状态：

  • Dis：接口没有使能MPLS TE

  • En：接口上使能了MPLS TE

LSP Count

接口上建立的LSP数量

CRLSP Count

接口上建立的CR-LSP数量

Effective MTU
MPLS转发实际采用的MTU值

  • 如果没有配置MPLS MTU，取Interface MTU

  • 如果配置了MPLS MTU，取MPLS MTU和Interface MTU中的较小值

项目

描述

No

显示项的编号

Interface

处于stale状态的MPLS接口索引

Status

Stale接口的状态为Up

TE Attribute

接口是否使能MPLS TE：Disable或Enable

Static LSPCount

接口上建立的静态LSP的数量

Static CR-LSPCount

接口上建立的静态CR-LSP的数量

LDP LSPCount

接口上建立的LDP LSP的数量

RSVP LSPCount

接口上建立的RSVP LSP的数量

MPLS MTU

接口上使用mpls mtu命令配置的MPLS MTU值。“-”表示接口已处于stale状态，因此无法获得MTU

Interface MTU

接口上使用mtu命令配置的MTU值

Effective MTU
MPLS转发实际采用的MTU值

  • 如果没有配置MPLS MTU，取Interface MTU

  • 如果配置了MPLS MTU，取MPLS MTU和Interface MTU中的较小值

TE FRR
接口MPLS TE的使能状态：

  • Disable：没有手工FRR的旁路隧道保护该接口

  • Enable：有手工FRR的旁路隧道保护该接口

Manual Bypass

手工配置的旁路隧道名称。

Interface State

接口的状态。“Stale”表示该接口处于待删除状态。
