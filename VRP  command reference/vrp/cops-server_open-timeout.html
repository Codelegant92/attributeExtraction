cops-server open-timeout

命令功能

cops-server open-timeout命令用于设置COPS服务器的Open消息超时时间。

undo cops-server open-timeout命令用于恢复缺省值。

缺省情况下，CPOS服务器的Open消息超时时间为15秒。

命令格式

cops-server open-timeout time

undo cops-server open-timeout

参数说明

time：Open消息超时时间。取值范围是1～255，单位是秒。

使用指南

Open消息超时时间是指客户端设备向COPS服务器发送Open消息后，等待响应的超时时间。在该时间内无法收到COPS服务器的响应时，客户端认为和COPS服务器的通讯已经中断。

该命令全局有效，即对所有的COPS服务器都起作用。

在网络状态存在不稳定因素的情况下，建议适当延长COPS Open消息超时时间。

执行命令cops-server open-timeout之后，需要在COPS服务器组视图下执行命令undo active和active使配置生效。

使用实例

# 设置COPS服务器的Open消息超时时间为40秒。

<Quidway> system-view
[Quidway] cops-server open-timeout 40
 Info:For the active COPS group, execute "undo active" and "active" commands to
make the configuration take effect.
