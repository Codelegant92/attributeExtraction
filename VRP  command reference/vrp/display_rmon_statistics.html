display rmon statistics

命令功能

display rmon statistics命令用来查看RMON统计消息。

命令格式

display rmon statistics [ ethernet interface-number | gigabitethernet interface-number ]

参数说明

ethernet：以太网接口。

gigabitethernet：千兆以太网接口。

interface-number：接口号。
