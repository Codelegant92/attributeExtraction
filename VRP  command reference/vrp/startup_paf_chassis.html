startup paf（多框）

命令功能

startup paf命令用来设置下次启动的PAF文件。

命令格式

startup paf file-name [ all | slave-board | chassis chassis-number { master | slave } ]

参数说明

file-name：指定License文件的存储路径和文件名，字符串形式，长度范围1～48。

all：所有框的主控板。

slave-board：指定下次启动后备用主控板的License文件名。

chassis chassis-number：指定中央交换框或转发框的框号。

master：指定框的主板。

slave：指定框的备板。

用户视图

缺省级别

2：配置级

使用指南

PAF文件的扩展名应为.txt，必须存放在存储设备的根目录下，缺省的存储位置为CF卡。

使用实例

# 设置下一次启动的PAF文件。

<Quidway> startup paf cfcard:/paf.txt
Info: Succeeded in setting main board resource file for system.
