display snmp-agent notification-log

命令功能

display snmp-agent notification-log命令用于显示告警缓冲区中最新告警日志的内容。

命令格式

display snmp-agent notification-log [ info | logtime starttime to endtime | size size ]

参数说明

info：告警缓冲区的信息。包括告警日志缓冲区中全局告警日志的数目、丢弃的告警日志数目、告警日志的老化时间、告警日志条数的限制数目、当前的告警日志记录的总数目和告警日志记录的使能标志。

starttime：告警日志记录的起始时间。格式为HH:MM:SS YYYY/MM/DD，HH:MM:SS表示时:分:秒。HH范围是0～23，MM和SS范围是0～59。YYYY/MM/DD表示年/月/日，YYYY范围是2000～2099，MM范围是1～12，DD范围是1～31。

endtime：告警日志记录的终止时间。格式为HH:MM:SS YYYY/MM/DD，HH:MM:SS表示时:分:秒。HH范围是0～23，MM和SS范围是0～59。YYYY/MM/DD表示年/月/日，YYYY范围是2000～2099，MM范围是1～12，DD范围是1～31。

size size：需要显示的最近告警日志的指定条数，不同产品，可显示的最大的值是不一样的，VRP支持15000条。如果指定的条数超过目前系统中存在的日志条数，则输出当前告警缓冲区中所有的告警日志信息。此参数取值以各产品实现规格为准。

使用指南

用户通过该命令显示告警缓冲区中告警日志信息的内容，可以用三种方式来查询：

  • 显示最近告警日志的指定条数。

  • 根据告警日志记录的时间来显示。

  • 默认情况下，显示所有记录的告警日志信息。

使用实例

# 显示告警日志信息。

<Quidway> display snmp-agent notification-log info

描述

Notification Admin Status

告警日志使能状态

GlobalNotificationsLogged

GlobalNotificationsAgeout

全局告警日志配置的老化时间

Total number of notification log

当前的告警日志记录的总条数

参数

描述

LogIndex

日志索引。

LogTime

日志记录距离系统启动的相对时间，以10ms为单位。

LogDateAndTime

日志记录的绝对时间和日期。

LogEngineID

日志记录的SNMP报文的引擎ID。

LogEngineTAddress

日志记录的报文IP地址和端口号。

LogEngineTDomain

日志记录的报文的传输类型。

LogContextEngineID

日志记录的SNMP报文上下文的引擎ID。

LogContextName

安全用户名＋IP地址＋VPN实例名。

LogNotificationID

日志记录的节点OID。

LogVariableIndex

日志变量的索引。

LogVariableOID

日志变量节点的OID。

LogVariableValueType

日志变量值的类型。

LogVariableValue

日志变量值。
