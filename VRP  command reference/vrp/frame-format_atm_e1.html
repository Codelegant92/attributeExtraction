frame-format（ATM E1）

命令功能

frame-format命令用来设置ATM E1/T1线路的帧格式。

undo frame-format命令用来恢复缺省设置。

缺省情况下，ATM E1的帧格式为no-crc4-adm，ATM T1的帧格式为esf-adm。

命令格式

frame-format { crc4-adm | no-crc4-adm }

frame-format { sf-adm | esf-adm }

undo frame-format

参数说明

crc4-adm：设置ATM E1帧格式为CRC4 ADM。

no-crc4-adm：设置ATM E1帧格式为No-CRC4 ADM。

sf-adm：设置ATM T1帧格式为SF ADM。

esf-adm：设置ATM T1帧格式为ESF ADM。

视图

ATM接口视图

缺省级别

2：配置级

使用指南

ATM E1通道支持带4bit循环冗余校验码CRC（Cyclic Redundancy Check）的帧格式和no-crc4-adm。ATM T1支持sf-adm和esf-adm帧格式。

使用实例

# 设置ATM E1接口2/0/1使用No-CRC4 ADM帧格式。

<Quidway> system-view
[Quidway] interface atm 2/0/1
[Quidway-Atm2/0/1] frame-format no-crc4-adm
