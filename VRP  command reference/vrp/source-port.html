source-port

命令功能

source-port命令用来配置本次测试的源端口号。

undo source-port命令用来恢复缺省配置。

缺省情况下，不配置源端口号。

命令格式

source-port port-number

undo source-port

参数说明

port-number：测试的源端口号。取值范围是1～50000。

视图

NQA视图

缺省级别

2：配置级

使用指南

不能对LSP Ping、LSP Traceroute、DNS、ICMP、DHCP、ICMPJitter、PathJitter、Jitter、MPing、MTrace、PWE3Ping、PWE3Trace、使能接口板发包的Jitter、使能接口板发包的ICMPJitter或Traceroute类型的测试配置源端口号。

不能修改正在运行的测试例的此项参数。

使用实例

# 设置测试例user test的源端口号为1000。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type ftp
[Quidway-nqa-user-test] source-port 1000
