display mpls te link-administration admission-control

命令功能

display mpls te link-administration admission-control命令用来查看所有链路接纳的CR-LSP信息，包括带宽、优先级等。

命令格式

display mpls te link-administration admission-control [ interface interface-type interface-number | stale-interface interface-index ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：查看指定接口的准入控制情况。

stale-interface interface-index：查看stale接口的准入控制情况。interface-index表示stale接口的索引号。十六进制整数形式，取值范围是1～FFFFFFFF。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

stale接口的接口索引号可通过执行display mpls stale-interface命令查看。

使用实例

# 查看所有链路的准入控制信息。

<Quidway> display mpls te link-administration admission-control
LspID           In/Out IF         S/H Prio    CT       BW (kbps)
1.1.1.9:1024:1    --- / GE1/0/0     0 /0        0        1000

项目

描述
