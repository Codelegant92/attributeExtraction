interface tunnel

命令功能

interface tunnel命令用来创建一个Tunnel接口，并进入该Tunnel接口视图。

undo interface tunnel命令用来删除指定的Tunnel接口。

缺省情况下，系统无Tunnel接口。

命令格式

interface tunnel interface-number

undo interface tunnel interface-number

参数说明

interface-number：Tunnel接口的编号，具体编号形式与实际使用的硬件设备相关。Tunnel接口编号只具有本地意义，隧道两端可以使用不同的接口编号。

缺省级别

2：配置级

使用指南

在配置GRE、MPLS TE、IPv6 over IPv4等类型的隧道时，需要先使用interface tunnel命令创建一个Tunnel接口，并在该接口下进行相关配置。

使用实例

# 创建编号为1/0/1的Tunnel接口。

<Quidway> system-view
[Quidway] interface tunnel 1/0/1
