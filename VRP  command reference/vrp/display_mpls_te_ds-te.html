display mpls te ds-te

命令功能

display mpls te ds-te命令用来查看当前系统的DS-TE相关信息。

命令格式

display mpls te ds-te { summary | te-class-mapping [ default | config | verbose ] }

参数说明

summary：显示当前系统的DS-TE相关配置信息。

te-class-mapping：显示TE-Class映射表的信息。

default：显示系统缺省的TE-Class映射表的信息。

config：显示配置的TE-Class映射表的信息。

verbose：显示TE-Class映射表的详细信息。

项目

描述

DS-TE IETF Supported

当前系统是否支持IETF模式的DS-TE

DS-TE MODE

DS-TE模式：

  • IETF：IETF定义的DS-TE模式
  • NON-IETF：非IETF定义的DS-TE模式

Bandwidth Constraint Model

带宽约束模型，分为：

  • RDM：俄罗斯套娃模型
  • MAM：最大带宽分配模型
  • Extended MAM：扩展的最大带宽分配模型

TE-Class ID

TE-Class的ID，取值范围0～7

Class Type

服务类型，如果显示为“--”，表示用户没有配置该TE-Class

Priority

隧道的抢占优先级，取值范围0～7。值越小优先级越高
