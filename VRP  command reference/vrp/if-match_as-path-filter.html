if-match as-path-filter

命令功能

if-match as-path-filter命令用来设置一个基于AS路径过滤器的匹配规则。

undo if-match as-path-filter命令用来删除该匹配规则。

缺省情况下，没有配置基于AS路径过滤器的匹配规则。

命令格式

if-match as-path-filter as-path-filter-number &<1-16>

undo if-match as-path-filter [ as-path-filter-number &<1-16> ]

参数说明

as-path-filter-number：指定AS路径过滤器号。整数形式，取值范围是1～256。在一个命令行中最多可以配置16个此参数。

使用指南

该命令仅应用在BGP协议中。要使这个匹配条件生效需要使用命令ip as-path-filter定义AS路径过滤器。

使用实例

# 设置序号为2的AS路径过滤器，允许路由信息中包含AS200和AS300。然后定义一个名为test的Route-Policy，该策略的10号节点引用AS路径过滤器2定义了一个if-match子句。

<Quidway> system-view
[Quidway] ip as-path-filter 2 permit _200_300
[Quidway] route-policy test permit node 10
[Quidway-route-policy] if-match as-path-filter 2
