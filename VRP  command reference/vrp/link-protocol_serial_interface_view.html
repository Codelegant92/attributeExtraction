link-protocol（串行接口视图）

命令功能

link-protocol命令用来配置串口的链路层封装协议。

缺省情况下，串口封装PPP协议。

命令格式

link-protocol { fr | hdlc | lapb | ppp | x25 }

参数说明

fr：配置串口的链路层协议封装为帧中继。

hdlc：配置串口的链路层协议封装为HDLC。

lapb：配置串口的链路层协议封装为LAPB。

ppp：配置串口的链路层协议封装为PPP。

x25：配置串口的链路层协议封装为X.25。

缺省级别

2：配置级

使用指南

接口相关的参数都需要在接口视图下配置，只有配置了正确的链路层协议才能配置该协议相关的参数。

使用实例

# 配置串口Serial1/0/0:0链路层封装的协议类型为HDLC。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol hdlc
