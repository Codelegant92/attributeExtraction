pim timer graft-retry

命令功能

pim timer graft-retry命令用来在接口上配置重传Graft消息的时间间隔。

undo pim timer graft-retry命令用来恢复时间间隔为缺省值。

缺省情况下，接口上重传Graft消息的时间间隔是3秒。

命令格式

pim timer graft-retry interval

undo pim timer graft-retry

参数说明

interval：表示重传Graft消息的时间间隔。整数形式，取值范围是1～65535，单位是秒。

缺省级别

2：配置级

使用指南

在PIM-DM模式中，当有成员加入曾经被剪枝的组时，路由器发出嫁接消息，并等待上游确认。如果在该命令配置的时间内没有收到确认消息，路由器将重传嫁接消息，直到收到上游的确认消息。

使用实例

# 在接口上配置重传Graft消息的时间间隔为80秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim timer graft-retry 80
