pfs

命令功能

pfs命令用来设置使用此安全策略发起协商时的完善的前向安全PFS（Perfect Forward Secrecy）特性。

undo pfs命令用来设置在协商时不使用PFS特性。

缺省情况没有PFS特性。

命令格式

pfs { dh-group1 | dh-group2 }

undo pfs

参数说明

dh-group1：指定使用768-bit Diffie-Hellman组。

dh-group2：指定使用1024-bit Diffie-Hellman组。

视图

安全策略视图、安全策略模板视图

缺省级别

2：配置级

使用指南

pfs命令使IPSec在使用此安全策略发起一个协商时，在阶段2的协商中进行一次附加的密钥交换以提高通讯的安全性。本端和对端指定的Diffie-Hellman组必须一致，否则协商会失败。

此命令只能在ike协商模式以及安全策略模板起作用，对通过手工方式建立的安全联盟不起作用。

使用实例

# 设置使用安全策略shanghai 200进行协商时使用PFS特性。

<Quidway> system-view
[Quidway] ipsec policy shanghai 200 isakmp
[Quidway-ipsec-policy-isakmp-shanghai-200] pfs dh-group1
