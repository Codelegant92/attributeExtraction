telnet

命令功能

telnet命令用来从当前路由器用Telnet工具再登录到其它路由器。

缺省情况下，在不指定port-number时，Telnet端口号为23。

命令格式

telnet [ vpn-instance vpn-instance-name ] [ -a source-ip-address ] host-name [ port-number ]

telnet ipv6 [ -a source-ip-address ] [ vpn6-instance vpn6-instance-name ] host-name [ -i interface-type interface-number ] [ port-number ]

参数说明

vpn-instance vpn-instance-name：VPN实例名，字符串形式，长度范围是1～31。

-a source-ip-address：本端路由器的IPv4/IPv6地址。

vpn6-instance vpn6-instance-name：VPN6的实例名称。字符串形式，不支持空格，长度范围是1～31。

host-name：远端路由器的IPv4/IPv6地址或主机名，IPv4地址取值范围1～20，IPv6地址取值范围1～46。

port-number：远端路由器提供Telnet服务的TCP端口号，范围1～65535。缺省值是23。

-i interface-type interface-number：如果远端主机使用了IPv6链路本地地址，则需要指定本地的出接口。

使用指南

通过执行telnet命令，用户可以很方便地从一个路由器登录到另一个路由器进行远程管理。

在连接的过程中，可以使用“Ctrl_K”停止本端路由设备与远端路由设备的连接。

通过指定源地址，用户可以以指定的IP地址与服务端通信，从而达到进行安全校验的目的。
