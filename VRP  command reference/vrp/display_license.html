display license

命令功能

display license命令用来查看主用主控板和备用主控板的License文件。

命令格式

display license

display license { all | { resource | service } item-name }

参数说明

all：License文件中的所有信息。

resource：License文件中性能项的配置值。

service：License文件中属性项的配置值。

item-name：License文件性能项或属性项的名称，字符串形式，长度范围是1～64。

使用指南

执行display license命令后，会显示主用、备用主控板上激活的License文件。

  • 只有主用主控板，则显示主用主控板上激活的License文件。

  • 有主用、备用主控板，则同时显示主用、备用主控板上激活的License文件。

说明：

目前display license { all | { resource | service } item-name }命令仅在NE40E、NE80E、NE5000E上可用。
