display rip

命令功能

display rip命令用来显示RIP进程的当前运行状态及配置信息。

命令格式

display rip [ process-id | vpn-instance vpn-instance-name ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIP进程号。整数形式，取值范围是1～65535。

vpn-instance vpn-instance-name：指定VPN实例名。字符串形式，不支持空格，区分大小写，长度范围是1～31。

配置vpn-instance-name参数，可以查看指定VPN实例的RIP配置。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

可以通过指定vpn-instance-name，查看RIP进程下不同的VPN实例信息

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

RIP process

RIP进程号

RIP version

RIP版本1或2

Preference

RIP进程优先级

Checkzero

零域检查

Default-cost

缺省开销值

Summary

路由聚合是否使能

Hostroutes

主机路由是否使能

Maximum number of balanced paths

等价路由的最大数目

Update time

RIP更新间隔时间

Age time

RIP路由老化时间

Suppress time

路由抑制时间

Garbage collection time

垃圾路由回收时间

Silent interfaces

抑制接口数（这些接口不发送周期更新报文）

Default routes

缺省路由，用于报文无法在路由表中找到对应路由项

Verify-source

源验证

Networks

网络地址

Configured peers

配置的邻居

Number of routes in database

RIP数据库中路由的条数

Number of interfaces enabled

使能RIP协议的接口数

Triggered updates sent

发送的触发更新报文数

Number of route changes

RIP进程改变路由数据库的统计数据

Number of replies to queries

RIP请求的响应报文数

Number of sending routes

每次更新时发送的路由条数

Number of routes sendable in a periodic update

每次更新时发送的路由条数

Number of routes sent in last periodic update

最后一次更新时发送的路由条数
