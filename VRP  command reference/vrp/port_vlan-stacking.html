port vlan-stacking

命令功能

port vlan-stacking命令用来配置VLAN Stacking功能。即，在收到的帧的最外层VLAN tag外，再加上一层VLAN tag。

port vlan-stacking outside-vlan命令用于在接收tagged帧的端口上配置VLAN Stacking。

port vlan-stacking ce-default-vlan命令用于在接收untagged帧的端口上配置VLAN Stacking。

undo port vlan-stacking命令用来取消VLAN Stacking功能。

命令格式

port vlan-stacking outside-vlan vlan-id1 [ to vlan-id2 ] stack-vlan vlan-id3

undo port vlan-stacking outside-vlan vlan-id1 [ to vlan-id2 ] stack-vlan vlan-id3

port vlan-stacking ce-default-vlan stack-vlan vlan-id3

undo port vlan-stacking ce-default-vlan

参数说明

outside-vlan vlan-id1 [ to vlan-id2 ]：端口接收到的帧携带的外层VLAN ID，取值范围是1～4094。

stack-vlan vlan-id3：新加入帧中的VLAN ID，取值范围是1～4094。

ce-default-vlan：表示接收untagged帧的端口上的缺省VLAN ID。

缺省级别

使用指南

如果是三层接口，必须先执行portswitch（Ethernet接口视图）命令切换为二层端口。

使用实例

＃ 进入端口的VLAN为100的帧在原有tag之前再加入一个VID=9的tag。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port vlan-stacking outside-vlan 100 stack-vlan 9
