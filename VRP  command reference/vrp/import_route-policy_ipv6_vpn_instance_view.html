import route-policy(IPv6 VPN实例视图)

命令功能

import route-policy命令用于将当前IPv6 VPN实例与一条入方向路由策略进行关联。并且路由策略只能关联一个，以最后关联的路由策略为准。

undo import route-policy命令用于取消当前IPv6 VPN实例与入方向路由策略的关联。

入方向路由策略没有缺省值，如果不配置，则允许所有VPN-target属性匹配的路由加入IPv6 VPN实例。

命令格式

import route-policy policy-name

undo import route-policy

参数说明

policy-name：VPN实例的入方向路由策略名。字符串形式，长度范围是1～40。

视图

IPv6 VPN实例视图

缺省级别

2：配置级

使用指南

当要求更精确地控制引入VPN实例路由时，可以采用入方向路由策略。入方向路由策略可以过滤引入的路由信息以及为通过过滤的路由信息设置路由属性。。

使用实例

# 对名为vrf1的IPv6 VPN实例在入方向应用路由策略poly-1。

<Quidway> system-view
[Quidway] ipv6 vpn6-instance vrf1
[Quidway-vpn6-instance-vrf1] import route-policy poly-1
