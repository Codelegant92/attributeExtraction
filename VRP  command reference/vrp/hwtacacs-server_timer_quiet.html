hwtacacs-server timer quiet

命令功能

hwtacacs-server timer quiet命令用来设置主服务器恢复激活状态的时间。

undo hwtacacs-server timer quiet命令用来恢复缺省配置。

缺省情况下，主服务器恢复激活状态前需要等待5分钟。

命令格式

hwtacacs-server timer quiet value

undo hwtacacs-server timer quiet

参数说明

value：主服务器恢复激活状态的时间值。取值范围是1～255，单位是分钟。

缺省级别

2：配置级

使用指南

改变主服务器恢复激活状态的时间时，不检查当前是否有用户在使用HWTACACS模板。

使用实例

# 设置主服务器恢复激活状态的时间为10分钟。

<Quidway> system-view
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1] hwtacacs-server timer quiet 10
