ipv6 auto-cost enable

命令功能

ipv6 auto-cost enable命令用来使能SPF中的自动开销计算功能。

undo ipv6 auto-cost enable命令用来关闭SPF中的自动开销计算功能。

命令格式

ipv6 auto-cost enable

undo ipv6 auto-cost enable

参数说明

无

视图

IS-IS视图

缺省级别

2：配置级

使用指南

当使能此功能后，对于某个IS-IS接口来说，如果在接口视图下没有配置其开销值，而且在IS-IS视图下也没有配置全局开销值，则此接口的开销由系统自动计算。

当cost-style为wide或wide-compatible时，自动开销的计算公式是：接口的开销值 = (Bandwidth-reference/Link-bandwidth)×10，其中，参考带宽值Bandwidth-reference用ipv6 bandwidth-reference命令设置，Link-bandwidth为接口带宽值。

当cost-style为narrow、narrow-compatible或compatible时，各个接口的开销值根据下表来确定。

用命令ipv6 circuit-cost配置的IPv6全局开销的优先级高于自动开销。

使用此命令前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。

使用实例

# 使能用于SPF计算的自动开销计算。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] ipv6 auto-cost enable
