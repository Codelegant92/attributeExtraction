display efm session

命令功能

display efm session命令用来查看指定接口和对端的EFM OAM会话信息，包括该接口的：

  • EFM状态

  • 环回测试超时时间

命令格式

display efm session { all | interface interface-type interface-number } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：显示所有配置了EFM OAM功能的接口和对端的EFM OAM会话信息。

interface interface-type interface-number：显示指定接口和对端的EFM OAM会话信息。interface-type表示接口类型，取值包括：ethernet和gigabitethernet。interface-number表示接口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用efm enable命令全局使能EFM OAM功能后才能使用本命令。

使用本命令只显示配置了EFM OAM功能的接口和对端的EFM OAM会话信息。

使用实例

# 查看所有接口和对端的EFM OAM会话信息。

<Quidway> system-view
[Quidway] efm enable
[Quidway] display efm session all
  Interface                 EFM State                   Loopback Timeout
  ----------------------------------------------------------------------
  GigabitEthernet2/0/2      discovery                   --

项目

描述

EFM State
接口的EFM OAM协议状态。

  • disabled：接口处于EFM OAM协议禁止状态。

  • discovery：接口处于对端发现状态。

  • detect：接口处于握手状态。

  • loopback(control)：接口处于远端环回状态，丢弃返回的报文，且是远端环回的发起方。

  • loopback(control,forward)：接口处于远端环回状态，转发返回的报文，且是远端环回的发起方。

  • loopback (be controlled)：接口处于远端环回状态，且是远端环回的响应方。

LoopBack Timeout

接口的环回测试超时时间。
