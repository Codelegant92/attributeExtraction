import-route (RIPng)

命令功能

import-route命令用来从其它路由协议引入路由。使用filter-policy参数可以过滤引入的路由，并可以设置路由的属性。

undo import-route命令用来取消引入外部路由信息。

命令格式

import-route protocol [ process-id ] [ cost cost ] [ route-policy route-policy-name ]

undo import-route protocol [ process-id ]

参数说明

protocol：要引入的源路由协议，它可以是下列关键字中的一个：direct、static、ripng、isis、ospfv3或bgp。

process-id：被引入路由的进程号。只有当protocol是ripng、ospfv3或isis时需要指定进程号。

cost cost：引入路由的权值。整数形式，取值范围是0～15。如果没有指定权值，则使用default-cost命令设置的缺省权值。

route-policy route-policy-name：路由策略名称。字符串形式，不支持空格，区分大小写，长度范围是1～40。

2：配置级

使用指南

只有通过过滤的路由才能被加入到RIPng路由表。

使用实例

# 引入IS-IS协议（进程号7）的路由信息，并设置其权值为7。

<Quidway> system
[Quidway] ripng 1
[Quidway-ripng-1] import-route isis 7 cost 7
