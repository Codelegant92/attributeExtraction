display mac-tunnel

命令功能

display mac-tunnel命令用来查看PBB-TE Tunnel的信息。

命令格式

display mac-tunnel [ tunnel-name tunnel-name ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

tunnel-name tunnel-name：指定PBB-TE Tunnel的名字。长度范围是1～31个字符。

verbose：指定查看PBB-TE Tunnel的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不输入隧道名字，查看所有隧道的基本信息和详细信息。如果查看指定的PBB-TE Tunnel信息，则可以查看该PBB-TE Tunnel的基本信息和详细信息。

如果不输入verbose参数，则只显示待查看隧道的名字、隧道的index、目的MAC地址、VLAN、出接口编号、PBB-TE Tunnel组合状态信息、CFM运行状态。

如果输入verbose参数，显示内容包括：隧道名称、描述信息、目的MAC地址、源MAC地址、VLAN、VLAN协议号、出接口编号、隧道管理状态、CFM运行状态、是否信任业务实例优先级、是否使能水平分割、是否加入保护组、在保护组中的地位（工作隧道/保护隧道）、本隧道加入的保护组中的另一条隧道、保护组倒换结果、运行的时间、映射的业务实例的信息。

其中，隧道状态信息的显示内容会根据具体配置情况不同：

  • 如果隧道配置了shutdown命令，则Manage-status显示为DOWN状态

  • 如果隧道没有配置shutdown命令，则根据实际的隧道状态显示为Up或Down状态

表1 display mac-tunnel命令的输出信息描述表

项目

描述

Tunnel-name

PBB-TE Tunnel的名字。

Tunnel-index

PBB-TE Tunnel的索引。

Tunnel-Description

PBB-TE Tunnel的描述信息。

Destination-MAC

PBB-TE Tunnel目的MAC地址。

Source-MAC

PBB-TE Tunnel源MAC地址。

Port

PBB-TE Tunnel的主接口编号。

Vlan

PBB-TE Tunnel的BVLAN。

Vlan-protocol

PBB-TE Tunnel的BVLAN协议号。

Manage-status
PBB-TE Tunnel的管理状态：

  • down：关闭

  • up：打开

Cfm-status/CfmStatus
CFM的运行状态：

  • up：CC链路检测正常

  • down：CC链路检测故障

  • disable：没有配置CFM功能（即没有配置本端CC发送使能和远端CC接收使能）

CombinStatus
PBB-TE Tunnel、BVLAN以及属于BVLAN的端口的组合状态：

  • up：PBB-TE Tunnel、BVLAN以及属于BVLAN的端口状态都为Up。

  • down：PBB-TE Tunnel、BVLAN以及属于BVLAN的端口中，如果其中一个状态为down，则CombinStatus为down。

Total number of tunnels

隧道总数。

Number of tunnels with being Up

CombinStatus状态为Up的隧道数。

Number of tunnels with CfmStatus being Up

CfmStatus状态为Up的隧道数。

TrustInstancePriority
PBB-TE Tunnel是否信任业务实例优先级：

  • yes：信任业务实例优先级

  • no：不信任业务实例优先级

Priority

PBB-TE Tunnel的优先级，分为0～7八个级别。缺省情况下隧道信任业务实例优先级，该项显示内容为空。

Split-horizon
PBB-TE Tunnel的水平分割属性：

  • enable：水平分割使能

  • disable：水平分割去使能

Join-protection-group
PBB-TE Tunnel是否加入保护组：

  • yes：加入保护组

  • no：没有加入保护组

Work-or-Protect

该PBB-TE Tunnel在保护组中是工作隧道还是保护隧道（如果没有配置保护组，该项内容为-）。

Another-tunnel

该PBB-TE Tunnel加入的保护组中的另一条隧道（如果没有配置保护组，该项显示内容为-）。

Switch-result

保护组倒换结果（如果没有配置保护组，该项显示内容为-）。

Tunnel up-time

PBB-TE Tunnel运行的时间，单位是秒。

Bind service-instance

PBB-TE Tunnel上绑定的业务实例。

Total num of bind service-instance

PBB-TE Tunnel上绑定业务实例的总数。
