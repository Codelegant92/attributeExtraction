mpls l2vpn trigger if-down

命令功能

mpls l2vpn trigger if-down命令用来使能物理层故障通告功能。

undo mpls l2vpn peer message-merge命令用来去使能物理层故障通告功能。

缺省情况下，物理层故障通告功能未使能。

命令格式

mpls l2vpn trigger if-down

undo mpls l2vpn trigger if-down

参数说明

无

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

该命令只能作用于Ethernet主接口和GE主接口视图，并且接口已经配置了LDP方式或BGP方式的VLL。

当使能物理层故障通告功能后：

  • 当PW侧发生故障时，本端AC接口被shutdown。
  • 当本端AC侧发生故障时，PE将故障通告给对端PE，对端AC接口被shutdown。

当故障恢复时，被shutdown的AC接口被自动undo shutdown，物理状态恢复为UP。

物理层故障通告功能与OAM Mapping功能互斥，即同一接口视图下，不能同时配置mpls l2vpn trigger if-down命令和mpls l2vpn oam-mapping命令。

使用实例

# 使能物理层故障通告功能。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] quit
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mpls l2vc 1.1.1.1 1
[Quidway-GigabitEthernet1/0/0] mpls l2vpn trigger if-down
