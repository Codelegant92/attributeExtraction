dialer bundle

命令功能

dialer bundle命令用来使能共享DCC配置，并设置一个Dialer接口使用的拨号池。

undo dialer bundle命令用来禁止共享DCC配置并将dialer bundle与Dialer接口脱离。

缺省没有使能共享DCC，也没有指定dialer bundle。

命令格式

dialer bundle number

undo dialer bundle

参数说明

number：Dialer pool序号。整数形式，取值范围是1～255。

视图

Dialer接口视图

缺省级别

2：配置级

使用指南

此命令仅用于Dialer接口，并且一个Dialer接口只能对应一个dialer bundle。无论Dialer接口封装PPP或帧中继时，该命令不仅使能共享DCC功能，还能用于配置一个Dialer接口使用的拨号池。

在配置该命令前，必须先执行命令dialer user设置对端用户名。

使用实例

# 在Dialer1上配置接口使用Dialer bundle3，且Dialer bundle3中包含Serial1/0/0接口。

<Quidway> system-view
[Quidway] interface dialer 1
[Quidway-Dialer1] dialer bundle 3
[Quidway-Dialer1] quit
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] dialer bundle-member 3
