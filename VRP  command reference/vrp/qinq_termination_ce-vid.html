qinq termination ce-vid

命令功能

qinq termination ce-vid命令用来配置虚拟以太网子接口对两层Tag报文的终结功能。

undo qinq termination ce-vid命令用来取消虚拟以太网子接口对两层Tag报文的终结功能。

缺省情况，虚拟以太网子接口没有配置对两层Tag报文的终结功能。

命令格式

qinq termination ce-vid low-ce-vid [ to high-ce-vid ] [ vlan-group group-id ]

undo qinq termination

参数说明

low-ce-vid：CE的VLAN ID（内层Tag的值）取值范围的下限，十进制整数形式，取值范围是1～4094。

high-ce-vid：CE的VLAN ID（内层Tag的值）取值范围的上限，十进制整数形式，取值范围是1～4094。

group-id：用户组号，十进制整数形式，取值范围是1～16。

缺省级别

2：配置级

使用指南

当虚拟以太网子接口需要终结带有两层Tag的用户报文时，可以在子接口上配置qinq termination ce-vid命令。用户报文的内层Tag应该在命令中指定的CE的VLAN ID范围之内。

QinQ终结子接口的QoS策略需要配置在VLAN组中，可以对不同VLAN组的应用不同的QoS策略。在qinq termination ce-vid命令中，通过使用vlan-group参数引用相应的VLAN组，可以对匹配内层Tag的流量应用指定的QoS策略。

使用实例

# 在虚拟以太网子接口VE1/0/0.1上配置子接口终结功能，内层Tag分别为：100～200，400～500。

<Quidway> system-view
[Quidway] interface virtual-ethernet 1/0/0
[Quidway-Virtual-Ethernet1/0/0] ve-group 1 l3-access
[Quidway-Virtual-Ethernet1/0/0] mode user-termination
[Quidway-Virtual-Ethernet1/0/0] quit
[Quidway] interface virtual-ethernet 1/0/0.1
[Quidway-Virtual-Ethernet1/0/0.1] qinq termination ce-vid 100 to 200
[Quidway-Virtual-Ethernet1/0/0.1] qinq termination ce-vid 400 to 500
