reset bgp l2vpn

命令功能

reset bgp l2vpn命令用来复位BGP L2VPN的TCP连接。

命令格式

reset bgp l2vpn { as-number | peer-ip-address | all | internal | external }

参数说明

as-number：L2VPN的对等体所在的AS号。整数形式，取值范围是1~65535。

peer-ip-address：L2VPN的对等体的IP地址。

all：复位所有L2VPN的BGP连接。

internal：复位L2VPN相同AS域的BGP连接。

external：复位L2VPN跨域的BGP连接。

使用指南

BGP-L2VPN地址族视图下的配置参数改变了之后，可以使用reset bgp l2vpn命令复位BGP L2VPN的TCP连接，使该BGP重新协商参数、重新发送标签信息和建立会话。

如果BGP的L2VPN应用和其他应用使用同一个TCP连接，那么reset bgp l2vpn命令会导致建立在该TCP连接上的所有应用的BGP邻居关系被复位。

使用实例

# 复位所有的L2VPN BGP连接。

<Quidway> reset bgp l2vpn all
