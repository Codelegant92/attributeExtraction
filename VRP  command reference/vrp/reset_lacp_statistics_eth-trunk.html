reset lacp statistics eth-trunk

命令功能

reset lacp statistics eth-trunk命令用来重置指定接口或Eth-Trunk口的所有成员接口的报文统计计数器。

命令格式

reset lacp statistics eth-trunk [ trunk-id [ interface interface-type interface-number ] ]

参数说明

trunk-id：接口聚合组ID，取值范围是0～63。

interface-type：接口的接口类型。

interface-number：接口的接口编号。

用户视图

2：配置级

使用指南

本命令用来将指定的Eth-Trunk或其成员接口的LACP报文收发统计数目清零。

使用该命令的前提条件是Eth-Trunk存在，并且Eth-Trunk工作模式是静态LACP模式。

如果需要更改当前Eth-Trunk工作模式，可以通过执行mode命令修改。

如果需要重置接口的报文收发数统计，则要求确保此接口加入了相应的聚合组。

使用实例

# 重置Eth-Trunk1下接口GigabitEthernet 1/0/1的报文统计信息。

<Quidway> reset lacp statistics eth-trunk 1 interface gigabitethernet 1/0/1
