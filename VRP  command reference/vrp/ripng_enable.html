ripng enable

命令功能

ripng enable命令用来使能一个接口的RIPng路由协议。

undo ripng命令用来关闭一个接口的RIPng路由协议。

命令格式

ripng process-id enable

undo ripng

参数说明

process-id：RIPng进程号。整数形式，取值范围是1～65535。

视图

接口视图

缺省级别

2：配置级

使用指南

在接口视图下，如果没有配置IPv6地址，则此命令不可执行。在ATM接口上不支持此命令。

使用实例

# 在接口GigabitEthernet1/0/0上使能RIPng100。

[Quidway] ipv6
[Quidway] ripng 100
[Quidway-ripng-100] quit
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 address 2003:567D::1/64
[Quidway-GigabitEthernet1/0/0] ripng 100 enable
