priority（MAC Tunnel视图）

命令功能

priority命令用来配置当前MAC Tunnel的优先级，即隧道的骨干VLAN优先级字段。

undo priority命令用来恢复MAC Tunnel的优先级。

缺省情况下，隧道信任业务实例的优先级。

命令格式

priority { priority | trust instance-priority }

undo priority

参数说明

priority：指定MAC Tunnel的优先级。整数形式，取值范围是0～7，取值越大优先级越高。

trust instance-priority：配置隧道的优先级为信任业务实例的优先级。

MAC Tunnel视图

缺省级别

2：配置级

使用指南

如果MAC Tunnel配置了保护组，保护隧道的优先级将与工作隧道的优先级一致，即工作隧道的优先级变化会导致保护隧道的优先级也同步更新。

本命令只能在工作隧道上使用。

使用实例

# 配置当前隧道的优先级为1。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] priority 1
