anti-attack udp-flood enable

命令功能

anti-attack udp-flood enable命令用来使能UDP-Flood报文攻击防范。

undo anti-attack udp-flood enable命令用来去使能UDP-Flood报文攻击防范。

缺省情况下，使能UDP-Flood报文攻击防范。

命令格式

anti-attack udp-flood [ slot slot-number ] enable

undo anti-attack udp-flood [ slot slot-number ] enable

参数说明

slot slot-number：接口板槽位号。如果不指定槽位号，缺省为所有单板使能UDP-Flood报文攻击防范。

视图

系统视图

缺省级别

2：配置级

使用指南

使能UDP-Flood报文攻击防范之后，设备就会针对UDP-Flood攻击报文进行防攻击检测，如果检测到UDP-Flood攻击报文，将报文直接丢弃。

使用实例

# 在系统视图下使能UDP-Flood报文防攻击命令。

<Quidway> system-view
[Quidway] anti-attack udp-flood slot 6 enable
