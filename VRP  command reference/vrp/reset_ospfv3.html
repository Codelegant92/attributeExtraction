reset ospfv3

命令功能

reset ospfv3命令用来重启OSPFv3进程。

命令格式

reset ospfv3 { process-id | all } [ graceful-restart [ extend-period period ] ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

all：重启所有OSPFv3进程。

graceful-restart：平滑重启OSPFv3进程。

extend-period period：延长现有GR周期的时长。整数形式，取值范围是1～1800，单位是秒。

使用指南

可通过参数来选择进行普通重启还是以Graceful Restart方式重启。

通过配置extend-period参数，可以延长现有GR周期的时间。延长的GR周期时间在进程重启后失效。

配置该命令reset ospfv3 { process-id | all } graceful-restart extend-period period后，将改变下次GR重启的时长。此参数仅一次生效，即在路由器下次GR重启之后，周期还会恢复成graceful-restart [ period period ]所配置的时长。

使用实例

# 以GR方式重启OSPFv3进程1。

<Quidway>reset ospfv3 1 graceful-restart extend-period 200
