cluster-member ftp-timeout

命令功能

cluster-member ftp-timeout命令用来配置成员交换机通过FTP下载配置文件、版本文件或者补丁文件的超时时间。

缺省情况下，成员交换机FTP下载文件的超时时间是300秒。

命令格式

cluster-member ftp-timeout time

参数说明

ftp-timeout time：成员交换机通过FTP下载文件的超时时间。十进制整数形式，取值范围是60～3600，单位是秒。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

当命令交换机批量下发配置文件、版本文件或者补丁文件时，为了防止成员交换机获取文件的过程中没有响应，可以通过本命令设置超时时间。

使用实例

# 配置成员交换机通过FTP下载文件的超时时间是120秒。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] cluster-member ftp-timeout 120
