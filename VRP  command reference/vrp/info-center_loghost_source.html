info-center loghost source

命令功能

info-center loghost source命令用来指定发向日志主机的包的源接口。

undo info-center loghost source命令用来恢复缺省配置。

缺省情况下，从一台路由器发出的日志消息，源接口是发送消息的接口。

命令格式

info-center loghost source interface-type interface-number

undo info-center loghost source

参数说明

interface-type：接口类型。

interface-number：接口编号。

使用指南

如果多台设备向同一个日志主机发送信息，通过对不同的路由器设置不同的源接口，就可以通过源接口地址判断日志消息是从哪台路由器发出的，从而便于对收到的日志消息检索。

使用实例

# 将接口Loopback0的IP地址作为日志消息报文的源地址。

[Quidway] interface loopback 0
[Quidway-LoopBack0] ip address 1.1.1.1 255.255.255.0
[Quidway-LoopBack0] quit
[Quidway] info-center loghost source loopback 0
