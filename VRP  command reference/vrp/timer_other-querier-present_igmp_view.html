timer other-querier-present（IGMP视图）

命令功能

timer other-querier-present命令用来在全局性上设置IGMP查询器存活时间。

undo timer other-querier-present命令用来恢复该配置参数的缺省值。

缺省情况下，其他IGMP查询器的存活时间的计算公式是：其他IGMP查询器的存活时间 ＝ 健壮系数 × IGMP普遍查询消息发送间隔 +（1/2）× 最大查询响应时间。当健壮系数、IGMP普遍查询消息发送间隔和最大查询响应时间都取缺省值时，其他IGMP查询器的存活时间的值为125秒。

命令格式

timer other-querier-present interval

undo timer other-querier-present

参数说明

interval：表示其他IGMP查询器的存活时间。整数形式，取值范围是60～300，单位是秒。

使用指南

该命令和接口视图下的igmp timer other-querier-present命令功能完全相同，差别仅仅是作用范围不同。IGMP视图下的配置具有全局性，对所有接口有效；接口视图下的配置只对当前接口有效。系统优先采用接口视图下的配置。当接口视图下没有进行配置，才采用IGMP视图下的配置。

使用实例

# 在IGMP视图下，配置查询器存在时间的超时值为200秒。

<Quidway> system-view
[Quidway] igmp
[Quidway-igmp] timer other-querier-present 200
