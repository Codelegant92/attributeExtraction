display mbgp routing-table

命令功能

display mbgp routing-table命令用来查看MBGP的路由信息。

命令格式

display mbgp routing-table [ ipv4–address [ mask | mask-length ] ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv4–address：IPv4形式的网络地址。

mask | mask-length：点分十进制掩码/掩码长度。

verbose：显示激活和未激活路由的详细信息。不指定该参数，则只显示激活路由的概要信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过指定参数可以只查看特定的路由信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看MBGP的所有路由信息。

<Quidway> display mbgp routing-table
Route Flags: R - relay, D - download to ib
------------------------------------------------------------------------------
Routing Tables: MBGP
         Destinations : 1        Routes : 1

路由的状态

  • Active：激活路由

  • Invalid：无效路由

  • Inactive：非激活路由

  • NoAdv：不允许发布的路由

  • Adv：可以发布的路由

  • Del：即将被删除的路由

  • GotQ：迭代到下一跳及出接口的路由，或者迭代到隧道的路由

  • WaitQ：没有迭代到下一跳及出接口的路由，或者迭代到隧道的路由

  • Stale.：带Stale标记的路由，在GR中使用
