info-center channel

命令功能

info-center channel命令用来为指定编号的信息通道命名。

undo info-center channel命令用来恢复缺省的通道名。

命令格式

info-center channel channel-number name channel-name

undo info-center channel channel-number

参数说明

channel-number：通道号。取值为0～9，即系统有10个通道。

channel-name：通道名。取值为1～30个字符，首字符不能为数字，不支持“-”、“/”和“\”等字符。

使用指南

用户在命名时注意，通道不可重名。

使用实例

# 将0号通道命名为execconsole。

<Quidway> system-view
[Quidway] info-center channel 0 name execconsole
