display trapbuffer

命令功能

display trapbuffer命令用来显示告警缓冲区记录的信息。

缺省情况下，不带参数表示显示告警缓冲区记录的所有信息。

命令格式

display trapbuffer [ size value ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

size value：显示告警缓冲区内信息的条数。取值范围是1～1024。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果当前告警缓冲区内信息数少于用户指定的value，则显示实际条数的告警信息。

使用实例

# 显示告警缓冲区的信息。

<Quidway> display trapbuffer
Trapping Buffer Confiuration and contents:
enabled
allowed max buffer size : 1024
actual buffer size : 256
channel number : 3 , channel name : trapbuf
dropped messages : 0
overwrote messages : 0
current messages : 0

# 显示告警缓冲区的23条记录信息

<Quidway> display trapbuffer size 23
Trapping Buffer Confiuration and contents:
enabled
allowed max buffer size : 1024
actual buffer size : 256
channel number : 3 , channel name : trapbuf
dropped messages : 0
overwrote messages : 0
current messages : 0
