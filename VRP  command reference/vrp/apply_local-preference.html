apply local-preference

命令功能

apply local-preference命令用来指定BGP路由信息的本地优先级。

undo apply local-preference命令用来取消设置。

命令格式

apply local-preference preference

undo apply local-preference

参数说明

preference：设置的BGP本地优先级。整数形式，取值范围是0～4294967295。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

无

使用实例

# 设置BGP路由信息的本地优先级为130。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply local-preference 130
