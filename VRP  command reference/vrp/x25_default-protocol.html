x25 default-protocol

命令功能

x25 default-protocol命令用来为X.25接口设置X.25协议缺省承载的上层协议。

undo x25 default-protocol命令用来恢复缺省的上层承载协议。

缺省情况下，X.25承载的网络协议为IP。

命令格式

x25 default-protocol ip

undo x25 default-protocol

参数说明

ip：协议类型，只能选择为IP。

使用指南

在X.25 SVC建立过程中，被叫设备会检查X.25呼叫请求分组的呼叫用户数据（CUD，Call User Data）字段，如果为不可辨识的字段则会拒绝该呼叫连接的建立。

但是用户可以为X.25协议指定一个缺省承载的上层协议，当X.25协议接收到一个携带未知CUD的呼叫时，可以将其按照用户指定的缺省上层协议来对待。

使用实例

# 将X.25接口Serial1/0/0的缺省承载上层协议设置为IP协议。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 default-protocol ip
