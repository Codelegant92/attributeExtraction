display atm class

命令功能

display atm class命令用来显示ATM-Class的信息。如果不指定ATM-Class名，则缺省显示所有ATM-Class的信息。

命令格式

display atm class [ atm-class-name ]

参数说明

atm-class-name：ATM-Class名。字符串形式，长度范围是1～16。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行接口的故障诊断等。

使用实例

# 显示设备中名字为“main”的ATM-Class的信息。

<Quidway> display atm class main
ATM CLASS: main
encapsulation aal5snap
map ip inarp
oam frequency 3 up 3 down 3 retry-frequency 3
Service ubr
表1 display atm class命令输出信息描述

项目

描述

ATM CLASS:

ATM-Class的名字。

encapsulation aal5 snap

AAL封装类型为SNAP。

map ip inarp

虚电路IPoA映射的是动态IP地址。相关命令请参考map ip。

oam frequency 3 up 3 down 3 retry-frequency 3

OAM F5 Loopback信元的发送以及重传检测或修改该重传检测的时间参数。相关命令请参考oam frequency。

Service ubr

业务类型为非确定速率（Unspecified Bit Rate）。
