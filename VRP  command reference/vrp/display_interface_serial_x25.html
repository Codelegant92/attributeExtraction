display interface serial（X25）

命令功能

display interface命令用来显示X.25接口的信息。

命令格式

display interface serial [ interface-number ][ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-number：串口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。字符串格式，支持空格，区分大小写。

项目

描述

Link-protocol is X.25 DCE Ietf

该接口当前封装协议为X.25协议，工作在DCE方式下，数据报封装格式为IETF格式。

address is

该X.25接口的X.121地址；若没有，该域显示为none。

State

该X.25接口的当前状态。

Modulo

该X.25接口所发送的数据分组和流量控制分组缺省按照模8的方式编号。

window sizes input:7 output:7, packet sizes input:256 output:256

该X.25接口的流量控制参数，包括接收窗口尺寸、发送窗口尺寸、最大接收分组长度（以字节为单位）、最大发送分组长度（以字节为单位）四项内容。

Channels

该X.25接口的信道范围划分，依次为单向呼入信道区间、双向信道区间、单向呼出信道区间；如果某个区间的两个界定值均为0，表示该区间被禁止使用。

Timers

该X.25接口的各个定时器的时延值，以秒为单位。

New Configuration

该X.25接口的将在下一次重新启动后生效的新配置；如果该配置的某些内容错误，它们将被恢复成缺省值。

Restarts 0 (Restart Collisions 0)

该X.25接口的统计信息：重新启动的次数（其中包括重新启动碰撞的次数）。

Refused Incoming Call

该X.25接口的统计信息：拒绝收到的呼叫的次数。

Failing Outgoing Call

该X.25接口的统计信息：发起呼叫失败的次数。

input/output: RESTART 1/1 ... REJ 0/0

该X.25接口的统计信息：接收到的和发送的各种分组的数目，格式为：接收数目/发送数目。

Invalid Pr

该X.25接口的错误统计信息：接收到的携带错误确认编号的数据分组和流量控制分组的总数。

Invalid Ps

该X.25接口的错误统计信息：接收到的携带错误顺序编号的数据分组的总数。

Unknown

该X.25接口的错误统计信息：接收到的无法解析的分组的总数。

Link-protocol is LAPB

该接口当前封装协议为LAPB协议。

LAPB DCE

该接口LAPB工作在DCE方式。

modulo 8

该接口LAPB所发送的信息帧和监控帧按照模8方式编号。

window-size 7

该接口LAPB的窗口尺寸为7。

max-frame 12056

该接口LAPB所发送的帧的最大长度为12056比特。

retry 10

该接口LAPB信息帧最大重新发送次数为10次。

Timer

该接口LAPB的各个定时器的时延值，以毫秒为单位，T3以秒为单位。

State

该接口LAPB的当前状态。

VS

该接口LAPB的发送变量。

VR

该接口LAPB的接收变量。

Remote VR

该接口LAPB收到的对端的最后一次对信息帧的确认。

IFRAME 147/254 ... DISC 0/0

该接口LAPB所收发各种帧的统计信息，格式为：接收数目/发送数目。

invalid ns

该接口LAPB的错误统计：接收的携带错误顺序编号的信息帧的总数。

invalid nr

该接口LAPB的错误统计：接收的携带错误确认编号的信息帧和监控帧的总数。

link resets

该接口LAPB链路的重新启动次数。
