peer enable（MBGP）

命令功能

peer enable命令用来使能组播对等体（组）。

undo peer enable命令用来禁止与指定对等体（组）交换路由信息。

缺省情况下，未使能组播对等体（组）。

命令格式

peer { group-name | peer-address } enable

undo peer { group-name | peer-address } enable

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

视图

BGP-IPv4组播地址族视图

缺省级别

2：配置级

使用指南

只有对等体（组）使能之后，才能建立与此组播对等体的连接。

使用对等体组可以简化配置。用户可以先创建一个对等体组，然后进行配置。用户可以在此对等体组中增加适用此配置的对等体。

使用实例

# 使能组播对等体 1.1.11.1。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.11.1 as-number 200
[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] peer 1.1.11.1 enable
