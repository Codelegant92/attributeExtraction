reset ipv6 pathmtu

命令功能

reset ipv6 pathmtu all命令用来清除所有PMTU项，无论是通过CLI配置的静态项还是动态项。

reset ipv6 pathmtu dynamic命令只用来清除动态项。

reset ipv6 pathmtu static命令只用来清除CLI配置（静态）项。

命令格式

reset ipv6 pathmtu [ vpn6-instance vpn6-instance-name ]{ all | dynamic | static }

参数说明

vpn6-instance-name：IPv6 VPN实例名称，用来清除指定IPv6 VPN实例的PMTU项。字符串形式，长度范围是1～31，区分大小写。

all：清除缓存中所有PMTU项。

dynamic：清除缓存中所有动态PMTU项。

static：清除缓存中所有静态PMTU项。

使用实例

# 清除所有PMTU项，无论是通过CLI配置的静态项还是动态项。

<Quidway> reset ipv6 pathmtu all

# 只清除CLI配置的静态项。

<Quidway> reset ipv6 pathmtu static

# 只清除动态项。

<Quidway> reset ipv6 pathmtu dynamic

# 清除vpn1的所有PMTU项，无论是通过CLI配置的静态项还是动态项。

<Quidway> reset ipv6 pathmtu vpn6-instance vpn1 all
