display mpls rsvp-te session

命令功能

display mpls rsvp-te session命令用来查看指定RSVP会话的所有信息。

命令格式

display mpls rsvp-te session ingress-lsr-id tunnel-id egress-lsr-id [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

ingress-lsr-id：入节点的LSR ID。

tunnel-id：隧道ID，取值范围是0～65535。

egress-lsr-id：出口设备的LSR ID。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

RSVP会话的信息包括PSB（Path State Block）、RSB（Reservation State Block）、RFSB（ReFresh State Block）和TCSB（Traffic Control State Block）的信息。

使用实例

项目

描述

The PSB Content

路径状态块（Path State Block）的内容

Tunnel Addr

隧道目的地址，格式为X.X.X.X

Exist time

PSB从创建到现在的时间值

Tunnel ExtID

隧道扩展ID（入节点的LSR ID），格式为X.X.X.X

Session ID/ Session Tunnel ID

隧道ID

Ingress LSR ID

入节点的LSR ID，格式为X.X.X.X

Local LSP ID

LSP的本地标识符

Previous Hop

前一跳地址，格式为X.X.X.X

Next Hop

下一跳地址，格式为X.X.X.X

Incoming / Outgoing Interface

入接口或出接口名称

InLabel

入标签值

OutLabel

出标签值

Send Message ID

发送的刷新消息ID

Recv Message ID

接收的刷新消息ID

Session Attribute

会话的属性

SetupPrio

会话建立优先级

HoldPrio

会话保持优先级

SessionAttrib

会话的特性，包括需要的本地保护、带宽保护、节点保护、标签记录和预留风格。

FRR Flag
是否提供FRR保护的标志位：

  • No protection：没有提供FRR保护

  • PLR in use：本地修复节点提供保护

  • MP in use：汇聚节点提供保护

  • Under protecting：提供了FRR保护

Local RRO Flag

是否记录本地路由信息的标志位

FAST_REROUTE Information

快速重路由信息

ERO Information

ERO（Explicit Route Object）信息

L-Type
显式路由类型：

  • Strict：严格显示路由

  • Loose：松散显示路由

ERO-IPAddr

显式路由地址

ERO-PrefixLen

显式路由地址的前缀长度

RRO Information

Record Route Object，记录路由信息

RRO-CType

记录C类路由，包括IPv4/IPv6地址和标签。

RRO-IPAddress

记录路由的IPv4或IPv6地址

RRO-IPPrefixLen

记录路由前缀的长度

SenderTspec Information

发送方的规格信息

Token bucket rate

流量参数，令牌桶速率（单位：kbit/s）

Token bucket size

流量参数，令牌桶大小

Peak data rate

流量参数：峰值数据速率（单位：kbit/s）

Minimum policed unit

流量参数，最小监管单元

Maximum packet size

流量参数，最大报文长度（单位：byte）

CT-BandWidth Information

CT及对应的带宽信息

CT0～CT7 Bandwidth

CT0～CT7的带宽值

Path Message sent to

Path消息的下一跳地址及出接口

Resource Reservation OK

只有设置了资源预留标记时才会显示此字段

LSP Statistics Infomation

LSP的统计信息

SendPacketCounter

发送的报文数

RecvPacketCounter

接收的报文数

SendPathounter

发送的Path消息数

RecvPathCounter

接收的Path消息数

SendResvCounter:

发送的Resv消息数

RecvResvCounter

接收的Resv消息数

The RSB Content

预留状态块（Reservation State Block）的内容

Reservation Style
预留风格：

  • SE：Shared Explicit Style

  • FF：Fixed-Filter Style

  • WF：Wildcard-Filter Style

Reserve Incoming Interface

预留消息的输入接口

Reservation Interface

预留的接口名

Message ID

本地分配的消息ID，用于RSVP摘要刷新特性和可靠性。

Filter Spec Information

过滤规格信息

The filter number

过滤器总数

Label RRO-Label

记录的标签值信息

FlowSpec Information

包括流规格信息

Bandwidth guarantees

预留规格参数，带宽保证

Delay guarantees

预留规格参数，延迟保证

Qos Service is Controlled

提供QoS保证

The RFSB Content

刷新状态块（ReFresh State Block）的内容

Resv Message arrive on

Resv消息的入接口及上一跳地址

The TCSB Content

流量控制状态块（Traffic Control State Block）的内容
