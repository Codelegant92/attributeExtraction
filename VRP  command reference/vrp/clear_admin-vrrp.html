clear admin-vrrp

命令功能

clear admin-vrrp命令用来在系统视图下清除当前设备上遗留的不在位单板上配置的所有管理VRRP绑定关系。

命令格式

clear admin-vrrp binding interface interface vrid virtual-router-id

参数说明

interface interface：管理VRRP所在接口。

vrid virtual-router-id：管理VRRP的ID。取值范围是1～255。

视图

系统视图

缺省级别

2：配置级

使用指南

由于管理VRRP所在单板不在位时业务模块的绑定关系仍然保留，在该管理VRRP配置恢复失败时，可以使用本条命令清除当前遗留的与业务模块的绑定关系。

由于指定的是设备上已不可见的管理VRRP，管理VRRP接口可能是临时口，所以在输入管理VRRP接口时命令行没有做检查，需作有效的输入。

使用实例

# 清除指定的单板已不在位的管理VRRP当前所有绑定关系

<Quidway> system-view
[Quidway] clear admin-vrrp binding interface gigabitethernet1/0/0.1 vrid 1
Info: The action will clear all the current tracking configurations with the administrator VRRP automatically.
