display mpls te cspf tedb

命令功能

display mpls te cspf tedb命令用来查看满足指定条件的基于CSPF的TEDB信息。可以使用正则表达式对输出信息进行过滤。

命令格式

display mpls te cspf tedb { all | area area-id | interface ip-address | network-lsa | node [ router-id ] | srlg srlg-number } [ | { begin | include | exclude } regular-expression ]

参数说明

all：所有TEDB节点信息。

area-id：区域ID。对于OSPF，取值范围是0～4294967295；对于IS-IS，取值为1或2。

ip-address：接口IP地址。

network-lsa：所有的Network-LSA信息。

node：节点信息。

router-id：设备ID，点分十进制格式。

srlg srlg-number：SRLG号。srlg-number为整数形式，取值范围是0～4294967295。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
