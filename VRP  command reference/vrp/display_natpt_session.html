display natpt session

命令功能

display natpt session命令用来查看NAT-PT动态会话信息。

命令格式

display natpt session { all | icmp | tcp | udp } slot slot-id [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：显示所有会话信息。

icmp：显示ICMP会话信息。

tcp：显示TCP会话信息。

udp：显示UDP会话信息。

slot slot-id: 接口板的插槽号，在分布式环境中有效。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
