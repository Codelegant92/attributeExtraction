display mpls rsvp-te sender

命令功能

display mpls rsvp-te sender命令用来查看使能了RSVP-TE的接口的RSVP-TE发送方消息信息。

命令格式

display mpls rsvp-te sender [ interface interface-type interface-number peer-ip-address ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：本地收到Path消息的入接口的类型和接口号。

peer-ip-address：发送Path消息的上一跳邻居的出接口地址。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
