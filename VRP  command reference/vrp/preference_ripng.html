preference (RIPng)

命令功能

preference命令用来指定RIPng路由优先级的值。

undo preference命令用来恢复RIPng路由优先级的缺省值。

RIPng路由优先级的缺省值是100。

命令格式

preference { preference | route-policy route-policy-name } *

undo preference

参数说明

route-policy：指定路由策略，对满足特定条件的路由设置优先级。

route-policy-name：路由策略名称。字符串形式，不支持空格，区分大小写，长度范围是1～40。

prefrence：RIPng路由优先级的值。整数形式，取值范围是1～255。

缺省级别

2：配置级

使用指南

通过路由策略可改变指定路由的优先级。

使用实例

# 设置RIPng路由优先级为120。

[Quidway] ripng 100
[Quidway-ripng-100] preference 120

# 使用路由策略改变路由的优先级。

[Quidway-ripng-100] preference route-policy policy1 60

# 恢复RIPng路由优先级的缺省值。

[Quidway-ripng-100] undo preference
