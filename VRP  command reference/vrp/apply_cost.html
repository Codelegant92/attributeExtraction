apply cost

命令功能

apply cost命令用来设置路由的开销。

undo apply cost命令用来取消设置。

缺省情况下，没有设置路由的开销值。

命令格式

apply cost [ + | - ] cost

undo apply cost

参数说明

+：增加开销值。

-：减少开销值。

cost：指定路由信息的路由开销。整数形式，取值范围0～4294967295。

无

使用实例

# 定义一个设置路由信息属性的应用子句，设置路由开销为120。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply cost 120
