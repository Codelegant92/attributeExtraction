t1 shutdown（CT3接口视图）

命令功能

t1 shutdown命令用来禁止指定T1通道。

undo t1 shutdown命令用来使能指定T1通道。

缺省情况下，T1通道是使能的。

命令格式

t1 t1-number shutdown

undo t1 t1-number shutdown

参数说明

t1-number：T1通道号。整数形式，取值范围是1～28。

视图

CT3接口视图

缺省级别

2：配置级

使用指南

只有当CT3接口工作在CT3工作模式下时，才能配置该命令。

禁止或使能T1通道，对于T1通道形成的串口及T1通过时隙捆绑形成的串口都有效。

对于通道化的串口，可以在串口视图下执行命令shutdown关闭串口。

使用实例

# 禁止T3 1/0/0的第28个T1通道。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] t1 28 shutdown
