reset rrpp statistics

命令功能

reset rrpp statistics命令用于清除RRPP所有报文统计信息。

如果指定ring-id，则清除本交换机所属域中指定环的主、副端口的RRPP报文统计信息；否则清除本交换机所属域中所有环上RRPP端口的报文统计信息。

命令格式

reset rrpp statistics domain domain-id [ ring ring-id ]

参数说明

domain-id：RRPP域的ID。整数形式，取值范围是1～8。

ring-id：RRPP环的ID。整数形式，取值范围是1～64。

使用指南

清除报文统计信息可以重新开始统计报文数量，用来分析网络状况。报文统计信息被清除后，不可恢复。

使用实例

# 清除RRPP域ID为1，环ID为10的报文统计信息。

<Quidway> reset rrpp statistics domain 1 ring 10

# 清除RRPP域1上所有环的报文统计信息。

<Quidway> reset rrpp statistics domain 1
