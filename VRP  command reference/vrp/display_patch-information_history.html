display patch-information history

命令功能

display patch-information history命令用来查看补丁操作的历史记录。

命令格式

display patch-information history [ slave | slot slot-id ]

参数说明

slave：查看备用主控板的补丁操作记录。

slot slot-id：查看指定单板的补丁操作记录。

所有视图

缺省级别

2：配置级

使用指南

display patch-information history命令用来查看最近几次的补丁操作记录，包括各种补丁状态的起始时间和终止时间。系统掉电后，历史记录内容不会丢失。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

表1 display patch-information history命令输出信息描述

项目

描述

Type
补丁类型

  • C：C语言补丁

  • NP：NP类型的补丁

Slot

槽位号

ID

补丁号

State
补丁状态

  • active：激活态

  • deactive：去激活态

  • run：运行态

  • idle：无补丁

From

补丁状态的起始时间

To
补丁状态的结束时间，可取值如下

  • YYYY-MM-DD HH:MM:SS：标识具体的时间

  • now：状态一直持续到现在

  • ****：设备重启(用于记录运行态的补丁遇到设备重启的历史信息；当前补丁状态不会出现此取值)

  • -：处于idle状态的补丁，尚无激活、去激活或者运行的时间记录
