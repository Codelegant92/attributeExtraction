label-retention

命令功能

label-retention命令用来配置标签保持方式。

undo label-retention命令恢复缺省配置。

缺省情况下，标签保持方式为自由标签保持方式（liberal）。

命令格式

label-retention { conservative | liberal }

undo label-retention

参数说明

conservative：保守标签保持方式。这种方式下，LSR只保留路由表中下一跳邻居发布的标签映射。

liberal：自由标签保持方式。这种方式下，不论发来标签映射（Mapping）的LSR是否是所发布映射中的下一跳，都将保留此标签映射。

MPLS-LDP-VPN实例视图、MPLS-LDP视图

缺省级别

2：配置级

使用指南

标签保持方式在建立LSP之前配置，并且，只有在相应视图中使能了MPLS LDP后，该配置才会生效。

说明：
建议用户不要轻易更改标签保持方式的配置，如需更改，请了解以下事项：

  • 当从保守方式改为自由方式时，由于DU模式下的标签重发布特性，将影响标签映射表项；

  • 当从自由方式改为保守方式时，原自由方式的标签映射表项不会被删除。这种情况下，如果改变LSP的配置，LDP中LSP的内容将不可预期。

  • 建立LSP时，推荐采用的组合是：du＋ordered＋liberal。

使用实例

# 配置标签保持方式为自由方式。

<Quidway> system-view
[Quidway] mpls ldp
[Quidway-mpls-ldp] label-retention liberal
