mode user-termination

命令功能

mode user-termination命令用来配置接口的工作模式为用户终结模式。

undo mode user-termination命令用来配置接口的工作模式为非用户终结模式。

默认情况下，接口工作在非用户终结的模式下。

命令格式

mode user-termination

undo mode user-termination

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

接口在默认情况下工作在非用户终结模式下，此时可以进行子接口的vlan-type的相关配置命令，不能使用子接口下的dot1q和QinQ的相关配置命令。

接口工作在用户终结模式下，此时可以使用子接口下的dot1q和QinQ的相关配置命令，不能使用vlan-type的相关配置命令。

在以太网主接口上使用undo mode user-termination命令来配置接口的工作模式为非用户终结模式前，应先删除该主接口下的所有子接口，否则系统会提示配置错误。

使用实例

# 配置接口GE1/0/0为用户终结模式。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mode user-termination
