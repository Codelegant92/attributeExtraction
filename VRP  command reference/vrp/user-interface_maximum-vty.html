user-interface maximum-vty

命令功能

user-interface maximum-vty命令用来设置登录用户的最大数目。

undo user-interface maximum-vty命令用来恢复登录用户最大数目为缺省值。

缺省情况下，telnet、SSH用户最大数目为5个。

命令格式

user-interface maximum-vty number

undo user-interface maximum-vty

参数说明

number：用来设置Telnet、SSH用户最大数目。整数形式，取值范围0～15。

视图

系统视图

缺省级别

3：管理级

使用指南

如果所有登录用户对应的VTY节点索引都小于用户的配置数值，则该配置被接受；否则，报告配置失败。

网管用户除可以通过预留通道登录路由器外，还可以通过没有被占用的VTY登录到路由器。网管专用通道不支持RSA认证方式，只支持密码认证方式。

如果配置的登录用户最大数目为零，则路由器不再接受任何用户通过Telnet、SSH登录路由器。

使用实例

# 配置VTY用户界面最大数目为7个。

<Quidway > system-view
[Quidway] user-interface maximum-vty 7
