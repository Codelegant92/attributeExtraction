pim ipv6 silent

命令功能

pim ipv6 silent命令用来在指定接口上使能PIM Silent功能。

undo pim ipv6 silent命令用来在指定接口上取消PIM Silent功能。

缺省情况下，接口未使能PIM Silent功能。

命令格式

pim ipv6 silent

undo pim ipv6 silent

参数说明

无

使用指南

在接入层上，路由器直连用户的接口上需要使能PIM协议，在该接口上可以建立PIM邻居，处理各类PIM协议报文。此配置同时存在着安全隐患：当恶意主机大量发送模拟的PIM Hello报文时，有可能导致路由器瘫痪。

为了避免这样的情况发生，可以在该接口上执行pim ipv6 silent命令，将接口设置为PIM消极模式。当接口进入消极状态后，将禁止接收和转发任何PIM协议报文，删除该接口上的所有PIM邻居以及PIM状态机，并自动成为DR。同时，该接口上的MLD功能不受影响。

PIM silent仅适用于与用户主机网段直连的接口，且该网段上只能连接一台PIM路由器。

使用实例

# 使能接口GE1/0/0的PIM Silent功能。。

<RouteA> system-view
[RouteA] interface gigabitethernet1/0/0
[RouterA-GigabitEthernet1/0/0] pim ipv6 silent
