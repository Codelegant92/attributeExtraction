hwtacacs-server timer response-timeout

命令功能

hwtacacs-server timer response-timeout命令用来设置HWTACACS服务器应答超时时间。

undo hwtacacs-server timer response-timeout命令用来恢复缺省配置。

缺省情况下，HWTACACS服务器应答超时时间为5秒。

命令格式

hwtacacs-server timer response-timeout value

undo hwtacacs-server timer response-timeout

参数说明

value：HWTACACS服务器应答超时时间值。取值范围是1～300，单位是秒。

视图

HWTACACS服务器模板视图

缺省级别

2：配置级

使用指南

由于HWTACACS是基于TCP实现的，因此，服务器应答超时或TCP超时都可能导致与HWTACACS服务器的连接断开。

使用实例

# 设置HWTACACS服务器应答超时时间为30秒。

<Quidway> system-view
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1] hwtacacs-server timer response-timeout 30
