screen-length

命令功能

screen-length命令用来设置终端屏幕的每屏行数。

undo screen-length命令用来恢复缺省设置。

缺省情况下，终端屏幕的每屏显示24行。

命令格式

screen-length screen-length

undo screen-length

参数说明

screen-length：屏幕分屏显示的行数。整数形式，取值范围是0～512。取值为0时表示关闭分屏功能。

使用指南

通常情况，无需调整终端屏幕的每屏行数。

使用实例

# 设置屏幕的每屏行数为30。

<Quidway> system
[Quidway] user-interface console 0
[Quidway-ui-console0] screen-length 30
