display dhcp snooping bind-table（二层设备）

命令功能

display dhcp snooping bind-table命令用来显示DHCP Snooping绑定表的表项信息。

命令格式

display dhcp snooping bind-table { ip-address ip-address | mac-address mac-address | vlan vlan-id [ interface interface-type interface-number ] | vsi vsi-name | static | dynamic | all } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

vlan vlan-id：VLAN ID值，取值范围1~4094。

interface-type：接口类型。

interface-number：接口号。

static：静态的绑定表项。

dynamic：动态的绑定表项。

all：所有的绑定表项。

vsi vsi-name：虚拟交换实例名。字符串形式，不支持空格，长度范围是1～31。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display dhcp snooping bind-table命令后，设备将显示绑定表的所有信息。

当执行display dhcp snooping bind-table vsi vsi-name命令，设备将显示指定VSI的DHCP Snooping动态绑定表的表项信息。当接口绑定的VSI不存在，则系统会提示配置错误。

使用实例

# 显示DHCP Snooping绑定表的静态表项信息。

<Quidway> display dhcp snooping bind-table static
bind-table:
ifname         p/cvlan      mac-address      ip-address       tp lease
ethernet6/1/0 0001/0000     0024-e801-8f01   192.168.010.002  S  2006-7-1
--            0001/0000     0025-e811-8d23   192.168.010.003  S  2006-7-3

# 显示指定VSI下DHCP Snooping绑定表的表项信息。

<Quidway> display dhcp snooping bind-table vsi vsiname1

bind-table:
ifname       vrf  vsi  p/cvlan   mac-address    ip-address      tp lease
-------------------------------------------------------------------------------
Eth1/0/1     0000 0000 0001/0000 0200-4c4f-4f50 100.001.002.001 D  050728-1132
-------------------------------------------------------------------------------
binditem count:      1                   binditem total count: 1

项目

描述

Ifname

配置绑定的接口名称

vrf

L3VPN的标识，不绑定L3VPN时缺省值是0

vsi

虚拟交换实例的名称

p/cvlan

Port VLAN和Client VLAN的值（Q-in-Q中使用的VLAN）

mac-address

绑定的MAC 地址

ip-address

绑定的IP地址

tp

绑定表项类型，s是静态表项，d是动态表项

lease

租期，当到达这个租期后，分配给用户的IP地址将被回收
