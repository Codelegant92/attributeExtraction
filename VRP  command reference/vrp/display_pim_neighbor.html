display pim neighbor

命令功能

display pim neighbor命令用来查看PIM邻居信息。

命令格式

display pim [ vpn-instance vpn-instance-name | all-instance ] neighbor [ neighbor-address | interface interface-type interface-number | verbose ] * [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

interface interface-type interface-number：表示接口类型和接口号。

neighbor-address：表示PIM邻居的IP地址。点分十进制形式。

verbose：表示PIM邻居的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display pim neighbor命令时：

  • 如果不指定vpn-instance或all-instance，则只显示公网实例的相关信息。

  • 如果指定interface interface-type interface-number，则仅显示该接口下的PIM邻居信息。

  • 如果不指定interface interface-type interface-number，则列举当前路由器上的所有PIM邻居信息。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

VPN-Instance

公网实例

Neighbor

PIM邻居的地址

Interface

PIM邻居所在的接口

Uptime

PIM邻居已经存在的时间

Expiry time

PIM邻居还有多少时间就要超时

DR Priority

DR优先级

Generation ID

PIM邻居状态随机数

Holdtime

PIM邻居生存时间

LAN delay

传递Prune剪枝消息的延迟时间

Override interval

否决Prune剪枝的时间间隔

Neighbor tracking

是否使能邻居跟踪功能

PIM BFD-session

是否建立BFD session

PIM BFD-Session min-tx-interval

PIM BFD检测消息最小发送时间间隔

PIM BFD-session min-rx-interval

PIM BFD检测消息最小接收时间间隔

PIM BFD-session detect-multiple

PIM BFD检测倍数
