mfr window-size

命令功能

mfr window-size命令用来配置多链路帧中继对接收的分片报文进行重组时，使用的滑动窗口算法中窗口所容纳的分片数。

缺省情况下，滑动窗口尺寸等于MFR捆绑的物理接口数。

命令格式

mfr window-size number

undo mfr window-size

参数说明

number：分片数。取值范围是1～16。

视图

MFR接口视图

缺省级别

2：配置级

使用指南

窗口的大小决定重组的速度，但是不是窗口越大越好，需要权衡MFR捆绑链路个数和窗口大小的关系，一般建议使用缺省值。

使用实例

# 设置多链路帧中继捆绑接口MFR4/0/123的滑动窗口尺寸为8。

<Quidway> system-view
[Quidway] interface mfr 4/0/123
[Quidway-MFR4/0/123] mfr window-size 8
