rpr protection timer wtr-timeout

命令功能

rpr protection timer wtr-timeout命令用来配置本地节点某个SPAN的WTR超时时间，即该SPAN进入WTR状态后经过多长时间后进行故障恢复。

undo rpr protection timer wtr-timeout命令用来恢复缺省值。

缺省情况下，WTR的超时时间值为10秒。

命令格式

rpr protection timer wtr-timeout value

undo rpr protection timer wtr-timeout

参数说明

value：超时时间值。整数形式，取值范围是0～1440，单位是秒。缺省值为10秒。

使用指南

如果网络不太稳定或者时延要求比较低，可以把时间设置的长些。

使用实例

# 配置本地节点东向SPAN的WTR超时时间为100秒。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr protection timer wtr-timeout 100
