ssm-mapping（MLD视图）

命令功能

ssm-mapping命令用来设置静态SSM源/组映射规则。

undo ssm-mapping命令用来恢复配置的缺省情况。

缺省情况下，未配置静态SSM源/组映射规则。

命令格式

ssm-mapping ipv6-group-address ipv6-group-mask-length ipv6-source-address

undo ssm-mapping { ipv6-group-address ipv6-group-mask-length [ ipv6-source-address ] | all }

参数说明

ipv6-group-address：指定进行映射的组播组的IPv6地址。十六制形式，取值范围是FF00::～FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。

ipv6-group-mask-length：指定组地址的掩码长度。取值范围是16、32、64和128。

ipv6-source-address：指定组播源的IPv6地址。

all：删除所有的静态SSM源/组映射规则。

使用实例

# 设置静态SSM源/组映射规则：组播组FF35::1/128，映射组播源2000::1。
