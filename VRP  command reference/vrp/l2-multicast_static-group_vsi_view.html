l2-multicast static-group (VSI视图)

命令功能

l2-multicast static-group 命令用来将PW静态加入VSI内二层组播组。

undo l2-multicast static-group命令用来取消PW静态加入VSI内二层组播组。

命令格式

l2-multicast static-group group-address group-address remote-peer ip-address [ negotiation-vc-id vc-id ]

undo l2-multicast static-group group-address { group-address | all } remote-peer ip-address [ negotiation-vc-id vc-id ]

参数说明

group-address group-address：组播IP地址，取值范围是224.0.1.0～239.255.255.255。

remote-peer ip-address：远端对等体的IP地址。

negotiation-vc-id vc-id：虚电路的唯一标识，一般用于两端VSI ID不同但要求互通的情况。参数vc-id不能与本端其他VSI配置的VSI ID相同，也不能与本端其他negotiation-vc-id指定的vc-id相同，即vc-id是没有被用过的虚电路号，整数形式，取值范围是1～4294967295。

使用指南

使用此命令之前，全局使能IGMP Snooping功能。

当使用LDP作为信令协商协议时，使用l2-multicast static-group group-address ip-group-address remote-peer ip-address [ negotiation-vc-id vc-id]命令将PW静态加入VSI内二层组播组。如果事先没有配置remote-peer的IP地址，仍然允许PW静态加入VSI内二层组播组，只是此时不起作用，等到对应的remote-peer的IP地址配置好后，此命令才生效。

使用实例

# 当使用LDP作为信令协商协议时，将VSI vsi0中的PW加入组播组225.1.1.2中。

<Quidway> system-view
[Quidway] vsi vsi0
[Quidway-vsi-vsi0] l2-multicast static-group group-address 225.1.1.2 remote-peer 3.3.3.3
