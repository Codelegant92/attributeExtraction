export route-policy（VPN实例视图）

命令功能

export route-policy命令用于将当前VPN实例与一条出方向路由策略进行关联。并且路由策略只能关联一个，以最后关联的路由策略为准。

undo export route-policy命令用于取消当前VPN实例与出方向路由策略的关联。

缺省情况下，没有配置出向路由策略。

命令格式

export route-policy policy-name

undo export route-policy

参数说明

policy-name：VPN实例的出方向路由策略名。区分大小写。

缺省级别

2：配置级

使用指南

当要求比采用扩展团体属性方式更精确地控制发布的VPN实例路由时，可以使用出方向路由策略。出方向路由策略可以过滤发布的路由信息以及为通过过滤的路由信息设置路由属性。

使用实例

# 对名为vrf1的VPN实例在出方向应用路由策略poly-1。

<Quidway> system-view
[Quidway] ip vpn-instance vrf1
[Quidway-vpn-instance-vrf1] export route-policy poly-1
