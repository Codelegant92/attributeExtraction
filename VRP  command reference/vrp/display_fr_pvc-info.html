display fr pvc-info

命令功能

命令display fr pvc-info用来查看帧中继虚电路的配置情况和统计信息：

  • 不指定任何参数时，显示所有接口DLCI的帧中继基本配置和统计信息。

  • 只指定接口号，不指定DLCI号时，显示所有指定接口下DLCI的帧中继基本配置和统计信息。

  • 既指定接口号，也指定DLCI号时，显示指定接口下指定DLCI的帧中继基本配置和统计信息以及帧中继QoS的配置和统计信息。

命令格式

display fr pvc-info [ interface interface-type interface-number ] [ dlci dlci-number ]

参数说明

interface-type：接口类型。

interface-number：接口编号，既可以指定主接口也可以指定子接口。

dlci dlci-number：DLCI号。取值范围是16～1007。

使用指南

如果配置了帧中继压缩（FRF.9），将显示帧中继压缩的状态信息。
