display mpls te link-administration srlg-information

命令功能

display mpls te link-administration srlg-information命令用来查看接口所属的SRLG。

命令格式

display mpls te link-administration srlg-information [ interface interface-type interface-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface interface-type interface-number：查看指定接口所属的SRLG。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
