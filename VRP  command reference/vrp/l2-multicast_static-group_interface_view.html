l2-multicast static-group(接口视图)

命令功能

l2-multicast static-group命令用来配置接口静态加入组播组。

undo l2-multicast static-group命令用来取消接口静态加入组播组的配置。

命令格式

l2-multicast static-group group-address group-address vlan { { vlan-id1 [to vlan-id2 ] }&<1-10> }

undo l2-multicast static-group group-address {group-address | all }vlan {all | {vlan-id1 [ to vlan-id2] }&<1-10> }

参数说明

group-address group-address：表示组播IP地址。点分十进制格式，取值范围是224.0.2.0～239.255.255.255。

vlan vlan-id1：表示VLAN的编号，要求小于vlan-id2。整数形式，取值范围是1～4094。

vlan vlan-id2：表示VLAN的编号，要求大于vlan-id1。整数形式，取值范围是1～4094。

all：表示所有组播组的配置信息。

使用指南

使用此命令之前，全局使能IGMP Snooping功能。

配置静态二层组播功能时需要配置这条命令。

参数中的VLAN可以是普通VLAN或用户VLAN，并且相应的成员接口已经加入该VLAN。

使用实例

# 配置VLAN内的接口加入IP地址形式的组播组。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] quit
[Quidway] interface gigabitethernet 6/0/0
[Quidway-GigabitEthernet6/0/0] port trunk allow-pass vlan 2
[Quidway-GigabitEthernet6/0/0] l2-multicast static-group ip-group-address 224.1.1.1 vlan 2
