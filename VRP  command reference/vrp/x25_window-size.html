x25 window-size

命令功能

x25 window-size命令用来设置X.25接口接收及发送窗口的尺寸。

undo x25 window-size命令用来恢复X.25接口接收及发送窗口尺寸的缺省值。

命令格式

x25 window-size input-window-size output-window-size

undo x25 window-size

参数说明

input-window-size：接收窗口的尺寸。当X.25的窗口模数为8时，取值范围是1～7；当X.25的窗口模数为128时，取值范围是1～127。缺省情况下，X.25接口接收窗口的尺寸为2。

output-window-size：发送窗口的尺寸。当X.25的窗口模数为8时，取值范围是1～7；当X.25的窗口模数为128时，取值范围是1～127。缺省情况下，X.25接口发送窗口的尺寸为2。

使用指南

in-packets决定了X.25在发送确认信息之前，可以接收的正确数据分组的最大个数。在带宽允许的情况下，窗口尺寸越大，传输效率越高。

out-packets决定了X.25在接收正确的确认信息之前，可以发送的数据分组的最大个数。在带宽允许的情况下，窗口尺寸越大，传输效率越高。

请向接入服务商咨询发送窗口和接收窗口的尺寸，而且，除非网络支持，请不要将这两个参数设置成不一样的。

执行完该命令后，必须重新启动接口，使配置生效。

使用实例

# 将X.25接口Serial1/0/0的接收窗口和发送窗口尺寸均设置为5。

<Quidway>system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0]link-protocol x25
[Quidway-Serial1/0/0] x25 window-size 5 5
[Quidway-Serial1/0/0] shutdown
[Quidway-Serial1/0/0] undo shutdown
