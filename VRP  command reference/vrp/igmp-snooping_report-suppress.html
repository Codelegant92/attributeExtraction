igmp-snooping report-suppress

命令功能

igmp-snooping report-suppress命令用来使能 IGMP Report报文抑制。

undo igmp-snooping report-suppress命令用来去使能 IGMP Report报文抑制。

缺省情况下，去使能 IGMP Report报文抑制。

命令格式

igmp-snooping report-suppress

undo igmp-snooping report-suppress

参数说明

无

视图

VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

使能igmp-snooping report-suppress后，当上游组播路由器发送IGMP查询报文时，无论该组播组内有多少端口，都只需要发送一份IGMP Report报文给上游组播路由器，减少网络侧流量。

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定了VSI；

  • VLAN内包含VE接口。

使用实例

# 在VLAN 10视图下使能 IGMP Report 报文抑制。

<Quidway> system-view
[Quidway] vlan 10
[Quidway-vlan10] igmp-snooping report-suppress

# 在VSI vsi0视图下去使能 IGMP Report 报文抑制。

<Quidway> system-view
[Quidway] vsi vsi0
[Quidway-vsi-vsi0] undo igmp-snooping report-suppress
