code nrzi

命令功能

code nrzi命令用来配置链路的编码方式为nrzi编码。

undo code命令用来取消nrzi编码，使链路的编码方式恢复缺省的nrz编码。

缺省情况下，链路的编码方式为nrz编码。

命令格式

code nrzi

undo code

参数说明

无

视图

串行接口视图

缺省级别

2：配置级

使用指南

两个同步串口相连时，接收端和发送端配置的链路编码方式必须相同，否则收到的数据帧会被解码错误，认为是错误帧而丢弃。

使用实例

# 设置链路编码方式为nrzi编码。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] code nrzi
