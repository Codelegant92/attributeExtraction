apply ip-precedence（Policy-Based-Route视图）

命令功能

apply ip-precedence命令用来设置IP报文优先级。

undo apply ip-precedence命令用来取消报文优先级设置。

命令格式

apply ip-precedence precedence

undo apply ip-precedence

参数说明

precedence：指定策略路由的优先级。整数形式，取值范围是0～7。也可以使用优先级关键字代替优先级的值，它们之间的对应关系如下表所示。

表1 优先级取值与关键字的对应关系
