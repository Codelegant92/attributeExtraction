display rmon event

命令功能

display rmon event命令用来查看RMON事件。

命令格式

display rmon event [ entry-number ]

参数说明

entry-number：事件表行索引。整数形式，取值范围是1～65535。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定事件表行的索引，将显示事件表中所有事件表项。

使用实例

# 查看RMON事件。

<Quidway> display rmon event
Event table 1 owned by Huawei is VALID.
Description: null.
  Will cause log  when triggered, last triggered at 0days 00h:24m:10s.
