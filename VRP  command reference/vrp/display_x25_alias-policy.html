display x25 alias-policy

命令功能

display x25 alias-policy命令用来显示X.25别名表。

命令格式

display x25 alias-policy [ interface serial interface-number ]

参数说明

interface-number：接口编号。

视图

所有视图

缺省级别

1：监控级

使用指南

在检查接口的故障原因时，可执行此命令获取接口的别名信息。用户可以根据这些信息进行故障定位和诊断等。

使用实例

# 查看X.25的别名表。

<Quidway> display x25 alias-policy
Alias for interface Serial2/0/0:
  Alias-1:$20112405$strict
  Alias-2:$20112450left
  Alias-3:20112450$right
表1 display x25 alias-policy命令输出信息描述

项目

描述

Alias for interface Serial2/0/0
接口Serial2/0/0设置了3个别名，它们分别是：

  • $20112405$（匹配模式为严格匹配）。

  • $20112450（匹配模式为左对齐匹配）。

  • 20112450$（匹配模式为右对齐匹配）。
