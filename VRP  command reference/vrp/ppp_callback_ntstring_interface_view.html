ppp callback ntstring（接口视图）

命令功能

ppp callback ntstring命令用来设置从Windows NT Server回呼路由器时所需要的拨号串。

undo ppp callback ntstring命令用来取消设置的回呼拨号串。

命令格式

ppp callback ntstring dial-string

undo ppp callback ntstring [ dial-string ]

参数说明

dial-string：回呼的拨号串，长度为1～64。

视图

接口视图

缺省级别

2：配置级

使用指南

当路由器作为回呼的主呼端呼叫Windows NT Server时，如果服务器需要路由器发送回呼的号码，则需要配置此条命令。

回呼功能一般只在PPP链路的低速接口和Dialer接口上可用。

使用实例

# 设置从Windows NT Server回呼路由器时所需要的拨号串为660068。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] ppp callback ntstring 660068
