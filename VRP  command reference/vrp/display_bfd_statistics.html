display bfd statistics

命令功能

display bfd statistics命令用来查看BFD全局统计信息。

命令格式

display bfd statistics [ slot slot-id ]

参数说明

slot slot-id：指定接口板号。

视图

所有视图

缺省级别

2：配置级

使用指南

在与主控板建立会话的初始阶段，接口板不能正常的向主控板上报统计信息，因此会出现接口板和主控板上统计信息的差异。如果使用以不同的板号为参数的display bfd statistics命令可能发现这种差异，但会话稳定之后统计信息最终会相同。

表1 display bfd statistics命令输出信息描述

项目

描述

Current Display Board Number

当前显示信息所属的板号，“Main”表示显示主控板上的信息

IP Multihop Destination Port

BFD多跳会话报文的目的端口号，为3784或者4784

Total Up/Down Session Number

当前所有处于UP/DOWN状态的BFD会话数目

Current Product Register Type

当前产品的类型

Current Session Number

当前的会话数

Static session

静态会话数

Dynamic session

动态会话数

E_Dynamic session

完全动态会话数

Static_Auto session

静态自协商会话数目

LDP_LSP session

检测 LDP LSP的会话数

STATIC_LSP session

检测静态LSP的会话数

TE_TUNNEL session

检测TE Tunnel的会话数

TE_LSP session

检测TE LSP的会话数

PW session

检测PW的会话数

IP session

BFD for IP的会话数

VSI_PW session

BFD for VSI_PW的会话数

PAF/LCS Name

许可证文件中的项目名称

BFD_CFG_NUM

路由器全局可以配置的BFD最大数

BFD_IF_NUM

每个接口最多绑定的BFD会话个数

BFD_SESSION_NUM

路由器全局可以建立BFD会话的最大数

BFD_IO_SESSION_NUM

路由器每接口板上可以建立的BFD会话的最大数

BFD_PER_TUNNEL_CFG_NUM

每个TUNNEL上可以配置的BFD最大数

IO Board Current Created Session Statistics Information : (slot/number):

接口板上创建会话的个数

Current Total Used Discriminator Num

当前配置的标识符总数

IO Board Reserved Sessions Number Information : (slot/number)

保留的绑定BFD会话的接口板数目

BFD HA Information

BFD HA 信息

Core Current HA Status
CORE中HA当前的状态，可能取值有：

  • Normal ：正常

  • Batch Backup Going：正在备份

  • Slave Not Ready：备板还没有准备好

  • Smooth Going ：正在进行平滑处理

Shell Current HA Status
SHELL中HA当前的状态，可能取值有：

  • Normal ：正常

  • Batch Backup Going：正在备份

  • Slave Not Ready：备板还没有准备好

  • Smooth Going ：正在进行平滑处理

Ability of auto creating BFD session on egress

是否使能创建完全动态会话能力

Period of LSP Ping

动态BFD for LSP/PW会话发送LSP Ping报文的时间间隔

System Session Delay Up Timer
显示当前延迟UP定时器的状态
  • OFF：系统处于正常状态
  • ×s：表明X秒后系统恢复正常，BFD会话可以UP
