t1 shutdown（CPOS接口视图）

命令功能

t1 shutdown命令用来禁止T1通道。

undo t1 shutdown命令用来使能T1通道。

缺省情况下，T1通道是使能的。

命令格式

t1 t1-number shutdown

undo t1 t1-number shutdown

参数说明

t1-number：CPOS的T1通道号。整数形式，取值范围是1～84。

视图

CPOS接口视图

缺省级别

2：配置级

使用指南

禁止T1通道后，如果有捆绑形成的串口，则串口也被禁止。

使用实例

# 禁止T1通道1。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] t1 1 shutdown
