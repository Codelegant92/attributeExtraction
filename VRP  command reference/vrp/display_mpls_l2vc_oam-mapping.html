display mpls l2vc oam-mapping

命令功能

display mpls l2vc oam-mapping命令用来查看以LDP为信令的L2VPN的AC状态和PSN侧状态的映射(OAM Mapping)表信息。

命令格式

display mpls l2vc oam-mapping [ interface interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type interface-number：接口类型和编号。用于显示指定VC(LDP信令)接口的OAM Mapping信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display mpls l2vc oam-mapping命令后：

  • 如果指定接口名称，则显示该接口下详细的OAM Mapping信息。

  • 如果没有指定接口名称，则只显示所有接口的基本OAM Mapping信息。

使用实例

# 查看所有以LDP为信令的VC接口的OAM Mapping信息。

<Quidway> display mpls l2vc oam-mapping
Interface                    Active VC-ID      VC-status AC-OAM  BFD-for-PW
Pos1/0/0                     Yes    100        Main      Enable  Enable
表1 display mpls l2vc oam-mapping命令输出信息描述

项目

描述

Interface

AC接口类型和编号。

Active
该标识仅在主、备PW同时存在时才有意义。表示PW是否为活动状态：

  • Yes：表明该PW为活动状态

  • No：表明该PW为非活动状态

VC-ID

本端VC的ID值。

VC-status
本端PW是主用PW还是备用PW。

  • Main：主用PW。

  • Secondary：备用PW。

AC-OAM
表示本PE设备上是否使用OAM通告AC故障给远端PE：

  • Enable：表明本PE使用OAM通告AC故障给远端PE

  • Disable：表明本PE不使用OAM通告AC故障给远端PE

BFD-for-PW
表示本PE设备上是否配置了使用BFD检测PW状态故障的功能。

  • Enable：表明本PE使用BFD检测PW状态

  • Disable：表明本PE不使用BFD检测PW状态

项目

描述

AC OAM Info

表示以下5行内容是AC OAM相关信息。

ACFD Index

AC转发信息索引。

Notify
表示本地PE上是否配置了将PSN故障通告给本地CE的功能：

  • Enable：配置了将PSN故障通告给本地CE的功能

  • Disable：未配置将PSN故障通告给本地CE的功能

Detect
表示本地PE上是否配置了检测本地AC故障：

  • Enable：配置了检测本地AC故障

  • Disable：未配置检测本地AC故障

AC OAM State
本地AC故障检测的状态。

  • Up：未使能AC故障检测，或使能了AC故障并且未检测到AC故障。

  • Down：使能了AC故障并且检测到AC故障。

OAM-mapping
表示本地PE上是否配置OAM Mapping功能：

  • Enable：本地PE上配置了OAM Mapping功能

  • Disable：本地PE上未配置OAM Mapping功能

PSN info

表示以下内容是PW的相关信息

VC-ID

VC的标识符

VC status
表示该PW为主用PW还是备用PW

  • Primary：该PW为主用PW

  • Secondary：该PW为备份PW

Active state
该标识仅在主、备PW同时存在时才有意义。表示该PW是否为活动状态：

  • Active：该PW处于活动状态

  • InActive：该PW处于非活动状态

当一个AC接口上只配置了一个PW时，该PW为活动状态。如果一个AC接口上配置了主备PW，则正常情况下主PW为活动状态；备份PW为非活动状态。当主PW出现故障时，备份PW变为活动状态，主PW为非活动状态。

Link State
AC接口的链路状态：

  • Up：该接口处于物理可用状态

  • Down：该接口处于不可用状态

BFD for PW

本端PE设备上是否配置了使用BFD检测PW状态故障的功能。

BFDSessionIndex

BFD会话的索引值，该内容只有使能了BFD检测PW状态的功能后才可见。

BFD State
BFD会话的状态：

  • Up：BFD会话处于Up状态

  • Down：BFD会话处于Down状态

  • Admin down：BFD会话处于管理Down状态

该内容只有使能了BFD检测PW状态的功能后才可见。

BFD for LSP

表示该PW是否使用BFD检测隧道。值为0表示没有使用BFD检测隧道。值为非0表示使用了BFD检测隧道

TunnelNum

被BFD检测的隧道的数目

PSN State

因没有使用BFD检测隧道，故隧道的状态默认为Up。当PW使用没有使能BFD的隧道，PSN State也为Up
