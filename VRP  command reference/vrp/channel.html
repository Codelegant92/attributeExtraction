channel

命令功能

channel命令用来将一个串口X.25接口或XOT通道加入至当前搜索群（hunt group）。

undo channel命令用来将指定的接口或XOT通道从当前搜索群内删除。

命令格式

channel { interface serial interface-number | xot ip-address }

undo channel { interface serial interface-number | xot ip-address }

参数说明

interface-number：接口编号，三维形式：槽号/卡号/接口号。

ip-address：对端XOT主机的IP地址。

使用指南

一个接口最多可以同时属于6个搜索群。

使用实例

# 将串口serial1/0/0加入到搜索群hg1之中。

<Quidway> system-view
[Quidway] x25 hunt-group hg1 round-robin
[Quidway-hg-hg1] channel interface serial1/0/0
