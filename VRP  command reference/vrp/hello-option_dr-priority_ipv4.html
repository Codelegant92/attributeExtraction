hello-option dr-priority（IPv4）

命令功能

hello-option dr-priority命令用来配置路由器竞选成为DR（Designated Router）的优先级。

undo hello-option dr-priority命令用来恢复配置参数的缺省值。

缺省情况下，路由器竞选成为DR的优先级是1。

命令格式

hello-option dr-priority priority

undo hello-option dr-priority

参数说明

priority：表示路由器竞选成为DR的优先级，数值越大，优先级越高。整数形式，取值范围是0～4294967295。

缺省级别

使用指南

该命令仅对PIM-SM有效。

在PIM-SM中，共享网段需要竟选DR，负责本地组播源注册和接收者加入。

竞选DR的依据是优先级和IP地址。路由器之间通过互相发送携带优先级参数的Hello消息来竞选成为DR，具有最高优先级的路由器将成为DR，优先级相同的情况下，IP地址大的作为DR。

假如网络中至少有一台路由器不支持在Hello报文中携带优先级参数，则根据各路由器的IP地址来选择DR，具有最大IP地址的路由器将成为DR。

使用实例

# 在公网实例中，配置路由器竞选成为DR的优先级为3。

<Quidway> system-view
[Quidway] pim
[Quidway-pim] hello-option dr-priority 3
