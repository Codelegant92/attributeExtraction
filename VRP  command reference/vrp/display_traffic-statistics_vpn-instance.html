display traffic-statistics vpn-instance

命令功能

display traffic-statistics vpn-instance命令用来显示指定L3VPN的统计信息。

命令格式

display traffic-statistics vpn-instance vpn-instance-name

参数说明

vpn-instance-name：VPN实例的名称。区分大小写。
