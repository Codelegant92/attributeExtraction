cfm mac-tunnel mip trace enable

命令功能

cfm mac-tunnel mip trace enable命令用来使能当前设备的MAC tunnel中间节点Trace回复功能。

undo cfm mac-tunnel mip trace enable命令用来禁止当前设备的MAC tunnel中间节点Trace回复功能。

缺省情况下，当前设备的MAC tunnel中间节点Trace回复功能处于禁止状态。

命令格式

cfm mac-tunnel mip trace enable

undo cfm mac-tunnel mip trace enable

参数说明

无

使用指南

在MAC tunnel中，只能Trace对端端点，中间节点是否回应LTM消息可由本命令进行配置。如果当前设备是某个MAC tunnel的中间节点，并且使能了MAC tunnel中间节点Trace回复功能，而且该设备连接隧道的两个接口均已配置允许该隧道的骨干VLAN内的报文通过，那么本设备在收到该骨干VLAN内的LTM消息时回复LTR消息。

本命令对MAC tunnel两端MEP没有影响。

使用cfm enable（系统视图）命令全局使能设备的CFM功能后才能使用本命令。

使用实例

# 使能MAC tunnel中间节点的Trace回复功能。

<Quidway> system-view
[Quidway] cfm enable
[Quidway] cfm mac-tunnel mip trace enable
