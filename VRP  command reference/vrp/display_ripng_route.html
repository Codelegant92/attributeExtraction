display ripng route

命令功能

display ripng route命令用来显示所有从其它路由器学来的激活和非激活RIPng路由信息，以及与每条路由相关的不同定时器的值。

命令格式

display ripng process-id route [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：显示指定RIPng进程路由信息。整数形式，取值范围是1～65535。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

该命令有助于调试和检测协议。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
