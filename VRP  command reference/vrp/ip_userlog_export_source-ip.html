ip userlog export source-ip

命令功能

ip userlog export source-ip命令用来配置输出的用户日志报文的源IP地址。

undo ip userlog export source-ip命令用来恢复缺省设置。

缺省情况下，日志输出报文的源IP地址为路由器输出用户日志报文的接口的IP地址。

命令格式

ip userlog nat export source-ip ip-address

undo ip userlog nat export source-ip

参数说明

nat：NAT日志功能。

ip-address：输出的日志报文中的源IP地址。

使用指南

在用户日志服务器端，通过识别日志的源IP地址，可以迅速定位日志信息的来源。建议配置Loopback地址或Router ID作为日志报文的源IP地址。

使用实例

# 将169.254.1.2作为输出到用户日志服务器上的日志报文的源IP地址。

<Quidway> system-view
[Quidway] ip userlog nat export source-ip 169.254.1.2
