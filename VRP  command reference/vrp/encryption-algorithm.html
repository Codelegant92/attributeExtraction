encryption-algorithm

命令功能

encryption-algorithm命令用来指定一个IKE提议使用的加密算法。

undo encryption-algorithm命令用来设置加密算法为缺省值。

缺省情况下使用CBC模式的56 bits DES加密算法。

命令格式

encryption-algorithm { des-cbc | 3des-cbc | aes-cbc }

undo encryption-algorithm

参数说明

des-cbc：指定IKE安全提议采用的加密算法为CBC模式的DES算法。

3des-cbc：指定IKE安全提议采用的加密算法为CBC模式的168 bits的3DES算法。

aes-cbc：指定IKE安全提议采用的加密算法为AES算法（Advanced Encryption Standard）（该参数只使用于NE16E、NE08E、NE05产品并由加密卡硬件提供）。

使用指南

对于保密及安全性要求非常高的地方采用3DES算法可以满足需要，但3DES加密速度比较慢；对于普通的安全要求，DES算法则可以满足需要。

使用实例

# 指定IKE提议10的加密算法为CBC模式的56 bits DES。

<Quidway> system-view
[Quidway] ike proposal 10
[Quidway-ike-proposal-10] encryption-algorithm des-cbc
