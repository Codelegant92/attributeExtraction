ftp-operation

命令功能

ftp-operation命令用来配置NQA测试中FTP测试例的操作类型。

命令格式

ftp-operation { get | put }

参数说明

get：下载指定文件到客户端，并记录下载速度。

put：将本地文件或构造的数据包上载到服务器，并记录上载速度。

缺省情况下，FTP测试的操作类型是get。

缺省级别

2：配置级

使用指南

通过FTP服务器的地址、用户名、密码与其建立控制连接，并记录下连接建立的时间。

只能对FTP类型的测试例配置FTP操作类型。

不能更改正在运行的测试例的操作类型。

使用实例

# 设置测试例user test进行FTP下载速度测试。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type ftp
[Quidway-nqa-user-test] ftp-operation get
