display member-reboot-stat

命令功能

display member-reboot-stat命令用来查看成员交换机重启的状态。

命令格式

display member-reboot-stat

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

当使用cluster-member reboot命令配置指定的成员交换机进行重启后，可以使用本命令用来查看成员交换机重启的状态。

指定成员交换机的成员编号时不能超过集群支持的最大编号。

使用实例

# 查看集群中成员交换机重启的状态。

<Quidway> display member-reboot-stat 
The result of member switchs rebooting:
------------------------------------------------------------------------
SN    Device              MacAddress        IPAddress        Result
------------------------------------------------------------------------
1     NE40E               0002-0002-0002    10.0.0.2         Succeed
2     NE40E               0003-0003-0003    10.0.0.3         Succeed
------------------------------------------------------------------------

该显示信息主要内容解释见表1。

表1 display member-reboot-stat命令输出信息描述

项目

描述

SN
成员编号。

  • “0”表示命令交换机。

  • 其它数字表示成员交换机。

Device

设备的类型。

MAC Address

设备的MAC地址。

IpAddress

设备的集群IP地址。

Result
成员交换机重启的结果。

  • Waiting

  • Runing

  • Timeout

  • Failed

  • Succeed
