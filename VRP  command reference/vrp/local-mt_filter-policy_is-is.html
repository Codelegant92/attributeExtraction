local-mt filter-policy (IS-IS)

命令功能

local-mt filter-policy命令用来配置IS-IS本地组播拓扑特性的策略。

undo local-mt filter-policy命令用来取消该策略。

缺省情况下，不配置本地组播拓扑特性的策略。

命令格式

local-mt filter-policy { acl acl-number | ip-prefix ip-prefix-name }

undo local-mt filter-policy

参数说明

acl acl-number：基本ACL的编号。

ip-prefix ip-prefix-name：IP地址前缀列表名称。

视图

IS-IS视图

缺省级别

2：配置级

使用指南

通过配置本地组播拓扑的策略，可以只允许到组播源地址的路由加入到MIGP路由表中，减小对路由器CPU、内存的占用。

使用实例

# 配置IS-IS本地组播拓扑特性的策略。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] local-mt filter-policy acl 2000
