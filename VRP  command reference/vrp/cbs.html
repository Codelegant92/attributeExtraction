cbs

命令功能

cbs命令用来设置帧中继虚电路的承诺突发流量（Committed burst size）。

undo cbs命令用来恢复缺省值。

命令格式

cbs [ inbound | outbound ] cbs

undo cbs [ inbound | outbound ]

参数说明

inbound：设置报文入方向的承诺突发流量，本参数仅当接口使能帧中继流量监管时有效。

outbound：设置报文出方向的承诺突发流量，本参数仅当接口使能帧中继流量整形时有效。

cbs：承诺突发流量，单位是bit。取值范围是300～16000000。缺省值为56000 bits。

缺省级别

2：配置级

使用指南

如果配置时不指定报文方向，则参数同时设置在入方向和出方向。

承诺突发流量是帧中继网络在一个Tc的时间间隔内，承诺可以发送的报文流量。当网络没有发生拥塞时，帧中继网络保证这部分流量可以被成功发送。

使用实例

# 设置名为test1的帧中继class的承诺突发流量为64000 bits。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] cbs 64000
