reset vrrp-group statistics

命令功能

reset vrrp-group statistics命令用来重置管理组的统计信息。

命令格式

reset vrrp-group [ vrrp-group-id ] statistics

参数说明

vrrp-group-id：VRRP管理组ID，取值范围是1～16。

视图

用户视图

缺省级别

2：配置级

使用指南

该命令用来清空VRRP管理组的统计信息。该命令将删除历史统计信息，如果需要该信息，请慎用该命令。

使用实例

# 重置管理组1的统计信息

<Quidway> reset vrrp-group 1 statistics
