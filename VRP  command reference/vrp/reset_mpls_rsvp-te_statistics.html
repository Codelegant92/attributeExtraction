reset mpls rsvp-te statistics

命令功能

reset mpls rsvp-te statistics命令用来清除RSVP-TE的运行统计信息。

命令格式

reset mpls rsvp-te statistics { global | interface [ interface-type interface-number ] }

参数说明

global：清除全局RSVP-TE运行统计信息。

interface：清除接口上的RSVP-TE运行统计信息。

interface-type interface-number：运行RSVP-TE的接口的类型和编号。如果不指定接口类型和编号，将清除所有RSVP-TE接口上的统计信息。
