fr inarp

命令功能

fr inarp命令用来允许帧中继逆向地址解析功能。

undo fr inarp命令用来禁止该项功能。

缺省情况下，允许逆向地址解析。

命令格式

fr inarp [ ip [ dlci-number ] ]

undo fr inarp [ ip [ dlci-number ] ]

参数说明

ip：对IP网络协议进行逆向地址解析。

dlci-number：对指定虚电路进行逆向地址解析。取值范围是16～1007。

视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

如果不指定DLCI参数，则对所有PVC的逆向地址解析功能生效。

使用实例

# 在帧中继接口Serial1/0/0:0上的所有PVC都允许进行逆向地址解析。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr
[Quidway-Serial1/0/0:0] fr inarp
