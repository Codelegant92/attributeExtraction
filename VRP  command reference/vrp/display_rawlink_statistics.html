display rawlink statistics

命令功能

display rawlink statistics命令用来查看RawLink统计信息。

命令格式

display rawlink statistics [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display rawlink statistics命令用来查看RawLink统计信息，包括：

  • RawLink输入统计
  • RawLink输出统计
  • IFNET指针是否为空
  • 因没有对应的协议控制块（PCB）而丢失的分组数
  • 数据是否是通过多播接收的
  • 是否加入了多播组

使用实例

# 显示RawLink统计信息。

<Quidway> display rawlink statistics

full sock and nothing to be freed
