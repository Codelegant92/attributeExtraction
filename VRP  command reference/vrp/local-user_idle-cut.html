local-user idle-cut

命令功能

local-user idle-cut命令用来使能本地用户的闲置切断功能。

undo local-user idle-cut命令用来禁止本地用户的闲置切断功能。

缺省情况下，对本地用户进行闲置切断功能。

命令格式

local-user user-name idle-cut

undo local-user user-name idle-cut

参数说明

user-name：用户名。字符串形式，长度范围是1～64，支持通配符“*”。

视图

AAA视图

缺省级别

2：配置级

使用指南

AAA用户的闲置切断时间取值范围是有优先级的，具体顺序为：服务器下发的优先级最高，AAA域配置的优先级次之，VTY配置的优先级最低。

使用实例

# 使能本地用户hello@163.net的闲置切断功能。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] local-user hello@163.net idle-cut
