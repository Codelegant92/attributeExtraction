display radius-server configuration

命令功能

display radius-server configuration命令用来查看RADIUS服务器的配置信息。

命令格式

display radius-server configuration [ template template-name ]

参数说明

template-name：RADIUS服务器模板名，字符串形式，长度范围是1～32。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定使用的服务器模板名称，将显示所有RADIUS服务器的配置信息。

使用实例

# 查看RADIUS服务器的配置信息。

<Quidway> display radius-server configuration
------------------------------------------------------------------
  Server-template-name             :  test1
  Protocol-version                 :  standard
  Traffic-unit                     :  B
  Shared-secret-key                :  huawei
  Timeout-interval(in second)      :  5
  Primary-authentication-server    :  0.0.0.0:0:LoopBack0
  Primary-accounting-server        :  0.0.0.0:0:LoopBack0
  Secondary-authentication-server  :  0.0.0.0:0:LoopBack0
  Secondary-accounting-server      :  0.0.0.0:0:LoopBack0
  Retransmission                   :  3
  EndPacketSendTime                :  0
  Domain-included                  :  YES
  -------------------------------------------------------------------
表1 display radius-server configuration命令显示信息说明

项目

描述

Server-template-name

RADIUS服务器的模板名称

Protocol-version

RADIUS协议的版本

Traffic-unit

RADIUS服务器使用的流量单位

Shared-secret-key

RADIUS服务器的共享密钥

Timeout-interval(in second)

RADIUS服务器应答超时时间。

Primary-authentication-server

RADIUS主认证服务器IP地址及端口号

Primary-accounting-server

RADIUS主计费服务器IP地址及端口号

Secondary-authentication-server

RADIUS 次认证服务器IP地址及端口号

Secondary-accounting-server

RADIUS次计费服务器IP地址及端口号

Retransmission

RADIUS服务器的重传次数

EndPacketSendTime

计费结束报文的重传次数

Domain-included

RADIUS服务器的用户名中是否带域名
