atm class

命令功能

atm class命令用来创建一个ATM-Class并进入ATM-Class视图。

undo atm class命令用来删除指定的ATM-Class。

命令格式

atm class atm-class-name

undo atm class atm-class-name

参数说明

atm-class-name：ATM-Class名。字符串形式，不支持空格，区分大小写，长度为1～16。

视图

系统视图

缺省级别

2：配置级

使用指南

一个ATM-Class是一组预先配置的参数，可以将ATM-Class应用于ATM接口或PVC。

使用实例

# 创建一个名为“main”的ATM-Class。

<Quidway> system-view
[Quidway] atm class main
