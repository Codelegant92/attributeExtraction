local-user change-password

命令功能

local-user change-password命令用来使本地用户修改自己的密码。

命令格式

local-user change-password

参数说明

无

视图

用户视图

缺省级别

0：参观级

使用指南

本地认证通过的用户才可以执行该命令修改自己的密码。即管理员已经使用local-user password命令创建了该本地用户，登录用户通过了本地认证成功登录之后，才可以执行该命令修改自己的密码。

通过本地认证的用户执行该命令成功修改自己的密码后，下次登录需要输入新密码才可以通过认证。

该命令本身是修改本地用户密码，不直接保存配置，但会以local-user password命令的形式保存修改结果。

使用实例

# 本地用户修改自己的密码。

<Quidway> local-user change-password
Please enter old password: old-password
Please enter new password: new-password
Please confirm new password: new-password
Info: The password is changed successfully.
