tracert-livetime

命令功能

tracert-livetime命令用来配置NQA测试Traceroute测试例的生存时间。

undo tracert-livetime命令用来恢复缺省配置。

命令格式

tracert-livetime first-ttl first-ttl max-ttl max-ttl

undo tracert-livetime

参数说明

first-ttl first-ttl：设置报文的初始TTL值。取值范围是1～255。

max-ttl max-ttl：设置报文的最大TTL值。取值范围是1～255。max-ttl的值不能小于first-ttl。缺省情况下，报文的最大TTL值为30。

缺省级别

2：配置级

使用指南

只能对Traceroute类型、MTrace类型、PWE3Trace类型、LSPTrace类型的测试例配置tracert-livetime命令。
说明：

在MTrace测试例中，first-ttl值必须为1。

不能改变正在运行的测试例的此项配置。

使用实例

# 设置测试例user test的生存时间，初始TTL为5，最大TTL为20。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type trace
[Quidway-nqa-user-test] tracert-livetime first-ttl 5 max-ttl 20

# 设置测试例user test的生存时间为缺省值。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type trace
[Quidway-nqa-user-test] undo tracert-livetime
