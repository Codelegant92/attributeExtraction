format

命令功能

format命令用来格式化存储设备。

命令格式

format device-name

参数说明

device-name：设备名。可以是flash、hda1或者CF卡。

视图

用户视图

缺省级别

3：管理级

使用指南

格式化操作将导致指定的存储设备上所有的文件丢失，并且不可恢复。

使用实例

#格式化flash。

<Quidway> format flash:
All data on flash: will be lost, proceed with format ? [Y/N]:y
%Format flash: completed.
