ipv6 fib-loadbalance-type hash-based

命令功能

ipv6 fib-loadbalance-type hash-based命令用来确保从指定源发送到指定目的地的报文使用相同的路由。

undo ipv6 fib-loadbalance-type hash-based命令用来恢复默认的负载分担。

命令格式

ipv6 fib-loadbalance-type hash-based

undo ipv6 fib-loadbalance-type hash-based

参数说明

无

说明：

这个命令只在FIB缓存失效时有效。
