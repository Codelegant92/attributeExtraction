set flow-stat interval

命令功能

set flow-stat interval命令用来设定接口流量统计的时间间隔。

undo set flow-stat interval命令用来恢复接口流量统计的时间间隔为默认值。

缺省情况下，接口流量统计时间间隔为300秒。

命令格式

set flow-stat interval interval

undo set flow-stat interval

参数说明

interval：接口流量统计时间间隔。整数形式，取值范围为10～600，且必须为10的整数倍。单位是秒。

视图

系统视图、接口视图

缺省级别

2：配置级

使用指南

在系统视图下配置的接口流量统计时间间隔对接口下的时间间隔为缺省值的所有接口都生效。

在接口视图下配置的接口流量统计时间间隔只对本接口生效，不影响其他接口。

在接口视图下配置的时间间隔的优先级高于在系统视图下配置的时间间隔。

说明：

  • 主控板上用于连接网管站和多框互连的以太网接口不能配置接口流量统计功能。

  • 新的时间间隔将在原时间间隔超时后生效，流量显示将在新的时间间隔生效后第二个周期更新。

使用实例

# 在系统视图下配置接口的流量统计时间间隔为200秒。

<Quidway> system-view
[Quidway] set flow-stat interval 200

# 配置接口VLANIF1的流量统计时间间隔为120秒。

[Quidway] interface vlanif 1
[Quidway-Vlanif1] set flow-stat interval 120
