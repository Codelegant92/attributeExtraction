display cpu-packet statistics interface vlan

命令功能

display cpu-packet statistics interface vlan命令用来查看指定的接口指定VLAN下所有上送到CPU的报文的统计信息。

命令格式

display cpu-packet statistics interface interface-type interface-number vlan vlan-id

参数说明

interface-type interface-number：接口类型和接口编号。

vlan-id：VLAN ID。整数形式，取值范围是1～4094。

使用指南

使用display cpu-packet statistics interface vlan命令查看指定接口的指定VLAN下所有上送到CPU的报文统计信息之和。

使用实例

# 查看接口GigabitEthernet 1/0/0下VLAN 1的所有上送到CPU的报文的统计信息。

<Quidway> display cpu-packet statistics interface gigabitethernet 1/0/0 vlan 1
GigabitEthernet1/0/0, VLAN: 1, 100 packets, 2310 bytes
