display mpls ldp adjacency

命令功能

display mpls ldp adjacency命令用来查看LDP邻接体的信息。

命令格式

display mpls ldp adjacency [ vpn-instance vpn-instance-name ] [ interface interface-type interface-number | remote ] [ peer peer-id ] [ verbose ]

display mpls ldp adjacency [ vpn-instance vpn-instance-name ] [ interface interface-type interface-number | remote ] [ peer peer-id ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：显示指定VPN实例的LDP邻接体信息。字符串形式，不支持空格，区分大小写，长度范围是1～31。

interface interface-type interface-number ：显示指定接口名称的LDP邻接体信息。

remote peer peer-id：显示指定远端对等体的LDP邻接体信息。

peer peer-id：显示指定对等体的LDP邻接体信息。

verbose：显示LDP邻接体的详细信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行此命令可以查看邻接体相关的实时信息，例如收到的hello消息的个数。

使用实例

# 查看LDP邻接体的信息。

<Quidway> display mpls ldp adjacency
 LDP Adjacency Information
 ------------------------------------------------------------------------------
 SN   PeerID         VrfID  CreateDate  CreateTime  AdjType
 ------------------------------------------------------------------------------
 1    2.2.2.9        0      2008-09-02  14:50:12    L
 2    3.3.3.9        0      2008-09-02  14:51:54    R
 ------------------------------------------------------------------------------
 TOTAL: 2 Record(s) found.
 R: Remote Adjacency.
 L: Local Adjacency.
表1 display mpls ldp adjacency命令输出信息描述

项目

描述

SN

序列号

PeerID

LDP对等体的LSR ID

VrfID

VPN实例ID

CreateDate

LDP邻接体的创建日期

CreateTime

LDP邻接体的创建时间

AdjType

LDP邻接体的类型：

  • L：本地LDP邻接体

  • R：远端LDP邻接体

# 查看远端对等体的LSR ID为3.3.3.9/32的LDP邻接体信息。

<Quidway> display mpls ldp adjacency remote peer 3.3.3.9 verbose
 LDP Adjacency Information
 ------------------------------------------------------------------------------
                           LDP Peer ID : 3.3.3.9
                      VPNInstance name : -
                            CreateDate : 2008-09-02
                            CreateTime : 14:51:54
                         AdjacencyType : Remote Adjacency
                       DiscoverySource : -
                           Sequence No.: 0
   Configuration Hello Hold Timer(sec) : 45
 ------------------------------------------------------------------------------
 TOTAL: 2 Record(s) found.
表2 display mpls ldp adjacency remote peer命令输出信息描述

项目

描述

LDP Peer ID

LDP对等体的LSR ID

VPNInstance name

VPN实例名称

AdjacencyType

LDP邻接体的类型：

  • Local Adjacency：本地邻接体

  • Remote Adjacency：远端邻接体

DiscoverySource

LDP邻接体的发现源。

  • 本地LDP邻接体的发现源为接口

  • 远端LDP邻接体的发现源为空

Sequence No

收到的hello消息中携带的序列号，缺省值为0

Configuration Hello Hold Timer(sec)

对端配置的Hello保持定时器的值，单位是秒

  • 链路Hello保持定时器用于维护本地LDP邻接体，缺省值为15秒

  • 目标Hello保持定时器用于维护远端LDP邻接体，缺省值为45秒
