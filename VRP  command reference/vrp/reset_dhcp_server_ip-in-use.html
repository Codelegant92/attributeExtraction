reset dhcp server ip-in-use

命令功能

reset dhcp server ip-in-use命令用来清除DHCP动态地址绑定信息。

命令格式

reset dhcp server ip-in-use { all | ip ip-address | pool [ pool-name ] | interface [ interface-type interface-number ] | vlan vlan-id }

参数说明

ip ip-address：指定IP地址的绑定信息。

pool [ pool-name ]：指定全局地址池。如果不指定地址池名，表示所有全局地址池。字符串形式，长度范围是1～35。

interface [ interface-type interface-number ]：指定接口地址池，如果不指定接口名，表示所有接口地址池。

vlan vlan-id：表示范围为指定的VLAN的接口地址池，没指定VLAN时指所有VLAN的接口地址池。

all：表示清除所有的动态地址绑定信息。

使用指南

当需要重新分配IP地址时，使用reset dhcp server ip-in-use命令用来清除DHCP动态地址绑定信息。

执行此命令既可以清除过期的地址绑定信息，也可以清除未过期的动态绑定信息。

使用实例

# 清除地址10.110.1.1的绑定信息。

<Quidway> reset dhcp server ip-in-use ip 10.110.1.1
