process-pst

命令功能

process-pst命令用来允许BFD会话修改端口状态表PST（Port State Table）。

undo process-pst命令用来恢复缺省设置。

缺省情况下，BFD不修改端口状态表。

命令格式

process-pst

undo process-pst

参数说明

无

视图

BFD会话视图

缺省级别

2：配置级

使用指南

process-pst命令只能在绑定了接口的BFD会话视图下配置，即，只能配置在BFD单跳检测的情况下。

对于NE5000E/80E/40E路由器，在部署基于BFD检测的IP FRR和LDP FRR时，必须配置process-pst命令。

VE接口不支持组播BFD会话修改端口状态表。

对于不通过PST来感知BFD检测故障的应用，不需要配置process-pst命令。

使用实例

# 禁止BFD会话修改端口状态表。

<Quidway> system-view
[Quidway] bfd pos0
[Quidway-bfd-session-pos0] undo process-pst
