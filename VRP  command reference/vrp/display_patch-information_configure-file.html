display patch-information configure-file

命令功能

display patch-information configure-file命令用来查看补丁状态文件。

命令格式

display patch-information configure-file [ next-startup ]

参数说明

next-startup：查看下一次启动后的补丁状态文件。

说明：

如果没有设置下一次启动后的补丁状态，则执行display patch-information configure-file next-startup命令查看到的下次重启后的补丁状态文件与当前的补丁状态文件相同。

使用指南

执行display patch-information configure-file命令可以查看当前的补丁状态文件。

补丁状态文件仅用于设备重启后恢复单板上的补丁状态，查看当前的补丁状态文件不等于查看当前的补丁状态。例如，补丁状态文件中记录某单板的补丁状态为激活态，设备重启后，该单板补丁状态变为去激活态，但补丁状态文件中该单板状态仍为激活态，此时补丁状态文件中记录的补丁状态与实际情况不符。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

# 查看设备当前的补丁状态文件。

<Quidway> display patch-information configure-file

表1 display patch-information configure-file命令输出信息描述

项目

描述

Slot

单板的槽位号

State
单板的状态

  • unregistered：单板不在槽位中

Run

补丁状态为运行态

Active

补丁状态为激活态

Deactive

补丁状态为去激活态

NPPatch

产品补丁状态

idle

无补丁
