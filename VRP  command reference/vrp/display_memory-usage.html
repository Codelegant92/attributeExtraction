display memory-usage

命令功能

display memory-usage命令用来查看路由器的内存占用率信息。

命令格式

display memory-usage [ slave | slot slot-id ]

参数说明

slave：显示备用主控板的内存信息。

slot slot-id：显示指定接口板的内存信息。接口板槽号的取值范围因设备不同而不同。

所有视图

使用指南

执行该命令后，可以查看主控板和指定接口板的内存信息。

使用实例

# 查看当前主控板的内存占用率信息。

<Quidway> display memory-usage
 Memory utilization statistics at 2005-01-06 14:59:12 872 ms
 System Total Memory Is: 183352736 bytes
Total Memory Used Is: 69294044 bytes
 Memory Using Percentage Is: 37%
