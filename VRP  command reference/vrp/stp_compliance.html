stp compliance

命令功能

stp compliance命令用来配置交换机收发MSTP报文的协议格式。

undo stp compliance命令用来恢复交换机收发MSTP报文格式的缺省值。

缺省情况下，MSTP报文格式为auto。

命令格式

stp compliance { auto | dot1s | legacy }

undo stp compliance

参数说明

auto ：协议格式自适应。

dot1s ：标准IEEE 802.1s报文格式。

legacy ：私有协议报文格式。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

本命令只能在二层端口视图下配置。

对于同一个端口，如果分别在系统视图和接口视图下配置了不同的报文格式，则以后配置的为准。

使用实例

# 设置端口GigabitEthernet1/0/0的报文收发格式为标准格式。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quideway-GigabitEthernet1/0/0] portswitch
[Quideway-GigabitEthernet1/0/0] stp compliance dot1s

# 恢复交换机的报文收发格式为自适应。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quideway-GigabitEthernet1/0/0] portswitch
[Quideway-GigabitEthernet1/0/0] undo stp compliance
