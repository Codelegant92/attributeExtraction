ebgp-interface-sensitive

命令功能

ebgp-interface-sensitive命令可以在某个接口状态变为Down时，立即清除建立在该接口上的直连EBGP会话。

undo ebgp-interface-sensitive命令用来取消该功能。

缺省情况下，使能该功能。

命令格式

ebgp-interface-sensitive

undo ebgp-interface-sensitive

参数说明

无

视图

BGP视图、BGP-IPv6单播地址族视图、BGP-VPN实例视图、BGP-VPN6实例视图

缺省级别

2：配置级

使用指南

配置undo ebgp-interface-sensitive命令后，EBGP直连会话不会随着接口不断Up和Down而重建与删除。

使用实例

# 使能自动清除BGP会话能力。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ebgp-interface-sensitive
