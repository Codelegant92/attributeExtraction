save logfile

命令功能

save logfile用来把日志文件缓冲区中的日志保存到日志文件中。

命令格式

save logfile

参数说明

无

视图

用户视图

缺省级别

0：参观级

使用指南

日志缓冲区中的日志会定时自动保存到日志文件中，不同产品定时保存日志到日志文件中的时间间隔不同。如需要查看当前日志信息，可以通过命令save logfile把日志保存到日志文件后进行查看。

使用实例

# 保存日志文件缓冲区中的日志到日志文件。

<Quidway> save logfile
Save log file successfully.
