display atm pvc-info

命令功能

display atm pvc-info命令用来显示PVC的信息。

命令格式

display atm pvc-info [ interface atm interface-number [ pvc { pvc-name [ vpi/vci ] | vpi/vci } ] ]

参数说明

interface-number：接口编号。

pvc-name：PVC名，字符串形式，长度为1～16。如果不指定PVC名且不指定VPI/VCI值对，则缺省显示指定ATM接口范围内所有PVC的信息。

vpi/vci：VPI/VCI值对。详见pvc命令中的参数说明。

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 显示设备中所有ATM接口下的PVC的信息。

<Quidway> display atm pvc-info
VPI/VCI |STATE|PVC-NAME        |INDEX   |ENCAP|PROT |INTERFACE
--------|-----|----------------|--------|-----|-----|-------------------
  0/40  |UP   |                |1       |SNAP  |IP   |Atm1/0/0 (UP)
  0/60  |UP   |to_adsl_a       |2       |SNAP  |ETH  |Atm1/0/0.1 (UP)
  0/61  |UP   |to_adsl_b       |3       |SNAP  |PPP  |Atm1/0/0.1 (UP)
VPI/VCI |STATE|PVC-NAME        |INDEX   |ENCAP|PROT |INTERFACE

  1/10  |UP   |                |2       |SNAP |None |Atm2/0/0.10 (UP)

项目

描述

VPI/VCI

VPI/VCI值对。

STATE

PVC的状态。

PVC-NAME

PVC名称。

INDEX

PVC的索引号。根据PVC建立的先后，系统自动为其分配索引号。索引编号从0开始还是从1开始和产品有关。当前一个索引编号被删除时，后一个索引编号会自动减1，来填充前一个编号。索引编号的取值范围是0～64K。

ENCAP

AAL的封装类型。

PROT
应用方式。

  • IP表示IPoA。

  • ETH表示PPPoEoA或IPoEoA。

  • PPP表示PPPoA。

INTERFACE

所在的ATM接口。

项目

描述

VPI: VCI:

VPI/VCI值对中的VPI和VCI值。

INDEX

PVC的索引号。

根据PVC建立的先后，系统自动为其分配索引号。索引编号从0开始还是从1开始和产品有关。当前一个索引编号被删除时，后一个索引编号会自动减1，来填充前一个编号。索引编号的取值范围是0～64K。

AAL5 Encaps:

AAL5封装的协议。

Protocol:

网络层协议类型。

Service-type:
业务类型。主要的业务类型有：

  • CBR：Constant Bit Rate，确定速率。

  • UBR：Unspecified Bit Rate，非确定速率。

  • VBR-NRT：Variable Bit Rate-Non Real Time，非实时可变速率。

  • VBR-RT：Variable Bit Rate-Real Time，实时可变速率。
