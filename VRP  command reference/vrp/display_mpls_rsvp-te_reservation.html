display mpls rsvp-te reservation

命令功能

display mpls rsvp-te reservation命令用来查看使能了RSVP-TE的接口的RSVP-TE预留消息。

命令格式

display mpls rsvp-te reservation [ interface interface-type interface-number peer-ip-address ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface：显示处于非stale状态的接口RSVP-TE隧道资源预留相关信息。

interface-type interface-number：接口类型和接口编号。

peer-ip-address：RSVP-TE邻居的IP地址。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定接口名，将显示所有使能了RSVP-TE的接口的请求消息信息。
