mfr timer ack

命令功能

mfr timer ack命令用来设置帧中继捆绑链路（bundle link）在重发hello消息前等待hello应答消息的时间。

undo mfr timer ack命令用来恢复缺省设置。

命令格式

mfr timer ack seconds

undo mfr timer ack

参数说明

seconds：重发hello消息前等待hello应答消息的时间。整数形式，取值范围是1～10，单位是秒。缺省值是4秒。

视图

帧中继接口视图

缺省级别

2：配置级

使用指南

帧中继捆绑链路的两端定期向对端发送hello消息，对端收到后，回应一个hello应答消息。

使用实例

# 设置帧中继捆绑链路Serial1/0/0:0在重发hello消息前等待6秒。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr mfr 1/0/123
[Quidway-Serial1/0/0:0] mfr timer ack 6
