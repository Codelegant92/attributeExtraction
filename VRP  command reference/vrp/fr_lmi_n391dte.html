fr lmi n391dte

命令功能

fr lmi n391dte命令用来配置DTE侧N391参数。

undo fr lmi n391dte命令用来恢复到缺省值。

命令格式

fr lmi n391dte n391-value

undo fr lmi n391dte

参数说明

n391-value：请求PVC状态的计数器值。整数形式，取值范围是1～255。缺省值是6。

视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

DTE设备每隔一定的时间（间隔由T391决定）要发送一个状态请求报文（Status-Enquiry报文）。状态请求报文有两种类型：链路完整性验证报文和链路状态查询报文。参数N391定义两种报文的发送比例，即，链路完整性验证报文数：链路状态查询报文数＝（N391－1）：1。

使用实例

# 设置帧中继接口Serial1/0/0:0工作在DTE方式，请求PVC状态的计数器值为10。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr
[Quidway-Serial1/0/0:0] fr interface-type dte
[Quidway-Serial1/0/0:0] fr lmi n391dte 10
