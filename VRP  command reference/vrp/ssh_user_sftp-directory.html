ssh user sftp-directory

命令功能

ssh user sftp-directory 命令用来配置SSH用户的SFTP服务授权目录。

undo ssh service-type命令用来取消SSH用户的SFTP服务授权目录。

缺省情况下，SSH用户的SFTP服务授权目录是flash:。

命令格式

ssh user user-name sftp-directory path

undo ssh user user-name sftp-directory

参数说明

user-name：SSH用户名，字符串形式，长度范围为1～64。

path：SFTP服务器上的目录名，字符串形式，长度范围为1～64。

使用指南

用户只能访问SFTP服务设定目录。如果用户user-name不存在，则新建一个用户名为user-name的SSH用户，SFTP服务授权目录为所配置的目录。如果所配置的目录不存在，SFTP客户端用该用户连接SSH服务器时失败。

使用实例

# 配置SSH用户的SFTP服务授权目录。

<Quidway> system-view
[Quidway] ssh user zhangsan sftp-directory flash:/ssh
