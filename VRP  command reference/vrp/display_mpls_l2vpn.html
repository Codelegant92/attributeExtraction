display mpls l2vpn

命令功能

display mpls l2vpn命令用来查看PE上的L2VPN信息。

命令格式

display mpls l2vpn [ l2vpn-name [ local-ce | remote-ce ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

l2vpn-name：用于显示的L2VPN名称。字符串格式，不支持空格，区分大小写，长度范围是1～31。如果不指定，将显示所有L2VPN信息。

local-ce：查看指定VPN上所有本地CE的配置情况和状态。

remote-ce：查看从其它PE学习到的远端CE的配置情况和状态。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display mpls l2vpn命令后，如果指定参数，则只显示和参数相关的信息；如果不指定参数，则显示全部信息。

使用实例

# 查看PE上配置的所有VPN。

<Quidway> display mpls l2vpn
VPN number: 1
vpn-name        encap-type   route-distinguisher     mtu   ce(L)  ce(R)
vpn1            ppp          100:1                   128   1      1
表1 display mpls l2vpn命令输出信息描述

项目

描述

VPN number

已创建的L2VPN实例的数量。

vpn-name

已创建的VPN实例的名称。

encap-type

L2VPN的封装类型。

route-distinguisher

L2VPN的路由标志符RD。

mtu

L2VPN的MTU。

ce(L)

本地CE连接的数量，“L”表示“local”。

ce(R)

远程CE连接的数量，“R”表示“remote”。

# 查看名为vpn1的L2VPN信息。

<Quidway> display mpls l2vpn vpn1
VPN name: vpn1, encap type: interworking, local ce number(s): 1, remote ce number(s): 1
route distinguisher: 100:1, MTU: 128
import vpn target: 1:1,
export vpn target: 1:1,
 
remote vpn site(s) :
no.  remote-pe-id    route-distinguisher
1    3.3.3.9         100:1
表2 display mpls l2vpn vpn-instance-name命令输出信息描述

项目

描述

VPN number

已创建的L2VPN实例的数量。

encap-type

L2VPN的封装类型。

local ce number(s)

本地CE连接的数量。

remote ce number(s)

远程CE连接的数量。

route-distinguisher

L2VPN的路由标志。

MTU

该L2VPN关联的接口的最大传输单元。

import vpn target

接收的VPN路由属性。

export vpn target

对发送的VPN路由配置的属性。

remote vpn site(s)

表示以下显示信息为远端站点的相关信息。

no.

远端对等体的序号

remote-pe-id

远端PE对等体的ID，一般为MPLS LSR-ID或BGP对等体的会话地址。

route-distinguisher

路由标志符，相关命令请参见vpn-target（MPLS-L2VPN实例视图）。

# 查看名为vpn1的L2VPN上本地CE的信息。

<Quidway> display mpls l2vpn vpn1 local-ce
ce-name              ce-id   range   conn-num  LBBase/Offset/Range
ce1                  1       10      1         21504/0/10
ce2                  2       10      1         21514/0/10
表3 display mpls l2vpn vpn-name local-ce命令输出信息描述

项目

描述

ce-name

定义的CE名字

ce-id

定义的CE标志符，唯一标志一个CE。

range

指定本地CE的范围（CE range），表明这个本地CE能与多少个CE建立连接

conn-num

为本CE创建的本地连接数。

LBBase/Offset/Range

本端为与之建立连接的CE分配的标签块。21504/0/10表示标签块的起始值/初始CE偏差/标签的个数。

# 查看名为vpn1的L2VPN上远端CE的信息。

<Quidway> display mpls l2vpn vpn1 remote-ce
no.  ce-id peer-id         route-distinguisher     LBBase/Offset/Range
1    2     3.3.3.9         100:1                   19456/0/10
表4 display mpls l2vpn vpn-name remote-ce命令输出信息描述

项目

描述

no.

连接的序列号。

ce-id

远端的CE-ID。

peer-id

远端的IP地址。

route-distinguisher

远端的路由标志符。

LBBase/Offset/Range

远端标签块。19456/0/10表示远端标签块起始值/远端的初始CE偏差/远端标签个数
