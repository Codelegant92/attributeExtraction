cable（CT3接口视图）

命令功能

cable命令用来配置CT3的电缆长度，是路由器到配线架之间的距离。

undo cable命令用来恢复缺省值。

缺省情况下，CT3的电缆长度为350英尺。

命令格式

cable feet

undo cable

参数说明

feet：电缆长度。取值范围是0～450，单位是英尺（feet）。

视图

CT3接口视图

缺省级别

2：配置级

使用指南

当物理接口闲置，没有连接电缆时，请使用shutdown命令禁止该接口，以防止由于干扰导致接口异常。

使用实例

# 设置CT3接口1/0/0的电缆长度为50英尺。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] cable 50
