efm mode

命令功能

efm mode命令用来配置当前接口的EFM OAM工作模式。

缺省情况下，接口的EFM OAM工作模式是主动模式。

命令格式

efm mode { active | passive }

参数说明

active：指定接口的EFM OAM工作模式为主动模式。

passive：指定接口的EFM OAM工作模式为被动模式。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

具备EFM OAM功能的两端中，应至少有一端配置为主动模式。被配置为主动模式的一端，在端口使能了EFM OAM功能后，会主动发起EFM OAM对端发现过程，而被配置为被动模式的一端则不会主动发起EFM OAM对端发现过程，而是等待对端发送的协议报文。如果两端都被配置成为被动模式，则链路检测无法进行。

工作模式为主动模式的接口才能发起环回测试。

注意：

只能在EFM OAM功能全局使能后，接口的EFM OAM功能使能前，配置接口的EFM OAM工作模式。在接口的EFM OAM功能使能后，设备不允许再改变接口的EFM OAM工作模式。

使用实例

# 配置接口GigabitEthernet1/0/0的EFM OAM工作模式为被动模式

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] efm mode passive
