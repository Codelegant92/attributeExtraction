source-address

命令功能

source-address命令用来配置本次测试的源IP地址。

undo source-address命令用来恢复缺省配置。

缺省情况下，使用发送测试报文的接口IP地址作为源IP地址。

命令格式

source-address ipv4 ip-address

undo source-address

参数说明

ipv4 ip-address：设置目的地址为IPv4类型。IP地址为点分十进制格式。

视图

NQA视图

缺省级别

2：配置级

使用指南

不能对DHCP、DNS、MPing、PWE3Ping和PWE3Trace类型的测试例配置源地址。

不能对正在运行的测试例修改此项参数。

使用实例

# 设置测试例user test的源地址为1.1.1.1。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] source-address ipv4 1.1.1.1
