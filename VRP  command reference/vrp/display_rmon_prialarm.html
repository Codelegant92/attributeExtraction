display rmon prialarm

命令功能

display rmon prialarm命令用来查看RMON扩展告警信息。

命令格式

display rmon prialarm [ entry-number ]

参数说明

entry-number：扩展报警表行索引。整数形式，取值范围是1～65535。

视图

所有视图

缺省级别

1：监控级

使用指南

无

使用实例

# 查看RMON扩展告警信息。

<Quidway> display rmon prialarm
Prialarm table 1 owned by Huawei is VALID.
  Samples absolute value: .1.3.6.1.2.1.16.1.1.1.6.1+.1.3.6.1.2.1.16.1.1.1.7.1
  Sampling interval         : 30(sec)
  Rising threshold          : 1000(linked with event 3)
  Falling threshold         : 100(linked with event 3)
  When startup enables      : risingOrFallingAlarm
  This entry will exist     : forever.
  Latest value               : 557
表1 display rmon prialarm命令输出信息描述

项目

描述

Prialarm table 1

告警表中的索引1

VALID

索引对应的行状态有效

Huawei

所有者

Samples absolute value

采样为绝对值，后面跟告警变量表达式

Sampling interval

采样间隔时间

Rising threshold

上限阈值为1000

Falling threshold

下限阈值为100

linked with event 3

表示与事件表中索引为3的行相联系

risingOrFallingAlarm

初次报警类型：超过上限阈值或者低于下限阈值时报警

Latest value

最近一次的采样值
