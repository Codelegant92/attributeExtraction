snmp-agent trap enable vrrp-group

命令功能

snmp-agent trap enable vrrp-group命令用来配置VRRP管理组向网管中心发送告警功能。

undo snmp-agent trap enable vrrp-group命令用来去使能VRRP管理组向网管中心发送告警功能。

缺省情况下，VRRP管理组不向网管中心发送告警。

命令格式

snmp-agent trap enable vrrp-group

undo snmp-agent trap enable vrrp-group

参数说明

无

系统视图

缺省级别

2：配置级

使用指南

snmp-agent trap enable vrrp-group命令必须与snmp-agent target-host命令协同使用。

snmp-agent target-host命令指定向哪些主机发送Trap信息。

为了发送Trap信息，用户必须配置至少一条snmp-agent target-host命令。

使用实例

# 为路由器Quidway上的所有VRRP管理组打开Trap告警使能开关。

<Quidway> snmp-agent trap enable vrrp-group
