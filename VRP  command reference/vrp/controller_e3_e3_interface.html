controller e3（E3接口）

命令功能

controller e3命令用来进入指定E3接口的接口视图。

命令格式

controller e3 controller-number

参数说明

controller-number：接口号，包括槽号、卡号、端口号。

视图

系统视图

缺省级别

2：配置级

使用指南

当需要对E3接口进行配置时，可使用该命令进入E3接口视图。

使用实例

# 进入E3接口3/0/0的接口视图。

<Quidway> system-view
[Quidway] controller e3 3/0/0
[Quidway-E3 3/0/0]
