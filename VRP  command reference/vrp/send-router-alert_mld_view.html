send-router-alert（MLD视图）

命令功能

send-router-alert命令用来全局设置发送的MLD报文在IPv6报文头中应包含Router-Alert选项。

undo send-router-alert命令用来全局设置发送的MLD报文在IPv6报文头里不包含Router-Alert选项。

缺省情况下，路由器发送包含Router-Alert选项的MLD报文。

命令格式

send-router-alert

undo send-router-alert

参数说明

无

视图

MLD视图

缺省级别

2：配置级

使用指南

此命令与接口视图下的mld send-router-alert的含义是相同的，只是视图不同。缺省情况下，如果接口上没有配置，取MLD视图下配置的值；否则取接口视图下配置的值。

使用实例

# 指定发送的MLD报文中IPv6报文头应包含Router-Alert选项。

<Quidway> system-view
[Quidway] mld
[Quidway-mld] send-router-alert
