dns server ipv6 source-ip

命令功能

dns server ipv6 source-ip命令用来指定本端路由设备作为IPv6 DNS client进行IPv6 DNS通信时的源地址。

undo dns server ipv6 source-ip命令用来删除本端路由设备的源地址。

命令格式

dns server ipv6 source-ip ipv6-address

undo dns server ipv6 source-ip

参数说明

ipv6-address：本端路由设备的源地址。

使用指南

指定本端路由设备的源地址，以指定的源地址与IPv6 DNS服务端通信，从而达到进行安全校验的目的。IPv6 DNS服务器会将域名请求按照指定IPv6地址的路由发送回本端路由设备上。

使用实例

# 指定本端路由设备的源地址为2001:10::100/64。

<Quidway> system-view
[Quidway] dns server ipv6 source-ip 2001:10::100
