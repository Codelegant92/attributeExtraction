mac-limit rule-name（接口视图）

命令功能

mac-limit rule-name命令用来在主接口或者子接口上应用全局MAC地址学习限制规则。

undo mac-limit命令用来删除主接口或者子接口上的全局MAC地址学习限制规则。

命令格式

mac-limit [ vlan vlan-id1 [ to vlan-id2 ] rule-name rule-name

undo mac-limit [ vlan vlan-id1 [ to vlan-id2 ]

参数说明

rule-name rule-name：全局MAC地址限制规则名。字符串形式，1～31字符。规则名不能有空格、问号、减号。

vlan-id1、vlan-id2：接口所属的VLAN ID，取值范围1～4094。

视图

Ethernet接口视图、GE接口视图、Eth-trunk接口视图、Ethernet子接口视图、GE子接口视图、Eth-trunk子接口视图

缺省级别

2：配置级

使用指南

配置本命令时，应注意以下几点：

  • Ethernet接口、GE接口、Eth-trunk接口必须是二层接口。

  • 配置本命令前，子接口已经绑定了VSI。

  • 如果要配置VLAN ID，接口必须已经加入了指定的VLAN。

  • 一条全局MAC地址学习限制规则最多应用在128个接口上。

  • 当接口下已经存在其他MAC地址学习限制规则时，不能再应用全局限制规则。

  • 接口和接口所属VLAN不能应用相同的全局MAC地址学习限制规则。

使用实例

# 在GE2/0/1.1上应用规则名为name1的全局MAC地址学习限制规则。

<Quidway> system-view
[Quidway] interface gigabitethernet2/0/1.1
[Quidway－GigabitEthernet2/0/1.1] mac-limit rule-name name1

# 在GE2/0/1上应用规则名为name1的全局MAC地址学习限制规则。

<Quidway> system-view
[Quidway] interface gigabitethernet2/0/1
[Quidway－GigabitEthernet2/0/1] portswitch
[Quidway－GigabitEthernet2/0/1] mac-limit rule-name name1
