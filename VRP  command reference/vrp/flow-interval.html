flow-interval

命令功能

flow-interval命令用来配置流控的间隔时间。

undo flow-interval命令用来恢复流控的间隔时间的缺省值。

命令格式

flow-interval interval

undo flow-interval

参数说明

interval：流控的间隔时间。取值范围是1～1500，单位是秒。缺省值是20秒。

视图

系统视图

缺省级别

2：配置级

使用指南

该命令仅对DCC流量触发拨号有效。

使用实例

# 配置流控的间隔时间为3秒。

<Quidway> system-view
[Quidway] flow-interval 3
