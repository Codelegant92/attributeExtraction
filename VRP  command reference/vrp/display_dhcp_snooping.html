display dhcp snooping

命令功能

display dhcp snooping命令用来显示接口上的DHCP Snooping相关信息。

命令格式

display dhcp snooping { interface interface-type interface-number | vlan vlanid [ interface interface-type interface-number ] }

参数说明

vlan vlanid：表示VLAN ID，取值范围1~4094。此参数仅在二层设备上可配置。

interface-type：接口类型。

interface-number：接口编号。

使用指南

执行display dhcp snooping后，设备将显示VLAN和接口下所有的DHCP Snooping配置信息。显示信息的内容和含义，请参见下表。

使用实例

# 显示Ethernet4/0/0接口相关的信息。

<Quidway> display dhcp snooping interface ethernet 4/0/0

dhcp snooping trusted
dhcp snooping check arp enable
dhcp snooping alarm arp enable
dhcp snooping alarm arp threshold 50
dhcp snooping nomatch-packet arp action discard
dhcp snooping check ip enable
dhcp snooping alarm ip enable
dhcp snooping alarm ip threshold 50
dhcp snooping nomatch-packet ip action discard
dhcp snooping alarm dhcp-reply enable

dhcp snooping alarm dhcp-chaddr enable
dhcp snooping alarm dhcp-chaddr threshold 50
dhcp snooping check dhcp-request enable
