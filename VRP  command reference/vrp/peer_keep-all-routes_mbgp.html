peer keep-all-routes（MBGP）

命令功能

peer keep-all-routes命令用来保存自MBGP连接建立起来之后的所有来自指定对等体（组）的BGP路由更新信息，即使这些路由没有通过已配置的入口策略。

undo peer keep-all-routes命令用来取消该功能。

缺省情况下，只保存来自对等体的通过已配置入口策略的BGP路由更新信息。

命令格式

peer { group-name | peer-address } keep-all-routes

undo peer { group-name | peer-address } keep-all-routes

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。
