radius-server shared-key

命令功能

radius-server shared-key命令用来设置RADIUS服务器的共享密钥。

命令格式

radius-server shared-key key-string

参数说明

key-string：共享密钥，字符串形式，长度范围是1～16，缺省值是“huawei”。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

密钥用于加密用户口令及生成回应认证符（Response Authenticator）。RADIUS发送认证报文时，对口令等重要信息使用MD5加密，确保认证信息在网络中传输的安全性。为了确保认证双方的身份合法性，要求路由器上的密钥与RADIUS服务器间设定的密钥必须相同。

只有当该RADIUS Server模板没有用户使用时，才能改变此配置。

使用实例

# 配置RADIUS服务器共享密钥为hello。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server shared-key hello
