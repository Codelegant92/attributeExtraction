display ip pool

命令功能

display ip pool命令主要用来查看地址池中地址的配置和使用情况。

命令格式

display ip pool { global | domain domain-name }

参数说明

global：全局下的地址池。

domain-name：显示指定域下的地址池。字符串新式，长度范围是1～20，不区分大小写。

视图

所有视图

缺省级别

2：配置级

使用指南

如果参数为global，则显示在AAA视图下的址址池的情况，AAA视图下的地址池是为default域的用户以及不认证的用户分配地址使用的。

如果参数为domain domain-name则显示该域下的地址池的配置和使用情况，一个PPP用户如果通过了认证并且不在default域下，则使用相应域下的地址池。

使用实例

# 查看全局下的地址池使用情况。

<Quidway> display ip pool global
------------------------------------------------------------------ 
Pool-number Pool-start-addr Pool-end-addr Pool-length  Used-addr-number
-------------------------------------------------------------------- 
0           190.1.1.2        190.1.1.100     99              0             
--------------------------------------------------------------------
 Total pool number:    1

# 查看域huawei下的地址池使用情况。

<Quidway> display ip pool domain huawei
------------------------------------------------------------------
Pool-number Pool-start-addr Pool-end-addr Pool-length  Used-addr-number 
-------------------------------------------------------------------- 
1        180.1.10.1        180.1.13.254        1022           1           
2        180.1.14.1        180.1.16.254        766            0                       
--------------------------------------------------------------------
 Total pool number:    2

项目

描述

Pool-number

地址池编号

Pool-start-addr

地址池起始地址

Pool-end-addr

地址池结束地址

Pool-length

地址池中包含的地址数

Used-addr-number

地址池中被使用的地址数量
