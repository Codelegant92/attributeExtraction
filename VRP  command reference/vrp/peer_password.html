peer password

命令功能

peer password命令用来配置BGP对等体建立TCP连接时以及对BGP消息进行MD5认证。

undo peer password命令用来取消此功能。

命令格式

peer { group-name | ipv4-address | ipv6-address } password { cipher cipher-password | simple simple-password }

undo peer { group-name | ipv4-address | ipv6-address } password

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

cipher cipher-password：密文密码字符串。字符串类型，不允许空格，可以输入1～16个字符的明文，也可以输入24个字符的密文。

simple simple-password：明文密码字符串。字符串类型，不允许空格，长度为1～16。
