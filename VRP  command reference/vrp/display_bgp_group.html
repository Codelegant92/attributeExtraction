display bgp group

命令功能

display bgp group命令用来显示对等体组信息。

命令格式

display bgp group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp l2vpn group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpls group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp ipv6 group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 all group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 vpn6-instance vpn6-instance-name group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

group-name：指定的对等体组名。

vpnv4：查看VPNv4的BGP对等体组信息。

all：查看所有VPNv4的对等体组信息。

vpn-instance vpn-instance-name：指定VPN实例名称。区分大小写。

l2vpn：查看L2VPN的BGP对等体组信息。

vpls：查看VPLS的BGP对等体组信息。

ipv6：查看IPv6对等体组信息。

vpnv6：查看BGP VPNv6对等体组信息。

vpnv6 vpn6-instance group：查看指定IPv6 VPN实例的对等体组信息。

vpn6-instance-name：指定IPv6 VPN实例名称。区分大小写。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Remote AS

对等体所在的AS号

Group's BFD has been enabled,

对等体组已经使能BFD功能。

Type
对等体组的类型

  • internal：对等体组类型为IBGP

  • external：对等体组类型为EBGP

Configured hold timer value

所配置的保持时间的值

Keepalive timer value

存活时间的值

Minimum route advertisement interval

路由通告的最短时间间隔

Listen-only has been configured

仅监听连接请求，不主动发起连接功能

PeerSession Members

对等体成员之间建立了会话连接的成员

Maximum allowed route limit

最大可允许的BGP路由的数量

Threshold

告警阈值，即BGP路由数量占可允许接收的路由数量的百分比

Peer Preferred Value

对等体的首选值

Peer Members

表示以下显示的是对等体的相关信息

Peer

对等体的IP地址

V

BGP版本号

As

对等体所在的自治系统号

MsgRcvd

收到的消息数目

MsgSent

发送的消息数目

OutQ

等待向邻居发送的消息数目

Up/Down

BGP会话处于当前状态的时长

State
BGP状态机

  • Idle：BGP拒绝任何进入的连接请求，是BGP初始状态

  • Active：BGP将尝试进行TCP连接的建立，是BGP的中间状态

  • Established：BGP对等体间可以交换Update报文、Notification报文和Keepalive报文

  • Connect：此状态下，BGP等待TCP连接的建立完成后再决定后续操作

  • OpenSent：此状态下，BGP等待对等体的Open消息

  • OpenConfirm：此状态下，BGP等待一个Notification报文或Keepalive报文

PrefRcv

本端从对等体上收到路由前缀的数目
