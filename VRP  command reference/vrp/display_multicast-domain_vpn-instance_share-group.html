display multicast-domain vpn-instance share-group

命令功能

display multicast-domain vpn-instance share-group命令用来查看组播域中指定VPN实例的Share-group组信息。

命令格式

display multicast-domain vpn-instance vpn-instance-name share-group [ local | remote ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance-name：表示指定VPN实例的名称。字符串形式，长度范围是1～31。

local：本机配置的Share-group的组地址。

remote：从BGP的Update报文中学到的Share-group的组地址。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display multicast-domain vpn-instance share-group命令：

  • 如果指定local关键字，则显示本地配置的Share-group组信息。

  • 如果指定remote关键字，则显示通过BGP的Update报文学到的Share-group组信息。

  • 如果既没有指定local关键字，也没有指定remote关键字，则显示所有Share-group组信息。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

显示通过BGP的Update报文学到的Share-group组信息时，以Share-group组从小到大的顺序进行排列。Share-group组相同时以下一跳地址从小到大的顺序进行排列。

项目

描述

MD local share-group information for VPN-Instance

组播域中本地Share-group信息所属的VPN实例。

Share-group

本机配置的Share-MDT的组地址。

MTunnel address

本机配置的Share-group绑定的MTI接口地址。

Route-distinguisher

路由标识符，区分VPN实例使用相同地址空间的IPv4前缀。

Router-id

路由器标识符。

Share-group

远端配置的Share-group的组地址。

Next-hop

到远端PE的下一跳地址。
