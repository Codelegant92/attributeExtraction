clock daylight-saving-time

命令功能

clock daylight-saving-time命令用来设置夏令时的名称和起始、终止时间。

undo clock daylight-saving-time命令用来取消当前设置。

命令格式

clock daylight-saving-time time-zone-name one-year start-time start-date end-time end-date offset

clock daylight-saving-time time-zone-name repeating start-time { { first | second | third | fourth | last } weekday month | start-date } end-time { { first | second | third | fourth | last } weekday month | end-date } offset [ start-year [ end-year ] ]

undo clock daylight-saving-time

参数说明

time-zone-name：指定夏令时区名称。字符串形式，取值范围是1～32。

one-year：仅对某一年的夏令时时间进行设置。

repeating：对从某一年开始的以后每年的夏令时时间的设置。

start-time：指定起始时间。格式是HH:MM，24小时制，HH表示小时，取值范围是0～23，MM表示分钟，取值范围是0～59。其中，可以不输入MM，表示0分0秒。但至少需要输入一位数的HH的值，例如输入0，则表示0小时0分0秒。

start-date：指定起始日期。格式是MM-DD，MM取值范围是1～12，DD取值范围是1～31。

end-time：指定结束时间。格式是HH:MM，24小时制，HH表示小时，取值范围是0～23，MM表示分钟，取值范围是0～59。其中，可以不输入MM，表示0分0秒。但至少需要输入一位数的HH的值，例如输入0，则表示0小时0分0秒。

end-date：指定结束日期。格式是MM-DD，MM取值范围是1～12，DD取值范围是1～31。

offset：指定采用夏令时的时差。格式是HH:MM，24小时制，HH表示小时，取值范围是0～23，MM表示分钟，取值范围是0～59。其中，可以不输入MM，表示0分0秒。但至少需要输入一位数的HH的值。

first：指定月份中起始的第一个工作日。

second：指定月份中起始的第二个工作日。

third：指定月份中起始的第三个工作日。

fourth：指定月份中起始的第四个工作日。

last：指定月份中起始的最后一个工作日。

weekday：指定工作日。取值：Monday、Tuesday、Wednesday、Thursday、Friday、Saturday、Sunday，分别表示从星期一到星期日。

month：指定月份。取值：January、February、March、April、May、June、July、August、September、October、November、December，分别表示从一月份到12月份。

start-year：指定起始年份。格式是YYYY，取值范围是2000～2099。

end-year：指定结束年份。格式是YYYY，取值范围是2000～2099。

使用指南

系统默认情况下，无夏令时。此时使用display clock命令并不显示夏令时相关信息。当需要使用夏令时时，用该命令配置。

结束时间一定要比开始时间大于一天且不大于一年，才能使得设置生效。

设置生效后，可以用display clock命令进行查看。另外，日志、诊断等信息的时间采用经过时区和夏令时调整过的本地时间。

使用实例

# 按周期设置夏令时。

<Quidway> clock daylight-saving-time test repeating 0 first Sun Jan 0 first Sun Apr 4 2009 2009
<Quidway> display clock

Tuesday

         Repeat mode : repeat
         Start year  : 2009
         End year    : 2009

         End time    : first Sunday April 00:00:00
         Saving time : 04:00:00

<Quidway> clock daylight-saving-time test repeating 12:11 1-1 1:0 3-4 1
<Quidway> display clock

Tuesday

         Start year  : 2000

         Start time  : 01-01 12:11:00
         End time    : 03-04 01:00:00
         Saving time : 01:00:00
