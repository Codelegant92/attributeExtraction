display ipsec policy

命令功能

display ipsec policy命令用来查看安全策略的信息。

命令格式

display ipsec policy [ brief | name policy-name [ seq-number ] ]

参数说明

brief：表示显示所有安全策略的简要信息。

name：表示显示名字为policy-name，顺序号为seq-number的安全策略的信息。

policy-name：指定安全策略的名字，取值范围是1～15。

seq-number：指定安全策略的顺序号，取值范围1～10000。

如果不指定任何参数，则表示要显示所有安全策略的详细信息。如果指定了name policy-name，而没有指定seq-number，则表示要显示指定的安全策略组的信息。

使用指南

display ipsec policy brief命令可以快速查看所有的安全策略，观察到的简要信息包括：

  • 安全策略的名字和顺序好

  • 安全策略采用的协商方式

  • 安全策略引用的访问控制列表

  • 安全策略引用的对端

  • 本地的IP地址

  • 对端的IP地址

在指定安全策略的名字时，命令显示安全策略的详细信息，显示格式为详细格式。

使用实例

# 显示所有安全策略的简要信息。

<Quidway> display ipsec policy brief
IPsec-policy-Name  Mode     acl ike-peer      Local Address   Remote Address
policy1-100       manual   3100                         150.1.1.1
test-300          isakmp   3120 test
