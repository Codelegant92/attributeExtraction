display time-range

命令功能

display time-range命令用来显示当前时间段的配置和状态。

命令格式

display time-range { time-name | all } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

time-name：指定时间段的名称。

all：所有配置的时间段。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

显示信息中，对于当前处在激活状态的时间段将显示Active，对于非激活状态的时间段将显示Inactive。

由于系统更新ACL状态有一个误差，大约1分钟，而display time-range会采用当前时间对其进行判断，所以有可能出现display time-range看到一个时间段已经激活，而引用它的ACL没有激活。这种情况是正常的。

使用实例

# 显示所有时间段。

<Quidway> display time-range all
Current time is 14:19:16 3-15-2006 Wednesday
Time-range : time1 ( Inactive )
 10:00 to 12:00 daily
Time-range : time2 ( Inactive )
 from 13:00 2006/4/1 to 23:59 2099/12/31  
Time-range : active1 ( Active )
 14:00 to 00:00 daily

# 显示名字为active1的时间段。

<Quidway> display time-range active1
Current time is 14:22:36 3-15-2006 Wednesday
Time-range : active1 ( Active )
 14:00 to 00:00 daily
