display fr lmi-info

命令功能

display fr lmi-info命令用来显示LMI协议帧的统计信息。

命令格式

display fr lmi-info [ interface interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口编号，只能是主接口，三维形式：槽号/卡号/接口号。不指定接口则显示指定接口类型的全部接口信息。

使用指南

LMI协议用于维护当前帧中继链路，协议报文包括状态请求报文和状态报文。

使用实例

# 查看LMI协议帧的统计信息。

<Quidway> display fr lmi-info
Frame relay LMI statistics for interface MFR6/0/0 (DTE, Q933)
  T391DTE = 10 (hold timer 10)
  N391DTE = 5, N392DTE = 3, N393DTE = 4
  out status enquiry = 0, in status = 0
  status timeout = 0, discarded messages = 0
Frame relay LMI statistics for interface Serial6/0/0 (DCE, Q933)
  T392DCE = 15, N392DCE = 3, N393DCE = 4
  in status enquiry = 250, out status = 250
  status enquiry timeout = 0, discarded messages = 15
表1 display fr lmi-info输出信息描述

项目

描述

T391DTE = 10 (hold timer 10)

用户侧轮询定时器。它定义了DTE设备发送状态请求报文的间隔。

N391DTE =

发送全状态查询报文的轮询个数，即计数器值达到N391时，发送全状态查询报文。

N392DTE =

错误门限。表示在被观察的事件总数中发生错误的门限。

N393DTE =

事件计数器。表示被观察的事件总数。

out status enquiry =

发送状态查询消息数。

in status =

接收状态应答消息数。

status timeout =

请求状态报文超时次数。

discarded messages=

丢弃的消息数。

DCE, Q933

DCE端，Q933类型消息格式。LMI消息类型有ANSI、Q933等。
