cops-server shared-key

命令功能

cops-server shared-key命令用于设置COPS服务器组的报文密钥。

undo cops-server shared-key命令用于恢复缺省值。

缺省情况下，COPS服务器组的报文密钥为huawei。

命令格式

cops-server shared-key keystring

undo cops-server shared-key

参数说明

keystring：COPS服务器组的报文密钥。1～16位字符串。

视图

COPS服务器组视图

缺省级别

2：配置级

使用指南

密钥用于对COPS报文进行加密，在设备和COPS服务器上必须配置相同密钥。通过设置报文密钥，可以提高客户端和COPS服务器组之间报文交互的安全性。

当客户端与服务器组端之间收发报文的安全级别较高时，建议配置此选项。

使用实例

# 设置COPS服务器组huawei的报文密钥为hello。

<Quidway> system-view
[Quidway] cops-server group huawei
[Quidway-cops-huawei] cops-server shared-key hello
