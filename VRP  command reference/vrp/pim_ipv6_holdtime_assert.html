pim ipv6 holdtime assert

命令功能

pim ipv6 holdtime assert命令用来设置断言消息的保持时间。

undo pim ipv6 holdtime assert命令用来恢复缺省值。

缺省情况下，断言消息的保持时间是180秒。

命令格式

pim ipv6 holdtime assert interval

undo pim ipv6 holdtime assert

参数说明

interval：指定断言消息的保持时间。整数形式，取值范围是7～65535，单位是秒。

使用指南

该命令与IPv6 PIM视图下的holdtime assert（IPv6）命令作用相同，只是视图不同。缺省情况下，如果接口上没有配置，取IPv6 PIM视图下配置的值；否则取接口视图下的值。

请参见IPv4 PIM配置命令中的holdtime assert（IPv6）。

使用实例

# 设置接口GE1/0/0上断言消息的保持时间是100秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 holdtime assert 100
