fr compression frf9

命令功能

fr compression frf9命令用来使能帧中继压缩功能。

undo fr compression命令用来禁止帧中继的压缩功能。

缺省情况下，禁止帧中继压缩功能。

命令格式

fr compression frf9

undo fr compression frf9

参数说明

无

帧中继子接口视图

缺省级别

2：配置级

使用指南

在配置本命令前，必须先配置接口的DLCI。

只能在点到点类型的帧中继子接口下配置该命令，且报文封装类型必须是IETF。如果报文封装类型是nonstandard，当执行该命令使能帧中继FRF.9压缩时，系统会提示nonstandard封装类型不支持FRF.9压缩。

说明：

  • FRF.9压缩适用于低速链路。

  • MFR链路不使用FRF.9压缩功能。

  • FRF.9压缩有一个同步的过程。压缩时如果报文乱序，通常会压缩不成功。

对于帧中继主接口或点到多点类型的帧中继子接口，使用命令fr map ip compression使能帧中继FRF.9压缩。

使用实例

# 在点到点类型的帧中继子接口Serial1/0/0.1上使能帧中继压缩。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol fr
[Quidway-Serial1/0/0] quit
[Quidway] interface serial 1/0/0.1 p2p
[Quidway-Serial1/0/0.1] fr dlci 100
[Quidway-fr-dlci-Serial1/0/0.1-100] quit
[Quidway-Serial1/0/0.1] fr compression frf9
