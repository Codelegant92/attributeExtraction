display qinq statistics

命令功能

display qinq statistics用来显示指定接口下用户组的QinQ的统计信息。

命令格式

display qinq statistics [ interface interface-type interface-number [ .subinterface-number ] [ vlan-group group-id ] ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：需要查看QinQ的统计信息的用户组所在的接口类型和编号。

vlan-group group-id：用户组号，十进制整数形式，取值范围是1～16。

verbose：显示用户组详细的QinQ统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display qinq statistics命令来查看QinQ的统计信息前，应该先进行如下配置：

  • 在VLAN-Group接口视图下通过命令statistic enable使能QinQ报文的统计功能。

  • 使用命令dot1q termination vid或qinq termination pe-vid ce-vid或qinq stacking vid，将子接口终结或者Stacking的Tag范围与用户VLAN组绑定。

项目

描述

GigabitEthernet1/0/0.1 vlan-group[1]:Single

子接口的名称和子接口下用户VLAN组的名称及模式。

QinQ termination tag:10/100-200

QinQ终结子接口终结的Tag范围。“/”左侧是外层Tag（pe-vid）；“/”右侧是内层Tag（ce-vid）。

Dot1q termination tag:80-90

Dot1q终结子接口终结的Tag范围。

Stacking tag:20-50

Stacking子接口进行Stacking操作的Tag范围。

Statistics on VRP:

VRP平台的统计信息。

0 Packets received, 0 bytes

收到报文的数量。

0 Error Packets received

收到错误报文的数量。

0 Packets transmitted, 0 bytes

发送报文的数量。

0 Error Packets transmitted

发送错误报文的数量。

Total statistics for QinQ on VRP

VRP平台对QinQ报文的所有统计信息。

0 Packets received, 0 bytes

接收的报文总数。

0 Packets transmitted, 0 bytes

发送的报文总数。
