ipv4-family

命令功能

ipv4-family命令用来进入各个BGP的IPv4地址族视图。

undo ipv4-family命令用来删除相应地址族视图下的所有配置。

命令格式

ipv4-family unicast

ipv4-family vpnv4 [ unicast ]

ipv4-family vpn-instance vpn-instance-name

undo ipv4-family { vpnv4 [ unicast ] | vpn-instance vpn-instance-name }

参数说明

unicast：进入单播地址族视图。

vpnv4：使用该参数将进入BGP-VPNv4地址族视图。

vpn-instance vpn-instance-name：将指定的VPN实例与IPv4地址族进行关联，使用该参数将进入BGP-VPN实例视图。

视图

缺省级别

2：配置级

使用指南

undo ipv4-family命令不能用来删除BGP IPv4单播地址族视图。

使用实例

# 进入BGP-IPv4单播地址族视图。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4]

# 进入BGP-VPNv4地址族视图。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family vpnv4
[Quidway-bgp-af-vpnv4]
