display ospfv3 lsdb

命令功能

display ospfv3 lsdb命令用来显示OSPFv3的链路状态数据库信息。

命令格式

display ospfv3 [ process-id ] lsdb [ external | grace | inter-prefix | inter-router | intra-prefix | link | network | router ] [ link-state-id ] [ originate-router advertising-router-id ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display ospfv3 [ process-id ] lsdb statistics [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

external：显示数据库中AS-external LSA的信息。

grace：显示数据库中Grace-LSA的信息。

inter-prefix：显示数据库中Inter-area-prefix LSA的信息。

inter-router：显示数据库中Inter-area-router LSA的信息。

intra-prefix：显示数据库中Intra-area-prefix LSA的信息。

link：显示数据库中Link-local LSA的信息。

network：显示数据库中Network-LSA的信息。

router：显示数据库中Router-LSA的信息。

link-state-id：链路状态ID，IPv4地址形式。

originate-router advertising-router-id：指定发布LSA的路由器的Router ID。

statistics：显示LSDB的统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
