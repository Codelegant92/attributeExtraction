igmp-snooping router-learning

命令功能

igmp-snooping router -learning命令用来使能动态学习路由端口。

undo igmp-snooping router -learning命令用来去使能动态学习路由端口。

缺省情况下，路由器使能了动态学习路由端口。

命令格式

igmp-snooping router-learning

undo igmp-snooping router-learning

参数说明

无

视图

VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

对于已经绑定到VSI下的VLAN，需要在VSI视图下配置IGMP Snooping的功能特性。

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定了VSI；

  • VLAN内包含VE接口。

使用实例

# 配置VLAN 10内端口为动态学习路由端口。

<Quidway> system-view
[Quidway] vlan 10
[Quidway-vlan10] igmp-snooping router-learning

# 配置VSI vsi0内端口为动态学习路由端口。

<Quidway> system-view
[Quidway] vsi vsi0
[Quidway-vsi-vsi0] igmp-snooping router-learning
