least active-linknumber（IP-Trunk接口视图）

命令功能

least active-linknumber命令用来设置IP-Trunk接口成员链路UP的下限阈值。

undo least active-linknumber命令用来恢复Trunk接口成员链路Up的下限阈值为缺省值。

命令格式

least active-linknumber link-number

undo least active-linknumber

参数说明

link-number：成员链路UP的下限阈值。整数形式，取值范围是1～16。缺省值是1。

视图

IP-Trunk接口视图

缺省级别

2：配置级

使用指南

当捆绑组中处于UP状态的成员端口数小于设置的下限阈值时，IP-Trunk接口状态转为Down。当捆绑组中处于UP状态的成员端口数达到设置的下限阈值时，IP-Trunk接口状态转为UP。

使用实例

# 设置当不少于3条的成员链路处于UP状态时，IP-Trunk 1为UP状态。

<Quidway> system-view
[Quidway] interface ip-trunk 1
[Quidway-Ip-Trunk1] least active-linknumber 3
