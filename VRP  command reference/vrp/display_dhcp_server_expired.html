display dhcp server expired

命令功能

display dhcp server expired命令用来查看DHCP地址池中已经超期的租约，在一定条件下这类租约的地址将被分配给其它DHCP客户端。

命令格式

display dhcp server expired { all | interface [ interface-type interface-numaber ] | ip ip-address | pool [ pool-name ] | vlan vlan-id }

参数说明

interface [ interface-type interface-numaber ]：指定要查看的租约超期的接口地址池，不指定参数interface-type interface-numaber则表示查看所有的接口地址池。

ip ip-address：指定要查看的租约超期的IP地址。

pool [ pool-name ]：指定全局地址池的名字。如果不指定参数pool-name则表示查看所有全局地址池。字符串形式，长度范围是1～35。

vlan vlan-id：表示范围为指定的VLAN的接口地址池，没指定参数vlan-id时指所有VLAN的接口地址池。

all：表示查看所有超期的IP地址。

使用指南

在DHCP地址池中的地址无法分配时，可以使用display dhcp server expired来是否有超期的地址仍然在使用，可以释放掉此地址，分配给其他的客户端。

使用实例

# 查看DHCP地址池的超期租约信息。

<Quidway> display dhcp server expired all
Global pool:
 IP address   Hardware address    Lease expiration      Type
 2.2.2.2      4444-4444-4444      NOT Used              Manual
Interface pool:
 IP address   Hardware address    Lease expiration      Type
