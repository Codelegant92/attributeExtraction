display info-center filter-id

命令功能

display info-center filter-id命令用来显示过滤表中单条日志ID信息或过滤列表中所有的日志ID信息。

命令格式

display info-center filter-id [ id ]

参数说明

id：每一条日志唯一对应的标识。整数形式，取值范围0～4294967295。

使用指南

被屏蔽的ID集合称为ID过滤列表。

在不指定ID时，显示的是已添加的所有过滤ID信息。ID是各功能模块注册日志时对应的唯一标志。

如果用户不希望某条日志输出到文件、缓冲区等，可以到数据字典里找该日志对应的ID，通过添加过滤ID命令info-center filter-id将ID添加到过滤表中，然后通过display info-center filter-id查看该ID是否已经添加为过滤ID。

使用实例

# 显示过滤表中ID为1073844227的过滤信息。

<Quidway> display info-center filter-id 1073844227
ID:              1073844227
Content:         The overloadState of IO board [ULONG] in master board is not ov
erload suspend.
Filtered Number: 0
