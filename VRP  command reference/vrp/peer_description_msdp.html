peer description（MSDP）

命令功能

peer description命令用来为MSDP对等体配置描述性文本。

undo peer description命令用来恢复缺省配置。

缺省情况下，MSDP对等体没有描述性文本。

命令格式

peer peer-address description text

undo peer peer-address description

参数说明

peer-address：表示MSDP对等体的地址。点分十进制形式。

text：描述性文本。区分大小写，最长为80个字符。

视图

公网实例MSDP视图、VPN实例MSDP视图

缺省级别

2：配置级

使用指南

管理员可以通过配置描述性文本方便地区分MSDP对等体。

使用实例

# 在公网实例中，为路由器125.10.7.6添加描述性文本：ClientA，指明该路由器是客户A。

<Quidway> system-view
[Quidway] msdp
[Quidway-msdp] peer 125.10.7.6 description ClientA
