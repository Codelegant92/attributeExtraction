speed（用户界面视图）

命令功能

speed命令用来设置用户界面的传输速率。

undo speed命令用来恢复用户界面的缺省传输速率。

缺省情况下，传输速率为9600bit/s。

命令格式

speed speed-value

undo speed

参数说明

speed-value：指定用户界面的传输速率，单位为bit/s。

视图

用户界面视图

缺省级别

3：管理级

使用指南

只有串口工作在异步交互方式下时，配置有效。

异步串口支持的传输速率有：

  • 300bit/s

  • 600bit/s

  • 1200bit/s

  • 4800bit/s

  • 9600bit/s

  • 19200bit/s

  • 38400bit/s

  • 57600bit/s

  • 115200bit/s

使用实例

# 将用户界面的传输速率为19200bit/s。

<Quidway > system-view
[Quidway] user-interface aux 0
[Quidway-ui-aux0] speed 19200
