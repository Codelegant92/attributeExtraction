dialer timer autodial

命令功能

dialer timer autodial命令用来配置DCC自动拨号的间隔。

undo dialer timer autodial命令用来恢复缺省间隔时间。

命令格式

dialer timer autodial seconds

undo dialer timer autodial

参数说明

seconds：发起下次呼叫尝试的间隔时间。整数形式，取值范围是1～604800，单位是秒。缺省值是300秒。

缺省级别

2：配置级

使用指南

在配置该命令前，必须先执行命令dialer user设置对端用户名，或执行命令dialer enable-circular使能轮询DCC。

该命令必须与dialer route配置命令中的关键字autodial结合使用。配置该命令后，DCC将每隔seconds时间自动尝试拨号一次，直至连接建立。自动拨号功能无需数据包的触发，并且在连接建立后不会因超时而自动挂端，即dialer timer idle命令配置对其无效。

使用实例

# 在接口Serial1/0/0上设置DCC自动呼叫间隔时间为60秒。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] dialer timer autodial 60
