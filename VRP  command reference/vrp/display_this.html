display this

命令功能

display this命令用来显示系统当前视图的运行配置。

命令格式

display this [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

视图

所有视图

缺省级别

1：监控级

使用指南

当用户在某一视图下完成一组配置之后，需要验证是否配置正确，则可以执行display this命令来查看当前生效的参数。

对于某些正在生效的配置参数，如果与缺省工作参数相同，则不显示；对于某些参数，虽然用户已经配置，但如果这些参数所在的功能如果没有生效，则不予显示。

在不同接口视图下使用该命令，将直接显示相应接口下的配置；在不同协议视图下使用该命令，将显示相应协议视图的配置；在协议子视图下，不显示任何配置信息。

使用实例

# 显示路由器系统当前视图下生效的运行配置参数。

[Quidway] display this
#
 sysname Quidway
#
 command-privilege level 0 view system interface
#
