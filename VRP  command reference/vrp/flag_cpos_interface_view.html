flag（CPOS接口视图）

命令功能

flag命令用来设置SONET/SDH帧的开销字节。

undo flag命令用来恢复缺省设置。

命令格式

flag { j0 j0-string | j1 j1-string | c2 c2-value }

undo flag { j0 | j1 | c2 }

参数说明

j0：再生段跟踪字节。字符串形式，长度范围是1～15。缺省值是0x01。

j1：通道跟踪字节。字符串形式，长度范围是1～15。缺省值是NetEngine。

c2：信号标记字节。整数形式，十六进制，取值范围是00～FF。缺省值是0x02。

缺省级别

2：配置级

使用指南

SONET/SDH帧具有丰富的开销字节，可完成对传输网的分层管理等运行维护功能OAM（Operation Administration & Maintenance）。j1、j0和c2主要用于在不同国家、不同地区、或不同厂商的设备之间提供互通支持。

使用实例

# 设置CPOS 1/0/0的通道跟踪字节j1为字符串HUAWEI。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] flag j1 HUAWEI
