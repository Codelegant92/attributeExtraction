display mpls te session-entry

命令功能

display mpls te session-entry用来查看Session Entry数据结构的详细信息。

命令格式

display mpls te session-entry [ ingress-lsr-id tunnel-id egress-lsr-id ]

参数说明

ingress-lsr-id：入节点的LSR ID，点分十进制格式。

tunnel-id：隧道标识符。整数形式，取值范围是0～65535。

egress-lsr-id：出节点的LSR ID，点分十进制格式。

使用实例

# 标准DS-TE模式下，查看SessionEntry数据结构的详细信息。

<Quidway> display mpls te session-entry
  Ingress-ID                :  1.1.1.1
  Tunnel-ID                 :  100
  Egress-ID                 :  2.2.2.2
  Crlsp num                 :  1
  First TunnelTable index   :  0
  LSP No.     :  1
  LSP ID      :  1.1.1.1:100:1
  In/Out IF   :  -/S1/0/0
  Bandwidth(Kbit/sec):
    CT0 : 100         CT1 : 0
    CT2 : 0           CT3 : 0
    CT4 : 0           CT5 : 0
    CT6 : 0           CT7 : 0
