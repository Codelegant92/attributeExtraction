reset control-flap

命令功能

reset control-flap命令用来清除接口的control-flap特性的抑制相关值。

命令格式

reset control-flap { penalty | counter } interface interface-type interface-number

参数说明

penalty：清除接口的control-flap惩罚值，并解除接口的抑制状态。

counter：清除接口的振荡次数。

interface-type：接口类型。

interface-number：接口编号。

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除接口POS1/0/0的control-flap特性抑制惩罚值。

<Quidway> reset control-flap penalty interface pos 1/0/0
