radius-server template

命令功能

radius-server template命令用来进入RADIUS模板视图，如果指定的RADIUS服务器模板名不存在，则同时创建一个新的RADIUS服务器模板。

undo radius-server template命令用来删除一个RADIUS服务器模板。

命令格式

radius-server template template-name

undo radius-server template template-name

参数说明

template-name：RADIUS服务器模板名。字符串形式，长度范围是1～32。字符包括大写字母A～Z、小写字母a～z、数字0～9、点号“.”、短线“-”和下划线“_”。

缺省级别

2：配置级

使用指南

在RADIUS模板视图下可以对此RADIUS服务器模板进行具体配置。

系统最多支持配置128个RADIUS服务器模板。只有当一个模板没有用户使用时，才能修改模板内容。

删除服务器模板时，如果此模板有用户正在使用或者没有被用户使用但被某个域引用，则系统会提示该模板正在被用户或域使用，无法删除。

使用实例

# 创建名为test1的RADIUS服务器模板并进入相应的RADIUS模板视图。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1]
