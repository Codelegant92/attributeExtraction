cops-server pep-id

命令功能

cops-server pep-id命令用于配置本设备的COPS客户端标识，

undo cops-server pep-id命令用于删除本设备的COPS客户端标识。

命令格式

cops-server pep-id client-id

undo cops-server pep-id

参数说明

client-id：本设备的COPS客户端标识，通常可设置为本设备的某个Loopback接口的IP地址。字符串形式，范围为1～64个字符。

视图

COPS服务器组视图

缺省级别

2：配置级

使用指南

客户端标识（PEP ID）用于COPS服务器确认本客户端的身份。一般情况下，可配置为设备Loopback端口的IP地址。

客户端标识基于COPS服务器组进行配置，即对应不同的COPS服务器组，可以设置不同的客户端标识。缺省的客户端标识是huawei。

当COPS服务器被使用时，COPS客户端标识不可删除和修改。

使用实例

# 配置COPS客户端标识为10.10.10.1。

<Quidway> system-view
[Quidway] cops-server group huawei
[Quidway-cops-huawei] cops-server pep-id 10.10.10.1
