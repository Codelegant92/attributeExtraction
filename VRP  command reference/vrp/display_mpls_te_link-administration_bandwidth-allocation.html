display mpls te link-administration bandwidth-allocation

命令功能

display mpls te link-administration bandwidth-allocation命令用来查看指定的或所有的MPLS TE接口的带宽分配情况。

命令格式

display mpls te link-administration bandwidth-allocation [ interface interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：查看指定接口的带宽分配情况。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

表1 display mpls te link-administration bandwidth-allocation命令输出信息描述

项目

描述

Bandwidth Constraint Model
带宽约束模型，有如下三种模型：

  • RDM：Russian Dolls Model

  • RAM：Maximum Allocation Model

  • Extended Maximum Allocation Model (Extended MAM)

Maximum Link Reservable Bandwidth(Kbit/sec)

链路的最大可预留带宽

Reservable Bandwidth BCi(0≤i≤7)

预留给BCi的带宽

Downstream Bandwidth

下游的带宽

IPUpdown Link Status
IP链路状态：

  • UP

  • DOWN

PhysicalUpdown Link Status
物理链路状态：
  • UP：链路可用
  • DOWN：链路不可用

TE-CLASS

TE的类别

CT

Class Type

PRIORITY

隧道的抢占优先级

BW RESERVED

已经保留给该CT的带宽

BW AVAILABLE

可以预留给该CT的带宽

DOWNSTREAM RSVPLSPNODE COUNT

下游RSVP LSP节点的数目
