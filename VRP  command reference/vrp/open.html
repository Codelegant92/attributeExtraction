open

命令功能

open命令用来与远程FTP server建立控制连接。命令中VPN实例名，标识FTP到哪一个VPN实例中的FTP server。

命令格式

open [ -a source-ip-address | -i interface-type interface-num ] [ host [ port-number ] ] [ vpn-instance vpn-instance-name ]

参数说明

-a source-ip-address：指定源IP地址。

host：远程FTP server的IP地址或主机名。

-i interface-type interface-num：指定本端可以连接出去的源接口。

port-number：远程FTP server的端口号，取值范围0～65535。

vpn-instance-name：服务器端的VPN实例名，字符串形式，长度范围是1～31。
