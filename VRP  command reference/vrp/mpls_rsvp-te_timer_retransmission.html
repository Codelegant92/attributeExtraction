mpls rsvp-te timer retransmission

命令功能

mpls rsvp-te timer retransmission命令用来在接口上配置重传功能参数。

undo mpls rsvp-te timer retransmission命令用来关闭重传功能。

命令格式

mpls rsvp-te timer retransmission { increment-value increment | retransmit-value interval }*

undo mpls rsvp-te timer retransmission [ increment-value | retransmit-value ]

参数说明

increment：重传增量。整数形式，取值范围是1～10，缺省值为1。

interval：重传定时器间隔。整数形式，取值范围是500～3000，单位是毫秒，缺省值为500毫秒。

使用指南

执行该命令前，应先使能接口的RSVP-TE功能。

使用实例

# 设定重传定时器间隔为500毫秒，重传增量值为2。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] mpls
[Quidway-Pos1/0/0] mpls te
[Quidway-Pos1/0/0] mpls rsvp-te
[Quidway-Pos1/0/0] mpls rsvp-te timer retransmission retransmit-value 500 increment-value 2
