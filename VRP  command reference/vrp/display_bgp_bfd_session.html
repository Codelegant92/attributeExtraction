display bgp bfd session

命令功能

display bgp bfd session命令用来显示BGP建立的BFD会话信息。

命令格式

display bgp bfd session { [ vpnv4 vpn-instance vpn-instance-name ] peer ipv4-address | all } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpnv4 vpn-instance vpn-instance-name：指定的IPv4私网实例。

peer ipv4-address：对等体的IPv4地址。

all：查看BGP建立的所有BFD会话。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示BGP建立的所有BFD会话。

<Quidway> display bgp bfd session all

项目

描述

Local_Address

本地地址

Peer_Address

邻居地址

LD/RD

本地/远端标识符

Interface
创建BFD会话的接口。
说明：

只有使用直连接口建立EBGP邻居时，才显示创建BFD会话的实际接口。其它情况下显示为Unknow。

Tx-interval(ms)

发送BFD报文的时间间隔（单位为毫秒）

Rx-interval(ms)

接收BFD报文的时间间隔（单位为毫秒）

Session-State
BFD状态：

  • BFD global disable

  • BFD session number exceed

  • Detect down

  • Init

  • Receive admin down

  • Up
