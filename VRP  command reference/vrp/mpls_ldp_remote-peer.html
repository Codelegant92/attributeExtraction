mpls ldp remote-peer

命令功能

mpls ldp remote-peer命令用来创建远端对等体实体并进入远端对等体视图。

undo mpls ldp remote-peer命令用来删除远端对等体实体。

缺省情况下，没有创建远端对等体实体。

命令格式

mpls ldp remote-peer remote-peer-name

undo mpls ldp remote-peer remote-peer-name

参数说明

remote-peer-name：指定远端对等体名称。字符串形式，不支持空格，不区分大小写，长度范围是1～32。

使用指南

创建远端对等体的时候，必须在对等体的两端分别执行此命令。

使用实例

# 建立远端对等体。

<Quidway> system-view
[Quidway] mpls ldp remote-peer BJI
[Quidway-mpls-ldp-remote-bji] quit
