description（Tunnel接口视图）

命令功能

description命令用来设置当前Tunnel接口的描述信息。

undo description命令用来删除当前Tunnel接口的描述信息。

缺省情况下，系统不显示接口的默认描述信息。Tunnel接口默认描述信息为“HUAWEI, Quidway Series, Tunnel tunnel-number Interface”。

命令格式

description text

undo description

参数说明

text：Tunnel接口描述信息。字符串形式，长度范围是1～242。

视图

Tunnel接口视图

缺省级别

2：配置级

使用指南

无

使用实例

# 设置Tunnel1/0/0的描述信息。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] description This is a tunnel from 1.1.1.1 to 2.2.2.2

# 删除Tunnel1/0/0的描述信息。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] undo description
