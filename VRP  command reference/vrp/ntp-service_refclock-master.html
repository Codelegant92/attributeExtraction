ntp-service refclock-master

命令功能

ntp-service refclock-master命令用来设置外部参考时钟或本地时钟作为NTP主时钟，为其它设备提供同步时间。

undo ntp-service refclock-master命令用来取消NTP主时钟设置。

缺省情况下，没有设置NTP主时钟。

命令格式

ntp-service refclock-master [ ip-address ] [ stratum ]

undo ntp-service refclock-master [ ip-address ]

参数说明

ip-address：本地参考时钟IP地址127.127.t.u。t的取值范围是0～37，目前只能取t=1，表示本地参考时钟；u的取值范围是0～3，表示NTP的进程号。当不指定IP地址时，默认设置本地时钟127.127.1.0为NTP主时钟。

stratum：NTP主时钟所处的层数。整数形式，取值范围是1～15。缺省值是8。值越小表示时钟准确度越高。

使用指南

在网络中选取时钟精度比较高的设备，执行ntp-service refclock-master配置为NTP的主时钟，为其它设备提供同步时间。

使用实例

# 设置本地设备时钟作为NTP主时钟，层数为3。

<Quidway> system-view
[Quidway] ntp-service refclock-master 3
