display { multicast-vlan | user-vlan } vlan

命令功能

display multicast-vlan vlan [ vlan-id ] 命令用来显示组播VLAN的信息。

display user-vlan vlan [ vlan-id ] 命令用来显示用户VLAN的信息。

命令格式

display { multicast-vlan | user-vlan } vlan [ vlan-id ]

参数说明

vlan vlan-id：需要显示组播VLAN或者用户VLAN编号。取值范围是1～4094。

视图

所有视图

缺省级别

1：监控级

使用指南

使用命令display multicast-vlan vlan 不指定参数vlan-id时，显示所有组播VLAN的信息。

使用命令display user-vlan vlan不指定参数vlan-id时，显示所有用户VLAN的信息。

使用实例

# 显示所有组播VLAN的信息

<Quidway> display multicast-vlan vlan
Total multicast vlan    3
multicast-vlan    user-vlan   snooping-state
--------------------------------------------------------
1000                1                 Enable  
1020                0                 Enable  
1030                0                 Enable
表1 display multicast-vlan vlan输出信息描述

项目

描述

Total multicast vlan

组播VLAN的信息的条数。

multicast-vlan

组播VLAN编号。

user-vlan

该组播VLAN下包含的用户VLAN个数。

snooping-state
IGMP Snooping状态为Enable：

  • 已使能IGMP Snooping。

# 显示所有用户VLAN的信息

<Quidway> display user-vlan vlan
 Total user vlan   6
 user-vlan  snooping-state  multicast-vlan  snooping-state
 ---------------------------------------------------------
 3              Enable              2               Enable
 223            Enable              222             Enable
 232            Enable              231             Enable
 601            Enable              600             Enable
 2001           Enable              2000            Enable
 2002           Enable              2000            Enable
表2 display user-vlan vlan输出信息描述

项目

描述

Total user vlan

用户VLAN的信息的条数。

user-vlan

用户VLAN编号。

snooping-state
用户VLAN的IGMP Snooping状态：

  • enable：已是能IGMP Snooping。

multicast-vlan

该用户VLAN所属组播VLAN的编号。

snooping-state

该组播VLAN的IGMP Snooping状态。
