outbound recording-scheme

命令功能

outbound recording-scheme命令用来设置对于路由器做为客户端进行的操作的记录策略。

undo outbound recording-scheme命令用来删除记录策略，即不进行相应的记录。

缺省情况下，不记录路由器做为客户端进行的操作。

命令格式

outbound recording-scheme recording-scheme-name

undo outbound recording-scheme

参数说明

outbound：记录连接信息，目前支持对Telnet客户端的记录。

recording-scheme-name：记录方案名称，字符串形式，长度范围是1～32。不区分大小写，方案名需要符合Windows的命名规范，不能包括斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“"”和尖括号“<”、“>”等字符。

使用指南

命令中引用的记录方案名称必须是已经创建完成的记录方案。

使用实例

# 设置路由器做为客户端进行的操作的记录策略，方案名为test。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] outbound recording-scheme test
