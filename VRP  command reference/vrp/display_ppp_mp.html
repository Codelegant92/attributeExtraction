display ppp mp

命令功能

display ppp mp命令用来显示MP的捆绑信息及捆绑链路的统计信息，如果不指定接口类型和接口编号，将显示所有虚拟接口模板上的绑定信息和捆绑链路的统计信息。

命令格式

display ppp mp [ interface interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type interface-number：接口类型和接口编号，用于指定查看的接口。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。字符串格式，支持空格，区分大小写。

使用指南

用此命令可以查看到物理接口是否成功绑定到了指定的虚拟接口模板上。同时还可以看到被绑定的方式是直接绑定还是通过验证方式的绑定等。

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行接口的故障诊断等。

使用实例

# 查看Virtual-Template接口的MP绑定信息。

<Quidway> display ppp mp interface virtual-template 1
Template is Virtual-Template1
 Bundle quid0, 2 member, slot 4, Master link is Virtual-Template1:0
  0 lost fragments, 0 reordered, 0 unassigned, 0 interleaved,
 sequence 0/0 rcvd/sent
 The bundled son channels are:
      Serial4/0/0:0
      Serial4/0/0:1
表1 display ppp mp interface virtual-template命令输出信息描述

项目

描述

Template is Virtual-Template1

使用的虚拟接口模板：Virtual-Template1。

Bundle quid0

被绑定的物理接口的用户名是quid0。当配置通过终端标识符进行验证绑定时，此处显示的唯一标识这台路由器的终端标识符。

2 member

绑定的通道数目：2。

slot 4

绑定在4号槽。

Master link is Virtual-Template1:0

主通道。

0 lost fragments

接收方向丢弃的分片。

0 reordered

接收方向重组报文数。

0 unassigned

接收方向等待重组的分片数。

0 interleaved

发送方向LFI的分片个数。

sequence 0/0 rcvd/sent

接收序列号/发送序列号。

The bundled son channels are

此MP中捆绑的子通道。

# 查看MP-Group接口的MP绑定信息和捆绑链路的运行状态。

<Quidway> display ppp mp interface mp-group 1/0/1
Mp-group is Mp-group1/0/1
 ===========Sublinks status begin======           
 Serial1/0/1:1 physical UP,protocol UP        
 Serial1/0/1:2 physical UP,protocol UP            
 Serial1/0/1:3 physical UP,protocol DOWN(damping)        
 Serial1/0/1:4 physical UP,protocol DOWN     
 ===========Sublinks status end========       
 Bundle Multilink, 2 member, slot 1, Master link is Mp-group1/0/1 
  0 lost fragments, 0 reordered, 0 unassigned, 0 interleaved,
 sequence 0/0 rcvd/sent                 
 The bundled son channels are:           
      Serial1/0/1:1                    
      Serial1/0/1:2
表2 display ppp mp interface mp-group命令输出信息描述

项目

描述

Mp-group is Mp-group1/0/1

使用的MP-Group接口是MP-Group1/0/1。

Serial physical, protocol

捆绑链路的物理接口及其物理状态和链路层协议状态。

  • UP：表示接口的物理层或链路层工作正常。

  • DOWN：表示接口的物理层或链路层出现了故障。其中damping表示接口因为震荡频繁而被抑制。

Bundle Multilink, member, slot, Master link is

表示多链路捆绑的有效捆绑链路数、槽号和绑定的接口。

lost fragments

接收方向丢弃的分片。

reordered

接收方向重组报文数。

unassigned

接收方向等待重组的分片数。

interleaved

发送方向LFI的分片个数。

sequence rcvd/sent

接收序列号/发送序列号。

The bundled son channels are

此MP中捆绑的捆绑链路。
