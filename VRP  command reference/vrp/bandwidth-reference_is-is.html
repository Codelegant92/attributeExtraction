bandwidth-reference (IS-IS)

命令功能

bandwidth-reference命令用来设置接口开销自动计算功能中所使用的带宽参考值。

undo bandwidth-reference命令用来取消此设置。

缺省情况下，带宽参考值为100。

命令格式

bandwidth-reference value

undo bandwidth-reference

参数说明

value：接口的带宽参考值。整数形式，取值范围是1～2147483648，单位是Mbit/s。

使用实例

# 设置系统带宽的参考值为1000（兆比特每秒）

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] bandwidth-reference 1000
