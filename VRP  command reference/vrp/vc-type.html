vc-type

命令功能

vc-type命令用来在NQA视图下配置建立L2VPN VC的协议类型。

undo vc-type命令用来取消在NQA视图下配置的L2VPN VC的协议类型。

缺省情况下，建立L2VPN VC的协议类型为LDP方式。

命令格式

vc-type { ldp | bgp }

undo vc-type

参数说明

ldp：指定LDP作为信令协议传播内层标签。

bgp：指定BGP作为信令协议传播内层标签。

视图

NQA视图

缺省级别

2：配置级

使用指南

使用vc-type命令前，需要先在NQA视图下配置NQA测试例的测试类型为PWE3 Trace或者PWE3 Ping，否则无法配置vc-type命令。

PWE3 Ping不支持VC-TYPE配置为BGP。

使用实例

# 在NQA视图下配置建立L2VPN VC的协议类型为BGP。

<Quidway> system-view
[Quidway] nqa test-instance admin pwe3
[Quidway-admin-pwe3] test-type pwe3trace
[Quidway-admin-pwe3] vc-type bgp
