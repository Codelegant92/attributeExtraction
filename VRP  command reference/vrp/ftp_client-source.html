ftp client-source

命令功能

ftp client-source命令用于设置当设备作为FTP客户端时的源为指定源，其中包括LoopBack源地址和LoopBack源接口。

命令格式

ftp client-source { -a source-ip-address | -i interface-type interface-num }

undo ftp client-source

参数说明

-a source-ip-address：本端路由器的LoopBack地址。

-i interface-type interface-num：本端路由器的LoopBack接口。

使用指南

ftp登录命令如果不带源选项，则使用该命令配置的默认源，即ftp client-source指定的源；如果ftp命令显式指定源，即登录命令带源选项，则使用显式源。从服务器端查看当前FTP连接，显示的用户IP就是用户指定的源IP或指定LoopBack接口下的主IP。

使用实例

# 设置FTP客户端的源地址为1.1.1.1。

<Quidway> ftp client-source -a 1.1.1.1
Info: Succeeded in setting the source address of the FTP client to 1.1.1.1.
