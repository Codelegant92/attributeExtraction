display qos pql

命令功能

display qos pql命令用来显示优先列表的内容。

命令格式

display qos pql [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display qos pql命令可以观察到的信息包括：

  • PQ列表号

  • 队列类型

  • 队列配置参数

使用缺省配置的项不被显示。

使用实例

# 显示优先列表。

<Quidway> display qos pql
