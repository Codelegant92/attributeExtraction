display channel

命令功能

display channel命令用来显示信息通道的内容。

display channel不带参数表示显示所有通道的设置情况。

命令格式

display channel [ channel-number | channel-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

channel-number：通道号可以取的值为0～9，即系统可以有10个通道。

channel-name：通道名。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行命令display channel可以显示，日志、告警和debugging信息是否允许从指定的通道通过和允许通过信息的级别。在不指定任何的参数情况下，显示日志、告警和debugging信息在所有通道中的是否允许通过和允许通过信息的级别。

使用实例

# 显示0号信息通道的内容。

<Quidway> display channel 0
channel number:0, channel name:console
MODU_ID  NAME     ENABLE LOG_LEVEL     ENABLE TRAP_LEVEL    ENABLE DEBUG_LEVEL
ffff0000 default  Y      warning       Y      debugging     Y      debugging
表1 display channel命令输出信息描述

项目

描述

MODU_ID

模块号，ffff0000表示缺省的条目。

NAME

模块名，default表示缺省的模块。

ENABLE（first）

日志信息的使能状态。

LOG_LEVEL

允许输出的日志信息的级别。

ENABLE（second）

告警信息的使能状态。

TRAP_LEVEL

允许输出的告警信息的级别。

ENABLE（third）

调试信息的使能状态。

DEBUG_LEVEL

允许输出的调试信息的级别。
