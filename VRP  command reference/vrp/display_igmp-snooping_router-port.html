display igmp-snooping router-port

命令功能

display igmp-snooping router-port命令用来显示路由端口信息。

命令格式

display igmp-snooping router-port { vlan vlan-id | vsi vsi-name }

参数说明

vlan vlan-id：显示指定VLAN的路由端口信息。vlan-id 表示VLAN编号，取值范围是1～4094。

vsi vsi-name ：显示指定VSI的路由端口信息。vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

使用指南

只有当路由端口处于Up状态时，使用本命令时才显示该端口的信息。

使用实例

# 显示VLAN 10的路由端口表

<Quidway> display igmp-snooping router-port vlan 10
Port Name                         UpTime           Expires         Flags
 ---------------------------------------------------------------------
 VLAN 10, 1 router-port(s)
 GigabitEthernet2/0/1            00:00:17         --            STATIC
表1 display igmp-snooping router-port vlan命令输出信息描述

项目

描述

Port Name

端口类型和编号。

UpTime

端口作为路由端口的存在时间。

Expires

端口作为路由端口的剩余老化时间。

Flags
路由端口的类型，显示为以下两种类型中的一种或它们的组合：

  • STATIC：静态路由器端口

  • DYNAMIC：动态路由器端口

# 显示VSI v123的路由端口表

<Quidway> display igmp-snooping router-port vsi v123

 ---------------------------------------------------------------------
 VSI v123, 1 router-port(s)
 PW(1.1.1.1/123)                 00:05:16      --            STATIC
