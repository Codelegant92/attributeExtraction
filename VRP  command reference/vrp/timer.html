timer

命令功能

timer命令用来设置握手报文的发送时间间隔。

undo timer命令用来恢复握手报文的发送时间间隔为缺省值。

缺省情况下，握手报文的发送时间间隔是10秒。

命令格式

timer interval

undo timer

参数说明

interval：指定握手报文的发送时间间隔。十进制整数形式，取值范围是1～80，单位是秒。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。该取值必须小于或等于有效保留时间的三分之一。

说明：

配置握手超时后保持状态的时间，请参见holdtime命令

使用实例

# 设置握手报文定时发送的时间间隔为3秒。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] timer 3
