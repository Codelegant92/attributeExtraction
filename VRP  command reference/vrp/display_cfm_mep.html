display cfm mep

命令功能

display cfm mep命令用来查看指定MD和MA内的MEP的信息，包括：

  • 所在MD的名称和级别

  • 所在MA的名称和MA关联的VLAN

  • MEP的ID

  • 所在的接口

  • CCM消息发送功能的使能状态

  • 类型

命令格式

display cfm mep [ md md-name [ ma ma-name [ mep-id mep-id ] ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

md md-name：显示指定MD内MEP的信息。md-name表示MD的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。不输入本参数时，显示本设备上所有MD和MA内的MEP的信息。

ma ma-name：显示指定MA内MEP的信息。ma-name表示MA的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。输入参数md-name，但不输入本参数时，显示本设备上指定MD内所有MA内的MEP的信息。

mep-id mep-id：显示指定MEP的信息。mep-id表示MEP的ID，整数形式，取值范围是1～8191。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

当配置完MEP后或CFM功能出现故障时，可使用本命令验证或了解MEP的配置。

表1 display cfm mep命令输出信息描述表

项目

描述

MD Name

MD的名称。

Level

MD的级别，范围是0～7，级别越高，MD的优先级越高。

MA Name

MA的名称。

MEP ID

MEP节点标识。

VLAN ID

MA关联的VLAN ID。

VSI Name

MA关联的VSI名字。

Interface Name

MEP所在的接口。如果MEP是VLANIF型的，本字段显示为“--”。

CCM send
MEP的CCM消息发送功能的使能状态，取值包括：

  • enabled：MEP的CCM消息发送功能处于使能状态。

  • disabled：MEP的CCM消息发送功能处于禁止状态。

Direction
MEP的类型，取值包括：

  • inward：表示该MEP是一个inward型的MEP。inward型MEP发出的802.1ag协议报文在当前设备上的发送方式为在当前MA关联的VLAN内广播，但不会发送到该MEP所在的接口上。

  • outward：表示该MEP是一个outward型的MEP。outward型MEP发出的协议报文直接通过该MEP所在的接口向外发送。
