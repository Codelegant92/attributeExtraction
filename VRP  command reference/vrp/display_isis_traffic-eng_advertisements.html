display isis traffic-eng advertisements

命令功能

display isis traffic-eng advertisements命令用来查看流量工程TE（Traffic Engineering）最近一次发布的TE信息。

命令格式

display isis traffic-eng advertisements [ level-1 | level-2 | level-1-2 ] [ lsp-id | local ] [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

level-1：Level-1 LSDB中的TE信息。

level-2：Level-2 LSDB中的TE信息。

level-1-2：根据本节点的类别显示Level-1或Level-2 LSDB中的TE信息。即，若本节点属Level-1或Level-2，则显示Level-1或Level-2 LSDB中的TE信息；若本节点属Level-1-2，则显示Level-1和Level-2 LSDB中的TE信息。

lsp-id：查看IS-IS指定LSP ID发布的TE信息。

local：查看本地发布的TE信息。

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

该命令的显示信息有两种：标准DS-TE模式和非标准DS-TE模式。
说明：

有关DS-TE的详细介绍请参见《VRP 特性描述-MPLS》

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

LSPID

链路状态报文ID

LSP Seq Num

LSP序列号

LSP Checksum

LSP校验和

LSP Holdtime

LSP保持时长

ATT/P/OL

Attach bit（ATT）：连接位；Partition bit（P）：分区位；Overload bit（OL）：过载位。

NLPID

网络协议

AREA ADDR

区域地址

Router ID

路由器的Router ID

+NBR

开销类型为wide，wide-compatible和compatible的邻居系统ID

COST

开销

Affinity:

链路的亲和属性

Interface IP Address

接口的IP地址

Peer IP Address

接口对端的IP地址

Physical BW :

链路的物理带宽

Reservable BW:

链路的可预留带宽

Unreserved BW

未预留带宽

Sub Unreserved BW for Class Type 1

CT1的未预留带宽

BW Unresrv [ x ]

优先级为x的链路的未预留带宽

TE Cost

链路的TE度量

Bandwidth Constraint Model

链路使用的带宽约束模型

Bandwidth Constraints

带宽约束

BC[x]

BCx的带宽约束

Local Overbooking Multiplier

本地过预留倍数

LOM[x]

BCx的本地过预留倍数

+SRLG NBR ID

一个共享风险链路组中邻居的系统ID

Neighbor IP Address

邻居的IP地址

Shared Risk Link Group

共享风险链路组

说明：

在接口下使能SRLG功能时，TE信息中带有SRLG信息。
