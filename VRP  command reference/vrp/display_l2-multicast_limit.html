display l2-multicast limit

命令功能

display l2-multicast limit命令用来显示二层组播CAC限制的配置和当前统计情况。

命令格式

display l2-multicast limit [ configuration | statistics ] { interface { interface-type interface-number } ] | vlan [ vlanid [ interface { interface-type interface-number } ] ] | vsi [ vsi-name [ interface { interface-type interface-number } ] | remote-peer ip-address [ negotiation-vc-id vc-id ] | remote-site remote-site-id ] ] } [ channel channel-name ]

参数说明

configuration：配置信息。表示执行此命令后显示当前对应的各配置信息。

statistics：统计信息。表示执行此命令后显示当前对应的统计信息。

interface interface-type interface-number：对应端口的端口类型和端口编号。支持GigabitEthernet和Ethernet、Eth-Trunk类型的接口。

vlan vlan-id：已配置组播CAC限制的VLAN编号。整数形式，取值范围是1～4094。

vsi vsi-name：已配置组播CAC限制的VSI名称。字符串形式，大小写敏感，取值范围是1～31。

remote-peer ip-address：指定PW通道的远端IP地址。点分十进制形式。

negotiation-vc-id vc-id：PW信令为LDP时的对端协商VC-ID。整数形式，取值范围是1～4294967295。

remote-site remote-site-id：PW信令为BGP时的协商站点号。整数形式，取值范围是0～65534。

channel channel-name：节目组名。字符串形式，大小写敏感，取值范围是1～31。

缺省级别

1 ：监控级

使用指南

按照此命令行，如果未输入configuration或statistics，则同时显示配置和统计信息。如果输入了configuration，则只显示配置信息。如果输入了statistics，则只显示统计信息。

此命令为不可覆盖式命令。
