map bridge

命令功能

map bridge命令用来创建PVC上的IPoEoA映射或者PPPoEoA映射。

undo map bridge命令用来删除该映射。

缺省情况下，不配置任何映射。

命令格式

map bridge virtual-ethernet interface-number

undo map bridge

参数说明

interface-number：VE接口编号，编号为三维形式，包括槽号/卡号/顺序号三部分。顺序号的取值范围是0～1023。

视图

ATM-PVC视图

缺省级别

2：配置级

使用指南

在使用本命令之前，应该保证该VE已经创建成功。对于分布式设备，还应保证VE的槽号与ATM接口的槽号相同。

说明：

由于VE接口的链路层上层为以太网，下层由AAL5来承载，所以VE接口使用的MAC地址不是实际MAC地址，无法从硬件获取，必须手工配置。用户需要自己配置正确的MAC地址。

使用实例

下面这个例子展示了一个完整的IPoEoA配置过程。

# 创建VE接口Virtual-Ethernet2/0/0。

<Quidway> system-view
[Quidway] interface virtual-ethernet 2/0/0

# 为该VE接口配置IP地址10.1.1.1/16。

[Quidway-Virtual-Ethernet2/0/0] ip address 10.1.1.1 255.255.0.0
[Quidway-Virtual-Ethernet2/0/0] quit

# 在ATM接口Atm2/0/0下创建PVC 1/102。

[Quidway] interface atm 2/0/0
[Quidway-Atm2/0/0] pvc 1/102

# 在PVC视图下使用已创建好的VE接口创建IPoE映射。

[Quidway-atm-pvc-Atm2/0/0-1/102] map bridge virtual-ethernet2/0/0
