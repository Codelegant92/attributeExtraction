mib-collect enable

命令功能

mib-collect enable命令用来使能相应的接口上收集错误统计信息的功能。

undo mib-collect enable命令用来禁止相应接口上收集错误统计信息功能。

缺省情况下，不收集任何接口的信息。

命令格式

mib-collect enable

undo mib-collect enable

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

目前仅在E1/CE1/CT1/POS/CPOS接口中可应用本命令。

使用实例

# 使能POS1/0/0的MIB信息收集功能。

<Quidway> system-view
[Quidway] controller pos 1/0/0
[Quidway-Pos 1/0/0] mib-collect enable
