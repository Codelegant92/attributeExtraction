stp edged-port

命令功能

stp edged-port enable命令用来将当前端口配置为边缘端口。

stp edged-port disable命令用来将当前端口配置为非边缘端口。

undo stp edged-port命令用来将当前端口的边缘端口属性恢复为缺省值。

缺省情况下，交换机的所有端口均为非边缘端口。

命令格式

stp edged-port { enable | disable }

undo stp edged-port

参数说明

enable：配置当前的端口为边缘端口。

disable：配置当前的端口为非边缘端口。

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

边缘端口是交换机上直接与终端相连的端口。边缘端口不参与生成树的计算，能快速迁移到Forwarding状态。

如果交换机上有与终端直接相连的端口，可通过本命令配置为边缘端口，以实现这些端口的快速迁移。

使用实例

# 配置以太网接口GE 1/0/0为非边缘端口。

<Quidway> system-view
[Quidway] interface gigabitethernet0/0/1
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet0/0/1] stp edged-port disable
