c-bsr interval（IPv6）

命令功能

c-bsr interval命令用来配置BSR路由器连续发送Bootstrap自举消息的时间间隔。

undo c-bsr interval命令用来恢复时间间隔为缺省值。

缺省情况下，BSR路由器连续发送Bootstrap自举消息的时间间隔是60秒。

命令格式

c-bsr interval interval

undo c-bsr interval

参数说明

interval：表示BSR连续发送Bootstrap自举消息的时间间隔。整数形式，取值范围是1～107374177，单位是秒。

视图

IPv6 PIM视图

缺省级别

2：配置级

使用指南

路由器发送自举消息的时间间隔应该小于自举消息的超时时间。使用c-bsr holdtime命令配置自举消息的超时时间。

使用实例

# 在公网实例下，在C-BSR上配置连续发送Bootstrap自举消息的时间间隔为30秒。

<Quidway> system-view
[Quidway] pim-ipv6
[Quidway-pim6] c-bsr interval 30
