ipv6 nd nud reachable-time

命令功能

ipv6 nd nud reachable-time命令用来配置邻居可达时间。

undo ipv6 nd nud reachable-time命令用来将此设置恢复为缺省值。

命令格式

ipv6 nd nud reachable-time value

undo ipv6 nd nud reachable-time

参数说明

value ：指定定时器的取值。整数形式，单位是毫秒，取值范围是0～3600000。缺省值是30000毫秒。

使用指南

可达时间的作用包括：

  • 控制本路由设备邻居表项的老化时间；

  • 作为RA报文的参数，帮助主机配置可达时间。

说明：

一般情况下，邻居可达时间与RA报文的可达时间取值相同。但在下述情况中，邻居可达时间为缺省值30000毫秒，RA报文中的可达时间为0毫秒。

  • 缺省情况下，邻居可达时间为缺省值30000毫秒。

  • 执行undo ipv6 nd nud reachable-time命令将邻居可达时间恢复为缺省值。

  • 执行ipv6 nd nud reachable-time 0命令，“0”表示不对邻居可达时间进行设置。

当主机收到可达时间为0的RA报文后，使用缺省值30000毫秒，与路由设备保持一致。

路由设备在接口上每次发布RA报文时都携带该时间，以便同一链路上的所有节点都使用相同的时间。

此命令支持Ethernet接口及其子接口、GigabitEthernet接口及其子接口、POS接口以及Tunnel接口。

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 配置GE接口上的可达时间。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 nd nud reachable-time 10000
