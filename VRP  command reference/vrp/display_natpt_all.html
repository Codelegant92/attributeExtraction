display natpt all

命令功能

display natpt all命令用来查看所有NAT-PT配置信息。

命令格式

display natpt all [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

表1 display natpt all命令输出信息描述

项目

描述

NATPT IPv4 Address Pool Information:

所有IPv4地址池信息

NATPT address mapping(v6bound view):

IPv6侧的NAT-PT地址映射信息。IPv4侧的NAT-PT地址映射信息与IPv6侧的基本相同，只需将Type字段中的DESTINATION和SOURCE互换。

NATPT V6Server static mapping:

从IPv4侧到目的地址的静态PAT映射

NATPT V4bound information:

使用natpt v4bound配置的IPv4绑定关系

NATPT V6bound information:

使用natpt v6bound配置的IPv6绑定关系

NATPT Prefix Info:

NAT-PT发布的前缀信息

NATPT aging-time value information:

NAT-PT Session的有效时间

NATPT Statistics

NAT-PT的统计信息，包括：

  • Total Sessions：当前Session的数目
  • Expired Sessions：超时Session的数目
  • Hits：地址转换成功的次数
  • Misses：地址转换失败的次数
  • Total Fragment Sessions：已存在的分片报文的Session数目
  • Expired Fragment Sessions：超时的分片报文的Session数目
  • Fragment Hits：分片报文转换成功的次数
  • Fragment Misses：分片报文转换失败的次数
  • Total Address Mappings：地址映射数目
  • Total V6Server Mappings：IPv4侧的静态PAT映射数目

NATPT Interfaces

使能NAT-PT的接口
