t1 set frame-format（CT3接口视图）

命令功能

t1 set frame-format命令用来配置T1通道的帧格式。

undo t1 set frame-format命令用来恢复缺省设置。

缺省情况下，T1通道使用的帧格式为ESF。

命令格式

t1 t1-number set frame-format { esf | sf }

undo t1 t1-number set frame-format

参数说明

t1-number：T1通道号。整数形式，取值范围是1～28。

esf：设置T1通道使用扩展超帧（Extended Super Frame）格式。

sf：设置T1通道使用超帧（Super Frame）格式。

视图

CT3接口视图

缺省级别

2：配置级

使用指南

只有当CT3接口工作在CT3工作模式下时，才能配置该命令。

配置时需与对端保持一致。

使用实例

# 设置CT3接口1/0/0的1号T1通道的帧格式为SF。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] t1 1 set frame-format sf
