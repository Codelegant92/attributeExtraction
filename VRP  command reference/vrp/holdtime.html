holdtime

命令功能

holdtime命令用于设置握手超时后保持状态的时间。

undo holdtime命令用来将有效保留时间恢复为缺省值。

缺省情况下，握手超时后保持状态的时间是60秒。

命令格式

holdtime hold-time

undo holdtime

参数说明

hold-time：握手报文超时后，状态的有效保留时间。整数形式，取值范围是3～240，单位是秒。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。握手超时后保持状态的时间值必须大于或等于握手报文发送时间间隔的三倍。

说明：

配置握手报文的发送时间间隔，请参见timer命令

使用实例

# 配置握手超时后保持状态的时间为30秒。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] holdtime 30
