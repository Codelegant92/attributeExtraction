display rsa peer-public-key

命令功能

display rsa peer-public-key命令用来显示指定的RSA公共密钥，如果没有指定，则显示所有的公共密钥。

命令格式

display rsa peer-public-key [ brief | name key-name ]

参数说明

brief：显示所有远端公钥的简明信息。

key-name：指定要显示的密钥名称，字符串形式，长度为1～30。

视图

所有视图

缺省级别

1：监控级

使用指南

运行此命令可显示所有公共密钥或者某个指定公共密钥的详细信息。

使用实例

# 显示所有RSA公共密钥的简要信息。

<Quidway> display rsa peer-public-key brief
Address        Bits   Name
                512   rsakey001

# 显示名为rsakey001的RSA公共密钥的详细信息。

<Quidway> display rsa peer-public-key name rsakey001
