display isis spf-log

命令功能

display isis spf-log命令用来显示IS-IS的SPF计算日志记录。

命令格式

display isis spf-log [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ ipv6 | [ level-1 | level-2 ] | verbose ] * [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称，字符串形式，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，字符串形式，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

ipv6：IPv6拓扑的SPF日志。

level-1：Level-1的SPF日志。

level-2：Level-2的SPF日志。

verbose：SPF日志的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用此命令可以查看SPF计算的开始和持续时间，节点数，触发事件等信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
