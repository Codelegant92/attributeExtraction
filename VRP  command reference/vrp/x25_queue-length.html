x25 queue-length

命令功能

x25 queue-length命令用来设置X.25虚电路的数据队列的长度。

undo x25 queue-length命令用来恢复X.25虚电路的数据队列的长度的缺省值。

命令格式

x25 queue-length queue-length

undo x25 queue-length

参数说明

queue-length：队列长度（以数据报的个数为单位），取值范围是0～9999。X.25虚电路的数据队列的长度缺省为200。

视图

接口视图

缺省级别

2：配置级

使用指南

当数据流量过大时，可以使用该命令加长X.25虚电路的发送队列和接收队列，以免造成数据丢失而影响传输效果。需要注意的一点是，更改该参数不会影响已经存在的虚电路的数据队列。

执行完该命令后，必须重新启动接口，使配置生效。

使用实例

# 将X.25接口Serial1/00的虚电路数据队列的长度设置为75个数据报。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 queue-length 75
[Quidway-Serial1/0/0] shutdown
[Quidway-Serial1/0/0] undo shutdown
