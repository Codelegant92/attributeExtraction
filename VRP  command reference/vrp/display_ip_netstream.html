display ip netstream

命令功能

display ip netstream命令用来显示NetStream的配置信息。可以显示单个接口下的NetStream配置信息，可以显示系统视图下和聚合视图下的NetStream配置信息。也可以显示系统中所有的NetStream配置信息。

命令格式

display ip netstream { all | global | interface interface-type interface-number }

参数说明

all：显示NetStream流统计的所有配置信息。包括系统视图下的流统计配置信息、聚合视图下的流统计配置信息和接口视图下的流统计配置信息。

global：显示系统视图下流统计的配置信息和聚合视图下的流统计配置信息。

interface-type：接口类型。

interface-number：接口编号。

使用指南

只有接口配置了NetStream采样相关的配置才会显示出来，否则不会显示该接口。

显示的内容有IPv4、IPv4组播、MPLS、IPv6、IPv6组播、RPF/URPF的入出采样使能，采样模式和采样报文间隔，如果是二层接口，显示二层接口所属的VLAN ID。

使用实例

# 显示接口Ethernet 1/0/0的NetStream配置信息。

<Quidway> display ip netstream interface Ethernet 1/0/0
 ip netstream inbound
 ip netstream outbound
 ip netstream sampler inbound 10
 ip netstream sampler outbound 10
 ip netstream urpf enable
表1 display ip netstream命令输出信息描述

项目

描述

ip netstream inbound

使能了NetStream入统计功能

ip netstream outbound

使能了NetStream出统计功能

ip netstream sampler inbound

接口下配置NetStream入统计的采样报文的间隔

ip netstream sampler outbound

接口下配置NetStream出统计的采样报文的间隔

ip netstream urpf enable

使能NetStream统计URPF报文的功能

# 显示全局和聚合的netstream配置信息。

<Quidway> display ip netstream global
 ip netstream export version 9 peer-as
  ip netstream export manager version 0 
  ip netstream sampler inbound 12
  ip netstream aggregation as
  enable
  ip netstream export source 2.2.22.2
  ip netstream host 1.1.1.1 5555
表2 display ip netstream命令输出信息描述

项目

描述

ip netstream export version 9 peer-as

输出v9报文的自治系统号为对端自治系统号

ip netstream export manager version 0

输出给网管的报文格式是原有的报文格式

ip netstream sampler inbound

全局下配置的入统计的采样间隔

ip netstream sampler outbound

NetStream出统计的采样报文的间隔

ip netstream aggregation as

NetStream按照AS号进行聚合

ip netstream export source

输出NetStream报文的源接口

ip netstream host

NetStream报文输出的目的主机地址
