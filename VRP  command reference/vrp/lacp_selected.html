lacp selected

命令功能

lacp selected命令用来配置静态LACP模式Eth-Trunk选择活动接口的方式。

缺省情况下，依据接口优先级选择活动接口。

命令格式

lacp selected { priority | speed }

参数说明

priority：依据接口优先级选择活动接口。接口优先级高的接口将被优先选为活动接口。

speed：依据接口速率选择活动接口。接口速率高的接口将被优先选为活动接口。

缺省级别

2：配置级

使用指南

执行该命令前需首先保证Eth-Trunk工作模式为静态LACP模式。

在一个静态Eth-Trunk中有速率不同的成员口，例如，同时将100M bit/s、1G bit/s的以太口加入同一个静态Eth-Trunk。按照LACP协议，依据接口优先级来选择活动接口，可能会选中速率低的成员口。如果希望选中接口速率高的成员口，可以使用lacp selected speed命令配置静态Eth-Trunk依据接口速率选择活动接口。

使用实例

# 指定Eth-trunk 1工作在静态模式，并且根据端口速率来选择端口。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] mode lacp-static
[Quidway-Eth-Trunk1] lacp selected speed
