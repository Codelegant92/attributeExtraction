tos

命令功能

tos命令用来配置测试包的服务类型TOS。

undo tos命令用来将服务类型恢复到缺省值。

缺省情况下，TOS的值为0。

命令格式

tos value

undo tos

参数说明

value：报文的服务类型。取值范围是0～255。

视图

NQA视图

缺省级别

2：配置级

使用指南

不能对DNS、DHCP、LSP Ping、LSP Jitter、LSP Trace、MTrace、PWE3 Ping、PWE3 Trace、Trace、Path MTU类型的测试例配置服务类型。

不能修改正在运行的测试例的此项配置。

使用实例

# 设置测试例user test的服务类型。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] tos 10
