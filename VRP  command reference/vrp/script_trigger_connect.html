script trigger connect

命令功能

script trigger connect命令用来配置当成功建立呼入连接时执行的Modem脚本。

undo script trigger connect命令用来取消这个特征。

系统缺省不指定该脚本。

命令格式

script trigger connect script-name

undo script trigger connect

参数说明

script-name：脚本集中的一个脚本，字符串形式，长度为1～50。

使用指南
