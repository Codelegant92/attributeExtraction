display memory-usage threshold

命令功能

display memory-usage threshold命令用来显示内存占用率阈值。

命令格式

display memory-usage threshold [ slot slot-id ]

参数说明

slot slot-id：显示指定接口板的内存占用率阈值。接口板槽号的取值范围因设备不同而不同。

所有视图

使用指南

执行该命令后，可以查看主控板和指定接口板的内存占用率阈值。

使用实例

# 查看当前主控板的内存占用率阈值。

<Quidway> display memory-usage threshold
 Current memory-usage threshold is 100%
