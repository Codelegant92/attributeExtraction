cluster switch-to administrator

命令功能

cluster switch-to administrator命令用来从成员交换机telnet到命令交换机。

命令格式

cluster switch-to administrator

参数说明

无

视图

用户视图

缺省级别

0：参观级

使用指南

本命令只能在成员交换机上执行。

在命令交换机的用户视图下输入quit命令，则结束telnet，返回成员交换机。

使用实例

# 从编号为1的成员交换机telnet到命令交换机，再结束telnet，返回成员交换机。
