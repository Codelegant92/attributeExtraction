ipv6 address

命令功能

ipv6 address命令用来手工配置接口的全球单播地址。

undo ipv6 address命令用来删除手工配置的接口地址，如果未指定IPv6地址和前缀长度这个参数，则缺省删除该接口上配置的所有地址，但自动配置的链路本地地址除外。

命令格式

ipv6 address { ipv6-address prefix-length | ipv6-address/prefix-length }

undo ipv6 address [ ipv6-address prefix-length | ipv6-address/prefix-length ]

参数说明

ipv6-address：指定IPv6地址。

prefix-length：指定前缀长度。整数形式，取值范围是1～128。

使用指南

IPv6的128位IP地址有以下两种表示形式。

  • X:X:X:X:X:X:X:X

    在这种形式中，128位的IP地址被分为8组，每组的16位用4个十六进制字符（0～9，A～F）来表示，组和组之间用冒号（:）隔开。其中每个“X”代表一组十六进制数值。

  • X:X:X:X:X:X:d.d.d.d

    分为IPv4兼容IPv6地址和IPv4映射IPv6地址两种类型 。其中IPv4兼容IPv6地址用于配置IPv6 over IPv4隧道。

    在这种形式中，“X”代表高阶的六组数字，用十六进制数来表示每组的16比特。“d”代表低阶的四组数字，用十进制数表示每组的8比特。后边的部分（d.d.d.d）其实就是一个标准的IPv4地址。

一个IPv6地址可以分为如下两部分：
  • 网络前缀：n比特，相当于IPv4地址中的网络ID。
  • 接口标识：128-n比特，相当于IPv4地址中的主机ID。

ipv6 address命令支持Ethernet接口及其子接口、GigabitEthernet接口及其子接口、POS接口、Serial接口、Loopback接口、Eth-Trunk接口及其子接口、IP-Trunk接口以及Tunnel接口。

接口上最多可配置10个全球单播地址。

不能配置在接口上的IPv6地址有：

  • 环回地址（::1/128）

  • 未指定地址（::/128）

  • 组播地址

  • 任播地址

使用实例

# 配置GE接口的全球单播地址。

<Quidway> system-view
[Quidway] interface gigaitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 enable
[Quidway-GigabitEthernet1/0/0] ipv6 address 2001::1/64
