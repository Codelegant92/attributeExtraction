firewall blacklist enable

命令功能

firewall blacklist enable命令可以用来使能黑名单功能。

undo firewall blacklist enable命令用来停止黑名单功能。

系统缺省情况下为停止黑名单功能。

命令格式

firewall blacklist enable [ acl-number acl-number ]

undo firewall blacklist enable

参数说明

acl-number：高级访问控制列表号（3000～3999）。对于ACL规则允许的访问，则黑名单不阻挡这些流量。

视图

系统视图

缺省级别

2：配置级

使用指南

使能黑名单还可以同时通过ACL指定一些规则，对于ACL规则允许的访问，则黑名单不阻挡这些流量。注意：此时ACL仅支持5元组和时间段，5元组即协议、源IP地址及掩码、目的IP地址及掩码、源端口范围、目的端口范围。

使用实例

# 使能黑名单功能。

<Quidway> system-view
[Quidway] firewall blacklist enable

# 使能黑名单功能，同时指定ACL规则组3000。

<Quidway> system-view
[Quidway] firewall blacklist enable acl-number 3000
