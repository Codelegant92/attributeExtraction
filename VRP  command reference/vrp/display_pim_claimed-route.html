display pim claimed-route

命令功能

display pim claimed-route命令用来查看PIM使用的单播路由信息。

命令格式

display pim [ vpn-instance vpn-instance-name | all-instance ] claimed-route [ source-address ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

source-address：表示组播源地址。点分十进制形式。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display pim claimed-route命令时：

  • 如果不指定vpn-instance或all-instance，则只显示公网实例的相关信息。

  • 如果指定source-address地址，则仅显示到达source-address的单播路由信息。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
