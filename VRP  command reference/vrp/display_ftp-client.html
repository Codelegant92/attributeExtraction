display ftp-client

命令功能

display ftp-client命令用来显示设备作为FTP客户端时的源参数。

命令格式

display ftp-client

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

进行FTP客户端源参数设置后，可通过此命令查看设置结果。若未配置ftp client-source，则默认源为：0.0.0.0。

使用实例

# 显示FTP客户端源参数设置结果。

<Quidway> display ftp-client
The source address of FTP client is 1.1.1.1.
表1 display ftp-client命令输出信息描述

项目

描述

The source address of the FTP client is 1.1.1.1

FTP客户端的源地址设置为1.1.1.1。
