port-standby（Ethernet接口视图）

命令功能

port-standby命令用来为捆绑组的成员端口配置备份端口。

undo port-standby命令用来取消配置。

缺省情况下，采用组内备份。

命令格式

port-standby interface-type interface-number

undo port-standby

参数说明

interface-type：备份端口的接口类型，必须与主端口的类型相同。

interface-number：备份端口的接口编号。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

如果为成员端口配置了备份端口，当成员端口处于Down状态时，流量通过备份端口转发。

如果备份端口状态也处于Down状态，仍采用组内备份方式，流量通过组内状态为UP的端口转发。

使用实例

# 为Eth-Trunk1的成员端口GE1/0/0配置备份端口。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] port-standby gigabitethernet 2/0/0
