flag（POS接口视图）

命令功能

flag命令用来配置SDH开销字节。

undo flag命令用来恢复缺省值。

命令格式

flag { c2 | j0 | j1 } hex-value

undo flag { c2 | j0 | j1 }

参数说明

c2：信号标记字节（Path signal label byte），属于高阶通道开销字节，用来指示VC帧的复接结构和信息净负荷的性质。缺省值是16（十六进制）。

j0：再生段踪迹字节（Regeneration Section Trace Message），属于段开销字节，用于检测两个端口之间的连接在段层次上的连续性。缺省值是0（十六进制）。

j1：通道踪迹字节（Higher-Order VC-N path trace byte），属于高阶通道开销字节，用于检测两个端口处于持续连接状态。缺省值是0（十六进制）。

hex-value：16进制的数字，取值范围是0～ff。

缺省级别

2：配置级

使用指南

C2字节和J1字节的设置一定要使收/发两端相匹配。J0在同一个运营者的网络内该字节可为任意字符，而在不同两个运营者的网络边界处要使设备收、发两端的J0字节相匹配。通过J0字节可使运营者提前发现和解决故障，缩短网络恢复时间。

使用实例

# 配置开销字节J0。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] flag j0 ff
