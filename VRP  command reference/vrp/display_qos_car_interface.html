display qos car interface

命令功能

display qos car interface命令用来显示CAR在指定接口上的参数设置情况和运行统计信息。

命令格式

display qos car interface [ interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口编号。

所有视图

缺省级别

1：监控级

使用指南

display qos car interface命令可以观察到的内容包括：

  • 接口上的承诺信息速率

  • 接口上的承诺突发尺寸

  • 接口上的过度突发尺寸

  • 通过CIR的报文的动作

  • 超过CIR的报文的动作

  • 通过的报文统计信息

  • 丢弃的报文统计信息

如不指定接口，将显示所有接口的CAR参数设置情况和运行统计信息。

使用实例

表1 display qos car interface命令输出信息描述

主要字段

描述

Interface

配置流量参数的接口

Direction

接口上应用流量参数的方向

Rule(s)

匹配的规则

CIR 8000(Bps)

承诺信息速率（Committed Information Rate）

CBS 15000(Bit)

承诺突发尺寸（Committed Burst Size）

EBS 0(Bit)

过度突发尺寸（Excess Burst Size）

Conform Action: remark ip-precedence 3 and pass

通过CIR的报文的动作，重新标记优先级并发送

Exceed Action: remark ip-precedence 4 and continue

超过CIR的报文的动作，重新标记优先级并由下一个CAR处理

Conformed: (Packets/Bytes)

通过的报文统计信息(包数/字节数)

Exceeded: (Packets/Bytes)

丢弃的报文统计信息(包数/字节数)
