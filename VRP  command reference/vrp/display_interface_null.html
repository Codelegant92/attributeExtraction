display interface null

命令功能

display interface null命令用来查看Null接口的状态信息。

命令格式

display interface null [ 0 ] [ | { begin | exclude | include } regular-expression ]

参数说明

0：Null接口的编号。Null接口的编号只有0。

|：使用正则表达式过滤输出的显示信息。

begin：显示以包含指定正则表达式的行开始的配置信息。

exclude：显示不包含指定正则表达式的行的配置信息。

include：显示包含指定正则表达式的行的配置信息。

regular-expression：正则表达式。字符串形式，支持空格，区分大小写。

项目

描述

NULL0 current state :

NULL接口当前的物理状态。NULL接口的物理状态一直都是Up。

Line protocol current state :

显示该接口的链路协议状态。NULL接口的协议状态一直都是Up。

Description:

对接口的说明，支持空格，区分大小写，最多可输入242个字符。该说明信息便于用户了解该接口的作用。

The Maximum Transmit Unit is

接口的最大传输单元（MTU），是1500字节。

Internet protocol processing : disabled

接口没有配置IP地址。

Physical is NULL DEV

物理接口为空设备。

Statistics last cleared:

上次清除本接口统计信息的时间。

Last 300 seconds input rate Last 300 seconds output rate

这一字段显示了在前5分钟内通过接口发送和接收的字节速率和报文速率。
