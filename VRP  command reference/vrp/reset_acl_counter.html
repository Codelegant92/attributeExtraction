reset acl counter

命令功能

命令reset acl counter用来清除访问控制列表的统计信息。

命令格式

reset acl counter { acl-number | all }

参数说明

acl-number：指定访问控制列表的编号。整数形式，取值范围是1000～3999。其中1000～1999是基于接口的访问控制列表，2000～2999是基本访问控制列表，3000～3999是高级访问控制列表。

all：所有的ACL。

使用指南

该命令用来清除访问控制列表的统计信息。请使用前确认需要清除统计信息。

使用实例

# 清除访问控制列表2001的统计信息。

<Quidway> reset acl counter 2001
