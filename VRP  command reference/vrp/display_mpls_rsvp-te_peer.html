display mpls rsvp-te peer

命令功能

display mpls rsvp-te peer命令用来查看使能RSVP-TE的接口的RSVP-TE邻居信息。

命令格式

display mpls rsvp-te peer [ interface interface-type interface-number ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：显示指定接口的RSVP-TE邻居信息。interface-type表示接口类型，interface-number表示接口编号。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

邻居地址

SrcInstance

源实例

对端源实例

PSB Count

路径状态块（Path State Block）数量

RSB Count

预留状态块（Reservation State Block）数量

Hello Type Sent
向对端发送的Hello类型：
  • REQ
  • ACK
  • NONE

Neighbor Hello Extension

邻居的Hello特性扩展的使能状态

SRefresh Enable
是否使能摘要刷新：
  • YES
  • NO
