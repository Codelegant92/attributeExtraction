port-master

命令功能

port-master用来指定当前接口为1:1主备模式下的主接口。

undo port-master用来取消当前接口为1:1主备模式下的主接口。

缺省情况下，Eth-Trunk成员接口的类型为备份接口。

命令格式

port-master

undo port-master

参数说明

无

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

使用本命令的前提条件是要求接口所属的Eth-Trunk模式为1:1主备模式。如果需要更改当前Eth-Trunk工作模式，可以通过执行mode命令修改。

要更改当前的备份接口为主接口，必须首先在当前主接口上执行undo port-master命令取消当前接口为主接口，然后再配置当前的备份接口为主接口即可。但这种方法切换主备接口有可能造成业务的短暂中断，要避免业务的短暂中断可以在Eth-Trunk接口视图下执行protect-switch命令强制手工切换非活动接口为活动接口，这时备份接口成为活动接口并负责转发数据。

使用实例

# 将GigabitEthernet 1/0/0接口加入1:1主备模式Eth-Trunk 1，并把它设置成主接口。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] eth-trunk 1
[Quidway-GigabitEthernet1/0/0] port-master
