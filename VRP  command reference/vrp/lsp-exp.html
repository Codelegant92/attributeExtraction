lsp-exp

命令功能

lsp-exp命令用来配置NQA测试的LSP EXP值。

undo lsp-exp命令用来恢复缺省配置。

缺省情况下，LSP EXP值是0。

命令格式

lsp-exp exp

undo lsp-exp

参数说明

exp：EXP值。取值范围是0～7。

视图

NQA视图

缺省级别

2：配置级

使用指南

只能对LSP类型的测试例配置LSP EXP。

不能更改正在运行的测试例的LSP EXP值。

使用实例

# 设置测试例user test的LSP EXP值为5。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] test-type lspping
[Quidway-nqa-user-test] lsp-exp 5
