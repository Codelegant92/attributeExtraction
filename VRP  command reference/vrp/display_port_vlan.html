display port vlan

命令功能

display port vlan命令用来显示VLAN中包含的端口信息。

命令格式

display port vlan [ interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type：接口类型，以太类接口，例如Ethernet、GigabitEthernet、Eth-Trunk、Virtual-Ethernet。

interface-number：接口序号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

当指定接口类型和编号时，若该接口为交换式端口，则显示该端口的Link Type、PVID、Trunk VLAN List信息。

若不指定接口类型和编号，则显示所有交换式端口的Link Type、PVID、Trunk VLAN List信息。

使用实例

# 查询所有交换式端口的Link Type、PVID、Trunk VLAN List信息。

<Quidway> display port vlan 
Port                     Link Type            PVID         Trunk VLAN List  
--------------------------------------------------------------------------- 
GigabitEthernet1/0/0     trunk                0            5
Eth-Trunk2               hybrid               2            10-12, 2049
Eth-Trunk3               trunk                0            10-12, 2049
Eth-Trunk4               dot1q-tunnel         10

项目

描述

Port

端口类型和编号。

Link Type
端口链路类型，有以下类型。

  • access

  • trunk

  • hybrid

  • dot1q-tunnel

PVID

端口所属的缺省VLAN ID。

Trunk VLAN List

端口允许通过的VLAN ID。
