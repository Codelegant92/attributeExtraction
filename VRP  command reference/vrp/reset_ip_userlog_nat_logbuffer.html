reset ip userlog nat logbuffer

命令功能

reset ip userlog logbuffer命令用来清除当前用户日志缓存中的记录，该操作将导致这些记录被直接清除，不再被输出。

命令格式

reset ip userlog nat logbuffer slot slot-number

参数说明

nat：NAT日志记录。

slot-number：接口板槽号。

用户视图

缺省级别

2：配置级

使用指南

建议用户不要轻易使用该命令，除非确实对过去生成的用户日志记录没有需求了。

使用实例

# 在路由器的2号接口板上，清除当前用户日志缓存中的NAT日志记录。

<Quidway> reset ip userlog nat logbuffer slot 2
