shutdown（接口视图）

命令功能

shutdown命令用来关闭接口。

undo shutdown命令用来启动该接口。

缺省情况下，在路由器上电时，初始化并启动各物理接口。

命令格式

shutdown

undo shutdown

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

请慎重使用关闭和重启接口命令。但在某些特殊情况下，例如配置了接口的工作参数后，不能立即生效，需要关闭和重启接口后，才能使配置生效。

由于产品或版本不同，缺省情况下接口是否处于启动状态也不同。如果缺省情况下接口是关闭状态，需要执行undo shutdown配置才能生效。

在某些逻辑接口（如Logic-Channel/Virtual-Template/Loopback/NULL/Dialer等接口）的接口视图下不支持关闭和重启命令。

说明：

  • 当路由器的某物理接口闲置，没有连接电缆时，请使用shutdown命令禁止该接口，以防止由于干扰导致接口异常。

  • 当接口下配有子接口时，在主接口连续执行shutdown和undo shutdown操作之间的时间间隔应至少为15秒。

使用实例

# 关闭POS接口1/0/0。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] shutdown
