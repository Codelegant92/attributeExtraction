mld timer query

命令功能

mld timer query命令用来设置接口上发送MLD普遍组查询消息的时间间隔。

undo mld timer query命令用来恢复配置的缺省情况。

缺省情况下，发送MLD查询消息的时间间隔为125s。

命令格式

mld timer query inerval

undo mld timer query

参数说明

interval：指定接口上发送MLD普遍组查询消息的时间间隔。整数形式，取值范围是1～18000，单位是s。

视图

接口视图

缺省级别

2：配置级

使用指南

组播路由器在指定的间隔时间内发送MLD普遍组查询消息，以发现网络上是否存在组播组成员。可以根据网络的实际情况来修改此查询间隔。

如果接口视图和MLD视图下都配置了此参数，优先取接口视图下的值。

使用实例

# 设置接口GE1/0/0发送普遍组查询报文的时间间隔是200s。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld timer query 200
