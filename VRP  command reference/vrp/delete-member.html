delete-member

命令功能

delete-member命令用来删除成员交换机。

命令格式

delete-member member-number

参数说明

member-number：成员交换机的成员编号。十进制整数形式，取值范围是1～255。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

使用实例

# 删除成员编号为1的成员交换机。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] delete-member 1
