interface mp-group

命令功能

interface mp-group命令用来创建一个MP-Group类型的接口。

undo interface mp-group命令用来删除指定的MP-Group接口。

命令格式

interface mp-group number

undo interface mp-group number

参数说明

number：MP-Group接口的编号，编号为三维形式，包括槽号/卡号/顺序号三部分。顺序号的取值范围是0～1023。即一块接口板最多可支持1024个MP-Group接口。MP-Group接口的槽号需要与所绑定的接口槽号保持一致。

使用指南

该命令与ppp mp mp-group命令配合使用，可以先创建MP-Group接口，也可以先配置接口加入MP-Group组。

使用实例

# 创建MP-Group接口1/0/1。

<Quidway> system-view
[Quidway] interface mp-group 1/0/1
