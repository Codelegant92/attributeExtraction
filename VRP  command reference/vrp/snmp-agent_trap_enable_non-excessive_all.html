snmp-agent trap enable non-excessive all

命令功能

snmp-agent trap enable non-excessive all命令用来使能向网管发送所有的非海量告警消息。

undo snmp-agent trap enable non-excessive all命令用来取消当前设置。

缺省情况下，任何告警消息都不会向网管发送。

命令格式

snmp-agent trap enable non-excessive all

undo snmp-agent trap enable non-excessive all

参数说明

无
