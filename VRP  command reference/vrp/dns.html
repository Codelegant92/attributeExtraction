dns

命令功能

dns命令用来为当前域指定DNS服务器。

undo dns命令用来删除当前域的DNS服务器。

缺省情况下，域没有DNS服务器。

命令格式

dns { primary-ip |second-ip } ip-address

undo dns { primary-ip | second-ip }

参数说明

primary-ip：设置主DNS服务器。

second-ip：设置从DNS服务器。

ip-address：DNS服务器的IP地址。点分十进制形式。必须使用合法的单播地址。

缺省级别

2：配置级

使用指南

修改域的DNS服务器的配置，在用户下一次登录时生效。

使用实例

# 设置当前域使用IP地址为10.1.1.1的服务器做为主DNS服务器。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei] dns primary-ip 10.1.1.1
