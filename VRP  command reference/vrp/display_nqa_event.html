display nqa event

命令功能

display nqa event命令用来显示和告警相关联的事件信息。

命令格式

display nqa event [ event-entry ]

参数说明

event-entry：和告警相关联的事件表号。整数形式，取值范围是1～65535。

使用指南

执行display nqa event命令，将输出和告警相关联的事件信息。

使用实例

# 显示事件号是10的事件信息。

<Quidway> display nqa event 10
