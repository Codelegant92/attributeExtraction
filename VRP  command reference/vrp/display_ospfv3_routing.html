display ospfv3 routing

命令功能

display ospfv3 routing命令用来显示OSPFv3路由表的信息。

命令格式

display ospfv3 [ process-id ] routing [ [ ipv6-address prefix-length ] | abr-routes | asbr-routes | all | statistics ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

ipv6-address：IPv6地址。

prefix-length：指定前缀长度。整数形式，取值范围是0～128。

abr-routes：显示OSPFv3中所有ABR的路由表信息。

asbr-routes：显示OSPFv3中所有ASBR的路由表信息。

all：显示OSPFv3中所有的路由表信息。

statistics：显示OSPFv3中所有路由表的统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
