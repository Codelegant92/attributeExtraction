display acl ipv6

命令功能

display acl ipv6命令用来查看指定ACL6或所有ACL6的配置信息和统计信息。

命令格式

display acl ipv6 { acl6-number | all } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

acl6-number：指定ACL6编号，整数形式，取值范围是1000～3999。其中1000～1999是基于接口的ACL6，2000～2999是基本ACL6，3000～3999是高级ACL6。

all：显示所有ACL6的配置。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
