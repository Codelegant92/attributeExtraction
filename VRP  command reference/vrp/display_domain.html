display domain

命令功能

display domain命令用来查看域的配置信息。

命令格式

display domain [ domain-name ]

参数说明

domain-name：域名，字符串形式，长度范围是1～20，不区分大小写。

所有视图

缺省级别

使用指南

display domain命令查看域的配置信息包括：

  • 域的名字

  • 状态

  • 认证方案表

  • 计费方案表

  • 授权方案表

  • Web的IP地址

  • DNS服务器信息

  • NBNS服务器信息

  • 闲置切断数据

  • 缺省用户优先级

  • 接入用户数目限制

  • 在线用户数目和索引号

  • 域下的radius模板信息

  • 域下的hwtacacs模板信息

如果不指定域名，则显示所有当前存在的域的配置信息。

使用实例

# 查看所有当前存在的域的配置信息。

<Quidway> display domain
    -----------------------------------------------------------------------
  -----------------------------------------------------------------------
  default                             Active     0         256            0
  huawei                              Active     5         256            0
  -----------------------------------------------------------------------
  Total 2,2 printed
