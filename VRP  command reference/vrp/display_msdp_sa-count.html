display msdp sa-count

命令功能

display msdp sa-count命令用来查看SA缓存中的（S，G）项的数量。

命令格式

display msdp [ vpn-instance vpn-instance-name | all-instance ] sa-count [ as-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

as-number：表示自治系统号。整数形式，取值范围是1～65535。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

只有使能了SA缓存的MSDP对等体才能显示SA缓存中的（S，G）项的数量（缺省情况下，SA缓存是使能的）。

使用display msdp sa-count命令时，若不指定vpn-instance或all-instance，则只显示公网实例的相关信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
