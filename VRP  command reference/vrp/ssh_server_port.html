ssh server port

命令功能

ssh server port port-number命令用来更改SSH服务器的监听端口号。

undo ssh server port命令用来取消SSH服务器端设置的当前监听端口号，恢复为缺省值。

缺省情况下，SSH服务器监听端口号为22。

命令格式

ssh server port port-number

undo ssh server port

参数说明

port-number：SSH服务器监听端口号，整数形式，取值范围为22或1025～55535。

缺省值是协议标准端口号22。

使用指南

设置了SSH服务器的监听端口号以后，只有当服务器正在监听的端口号是22时，SSH客户端登录时可以不指定端口号；否则如果是其他监听端口号，SSH客户端登录时必须指定端口号。

使用实例

# 配置SSH服务器的监听端口号。

<Quidway> system-view
[Quidway] ssh server port 1025
Info: Succeeded in changing SSH listening port.
