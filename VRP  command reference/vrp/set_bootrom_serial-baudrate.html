set bootrom serial-baudrate

命令功能

set bootrom serial-baudrate命令用来设置串行接口的波特率。

命令格式

set bootrom serial-baudrate baudrate-value [ slave ]

参数说明

baudrate-value：指定波特率的取值，只能选取9600，19200，38400，57600，115200，单位为bit/s。

slave：设置备用主控板的串行接口波特率。

视图

用户视图

缺省级别

2：配置级

使用指南

可以通过设置BootROM中串行接口的波特率的方法，到指定的服务器上重新下载资源文件。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 设置串行接口的波特率为9600。

<Quidway> set bootrom serial-baudrate 9600
Info: set Bootrom Serial successful
