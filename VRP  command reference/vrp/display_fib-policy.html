display fib-policy

命令功能

display fib-policy命令用来查看IPv4 FIB策略。

命令格式

display fib-policy slot slot-number

参数说明

slot slot-number：需要查看IPv4 FIB策略的接口板编号。

所有视图

缺省级别

1：监控级

使用指南

执行命令display fib-policy，可以查看接口板配置的策略，如果没有配置策略也会提示用户策略没有配置。

使用实例

# 查看2号接口板IPv4 FIB策略的应用信息。

<Quidway> display fib-policy slot 2
  Fib-policy:
    public route
    vpn route: vpna
表1 display fib-policy输出信息描述

项目

描述

Fib-policy:
本接口板上应用的IPv4 FIB策略。可能显示的配置信息有：

  • Have not configured FIB policy：接口板上没有配置任何IPv4 FIB策略

  • public route：接口板上配置了IPv4公网策略

  • VPN route：接口板上配置了IPv4 VPN转发表项策略

  • Have configured no-route FIB policy：接口板不接收主控板下发的任何IPv4转发表项

public route

IPv4公网转发表项。

vpn route:

IPv4 VPN转发表项。
