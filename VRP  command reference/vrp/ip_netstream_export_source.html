ip netstream export source

命令功能

ip netstream export source命令用来配置NetStream统计输出报文的源地址。

undo ip netstream export source命令用来删除配置的NetStream统计输出报文的源IP地址。

命令格式

ip netstream export source ip-address

undo ip netstream export source

参数说明

ip-address：指定发送的承载统计信息的UDP报文的源地址。

视图

系统视图，NetStream聚合视图

缺省级别

2：配置级

使用指南

针对聚合模式，优先采用聚合模式下配置的源地址，当某种聚合没有配置对应的输出源地址信息时，将采用系统视图下配置的源地址。

在系统视图和聚合视图下都只能配置一个源地址，多次配置源地址，只有最后一个源地址生效。

使用实例

# 设置原始流统计报文的源IP地址为19.0.0.3。

<Quidway> system-view
[Quidway] ip netstream export source 19.0.0.3

# 设置聚合流统计报文的源IP地址为19.0.0.4。

<Quidway> system-view 
[Quidway] ip netstream aggregation as
[Quidway-aggregation-as] ip netstream export source 19.0.0.4
