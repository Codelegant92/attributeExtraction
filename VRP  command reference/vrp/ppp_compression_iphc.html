ppp compression iphc

命令功能

ppp compression iphc命令用来启动某接口上的IP头压缩。

undo ppp compression iphc命令用来关闭IP头压缩功能。

缺省情况下，不启动接口上的IP头压缩功能。

命令格式

ppp compression iphc [ nonstandard ]

undo ppp compression iphc

参数说明

nonstandard：非标准兼容的封装格式。

视图

POS接口视图、串行接口视图、虚拟接口模板视图、MP-Group接口视图、Dialer接口视图

缺省级别

2：配置级

使用指南

IP报文头压缩命令将启动IP/UDP/RTP头压缩，以及建立RTP会话的IP/TCP头压缩。

用户必须在链路的两端同时配置IP头压缩命令。

使用实例

# 接口使能IP报文头压缩。

<Quidway> system-view
[Quidway] interface serial 2/0/0:0
[Quidway-serial 2/0/0:0] link-protocal ppp
[Quidway-serial 2/0/0:0] ppp compression iphc
