display rpr traffic interface ring-if

命令功能

display rpr traffic interface ring-if命令用来查看本地节点发送和接收帧的统计信息和错误信息。

命令格式

display rpr traffic interface ring-if interface-number

参数说明

interface-number：接口编号。整数形式，取值范围是0～31。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

表1 display rpr traffic interface ring-if命令输出信息描述

项目

描述

Unicast：

单播的包数和字节数。

Multicast：

组播的包数和字节数。

Host-Received

上送主机的流量。

Host-Send

主机发送的流量。

Ring-Received

从环上接收的流量。

Ring-Send

从环上发送的流量。

Transitted

过环流量（Transitted =Ring-Received- Host-Received）。

ClassA
服务类：

  • 最小时延

  • 最小抖动

  • 保留的业务带宽

ClassB-CIR
服务类：

  • 绑定的时延和抖动

  • 在契约速率范围内的业务保证传输

  • 契约范围的尽力传输

ClassB-EIR

服务类：指超出契约范围的流量。

ClassC
服务类：

  • 最大限度的利用带宽

  • 尽力传输

不同节点根据权重分享剩余带宽，不存在一个节点0带宽的情况。
