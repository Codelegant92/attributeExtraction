next hop

命令功能

next hop命令用来在显式路径中指定下一跳IP地址，按配置的顺序创建路径。

命令格式

next hop ip-address [ include [ strict | loose ] | exclude ]

参数说明

ip-address：显式路径中的下一跳IP地址，取节点接口的IP地址，点分十进制格式。

include：在显式路径中包含此IP地址。

strict：严格下一跳，即，此地址与上一节点必须直连。

loose：松散下一跳，即，此地址与上一节点可以不是直连的。

exclude：在后续的部分路径段中不包括此IP地址。
