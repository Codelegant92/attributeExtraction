display snmp-agent extend error-code status

命令功能

display snmp-agent extend error-code status命令用于查看扩展错误码返回开关是否打开。

命令格式

display snmp-agent extend error-code status

参数说明

无

所有视图

缺省级别

1：监控级

使用指南

网管端没有得到支持扩展错误码的设备返回扩展错误码，可以通过执行命令display snmp-agent extend error-code status，查看设备端的扩展错误码是否已经开启。

使用实例

# 查看当前设备扩展错误码返回开关是否打开。

<Quidway> display snmp-agent extend error-code status
Extend error-code status: enabled
表1 display snmp-agent extend error-code status输出信息描述

项目

描述

Extend error-code status
扩展错误码返回开关的状态：
  • enabled：扩展错误码返回开关处于开启状态。
  • disabled：扩展错误码返回开关处于关闭状态。
