display trunkmembership eth-trunk

命令功能

display trunkmembership eth-trunk命令用来查看Trunk口的成员端口信息。

命令格式

display trunkmembership eth-trunk trunk-id [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

trunk-id：表示Trunk ID，取值范围是0～63。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看Trunk成员。

<Quidway> display trunkmembership eth-trunk 0

项目

描述

Trunk ID

查看的Eth-Trunk接口号。

used status

使用状态，包括VALID和INVALID。

TYPE
trunk的类型。

  • Ethernet：Eth-Trunk

  • Pos：Ip-Trunk

Working Mode
Eth-Trunk接口的工作模式。
  • Normal:表示手工负载分担模式。
  • Static:表示静态LACP模式。

Working State

Eth-Trunk接口的工作状态。

Number Of Ports in Trunk ＝

加入Eth-Trunk接口的端口总数。

Number Of UP Ports in Trunk =

加入Eth-Trunk接口的端口中处于UP状态的端口总数。

operate status

执行状态。
