reset ipv6 routing-table statistics protocol

命令功能

reset ipv6 routing-table statistics protocol命令用来清除IPv6路由表中的统计信息。

命令格式

reset ipv6 routing-table [ vpn6-instance vpn6-instance-name ] statistics protocol { all | protocol }

参数说明

vpn6-instance vpn6-instance-name：VPNv6实例的名称。

all：清除路由表中所有IPv6路由协议的统计信息。

protocol：清除路由表中指定路由协议的统计信息。它的取值可以是下列关键字中的一个：bgp、direct、isis、ospfv3、ripng或static。

使用指南

清除IPv6路由表中的统计信息后，以前的统计信息将无法恢复，务必仔细确认。

使用实例

# 清除路由表中所有IPv6路由协议的统计信息。

<Quidway> reset ipv6 routing-table statistics protocol all
