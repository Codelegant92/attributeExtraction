mpls ldp timer keepalive-hold

命令功能

mpls ldp timer keepalive-hold命令用来配置keepalive保持定时器的值。

undo mpls ldp timer keepalive-hold命令用来恢复缺省值。

缺省情况下，本地会话和远端会话的keepalive保持定时器的值为45秒。

命令格式

mpls ldp timer keepalive-hold interval

undo mpls ldp timer keepalive-hold

参数说明

interval：keepalive保持定时器的值。整数形式，取值范围是30～65535，单位是秒。

视图

接口视图、MPLS LDP远端对等体视图

缺省级别

2：配置级

使用指南

接口视图下配置的是本地会话keepalive保持定时器，远端对等体视图下配置的是远端会话keepalive保持定时器。

如果两个LSR之间使能LDP的链路条数超过1条，所有链路的keepalive保持时间必须相同，否则LDP会话可能不稳定。

说明：

修改keepalive保持定时器的值会导致LDP会话重建。同时，基于此会话所创建的LSP也将重建。

使用实例

# 配置本地会话keepalive保持定时器的值为90秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mpls ldp timer keepalive-hold 90
Warning: All the related sessions will be deleted if the operation is performed
!Continue? (y/n)y
[Quidway-GigabitEthernet1/0/0] quit

# 配置远端会话keepalive保持定时器的值为40秒。

<Quidway> system-view
[Quidway] mpls ldp remote-peer bji
[Quidway-mpls-ldp-remote-bji] mpls ldp timer keepalive-hold 40
Warning: All the related sessions will be deleted if the operation is performed
!Continue? (y/n)y
[Quidway-mpls-ldp-remote-bji] quit
