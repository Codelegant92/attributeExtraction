ip netstream export template option application-label

命令功能

ip netstream export template option application-label命令用来使能TAL信息向NSC发送。

undo ip netstream export template option application-label命令用来禁止TAL信息向NSC发送。

缺省不使能TAL信息发布。

命令格式

ip netstream export template option application-label

undo ip netstream export template option application-label

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

只有当向NSC输出的NetStream版本为v9时，发送的选项模板中才能携带TAL信息。

TAL发布条件满足后，各应用模块发布标签或其它应用信息时，将选项数据通过NetStream发送给NSC。

使用实例

# 在系统视图下使能TAL信息箱NSC发送。

<Quidway> system-view
[Quidway] ip netstream aggregation mpls-label
[Quidway] ip netstream export template option application-label
