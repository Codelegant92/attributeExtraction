e1 channel-set（CE3接口视图）

命令功能

e1 channel-set命令用来对CE3接口的E1通道进行时隙捆绑。

undo e1 channel-set命令用来取消指定的时隙捆绑。

命令格式

e1 e1-number channel-set set-number timeslot-list slot-list

undo e1 e1-number channel-set set-number

参数说明

e1-number：E1通道的顺序号。整数形式，取值范围是1～16。

set-number：指定E1上时隙捆绑形成的channel-set的索引号。整数形式，取值范围是0～30。

slot-list：用于捆绑的时隙编号或时隙范围。整数形式，取值范围是1～31。指定捆绑时隙时，可以指定单个时隙，也可以指定时隙范围。使用“,”分隔多个时隙，使用“-”表示时隙范围。

缺省级别

2：配置级

使用指南

只有当CE3接口工作在通道化模式下时，才能配置该命令。

CE3接口支持通道化到64kbit/s，每个E1最多可捆绑出31个通道，但一个CE3的16个E1捆绑的通道数最多为256个。

使用实例

# 在CE3接口3/0/0接口第3个E1通道上捆绑1个128K的时隙接口

<Quidway> system-view
[Quidway] controller e3 3/0/0
[Quidway-E3 3/0/0] e1 3 channel-set 0 timeslot-list 1,2
