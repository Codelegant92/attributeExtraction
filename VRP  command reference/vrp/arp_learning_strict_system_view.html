arp learning strict（系统视图）

命令功能

arp learning strict命令用来使能ARP严格学习功能，使路由器只学习自己发送的ARP请求报文的应答报文。

undo arp learning strict命令去使能ARP严格学习功能。

命令格式

arp learning strict

undo arp learning strict
