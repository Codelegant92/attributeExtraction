display keepalive packets count

命令功能

display keepalive packets count命令用来查看GRE隧道接口发送给对端的Keepalive报文的数量和Keepalive响应报文的数量，以及从对端接收的Keepalive报文的数量和Keepalive响应报文的数量。

命令格式

display keepalive packets count

参数说明

无

视图

Tunnel接口视图

缺省级别

1：监控级

使用指南

只有将Tunnel接口的隧道类型配置为GRE后才能执行此命令。

使用实例

# 查看GRE隧道接口Tunnel1/0/0发送给对端的Keepalive报文的数量和Keepalive响应报文的数量，以及从对端接收的Keepalive报文的数量和Keepalive响应报文的数量。

<Quidway1> system-view
[Quidway1] interface tunnel 1/0/0
[Quidway1-Tunnel1/0/0] tunnel-protocol gre
[Quidway1-Tunnel1/0/0] display keepalive packets count
Send 34 keepalive packets to peers, Receive 34 keepalive response packets from peers
Receive 34 keepalive packets from peers, Send 34 keepalive response packets to peers
