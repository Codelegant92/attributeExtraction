rmdir（用户视图）

命令功能

rmdir命令用来删除目录。

命令格式

rmdir directory

参数说明

directory：目录名，字符串形式，长度范围是1～64。

视图

用户视图

缺省级别

3：管理级

使用指南

删除的目录必须为空目录。

使用实例

# 删除目录test。

<Quidway> rmdir test
