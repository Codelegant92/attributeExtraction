nqa-ftp-record enable

命令功能

nqa-ftp-record enable命令用来使能把测试结果通过FTP方式发送到FTP服务器。

undo nqa-ftp-record enable命令用来去使能把测试结果通过FTP方式发送到FTP服务器。

缺省情况下，不通过FTP保存NQA测试结果

命令格式

nqa-ftp-record enable

undo nqa-ftp-record enable

参数说明

无

视图

系统视图

缺省级别

3：管理级

使用指南

在配置把测试结果发送到FTP服务器时，需要先配置FTP连接所需要的参数，再使能通过FTP方式把测试结果发送到FTP服务器。

在执行命令nqa-ftp-record enable前，需要确保命令nqa-ftp-record ip-address、nqa-ftp-record username、nqa-ftp-record password、nqa-ftp-record filename已经执行。
