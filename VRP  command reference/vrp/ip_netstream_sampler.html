ip netstream sampler

命令功能

ip netstream sampler { fix-packet fix-packet-number | random-packet random-packet-number | fix-time fix-time-value | random-time random-time-value } { inbound | outbound }命令用来使能NetStream入或出统计的采样功能。

undo ip netstream sampler { inbound | outbound }命令用来禁止NetStream入或出统计的采样功能。

缺省情况下，禁止NetStream入或出统计的采样功能，对报文逐个进行统计。

命令格式

ip netstream sampler { inbound | outbound } packet-interval

undo ip netstream sampler { inbound | outbound }

参数说明

fix-packet-number：固定报文采样的报文间隔。

random-packet-number：随机报文采样的最大报文间隔。

fix-time-value：固定时间采样的时间间隔。

random-time-value：随机时间采样的最大时间间隔。

inbound：对报文的入采样。

outbound：对报文的出采样。

3：管理级

使用指南

系统视图下定义的采样优先级低于接口视图下定义的采样优先级。系统视图下设置了采样功能，若接口下没有设置，但该接口使能了NetStream入或出统计功能，则该接口启动采样功能，且采用系统视图下的设置。

使用实例

# 在接口上定义NetStream入统计每隔1000个报文进行一次采样。

<Quidway>  system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ip netstream sampler fix-packet 1000 inbound
