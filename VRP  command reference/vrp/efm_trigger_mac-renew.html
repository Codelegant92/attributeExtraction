efm trigger mac-renew

命令功能

efm trigger mac-renew命令用来配置在物理接口上清除MAC地址表。

undo efm trigger mac-renew命令用来取消配置在物理接口上清除MAC地址表。

缺省情况下，接口不清除MAC地址表。

命令格式

efm trigger mac-renew

undo efm trigger mac-renew

参数说明

无

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

当需要在CE端感知相连的PE上配置的VLL的相关故障，并通过清除MAC来实现PW的倒换时，需要在CE的对应接口上配置此命令。

当前接口必须已使用efm enable命令使能了EFM OAM。

使用实例

# 配置接口GigabitEthernet1/0/1收到故障通知后清除MAC表项。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/1
[Quidway-GigabitEthernet1/0/1] efm enable
[Quidway-GigabitEthernet1/0/1] efm trigger mac-renew
[Quidway-GigabitEthernet1/0/1] quit
