bpdu-tunnel

命令功能

bpdu-tunnel enable命令用来使能接口的BPDU Tunnel功能。透传来自用户网络的BPDU报文。

bpdu-tunnel disable命令用来取消接口的BPDU Tunnel功能。不透传来自用户网络的BPDU报文。

缺省情况下，设备未使能接口的BPDU Tunnel功能。

命令格式

bpdu-tunnel enable

bpdu-tunnel disable

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

此命令需要配置在二层接口上。

当不同的用户网络需要通过ISP的网络来完成生成树功能，可以在ISP网络的PE设备的用户侧接口上使能BPDU Tunnel功能。这样来自用户网络的BPDU报文就不会上送CPU处理，而是被直接进行转发，穿越ISP的网络。

一般情况下，bpdu-tunnel enable命令配置在PE设备的用户侧接口上。

不能把bpdu-tunnel stp vlan和bpdu-tunnel命令同时配置在同一个接口下，否则会提示配置冲突。

使用实例

# 使能接口GigabitEthernet1/0/0的BPDU Tunnel功能，转发接口收到的BPDU报文。

<Quidway> system-view
[Quidway] interface gigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] port default vlan 1
[Quidway-GigabitEthernet1/0/0] bpdu-tunnel enable
[Quidway-GigabitEthernet1/0/0] stp disable
说明：

当接口为二层接口时，portswitch命令不可用，此时就不需要进行转换。
