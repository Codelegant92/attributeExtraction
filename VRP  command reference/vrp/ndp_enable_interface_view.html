ndp enable（接口视图）

命令功能

ndp enable命令用来使能接口NDP功能。

undo ndp enable命令用来禁止接口NDP功能。

缺省情况下，接口NDP功能处于禁止状态。

命令格式

ndp enable

undo ndp enable

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图和VE接口视图

缺省级别

2：配置级

使用指南

NDP协议是集群管理的基础，建立集群前需要使能全局和相关接口下NDP功能。

说明：
相关接口就是设备上用于集群管理的接口,集群中的设备通过这些接口互相连接。

使用实例

# 在GE1/0/1接口视图下使能接口NDP功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] ndp enable
