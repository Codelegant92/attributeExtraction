display qos wfq

命令功能

display qos wfq命令用来显示接口加权公平队列的配置情况和统计信息。

命令格式

display qos wfq

参数说明

queue-length：队列最大长度

视图

所有视图

缺省级别

1：监控级

使用指南

display qos wfq命令能够观察到的信息包括：

  • 协议队列信息：大小/长度/丢弃包数

  • 加权公平队列信息：大小/长度/丢弃包数

  • 哈希队列的信息

使用实例

# 显示队列配置信息。

<Quidway> display qos wfq interface GigabitEthernet 6/0/0
Interface: GigabitEthernet6/0/0
Output queue : (Protocol queue : Size/Length/Discards) 0/1000/0
Output queue : (Weighted Fair queuing : Size/Length/Discards)  277/100/2570220
Hashed queues :  0/0/64 (Active/Max active/Total)
