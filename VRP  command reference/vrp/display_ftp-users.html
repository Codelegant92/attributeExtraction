display ftp-users

命令功能

display ftp-users命令用来显示当前FTP用户的各项参数。

命令格式

display ftp-users

参数说明

无

视图

所有视图

缺省级别

3：管理级

使用指南

运行该命令可以显示诸如FTP用户名，客户端主机IP地址、端口号，用户的空闲时间和授权目录等信息。

使用实例

# 显示FTP用户参数设置。

<Quidway> display ftp-users
  username            host         port     idle     topdir
  ftp          192.168.200.100     1130        0     hda1:

以上显示信息表示有一个FTP用户和FTP服务器建立了连接。
