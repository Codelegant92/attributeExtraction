display ipsec policy-template

命令功能

display ipsec policy-template命令用来查看安全策略模板的信息。

命令格式

display ipsec policy-template [ brief | name template-name [ seq-number ] ]

参数说明

brief：表示显示所有安全策略模板的简要信息。

name：表示显示名字为template-name，顺序号为seq-number的安全策略模板的信息。

template-name：指定安全策略模板的名字，取值范围是1～15。

seq-number：指定安全策略模板的顺序号，取值范围是1～10000。

如果不指定任何参数，则表示要显示所有安全策略模板的详细信息。如果指定了name template-name，而没有指定seq-number，则表示要显示指定的安全策略模板组的信息。

使用指南

display ipsec policy-template brief命令显示所有安全策略模板的简要信息，如下：

  • 安全策略模板名及顺序号

  • 访问控制列表号

  • 对端地址

在指定安全策略模板的名字时，命令显示安全策略模板的详细信息。

使用实例

# 显示所有安全策略模板的简要信息。

<Quidway> display ipsec policy-template brief
Policy-template-Name     acl            remote-address
------------------------------------------------------
test-tplt300             3120           192.168.0.1
