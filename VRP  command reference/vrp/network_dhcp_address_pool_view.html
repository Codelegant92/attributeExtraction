network（DHCP地址池视图）

命令功能

network命令用来配置动态分配的IP地址范围。

undo network命令用来删除配置。

缺省情况下，未配置动态分配的IP地址范围。

命令格式

network ip-address [ mask { mask | mask-length } ]

undo network

参数说明

ip-address：指定用于动态分配的IP地址池子网地址。

mask：IP地址池网络掩码，不指定该参数时表示采用自然掩码。

mask：指定子网掩码，点分十进制格式。

mask-length：指定掩码的长度。整数形式，取值范围是0～32。

说明：

虽然VRP支持为DHCP地址池配置31位掩码的IP地址，但是因为目前还不能为用户分配该类地址，所以不建议在地址池中配置31位掩码的IP地址。

缺省级别

2：配置级

使用指南

每个DHCP地址池只能配置一个网段，新的配置将覆盖原先的配置。如果系统需要多段地址，则应该使用多个地址池。

使用实例

# 将192.168.8.0/24作为DHCP地址池0的地址空间。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] network 192.168.8.0 mask 255.255.255.0
