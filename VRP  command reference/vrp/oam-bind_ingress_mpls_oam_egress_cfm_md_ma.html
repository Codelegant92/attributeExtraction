oam-bind ingress mpls oam egress cfm md ma

命令功能

oam-bind ingress mpls oam egress cfm md ma命令配置MPLS OAM向以太网CFM传递故障。

undo oam-bind ingress mpls oam egress cfm md ma命令取消MPLS OAM向以太网CFM传递故障。

命令格式

oam-bind ingress mpls oam { lsp-name lsp-name | lsr-id lsr-id tunnel-id tunnel-id } egress cfm md md-name ma ma-name

undo oam-bind ingress mpls oam { lsp-name lsp-name | lsr-id lsr-id tunnel-id tunnel-id } egress cfm md md-name ma ma-name

参数说明

md-name：用来标识一个维护域，与ma-name组合最长为44个字符。

ma-name：用来标识一个维护域，与md-name组合最长为44个字符。

lsp-name：LSP名称，字符串形式，长度范围是1～15。

lsr-id：绑定被检测LSP的隧道出节点的IP地址。

tunnel-id：绑定的被检测LSP的隧道ID，取值范围是1～65534。

使用指南

配置LSP出节点与以太网CFM绑定，OAM管理模块通过绑定关系向以太网CFM传递故障。

配置该命令时，相关的MD、MA和MPLS隧道必需已经创建。
