mac-learning

命令功能

mac-learning enable命令用于使能VSI实例的MAC地址学习能力。

mac-learning disable命令用于去使能VSI实例的MAC地址学习能力。

缺省情况下，使能VSI实例的MAC地址学习能力。

命令格式

mac-learning { enable | disable }

参数说明

enable：使能VSI实例的MAC地址学习能力。

disable：取消VSI实例的MAC地址学习能力。

使用实例

<Quidway> system-view

[Quidway-vsi-company1] mac-learning disable
