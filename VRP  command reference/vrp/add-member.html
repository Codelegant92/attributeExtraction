add-member

命令功能

add-member命令用来将候选交换机加入到HGMP集群中。

命令格式

add-member [ member-number ] mac-address mac-address [ password password ]

参数说明

member-number：指定为候选交换机分配的成员编号。十进制整数形式，取值范围是1～255。

mac-address mac-address：指定候选交换机的MAC地址。mac-address是用16进制表示的MAC地址，格式为H-H-H，每个“H”代表4位16进制数。

password password：指定在侯选交换机上将用户级别切换为3级时所需要的密码。password表示密码。字符串形式，区分大小写，不支持空格，长度范围是1～256。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

当使用本命令却没有指定成员编号时，命令交换机为候选交换机分配成员编号的规则：

  • 如果集群中从未删除过成员交换机，那么命令交换机使用未使用过的成员编号。

  • 如果集群中删除过成员交换机，那么命令交换机使用上一次删除成员交换机时释放的成员编号。

如果在候选交换机上通过super password命令设置了切换到3级用户时所需要的密码，则必须使用参数password password。如果候选交换机上没有进行此项设置，则不需要使用参数password password。

使用实例

# 配置当前设备为命令交换机，并配置集群名称为HUAWEI。该命令交换机与成员交换机连接的接口为GE1/0/1，管理VLAN的编号为1，内部通信地址池为私网IP地址池10.0.0.1/8。并将MAC地址为00E0-FC00-35E7，3级用户密码为123的候选交换机加入HUAWEI集群，成员编号为6。

<Quidway> system-view
[Quidway] vlan 1
[Quidway-vlan1] port gigabitethernet 1/0/1
[Quidway-vlan1] quit
[Quidway] interface vlanif 1
[Quidway-Vlanif1] quit
[Quidway] cluster
[Quidway-cluster] ip-pool 10.0.0.1 8
[Quidway-cluster] build HUAWEI
[HUAWEI_0.Quidway-cluster] add-member 6 mac-address 00e0-fc00-35e7 password 123
