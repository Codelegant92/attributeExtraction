nat traversal

命令功能

nat traversal命令用来使能NAT穿越功能。

undo nat traversal命令用来禁止NAT穿越功能。

缺省情况下，禁止NAT穿越功能。

命令格式

nat traversal

undo nat traversal

参数说明

无

视图

IKE对等体视图

缺省级别

2：配置级

使用指南

NAT穿越功能使IKE在协商过程中发现两个端点之间存在NAT网关，使ESP报文正常穿越NAT网关。

使用实例

# 使能NAT穿越功能。

<Quidway> system-view
[Quidway] ike peer huawei
[Quidway-ike-peer-huawei] nat traversal
