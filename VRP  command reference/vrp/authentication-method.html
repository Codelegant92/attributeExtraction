authentication-method

命令功能

authentication-method命令用来指定一个IKE提议使用的认证方法。

undo authentication-method命令用来设置认证方法为缺省值。

缺省情况下，使用pre-shared key的认证方法。

命令格式

authentication-method { pre-share | rsa-encr }

undo authentication-method

参数说明

pre-share：指定IKE安全提议的认证方法为pre-shared key认证。

rsa-encr：指定IKE安全提议使用RSA加密nonce的认证方法，这种认证方法提供“拒认”（Repudiation）功能，使通信过程无法被回溯（该参数只使用于NE16E、NE08E、NE05产品并由加密卡硬件提供）。

使用指南

使用pre-shared key的认证方法时必须配置认证字。

使用实例

# 指定IKE提议10的认证方法为pre-shared key。

<Quidway> system-view
[Quidway] ike proposal 10
[Quidway-ike-proposal-10] authentication-method pre-share
