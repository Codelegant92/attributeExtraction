map pvp

命令功能

map pvp命令用来配置对端和本地VPI的映射。

undo map pvp命令用来删除该映射。

缺省情况下，不配置任何映射。

命令格式

map pvp remote-vpi

undo map pvp

参数说明

remote-vpi：对端VPI值。整数形式，取值范围是0～255。

视图

ATM-PVP视图

缺省级别

2：配置级

使用指南

本命令只用于ATM透传。

如果对端vpi和本地不同，需要配置vpi映射。对于从本地发出的信元，vpi会被更改为对端的vpi。

使用实例

# 创建ATM子接口上的映射。

<Quidway> system-view
[Quidway] interface atm 1/0/0.1
[Quidway-Atm1/0/0.1] pvp 1
[Quidway-atm-pvp-Atm1/0/0.1-1] map pvp 2
