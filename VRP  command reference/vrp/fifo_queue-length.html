fifo queue-length

命令功能

fifo queue-length命令用来设置帧中继虚电路的FIFO队列长度。

undo fifo queue-length命令用来恢复缺省值。

命令格式

fifo queue-length queue-size

undo fifo queue-length

参数说明

queue-size：FIFO队列长度，即队列能够容纳的数据报文最大个数，取值范围是1～1024。缺省值为256。

缺省级别

2：配置级

使用指南

当路由器作为DCE交换时，若DLCI应用了FRTS，可以设置DLCI的FIFO队列长度。

使用实例

# 设置名为test1的帧中继class的FIFO队列最多可容纳80个报文。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] fifo queue-length 80
