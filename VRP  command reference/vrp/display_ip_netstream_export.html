display ip netstream export

命令功能

display ip netstream export命令用来查看NetStream 统计输出报文的各种信息。

命令格式

display ip netstream export [ ip | ipv6 | mpls-ip | mpls-ipv6 | mpls-label aggregation | mpls-unknown | multicast-vpn ] slot slot-id

参数说明

ip：显示IPv4流量统计输出报文的各种信息。

ipv6：显示IPv6流量统计输出报文的各种信息。

mpls-ip：显示MPLS和IPv4流量统计输出报文的各种信息。

mpls-ipv6：显示MPLS和IPv6流量统计输出报文的各种信息。

mpls-label aggregation：显示MPLS标签聚合流量统计输出报文的各种信息。

mpls-unknown：显示未知MPLS标签流量统计输出报文的各种信息。

multicast-vpn：显示多播VPN流量统计输出报文的各种信息。

slot slot-id：指定显示NetStream统计输出报文信息的接口板槽号。
