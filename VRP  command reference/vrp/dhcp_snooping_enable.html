dhcp snooping enable

命令功能

dhcp snooping enable命令用来使能DHCP Snooping功能。

undo dhcp snooping enable命令用来去使能DHCP Snooping功能。

缺省情况下，DHCP Snooping功能是关闭的。

命令格式

dhcp snooping enable [ interface interface-type interface-number ]

undo dhcp snooping enable [ interface interface-type interface-number ]

参数说明

interface-type：表示接口类型。此参数仅在VLAN视图下才可配置。

interface-number：表示接口号。此参数仅在VLAN视图下才可配置。

视图

系统视图、接口视图、VLAN视图

缺省级别

2：配置级

使用指南

只有打开了全局DHCP Snooping功能开关，与DHCP Snooping功能相关的其他配置命令才能生效。使能DHCP Snooping功能的顺序是先使能全局下的DHCP Snooping功能，再使能VLAN或者接口下的DHCP Snooping功能。

在系统视图下执行此命令表示使能全局的DHCP Snooping功能。

在接口视图、VLAN视图下执行此命令表示仅在本接口或VLAN下使能DHCP Snooping功能开关。支持此功能的接口包括：ethernet接口及其子接口、GE接口及其子接口、Vlanif接口、Eth-Trunk接口及其子接口。

使用实例

# 打开DHCP Snooping功能的全局开关。

<Quidway> system-view
[Quidway] dhcp snooping enable
