pvc-pq

命令功能

pvc-pq命令用来设置帧中继虚电路发送的报文进入的PVC PQ队列类型。

undo pvc-pq命令用来恢复缺省进入的PVC PQ队列类型。

缺省情况下，帧中继虚电路发送的报文进入的PVC PQ队列类型为normal。

命令格式

pvc-pq { top | middle | normal | bottom }

undo pvc-pq

参数说明

top：设置虚电路发送的报文进入的PVC PQ队列类型为top，即高优先队列。

middle：设置虚电路发送的报文进入的PVC PQ队列类型为middle，即中优先队列。

normal：设置虚电路发送的报文进入的PVC PQ队列类型为normal，即正常优先队列。

bottom：设置虚电路发送的报文进入的PVC PQ队列类型为bottom，即低优先队列。

使用指南

PVC PQ队列分为top、middle、normal、bottom四种类型。PVC PQ的队列是相对于DLCI而言的，将接口的队列设置为PVC PQ队列后，每个虚电路的报文只能进入一种类型的PVC PQ队列。

使用实例

# 设置与名为test1的帧中继class关联的虚电路发送的报文进入top类型的PVC PQ队列。

<Quidway> system-view
[Quidway] fr-class test1
[Quidway-fr-class-test1] pvc-pq top
