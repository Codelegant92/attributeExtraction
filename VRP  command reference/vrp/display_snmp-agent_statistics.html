display snmp-agent statistics

命令功能

display snmp-agent statistics命令用来显示SNMP的报文处理统计数据信息。

命令格式

display snmp-agent statistics

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

通过对报文统计信息进行分析，能够得到代理和网管间通信的相关信息，为故障定位提供依据。

使用实例

# 查看SNMP通讯的统计数据信息。

<Quidway> display snmp-agent statistics
