display version

命令功能

display version命令用来显示系统版本信息。

命令格式

display version [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

通过查看版本信息可以获知系统当前使用的软件版本、机架类型、主控板及接口板的相关信息。
