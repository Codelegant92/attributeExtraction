ima-clock

命令功能

ima-clock命令用来设置IMA组的时钟模式。

undo ima-clock命令用来恢复缺省设置。

缺省情况下，IMA组使用共用模式时钟（ctc）。

命令格式

ima-clock { ctc [ link_number number ] | itc }

undo ima-clock

参数说明

ctc：Common Transmit Clock Configuration，公共时钟传输模式。

link_number：使用成员链路的时钟作为IMA组的时钟源。

number：成员链路的链路号。

itc：Independent Transmit Clock Configuration，独立时钟传输模式。

缺省级别

2：配置级

使用指南

IMA组设置使用ITC模式时，作为组成员的ATM E1或ATM T1必须设置为从时钟模式（clock slave）。

使用实例

# 设置IMA组1/0/1采用独立时钟传输模式。

<Quidway> system-view
[Quidway] interface ima-group 1/0/1
[Quidway-Ima-group 1/0/1] ima-clock itc
