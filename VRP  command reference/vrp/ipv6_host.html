ipv6 host

命令功能

ipv6 host命令用来配置静态域名解析表项。每个域名最多可包含8个IPv6地址。

undo ipv6 host命令用来取消已配置的静态域名解析表项。

命令格式

ipv6 host host-name ipv6-address

undo ipv6 host host-name [ ipv6-address ]

参数说明

host-name：指定主机名。字符串形式，长度范围是1～20。

ipv6-address：指定主机名所对应的IPv6地址。

使用指南

如果一个主机名配置了多个与之对应的IPv6地址，当使用主机名查找地址，如ping主机时，使用第一个配置的地址。

使用实例

# 配置静态域名解析表项。

<Quidway> system-view
[Quidway] ipv6 host RTA 20::1
