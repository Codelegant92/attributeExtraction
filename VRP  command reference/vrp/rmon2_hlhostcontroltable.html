rmon2 hlhostcontroltable

命令功能

rmon2 hlhostcontroltable命令用来创建或修改主机控制表中的一行。

undo rmon2 hlhostcontroltable命令用来删除主机控制表中的指定行信息或整个表信息。

命令格式

rmon2 hlhostcontroltable index ctrl-index [ datasource interface interface-type interface-number ] [ maxentry maxentry-value ] [ owner owner-name ] [ status { active | inactive } ]

undo rmon2 hlhostcontroltable [ index ctrl-index ]

参数说明

ctrl-index：唯一识别主机控制表中的行数，是该表的索引。整数形式，取值范围是1～65535。

datasource interface interface-type interface-number：即hlHostControlDataSource，用来识别接口，从而识别子网。该值即接口索引，是该行定义的数据表条目的数据源。在命令行中直接使用接口类型和编号。

maxentry maxentry-value：主机表的最大条目数，取值范围是1～100000，缺省值为50。

owner owner-name：所有者描述信息。字符串类型，长度范围是1～127，不可为空。

status：即hlHostControlStatus，表示主机控制表条目的状态。

active：表示该条目已激活，可以使用。

inactive：表示该条目没有激活，状态为notinservice，不能使用。

使用指南

主机表条目过多会影响到系统转发性能，一般情况下使用缺省值即可。

使用实例

# 创建主机控制表中的一行。

[Quidway] rmon2 hlhostcontroltable index 1 datasource interface GigabitEthernet2/0/0 maxentry 100 owner huawei status active

# 修改主机控制表中的一行hlHostControlStatus为notinservice。

[Quidway] rmon2 hlhostcontroltable index 1 status inactive
