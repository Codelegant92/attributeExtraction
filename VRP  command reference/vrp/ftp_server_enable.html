ftp server enable

命令功能

ftp server enable命令用来启动FTP服务器，允许FTP用户登录。

undo ftp server命令用来关闭FTP服务器，禁止FTP用户登录。

缺省情况下，系统关闭FTP服务器功能。

命令格式

ftp server enable

undo ftp server

参数说明

无

视图

系统视图

缺省级别

3：管理级

使用指南

关闭FTP服务器后，已经登录到该FTP服务器上的用户，除了退出登录的操作外，不能再执行任何操作。

使用实例

# 关闭FTP服务器

[Quidway] undo ftp server
Info: Successed in closing the FTP server.
