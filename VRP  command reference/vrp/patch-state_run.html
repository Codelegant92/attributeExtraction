patch-state run

命令功能

patch-state run命令用来指定重启后的补丁状态为运行态。

命令格式

patch-state run { all | slot slot-id }

参数说明

all：所有单板的补丁状态均为运行态。

slot slot-id：指定单板的补丁状态为运行态。

视图

用户视图

缺省级别

2：配置级

使用指南

此命令用于指定重启后的补丁状态，供下次启动时使用。重启后的补丁状态只支持运行态和非运行态两种。

执行该命令时，需确保startup patch命令设置的下次启动补丁不为空，否则该命令将无法执行。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 指定重启后2号单板的补丁状态为运行态。

<Quidway> patch-state run slot 2
