tftp-server

命令功能

tftp-server命令用来设置HGMP集群公用TFTP服务器的IP地址。

undo tftp-server命令用来删除HGMP集群公用TFTP服务器的IP地址。

缺省情况下，没有设置HGMP集群公用TFTP服务器的IP地址。

命令格式

tftp-server ip-address

undo tftp-server

参数说明

ip-address：指定HGMP集群公用TFTP服务器的IP地址。请将其设置为合法的单播地址，否则将会提示出错信息。

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

重复执行该命令来配置集群公用TFTP服务器，新配置将替代旧配置。

对于NE40E、NE80E、NE5000E和CX600产品，目前暂不支持HGMP集群的TFTP功能。

使用实例

# 配置集群公用TFTP服务器的IP地址为1.1.1.1。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] tftp-server 1.1.1.1
