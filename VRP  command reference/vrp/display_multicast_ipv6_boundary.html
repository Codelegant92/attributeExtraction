display multicast ipv6 boundary

命令功能

display multicast ipv6 boundary命令用来查看所有接口的边界信息。

命令格式

display multicast ipv6 boundary [ ipv6-group-address [ ipv6-group-mask-length ] | interface interface-type interface-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv6-group-address：指定IPv6组播组地址，显示相应的组播路由表信息。取值范围是FF00::0～FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。

ipv6-group-mask-length：指定IPv6组地址的掩码长度。取值范围是8～128。

interface interface-type interface-number：指定接口的类型和编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display multicast ipv6 boundary命令：

  • 如果指定ipv6-group-address，则显示为指定组配置的边界信息。

  • 如果指定ipv6-group-address ipv6-group-mask-length，则显示为指定范围内的组地址配置的边界信息。

  • 如果指定interface，则仅显示在指定接口上配置的组播边界。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
