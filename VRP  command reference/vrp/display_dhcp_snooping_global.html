display dhcp snooping global

命令功能

display dhcp snooping global命令用来显示所有跟DHCP Snooping相关的信息。

命令格式

display dhcp snooping global

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

执行display dhcp snooping global后，设备将显示全局下所有的DHCP Snooping配置信息。显示信息的内容和含义，请参见表1。

使用实例

# 显示DHCP Snooping所有全局相关的信息。

<Quidway> display dhcp snooping global
dhcp snooping enable
dhcp snooping bind-table autosave hda1:/backup.tbl
dhcp snooping nomatch-packet arp action discard
dhcp snooping nomatch-packet ip action discard
dhcp-rate discard count 40 dhcp-rate discard total count 130
dhcp snooping alarm threshold 50
dhcp snooping check dhcp-rate enable
dhcp snooping check dhcp-rate alarm enable
dhcp snooping check dhcp-rate 100
dhcp snooping check dhcp-rate alarm threshold 50
dhcp snooping check dhcp-rate nomatch-packet count-total 200
表1 display dhcp snooping global命令输出信息描述

项目

描述

dhcp snooping enable

全局DHCP Snooping功能已经使能

dhcp snooping bind-table autosave hda1:/backup.tbl

向硬盘备份绑定表，hda1:/backup.tbl是输入的备份绑定表的路径

dhcp snooping nomatch-paket ip action discard

对不匹配的ip报文的转发行为，是discard，默认值是forward

dhcp snooping nomatch-paket arp action discard

对不匹配的arp报文的转发行为，是discard，默认值是forward

dhcp-rate discard count 40

上送协议栈的DHCP报文，本次计数中超过规定的速率而被丢弃的报文数目，这个计数器达到设置的告警阈值后就清零

dhcp-rate discard total count 130

上送协议栈的DHCP报文，所有超过规定的速率而被丢弃报文的总数目，这个计数器不会被清零

dhcp snooping alarm threshold 50

全局告警阈值是50，各个接口上的告警阈值是以这个全局告警阈值为默认值的

dhcp snooping check dhcp-rate enable

DHCP上送速率检查功能已经使能

dhcp snooping check dhcp-rate alarm enable

DHCP 上送速率检查告警已经使能

dhcp snooping check dhcp-rate 100 ]

DHCP上送速率限制是 100个/秒

dhcp snooping check dhcp-rate alarm threshold 50

配置DHCP上送速率检查的告警阈值，超过速率限制而被丢弃的报文达到50个就产生告警

dhcp snooping check dhcp-rate nomatch-packet count-total 200

全局总的丢弃报文数是200个
