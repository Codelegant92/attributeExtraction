shutdown（Eth-Trunk接口视图）

命令功能

shutdown命令用来关闭当前接口。

undo shutdown命令用来启用当前接口。

命令格式

shutdown

undo shutdown

参数说明

无

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

当Eth-Trunk接口没有处于shutdown状态时，可以在成员端口执行shutdown/undo shutdown命令。

当在Eth-Trunk接口执行shutdown后，所有成员端口被自动shutdown，不可以在成员端口执行shutdown/undo shutdown命令。

当Eth-Trunk接口处于shutdown状态时，成员端口加入该Eth-Trunk接口后，自动处于shutdown状态；成员端口退出该Eth-Trunk接口后，shutdown状态不变。

说明：

当Eth-Trunk接口下配有子接口时，在Eth-Trunk主接口连续执行shutdown和undo shutdown操作之间的时间间隔应至少为15秒。

使用实例

# 关闭接口Eth-Trunk1。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] shutdown
