display isis name-table

命令功能

display isis name-table命令用来查看本地和远端路由器名称到系统ID的映射关系表。

命令格式

display isis name-table [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name为IPv4 VPN实例的名称，，区分大小写。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程。vpn6-instance-name为IPv6 VPN实例的名称，，区分大小写。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 给本地IS-IS系统配置一个名称。

[Quidway] isis
[Quidway-isis-1] is-name RUTA

# 为远端IS-IS系统（0000.0000.0041）配置静态名称映射。

[Quidway-isis-1] is-name map 0000.0000.0041 RUTB

# 显示IS-IS名称表。

<Quidway> display isis name-table
              Name table information for ISIS(1)
---------------------------------------------------------------
System ID               Hostname       Type
6789.0000.0001          RUTA           DYNAMIC
0000.0000.0041          RUTB           STATIC
