l2-multicast forwarding-mode

命令功能

l2-multicast forwarding-mode { mac | ip }命令用来配置转发模式。

缺省情况下，转发模式为基于MAC地址转发。

命令格式

l2-multicast forwarding-mode { mac | ip }

参数说明

mac：配置组播数据按MAC地址转发。

ip：配置组播数据按IP地址转发。

说明：

  • 一般情况下，建议使用默认值。

  • 由IPv4的组播地址与MAC地址的对应关系可知，32个IP地址对应一个相同的MAC地址。当配置按MAC地址转发时，如果发生地址冲突，请将配置修改成按IP地址转发组播数据。

视图

VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

使用此命令之前，不允许在VLAN/VSI视图使能IGMP Snooping功能。

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定了VSI；

  • VLAN内包含VE接口。

使用实例

# 配置VLAN 10的转发模式为按MAC地址转发

<Quidway> system-view
[Quidway] vlan 10
[Quidway-vlan10] l2-multicast forwarding-mode mac

# 配置VSI vsi0的转发模式为按IP地址转发

<Quidway> system-view
[Quidway] vsi vsi0
[Quidway-vsi-vsi0] l2-multicast forwarding-mode ip
