cir allow

命令功能

cir allow命令用来设置帧中继虚电路的承诺信息速率（CIR，Committed Information Rate）。

undo cir allow命令用来恢复缺省值。

命令格式

cir allow [ inbound | outbound ] rate-limit

undo cir allow [ inbound | outbound ]

参数说明

inbound：设置报文入方向的承诺信息速率，本参数仅当接口使能帧中继流量监管时有效。

outbound：设置报文出方向的承诺信息速率，本参数仅当接口使能帧中继流量整形时有效。

rate-limit：承诺信息速率，单位是bit/s，取值范围是1～45000000。缺省值是56000 bit/s。

缺省级别

2：配置级

使用指南

承诺信息速率是正常情况下帧中继网络所能提供的发送速率，当网络没有发生拥塞时，它保证用户能够以此速率发送数据。

如果配置时不指定报文方向，则参数同时设置在入方向和出方向。

使用实例

# 设置名为test1的帧中继class承诺信息速率为64000 bit/s。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] cir allow 64000
