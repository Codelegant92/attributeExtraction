igmp-snooping send-query source-address

命令功能

igmp-snooping send-query source-address命令用来配置IGMP通用查询消息的源IP地址。

undo igmp-snooping send-query source-address命令用来恢复该配置的缺省情况。

缺省情况下，IGMP通用查询消息的源IP地址为192.168.0.1。

命令格式

igmp-snooping send-query source-address ip-address

undo igmp-snooping send-query source-address

参数说明

source-address ip-address：指定IGMP通用查询消息的源IP地址，点分十进制格式。

视图

缺省级别

2：配置级

使用指南

当二层网络拓扑变化时，路由器主动向接收到事件通知的端口所在的VLAN/VSI内的所有非路由端口发送IGMP通用查询消息，以便其它路由器快速更新路由端口。

当192.168.0.1已被本路由器所在网络中的其它设备占用时，可使用igmp-snooping send-query source-address命令改变IGMP Snooping模块发送的IGMP通用查询消息的源IP地址。

使用实例

# 配置IGMP Snooping模块响应二层网络拓扑变化事件时发送的IGMP通用查询消息的源IP地址为192.168.10.1。
