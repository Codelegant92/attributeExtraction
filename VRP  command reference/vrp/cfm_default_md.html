cfm default md

命令功能

cfm default md命令用来创建默认MD。

undo cfm default md命令用来删除默认MD。

缺省情况下，默认MD的级别为7。

命令格式

cfm default md [ level level ]

undo cfm default md

参数说明

level level：指定默认MD的级别。整数形式，取值范围是0～7，值越大默认MD的优先级越高。

使用指南

现如今，802.1ag协议分为draft7草案版本和standard2007标准版本，本命令只支持standard2007标准版本使用。

使用cfm enable（系统视图）命令全局使能CFM功能后才能使用本命令。

默认MD的级别必须高于本设备配置MEP节点所有MD的级别，和高优先级MD的级别相等，用于高级别CCM报文穿越，创建MIP节点回复LTR报文。

在设备上配置默认MD，没有关联MA的VLAN内的MIP将基于该默认MD创建。所属默认MD的端口将根据MIP生成规则生成相应的MIP节点。

使用实例

# 创建级别为6的默认MD。

<Quidway> system-view
[Quidway] cfm enable
[Quidway] cfm default md level 6
