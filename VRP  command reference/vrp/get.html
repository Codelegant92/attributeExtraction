get

命令功能

get命令用来下载远程文件并存储在本地。

命令格式

get remote-filename [ local-filename ]

参数说明

local-filename：本地文件名，字符串形式，长度范围是1～64。

remote-filename：远程FTP server上文件名，字符串形式，长度范围是1～64。

视图

FTP客户端视图

缺省级别

3：管理级

使用指南

如果没有指定本地文件名，则认为与远程FTP server上文件同名。
