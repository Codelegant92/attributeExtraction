snmp-agent trap enable port

命令功能

snmp-agent trap enable port命令用来使能所有Ethernet子接口、GE子接口的出、入带宽利用率告警功能。

undo snmp-agent trap enable port命令去使能出、入带宽利用率的告警功能。

缺省情况下，去使能带宽利用率的告警功能。

命令格式

snmp-agent trap enable port { input-rate | output-rate }

undo snmp-agent trap enable port { input-rate | output-rate }

参数说明

input-rate：入带宽利用率告警功能。

output-rate：出带宽利用率告警功能。

视图

系统视图

缺省级别

2：配置级

使用指南

如果使能带宽利用率的告警功能，当带宽利用率超出阈值，或从阈值以上降到阈值以下时，系统将产生告警信息。

使用实例

# 使能所有子接口的出带宽利用率告警功能。

<Quidway> system-view
[Quidway] snmp-agent trap enable port output-rate
