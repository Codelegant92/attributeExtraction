vrrp vrid preempt-mode disable

命令功能

vrrp vrid preempt-mode disable命令用来设置备份组中路由器采用非抢占方式。

undo vrrp vrid preempt-mode命令用来恢复缺省的抢占方式。

缺省情况下，采用抢占方式，并且是立即抢占。

命令格式

vrrp vrid virtual-router-id preempt-mode disable

undo vrrp vrid virtual-router-id preempt-mode

参数说明

virtual-router-id：VRRP备份组号，备份组号范围1～255。

接口视图

缺省级别

2：配置级

使用指南

如果需要优先级高的路由器能够主动成为Master，应将这台路由器设置为抢占方式。

说明：

取消抢占方式，则延迟时间将自动恢复为0秒。

使用实例

# 设置备份组1采用非抢占方式。

[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] vrrp vrid 1 preempt-mode disable
