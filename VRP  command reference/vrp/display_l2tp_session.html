display l2tp session

命令功能

display l2tp session命令用来显示当前设备上建立的L2TP会话信息。

命令格式

display l2tp session

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display l2tp session命令显示当前设备上建立的L2TP会话条数，各条会话的本地会话ID，远端会话ID及其对应的本地隧道ID。配置完L2TP隧道后，可执行display l2tp session命令，检查建立的会话中，是否存在对应的本地隧道ID是属于该L2TP隧道的，存在几条这样的会话。

使用实例

# 查看当前L2TP会话信息。

<Quidway> display l2tp session
Total session = 1
LocalSID      RemoteSID   LocalTID
      1       1           2
表1 display l2tp session命令输出信息描述

项目

描述

Total session

本端建立的会话总数。

LocalSID

隧道进行会话本端所使用的标识号。

RemoteSID

隧道进行会话时远端所使用的标识号。

LocalTID

本端的隧道ID。
