display logfile

命令功能

display logfile命令用来显示日志文件信息。

命令格式

display logfile driver path file-name [ offset | hex ] *

参数说明

driver：日志文件的盘符。

path：日志文件的路径。

file-name：日志文件名。

offset：设置待显示文件的偏移（字节）。

hex：16进制形式显示日志文件。

使用指南

在不选择hex显示方式时，缺省的显示方式是文本显示方式。

日志的时间显示采用基准时间＋偏移时间的方式。当使用offset进行偏移显示时，出现“%0-0-0 00:00:00”时间是属正常现象，因为所选择的offset位置不一定是基准时间，所以在没有基准时间的情况下，偏移时间没有意义，时间都显示为“%0-0-0 00:00:00”。

日志文件中保存的版本信息和设备版本信息不一致时，按十六进制形式显示。

使用实例

# 显示指定路径的日志文件中保存的日志信息。
<Quidway> display logfile hda1:/logfile/2009_01.blg
%Dec 31 19:48:12 2006 NE08-2 HSC/5/LOG:VIU2 power off, in board reset
%Dec 31 19:48:13 2006 NE08-2 HSC/5/LOG:VIU2 power on, in board reset
%Dec 31 19:48:13 2006 NE08-2 HSC/5/LOG:In IntProcess, int type is VIU2 undefine
%Dec 31 19:48:13 2006 NE08-2 HSC/5/LOG:In IntProcess, int type is VIU2 undefine
%Dec 31 19:57:54 2006 NE08-2 HSC/5/LOG: VIU board in slot 2 doesn't start up till now!
# 进入到目录下直接显示日志文件中保存的日志信息。
<Quidway> cd hda1:/logfile/
<Quidway> display logfile 2009_01.blg

# 日志文件版本和系统版本不一致时提示无法解析。
