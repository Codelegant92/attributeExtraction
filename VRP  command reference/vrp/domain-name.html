domain-name

命令功能

domain-name命令用来配置DHCP服务器的全局地址池分配给客户端的域名。

undo domain-name命令用来删除分配的域名。

缺省情况下，未配置分给DHCP客户端的域名，域名为空。

命令格式

domain-name domain-name

undo domain-name

参数说明

domain-name：指定DHCP服务器分配给客户机的域名。字符串形式，长度范围是3～50。

DHCP地址池视图

缺省级别

2：配置级

使用指南

用户通过域名访问Internet时，需要把域名解析为IP地址，使用domain-name命令用来配置DHCP服务器的全局地址池分配给客户端的域名。

使用实例

# 将mydomain.com.cn设定为DHCP地址池0的域名。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] domain-name mydomain.com.cn
