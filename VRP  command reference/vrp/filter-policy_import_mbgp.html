filter-policy import（MBGP）

命令功能

filter-policy import命令用来配置对接收的路由信息进行过滤。

undo filter-policy import命令用来取消对接收的路由信息进行过滤。

缺省情况下，不对接收的路由信息进行过滤。

命令格式

filter-policy { basic-acl-number | ip-prefix ip-prefix-name } import

undo filter-policy import

参数说明

basic-acl-number：基本ACL序号。

ip-prefix ip-prefix-name：指定过滤路由的地址前缀列表名。

视图

BGP-IPv4组播地址族视图

缺省级别

2：配置级

使用指南

该命令可以对MBGP接收路由进行过滤，以决定是否将路由添加到MBGP路由表中。

使用实例

# 使用ACL 2001对组播MBGP接收的路由信息进行过滤。

<Quidway> system-view
[Quidway] bgp 100
[Quidway–bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] filter-policy 2001 import
