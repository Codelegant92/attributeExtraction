dot1q termination vid

命令功能

dot1q termination vid命令用来配置子接口对一层Tag报文的终结功能。

undo dot1q termination vid命令用来取消子接口对一层Tag报文的终结功能。

缺省情况，子接口没有配置对一层Tag报文的终结功能。

命令格式

dot1q termination vid low-pe-vid [ to high-pe-vid ] [ vlan-group group-id ]

undo dot1q termination vid low-pe-vid [ to high-pe-vid ] [ vlan-group group-id ]

参数说明

low-pe-vid：用户报文中的Tag的取值下限，十进制整数形式，取值范围是1～4094。

high-pe-vid：用户报文中的Tag的取值上限，十进制整数形式，取值范围是1～4094。

vlan-group group-id：用户组号，十进制整数形式，取值范围是1～16。

视图

Ethernet子接口视图、GE子接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

在以太网子接口上使用dot1q termination vid命令前，应先在该子接口下配置control-vid vid dot1q-termination命令，否则无法看到dot1q termination vid命令。

子接口收到的用户报文的Tag值应该在命令中指定的pe-vid范围内，否则该报文将被丢弃。

使用实例

在子接口上GE1/0/0.1上配置vlan-group 1的终结功能，外层Tag为100～200。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] control-vid 1 dot1q-termination
[Quidway-GigabitEthernet1/0/0.1] vlan-group 1
[Quidway-GigabitEthernet1/0/0.1] dot1q termination vid 100 to 200 vlan-group 1
