display l2-multicast protection

命令功能

display l2-multicast protection命令用来查看保护组的详细信息。

命令格式

display l2-multicast protection vlan [ vlan-id ]

参数说明

vlan vlan-id：表示VLAN的编号。整数形式，取值范围是1～4094。

视图

所有视图

缺省级别

1：监控级

使用指南

在配置了组播VLAN 1＋1保护组的情况下，命令中加上vlan-id用来查看指定VLAN所在保护组的详细信息。

使用实例

# 查看所有组播组的详细信息。

<Quidway> display l2-multicast protection vlan
PG    Main   Prot   Rest   Wtr   Hold   Main    Prot   Prot      FM   Switch  Group
Indx  vlan   vlan   mode   time  time   state   state  enable    board  cmd   state
------------------------------------------------------------------------------
1     1000   2000   Rev    5     100     OK      OK     Yes      3     none     normal
表1 display l2-multicast protection命令输出信息描述

项目

描述

PG Indx

保护组索引。

Main vlan

工作VLAN的编号。

Prot vlan

保护VLAN的编号。

Rest mode
恢复模式：

  • Rev：代表恢复式。

  • No：非恢复式。

Wtr time

等待恢复时间。

Hold time

倒换延迟时间。

Main state
工作VLAN状态：

  • OK：正常

  • DE：异常

Prot state

保护VLAN状态：

  • OK：正常

  • DE：异常

Prot enable

保护协议使能标志。

FM board

状态机所在单板。

Switch cmd
倒换控制命令。

  • lock：强制从主用VLAN接收组播数据

  • force：强制从保护VLAN接收组播数据

  • manual：当工作VLAN和保护VLAN都正常的时候，强制从保护VLAN接收组播数据

Group state
保护组状态。

  • wtr：用于“恢复”模式下，倒换之后，如果主用VLAN恢复了，则进入该状态，而不是立刻回切到主用VLAN

  • dnr：用于“非恢复”模式，倒换之后，如果主用VLAN恢复了，则进入此状态，并且业务不会回切到主用VLAN

  • normal：当前主用VLAN和保护VLAN都是正常。业务工作在主用VLAN上

  • sf_p：表示备链路故障

  • sf_w：表示主链路故障,备链路无故障
