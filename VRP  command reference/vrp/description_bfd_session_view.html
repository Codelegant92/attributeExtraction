description（BFD会话视图）

命令功能

description命令用来配置BFD会话的描述信息。

undo description命令用来取消BFD会话的描述信息。

缺省情况下，BFD会话的描述信息是空。

命令格式

description description

undo description

参数说明

description：BFD会话描述信息。字符串形式，不支持空格，长度范围是1～51。

视图

BFD会话视图

缺省级别

2：配置级

使用指南

建立BFD会话时，可以使用description命令配置该会话的描述信息。

使用实例

# 配置BFD会话atoc的描述信息。

[Quidway] bfd atoc
[Quidway-bfd-session-atoc] description RouterA_to_RouterC

# 查看BFD会话的配置信息。

[Quidway-bfd-session-atoc] display bfd configuration all verbose
