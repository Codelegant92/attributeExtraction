nqa-ftp-record username

命令功能

nqa-ftp-record username命令用来配置保存NQA测试结果到FTP服务器时使用的用户名。

undo nqa-ftp-record username 命令用来取消配置。

命令格式

nqa-ftp-record username username

undo nqa-ftp-record username

参数说明

username：保存NQA测试结果的FTP服务器用户名，字符串形式，长度范围是1～32。

视图

系统视图

缺省级别

3：管理级

使用指南

配置的保存NQA测试结果到FTP服务器的用户名必须是FTP服务器端已经存在的用户名。

使用实例

# 配置登录FTP服务器需要的用户名。

<Quidway> system-view
[Quidway] nqa-ftp-record username ftp
