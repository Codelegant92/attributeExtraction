display interface-statistics

命令功能

display interface-statistics命令用来查看接口的流量统计信息。

命令格式

display interface-statistics { interface-type interface-number } &<1-5> [ interval interval ] [ times times ]

参数说明

interface-type interface-number：显示指定接口的流量统计信息。

interval interval：指定显示流量统计信息的时间间隔。整数形式，取值范围是10～600，单位是秒。缺省值是10秒。

times times：指定显示流量统计信息的次数。整数形式，取值范围是1～999。缺省值是5。

使用指南

display interface-statistics命令用来查询接口的流量统计信息。缺省情况下，执行该命令后，系统每隔10秒钟实时查询并显示一次接口流量统计信息，一共查询并显示5次。若想终止命令的执行，可以使用快捷键Ctrl+C。

使用实例

# 以10秒为间隔时间，显示接口GE4/0/0流量统计信息，统计3次。

<Quidway> display interface-statistics gigabitethernet 4/0/0 times 3

表1 display interface-statistics命令输出信息描述

项目

描述

Display statistics of the interface at the interval of 10 secondes for up to 3 times. Preess Ctrl + C to stop.

查询接口的流量统计信息，时间间隔是10秒，重复次数3次。该命令可以通过Ctrl+C终止。

seconds left

距离下一次显示接口流量统计信息的时间。缺省情况下，每隔10秒钟显示一次。

times 1

查询第一次。每一次的查询开始时，都会有该字段提示。

Interface

接口名。表示下面的显示信息是该接口的流量统计信息。

Input rate

接收报文的速率。

Output rate

发送报文的速率。

Input

收到报文的字节数和包数。

Output

发送报文的字节数和包数。

Input error

接受到的错误的报文数。

Output error

发送出的错误的报文数。

Input bandwidth utilization

接受报文的速率占总带宽的百分比。

Output bandwidth utilization

发送报文的速率占总带宽的百分比。
