reset traffic-statistics vpn-instance

命令功能

reset traffic-statistics vpn-instance命令用来清除指定L3VPN或所有L3VPN的统计信息。

命令格式

reset traffic-statistics vpn-instance { name vpn-instance-name | all }

参数说明

vpn-instance-name：VPN实例的名称。区分大小写。

all：清除所有L3VPN的统计信息。

视图

用户视图

缺省级别

3：管理级

使用指南

在用户视图下清除指定L3VPN的统计信息，或在用户视图下清除所有L3VPN的统计信息，命令执行成功时请用户确认是否清除所有L3VPN的统计信息，输入Y确认清除，输入N取消清除操作。

使用实例

# 在用户视图下清除所有L3VPN的统计信息

<Quidway> reset traffic-statistics vpn-instance all
Warning:Traffic-statistics information will be cleared! Continue? [Y/N]:Y
