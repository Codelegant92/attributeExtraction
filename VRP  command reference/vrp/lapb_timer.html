lapb timer

命令功能

lapb timer命令用来配置LAPB系统定时器T1、T2、T3。

undo lapb timer命令用来恢复T1、T2、T3的缺省值。

命令格式

lapb timer { t1 t1-value | t2 t2-value | t3 t3-value }

undo lapb timer { t1 | t2 | t3 }

参数说明

t1-value：定时器T1值。整数形式，取值范围是1～64000，单位是毫秒。T1的缺省值是3000毫秒。

t2-value：定时器T2值。整数形式，取值范围是1～32000，单位是毫秒。T2的缺省值是1500毫秒。

t3-value：定时器T3值。整数形式，取值范围是0～255，单位是秒。T3的缺省值是0。

使用指南

T1为发送计时器，当T1计时器到时后，DTE或DCE就启动重发。T1的值应大于从发送方发送一个帧到接收方，由接收方对该帧作出应答，发出应答帧，并且发送方接收到该应答帧之间的最大时间。

T2为接收计时器，当T2计时器到时后，DTE或DCE必须发送证实帧，使得对方DTE(DCE)的T1定时器超时之前能接收到证实帧（T2<T1）。

T3为空闲通道计时器，当T3计时器到时后，DCE向分组层报告链路出现了长时间的空闲通道状态，T3应当大于DCE中的计时器T1。T3为0表示定时器未起作用。

使用实例

# 设置接口Serial1/0/0上LAPB的定时器T1为1000 ms。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol lapb
[Quidway-Serial1/0/0] lapb timer t1 1000
