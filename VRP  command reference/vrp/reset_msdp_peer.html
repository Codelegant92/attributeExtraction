reset msdp peer

命令功能

reset msdp peer命令用来重置与指定MSDP对等体的TCP连接，并清除指定MSDP对等体的所有统计信息。

命令格式

reset msdp [ vpn-instance vpn-instance-name | all-instance ] peer [ peer-address ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

peer-address：表示MSDP对等体的地址。点分十进制形式。如果没有指定peer-address，则重置所有对等体。

使用指南

使用reset msdp peer命令后，与指定对等体之间的TCP连接将断开，并立即重新建立TCP连接，其间MSDP服务的中断有可能影响组播业务。

如果不指定vpn-instance或all-instance，则只重置公网实例中MSDP的TCP连接，并清除MSDP对等体的统计信息。

如果希望在不重置MSDP对等体的情况下清除MSDP对等体的统计信息，请使用reset msdp statistics命令。

使用实例

# 重置公网实例中MSDP对等体125.10.7.6的TCP连接，并清除其统计信息。

<Quidway> reset msdp peer 125.10.7.6
