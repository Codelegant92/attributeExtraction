x25 timer tx2

命令功能

x25 timer tx2命令用来设置DTE（或DCE）的复位请求（指示）发送定时器时延。

undo x25 timer tx2命令用来恢复该定时器时延的缺省值。

命令格式

x25 timer tx2 seconds

undo x25 timer tx2

参数说明

seconds：复位请求（指示）定时器的时延值。整数形式，取值范围是1～1000，单位是秒。缺省情况下，DTE侧复位请求定时器的时延为180秒，DCE侧复位指示定时器的时延为60秒。

使用指南

X.25协议规定，当DTE发送了复位请求（或DCE发送了复位指示）后，应该启动一个定时器；如果在该定时器超时后仍没有收到对端的响应，发送端将采取一定的措施保证本地规程的正常进行。该参数指定了这个定时器在超时到来之前的时延值。

使用实例

# 将X.25接口Serial1/0/0的复位定时器时延值设置为120秒。

<Quidway>system-view
[Quidway]interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 timer tx2 120
