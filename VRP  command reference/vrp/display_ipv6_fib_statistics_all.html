display ipv6 fib statistics all

命令功能

display ipv6 fib statistics all命令用来查看IPv6 FIB的统计信息。主要包括接口板容量及各个转发表项实例的转发表项数量。

命令格式

display ipv6 fib [ slot-number ] statistics all [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

slot-number：需要查看IPv6 FIB策略统计信息的槽号。当不指定槽号时，查看的是主控板上所有的IPv6 FIB转发表项信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行命令display ipv6 fib statistics all时，

如果指定槽号，则可以查看指定槽号的接口板上的IPv6 FIB路由前缀容量及各个转发表项实例的转发表项数量。

如果不指定槽号，则可以查看主控板上所有IPv6 FIB转发表项信息。

使用实例

# 查看2号槽的接口板上的IPv6 FIB策略的统计信息。

<Quidway> display ipv6 fib 2 statistics all
IPv6 FIB Route Prefix Capacity : 320000
IPv6 FIB Total Route Prefix Count: 1 ; Entry Count : 1

IPv6 FIB Public Route Prefix Count :1 ; Entry Count : 1
IPv6 Vpn-instance vpna6 Route Prefix Count :0 ; Entry Count : 0
表1 display ipv6 fib statistics all输出信息描述

项目

描述

IPv6 FIB Route Prefix Capacity :

本接口板的IPv6 FIB路由前缀容量。

IPv6 FIB Total Route Prefix Count : Entry Count :

IPv6转发表项的路由前缀总数和转发表项总数。

IPv6 FIB Public Route Prefix Count : Entry Count :

IPv6公网转发表项的路由前缀总数和转发表项总数。

IPv6 Vpn-instance vpna6 Route Prefix Count : Entry Count :

名为vpna6的IPv6 VPN实例的路由前缀总数和转发表项总数。
