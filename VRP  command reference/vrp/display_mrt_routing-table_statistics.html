display mrt routing-table statistics

命令功能

display mrt routing-table statistics命令用来查看MRT路由表的统计信息。

命令格式

display mrt routing-table [ vpn-instance vpn-instance-name ] statistics [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：VPN实例的名称。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

路由统计信息包括可通过协议增加或删除的路由总数，以及已带删除标签但未删除的激活或未激活路由数。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看MRT路由表的统计信息。

<Quidway> display mrt routing-table statistics
Proto     total      active      added        deleted      freed
          routes     routes      routes       routes       routes
MSTATIC      1          0           1            0            0
