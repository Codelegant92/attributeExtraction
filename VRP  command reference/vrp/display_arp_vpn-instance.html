display arp vpn-instance

命令功能

display arp vpn-instance命令用来查看指定的VPN实例的ARP映射表。

命令格式

在NE16E、NE08E、NE05、NE40E、NE80E和NE5000E上，该命令格式如下：

display arp vpn-instance vpn-instance-name slot slot-id [ dynamic | static ] [ | { begin | exclude | include } regular-expression ]

在NE20、NE20E上，该命令格式如下：

参数说明

vpn-instance-name：指定MPLS VPN的VPN实例名称。

slot slot-id：指定接口板的槽号。

dynamic：显示动态ARP表项。

static：显示静态ARP表项。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在监控ARP表项或ARP出现故障时，可以使用此命令查看ARP表项的内容。

说明：

在不同设备上，该命令执行后得到的显示信息可能存在差别，详细内容请参考相关设备分册。

项目

描述

IP ADDRESS

ARP表项中的IP地址

MAC ADDRESS

ARP表项中的MAC地址

EXPIRE(M)

ARP表项的剩余存活时间（单位是分钟）

TYPE
表项类型及获取该表项的槽位号。类型共有三位。第一位包括以下标志：

  • I：Interface，接口本身的MAC地址；

  • D：Dynamic，通过ARP协议报文获取的动态表项；

  • S：Static，通过静态配置获取的静态表项。

第二位只有标志F，表示该表项已上报路由模块，计算到该IP地址的路由并且更新FIB表项。
说明：

对于没有上报路由模块的表项将显示“-”。对于I类型的ARP表项，该标志位不存在。

第三位表示获取该表项的槽位号。
说明：

对于I类型和S类型的ARP表项将显示“-”。

VPN-INSTANCE

ARP表项所属的VPN实例的名称

INTERFACE

学到ARP表项的接口类型和编号

VLAN/CEVLAN

ARP表项所属的VLAN ID

PVC

PVC所在的接口与VPI/VCI

Total：2

ARP映射表中共有2条ARP表项

Dynamic：1

ARP映射表中共有1条动态ARP表项

Static：0

ARP映射表中共有0条静态ARP表项

Interface：1

ARP映射表中共有1条本身接口的ARP表项
