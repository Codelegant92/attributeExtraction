preference (RIP)

命令功能

preference命令用来指定RIP路由的优先级。并且通过应用路由策略可以对特定的路由设置优先级。

undo preference命令用来恢复路由优先级的缺省值。

命令格式

preference { preference | route-policy route-policy-name } *

undo preference

参数说明

preference：路由的优先级。整数形式，取值范围是1～255。缺省值是100。

route-policy：应用路由策略，对满足条件的特定路由设置优先级。

route-policy-name：路由策略名称。字符串形式，不支持空格，区分大小写，长度范围是1～40。

缺省级别

2：配置级

使用指南

优先级的值越小，实际优先级越高。如果想让RIP路由具有比从其它IGP协议学来的路由更高的优先级，需要配置小的优先级值。优先级的高低将最后决定IP路由表中的路由采取哪种路由算法获取的最佳路由。

使用实例

# 设置RIP路由的优先级为120。

<Quidway> system-view
[Quidway] rip 1
[Quidway-rip-1] preference 120
