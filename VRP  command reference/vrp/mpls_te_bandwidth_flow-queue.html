mpls te bandwidth flow-queue

命令功能

mpls te bandwidth命令用来配置TE隧道的带宽和FQ模板。

undo mpls te bandwidth命令用来删除TE隧道的带宽和FQ模板。

命令格式

mpls te bandwidth bandwidth [ flow-queue flow-queue-name ]

undo mpls te bandwidth

参数说明

bandwidth bandwidth：TE的带宽。取值范围是1～32000000，单位是kbit/s。

flow-queue flow-queue-name：Flow-queue模板名称。字符串格式，长度范围是1～31。

使用指南

mpls te bandwith命令用来配置TE隧道的带宽和Flow-queue 模板。其中Flow-queue 模板配置TE中各种优先级流量的带宽比例。

使用实例

# 在TE下配置带宽为2000kbit/s，Flow-queue 模板使用已经配置的fq1。

<Quidway> system
[Quidway] interface Tunnel 2/0/7
[Quidway- Tunnel2/0/7] mpls te bandwidth 2000 flow-queue fq1
