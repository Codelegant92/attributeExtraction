display cpu-packet statistics interface pvc

命令功能

display cpu-packet statistics interface pvc命令用来查看指定接口的指定PVC下所有上送到CPU的报文的统计信息。

命令格式

display cpu-packet statistics interface interface-type interface-number pvc vpi/vci

参数说明

interface-type interface-number：接口名称和接口编号。

vpi/vci：指定的PVC。

使用指南

使用display cpu-packet statistics interface pvc命令查看指定接口的指定PVC下所有上送到CPU的报文统计信息之和。

使用实例

# 查看接口ATM 1/0/0下VPI/VCI 25/52下所有上送到CPU的报文的统计信息。

<Quidway> display cpu-packet statistics interface atm 1/0/0 pvc 25/52

VPI: 25
