MSTP配置命令

功能描述

MSTP配置命令涉及MSTP基本配置、MSTP的保护功能、MSTP的参数优化等。

说明：

只能对二层以太口进行MSTP配置。对于三层以太网口，应先执行portswitch命令将其切换到二层模式。
