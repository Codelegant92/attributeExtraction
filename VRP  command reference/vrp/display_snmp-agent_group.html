display snmp-agent group

命令功能

display snmp-agent group命令用来显示基于USM的组信息。

命令格式

display snmp-agent group [ group-name ]

参数说明

group-name：指定要显示SNMP信息的组名，1～32字符。

视图

所有视图

缺省级别

1：监控级

使用指南

在管理实体配置基于SNMPv3的组信息时，需要使用display snmp-agent group命令查询代理的组信息，不带参数的情况下显示所有的组信息，包括组名、安全模式、存储方式等。

使用实例

# 显示SNMP组名以及安全模式。

<Quidway> display snmp-agent group

Group name: gg
       Security model: v3 noAuthnoPriv
       Readview: ViewDefault
       Writeview: <no specified>
       Notifyview :<no specified>
       Storage-type: nonVolatile
       Acl:2000
表1 display snmp-agent group命令输出信息描述

项目

描述

Security model

组所用的安全模式

该组对应的只读的MIB视图名

Writeview

该组对应的可写的MIB视图名

Notifyview

该组对应的通知的MIB视图名

Storage-type

存储方式

Acl

该组对应的访问控制列表号
