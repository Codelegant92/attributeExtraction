snmp-agent packet max-size

命令功能

snmp-agent packet max-size命令用来设置Agent能接收和发送的SNMP消息包的最大值。

undo snmp-agent packet max-size命令用来取消当前设置。

命令格式

snmp-agent packet max-size byte-count

undo snmp-agent packet max-size

参数说明

byte-count：Agent能接收和发送的SNMP消息包最大值，单位字节，取值范围484～17940。缺省值为1500字节。

使用指南

根据网络的情况，使用snmp-agent packet max-size命令来设置Agent能接受/发送的SNMP消息包的最大值。

使用实例

# 设置Agent能接收/发送的SNMP消息包的最大值为1042字节。

[Quidway] snmp-agent packet max-size 1042
