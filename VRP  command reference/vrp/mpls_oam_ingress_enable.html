mpls oam ingress enable

命令功能

mpls oam ingress enable命令用来在LSP入节点启动指定的MPLS OAM功能。

undo mpls oam ingress enable命令用来在LSP入节点取消指定的MPLS OAM功能，命令执行后仍保留原来配置的MPLS OAM参数。

命令格式

mpls oam ingress enable { all | tunnel tunnel-number }

undo mpls oam ingress enable { all | tunnel tunnel-number }

参数说明

tunnel tunnel-number：隧道的接口编号。

使用指南

在入节点上，MPLS OAM参数配置完毕后并不生效，需要使用mpls oam ingress enable命令启动。

使用实例

# 启动Tunnel1/0/0的MPLS OAM功能。

<Quidway> system-view
[Quidway] mpls oam ingress enable tunnel 1/0/0
