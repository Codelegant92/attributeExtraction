display isis error

命令功能

display isis error命令用来查看接口或进程收到的错误LSP报文和Hello报文的统计信息。

命令格式

display isis error [ { process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] } [ interface ] ]

display isis error interface { interface-name | interface-type interface-number }

display isis error packet process-id [ packet-number ]

参数说明

process-id：IS-IS进程号。整数形式，取值范围是1～65535。

vpn-instance vpn-instance-name：指定VPN实例名。vpn-instance-name为字符串形式，不支持空格，区分大小写，长度范围是1～31。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定VPN6实例名。vpn6-instance-name为字符串形式，不支持空格，区分大小写，长度范围是1～31。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

interface：按接口显示错误报文的统计信息。

interface-name：接口名称。

interface-type interface-number：指定错误报文来源接口的类型和编号。

packet：显示进程收到的错误LSP报文信息。

packet-number：错误报文个数。整数类型，取值范围是1～10，缺省值为10。

使用指南

当IS-IS出现异常时，使用该命令收集信息，查看各种错误类型的报文。

当指定了IS-IS进程号或VPN或者VPN6实例名，使用参数interface时，显示IS-IS进程下所有接口收到的各种错误报文统计数量。

使用命令display isis error interface interface-type interface-number可以查看指定接口收到的各种错误报文统计数量。

使用参数packet时，必须指定进程号，显示一个进程收到的错误LSP报文相关信息和内容。目前支持最多显示10个LSP错误报文。
