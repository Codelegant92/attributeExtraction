ip address unnumbered

命令功能

ip address unnumbered命令用来配置接口可借用其它接口的IP地址。

undo ip address unnumbered命令用来取消借用其它接口的IP地址。

缺省情况下，本接口不借用其它接口的IP地址。

命令格式

ip address unnumbered interface interface-type interface-number

undo ip address unnumbered

参数说明

interface-type：指定被借用接口的接口类型。

interface-number：指定被借用接口的接口编号。

接口视图

缺省级别

2：配置级

使用指南

封装PPP、HDLC、帧中继的接口以及ATM接口和Tunnel接口可以使用该命令借用以太网接口、Loopback接口或其它接口的IP地址。以太网口不能借用其它接口的IP地址。

由于借用接口本身没有IP地址，无法加上路由，所以必须手工配置路由才能实现路由设备之间的连通。

使用实例

# 封装PPP的POS1/0/0借用GE2/0/0的IP地址。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ip address unnumbered interface gigabitethernet 2/0/0
