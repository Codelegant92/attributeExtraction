peer route-update-interval (BGP)

命令功能

peer route-update-interval命令用来配置向对等体/对等体组发送相同路由前缀更新报文的时间间隔。

undo peer route-update-interval命令用来恢复发送路由更新的时间间隔为缺省值。

缺省情况下，IBGP对等体的路由更新时间间隔为15秒，EBGP对等体的路由更新时间间隔为30秒。

命令格式

peer { group-name | ipv4-address | ipv6-address } route-update-interval interval

undo peer { group-name | ipv4-address | ipv6-address } route-update-interval

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图、BGP-IPv4单播地址族视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP-IPv6单播地址族视图和BGP-VPN6实例视图下生效。

  • group-name参数仅在BGP视图、BGP-IPv4单播地址族视图、BGP-VPN实例视图和BGP-VPN6实例视图下生效。

interval：为发送BGP更新报文的最小时间间隔。整数形式，取值范围为5～600秒。

视图

BGP视图，BGP-IPv4单播地址族视图、BGP-IPv6单播地址族视图，BGP-VPN实例视图、BGP-VPN6实例视图

缺省级别

2：配置级

使用指南

当路由变化时，路由器会发送Update报文通知对等体。但如果同一路由频繁变化时，为避免每次变化路由器都要发送Update报文给对等体，可以通过该命令配置发送该同一路由的Update报文的时间间隔。此命令仅对从对等体学来的路由生效，对自身发布的路由无效。

使用实例

# 配置向对等体发送路由更新的时间间隔。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200
[Quidway-bgp] peer 1.1.1.2 route-update-interval 10
