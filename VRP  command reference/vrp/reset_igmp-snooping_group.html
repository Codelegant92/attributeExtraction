reset igmp-snooping group

命令功能

reset igmp-snooping group命令用来清除组播出端口信息中的动态出端口。

命令格式

reset igmp-snooping group { vlan { vlan-id | all } | vsi { name vsi-name | all } | all }

参数说明

vlan vlan-id：清除指定VLAN的动态出端口。vlan-id表示VLAN编号，取值范围是1～4094。

name vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

vlan all：清除所有VLAN的组播出端口信息中的动态出端口。

vsi all：清除所有VSI的组播出端口信息中的动态出端口。

all：清除所有VSI和VLAN的组播出端口信息中的动态出端口。

使用指南

使用本命令清除某VLAN/VSI的动态出端口，会引起该VLAN/VSI内的主机接收组播流暂时性中断，直到主机再次发出IGMP成员报告消息，路由器重新生成出端口信息后，主机才能再收到组播流。

如果VLAN已经绑定到VSI，在配置reset igmp-snooping group vlan all时，不会提示出错。

使用实例

# 清除所有VLAN和VSI的所有动态出端口。

<Quidway> reset igmp-snooping group all

# 清除VLAN 2的所有动态出端口。

<Quidway> reset igmp-snooping group vlan 2
