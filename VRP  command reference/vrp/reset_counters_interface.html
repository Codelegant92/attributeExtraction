reset counters interface

命令功能

reset counters interface命令用来清除命令display interface显示的接口流量统计信息。

命令格式

reset counters interface [ interface-type [ interface-number ] ]

参数说明

interface-type interface-number：接口的类型及编号。不指定时将清除所有接口的统计信息。

视图

用户视图

缺省级别

2：配置级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

reset counters interface命令清除的是display interface显示信息的最后一部分：接口输入输出报文的统计信息。执行该命令后，display interface显示的报文统计信息会全部清零。所以请务必仔细确认是否要执行该命令。

执行命令reset counters interface对网管的接口流量统计信息不影响。当需要清除网管的接口流量统计信息时，可以执行命令reset counters if-mib interface。

使用实例

# 清除所有接口的统计信息。

<Quidway> reset counters interface
