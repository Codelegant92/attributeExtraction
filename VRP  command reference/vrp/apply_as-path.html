apply as-path

命令功能

apply as-path命令用来替换原有AS_Path属性或在AS_Path属性中增加指定的AS号。

undo apply as-path命令用来取消该设置。

缺省情况下，不替换或追加AS号。

命令格式

apply as-path as-number &<1-10> [ additive ]

undo apply as-path

参数说明

as-number：指定要替换或增加的AS号。在同一个命令行中最多可以同时指定10个AS号。

additive：在原有的AS_Path属性中追加指定的AS号。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

配置此命令后，符合匹配条件的BGP路由的AS_Path属性将会改变。假设原来AS_Path为（30，40，50），在符合匹配条件的情况下，如果配置了apply as-path 60 70 80 additive命令，则AS_Path改为（60，70，80，30，40，50）。如果配置了apply as-path 60 70 80命令，则AS_Path更改为（60，70，80）。

使用实例

# 将原来的AS_Path属性中的AS号替换为200。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply as-path 200
