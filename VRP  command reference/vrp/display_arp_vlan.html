display arp vlan

命令功能

display arp vlan命令用来查看指定VLAN内的ARP映射表。

命令格式

display arp vlan vlan-id interface interface-type interface-number [ | { begin | include | exclude } regular-expression ]

参数说明

vlan vlan-id：VLAN ID，用来标识一个VLAN，取值范围是1～4094。

interface interface-type interface-number：指定的接口类型和接口编号。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在监控ARP表项或ARP出现故障时，可以使用此命令查看ARP表项的内容。

使用实例

# 查看VLAN1的成员接口GE1/0/0的ARP表项。

<Quidway> display arp vlan 1 interface gigabitethernet 1/0/0
IP ADDRESS      MAC ADDRESS     EXPIRE(M) TYPE        INTERFACE   VPN-INSTANCE
                                          VLAN/CEVLAN PVC
------------------------------------------------------------------------------
10.0.0.1        00e0-fcb8-d6b6  19        D-1         GE1/0/0
                                             1/-
------------------------------------------------------------------------------
Total:1         Dynamic:1       Static:0    Interface:0
表1 display arp vlan命令输出信息描述

项目

描述

IP ADDRESS

ARP表项中的IP地址

MAC ADDRESS

ARP表项中的MAC地址

EXPIRE(M)

ARP表项的剩余存活时间（单位是分钟）

TYPE
表项类型及获取该表项的槽位号。类型共有三位。第一位包括以下标志：

  • I：Interface，接口本身的MAC地址；

  • D：Dynamic，通过ARP协议报文获取的动态表项；

  • S：Static，通过静态配置获取的静态表项。

第二位只有标志F，表示该表项已上报路由模块，计算到该IP地址的路由并且更新FIB表项。
说明：

对于没有上报路由模块的表项将显示“-”。对于I类型的ARP表项，该标志位不存在。

第三位表示获取该表项的槽位号。
说明：

对于I类型和S类型的ARP表项将显示“-”。

VPN-INSTANCE

ARP表项所属的VPN实例的名称

INTERFACE

学到ARP表项的接口类型和编号

VLAN/CEVLAN

ARP表项所属的VLAN ID

PVC

PVC所在的接口与VPI/VCI

Total：

ARP映射表中ARP表项的个数

Dynamic：

ARP映射表中动态ARP表项的个数

Static：

ARP映射表中静态ARP表项的个数

Interface：

ARP映射表中本身接口的ARP表项的个数
