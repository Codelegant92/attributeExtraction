display mpls lsp

命令功能

display mpls lsp命令用来查看LSP信息。

display mpls lsp lsp-id命令只能用来查看CR-LSP的信息。

命令格式

display mpls lsp [ | count ] [ | { begin | exclude | include } regular-expression ]

display mpls lsp lsp-id ingress-lsr-id session-id lsp-id [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display mpls lsp attribute { bypass-inuse { inuse | not-exists | exists-not-used } | bypass-tunnel tunnel-name } [ | count ] [ | { begin | exclude | include } regular-expression ]

display mpls lsp [ [ vpn-instance vpn-instance-name | vpn6-instance vpn6-instance-name ] [ protocol { bgp | bgp-ipv6 | rsvp-te | static | static-cr } ] | asbr ] [ { exclude | include } ip-address mask-length ] [ incoming-interface interface-type1 interface-number1 ] [ outgoing-interface interface-type2 interface-number2 ] [ in-label in-label-value ] [ out-label out-lable-value ] [ lsr-role { egress | ingress | transit } ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display mpls lsp [ [ vpn-instance vpn-instance-name | vpn6-instance vpn6-instance-name ] [ protocol ldp ] | asbr ] [ { exclude | include } ip-address mask-length ] [ outgoing-interface interface-type interface-number ] [ in-label in-label-value ] [ out-label out-lable-value ] [ lsr-role { egress | ingress | transit } ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display mpls lsp { stale-incoming-interface | stale-outgoing-interface } interface-index[ in-label in-label-value ] [ out-label out-lable-value ] [ lsr-role { egress | ingress | transit } ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

ingress-lsr-id：入节点的LSR ID，点分十进制格式。

session-id：会话标识。整数形式，取值范围是0～65535。

lsp-id：LSP的标识号。整数形式，取值范围是0～65535。

attribute：查看本地LSP属性。

bypass-inuse：查看使用中的旁路LSP属性。

inuse：查看使用中的旁路LSP属性。

not-exists：没有使用旁路隧道的属性。

exists-not-used：查看存在但未使用的旁路LSP属性。

bypass-tunnel：查看旁路隧道的属性。

tunnel-name：隧道的名称，字符串形式，区分大小写，不支持空格，长度为1～63。

vpn-instance vpn-instance-name：VPN实例名称，字符串形式，区分大小写，不支持空格，长度范围是1～31，用于查看指定VPN实例的LSP详细配置信息。

vpn6-instance vpn6-instance-name：IPv6 VPN实例名称，字符串形式，区分大小写，不支持空格，长度范围是1～31，用于查看指定IPv6 VPN实例的LSP详细配置信息。

protocol：根据建立LSP的协议类型查看LSP信息。

asbr：根据ASBR查看LSP信息。

exclude：输出不包含指定FEC的LSP信息。

include：输出包含指定FEC的LSP信息。

ip-address：显示指定IPv4或者IPv6地址的LSP信息。

mask-length：IPv4地址的掩码长度。整数形式，取值范围是0～32。或者IPv6地址的掩码长度。整数形式，取值范围是0～128。

incoming-interface：根据入接口查看LSP。

interface-type interface-number：接口类型和编号。使用此参数查看指定接口上LSP的配置信息。

outgoing-interface ：根据出接口查看LSP。

in-label in-label-value：显示指定入标签值的LSP信息。整数形式，取值范围是0～1048575。

out-label out-lable-value：显示指定出标签值的LSP信息。整数形式，取值范围是0～1048575。

egress：显示LSR作为Egress节点的LSP信息。

ingress：显示LSR作为Ingress节点的LSP信息。

transit：显示LSR作为Transit节点的LSP信息。

verbose：显示LSP的详细信息。

stale-incoming-interface：显示处于stale状态的LSP入接口信息。

stale-outgoing-interface：显示处于stale状态的LSP出接口信息。

interface-index：指定stale接口的索引号。十六进制整数形式，取值范围是0～FFFFFFFF。

使用指南

如果不使用任何可选参数，将显示所有LSP的信息。

stale接口的接口索引号可通过执行不带参数的display mpls stale-interface命令查看。

项目

描述

LSP Information:
LSP信息：

  • STATIC LSP：此类LSP是手工配置的

  • LDP LSP：此类LSP是通过LDP建立

  • STATIC CR-LSP：此类LSP是手工配置的静态MPLS TE隧道

  • RSVP LSP：此类LSP是通过信令协议RSVP-TE建立的MPLS TE隧道

  • BGP LSP：此类LSP是通过BGP建立

FEC

转发等价类，取值通常为LSP的目的地址。

In/Out Label

入标签/出标签的值。

In/Out IF

入接口/出接口的名称。

Vrf Name

VPN实例的名称。

表2 display mpls lsp verbose命令输出信息描述

项目

描述

No

LSP的编号。

SessionID

CR-LSP的会话ID值。

IngressLsrID

CR-LSP入节点的LSR ID。

LocalLspID

CR-LSP的LSP ID值。

Tunnel-Interface

隧道接口名称。

FEC

转发等价类，取值通常为LSP的目的地址。

TunnelTableIndex

隧道表项索引。

Nexthop

LSP的下一跳IP地址。

In-Label

入标签的值。

Out-Label

出标签的值。

In-Interface

入接口的名称。

Out-Interface

出接口的名称。

LspIndex

LSP的索引号。

Token

LSP的Token值，主要用于指导转发。

LsrType

当前LSR在LSP中所担当的角色：

  • Ingress

  • Transit

  • Egress

Mpls-Mtu

LSP接口的最大传输单元。

TimeStamp

时间戳，表示LSP建立至今的时间。

Bfd-State

BFD的状态。

VrfIndex

VPN实例索引值，该字段仅适用于非CR-LSP。

FrrToken

备份LDP LSP的Token值，该字段仅适用于非CR-LSP。

Outgoing token

出方向的Token值，主要用于指导转发，该字段仅适用于非CR-LSP。

Label Operation

标签操作类型：

  • PUSH

  • SWAP

  • POP

该字段仅适用于非CR-LSP。
