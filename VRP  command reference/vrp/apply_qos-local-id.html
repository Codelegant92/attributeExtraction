apply qos-local-id

命令功能

apply qos-local-id命令用来设置本地QoS ID值。

undo apply qos-local-id命令用来取消配置该设置。

缺省情况下，不设置qos-local-id值。

命令格式

apply qos-local-id qos-local-id

undo apply qos-local-id

参数说明

qos-local-id：本地QoS ID值。整数形式，取值范围是1～4095。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

这是QPPB可以设置的三种QoS参数中的一种，与apply behavior，apply ip-precedence（Route-Policy视图）命令互斥。

QoS Local ID是用于QoS的一个本地标识。应用时，在Route-Policy中设置QoS Local ID的命令，在QoS策略中配置匹配QoS Local ID的命令。在Route-Policy中设置的QoS Local ID会被下发到FIB表中，转发时，系统从FIB表中取出QoS Local ID并根据QoS Local ID执行相应的QoS策略。

使用实例

# 在名为test的Route-Policy下设置QoS ID。

<Quidway> system-view
[Quidway] route-policy test permit node 10
[Quidway-route-policy] apply qos-local-id 10
