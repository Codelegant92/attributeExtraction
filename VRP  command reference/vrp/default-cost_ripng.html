default-cost (RIPng)

命令功能

default-cost命令用来为引入的路由设置RIPng权值。

undo default-cost命令用来恢复缺省值。

命令格式

default-cost cost

undo default-cost

参数说明

cost：引入路由的缺省权值。整数形式，取值范围是0～15。缺省值为0。

视图

RIPng视图

缺省级别

2：配置级

使用指南

该命令用来从其他路由协议引入路由但没有指定权值的情况。

使用实例

# 设置引入路由的缺省权值为2。

[Quidway] ripng 100
[Quidway-ripng-100] default-cost 2

# 恢复引入路由的缺省权值。

[Quidway-ripng-100] undo default-cost
