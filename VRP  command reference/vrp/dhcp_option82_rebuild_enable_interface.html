dhcp option82 rebuild enable interface

命令功能

dhcp option82 rebuild enable interface命令用来使能对DHCP报文插入Option82的功能。

undo dhcp option82 rebuild enable interface命令用来取消对DHCP报文插入Option82的功能。

缺省情况下，不对DHCP报文插入Option82。

命令格式

dhcp option82 rebuild enable interface interface-type interface-number

undo dhcp option82 rebuild enable interface interface-type interface-number

参数说明

interface-type：接口类型。此参数仅在VLAN视图下才可配置。

interface-number：接口号。此参数仅在VLAN视图下才可配置。

视图

VLAN视图

缺省级别

2：配置级

使用指南

本命令只用于在配置了DHCP Relay的接口上配置强插Option82的功能。

配置dhcp option82 rebuild后，如果原来的DHCP报文中没有option82选项，则在DHCP报文中插入option82选项；如果原来的DHCP报文中有option82选项，则去掉原DHCP报文中的option82选项，插入新的option82选项。

使用实例

# 在VLAN100的Ethernet4/0/0接口上使能对DHCP报文强插入Option82功能。

<Quidway> system-view
[Quidway] vlan 100
[Quidway-vlan100] dhcp option82 rebuild enable interface ethernet 4/0/0
