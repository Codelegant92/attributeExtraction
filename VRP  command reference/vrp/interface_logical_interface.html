interface（逻辑接口）

命令功能

interface命令用来创建点到多点或点到点类型的子接口。

undo interface命令用来删除指定的子接口。

命令格式

interface interface-type interface-number.subinterface-number [ p2mp | p2p ]

undo interface interface-type interface-number.subinterface-number

参数说明

interface interface-type interface-number：接口类型和接口编号。

subinterface-number：子接口编号。取值范围是1～4096。整数形式，不同物理接口、不同产品上的子接口取值范围可能不同。所有产品上，一个主接口可创建的子接口数量最大为1024个。

p2mp：配置子接口类型为点到多点。这是缺省的子接口类型。

p2p：配置子接口类型为点到点。

使用指南

目前，只有封装ATM、帧中继或X.25的接口可以配置点到多点或点到点类型的子接口，以太网子接口则是广播类型的子接口。

实际使用中，一个主接口可创建的子接口数量最大为1024个。

说明：

  • 当接口下配有子接口时，在主接口连续执行shutdown（接口视图）和undo shutdown操作的间隔应至少为15秒。

  • 当子接口配置了与L2VPN（VLL或VPLS）的绑定关系时，必须先删除子接口和L2VPN的绑定关系，才能删除子接口。

使用实例

# 在ATM接口2/0/0上创建一个子接口。

<Quidway> system-view
[Quidway] interface atm 2/0/0.1
