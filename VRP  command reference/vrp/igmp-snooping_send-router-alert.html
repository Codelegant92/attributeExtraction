igmp-snooping send-router-alert

命令功能

igmp-snooping send-router-alert命令用来配置路由器向当前VLAN/VSI内发送的IGMP报文的IP报文头中带有Router Alert。

undo igmp-snooping send-router-alert命令用来配置路由器向当前VLAN/VSI内发送的IGMP报文的IP报文头中不带有Router Alert。

缺省状况下，路由器向VLAN/VSI内发送的IGMP报文的IP报文头中带有Router Alert。

命令格式

igmp-snooping send-router-alert

undo igmp-snooping send-router-alert

参数说明

无

视图

VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

通常情况下路由器不会发送IGMP查询报文，只有当网络拓扑发生变化引发MSTP重新计算后，由于VLAN/VSI报文的转发路径可能发生变化，因此路由器会发送IGMP通用查询消息，查询各端口下是否有组播组成员。

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定了VSI；

  • VLAN内包含VE接口。

使用实例

# 配置路由器向VLAN 2内发送的IGMP报文的IP报文头中不带有Router Alert。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] undo igmp-snooping send-router-alert
