display pim grafts

命令功能

display pim grafts命令用来查看未确认的PIM-DM嫁接信息。

命令格式

display pim [ vpn-instance vpn-instance-name | all-instance ] grafts [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在PIM-DM中，当路由器发出嫁接消息，需要等待上游回复确认消息。

使用display pim grafts命令可以查看已发出但未收到确认的PIM-DM嫁接信息。

若不指定vpn-instance或all-instance，则只显示公网实例的相关信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示公网实例未确认的PIM-DM嫁接信息。

<Quidway> display pim grafts
VPN-Instance: public net

20.0.5.200                 226.3.3.3                 00:02:52 00:00:02
