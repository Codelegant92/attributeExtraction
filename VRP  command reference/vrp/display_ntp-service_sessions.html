display ntp-service sessions

命令功能

display ntp-service sessions命令用来查看本地NTP服务维护的所有会话信息。

命令格式

display ntp-service sessions [ verbose ]

参数说明

verbose：显示NTP会话的详细信息。

使用指南

执行命令display ntp-service sessions如果不使用verbose选项，将显示会话的简要信息。

使用实例

# 查看本地设备的NTP会话信息。

项目

描述

clock source

时钟源地址。

clock stratum

本地系统所处的NTP stratum。

reference clock ID

当本地系统已被同步到一个远程NTP服务器或某个时钟源时，指示远程服务器的地址或时钟源的标识。当该服务器处于某一VPN内时，将显示该VPN实例的名称。

local mode

本地系统模式。

peer mode

对端系统模式。

offset

表示相对上一级时钟源的时差。

delay

表示相对上一级时钟源的延时。

disper

表示相对于上一级时钟源的离差。

root delay

本地到主参考时钟总的系统延迟。

root disper

本地相对主参考时钟的系统离差。

reach

可达性标记。表明了相对于时钟源的可达性。

sync dist

表示相对上一级时钟源的同步距离，此参数评价描述时钟源，NTP选择同步距离最小的时钟源。

sync state
同步的状态：

  • 0：时钟没有被更新过

  • 1：从配置信息中得到频率信息

  • 2：时钟被设定

  • 3：时钟被设定，但是还没有决定频率

  • 4：时钟被同步

  • 5：发现错误
