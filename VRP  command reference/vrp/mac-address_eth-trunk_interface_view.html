mac-address（Eth-Trunk接口视图）

命令功能

mac-address命令用来配置当前Eth-Trunk口的MAC地址。

undo mac-address命令用来删除当前Eth-Trunk口的MAC地址。

缺省情况下，Eth-Trunk接口的MAC地址为系统的MAC。

命令格式

mac-address mac-address

undo mac-address

参数说明

mac-address：Eth-Trunk口MAC地址，格式为H-H-H。

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

只有当Eth-Trunk处于三层模式时，才能使用mac-address命令。

使用实例

# 配置Eth-Trunk 1的MAC地址。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] mac-address 0001-2345-abcd
