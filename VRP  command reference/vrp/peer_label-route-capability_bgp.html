peer label-route-capability (BGP)

命令功能

peer label-route-capability命令用来使能发送标签路由能力。

undo peer label-route-capability命令用来取消该功能。

命令格式

peer { group-name | ipv4-address } label-route-capability

undo peer { group-name | ipv4-address } label-route-capability

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

视图

BGP视图、BGP-IPv4单播地址族视图、BGP-VPN实例视图、BGP-IPv6单播地址族视图

缺省级别

2：配置级

使用指南

使能或禁止发送标签路由能力时，BGP连接会自动断开，然后重新进行邻居能力协商。

说明：

BGP-IPv6单播地址族视图下，仅支持参数ipv4-address。

如果需要配置6PE，请在IPv6单播地址族视图下指定Peer为IPv4单播地址。

使用实例

# 使能BGP发送标签路由能力。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 100
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp] peer 1.1.1.2 label-route-capability

# 使能BGP发送标签路由能力。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 100
[Quidway-bgp] ipv6-family unicast
[Quidway-bgp-af-ipv6] peer 1.1.1.2 enable
[Quidway-bgp-af-ipv6] peer 1.1.1.2 label-route-capability
