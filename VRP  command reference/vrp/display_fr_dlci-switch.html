display fr dlci-switch

命令功能

display fr dlci-switch命令用来显示配置的帧中继交换的信息，可以查看用户的帧中继交换配置是否正确。

命令格式

display fr dlci-switch [ interface interface-type interface-number ]

参数说明

interface-type：查看指定接口类型的虚电路交换信息。

interface-number：查看指定接口类型下指定接口编号的虚电路交换信息。

说明：

指定接口只能是主接口。不指定接口则显示全部信息。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 显示配置的帧中继交换的信息。

<Quidway> display fr dlci-switch
Frame relay switch statistics for board 1
Status   Interface(Dlci)      <----->      Interface(Dlci)
Inactive Serial1/1/1:0(100)               Serial1/1/1:1(300)
表1 display fr dlci-switch命令输出信息描述

项目

描述

输入接口和输入DLCI和输出接口和输出DLCI。“<----->”表示帧中继交换两端接口的接口对应关系。
