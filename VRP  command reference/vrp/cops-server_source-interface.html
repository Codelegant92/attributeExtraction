cops-server source-interface

命令功能

cops-server source-interface命令用于配置发送COPS报文的源接口。

undo cops-server source-interface命令用于取消源接口的配置。

缺省情况下，系统未配置发送COPS报文的源接口。

命令格式

cops-server source-interface interface-type interface-number

undo cops-server source-interface

参数说明

interface-type：源接口类型，可以选择物理接口或者环回接口。

interface-number：源接口编号，不同的接口类型有不同编号格式。

使用指南

发送COPS报文的源接口是指发送COPS报文时，使用该接口的IP地址作为COPS报文的源IP地址。所以指定的源接口必须有IP地址。

必须配置COPS报文的源接口，COPS连接才能建立。

建议配置逻辑接口作为COPS报文的源接口，如Loopback接口。物理接口的状态失效可能会导致系统不能正常接收COPS服务器发回的报文。

执行命令cops-server source-interface之后，需要在COPS服务器组视图下执行命令undo active和active使配置生效。

使用实例

# 配置发送COPS报文的源接口为Loopback 0接口。

<Quidway> system-view
[Quidway] interface loopback 0
[Quidway-LoopBack0] ip address 1.1.1.1 24
[Quidway-LoopBack0] quit
[Quidway] cops-server source-interface loopBack 0
