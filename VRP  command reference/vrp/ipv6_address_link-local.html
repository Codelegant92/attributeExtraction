ipv6 address link-local

命令功能

ipv6 address link-local命令用来手动配置指定接口的链路本地地址。

undo ipv6 address link-local命令用来删除接口上手动配置的链路本地地址。

应尽量避免链路本地地址的改变。

命令格式

ipv6 address ipv6-address link-local

undo ipv6 address ipv6-address link-local

参数说明

ipv6-address：指定IPv6地址。在配置链路本地地址时，指定的IPv6地址的前缀必须匹配FE80::/10。

视图

接口视图

缺省级别

2：配置级

使用指南

此命令支持Ethernet接口及其子接口、GigabitEthernet接口及其子接口、POS接口、Serial接口、Loopback口、Eth-Trunk接口及其子接口、IP-Trunk接口以及Tunnel接口。

使用undo ipv6 address link-local命令只能删除接口上手动配置的链路本地地址，若要删除自动配置的链路本地地址，请使用命令undo ipv6 address auto link-local。

使用实例

# 手动配置以太网口的链路本地地址。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 enable
[Quidway-GigabitEthernet1/0/0] ipv6 address fe80::1 link-local
