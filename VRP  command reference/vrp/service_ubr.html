service ubr

命令功能

service ubr命令用来指定PVC的业务类型为非确定速率（Unspecified Bit Rate，UBR）。

缺省情况下，创建了一个PVC后，业务类型即为UBR。

命令格式

service ubr

参数说明

无

视图

ATM-PVC视图

缺省级别

2：配置级

使用指南

可以使用本命令以及service vbr-nrt，service vbr-rt，service cbr命令来设置PVC的业务类型和业务参数。新指定的PVC业务类型将会覆盖已有的业务类型。

使用实例

# 创建一条名为“huawei”、VPI/VCI为1/101的PVC。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvc huawei 1/101

# 指定该PVC的业务类型为ubr。

[Quidway-atm-pvc-Atm1/0/0-1/101-huawei] service ubr
