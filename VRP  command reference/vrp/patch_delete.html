patch delete

命令功能

patch delete命令用来删除补丁。

缺省情况下为在主用主控板上删除补丁。

命令格式

patch delete [ all | slave | slot slot-id ]

参数说明

all：删除所有单板上的补丁。

slave：删除备用主控板上的补丁。

slot slot-id：删除指定单板上的补丁。

视图

用户视图

缺省级别

3：管理级

使用指南

执行patch delete all命令，将在所有在位单板上（包括主用主控板、备用主控板和所有接口板）删除补丁。执行此命令后，不论补丁处于任何状态，都将被删除。

当备用主控板正在注册中且尚未注册成功时，如果进行补丁删除操作，则系统会提示：是否确认继续执行补丁操作。

说明：

  • 执行此命令可能影响系统的正常运行，在删除前应仔细确认。

  • 目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 删除所有补丁。

<Quidway> patch delete all
