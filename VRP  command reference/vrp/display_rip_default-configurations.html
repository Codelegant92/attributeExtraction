项目

描述

Protocol Level Default Configurations

RIP协议级的缺省信息

RIP version

RIP版本1或2

Preference

RIP进程优先级

Checkzero

零域检查

Default-cost

缺省开销值

Auto Summary

路由聚合是否使能

Hostroutes

主机路由是否使能

Maximum Balanced Paths

等价路由的最大数目

Update time

RIP更新间隔时间

Suppress time

路由抑制时间

Default Route

缺省路由

Verify-source

源验证

Interface Level Default Configurations

RIP接口级的缺省信息

Metricin

接口接收RIP报文时给路由增加的度量值

Metricout

接口发送RIP报文时给路由增加的度量值

Input Packet Processing

是否使能了入报文处理

Output Packet Processing

是否使能了出报文处理

Poison Reverse

接口是否使能了毒性反转特性

Split Horizon

  • For Broadcast and P2P Interfaces

  • For NBMA Interfaces

接口是否使能了水平分割特性

  • 广播接口和P2P接口是否使能了水平分割特性

  • NBMA接口是否使能了水平分割特性

Packet Transmit Interval

报文转发间隔（毫秒）

Packet Transmit Number

报文转发数目

RIP Protocol Version

RIP版本1或2
