display bgp multicast routing-table community

命令功能

display bgp multicast routing-table community命令用来查看指定MBGP团体的路由信息。

命令格式

display bgp multicast routing-table community [ aa:nn | community-number ] & <0-13> [ internet | no-advertise | no-export | no-export-subconfed ] * [ whole-match ]

[ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

aa:nn：表示MBGP团体号，最多可同时指定13个MBGP团体。

community-number：表示MBGP团体号，最多可同时指定13个MBGP团体。

internet：显示带有internet团体属性的MBGP路由信息。internet团体属性表示向所有远端对等体发送匹配路由。

no-advertise：显示带有no-advertise团体属性的MBGP路由信息。no-advertise团体属性表示不向任何对等体发送匹配路由。

no-export：显示带有no-export团体属性的MBGP路由信息。no-export团体属性表示不向自治系统外部通告路由，但会通告联盟中的其它子自治系统。

no-export-subconfed：显示带有no-export-subconfed团体属性的MBGP路由信息。no-export-subconfed团体属性表示不向本地自治系统外发送匹配路由。

whole-match：精确匹配。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
