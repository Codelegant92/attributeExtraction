radius-server user-name domain-included

命令功能

radius-server user-name domain-included命令用来设置RADIUS服务器的用户名包含域名。

undo radius-server user-name domain-included命令用来设置RADIUS服务器的用户名不包含域名。

缺省情况下，用户名中包括域名。

命令格式

radius-server user-name domain-included

undo radius-server user-name domain-included

参数说明

domain-included：用户名格式中包括域名，即：用户名@域名。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

用户名通常采用“纯用户名@域名”格式，“@”后面的部分为域名。

如果RADIUS服务器不接受带域名的用户名时，可以配置将用户名的域名去除后再传送给RADIUS服务器。

执行undo radius-server user-name domain-included命令后，客户端向RADIUS服务器发送认证、授权、计费请求时，会将用户名中的域名都去掉。

只有当该RADIUS Server模板没有用户使用时，才能改变此配置。

使用实例

# 设置用户名格式中包含域名。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server user-name domain-included
