display port-mapping

命令功能

display port-mapping命令用来查看用户定义的端口映射情况。

命令格式

display port-mapping [ application-name | port port-number ]

参数说明

application-name：指定用于端口映射的应用的名称。可选的应用有：ftp、http、h323、smtp、rtsp。

port-number：端口号（0～65535)。

使用指南

display port-mapping命令可以查看用户定义的端口映射信息包括：

  • 系统定义的端口映射表

  • 用户自定义的端口映射表

使用实例

# 显示端口映射的所有信息。

<Quidway> display port-mapping
SERVICE    PORT       ACL        TYPE
 -------------------------------------------------
  ftp          21                  system defined
  smtp         25                  system defined
  http         80                  system defined
  rtsp        554                  system defined
  h323       1720                  system defined
