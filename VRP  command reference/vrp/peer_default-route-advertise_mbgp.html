peer default-route-advertise（MBGP）

命令功能

peer default-route-advertise命令用来设置给对等体或对等体组发布缺省路由。

undo peer default-route-advertise命令用来取消已有的配置。

缺省情况下，不向对等体或对等体组发布缺省路由。

命令格式

peer { group-name | peer-address } default-route-advertise [ route-policy route-policy-name ]

undo peer { group-name | peer-address } default-route-advertise

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

route-policy route-policy-name：指定路由策略。

视图

缺省级别

2：配置级

使用指南

该命令不需要在路由表中存在缺省路由，而是无条件地向对等体发送一个下一跳为自身的缺省路由。

使用实例

# 指定给对等体组test发布缺省路由的对等体组。

<Quidway> system-view
[Quidway] bgp 100
[Quidway] ipv4-family multicast
[Quidway-bgp-af-multicast] peer test enable
[Quidway-bgp-af-multicast] peer test default-route-advertise
