embedded-rp

命令功能

embedded-rp命令用来使能嵌入式RP功能。

undo embedded-rp命令用来关闭嵌入式RP支持。

缺省情况下，嵌入式RP是使能的。

命令格式

embedded-rp [ basic-acl6-number ]

undo embedded-rp [ basic-acl6-number ]

参数说明

basic-acl6-number：指定基本访问控制列表序列号。整数形式，取值范围是2000～2999。

视图

IPv6 PIM视图

缺省级别

2：配置级

使用指南

embedded-rp命令适用的组范围是FF70::/12。

embedded-rp命令与acl命令配合使用。在ACL视图下，通过rule命令的source参数定义组播组地址范围。

使用实例

# 使能组播组ff72:13::对应的嵌入式RP功能。

<Quidway> system-view
[Quidway] acl ipv6 number 2000
[Quidway-acl6-basic-2000] rule permit source ff72:13:: 128
[Quidway-acl6-basic-2000] quit
[Quidway] pim-ipv6
[Quidway-pim6] embedded-rp 2000
