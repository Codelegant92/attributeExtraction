dhcp server dns-list（接口视图）

命令功能

dhcp server dns-list命令用来指定本接口所连接的DHCP客户端的DNS服务器的IP地址。

undo dns-list命令用来删除为接口下的DHCP客户端指定的DNS服务器的IP地址。

缺省情况下，在接口上未指定DNS服务器的IP地址。

命令格式

dhcp server dns-list ip-address &<1-8>

undo dhcp server dns-list { ip-address | all }

参数说明

ip-address：指定DNS服务器的IP地址。一条命令中最多可以配置8个DNS服务器的IP地址，用空格分隔。

all：删除为客户端指定的所有DNS服务器的IP地址。

使用指南

实现通过域名方式访问主机，使用dhcp server dns-list命令用来指定本接口所连接的DHCP客户端的DNS服务器的IP地址。在目前的实现中，每个DHCP地址池最多可以设置8个DNS服务器地址。

使用实例

# 为GE1/0/0的接口DHCP地址池指定DNS服务器的地址为1.1.1.254。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet 1/0/0] dhcp server dns-list 1.1.1.254
