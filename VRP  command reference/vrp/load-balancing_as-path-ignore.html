load-balancing as-path-ignore

命令功能

load-balancing as-path-ignore命令用来设置路由在形成负载分担时不比较路由的AS-Path属性。

undo load-balancing as-path-ignore命令用来关闭该功能。

缺省情况下，路由在形成负载分担时比较路由的AS-Path属性。

命令格式

load-balancing as-path-ignore

undo load-balancing as-path-ignore

参数说明

无

视图

BGP视图、BGP-VPN实例视图和BGP-IPv4单播地址族视图

缺省级别

1：监控级

使用指南

配置load-balancing as-path-ignore命令后，路由形成负载分担时不再比较as-path属性。该命令主要用于EBGP和IBGP路由负载分担场景，由于该命令会改变路由负载分担的条件，配置时需要慎重考虑。

说明：

load-balancing as-path-ignore命令和bestroute as-path-neglect命令互斥。即在配置bestroute as-path-neglect命令后不能配置load-balancing as-path-ignore命令。

使用实例

# 路由在形成负载分担时，忽略比较路由的as-path属性。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family vpn-instance vpna
[Quidway-bgp-vpna] load-balancing as-path-ignore
