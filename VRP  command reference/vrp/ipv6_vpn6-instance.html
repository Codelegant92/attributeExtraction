ipv6 vpn6-instance

命令功能

ipv6 vpn6-instance命令用来创建IPv6 VPN实例。

undo ipv6 vpn6-instance命令用来删除指定的IPv6 VPN实例。

缺省情况下，未定义IPv6 VPN实例。

命令格式

ipv6 vpn6-instance vpn6-instance-name

undo ipv6 vpn6-instance vpn6-instance-name

参数说明

vpn6-instance-name：IPv6 VPN实例的名称。字符串形式，区分大小写，长度范围是1～31。如果本设备不存在该名称的VPN实例，系统执行该命令后将创建一个使用该名称的VPN实例。
