display mpls te rrvpn-tunnel（VLL）

命令功能

display mpls te rrvpn-tunnel命令用来查看为L2VPN分配资源的情况。

命令格式

display mpls te rrvpn-tunnel [ tunnel interface-number | auto-bypass-tunnel tunnel-name ]

参数说明

tunnel interface-number：指定隧道接口。如果指定隧道接口，只显示使用该隧道的VPN占用资源的情况。

auto-bypass-tunnel tunnel-name：显示某一条指定的自动生成的Bypass tunnel信息。隧道名称为字符串形式，不支持空格，长度范围是1～63。

使用指南

配置资源隔离VPN（包括L3VPN和VLL）后，可使用display mpls te rrvpn-tunnel命令验证配置结果。

使用实例

# 在设备上查看指定隧道的资源被VPN占用的情况。

<Quidway> display mpls te rrvpn-tunnel

表1 display mpls te rrvpn-tunnel命令输出信息描述

项目

描述

Tunnel Name

VPN资源隔离所使用的隧道接口。

Interface Id

该隧道接口的ID。

Status
第一个“Staus”表示主隧道的接口状态，其他的“Status”表示子隧道接口的接口状态。

  • up：表示该隧道可用。

  • down：表示该隧道不可用。

Bandwidth

带宽值。主隧道的带宽值大于或等于子隧道带宽值的总和。

L3V Count

使用该隧道的L3VPN的个数。

VLL Count

使用该隧道的VLL的个数。

Vpn Instance

绑定该VLL的接口。

Qos Policy

为该VLL配置的QoS策略。
