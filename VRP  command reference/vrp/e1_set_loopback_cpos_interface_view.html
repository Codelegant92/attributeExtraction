e1 set loopback（CPOS接口视图）

命令功能

e1 set loopback命令用来设置E1通道的环回模式。

undo e1 set loopback命令用来禁止环回。

缺省情况下，禁止任何形式的环回。

命令格式

e1 e1-number set loopback { local | payload | remote }

undo e1 e1-number set loopback

参数说明

e1-number：CPOS的E1通道号。整数形式，取值范围是1～63。

local：使能E1通道对内自环。

payload：使能E1通道对外载荷环回。

remote：使能E1通道对外远端环回。

缺省级别

2：配置级

使用指南

E1通道提供丰富的环回功能，可用于不同层次的测试。

在对内自环模式下，发端的数据直接被环回到收端。

在对外载荷环回模式下，收端接收的数据经过E1成帧器，生成载荷后再进行环回。

在对外远端环回模式下，收端接收的数据不经过E1成帧器，未生成载荷即进行环回。

当对E1通道设置环回时，该环回也会在该E1通道下的同步串口生效：
  • 如果同步串口的链路层协议是PPP，当设置环回时，接口的物理状态变为Up，而链路协议状态为Down。
  • 如果同步串口的链路层协议是HDLC，当设置环回时，接口的物理状态和链路协议状态都是Up。

使用实例

# 设置E1通道1进行对外远端环回。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] e1 1 set loopback remote
