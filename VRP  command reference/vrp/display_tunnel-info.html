display tunnel-info

命令功能

display tunnel-info命令用来查看隧道信息。

命令格式

display tunnel-info { tunnel-id | all | statistics [ slots ] } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

tunnel-id：隧道ID。如果指定的隧道ID不存在，将提示命令执行错误。

all：显示所有隧道的信息。

statistics：显示所有隧道的统计信息。

slots：按接口板顺序显示隧道信息的统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display tunnel-info all命令可以用来

  • 查看本设备上存在哪些建立成功的隧道，及这些隧道的Tunnel ID、隧道类型、目的IP地址、Token信息。

  • 用来判断某隧道是否建立成功，因为如果隧道建立不成功，其Tunnel ID为0x0。

display tunnel-info all命令的显示信息中只包含建立成功的隧道的信息，而不包含Tunnel ID为0x0的隧道的信息。

如果只知道隧道的Tunnel ID，但想了解该隧道的详细信息，可以使用display tunnel-info tunnel-id命令查看。

display tunnel-info statistics命令可以用来查看本设备上成功建立了几条各类型的隧道。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看隧道信息。

<Quidway> display tunnel-info 1100200a
Tunnel ID:                    0x1100200a
Tunnel Token:                 115712
Type:                         cr lsp
Destination:                  3.3.3.3
Out Slot:                     1
Instance ID:                  0
Interface:                    Tunnel1/0/1
Sub Tunnel ID:                0x0
表1 display tunnel-info命令输出信息描述

项目

描述

Tunnel ID

隧道ID，十六进制形式，是系统自动分配的。

Tunnel Token

用于MPLS转发的Token值，是Tunnel ID的一部分，系统自动分配的。

Type

隧道类型，例如GRE、MPLS LSP、CR-LSP等。不同类型隧道的显示信息也有所区别。

Destination

隧道的目的IP地址。

Out Slot

实际发出报文时使用的槽位号。

Instance ID

VPN实例ID，值为0表示该隧道为公网隧道。

Interface

隧道所对应的本端Tunnel接口。

Sub Tunnel ID

资源隔离VPN中的子隧道ID，十六进制形式，是系统自动分配的。

项目

描述

Tunnel Allocation Method

隧道Token的分配方法。有8种：
  • Global
  • Global with Reserved
  • Per Slot
  • Per Slot with Reserved Token
  • Per Slot with Different Avail Value
  • Mixed
  • Mixed with Global Space 2
  • Global Space 2

Global-3 Avail Tunnel-ID Num

隧道Token全局空间3能分配的token数量。

Per-Board Avail Tunnel-ID Num

隧道Token每板空间能分配的token数量。

Global-2 Avail Tunnel-ID Num

隧道Token全局空间2能分配的token数量。

Global-1 Avail Tunnel-ID Num

隧道Token全局空间1能分配的token数量。

Separation point b/w product and Global3

隧道Token全局空间3与产品空间分界点的Token值。

Separation point b/w Global2 and Per-Board

隧道Token全局空间2与每板空间分界点的Token值。

Separation point b/w Global1 and Global2

隧道Token全局空间1与全局空间2分界点的Token值。

Total Allocated Tunnel-ID Num

表示已分配的Tunnel ID总数。

Globa3 Allocated Tunnel-ID Num

表示全局空间3已分配的Tunnel ID总数。

Per-Board Allocated Tunnel-ID Num

表示每板空间已分配的Tunnel ID总数。

Globa2 Allocated Tunnel-ID Num

表示全局空间2已分配的Tunnel ID总数。

Globa1 Allocated Tunnel-ID Num

表示全局空间1已分配的Tunnel ID总数。

LSP/32bit LSP

分配给LSP类型隧道的Tunnel ID数/32位地址的主机IP路由触发建立的LSP数。

GRE

分配给GRE类型隧道的Tunnel ID数。

CRLSP

分配给CR-LSP类型隧道的Tunnel ID数。

LOCAL IFNET

被VPN内部模块使用的隧道数量。

MPLS LOCAL IFNET

被MPLS内部模块使用的隧道数量。

VPN QOS LSP

分配给应用于VPN QoS中的LSP的Tunnel ID数。

Reserved

分配给产品的TunnelID数。
