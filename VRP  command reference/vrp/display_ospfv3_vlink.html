display ospfv3 vlink

命令功能

display ospfv3 vlink命令用来显示OSPFv3的虚连接信息。

命令格式

display ospfv3 [ process-id ] vlink [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

Transit Area

如果当前接口为虚连接，则显示传输区域ID

Local address

本地地址

Remote address

远端地址

Transmit Delay

接口传输延迟

State

接口状态，可见状态有：Down、Point-to-Point

Timer intervals
相关定时器

  • Hello：Hello间隔

  • Dead：Dead定时器时间间隔

  • Wait：Wait定时器时间间隔，同普通接口，但是对vlink没有作用。

  • Retransmit：重传间隔

Adajcency state

虚连接状态，如Down/Attempt/Init/2 Way/Exch Start/Exchange/Loading/Full
