mld robust-count

命令功能

mld robust-count命令用来设置MLD查询器的健壮系数。

undo mld robust-count命令用来恢复配置的缺省情况。

缺省情况下，MLD查询器的健壮系数是2。

命令格式

mld robust-count robust-value

undo mld robust-count

参数说明

robust-value：指定MLD健壮系数。整数形式，取值范围是2～5，缺省值是2。

缺省级别

2：配置级

使用指南

健壮系数用来规定：

  • MLD查询器启动时发送普遍组查询消息的次数

  • MLD查询器收到离开消息后发送指定组查询消息的次数

运行MLD的主机离开组时，发送离开报文。如果查询器接收到此报文，必须根据mld lastlistener-queryinterval设置的查询间隔Interval（如果没有设置，则取缺省值1秒），按健壮性变量Robust_Count发送MLD最后侦听者查询报文。

如果另一个主机从MLD查询器接收到MLD最后侦听者查询报文，希望加入该组，则在报文规定的最大响应时间内发送MLD成员关系报告报文。如果MLD查询器在（Interva × Robust_Count）的时间内从其他主机接收到MLD成员报告报文，就继续维护组成员关系。如果没有，则认为这个组超时，停止维护组成员。

如果接口视图和MLD视图都设置了健壮性变量，优先取接口视图下的值。

使用实例

# 设置接口GE1/0/0的健壮性变量是3。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld robust-count 3
