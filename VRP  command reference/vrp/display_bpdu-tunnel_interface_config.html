display bpdu-tunnel interface config

命令功能

display bpdu-tunnel interface config命令用来显示接口的BPDU配置信息。

命令格式

display bpdu-tunnel interface config

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

当需要查看以下内容时：

  • 接口发出的BPDU报文是否带有Tag值

  • 是否配置了基于端口的BPDU Tunnel功能

  • 接口是否允许带有Tag值的BPDU报文通过

  • 外层Tag的以太封装类型值

  • 接口发出的BPDU报文所带有Tag值

  • 接口允许通过的BPDU报文所带的Tag值

可以使用display bpdu-tunnel interface config命令查看。

display bpdu-tunnel interface config命令只能在接口视图下使用。

使用实例

# 查看接口GE1/0/0的QinQ的配置信息。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] display bpdu-tunnel interface config
BpduDot1qStatus  disable
BpduOneQStatus   disable
BpduTwoQStatus   enable
EtherType        8100
Dot1qVlan
TwoQList         10
表1 display bpdu tunnel interface config命令输出信息描述

项目

描述

BpduDot1qStatus

接口发出的BPDU报文是否带有Tag值。当配置了stp bpdu vlan vlan-id命令后，此项内容状态变为enable。

BpduOneQStatus

是否配置了基于端口的BPDU Tunnel功能，接口可以透传来自用户网络的BPDU报文。当配置了bpdu-tunnel enable命令后，此项内容状态变为enable。

BpduTwoQStatus

接口是否允许带有Tag值的BPDU报文通过。当配置了bpdu-tunnel stp vlan命令后，此项内容状态变为enable。

EtherType

外层Tag的以太封装类型值，其缺省值是0x8100。

Dot1qVlan

接口发出的BPDU报文所带有Tag值。即使用stp bpdu vlan vlan-id命令中所配置的vlan-id值。

TwoQList

接口允许通过的BPDU报文所带的Tag值。即使用bpdu-tunnel stp vlan vlan-id1 [ to vlan-id2 ] 命令中所配置的vlan-id值。
