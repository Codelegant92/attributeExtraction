fr iphc

命令功能

fr iphc命令用来提供IP头压缩功能，包括RTP/TCP头压缩。

undo fr iphc命令用来禁止该项功能。

缺省情况下，禁止IP头压缩功能。

命令格式

fr iphc { nonstandard | rtp-connections number1 | tcp-connections number2 | tcp-include }

undo fr iphc { nonstandard | rtp-connections | tcp-connections | tcp-include }

参数说明

nonstandard：非标准兼容压缩格式。

rtp-connections number1：指定RTP压缩连接的数目。整数形式，取值范围因产品不同而不同，初始值是256。

tcp-connections number2：指定TCP压缩连接的数目。整数形式，取值范围是3～255，初始值是256。

tcp-include：进行RTP压缩时包含TCP头压缩。

视图

缺省级别

2：配置级

使用指南

配置fr iphc命令之前，必须先执行命令fr compression iphc使能帧中继IP头压缩功能。

当使用undo fr compression iphc命令取消帧中继IP头压缩功能时，fr iphc命令的设置自动被删除。

使用实例

# 在帧中继接口Serial1/0/0上，配置RTP压缩连接的数目为200。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol fr
[Quidway-Serial1/0/0] fr iphc rtp-connections 200
