set memory-usage threshold

命令功能

set memory-usage threshold threshold-value命令用来设置主控板的内存占用率阈值；

set memory-usage threshold threshold-value [ slot slot-id ]命令用来设置接口板内存占用率阈值。

undo set memory-usage threshold命令用来恢复缺省设置。

缺省情况下：

  • 接口板的内存容量在小于128MB时，内存占用率的阈值是80%；

  • 接口板的内存容量在128MB～256MB时，内存占用率的阈值是85%；

  • 接口的内存容量在256MB～512MB时，内存占用率的阈值是90%；

  • 接口的内存容量大于512MB时，内存的占用率的阈值是95%。

命令格式

set memory-usage threshold threshold [ slot slot-id ]

undo set memory-usage threshold

参数说明

threshold threshold-value：内存占用率阈值。整数形式，取值范围是75～100。

slot-id：指定的接口板所在的槽位。

使用指南

当内存占用率接近阈值时，终端界面自动输出提示信息，告知用户内存空间已不充裕。

当内存占用率超过阈值时，写日志。通过查看日志信息，方便用户掌握内存的使用情况。

使用实例

# 配置内存占用率阈值为80%。

<Quidway> system-view
[Quidway] set memory-usage threshold 80
