ignore-mtu-match（MPLS-L2VPN实例视图）

命令功能

ignore-mtu-match命令配置在PE设备上，使PE忽略对L2VPN实例的MTU值的匹配检查。

undo ignore-mtu-match命令用来使能PE设备对L2VPN实例下的MTU值的匹配检查。

命令格式

ignore-mtu-match

undo ignore-mtu-match

参数说明

无

视图

MPLS-L2VPN实例视图

缺省级别

2：配置级

使用指南

使用该命令前，请先确认没有为该L2VPN实例配置vpn-target。

缺省情况下，PE设备是对L2VPN实例下的MTU值进行匹配检查的。如果两端PE设备的L2VPN实例下的MTU值不同，VC链路无法UP。

当需要和其他厂商的设备进行Kompella方式互通时，如果其他厂商的设备不支持L2VPN实例的MTU值匹配的检查，使用ignore-mtu-match命令来忽略匹配检查。

使用实例

# 设备上存在名为vpn1的L2VPN实例，配置忽略对该实例下的MTU值的匹配检查。

<Quidway> system-view
[Quidway] mpls l2vpn vpn1
[Quidway-mpls-l2vpn-vpn1] ignore-mtu-match
