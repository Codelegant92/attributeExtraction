peer next-hop-local（MBGP）

命令功能

peer next-hop-local命令用来设置MBGP向IBGP对等体/对等体组通告路由时，把下一跳属性设为自身的IP地址。

undo peer next-hop-local命令用来恢复缺省配置。

缺省情况下，MBGP在向EBGP对等体通告路由时，将下一跳属性设为自身的IP地址。MBGP在向IBGP对等体通告路由时，不改变下一跳属性。

命令格式

peer { group-name | peer-address } next-hop-local

undo peer { group-name | peer-address } next-hop-local

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

视图

缺省级别

2：配置级

使用指南

该命令应用于IBGP对等体间。

使用实例

# MBGP向对IBGP等体组test发布路由时，将自身地址作为下一跳。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] IPv4-family multicast
[Quidway-bgp-af-multicast] peer test enable
[Quidway-bgp-af-multicast] peer test next-hop-local
