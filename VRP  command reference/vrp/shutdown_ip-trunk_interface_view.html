shutdown（IP-Trunk接口视图）

命令功能

shutdown命令用来关闭当前IP-Trunk接口。

undo shutdown命令用来启用当前IP-Trunk接口。

命令格式

shutdown

undo shutdown

参数说明

无

视图

IP-Trunk接口视图

缺省级别

2：配置级

使用指南

当IP-Trunk接口没有处于Administratively DOWN状态时，可以在成员端口执行命令shutdown。

当在IP-Trunk接口执行命令shutdown后，所有成员端口被自动关闭，此时在成员端口执行undo shutdown命令是无效的。

当IP-Trunk接口处于Administratively DOWN状态时，

  • 如果成员端口加入该IP-Trunk接口，则成员端口自动变为Administratively DOWN状态；

  • 如果成员端口退出该IP-Trunk接口，则成员端口的Administratively DOWN状态不变。

使用实例

# 关闭接口IP-Trunk1。

<Quidway> system-view
[Quidway] interface ip-trunk 1
[Quidway-Ip-Trunk1] shutdown
