nqa-server tcpconnect

命令功能

nqa-server tcpconnect命令用来配置NQA测试的TCP服务器。

undo nqa-server tcpconnect命令用来删除NQA测试的TCP服务器。

命令格式

nqa-server tcpconnect [ vpn-instance vpn-instance-name ] ip-address port-number

undo nqa-server tcpconnect { all | [ vpn-instance vpn-instance-name ] ip-address port-number }

参数说明

vpn-instance vpn-instance-name：设置TCP服务器所属的VPN实例。VPN实例名为字符串形式，长度范围是1～31。

all : NQA服务器端的所有TCP服务连接。

ip-address：NQA服务器端TCP服务监听的地址。点分十进制格式。

port-number：NQA服务器端TCP服务监听的端口号。取值范围是1～50000。
