nqa-server udpecho

命令功能

nqa-server udpecho命令用来配置NQA测试的UDP服务器。

undo nqa-server udpecho命令用来删除NQA测试的UDP服务器。

命令格式

nqa-server udpecho [ vpn-instance vpn-instance-name ] ip-address port-number

undo nqa-server udpecho { [ vpn-instance vpn-instance-name ] ip-address port-number | all }

参数说明

vpn-instance vpn-instance-name：设置UDP服务器所属的VPN实例。VPN实例名为字符串形式，长度范围是1～31。

all：所有NQA测试的UDP服务器。

ip-address：NQA服务器端UDP服务监听的地址。点分十进制格式。

port-number：NQA服务器UDP服务监听的端口号。取值范围是1～50000。
