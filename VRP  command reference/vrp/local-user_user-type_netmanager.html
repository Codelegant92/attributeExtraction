local-user user-type netmanager

命令功能

local-user user-name user-type netmanager命令用来增加网管类型用户。

undo local-user user-name user-type netmanager命令用来删除网管类型用户。

命令格式

local-user user-name user-type netmanager

undo local-user user-name user-type netmanager

参数说明

无

视图

AAA视图

缺省级别

2：配置级

使用指南

在AAA视图下，增加的网管类型用户，需要是已经存在的本地用户。

网管类型用户只有通过AAA认证后才能登录到设备。

网管类型用户不支持RSA认证通过登录设备。

当VTY用户数量没有达到最大的登录数量时，网管用户可以占用当前登录用户没有占用的VTY进行登录。

系统为网管用户保留了5个VTY，供网管用户登录。

当前VTY登录用户达到最大数后，对新登录的用户进行判断，如果是非网管用户，则不允许登录；如果是网管用户则进行AAA认证通过认证后才能登录。

使用实例

# 增加网管类型用户。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] local-user mmi password simple 123456
[Quidway-aaa] local-user mmi user-type netmanager
