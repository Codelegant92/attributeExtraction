display x25 pad

命令功能

display x25 pad命令用来显示X.25 PAD连接信息。

命令格式

display x25 pad [ pad-id ]

参数说明

pad-id：PAD标识号。整数形式，取值范围是0～255，如果不指定则显示所有PAD连接信息

视图

所有视图

缺省级别

1：监控级

使用指南

PAD是一种类似于telnet的应用，可以从一端通过X121地址建立到另一端的PAD连接进行配置等操作。

使用实例
