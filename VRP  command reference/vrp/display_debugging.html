display debugging

命令功能

display debugging命令用来显示已经打开的调试开关。

缺省情况下，不带任何参数显示所有打开的调试开关。

命令格式

display debugging [ interface interface-type interface-number ] [ module-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type：接口类型。

interface-number：接口编号。

module-name：模块名。比如RIP模块等

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
