cops-server group

命令功能

cops-server group命令用于创建COPS服务器组，并进入到该COPS服务器组视图。

undo cops-server group命令用于删除COPS服务器组。

缺省情况下，未配置COPS服务器组。

命令格式

cops-server group group-name [ client-type ssg ]

undo cops-server group group-name

参数说明

group-name：COPS服务器组名。字符串形式，长度范围是为1～32。

client-type：指定COPS服务器组的客户端类型。

ssg：指定COPS服务器组的客户端类型为SSG（Service Select Gateway）。

使用指南

说明：

在首次创建COPS服务器组时，必须使用参数client-type指定COPS服务器组的客户端类型，即指定客户端是为开展何种业务而连接COPS服务器的。这里需要配置COPS服务器组的客户端类型为SSG。

要使COPS客户端能够向相应的COPS服务器发送用户上下线信息，需要执行命令cops-group iptn-binding，将COPS服务器组绑定到IPTN业务上。

使用实例

# 添加名为huawei的COPS服务器组，客户端类型为SSG。

<Quidway> system-view
[Quidway] cops-server group huawei client-type ssg
[Quidway-cops-huawei]
