display acl

命令功能

命令display acl用来查看配置的访问控制列表的规则。

命令格式

display acl { acl-number | all } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：所有的ACL。

acl-number：指定访问控制列表的编号。整数形式，取值范围是1000～3999。其中1000～1999是基于接口的访问控制列表，2000～2999是基本访问控制列表，3000～3999是高级访问控制列表。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用该命令可以查看ACL的配置信息，包括规则数目、ACL步长等信息。

使用实例

# 显示ACL 2001的规则的内容。

<Quidway> display acl 2001
Basic acl  2001, 2 rules,
Acl's step is 5  
 rule 5  permit (0 times matched)
 rule 10 permit source 1.1.1.1 0 (0 times matched)
