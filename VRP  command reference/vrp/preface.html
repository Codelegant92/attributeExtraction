前 言

概述

本文档介绍了VRP中各特性的配置命令，包括每条命令的格式、视图、缺省级别、描述、参数、使用指南、相关命令和举例。

本文档与VRP配置指南的各分册配合，帮助读者掌握具体命令的使用。

产品版本

与本文档相对应的产品版本如下所示。

产品名称

产品版本

VRP

VRP5.60

读者对象

本文档（本指南）主要适用于以下工程师：

  • 调测工程师
  • 数据配置工程师
  • 网络监控工程师
  • 系统维护工程师

内容简介

本文档包括13章，内容如下。

章节

内容

基础配置命令

主要介绍配置基础、用户管理、文件系统、FTP、TFTP、XMODEM、Telnet、SSH、升级、补丁、重启的命令。

系统管理命令

主要介绍信息中心、SNMP、RMON和RMON2、NTP、NQA、NetStream、Ping和Tracert的配置命令。

可靠性命令

主要介绍接口备份、VRRP、BFD、GR的配置命令。

接口管理命令

主要介绍接口基础配置、逻辑接口、接口板混插、快速感知、控制接口震荡和传输告警定制与抑制的配置命令。

局域网与城域网接入命令

主要介绍MAC地址表、以太网接口、VLAN、Eth-Trunk、QinQ、PBB-TE、MSTP、BPDU-Tunnel、RRPP和RPR的配置命令。

广域网接入命令

主要介绍串口、E1、CE1、CT1、E3、CE3、T3、CT3、POS、CPOS、PPP和MP、ATM、帧中继、HDLC和IP-Trunk、LAPB和X.25、PPPoE、Modem和DCC的配置命令。

IP业务命令

主要介绍IP地址、ARP、DNS、DHCP、IP性能、IP单播策略路由、ACL、IPv6基础、IPv6应用、ACL6、NAT-PT、IPv6 over IPv4隧道和IPv4 over IPv6隧道的配置命令。

IP路由命令

主要介绍IP路由表的显示命令及静态路由、RIP、RIPng、OSPF、OSPFv3、IS-IS、BGP、BGP4+、路由策略的配置命令。

IP组播命令

主要介绍IGMP、二层组播、PIM、MSDP、组播BGP、组播VPN、组播路由管理的配置命令。

MPLS命令

主要介绍MPLS、MPLS TE、MPLS OAM的配置命令。

VPN命令

主要介绍VPN隧道管理、L2TP、GRE、BGP/MPLS IP VPN、BGP/MPLS IPv6 VPN、VLL、PWE3、VPLS和VPLS汇聚的配置命令。

QoS命令

主要介绍流量监管和流量整形、拥塞管理、WRED、基于类的QoS、链路效率机制、帧中继QoS、MPLS QoS、VPN QoS的配置命令。

安全命令

主要介绍AAA及用户管理、NAT及NAT用户日志、DHCP Snooping、ARP安全、防火墙、URPF、IPSec的配置命令。

多框集群命令

主要介绍多框集群的文件系统和升级等相关命令。

命令行格式约定

格式

意义

粗体

命令行关键字（命令中保持不变、必须照输的部分）采用加粗字体表示。

斜体

命令行参数（命令中必须由实际值进行替代的部分）采用斜体表示。

[ ]

表示用“[ ]”括起来的部分在命令配置时是可选的。

{ x | y | ... }

表示从两个或多个选项中选取一个。

[ x | y | ... ]

表示从两个或多个选项中选取一个或者不选。

{ x | y | ... }*

表示从两个或多个选项中选取多个，最少选取一个，最多选取所有选项。

[ x | y | ... ]*

表示从两个或多个选项中选取多个或者不选。

&<1-n>

表示符号&的参数可以重复1～n次。

#

由“#”开始的行表示为注释行。

图形界面元素引用约定

格式

意义

“”

带双引号“”的格式表示各类界面控件名称和数据表，如单击“确定”。

多级菜单用“>”隔开。如选择“文件 > 新建 > 文件夹”，表示选择“文件”菜单下的“新建”子菜单下的“ 文件夹”菜单项。

键盘操作约定

格式

意义

加“”的字符

表示键名。如“Enter”、“Tab”、“Backspace”、“a”等分别表示回车、制表、退格、小写字母a。

“键1+键2”

表示在键盘上同时按下几个键。如“Ctrl+Alt+A”表示同时按下“Ctrl”、“Alt”、“A”这三个键。

“键1，键2”

表示先按第一键，释放，再按第二键。如“Alt，F”表示先按“Alt”键，释放后再按“F”键。
