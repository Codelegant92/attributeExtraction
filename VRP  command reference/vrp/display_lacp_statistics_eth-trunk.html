display lacp statistics eth-trunk

命令功能

display lacp statistics eth-trunk命令用来查看静态LACP模式下的LACP报文收发统计信息。

命令格式

display lacp statistics eth-trunk [ trunk-id [interface interface-type interface-number ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

trunk-id：接口聚合组ID，取值范围是0～63。

interface interface-type interface-number：指定查看的接口类型及接口号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用此命令需要聚合组的工作模式为静态LACP模式。要更改工作模式为静态LACP模式，请在Eth-Trunk接口视图下执行mode lacp-static命令。

如果查看特定聚合组的统计信息，需要此聚合组已经存在。如果需要查看特定聚合组的特定接口统计信息，需要此接口已经加入到了此聚合组。

本命令的显示信息如下：

  • 收发LACPDU报文的数量

  • 收发Marker报文的数量

Marker协议用于业务流量从一条链路切换到另一条链路时，保证数据帧在接收端不会出现错序。

# 查看静态Eth-Trunk 4成员接口收发报文的统计信息。

<Quidway> display lacp statistics eth-trunk 4
