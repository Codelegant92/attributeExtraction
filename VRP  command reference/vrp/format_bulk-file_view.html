format (bulk-file视图)

命令功能

format format-type命令用来配置统计文件的文件格式。

undo format format-type命令用来将统计文件的文件格式恢复到缺省的存储方式。

缺省的统计文件的文件格式是bulkascii。

命令格式

format bulkascii

undo format

参数说明

无

视图

bulk-file视图

缺省级别

2：配置级

使用指南

现阶段只支持bulkascii的存储方式。 此命令接口可以后续进行扩展，以支持多种文件格式。

使用实例

# 配置批量统计文件“iftable”的文件格式为bulkascii。

<Quidway> system-view
[Quidway] bulk-stat enable
Info: Succeeded in enabling the bulk stat function.
[Quidway] bulk-file iftable
[Quidway-bulk-file-iftable] format bulkascii
