display igmp-snooping querier

命令功能

display igmp-snooping querier命令用来显示IGMP Snooping查询器信息。

命令格式

display igmp-snooping querier { vlan [ vlan-id ] | vsi [ vsi-name ] }

参数说明

vlan vlan-id：VLAN编号，取值范围是1～4094。

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

视图

所有视图

缺省级别

1：监控级

使用指南

display igmp-snooping querier vlan不指定参数vlan-id时，显示所有VLAN的IGMP Querier配置信息。

display igmp-snooping querier vsi不指定参数vsi-name时，显示所有VSI的IGMP Querier配置信息。

使用实例

# 显示所有VLAN的查询器配置状况

<Quidway> display igmp-snooping querier vlan
VLAN                            Querier-state
-----------------------------------------------
1                                Enable
2                                Disable
3                                Disable
-----------------------------------------------
 total entry 3
表1 display igmp-snooping querier vlan输出信息描述

项目

描述

VLAN

VLAN编号。

Querier-state
查询器状态：

  • Disable：查询器未启动。

  • Enable：查询器已启动。

total entry 3

总共显示查询器的三条信息。
