peer listen-only

命令功能

peer listen-only命令用来使能对等体（组）仅监听连接请求，而不主动发送连接请求功能。

undo peer listen-only命令用来取消该功能。

缺省情况下，对等体（组）发送并监听连接请求。

命令格式

peer { group-name | ipv4-address | ipv6-address } listen-only

undo peer { group-name | ipv4-address | ipv6-address } listen-only

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP视图和BGP-VPN6实例视图下生效。

使用实例

# 配置对等体1.1.1.1只监听对方连接请求。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.1 listen-only
