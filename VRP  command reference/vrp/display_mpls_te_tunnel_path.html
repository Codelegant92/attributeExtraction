display mpls te tunnel path

命令功能

display mpls te tunnel path命令用来查看隧道的路径信息。

命令格式

display mpls te tunnel path [ [ tunnel-name ] [ lsp-id ingress-lsr-id session-id local-lsp-id ] | fast-reroute { local-protection-available | local-protection-inuse } ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

tunnel-name：隧道的名称，字符串形式，不支持空格，区分大小写，长度范围是1～255。隧道名称的形式必须与配置文件中的名称形式完全一致，包括大小写和空格。例如，如果配置文件中的Tunnel接口信息为“interface Tunnel1/0/0”，此处指定隧道名称时，只能用“Tunnel1/0/0”，不能更改大小写或增加空格，“tunnel1/0/0”或“Tunnel 1/0/0”都是错误的写法。

ingress-lsr-id：入节点的LSR-ID，点分十进制格式。

session-id：会话标识。整数形式，取值范围是0～65535。

local-lsp-id：LSP标识号。整数形式，取值范围是0～65535。

fast-reroute local-protection-available：快速重路由的本地保护功能可用。

fast-reroute local-protection-inuse：快速重路由的本地保护功能正在使用。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果不指定隧道接口的编号，将显示所有隧道的路径属性信息。

说明：

只有用户配置了mpls te record-route命令，系统才能记录并显示路径信息。
