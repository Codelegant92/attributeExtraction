ntdp enable

命令功能

ntdp enable命令用来使能系统或指定接口的NTDP功能。

undo ntdp enable用来禁止系统或接口的NTDP功能。

缺省情况下，系统NTDP处于禁止状态，接口NTDP处于禁止状态。

命令格式

ntdp enable

undo ntdp enable

参数说明

无

视图

系统视图、Ethernet接口视图、Eth-Trunk接口视图、VE接口视图和GE接口视图

缺省级别

2：配置级

使用指南

在系统视图和接口视图下都使能NTDP功能，当前接口的NTDP使能才生效。

在系统视图下禁止NTDP功能，会关闭所有接口的NTDP功能。

使用实例

# 使能系统NTDP功能。

<Quidway> system-view
[Quidway] ntdp enable

# 使能GE1/0/1接口的NTDP功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] ntdp enable
