apply access-vpn

命令功能

apply access-vpn命令用来设置报文转发的VPN实例。

undo apply access-vpn命令用来取消报文转发的VPN实例设置。

本命令对已经匹配的IP报文设置VPN实例，最多可以指定6个VPN实例。

命令格式

apply access-vpn vpn-instance vpn-instance-name &<1-6>

undo apply access-vpn vpn-instance vpn-instance-name &<1-6>

参数说明

vpan-instance vpn-instance-name：VPN实例名称。字符串类型，长度是1～31。在同一条命令中最多可以有6个VPN实例名称。

使用实例

# 设置报文转发所应用的VPN实例为vpn1和vpn2。

<Quidway> system-view
[Quidway] policy-based-route policy1 permit node 10
[Quidway-policy-based-route-policy1-10] apply access-vpn vpn-instance vpn1 vpn2
说明：

在应用VPN实例之前，必须先创建VPN实例。
