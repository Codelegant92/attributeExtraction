timer other-querier-present（MLD视图）

命令功能

timer other-querier-present命令用来全局设置“其他MLD查询器的存活时间”。

undo timer other-querier-present命令用来恢复其他MLD查询器的存活时间为缺省值。

缺省情况下，其他MLD查询器的存活时间的计算公式是：其他MLD查询器的存活时间 ＝ 健壮系数 × MLD普遍查询消息发送间隔 +（1/2）× MLD查询消息的最大响应时间。当健壮系数、MLD查询消息发送间隔和MLD查询消息最大响应时间都取缺省值时，其他MLD查询器的存活时间的值为255s。

命令格式

timer other-querier-present interval

undo timer other-querier-present

参数说明

interval：指定“其他MLD查询器的存活时间”。整数形式，取值范围是60～300，单位是s。

视图

MLD视图

缺省级别

2：配置级

使用指南

此命令与接口视图下的mld timer other-querier-present命令作用相同，只是视图不同。缺省情况下，如果接口上没有配置，取MLD视图下配置的值；否则取接口视图下配置的值。

使用实例

# 设置查询器的超时间隔是200s。

<Quidway> system-view
[Quidway] mld
[Quidway-mld] timer other-querier-present 200
