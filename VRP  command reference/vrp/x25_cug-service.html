x25 cug-service

命令功能

x25 cug-service命令用来配置本地CUG设施到网络CUG映射，对于带有CUG设施的呼叫满足CUG抑制条件时进行处理。

undo x25 cug-service命令用来删除CUG抑制。

缺省情况下，不定义任何CUG抑制。

命令格式

x25 cug-service [ incoming-access | outgoing-access | suppress { all | preferential } ] *

undo x25 cug-service

参数说明

incoming-access：对从DCE到DTE的呼入进行抑制处理。

outgoing-access：对从DTE到DCE的呼出进行抑制处理。

all：抑制所有CUG，包括配置了抑制优先的。

preferential：只对配置了抑制优先的进行处理。

使用指南

该命令使用在DCE侧，可以使用命令link-protocol x25设定该接口的工作方式为DCE。

使用实例

# 在Serial1/0/0上定义对呼入进行抑制处理。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25 dce
[Quidway-Serial1/0/0] x25 cug-service incoming-access
