display nqa-agent

命令功能

display nqa-agent命令用来查看NQA测试的客户端状态。

命令格式

display nqa-agent [ | count ] [ | { begin | include | exclude } regular-expression ] [ verbose ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

verbose：显示NQA测试客户端状态的详细信息。

使用实例

# 查看NQA测试的客户端状态。

<Quidway> display nqa-agent
  NQA Tests Max:2000          NQA Tests  Number:   1
  NQA Flow  Max:1000          NQA Flow Remained:1000
nqa test-instance icmp admin
 test-type icmp
 destination-address ipv4 100.1.1.201
 nqa status : normal

项目

描述

NQA Tests Max

可以支持的NQA测试例的最大数目。

NQA Tests Number

当前测试例的数目。

NQA Flow Max

NQA测试例支持的最大并发数。

NQA Flow Remained

仍然可以启动的测试例最大并发数。

test-type

测试类型。

destination-address

目的地址。

nqa status
NQA测试组状态，包括三种：

  • group leader, group members number：表示该测试例已经升级为测试组，以及该测试组目前包括的成员测试例数目。

  • group member, belong to group：表示该测试例目前已经加入一个测试组，以及所加入测试组的名称。

  • normal：表示该测试例没有升级为测试组，也不是任何测试组的成员测试例。
