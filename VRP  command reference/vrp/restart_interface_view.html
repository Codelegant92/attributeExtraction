restart

命令功能

restart命令用来重启当前接口。

命令格式

restart

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

执行restart命令的效果相当于连续执行shutdown（接口视图）和undo shutdown命令。

在某些逻辑接口（如Logic-Channel/Virtual-Template/Loopback/NULL/Dialer等接口）的接口视图下不支持重启命令。

说明：

当接口下配有子接口时，在主接口连续执行restart操作之间的时间间隔应至少为15秒。

使用实例

# 重启接口POS1/0/0。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] restart
