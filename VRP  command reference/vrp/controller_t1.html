controller t1

命令功能

controller t1命令用来进入指定CT1接口的接口视图。

命令格式

controller t1 controller-number

参数说明

controller-number：CT1接口编号，包括槽号、接口卡号、端口号三部分。

视图

系统视图

缺省级别

2：配置级

使用指南

当需要对CT1接口进行配置时，可使用该命令进入CT1接口视图。

使用实例

# 进入CT1接口1/0/0的接口视图。

<Quidway> system-view
[Quidway] controller t1 1/0/0
[Quidway-T1 1/0/0]
