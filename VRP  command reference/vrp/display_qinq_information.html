display qinq information

命令功能

display qinq information命令用来查看配置了QinQ终结或者Stacking的所有接口的名称以及这些接口下QinQ和VLAN-Group的数量。

命令格式

display qinq information { termination | stacking } [ interface interface-type interface-number [.subinterface-number ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：指定接口的类型和编号。

subinterface-number：指定子接口的编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

配置完QinQ终结和Stacking后，可以使用display qinq information命令来查看哪些接口配置了QinQ终结和Stacking，以及QinQ终结和Stacking的相关配置。

使用实例

# 查询配置了QinQ终结的所有子接口。

<Quidway> display qinq information termination
  GigabitEthernet1/0/0.1
    Total QinQ Num: 1
      qinq termination pe-vid 100 ce-vid 10 to 20
    Total vlan-group Num: 0
    control-vid 1 qinq-termination
  GigabitEthernet1/0/0.2
    Total QinQ Num: 1
      qinq termination pe-vid 200 ce-vid 10 to 20
    Total vlan-group Num: 0
    control-vid 2 qinq-termination dynamic

# 查询配置了QinQ stacking的所有子接口。

<Quidway> display qinq information stacking
GigabitEthernet1/0/0.1
    pwe3/vll/vpls bound
    qinq 8021p-mode trust pe-vid-8021p
    Total QinQ Num: 2
      qinq stacking vid 1 to 200 pe-vid 100 vlan-group 1
      qinq stacking vid 300 vlan-group 1
    Total vlan-group: 1
      vlan-group 1
表1 display qinq information命令输出信息描述

项目

描述

GigabitEthernet1/0/0.1

终结子接口的名称。

pwe3/vll/vpls bound

终结子接口绑定的业务类型，可以是L3VPN bound、VLL/PWE3 bound或VPLS bound。如果没有绑定任何业务，则不显示此项内容。

Total QinQ Num

终结子接口对用户报文终结的规则数量。

Total vlan-group

终结子接口下配置的用户VLAN组的数量。
说明：

除表中所列内容外，其他显示信息均为终结子接口的配置内容，与具体配置有关。
