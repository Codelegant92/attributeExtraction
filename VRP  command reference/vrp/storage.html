storage

命令功能

storage ephemeral命令用来配置统计文件存储方式。

undo storage ephemeral命令用来将统计文件存储方式.恢复到缺省的存储方式。

缺省情况下，统计文件的存储方式是ephemeral。

命令格式

storage ephemeral

undo storage

参数说明

无

视图

bulk-file视图

缺省级别

2：配置级

使用指南

现阶段只支持ephemeral的存储方式，缺省就是ephemeral。 此命令接口可以后续进行扩展，以支持多种存储方式。

使用实例

# 配置批量统计文件“iftable”的存储方式为ephemeral。

<Quidway> system-view
[Quidway] bulk-stat enable
Info：Succeeded in enabling the bulk stat function.
[Quidway] bulk-file iftable
[Quidway-bulk-file-iftable] storage ephemeral
