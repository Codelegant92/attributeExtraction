display transmission-alarm configuration

命令功能

display transmission-alarm configuration命令用来查看指定接口的告警定制与抑制配置。如果没有指定接口，则查看所有接口的告警配置。

命令格式

display transmission-alarm configuration [ interface-type interface-number ]

参数说明

interface-type interface-number：接口名和接口编号。

使用指南

执行display transmission-alarm configuration命令时若不指定接口，则查看所有接口的告警配置信息。

使用实例

# 查看POS1/2/0接口的告警配置信息。

<Quidway> display transmission-alarm configuration pos 1/2/0
 Interface: Pos1/2/0
  Filter function: enabled (Holdtime is 55)
  Damping function: enabled
  Suppress value: 999
  Ceiling value: 6000
  Reuse value: 500
  OK half decay value: 500
  NG half decay value: 1000
表1 display transmission-alarm configuration命令显示说明

项目

描述

Interface:

接口名称和接口编号。

Filter function:
是否使能过滤功能。

  • enable：使能了过滤功能。此时会通过holdtime表示过滤时间间隔。

  • disable：没有使能过滤功能。

Damping function:
是否使能抑制功能。

  • enable：使能了抑制功能。

  • disable：没有使能抑制功能。

Suppress value:

抑制门限。

Ceiling value:

最高门限。

Reuse value:

重用门限。

OK half decay value:

告警消除时的半衰期。

NG half decay value:

告警产生时的半衰期。
