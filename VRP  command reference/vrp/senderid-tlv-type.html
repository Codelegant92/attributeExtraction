senderid-tlv-type

命令功能

senderid-tlv-type命令用来配置Sender ID TLV类型在CFM报文中。

缺省情况下，Sender ID TLV类型为defer。

命令格式

senderid-tlv-type { none | chassis | manage | chassis-manage | defer }

参数说明

none：表示发送的CFM报文中不含Sender ID TLV类型。

chassis：表示发送的CFM报文中含有Chassis ID相关信息。

manage：表示发送的CFM报文中含有管理地址相关信息。

chassis-manage：表示发送的CFM报文中含有Chassis ID和管理地址相关信息。

defer：表示Sender ID TLV的内容由MD管理对象决定。

2：配置级

使用指南

现如今，802.1ag协议分为draft7草案版本和standard2007标准版本，本命令只支持standard2007标准版本使用。

此命令用于在MD视图和默认级别MD视图下配置Sender ID TLV类型，此类型将用于填写CFM报文。

使用实例

# 配置Sender ID TLV的类型为manage。

<Quidway> system-view
[Quidway] cfm enable
[Quidway] cfm default md
[Quidway-default-md] senderid-tlv-type manage
