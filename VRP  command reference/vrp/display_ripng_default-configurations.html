项目

描述

Protocol Level Default Configurations

RIPng协议级的缺省信息

Preference

RIPng进程优先级

Checkzero

零域检查

Default-cost

缺省路由开销值

Maximum number of balanced paths

等价路由的最大数目

Update time

RIPng更新间隔时间

Suppress time

路由抑制时间

Interface Level Default Configurations

RIPng接口级的缺省信息

Metricin

接口接收RIPng报文时给路由增加的度量值

Metricout

接口发送RIPng报文时给路由增加的度量值

Poison Reverse

接口是否使能了毒性反转特性
Split Horizon

  • For Broadcast and P2P Interfaces

  • For NBMA Interfaces
接口是否使能了水平分割特性

  • 广播接口和P2P接口是否使能了水平分割特性

  • NBMA接口是否使能了水平分割特性

Default-route

缺省路由

Packet Transmit Interval

报文转发间隔（毫秒）

Packet Transmit Number

报文转发数目
