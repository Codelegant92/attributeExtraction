ip netstream multicast

命令功能

ip netstream multicast命令用来使能对组播报文的NetStream入或出统计功能。

undo ip netstream multicast命令用来禁止对组播报文的NetStream入或出统计功能。

只有在接口使能了NetStream功能，配置才会生效。

缺省情况下，禁止组播报文的NetStream入或出统计功能。

命令格式

ip netstream multicast { inbound | outbound }

undo ip netstream multicast { inbound | outbound }
