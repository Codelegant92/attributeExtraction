reset lspv statistics

命令功能

reset lspv statistics命令用来清除PWE3 tracert的统计信息。

命令格式

reset lspv statistics

参数说明

无

视图

用户视图

缺省级别

2：配置级

使用指南

reset lspv statistics命令用来清除PWE3 tracert的统计信息。包括发送的MPLS echo request和MPLS echo reply报文总数，接收到的MPLS echo request和MPLS echo reply报文总数。

使用reset lspv statistics前请先确认好是否需要清除PWE3 tracert的统计信息，删除后的PWE3 tracert的统计信息不能被恢复。

使用实例

# 清除设备上PWE3 tracert的统计信息。

<Quidway> reset lspv statistics
