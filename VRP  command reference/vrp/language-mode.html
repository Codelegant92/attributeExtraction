language-mode

命令功能

language-mode命令用来切换命令行接口的语言模式。

缺省情况下，为英文模式。

命令格式

language-mode { chinese | english }

参数说明

chinese：命令行接口采用中文语言模式。

english：命令行接口采用英文语言模式。

视图

用户视图

缺省级别

0：参观级

使用指南

切换为中文模式后，系统界面的命令行提示信息和回显信息将以中文形式显示。

说明：
目前只支持中文和英文两种语言模式。

使用实例

# 将英文模式切换为中文模式。

<Quidway> language-mode chinese
Change language mode, confirm? [Y/N] y
% 改变到中文模式。
