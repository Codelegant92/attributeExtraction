display ipv6 vpn6-instance

命令功能

display ipv6 vpn6-instance命令用来查看VPN实例信息。

命令格式

display ipv6 vpn6-instance [ brief | verbose ] [ vpn6-instance-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

brief：显示IPv6 VPN实例的简要信息。

verbose：显示IPv6 VPN实例和关联接口的详细信息。

vpn6-instance-name：IPv6 VPN实例的名称，字符串形式，区分大小写，长度范围是1～31。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

完成IPv6 VPN实例的配置时，可用display ipv6 vpn6-instance命令检查IPv6 VPN实例的配置情况。还可以用此命令查看本设备上配置了哪些IPv6 VPN实例。

如果不指定可选参数，此命令将显示本设备所有IPv6 VPN实例的简要信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看本路由器上所有IPv6 VPN实例的简要信息。

<Quidway> display ipv6 vpn6-instance
  Total VPN6-Instances configured : 2
  VPN6-Instance Name               RD                   Creation Time
  vpn6                            100:1                2006/08/24 10:04:48
  vpn1                            100:2                2006/08/24 11:42:46
表1 display ipv6 vpn6-instance命令输出信息描述

项目

描述

Total VPN-Instances configured

表示本端配置的IPv6 VPN实例总数。

VPN6-Instance Name

表示VPN实例的名称。

RD

表示该VPN实例的路由标志符，具体请参见命令route-distinguisher(IPv6 VPN实例视图)。

Creation Time

显示创建该IPv6 VPN实例的具体时间。

 
<Quidway> display ipv6 vpn6-instance brief
  Total VPN-Instances configured : 1
  VPN-Instance Name               RD                Creation Time
  vpna                            200:1             2006/11/20 14:41:28

# 查看所有VPN实例的详细信息。

<Quidway> display ipv6 vpn6-instance verbose

  Create date : 2006/11/20 14:41:28
  Up time : 0 days, 00 hours, 11 minutes and 00 seconds
  Route Distinguisher : 1:1
  Export VPN Targets :  1:1
  Import VPN Targets :  1:1
  Label policy : label per route
  Import Route Policy : p1
  Export Route Policy : p2
  Tunnel Policy : tnlpolicy1
  Description : This is a VPN for company1.
  Maximum Routes Limit : 100
  Interfaces : GigabitEthernet1/0/0
表2 display ipv6 vpn6-instance verbose命令输出信息描述

项目

描述

Total VPN-Instances configured

表示本端配置的IPv6 VPN实例总数。

VPN-Instance Name and ID

表示IPv6 VPN实例的名称和标识号。标识号是系统分配的，便于查找。

Create date

显示创建该IPv6 VPN实例的具体时间。

Up time

该IPv6 VPN实例处于可用状态的持续时间。

Route Distinguisher

表示该IPv6 VPN实例的路由标志符，具体请参见命令route-distinguisher(IPv6 VPN实例视图)。

Export VPN Targets

表示该IPv6 VPN实例发出的路由扩展团体列表，具体请参见命令vpn-target。

Import VPN Targets

表示该IPv6 VPN实例收到的路由扩展团体列表，具体请参见命令vpn-target。

Label policy
标签策略：

  • label per route表示给每一条路由都分配一个标签

    ；

  • label per instance表示同一个VPN实例的所有路由共用一个标签

    。

Import Route Policy

应用于该IPv6 VPN实例的入方向路由策略。只有配置了import route-policy(IPv6 VPN实例视图)命令才有该显示项。

Export Route Policy

应用于该IPv6 VPN实例的出方向路由策略。只有配置了export route-policy(IPv6 VPN实例视图)命令才有该显示项。

Tunnel Policy

应用于该IPv6 VPN实例的隧道策略。只有配置了tnl-policy命令才有该显示项。

Description

IPv6 VPN实例的描述信息。只有配置了description命令才有该显示项。

Maximum Routes Limit

IPv6 VPN实例支持的最大路由数。只有配置了routing-table limit命令才有该显示项。

Interfaces

表示绑定了该IPv6 VPN实例的接口。只有配置了ipv6 binding vpn6-instance命令才有该显示项。
