enable traffic-adjustment

命令功能

enable traffic-adjustment命令用来使能IGP Shortcut特性；

enable traffic-adjustment advertise命令用来使能转发邻接特性。

undo enable traffic-adjustment命令用来关闭IGP Shortcut特性；

undo enable traffic-adjustment advertise命令用来关闭转发邻接特性。

缺省情况下，不支持IGP Shortcut和转发邻接。

命令格式

enable traffic-adjustment [ advertise ]

undo enable traffic-adjustment [ advertise ]

参数说明

advertise：通告路由，用于转发邻接（Forwarding Adjacency）。

视图

OSPF视图

缺省级别

2：配置级

使用指南

配置后，在进行SPF计算和泛洪（flooding）时，所有TE隧道也被包括在内。

使用实例

# 使能OSPF中的转发邻接。

<Quidway> system-view
[Quidway] ospf 100
[Quidway-ospf-100] enable traffic-adjustment advertise
