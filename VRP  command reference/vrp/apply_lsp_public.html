apply lsp public

命令功能

apply lsp public命令用来配置使用指定的LSP转发满足策略路由过滤条件的公网报文。

undo apply lsp public命令用来取消配置。

缺省情况下，不对公网报文进行策略路由。

命令格式

apply lsp public pe-address [ p-address | interface-type interface-number | secondary ]

undo apply lsp public pe-address

参数说明

pe-address：对端PE设备的IP地址。

p-address：P设备的IP地址，与pe-address配合，指定LSP接下来经过的P设备。

interface-type interface-number：与pe-address配合，指定LSP接下来使用的出接口。

secondary：指定使用LDP FRR生成的备份LSP。如果不配置secondary参数，只选择主LSP转发；如果配置secondary参数，只选择备份LSP转发。

缺省级别

2：配置级

使用指南

配置了策略路由到PE、P的地址后，如果去往PE、P没有可达的LSP，则策略路由不起作用。

使用实例

# 配置一条策略路由，使满足ACL 2000过滤的公网报文使用地址为10.1.1.1的PE的LSP转发。

<Quidway> system-view
[Quidway] policy-based-route map1 permit node 10
[Quidway-policy-based-route-map1-10] if-match acl 2000
[Quidway-policy-based-route-map1-10] apply lsp public 10.1.1.1
