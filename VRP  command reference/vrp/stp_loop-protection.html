stp loop-protection

命令功能

stp loop-protection命令用来启动当前端口的环路保护功能。

undo stp loop-protection命令用来恢复当前端口的环路保护功能的缺省运行状态。

缺省情况下，端口的环路保护功能处于关闭状态。

命令格式

stp loop-protection

undo stp loop-protection

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

在启动了环路保护功能后，如果根端口收不到来自上游的BPDU时，根端口会被设置进入阻塞状态；而阻塞端口则会一直保持在阻塞状态，不转发报文，从而不会在网络中形成环路。

使用实例

# 开启以太网接口GE1/0/0的环路保护功能。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] stp loop-protection
