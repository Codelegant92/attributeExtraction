display mpls static-cr-lsp

命令功能

display mpls static-cr-lsp命令用来查看静态CR-LSP信息。

命令格式

display mpls static-cr-lsp [ lsp-name ] [ { include | exclude } ip-address mask-length ] [ verbose ] [ | count ] [ | { begin regular-expression | include regular-expression | exclude regular-expression } ]

参数说明

lsp-name：LSP名称，字符串形式，区分大小写，不支持空格，长度范围是1～19个字符。

exclude：显示除去指定目的IP地址的CR-LSP信息。

include：显示包含指定目的IP地址的CR-LSP信息。

ip-address：目的IP地址。

mask-length：目的IP地址掩码长度，取值范围是0～32。

verbose：查看详细信息。

|：使用正则表达式对输出信息进行过滤。

[ | count ]

begin regular-expression：从包含匹配字符串的行开始输出。

exclude regular-expression：只输出不包含匹配字符串的行。

include regular-expression：只输出包含匹配字符串的行。

使用实例

# 查看静态CR-LSP的简要信息。

<Quidway> display mpls static-cr-lsp
TOTAL         : 2     STATIC CRLSP(S)
UP            : 2     STATIC CRLSP(S)
DOWN          : 0     STATIC CRLSP(S)
Name            FEC               I/O Label    I/O If            Stat
Tunnel1/0/0     3.3.3.3/32        NULL/20      -/GE1/0/0         Up
tunnel2/0/0     -/-               130/NULL     GE1/0/0/-         Up
表1 display mpls static-cr-lsp命令输出信息描述

项目

描述

TOTAL

静态CR-LSP的总数

UP

处于UP状态的静态CR-LSP数量

DOWN

处于DOWN状态的静态CR-LSP的数量

Name

静态CR-LSP的名称

FEC

目的地址及掩码长度

I/O Label

入标签和出标签

I/O If

入接口和出接口

Stat

此LSP当前的状态

# 查看静态CR LSP的详细信息。

<Quidway> display mpls static-cr-lsp verbose
No             : 1
LSP-Name       : tunnel2/0/0
LSR-Type       : Transit
FEC            : -/-
In-Label       : 20
Out-Label      : 30
In-Interface   : GigabitEthernet1/0/0
Out-Interface  : GigabitEthernet2/0/0
NextHop        : 10.1.3.2
Lsp Status     : Up
表2 display mpls static-cr-lsp verbose命令输出信息描述

项目

描述

No

静态CR-LSP的显示序号

LSP-Name

LSP的名称

LSR-Type
LSR的类型：
  • Ingress
  • Transit
  • Egress

FEC

目的地址及掩码长度，对于Transit节点和Egress节点，FEC字段值显示为“-/-”

In-Label

入标签值

Out-Label

出标签值

In-Interface

入接口

Out-Interface

出接口

NextHop

下一跳地址，格式为X.X.X.X

LSP Status
表示该静态CR-LSP的状态：
  • Up：CR-LSP建立成功
  • Down：CR-LSP建立不成功
