vrrp vrid track efm interface

命令功能

vrrp vrid track efm interface命令用来配置VRRP通过监视EFM会话状态来实现主备快速切换的功能。

缺省情况下，VRRP备份组不监视EFM会话状态。

命令格式

vrrp vrid virtual-router-id track efm interface interface-type interface-number

undo vrrp vrid virtual-router-id track efm interface interface-type interface-number

参数说明

virtual-router-id：指定VRRP的ID，取值范围是1～255，无缺省值。

interface interface-type interface-number：接口编号，包括槽号、卡号、端口号。

缺省级别

2：配置级

使用指南

每个VRRP只能支持监视一个EFM会话。执行该命令后，EFM状态变化将会影响VRRP的状态。

使用实例

# 配置VRRP监视EFM功能：

<Quidway> system-view
[Quidway] interface gigabitethernet 3/0/22
[Quidway-GigabitEthernet3/0/22] vrrp vrid 1 track efm interface GigabitEthernet3/0/22

# 配置VRRP不监视EFM功能：

[Quidway-GigabitEthernet3/0/22] undo vrrp vrid 1 track efm
