x29 timer inviteclear-time

命令功能

x29 timer inviteclear-time命令用来设置当发起PAD清除规程后等待对方应答的时延，超过这个时间后强制退出PAD连接并启动x25清除规程。

undo x29 timer inviteclear-time可使时延值恢复为缺省值5秒。

命令格式

x29 timer inviteclear-time seconds

undo x29 timer inviteclear-time

参数说明

seconds：时延值。取值范围是5～2147483，单位是秒。当发起PAD清除规程后等待对方应答的时延，缺省情况时延值为5。
