display multicast limit out-interface

命令功能

display multicast limit out-interface命令用来查看出接口上所有组播组的表项数量、带宽的限制值和统计值。

命令格式

display multicast [ vpn-instance vpn-instance-name ] limit out-interface { interface-type interface-number | all } [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例名称。字符串形式，区分大小写。

out-interface interface-type interface-number：查看指定出接口上的表项数量、带宽的限制值和统计值。

all：查看所有出接口上的表项数量、带宽的限制值和统计值。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

对于ASM模型的一个组播组G，(*,G)和所有的(S,G)计为一个表项；对于SSM模型的一个组播组G，每个(S,G)计为一个表项。

缺省情况下，每个表项的带宽是2Mbit/s。可以通过channel-name视图下的group group-address mask { group-mask-length | group-mask } per-bandwidth traffic-rate命令配置节目组下每个表项的带宽。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
