igmp version

命令功能

igmp version命令用来在接口上配置运行的IGMP版本。

undo igmp version命令用来恢复缺省配置。

缺省情况下，运行IGMPv2。

命令格式

igmp version { 1 | 2 | 3 }

undo igmp version

参数说明

1：表示IGMPv1。

2：表示IGMPv2。

3：表示IGMPv3。

视图

接口视图

缺省级别

2：配置级

使用指南

子网上的所有路由器接口应支持同一版本的IGMP。

IGMP各版本之间不能自动转换。

使用实例

# 指定在GE1/0/0上使用IGMPv1版本。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] igmp version 1
