rpr fairness weight

命令功能

rpr fairness weight命令用来配置本地节点在某个方向上发送链路的权重。

undo rpr fairness weight命令用来恢复缺省值。

缺省情况下，本地节点发送链路的权重值为1。

命令格式

rpr fairness weight value { west | east }

undo rpr fairness weight { west | east }

参数说明

value：公平算法权重值，取值为2的幂次方：1、2、4、8、16、32、64或128。其中的最大值由具体的产品指定，缺省最大值为128。

west：本地节点的西向连接。

east：本地节点的东向连接。

缺省级别

2：配置级

使用指南

链路的权重会影响公平算法的计算结果，权重值高的链路上帧的流量大。

使用实例

# 配置本地节点的西向发送链路的权重值为16。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr fairness weight 16 west
