display ntdp device-list

命令功能

display ntdp device-list命令用来查看通过NTDP收集到的设备信息。

命令格式

display ntdp device-list [ verbose ]

参数说明

verbose：显示设备的详细信息。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定可选参数，本命令将显示设备的简要信息。

使用实例

# 查看NTDP收集到的设备信息。

<HUAWEI_0.Quidway> display ntdp device-list
The device-list of NTDP:
------------------------------------------------------------------------------
 MAC             HOP  IP                  PLATFORM

------------------------------------------------------------------------------
 00e0-fcb8-d6b6  0    10.0.0.1/24         NE40E
 00e0-0003-0003  1    10.0.0.3/24         NE40E

该显示信息主要内容解释见表1。

表1 display ntdp device-list命令输出信息描述（简要信息）

项目

描述

MAC

设备的MAC地址。

HOP

设备距离拓扑收集设备的跳数。

IP

设备在集群内通信时所使用的IP地址。

PLATFORM

设备类型。

该显示信息主要内容解释见表2。

表2 display ntdp device-list命令输出信息描述（详细信息）

项目

描述

Hostname

设备的主机名。

MAC

设备的MAC地址。

Hop

本设备距离拓扑收集设备的跳数。

Platform

设备型号。

IP

设备的私有IP地址。

Version

设备的系统软件版本。

Cluster

设备所处集群的信息。包括集群名称、在集群中角色，命令交换机的MAC地址。

Peer MAC

邻居节点的MAC地址。

Native Port ID

本设备与邻居节点连接时所使用的接口。

Peer Port ID

邻居节点与本设备连接时所使用的接口。

N-Index

本地接口索引。

P-Index

对端接口索引。

Speed

邻居节点与本设备连接时的接口速率。

Dup

邻居节点与本设备连接时的接口双工方式。
