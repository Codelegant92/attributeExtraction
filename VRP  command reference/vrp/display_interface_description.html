display interface description

命令功能

display interface description命令用来查看设备上所有接口的接口描述信息。

命令格式

display interface description [ interface-type [ interface-number ] | slot slot-id [ card card-number ] ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-type interface-number：接口类型和接口编号。如果不指定接口类型，将显示所有接口的接口描述信息。如果不指定接口编号，将显示所有该类型接口的接口描述信息。

slot slot-id：板号。接口不指定板号，将显示所有接口板和主控板的所有接口的描述信息。

card card-number：卡号。如果不指定卡号，将显示指定板号上的所有卡上的所有接口的描述信息。

|：使用正则表达式过滤输出的显示信息。

begin：显示以包含指定正则表达式的行开始的配置信息。

exclude：显示不包含指定正则表达式的行的配置信息。

include：显示包含指定正则表达式的行的配置信息。

regular-expression：正则表达式。字符串形式，支持空格，区分大小写。

使用指南

为了方便记忆和管理，可以对接口执行description命令，记录接口的用途。如果接口比较多，使用display interface description可以快速查看到这些接口的描述信息。

说明：

终端用户接口视图不能配置接口的描述信息。

使用实例

# 查看路由器上所有接口的描述信息。

<Quidway> display interface description
Interface                   Description
GigabitEthernet1/0/0               Connecting to Beijing
GigabitEthernet1/1/0               HUAWEI, Quidway Series, GigabitEthernet1/1/0 Interface
NULL0                       HUAWEI, Quidway Series, NULL0 Interface
Serial1/0/0                  HUAWEI, Quidway Series, Serial1/0/0 Interface

# 查看1号板上的所有卡上的接口的描述信息。

<Quidway> display interface description slot 1
Interface                   Description
GigabitEthernet1/0/0               Connecting to Beijing
GigabitEthernet1/1/0               HUAWEI, Quidway Series, GigabitEthernet1/1/0 Interface
NULL0                  HUAWEI, Quidway Series, NULL0 Interface
Serial1/0/0                 HUAWEI, Quidway Series, Serial1/0/0 Interface

# 查看1号板上的1号卡上的所有接口的描述信息。

<Quidway> display interface description slot 1 card 1
Interface                  Description
GigabitEthernet1/1/0              HUAWEI, Quidway Series, GigabitEthernet1/1/0 Interface

# 查看路由器上所有以太网接口的描述信息。

<Quidway> display interface description gigabitethernet
Interface                  Description
GigabitEthernet1/0/0              Connecting to Beijing
GigabitEthernet1/1/0              HUAWEI, Quidway Series, GigabitEthernet1/1/0 Interface

# 查看路由器上接口GigabitEthernet1/0/0的描述信息。

<Quidway> display interface description GigabitEthernet 1/0/0
Interface                   Description
GigabitEthernet1/0/0               Connecting to Beijing

# 用正则表达式过滤查看路由器上GigabitEthernet接口的描述信息。

<Quidway> display interface description | include Giga
Interface                   Description
GigabitEthernet1/0/0               Connecting to Beijing
