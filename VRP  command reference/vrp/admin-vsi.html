admin-vsi

命令功能

admin-vsi命令用来把当前业务VSI配置为管理VSI。

undo admin-vsi命令用来把当前管理VSI配置为业务VSI。

缺省情况下，没有配置管理VSI。

命令格式

admin-vsi

undo admin-vsi

参数说明

无

视图

VSI视图

缺省级别

2：配置级

使用指南

admin-vsi命令目前只支持Martini方式的VPLS，使用该命令前，需要先执行pwsignal命令和vsi-id命令。

在城域以太网中，主备NPE之间通过VRRP实例进行主备检测与倒换控制，UPE上建立的管理VSI，专门用于NPE之间传送主备检测报文。

在管理VSI下，不允许执行shutdown命令。

管理VSI的Up条件与普通业务VSI的Up条件有所不同。

管理VSI：只要VSI中有一个PW为Up或者一个AC为Up。

业务VSI：要有两个或两个以上Up的AC接口，或者有一个Up的AC接口和一个Up的PW。

管理VSI收到免费ARP后，相关的业务VSI需要清除MAC地址。管理VSI判定所绑定的业务VSI的个数是否达到阈值：

如达到阈值，则清除所有VSI的MAC，并记录日志。

如未达到阈值，则清除单个VSI的MAC，并记录日志。

使用实例

# 配置V100为管理VSI。

<Quidway> system-view
[Quidway] mpls l2vpn 
[Quidway-l2vpn] quit
[Quidway] vsi v100 static
[Quidway-vsi-v100] pwsignal ldp
[Quidway-vsi-v100-ldp] vsi-id 100
[Quidway-vsi-v100-ldp] quit
[Quidway-vsi-v100] admin-vsi
