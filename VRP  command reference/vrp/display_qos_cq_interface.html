display qos cq interface

命令功能

display qos cq interface命令用来显示接口的定制队列配置情况和统计信息。

命令格式

display qos cq interface [ interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口号。

使用指南

如果不指定接口，将显示所有接口的CQ配置情况和统计信息。
