snmp-agent extend error-code enable

命令功能

snmp-agent extend error-code enable命令用于使能设备扩展错误码开关。

命令格式

snmp-agent extend error-code enable

undo snmp-agent extend error-code enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

MIB对扩展错误码的支持：

  • 对于支持扩展错误码的MIB，可以通过使能扩展错误码并且在使用华为公司的网管情况下，网管端得到更加丰富的错误码；
  • 对于不支持扩展错误码的MIB在使能扩展错误码功能后，无论是华为公司的网管还是其他厂商的网管都只能得到标准的错误码。

使用实例

# 打开扩展错误码的控制开关。

<Quidway> system-view
[Quidway] snmp-agent extend error-code enable
Now setting the error-code status to the device
