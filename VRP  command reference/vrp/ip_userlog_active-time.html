ip userlog active-time

命令功能

ip userlog active-time命令用来为监视长时间活跃的流设置定时器时长。

undo ip userlog active-time命令用来取消此功能。

缺省情况下，不使能此功能。

命令格式

ip userlog nat active-time minutes

undo ip userlog nat active-time

参数说明

nat：NAT日志功能。

minutes：活跃流超时的时长。取值范围：10分钟～120分钟，无缺省值。

使用指南

如果数据流一直是活跃的，每隔一定的时间，就将该流记录一次。

使用实例

# 对于长时间活跃的NAT流，每隔1小时记录一次。

<Quidway> system-view
[Quidway] ip userlog nat active-time 60
