snmp-agent trap enable feature-name l2-multicast

命令功能

snmp-agent trap enable feature-name l2-multicast命令用来打开二层组播告警功能。

undo snmp-agent trap enable l2-multicast命令用来关闭二层组播告警功能。

缺省情况下，二层组播告警功能默认关闭。

命令格式

snmp-agent trap enable feature-name l2-multicast

undo snmp-agent trap enable feature-name l2-multicast

参数说明

无

使用指南

本功能主要用于打开二层组播告警功能，在发生Trap时，向网管报告。

使用实例

# 打开二层组播告警功能。

<Quidway> system-view
[Quidway] snmp-agent trap enable feature-name l2-multicast
