display ipv6 routing-table protocol

命令功能

display ipv6 routing-table protocol命令用来显示指定协议的路由信息。

命令格式

display ipv6 routing-table protocol protocol [ inactive | verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

inactive：如果配置了该参数，display ipv6 routing-table protocol命令将只显示未激活路由信息。不指定该参数，显示所有激活和未激活路由信息。

verbose：显示通过过滤规则的激活和未激活路由的详细信息。不指定该参数，只显示通过过滤规则的激活路由的概要信息。

protocol：显示指定协议的路由信息。该参数可为下列关键字：

  • direct：显示直连IPv6路由信息。

  • static：显示静态IPv6路由信息。

  • bgp：显示BGP IPv6路由信息。

  • ripng：显示RIPng IPv6路由信息。

  • isis：显示ISIS IPv6路由信息。

  • ospfv3：显示OSPF IPv6路由信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例
