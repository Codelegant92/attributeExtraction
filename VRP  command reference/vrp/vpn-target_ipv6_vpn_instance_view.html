vpn-target(IPv6 VPN实例视图)

命令功能

vpn-target命令用来将当前IPv6 VPN实例与一个或多个IPv6 VPN Target进行关联。

undo vpn-target命令用于删除当前IPv6 VPN实例关联的IPv6 VPN target。

IPv6 VPN Target没有缺省值，必须在创建IPv6 VPN实例时配置。

如果用户不指定关键字“both/export/import”，则缺省为“both”。

命令格式

vpn-target vpn-target &<1-8> [ both | export-extcommunity | import-extcommunity ]

undo vpn-target { all | vpn-target &<1-8> [ both | export-extcommunity | import-extcommunity ] }

参数说明

vpn-target：添加IPv6 VPN Target扩展团体属性到IPv6 VPN实例入或出方向IPv6 VPN Target扩展团体列表，或是出入方向IPv6 VPN Target扩展团体。用户可以使用以下两种格式之一来表示IPv6 VPN Target值。

  • 16位自治系统号:32位用户自定义数，例如：1:3。自治系统号的取值范围是0～65535；用户自定义数的取值范围是0～4294967295。其中，自治系统号和用户自定义数不能同时为0，即VPN Target的值不能是0:0。

  • 32位IP地址:16位用户自定义数，例如：192.168.122.15:1。IP地址的取值范围是0.0.0.0～255.255.255.255；用户自定义数的取值范围是0～65535。

both：添加vpn-target扩展团体属性到IPv6 VPN实例入方向和出方向的扩展团体属性列表中。

export-extcommunity：指定出方向到目的IPv6 VPN的路由信息的扩展团体属性值。

import-extcommunity：定义可以接收带有指定扩展团体属性值的路由信息。

使用指南

通过IPv6 VPN Target可以控制节点之间的路由发布。

当PE根据VPN实例向其他PE发送路由时，它将向发送的路由附加Export IPv6 VPN Target。当PE从其他PE接收路由时，它根据本地的Import IPv6 VPN Target和接收的Export IPv6 VPN Target决定这些路由是否可以被添加到相应的IPv6 VPN实例中。

使用实例

# 为VPN实例vrf1的Export IPv6 VPN Target列表添加3:3，Import IPv6 VPN Target列表添加4:4。

<Quidway> system-view
[Quidway] ipv6 vpn6-instance vrf1
[Quidway-vpn6-instance-vrf1] vpn-target 3:3 export-extcommunity
[Quidway-vpn6-instance-vrf1] vpn-target 4:4 import-extcommunity

# 为VPN实例vrf1的Export IPv6 VPN Target列表和Import IPv6 VPN Target列表添加5:5。

<Quidway> system-view
[Quidway] ipv6 vpn6-instance vrf1
[Quidway-vpn6-instance-vrf1] vpn-target 5:5 both
