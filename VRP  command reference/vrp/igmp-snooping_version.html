igmp-snooping version

命令功能

igmp-snooping version命令用来配置IGMP Snooping在当前VLAN/VSI内可以处理的IGMP报文的版本。

undo igmp-snooping version命令用来恢复该配置的缺省值。

缺省情况下，既可以处理IGMPv1的报文，也可以处理IGMPv2的报文。

命令格式

igmp-snooping version number

undo igmp-snooping version

参数说明

version number ：表示IGMP的版本号。取值只能是1或者2。

视图

VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定到VSI；

  • VLAN内包含VE接口。

使用实例

# 配置IGMP Snooping在VLAN 2内只可以处理IGMPv1报文。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] igmp-snooping version 1
