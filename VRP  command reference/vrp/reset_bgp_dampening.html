reset bgp dampening

命令功能

reset bgp dampening命令用来清除路由的衰减信息并释放被抑制的路由。

命令格式

reset bgp [ vpn-instance vpn-instance-name ] dampening [ ipv4-address [ mask | mask-length ] ]

reset bgp { ipv6 | vpn6-instance vpn6-instance-name } dampening [ ipv6-address prefix-length ]

参数说明

vpn-instance vpn-instance-name：清除指定VPN实例的路由衰减信息。

ipv4-address：指定IPv4网络地址。

mask | mask-length：指定网络掩码（点分十进制形式）或掩码长度。如果没有指定掩码，则按有类地址处理。

ipv6：清除IPv6路由的衰减信息并释放被抑制的路由。

ipv6-address：要清除衰减信息的网络IPv6地址。

prefix-length：IPv6前缀的长度，是十进制数字，指定地址中有多少连续的位组成前缀，即地址中的网络地址部分。

vpn6-instance dampening：清除BGP IPv6 VPN对等体连接的BGP路由衰减信息。

使用指南

不指定ipv6-address prefix-length时，执行命令reset bgp { ipv6 | vpn6-instance vpn6-instance-name } dampening将清除整个BGP路由表中的IPv6路由衰减信息。

使用实例

# 清除20.1.0.0网段路由的衰减信息并释放被抑制的路由。

<Quidway> reset bgp dampening 20.1.0.0 255.255.0.0

# 清除到2345::0的路由衰减信息并释放被抑制的路由。

<Quidway> reset bgp ipv6 dampening 2345:: 64

# 清除VPNv6实例1的路由衰减信息。

<Quidway> reset bgp vpn6-instance 1 dampening
