cops-server

命令功能

cops-server命令用于配置COPS服务器，包括IP地址、服务器端口号、客户端端口号、VPN实例、权重等参数。

undo cops-server命令用于删除COPS服务器或者COPS服务器的VPN实例。

缺省情况下，COPS服务器组中未配置COPS服务器。

命令格式

cops-server ip-address [ server-port | client-port client-port | vpn-instance instance-name | weight value ] *

undo cops-server ip-address [ vpn-instance instance-name ]

参数说明

ip-address:：COPS服务器的IP地址。

server-port:：COPS服务器的端口号。整数形式，取值范围是1～49151，缺省值是3288。

client-port server-port:：COPS客户端的端口号。整数形式，取值范围是0～49151，缺省值是7374。0代表任意端口。

vpn-instance instance-name:：指定COPS服务器的VPN实例名。字符串形式，长度范围是1～31。必须是已配置的VPN实例。

weight value:：COPS服务器的权重。取值范围是0～100，缺省值是0。value值越大，权重越大。当一个服务器组中配置多台COPS服务器时，可使用权重来控制服务器之间的负荷分担。

使用指南

只有当COPS服务器组绑定到IPTN业务后，设备才能上报IPTN业务的用户信息给指定的COPS服务器组。

说明：
  • 建议用户不要另行指定COPS服务器的端口号，因为如果指定的端口号已被占用，将影响设备和COPS服务器之间建立TCP连接。
  • 设备与COPS服务器之间应保证至少有一条可达路由存在。

使用实例

# 为名字为huawei的COPS服务器组添加一个COPS服务器。

<Quidway> system-view
[Quidway] cops-server group huawei client-type ssg
Create a new cops-server group
[Quidway-cops-huawei] cops-server 10.10.10.2
