upe-npe mac-withdraw enable

命令功能

upe-npe mac-withdraw enable命令用来使能NPE将从UPE收到的LDP MAC-Withdraw消息转发给其他NPE。

undo upe-npe mac-withdraw enable命令用来去使能NPE将从UPE收到的LDP MAC-Withdraw消息转发给其他NPE。

缺省情况下，NPE不会将从UPE收到的LDP MAC-Withdraw消息转发给其他NPE。

命令格式

upe-npe mac-withdraw enable

undo upe-npe mac-withdraw enable

参数说明

无

视图

VSI-LDP视图

缺省级别

2：配置级

使用指南

该命令主要在城域以太网ME（Metro Ethernet）中的PE-AGG设备上使用。其中：
  • NPE是指HVPLS业务中，本端设备上VSI的对等体。需要通过命令peer peer-address [ negotiation-vc-id vc-id ] [ tnl-policy policy-name ]指定。
  • UPE是指HVPLS业务中，本端设备上VSI的用户端对等体。需要通过命令peer peer-address [ negotiation-vc-id vc-id ] [ tnl-policy policy-name ] upe指定。
  • 如果本端设备没有配置HVPLS业务，则不需要配置upe-npe mac-withdraw enable命令。

使用实例

# 使能NPE将从UPE收到的LDP MAC-Withdraw消息转发给其他NPE。

<Quidway> system-view
[Quidway] vsi v1 static
[Quidway-vsi-v1] pwsignal ldp
[Quidway-vsi-v1-ldp] upe-npe mac-withdraw enable
