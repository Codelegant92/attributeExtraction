accounting-scheme（AAA视图）

命令功能

accounting-scheme命令用来创建一个计费方案，并进入计费方案视图。

undo accounting-scheme命令用来删除一个计费方案。

系统固定有一个“default”方案，不能删除，只能修改。

命令格式

accounting-scheme accounting-scheme-name

undo accounting-scheme accounting-scheme-name

参数说明

accounting-scheme-name：计费方案名称，字符串形式，长度范围是1～32。不区分大小写，方案名需要符合Windows的命名规范，不能包括斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“””和尖括号“<”、“>”等字符。

视图

AAA视图

缺省级别

3：管理级

使用指南

accounting-scheme命令在创建一个计费方案时，如果同名的计费方案已经存在，将直接进入该计费方案视图。

undo accounting-scheme只能删除一个已经存在的、没有被任何域绑定的计费方案。

系统最多支持128个计费方案。

使用实例

# 增加一个新的计费方案，名为newscheme。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] accounting-scheme newscheme
[Quidway-aaa-accounting-newscheme]

# 删除一个已经存在的计费方案，名为oldscheme。

[Quidway-aaa] undo accounting-scheme oldscheme

# 进入缺省计费方案的视图。

[Quidway-aaa] accounting-scheme default
[Quidway-aaa-accounting-default]
