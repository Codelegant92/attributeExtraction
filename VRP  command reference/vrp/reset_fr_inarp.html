reset fr inarp

命令功能

reset fr inarp命令用来清除逆向地址解析协议建立的地址映射。

命令格式

reset fr inarp

参数说明

无

视图

用户视图

缺省级别

2：配置级

使用指南

注意：

reset fr inarp命令可以删除路由器上所有动态MAP表项。执行该命令后可能会导致业务中断，所以请谨慎使用。

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除全部帧中继动态地址映射。

<Quidway> reset fr inarp
