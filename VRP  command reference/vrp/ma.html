ma

命令功能

ma命令用来在MD内创建MA，并进入MA视图。如果该MA已存在，本命令用来进入MA视图。

undo ma命令用来删除MA。

命令格式

ma ma-name

undo ma { all | name ma-name }

参数说明

ma-name：指定MA的名称。字符串形式，不支持空格、“-”和“?”，MA名称长度和MD名称长度之和不超过44个字符。同一MD内，MA的名称不能重复。

all：表示删除当前MD内的所有MA。

使用指南

同一个MD内，最多可创建4K个MA。一台VRP设备上最多可创建4K个MA。

删除一个MA时，设备自动删除该MA内的所有MEP、RMEP，并停止CC检测、清除该MA相关的告警。

使用实例

# 创建名称为ma1的MA。

<Quidway> system-view
[Quidway] cfm enable
[Quidway] cfm md md1
[Quidway-md-md1] ma ma1
[Quidway-md-md1-ma-ma1]
