x25 modulo

命令功能

x25 modulo命令用来设置一个X.25接口的窗口模数。

undo x25 modulo命令用来恢复X.25接口窗口模数的缺省值。

缺省情况下，X.25接口的窗口模数为模8方式。

命令格式

x25 modulo modulus

undo x25 modulo

参数说明

modulus：模数，取值为8或128。

使用指南

X.25的流量控制的基础就是滑动窗口，而滑动窗口的关键在于为所发送的数据分组顺序循环编号并等待对端确认。

在编号时，所谓顺序是指顺序向上递增，如“……2，3，4，5，6，……”；所谓循环是指到达某个数后（我们称这个数为“模数”），编号将再次重头开始，如模数为8时，“……4，5，6，7，0，1，……”。

X.25协议规定了两种编号模数：8（也称基本编号方式）和128（也称扩展编号方式），X.25支持以上两种方式。

执行完该命令后，必须重新启动接口，使配置生效。

使用实例

# 将X.25接口Serial1/0/0的模数设置为128。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol x25
[Quidway-Serial1/0/0] x25 modulo 128
[Quidway-Serial1/0/0] shutdown
[Quidway-Serial1/0/0] undo shutdown
