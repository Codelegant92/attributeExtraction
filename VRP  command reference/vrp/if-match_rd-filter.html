if-match rd-filter

命令功能

if-match rd-filter命令用来设置一个基于RD属性过滤器的匹配规则。

undo if-match rd-filter 命令用来删除一个基于RD属性过滤器的匹配规则。

命令格式

if-match rd-filter rd-filter-number

undo if-match rd-filter

参数说明

rd-filter-number：RD属性过滤器的编号。整数形式，取值范围是1～199。

使用实例

# 定义一条匹配规则，用来匹配RD属性过滤器。

[Quidway-route-policy] if-match rd-filter 1
