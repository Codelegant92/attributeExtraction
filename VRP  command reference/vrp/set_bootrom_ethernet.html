set bootrom ethernet

命令功能

set bootrom ethernet命令用来设置以太接口的参数信息。

命令格式

set bootrom ethernet { host-ip ip-address | ethernet-ip ip-address | filename file-name | user user-name | password password } [ slave ]

参数说明

host-ip ip-address：FTP服务器的IP地址，点分十进制形式。

ethernetip ip-address：路由器的IP地址，点分十进制形式。

filename file-name：待下载的文件名，字符串形式，长度范围是1～31。

user user-name：FTP的用户名，字符串形式，长度范围是1～16。

password password：FTP的密码，字符串形式，长度范围是1～16。

slave：设置备用主控板的以太接口参数信息。

使用指南

可以通过设置BootROM中以太接口的参数信息，到指定的服务器上下载资源文件。

说明：

目前此命令仅在NE40E、NE80E、NE5000E上可用。

使用实例

# 设置FTP服务器IP地址为192.168.0.34。

<Quidway> set bootrom ethernet host-ip 192.168.0.34
Info: set Bootrom Ethernet successful
