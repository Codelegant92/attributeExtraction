ripng split-horizon

命令功能

ripng split-horizon命令用来配置对RIPng更新报文进行水平分割。

undo ripng split-horizon命令用来关闭水平分割。

缺省情况下，启用水平分割功能。

命令格式

ripng split-horizon

undo ripng split-horizon

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

当水平分割功能被配置时，从一个接口学到的路由不能通过此接口向外发布。在毒性反转和水平分割都配置的情况下，采用毒性反转。

使用实例

# 配置对RIPng更新报文进行水平分割。

[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ripng split-horizon

# 取消对RIPng更新报文进行水平分割。

[Quidway-GigabitEthernet1/0/0] undo ripng split-horizon
