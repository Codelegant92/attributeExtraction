speed（Ethernet接口视图）

命令功能

speed命令用来设置快速以太网电接口或千兆以太网电接口的工作速率。

undo speed命令用来恢复缺省设置。

缺省情况下，接口采用自协商方式。

命令格式

speed { 10 | 100 | negotiation }

speed { 10 | 100 | auto }

undo speed

参数说明

10：设置速率为10Mbit/s。

100：设置速率为100Mbit/s。

negotiation：设置速率为自协商方式。是否支持该参数与具体产品有关。

auto：设置速率为自协商方式。该速率值只在千兆以太网电口上提供。

缺省级别

2：配置级

使用指南

对于快速以太网的电接口，可以设置速率为10Mbit/s、100Mbit/s或自协商方式。

对于千兆以太网的电接口，可以设置速率为10Mbit/s、100Mbit/s或自协商方式。

快速以太网的光接口速率固定为100Mbit/s，千兆以太网的光接口速率固定为1000Mbit/s，都不能通过配置命令改变。

改变了接口的速率模式后，一般要重启接口才可以使配置生效。

对于两端处于自协商工作方式的设备，最终协商的结果是采用二者技术能力域中优先级低的网口提供的工作方式工作。

自协商机制是建立在对接设备两端均工作在自协商状态下的。当一端处于协商状态，一端处于固定状态，自协商机制是不起作用的，协商一侧采用的是“并行检测”机制，但是只能检测到对端的工作速度，而检测不到对端的双工方式，为了与以前的只能工作在半双工方式的老设备兼容，所以这种情况下大多设备均默认为半双工方式。

说明：

对于GE电接口，工作速率1000Mbit/s与半双工模式是互斥的，不能同时配置。

使用实例

# 设置以太网接口1/0/0的工作速度是100Mbit/s。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] speed 100
[Quidway-GigabitEthernet1/0/0] shutdown
[Quidway-GigabitEthernet1/0/0] undo shutdown
