vrrp vrid timer advertise

命令功能

vrrp vrid timer advertise命令用来设置备份组中Master发送VRRP报文的间隔时间。

undo vrrp vrid timer advertise命令用来恢复缺省值。

缺省情况下，Master发送VRRP报文的间隔时间为1秒。

命令格式

vrrp vrid virtual-router-idtimer advertise adver-interval

undo vrrp vrid virtual-router-id timer advertise

参数说明

vrid virtual-router-id：VRRP备份组号，备份组号范围1～255。

timer：指定Master路由器发送VRRP广播报文的定时器的长度。

advertise adver-interval：备份组中的Master发送VRRP报文的时间间隔，单位为秒，取值范围1～255。

使用指南

VRRP备份组中的Master路由器定时（adver-interval）向组内的路由器发送VRRP报文通知自己工作正常。如果Backup超过一定时间（master-down-interval）没有收到Master发送来的VRRP报文，则认为它已经无法正常工作，同时会将自己的状态转变为Master。Backup的master-down-interval的间隔时间大约是adver-interval的3倍。

网络流量过大或者不同路由器的定时器差异等因素会导致master-down-interval异常超时而发生状态转换，这类问题可以通过延长adver-interval的间隔时间和设置延迟时间来解决。

使用实例

# 设置备份组1的Master发送VRRP报文的时间间隔为5秒。

[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] vrrp vrid 1 timer advertise 5
