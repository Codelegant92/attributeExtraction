apply preference

命令功能

apply preference命令用来为路由指定优先级。

undo apply preference命令用来取消该功能。

缺省情况下，不设置路由协议的优先级。

命令格式

apply preference preference

undo apply preference

参数说明

preference：路由的优先级。整数形式，取值范围1～255。

Route-Policy视图

缺省级别

2：配置级

使用指南

无

使用实例

# 为路由指定优先级。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply preference 90
