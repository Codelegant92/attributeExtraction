display stp region-configuration

命令功能

display stp region-configuration命令用来显示交换机上当前生效的MST域配置信息，包括域名、域的修订级别以及VLAN与生成树实例的映射关系。

命令格式

display stp [ process processs-id ] region-configuration [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

process process-id：MSTP进程的ID值，十进制整数形式，取值范围是1～256。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

  • 在系统视图中使用本命令，如果不指定MSTP进程的ID，则默认显示MSTP进程0的相关信息，否则显示指定ID的MSTP进程信息。

  • 在MSTP进程视图中使用本命令，如果不指定MSTP进程的ID，则默认显示所在视图MSTP进程的信息，否则显示指定ID的MSTP进程信息。

说明：

关于MSTP进程0的内容，请参见stp process命令。

使用实例

# 显示MST域的配置信息。

<Quidway> display stp region-configuration
Oper Configuration:
   Format selector :0
   Region name     :huawei

   Instance   Vlans Mapped

      1       1 to 10
      2       11 to 20
