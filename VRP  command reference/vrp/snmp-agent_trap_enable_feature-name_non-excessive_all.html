snmp-agent trap enable feature-name non-excessive all

命令功能

snmp-agent trap enable feature-name non-excessive all命令用来使能向网管发送指定特性下的所有非海量告警。

undo snmp-agent trap enable feature-name non-excessive all命令用来关闭向网管发送指定特性下的所有非海量告警。

undo snmp-agent trap enable feature-name excessive all命令用来关闭向网管发送指定特性下的所有海量告警。

undo snmp-agent trap enable feature-name all命令用来关闭向网管发送指定特性下的所有告警。

缺省情况下，不使能向网管发送告警。

命令格式

snmp-agent trap enable feature-name feature-name non-excessive all

undo snmp-agent trap enable feature-name feature-name non-excessive all

undo snmp-agent trap enable feature-name feature-name excessive all

undo snmp-agent trap enable feature-name all

参数说明

feature-name：产生告警的特性名称。

使用指南

在需要把某个特性下的所有非海量告警发送到网管时，可以执行命令snmp-agent trap enable feature-name non-excessive all用来配置。

执行命令snmp-agent trap enable feature-name non-excessive all 后，只有执行命令snmp-agent target-host inform或执行命令snmp-agent target-host trap 配置了接收告警的网管地址后，网管才能接收到此告警信息。

使用实例

# 配置向网管发送MSDP特性下的所有非海量告警。

<Quidway> system-view
[Quidway] snmp-agent trap enable feature-name msdp non-excessive all
