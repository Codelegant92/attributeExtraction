display mac-address summary

命令功能

display mac-address summary命令用来查看设备上所有MAC地址表项的统计信息。

命令格式

display mac-address summary

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

本命令可查看当前在位的单板上的所有MAC地址表项统计数据。这些表项根据不同的槽位，按照Blackhole、Static和Dynamic类型分别显示。

使用实例

# 查看设备上MAC地址表项的统计信息。

<Quidway> display mac-address summary
--------------------------------------------------------
Slot        Total    Blackhole  Static  Dynamic
--------------------------------------------------------
1               2            1       0        1
6               3            1       0        2
8               2            1       0        1
--------------------------------------------------------
