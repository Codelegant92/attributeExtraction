display ospfv3 db-summary-list

命令功能

display ospfv3 db-summary-list命令用来查看OSPFv3的db-summary列表。如果不指定OSPFv3进程号，该命令将显示所有OSPFv3进程的db-summary信息。

命令格式

display ospfv3 [ process-id ] db-summary-list [ statistics ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

statistics：显示所有OSPFv3进程的统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
