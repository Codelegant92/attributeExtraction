hwtacacs-server accounting

命令功能

hwtacacs-server accounting命令用来配置HWTACACS计费服务器的地址。

undo hwtacacs-server accounting命令用来删除配置的HWTACACS计费服务器。

缺省情况下，HWTACACS计费服务器的IP地址为全零，不绑定VPN实例。

命令格式

hwtacacs-server accounting ip-address [ port ] [ vpn-instance vpn-instance-name ] [ secondary ]

undo hwtacacs-server accounting [ secondary ]

参数说明

ip-address：服务器的IP地址，点分十进制格式。必须是合法的单播地址。

port：服务器的端口号，取值范围是1～65535，缺省端口号为49。

secondary：备份服务器，如果不使用此参数，则表示主服务器。

vpn-instance vpn-instance-name：绑定的VPN实例名称。字符串形式，长度范围是1～31。

缺省级别

2：配置级

使用指南

主计费服务器和备用计费服务器的IP地址和绑定的VPN实例不能完全相同，否则将提示配置不成功。

当指定VPN实例时，该VPN实例必须是已经在系统视图下创建好的。

如果多次执行此命令，新的配置将覆盖原来的配置。

只有当没有活跃、用于发送计费报文的TCP连接使用该计费服务器时，才允许删除该服务器。

使用实例

# 配置主计费服务器。

<Quidway> system-view
[Quidway] ip vpn-instance vpna
[Quidway-vpn-instance-vpna] route-distinguisher 100:1
[Quidway-vpn-instance-vpna] quit
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1] hwtacacs-server accounting 10.163.155.12 49 vpn-instance vpna
