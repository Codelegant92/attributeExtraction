frame-format（CT3接口视图）

命令功能

frame-format命令用来配置CT3的帧格式。

undo frame-format命令用来恢复缺省设置。

缺省情况下，CT3的帧格式为c-bit。

命令格式

frame-format { c-bit | m23 }

undo frame-format

参数说明

c-bit：设置帧格式为C-bit。

m23：设置帧格式为m23。

视图

CT3接口视图

缺省级别

2：配置级

使用指南

T1对应的数字信号级别为DS-1，28个T1以位复用的方式形成T3，使用的帧格式有两种：C-bit和M23。

C-bit又称为C-bit奇偶校验，配置为这种格式时，帧中的填充（stuffing）指示位可用于维护功能。

M23是DS-2到DS-3的复用过程。4个T1（DS-1）通过M12复用为一条DS-2线路，7个DS-2线路再通过M23复用为T3（DS-3）。

只有当CT3接口工作在CT3工作模式下时才能进行帧格式的配置。

使用实例

# 设置CT3接口1/0/0的帧格式为m23。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] frame-format m23
