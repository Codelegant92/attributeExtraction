display vrrp binding member-pw

命令功能

display vrrp binding member-pw命令用来显示管理VRRP备份组与成员PW的绑定关系。

命令格式

display vrrp binding admin-vrrp [ interface interface-type1 interface-number1 ] [ vrid virtual-router-id ] member-pw [ | count ] [ | { begin | include | exclude } regular-expression ]

display vrrp binding admin-vrrp [ interface interface-type1 interface-number1 ] [ vrid virtual-router-id ] member-pw vc interface interface-type2 interface-number2

display vrrp binding admin-vrrp [ interface interface-type1 interface-number1 ] [ vrid virtual-router-id ] member-pw vsi vsi-name peer ip-address [ negotiation-vc-id vc-id1 ]

display vrrp binding admin-vrrp [ interface interface-type1 interface-number1 ] [ vrid virtual-router-id ] member-pw vc switch-vc peer ip-address vc-id2

参数说明

interface interface-type1 interface-number1：管理VRRP备份组所在的接口类型和接口编号。

vrid virtual-router-id：管理VRRP备份组号。十进制整数形式，取值范围是1～255。

member-pw：查看指定的成员PW信息。

interface interface-type2 interface-number2：业务PW所在的接口类型和接口编号。

vc：查看业务VLL的PW与管理VRRP进行绑定。

vsi：查看业务VSI的PW与管理VRRP进行绑定。

switch-vc：查看交换VLL的PW与管理VRRP进行绑定。

vsi-name：要显示的VSI实例的名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

negotiation-vc-id vc-id：虚电路的唯一标识，一般用于两端VSI ID不同但要求互通的情况。参数vc-id不能与本端其他VSI配置的VSI ID相同，也不能与本端其他negotiation-vc-id指定的VSI ID相同，即vc-id是没有被用过的虚电路号，十进制整数形式，取值范围是1～4294967295。

peer ip-address：PW对端设备的MPLS LSR ID。

vc-id2：交换VLL的二层虚电ID。十进制整数形式，取值范围是1～4294967295。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display vrrp binding member-pw命令时，如果不指定管理VRRP备份组号和成员PW的VC ID，则显示所有管理VRRP备份组和成员PW的绑定信息。

使用实例

# 显示接口GE1/0/0.1下的管理VRRP备份组和业务VSI中的PW的绑定关系，其中VSI的名称是v1，对等体是1.1.1.1。

<Quidway> display vrrp binding admin-vrrp interface gigabitethernet 1/0/0.1 member-pw vsi v1 peer 1.1.1.1
interface: GigabitEthernet1/0/0.1,  admin-vrrp vrid: 100, state:Master
VSI PW number: 1
  VSI name: v1, peer router ID: 1.1.1.1, vcid: 101, state: Up

# 显示接口GE1/0/0.1下的管理VRRP备份组业务VLL中的PW的绑定关系，其中业务VLL中的PW所在的接口是GE1/0/1.1。

<Quidway> display vrrp binding admin-vrrp interface gigabitethernet 1/0/0.1 member-pw vc interface gigabitethernet 1/0/1.1
Interface: GigabitEthernet1/0/0.1, admin-vrrp vrid: 1, state: Master
VC number: 1
  Interface name: GigabitEthernet1/0/1.5, type: VLAN, state: Up

# 显示接口GE1/0/0.1下的管理VRRP备份组和交换VLL中的PW的绑定关系。其中交换PW的对端设备的LSR ID是1.1.1，VC ID是100。

<Quidway> displa vrrp binding admin-vrrp interface gigabitethernet1/0/0.1 member-pw vc switch-vc peer 1.1.1.1 100

Switch VC number: 1
  Peer router ID: 1.1.1.1, vcid: 100, type: Ethernet, state: Up
