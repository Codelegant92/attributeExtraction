efm enable

命令功能

efm enable命令用来全局使能EFM OAM功能或使能接口的EFM OAM功能。

undo efm enable命令用来全局禁止EFM OAM功能或禁止接口的EFM OAM功能。

缺省情况下，EFM OAM功能处于全局禁止状态，各接口的EFM OAM功能处于禁止状态。

命令格式

efm enable

undo efm enable

参数说明

无

视图

系统视图、Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

在系统视图下使用undo efm enable命令全局禁止EFM OAM功能时，设备自动删除所有EFM相关配置。如果在本设备进行远端环回并发送大量测试报文时使用此undo命令，可能会导致大量测试报文被转发。

使用本命令全局使能EFM OAM功能是使用所有EFM OAM功能的前提条件。

当使能接口的EFM OAM功能后，接口开始发送OAMPDU，开始点到点的EFM链路检测。只有当对端端口也使能了EFM OAM功能，才能进入EFM链路检测阶段。

只能在EFM OAM功能全局使能后，接口的EFM OAM功能使能前，才能使用efm mode命令配置接口的EFM OAM工作模式。

如果遇到提示GTL资源不足时，用户需要释放已经使用的GTL资源或者到公司买新的资源。

使用实例

# 全局使能EFM OAM功能。

<Quidway> system-view
[Quidway] efm enable

# 使能接口GigabitEthernet1/0/0的EFM OAM功能。

<Quidway> system-view
[Quidway] efm enable
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] efm enable
