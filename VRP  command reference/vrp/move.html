move

命令功能

move命令用来移动文件。不能在设备之间移动文件。

命令格式

move source-filename destination-filename

参数说明

source-filename：源文件名，可以使用通配符“*”。字符串形式，长度范围是1～64。

destination-filename：目标文件名。字符串形式，长度范围是1～64。

视图

用户视图

缺省级别

3：管理级

使用指南

如果目标文件名为已经存在的目录名，则源文件会移动到该指定目录，新生成的文件名与源文件同名。

如果目标文件名与已经存在的文件名重名，系统将提示是否覆盖。

该命令不支持跨存储设备的文件移动。

使用实例

移动文件从flash:/test/sample.txt到flash:/sample.txt。

<Quidway> move flash:/test/sample.txt flash:/sample.txt
Warning: Move file flash:/test/sample.txt to flash:/sample.txt ?[Y/N] :y
Info: File flash:/test/sample.txt to flash:/sample.txt
