file prompt

命令功能

file prompt命令用来修改路由器文件操作的提醒方式。

缺省情况下，为alert提醒方式。

命令格式

file prompt { alert | quiet }

参数说明

alert：对由于用户操作（比如删除文件操作）可能会导致数据丢失或破坏的情况进行交互式确认。

quiet：对由于用户操作（比如删除文件操作）可能会导致数据丢失或破坏的情况进行不作提示。

视图

系统视图

缺省级别

3：管理级

使用指南

如果将文件操作的提醒方式设置为quiet，则对由于用户误操作（比如删除文件操作）而导致数据丢失的情况不作提示。

使用实例

# 设置路由器文件操作的提醒方式为quiet。

<Quidway> system-view
[Quidway] file prompt quiet

# 设置路由器文件操作的提醒方式为alert。

<Quidway> system-view
[Quidway] file prompt alert
