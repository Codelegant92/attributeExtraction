display mpls l2vc

命令功能

display mpls l2vc命令用来查看VRP上Martini方式的虚电路VC（Virtual Circuit）。

命令格式

display mpls l2vc [ vc-id | interface interface-type interface-number | remote-info [ vc-id ] | state { down | up } ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：与CE相连的接口类型和接口编号。

remote-info：显示远端对等体上的VC信息。

vc-id：二层虚电路的ID。十进制整数形式，取值范围是1～4294967295。

state：对等体之间VC的状态信息。

down：状态为Down的VC信息。

up：状态为Up的VC信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display mpls l2vc命令后，如果指定接口，则只显示指定CE接口上Martini方式的VC。

如果指定参数remote-info但不指定vc-id，则显示对等体上的，与本端共同建立的所有VC的信息。

项目

描述

Client interface

用户接口及其状态。

Session State
VC两端的LDP会话状态：

  • up表示VC两端LDP会话建立成功；

  • down表示VC两端的会话LDP建立不成功。

AC Status

本PE与直连CE之间的链路状态。

VC State
VC状态，up或down：

  • up表示PW建立成功；

  • down表示PW建立不成功。

VC ID

虚电路号，十进制整数，缺省值为0。

VC Type

L2VC的封装类型。

Destination

对端设备的LSR ID。

Local Group ID

本端Group ID。

Remote Group ID

远端Group ID。

local VC label

本端VC标签值。

remote VC label

远端VC标签值。

local AC OAM State

本端AC的OAM状态。

  • 配置mpls l2vpn oam-mapping命令后，VC下的Local AC OAM State表示AC链路的链路状态。

  • 如果不配置mpls l2vpn oam-mapping命令，那么Local AC OAM State的状态一直保持up，与链路状态没有关联关系。

local PSN State

本端设备网络侧的状态。

local forwarding state

本端转发表的状态。

remote AC OAM state

远端AC的OAM状态。

remote PSN state

远端设备网络侧的状态。

remote forwarding state

远端转发表的状态。

BFD for PW

是否使能了BFD检测PW状态的功能。

manual fault

是否手工设置了PW故障。

active state

是否处于活跃状态（活跃状态表示可以转发）。

forwarding entry

是否存在转发表项。

link state

AC接口的链路状态：

  • Up：该接口处于物理可用状态

  • Down：该接口处于不可用状态

Local VC MTU

本端VC最大传输单元。

Remote VC MTU

远端VC最大传输单元。

Local VCCV
是否使能本端VC连接检测功能。

  • disable表示本端没有使能VC连接检测功能；

  • enable表示本端使能了VC连接检测功能。

Remote VCCV
是否使能远端VC连接检测功能。

  • disable表示远端没有使能VC连接检测功能；

  • enable表示远端使能了VC连接检测功能。

tunnel policy name

应用于该VC的隧道策略名。“--”表示没有为该VC配置隧道策略。

traffic behavior name

应用于该VC的流量策略名。“--”表示没有为该VC配置流量策略。

PW Template Name

模板名称。“--”表示没有为该PW配置PW模板。

primary or secondary

使用主用PW还是备用PW。

VC Tunnel/token info: 1 tunnels/tokens

VC Tunnel/token信息。“1 tunnels/tokens”表示该PW使用一个隧道/Token。

TNL Type

该PW使用的隧道的类型。

TNL ID

该PW使用的隧道ID。

Create Time

该VC已经创建了多长时间。

Up Time

该VC保持UP状态多长时间。

Last Change Time

该VC最后一次状态变化距现在多长时间。

VC last up time :

VC最近一次UP的时刻。

VC total up time:

VC累积UP的时间（除去中间DOWN的时间）。

项目

描述

Total remote ldp vc

所有已建立的远程标签LDP VC条数

Transport VC ID

传输的VC ID，此ID唯一标志了一条VC

Group ID

L2VPN所属的组号，默认值为0

Peer Addr

远端对等体的地址

Remote Encap

远端的VC封装类型

C Bit
是否使能控制字（Control Word），

  • 1：表示使能控制字

  • 0：表示未使能控制字

Mtu/CELLS

L2VPN的最大传输单元

N Bit
是否支持Notification报文。

  • 1：表示支持

  • 0：表示不支持

S Bit
状态码（Status Code）。

  • 0：表示转发状态

  • 1：表示非转发状态
