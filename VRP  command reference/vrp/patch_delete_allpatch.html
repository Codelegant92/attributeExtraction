patch delete allpatch

命令功能

patch delete allpatch命令用来删除系统中所有的补丁，此时系统未运行任何补丁。

命令格式

patch delete allpatch

参数说明

无

视图

用户视图

缺省级别

3：管理级

使用指南

系统中的任何一个补丁，无论其处于什么状态，执行该命令后，都被从内存补丁区中删除。

使用实例

# 将系统中所有板的运行补丁软件状态恢复到运行补丁前的状态。

<Quidway> patch delete allpatch
