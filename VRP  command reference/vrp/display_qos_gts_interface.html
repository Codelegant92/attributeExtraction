display qos gts interface

命令功能

display qos gts interface命令用来显示某个接口或所有接口的GTS配置情况和统计信息。

命令格式

display qos gts interface [ interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口号。

所有视图

缺省级别

1：监控级

使用指南

display qos gts interface命令可以观察到的信息包括：

  • 接口上的承诺信息速率

  • 接口上的承诺突发尺寸

  • 接口上的过度突发尺寸

  • 缓存队列长度

  • 队列尺寸

  • 发送的报文数量

  • 丢弃的报文数量

  • 延迟的报文数量

如不指定接口，将显示所有接口的GTS配置情况和运行统计信息。

使用实例

<Quidway> display qos gts interface GigabitEthernet6/0/0
