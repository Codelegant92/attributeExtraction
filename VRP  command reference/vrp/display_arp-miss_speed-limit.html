display arp-miss speed-limit

命令功能

display arp-miss speed-limit souce-ip命令用来查询基于源IP地址对ARP Miss消息进行时间戳抑制的抑制速率。

命令格式

display arp-miss speed-limit source-ip [ slot slot-id ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

slot-id：当前在位的接口板的槽位号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display arp-miss speed-limit souce-ip命令的输出信息包括：

  • 指定的接口板槽号
  • 时间戳抑制的消息类型
  • 时间戳抑制的抑制速率

使用实例

# 查询3号接口板上基于源IP地址对ARP Miss消息进行时间戳抑制的抑制速率。

<Quidway> display arp-miss speed-limit  source-ip slot 3
Slot     SuppressType    SuppressValue
 ---------------------------------------------------
 3        ARP-miss       500
