hwtacacs-server template

命令功能

hwtacacs-server template命令用来创建一个新的HWTACACS服务器模板。

undo hwtacacs-server template命令用来删除一个HWTACACS服务器模板。

在HWTACACS服务器模板视图下，可以对此HWTACACS服务器模板进行具体配置。

命令格式

hwtacacs-server template template-name

undo hwtacacs-server template template-name

参数说明

template-name：HWTACACS服务器模板名称。字符串形式，长度范围是1～32。字符包括大写字母A～Z、小写字母a～z、数字0～9、点号“.”、短线“-”和下划线“_”。

缺省级别

2：配置级

使用指南

在HWTACACS服务器模板视图下可以对此HWTACACS服务器模板进行具体配置。

系统最多支持配置128个HWTACACS服务器模板。只有当模板没有用户使用时，才能修改模板内容。

删除服务器模板时，如果此模板有用户正在使用，则系统将提示删除失败。

使用实例

# 创建名为test1的HWTACACS服务器模板并进入相应的HWTACACS Server视图。

<Quidway> system-view
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1]
