vrrp vrid track admin-vrrp vrid

命令功能

vrrp vrid track admin-vrrp vrid 命令用来配置成员VRRP备份组与管理VRRP备份组的绑定关系。

undo vrrp vrid track admin-vrrp vrid命令用来取消成员VRRP备份组与管理VRRP备份组之间的绑定关系。

缺省情况下，成员VRRP备份组与管理VRRP备份组之间没有绑定关系。

命令格式

vrrp vrid virtual-router-id1 track admin-vrrp interface interface-type interface-number vrid virtual-router-id2

undo vrrp vrid virtual-router-id1 track admin-vrrp

参数说明

vrid virtual-router-id1：成员VRRP备份组号。十进制整数形式，取值范围是1～255。

vrid virtual-router-id2：管理VRRP备份组号。十进制整数形式，取值范围是1～255。

interface interface-type interface-number：管理VRRP备份组所在的接口类型和接口编号。

视图

Ethernet接口视图、Ethernet子接口视图、GE接口视图、GE子接口视图、Eth-Trunk接口视图、Eth-Trunk子接口视图、VE接口视图、VE子接口视图

缺省级别

2：配置级

使用指南

vrrp vrid track admin-vrrp vrid命令需要在成员VRRP备份组接口下配置，不能在管理VRRP备份组的接口下配置。

业务VRRP接口与管理VRRP接口不能配置在不同的接口板，而必须配置在同一物理接口的不同子接口，否则管理VRRP的状态不能反映业务VRRP接口的真实状态。

使用实例

# 配置接口GE1/0/0下的成员VRRP备份组1绑定到接口GE2/0/0下的管理VRRP备份组2。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] vrrp vrid 1 track admin-vrrp interface gigabitethernet 2/0/0 vrid 2
