delete（FTP客户端视图）

命令功能

delete命令用来删除指定的文件。

命令格式

delete remotefile

参数说明

remotefile：文件名，字符串形式，长度范围是1～64。

视图

FTP客户端视图

缺省级别

3：管理级

使用指南

在FTP客户端视图下执行此命令删除的文件将无法被恢复。

使用实例

# 删除文件temp.c。

<Quidway> ftp 172.16.104.110

Press CTRL+K to abort
Connected to ftp 172.16.104.110

User(ftp 172.16.104.110:(none)):huawei
331 Password required for huawei
Password:
230 User logged in.
[ftp] delete temp.c
Warning: File temp.c will be deleted into the recycle bin. Continue? [Y/N]: y
250 DELE command successful.
