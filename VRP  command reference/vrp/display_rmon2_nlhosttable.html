display rmon2 nlhosttable

命令功能

display rmon2 nlhosttable命令用来查看主机表。

命令格式

display rmon2 nlhosttable [ hostcontrolindex ctrl-index ] [ timemark time-value ] [ protocoldirlocalindex protocol-local-index ] [ hostaddress ip-address ]

参数说明

hostcontrolindex ctrl-index：索引值，唯一识别主机控制表中的行数。整数形式，取值范围是1～65535。

timemark time-value：启用时间过滤器。时间过滤器取值范围是0～4294967295。主机表中，ChgTm项大于该值的条目将被显示。

protocoldirlocalindex protocol-local-index：标识nlHostAddress的网络层协议。协议索引值的取值范围是1～2147483647。

hostaddress ip-address：查看指定主机的流量。

使用指南

对于每个接口下每一种协议生成的主机条目，最多可显示前5000行的信息。

使用实例

# 查看主机表中的第一行信息。

<Quidway> display rmon2 nlhosttable hostcontrolindex 1 protocoldirlocalindex 2 hostaddress 10.110.94.177
