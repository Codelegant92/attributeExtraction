ripng summary-address

命令功能

ripng summary-address命令用来配置RIPng路由器在接口发布聚合IPv6地址并指定被聚合的路由的IPv6地址前缀。

undo ripng summary-address命令用来禁止RIPng路由器发布聚合的IPv6地址。

缺省情况下，不使能该功能。

命令格式

ripng summary-address ipv6-address prefix-length [ avoid-feedback ]

undo ripng summary-address ipv6-address prefix-length

参数说明

ipv6-address：指定聚合后的IPv6网络地址。

prefix-length：IPv6网络前缀长度。整数形式，取值范围是0～128。它指定地址中有多少连续的位组成前缀，即地址中的网络地址部分。

avoid-feedback：禁止从此接口学习到相同的聚合路由。

使用指南

如果一条路由的前缀和前缀长度与定义的IPv6前缀匹配，则这个自定义的IPv6前缀将取代原来的路由被发布出去。这样，多条路由将由一条路由所代替，并且，这条路由的度量值是原多条路由中最低的。

通过指定avoid-feedback关键字，本接口将不再学习到和已发布的聚合IP地址相同的聚合路由，从而可以起到防止产生路由环路的作用。

使用实例

# 给接口GigabitEthernet1/0/0配置IPv6地址2001:200::3EFF:FE11:6770，其地址前缀长度为64位。通过RIPng100聚合为IPv6地址前缀2001:200::/35。

[Quidway] ipv6
[Quidway] ripng 100
[Quidway-ripng-100] quit
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 address 2001:200::3EFF:FE11:6770/64
[Quidway-GigabitEthernet1/0/0] ripng 100 enable 
[Quidway-GigabitEthernet1/0/0] ripng summary-address 2001:200:: 35
