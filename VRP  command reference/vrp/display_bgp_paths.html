display bgp paths

命令功能

display bgp paths命令用来显示BGP的路径信息。

命令格式

display bgp paths [ as-regular-expression [ | count ] | | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } paths [ as-regular-expression [ | count ] | | { begin | exclude | include } regular-expression ]

display bgp ipv6 paths [ as-regular-expression [ | count ] | | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name } paths [ as-regular-expression [ | count ] | | { begin | exclude | include } regular-expression ]

参数说明

as-regular-expression：匹配AS_Path的正则表达式。

vpnv4：查看VPNv4的AS路径信息。

all：查看所有的AS路径信息。

vpn-instance vpn-instance-name：查看指定VPN实例的AS路径信息。

ipv6：查看本地BGP数据库中的IPv6-BGP路由的路径属性。

vpnv6 paths：查看BGP VPNv6的AS路径信息。

vpn6-instance vpn6-instance-name：显示本端指定的BGP VPNv6的AS路径信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示BGP路径信息。

<Quidway> display bgp paths

    0x50EEEB8     1       1         0            ?
    0x50EEF88     1       1                     i

表1 display bgp vpnv4 paths命令输出信息描述

项目

描述

Address

本地数据库中的路径属性节点地址，以16进制形式显示

Hash

存放路径的哈希桶

Refcount

路由被引用的次数

MED

路由的度量值

Path

报文经过的AS路径列表

Origin

路由来源
