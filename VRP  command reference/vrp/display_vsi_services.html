display vsi services

命令功能

display vsi services命令用来显示与VSI相关联的AC接口信息。

命令格式

display vsi services { all | vsi-name | interface interface-type interface-number | vlan vlan-id } [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：显示所有VSI与AC接口的关联信息。

vsi-name：显示指定VSI与AC接口的关联信息。字符串形式，不支持空格，区分大小写，取值范围是1～31。

interface interface-type interface-number：显示指定AC接口与VSI的关联信息。

vlan vlan-id：显示属于指定VLAN的AC接口与VSI的关联信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用实例

# 显示所有VSI与AC接口的关联信息。

<Quidway> display vsi services all

--------------------------------------------------------------------------------

Eth-Trunk1.1                    vsi1                           down    down

Eth-Trunk1.1                   vsi1                            down    down

项目

描述

Interface

接口名称。

Vsi Name

接口所绑定的VSI名称。

Admin Status

接口所绑定的VSI状态。具体分为：up：VSI的状态为Updown：VSI的状态为Down*down：VSI的状态为管理Down

Physical Status

接口的物理状态。具体分为：up：接口的物理状态为Updown：接口的物理状态为Down
