display mpls te tunnel statistics

命令功能

display mpls te tunnel statistics命令用来查看隧道数量及状态的统计信息。

命令格式

display mpls te tunnel statistics [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用实例

# 查看示MPLS TE隧道数量及状态的统计信息。

<Quidway> display mpls te tunnel statistics
Ingress:    84 Tunnels     83 Up,     83 CRLSPs Up
             0 Modified,    0 In-Progress,    1 Failed
Transit:     2 Up
Egress :     3 Up
表1 display mpls te tunnel statistics命令输出信息描述

项目

描述

Ingress
入口节点处于该节点的隧道数量及状态：

  • Tunnel：主隧道总数；

  • Up：处于Up状态的隧道数量；

  • CRLSPs Up：其中处于Up状态的CR-LSP的数目

  • Modified：原up的隧道由于某原因处于重建状态的数量

说明：
可能导致隧道处于这种状态的原因包括：

  • 用户手工修改了隧道的配置

  • 隧道正在进行重优化

  • 隧道处于FRR INUSE状态

  • 隧道处于BACKUP保护状态

  • In-Progress：处于建立过程中的隧道的数量，此时的隧道还没有Up

  • Failed：处于Down状态的隧道数量

Transit

中间转发节点处于该节点的隧道Up数量

Egress

出口节点处于该节点的隧道Up数量
