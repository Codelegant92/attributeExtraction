t3 framed（CPOS接口视图）

命令功能

t3 framed（CPOS接口视图）命令用来配置CPOS T3通道工作在非通道化模式。

undo t3 framed（CPOS接口视图）命令用来恢复缺省配置。

缺省情况下，CPOS的T3通道不可用。必须配置CPOS T3通道的工作模式后，T3通道才可用。

命令格式

t3 t3-number framed

undo t3 t3-number framed

参数说明

t3-number：CPOS的T3通道编号。整数形式，取值范围是1～3。

视图

CPOS接口视图

缺省级别

2：配置级

使用指南

当在CPOS接口下执行命令t3 framed后，该T3通道形成一个速率为44.210Mbit/s的同步串口。

当在CPOS接口下执行命令t3 unframed后，该T3通道形成一个速率为44.736Mbit/s的同步串口。

使用实例

# 配置CPOS 1/0/0的第3个T3通道工作在非通道化模式。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos 1/0/0] t3 3 framed
