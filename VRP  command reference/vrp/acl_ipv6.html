acl ipv6

命令功能

acl ipv6命令用来创建基于IPv6的访问控制列表并进入ACL6视图。

如果已存在ACL6，此命令只用来进入ACL6配置模式。

undo acl ipv6命令用来删除ACL6。

命令格式

acl ipv6 [ number ] acl6-number [ match-order { auto | config } ]

undo acl ipv6 { all | number acl6-number }

参数说明

number：利用数字标识一个ACL6。

acl6-number：指定ACL6编号，整数形式，取值范围是1000～3999。其中2000～2999是基本ACL6，3000～3999是高级ACL6，1000～1999是基于接口的ACL6。

auto：表示ACL6规则是自动排序，也就是“深度优先”的顺序。

说明：

若“深度优先”的顺序相同，则匹配该规则时按用户的配置顺序。

config：表示ACL6规则按照配置顺序进行匹配。

说明：

指定匹配该规则时按用户的配置顺序，是指在用户没有指定rule-id的前提下。若用户指定了rule-id，则匹配规则时，按rule-id由小到大的顺序。

all：删除所有配置的ACL6。

使用指南

创建ACL6时需要明确定如下参数：

  • 指定数字型ACL6

  • 指定ACL6的匹配顺序(可选)，缺省情况下，匹配顺序是配置顺序

使用实例

# 创建匹配顺序为config，序列号为1000的ACL6。
<Quidway> system-view
[Quidway] acl ipv6 number 1000 match-order config
[Quidway-acl6-if-1000]
# 创建一个序列号为1500的ACL6。
<Quidway> system-view
[Quidway] acl ipv6 number 1500
[Quidway-acl6-if-1500]
