display mip create-type

命令功能

display mip create-type命令用来查看全局或指定接口的MIP生成规则。

命令格式

display mip create-type [ interface interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：表示查看指定接口的MIP生成规则。interface-type表示接口类型，取值包括：ethernet、eth-trunk和gigabitethernet。interface-number表示接口编号。不指定本参数时，表示显示全局的MIP生成规则。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当使用mip create-type（系统视图）命令配置完MIP生成规则后或CFM功能出现故障时，可使用本命令验证或了解MIP生成规则的配置情况。

使用实例

# 查看以太网接口GigabitEthernet1/0/1的MIP生成规则。

<Quidway> display mip create-type interface gigabitethernet1/0/1
Interface Name                MIP Create-Type  MIP Create-Type On Interface
---------------------------------------------------------------------------
GigabitEthernet1/0/1          default          --
表1 display mip create-type命令输出信息描述表

项目

描述

MIP Create-Type
全局MIP生成规则：

  • default：如果指定接口不存在更高级别的MEP，并且不存在更低级别的MIP，则可以在该接口上创建MIP。

  • explicit：如果在指定接口下存在更低级的MEP但不存在更高级别的MEP，而且不存在更低级别的MIP，则可以创建MIP，则可以创建MIP节点。

  • none：不自动创建MIP。

MIP Create-Type On Interface
接口的MIP生成规则：

  • default：如果指定接口不存在更高级别的MEP，并且不存在更低级别的MIP，则可以在该接口上创建MIP。

  • explicit：如果在指定接口下存在更低级的MEP但不存在更高级别的MEP，而且不存在更低级别的MIP，则可以创建MIP节点。

  • none：不自动创建MIP。

  • --：表示接口的MIP生成规则和全局MIP生成规则相同。
