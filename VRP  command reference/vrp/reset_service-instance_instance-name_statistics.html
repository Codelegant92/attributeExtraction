reset service-instance instance-name statistics

命令功能

reset service-instance instance-name statistics命令用来清除当前业务实例的流量统计数据。

命令格式

reset service-instance instance-name service-instance-name statistics

参数说明

instance-name service-instance-name：指定业务实例的名字。长度范围是1～31个字符，首位不能是空格。

视图

用户视图

缺省级别

3：监控级

使用指南

为了获得某个时间段内的业务实例的流量统计数据，首先执行该命令清除当前的业务实例统计数据，经过一段时间后，再使用命令display service-instance instance-name statistic查看业务实例的统计数据。

使用实例

# 清除业务实例的流量统计数据。

<Quidway> reset service-instance instance-name mysi statistics
