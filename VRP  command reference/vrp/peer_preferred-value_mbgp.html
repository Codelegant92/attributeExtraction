peer preferred-value（MBGP）

命令功能

peer preferred-value命令用来为对等体分配首选值。

undo peer preferred-value命令用来删除分配给对等体的首选值。

缺省情况下，从其它MBGP对等体学来的路由的首选值为0。

命令格式

peer { group-name | peer-address } preferred-value preference-value

undo peer { group-name | peer-address } preferred-value

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

preference-value：要分配的路由首选值。整数形式，取值范围是0～65535。

视图

BGP-IPv4组播地址族视图

缺省级别

2：配置级

使用指南

所有从该对等体学来的路由都有一个初始首选值。在多条路由中，拥有最高首选值的路由将被选作到达指定网络的路由。

使用实例

# 分配对等体首选值。

<Quidway> system-view
[Quidway] bgp 109
[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] peer 131.108.1.1 enable
[Quidway-bgp-af-multicast] peer 131.108.1.1 preferred-value 50
