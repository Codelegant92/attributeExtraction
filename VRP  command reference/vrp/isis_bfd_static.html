isis bfd static

命令功能

isis bfd static命令用来在使能IS-IS的特定接口下配置BFD特性。

undo isis bfd static命令用来取消该BFD特性。

缺省情况下不创建静态的BFD会话。

命令格式

isis bfd static

undo isis bfd static

参数说明

无

接口视图

缺省级别

2：配置级

使用指南

使能isis fast-sense命令后，系统自动识别为isis bfd static和isis fast-sense rpr两条命令。

isis bfd block、isis bfd enable和isis bfd static是三条互斥命令，不能同时配置。

使用实例

# 配置POS 1/0/0接口的静态BFD特性。

<Quidway> system-view
[Quidway] interface pos1/0/0
[Quidway-Pos1/0/0] isis bfd static
