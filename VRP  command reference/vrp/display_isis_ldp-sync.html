display isis ldp-sync

命令功能

display isis ldp-sync命令用来查看接口上LDP和IS-IS同步状态的信息。

命令格式

display isis [ process-id | vpn-instance vpn-instance-name ] ldp-sync interface [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：指定IS-IS进程号。整数形式，取值范围是1～65535。

vpn-instance vpn-instance-name：指定VPN实例名。vpn-instance-name为字符串形式，不支持空格，区分大小写，长度范围是1～31。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

所有使能了LDP和IS-IS同步功能的接口，都可以通过该命令查看LDP和IS-IS同步的状态信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看接口LDP和IS-IS同步状态的信息。

<Quidway> display isis ldp-sync interface
Ldp Sync interface information for ISIS(1)
-----------------------------------------------------------------------
Interface    HoldDownTimer    HoldMaxCostTimer   LDP State   Sync State
GigabitEthernet1/0/0    10           10              Down        Init

项目

描述

Interface

与邻居相连的接口。

HoldDownTimer

接口不建立IS-IS邻居而等待LDP会话建立的时间间隔，缺省值是10秒。

HoldMaxCostTimer

IS-IS在本地设备的LSA中通告最大metric值的时间间隔，缺省值是10秒。

LDP State
LDP会话的状态，包括以下几种：

  • Up

  • Down

Sync State
LDP和IS-IS的同步状态，包括以下几种：

  • Sync-Achieved：已同步。

  • HoldDown：表示接口不建立IS-IS邻居而等待LDP会话建立的状态。

  • HoldMaxCost：表示IS-IS在本地设备的LSA中通告最大metric值的状态。

  • Init：初始状态。
