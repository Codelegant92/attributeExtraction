display rm bfd-session

命令功能

display rm bfd-session命令用来查看路由管理(RM)中保存的BFD会话配置信息，可显示的内容包括BFD的全局状态、BFD会话数量、每个BFD会话的VPN实例、目的地址、源地址、接口、会话状态等信息。

命令格式

display rm bfd-session [ all ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display rm bfd-session [ vpn-instance vpn-instance-name ] [ destination destination-address ] [ source source-address ] [ interface interface-type interface-number ] [ protocol { bgp | isis-l1 | isis-l2 | isisl1l2 | ospf | pim }] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：显示RM中BFD会话的全部配置信息（包括公网和私网）。如果不指定参数all，则仅显示公网的BFD会话配置信息。

vpn-instance vpn-instance-name：VPN实例的名称。

destination destination-address：对端目的地址。

source source-address：本地源地址。

interface interface-type interface-number：出接口类型和出接口号。

protocol：指定路由协议。

  • bgp：BGP协议。

  • isis-l1：IS-IS协议，Level-1级别。

  • isis-l2：IS-IS协议，Level-2级别。

  • isisl1l2：IS-IS协议，Level-1-2级别。

  • ospf：OSPF协议。

  • pim：PIM协议。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通过命令可以查看路由协议动态创建的BFD会话。如果需要更改BFD会话参数值请参见命令bfd all-interfaces、ospf bfd、isis bfd、peer bfd和pim bfd。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示RM中BFD会话的配置信息。

<Quidway> display rm bfd-session vpn-instance a destination 1.1.1.1 source 1.1.1.2 interface GigabitEthernet 2/0/0

项目

描述

BFD Global Status
BFD全局状态

  • ON：全局使能BFD功能

  • OFF：全局关闭BFD功能

BFD Session Total Number

BFD会话的总数

BFD Session

BFD会话的编号

Destination

BFD会话绑定的对端IP地址

Source

BFD会话绑定的源端IP地址

Session-State
BFD会话当前状态

  • AdminDown

  • Down

  • Init

  • Up

Create-State
RM模块中BFD会话建立状态

  • Creating：表示会话已经由路由协议通告给RM，但RM还没有通告BFD进行创建

  • Established：表示RM已经通告BFD进行会话创建（BFD会话最终是否建立需要查看Session-State）

Interface

BFD绑定的本端物理接口

Route-Protocol
使能BFD的路由协议

  • BGP

  • IS-IS Level-1

  • IS-IS Level-2

  • OSPF

  • PIM

Rx-Interval(ms)

配置的接收间隔（毫秒）

Tx-Interval(ms)

配置的发送间隔（毫秒）

Multiplier

配置的本地检测倍数

BFD Session exceed The Limit number

当配置的BFD会话数目超过系统规格上限的时候，会提示此信息。
