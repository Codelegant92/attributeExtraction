display ip rd-filter

命令功能

display ip rd-filter命令用来显示已经配置的RD属性过滤器。

命令格式

display ip rd-filter [ rd-filter-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

rd-filter-number：RD属性过滤器的编号。整数形式，取值范围是1～199。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示已经配置的RD属性过滤器。

<Quidway> display ip rd-filter
Route Distinguisher Filter 1
        permit 1.1.1.1:1 2.2.2.2:* 100:1 200:*
Route Distinguisher Filter 2
        deny 1:1 2:2
        permit 1:* 2:*
