vlan-group

命令功能

vlan-group命令用来配置用户的VLAN组。

undo vlan-group命令用来取消指定ID的用户VLAN组。

缺省情况下，子接口下没有配置任何的用户VLAN组。

命令格式

vlan-group group-id

undo vlan-group group-id

参数说明

group-id：用户组号，十进制整数形式，取值范围是1～16。

视图

Ethernet子接口视图、GE子接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

在以太网子接口上使用vlan-group命令前，应先在该子接口对应的主接口下配置mode user-termination命令，否则系统会提示配置错误。

配置用户的VLAN组是为了对不同的用户组应用不同的QoS策略。

使用实例

# 在子接口GE1/0/0.1下配置用户组，其用户组号是1。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mode user-termination
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] interface gigabitethernet1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] vlan-group 1
[Quidway-GigabitEthernet1/0/0.1-vlangroup1]
