rrpp enable

命令功能

rrpp enable命令用来使能RRPP协议。

undo rrpp enable或rrpp disable命令用来关闭RRPP协议。

缺省情况下，关闭RRPP协议。

命令格式

rrpp enable

undo rrpp enable

rrpp disable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

激活RRPP环需满足以下条件：

  • 使能RRPP协议

  • 使能RRPP环

也就是RRPP协议使能状态与RRPP环使能状态的“与”值决定某个RRPP环是否处于激活状态。

使用实例

# 使能RRPP协议。

<Quidway> system-view
[Quidway] rrpp enable

# 关闭RRPP协议。

<Quidway> system-view
[Quidway] undo rrpp enable

# 关闭RRPP协议。

<Quidway> system-view
[Quidway] rrpp disable
