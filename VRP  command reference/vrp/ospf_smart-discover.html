ospf smart-discover

命令功能

ospf smart-discover命令用在接口上使能Smart-discover功能。

undo ospf smart-discover命令用来在接口上关闭Smart-discover功能。

缺省情况下，接口不使能Smart-discover功能。

命令格式

ospf smart-discover

undo ospf smart-discover

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

本命令可以使路由器的邻居状态或者多址网络（广播型或NBMA）上的DR、BDR发生变化时，不必等到Hello定时器到时，就立刻主动的向邻居发送Hello报文。

使用实例

# 使能Smart-discover功能。

<Quidway> system-view 
[Quidway] interface pos1/0/0
[Quidway-Pos1/0/0] ospf smart-discover
