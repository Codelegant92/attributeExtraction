security-key

命令功能

security-key命令用来配置加密的密码。

undo security-key命令用来恢复密码到缺省值。

缺省情况下，simple方式密码为00E0FC0000000000。

命令格式

security-key { simple simple-key | cipher cipher-key }

undo security-key

参数说明

simple simple-key ：以明文方式加密安全密钥。密码是长度小于等于16字节的连续字符串。缺省值是00E0FC0000000000。

cipher cipher-key ：以密文方式加密安全密钥。密码是长度小于等于16字节的连续字符串。查询时，显示24字节字符串。缺省值是00E0FC0000000000。

使用指南

注意：

如果使用simple选项，密码将以明文形式保存在配置文件中，以低级别登录的用户可以通过查看配置方式获取密码，造成安全隐患。因此，建议使用cipher选项，将密码加密保存。

用户可以选择采用明文加密或密文加密。

  • 明文加密时，在配置文件中采用明文反编译后显示。

  • 密文加密时，在配置文件中采用加密后的乱码显示，不显示真正的密码。

为了提高系统的安全性可以配置加密密码。MC-Trunk中的两台设备上的加密密码必须配置为一致。

使用实例

# 配置MC-Trunk的密码为密文方式，密码为00E0FC000000。

<Quidway> system-view
[Quidway] mc-trunk 1
[Quidway-mc-trunk-1] security-key cipher 00E0FC000000
