display isis brief

命令功能

display isis brief命令用来显示IS-IS的概要信息。

命令格式

display isis brief [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。整数形式，取值范围是1～65535。

vpn-instance vpn-instance-name：显示指定IPv4 VPN的IS-IS多实例进程的概要信息。vpn-instance-name为IPv4 VPN实例的名称，区分大小写。

vpn-instance default：显示除IPv4 VPN实例外的所有IS-IS进程的概要信息，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程的概要信息。

vpn6-instance vpn6-instance-name：显示指定IPv6 VPN的IS-IS多实例进程的概要信息。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：显示除IPv6 VPN实例外的所有IS-IS进程的概要信息，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程的概要信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display isis brief和display isis brief vpn-instance default vpn6-instance default命令的作用相同，都可以显示所有公网IS-IS进程的概要信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

项目

描述

ISIS Protocol Information for ISIS(3)

3：IS-IS进程号

SystemId

当前IS-IS路由器的主机系统ID

System Level
接口的Level级别：

  • L1

  • L2

  • L12

Area-Authentication-mode

区域的认证模式

Domain-Authentication-mode

域的认证模式

Ipv6 is not enabled

本IS-IS进程没有使能IPv6

ISIS is in starting status

IS-IS正处于运行状态

ISIS is in protocol hot standby state: Real-Time Backup

IS-IS热备份状态是实时备份

Interface

使能了IS-IS的接口类型、接口号和接口地址

Cost

接口IPv4的开销值

Ipv6 Cost

接口IPv6的开销值

State

使能了IS-IS的接口状态

Type

接口的类型，可以是P2P、Broadcast、NBMA、PTMP、Vlink

MTU

使能了IS-IS的接口的MTU值

Priority
使能了IS-IS的接口的协议优先级
说明：

当前接口为虚链路的接口时，不显示优先级字段

Timers

IS-IS定时器

Csnp

CSNP报文定时器的时间间隔

Retransmit

LSP在点到点链路上的重传间隔

Hello

Hello定时器的时间间隔

Hello Multiplier

用于判断邻居是否失效的Hello报文的数目

LSP-Throttle Timer

发送LSP的最小间隔时间
