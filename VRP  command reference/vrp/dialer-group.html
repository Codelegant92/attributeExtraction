dialer-group

命令功能

dialer-group命令用来对通过DCC接口发送的报文进行访问控制，将接口置于一个拨号访问组（dialer access group）中。

undo dialer-group命令用来将接口从此拨号访问组中删除。

缺省不配置此命令。

命令格式

dialer-group group-number

undo dialer-group

参数说明

group-number：接口属于的拨号访问组的序号，这个拨号访问组由dialer-rule命令设定。取值范围是1～255。

视图

接口视图、Dialer接口视图

缺省级别

2：配置级

使用指南

在配置该命令前，必须先执行命令dialer user设置对端用户名，或执行命令dialer enable-circular使能轮询DCC。

此命令将接口置于一个拨号访问组中，通过dialer-rule命令将拨号访问组与acl命令对应起来。一个DCC接口只能属于一个dialer group，若配置第二次，则覆盖第一次的配置。

说明：

接口的缺省配置中，dialer-group是未配置的。用户必须配置此命令，否则DCC将无法发送报文。

使用实例

# 将接口Serial1/0/0置入dialer-group 1。

<Quidway> system-view
[Quidway] dialer-rule 1 acl 3001
[Quidway] interface serial1/0/0
[Quidway-Serial1/0/0] dialer-group 1
