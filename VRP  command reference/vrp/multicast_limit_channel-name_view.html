multicast limit（channel-name视图）

命令功能

multicast limit命令用来配置当前节目组的全局PIM表项限制和预警值。

undo multicast limit命令用来删除当前节目组的全局PIM表项限制和预警值。

命令格式

multicast limit limit [ threshold threshold ]

undo multicast limit

参数说明

limit：指定PIM路由表中与当前节目组相关的最大表项个数。整数形式，取值范围是1～16384。

threshold threshold：指定预警值。整数形式，取值范围是1～16383。预警值必须小于limit。

视图

channel-name视图

缺省级别

2：配置级

使用指南

如果指定threshold，则当PIM路由表项计数超出threshold时系统会在日志中输出预警信息。

使用实例

# 配置PIM路由表中与节目组SDTV相关的最大PIM路由表项个数是100，预警值是50。

<Quidway> system-view
[Quidway] multicast-channel
[Quidway-multicast-channel] channel SDTV type asm
[Quidway-multicast-channel-SDTV] multicast limit 100 threshold 50
