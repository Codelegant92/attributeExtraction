rpr protection timer atd-timeout

命令功能

rpr protection timer atd-timeout命令用来配置本地节点发送ATD帧的间隔时间。

undo rpr protection timer atd-timeout命令用来恢复缺省值。

缺省情况下，本地节点发送ATD的间隔时间为1秒。

命令格式

rpr protection timer atd-timeout value

undo rpr protection timer atd-timeout

参数说明

value：时间值。整数形式，取值范围是1～10，单位是秒。缺省值为1秒。

RPR逻辑接口视图

缺省级别

2：配置级

使用指南

一般可以不配置而直接采用默认值。

使用实例

# 配置本地节点发送ATD帧的间隔时间为3秒。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr protection timer atd-timeout 3
