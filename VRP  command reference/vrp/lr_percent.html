lr percent

命令功能

lr percent命令用来启动限速。

undo lr命令用来取消限速。

缺省情况下，不启动限速。

命令格式

lr percent cir bandwidth-percent [ cbs cbs-interval [ ebs ebs-interval ] ]

undo lr

参数说明

cir bandwidth-percent：带宽的百分比。承诺信息速率的百分比形式，取值范围是1～100。

cbs cbs-interval：C桶的填充时间间隔。承诺突发尺寸等于承诺信息速率与该时间间隔的乘积。取值范围是50～2000，单位是ms。缺省取值是500ms。

ebs ebs-interval：E桶的填充时间间隔。过度突发尺寸等于承诺信息速率与该时间间隔的乘积。取值范围是0～2000，单位是ms。缺省取值是0ms，即只采用一个令牌桶监管。

使用指南

限速（LR，Line Rate）同GTS的区别仅仅在于GTS工作于IP层，LR工作于链路层。

lr percent命令只在NE16E/08E/05路由器上可用。

使用实例

# 为行为配置使用限速（LR），正常流量为50%的接口带宽，在第一时间可以有200毫秒×50％接口带宽的突发流量通过，以后速率小于等于50%的接口带宽时正常发送，速率大于50%的接口带宽时，将进入队列缓存。
