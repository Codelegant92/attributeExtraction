peer log-change

命令功能

peer log-change命令用来使能BGP记录指定对等体/对等体组会话状态和事件信息的功能。

undo peer log-change命令用来取消该功能。

缺省情况下，使能该功能。

命令格式

peer { group-name | ipv4-address | ipv6-address } log-change

undo peer { group-name | ipv4-address | ipv6-address } log-change

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP视图和BGP-VPN6实例视图下生效。

视图

BGP视图、BGP-VPN实例视图、BGP-VPN6实例视图

缺省级别

2：配置级

使用指南

无

使用实例

# 记录来自对等体1.1.1.2的状态和事件信息。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200
[Quidway-bgp] peer 1.1.1.2 log-change
