ip userlog syslog

命令功能

ip userlog syslog命令用来配置用户日志以系统日志的方式输出。

undo ip userlog syslog命令用来切换到UDP报文的输出方式。

缺省情况下，用户日志以UDP报文的形式发到外部用户日志服务器。

命令格式

ip userlog nat syslog

undo ip userlog nat syslog

参数说明

nat：NAT日志功能。

syslog：将日志信息以syslog的方式输出。

使用指南

以syslog方式输出时，日志信息的优先级为informational，即一般提示信息。

建议采用缺省配置，syslog方式可用于日志量比较小的情况。

使用实例

# 设置NAT日志的输出方式为syslog。

<Quidway> system-view
[Quidway] ip userlog nat syslog
