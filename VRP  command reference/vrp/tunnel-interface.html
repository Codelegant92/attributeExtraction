tunnel-interface

命令功能

tunnel-interface命令用来配置和修改MAC Tunnel使用的接口。

undo tunnel-interface命令用来删除MAC Tunnel的接口。

命令格式

tunnel-interface interface-type interface-number

undo tunnel-interface

参数说明

interface-type interface-number：指定MAC Tunnel使用的接口。

其中：

  • interface-type表示接口的类型，包括以太网接口和Eth-Trunk接口。

  • interface-number表示接口的编号。

使用指南

在MAC Tunnel的每个端节点设备上都有一个入接口，这个接口同时也是反向MAC Tunnel的出接口。

如果已经配置了隧道的骨干VLAN，在执行tunnel-interface命令配置隧道接口时，该接口必须已经加入了隧道的骨干VLAN。

如果隧道已经绑定业务实例或加入保护组，则不能再修改或删除隧道的接口。

执行undo tunnel-interface命令后隧道的接口将被删除。如果隧道原先状态是Up，执行了该命令后，隧道状态将被置为Down。

使用实例

# 配置当前隧道的接口为GigabitEthernet 1/0/1。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] tunnel-interface gigabitethernet 1/0/1

# 配置当前隧道的接口，接口名为Eth-Trunk1。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] tunnel-interface eth-trunk 1
