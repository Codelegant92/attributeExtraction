router-id (BGP)

命令功能

router-id命令用来为路由器指定ID。每个ID在自治系统AS中只能标识一个路由器。

undo router-id命令用来删除配置的ID。

命令格式

router-id ipv4-address

undo router-id

参数说明

ipv4-address：指定路由器ID。用IPv4地址的形式标识。

视图

BGP视图

缺省级别

2：配置级

使用指南

改变路由器ID的配置或删除已配置的ID时，BGP会话将重置，请确认后再执行此命令。

使用实例

# 指定路由器ID。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] router-id 1.1.1.1
Warning: Changing the parameter in this command resets the peer session. Continue?(Y/N):y

# 删除路由器ID。

[Quidway-bgp] undo router-id
Warning: Changing the parameter in this command resets the peer session. Continue?(Y/N):y
