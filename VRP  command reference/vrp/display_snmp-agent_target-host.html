display snmp-agent target-host

命令功能

display snmp-agent target-host命令用于显示所有目标主机的信息。包括目标主机的IP地址、接收告警的端口号、所在VPN实例的名字、告警的发送方式、告警发送使用的安全名字符串、协议版本以及安全级别。

命令格式

display snmp-agent target-host

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

可以使用该命令查看当前所有目标主机的配置信息。系统现在最多支持保存20条目标主机信息，因此最多可显示20个目标主机信息。

使用实例

# 显示系统中所有告警主机的配置信息。

<Quidway> display snmp-agent target-host
  Target-host number: 4
  Level: (A)authentication; (P)privacy; (N)no authentication and privacy
  No. IP-address       VPN instance/Security name        Port Type   Ver. Level
  ------------------------------------------------------------------------------
  1   100.1.1.48       -/NMS1                            162  inform v2c  N
  2   1.1.1.2          -/NMS2                            162  inform v3   N
  3   1.1.1.1          -/a                               162  inform v3   N
  4   100.1.1.48       -/a                               162  trap   v1   N
表1 display snmp-agent target-host命令输出信息描述

项目

描述

IP-address

告警主机的IP地址。

VPN instance/Security name

VPN实例名/安全用户名。

Port

发送请求的UDP端口号。

Type

请求类型，Trap或者Inform。

Ver

SNMP协议版本号。

Level
安全用户名级别。可以有以下级别:
  • (A)authentication：对报文进行认证但不加密。
  • (P)privacy：对报文进行认证并且加密。
  • (N)no authentication and privacy：对报文既不认证也不加密。
