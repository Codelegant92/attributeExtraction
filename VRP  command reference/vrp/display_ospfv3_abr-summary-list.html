display ospfv3 abr-summary-list

命令功能

display OSPFv3 abr-summary-list命令用来查看OSPFv3的引入路由聚合信息。

如果不指定IP地址和掩码，将显示所有引入路由聚合信息。

命令格式

display ospfv3 [ process-id ] abr-summary-list [ ipv6-address prefix-length ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

ipv6-address：匹配的IPv6地址。

prefix-length：指定前缀长度。整数形式，取值范围是1～128。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看OSPFv3的所有引入路由聚合信息。

<Quidway> display OSPFv3 abr-summary-list

3001::                                  16          0[NotActive]   Advertise
