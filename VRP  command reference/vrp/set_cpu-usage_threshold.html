set cpu-usage threshold

命令功能

set cpu-usage threshold threshold-value命令用来设置主控板的CPU占用率阈值；

set cpu-usage threshold threshold-value [ slot slot-id ]命令用来设置指定槽号的接口板的CPU占用率阈值。

undo set cpu-usage threshold命令用来恢复缺省设置。

缺省情况下，CPU占用率阈值是95%。

命令格式

set cpu-usage threshold threshold-value [ slave | slot slot-id ]

undo set cpu-usage threshold

参数说明

threshold-value：CPU占用率阈值。整数形式，取值范围是1～100。

slot-id：指定接口板所在的槽位。

视图

系统视图

缺省级别

2：配置级

使用指南

当CPU占用率超过阈值时，记录日志。此时当CPU的占用率下降幅度在5%的范围内并且再次超过阈值时，不会记录日志。直到CPU的占用率下降幅度超出5%的范围并重新达到阈值时，才记录日志。通过查看日志信息，方便用户掌握CPU的使用情况。

使用实例

# 设置主控板CPU占用阈值为80％。

[Quidway] set cpu-usage threshold 80

# 设置接口板CPU占用阈值为80％。

[Quidway] set cpu-usage threshold 80 slot 2
