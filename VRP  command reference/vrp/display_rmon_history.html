display rmon history

命令功能

display rmon history命令用来查看RMON控制表配置信息以及最近一次历史采样信息，包括以太网接口利用率、校验错误包数以及总包数等。

命令格式

display rmon history [ ethernet interface-number | gigabitethernet interface-number ]

参数说明

ethernet：以太网接口。

gigabitethernet：千兆以太网接口。

interface-number：接口号。
