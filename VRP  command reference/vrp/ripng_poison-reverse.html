ripng poison-reverse

命令功能

ripng poison-reverse命令用来配置对RIPng路由更新报文进行毒性反转。

undo ripng poison-reverse命令用来关闭毒性反转进程。

缺省情况下，不启用毒性反转功能。

命令格式

ripng poison-reverse

undo ripng poison-reverse

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

当毒性反转被使能时，从一个接口学到的路由还可以从这个接口向外发布，但metric值必须为16。在毒性反转和水平分割都配置的情况下，采用毒性反转。

使用实例

# 配置对RIPng更新报文进行毒性反转。

[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ripng poison-reverse

# 关闭RIPng更新报文的毒性反转。

[Quidway-GigabitEthernet1/0/0] undo ripng poison-reverse
