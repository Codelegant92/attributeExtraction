display mac-address aging-time

命令功能

display mac-address aging-time命令用来查看地址表老化时间。

命令格式

display mac-address aging-time [ vlan ] [ vlanid ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
