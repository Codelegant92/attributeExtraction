display diagnostic-information

命令功能

display diagnostic-information命令用来查看系统当前各模块的运行信息。

命令格式

display diagnostic-information [ file-name ]

参数说明

file-name：存储诊断信息的文件名。字符串形式，取值范围是5～64。

视图

所有视图

缺省级别

1：监控级

使用指南

display diagnostic-information命令集合了多条常用display命令的输出信息，包括display clock、display version、display current-configuration等等，可以看作是对系统常用display命令的批量执行。

当系统出现故障时，如果难以判断产生故障的模块，可以使用display diagnostic-information命令收集信息，用于故障定位。

说明：

由于不同设备的故障诊断重点不同，因此不同设备上display diagnostic-information命令所包含的具体display命令也有所差异。

使用实例

# 显示诊断信息。

<Quidway> display diagnostic-information
