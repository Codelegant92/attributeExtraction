tftp-server acl

命令功能

tftp-server acl命令用来设置本端设备可以访问的TFTP服务器，该配置通过为本端设备指定访问控制列表号来实现。

undo tftp-server acl命令用来取消本端设备可以访问的TFTP服务器的访问控制。

命令格式

tftp-server acl acl-number

undo tftp-server acl

tftp-server ipv6 acl acl6-number

undo tftp-server ipv6 acl

参数说明

acl-number：基本访问控制列表编号，整数形式，取值范围2000～2999。

acl6-number：（当服务器端是IPv6地址时）基本访问控制列表编号，整数形式，取值范围2000～2999。

使用指南

路由器作为TFTP客户端时，可以在路由器上配置访问控制列表，控制允许本设备以TFTP方式登录到哪些TFTP服务器。

使用实例

# 设置本端设备可以访问满足ACL 2000匹配条件的TFTP服务器。

[Quidway] tftp-server acl 2000
