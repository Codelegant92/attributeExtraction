display ip routing-table statistics

命令功能

display ip routing-table statistics命令用来显示路由表中的综合路由统计信息。

命令格式

display ip routing-table [ all-vpn-instance ] statistics [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all-vpn-instance：显示所有VPN实例的路由表中的综合路由统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

综合路由统计信息包括可通过路由协议增加或删除的路由总数，以及已带删除标签但未删除的激活或未激活路由数。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示综合路由统计信息。

<Quidway> display ip routing-table statistics
Summary prefixes: 9

        routes  routes  routes     routes      routes

STATIC   3        3      21          18         18
RIP      0        0       0           0          0
OSPF     0        0       0           0          0
IS-IS    0        0       0           0          0
BGP      0        0       0           0          0
Total    9        9      63          54         54

项目

描述

Summary prefixes

当前路由表中路由前缀总数

Proto

路由协议

total routes

当前路由表中的路由总数

active routes

路由表中激活路由的数目

added routes

路由表中增加的路由数目（激活和未激活）

deleted routes

路由表中添加了删除标记的路由数目

freed routes

路由表中释放（永久删除）的路由数目
