display vsi pw out-interface

命令功能

display vsi pw out-interface命令用来显示VSI PW的出接口信息。

命令格式

display vsi pw out-interface [ vsi vsi-name ]

参数说明

vsi vsi-name：VSI名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

视图

所有视图

缺省级别

1：监控级

使用指南

display vsi [ name vsi-name ] [ verbose ]命令中只显示了VSI PW的隧道策略，没有显示PW的实际出接口的信息。通过此命令可以查看PW的实际出接口信息。此命令只支持显示Martini方式VPLS PW的出接口信息。

使用实例

# 显示VSI PW的出接口信息。

<Quidway> system-view
[Quidway]display vsi pw out-interface
