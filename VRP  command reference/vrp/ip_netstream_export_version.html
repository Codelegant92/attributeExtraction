ip netstream export version

命令功能

ip netstream export version命令用来配置原始流统计信息的输出报文版本号和自治系统选项。

undo ip netstream export version命令用来将版本号和自治系统选项设为缺省值。

缺省情况下，原始流统计信息的输出为版本5格式，自治系统选项为邻接自治系统peer-as。

命令格式

ip netstream export version version [ origin-as | peer-as ] [ bgp-nexthop ]

undo ip netstream export version

参数说明

version：指定NetStream 统计输出报文版本号。整数形式，取值为5或9。目前只支持版本5和版本9。

origin-as：自治系统选项，起始自治系统号作为给定IP地址的自治系统号。

peer-as：自治系统选项，邻接自治系统号作为给定IP地址的自治系统号。

bgp-nexthop：BGP下一跳，决定统计输出时是否携带BGP下一跳信息。版本5不支持BGP下一跳。
