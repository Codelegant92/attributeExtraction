display mpls te rrvpn-tunnel(qos)

命令功能

display mpls te rrvpn-tunnel命令用来查看为VPN分配资源的情况。

命令格式

display mpls te rrvpn-tunnel [ tunnel tunnel-number ] [ auto-bypass-tunnel tunnel-name ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

tunnel interface-number：指定隧道接口。如果指定隧道接口，只显示使用该隧道的VPN占用资源的情况。

auto-bypass-tunnel tunnel-name：指定auto-bypass隧道名称。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

配置资源隔离VPN后，可使用该命令验证配置结果。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

表1 display mpls te rrvpn-tunnel命令输出信息描述

项目

描述

Tunnel Name

隧道接口名称。

Interface Id

隧道接口的ID。

Status

第一个“Staus”表示主隧道的接口状态，其他的“Status”表示子隧道接口的接口状态。up：表示该隧道可用。down：表示该隧道不可用。

Token

该隧道的Token值。

L3RRVPN

使用该隧道的L3VPN的个数。

VLLRRVPN

使用该隧道的VLL VPN的个数。

VPLSRRVPN

使用该隧道的VPLS VPN的个数。

Vpn Name

VPN实例的名称。

Vpn Type

绑定MPLS TE隧道的VPN类型，包括 L3RRVPN,VLLRRVPN,VPLSRRVPN。

Sub Tunnel Id

主隧道的子隧道Tunnel ID。

Traff Behavior Name

应用于VPN的流行为名称。

Qos Policy

应用于VPN的QoS策略。

Qos Policy Value

如果Qos Policy为“Qos Local Id”，则Qos Policy Value指的是qos-local-id的值。

Cir Bandwidth

该VPN的承诺信息速率。

Pir Bandwidth

该VPN的突发信息速率。

Flow Queue Id

该VPN的流队列ID。

Group Queue Id

该VPN对应的Group Queue的ID。

Qos Token

子Tunnel的转发Token。

Backup Qos Token

子Tunnel的备份转发token。

Modify Cir Bandwidth

修改的承诺信息速率。

Modify Pir Bandwidth

修改的突发信息速率。

Modify Flow Queue Id

修改的流队列ID。

Modify Group Queue Id

修改的该VPN对应的Group Queue的ID。

Modify Qos Token

修改的子Tunnel的转发Token。
