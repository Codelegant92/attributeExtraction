dns resolve

命令功能

dns resolve命令用来使能基于DNS的动态域名解析功能。

undo dns resolve命令用来禁止动态域名解析功能。

缺省情况下，动态域名解析功能被禁止。

命令格式

dns resolve

undo dns resolve

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

由于动态域名解析可能会花费一定的时间，所以有些时候用户可能不希望进行动态域名解析。使用本命令可以实现对动态域名解析的使能或禁止。

使用实例

# 使能动态域名解析。

<Quidway> system-view
[Quidway] dns resolve
