timer hello（IPv4）

命令功能

timer hello命令用来配置路由器发送Hello消息的时间间隔。

undo pim timer hello命令用来恢复时间间隔为缺省值。

缺省情况下，路由器发送Hello消息的时间间隔是30秒。

命令格式

timer hello interval

undo timer hello

参数说明

interval：表示发送Hello消息的时间间隔。整数形式，取值范围是1～2147483647，单位是秒。

缺省级别

2：配置级

使用指南

该命令对于PIM-DM和PIM-SM都有效。

路由器发送Hello消息的时间间隔应该小于PIM邻居超时时间。使用hello-option holdtime命令可以配置PIM邻居超时时间。

使用实例

# 在公网实例中，配置发送Hello消息的时间间隔为40秒。

<Quidway> system-view
[Quidway] pim
[Quidway-pim] timer hello 40
