display pim rp-info

命令功能

display pim rp-info命令用来查看组播组对应的RP信息。

命令格式

display pim [ vpn-instance vpn-instance-name | all-instance ] rp-info [ group-address ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

group-address：表示组播组地址，用来指定一个组播组，显示该组对应的RP信息。点分十进制形式，取值范围是224.0.1.0～239.255.255.255。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display pim rp-info命令时，若不指定vpn-instance或all-instance，则只显示公网实例的相关信息。

RP信息包括BSR机制发现的RP、静态RP和Auto-RP的信息。

如果此命令不指定组地址，将显示所有组对应的RP信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看公网实例中所有组对应的RP信息。

<Quidway> display pim rp-info
