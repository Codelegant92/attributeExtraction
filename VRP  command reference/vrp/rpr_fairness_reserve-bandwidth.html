rpr fairness reserve-bandwidth

命令功能

rpr fairness reserve-bandwidth命令用来配置节点东向或西向的预留带宽。

undo fairness reserve-bandwidth命令用来恢复缺省值。

缺省情况下，预留带宽绝对值是0 Mbit/s。

命令格式

rpr fairness reserve-bandwidth abs-value { west | east }

undo rpr fairness reserve-bandwidth { west | east }

参数说明

abs-value：预留带宽的绝对值。整数形式，取值范围和具体产品有关。

west：本地节点的西向发送链路。

east：本地节点的东向发送链路。

缺省级别

2：配置级

使用指南

预留带宽用于保证某些帧的实时传输，只能应用于subclass A0子类。

使用实例

# 分配给本节点的西向链路2000Mbit/s的预留带宽。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway-ring-if1] rpr fairness reserve-bandwidth 2000 west
