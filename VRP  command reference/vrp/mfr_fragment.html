mfr fragment

命令功能

mfr fragment命令用来使能多链路帧中继捆绑的分片功能。

undo mfr fragment命令用来禁止此功能。

缺省情况下，多链路帧中继捆绑的分片功能是禁止的。

命令格式

mfr fragment

undo mfr fragment

参数说明

无

视图

MFR接口视图

缺省级别

2：配置级

使用指南

如果网络的数据流很大，可以设置MFR分片，降低传输时延。配置时要考虑与对端的匹配，尽量使两端的分片一致，以提高效率。

使用实例

# 对MFR接口1/0/123使能分片功能。

<Quidway> system-view
[Quidway] interface mfr 1/0/123
[Quidway-MFR1/0/123] mfr fragment
