snmp-agent trap enable feature-name isis

命令功能

snmp-agent trap enable feature-name isis命令用来打开ISIS模块的告警开关。

undo snmp-agent trap enable feature-name isis命令用来关闭ISIS模块的告警开关。

缺省情况下，ISIS模块的所有告警开关均处于关闭状态。

命令格式

snmp-agent trap enable feature-name isis { non-excessive all | trap-name { isisAdjacencyChange | isisAreaMismatch | isisAttemptToExceedMaxSequence | isisAuthenticationFailure | isisAuthenticationTypeFailure | isisCorruptedLSPDetected | isisDatabaseOverload | isisIDLenMismatch | isisManualAddressDrops | isisMaxAreaAddressesMismatch | isisOrigLSPBuffSizeMismatch | isisOwnLSPPurge | isisProtocolsSupportedMismatch | isisRejectedAdjacency | isisSequenceNumberSkip | isisVersionSkew } }

undo snmp-agent trap enable feature-name isis { non-excessive all | trap-name { isisAdjacencyChange | isisAreaMismatch | isisAttemptToExceedMaxSequence | isisAuthenticationFailure | isisAuthenticationTypeFailure | isisCorruptedLSPDetected | isisDatabaseOverload | isisIDLenMismatch | isisManualAddressDrops | isisMaxAreaAddressesMismatch | isisOrigLSPBuffSizeMismatch | isisOwnLSPPurge | isisProtocolsSupportedMismatch | isisRejectedAdjacency | isisSequenceNumberSkip | isisVersionSkew } }

参数说明

non-excessive all：打开IS-IS模块的所有非海量告警开关。包括isisAttemptToExceedMaxSequence、isisDatabaseOverload、isisManualAddressDrops、isisOwnLSPPurge和isisSequenceNumberSkip。

isisAdjacencyChange：打开邻居变化的告警开关。

isisAreaMismatch：打开收到的Level-1 Hello报文的区域地址与本地区域地址不匹配的告警开关。

isisAttemptToExceedMaxSequence：打开LSP序列号将达最大值的告警开关。

isisAuthenticationFailure：打开认证码不匹配的告警开关。

isisAuthenticationTypeFailure：打开认证类型不匹配的告警开关。

isisCorruptedLSPDetected：打开LSP校验和验证失败的告警开关。

isisDatabaseOverload：打开LSDB过载状态变化的告警开关。

isisIDLenMismatch：打开system ID长度不匹配的告警开关。

isisManualAddressDrops：打开Level-2区域地址发布时，不同区域地址个数超过最大值且有可能丢弃部分区域地址的告警开关。

isisMaxAreaAddressesMismatch：打开最大区域地址不匹配的告警开关。

isisOrigLSPBuffSizeMismatch：打开LSP缓冲区大小不匹配的告警开关。

isisOwnLSPPurge：打开IS感知自己的LSP在网络中被清除的告警开关。

isisProtocolsSupportedMismatch：打开所支持的协议类型不匹配的告警开关。

isisRejectedAdjacency：打开IS-IS收到邻居Hello报文但无法建立邻接关系的告警开关。

isisSequenceNumberSkip：打开收到自己发出的LSP序列号比LSDB中该LSP序列号大的告警开关。

isisVersionSkew：打开收到Hello报文版本号错误的告警开关。

使用指南

如果要打开ISIS模块所有事件的非海量告警开关，则选择non-excessive all批量处理。

如果只打开某个或几个事件的告警开关时，则选择trap-name。

用display snmp-agent trap feature-name isis all命令可以查看ISIS模块的所有告警信息。

目前定义的ISIS模块的非海量告警包括：isisAttemptToExceedMaxSequence、isisDatabaseOverload、isisManualAddressDrops、isisOwnLSPPurge和isisSequenceNumberSkip。

说明：

如果在较低版本上打开了IS-IS告警开关，升级到该版本后，需要手工配置该命令重新打开告警开关，否则原有配置会丢失。

使用实例

# 打开IS-IS邻居变化事件的告警开关。

<Quidway> system-view
[Quidway] snmp-agent trap enable feature-name isis trap-name isisAdjacencyChange
