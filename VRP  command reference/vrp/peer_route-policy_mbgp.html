peer route-policy（MBGP）

命令功能

peer route-policy命令用来为指定对等体或对等体组配置基于route-policy的路由策略。

undo peer route-policy命令用来删除指定的Route-Policy。

缺省情况下，不指定对等体或对等体组的路由策略。

命令格式

peer { group-name | peer-address } route-policy route-policy-name { import | export }

undo peer { group-name | peer-address } route-policy route-policy-name { import | export }

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

route-policy route-policy-name：指定的路由策略。

import：对从对等体或对等体组收到的路由应用路由策略。

export：对向对等体或对等体组发布的路由应用路由策略。

缺省级别

2：配置级

使用指南

可以根据Route-Policy为对等体灵活的配置策略。

命令peer route-policy中所应用的Route-Policy不支持将特定接口作为匹配条件，即在Route-Policy中配置if-match interface命令不能生效。

使用实例

# 对来自对等体组test的路由应用名为policy1的路由策略。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] peer test enable
[Quidway-bgp-af-multicast] peer test route-policy policy1 import
