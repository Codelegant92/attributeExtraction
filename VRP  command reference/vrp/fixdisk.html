fixdisk

命令功能

fixdisk命令用来对文件系统异常的存储设备进行修复，不能确保成功。

命令格式

fixdisk device-name

参数说明

device-name：设备名称。

视图

用户视图

缺省级别

3：管理级

使用指南

当终端显示系统出错时，可以使用此命令进行修复。

使用实例

# 终端显示存储设备flash出错，进行修复。

Lost chains in flash detected, please use fixdisk to recover them!
<Quidway> fixdisk flash:
Fixdisk flash: will take long time if needed.
Fixdisk flash: completed.
