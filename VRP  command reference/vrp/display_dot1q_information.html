display dot1q information

命令功能

display dot1q information termination命令用来查看配置了dot1q终结的所有接口的名称以及这些接口下QinQ和VLAN-Group的数量。

命令格式

display dot1q information termination [ interface interface-type interface-number [.subinterface-number ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：指定接口的类型和编号。

subinterface-number：指定子接口的编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

配置完dot1q终结后，可以使用display dot1q information termination命令来查看哪些接口配置了dot1q终结，以及dot1q终结的相关配置。

使用实例

# 查询配置了Dot1q终结的所有子接口。

<Quidway> display dot1q information termination
Eth-Trunk1.1
    qinq 8021p-mode trust pe-vid-8021p
    Total QINQ Num: 4
      dot1q  termination vid 1 to 100 vlan-group 1
      dot1q  termination vid 101 to 200 vlan-group 2
      dot1q  termination vid 301 to 400 vlan-group 2
      dot1q  termination vid 201 to 300 vlan-group 3
    Total vlan-group: 3
      vlan-group 1
      vlan-group 2
      vlan-group 3
    control-vid 1 dot1q-termination
Eth-Trunk1.2
    qinq 8021p-mode trust pe-vid-8021p
    Total QINQ Num: 1
      dot1q  termination vid 401 to 500 vlan-group 4
    Total vlan-group: 1
      vlan-group 4
    control-vid 2 dot1q-termination rt-protocol
表1 display dot1q information命令输出信息描述

项目

描述

Eth-Trunk1.1

终结子接口的名称。

VSI bound

终结子接口绑定的业务类型，可以是L3VPN bound、VLL/PWE3 bound或VSI bound。如果没有绑定任何业务，则不显示此项内容。

Total QinQ Num

终结子接口对用户报文终结的规则数量。

Total vlan-group

终结子接口下配置的用户VLAN组的数量。
说明：

除表中所列内容外，其他显示信息均为终结子接口的配置内容，与具体配置有关。
