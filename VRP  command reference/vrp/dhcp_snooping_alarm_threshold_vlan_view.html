dhcp snooping alarm threshold（VLAN视图）

命令功能

dhcp snooping alarm threshold命令用来配置VLAN丢弃的ARP报文、IP报文、DHCP Reply报文（Untrusted端口收到的）和DHCP Request报文数量的告警阈值以及用户数达到最大用户数限制的百分比告警阈值。

undo dhcp snooping alarm threshold命令用来恢复告警阈值的缺省值。

缺省情况下：

  • VLAN丢弃报文的告警阈值是全局模式下的告警阈值（全局模式下，默认的告警阈值为100，也可手动配置），单位是报文的个数。

  • VLAN内用户数达到最大用户数的告警阈值是100，单位是“％”。

命令格式

dhcp snooping alarm { arp | ip | dhcp-request | dhcp-chaddr | dhcp- reply | user-limit } threshold threshold [ interface interface-type interface-number ]

undo dhcp snooping alarm { arp | ip | dhcp-request | dhcp-chaddr | dhcp- reply | user-limit } threshold [ interface interface-type interface-number ]

参数说明

arp：丢弃ARP报文的告警阈值。

ip：丢弃IP报文的告警阈值。

dhcp-reply：丢弃untrusted接口上收到的DHCP应答报文（包括Offer、ACK和NAK报文）的告警阈值。

dhcp-chaddr：丢弃DHCP报文中CHADDR字段与以太帧头中的源 MAC不匹配报文的告警阈值。

dhcp-request：丢弃DHCP Request报文的告警阈值。

user-limit：用户数达到最大用户数限制的告警阈值。

threshold：告警阈值，VLAN丢弃报文的告警阈值取值范围是1~1000，缺省为100；DHCP Snooping用户数达到最大用户数限制的百分比告警阈值取值范围是1~100，缺省为100。

interface-type interface-number：接口类型、接口号。此参数仅在VLAN视图下才可配置。

使用指南

在VLAN视图下执行此命令表示仅配置该VLAN丢弃的报文的告警阈值以及用户数达到最大用户数限制的百分比告警阈值。

VLAN丢弃的报文告警阈值,有以下两种情况：

  • 如果没有配置VLAN的告警阈值，那么VLAN的告警阈值就是全局的默认值，可以通过配置全局的告警阈值来修改默认值。

  • 如果VLAN上配置了告警阈值，那么以VLAN上配置的告警阈值为准。

在VLAN视图下配置用户数达到最大用户数限制的百分比告警阈值前，必须先执行以下命令：

  • 在系统视图下，执行dhcp snooping enable命令，使能全局的DHCP Snooping功能。

  • 在VLAN视图下，执行dhcp snooping max-user-number（VLAN视图）命令，配置用户数限制功能。

  • 在VLAN视图下，执行dhcp snooping alarm命令，使能DHCP Snooping告警功能。

若配置了用户数最大限制数量为n，用户数达到最大用户数限制的百分比为m，那么当该VLAN下用户数达到n×m，将产生告警。但是当用户数达到或超过(n×m)＋1都不会再告警。只有用户租期到期或用户主动释放IP地址，VLAN下的用户数会下降到n×m以下。当用户数再一次达到n×m，这时将会产生告警。

当用户需要修改已经配置的百分比告警阈值，重新配置即可，只有当前配置生效。

说明：
如果当前VLAN是RRPP的控制VLAN，则无法配置最大用户数百分比告警阈值。

使用实例

# 配置VLAN 100丢弃ARP报文的告警阈值为200。

<Quidway> system-view
[Quidway] vlan 100
[Quidway-Vlan100] dhcp snooping alarm arp threshold 200
# 配置VLAN100内用户数达到最大用户数限制的百分比告警阈值为50％。
<Quidway> system-view
[Quidway] dhcp snooping enable
[Quidway] vlan 100
[Quidway-vlan100] dhcp snooping max-user-number 3000
[Quidway-vlan100] dhcp snooping alarm user-limit enable
[Quidway-vlan100] dhcp snooping alarm user-limit threshold 50
# 配置VLAN 100内接口GE1/0/1上用户数达到最大用户数限制的百分比告警阈值为50%。
<Quidway> system-view
[Quidway] dhcp snooping enable
[Quidway] vlan 100
[Quidway-vlan100] dhcp snooping max-user-number 3000 interface gigabitethernet 1/0/1
[Quidway-vlan100] dhcp snooping alarm user-limit enable
[Quidway-vlan100] dhcp snooping alarm user-limit threshold 50 interface gigabitethernet 1/0/1
