template

命令功能

template命令用来定义聚合流信息以版本9输出时，对应的模板刷新参数。

undo template命令用来恢复模板刷新参数的缺省值。

缺省情况下，模板刷新的报文间隔数为20个，时间间隔数为30分钟。

报文间隔数与时间间隔数可以同时配置。

命令格式

template { refresh-rate packet-interval | timeout-rate timeout-interval }

undo template { refresh-rate | timeout-rate }

参数说明

packet-interval：指定模板刷新的报文间隔数。整数形式，取值范围1～600。缺省值是20。

timeout-interval：指定模板刷新的时间间隔。整数形式，单位是分钟，取值范围1～3600分钟。缺省值是30分钟。
