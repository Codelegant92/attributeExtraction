display ip host

命令功能

display ip host命令用来查看主机名和IP地址的对应关系。

命令格式

display ip host [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当配置静态DNS表项后，可以使用命令display ip host查看主机名和IP地址是否已经正确匹配。

使用实例

# 查看系统配置的主机名与IP地址的对应关系。

<Quidway> display ip host
Host          Age       Flags           Address
hw            0         static          10.1.1.1
gww           0         static          192.168.1.1
表1 display ip host命令输入信息描述

项目

描述

Host

主机名称。

Age

老化时间。0表示是静态域名，不老化。

Flags

标识。Static表示是静态域名。

Address

表示主机名对应的IP地址。
