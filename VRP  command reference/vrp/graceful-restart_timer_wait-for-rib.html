graceful-restart timer wait-for-rib

命令功能

graceful-restart timer wait-for-rib命令用来设置等待End-Of-RIB标记的时间。

undo graceful-restart timer wait-for-rib命令用来删除设置的时间。

命令格式

graceful-restart timer wait-for-rib time

undo graceful-restart timer wait-for-rib

参数说明

time：等待End-Of-RIB标记的时间。整数形式，单位是秒，取值范围是3～3000，缺省值为600秒。

视图

BGP视图

缺省级别

2：配置级

使用指南

当建立或重建BGP会话时，应在此命令设置的时间内收到End-Of-RIB标记，如果收不到End-Of-RIB标记，确保能够退出GR过程。

说明：

只有先使能了Graceful Restart特性后，才能配置该命令。

使用实例

# 设置等待End-Of-RIB标记的时间为100秒。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] graceful-restart timer wait-for-rib 100
