reset mpls ldp

命令功能

reset mpls ldp命令用来重启全局LDP能力，即重启LDP公网实例。

reset mpls ldp vpn-instance命令用来重启指定LDP私网实例。

reset mpls ldp all命令用来重启所有LDP公网实例和私网实例。

命令格式

reset mpls ldp [ vpn-instance vpn-instance-name ] [ peer peer-id ]

reset mpls ldp all

参数说明

vpn-instance vpn-instance-name：VPN实例的名称，字符串形式，不支持空格，长度范围是1～31。

peer peer-id：对等体的LSR ID，点分十进制格式。

all：所有LDP实例。

使用指南

reset mpls ldp vpn-instance命令用来重启指定的LDP私网实例，被重启的LDP实例上存在的接口、对等体、会话、LSP和CR-LSP都将被删除重建。

reset mpls ldp peer命令将重启指定对等体的会话、LSP和CR-LSP。如果需要重启某个VPN实例的指定对等体，可以使用reset mpls ldp vpn-instance vpn-instance-name peer peer-id 命令。

说明：

GR期间禁止执行reset mpls ldp命令。

使用实例

# 重启全局LDP能力。

<Quidway> reset mpls ldp

# 重启所有LDP实例，包括公网LDP实例和私网LDP实例。

<Quidway> reset mpls ldp all

# 重启指定VPN实例的LDP。

<Quidway> reset mpls ldp vpn-instance vrf1

# 重启对等体。

<Quidway> reset mpls ldp peer 2.2.2.9

# 重启VPN实例vrf1的指定对等体。

<Quidway> reset mpls ldp vpn-instance vrf1 peer 3.3.3.9
