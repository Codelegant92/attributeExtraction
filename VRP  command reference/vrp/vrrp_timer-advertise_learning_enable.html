vrrp timer-advertise learning enable

命令功能

vrrp timer-advertise learning enable命令用来使能VRRP协议报文接收时间间隔学习功能。

undo vrrp timer-advertise learning enable命令用来关闭VRRP协议报文接收时间间隔学习功能。

命令格式

vrrp timer-advertise learning enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

使能VRRP协议报文时间间隔学习功能后，设备上非Master状态的VRRP会进行协议报文接收时间间隔学习，调整自己定时器与对应Master的VRRP保持同步。

如果关闭学习功能时设备已经使能整机VRRP平滑倒换功能，则执行此命令时整机VRRP平滑倒换功能会同时被关闭。

使用实例

# 使能VRRP协议报文时间间隔学习功能。

<Quidway> system-view
[Quidway] vrrp timer-advertise learning enable
Error: The specific VRRP timer-advertise learning function is already enabled.
