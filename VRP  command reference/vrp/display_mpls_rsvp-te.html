display mpls rsvp-te

命令功能

display mpls rsvp-te命令用来查看RSVP-TE配置信息。

命令格式

display mpls rsvp-te [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

LSR ID

LSR ID，格式为X.X.X.X

Resv Confirmation Request

是否启用预留确认

RSVP Hello Extension

是否启用RSVP的Hello扩展特性

Hello Interval

Hello定时器时长，单位为秒

Max Hello misses

最多可丢失的Hello报文数

Path and Resv message refresh interval

路径和预留消息的刷新定时器时间，单位为秒

Path and Resv message refresh retries count

路由和预留消息刷新重发的次数

Blockade Multiplier

阻塞状态保持倍数

Graceful-Restart Capability
表示设备的RSVP GR支持情况：

  • GR-Self GR-Support：具有自身RSVP GR能力，又有支持邻居RSVP GR的能力

  • GR-Self：只具有自身RSVP GR能力

  • GR-Support：只具有支持邻居RSVP GR的能力

Restart Time

GR开始时间，单位为毫秒

Recovery Time

恢复所有LSP所需时间，单位是毫秒

Bfd Enabled
MPLS视图下是否全局使能了BFD for RSVP：
  • ENABLE：MPLS视图下全局使能了BFD for RSVP
  • DISABLE：MPLS视图下未全局使能BFD for RSVP

Bfd Min-Tx

本地配置的BFD消息最小发送时间间隔，单位是毫秒

Bfd Min-Rx

本地配置的BFD消息最小接收时间间隔，单位是毫秒

Bfd Detect-Multi

本地BFD检测倍数
