reset dhcp server conflict

命令功能

reset dhcp server conflict命令用来释放DHCP地址池中的冲突地址。

命令格式

reset dhcp server conflict { all | ip ip-address }

参数说明

ip ip-address：指定冲突的IP地址。

all：释放DHCP地址池中所有的冲突地址。

视图

用户视图

缺省级别

2：配置级

使用指南

DHCP服务器通过检测到地址冲突，需要使用reset dhcp server conflict命令用来释放DHCP地址池中冲突的IP地址。

使用实例

# 释放DHCP地址池中所有的冲突地址。

<Quidway> reset dhcp server conflict all
