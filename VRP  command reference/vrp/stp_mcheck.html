stp mcheck

命令功能

stp mcheck命令用来对当前端口执行MCheck操作。

  • 在系统视图下执行stp mcheck命令，将会对交换机上所有端口执行MCheck操作。

  • 在Ethernet接口视图、GE接口视图、Eth-Trunk接口视图或VE视图下执行stp mcheck命令，将只对当前端口执行MCheck操作。

命令格式

stp mcheck

参数说明

无

视图

系统视图、Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

在运行MSTP的交换机上，如果某个端口和另一台运行STP的交换机连接，则该端口会自动迁移到STP兼容工作模式。

如果运行STP的交换机被关机或移走，该端口无法自动迁移回MSTP模式，此时需要在该端口上执行MCheck操作，将端口手动迁移到MSTP模式。

使用实例

# 对以太网接口GE 1/0/0执行MCheck操作，使它迁移到MSTP模式下工作。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] stp mcheck
