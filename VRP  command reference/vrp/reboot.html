reboot

命令功能

reboot命令用来重启路由器，重启前提示用户是否保存配置。

reboot fast命令用来快速重启路由器，重启前不提示用户是否保存配置。

命令格式

reboot [ fast ]

参数说明

无

视图

用户视图

缺省级别

3：管理级

使用指南

该命令功能与路由器下电再上电效果相同，但在对路由器远程维护时，不需用户到路由器所在地重启路由器，而直接在远地即可重启路由器。

注意：

在一般情况下，请勿使用该命令，因为它将导致网络工作在短时间内瘫痪，另外在重启路由器时，要确保路由器配置文件是否已经保存了。

说明：

如果用户不输入，则超时后会退回到用户视图，不重新启动。
