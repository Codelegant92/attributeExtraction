igmp-snooping static-router-port (VSI视图)

命令功能

igmp-snooping static-router-port remote-peer ip-address [ negotiation-vc-id vc-id ]命令用来将PW配置为VSI中的静态路由端口。

undo igmp-snooping static-router-port remote-peer ip-address [ negotiation-vc-id vc-id ]命令用来取消将PW配置为VSI中的静态路由端口。

命令格式

igmp-snooping static-router-port remote-peer ip-address [ negotiation-vc-id vc-id ]

undo igmp-snooping static-router-port remote-peer ip-address [ negotiation-vc-id vc-id ]

参数说明

remote-peer ip-address：VSI对端的IP地址。

negotiation-vc-id vc-id：虚电路的唯一标识，一般用于两端VSI ID不同但要求互通的情况。参数vc-id不能与本端其他VSI配置的VSI ID相同，也不能与本端其他negotiation-vc-id指定的vc-id相同，即vc-id是没有被使用的虚电路号，整数形式，取值范围是1～4294967295。

缺省级别

2：配置级

使用指南

当使用LDP作为信令协商协议时，使用igmp-snooping static-router-port remote-peer ip-address [ negotiation-vc-id vc-id] 命令将PW配置为VSI中的静态路由端口。如果事先没有配置remote-peer的IP地址，仍然允许PW配置为VSI中静态路由端口，只是此时不起作用，等到对应的remote-peer的IP地址配置好后，此命令才生效。

使用实例

# 当使用LDP作为信令协商协议时，将PW配置为VSI vsi0内的路由端口。

<Quidway> system-view
[Quidway] vsi vsi0
[Quidway-vsi-vsi0] igmp-snooping static-router-port remote-peer 3.3.3.3
