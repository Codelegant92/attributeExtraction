t1 set bert

命令功能

t1 set bert命令用来启动T1通道的比特错码率检测功能。

undo t1 set bert命令用来关闭T1通道的比特错码率检测功能。

缺省情况下，禁止比特错码率检测功能。

命令格式

t1 t1-number set bert { 2^11 | 2^15 | 2^20 } timer display interval-time

undo t1 t1-number set bert

参数说明

t1-number：T1通道号。整数形式，取值范围是1～28。

2^11：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为211，O.153或O.152模式。

2^15：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为215，O.151模式。

2^20：用于指定BERT的伪随机重复模式，使用该参数后，伪随机序列长度为220，O.153模式。

interval-time：指定BERT的持续时间。取值范围是1～4000，单位是分钟。

使用指南

只有当CT3接口工作在CT3工作模式下时，才能配置该命令。

T1支持比特错码率检测（BERT，Bit Error Rate Testing），如果启动了BERT，T1通道的链路状态将显示为Down，这是正常现象。

BERT功能的执行会中断正常业务，所以，建议在T1处于通道化模式，且没有配置任何时隙捆绑时才使用BERT功能，否则在关闭BERT功能后，还需要对T1重新配置时隙捆绑等。

使用实例

# 在CT3接口1/0/0的1号T1通道上执行BERT功能，采用伪随机序列长度215，显示周期是10分钟。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] t1 1 set bert 2^15 timer display 10
