pim

命令功能

pim命令用来进入PIM视图。

undo pim命令用来清除PIM视图下进行的配置。

命令格式

pim [ vpn-instance vpn-instance-name ]

undo pim [ vpn-instance vpn-instance-name ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例的名称。字符串形式，区分大小写。如果不指定此参数，则进入公网实例的PIM视图。

视图

系统视图

缺省级别

2：配置级

使用指南

与PIM相关的全局参数必须在公网实例PIM视图、VPN实例PIM视图下进行配置。

使用实例

# 进入公网实例PIM视图。

<Quidway> system-view
[Quidway] multicast routing-enable
[Quidway] pim
[Quidway-pim]

# 进入VPN实例mytest的PIM视图。

<Quidway> system-view
[Quidway] ip vpn-instance mytest
[Quidway-vpn-instance-mytest] route-distinguisher 100:1
[Quidway-vpn-instance-mytest] multicast routing-enable
[Quidway-vpn-instance-mytest] quit
[Quidway] pim vpn-instance mytest
[Quidway-pim-mytest]
