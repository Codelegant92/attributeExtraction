display ospfv3 interface

命令功能

display ospfv3 interface命令用来查看OSPFv3的接口信息。

命令格式

display ospfv3 interface [ interface-type interface-number ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-type interface-number：接口类型和接口号。如果不指定接口，则显示OSPFv3所有接口信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
