display mac-limit rule-name

命令功能

display mac-limit rule-name命令用来显示全局MAC地址学习限制规则详细信息。

命令格式

display mac-limit rule-name [ rule-name ]

参数说明

rule-name：全局MAC地址限制规则名。字符串形式，1～31字符。规则名不能有空格、问号、减号。

项目

描述

RuleName

全局MAC地址学习限制规则名。

Maximum

MAC地址学习的最大数量。

Rate(ms)

MAC地址的学习速率。

Action
当MAC地址达到限制后采取的动作。

  • discard：丢弃

  • forward：转发

Alarm
当MAC地址达到限制后是否进行提示。

  • enable：进行提示

  • disable：不进行提示

PortNum

应用本全局MAC地址学习限制规则的接口总数。

Port

被限制MAC地址学习的端口号。

Vlan

接口所属的VLAN ID。
