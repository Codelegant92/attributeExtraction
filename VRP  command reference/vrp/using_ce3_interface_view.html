using（CE3接口视图）

命令功能

using命令用来设置CE3接口的工作模式，创建不同带宽的同步串口。

undo using命令用来恢复缺省设置。

缺省情况下，CE3接口工作在通道化模式，需要对CE3接口下的E1通道进行配置，才能形成同步串口。

命令格式

using { e3 | ce3 }

undo using

参数说明

e3：工作模式为净通道模式。

ce3：工作模式为通道化模式。

视图

CE3接口视图

缺省级别

2：配置级

使用指南

CE3接口有两种工作模式，只有在CE3模式下才能对E1通道进行配置，具体配置方法请参见《VRP 配置指南-广域网接入》。

使用实例

# 设置CE3接口3/0/0接口工作在净通道模式。

<Quidway> system-view
[Quidway] controller e3 3/0/0
[Quidway-E3 3/0/0] using e3
