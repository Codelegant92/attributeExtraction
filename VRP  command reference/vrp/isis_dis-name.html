isis dis-name

命令功能

isis dis-name命令用来为DIS配置名称。

undo isis dis-name命令用来删除为DIS配置的名称。

命令格式

isis dis-name symbolic-name

undo isis dis-name

参数说明

symbolic-name：DIS的主机名称。字符串形式，长度范围是1～64。

视图

接口视图

缺省级别

2：配置级

使用指南

只有用isis enable命令使能了接口的IS-IS后，才能在接口上配置DIS名称，并以伪节点LSP报文的形式发布出去，从而使DIS名称与系统ID相关联。该配置仅在广播网的DIS上有意义。

说明：

如果通过isis circuit-type命令将广播接口模拟为P2P接口，则isis dis-name命令在该接口失效；如果通过undo isis circuit-type命令将该接口恢复为广播接口，则isis dis-name命令在该接口生效。

使用实例

# 配置DIS的主机名称。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] isis enable
[Quidway-GigabitEthernet1/0/0] isis dis-name LOCALAREA
