display l2-multicast forwarding-mode

命令功能

display l2-multicast forwarding-mode vlan [ vlan-id ] 命令用来查看VLAN的转发模式。

display l2-multicast forwarding-mode vsi [ vsi-name ] 命令用来查看VSI的转发模式。

命令格式

display l2-multicast forwarding-mode { vlan [ vlan-id ] | vsi [ vsi-name ] }

参数说明

vlan vlan-id：VLAN编号，取值范围是1～4094。

vsi vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

使用指南

使用命令display l2-multicast forwarding-mode vlan 不指定参数vlan-id时，显示所有VLAN的转发模式。

使用命令display l2-multicast forwarding-mode vsi 不指定参数vsi-name 时，显示所有VSI的转发模式。

使用实例

# 显示VLAN 10的转发模式

<Quidway>display l2-multicast forwarding-mode vlan 10
