controller cpos

命令功能

controller cpos命令用来进入CPOS的接口视图。

命令格式

controller cpos cpos-number

参数说明

cpos-number：CPOS的编号，包括槽号/卡号/端口号。

视图

系统视图

缺省级别

2：配置级

使用指南

如果需要对CPOS物理端口进行配置，则必须先进入CPOS接口视图，这与其它一些支持通道化的接口，如CE1、CE3和CT1是类似的。

使用实例

# 进入CPOS 1/0/0的接口视图。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0]
