clock（E3接口视图）

命令功能

clock命令用来设置E3接口的时钟模式。

undo clock命令用来恢复缺省设置。

缺省情况下，E3接口作为从时钟还是主时钟根据产品的不同而不同。

命令格式

clock { master | slave }

undo clock

参数说明

master：设置E3作为主时钟。

slave：设置E3作为从时钟。

E3接口视图

缺省级别

2：配置级

使用指南

当两台路由器的接口直接相连时，应设置一端作为主时钟，使用内部时钟信号，另一端作为从时钟，使用线路提供的时钟信号。

使用实例

# 设置E3接口1/0/0作为Master，使用内部时钟。

<Quidway> system-view
[Quidway] controller e3 1/0/0
[Quidway-E3 1/0/0] clock master
