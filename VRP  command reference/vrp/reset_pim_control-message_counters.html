reset pim control-message counters

命令功能

reset pim control-message counters命令用来清除PIM控制报文统计信息。

命令格式

reset pim [ vpn-instance vpn-instance-name | all-instance ] control-message counters [ interface interface-type interface-number ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称，字符串形式，长度范围是1～31。

all-instance：指定所有实例。

interface interface-type interface-number：指定接口类型和接口号，清除指定接口上的PIM控制报文统计信息。

视图

用户视图

缺省级别

1：监控级

使用指南

使用reset pim control-message counters命令：

  • 如果不指定vpn-instance或all-instance，则只清除公网实例中的统计信息。

  • 如果不指定路由器的interface，将清除路由器所选实例下所有接口上的PIM控制报文统计信息。清除PIM控制报文统计信息后，不影响PIM的正常运行。

使用实例

# 清除公网实例所有接口上的PIM控制报文统计信息。

<Quidway> reset pim control-message counters

# 清除公网实例接口GE1/0/0上的PIM控制报文统计信息。

<Quidway> reset pim control-message counters interface gigabitEthernet 1/0/0
