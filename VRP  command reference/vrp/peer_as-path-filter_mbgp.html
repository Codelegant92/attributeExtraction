peer as-path-filter（MBGP）

命令功能

peer as-path-filter命令用来为对等体或对等体组设置基于AS路径列表的MBGP路由过滤策略。

undo peer as-path-filter命令用来取消已有的配置。

缺省情况下，对等体或对等体组没有配置基于AS路径列表的路由过滤策略。

命令格式

peer { group-name | peer-address } as-path-filter filter-number { import | export }

undo peer { group-name | peer-address } as-path-filter filter-number { import | export }

参数说明

group-name：指定MBGP对等体组的名称。

peer-address：指定MBGP对等体的地址。点分十进制形式。

as-path-filter filter-number：指定AS路径过滤列表号。filter-number是AS路径过滤列表号。

import：对收到的路由应用路由策略。

export：对发布的路由应用路由策略。

# 设置对等体组test基于AS路径列表的MBGP路由过滤策略。

[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] peer test enable
