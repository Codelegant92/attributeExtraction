mpls te signal-protocol

命令功能

mpls te signal-protocol命令用来设置MPLS TE建立隧道使用的信令协议。

缺省情况下，使用RSVP-TE。

命令格式

mpls te signal-protocol { cr-static | rsvp-te | static }

参数说明

cr-static：使用static CR-LSP建立TE隧道。

rsvp-te：使用RSVP-TE做为信令协议。

static：使用static LSP建立TE隧道。

缺省级别

2：配置级

无

使用实例

# 设置使用RSVP-TE做为MPLS-TE的信令协议。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol mpls te
[Quidway-Tunnel1/0/0] mpls te signal-protocol rsvp-te
