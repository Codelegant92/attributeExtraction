igmp-snooping enable

命令功能

igmp-snooping enable命令用来使能IGMP Snooping功能。

undo igmp-snooping enable命令用来禁止IGMP Snooping功能。

缺省情况下，IGMP Snooping功能处于禁止状态。

命令格式

igmp-snooping enable

undo igmp-snooping enable

参数说明

无

视图

系统视图、VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

只有当全局使能IGMP Snooping后，才能配置VLAN/VSI的IGMP Snooping使能。

在系统视图下和VLAN/VSI视图下使用此命令的区别如下：

  • 在系统视图下执行此命令，将使能或禁止路由器的IGMP Snooping功能。如果需要使用IGMP Snooping功能，必须首先在系统视图下使用igmp-snooping enable命令使能路由器的IGMP Snooping功能，否则其它和IGMP Snooping功能相关的命令不可用。

  • 如果在系统视图下使用undo igmp-snooping enable命令，将关闭整个路由器的IGMP Snooping功能。

  • 在VLAN/VSI视图下执行此命令，将使能或禁止该VLAN/VSI的IGMP Snooping功能。在系统视图下使能路由器的IGMP Snooping功能后，缺省情况下VLAN/VSI的IGMP Snooping功能仍处于禁止状态。要使能某个VLAN/VSI的IGMP Snooping功能，还需要在该VLAN/VSI视图下执行igmp-snooping enable命令。

  • 如果在VLAN/VSI视图下使用undo igmp-snooping enable命令，将关闭整个VLAN/VSI内的IGMP Snooping功能。

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定了VSI；

  • VLAN内包含VE接口。

使用实例

# 使能路由器的IGMP Snooping功能。

<Quidway> system-view
[Quidway] igmp-snooping enable

# 使能VLAN 2的IGMP Snooping功能。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] igmp-snooping enable
