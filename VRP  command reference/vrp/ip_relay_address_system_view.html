ip relay address（系统视图）

命令功能

ip relay address命令用来配置中继设备所代理的DHCP服务器地址。

undo ip relay address命令用来删除配置的DHCP服务器地址。

缺省情况下，系统视图下没有配置中继设备所代理的DHCP服务器地址。

命令格式

ip relay address ip-address { all | interface interface-type interface-number.sub-interface-number1 [ to interface-type interface-number.sub-interface-number2 ] | interface interface-type interface-number | vlan vlan-id }

undo ip relay address { ip-address | all } { all | interface interface-type interface-number.sub-interface-number1 [ to interface-type interface-number.sub-interface-number2 ] | interface interface-type interface-number | vlan vlan-id }

参数说明

ip-address：指定所代理的DHCP服务器的IP地址。

interface interface-type interface-number.sub-interface-number1 [ to interface-type interface-number.sub-interface-number2 ]：用关键字to连接两个子接口，表示子接口号是在两个接口之间的所有子接口（包含这两个子接口）。

vlan vlan-id：IP地址所对应的VLAN ID。整数形式，取值范围是1～4094。

all：undo命令中的前一个all指中继所代理的所有的DHCP服务器的IP地址，后一个all指所有接口。

使用指南

在系统视图下配置DHCP服务器地址，可以全局使能设备的DHCP中继功能。中继设备接收到客户端发送的广播的DHCP请求报文后，将其转发至指定的DHCP服务器。

使用实例

# 配置GE 2/0/0.1到GE 2/0/0.5之间的接口所代理的DHCP服务器地址为202.38.1.2。

<Quidway> system-view
[Quidway] ip relay address 202.38.1.2 interface gigabitethernet 2/0/0.1 to gigabitethernet 2/0/0.5
