display ospfv3 area

命令功能

display ospfv3 area命令用来查看OSPFv3的区域信息。

命令格式

display ospfv3 area [ process-id ] area [ area-id ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：指定OSPFv3的进程号。整数形式。

area area-id：指定所要显示区域的区域号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
