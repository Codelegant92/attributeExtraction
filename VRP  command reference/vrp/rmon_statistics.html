rmon statistics

命令功能

rmon statistics用来在统计表中添加一行。

undo rmon statistics用来在统计表中删除一行。

命令格式

rmon statistics entry-number [ owner owner-name ]

undo rmon statistics entry-number

参数说明

entry-number：添加或删除行的行号。整数形式，取值范围是1～65535。

owner-name：该行的创建者描述信息。字符串类型，长度范围是1～127。

缺省级别

2：配置级

使用指南

RMON统计管理可以设置对监视端口的使用及进行错误统计。

统计信息包括冲突、循环冗余校验和错误数、太小或超大包、超时传送、碎片、广播、多播、单播消息等。

使用实例

# 在GE1/0/0的统计表中添加第20行。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] rmon statistics 20 owner creater
