firewall packet-filter

命令功能

firewall packet-filter命令用来将访问控制列表应用到相应的接口上。

undo firewall packet-filter命令用来表示删除相应的设置。

命令格式

firewall packet-filter [ ipv6 ] acl-number { inbound | outbound }

undo firewall packet-filter [ ipv6 ] acl-number { inbound | outbound }

参数说明

acl-number：ACL的序号。

inbound：过滤从接口收上来的数据包。

outbound：过滤从接口转发的数据包。

缺省级别

使用指南

可以对接口的收发报文分别指定访问规则。

将访问规则列表应用到接口时，同时会遵循时间段过滤原则。

基于接口的访问控制列表（即序号为1000到1999的ACL）只能用参数outbound。

使用实例

# 在接口Pos1/0/0应用访问控制列表3101，过滤接收到的报文。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] firewall packet-filter 3101 inbound
