cmd recording-scheme

命令功能

cmd recording-scheme命令用来设置用户在路由器上所执行的命令的记录策略。

undo cmd recording-scheme命令用来删除记录策略，即不进行相应的记录。

缺省情况下，不记录用户在路由器上所执行的命令。

命令格式

cmd recording-scheme recording-scheme-name

undo cmd recording-scheme

参数说明

cmd：记录当前路由器上执行的命令。

recording-scheme-name：记录方案名称，字符串形式，长度范围是1～32。不区分大小写，方案名需要符合Windows的命名规范，不能包括斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“””和尖括号“<”、“>”等字符。

使用指南

配置该命令后，可以对设备情况进行记录，对监控和故障处理有一定的帮助。

使用实例

# 设置用户在路由器上所执行的命令的记录策略，方案名为test。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] cmd recording-scheme test
