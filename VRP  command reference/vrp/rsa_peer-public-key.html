rsa peer-public-key

命令功能

rsa peer-public-key命令用来进入公共密钥视图。

命令格式

rsa peer-public-key key-name

参数说明

key-name：公共密钥名称，字符串形式，长度为1～30。

视图

系统视图

缺省级别

2：配置级

使用指南

输入该命令后，将进入公共密钥视图，和public-key-code begin一同使用，可以对客户端的公钥进行配置。

客户端的公钥是由客户端软件随机生成的。

使用实例

# 进入公共密钥视图

[Quidway] rsa peer-public-key quidway002
Enter "RSA public key" view, return system view with "peer-public-key end".
