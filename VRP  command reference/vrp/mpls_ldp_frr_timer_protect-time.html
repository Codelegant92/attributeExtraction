mpls ldp frr timer protect-time

命令功能

mpls ldp frr timer protect-time命令用来为接口配置或调整LDP FRR保护定时器的时间。

undo mpls ldp frr timer protect-time命令用来恢复接口的LDP FRR保护定时器的默认时间。

缺省情况下，LDP FRR保护定时器的时间是10秒。

命令格式

mpls ldp frr timer protect-time protect-time

undo mpls ldp frr timer protect-time

参数说明

protect-time：LDP FRR保护时间。整数形式，取值范围是5～30，单位是秒。

使用指南

启用LDP FRR保护定时器后，当主链路故障导致LDP会话状态变为Down时，保持主链路的LDP LSP不被删除，直到出现以下任意一种情况才删除：

  • LDP FRR保护定时器超时

  • 主链路的路由被删除

  • 主链路的接口状态变为Down

如果接口没有使能LDP，不能为该接口配置LDP FRR保护定时器的时间。

使用实例

# 在接口GE1/0/0上使能LDP，并为该接口配置LDP FRR保护定时器的时间。

<Quidway> system-view
[Quidway] mpls lsr-id 1.1.1.1
[Quidway] mpls 
[Quidway-mpls] quit
[Quidway] mpls ldp
[Quidway-mpls-ldp] quit
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] mpls ldp
[Quidway-GigabitEthernet1/0/0] mpls ldp frr timer protect-time 5
