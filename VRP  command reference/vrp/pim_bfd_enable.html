pim bfd enable

命令功能

pim bfd enable命令用来在接口上使能PIM BFD功能。

undo pim bfd enable命令用来取消接口上的PIM BFD功能。

缺省情况下，接口没有使能PIM BFD功能。

命令格式

pim bfd enable

undo pim bfd enable

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

使用pim bfd enable命令时，存在如下限制：

  • 该命令只适用于NBMA接口和Broadcast接口，且不适用于MTunnel接口。

  • 该命令配置之前，接口上必须执行命令pim sm使能PIM-SM。执行undo pim sm命令将同时取消接口上的PIM BFD功能。

  • PIM BFD功能依赖于BFD协议，如果没有使用命令bfd使能全局BFD功能，仍可以建立PIM BFD Session，但session状态为BFD global disable。

使用实例

# 在GE1/0/0上使能PIM BFD功能。

<Quidway> system-view
[Quidway] bfd
[Quidway-bfd] quit
[Quidway] multicast routing-enable
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim sm
[Quidway-GigabitEthernet1/0/0] pim bfd enable
