display interface vlanif

命令功能

display interface vlanif命令用来查看VLANIF接口的使能状态、协议使能状态、接口描述信息和接口IP地址等。

命令格式

display interface vlanif [ vlan-id ]

参数说明

vlanif vlan-id：VLANIF接口的ID。整数形式，取值范围是1～4094，必须是已经存在的VLANIF接口。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

项目

描述

Vlanif2 current state :
NULL接口当前的物理状态：

  • UP：物理状态正常启动。

  • DOWN：物理连接不正常。

  • Administratively down：如果网络管理员在该接口执行shutdown命令，将显示状态为Administratively down。

Line protocol current state :

显示该接口的链路协议状态：

  • UP：接口的链路协议处于正常的启动状态。

  • DOWN：接口的链路协议不正常，或者没有在此接口配置IP地址。

Last line protocol up time:

接口的链路协议上次Up的时刻，“-”表示接口的链路协议状态没有发生过变化。

The Maximum Transmit Unit is

接口的最大传输单元（MTU）。例如串口默认值是1500字节。长度大于MTU的报文，将会被分片后再发送。如果设置了不准分片，会被丢弃。

Internet Address is

接口配置的IP地址和子网掩码。

Physical is

接口物理类型。

Statistics last cleared

上次清除本接口统计信息的时间。

Last 5 minutes input rate Last 5 minutes output rate

这一字段显示了在前5分钟内通过接口发送和接收的字节速率和报文速率。

Last 300 seconds input/output rate

这一字段显示了在前300秒内通过接口发送和接收的字节速率和报文速率。

Input/Output

接口收到或发送报文的方向。

packets

接口接收或发送总报文数。

bytes

接口接收或发送总字节数。

unicast

接口接收或发送的单播报文的数。

broadcast

接口接收或发送的广播报文的数。

multicast

接口接收或发送的组播报文的数。

errors

接口在物理层检测时发现的错误报文数目。

drops

在物理层丢弃的报文数。

unknown protocol

接收报文时，在物理层检测时发现的不能识别的协议报文数。
