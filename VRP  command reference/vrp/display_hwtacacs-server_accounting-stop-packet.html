display hwtacacs-server accounting-stop-packet

命令功能

display hwtacacs-server accounting-stop-packet命令用来查看HWTACACS服务器的计费停止报文情况。

命令格式

display hwtacacs-server accounting-stop-packet { all | number | ip ip-address }

参数说明

all：显示所有计费停止报文。

number：显示开始的指定数目的计费停止报文，取值范围是1～65535。

ip：显示包含指定IP地址的计费停止报文。

ip-address：IP地址，点分十进制形式。

使用指南

在查看配置或进行HWTACACS故障处理的时候，可以使用display hwtacacs-server accounting-stop-packet命令来查看HWTACACS服务器的计费停止报文情况。

使用实例

# 查看所有计费停止报文。

<Quidway> display hwtacacs-server accounting-stop-packet all
