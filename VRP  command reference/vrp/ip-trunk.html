ip-trunk

命令功能

ip-trunk命令用来将当前POS接口绑定到IP-Trunk捆绑组。

undo ip-trunk命令用来从IP-Trunk捆绑组中删除当前POS接口。

命令格式

ip-trunk trunk-id

undo ip-trunk trunk-id

参数说明

trunk-id：IP-Trunk的接口编号，取值范围是0～63。

视图

POS接口视图

缺省级别

2：配置级

使用指南

IP-Trunk只能由POS接口构成。

说明：

加入IP-Trunk的接口，链路层协议必须为HDLC。

使用实例

# 将POS接口1/0/0加入IP-Trunk 1中。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] link-protocol hdlc
[Quidway-Pos1/0/0] ip-trunk 1
