radius-server timeout

命令功能

radius-server timeout命令用来设置RADIUS服务器的重传超时时间。

undo radius-server timeout命令用来恢复缺省设置。

命令格式

radius-server timeout seconds

undo radius-server timeout

参数说明

timeout-value：重传超时时间，取值范围是3～10，单位是秒，缺省值是5。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

为判断某个RADIUS服务器是否失效，路由器会向RADIUS服务器周期性地发送请求报文，如果在规定的时间内未得到RADIUS服务器发回的应答，需重传请求报文。

只有当该RADIUS模板没有用户使用时，才能改变此配置。

该命令可以与radius-server retransmit命令同时使用。

使用实例

# 配置服务器重传超时时间为6秒。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server timeout 6
