authorization-scheme（AAA域视图）

命令功能

authorization-scheme命令用来设置当前域的授权方案。

undo authorization-scheme命令用来删除域的授权方案，恢复缺省设置。

缺省情况下，域使用授权方案default。

命令格式

authorization-scheme authorization-scheme-name

undo authorization-scheme

参数说明

authorization-scheme-name：授权方案名称，字符串形式，长度范围是1～32。不区分大小写，方案名需要符合Windows的命名规范，不能包括斜线“\”、反斜线“/”、冒号“:”、星号“*”、问号“?”、引号“””和尖括号“<”、“>”等字符。

视图

AAA域视图

缺省级别

3：管理级

使用指南

authorization-scheme命令把域和某个特定的授权方案关联后，该域就可以用此授权方案对用户进行授权。

在使用authorization-scheme命令前，授权方案必须已经创建完成。

使用实例

# 设置当前域使用名为test的授权方案。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] domain Huawei
[Quidway-aaa-domain-huawei] authorization-scheme test

# 删除当前域的授权方案。

[Quidway-aaa-domain-huawei] undo authorization-scheme
