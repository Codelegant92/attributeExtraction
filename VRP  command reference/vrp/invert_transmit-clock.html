invert transmit-clock

命令功能

invert transmit-clock命令用来翻转DTE侧同步串口的发送时钟信号。

undo invert transmit-clock命令用来禁止该功能。

缺省情况下，同步串口作为DTE侧设备时，禁止翻转发送时钟信号。

命令格式

invert transmit-clock

undo invert transmit-clock

参数说明

无

视图

串行接口视图

缺省级别

2：配置级

使用指南

说明：

由其它接口板通道化成的serial接口不支持invert transmit-clock命令。

建议不做此类配置，现在的系统中已实现自动翻转功能。

使用实例

# 将DTE侧同步串口的发送时钟翻转。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] invert transmit-clock
