statistics interval

命令功能

statistics interval命令用来配置上报统计信息的时间间隔。

undo statistics interval命令用来恢复缺省值。

缺省情况下，时间间隔为0，即不上报统计信息。

命令格式

statistics interval interval

undo statistics interval

参数说明

interval-time：时间间隔。单位为秒，取值范围是30～65535。

使用实例

# 配置上报统计信息的时间间隔为30秒。
