startup system-software（多框）

命令功能

startup system-software命令用来配置系统下次启动时使用的系统软件文件名。

缺省情况下，配置主控板下次启动时使用的系统软件文件名。

命令格式

startup system-software system-file [ all | slave-board | chassis chassis-number { master | slave } ]

参数说明

system-file：系统软件的文件名。字符串形式，长度范围是4～48。

slave-board：配置备板下次启动时使用的系统软件文件名。

chassis chassis-number：框号，整数形式，当框类型为线卡框时，框号取值范围1～8，例如：clc1。当框类型为中央交换框时，框号取值为1和2，例如：ccc1

使用指南

系统软件文件名必须以.cc作为扩展名，而且必须存放在存储设备（CF卡）的根目录下，如果您选择缺省路径，系统软件将存储在中央交换框CF卡的根目录下。

使用实例

# 配置下次启动使用的系统文件。

<Quidway> startup system-software NE5000E.cc
