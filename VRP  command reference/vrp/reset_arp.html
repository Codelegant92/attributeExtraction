reset arp

命令功能

reset arp命令用来清除ARP映射表中的ARP项。

命令格式

reset arp { all | dynamic | interface interface-type interface-number | slot slot-id | static }

参数说明

all：表示清除所有的ARP项。

dynamic：表示清除动态ARP项。

interface：表示选择接口。

interface-type：接口类型。

interface-number：接口序号。

slot slot-id：表示接口板的槽号。

static：表示清除静态ARP项。

使用指南

注意：

执行reset arp命令将取消IP地址和MAC地址的映射关系，可能导致无法访问某些节点。

使用reset arp命令对ARP表项进行维护，重新建立ARP表项。

使用reset arp命令时，对于分布式设备，如果不指定接口板槽号，表示对主控板上的ARP表项进行操作。

对指定slot-id的板子进行操作：

  • 如果是主控板，则可以删除动态（dynamic）和静态（static）的表项
  • 如果是接口板，则只能删除动态（dynamic）表项

对指定interface的接口进行操作时，接口类型只能是Ethernet口、GE口、虚拟以太网口三种，而且只能删除该接口的动态（dynamic）表项。

使用实例

# 删除主控板的ARP映射表中的static项。

<Quidway> reset arp static

# 删除接口GE 1/0/0的ARP映射表中的dynamic项。

<Quidway> reset arp interface gigabitethernet 1/0/0
