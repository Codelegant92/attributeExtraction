description（NQA视图）

命令功能

description命令用来配置NQA测试例的描述信息。

undo description命令用来取消NQA测试例的描述信息。

缺省情况下，NQA测试例没有配置描述信息。

命令格式

description string

undo description

参数说明

string：NQA测试例的描述信息。字符串形式，长度范围是1～230。

视图

NQA视图

缺省级别

2：配置级

使用指南

用户可以对一个测试例进行简要的描述，以方便维护。通常用于描述一个测试例所做的测试项或测试目的。

不能修改正在执行的测试例的描述信息。

使用实例

# 设置测试例user test的描述信息为“fortest”。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway] test-type jitter
[Quidway-nqa-user-test] description fortest
