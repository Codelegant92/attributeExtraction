cluster-snmp-nat enable

命令功能

cluster-snmp-nat enable命令用来使能命令交换机的SNMP NAT功能。

undo cluster-snmp-nat enable命令用来禁止命令交换机的SNMP NAT功能。

缺省情况下，命令交换机的SNMP NAT功能处于使能状态。

命令格式

cluster-snmp-nat enable

undo cluster-snmp-nat enable

参数说明

无

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

使用实例

# 禁止命令交换机的SNMP NAT功能。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] undo cluster-snmp-nat enable
