mib-binding vpn-instance

命令功能

mib-binding vpn-instance命令用来绑定组播MIB操作对应的实例。

undo mib-binding vpn-instance命令用来取消绑定。

缺省情况下，组播MIB不绑定任何VPN实例，而是对应公网。

命令格式

mib-binding vpn-instance vpn-instance-name

undo mib-binding vpn-instance

参数说明

vpn-instance-name：VPN实例名称。字符串形式，区分大小写，长度范围是1～31。

视图

组播MIB视图

缺省级别

2：配置级

使用指南

当组播MIB绑定到某VPN实例时，组播MIB操作都针对该实例。

目前组播MIB不支持网管站（NMS）同时对一台路由器多个实例的组播MIB管理。组播MIB只能绑定一个实例。

当再次配置mib-binding vpn-instance的时候，会覆盖原有的配置。

使用实例

# 配置组播MIB绑定到实例VPN。

<Quidway> system-view
[Quidway] ip vpn-instance vpna
[Quidway-vpn-instance-vpna] route-distinguisher 11:1
[Quidway-vpn-instance-vpna] vpn-target 11:1
[Quidway-vpn-instance-vpna] quit
[Quidway] multicast-mib
[Quidway-multicast-mib] mib-binding vpn-instance vpna
