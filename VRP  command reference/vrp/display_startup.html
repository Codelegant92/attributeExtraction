display startup

命令功能

display stratup命令用来显示与本次及下次启动相关的系统软件、配置文件名和Paf、License文件、补丁文件。

命令格式

display startup

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

使用该命令可以查看的信息包括：

  • 用户配置的用于本次启动的系统软件的文件名；

  • 本次启动实际使用的系统软件的文件名；

  • 配置的用于下次启动的系统软件的文件名；

  • 本次启动采用的配置文件的文件名；

  • 配置的用于下次启动的配置文件的文件名；

  • 本次启动采用的PAF文件的文件名；

  • 配置的用于下次启动的PAF文件的文件名；

  • 本次启动采用的License文件的文件名；

  • 配置的用于下次启动的License文件的文件名；

  • 本次启动采用的补丁包；

  • 配置的用于下次启动的补丁包。

如果在启动过程中没有找到用户配置的用于启动的系统软件的文件名，启动程序将自动搜索Flash目录下扩展名为.cc的文件，并使用找到的第一个.cc文件启动系统。

使用实例

# 显示本次及下次启动相关的文件名。

<Quidway> display startup

  Next startup saved-configuration file:   flash:/vrpcfg.cfg

  Next startup license file:               flash:/license-b291smk3.txt
  Startup patch package:                   flash:/patch.bat
