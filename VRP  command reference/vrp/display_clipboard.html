display clipboard

命令功能

display clipboard命令用来显示剪贴板的内容。

命令格式

display clipboard [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

通常用快捷键复制完命令后，可以通过此命令查看剪贴板中的内容。

使用快捷键复制命令方法：

  1. 任何视图下输入命令。

  2. 将光标移到命令开始处，同时按下快捷键ESC_SHIFT_<。

  3. 再将光标移到命令末尾，同时按下快捷键ESC_SHIFT_>。

使用实例

# 查看系统剪贴板的内容。
