display rrpp ring-group

命令功能

display rrpp ring-group命令用来显示环组配置，当不指定环组ID时显示所有环组的配置。

命令格式

display rrpp ring-group [ ring-group-id ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

ring-group ring-group-id：环组ID。整数形式，取值范围是1～16。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果是边缘节点，会显示当前发送Edge-Hello报文的环。

如果是辅助边缘节点，会显示当前接收Edge-Hello报文的环。

使用实例

# 显示所有环组信息。

<Quidway> display rrpp ring-group
Ring Group 1：



domain 1 ring 1 to 3, 5
domain 2 ring 1 to 3, 5
domain 1 ring 1 send Edge-Hello packet

Ring Group 2：



domain 1 ring 4, 6 to 7
domain 2 ring 4, 6 to 7

# 显示环组2的配置。

<Quidway> display rrpp ring-group 2
Ring Group 2：



domain 1 ring 4, 6 to 7
domain 2 ring 4, 6 to 7
