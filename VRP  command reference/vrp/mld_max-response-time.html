mld max-response-time

命令功能

mld max-response-time命令用来设置MLD查询报文中最大响应延迟字段的值。

undo mld max-response-time命令用来恢复配置的缺省情况。

缺省情况下，MLD查询报文的最大响应时间为10s。

命令格式

mld max-response-time interval

undo mld max-response-time

参数说明

interval：指定MLD查询报文的最大响应时间。整数形式，取值范围1～25，单位是s。

使用指南

最大查询响应时间决定了路由器快速检测局域网中是否存在直连组成员的时间。

如果接口视图和MLD视图下都配置了最大响应时间，优先取接口视图下的值。

使用实例

# 设置接口GE1/0/0的最大响应时间是8s。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld max-response-time 8
