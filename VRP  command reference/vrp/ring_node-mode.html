ring node-mode

命令功能

ring node-mode命令用于以下配置：

  • 当前设备的节点模式

  • 主环节点的主端口和副端口

  • 子环节点的主端口和副端口

  • 子环节点的公共端口和边缘端口

  • 环的级别

undo ring命令用于删除RRPP环。

命令格式

ring ring-id node-mode { master | transit } primary-port interface-type interface-number secondary-port interface-type interface-number level level-value

ring ring-id node-mode { edge | assistant-edge } common-port interface-type interface-number edge-port interface-type interface-number

undo ring ring-id

参数说明

ring-id：待创建的RRPP环的ID。整数形式，取值范围是1～64。

master：指定当前交换机为待创建RRPP环中的主节点。配置时需要指定主节点的主端口和副端口，以及环的级别。如果指定level-value级别为0，表明配置主环的主节点；如果级别为1，表明配置子环的主节点。

transit：指定当前交换机为待创建RRPP环中的传输节点。配置时需要指定传输节点的主端口和副端口，以及环的级别。如果指定level-value级别为0，表明配置主环的传输节点；如果级别为1，表明配置子环的传输节点。

primary-port：指定本节点的主端口。

interface-type：接口类型。可以是Ethernet，GigabitEthernet，Eth-Trunk或Virtual-Ethernet等类型。

interface-number：接口号，采用槽位编号/子卡号/接口编号的格式。和interface-type一起表示主端口、副端口、公共端口或边缘端口。

secondary-port：指定本节点的副端口。

level level-value：RRPP环的级别，取值是0或1。0表示创建主环，1表示创建子环。

edge：指定当前交换机为待创建RRPP环中的边缘节点。配置时需要指定边缘节点的公共端口和边缘端口。

assistant-edge：指定当前交换机为待创建RRPP环中的辅助边缘节点。配置时需要指定辅助边缘节点的公共端口和边缘端口。

common-port：指定本节点的公共端口。

edge-port：指定本节点的边缘端口。

使用指南

同一个RRPP域内配置的任何两个环的ID必须不同。

RRPP域的主环上只有主节点和传输节点两种节点模式，子环上的交换机有主节点、传输节点、边缘节点和辅助边缘节点四种节点模式。

一台交换机如果被配置为RRPP子环的主节点或传输节点，其上不允许再配置相同RRPP域的其它环。

在交换机上配置子环时，注意以下几点：

  • 对于边缘端口和辅助边缘端口，必须先配置其主环，再配置子环，否则子环无法创建成功。

  • 子环的公共端口必须是和主环公用的端口，边缘端口必须不属于同一域的任何其他RRPP环。

删除RRPP环时需注意：

  • 删除主环时必须确认其子环已经删除，否则操作失败。

  • 如果已经使能了RRPP协议和RRPP环，在删除RRPP环前，必须先去使能RRPP环，否则操作失败。

RRPP功能和RRPP Snooping功能不能同时配置在相同的接口上。

使用实例

# 配置RRPP域1的主环10的主节点，主端口为Ethernet1/0/5，副端口为Ethernet1/0/6。

<Quidway> system-view
[Quidway] rrpp domain 1
[Quidway-rrpp-domain-region1] ring 10 node-mode master primary-port ethernet 1/0/5 secondary-port ethernet 1/0/6 level 0

# 配置RRPP域1的主环10的传输节点，主端口为Ethernet1/0/3，副端口为Ethernet1/0/4。

[Quidway-rrpp-domain-region1] ring 10 node-mode transit primary-port ethernet 1/0/3 secondary-port ethernet 1/0/4 level 0

# 配置RRPP域1的子环20的主节点，主端口为Ethernet 1/0/1，副端口为Ethernet 1/0/2。

[Quidway-rrpp-domain-region1] ring 20 node-mode master primary-port ethernet 1/0/1 secondary-port ethernet 1/0/2 level 1

# 配置RRPP域1的子环20的传输节点，主端口为Ethernet 1/0/10，副端口为Ethernet 1/0/11。

[Quidway-rrpp-domain-region1] ring 20 node-mode transit primary-port ethernet 1/0/10 secondary-port ethernet 1/0/11 level 1

# 配置RRPP域1的子环20的边缘节点，公共端口为Ethernet 1/0/7，边缘端口为Ethernet 1/0/8。

[Quidway-rrpp-domain-region1] ring 20 node-mode edge common-port ethernet 1/0/7 edge-port ethernet 1/0/8

# 配置RRPP域1的子环20的辅助边缘节点，公共端口为Ethernet 1/0/6，边缘端口为Ethernet 1/0/9。

[Quidway-rrpp-domain-region1] ring 20 node-mode assistant-edge common-port ethernet 1/0/6 edge-port ethernet 1/0/9

# 删除环10。

[Quidway-rrpp-domain-region1] undo ring 10
