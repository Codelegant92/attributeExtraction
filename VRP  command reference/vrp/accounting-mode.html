accounting-mode

命令功能

accounting-mode命令用来配置当前计费方案使用的计费模式。

缺省情况下，对用户采用不计费模式。

命令格式

accounting-mode { hwtacacs | radius | local | none | local-radius | local-hwtacacs }

参数说明

none：不计费。

local：使用本地计费。

radius：使用RADIUS服务器计费。

hwtacacs：使用HWTACACS服务器计费。

local-radius：同时向本地和RADIUS服务器发送计费报文。

local-hwtacacs：同时向本地和HWTACACS服务器发送计费报文。

视图

计费方案视图

缺省级别

3：管理级

使用指南

accounting-mode命令在配置当前计费方案使用的计费模式时，如果要对用户采用RADIUS/HWTACACS计费方案，必须预先在系统视图下配置RADIUS/HWTACACS服务器模板，然后在用户所属域的视图下应用该服务器模板。

使用实例

# 配置名为scheme1的计费方案采用HWTACACS计费。

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] accounting-scheme scheme1
[Quidway-aaa-accounting-scheme1] accounting-mode hwtacacs
