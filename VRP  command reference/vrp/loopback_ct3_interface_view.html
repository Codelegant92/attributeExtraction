loopback（CT3接口视图）

命令功能

loopback命令用来配置CT3的环回模式。

undo loopback命令用来取消环回。

缺省情况下，禁止环回。

命令格式

loopback { local | payload | remote }

undo loopback

参数说明

local：设置CT3进行对内自环。

payload：设置CT3进行对外载荷环回，这种环回将处理帧头开销。

remote：设置CT3进行对外远端环回，这种环回不对帧进行处理。

CT3接口视图

缺省级别

2：配置级

使用指南

环回功能通常用于进行某些特殊测试，正常工作时不要启动环回。

当CT3接口设置为本端环回时，CT3接口的物理状态变为Up。

当对CT3接口设置环回时，该环回也会在该CT3接口下的同步串口生效：
  • 如果同步串口的链路层协议是PPP，当设置环回时，接口的物理状态变为Up，而链路协议状态为Down。
  • 如果同步串口的链路层协议是HDLC，当设置环回时，接口的物理状态和链路协议状态都是Up。

使用实例

# 使能CT3接口1/0/0对内自环。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] loopback local
