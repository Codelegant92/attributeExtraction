mpls ldp（系统视图）

命令功能

mpls ldp命令用来使能LDP对公网实例或VPN实例的支持。

undo mpls ldp命令用来全局关闭LDP，并删除全部LDP运行实例。

缺省情况下，没有使能LDP对公网实例或VPN实例的支持。

命令格式

mpls ldp [ vpn-instance vpn-instance-name ]

undo mpls ldp [ vpn-instance vpn-instance-name ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例名称。字符串形式，不支持空格，长度范围是1～31。

缺省级别

2：配置级

使用指南

配置了MPLS LSR ID并全局使能MPLS后，可以配置此命令。

mpls ldp命令将在设备上全局使能LDP，并创建LDP的公网实例。

mpls ldp vpn-instance命令对VPN实例使能LDP，LDP支持多实例。

undo mpls ldp vpn-instance命令关闭VPN实例的LDP。

使用实例

# 使能MPLS和LDP。

<Quidway> system-view
[Quidway] mpls lsr-id 1.1.1.1
[Quidway] mpls
[Quidway-mpls] quit
[Quidway] mpls ldp

# 对VPN实例使能LDP。

<Quidway> system-view
[Quidway] mpls ldp vpn-instance vpn1
[Quidway-mpls-ldp-vpn-instance-vpn1] quit
