hwtacacs-server accounting-stop-packet

命令功能

hwtacacs-server accounting-stop-packet命令用来设置是否允许重发计费停止报文，以及可重发的计费停止报文个数。

缺省情况下，使能计费停止报文重发功能，可重发的报文个数为100。

命令格式

hwtacacs-server accounting-stop-packet resend { disable | enable number }

参数说明

disable：禁止重发计费停止报文。

enable：使能重发计费停止报文。

number：重发的计费停止报文个数，取值范围是1～300。

使用指南

可以根据链路的情况和用户的需求来决定是否允许重发计费停止报文以及报文的数量。

使用实例

# 使能计费停止报文重发功能，每次可重发50个报文。

<Quidway> system-view
[Quidway] hwtacacs-server accounting-stop-packet resend enable 50
