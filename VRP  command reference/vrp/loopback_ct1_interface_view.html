loopback（CT1接口视图）

命令功能

loopback命令用来使能CT1的环回功能。

undo loopback命令用来禁止环回功能。

缺省情况下，禁止环回功能。

命令格式

loopback { local | payload | remote }

undo loopback

参数说明

local：使能CT1进行对内自环。

payload：使能CT1进行对外载荷环回。

remote：使能CT1进行对外远端环回。

视图

CT1接口视图

缺省级别

2：配置级

使用指南

环回功能主要用于检测接口或电缆本身的状况，正常工作时应禁止这些功能。

当接口设置为本端环回时，接口的物理状态变为Up。

当对CT1接口设置环回时，该环回也会在该CT1接口下的同步串口生效：
  • 如果同步串口的链路层协议是PPP，同步串口的物理状态变为Up，而链路协议状态为Down。
  • 如果同步串口的链路层协议是HDLC，同步串口的物理状态和链路协议状态都是Up。

使用实例

# 使能CT1接口1/0/0进行对内自环。

<Quidway> system-view
[Quidway] controller t1 1/0/0
[Quidway-T1 1/0/0] loopback local
