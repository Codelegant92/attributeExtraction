source

命令功能

source命令用来指定Tunnel接口的源地址或源接口。

undo source命令用来删除设置的源地址或源接口。

缺省情况下，系统不指定隧道的源地址或源接口。

命令格式

source { source-ip-address | interface-type interface-number }

undo source

参数说明

source-ip-address：指定Tunnel的源地址。

interface-type interface-number：指定Tunnel源接口的接口类型和接口编号。可使用的接口类型包括：Gigabit Ethernet、Ethernet、Serial、ATM、POS、Tunnel和Loopback等。

缺省级别

2：配置级

使用指南

在配置GRE、IPv6 over IPv4隧道和IPv4 over IPv6隧道时，需要创建Tunnel接口，并使用source命令配置该隧道的源地址或源接口，即指定隧道封装报文的源地址。

不能对两个或两个以上使用同种封装协议的Tunnel接口配置完全相同的源地址和目的地址。

使用实例

# 配置IPv6 over IPv4类型的Tunnel1/0/0接口，此接口封装的报文实际出口IP地址为192.100.1.1。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol ipv6-ipv4
[Quidway-Tunnel1/0/0] source 192.100.1.1

# 配置IPv6 over IPv4类型的Tunnel1/0/0接口，此接口封装的报文实际出口为Loopback1。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol ipv6-ipv4
[Quidway-Tunnel1/0/0] source loopback 1

# 配置IPv6 over IPv4类型的Tunnel1/0/0接口，并指定发出报文的源接口为POS1/0/0。

<Quidway> system-view
[Quidway] interface tunnel 1/0/0
[Quidway-Tunnel1/0/0] tunnel-protocol ipv6-ipv4
[Quidway-Tunnel1/0/0] source pos 1/0/0
