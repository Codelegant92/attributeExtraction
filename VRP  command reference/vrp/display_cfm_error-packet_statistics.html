display cfm error-packet statistics

命令功能

display cfm error-packet statistics命令用来查看本设备接收错误CCM消息的统计信息，包括：

  • CCM消息中携带的MA ID与本端配置不匹配的CCM消息的数量。

  • CCM消息中携带的CC检测时间间隔与本端配置不匹配的CCM消息的数量。

  • CCM消息中携带的源MAC地址与本端配置的RMEP的MAC地址不匹配的CCM消息的数量。

命令格式

display cfm error-packet statistics [ md md-name [ ma ma-name [ remote-mep mep-id mep-id ] ] ]

参数说明

md md-name：表示查看指定MD内接收错误CCM消息的统计信息。md-name表示MD的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。

ma ma-name：表示查看指定MA内接收错误CCM消息的统计信息。ma-name表示MA的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。

remote-mep mep-id mep-id：表示查看本设备从指定RMEP接收到的错误CCM消息的统计信息。mep-id表示RMEP的ID，整数形式，取值范围是1～8191。

缺省级别

使用指南

当网络或设备的CFM功能出现问题时，可使用本命令查看错误CCM消息的统计信息。得到的统计信息可作为定位问题的依据之一。

使用实例

# 查看md1内接收错误CCM消息的统计信息。

<Quidway> display cfm error-packet statistics md md1
 MD Name               : md1
 MA Name               : ma1
 RMEP ID               : --
 MAID Error           : 300
 CCM-Interval Error : 1000
 MAC Error            : 0

# 查看md1中ma2内接收错误CCM消息的统计信息。

<Quidway> display cfm error-packet statistics md md1 ma ma2
 MD Name               : md1
 MA Name               : ma2
 RMEP ID               : --
 MAID Error           : 270
 CCM-Interval Error : 12
 MAC Error            : 0

# 查看本设备从md1内ma2中ID为3的RMEP接收的错误报文的统计信息。

<Quidway> display cfm error-packet statistics md md1 ma ma2 remote-mep mep-id 3
 MD Name               : md1
 MA Name               : ma2
 RMEP ID               : 3
 MAID Error           : 0
 CCM-Interval Error : 0
 MAC Error            : 0
