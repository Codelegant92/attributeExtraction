dhcp snooping check

命令功能

dhcp snooping check命令用来使能对报文的检查功能。

undo dhcp snooping check命令用来去使能对报文的检查功能。

缺省情况下，不检查任何报文。

命令格式

dhcp snooping check { arp | ip | dhcp-chaddr | dhcp-request } enable [ interface interface-type interface-number ]

undo dhcp snooping check { arp | ip | dhcp-chaddr | dhcp-request } enable [ interface interface-type interface-number ]

参数说明

arp：检查ARP报文。

ip：检查IP报文。

dhcp-chaddr：检查DHCP报文里的CHADDR字段和以太帧头的源MAC是否一致。

dhcp-request：检查DHCP Request报文。

interface-type：接口类型。此参数仅在VLAN视图下才可配置。

interface-number：接口号。此参数仅在VLAN视图下才可配置。

视图

以太网接口及其子接口视图、VLAN视图

缺省级别

2：配置级

使用指南

只有打开了DHCP Snooping功能开关，dhcp snooping check命令才能生效。

使用实例

# 在Ethernet4/0/0接口上打开DHCP Snooping对报文的检测功能。

<Quidway> system-view
[Quidway] interface ethernet 4/0/0
[Quidway-Ethernet4/0/0] dhcp snooping check ip enable
