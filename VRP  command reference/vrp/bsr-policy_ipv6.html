bsr-policy（IPv6）

命令功能

bsr-policy命令用来限定合法BSR地址范围，使路由器丢弃来自该地址范围之外的BSR报文，从而防止BSR欺骗。

undo bsr-policy命令用来恢复缺省配置。

缺省情况下，BSR地址范围不受任何限制，即接收到的所有BSR报文都认为是有效的。

命令格式

bsr-policy basic-acl6-number

undo bsr-policy

参数说明

basic-acl6-number：表示基本访问控制列表号，该列表定义了针对BSR报文源地址范围的过滤策略。整数形式，取值范围是2000～2999。

缺省级别

2：配置级

使用指南

bsr-policy命令与acl命令配合使用。在ACL6视图下，通过rule命令的source参数定义BSR报文源地址范围。

请参见IPv4 PIM配置命令中的bsr-policy（IPv4）。

使用实例

# 在公网实例下，配置合法BSR地址范围是2001::2/64网段。

<Quidway> system-view
[Quidway] acl ipv6 2001
[Quidway-acl6-basic-2001] rule permit source 2001::2 64
[Quidway-acl6-basic-2001] quit
[Quidway] pim-ipv6
[Quidway-pim] bsr-policy 2001
