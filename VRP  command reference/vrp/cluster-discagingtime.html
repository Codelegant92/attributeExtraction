cluster-discagingtime

命令功能

cluster-discagingtime命令用来设置成员交换机处于disconnect状态时的老化时间。如果成员交换机处于disconnect状态的时间超过该老化时间，命令交换机就将此成员交换机从本HGMP集群删除，使之成为候选交换机。

undo cluster-discagingtime用来恢复缺省设置。

缺省情况下，没有配置老化时间，表示不进行老化处理。

命令格式

cluster-discagingtime disconnect-aging-time

undo cluster-discagingtime

参数说明

disconnect-aging-time：成员交换机处于disconnect状态时的老化时间。整数形式，取值范围是0～18，单位是小时。缺省情况下，取值为0，即没有配置老化时间，表示不进行老化处理。

使用指南

本命令只能在命令交换机上执行。

在执行cluster-discagingtime命令前，必须先通过cluster-autojoin命令，使能候选交换机自动加入集群的功能，否则从HGMP删除的成员交换机无法再次自动加入集群。

使用实例

# 设置成员交换机处于disconnect状态时的老化时间为1小时。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] cluster-discagingtime 1
