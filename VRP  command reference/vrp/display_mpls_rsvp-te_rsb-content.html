display mpls rsvp-te rsb-content

命令功能

display mpls rsvp-te rsb-content命令用来查看RSVP-TE RSB（Reserve State Block）信息。

命令格式

display mpls rsvp-te rsb-content ingress-lsr-id tunnel-id lsp-id [ | { begin | include | exclude } regular-expression ]

参数说明

ingress-lsr-id：入节点的LSR ID。

lsp-id：LSP ID，取值范围是0～65535。

tunnel-id：隧道ID，取值范围是0～65535。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

如果不指定参数，则显示所有接口的RSVP-TE邻居信息。
