display rip database

命令功能

display rip database命令用来查看RIP发布数据库的所有激活路由。这些路由以常规RIP更新报文的形式发送。

命令格式

display rip process-id database [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIP进程号。整数形式，取值范围是1～65535。

verbose：显示RIP发布数据库中路由的详细信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看RIP数据库信息。

<Quidway> display rip 100 database
   10.0.0.0/8, cost 1, ClassfulSumm
       10.0.0.0/24, cost 1, nexthop 10.0.0.1, Rip-interface
   11.0.0.0/8, cost 1, ClassfulSumm
       11.0.0.0/24, cost 1, nexthop 10.0.0.1, Imported
