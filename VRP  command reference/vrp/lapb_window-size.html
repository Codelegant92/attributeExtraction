lapb window-size

命令功能

lapb window-size命令用来配置LAPB窗口参数K。

undo lapb window-size命令用来恢复LAPB窗口参数K的缺省值。

命令格式

lapb window-size k-value

undo lapb window-size

参数说明

k-value：DTE或DCE可以发送的未证实顺序编号的I帧的最大数量，如果模数为8，窗口参数K范围是1～7；如果模数为128，窗口参数K范围是1～127。缺省情况下，窗口参数K值是7。

执行完该命令后，必须重新启动接口，使配置生效。

使用实例

# 设置接口Serial1/0/0上LAPB窗口参数K为5。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol lapb
[Quidway-Serial1/0/0] lapb window-size 5
[Quidway-Serial1/0/0] shutdown
[Quidway-Serial1/0/0] undo shutdown
