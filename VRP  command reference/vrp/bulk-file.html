bulk-file

命令功能

bulk-file命令用来创建批量性能统计文件并进入bulk-file视图。

undo bulk-file命令用来删除批量性能统计文件。

命令格式

bulk-file file-name

undo bulk-file file-name

参数说明

file-name：指定批量性能统计文件的文件名。字符串形式，长度范围是1～31。文件名称中只能包含：数字、字母、“_”、“-”。

视图

系统视图

缺省级别

2：配置级

使用指南

需要先执行bulk-stat enable命令来使能批量性能统计功能，才能建立批量统计文件。

如果指定名为file-name的文件已经创建，则进入bulk-file视图。

删除统计文件前，必须确认统计文件存在，并且文件统计任务已经被停止；如果条件不满足，则输出相应的提示信息。

使用实例

# 创建批量性能统计文件iftable。

<Quidway> system-view
[Quidway] bulk-stat enable
Info: Succeeded in enabling the bulk stat function.
[Quidway] bulk-file iftable
[Quidway-bulk-file-iftable]
