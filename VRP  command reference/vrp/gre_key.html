gre key

命令功能

gre key命令用来设置GRE隧道的识别关键字，通过这种弱安全机制防止错误识别、接收其它地方来的报文。

undo gre key命令用来取消当前配置。

缺省情况下，GRE隧道没有设置识别关键字。

命令格式

gre key key-number

undo gre key

参数说明

key-number：标识隧道两端的识别关键字。整数形式，取值范围是0～4294967295。

视图

Tunnel接口视图

缺省级别

2：配置级

使用指南

如果配置了gre key命令，则收发双方将进行通道识别关键字的验证。只有Tunnel两端设置的识别关键字完全一致时才能通过验证，否则将报文丢弃。所以，如果在隧道两端设置key-number，则必须指定相同的key-number；或隧道两端都不设置key-number。

使用实例

# 路由器Quidway1和Quidway2之间建立隧道，设置隧道的识别关键字。

<Quidway1> system-view
[Quidway1] interface tunnel 1/0/0
[Quidway1-Tunnel1/0/0] gre key 123
<Quidway2> system-view
[Quidway2] interface tunnel 2/0/0
[Quidway2-Tunnel2/0/0] gre key 123
