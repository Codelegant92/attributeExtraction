mtrace-last-hop-address

命令功能

mtrace-last-hop-address ipv4命令配置nqa测试例的最后一跳地址。

undo mtrace-last-hop-address ipv4命令用来恢复缺省情况。

缺省情况下，NQA mtrace测试例未指定最后一跳路由器地址。

命令格式

mtrace-last-hop-address ipv4 ip-address

undo mtrace-last-hop-address

参数说明

ipv4 ip-address：指定最后一跳路由器地址，属于单播IPv4地址。

视图

NQA视图

缺省级别

2：配置级

使用指南

如果指定IGMP-Tracert-Query报文发送方式为last-hop，则必须配置此参数。

使用实例

# 设置测试例user test的最后一跳地址

<Quidway> systerm-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test]mtrace-last-hop-address ipv4 192.168.0.1

# 取消测试例user test的最后一跳地址

[Quidway-nqa-user-test] undo mtrace-last-hop-address
