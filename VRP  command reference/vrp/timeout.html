timeout

命令功能

timeout命令用来配置NQA测试例的操作超时时间。

undo timeout命令用来恢复缺省设置。

命令格式

timeout time

undo timeout

参数说明

time：设置一次探测的超时时间。单位是秒，取值范围是1～60。

视图

NQA视图

缺省级别

2：配置级

使用指南

测试超时时间是指发送完探测报文后，等待响应报文的时间。

如果超过此时间没有收到响应报文，认为该次测试失败。

不能修改正在运行的测试例的此项配置。

使用实例

# 设置测试例user test的超时时间为60秒。

<Quidway> system-view
[Quidway] nqa test-instance user test
[Quidway-nqa-user-test] timeout 60
