ssh user

命令功能

ssh user命令用来新建SSH用户。

undo ssh user命令用来删除SSH用户。

命令格式

ssh user user-name

undo ssh user [ user-name ]

参数说明

user-name：标识SSH用户的用户名。字符串形式，不支持空格，长度范围是1～64。

视图

系统视图

缺省级别

2：配置级

使用指南

可以执行该命令单独新建一个用户；也可以在执行ssh user authentication-type、ssh user service-type、ssh user sftp-directory命令时，如果检测user-name不存在，则新建一个名为user-name的用户。

使用实例

# 新建SSH用户testuser。

<Quidway> system-view
[Quidway] ssh user testuser
Info: Succeeded in adding a new SSH user.

# 删除SSH用户testuser。

<Quidway> system-view
[Quidway] undo ssh user testuser
Info: Succeeded in deleting the SSH user.
