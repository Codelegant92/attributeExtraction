shutdown（VLANIF接口视图）

命令功能

shutdown命令用于关闭VLANIF接口，使VLAN内的用户不能与其他VLAN互通。

undo shutdown命令用来打开VLANIF接口。

缺省情况下，接口处于打开状态。

命令格式

shutdown

undo shutdown

参数说明

无

视图

VLANIF接口视图

缺省级别

2：配置级

使用指南

VLANIF接口处于shutdown状态后，VLAN内的所有用户都不能通过VLANIF接口与其他VLAN内用户通信。但是VLAN内用户还可以互相通信。

VLANIF接口上的流量包括二层流量和三层流量，如果在VLANIF接口视图下执行shutdown命令，则只能禁止三层流量，不能禁止二层流量。此时，执行display interface vlanif命令可以看到VLANIF接口下的流量计数仍在增加。

如果需要禁止VLANIF接口下的所有流量，请在VLAN视图下执行shutdown vlan命令。

使用实例

# 关闭接口VLAN8，VLAN8内的用户可以相互访问，但不能访问其它VLAN。

<Quidway> system-view
[Quidway] interface vlanif 8
[Quidway-Vlanif8] shutdown
