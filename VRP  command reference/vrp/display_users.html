display users

命令功能

display users命令用来显示每个用户界面的用户登录信息。

命令格式

display users [ all ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：显示所有通过用户界面登录的用户的信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在执行命令时，如果未使用all选项，则仅显示当前已连接的用户；如果使用all选项，则显示所有用户，包括未连接的用户。

执行该命令后，可显示当前路由器上接入了哪些用户，用户的用户名、地址等登录信息，以及用户的验证和授权信息等。

项目

描述

+

正在使用的终端线

User-Intf

第一列数字是用户界面的绝对编号，第二列数字是用户界面的相对编号。

Delay

表明用户自最近一次输入到现在的时间间隔，单位是秒。

Type

连接类型，有Telnet、AUX、Console、SSH等类型。

Network Address

显示起始连接位置，即入连接的主机IP地址。

Username

显示使用该用户界面的用户名，即该用户的登入username。目前没有AAA验证，所以此项为空。

AuthenStatus

标识验证是否通过。

AuthorcmdFlag

命令行授权标志，分为：命令行授权和不进行命令行授权。
