display icmpv6 statistics

命令功能

display icmpv6 statistics命令用来查看ICMPv6流量统计信息。

命令格式

display icmpv6 statistics [ slot slot-id | interface interface-type interface-number ]

参数说明

slot slot-id：显示分布式环境中相关I/O板上的ICMPv6统计信息。在集中式环境中此项无效。

interface interface-type interface-number：显示指定接口的ICMPv6统计信息。

使用指南

例如报文传输，报文接收，报文封装与拆封，这些都有助于定位故障原因。本地路由设备收到的报文数指的是接收的报文总数，无论这些报文被转发、传入上层还是被丢弃。
