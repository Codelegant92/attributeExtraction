frame-format（CPOS接口视图）

命令功能

frame-format命令用来设置CPOS使用的帧类型。

undo frame-format用来恢复缺省设置。

缺省情况下，CPOS的帧类型为SDH。

命令格式

frame-format { sdh | sonet }

undo frame-format

参数说明

sdh：帧类型为SDH（Synchronous Digital Hierarchy）。

sonet：帧类型为SONET（Synchronous Optical Network）。

视图

CPOS接口视图

缺省级别

2：配置级

使用指南

配置时需与对端保持一致。

使用实例

# 设置指定的CPOS物理端口工作在SONET模式。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] frame-format sonet
