pppoe-server bind virtual-template

命令功能

pppoe-server bind virtual-template命令用来将当前以太网接口绑定到指定的虚接口模板上，并使能PPPoE。

undo pppoe-server bind virtual-template命令用来在当前接口禁止PPPoE协议。

缺省情况下，禁止PPPoE协议。

命令格式

pppoe-server bind virtual-template number

undo pppoe-server bind

参数说明

number：接入PPPoE的虚接口模板的接口编号。整数形式，取值范围是0～1023。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

它是为VT指定一个物理通道。如果不配置该命令，就无法配置接入服务器的服务名，用户就无法拨入。

使用实例

# 在GE1/0/0的虚拟模板1上使能PPPoE。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pppoe-server bind virtual-template 1
