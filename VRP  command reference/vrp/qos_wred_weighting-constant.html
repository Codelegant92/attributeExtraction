qos wred weighting-constant

命令功能

qos wred weighting-constant命令用来设置WRED计算平均队长的指数。

undo qos wred weighting-constant命令用来恢复缺省值。

命令格式

qos wred weighting-constant exponent

undo qos wred weighting-constant

参数说明

exponent：计算平均队长的指数，取值范围是1～16，缺省值为9。

视图

接口视图

缺省级别

2：配置级

使用指南

必须先使用qos wred在接口上应用WRED，才可以配置WRED的参数。

使用实例

# 以下命令将在GE6/0/0接口上配置计算平均队长的指数为6。（假设已在接口上使能了WRED）。

<Quidway> system-view
[Quidway] interface GigabitEthernet0/0/0
[Quidway-GigabitEthernet0/0/0] qos wred weighting-constant 6
