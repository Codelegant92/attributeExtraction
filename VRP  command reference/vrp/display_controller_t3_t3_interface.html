display controller t3（T3接口）

命令功能

display controller t3命令用来查看T3的配置和状态信息。

命令格式

display controller t3 [ controller-number ]

参数说明

controller-number：T3接口的编号，包括槽号、接口卡号、端口号三部分。如果不指定编号，将显示所有T3接口的配置和状态信息。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看T3 1/1/0的配置情况和状态信息。

<Quidway> display controller t3 1/1/0
T3 1/1/0 current state : DOWN
  Cable length is 350 feet
  Frame-format c-bit, code B3ZS, clock slave
  Current mode is T3, loopback not set
  Alarm: LOS LOF RED
  Error: 0 CPERR, 65535 EXZS, 65535 FEBE, 65535 LCV, 0 PERR, 65535 FERR

项目

描述

T3 1/1/0 current state :
T3接口当前的物理状态：

  • UP：物理状态正常启动。

  • DOWN：物理连接不正常。

  • Administratively down：如果网络管理员在该接口执行shutdown命令，将显示状态为Administratively down。

Cable length

电缆长度，是路由器到配线架之间的距离。可以使用cable（T3接口视图）命令改变电缆长度。

Frame-format

T3接口的帧格式。

code

T3接口的线路编解码格式。

clock
时钟模式：

  • slave：从时钟模式。

  • master：主时钟模式。

Current mode is

接口当前的工作模式：

  • CT3：接口工作在CT3模式下，表示通道化工作模式。

  • T3：接口工作在T3模式下，表示净通道化工作模式。

loopback not set

没有使能T3接口环回功能。只有在进行某些特殊功能测试时，才将T3接口设为环回。

Alarm:ERROR:
告警信息和错误信息。可能出现的错误类型有以下几类。

  • LOS：Loss of Signal，帧同步丢失。

  • LOF：Loss of Frame，帧丢失。

  • CERR：CRC Error，循环冗余校验错。

  • LCV：Line Code Violation，线路码错误。

  • BPV：Bipolar Violation，线路码极性错误。

  • EXZ：Excessive Zeros，连续多0错误。

  • FrmErr：FRAME ERROR，帧错误。
