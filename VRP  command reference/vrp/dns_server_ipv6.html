dns server ipv6

命令功能

dns server ipv6命令用来指定一个IPv6主机作为域名服务器，提供主机名称信息。可反复使用dns server ipv6命令。最多可以设置6个域名服务器。

undo dns server ipv6命令用来删除已配置的域名服务器。

命令格式

dns server ipv6 ipv6-address [ interface-type interface-number ]

undo dns server ipv6 ipv6-address [ interface-type interface-number ]

参数说明

ipv6-address：指定域名服务器的IPv6地址。

interface-type interface-number：指定DNS设置为IPv6链路本地地址时的出接口类型和接口编号。
