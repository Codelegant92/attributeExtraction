display firewall session statistics

命令功能

display firewall session statistics命令显示表项的统计信息。

命令格式

display firewall session statistics slot slot-id

参数说明

slot-id：为指定的接口板槽号。

视图

所有视图

缺省级别

1：监控级

使用指南

通过display firewall session statistics命令查看的表项统计信息包括：

  • 服务器表项

  • 会话表项

  • 分片表项

  • 半连接数

使用实例

#显示会话表项的统计信息。

<Quidway> display firewall session statistics slot 4
server table count: 0
  session table count: 0
  fragment table count: 0
  half connect count: 0
