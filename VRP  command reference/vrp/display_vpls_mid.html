display vpls mid

命令功能

display vpls mid命令用于显示MID（Multicast Information Description）信息。

命令格式

display vpls mid token vsi vsi-name

display vpls mid interface vsi vsi-name [ vlan vlan-id ]

参数说明

token：用于显示Token的MID（Multicast Information Description），即公网侧的广播表信息。

interface：用于显示Interface的MID信息，即AC侧的广播表信息。

vsi vsi-name：用于显示指定VSI实例的MID信息。字符串形式，不支持空格，区分大小写，取值范围是1～31。

vlan vlan-id：用于显示指定VLAN ID的AC侧的MID信息。十进制整数形式，取值范围是1～4094。

使用指南

MID信息包括Token信息和Interface信息。可以通过在命令行中指定不同的关键字或参数来显示所需要的MID信息。

使用实例

# 显示名称为company1的VSI实例MID公网侧的信息。

<Quidway> display vpls mid token vsi company1
VSI Name : company1, Total PW Number : 1
 SLOT ID   LspToken(s)
3          0x7d3
表1 display vpls mid token vsi命令输出信息描述

项目

描述

VSI Name

VSI名称。

Total PW Number

属于该VSI的PW的总数，包括处于up状态和处于down状态的。

SLOT ID

公网侧发送该VSI用户数据的接口所在的槽位号。

LspToken

MID表项对应的PW的Token值，系统用其唯一确定一条PW。

# 显示名称为company1的VSI实例的VLAN ID为100的MID中AC侧的信息。

<Quidway> display vpls mid interface vsi company1
VSI Name : company1, Total AC Number : 1
 VLAN ID   SLOT ID    Interface(s)
 100       2        Ethernet2/0/1.1
表2 display vpls mid interface vsi命令输出信息描述

VLAN ID

SLOT ID
