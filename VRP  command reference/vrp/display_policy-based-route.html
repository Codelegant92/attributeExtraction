display policy-based-route

命令功能

display policy-based-route命令用来查看已创建的策略内容。

命令格式

display policy-based-route [ policy-name [ verbose ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

policy-name：指定要查看的策略路由的名称，字符串形式，长度范围是1～19。

verbose：查看指定策略路由的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
