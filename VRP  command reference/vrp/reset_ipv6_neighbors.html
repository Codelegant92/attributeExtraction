reset ipv6 neighbors

命令功能

reset ipv6 neighbors命令用来清除IPv6邻居表项。

命令格式

reset ipv6 neighbors { all | dynamic | static | vid vlan-id [ interface-type interface-number ] | interface-type interface-number }

参数说明

all：清除所有接口上的静态与动态邻居表项。

dynamic：表示清除所有接口上的动态邻居表项。

static：表示清除所有接口上的静态邻居表项。

vid vlan-id：清除指定VLAN上的邻居表项。

interface-type interface-number：清除指定接口上的邻居表项。

使用指南

该命令会清除IPv6邻居表项。使用该命令前请确认是否需要清除邻居表项。

使用实例

# 清除所有接口上的所有邻居项。

<Quidway> reset ipv6 neighbors all

# 清除所有接口上的所有静态邻居项。

<Quidway> reset ipv6 neighbors static

# 清除所有接口上的所有动态邻居项。

<Quidway> reset ipv6 neighbors dynamic

# 清除指定接口GE1/0/1上的所有邻居项。

<Quidway> reset ipv6 neighbors gigabitethernet 1/0/1
