network (RIP)

命令功能

network命令用来对指定网段接口使能RIP路由。

undo network命令用来对指定网段接口禁用RIP路由。

命令格式

network network-address

undo network network-address

参数说明

network-address：使能RIP的网络地址，必须是自然网段的地址。

视图

RIP视图

缺省级别

2：配置级

使用指南

说明：

RIP不支持将同一物理接口下的不同网段使能到不同的RIP进程中。

使用实例

# 对指定网段接口使能RIP路由。

<Quidway> system-view
[Quidway] rip 1
[Quidway-rip-1] network 10.0.0.0
