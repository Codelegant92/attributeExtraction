display vrrp binding admin-vrrp

命令功能

display vrrp binding admin-vrrp命令用来显示管理VRRP所有的绑定信息，包括成员VRRP、成员接口和成员PW。

命令格式

display vrrp binding admin-vrrp [ interface interface-type interface-number ] [ vrid virtual-router-id ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface interface-type interface-number：配置了管理VRRP的接口类型和编号。

vrid virtual-router-id：管理VRRP备份组号。十进制整数形式，取值范围是1～255。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display vrrp binding admin-vrrp命令时，如果不指定管理VRRP的接口名称则显示所有与管理VRRP的绑定信息。

使用实例

# 显示所有与接口GE1/0/0下的管理VRRP(VRID：1)的绑定信息。
