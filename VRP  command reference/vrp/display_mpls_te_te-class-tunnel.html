display mpls te te-class-tunnel

命令功能

display mpls te te-class-tunnel命令用来查看系统某个或所有与TE-Class关联的TE隧道。

命令格式

display mpls te te-class-tunnel { all | {ct0 | ct1 | ct2 | ct3 | ct4 | ct5 | ct6 | ct7 } priority priority } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：显示当前系统所有与TE-Class关联的TE隧道。

ct0～ct7：指定TE-Class中的CT。

priority priority：指定TE-Class中的隧道优先级。

| ：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

No.

表项编号

CT

服务类型

priority

隧道的保持优先级

status

标识该TE-Class是否生效：

  • Valid：该TE-Class已生效
  • Invalid：该TE-Class未生效

tunnel name

该CT对应的TE隧道接口的名称

tunnel commit

隧道接口下的配置是否提交：

  • Yes：隧道接口的配置已提交
  • No：隧道接口的配置未提交
