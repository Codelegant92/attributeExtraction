reset msdp sa-cache

命令功能

reset msdp sa-cache命令用来清除SA缓存中的（S，G）表项。

命令格式

reset msdp [ vpn-instance vpn-instance-name | all-instance ] sa-cache [ group-address ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

group-address：表示组播组地址。点分十进制形式，取值范围是224.0.0.0～239.255.255.255。

使用指南

使用reset msdp sa-cache命令时：

  • 如果不指定vpn-instance或all-instance，则只清除公网SA缓存中的（S，G）表项。

  • 如果指定group-address，则清除SA缓存中匹配该地址的（S，G）表项。

  • 如果不指定group-address，则清除SA缓存中所有的（S，G）表项。

使用实例

# 从公网实例的SA缓存中清除组地址为225.5.4.3的（S，G）表项。

<Quidway> reset msdp sa-cache 225.5.4.3
