maximum-routes

命令功能

maximum-routes命令用来确定OSPF支持的路由最大数目，可以设置的路由类型为区域内路由、区域间路由和区域外部路由。

undo maximum-routes命令用来恢复不同类型路由最大数量的缺省值。

命令格式

maximum-routes { external | inter | intra } number

undo maximum-routes { external | inter | intra }

参数说明

number：路由的数量，不同类型路由的取值范围不同。

external：OSPF生成的区域外部路由的最大数目。整数形式，取值范围是100～5000000，单位是条。缺省值是5000000。

inter：OSPF生成的区域间路由的最大数目。整数形式，取值范围是100～1000000，单位是条。缺省值是1000000。

intra：OSPF生成的区域内路由的最大数目。整数形式，取值范围是100～100000，单位是条。缺省值是100000。
