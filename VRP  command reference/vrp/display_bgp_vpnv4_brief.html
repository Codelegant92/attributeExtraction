display bgp vpnv4 brief

命令功能

display bgp vpnv4 brief命令用来显示VPN实例的简要信息。

命令格式

display bgp vpnv4 { all | vpn-instance vpn-instance-name } brief [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

all：查看所有VPN实例的信息。

vpn-instance vpn-instance-name：VPN实例名称。字符串形式，不支持空格，长度范围是1～31。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用命令dispay bgp vpnv4 brief查看VPNv4和VPN实例的信息时，各VPN实例按照实例名的字典序排列。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
