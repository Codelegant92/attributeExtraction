parity

命令功能

parity命令用来设置用户界面的校验位。

undo parity命令用来恢复用户界面的校验方式为无校验。

缺省情况下，无校验。

命令格式

parity { even | mark | none | odd | space }

undo parity

参数说明

even：偶校验。

mark：mark校验。

none：无校验。

odd：奇校验。

space：space校验。

用户界面视图

缺省级别

3：管理级

使用指南

只有串口工作在异步交互方式下时，配置有效。

使用实例

# 将AUX口传输校验位设为奇校验。

<Quidway> system
[Quidway] user-interface aux 0
[Quidway-ui-aux0] parity odd
