display dns dynamic-host

命令功能

display dns dynamic-host命令用来查看域名缓存区中的动态DNS表项信息。

命令格式

display dns dynamic-host [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在任一视图下执行display dns dynamic-host命令，查看DNS的域名缓存区中的表项信息，检测域名是否匹配正确。

# 查看动态DNS表项。

<Quidway> display dns dynamic-host
No  Domain-name         IpAddress      TTL    Alias
1   www.huawei.com      91.1.1.1       3521
2   www.huawei.com.cn   87.1.1.1       3000
