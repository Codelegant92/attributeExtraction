frame-length

命令功能

frame-length命令用来设置IMA帧中的ATM信元个数。

undo frame-length命令用来恢复缺省设置。

缺省情况下，一个IMA帧中包含128个ATM信元。

命令格式

frame-length { 32 | 64 | 128 | 256 }

undo frame-length

参数说明

32、64、128、256：IMA帧中的ATM信元个数。

视图

IMA组接口视图

缺省级别

2：配置级

使用指南

建议使用缺省值。

使用实例

# 设置IMA组1/0/1的IMA帧中ATM信元个数为64。

<Quidway> system-view
[Quidway] interface ima-group 1/0/1
[Quidway-Ima-group 1/0/1] frame-length 64
