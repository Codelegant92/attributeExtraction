if-match cost

命令功能

if-match cost命令用来设置一个基于路由开销的匹配规则。

undo if-match cost命令用来删除该匹配规则。

缺省情况下，没有配置基于路由开销的匹配规则。

命令格式

if-match cost cost

undo if-match cost

参数说明

cost：指定路由开销值。整数形式，取值范围是0～4294967295。

缺省级别

2：配置级

使用指南

无

使用实例

# 匹配路由开销值为8的路由信息。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] if-match cost 8
