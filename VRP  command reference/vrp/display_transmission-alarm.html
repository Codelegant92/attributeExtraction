display transmission-alarm

命令功能

display transmission-alarm命令用来查看接口告警状态和告警统计信息。如果没有指定告警类型，则缺省显示该接口的所有告警信息。

命令格式

display transmission-alarm interface-type interface-number [ auais | b3tca | lais | lcd | lof | lom | lop | los | lrdi | lrei | oof | pais | prdi | prei | pplm | puneq | rdool | rrool | sdbere | sfbere | trool | wlnk ] *

参数说明

interface-type interface-number：接口类型和接口编号。

auais：Administrative Unit Alarm Indication Signal，高阶告警指示。

b3tca：B3 Threshold Crossing Alarm，B3超越门限值告警指示。

lais：Line Alarm Indication Signal，线路告警指示。

lcd：Loss of Code Group Synchronization，同步码丢失告警指示。只能在10GE-WAN接口上查看到该告警的相关信息。

lof：Loss of Frame，帧丢失告警指示。

lom：Loss of Tributary Multiframe，支路单元复帧丢失告警指示。

lop：Loss of Pointer，指针失步告警指示。

los：Loss of Signal，信号失步告警指示。

lrdi：Line Remote Defect Indication，线路远端缺陷告警指示。

lrei：Line Remote Error Indication，复用段远端误块告警指示。

oof：Out of Frame，帧失步告警指示。

pais：Path Alarm Indication Signal，通道级告警指示。

prdi：Path Remote Defect Indication，通道级远端缺陷告警指示。

prei：Path Remote Error Indication，通道远端误块告警指示。

pplm：Path Payload Mismatch，通道信号标签不匹配告警指示。

puneq：Path Unequipped Indication，通道信号标签不确定告警指示。

rdool：Receive Data Out of Lock，接收数据失锁告警指示。

rrool：Receive Reference Out of Lock，接收参考时钟失锁告警指示。

sdbere：Signal Degrade Threshold Alarm，信号损耗门限告警指示。

sfbere：Signal Fail Threshold Alarm，信号失效门限告警指示。

trool：Transmit Reference Out of Lock，发送参考时钟失锁告警指示。

wlnk：WAN Interface Link Stauts Alarm，WAN接口状态告警指示。只能在10GE-WAN接口上查看到该告警的相关信息。

项目

描述

Interface:

接口名称。

Filter function:

过滤功能。

Damping function:

抑制功能。

Last reset time:

最近一次执行命令reset transmission-alarm statistic的时间。

Alarm
告警类型。可能的告警类型有：

  • auais：Administrative Unit Alarm Indication Signal，高阶告警指示。

  • b3tca：B3 Threshold Crossing Alarm，B3超越门限值告警指示。

  • lais：Line Alarm Indication Signal，线路告警指示。

  • lcd：Loss of Code Group Synchronization，同步码丢失告警指示。只能在10GE-WAN接口上查看到该告警的相关信息。

  • lof：Loss of Frame，帧丢失告警指示。

  • lom：Loss of Tributary Multiframe，支路单元复帧丢失告警指示。

  • lop：Loss of Pointer，指针失步告警指示。

  • los：Loss of Signal，信号失步告警指示。

  • lrdi：Line Remote Defect Indication，线路远端缺陷告警指示。

  • lrei：Line Remote Error Indication，复用段远端误块告警指示。

  • oof：Out of Frame，帧失步告警指示。

  • pais：Path Alarm Indication Signal，通道级告警指示。

  • prdi：Path Remote Defect Indication，通道级远端缺陷告警指示。

  • prei：Path Remote Error Indication，通道远端误块告警指示。

  • pplm：Path Payload Mismatch，通道信号标签不匹配告警指示。

  • puneq：Path Unequipped Indication，通道信号标签不确定告警指示。

  • rdool：Receive Data Out of Lock，接收数据失锁告警指示。

  • rrool：Receive Reference Out of Lock，接收参考时钟失锁告警指示。

  • sdbere：Signal Degrade Threshold Alarm，信号损耗门限告警指示。

  • sfbere：Signal Fail Threshold Alarm，信号失效门限告警指示。

  • trool：Transmit Reference Out of Lock，发送参考时钟失锁告警指示。

  • wlnk：WAN Interface Link Stauts Alarm，WAN接口状态告警指示。只能在10GE-WAN接口上查看到该告警的相关信息。

Status:
告警状态。

  • Up：表示产生了此告警。

  • Down：表示没有产生此告警或此告警消除。

  • If down：表示此告警可以使接口的物理层状态变为Down。

  • Log：表示此告警可以产生日志信息。

  • In filtering：表示此告警正在被过滤。

  • In suppressing：表示此告警正在被抑制。

  • In aging：表示此告警正在老化。

Flapping count:

振荡次数。
