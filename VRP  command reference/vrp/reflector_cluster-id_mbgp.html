reflector cluster-id（MBGP）

命令功能

reflector cluster-id命令用来配置路由反射器的集群ID。

undo reflector cluster-id命令用来恢复缺省配置。

缺省情况下，每个路由反射器使用自己的路由器ID作为集群ID。

命令格式

reflector cluster-id { decimal-value | ipv4-address }

undo reflector cluster-id

参数说明

decimal-value：指定数值形式的cluster-id值。整数形式，取值范围是1～4294967295。

ipv4-address：指定IPV4点分十进制地址形式的cluster-id值。

缺省级别

2：配置级

使用指南

通常，一个集群仅有一个路由反射器。路由反射器的路由器ID用来标识这个集群。

在一个集群中设置多个路由反射器可提高网络的稳定性，此时需要使用该命令为所有的路由反射器配置相同的集群ID，从而避免路由循环。

使用实例

# 设置本地路由器是集群中的路由反射器。配置该集群ID为50。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv4-family multicast
[Quidway-bgp-af-multicast] reflector cluster-id 50
