reset ppp compression iphc

命令功能

reset ppp compression iphc命令用来删除已经无效的IP/UDP/RTP头压缩或解压上下文存储表，并清除IP/UDP/RTP头压缩的统计信息。

命令格式

reset ppp compression iphc [ interface interface-type interface-number ]

参数说明

interface interface-type interface-number：指定的接口类型和接口编号。如POS、Serial、VT等。

用户视图

缺省级别

使用指南

如果不指定参数，将清空所有接口的IP头压缩存储表项。

使用实例

# 删除已经无效的IP/UDP/RTP头压缩或解压上下文存储表，并清除IP/UDP/RTP头压缩的统计信息

<Quidway> reset ppp compression iphc serial 1/0/0
