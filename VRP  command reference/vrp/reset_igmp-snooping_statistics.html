reset igmp-snooping statistics

命令功能

reset igmp-snooping statistics命令用来清除IGMP Snooping统计信息。

命令格式

reset igmp-snooping statistics { vlan { vlan-id | all } | vsi { name vsi-name | all} | all }

参数说明

vlan vlan-id：vlan-id表示VLAN编号，取值范围是1～4094。

name vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，长度范围是1～31。

vlan all：清除所有VLAN的统计信息。

vsi all：清除所有VSI的统计信息。

all：清除所有VSI和VLAN的统计信息。

使用指南

清除所有VLAN和VSI内的IGMP Snooping 统计信息，使用命令reset igmp-snooping statistics all 。

清除指定VLAN的IGMP Snooping统计信息，使用命令reset igmp-snooping statistics vlan vlan-id。

清除所有VLAN的IGMP Snooping统计信息，使用命令reset igmp-snooping statistics vlan all 。如果VLAN已经绑定到VSI，在配置reset igmp-snooping group vlan all时，会提示出错。

清除指定VSI的IGMP Snooping统计信息，使用命令reset igmp-snooping statistics vsi name vsi-name。

清除所有VSI的IGMP Snooping统计信息，使用命令reset igmp-snooping statistics vsi all。

使用实例

# 清除所有VLAN的IGMP Snooping统计信息。

<Quidway> reset igmp-snooping statistics

# 清除VLAN2的IGMP Snooping统计信息。

<Quidway> reset igmp-snooping statistics vlan 2
