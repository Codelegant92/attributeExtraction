cache-sa-enable

命令功能

cache-sa-enable命令用来使能SA缓存功能，收到SA消息后缓存其中包含的（S，G）项。

undo cache-sa-enable命令用来关闭SA缓存功能。

缺省情况下，路由器使能SA缓存功能。

命令格式

cache-sa-enable

undo cache-sa-enable

参数说明

无

视图

公网实例MSDP视图、VPN实例MSDP视图

缺省级别

2：配置级

使用指南

当RP收到一个新的组加入消息（*，G）时：

  • 如果RP使能了SA缓存功能，则查找SA Cache。

    • 如果SA Cache中有对应的（S，G），则RP直接加入以S为根的SPT。

    • 如果SA Cache中没有对应的（S，G），则不予处理。

  • 如果关闭了SA缓存功能：

    • 如果配置了SA请求，则RP主动向指定的MSDP peer发送SA请求消息，并等待回复。

    • 如果没有配置SA请求，则等候其MSDP peer在下一个周期发来的SA消息。

使用实例

# 在公网实例下，关闭路由器的SA缓存功能。

<Quidway> system-view
[Quidway] msdp
[Quidway-msdp] undo cache-sa-enable
