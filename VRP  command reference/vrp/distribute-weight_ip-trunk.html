distribute-weight（IP-Trunk）

命令功能

distribute-weight命令用来配置IP-Trunk成员端口的负载分担权重。

undo distribute-weight命令用来删除配置的IP-Trunk成员端口负载分担权重。

缺省情况下，成员端口的负载分担权重为1。

命令格式

distribute-weight weight-value

undo distribute-weight

参数说明

weight-value：IP-Trunk成员端口的权重。整数形式，取值范围是1～16。

视图

接口视图

缺省级别

2：配置级

使用指南

在一个IP-Trunk接口中，某成员端口的权重值占所有成员端口权重之和的比例越大，该成员链路承担的负载就越大。

一个IP-Trunk接口最多允许16个负载分担项，且一个IP-Trunk中所有成员的权重之和不能超过16。

使用实例

# 设置IP-Trunk成员端口POS1/0/0的负载分担权重为3。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] distribute-weight 3
