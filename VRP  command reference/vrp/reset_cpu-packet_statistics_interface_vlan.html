reset cpu-packet statistics interface vlan

命令功能

reset cpu-packet statistics interface vlan命令用来清除指定接口下指定VLAN的所有上送到CPU的报文的统计信息。

命令格式

reset cpu-packet statistics interface interface-type interface-number vlan vlan-id

参数说明

interface-type interface-number：接口名称和接口编号。

vlan-id：VLAN ID。整数形式，取值范围是1～4094。

视图

用户视图

缺省级别

3：管理级

使用指南

在某些情况下，需要统计一定时间内某接口的某VLAN上送CPU的流量信息，这时必须在统计开始前清除该接口的该VLAN上原有的统计信息，使接口的VLAN流量重新进行统计。

使用实例

# 清除接口GigabitEthernet 1/0/0下VLAN 1上的所有上送到CPU的报文的统计功能。

<Quidway> reset cpu-packet statistics interface gigabitethernet 1/0/0 vlan 1
