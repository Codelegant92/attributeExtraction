cd（用户视图）

命令功能

cd命令用来修改路由器当前配置用户的当前路径，修改路由器当前工作路径为指定存在的目录。

缺省情况下，为设置好的缺省工作路径。

命令格式

cd { .. | directory }

参数说明

..：返回上一级目录。如果当前目录为根目录，系统将提示找不到路径的错误信息。

directory：目标目录名。字符串形式，取值范围是1～64。

视图

用户视图

缺省级别

3：管理级

使用指南

需注意用户权限，只能切换到允许用户登录目录的子目录中。

使用实例

# 修改路由器当前工作路径为test。

<Quidway> cd test
<Quidway> pwd
257 "flash:/test" is current directory
