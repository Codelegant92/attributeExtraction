interface mfr

命令功能

interface mfr命令用来创建多链路帧中继捆绑接口或子接口并进入相应的接口视图。

undo interface mfr命令用来删除指定的多链路帧中继捆绑接口或子接口。

缺省情况下，没有多链路帧中继捆绑接口或子接口。

命令格式

interface mfr interface-number[.subnumber [ p2mp | p2p ] ]

undo interface mfr interface-number[.subnumber ]

参数说明

interface-number：多链路帧中继捆绑的接口编号，包括槽号/卡号/接口号，其中接口号取值范围是0～1023。

subnumber：多链路帧中继捆绑的子接口编号。整数形式，取值范围是1～1024。

p2mp：配置子接口的类型为点到多点，这是缺省的帧中继子接口类型。

p2p：配置子接口的类型为点到点。

使用指南

在使用命令undo interface mfr删除MFR接口之前，必须已经将所有物理接口从该MFR接口中删除。

在创建MFR子接口之前，MFR接口必须已经建立。

使用实例

# 创建一个多链路帧中继捆绑接口，并为它创建一个点到多点类型的子接口。

<Quidway> system-view
[Quidway] interface mfr 1/0/123
[Quidway-MFR1/0/123] quit
[Quidway] interface mfr 1/0/123.1
[Quidway-MFR1/0/123.1]
