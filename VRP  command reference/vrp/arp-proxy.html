arp-proxy

命令功能

arp-proxy命令用来启动sub-VLAN内或sub-VLAN间的Proxy ARP功能。

undo arp-proxy 命令用来关闭sub-VLAN内或sub-VLAN间的Proxy ARP功能。

缺省情况下，关闭Proxy ARP功能。

命令格式

arp-proxy [ inner-sub-vlan-proxy | inter-sub-vlan-proxy ] enable

undo arp-proxy [ inner-sub-vlan-proxy | inter-sub-vlan-proxy ] enable

参数说明

inner-sub-vlan-proxy：配置同一个sub-vlan内主机互访的Proxy ARP功能。

inter-sub-vlan-proxy：配置sub-VLAN间的Proxy ARP功能。

enable：使能Proxy ARP功能。

视图

VLANIF接口视图

缺省级别

2：配置级

使用指南

对于同一Super-VLAN内的不同Sub-VLAN内的主机，默认是不能互访的，如果需要互访，就必须在Super-VLAN的VLANIF接口上使能VLAN间代理。

有的交换机可以使同一VLAN内的主机隔离，如果需要互访，就必须配置VLAN内代理。

使用实例

# 使能Proxy ARP功能。

<Quidway> system-view
[Quidway] interface vlanif 2
[Quidway-Vlanif2] arp-proxy enable
