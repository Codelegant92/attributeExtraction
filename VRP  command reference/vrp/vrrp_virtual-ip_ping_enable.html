vrrp virtual-ip ping enable

命令功能

vrrp virtual-ip ping enable命令用来使能Master设备响应Ping报文。

undo vrrp virtual-ip ping enable命令用来禁止Master设备响应Ping报文。

缺省情况下，Master设备能响应Ping报文。

命令格式

vrrp virtual-ip ping enable

undo vrrp virtual-ip ping enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

根据VRRP的标准协议，无法使用Ping命令来检测备份组虚拟IP地址的通达性，此时连接到路由器的用户将无法通过Ping命令来判断目的IP地址是否被备份组使用。

注意：

在配置虚拟IP地址时，一定不要和用户主机IP地址相同。如果用户主机IP地址和备份组虚拟IP地址相同，则本网段报文都将被发送到用户主机，从而导致本网段的数据不能被正确转发。

使用实例

# 禁止Master设备响应Ping报文。

[Quidway] undo vrrp virtual-ip ping enable
