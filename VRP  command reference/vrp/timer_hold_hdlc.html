timer hold（HDLC）

命令功能

timer hold命令用来设置轮询时间的间隔。

undo timer hold命令用来恢复缺省设置。

缺省状况下，接口的链路层协议轮询时间的间隔为10秒。

命令格式

timer hold seconds

undo timer hold

参数说明

seconds：为轮询时间的间隔。整数形式，取值范围是0～32767，单位是秒。设置为0表示禁止链路检测功能。缺省情况下，轮询时间为10秒。

缺省级别

2：配置级

使用指南

链路两端设备的轮询时间应设为相同的值。如果将两端的轮询时间的间隔都设为0，则禁止链路检测功能。

如果网络的延迟比较大，或拥塞程度较高，可以适当加大轮询时间的间隔，以减少网络震荡的发生。

使用实例

# 将接口POS1/0/0的轮询时间的间隔设置为20秒。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] timer hold 20
