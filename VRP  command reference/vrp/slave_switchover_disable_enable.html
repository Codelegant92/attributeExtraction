slave switchover { disable | enable }

命令功能

slave switchover { disable | enable } 命令用来使能或禁止强制进行主备倒换。

缺省情况下，使能强制进行主备倒换。

命令格式

slave switchover { disable | enable }

参数说明

disable：禁止强制进行主备倒换。

enable：使能强制进行主备倒换。

视图

系统视图

缺省级别

2：配置级

使用指南

只有在使能强制进行主备倒换之后，slave switchover命令才能有效；如果禁止强制进行主备倒换，则用户无法通过配置命令强制进行主备倒换。

说明：

以上命令在单主控板路由器不支持。

使用实例

# 首先使能强制进行主备倒换，然后使用命令slave switchover，使系统的主用板备用板进行切换。

<Quidway> system-view
[Quidway] slave switchover enable
