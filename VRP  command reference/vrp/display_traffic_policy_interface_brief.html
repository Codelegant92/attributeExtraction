display traffic policy interface brief

命令功能

display traffic policy interface brief命令用来显示指定接口或所有接口配置的流量策略信息。如果指定的接口为二层接口，将显示该接口所在VLAN下的流量策略信息。

命令格式

display traffic policy interface brief [ interface-type [ interface-number ] ]

参数说明

interface-type interface-number：指定的接口类型和接口编号。

视图

所有视图

缺省级别

1：监控级

使用指南

display traffic policy interface brief命令可以观察到指定接口或所有接口入方向和出方向配置的流量策略信息。

使用实例

# 显示所有接口的流量策略基本信息。

<Quidway> display traffic policy interface brief
 Interface                       InboundPolicy          OutboundPolicy
 Ethernet2/1/0                      tp3                    tp4
 Ethernet2/1/1                      -                      tp6
 Ethernet2/1/1.1                    -                      tp2
 GigabitEthernet3/2/0             tp1                    -
 Vlan 1 to 100                      -                      tp4
 Vlan 200 to 300                   tp3                    -
 Pos4/2/0                            -                      tp2

#显示所有Ethernet接口的流量策略基本信息。

<Quidway> display traffic policy interface brief ethernet

 Ethernet2/1/1                  -                     tp6
 Ethernet2/1/1.1                 -                    tp2

#显示Ethernet接口ethernet 2/1/1的流量策略基本信息。
