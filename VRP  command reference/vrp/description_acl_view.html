description（ACL视图）

命令功能

description命令用对ACL进行说明，可以表示该ACL规则组的具体用途。

undo description命令用来删除对ACL规则组的描述。

命令格式

description text

undo description

参数说明

text：描述该ACL的用途。字符串形式，长度范围是1～127。

视图

ACL视图

缺省级别

2：配置级

使用指南

使用命令display acl { acl-number | all }可以查看ACL的描述。

使用实例

# 给ACL 3100增加一个描述。

<Quidway> system-view
[Quidway] acl 3100
[Quidway-acl-adv-3100] description This acl is used in gigabitethernet 1/0/0
