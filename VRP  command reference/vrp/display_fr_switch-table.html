display fr switch-table

命令功能

display fr switch-table命令用来查看路由器中帧中继当前交换的表信息。

命令格式

display fr switch-table { all | name name }

参数说明

all：查看路由器所有帧中继交换表信息。

name：查看指定交换名称的帧中继交换表信息。字符串形式，取值范围1～31。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看路由器中帧中继当前交换的表信息。

<Quidway> display fr switch-table name pvc1
PVC-Name      Status   Interface(Dlci) <-----> Interface(Dlci)
pvc1           Active   Serial3/0:0(100)        Serial1/2(300)
<Quidway> display fr switch-table all
Total PVC switch records:1
PVC-Name   Status   Interface(Dlci) <-----> Interface(Dlci)
pvc1        Active   Serial3/0:0(100)        Serial1/2(300)
表1 display fr switch-table输出信息描述

项目

描述

PVC-Name

永久虚电路的名字。

Status

虚电路的状态。

Interface(Dlci) <-----> Interface(Dlci)

虚电路两端的接口类型、编号和各自的虚电路号。
