load-balance flow

命令功能

load-balance flow命令用来配置IP主机数据和MPLS三层转发数据采用逐流负载分担。

缺省情况下，IP主机数据和MPLS三层转发数据采用逐流负载分担。

命令格式

load-balance flow [ all | slot slot-id ]

参数说明

all：将配置应用于所有的接口板。

slot slot-id：将配置应用于指定槽号的接口板。
