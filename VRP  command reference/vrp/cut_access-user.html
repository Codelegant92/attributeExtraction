cut access-user

命令功能

cut access-user命令用来按照指定标准拆除一个或多个在线用户的连接。

命令格式

cut access-user domain domain-name

cut access-user user-id start-num [ end-num ]

cut access-user username { all | hwtacacs | local | none | radius } [ user-name ]

参数说明

domain：拆除指定域名下的所有连接。

domain-name：域名，字符串形式，长度范围是1～20，不区分大小写。

user-id：根据用户ID拆除连接。

start-num：起始在线索引号，最小值为0，最大取值与具体使用的产品相关。

end-num：结束在线索引号，最小值为0，最大取值与具体使用的产品相关。此参数必须大于start-num。

username：根据用户名拆除连接。

all：所有接入用户。

local：接入用户的认证方式为本地认证。

hwtacacs：接入用户的认证方式为HWTACACS认证。

radius：接入用户的认证方式是RADIUS认证。

none：接入用户的认证方式是不认证。

user-name：完整的用户名，格式为“纯用户名@域名”，字符串形式，取值范围是1～64，不区分大小写。

使用指南

根据用户名和认证方式拆除连接时，如果有多个符合条件的连接，将同时拆除。

使用实例

# 根据用户名，拆除在线连接：

<Quidway> system-view
[Quidway] aaa
[Quidway-aaa] cut access-user username local user0
[Quidway-aaa] cut access-user username radius user0
[Quidway-aaa] cut access-user username none user0
[Quidway-aaa] cut access-user username all

# 拆除在线索引号从0到100的在线连接，包括0和100。

[Quidway-aaa] cut access-user user-id 0 100

# 拆除名为huawei的域下的所有连接。

[Quidway-aaa] cut access-user domain huawei
