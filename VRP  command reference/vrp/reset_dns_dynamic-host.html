reset dns dynamic-host

命令功能

reset dns dynamic-host命令用来清除动态域名缓存区的动态域名映射项信息。

命令格式

reset dns dynamic-host

参数说明

无

视图

用户视图

缺省级别

2：配置级

使用指南

注意：

清除DNS表项后，将无法恢复，清除之前务必仔细确认。

在确认需要清除DNS的动态域名映射信息时，执行该reset命令。

使用实例

# 清除动态域名缓存区。

<Quidway> reset dns dynamic-host
