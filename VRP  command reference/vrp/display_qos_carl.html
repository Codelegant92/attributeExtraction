display qos carl

命令功能

display qos carl命令用来显示CAR列表的某条规则或所有规则。

命令格式

display qos carl [ carl-index ]

参数说明

carl-index：CAR列表的号码，取值范围是1～199。

视图

所有视图

缺省级别

1：监控级

使用指南

display qos carl命令可以观察到的命令包括：

  • 报文优先级

  • MAC地址的详细信息

如不指定carl-index，将显示所有的CAR列表的规则。

使用实例

# 显示CAR的第一条规则。

<Quidway> display qos carl
