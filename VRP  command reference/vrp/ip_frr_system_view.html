ip frr（系统视图）

命令功能

ip frr命令用来在系统视图下使能公网的IP FRR功能。

undo ip frr命令用来取消公网的IP FRR功能。

缺省情况下，没有使能公网的IP FRR功能。

命令格式

ip frr route-policy route-policy-name

undo ip frr [ all ]

参数说明

route-policy-name：Route-Policy名称。

all：取消所有的公网和私网IP FRR功能。

视图

系统视图

缺省级别

2：配置级

使用指南

同一时间内只能使能一个策略，如果重复配置，新的配置将覆盖之前的配置。系统视图下的配置与VPN实例视图下的配置互不影响。

该命令需要同apply backup-interface和apply backup-nexthop命令配合使用。

使用实例

# 在名为ip_frr_rp的Route-Policy中指定备份出接口，并在系统视图下使能公网IP FRR功能。

<Quidway> system-view
[Quidway] route-policy ip_frr_rp permit node 10
[Quidway-route-policy] apply backup-interface GigabitEthernet1/0/0
[Quidway-route-policy] apply backup-nexthop 192.168.20.2
[Quidway-route-policy] quit
[Quidway] ip frr route-policy ip_frr_rp
