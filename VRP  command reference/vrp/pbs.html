pbs

命令功能

pbs命令用来设置帧中继虚电路的峰值突发流量（Peak burst size）。

undo pbs命令用来恢复缺省值。

命令格式

pbs [ inbound | outbound ] pbs

undo pbs [ inbound | outbound ]

参数说明

inbound：设置报文入方向的过度突发流量，本参数仅当接口使能帧中继流量监管时有效。

outbound：设置报文出方向的过度突发流量，本参数仅当接口使能帧中继流量整形时有效。

pbs：峰值突发流量，单位是bit，取值范围是0～16000000。缺省值为0 bit。

缺省级别

2：配置级

使用指南

峰值突发流量（PBS）为时间间隔Tc内报文流量超过承诺突发流量（CBS）部分的最大值。当网络发生拥塞时，这部分超出的流量将会被首先丢弃。

在使用该命令时，如果不指定参数inbound和outbound，则设置的PBS值将同时在入接口方向和出接口方向生效。

使用实例

# 设置名为test1的帧中继class的过度突发流量为32000 bits。

<Quidway> system-view
[Quidway] fr class test1
[Quidway-fr-class-test1] pbs 32000
