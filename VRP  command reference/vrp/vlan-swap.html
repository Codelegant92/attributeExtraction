vlan-swap

命令功能

vlan-swap enable命令用来在主接口下使能VLAN Swap功能。

undo vlan-swap enable命令用来去使能VLAN Swap功能。

命令格式

vlan-swap enable

undo vlan-swap enable

参数说明

无

视图

Ethernet接口视图、GE接口视图、Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

配置该命令前，应先在主接口下配置mode user-termination命令，否则系统会提示配置错误。

当主接口配置vlan-swap enable后，所有该主接口下的子接口都会自动使能该功能。

该命令会将带有两层Tag的报文进行内、外层交换，但不影响带一层Tag的报文。

使用实例

# 在主接口GE1/0/1下使能Swap功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] mode user-termination
[Quidway-GigabitEthernet1/0/1] vlan-swap enable
[Quidway-GigabitEthernet1/0/1] quit
