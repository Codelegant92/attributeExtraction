timer hello (MC-Trunk 视图)

命令功能

timer hello命令用来设置MC-Trunk的Hello报文发送周期。

undo timer hello命令用来恢复Hello报文发送周期的缺省值。

缺省情况下，Hello报文发送周期值为10，单位是100毫秒，即1秒。

命令格式

timer hello hello-value

undo timer hello

参数说明

hello hello-value：Hello报文的发送周期。整数形式，取值范围是5～100，单位是100毫秒。缺省值是10。

缺省级别

2：配置级

使用指南

如果处于备用状态的MC-Trunk在超时时间内没有收到对端发送的Hello报文，则在定时器超时后进入主用状态。此处的超时时间是对端报文中所携带的超时时间，而不是本端设置的超时时间。

超时时间 = 发送周期 × 时间倍数。

如果对端报文中没有携带超时时间，则采用本端设置的超时时间。

使用实例

# 配置MC-Trunk的报文发送周期为9。

<Quidway> system-view
[Quidway] mc-trunk 1
[Quidway-mc-trunk-1] timer hello 9
