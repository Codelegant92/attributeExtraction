display fib

命令功能

display fib命令用来查看转发信息表的信息。

命令格式

display fib [ slot-id ] [ vpn-instance vpn-instance-name ] [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

slot-id：槽位号。根据路由设备硬件的不同而具有不同的取值范围。

vpn-instance vpn-instance-name：VPN实例的名称。字符串形式，区分大小写，长度范围是1～31。

verbose：查看转发信息表的详细信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

对于display fib [ slot-id ] | begin regular-expression：显示从包含字符串regular-expression的那一行开始到缓冲区结束的所有行。

对于display fib [ slot-id ] | exclude regular-expression：显示不包含字符串regular-expression的行。

对于display fib [ slot-id ] | include regular-expression：只显示包含字符串regular-expression的行。

该命令以列表的方式输出转发信息表，每一行代表一条路由。

display fib命令以列表的方式输出转发信息表，每一行代表一条路由。

display fib verbose命令显示转发表的详细信息。

<Quidway> display fib

<Quidway> display fib | begin 169.254.0.0

<Quidway> display fib | include GE1/0/0

项目

描述

Destination/Mask

目的地址/掩码长度

Nexthop

下一跳

Flag
当前标志，G、H、U、S、D、B、T的组合。

  • G（Gateway网关路由）：表示下一跳是网关。

  • H（Host主机路由）：表示下一跳为主机。

  • U（Up可用路由）：表示该路由状态是Up。

  • S（Static静态路由）

  • D（Dynamic动态路由）

  • B（Black Hole黑洞路由）：表示下一跳是空接口。

  • T（该路由已被LSPM添加了token值）

TimeStamp

时间戳，表示FIB生成时间与系统启动时间的相对值。

Interface

到目的地址的出接口。

TunnelID

表示转发表项索引。用于连接上行板和下行板之间的转发表项。该值不为0时，表示匹配该项的报文通过MPLS隧道转发。该值为0时，表示报文不通过MPLS隧道转发。
