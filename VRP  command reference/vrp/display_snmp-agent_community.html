display snmp-agent community

命令功能

display snmp-agent community命令用来显示配置的团体名信息。

命令格式

display snmp-agent community [ read | write ]

参数说明

read：显示只读访问权限的团体名信息。

write：显示读写访问权限的团体名信息。

视图

所有视图

缺省级别

1：监控级

使用指南

在对管理实体进行配置时，需要查看代理的团体名，使用display snmp-agent community命令来可以显示组名、存储方式、ACL等。

使用实例

# 显示当前配置的团体名。

<Quidway> display snmp-agent community
   Community name:aa
       Group name:aa
       Acl:2000
       Storage-type: nonVolatile
   Community name:public
       Group name:public
       Storage-type: nonVolatile
