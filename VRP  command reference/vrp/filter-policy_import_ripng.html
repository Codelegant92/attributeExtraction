filter-policy import (RIPng)

命令功能

filter-policy import命令用来指定接收路由过滤策略，符合过滤条件的路由才能被接收。

undo filter-policy ipv6-prefix命令用来取消对接收的路由信息的过滤。

命令格式

filter-policy { acl6-number | ipv6-prefix ipv6-prefix-name } import

undo filter-policy import

参数说明

acl6-number：基本ACL6的序号。整数形式，取值范围是2000～2999。

ipv6-prefix ipv6-prefix-name：IPv6地址前缀列表名称。字符串形式，不支持空格，区分大小写，长度范围是1～169。

使用指南

只有通过过滤的路由才能被加入到RIPng路由表。

使用实例

# 用地址前缀列表Filter1过滤收到的RIPng更新报文。

<Quidway> system-view
[Quidway] ripng 1
[Quidway-ripng-1] filter-policy ipv6-prefix Filter1 import
