radius-server type

命令功能

radius-server type命令用来配置RADIUS服务器使用的协议版本。

缺省情况下，服务器使用标准RADIUS协议（standard）。

命令格式

radius-server type { standard | portal }

参数说明

standard：服务器使用标准的RADIUS协议。

portal：服务器使用Portal RADIUS协议（也称为RADIUS+，V1.1）。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

只有当该RADIUS Server模板没有用户使用时，才能改变此配置。

使用实例

# 配置服务器使用RADIUS+协议。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server type portal
