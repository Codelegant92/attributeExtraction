reset msdp statistics

命令功能

reset msdp statistics命令用来在不重置MSDP对等体的情况下清除一个或多个MSDP对等体的统计信息。

命令格式

reset msdp [ vpn-instance vpn-instance-name | all-instance ] statistics [ peer-address ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

peer-address：表示MSDP对等体地址。点分十进制形式。如果不指定peer-address，则清除所有MSDP对等体的统计信息。

使用指南

使用reset msdp statistics命令后，统计信息自动清除，但是对等体之间的TCP连接不会断开，MSDP服务不会中断。

如果不指定vpn-instance或all-instance，则只删除公网中的MSDP对等体的统计信息。

如果希望在清除MSDP对等体的统计信息的同时重新建立MSDP对等体之间的TCP连接，请使用reset msdp peer命令。

使用实例

# 清除公网实例中MSDP对等体125.10.7.6的统计信息。

<Quidway> reset msdp statistics 125.10.7.6
