ip route-static default-preference

命令功能

ip route-static default-preference命令用来设置IPv4静态路由的缺省优先级。

undo ip route-static default-preference命令用来恢复IPv4静态路由的缺省优先级。

缺省情况下，IPv4静态路由协议的优先级为60。

命令格式

ip route-static default-preference preference

undo ip route-static default-preference

参数说明

preference：指定IPv4静态路由协议缺省优先级的值。整数形式，取值范围是1～255。
