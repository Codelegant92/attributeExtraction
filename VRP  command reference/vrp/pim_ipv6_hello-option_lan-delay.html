pim ipv6 hello-option lan-delay

命令功能

pim ipv6 hello-option lan-delay命令用来设置共享网络中传递剪枝消息的时延。

undo pim ipv6 hello-option lan-delay命令用来恢复缺省值。

缺省情况下，共享网络中传递剪枝消息的时延是500毫秒。

命令格式

pim ipv6 hello-option lan-delay interval

undo pim ipv6 hello-option lan-delay

参数说明

interval：指定共享网络中传递剪枝消息的时延。整数形式，取值范围是1～32767，单位是毫秒。

使用指南

此命令与IPv6 PIM视图下的hello-option lan-delay（IPv6）含义是相同的，只是视图不同。缺省情况下，如果接口上没有配置，取IPv6 PIM视图下配置的值；否则取接口视图下配置的值。

请参见IPv4 PIM配置命令中的pim hello-option lan-delay。

使用实例

# 设置接口GE1/0/0的局域网剪枝时延是200毫秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 hello-option lan-delay 200
