dialer user

命令功能

dialer user命令用来设置对端用户名，以便接收呼叫时能验证请求。

undo dialer user命令用来删除对端用户名。

缺省无对端用户名

命令格式

dialer user username

undo dialer user

参数说明

username：对端用户名，为长度为1～32的字符串，用于PPP验证。

视图

Dialer接口视图

缺省级别

2：配置级

使用指南

该命令仅在Dialer接口上有效。

当Dialer接口封装PPP时，利用PPP验证得到的对端用户名决定接收呼叫时的Dialer接口，此时可以选择配置dialer number命令。

当Dialer接口封装帧中继时，由于取消了用户名的协商过程，被叫端将根据接收到的主叫端拨号串来区分不同的Dialer接口，因此该命令无需配置，但是此时必须配置dialer number命令。

使用实例

# 设置对端用户名为“RouterB”。

<Quidway> system-view
[Quidway] interface dialer 3
[Quidway-Dialer3] dialer user RouterB
