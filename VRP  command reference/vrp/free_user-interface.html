free user-interface

命令功能

free user-interface命令用来断开与指定用户界面的连接。

命令格式

free user-interface { ui-number | ui-type ui-number1 }

参数说明

ui-number：用户界面绝对编号。最小值为0，最大值比系统支持的用户界面总数小1。不同设备用户界面取值范围不同。

ui-type：用户界面类型。

ui-number1：用户界面的相对编号。

用户视图

缺省级别

3：管理级

使用指南

也可以通过连续使用quit（所有视图）命令，最后从用户视图退出，从而断开与当前用户界面的连接。

使用实例

# 断开与user-interface 0的连接。

<Quidway> free user-interface 0
Warning: User interface con0 will be freed. Continue? [Y/N] y
