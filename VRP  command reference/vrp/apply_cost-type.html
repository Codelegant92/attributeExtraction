apply cost-type

命令功能

apply cost-type命令用来设置路由的开销类型。

undo apply cost-type命令用来取消设置。

缺省情况下，不设置路由开销类型。

命令格式

apply cost-type { external | internal | type-1 | type-2 }

undo apply cost-type

参数说明

external：IS-IS的外部路由开销类型。

internal：IS-IS的内部路由开销类型或者设置BGP路由的MED值为下一跳的IGP度量。

type-1：OSPF的外部Type-1路由。

type-2：OSPF的外部Type-2路由。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

apply cost-type internal策略应用在IS-IS路由和BGP路由的不同作用：

  • 应用在IS-IS路由时

    设置路由为IS-IS的内部路由类型。

  • 应用在BGP路由时

    路由器从IBGP对等体学到的路由在通告给EBGP对等体时，如果配置apply cost-type inernal命令，则路由器会将向EBGP对等体通告的路由的MED值设置为该路由的下一跳的IGP开销值。

使用实例

# 设置开销类型为OSPF External Type-1。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply cost-type type-1
