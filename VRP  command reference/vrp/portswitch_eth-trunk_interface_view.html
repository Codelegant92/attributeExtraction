portswitch（Eth-Trunk接口视图）

命令功能

portswitch命令用来将Eth-Trunk接口从三层模式切换到二层模式。

undo portswitch命令用来将Eth-Trunk接口从二层模式切换到三层模式。

命令格式

portswitch

undo portswitch

参数说明

无

视图

Eth-Trunk接口视图

缺省级别

2：配置级

使用指南

只有二层的Eth-Trunk才可以绑定或去绑定VLAN。当Eth-Trunk接口由三层模式切换到二层模式时，Eth-Trunk的三层功能和标识被禁止，采用系统MAC地址。

执行本命令时，接口下除了shutdown、undo shutdown和description，不能有其它配置。

连续执行portswitch、undo portswitch命令的最小时间间隔是30秒。

使用实例

# 将接口Eth-Trunk1切换为二层模式。

<Quidway> system-view
[Quidway] interface eth-trunk 1
[Quidway-Eth-Trunk1] portswitch
