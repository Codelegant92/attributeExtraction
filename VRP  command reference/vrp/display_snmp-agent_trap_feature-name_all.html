display snmp-agent trap feature-name all

命令功能

display snmp-agent trap feature-name all命令用来显示指定特性下向网管发送告警的使能情况和海量告警标志。

命令格式

display snmp-agent trap feature-name feature-name all

参数说明

feature-name：产生告警的特性名称。

所有视图

缺省级别

1：监控级

使用指南

在使能了向网管发送告警后，执行命令display snmp-agent trap feature-name all可以查看到在指定特性下所有告警的使能情况和该告警是否是海量告警。

使用实例

# 显示MSDP特性下所有告警的使能情况和海量标志。

<Quidway> display snmp-agent trap feature-name msdp all 
------------------------------------------------------------------------------
Feature name: MSDP
Trap number : 2
------------------------------------------------------------------------------
Trap name                       Status      Excessive Flag
establish                       on          N
backward                        off         N
表1 display snmp-agent trap feature-name all命令输出信息描述

项目

描述

Feature name

产生告警的特性名称。

Trap number

该特性下包涵的告警数量。

Trap name

告警名称。

Status
向网管发送告警的使能状态。
  • on：使能向网管发送告警。
  • off：去使能向网管发送告警。

Excessive Flag
海量告警的标志。
  • Y:表示该告警是海量告警。
  • N:表示该告警是非海量告警。
