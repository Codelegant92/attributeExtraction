scramble（POS接口视图）

命令功能

scramble命令用来使能POS接口对载荷数据的加扰功能。

undo scramble命令用来禁止加扰功能。

缺省情况下，使能POS接口对载荷数据的加扰功能。

命令格式

scramble

undo scramble

参数说明

无

视图

POS接口视图

缺省级别

2：配置级

使用指南

设置数据传输的加扰方式，注意与传输设备保持一致。

使用实例

# 禁止POS接口对载荷数据的加扰功能。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] undo scramble
