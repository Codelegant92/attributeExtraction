c-rp priority（IPv4）

命令功能

c-rp priority命令用来配置C-RP的全局性优先级。

undo c-rp priority命令用来恢复该优先级为缺省值。

缺省情况下，C-RP的全局性优先级是0。

命令格式

c-rp priority priority

undo c-rp priority

参数说明

priority：表示C-RP的全局性优先级，优先级数值越大，优先级越低。整数形式，取值范围是0～255，缺省值是0。

使用指南

该命令和c-rp interface-type interface-number priority priority命令功能完全相同，差别只是作用范围不同。c-rp priority priority命令具有全局性，对整个路由器有效；c-rp interface-type interface-number priority priority命令只对指定接口有效。

系统优先采用c-rp interface-type interface-number priority priority命令的配置。当未执行c-rp interface-type interface-number priority priority命令时，才采用c-rp priority priority命令的配置。

从众多C-RP中竞选出RP的规则如下：

  • C-RP接口地址掩玛最长者获胜。

  • C-RP优先级较高者获胜；

  • 如果优先级相同，则执行Hash函数，计算结果较大者获胜；

  • 如果以上都相同，则C-RP地址较大者获胜。

使用实例

# 在公网实例下，配置C-RP的全局性优先级为5。

<Quidway> system-view
[Quidway] pim
[Quidway-pim] c-rp priority 5
