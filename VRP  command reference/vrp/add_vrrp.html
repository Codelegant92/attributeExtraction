add vrrp

命令功能

add vrrp命令用来添加VRRP备份组到VRRP管理组中。

undo add vrrp命令用来删除VRRP管理组中的VRRP备份组成员。

命令格式

add vrrp vrid vrid [ data [ transfer-only ] ] to vrrp-group vrrp-group-id

undo add vrrp vrid vrid to vrrp-group vrrp-group-id

参数说明

vrid vrid：VRRP备份组ID，取值范围是1～255。

vrrp-group vrrp-group-id：VRRP管理组ID，取值范围是1～16。

data：数据通道。

data transfer-only：传输通道。

使用指南

对于VRRP管理组中的通道管理，有以下几种方式：

  • 属性data表明是一个数据通道，如果VRRP备份组以数据通道形式加入VRRP管理组，则VRRP管理组通过这个数据通道接收发送报文。

  • 属性data transfer-only表明是一个传输通道，VRRP管理组可以通过这个通道接收发送报文，但是这个通道的状态变化不影响VRRP管理组的优先级的改变。配置传输通道是为了提高VRRP管理组报文传输的可靠性。

  • 如果没有配置这两个属性，则VRRP备份组只是以普通成员的身份加入VRRP管理组，普通成员只能接收报文，不能发送。

使用实例

# 接口Ethernet1/0/0上VRID为10的VRRP备份组添加到VRRP管理组1，并且作为数据通道。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] vrrp vrid 10 virtual-ip 1.1.1.10
[Quidway-GigabitEthernet1/0/0] add vrrp vrid 10 data to vrrp-group 1
