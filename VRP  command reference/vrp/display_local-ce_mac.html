display local-ce mac

命令功能

display local-ce mac命令用来查看本地CE的以太接口MAC地址及VLAN ID信息。

命令格式

display local-ce mac [ interface-type interface-number ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-type：接口类型。此接口类型仅为Ethernet、GigabitEthernet、Eth-Trunk或Virtual-Ethernet。

interface-number：接口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

display local-ce mac命令仅用于封装为ip-interworking类型的L2VPN方式中。

执行display local-ce mac命令后，如果不指定接口，将显示所有接口的信息。

使用实例

# 查看本地CE的以太接口MAC地址及VLAN ID信息。

<Quidway> display local-ce mac

项目

描述

INTERFACE

封装了interworking类型的l2vpn的以太网接口名称。

MAC ADDRESS

本地CE的MAC地址。

EXPIRE

老化时间，单位为分钟。

TYPE
表示MAC地址的学习方式：

  • S：静态；

  • D：动态；

  • B：广播方式。

VLAN

该MAC地址所属的VLAN ID。

PVC

配置IPoEoA后，VE接口的VPI和VCI。

IP ADDRESS

动态学到的本地CE的IP地址。
