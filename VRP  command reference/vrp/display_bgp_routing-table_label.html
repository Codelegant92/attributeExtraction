display bgp routing-table label

命令功能

display bgp routing-table label命令用来查看BGP路由表中的标签路由信息。

命令格式

display bgp routing-table label [ statistics ] [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv4 { all | vpn-instance vpn-instance-name } routing-table [ statistics ] label [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp ipv6 routing-table [ statistics ] label [ | count ] [ | { begin | exclude | include } regular-expression ]

display bgp vpnv6 { all | vpn6-instance vpn6-instance-name } routing-table [ statistics ] label [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

statistics：查看标签路由的统计信息。

vpnv4：查看VPNv4的标签路由信息。

all：查看所有VPN实例的标签路由信息。

vpn-instance vpn-instance-name：VPN实例名。

ipv6：查看BGP4+的标签路由信息。

vpnv6：查看VPNv6的标签路由信息。

vpn6-instance vpn6-instance-name：VPN6实例名。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
