display aspf interface

命令功能

display aspf interface命令用来查看接口上配置ASPF策略。

命令格式

display aspf interface

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display aspf interface命令查看的信息包括接口上出、入方向上的ASPF策略。

使用实例

# 下面是display aspf interface的输出。

<Quidway> display aspf interface
[ASPF Interface Configuration]
    Interface         Inbound-Policy      Outbound-Policy
    ---------         --------------      ---------------
    Ethernet0         1                   none
表1 ASPF接口配置信息显示描述表

域名

描述

InboundPolicy

入方向的ASPF策略

OutboundPolicy

出方向的ASPF策略
