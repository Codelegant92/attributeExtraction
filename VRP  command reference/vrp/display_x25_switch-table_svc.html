display x25 switch-table svc

命令功能

display x25 switch-table svc命令用来显示X.25交换路由表。

命令格式

display x25 switch-table svc { dynamic | static }

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取SVC的交换路由表。用户可以根据这些信息进行接口的故障诊断等。

使用实例

# 显示X.25交换路由表

<Quidway> display x25 switch-table svc static
Number Destination Substitute-src Substitute-dst CUD   SwitchTo(type/name)
1      11                                                        I/Serial2/0/0
2      22                                                        I/Serial2/1/0
3      133                                                       H/hg1
4      132                                                       T/123.123.123.123
5      133                                                       T/123.123.123.123
6      111             222              333                     T/4.4.4.4
Total of static svc is 6.
The item type of SwitchTo meaning:
I: interface   H: hunt-group   T: xot
表1 display x25 switch-table svc命令输出信息描述

项目

描述

Number

该条路由在路由表中的序号。

Substitute-src

替换后的源X.121地址，内容为空表示不做替换。

Substitute-dst

替换后的目的X.121地址，内容为空表示不做替换。

CUD

Call User Data，呼叫用户数据。

SwitchTo

该条路由的转发地址，包括接口、XOT通道和搜索群。
