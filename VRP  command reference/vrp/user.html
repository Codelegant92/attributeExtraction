user

命令功能

user命令用来更改当前的FTP用户，重新登录FTP服务器。

命令格式

user user-name [ password ]

参数说明

user-name：登录用户名。字符串形式，长度范围是1～64。

password：登录密码.字符串形式，取值范围是1～16。

视图

FTP客户端视图。

缺省级别

3：管理级

使用指南

通过该命令，VRP提供了在不退出FTP客户端视图的情况下，以其他的用户名登录到FTP服务器。运行该命令后所建立的FTP连接，与执行ftp命令建立的FTP连接完全相同。

使用实例

# 以FTP服务器上用户名为tom，密码为bjhw的用户身份登录。

<Quidway> ftp 172.16.104.110
Trying ftp 172.16.104.110
Press CTRL+K to abort
Connected to ftp 172.16.104.110
220 FTP service ready.
User(ftp 172.16.104.110:(none)):huawei
331 Password required for huawei
Password:
230 User logged in.
[ftp] user tom bjhw
331 Password required for tom.
230 User logged in.
