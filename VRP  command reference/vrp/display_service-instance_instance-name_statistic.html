display service-instance instance-name statistic

命令功能

display service-instance instance-name statistic命令用来查看业务实例的流量统计信息。

命令格式

display service-instance instance-name service-instance-name statistic [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

instance-name service-instance-name：指定业务实例的名字。长度范围是1～31个字符。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

业务实例的统计信息是从启动流量统计开关（statistics enable（业务实例视图））到执行该命令这段时间段内的用户侧的流量信息统计，包括各个业务实例的收、发包数、收、发字节数。

在查看业务实例的统计信息之前，流量统计开关必须打开。
