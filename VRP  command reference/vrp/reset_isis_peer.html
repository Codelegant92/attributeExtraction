reset isis peer

命令功能

reset isis peer命令用来重置IS-IS的特定邻居。本命令在需要重建某个特定邻居的情况下使用。

命令格式

reset isis peer system-id [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ]

参数说明

system-id：IS-IS邻居的系统ID。字符串类型，长度范围是12～32。

process-id：指定要清除数据结构的IS-IS进程号。整数类型，取值范围是1～65535。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程（对于缺省的VPN实例，该参数可选）。vpn-instance-name为IPv4 VPN实例的名称，字符串形式，不支持空格，区分大小写，长度范围是1～31。

vpn-instance default：除IPv4 VPN实例外的所有IS-IS进程，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：指定IPv6 VPN的IS-IS多实例进程（对于缺省的VPN实例，该参数可选）。vpn6-instance-name为IPv6 VPN实例的名称，字符串形式，不支持空格，区分大小写，长度范围是1～31。

vpn6-instance default：除IPv6 VPN实例外的所有IS-IS进程，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。
