label-distribution

命令功能

label-distribution命令用来配置标签分配控制方式。

undo label-distribution命令用来恢复缺省设置。

缺省情况下，标签分配控制方式为有效模式（ordered）。

命令格式

label-distribution { independent | ordered }

undo label-distribution

参数说明

independent：标签分配控制方式为独立模式。即LSR未收到下游的标签映射消息时，也可以自主地为邻居分配标签。

ordered：标签分配控制方式为有序模式。即只有当LSR收到某一特定FEC下一跳的特定标签映射消息、或者此LSR是某一特定FEC的LSP的出口节点时，LSR才向上游发送标签映射消息。

视图

MPLS-LDP-VPN实例视图、MPLS-LDP视图

缺省级别

2：配置级

使用指南

只有在系统视图下使能了MPLS LDP后，该命令才会生效。

说明：

建立LSP时，VRP采用的组合是：du＋ordered＋liberal。

使用实例

# 配置标签分发模式为有序模式。

<Quidway> system-view
[Quidway] mpls ldp
[Quidway-mpls-ldp] label-distribution ordered
[Quidway-mpls-ldp] quit
