mpls te link administrative group

命令功能

mpls te link administrative group命令用来为接口设置链路管理属性。

undo mpls te link administrative group命令用来删除配置。

缺省情况下，链路管理属性的值为0x0。

命令格式

mpls te link administrative group value

undo mpls te link administrative group

参数说明

value：链路属性，32比特，在选路时将与隧道的affinity位进行比较。取值范围是0x0～0xFFFFFFFF，表示32个属性，每个属性占一位。

缺省级别

2：配置级

使用指南

接口属性被全局范围内扩散，从而可以用作隧道源端的路径选择标准。

更改链路的管理组属性配置，仅对新创建的LSP生效，不影响已建立的LSP。

该命令仅在TE Tunnel所经链路接口出方向配置即可

使用实例

# 将链路管理属性标志设置为0x0101。

<Quidway> system-view
[Quidway] interface pos1/0/0
[Quidway-Pos1/0/0] mpls te link administrative group 101
