remote-vpn-target refresh

命令功能

remote-vpn-target refresh用来更新VPN-Target。

命令格式

remote-vpn-target refresh

参数说明

无

视图

VSI-BGP视图

缺省级别

2：配置级

使用指南

VPLS故障定位时，可以使用remote-vpn-target refresh命令来手动更新远端标签。

使用实例

# 手动更新远端标签。

<Quidway> system-view
[Quidway] vsi company2
[Quidway-vsi-company2] pwsignal bgp
[Quidway-vsi-company2-bgp] remote-vpn-target refresh
