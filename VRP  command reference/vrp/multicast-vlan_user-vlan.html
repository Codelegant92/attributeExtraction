multicast-vlan user-vlan

命令功能

multicast-vlan user-vlan命令用来设置组播VLAN和用户VLAN的对应关系。

undo multicast-vlan user-vlan命令用来取消组播VLAN和用户VLAN的对应关系。

缺省情况下，组播VLAN没有对应的用户VLAN。

命令格式

multicast-vlan user-vlan { { vlan-id1 [to vlan-id2 ] } & <1-10> }

undo multicast-vlan user-vlan { all | { vlan-id1 [ to vlan-id2 ] } &<1-10> }

参数说明

vlan-id1：表示用户VLAN的编号，要求小于vlan-id2。整数形式，取值范围是1～4094。

vlan-id2：表示用户VLAN的编号，要求大于vlan-id1。整数形式，取值范围是1～4094。

all：表示所有的用户VLAN。

缺省级别

使用指南

配置组播VLAN复制功能时需要配置这条命令。

一个用户VLAN只能和一个组播VLAN绑定。

一个组播VLAN可以绑定多个用户VLAN。系统支持每个组播VLAN最多可以绑定512个用户VLAN。

当组播VLAN和用户VLAN绑定后，则不允许直接删除组播VLAN或者用户VLAN，需要解除绑定才允许删除。

使用实例

# 配置组播VLAN 10和用户VLAN 20的对应关系。

<Quidway> system-view
[Quidway] igmp-snooping enable 
[Quidway] vlan 10
[Quidway-vlan10] igmp-snooping enable
[Quidway-vlan10] multicast-vlan enable
[Quidway-vlan10] multicast-vlan user-vlan 20

# 配置组播VLAN 11和用户VLAN 2～VLAN 10的对应关系。

<Quidway> system-view
[Quidway] igmp-snooping enable 
[Quidway] vlan 11
[Quidway-vlan11] igmp-snooping enable
[Quidway-vlan11] multicast-vlan enable
[Quidway-vlan11] multicast-vlan user-vlan 2 to 10
