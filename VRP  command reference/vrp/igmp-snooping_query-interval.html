igmp-snooping query-interval

命令功能

igmp-snooping query-interval命令用来配置当前VLAN/VSI内的IGMP通用查询消息发送时间间隔。

undo igmp-snooping query-interval命令用来恢复该配置的缺省值。

命令格式

igmp-snooping query-interval query-interval

undo igmp-snooping query-interval

参数说明

query-interval：IGMP通用查询消息发送时间间隔，取值范围是1～65535，单位为秒，缺省值为60秒。

视图

VLAN视图、VSI视图

缺省级别

2：配置级

使用指南

端口收到report后，将组超时定时器设置为GMI(Group Membership Interval)，GMI = ((Robustness Variable)×(Query Interval)) + (Query Response Interval)。本命令用来设置上述公式中的Query Interval。

在VLAN视图下配置时，以下情况会导致配置不成功：

  • VLAN已经绑定了VSI；

  • VLAN内包含VE接口。

使用实例

# 配置VLAN 2内的IGMP通用查询消息发送时间间隔为100秒。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] igmp-snooping query-interval 100
