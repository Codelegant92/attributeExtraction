display mld group ssm-mapping

命令功能

display igmp group ssm-mapping命令用来查看通过ssm-mapping（MLD视图）命令配置了映射规则的的组播组信息。

命令格式

display mld group [ ipv6-group-address | interface interface-type interface-number ] * ssm-mapping [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

ipv6-group-address：指定组播组地址。

interface interface-type interface-number：显示指定接口上有SSM映射关系的组播组信息。interface-type interface-number表示接口类型和编号。

verbose：显示有SSM映射关系的组播组的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

Total entry on this router

路由器上SSM映射的组播组的个数

Interface group report information of VPN-Instance

接口上组播组信息所属的VPN实例

GigabitEthernet4/2/0(FE80::DD:84)

配置了SSM映射的接口（接口地址）

Total entry on this interface

接口上SSM映射的组播组个数

Total 1 MLD SSM-Mapping Group reported

总共有1个SSM映射组播组

Group

组播组地址

Uptime
发现组播组的运行时间。时间格式是：

  • 小于等于24小时：“时：分：秒”

  • 大于24小时但小于等于一周：“天：时”

  • 大于一周：“周：天”

Expires

从MLD组列表删除记录的预期时间间隔

Last reporter

报告组成员关系的最后主机的链路本地地址

Last-listener-query-counter

发送最后侦听者查询消息的次数

Last-listener-query-timer-expiry

发送最后侦听者查询消息的时间间隔

Group mode

对组播组的过滤模式，Exclude或Include

Version1-host-present-timer-expiry

MLDv1主机的超时时间
