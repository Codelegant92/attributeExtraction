static-lsp egress

命令功能

static-lsp egress命令用来将本LSR配置为静态LSP的Egress节点。

undo static-lsp egress命令用来恢复为缺省配置。

缺省情况下，没有配置静态LSP的Egress节点。

命令格式

static-lsp egress lsp-name incoming-interface interface-type interface-number in-label in-label [ lsrid ingress-lsr-id tunnel-id tunnel-id ]

undo static-lsp egress lsp-name

参数说明

lsp-name：LSP名称。字符串形式，区分大小写，不支持空格，长度范围是1～19个字符。

incoming-interface interface-type interface-number：接口类型和编号。

in-label：入标签值。整数形式，取值范围是16～1023。

lsrid ingress-lsr-id：隧道的LSR ID。点分十进制格式。

tunnel-id tunnel-id：隧道ID。取值范围是1～1024。

使用指南

配置完静态LSP后，可以执行display mpls static-lsp命令查看该静态LSP是否建立成功。

使用实例

# 配置本LSR作为名为“bj-sh”的静态LSP的Egress节点。

<Quidway> system-view
[Quidway] static-lsp egress bj-sh incoming-interface pos 1/0/0 in-label 233
