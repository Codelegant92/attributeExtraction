display rpr timer interface ring-if

命令功能

display rpr timer interface ring-if命令用来显示本地节点的所有的RPR定时器配置信息。

命令格式

display rpr timer interface ring-if interface-number

参数说明

interface-number：接口编号。取值范围是0～31。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 显示RPR接口1下的定时器信息。

<Quidway> display rpr timer interface ring-if 1
Protection wtr timers : timeout:10(s), time remain:0(s)
Protection discovery timers: txfasttimout:10(ms), txslowtimeout:1000(ms)
Protection attribute timers:1(s)
Protection Keepalivetimeout timers:3(ms)
Protection holdoff timers: west:0(ms), east:0(ms)
表1 display rpr timer interface ring-if命令输出信息描述

项目

描述

Protection wtr timers

节点的等待恢复时间，单位为s。

timeout

本地节点的WTR（Wait to Restore）超时时间，单位为s。

time remain

节点剩余的等待恢复时间，单位为s。

Protection discovery timers

TP/TC帧的转发时间，单位为ms。

txfasttimeout

TP/TC帧快发定时器超时时间，单位为ms。

txslowtimeout

TP/TC帧慢发定时器超时时间，单位为ms。

Protection attribute timers

属性发现定时器（ATD定时器）表示节点发送ATD帧的间隔时间，单位为s。

Protection Keepalivetimeout timers

节点的keepalive超时时间，单位为ms。

Protection holdoff timers

节点的故障延时上报时间，单位为ms。

流量路径分为：

  • west：逆时针方向

  • east：顺时针方向
