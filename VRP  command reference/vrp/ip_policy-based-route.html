ip policy-based-route

命令功能

ip policy-based-route命令用来在接口上使能策略路由。

undo ip policy-based-route命令用来去使能接口上已有的策略路由。

缺省情况下，接口没有使能策略路由。

命令格式

ip policy-based-route policy-name

undo ip policy-based-route policy-name

参数说明

policy-name：策略名称。字符串形式，长度范围是1～19。
