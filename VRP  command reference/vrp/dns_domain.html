dns domain

命令功能

dns domain命令用来添加域名后缀。

undo dns domain命令用来删除域名后缀。

命令格式

dns domain domain-name

undo dns domain [ domain-name ]

参数说明

domain-name：域名后缀。字符串形式，不支持空格，长度范围是1～20，可以设定为包含数字、字母和下划线“_”或“.”的组合。

使用指南

用户所访问的一些网站的后缀往往都是相同的。如huawei.com.cn。为了方便用户使用，可以设定一个domain后缀为com.cn，这样用户在访问huawei.com.cn时可以敲入命令ping huawei。系统先查找字符“huawei.com.cn”对应的IP地址，如果没有得到回应，就发送huawei进行解析查找“huawei”对应的IP地址。

系统最多支持10个域名后缀。在删除域名后缀的时候，输入后缀名，将删除指定后缀，否则删除全部后缀。

使用实例

# 配置一个域名后缀com.cn。

<Quidway> system-view
[Quidway] dns domain com.cn
