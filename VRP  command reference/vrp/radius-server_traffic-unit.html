radius-server traffic-unit

命令功能

radius-server traffic-unit命令用来设置RADIUS服务器使用的流量单位。

缺省情况下，使用字节做为流量的单位（byte）。

命令格式

radius-server traffic-unit { byte | kbyte | mbyte | gbyte }

参数说明

byte：以字节做为流量的单位。

kbyte：以千字节做为流量的单位。

mbyte：以兆（Mega）字节做为流量的单位。

gbyte：以吉（Giga）字节做为流量的单位。

视图

RADIUS模板视图

缺省级别

2：配置级

使用指南

只有当该RADIUS Server模板没有用户使用时，才能改变此配置。

此项配置对于非字节单位的流量单位以及使用标准RADIUS协议的服务器（standard类型）无效。

使用实例

# 设置RADIUS服务器使用千字节做为流量单位。

<Quidway> system-view
[Quidway] radius-server template test1
[Quidway-radius-test1] radius-server traffic-unit kbyte
