mpls rsvp-te hello

命令功能

在MPLS视图下，mpls rsvp-te hello命令用来全局启动Hello机制；全局启动后，在接口视图下，mpls rsvp-te hello命令用来启动接口的Hello机制。

在MPLS视图下，undo mpls rsvp-te hello命令用来关闭全局和接口的Hello机制；在接口视图下，undo mpls rsvp-te hello命令用来关闭接口的Hello机制。

缺省情况下，不启动全局或接口的Hello机制。

命令格式

mpls rsvp-te hello

undo mpls rsvp-te hello

参数说明

无

视图

MPLS视图、接口视图

缺省级别

2：配置级

使用指南

启动Hello机制前，应先使能RSVP-TE。

使用实例

# 启动Hello机制。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] mpls rsvp-te hello

# 在接口视图下启动Hello特性。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] mpls rsvp-te hello
