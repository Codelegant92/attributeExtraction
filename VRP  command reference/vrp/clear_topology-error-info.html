clear topology-error-info

命令功能

clear topology-error-info命令用来清除集群拓扑中的问题链路信息。

命令格式

clear topology-error-info

参数说明

无

视图

集群视图

缺省级别

2：配置级

使用指南

本命令只能在命令交换机上执行。

解决集群中对应成员交换机的故障后，需要手工执行本命令，清除集群拓扑信息中的问题链路信息。

# 清除问题链路及其子链路
