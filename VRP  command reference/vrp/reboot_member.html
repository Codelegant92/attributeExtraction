reboot member

命令功能

reboot member命令用来重新启动指定的成员交换机。

命令格式

reboot member { member-number | mac-address mac-address } [ eraseflash ]

参数说明

member-number：指定成员交换机的成员编号。十进制整数形式，取值范围是1～255。

mac-address mac-address：指定成员交换机的MAC地址。mac-address是用16进制表示的MAC地址，格式为H-H-H，每个“H”代表4位16进制数。

eraseflash：设置成员交换机在重新启动后，擦除本成员交换机本次启动时加载的配置文件。请谨慎使用本参数。

使用指南

本命令只能在命令交换机上执行。

使用实例

# 重新启动成员编号为1的成员交换机。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] reboot member 1
 This member will be rebooted. Continue?[Y/N] y

屏幕上显示的“[Y/N]”表示“YES或NO”。输入“y”则表示确认，输入“n”则表示放弃。
