display lapb status

命令功能

display lapb status命令用来查看LAPB的当前状态信息和状态变化信息。

命令格式

display lapb status

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

本命令只能在接口视图下使用。

使用实例

# 查看LAPB的状态信息。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] display lapb status
  Lapb Current Status:
  normal information transfer state:
  SABM(E) sent and UA received, or SABM(E) received and UA sent.
  the times of state changes: 2
  state change reason:
  SABM or UA.
  T1 timer out: 0
表1 display lapb status命令输出信息描述

项目

描述

Lapb Current Status:

LAPB当前的状态信息。

the times of state changes:

LAPB状态变化的次数。

state change reason:

LAPB状态变化的原因。

T1 timer out:

T1定时器超时次数。
