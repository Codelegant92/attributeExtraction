authentication-algorithm

命令功能

authentication-algorithm命令用来指定一个IKE提议使用的认证算法。

undo authentication-algorithm命令用来设置认证算法为缺省值。

缺省使用SHA1认证算法。

命令格式

authentication-algorithm { md5 | sha }

undo authentication-algorithm

参数说明

md5：指定认证算法为HMAC-MD5。

sha：指定认证算法为HMAC-SHA1。

视图

IKE提议视图

缺省级别

2：配置级

使用指南

AH协议没有加密的功能，只对报文进行认证。

MD5算法使用128位的密钥，SHA-1算法使用160位的密钥。MD5算法的计算速度比SHA-1算法快，而SHA-1算法的安全强度比MD5算法高。

使用实例

# 指定IKE提议10的认证算法为MD5。

<Quidway> system-view
[Quidway] ike proposal 10
[Quidway-ike-proposal-10] authentication-algorithm md5
