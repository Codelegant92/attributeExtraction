smart-link flush enable

命令功能

smart-link flush enable命令用来使能二层端口可接收SmartLink Flush协议报文。

undo smart-link flush enable命令取消端口的上述配置。

缺省情况下，不处理SmartLink Flush协议报文。

命令格式

smart-link flush enable control-vlan vlan-id

undo smart-link flush enable

参数说明

control-vlan vlan-id：端口接收SmartLink Flush报文的控制VLAN ID。取值范围1～4094。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

本命令只能在二层Ethernet端口或二层GE端口下执行。

本命令仅在路由器与支持发送SmartLink Flush报文的设备联合组网时，需要配置。

控制VLAN ID必须与SmartLink Flush报文中的控制VLAN ID一致，否则路由器不会处理该SmartLink Flush报文。

如果反复配置该命令行，且使用了不同的控制VLAN ID，则以最后一次的配置为准。

使用实例

# 使能端口GE 3/0/1接收SmartLink Flush报文的功能。控制VLAN ID为10。

<Quidway> system-view
[Quidway] interface gigabitethernet 3/0/1
[Quidway-GigabitEthernet3/0/1] portswitch
[Quidway-GigabitEthernet3/0/1] smart-link flush enable control-vlan 10
