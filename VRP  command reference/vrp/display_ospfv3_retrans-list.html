display ospfv3 retrans-list

命令功能

display ospfv3 retrans-list命令用来显示OSPFv3中重传列表的信息。

命令格式

display ospfv3 [ process-id ] retrans-list [ statistics ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：OSPFv3进程号。整数形式，取值范围是1～65535。

statistics：显示OSPFv3中重传列表的统计信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

该命令的显示信息可以用来帮助进行故障诊断。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示OSPFv3中的重传列表的信息。

<Quidway> display ospfv3 retrans-list
                                                                         
     OSPFv3 Process (1)

项目

描述

Interface

与邻居相连的接口

Area-ID

接口所属的区域ID

Nbr-ID

邻居路由器ID

LS-Type
LSA类型

  • Network-LSA：广播网络和NBMA网络的DR生成，描述本网段的链路状态，只在DR所处区域内传播。

  • Intra-Area-Prefix-LSA：由区域边界路由器ABR生成，在与该LSA相关的区域内传播。

LS-ID

Link State链路状态ID（来自LSA报文头）

AdvRouter

发布路由器（来自LSA报文头）

SeqNum

序列号（来自LSA报文头）

Age

老化时间（来自LSA报文头）
