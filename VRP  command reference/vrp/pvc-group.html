pvc-group

命令功能

pvc-group命令用来在ATM接口视图下使用pvc-group命令在ATM接口上创建一个PVC-Group或进入PVC-Group视图。

undo pvc-group命令用来删除指定的PVC-Group。

命令格式

pvc-group { pvc-name [ vpi/vci ] | vpi/vci }

undo pvc-group { pvc-name [ vpi/vci ] | vpi/vci }

参数说明

pvc-name：PVC名，最长为16个字符，在ATM接口上保持唯一，不区分大小写，并且不能是合法的VPI/VCI值对，如“1/20”就不允许作为PVC名。pvc-name对应的PVC必须已经建立。

vpi/vci：vpi为ATM网络虚路径标识（Virtual Path Identifier，VPI），取值范围是0～255；vci为ATM网络虚通道标识（Virtual Channel Identifier，VCI），取值范围是0～2047。通常，vci取值0到31保留用于特定用途，建议用户不要使用。vpi/vci对应的PVC必须已经建立。

视图

ATM接口视图

缺省级别

2：配置级

使用指南

一旦为某条PVC指定了名称pvc-name（例如“huawei”），则可以通过键入pvc-group pvc-name（如“pvc-group huawei”）进入PVC-Group视图

在删除该PVC-Group时，既可以通过键入undo pvc-group pvc-name（如“undo pvc-group huawei”），也可以通过undo pvc-group vpi/vci（假设该PVC的VPI/VCI为1/32，则就是“undo pvc-group 1/32”）来完成。

说明：

当链路状态不稳定，本端PVC-Group中的PVC需要感知到对端PVC的状态变化时，必须在本端PVC上通过执行命令oam frequency配置OAM。

使用实例

# 以一条名为“huawei”、VPI/VCI为1/32的PVC为基础创建一个PVC-Group。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-Atm1/0/0] pvc-group huawei 1/32
