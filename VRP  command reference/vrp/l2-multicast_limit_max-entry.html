l2-multicast limit max-entry

命令功能

l2-multicast limit max-entry命令用来配置全局、端口或VSI内的组播组数量限制。对于不需要做数量限制的组播组，可以使用ACL的permit规则定义范围区分出来。

undo l2-multicast limit max-entry命令用来取消全局、端口或VSI内的组播组数量限制。

缺省情况下，不对任何全局、端口或VSI内组播组进行数量限制。

命令格式

l2-multicast limit max-entry count [ except acl-number ]

undo l2-multicast limit max-entry

参数说明

max-entry count：组播组限制数量。整数形式，取值范围是1～8000。

except：排除不需要做数量限制的组播组。

acl-number：ACL号。整数形式，取值范围是2000～2999。

缺省级别

2 ：配置级

使用指南

此命令为覆盖式命令。

使用实例

# 将全局的组播组数量限制配置为100。

<Quidway> system-view
[Quidway] l2-multicast limit max-entry 100

# 将VSI 1000的组播组数量限制配置为100。

<Quidway> system-view
[Quidway] vsi 1000
[Quidway-vsi-1000]  l2-multicast limit max-entry 100

# 将接口GigabitEthernet2/0/1组播组数量限制配置为100。

<Quidway> system-view
[Quidway] interface GigabitEthernet2/0/1
[Quidway-GigabitEthernet2/0/1]  l2-multicast limit max-entry 100

# 将子接口GigabitEthernet2/0/1.1组播组数量限制配置为100。

<Quidway> system-view
[Quidway] interface GigabitEthernet2/0/1.1
[Quidway-GigabitEthernet2/0/1.1] l2-multicast limit max-entry 100
