display vlan statistics

命令功能

display vlan statistics命令用来查看指定VLAN或指定子接口的VLAN报文收发统计信息。

命令格式

display vlan statistics { vid vlan-id | interface interface-type interface-number.subinterface-number } *

display vlan vlan-id statistics

参数说明

vlan-id：VLAN ID，取值范围是1～4094。

interface-type：接口类型，可以指定的接口类型有Ethernet、Gigabitethernet或Eth-Trunk。

interface-number.subinterface-number：Ethernet、GigabitEthernet或Eth-Trunk的子接口编号。

缺省级别

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看子接口GE2/0/0.1的VLAN报文收发统计信息。

<Quidway> display vlan statistics interface gigabitethernet2/0/0.1
  VLAN packet statistics:
        0 Packets received, 0 bytes
        0 Packets transmitted, 0 bytes
        0 Received error ,0 Transmitted error

项目

描述

VLAN packet statistics

指定接口的VLAN报文统计信息。

Packets received, bytes

Packets transmitted, bytes

发送的报文数和字节数。

Received error ,Transmitted error
