remote-address

命令功能

remote-address命令用来配置IKE Peer对端的地址或地址段。

undo remote-address命令用来取消配置。

命令格式

remote address [vpn-instance vpn-instance ] low-ip-address [ high-ip-address ]

undo remote address

参数说明

low-ip-address：对端IP地址段的起始地址，点分十进制形式。

high-ip-address：对端IP地址段的终点地址，点分十进制形式。

vpn-instance：对端属于的VPN实例名称。

缺省级别

2：配置级

使用指南

remote-address命令配置时如果不指定high-ip-address，相当于只为IKE Peer配置一个地址。

对于一个MPLS VPN网络， VPN实例与连接CE的接口进行关联，此时如果再在该接口上应用IPSec策略，就需要通过本命令指明关联的VPN实例。

目前不支持CE的地址重叠，但能够支持CE连接私网的地址重叠。

使用实例

# 设置对端地址为202.38.0.1。

<Quidway> system-view
[Quidway] ike peer huawei
[Quidway-ike-peer-huawei] remote-address 202.38.0.1

# 设置对端地址为192.168.0.1，且本端地址的接口与VPN实例vpna关联。

<Quidway> system-view
[Quidway] ike peer huawei
[Quidway-ike-peer-huawei] remote-address vpn-instance vpna 192.168.0.1
