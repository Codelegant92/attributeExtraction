dhcp snooping alarm threshold（系统视图）

命令功能

dhcp snooping alarm threshold命令用来配置全局丢弃的报文数量的告警阈值。

undo dhcp snooping alarm threshold命令用来恢复告警阈值的缺省值。

缺省情况下，告警阈值是100。

命令格式

dhcp snooping alarm threshold threshold

undo dhcp snooping alarm threshold

参数说明

threshold：告警阈值，取值范围是1~1000，缺省为100 。

视图

系统视图

缺省级别

2：配置级

使用指南

在系统视图下执行此命令以设置所有的三层接口或VLAN上被丢弃报文的告警阈值。

如果没有配置三层接口或VLAN的告警阈值，那么三层接口和VLAN下的告警阈值就是全局的默认值，可以通过配置全局的告警阈值来修改默认值。

使用实例

# 配置全局默认的丢弃报文的告警阈值为200

<Quidway> system-view
[Quidway] dhcp snooping alarm threshold 200
