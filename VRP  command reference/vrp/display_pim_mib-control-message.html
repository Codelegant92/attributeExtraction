display pim mib-control-message

命令功能

display pim mib-control-message counters用来查看邻居丢失计数、DR当选计数、无效Join/Prune报文计数、无效注册报文计数、RP-Mapping改变计数、收到和发送的Assert报文计数。

display pim mib-control-message join-prune用来查看最后一次接收到的无效Join/Prune报文的源地址、组地址、RP地址。

display pim mib-control-message assert用来查看最后一次接收或者发送的Assert报文的源地址、组地址。

display pim mib-control-message register用来查看最后一次接收到的无效注册报文的源地址、组地址、RP地址。

命令格式

display pim mib-control-message { counters | join-prune | assert | register } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

counters：PIM相关计数。

join-prune：最后一次接收到的无效Join/Prune报文信息。

assert：最后一次接收或者发送的Assert报文信息。

register：最后一次无效的注册报文信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用实例

# 查看邻居丢失计数，DR当选计数，无效Join/Prune报文计数，无效注册报文计数，RP-Mapping-Change计数，收到和发送的Assert报文计数。

<Quidway> display pim mib-control-message counters
pim mib-control-message counters
Neighbor-loss（times）        : 0
DR-Election（times）          : 0
Invalid-join-prune（times）   : 0
RP-Mapping（times）           : 0
Invalid-register（times）     : 0
Assert-received（times）      : 0
Assert-send（times）          : 0

# 查看最后一次接收到的无效Join/Prune报文。

<Quidway> display pim mib-control-message join-prune
The last invalid join-prune message received information:
Group address  : 225.1.1.1
Source address : 0.0.0.0
RP address     : 20.0.5.5

# 查看最后一次发送或者接收的Assert报文。如果执行该命令时接收或发送Assert报文的接口板被拔掉，则显示“The last assert message was received or sent on lost interface”

<Quidway> display pim mib-control-message assert
The last assert message received or sent on Ethernet1/0/0
Group address   : 226.3.3.3
Source address  : 20.0.5.55

# 查看最后一次接收到的无效注册报文。

<Quidway> display pim mib-control-message register
The last invalid register message received information:
Group address  : 225.1.1.1
Source address : 20.0.5.100
RP address     : 20.0.3.1
