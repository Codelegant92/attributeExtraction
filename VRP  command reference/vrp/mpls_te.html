mpls te

命令功能

mpls te用来使能MPLS流量工程特性。

undo mpls te用来去使能MPLS流量工程特性。

缺省情况下，不使能MPLS流量工程特性。

命令格式

mpls te

undo mpls te

参数说明

无

视图

MPLS视图、接口视图

缺省级别

2：配置级

使用指南

在MPLS视图下执行mpls te命令，将全局使能MPLS流量工程特性；在接口视图下执行mpls te命令，将在接口上使能MPLS流量工程特性。必须先全局使能MPLS TE，才能在接口下使能MPLS TE。

使用实例

# 全局使能MPLS流量工程特性。

<Quidway> system-view
[Quidway] mpls lsr-id 1.1.1.9
[Quidway] mpls
[Quidway-mpls] mpls te

# 在接口GE1/0/0上使能MPLS TE。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] mpls
[Quidway-GigabitEthernet1/0/0] mpls te
