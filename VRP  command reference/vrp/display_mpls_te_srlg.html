display mpls te srlg

命令功能

display mpls te srlg命令用来查看SRLG（Shared Risk Link Group）配置和SRLG成员接口。

命令格式

display mpls te srlg { srlg-number | all } [ | count ] [ | { begin | exclude | include } regular-expression]

参数说明

srlg-number：接口所属的SRLG组号，整数形式，取值范围是0～4294967295。

all：所有属于SRLG组的节点和接口的SRLG信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

完成接口上的SRLG配置后，可执行display mpls te srlg命令查看配置结果。

display mpls te srlg命令可查看如下信息：
  • 节点上可支持的SRLG最大数
  • 节点上已配置的SRLG数目
  • SRLG的成员接口

使用实例

# 查看节点上的SLRG信息。

<Quidway>  display mpls te srlg all
