mpls l2vc track admin-vrrp vrid

命令功能

mpls l2vc track admin-vrrp vrid命令把业务VLL的PW与管理VRRP进行绑定。

undo mpls l2vc track admin-vrrp vrid命令用来删除业务VLL的PW与管理VRRP的绑定关系。

缺省情况下，没有配置业务VLL的PW与任何管理VRRP的绑定。

命令格式

mpls l2vc track admin-vrrp interface interface-type interface-number vrid virtual-router-id

undo mpls l2vc track admin-vrrp

参数说明

vrid virtual-router-id：管理VRRP备份组号。十进制整数形式，取值范围是1～255。

interface interface-type interface-number：配置了管理VRRP的接口类型和编号。

视图

Ethernet接口视图、Ethernet子接口视图、GE接口视图、GE子接口视图、Eth-Trunk接口视图、Eth-Trunk子接口视图、VE接口视图和VE子接口视图

缺省级别

2：配置级

使用指南

使用mpls l2vc track admin-vrrp vrid命令前，需要先使用mpls l2vc命令，创建VLL连接。

一个业务VLL只能绑定到一个管理VRRP中。

业务VLL的PW与管理VRRP接口不能配置在不同的接口板，而必须配置在同一物理接口的不同子接口，否则管理VRRP的状态不能反映PW的真实状态。

使用实例

# 配置VRRP备份组100，虚拟IP为 192.168.0.100。并配置该VRRP备份组为管理VRRP。指定的VC ID为100，在NPE上配置VRRP和业务VLL中的PW绑定。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ip address 192.168.0.1 255.255.255.0
[Quidway-GigabitEthernet1/0/0] vrrp vrid 100 virtual-ip 192.168.0.100
[Quidway-GigabitEthernet1/0/0] admin-vrrp vrid 100
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] mpls l2vpn 
[Quidway-l2vpn] quit
[Quidway] interface gigabitethernet 1/0/1
[Quidway-GigabitEthernet1/0/1] mpls l2vc 1.1.1.1 100
[Quidway-GigabitEthernet1/0/1] mpls l2vc track admin-vrrp interface gigabitethernet 1/0/1 vrid 100
