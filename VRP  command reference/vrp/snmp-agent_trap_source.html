snmp-agent trap source

命令功能

snmp-agent trap source命令用来指定发送Trap的源地址，即指定接口的IP地址。

undo snmp-agent trap source命令用来取消当前配置。

命令格式

snmp-agent trap source interface-type interface-number

undo snmp-agent trap source

参数说明

interface-type interface-number：指定发送Trap报文的源接口。

缺省级别

2：配置级

使用指南

从一台服务器发出的SNMP Trap消息，无论从那个接口发出，都有一个Trap地址。如果用户想使用Trap地址来跟踪特定事件时，可以使用这条配置命令。

使用实例

# 将GE1/0/0的IP地址作为Trap报文的源地址。

<Quidway> system-view
[Quidway] snmp-agent trap source gigabitethernet 1/0/0
