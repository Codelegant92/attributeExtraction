delete hop

命令功能

delete hop命令用来在MPLS TE显式路径中删除指定节点。

命令格式

delete hop ip-address

参数说明

ip-address：节点上接口的IP地址，点分十进制格式。

视图

显式路径视图

缺省级别

2：配置级

使用指南

使用delete hop命令删除显式路径中的节点时，必须是显式路径中存在的节点，并且需要保证本设备上没有任何Tunnel接口在使用该显式路径。

使用实例

# 在MPLS TE显式路径中删除IP地址为10.0.0.120的节点。

<Quidway> system-view
[Quidway] explicit-path p1
[Quidway-explicit-path-p1] list hop
 Path Name : path1     Path Status : Enabled
 1      10.0.0.110           Strict      Include
 2      10.0.0.120           Strict      Include
[Quidway-explicit-path-p1] delete hop 10.0.0.120
[Quidway-explicit-path-p1] list hop
 Path Name : path1     Path Status : Enabled
 1      10.0.0.110           Strict      Include
