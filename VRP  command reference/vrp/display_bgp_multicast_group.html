display bgp multicast group

命令功能

display bgp multicast group命令用来查看对等体组信息。

命令格式

display bgp multicast group [ group-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

group-name：对等体组的名称。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

如果指定了对等体组则显示该对等体组的详细信息。

如果没有指定对等体组，将显示所有MBGP对等体组信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

BGP peer-group: my-peer

对等体组：组名

remote AS

对等体所在的AS号

Type
对等体组的类型

  • internal：对等体组类型为IBGP

  • external：对等体组类型为EBGP

Configured hold timer value

所配置的保持时间的值

Keepalive timer value

存活时间的值

Minimum route advertisement interval

路由通告的最短时间间隔

PeerSession Members

对等体成员之间建立了会话连接的成员

Peer Preferred Value

对等体的首选值

Peer

对等体的IP地址

V

MBGP版本号

As

对等体所在的自治系统号

MsgRcvd

收到的消息数目

MsgSent

发送的消息数目

OutQ

等待向邻居发送的消息数目

Up/Down

MBGP会话处于当前状态的时长

State
MBGP状态机

  • Idle：MBGP拒绝任何进入的连接请求，是MBGP初始状态

  • Active：MBGP将尝试进行TCP连接的建立，是MBGP的中间状态

  • Established：MBGP对等体间可以交换Update报文、Notification报文和Keepalive报文

  • Connect：此状态下，MBGP等待TCP连接的建立完成后再决定后续操作

  • OpenSent：此状态下，MBGP等待对等体的Open消息

  • OpenConfirm：此状态下，MBGP等待一个Notification报文或Keepalive报文

PrefRcv

本端从对等体上收到路由前缀的数目
