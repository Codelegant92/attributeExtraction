命令功能

public-key-code end命令可以用来从公共密钥编辑视图退回到公共密钥视图，并且保存用户配置的公共密钥。

命令格式

public-key-code end

参数说明

无

视图

公共密钥编辑视图

缺省级别

2：配置级

使用指南

当执行此命令后，结束公钥编码的编辑过程，在保存之前要进行密钥合法性的检测。如果用户配置的公钥字符串中存在非法字符或不符合编码规则，那么将会显示相关提示信息，且用户配置的密钥将被丢弃，本次配置失败；如果配置的密钥合法，将会保存到客户公钥链表中。

使用实例

# 退出并保留配置。
