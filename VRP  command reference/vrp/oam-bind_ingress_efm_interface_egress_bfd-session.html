oam-bind ingress efm interface egress bfd-session

命令功能

oam-bind ingress efm interface egress bfd-session命令用于配置EFM OAM向BFD传递故障。

undo oam-bind ingress efm interface egress bfd-session命令用于取消EFM OAM向BFD传递故障。

命令格式

oam-bind ingress efm interface interface-type interface-number egress bfd-session bfd-session-id

undo oam-bind ingress efm interface interface-type interface-number egress bfd-session bfd-session-id

参数说明

bfd-session-id：BFD会话ID，取值范围1～8191。

interface-type interface-number：以太网接口类型和编号，取值包括：ethernet和gigabitethernet。该接口必须已经使能EFM OAM功能。

缺省级别

2：配置级

使用指南

当EFM OAM检测到故障后，OAM管理模块通过接口将该故障传递到BFD一侧。

配置该命令时，相关的BFD必需已经创建，指定的接口也已使能EFM功能。

使用实例

# 配置EFM OAM通过接口GE3/0/1向BFD传递故障。

<Quidway> system-view
[Quidway] oam-mgr
[Quidway-oam-mgr] oam-bind ingress efm interface gigabitethernet 3/0/1 egress bfd-session 50
