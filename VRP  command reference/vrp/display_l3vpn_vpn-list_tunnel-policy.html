display l3vpn vpn-list tunnel-policy

命令功能

display l3vpn vpn-list tunnel-policy用来显示L3VPN隧道策略与VPN实例的引用关系。

命令格式

display l3vpn vpn-list tunnel-policy tunnel-policy-name [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

tunnel-policy-name：隧道策略名。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
