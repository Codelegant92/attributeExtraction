mld static-group

命令功能

mld static-group命令用来使接口静态加入组播组。

undo mld static-group命令用来恢复配置的缺省情况。

缺省情况下，接口未静态加入组播组。

命令格式

mld static-group ipv6-group-address [ source ipv6-source-address ]

undo mld static-group { all | ipv6-group-address [ source ipv6-source-address ] }

参数说明

ipv6-group-address：指定静态加入的组播组地址。十六进制形式，可输入的地址格式是FFxA:xxxx:xxxx::xxxx，其中x的取值范围是0～F，A的取值范围是0和3～F。

source ipv6-source-address：指定组播源的IPv6地址。如果组播组地址在SSM范围内，可通过此关键字指定接口上的(S, G)转发项。

all：清除此接口加入的所有静态组。

使用指南

执行此命令后，路由器认为该接口上始终连接着组成员主机，并持续向该接口所在网段转发符合条件的组播报文。

使用实例

# 设置接口GE1/0/0静态加入组FF03::101。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/1] mld static-group ff03::101

# 设置接口GE1/0/0转发组播源2001::101到组播组FF04::202的报文。

[Quidway-GigabitEthernet1/0/0] mld static-group ff04::202 source 2001::101
