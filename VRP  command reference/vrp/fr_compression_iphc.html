fr compression iphc

命令功能

fr compression iphc命令用来使能接口IP头压缩。

undo fr compression iphc命令用来禁止此功能。

缺省情况下，禁止帧中继压缩功能。

命令格式

fr compression iphc

undo fr compression iphc

参数说明

无

视图

帧中继接口视图

缺省级别

2：配置级

使用指南

fr compression iphc命令和fr iphc命令可以分开配置，没有先后顺序。

当使用undo fr compression iphc命令取消帧中继IP头压缩功能时，fr iphc命令的设置仍然保留。

当使用fr compression iphc命令重新恢复帧中继IP头压缩功能时，也可以继续使用fr iphc命令以前配置的参数。

使用实例

# 配置帧中继接口Serial 1/0/0采用被动IP头压缩。

<Quidway> system-view
[Quidway] interface serial 1/0/0
[Quidway-Serial1/0/0] link-protocol fr
[Quidway-Serial1/0/0] fr compression iphc
