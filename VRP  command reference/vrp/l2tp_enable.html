l2tp enable

命令功能

l2tp enable命令用来启用L2TP功能。

undo l2tp enable命令用来禁止L2TP功能。

缺省情况下，L2TP功能被禁止。

命令格式

l2tp enable

undo l2tp enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

只有配置l2tp enable命令后，L2TP功能才能使用；如果禁止L2TP，则即便配置了L2TP的参数，路由器也不会提供相关功能。

使用实例

# 在路由器上启用L2TP功能。

<Quidway> system-view
[Quidway] l2tp enable
