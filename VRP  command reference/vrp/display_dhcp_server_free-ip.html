display dhcp server free-ip

命令功能

display dhcp server free-ip命令用来查看DHCP地址池的可用地址范围，即尚未分配的IP地址信息。

命令格式

display dhcp server free-ip [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当遇到客户端无法获取IP地址，或需要添加一定数量的客户端，可以使用display dhcp server free-ip命令来查看DHCP地址池可以使用的地址范围。

使用实例

# 查看DHCP地址池的可用地址范围。

<Quidway> display dhcp server free-ip
IP Range from 5.5.5.1              to  5.5.5.254
IP Range from 202.38.160.1         to  202.38.160.1
IP Range from 202.38.160.4         to  202.38.160.126
