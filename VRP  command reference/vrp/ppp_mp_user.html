ppp mp user

命令功能

ppp mp user命令用来根据用户名进行MP捆绑。

undo ppp mp user命令用来去掉MP捆绑。

命令格式

ppp mp user user-name bind virtual-template number

undo ppp mp user user-name

参数说明

user-name：MP用户名，字符串形式，长度范围是1～64。

number：为虚拟接口模板号，范围0～1023。

使用指南

在PPP建立连接的过程中，当PPP验证通过后，如果指定了虚拟接口模板，则将按照虚拟接口模板的参数进行MP捆绑，并形成一个新的虚拟接口进行数据传输。

当采用通过验证方式绑定物理接口时，接口下必须执行命令ppp mp。

在同一台路由器上，ppp mp user username bind virtual-template number命令最多可以配置200条。

在虚拟接口模板上可以配置的工作参数包括：

  • 本地IP地址和为PPP对端分配的IP地址（或IP地址池）

  • PPP工作参数

使用实例

# 指定用户名Quidway的对应虚接口模板为1，并配置该虚拟接口模板的IP地址是202.38.60.1。

<Quidway> system-veiw
[Quidway] ppp mp user Quidway bind virtual-template 1
[Quidway] interface virtual-template 1
[Quidway-virtual-template1] ip address 202.38.60.1 255.255.255.0
