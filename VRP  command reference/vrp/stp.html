stp

命令功能

stp命令用来启用或禁用交换机或端口上的MSTP功能。

缺省情况下，交换机上的MSTP功能处于禁用状态，但在二层接口下处于启用状态。

命令格式

stp { enable | disable }

参数说明

enable：启用MSTP功能。

disable：禁用MSTP功能。

视图

系统视图、Ethernet接口视图、GE接口视图、Eth-Trunk接口视图、VE接口视图

缺省级别

2：配置级

使用指南

注意：

禁用端口的MSTP功能可能导致环路产生。

在系统视图下执行stp命令，将会启用或禁用交换机上所有端口的MSTP功能。

在Ethernet接口视图、GE接口视图或Eth-Trunk接口视图下执行stp命令，将会启用或禁用当前端口的MSTP功能。本命令中的接口必须是二层口。

当端口的MSTP功能处于禁用状态时，该端口不参与生成树计算，且始终处在转发状态。

使用实例

# 启用交换机的MSTP功能。

<Quidway> system-view
[Quidway] stp enable

# 禁用以太网接口GE1/0/0的MSTP功能。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] stp disable
