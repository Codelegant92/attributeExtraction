display rip route

命令功能

display rip route命令用来显示所有从其它路由器学来的激活和非激活RIP路由信息，以及与每条路由相关的不同定时器的值。

命令格式

display rip process-id route [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIP进程号。整数形式，取值范围是1～65535。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

项目

描述

Destination(Dest)

目的IP地址

Nexthop

路由的下一跳

Cost

路由权值

Tag

区分内部RIP路由和外部路由的标识

Flags

第一个字符指示此路由是RIP或TRIP路由，第二个字符指明该路由状态

Sec

此路由项处于某种状态的时间
