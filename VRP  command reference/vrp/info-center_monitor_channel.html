info-center monitor channel

命令功能

info-center monitor channel命令用来设置向用户终端输出信息以及信息通道。

undo info-center monitor channel命令用来取消当前设置。

缺省情况下，不向用户终端输出信息。

命令格式

info-center monitor channel { channel-number | channel-name }

undo info-center monitor channel

参数说明

channel-number：通道号，取值为0～9，即系统有10个通道。

channel-name：通道名。

使用指南

只有启动了信息中心，该命令才会生效。
