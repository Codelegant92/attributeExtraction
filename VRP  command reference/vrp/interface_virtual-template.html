interface virtual-template

命令功能

interface virtual-template命令用来创建虚拟接口模板。

undo interface virtual-template命令用来删除指定虚拟接口模板。

命令格式

interface virtual-template vt-number

undo interface virtual-template vt-number

参数说明

vt-number：虚拟接口模板的编号，取值范围是0～1023。

视图

系统视图

缺省级别

2：配置级

使用指南

虚拟接口模板（VT）只能在系统创建虚拟访问接口（VA）之前被创建，在最后一个虚拟访问接口关闭后再被删除。

在删除时，要确保相关的虚拟访问接口都已经删除，而且该虚拟接口模板不再被使用。

使用实例

# 创建虚拟接口模板10。

<Quidway> system-view
[Quidway] interface virtual-template 10
