refresh bgp multicast internal

命令功能

refresh bgp multicast internal命令用来软复位IBGP对等体之间的连接。

命令格式

refresh bgp multicast internal { import | export }

参数说明

import：触发入方向的软复位。

export：触发出方向的软复位。

使用实例

# 对所有的IBGP连接触发入方向的软复位。

<Quidway> refresh bgp multicast internal import
