zip

命令功能

zip命令用来压缩文件。

unzip命令用来解压缩文件。

命令格式

zip source-filename destination-filename

unzip source-filename destination-filename

参数说明

source-filename：源文件名。字符串形式，格式为[ path ][ file-name ]，取值范围是1～64。

destination-filename：目标文件名。字符串形式，格式为[ path ][ file-name ]，取值范围是1～64。

使用指南

如果只指定了目标文件所在的路径，但未指定目标文件名，则目标文件名与源文件名相同。压缩后，源文件仍然存在。
