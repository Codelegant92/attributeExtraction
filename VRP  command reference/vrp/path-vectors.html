path-vectors

命令功能

path-vectors命令用来设置路径向量最大值。

undo path-vectors命令用来恢复路径向量最大值。

缺省情况下，路径向量最大值为32。

命令格式

path-vectors integer

undo path-vectors

参数说明

integer：指定路径向量。整数形式，取值范围是1～32。

视图

MPLS-LDP-VPN实例视图、MPLS-LDP视图

缺省级别

2：配置级

使用指南

此命令应该在所有接口LDP使能之前配置。只有在相应视图中使能了MPLS LDP后，该命令才会生效。

在采用路径向量方式配置环路检测时，本地路由触发和非本地路由触发的Egress LSP的处理方式不同。假设配置的路径向量最大跳数为n，本地路由触发的Egress LSP需要经过n+1跳才会检测到环路，而非本地路由触发的Egress LSP（Proxy Egress）经过n跳就能检测到环路。

使用实例

# 设置路径向量最大值为3。

<Quidway> system-view
[Quidway] mpls ldp
[Quidway-mpls-ldp] path-vectors 3
[Quidway] mpls ldp vpn-instance vpn1
[Quidway-mpls-ldp-vpn-instance-vpn1] path-vectors 3
