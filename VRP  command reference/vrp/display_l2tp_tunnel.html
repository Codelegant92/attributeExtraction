display l2tp tunnel

命令功能

display l2tp tunnel命令用来显示当前建立的L2TP隧道信息。

命令格式

display l2tp tunnel

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display l2tp tunnel命令可用来查看当前设备上成功建立的L2TP隧道的条数，及各条隧道的相关信息，如远端IP地址和远端隧道名、远端使用的UDP端口号，该隧道中存在的会话数。

使用实例

# 查看当前L2TP隧道信息。

<Quidway> display l2tp tunnel
Total tunnel = 1
LocalTID  RemoteTID  RemoteAddress  Port  Sessions  RemoteName
 2        22849      11.1.1.1       1701      1       lns
表1 display l2tp tunnel命令输出信息描述

项目

描述

Total tunnel

本端建立的L2TP Tunnel的数目

LocalTID

本端隧道ID

RemoteTID

对端隧道ID

RemoteAddress

隧道对端的IP地址

Port

协议规定的远端UDP知名端口号，值为1701

Sesssions

此隧道上的会话数目

Remote Name

对端的隧道名称
