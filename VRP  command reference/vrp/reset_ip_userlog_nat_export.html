reset ip userlog nat export

命令功能

reset ip userlog nat export命令用来清除用户日志的统计报文输出信息。

命令格式

reset ip userlog nat export slot slot-number

参数说明

slot-number：接口板槽号。

视图

用户视图

缺省级别

2：配置级

使用指南

清除统计信息后，以前的统计信息将无法恢复。

使用实例

# 清除路由器2号槽的用户日志统计报文输出信息。

<Quidway> reset ip userlog nat export slot 2
