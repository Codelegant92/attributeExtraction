bandwidth-reference (OSPF)

命令功能

bandwidth-reference命令用来设置计算链路开销时所依据的参考值。

undo bandwidth-reference命令用来恢复参考值为缺省设置。

缺省情况下，链路开销参考值为100M。即cost=100000000/bandwidth。

命令格式

bandwidth-reference value

undo bandwidth-reference

参数说明

value：计算链路开销时所依据的参考值。整数形式，取值范围是1～2147483648，单位是Mbit/s。

视图

OSPF视图

缺省级别

2：配置级

使用指南

如果没有配置链路的开销值，OSPF根据链路带宽来计算开销（开销 = 参考值（M）/带宽）。

使用实例

# 设置链路开销参考值为1000。

<Quidway> system-view 
[Quidway] ospf 100
[Quidway-ospf-100] bandwidth-reference 1000
