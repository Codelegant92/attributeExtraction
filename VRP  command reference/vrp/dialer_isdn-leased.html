dialer isdn-leased

命令功能

dialer isdn-leased命令用来配置专线连接的ISDN的B通道。

undo dialer isdn-leased命令用来取消该配置。

缺省不配置专线连接的ISDN B通道。

命令格式

dialer isdn-leased number

undo dialer isdn-leased number

参数说明

number：用于专线连接的ISDN的B通道号，在BRI接口下取值范围是1～2，在PRI接口下取值范围是0～30。

视图

接口视图

缺省级别

2：配置级

使用指南

可将ISDN的任意B通道配置成专线，该配置并不影响其它B通道的设置。

使用实例

# 将Bri1/0/0接口的第一个B通道设置成专线。

[Quidway-Bri1/0/0] dialer isdn-leased 1
