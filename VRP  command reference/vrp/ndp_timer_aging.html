ndp timer aging

命令功能

ndp timer aging命令用来设置NDP表项的老化时间。

undo ndp timer aging命令用来恢复缺省值。

缺省情况下，NDP表项的老化时间是180秒。

命令格式

ndp timer aging aging-time

undo ndp timer aging

参数说明

aging-time：指定NDP表项的老化时间。十进制整数形式，取值范围是6～255，单位是秒。

使用指南

NDP表项的老化时间必须大于NDP报文的发送时间间隔。

说明：

配置NDP报文的发送时间间隔，请参见ndp timer hello命令。

使用实例

# 设置NDP表项的老化时间为60秒。

<Quidway> system-view
[Quidway] ndp timer aging 60
