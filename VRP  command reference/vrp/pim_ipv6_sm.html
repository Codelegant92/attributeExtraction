pim ipv6 sm

命令功能

pim ipv6 sm命令用来在接口上使能IPv6 PIM-SM协议。

undo pim ipv6 sm命令用来关闭IPv6 PIM-SM协议。

缺省情况下，接口上IPv6 PIM-SM是关闭的。

命令格式

pim ipv6 sm

undo pim ipv6 sm

参数说明

无

缺省级别

2：配置级

使用指南

对于一台路由器上的一个VPN实例来说，在一个接口上使能IPv6 PIM-SM后，就不能在此接口或其他接口上配置IPv6 PIM-DM，即一台路由器的同一个实例上不能同时配置IPv6 PIM-DM和IPv6 PIM-SM。

使用实例

# 在接口GE1/0/0上使能IPv6 PIM-SM功能。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 sm
