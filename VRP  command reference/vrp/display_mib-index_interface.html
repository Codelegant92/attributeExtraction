display mib-index interface

命令功能

display mib-index interface命令用来显示接口在MIB中的索引值。

命令格式

display mib-index interface [ interface-type interface-number ]

参数说明

interface：查询指定接口对象的索引值。

interface-type：接口类型，如Ethernet。如果不指定接口类型，将显示所有接口的索引值。

interface-number：接口编号，与interface-type共同确定一个接口。如果不指定接口编号，将显示所有该类型接口的索引值。

为了方便网管查看接口名字所对应的接口索引，可以执行本命令查看接口索引的具体数值。
