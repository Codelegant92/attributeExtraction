dhcp option82 insert enable

命令功能

dhcp option82 insert enable命令用来使能对DHCP报文插入Option82。

undo dhcp option82 insert enable命令用来取消对DHCP报文插入Option82。

缺省情况下，不对DHCP报文插入Option82。

命令格式

dhcp option82 insert enable

undo dhcp option82 insert enable

参数说明

无

接口视图

缺省级别

2：配置级

使用指南

本命令只用于在配置了DHCP Relay的接口上配置Option82功能。

配置dhcp option82 insert enable后，如果原来的DHCP报文中没有option82选项，则在DHCP报文中插入option82选项；如果原来的DHCP报文中有Option82选项，则对DHCP报文插入子选项9。

dhcp option82 rebuild enable的优先级高于dhcp option82 insert enable，如果在一个接口上同时配置dhcp option82 insert enable和dhcp option82 rebuild enable，则等同于单独配置dhcp option82 rebuild enable。

使用实例

# 在Ethernet4/0/0接口上使能对DHCP报文插入Option82功能。

<Quidway> system-view
[Quidway] interface ethernet 4/0/0
[Quidway-Ethernet4/0/0] dhcp option82 insert enable
