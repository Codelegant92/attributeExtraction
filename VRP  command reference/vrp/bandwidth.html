bandwidth

命令功能

bandwidth bandwidth命令用来设置VLANIF接口的带宽。

undo bandwidth命令用来恢复缺省配置。

缺省情况下，VLANIF接口带宽为1000兆比特/秒。

命令格式

bandwidth bandwidth

undo bandwidth

参数说明

bandwidth：VLANIF接口的带宽。取值范围是100～1000000，单位是兆比特/秒，缺省值为1000兆比特/秒。

视图

VLANIF接口视图

缺省级别

2：配置级

使用指南

使用本命令可手工设置VLANIF接口的带宽。

使用实例

# 设置VLANIF2接口的带宽为10000兆比特/秒。

<Quidway> system-view
[Quidway] vlan 2
[Quidway-vlan2] quit
[Quidway] interface vlanif 2
[Quidway-Vlanif2] bandwidth 10000
