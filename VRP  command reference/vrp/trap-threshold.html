trap-threshold

命令功能

trap-threshold命令用来设置Ethernet子接口、GE子接口的出、入带宽利用率阈值。

undo trap-threshold命令用来将出、入带宽利用率阈值恢复为缺省值。

命令格式

trap-threshold { input-rate | output-rate } bandwidth-in-use

undo trap-threshold { input-rate | output-rate }

参数说明

input-rate：设置入带宽利用率阈值。

output-rate：设置出带宽利用率阈值。

bandwidth-in-use：带宽利用率的阈值。整数形式，取值范围是1～100，缺省值是100。

视图

Ethernet子接口视图、GE子接口视图

缺省级别

2：配置级

使用指南

出带宽利用率＝(子接口出流量速率/子接口所在物理接口的带宽)×100

入带宽利用率＝(子接口入流量速率/子接口所在物理接口的带宽)×100

当带宽利用率超出阈值，系统将产生告警信息；带宽利用率从阈值以上降到阈值以下，系统将产生告警恢复信息。

使用实例

# 设置端口输出流量超过端口带宽80%时告警。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] trap-threshold output-rate 80

# 恢复端口输入流量告警阈值，为默认值。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] undo trap-threshold input-rate
