display mpls bfd session

命令功能

display mpls bfd session命令用来查看BFD会话的相关信息。包括LDP LSP和TE CR-LSP的BFD会话信息。

命令格式

display mpls bfd session [ fec fec-address | monitor | nexthop ip-address | outgoing-interface interface-type interface-number | protocol { ldp [ fec fec-address ] | rsvp-te [ lsp-id ingress-lsr-id session-id lsp-id ] | cr-static [ lsp-id ingress-lsr-id session-id lsp-id ] } [ verbose ] | statistics | verbose ]

参数说明

fec fec-address：查看指定FEC的BFD会话相关信息。

monitor：查看BFD会话的监控信息。

nexthop ip-address：查看指定下一跳的BFD会话相关信息。

outgoing-interface interface-type interface-number：查看指定出接口的LSP的BFD会话的相关信息。

protocol ldp：查看协议类型为LDP的BFD会话相关信息。

protocol rsvp-te：查看协议类型为RSVP-TE的BFD会话相关信息。

lsp-id ingress-lsr-id：入节点的LSR ID，点分十进制格式。

session-id：会话标识。整数形式，取值范围是0～65535。

lsp-id：LSP的标识号。整数形式，取值范围是0～65535。

protocol cr-static：查看协议类型为CR-Static的BFD会话相关信息。

statistics：查看BFD会话的统计信息，如BFD会话的总数量等。

verbose：查看BFD会话的详细信息。
