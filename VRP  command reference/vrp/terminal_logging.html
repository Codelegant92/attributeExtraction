terminal logging

命令功能

terminal logging命令用来打开终端显示日志信息功能。

undo terminal logging命令用来关闭终端显示日志信息功能。

缺省情况下，系统配置为打开日志信息显示功能。

命令格式

terminal logging

undo terminal logging

参数说明

无

视图

用户视图

缺省级别

1：监控级

使用指南

使用terminal logging命令打开终端显示日志信息功能。

使用实例

# 关闭终端日志显示功能。

<Quidway> undo terminal logging
Info:Current terminal logging is off
