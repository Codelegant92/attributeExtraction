fr lmi t392dce

命令功能

fr lmi t392dce命令用来配置DCE侧T392，即DCE设备等待一个状态请求报文（Status-Enquiry报文）的最大时间。

undo fr lmi t392dce命令用来恢复缺省值。

命令格式

fr lmi t392dce t392-value

undo fr lmi t392dce

参数说明

t392-value：轮询定时器值。整数形式，取值范围是5～30，单位是秒。缺省值是15秒。

视图

帧中继接口视图、MFR接口视图

缺省级别

2：配置级

使用指南

DCE方的T392应大于DTE方的T391。

使用实例

# 配置帧中继接口Serial1/0/0:0工作在DCE方式，并配置T392为10。

<Quidway> system-view
[Quidway] interface Serial1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr
[Quidway-Serial1/0/0:0] fr interface-type dce
[Quidway-Serial1/0/0:0] fr lmi t392dce 10
