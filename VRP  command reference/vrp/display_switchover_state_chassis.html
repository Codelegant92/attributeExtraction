display switchover state（多框）

命令功能

display switchover state命令用来根据指定槽号显示主用板/备用板的备份状态。

命令格式

display switchover state [ all | chassis chassis-number ]

参数说明

all：中央交换框和转发框中的所有主备板。

chassis chassis-number：转发框的框号。

视图

所有视图

缺省级别

1：监控级

使用指南

根据主备倒换所处的不同状态，该命令显示的信息有所差异。

使用实例

# 显示主用板的备份状态。

<Quidway> display switchover state
Slot CCC1/9 HA FSM State(master), Realtime and routine backup.
Slot CCC1/10 HA FSM State(slave), Receiving realtime and routing data.
