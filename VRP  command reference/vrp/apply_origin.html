apply origin

命令功能

apply origin命令用来设置BGP路由信息的路由源。

undo apply origin命令用来取消设置。

命令格式

apply origin { egp as-number | igp | incomplete }

undo apply origin

参数说明

egp：设置BGP路由信息源为外部路由。

as-number：指定外部路由的AS号。整数形式，取值范围是1～65535。

igp：设置BGP路由信息源为内部路由。

incomplete：设置BGP路由信息源为未知源。

使用实例

# 设置路由信息的路由源为IGP。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply origin igp
