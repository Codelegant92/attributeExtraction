mpls rsvp-te authentication

命令功能

mpls rsvp-te authentication命令用来在接口启动验证功能并配置认证密钥。

undo mpls rsvp-te authentication命令用来关闭接口的验证功能。

命令格式

mpls rsvp-te authentication { cipher | plain } auth-key

undo mpls rsvp-te authentication

参数说明

cipher：以密文方式显示认证密钥。

plain：以明文方式显示认证密钥。

auth-key：字符串形式，不支持空格。以明文方式输入时，长度范围是8～16；以MD5密文方式输入时，长度为24位。

使用指南

执行该命令前，应先在接口上启动RSVP-TE特性。

在接口配置此命令后，所有从此接口发送的报文将携带由MD5算法根据验证密钥计算得到的认证信息，从此接口接收的报文也将使用此密钥来验证。

使用实例

# 使能验证机制，以密文方式显示认证密钥。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] mpls rsvp-te authentication cipher beijing123
