display gtsm statistics

命令功能

display gtsm statistics命令用来查看单板上GTSM的统计信息。

命令格式

display gtsm statistics { slot-id | all } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

slot-id：指定单板。

all：所有单板。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

用户可以查看所有单板或者指定单板的GTSM统计信息，包括BGP、LDP和OSPF的报文总数、通过的报文数量、丢弃的报文数量。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看所有单板的GTSM统计信息。

<Quidway> display gtsm statistics all
                         GTSM Statistics
