ppp timer negotiate

命令功能

ppp timer negotiate命令用来设置PPP协商超时的时间。

undo ppp timer negotiate命令用来恢复PPP协商超时的时间缺省值。

命令格式

ppp timer negotiate seconds

undo ppp timer negotiate

参数说明

seconds：协商超时的时间。取值范围是1～10，单位是秒。

视图

接口视图

缺省级别

2：配置级

使用指南

在PPP协商过程中，如果在这个时间的间隔内没有收到对端的应答报文，则PPP将会重发前一次发送的报文。该缺省情况下，PPP协商超时的时间为3秒。

说明：

MP-Group接口不支持本配置。

使用实例

# 设置PPP协商超时的时间为5秒。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ppp timer negotiate 5
