interface null

命令功能

interface null命令用来进入Null接口的视图。

命令格式

interface null 0

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

Null接口只能有一个，固定为null0，该接口始终处于Up状态，不能被关闭或删除。

使用实例

# 进入Null0接口的视图。

<Quidway> system-view
[Quidway] interface null 0
[Quidway-NULL0]
