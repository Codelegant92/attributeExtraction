ipv6 nd ns retrans-timer

命令功能

ipv6 nd ns retrans-timer命令用来设置路由设备的重传计时器。

undo ipv6 nd ns retrans-timer命令用来将此设置恢复为缺省值。

命令格式

ipv6 nd ns retrans-timer interval

undo ipv6 nd ns retrans-timer

参数说明

interval：指定重传时间间隔。整数形式，单位是毫秒，取值范围是1000～3600000。缺省值是1000毫秒。

视图

接口视图

缺省级别

2：配置级

使用指南

重传计时器的作用包括：

  • 控制本路由设备的地址解析；

  • 控制本路由设备重复地址检测时，发送NS报文的时间间隔；

  • 作为RA报文的参数，帮助主机完成重传计时器的设置。

说明：

一般情况下，路由设备的重传计时器与RA报文中的重传计时器取值相同，但在下述情况中，路由设备的重传计时器为缺省值1000毫秒，RA报文中的重传计时器为0毫秒。

  • 没有设置路由设备的重传计时器，即缺省情况下。

  • 执行undo ipv6 nd ns retrans-timer命令将路由设备的重传计时器恢复为缺省值。

主机收到重传计时器值为0的RA报文后，使用缺省值1000毫秒，与路由设备保持一致。

正常IPv6操作中建议不使用较短的间隔。

在配置该命令前，需要先在接口视图下执行ipv6 enable命令，使能IPv6功能。

使用实例

# 设置GE接口上路由设备的重传计时器为10000毫秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 nd ns retrans-timer 10000
