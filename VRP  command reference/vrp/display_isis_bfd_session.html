display isis bfd session

命令功能

display isis bfd session命令用来查看BFD特性会话的信息。

命令格式

display isis [ process-id | vpn-instance vpn-instance-name ] bfd session { peer ip-address | all } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：指定IPv4 VPN的IS-IS多实例进程。vpn-instance-name是VPN实例名，区分大小写。

peer ip-address：指定邻居的IP地址。

all：指定该IS-IS进程下所有使能了IS-IS的接口。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看邻居BFD信息。

<Quidway> display isis bfd session all
