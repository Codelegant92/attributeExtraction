interface loopback

命令功能

interface loopback命令用来创建Loopback接口。

undo interface loopback命令用来删除指定Loopback接口。

命令格式

interface loopback loopback-number

undo interface loopback loopback-number

参数说明

loopback-number：Loopback接口的编号，取值范围是0～1023。

视图

系统视图

缺省级别

2：配置级

使用指南

Loopback接口创建后一直保持Up状态，并具有环回的特性，因此常被用于提高配置的可靠性。

使用实例

# 创建Loopback接口5。

<Quidway> system-view
[Quidway] interface loopback 5
[Quidway-LoopBack5]
