qos wred ip-precedence

命令功能

qos wred ip-precedence命令用来设置WRED各优先级的下限、上限和丢弃概率分母。

undo qos wred ip-precedence命令用来恢复缺省值。

命令格式

qos wred ip-precedence ip-precedence low-limit low-limit high-limit high-limit discard-probability discard-probability

undo qos wred ip-precedence ip-precedence

参数说明

ip-precedence：IP报文的优先级，取值范围是0～7。

low-limit：优先级的WRED下限，取值范围是1～1024，low-limit的缺省值分别为10。

high-limit：优先级的WRED上限，取值范围是1～1024，high-limit的缺省值为30。

discard-probability：丢弃概率的分母，取值范围是1～255，缺省值为10。

使用指南

必须先使用qos wred在接口上应用WRED，才可以配置WRED的参数。阈值限制的是平均队列长度。

使用实例

# 在接口上设置优先级为3的报文的队列下限为20，上限为40，丢弃概率分母为15。

<Quidway> system-view
[Quidway] interface GigabitEthernet0/0/0
[Quidway-GigabitEthernet0/0/0] qos wred ip-precedence 3 low-limit 20 high-limit 40 discard-probability 15
