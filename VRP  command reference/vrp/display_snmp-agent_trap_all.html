display snmp-agent trap all

命令功能

display snmp-agent trap all命令用来显示所有特性下向网管发送告警的使能情况和海量告警标志。

命令格式

display snmp-agent trap all

参数说明

无

所有视图

缺省级别

1：监控级

使用指南

在使能了向网管发送告警后，执行命令display snmp-agent trap all可以查看到在所有特性下的告警的使能情况和该告警是否是海量告警。

使用实例

# 显示所有特性下所有告警的使能情况和海量标志。

<Quidway> display snmp-agent trap all
