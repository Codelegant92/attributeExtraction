ipv6 bandwidth-reference

命令功能

ipv6 bandwidth-reference命令用来设置用于计算接口开销的带宽参考值，

undo ipv6 bandwidth-reference命令用来设置带宽参考值为默认值。

缺省情况下，参考带宽为100Mbit/s。

命令格式

ipv6 bandwidth-reference bandwidth-reference-value

undo ipv6 bandwidth-reference

参数说明

bandwidth-reference-value：指定系统的参考带宽。整数形式，单位是Mbit/s，取值范围是1～2147483648。

视图

IS-IS视图

缺省级别

2：配置级

使用指南

使用此命令前，必须先使能IS-IS进程的IPv6能力。配置过程请参见isis ipv6 enable命令。

使用实例

# 设置用于计算接口开销的IPv6带宽参考值是200Mbit/s。

<Quidway> system-view
[Quidway] isis 1
[Quidway-isis-1] ipv6 bandwidth-reference 200
