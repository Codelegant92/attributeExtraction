display fr inarp-info

命令功能

display fr inarp-info命令用来显示帧中继逆向地址解析协议的报文统计信息。

命令格式

display fr inarp-info [ interface interface-type interface-number ]

参数说明

interface-type：查看指定接口类型的帧中继逆向地址解析协议的报文统计信息。

interface-number：查看指定接口类型下指定接口编号的帧中继逆向地址解析协议的报文统计信息。

指定接口只能是主接口。不指定接口则显示全部信息。

使用指南

帧中继逆向地址解析协议的报文包括地址解析请求报文和地址解析响应报文。根据该命令的输出信息，可以诊断逆向地址解析协议是否正常工作。

使用实例

# 显示帧中继逆向地址解析协议的报文统计信息。

<Quidway> display fr inarp-info
Frame relay InverseARP statistics for interface Serial1/0/0:0 (DTE)
In ARP request    Out ARP reply    Out ARP request    In ARP reply
0                 0                1                  1
