reset bgp multicast dampening

命令功能

reset bgp multicast dampening 命令用来复位MBGP路由表中路由的衰减信息。

命令格式

reset bgp multicast dampening [ ip-address [ mask | mask-length ] ]

参数说明

ip-address：表示网络地址。点分十进制形式。

mask：表示网络的掩码。

mask-length：表示网络地址的掩码长度。

视图

用户视图

缺省级别

2：配置级

使用指南

当指定网络地址后，使用此命令可以清除一条具体路由的振荡衰减信息并释放被抑制的路由。

使用实例

# 清除网段20.1.0.0路由的衰减信息并释放被抑制的路由。

<Quidway> reset bgp multicast dampening 20.1.0.0 255.255.0.0
