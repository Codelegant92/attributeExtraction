expired

命令功能

expired命令用来配置DHCP全局地址池中IP地址的租用有效期限。

undo expired命令用来恢复缺省设置。

缺省情况下，有效期限为1天。

命令格式

expired { day day [ hour hour [ minute minute ] ] | unlimited }

undo expired

参数说明

day day：指定租用有效期限的天数。整数形式，取值范围是0～365。单位是天。缺省值是1天。

hour hour：指定租用有效期限的小时数。整数形式，取值范围是0～23。单位是小时。缺省值是0。

minute minute：指定租用有效期限的分钟数。整数形式，取值范围是0～59。单位是分钟。缺省值是0。

unlimited：有效期限无限长。

使用指南

expired命令用来配置DHCP全局地址池中IP地址的租用有效期限，使不同的客户端得到不同的租用期限。

使用实例

# 配置全局地址池0的IP地址租用有效期为1天2小时30分。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] expired day 1 hour 2 minute 30
