reset loop-detect eth-loop vsi

命令功能

reset loop-detect eth-loop vsi命令用来恢复指定VSI下被阻断的接口或PW，并重新开始环路检测。

命令格式

reset loop-detect eth-loop vsi vsi-name { all | peer peer-ip-address | interface interface-type interface-number }

参数说明

vsi-name：VSI名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

all：复位VSI下所有被阻断的接口和PW。

peer peer-ip-address：复位VSI下指定的被阻断的PW。

interface interface-type interface-number：复位VSI下指定的被阻断的接口。interface-type interface-number是接口的类型和编号。

使用指南

reset loop-detect eth-loop vsi命令用来恢复VSI中以太环路检测后永久阻断的接口或PW，并且接口或PW在恢复后立即加入到新的以太环路检测中。

使用实例

# 恢复vsi1下以太网环路检测所有阻断的接口。

<Quidway> system-view
[Quidway] reset loop-detect eth-loop vsi vsi1 all

# 恢复vsi1下以太网环路检测阻断的接口GE1/0/0.1。

<Quidway> system-view
[Quidway] reset loop-detect eth-loop vsi vsi1 interface gigabitethernet 1/0/0.1
