display isis lsdb

命令功能

display isis lsdb命令用来查看IS-IS的链路状态数据库信息。

命令格式

display isis lsdb [ level-1 | level-2 ] [ verbose ] [ local | lsp-id | is-name symbolic-name ] [ process-id | [ vpn-instance { vpn-instance-name | default } | vpn6-instance { vpn6-instance-name | default } ] * ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

level-1：显示Level-1链路状态数据库。

level-2：显示Level-2链路状态数据库。

verbose：显示链路状态数据库的详细信息。使用verbose参数可以查看链路状态数据库中的详细信息，包括MT拓扑ID、MT拓扑邻居ID和IPv6前缀等。

local：显示本地数据库的信息。

lsp-id：指定要显示的LSP的ID。形式为####.####.####.##-##，例如0050.0500.5004.00-00。

is-name symbolic-name：动态主机名，形式为is-name或is-name.##-##（##-##表示伪节点ID-LSP分片ID）。

process-id：IS-IS进程号。

vpn-instance vpn-instance-name：显示指定IPv4 VPN的IS-IS多实例进程的LSDB信息。vpn-instance-name为IPv4 VPN实例的名称，区分大小写。

vpn-instance default：显示除IPv4 VPN实例外的所有IS-IS进程的LSDB信息，即所有的IPv6 VPN的IS-IS实例进程和公网IS-IS进程。

vpn6-instance vpn6-instance-name：显示指定IPv6 VPN的IS-IS多实例进程的LSDB信息。vpn6-instance-name为IPv6 VPN实例的名称，区分大小写。

vpn6-instance default：显示除IPv6 VPN实例外的所有IS-IS进程的LSDB信息，即所有的IPv4 VPN的IS-IS实例进程和公网IS-IS进程。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

缺省级别

1：监控级

使用指南

如果不指定级别，则显示Level-1和Level-2的链路状态数据库。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

LSPID

链路状态报文ID

Seq Num

LSP序列号

Checksum

LSP校验和

Holdtime

LSP保持时间

Length

LSP长度

ATT/P/OL

Attach bit（ATT）：连接位；Partition bit（P）：分区位；Overload bit（OL）：过载位。

SOURCE

源节点的系统ID

ORG ID

虚拟系统的源系统ID

HOST NAME

动态主机名

Auth

认证密码

Len

认证密码长度

Type

认证类型

  • Plain Text：密码以明文方式发送

  • MD5：密码以MD5加密方式发送

NLPID

支持的网络协议

AREA ADDR

区域地址

INTF ADDR

接口地址

INTF ADDR V6

接口的IPv6地址

Topology

拓扑类型

NBR ID

邻居的系统ID

+NBR ID

可以携带TE信息的邻居系统ID

+MT NBR ID

MT拓扑中的邻居ID

COST

指定路由的开销值

MT

拓扑号：

  • 0代表IPv4拓扑和标准IPv6拓扑
  • 2代表IPv6拓扑

IP-Internal

域内IPv4路由信息

IP-External

域外IPv4路由信息

+IP-Extended

扩展的IP路由信息，可以携带与TE相关的信息

IPV6

域内IPv6路由信息

Router ID

本地路由器ID

+SRLG NBR ID

同在一个共享风险链路组的邻居ID
