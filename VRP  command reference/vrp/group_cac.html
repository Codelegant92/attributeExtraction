group（CAC）

命令功能

group命令用来设置某全局节目组、某VLAN节目组或某VSI节目组内的组播组成员，并可以设置每个组播组成员的带宽，若未设置带宽的组播组，其带宽大小为0。

undo group命令用来取消对某全局节目组、某VLAN节目组或某VSI节目组内的组播组成员的设置。

命令格式

group group-address { group-mask-length | group-mask } [ per-bandwidth bandwidth ]

undo group group-address { group-mask-length | group-mask }

参数说明

group-address{ group-mask-length | group-mask }：组播组地址、地址掩码长度、地址掩码。用来指定所要删除的组播组地址，采用点分十进制表示。

per-bandwidth bandwidth：每个组播组的带宽。整数形式。取值范围是1～4194304，单位是千比特/秒。

视图

某全局节目组视图、某VLAN节目组视图和某VSI节目组视图

缺省级别

2 ：配置级

使用指南

全局节目组的组播组的地址不能与全局的所有节目组内的其他组播组地址重叠，不能与VLAN节目组内的组播组地址重叠，不能与VSI节目组内组播组地址重叠。

此命令为不可覆盖式命令。

使用实例

# 在VLAN10的节目组A1中设置节目组

<Quidway> system-view
[Quidway] l2-multicast-channel vlan 10
[Quidway-l2-channel-vlan10] channel A1
[Quidway-l2-channel-vlan10-A1] group 238.0.0.1 30 per-bandwidth 4
[Quidway-l2-channel-vlan10-A1]
