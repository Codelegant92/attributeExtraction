reset rpr traffic interface ring-if

命令功能

reset rpr traffic interface ring-if命令用来清除RPR的流量统计信息。

命令格式

reset rpr traffic interface ring-if interface-number

参数说明

interface-number：RPR逻辑接口编号。整数形式，取值范围是0～31。

视图

用户视图

缺省级别

1：监控级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除接口1的RPR流量统计信息。

<Quidway> reset rpr traffic interface ring-if 1
