display mpls l2vpn label-space

命令功能

display mpls l2vpn label-space命令用来显示label分配空间分布信息以及label二级管理缓冲区信息。

命令格式

display mpls l2vpn label-space

参数说明

无

视图

所有视图下

缺省级别

1：监控级

使用指南

使能L2VPN后，执行命令display mpls l2vpn label-space将显示label分配空间分布信息以及label二级管理缓冲区信息。

使用实例

# L2VPN没有使能时查看信息

<Quidway> display mpls l2vpn label-space
Warning: MPLS L2VPN disabled. Please enable MPLS L2VPN first

# L2VPN使能时，且缓存空间有标签时，查看信息

<Quidway> display mpls l2vpn label-space
Label:  17408
there are 1 labels or label blocks in label-space.
------------------------------------------------------------
label(s) released in SVC connection:              0
label(s) released in LDP connection:              1
label block(s) released in BGP connection:        0
label(s) released in VPLS LDP connection:         0
label block(s) released in VPLS BGP connection:   0

项目

描述

Label

标签缓存中VLL SVC方式标签的总数。

Label(s) released in LDP connection

标签缓存中VLL LDP方式标签的总数。

Label block(s) released in BGP connection

标签缓存中VLL BGP方式标签块的总数。

Label (s) released in VPLS LDP connection

标签缓存中VPLS LDP方式标签块的总数。

Label (s) released in VPLS BGP connection

标签缓存中VPLS BGP方式标签块的总数。
