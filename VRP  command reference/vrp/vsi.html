vsi

命令功能

vsi命令用来创建VSI实例或进入VSI实例视图。

undo vsi命令用来删除一个VSI实例。

缺省情况下，没有定义VSI实例。

命令格式

vsi vsi-name [ auto | static ]

undo vsi vsi-name

参数说明

vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

auto：采用自动成员发现方式。

static：采用静态成员发现方式。

视图

系统视图

缺省级别

2：配置级

使用指南

第一次创建VSI实例必须同时指定该实例的成员发现方式。信令方式在创建VSI实例后使用命令pwsignal设置，对于静态成员发现方式，配置使用LDP；对于自动成员发现方式配置BGP。为VSI实例指定成员发现方式后不可更改。如果需要更改VSI实例的成员发现方式，需删除该实例，重新创建实例后再指定成员发现方式。

如果系统已存在相应VSI实例时，vsi命令用来进入VSI视图。

使用实例

# 创建一个静态成员发现方式的VSI。

<Quidway> system-view
[Quidway] vsi aa static
[Quidway-vsi-aa]

# 删除VSI实例。

<Quidway> system-view
[Quidway] undo vsi aa
