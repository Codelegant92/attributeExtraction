traffic-statistics enable（VSI-LDP视图）

命令功能

traffic-statistics enable命令用来使能VSI LDP 方式下基于VSI的全局流量统计。

undo traffic-statistics enable命令用来去使能VSI LDP 方式下基于VSI的全局流量统计。

命令格式

traffic-statistics enable

undo traffic-statistics enable

参数说明

无

视图

VSI-LDP视图

缺省级别

2：配置级

使用指南

进入VSI-LDP视图后，使用此命令可以使能VSI LDP方式下的VSI全局流量统计。

当先配置使能全局流量统计再配置peer时，如果超过了流量统计的规格上限，此时peer依然可以配置，但对此peer的流量统计不能生效。而在重启系统后配置恢复时，先执行peer的配置，再执行使能全局流量统计的配置，此时系统会得出判断：如果使能全局流量统计将会使规格超过上限，因此不会使能，该配置无法恢复。

使用实例

# 使能VSI 1的全局流量统计功能。

[Quidway]vsi 1
[Quidway-vsi-1]pwsignal ldp
[Quidway-vsi-1-ldp]traffic-statistics enable
