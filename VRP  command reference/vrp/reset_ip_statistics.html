reset ip statistics

命令功能

reset ip statistics命令用来清除接口的IP流量统计信息。

命令格式

在分布式环境下，命令格式是：

reset ip statistics [ interface interface-type interface-number | slot slot-id ]

在集中式环境下，命令格式是：

reset ip statistics [ interface interface-type interface-number ]

参数说明

interface interface-type interface-number：指定接口类型和接口编号。

slot slot-id：指定槽号，以清除该接口板的IP统计信息。

使用指南

在某些情况下，需要统计一定时间内接口的IP统计信息，这时必须在统计开始前清除原有的统计信息，重新进行统计。如果不指定任何可选参数，将清除所有板的IP统计信息，对应的显示命令为display ip statistics。

使用实例

# 清除所有接口的IP统计信息。

<Quidway> reset ip statistics

# 清除插槽1上所有接口的IP统计信息。

<Quidway> reset ip statistics slot 1

# 清除接口POS1/0/0的IP统计信息。

<Quidway> reset ip statistics interface pos 1/0/0
