qinq 8021p-mode

命令功能

qinq 8021p-mode命令用来在子接口终结时配置802.1p的映射方式。

undo qinq 8021p-mode命令用在子接口终结时来取消802.1p的映射方式。

缺省情况下，QinQ子接口信任外层Tag的优先级。

命令格式

qinq 8021p-mode { trust { ce-vid-8021p | pe-vid-8021p } | precedence-value }

undo qinq 8021p-mode

参数说明

ce-vid-8021p：配置内层VLAN的802.1p值映射到DSCP或者映射到MPLS报文中exp字段。

pe-vid-8021p：配置外层VLAN的802.1p值映射到或者映射到MPLS报文中exp字段DSCP。

precedence-value：手工指定的优先级的映射值，十进制整数形式，取值范围为0～7。

视图

Ethernet子接口视图、GE子接口视图、Eth-Trunk子接口视图

缺省级别

2：配置级

使用指南

在以太网子接口上使用qinq 8021p-mode命令前，应先在该子接口下配置control-vid vid qinq-termination命令，否则无法看到qinq 8021p-mode命令。

配置外层VLAN的802.1p值映射到DSCP仅在子接口终结时可选。子接口Stacking时不可选。

使用实例

# 在子接口GE1/0/0.1上配置终结功能后，在该子接口配置8021p映射方式，使用外层VLAN的802.1p值映射到DSCP。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mode user-termination
[Quidway-GigabitEthernet1/0/0] quit
[Quidway] interface gigabitethernet1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] control-vid 1 qinq-termination
[Quidway-GigabitEthernet1/0/0.1] qinq termination pe-vid 100 ce-vid 500 to 600
[Quidway-GigabitEthernet1/0/0.1] qinq 8021p-mode trust pe-vid-8021p
