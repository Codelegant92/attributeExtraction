display x25 hunt-group-info

命令功能

display x25 hunt-group-info命令用来显示X.25搜索群的状态信息。

命令格式

display x25 hunt-group-info [ hunt-group-name ]

参数说明

hunt-group-name：搜索群名称，字符串形式，长度为1～24。

视图

所有视图

缺省级别

2：配置级

使用指南

利用此命令可以了解路由器的搜索群以及搜索群内接口和XOT通道的信息。

使用实例

# 显示X.25搜索群hg1的状态信息

<Quidway> display x25 hunt-group-info hg1
HG_ID ：hg1    HG_Type：round-robin  
member      state     vc-used   in-pkts   out-pkts
Serial1/0/0 Last      2         51        20
Serial2/0/0 Next      1         21        15
1.1.1.1     Normal    1         24        3
表1 display x25 hunt-group-info命令输出信息描述

项目

描述

hg1

搜索群名称。

round-robin

搜索群呼叫通道选择策略。

Member

搜索群内所包括的接口或XOT通道。

State
当前接口或XOT通道状态，包括：

  • Last：本次刚使用过。

  • Next：rotary选择策略下次将选择的接口或XOT通道。

  • Normal：普通状态。

vc-used

在接口或XOT通道上呼叫的次数（包括呼叫成功和呼叫失败）。

in-pkts

在接口或XOT通道上的输入流量，单位是传输的分组个数。

out-pkts

在接口或XOT通道上的输出流量，单位是传输的分组个数。
