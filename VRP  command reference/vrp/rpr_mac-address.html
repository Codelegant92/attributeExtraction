rpr mac-address

命令功能

rpr mac-address命令用来设置逻辑接口的MAC地址。

undo rpr mac-address命令用来取消设置。

命令格式

rpr mac-address mac-address

undo rpr mac-address

参数说明

mac-address：逻辑接口的MAC地址，格式为H-H-H。其中H为4位的十六进制数，可以输入1～4位，如00e0、fc01。当输入不足4位时，表示前面的几位为0，如：输入e0，等同于00e0。MAC地址不可设置为FFFF-FFFF-FFFF。

缺省级别

2：配置级

使用指南

MAC地址的变化，会影响环上拓扑变化，请用户慎重使用。需要修改时，建议如下：

  1. 先执行shutdown命令关闭RPR逻辑接口；

  2. 然后修改MAC地址；

  3. 最后再执行undo shutdown开启RPR逻辑接口，重新开始工作。

使用实例

# 创建一个MAC地址为2-2-2的静态逻辑接口。

<Quidway> system-view
[Quidway] interface ring-if 1
[Quidway -ring-if1] rpr mac-address 2-2-2
