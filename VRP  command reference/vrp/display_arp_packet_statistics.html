display arp packet statistics

命令功能

display arp packet statistics命令用来查询ARP报文的统计数据。

命令格式

display arp packet statistics [ slot slot-id ]

参数说明

slot slot-id：当前在位的接口板的槽位号。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定接口板槽号，则显示各接口板上ARP报文的统计信息。如果指定接口板槽号，则显示指定接口板上ARP报文的统计信息。

ARP报文的统计信息包括：

  • 收到的ARP报文数目

  • 丢弃的ARP报文数目(包括时间戳抑制导致的ARP报文，时间戳抑制包括对ARP报文和ARP Miss消息的时间戳抑制)

  • 学习ARP的次数

使用实例

# 查询所有ARP统计情况。

<Quidway> display arp packet statistics
ARP Pkt Received                       :   sum     23
ARP-Miss Msg Received                  :   sum      0
ARP Learnned Count                     :   sum      8
ARP Pkt Discard For Limit              :   sum      5
ARP Pkt Discard For SpeedLimit         :   sum      0
ARP Pkt Discard For Other              :   sum     10
ARP-Miss Msg Discard For SpeedLimit    :   sum      0
ARP-Miss Msg Discard For Other         :   sum      0
