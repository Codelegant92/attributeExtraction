display l2vpn ccc-interface vc-type

命令功能

display l2vpn ccc-interface vc-type命令用来查看L2VPN连接使用的接口的信息。

命令格式

display l2vpn ccc-interface vc-type { all | vc-type } [ down | up ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

all：显示所有L2VPN连接的接口信息。

vc-type：L2VPN连接的类型，此参数为bgp-vc、ccc、ldp-vc、rsvp-vc或static-vc。VPLS连接中此参数为vpls-vc。

down：显示处于Down状态的L2VPN连接的接口信息。

up：显示处于Up状态的L2VPN连接的接口信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

在定位各种L2VPN的故障时，可以使用display l2vpn ccc-interface vc-type命令查看该连接使用的AC接口的信息，包括使用于各类L2VPN连接的AC接口总数，各接口的状态、封装类型及其对应的L2VPN连接类型。

使用实例

# 查看所有L2VPN的接口信息。

<Quidway> display l2vpn ccc-interface vc-type all
Total ccc-interface of VPLS VC: 1
up (1), down (0)
Interface                  Encap Type                   State    VC Type
GegabitEthernet1/0/1.1     ethernet sub interface       up       vpls-vc
表1 display l2vpn ccc-interface vc-type命令的输出信息描述

项目

描述

Total ccc-interface of vpls VC:1up (1), down (0)

表示L2VPN连接的总数为1。其中有一个L2VPN状态为up；有0个L2VPN连接状态为down。

Interface

该L2VPN连接在本设备上关联的接口。

Encap Type

该L2VPN连接的封装类型。

State

该L2VPNl连接的当前状态。

VC Type

该L2VPN连接的类型。

  • CCC：交叉电路连接

  • ldp-vc：Martini方式的L2VPN连接

  • static-vc：SVC方式的L2VPN连接

  • bgp-vc：Kompella方式的L2VPN连接

  • vpls-vc：VPLS连接

# 查看所有vpls-vc类型的L2VPN的接口信息。

<Quidway> display l2vpn ccc-interface vc-type vpls-vc
Total ccc-interface of VPLS VC: 1
up (1), down (0)
Interface                    Encap Type                    State   VC Type
GigabitEthernet1/0/1.1    ethernet sub interface      up      vpls-vc
