bulk-stat enable

命令功能

bulk-stat enable命令用来使能批量性能统计特性。

undo bulk-stat enable命令用来去使能批量性能统计特性。

缺省情况下，没有使能批量性能统计特性。

命令格式

bulk-stat enable

undo bulk-stat enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

在启动批量性能统计特性后，被管理设备可以通过FTP方式把统计的结果发送到网管站。

该命令为交互式命令，在关闭时需要用户输入确认信息。

使用实例

# 使能批量性能统计特性。

<Quidway> system-view
[Quidway] bulk-stat enable
Info：Succeeded in enabling the bulk stat function.

# 关闭批量性能统计特性。

<Quidway> system-view
[Quidway] undo bulk-stat enable
Warning: All bulk stat configuration will be deleted. Continue? [Y/N]:y
Info: Succeeded in disabling the bulk stat function.
