display mpls ldp lsp

命令功能

display mpls ldp lsp命令用来查看使用LDP创建的LSP相关信息。

命令格式

display mpls ldp lsp [ vpn-instance vpn-instance-name ] [ destination-address mask-length ] [ | { begin | exclude | include } regular-expression ]

display mpls ldp lsp [ all ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：VPN实例的名字。使用此参数查看指定VPN下的LDP LSP信息。

destination-address：显示使用LDP创建的，指定目的IPv4地址的LSP信息。

mask-length：目的IPv4地址的掩码长度。整数形式，取值范围是0～32。

all：显示所有使用LDP创建的LSP的相关信息。

| ：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用LDP建立LSP的时候，可以执行此命令查看经过各设备的LSP的数量和详细信息。

表1 display mpls ldp lsp命令输出信息描述

项目

描述

SN

LSP的序号。

SN前面如果有“ * ”表示未建立LSP。

DestAddress/Mask

LSP的目的地址和掩码。

In/OutLabel

入/出标签值。

In/OutLabel前面有“ * ”表示LSP为Stale LSP，需要恢复。

Next-Hop

下一跳IP地址。

In/Out-Interface

入接口与出接口名称分别用“/”分开。
