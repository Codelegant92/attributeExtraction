display qos pq interface

命令功能

display qos pq interface命令用来显示接口的优先队列配置情况和统计信息。

命令格式

display qos pq interface [ interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口号。

视图

所有视图

缺省级别

1：监控级

使用指南

display qos pq interface命令显示的信息包括：

  • 接口上出队列的大小/长度/丢弃包数

  • 高优先级队列参数

  • 中优先级队列参数

  • 正常优先级队列参数

  • 低优先级参数

如不指定接口，将显示所有接口的优先队列配置情况和统计信息。

使用实例
