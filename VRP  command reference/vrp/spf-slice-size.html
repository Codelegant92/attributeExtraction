spf-slice-size

命令功能

spf-slice-size命令用来设置IS-IS每次路由计算的最大持续时间。

undo spf-slice-size命令用来恢复缺省设置。

缺省情况下，每次路由计算的最大持续时间为4毫秒。

命令格式

spf-slice-size duration-time

undo spf-slice-size

参数说明

duration-time：每次路由计算的最大持续时间。整数类型，取值范围是1～5000，单位是毫秒。缺省值是4。

视图

IS-IS视图

缺省级别

2：配置级

使用指南

当路由表中的路由数目非常多时，为防止路由计算占用系统资源的时间过长，可以通过本命令来缩短每次路由计算的最大持续时间。

使用实例

# 设置IS-IS每次路由计算的最大持续时间为50毫秒。

<Quidway> system-view
[Quidway] isis
[Quidway-isis-1] spf-slice-size 50
