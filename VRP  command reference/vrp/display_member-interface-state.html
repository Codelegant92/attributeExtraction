display member-interface-state

命令功能

display member-interface-state命令用来查看成员交换机不相关接口NDP或NTDP的状态。

命令格式

display member-interface-state { ndp | ntdp }

参数说明

无

视图

集群视图

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

当使用cluster-member unrelated-port命令配置成员交换机上不需要运行NDP和NTDP协议的接口，使能/禁止NDP和NTDP功能后，可以使用本命令用来查看成员交换机不相关接口NDP或NTDP的状态。

指定成员交换机的成员编号时不能超过集群支持的最大编号。

使用实例

# 查看成员交换机不相关接口NDP的状态。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] display member-interface-state ndp
The result of member switches executed disable member interface command:
------------------------------------------------------------------------------
SN        Device         MacAddress          IpAddress           result
------------------------------------------------------------------------------
3         NE40E          0004-0004-0004      10.0.0.4            success
2         NE40E          0003-0003-0003      10.0.0.3            success
1         NE40E          0002-0002-0002      10.0.0.2            success

# 查看成员交换机不相关接口NTDP的状态。

<HUAWEI_0.Quidway> system-view
[HUAWEI_0.Quidway] cluster
[HUAWEI_0.Quidway-cluster] display member-interface-state ntdp
The result of member switches executed disable member interface command:
------------------------------------------------------------------------------
