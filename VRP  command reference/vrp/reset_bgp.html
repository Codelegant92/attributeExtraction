reset bgp

命令功能

reset bgp命令用来复位指定的BGP连接。

命令格式

reset bgp [ vpn-instance vpn-instance-name | vpnv4 ] { all | as-number | ipv4-address | group group-name | external | internal }

reset bgp ipv4 all

reset bgp l2vpn { all | as-number | ipv4-address | external | internal }

reset bgp ipv6 { all | as-number | group group-name | ipv6-address | ipv4-address | external | internal }

reset bgp vpnv6 { all | as-number | ipv4-address | group group-name | external | internal }

reset bgp vpn6-instance vpn6-instance-name { all | as-number | ipv6-address | group group-name | external }

参数说明

vpn-instance vpn-instance-name：复位指定VPN实例的BGP连接。

vpnv4：复位VPNv4相关的BGP连接。

all：复位所有BGP连接。

as-number：复位与指定自治系统号的BGP邻居的BGP会话。

ipv4-address：复位与指定BGP对等体的连接。

group group-name：复位与指定BGP对等体组的连接。

external：复位所有EBGP连接。

internal：复位所有IBGP连接。

l2vpn：复位L2VPN相关的BGP连接。

ipv6-address：复位与指定BGP4+对等体的TCP连接，通过此连接学到的所有路由也同时被删除。

vpnv6：复位VPNv6相关的BGP连接。

vpn6-instance vpn6-instance-name：复位指定IPv6 VPN实例的BGP连接。

使用指南

注意：

使用该命令后将导致BGP建立的TCP连接复位，并重新建立对等体间的邻居关系。请务必谨慎使用。

使用实例

# 复位所有BGP连接。

<Quidway> reset bgp all

# 复位与对等体2345:ABCD::9的BGP会话连接。

<Quidway> reset bgp ipv6 2345:ABCD::9

# 复位所有BGP VPNv6连接。

<Quidway> reset bgp vpnv6 all
