ftp

命令功能

ftp命令用来与远程FTP服务器建立控制连接，并进入FTP客户端视图。

命令格式

ftp [ -a source-ip-address | -i interface-type interface-num ] [ host [ port-number ] ] [ { public-net | vpn-instance vpn-instace-name } ]

参数说明

-a source-ip-address：本端路由器的IPv4地址。

-i interface-type interface-num：本端路由器的出接口。

host：远程FTP服务器的IP地址或主机名称，字符串形式，取值范围1～20。

port-number：远程FTP服务器的端口号，整数形式，取值范围是1～65535。

public-net：在公网中连接FTP服务器。

vpn-instance vpn-instace-name：远程FTP服务器端的VPN实例名，字符串形式，取值范围1～31。

使用指南

命令中如果未带任何参数，则仅表示进入FTP视图，此时并未与任何FTP服务器建立连接。

命令中如果带有VPN实例名，则标识以FTP方式登录到哪一个VPN实例中的FTP服务器上。命令中的-a、-i选项使得用户可以设置源地址为Loopback地址，增加了安全性。
