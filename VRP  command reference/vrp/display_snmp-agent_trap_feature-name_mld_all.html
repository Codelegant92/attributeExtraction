表1 display snmp-agent trap feature-name mld all命令显示信息说明

项目

描述

Feature name

模块名称。

Trap number

告警事件的类型数。

Trap name
事件类型，有以下类型：
  • JOINGRP：收到MLD加入报文。
  • LEAVEGRP：收到MLD离开报文或组定时器超时。

Status
告警开关状态，有以下两种：
  • on：告警开关打开。
  • off：告警开关关闭。

Excessive Flag
海量信息标识，有以下两种取值：
  • Y：表示当前类型的信息是海量信息。
  • N：表示当前类型的信息不是海量信息。
