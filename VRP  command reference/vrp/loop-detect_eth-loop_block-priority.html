loop-detect eth-loop block-priority

命令功能

loop-detect eth-loop block-priority命令用来配置接口在以太环路检测中的阻断优先级。

undo loop-detect eth-loop block-priority命令用来恢复接口在以太环路检测中的阻断优先级。

缺省情况下，接口的优先级为0，即最先被阻断。

命令格式

loop-detect eth-loop block-priority block-priority

undo loop-detect eth-loop block-priority

参数说明

block-priority：接口在以太环路检测中的阻断优先级，整数形式，取值范围是1～256。

缺省级别

使用指南

可以配置Ethernet接口、Ethernet子接口、GE接口、GE子接口、VLANIF接口、Eth-Trunk、Eth-Trunk子接口和Dot1q终结子接口的阻断优先级。

在产生环路的情况下，block-priority值小的接口优先被阻断。

使用实例

# 配置GE1/0/0接口的以太环路检测阻断优先级为20。

<Quiway>system-view
[Quiway] interface gigabitEthernet 1/0/0
[Quiway-GigabitEthernet1/0/0]loop-detect eth-loop block-priority 20
