reset pw

命令功能

reset pw命令用于重新建立PW或PW模板。

命令格式

reset pw { pw-id pw-type | pw-template pw-template-name }

参数说明

pw-id：L2VPN连接的VC ID，十进制整数形式，取值范围是1～4294967295。

pw-type：PW的封装类型。支持reset pw命令的PW封装类型有atm-aal5-sdu、atm-trans-cell、ethernet、fr、hdlc、ip-interworking、ip-layer2、ppp、vlan。

pw-template：重建PW模板下所有PW。

pw-template-name：指定的PW模板名称。字符串形式，不支持空格，区分大小写，长度范围是1～19。

使用指南

改变了PW模板参数，则必须使用reset pw-template命令后，新配置才能起作用，而且对所有使用该模板的PW都起作用；PW的一些属性可以通过PW模板来配置，也可以通过命令行配置，通过命令行配置的优先级较高。如果在命令行中指定了PW属性，则PW模板中的相应PW属性不起作用，因此执行reset pw template和reset pw vc-id命令，PW属性不会改变。可以将需要更改的PW属性设置在PW模板上，而不是通过命令指定。

如果已建立PW模板，重新建立该PW模板就会重新建立该PW模板下的所有PW。如果PW引用了PW模板，当该PW模板属性发生变化时，该PW的属性也随之更新。

如果在命令行中指定了PW属性，则PW模板中的相应PW属性不起作用，因此执行reset pw pw-template或reset pw命令，PW属性不会改变。

使用实例

# 通过VC-ID和VC-Type重建PW。

<Quidway> reset pw 100 ppp

# 重建名为pwt1的PW模板下所有PW。

<Quidway> reset pw pw-template pwt1
