oam-detecting enable

命令功能

oam-detecting enable命令用于使能L2VPN与OAM对PSN隧道的检测功能。

undo oam-detecting enable命令用于取消L2VPN与OAM对PSN隧道的检测功能。

缺省情况下，L2VPN与OAM对PSN隧道不进行检测。

命令格式

oam-detecting enable

undo oam-detecting enable

参数说明

无

视图

L2VPN视图

缺省级别

2：配置级

使用指南

使能L2VPN与OAM对PSN隧道的检测功能后，可以

  • 快速而有效地检测、确认并定位出MPLS层网络内部的缺陷。

  • 报告缺陷并做出相应的处理。

  • 在出现故障的时候，使VSI感知到该LSP处于Down状态，然后进行删除MAC的处理。

使用实例

# 使能L2VPN与OAM对PSN隧道的检测功能。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] oam-detecting enable

# 取消L2VPN与OAM对PSN隧道的检测功能。

<Quidway> system-view
[Quidway] mpls l2vpn
[Quidway-l2vpn] undo oam-detecting enable
