portswitch batch

命令功能

portswitch batch命令用来批量将以太网接口从三层模式切换到二层模式。

undo portswitch batch命令用来批量将以太网接口从二层模式切换到三层模式。

命令格式

portswitch batch interface-type { interface-number1 [ to interface-number2 ] } &<1-10>

undo portswitch batch interface-type { interface-number1 [ to interface-number2 ] } &<1-10>

参数说明

interface-type：接口类型，必须是以太类接口，例如FE、GE、Eth-Trunk、Virtual-Ethernet。

interface-number：接口编号。

使用指南

输入端口范围时，应保证输入的端口格式正确：

  • 关键字to之后的端口号要大于to之前的端口号

  • 要保证采用to形式输入的端口类型相同

  • 两者之间包含的端口都存在

一条portswitch batch命令中，最多可以使用10次to形式输入10个端口范围。

如果其他接口已经配置了借用ethernet接口、GE接口或VE接口的IP地址，则不能对对应的ethernet接口、GE接口或VE接口执行portswitch batch命令。

使用实例

# 将接口GE1/0/0、2/0/0、3/0/0切换为二层模式。

<Quidway> system-view
[Quidway] portswitch batch gigabitethernet 1/0/0 2/0/0 3/0/0
