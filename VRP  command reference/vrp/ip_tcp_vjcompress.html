ip tcp vjcompress

命令功能

ip tcp vjcompress命令用来允许PPP接口进行VJ TCP头压缩。

undo ip tcp vjcompress命令用来禁止PPP接口进行VJ TCP头压缩。

缺省情况下，PPP接口禁止使用VJ TCP头压缩。

命令格式

ip tcp vjcompress

undo ip tcp vjcompress

参数说明

无

使用指南

如果允许PPP接口进行VJ TCP头压缩，线路对端的接口也应同时允许VJ TCP头压缩。

使用实例

# 在PPP接口上允许VJ TCP头压缩。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] ip tcp vjcompress
