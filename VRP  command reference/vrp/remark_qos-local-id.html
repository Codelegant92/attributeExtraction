remark qos-local-id

命令功能

remark qos-local-id命令用来配置标记QoS本地标识。

undo remark qos-local-id命令用来取消标记QoS本地标识。

命令格式

remark qos-local-id qos-local-id

undo remark qos-local-id

参数说明

qos-local-id：QoS本地标识，取值范围是1～4095。

流行为视图

缺省级别

2：配置级

使用指南

remark qos-local-id命令只在NE16E/08E/05路由器上可用。

使用实例

# 配置标记报文的QoS本地标识为200。

<Quidway> system-view
[Quidway] traffic behavior database
[Quidway-behavior-database] remark qos-local-id 200
