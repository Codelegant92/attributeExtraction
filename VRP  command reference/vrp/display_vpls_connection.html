display vpls connection

命令功能

display vpls connection命令用于显示VPLS的连接信息。

命令格式

display vpls connection [ ldp | bgp | vsi vsi-name ] [ down | up ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

ldp：用于显示LDP信令方式的各种连接信息。

bgp：用于显示BGP信令方式的各种连接信息。

vsi vsi-name：用于显示指定VSI实例的各种连接信息。字符串形式，不支持空格，区分大小写，取值范围是1～31。

down：用于显示处于Down状态的各种连接信息。

up：用于显示处于Up状态的各种连接信息。

verbose：用于显示详细的连接信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

执行display vpls connection命令后，可以通过在命令行中指定不同的关键字或参数来显示所需要的连接信息。

如果此命令不指定关键字或参数，将显示所有处于Up状态的连接信息。

使用实例

# 显示所有VSI实例的连接信息。

<Quidway> display vpls connection

VSI Name: bgp1                             Signaling: bgp
SiteID  RD                      PeerAddr          InLabel   OutLabel  VCState

项目

描述

VSI Name

虚拟交换实例VSI的名称。

Signaling

信令方式，为LDP或BGP。

VsiID

VSI的标志符。

EncapType

VSI的VPLS封装形式，即报文在VC上的封装形式。

PeerAddr

对等体的IP地址。

InLabel

本地分发的VC标签。

OutLabel

本地出标签，即对等体分发的VC标签。

VCState

VC的状态。

SiteID

远端VSI实例的Site编号。

RD

即Route Distinguisher，在以BGP为信令的VPLS中用来标识PE上的一个VSI实例。
