hwtacacs-server traffic-unit

命令功能

hwtacacs-server traffic-unit命令用来设置HWTACACS服务器使用的流量单位。

缺省情况下，使用字节做为流量的单位（byte）。

命令格式

hwtacacs-server traffic-unit { byte | kbyte | mbyte | gbyte }

参数说明

byte：以字节做为流量的单位。

kbyte：以千字节做为流量的单位。

mbyte：以兆（Mega）字节做为流量的单位。

gbyte：以吉（Giga）字节做为流量的单位。

视图

HWTACACS服务器模板视图

缺省级别

2：配置级

使用指南

只有当该hwtacacs Server模板没有用户使用时，才能改变此配置。

使用实例

# 设置HWTACACS服务器使用千字节做为流量单位。

<Quidway> system-view
[Quidway] hwtacacs-server template test1
[Quidway-hwtacacs-test1] hwtacacs-server traffic-unit kbyte
