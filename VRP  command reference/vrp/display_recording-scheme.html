display recording-scheme

命令功能

display recording-scheme命令用来查看记录方案的配置情况。

命令格式

display recording-scheme [ recording-scheme-name ]

参数说明

recording-scheme-name：记录方案名称，字符串形式，长度范围是1～32，不区分大小写。

视图

所有视图

缺省级别

1：监控级

使用指南

display recording-scheme命令查看记录方案的配置情况包括：

  • 记录方案名

  • 是否计费

  • 使用哪个模板计费

如果不指定记录方案名，将显示所有当前存在的记录方案配置情况。

使用实例

# 查看名为currentscheme的记录方案配置情况。

<Quidway> display recording-scheme currentscheme
-----------------------------------------------------------------
 Recording-scheme-name           : currentscheme
HWTACACAS-template-name          : NO SET
----------------------------------------------------------------

# 查看所有当前存在的计费方案配置情况。

<Quidway> display recording-scheme
------------------------------------------------------------------
 Recording-schemename                HWTACACS-template-name
 currentscheme                              NO SET
------------------------------------------------------------------
Total 1,1 printed
