if-match mpls-label

命令功能

if-match mpls-label命令用来设置一个基于MPLS标签的匹配规则，即匹配带有MPLS标签的路由。

undo if-match mpls-label命令用来取消该设置。

缺省情况下，没有配置基于MPLS标签的匹配规则。

命令格式

if-match mpls-label

undo if-match mpls-label

参数说明

无

使用实例

# 定义一条匹配规则，允许带标签的路由通过该规则的过滤。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] if-match mpls-label
