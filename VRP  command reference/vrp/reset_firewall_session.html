reset firewall session

命令功能

reset firewall session命令清除会话表项的相关信息。

命令格式

分布式下：

reset firewall session { table | statistics } slot slot-id

参数说明

slot-id：为指定的接口板槽号。

table：会话表项。

statistics：会话统计信息。

视图

用户视图

缺省级别

2：配置级

使用指南

清除统计信息后，以前的统计信息将无法恢复。

使用实例

# 删除会话表项的统计信息。

<Quidway> reset firewall session statistics slot 4
