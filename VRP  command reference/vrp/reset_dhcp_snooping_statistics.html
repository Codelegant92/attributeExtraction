reset dhcp snooping statistics

命令功能

reset dhcp snooping statistics命令用于清除全局下、VLAN下或者接口下的报文丢弃统计计数。

命令格式

reset dhcp snooping statistics [ [ vlan vlanid ] interface interface-type interface-number ]

参数说明

vlan vlan-id：虚拟局域网标识，取值范围1～4094。

interface-type：清除接口报文丢弃统计计数的接口类型。

interface-number：清除接口报文丢弃统计计数的接口号。

视图

系统视图

缺省级别

2：配置级

使用指南

丢弃的报文的种类包括ARP、IP、DHCP Requst、DHCP CHADDR、DHCP Reply。
