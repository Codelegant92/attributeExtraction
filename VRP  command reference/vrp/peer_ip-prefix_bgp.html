peer ip-prefix (BGP)

命令功能

peer ip-prefix命令用来配置对等体/对等体组基于IP前缀列表的路由过滤策略。

undo peer ip-prefix命令用来取消对等体/对等体组基于IP前缀列表的路由过滤策略。

缺省情况下，没有指定对等体/对等体组的关于IP前缀列表的过滤策略。

命令格式

peer { group-name | ipv4-address } ip-prefix ip-prefix-name { import | export }

peer { group-name | ipv4-address | ipv6-address } ipv6-prefix ipv6-prefix-name { import | export }

undo peer { group-name | ipv4-address } ip-prefix { import | export }

undo peer { group-name | ipv4-address | ipv6-address } ipv6-prefix { import | export }

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ip-prefix-name：指定的ip-prefix名称。

import：对由指定对等体/对等体组接收的路由应用过滤策略。

export：对向指定对等体/对等体组发送的路由应用过滤策略。

ipv6-address：对等体的IPv6地址。

ipv6-prefix：配置对等体/对等体组基于IPv6地址前缀列表的路由过滤策略。

ipv6-prefix-name：指定的IPv6地址前缀列表名称。

缺省级别

2：配置级

使用指南

如果需要配置6PE，请根据需要在IPv6单播地址族视图下指定Peer为IPv4单播地址。

使用实例

# 配置基于IP前缀列表的路由过滤策略。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] peer 1.1.1.2 ip-prefix list1 import

# 配置基于地址前缀列表的路由过滤策略。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] ipv6-family
[Quidway-bgp-af-ipv6] peer group1 ipv6-prefix list1 import
