ntdp timer

命令功能

ntdp timer命令用来设置定时拓扑收集的时间间隔。

undo ntdp timer命令用来恢复定时拓扑收集的时间间隔为缺省值。

缺省情况下，定时拓扑收集的时间间隔是0，即不进行定时拓扑收集。

命令格式

ntdp timer interval

undo ntdp timer

参数说明

interval：指定定时收集拓扑的时间间隔。十进制整数形式，取值范围是0～65535，单位是分钟。

使用指南

该命令只有在使能了设备的NTDP功能的情况下才能配置。

使用实例

# 设置定时拓扑收集的时间间隔为20分钟。

<Quidway> system-view
[Quidway] ntdp timer 20
