display clock

命令功能

display clock命令用来显示系统当前日期和时钟。

命令格式

display clock

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

用户可以通过display clock命令查看系统日期和时钟，如果发现系统时间有误，可及时调整。

如果使用clock daylight-saving-time设置过夏令时，则通过display clock命令可查看夏令时的相关信息；如果未配置过，则不显示夏令时相关信息。

使用实例

# 查看系统当前日期和时间。

<Quidway> display clock
