pim ipv6 hello-option dr-priority

命令功能

pim ipv6 hello-option dr-priority命令用来设置本路由器竞选DR的优先级。

undo pim ipv6 hello-option dr-priority命令用来恢复缺省值。

缺省情况下，路由器竞选DR的优先级是1。

命令格式

pim ipv6 hello-option dr-priority priority-value

undo pim ipv6 hello-option dr-priority

参数说明

priority-value：指定竞选DR的优先级。整数形式，取值范围是0～4294967295，数值越大，优先级越高。

使用指南

此命令与IPv6 PIM视图下的hello-option dr-priority（IPv6）命令作用相同，只是视图不同。缺省情况下，如果接口上没有配置优先级，取IPv6 PIM视图下配置的值；否则取接口视图下的值。

请参见IPv4 PIM配置命令中的pim hello-option dr-priority。

使用实例

# 设置接口GE1/0/0竞选DR的优先级为3。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 hello-option dr-priority 3
