pppoe up-down-log

命令功能

pppoe up-down-log命令用来配置是否对PPPoE用户的状态变化进行日志记录。

缺省情况下，对PPPoE用户的状态变化进行日志记录。

命令格式

pppoe up-down-log { enable | disable }

参数说明

enable：使能对PPPoE用户的Up/Down状态进行日志记录。

disable：禁止对PPPoE用户的Up/Down状态进行日志记录。

视图

系统视图

缺省级别

2：配置级

使用指南

当PPPoE用户的数量比较多，且状态变化比较频繁时，大量的日志记录可能会影响正常的系统监控，这时可以选择不对PPPoE用户的状态变化进行记录。

使用实例

# 使能对PPPoE用户的Up/Down状态进行日志记录。

<Quidway> system-view
[Quidway] pppoe up-down-log enable
