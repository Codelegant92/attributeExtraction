display ripng

命令功能

display ripng命令用来显示RIPng进程的当前运行状态及配置信息。

如果不指定可选参数，该命令将显示所有已配置的RIPng进程的参数。

命令格式

display ripng [ process-id ] [ | { begin | exclude | include } regular-expression ]

display ripng [ process-id | vpn6-instance vpn6-instance-name ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIPng进程号。整数形式，取值范围是1～65535。

vpn6-instance vpn6-instance-name：显示指定IPv6 VPN的RIPng多实例进程的概要信息。vpn6-instance-name为IPv6 VPN实例的名称，字符串形式，不支持空格，区分大小写，长度范围是1～31。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示RIPng进程信息。

<Quidway> display ripng 100
