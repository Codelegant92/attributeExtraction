encapsulation（VSI视图）

命令功能

encapsulation命令用来在VSI视图下使能、配置接口的封装能力。

缺省的封装类型设置为VLAN。

命令格式

encapsulation { ethernet | vlan }

参数说明

ethernet：指定封装格式为Ethernet封装格式。

vlan：指定封装格式为符合802.1Q标准的VLAN封装格式。

视图

VSI视图

缺省级别

2：配置级

使用指南

对于Kompella方式的VPLS，使用encapsulation命令前，需要先执行pwsignal命令和route-distinguisher（VSI-BGP视图）命令；对于Martini方式的VPLS，使用encapsulation命令前，需要先执行pwsignal命令和vsi-id命令。

使用实例

# 配置当前VSI实例的封装类型为Ethernet。

<Quidway> system-view
[Quidway] vsi company1 auto
[Quidway-vsi-company1] pwsignal bgp
[Quidway-vsi-company1-bgp] route-distinguisher 100:1
[Quidway-vsi-company1-bgp] quit
[Quidway-vsi-company1] encapsulation ethernet
