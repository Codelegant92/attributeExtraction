vsi-id

命令功能

vsi-id命令用于配置VSI实例的ID。

缺省情况下，没有配置VSI ID。

命令格式

vsi-id vsi-id

参数说明

vsi-id：用于标识一个VSI实例。十进制整数形式，取值范围是1～4294967295。

视图

VSI-LDP视图

缺省级别

2：配置级

使用指南

一个VSI实例的ID配置成功后，不可以再更改。任何两个VSI实例不可以复用一个ID。

使用实例

# 配置当前VSI实例的ID为1。

<Quidway> system-view
[Quidway] vsi company1 static
[Quidway-vsi-company1] pwsignal ldp
[Quidway-vsi-company1-ldp] vsi-id 1
