display fr map-info

命令功能

display fr map-info命令用来显示帧中继地址映射表。

命令格式

display fr map-info [ interface interface-type interface-number ] [ dlci dlci-number ]

参数说明

interface-type interface-number：用于指定查看的接口。指定接口可以是主接口，也可以是子接口。不指定接口则显示全部信息。

dlci dlci-number：DLCI号。取值范围是16～1007。

使用指南

根据此命令的显示信息可以查看用户配置的静态地址映射是否正确以及动态地址映射是否工作正常等。

使用实例

# 显示帧中继地址映射表。

<Quidway> display fr map-info
Map Statistics for interface Serial1/0/2:0 (DTE)
DLCI = 100, IP INARP 100.100.1.1, Serial1/0/2:0
create time = 2002/10/21 14:48:44, status = ACTIVE
encapsulation = ietf, vlink = 14, broadcast

项目

描述

Map Statistics for interface Serial1/0/2:0 (DTE)

工作方式是DTE的Serial1/0/2:0的地址映射信息表。

DLCI =

虚电路号。

IP INARP

动态IP地址。

Serial1/0/2:0

虚电路号和动态IP地址所在的具体接口类型和编号。

create time =

地址映射信息的创建时间。

status =

地址映射的状态。

encapsulation =

接口协议的封装格式。可以使用link-protocol fr命令来改变。

vlink =

虚电路连接的总数。

broadcast

支持广播。
