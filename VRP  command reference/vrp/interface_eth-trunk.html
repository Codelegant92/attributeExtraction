interface eth-trunk

命令功能

interface eth-trunk命令用来创建一个Eth-Trunk接口。

undo interface eth-trunk命令用来删除一个已存在的Eth-Trunk接口。

命令格式

interface eth-trunk trunk-id [.subnumber ]

undo interface eth-trunk trunk-id [.subnumber ]

参数说明

trunk-id：Trunk ID，取值范围是0～63。

subnumber：Eth-Trunk的子接口编号，在VRP上的取值范围是1～4094。但每个Eth-Trunk主接口下最多可创建的子接口数量为4094个。不同产品上实现的Eth-Trunk子接口取值范围不同，请以产品实现的规格为准。

使用指南

接口相关的参数都需要在接口视图下配置，只有先创建了Eth-Trunk接口，才可以对它进行配置。

使用实例

# 创建一个Eth-Trunk接口，编号为1。

<Quidway> system-view
[Quidway] interface eth-Trunk 1
[Quidway-Eth-Trunk1]

# 创建一个Eth-Trunk子接口，编号为10.4094。

<Quidway> system-view
[Quidway] interface eth-Trunk 10
[Quidway-Eth-Trunk10] quit
[Quidway] interface eth-Trunk 10.4094
[Quidway-Eth-Trunk10.4094]
