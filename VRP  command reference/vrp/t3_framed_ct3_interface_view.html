t3 framed（CT3接口视图）

命令功能

t3 framed（CT3接口视图）命令用来配置CT3接口的通道工作在非通道化模式。

undo t3 framed（CT3接口视图）命令用来恢复缺省配置。

缺省情况下，CT3接口工作在通道化模式。

命令格式

t3 framed

undo t3 framed

参数说明

无

视图

CT3接口视图

缺省级别

2：配置级

使用指南

当在CT3接口上执行命令t3 framed后，CT3接口工作在非通道化模式下。此时除0时隙外，CT3接口的所有时隙形成一个速率为44.210Mbit/s的同步串口。

当在CT3接口上执行命令using（CT3接口视图）后，CT3接口工作在净通道化模式下。此时，CT3接口的所有时隙形成一个速率为44.736Mbit/s的同步串口。

当在CT3接口上执行命令using（CT3接口视图）后，CT3接口工作在通道化模式下。此时需要执行命令t1 channel-set（CT3接口视图）或t1 unframed（CT3接口视图），配置CT3接口下的T1通道，CT3接口才能形成串口。

使用实例

# 将CT3 1/0/0配置为非通道化模式。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] t3 framed
