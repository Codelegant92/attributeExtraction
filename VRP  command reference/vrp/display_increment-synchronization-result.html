display increment-synchronization-result

命令功能

display increment-synchronization-result命令用来查看成员交换机的配置文件同步到FTP服务器的结果。

命令格式

display increment-synchronization-result

参数说明

无

视图

集群视图

缺省级别

1：监控级

使用指南

本命令只能在命令交换机上执行。

当使用increment-config synchronization命令把成员交换机的配置文件同步到FTP服务器后，可以使用本命令用来查看成员交换机的配置文件同步到FTP服务器的结果。

指定成员交换机的成员编号时不能超过集群支持的最大编号。

使用实例

# 查看成员交换机的配置文件同步到FTP服务器的结果。

<Quidway> system-view
[Quidway] cluster
[Quidway-cluster] display increment-synchronization-result
The result of member switchs' synchronization:
------------------------------------------------------------------------------
SN   Device              MacAddress     IpAddress      result
------------------------------------------------------------------------------
1    NE40E               0002-0002-0002 10.0.0.2       success
2    NE40E               0003-0003-0003 10.0.0.3       timeout
3    NE40E               0004-0004-0004 10.0.0.4       running

该显示信息主要内容解释见表1。

表1 display increment-synchronization-result命令输出信息描述

项目

描述

SN
成员编号。

  • “0”表示命令交换机。

  • 其它数字表示成员交换机。

Device

设备的类型。

MAC Address

设备的MAC地址。

IpAddress

设备的集群IP地址。

Result
下发增量配置的结果：

  • success

  • failed to get FTP configuration

  • failed to access FTP

  • failed to upload file

  • running

  • timeout

  • error
