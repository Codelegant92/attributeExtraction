peer filter-policy (BGP)

命令功能

peer filter-policy命令用来设置对等体/对等体组的过滤策略。

undo peer filter-policy命令用来取消已有的设置。

缺省情况下，对等体/对等体组无过滤策略。

命令格式

peer { group-name | ipv4-address | ipv6-address } filter-policy { acl-number | acl6-number } { import | export }

undo peer { group-name | ipv4-address | ipv6-address } filter-policy { acl-number | acl6-number } { import | export }

参数说明

group-name：对等体组的名称。不区分大小写。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

acl-number：基本ACL编号。

acl6-number：基本ACL6访问列表号。

import：对接收的路由进行过滤。

export：对发布的路由进行过滤。

说明：

  • ipv4-address和acl-number参数仅在BGP视图、BGP-IPv4单播地址族视图、BGP-IPv6单播地址族视图、BGP-VPN实例视图、BGP-VPNv4地址族视图和BGP-VPNv6地址族视图下生效。

  • ipv6-address和acl6-number参数仅在BGP-IPv6单播地址族视图和BGP-VPN6实例视图下生效。

缺省级别

2：配置级

使用指南

请根据对等体/组的地址族类型正确选择基本ACL或ACL6访问列表。

使用实例

# 设置对等体的过滤策略（IPv4）。

<Quidway> system-view
[Quidway] acl 2000
[Quidway -acl-basic-2000] rule permit
[Quidway -acl-basic-2000] quit
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200
[Quidway-bgp] ipv4-family unicast
[Quidway-bgp-af-ipv4] peer 1.1.1.2 filter-policy 2000 import

# 设置对等体的过滤策略（IPv6）。

<Quidway> system-view
[Quidway] acl ipv6 2001
[Quidway -acl6-basic-2001] rule permit
[Quidway -acl6-basic-2001] quit
[Quidway] bgp 100
[Quidway-bgp] ipv6-family unicast
[Quidway-bgp-af-ipv6] peer 1:2::3:4 filter-policy 2001 export
