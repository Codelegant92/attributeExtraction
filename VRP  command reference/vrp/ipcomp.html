ipcomp

命令功能

ipcomp命令用来设置IPComp的压缩算法。

undo ipcomp命令用来恢复缺省设置。

缺省情况下，IPComp使用LZS无损压缩算法。

命令格式

ipcomp lzs

undo ipcomp

参数说明

lzs：采用LZS无损压缩算法（该参数只使用于NE16E、NE08E、NE05产品并由加密卡硬件提供）。

视图

安全提议视图

缺省级别

2：配置级

使用指南

该配置只使用于NE16E、NE08E、NE05产品并由加密卡硬件提供，必须与AH和/或ESP配合使用，即只有当指定ipsec提议的安全协议为ah-ipcomp、esp-ipcomp、ah-esp-ipcomp时配置才有效，否则配置无效。

使用实例

# 设置提议test先对报文进行LZS无损压缩，再使用ESP。

<Quidway> system-view
[Quidway] ipsec proposal test
[Quidway-ipsec-proposal-test] transform esp-ipcomp
[Quidway-ipsec-proposal-test] ipcomp lzs

# 恢复压缩算法的缺省设置。

[Quidway-ipsec-proposal-test] undo ipcomp
