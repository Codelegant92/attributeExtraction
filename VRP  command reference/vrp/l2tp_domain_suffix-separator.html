l2tp domain suffix-separator

命令功能

l2tp domain suffix-separator命令用来设置作为后缀的分隔符。

undo l2tp domain suffix-separator命令用来删除设置的后缀分隔符。

缺省情况下，不存在作为后缀的域名分隔符。

命令格式

l2tp domain suffix-separator separator

undo l2tp domain suffix-separator separator

参数说明

separator：标识域名分隔符，目前支持的域名分隔符只有“@”。

使用指南

VRP支持使用包含域名的用户全名作为建立L2TP隧道的连接请求。此时，需要使用l2tp domain suffix-separator命令配置域名分隔符，区分域名和用户名。

使用实例

# 域名作为后缀，后缀与用户的名称之间以@号相隔。

<Quidway> system-view
[Quidway] l2tp domain suffix-separator @
