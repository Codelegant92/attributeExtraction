display rrpp snooping enable

命令功能

display rrpp snooping enable命令用来显示使能了RRPP Snooping功能的接口。

命令格式

display rrpp snooping enable { all |interface interface-type interface-number }

参数说明

all：表示查看所有使能了RRPP Snooping功能的接口。

interface interface-type interface-number：接口类型和接口编号。本命令支持的接口类型有Ethernet子接口、GE子接口、Eth-Trunk子接口或VLANIF接口。使用本参数表示只查看指定的接口使能RRPP Snooping功能的情况。

使用指南

通过本命令可以查看哪些接口使能了RRPP Snooping的功能。

使用实例

# 查看所有使能了RRPP Snooping的功能的接口。

<Quidway> display rrpp snooping enable all
  Port                             VsiName          Vlan  
-------------------------------------------------------- 
  GigabitEthernet1/0/0.1           name1            100
  GigabitEthernet2/0/0.1           name2            200

# 查看指定子接口使能RRPP Snooping的功能的情况。

<Quidway> display rrpp snooping enable interface gigabitethernet 1/0/0.1 
  Port                             VsiName          Vlan 
-------------------------------------------------------- 
  GigabitEthernet1/0/0.1           name1            100
