isis cost

命令功能

isis cost命令用来指定进行SPF计算时接口的链路开销。

undo isis cost命令用来恢复缺省的链路开销。

缺省情况下，IS-IS接口的链路开销为10。

命令格式

isis cost cost [ level-1 | level-2 ]

undo isis cost [ level-1 | level-2 ]

参数说明

cost：指定的链路开销值，整数类型，取值范围根据开销类型而定。当开销类型为narrow、narrow-compatible或compatible时，取值范围是1～63；当开销类型为wide或wide-compatible时，取值范围是1～16777215。

level-1：设置Level-1的链路开销。

level-2：设置Level-2的链路开销。

缺省级别

2：配置级

使用指南

如果配置中没有指定Level-1或者Level-2，则为Level-1和Level-2设置同样的链路开销。

建议用户为接口配置适当的链路开销，否则IS-IS路由计算出的链路开销无法反映真实情况。

对于IGP-Shortcut的TE-Tunnel接口，接口的链路开销值优先选择mpls te igp metric命令配置的值。否则选择IS-IS配置的开销值作为接口的链路开销值。

说明：

此命令的优先级比circuit-cost命令高。

使用实例

# 设置接口POS1/0/0上Level-2的链路开销为5。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] isis cost 5 level-2
