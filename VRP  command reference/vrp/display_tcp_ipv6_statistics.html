display tcp ipv6 statistics

命令功能

display tcp ipv6 statistics命令用来查看相关的IPv6 TCP统计信息。

命令格式

display tcp ipv6 statistics [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。
