pim ipv6 timer graft-retry

命令功能

pim ipv6 timer graft-retry命令用来设置重传嫁接消息的时间间隔。

undo pim ipv6 timer graft-retry命令用来恢复缺省值。

缺省情况下，重传嫁接消息的时间间隔是3秒。

命令格式

pim ipv6 timer graft-retry interval

undo pim ipv6 timer graft-retry

参数说明

interval：指定重传嫁接消息的时间间隔。整数形式，取值范围是1～65535，单位是秒。

缺省级别

2：配置级

使用指南

IPv6 PIM-DM模式下，系统连续重传嫁接消息，直至此消息被确认。

请参见IPv4 PIM配置命令中的pim timer graft-retry。

使用实例

# 设置接口GE1/0/0的嫁接重传定时器为80秒。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pim ipv6 timer graft-retry 80
