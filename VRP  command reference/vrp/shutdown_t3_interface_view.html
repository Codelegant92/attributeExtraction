shutdown（T3接口视图）

命令功能

shutdown命令用来禁止当前接口。

undo shutdown命令用来使能当前接口。

命令格式

shutdown

undo shutdown

参数说明

无

视图

T3接口视图、串行接口视图

缺省级别

2：配置级

使用指南

在T3接口视图下执行shutdown/undo shutdown命令可以禁止或使能T3物理端口，该操作对于T3形成的串口也有效。

在串口视图下执行shutdown/undo shutdown命令只禁止或使能当前所在的串口。

使用实例

# 禁止物理端口T3 1/0/0。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] shutdown
