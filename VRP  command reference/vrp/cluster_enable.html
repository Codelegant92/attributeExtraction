cluster enable

命令功能

cluster enable命令用来使能HGMP集群功能。

undo cluster enable命令用来禁止HGMP集群功能。

缺省情况下，HGMP集群功能处于禁止状态。

命令格式

cluster enable

undo cluster enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

在不同HGMP集群角色的设备上执行undo cluster enable命令，将具有不同的结果：

  • 如果在命令交换机上执行undo cluster enable命令，将删除集群和集群的成员，同时禁止本设备的集群功能。

  • 如果在成员交换机上执行undo cluster enable命令，本设备将退出集群，同时禁止本设备的集群功能。

  • 如果在支持集群功能，但没有加入任何集群的设备上执行undo cluster enable命令，将禁止本设备的集群功能。

使用实例

# 使能设备的集群功能。

<Quidway> system-view
[Quidway] cluster enable
