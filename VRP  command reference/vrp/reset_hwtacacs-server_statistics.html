reset hwtacacs-server statistics

命令功能

reset hwtacacs-server statistics命令用来清除HWTACACS服务器的统计信息。

命令格式

reset hwtacacs-server statistics { all | accounting | authentication | authorization }

参数说明

all：清除所有统计信息。

accounting：清除所有HWTACACS计费服务器的统计信息。

authentication：清除所有HWTACACS认证服务器的统计信息。

authorization：清除所有HWTACACS授权服务器的统计信息。

视图

用户视图

缺省级别

1：监控级

使用指南

清除统计信息后，以前的统计信息将无法恢复。

使用实例

# 清除所有统计信息。

<Quidway> reset hwtacacs-server statistics all
