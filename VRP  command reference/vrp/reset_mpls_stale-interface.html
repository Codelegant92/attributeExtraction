reset mpls stale-interface

命令功能

reset mpls stale-interface命令用来清除处于stale状态的MPLS接口的信息。

命令格式

reset mpls stale-interface [ interface-index ]

参数说明

interface-index：指定stale接口的索引号。十六进制整数形式，取值范围是0～FFFFFFFF。

视图

用户视图

缺省级别

2：配置级

使用指南

当MPLS接口所在接口板被拔出，MPLS接口被删除，或该接口去使能MPLS时，接口状态变为Stale。当出现异常时，可执行该命令，清除处于stale状态的MPLS接口信息。

stale接口的接口索引号可通过执行不带参数的display mpls stale-interface命令查看。

使用实例

# 清除指定stale接口信息。

<Quidway> reset mpls stale-interface 40265501

# 清除所有stale接口信息。

<Quidway> reset mpls stale-interface
