display access-user

命令功能

display access-user命令用来显示在线用户的详细信息。

命令格式

display access-user [ domain domain-name | mac-address mac-address | user-id user-id | username user-name | ip-address ip-address [ vpn-instance vpn-instance-name ] ]

参数说明

domain domain-name：指定的域名,字符串形式，取值范围1～20。

ip-address ip-address：IP地址。

vpn-instance vpn-instance-name：VPN实例名。字符串形式，长度范围是1～31。

mac-address mac-address：MAC地址。

user-id：用户ID，整数形式，取值0～383。

user-name：用户名，字符串形式，取值范围1～64。
