firewall default

命令功能

firewall default命令用来设置防火墙的过滤方式为“允许”或“禁止”。

系统缺省的过滤方式为允许。

命令格式

firewall default { permit | deny } { all | slot slot-id }

参数说明

permit：默认过滤规则为允许。

deny：默认过滤规则为禁止。

all：配置作用于所有接口板。

slot-id：为指定的接口板槽号。

使用指南

在没有一个合适的规则去判定用户数据包是否可以通过的时候，防火墙根据这条命令的配置允许或者禁止该数据包通过。

使用实例

# 设置防火墙缺省过滤规则为禁止。

<Quidway> system-view
[Quidway] firewall default deny all
