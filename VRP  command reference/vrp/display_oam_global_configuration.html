display oam global configuration

命令功能

display oam global configuration命令用来查看当前设备的以太网OAM全局配置，包括：

  • 以太网CFM功能的全局使能状态

  • 通用MAC Ping功能的使能状态

  • 通用MAC Trace功能的使能状态

  • MAC tunnel中间节点Trace回复功能的使能状态

  • EFM OAM功能的全局使能状态

  • draft7草案版本中MIP的全局生成规则

  • draft7草案版本中以太网CFM协议号

  • standard2007版本中以太网CFM版本信息

命令格式

display oam global configuration

参数说明

无

使用指南

如今，802．1ag协议分为draft7草案版本和standard2007标准版本。

不同的版本上在配置以太网OAM时，可使用本命令查看该以太网OAM功能是否已全局使能。

使用实例

# 查看draft7版本中当前设备的以太网OAM全局配置。

<Quidway> display oam global configuration
global conf                                          value
----------------------------------------------------------
cfm status                                         enabled
mip create-type                                    default
ping mac status                                   disabled
trace mac status                                  disabled
cfm mac-tunnel mip trace status                   disabled
efm status                                         enabled
cfm protocol                                          bbb0

# 查看standard2007版本中当前设备的以太网OAM全局配置。

global conf                                          value
----------------------------------------------------------
cfm status                                         enabled
ping mac status                                   disabled
trace mac status                                  disabled
cfm mac-tunnel mip trace status                   disabled
efm status                                         enabled
cfm version                                       standard
表1 display oam global configuration命令输出信息描述表

项目

描述

cfm status
以太网CFM功能的全局使能状态。

  • disabled：以太网CFM功能处于全局禁止状态。

  • enabled：以太网CFM功能处于全局使能状态。

mip create-type
draft7版本中MIP的全局生成规则。

  • default：如果指定接口不存在更高级别的MEP，并且不存在更低级别的MIP，则可以在该接口上创建MIP。

  • explicit：如果在指定接口下存在更低级的MEP但不存在更高级别的MEP，而且不存在更低级别的MIP，则可以创建MIP节点。

  • none：接口的MIP生成规则为none类型，即不自动创建MIP。

ping mac status
通用MAC Ping功能的使能状态。

  • disabled：通用MAC Ping功能处于禁止状态。

  • enabled：通用MAC Ping功能处于使能状态。

trace mac status
通用MAC Trace功能的使能状态。

  • disabled：通用MAC Trace功能处于禁止状态。

  • enabled：通用MAC Trace功能处于使能状态。

cfm mac-tunnel mip trace status
MAC tunnel中间节点Trace回复功能的使能状态。

  • disabled：MAC tunnel中间节点Trace回复功能处于禁止状态。

  • enabled：MAC tunnel中间节点Trace回复功能处于使能状态。

efm status
EFM OAM功能的全局使能状态。

  • disabled：EFM OAM功能处于全局禁止状态。

  • enabled：EFM OAM功能处于全局使能状态。

cfm protocol

draft7版本中以太网CFM协议号。

cfm version

standard2007版本中以太网CFM版本信息。
