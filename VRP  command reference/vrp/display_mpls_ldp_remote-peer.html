display mpls ldp remote-peer

命令功能

display mpls ldp remote-peer命令用来查看LDP会话的远端对等体信息。

命令格式

display mpls ldp remote-peer [ remote-peer-name ] [ | { begin | exclude | include } regular-expression ]

参数说明

remote-peer-name：显示指定远端对等体名称的远端对等体信息。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令和display mpls ldp peer命令不同，此命令用来查看本地配置了的远端对等体，只需要完成配置，就可以查看到相关信息。而display mpls ldp peer命令用来查看本地已经发现了的LDP对等体，只有本端和对端都已经完成了配置，并且建立了对等体关系，才能查看到相关信息。

使用实例

# 查看远端对等体信息。

<Quidway> display mpls ldp remote-peer

项目

描述

Remote Peer Name

远端对等体名称

Remote Peer IP

远端对等体IP

LDP ID

本地LDP ID

Transport Address

与远端对等体建立LDP会话的传输地址

Entity Status
远端对等体的状态：
  • Active：激活状态
  • Inactive：非激活状态

Configured Keepalive Timer

配置的keepalive定时器时间

Configured Hello Timer

配置的Hello定时器值

Negotiated Hello Timer

协商的Hello定时器值

Hello Packet sent/received

此远端对等体接收和发送的Hello消息数
