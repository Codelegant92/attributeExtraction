mld require-router-alert

命令功能

mld require-router-alert命令用来设置接口忽略不包含Router-Alert选项的MLD消息，即只接收处理包含Router-Alert选项的MLD消息。

undo mld require-router-alert命令用来恢复配置的缺省情况。

缺省情况下，路由器可接收处理不包含Router-Alert选项的MLD消息。

命令格式

mld require-router-alert

undo mld require-router-alert

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

此命令与require-router-alert（MLD视图）的含义是相同的，只是视图不同。缺省情况下，如果接口上没有配置，取MLD视图下配置的值；否则取接口视图下配置的值。

使用实例

# 设置接口GE1/0/0忽略不包含Router-Alert选项的消息。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld require-router-alert
