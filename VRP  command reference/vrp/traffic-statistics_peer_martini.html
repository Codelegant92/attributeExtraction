traffic-statistics peer（Martini）

命令功能

traffic-statistics peer命令用来使能指定Martini方式VPLS PW上的公网流量统计功能。

undo traffic-statistics peer命令用来去使能指定Martini方式VPLS PW上的公网流量统计功能。

缺省情况下，去使能指定Martini方式VPLS PW上的公网流量统计功能。

命令格式

traffic-statistics peer peer-address [ negotiation-vc-id vc-id ] enable

undo traffic-statistics peer peer-address [ negotiation-vc-id vc-id ] enable

参数说明

peer peer-address：PW的对端IP地址。

negotiation-vc-id vc-id：PW的ID值。十进制整数形式，取值范围是1～4294967295。

视图

VSI-LDP视图

缺省级别

2：配置级

使用指南

用户进入VSI-LDP视图下，对指定Martini方式VPLS PW使能/去使能公网流量统计功能，PW由用户所在的VSI视图与指定的对端IP地址和PW ID唯一确定。

如果已经在VSI-LDP视图下使用traffic-statistics enable命令使能了基于VSI的全局流量统计，则不能使用undo traffic-statistics peer命令删除指定peer的流量统计。

使用实例

# 在VSI-newvsi-ldp下使能指定PW上的公网流量统计功能。

<Quidway> system-view
[Quidway] vsi newvsi static
[Quidway-vsi-newvs] pwsignal ldp
[Quidway- vsi-newvsi-ldp]vsi-id 1
[Quidway- vsi-newvsi-ldp] traffic-statistics peer 100.22.33.20 negotiation-vc-id 2 enable
