display igmp group ssm-mapping

命令功能

display igmp group ssm-mapping命令用来查看通过ssm-mapping（IGMP视图）命令配置了映射规则的的组播组信息。

命令格式

display igmp [ vpn-instance vpn-instance-name | all-instance ] group [ group-address | interface interface-type interface-number ] * ssm-mapping [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

vpn-instance vpn-instance-name：指定VPN实例，vpn-instance-name表示VPN实例的名称。字符串形式，长度范围是1～31。

all-instance：指定所有实例。

group-address：表示组播组地址，用于指定一个组播组，显示该组播组的SSM映射信息。点分十进制形式。

interface interface-type interface-number：显示指定接口上有SSM映射关系的组播组信息。interface-type interface-number表示接口类型和编号。

verbose：表示有SSM映射关系的组播组的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display igmp group ssm-mapping命令时：

  • 如果不指定vpn-instance或all-instance参数，则只显示公网实例的相关信息。
  • 若指定了interface interface-type interface-number参数，则仅显示该接口下有SSM映射关系的组信息。
  • 若指定了group-address参数，则仅显示该组的SSM映射信息。
  • 若指定了verbose参数，则显示组播组运行SSM映射的各种状态参数。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

项目

描述

IGMP SSM mapping interface group report information of VPN-Instance

IGMP SSM-Mapping信息所属的VPN实例。

GigabitEthernet1/0/1(133.133.4.2)

接口编号（IP地址）。

Limited entry of this VPN-Instance

通过igmp limit命令限制的当前实例的所有接口上IGMP表项的最大个数。

Limited entries on this interface

通过igmp limit命令限制的当前接口上IGMP表项的最大个数。

Total 1 IGMP SSM-Mapping Group reported

配置了SSM Mapping的IGMP组播组被报告的个数。

Group

组地址。

Last reporter

最后一个报告此信息的主机地址。

Uptime

收到主机的report的时间。

Expires

组超时时间。

Last-member-query-counter

LMQC，最后组成员查询报文的发送次数。

Last-member-query-timer-expiry

LMQT，最后组成员查询报文的超时时间。

Group mode

组的模式（exclude还是include）。

Version1-host-present-timer-expiry

IGMP Version1主机的超时时间。

Version2-host-present-timer-expiry

IGMP Version2主机的超时时间。
