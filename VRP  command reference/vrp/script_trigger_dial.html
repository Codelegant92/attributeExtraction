script trigger dial

命令功能

script trigger dial命令用来指定DCC拨号时使用的Modem脚本。

undo script trigger dial命令用来取消这种特性。

系统缺省不指定该脚本。

命令格式

script trigger dial script-name

undo script trigger dial

参数说明

script-name：脚本集中的脚本名称，字符串形式，长度为1～50。

使用指南

如果配置了该项目，当DCC拨号时，则执行指定脚本。

使用实例

# 指定DCC拨号时使用“example”。
