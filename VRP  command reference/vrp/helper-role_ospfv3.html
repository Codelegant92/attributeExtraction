helper-role (OSPFv3)

命令功能

执行helper-role命令，配置路由器支持Helper模式。

命令格式

helper-role [ ip-prefix ip-prefix-name | acl-number acl-number ] [ max-grace-period period] [ planned-only ] [ lsa-checking-ignore ]

undo helper-role [ ip-prefix | acl-number ] [ max-grace-period ] [ planned-only ] [ lsa-checking-ignore ]

参数说明

ip-prefix ip-prefix-name：指定地址前缀列表名称。字符串形式，长度范围是1～169。

acl-number acl-number：指定基本访问控制列表的编号。整数形式，取值范围是2000～2999。

max-grace-period period：指定允许最大重启周期。整数形式，取值范围是1～1800，单位是秒。缺省值是120秒。

planned-only：指定路由器只支持Planned-GR。缺省情况下，路由器支持Planned-GR和Unplanned-GR。

lsa-checking-ignore：指定不执行LSA严格检查。缺省情况下，路由器对收到的各类LSA都进行严格检查。

使用指南

路由器在GR过程中禁止进入Helper模式。

使用实例

# 配置路由器支持Helper模式。

<Quidway> system-view
[Quidway] ospfv3 1
[Quidway-ospfv3-1] helper-role acl-number 2001 max-grace-period 250 planned-only lsa-checking-ignore
