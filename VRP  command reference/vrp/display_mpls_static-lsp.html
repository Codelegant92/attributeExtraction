display mpls static-lsp

命令功能

display mpls static-lsp命令用来查看静态LSP信息。

命令格式

display mpls static-lsp [ lsp-name ] [ { include | exclude } ip-address mask-length ] [ verbose ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

lsp-name：LSP的名称。字符串形式，区分大小写，不支持空格，长度范围是1～19。

include：输出包含指定FEC的LSP信息。

exclude：输出不包含指定FEC的LSP信息。

ip-address：IPv4目的地址。

mask-length：IPv4掩码长度。整数形式，取值范围是0～32。

verbose：查看详细信息。

| ：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

静态LSP只有本地的概念，因此要查看静态LSP能否正常工作，必须将本命令与ping lsp命令配合使用。

使用实例

# 查看静态LSP的简要信息

<Quidway> display mpls static-lsp
TOTAL          : 1       STATIC LSP(S)
UP             : 1       STATIC LSP(S)
DOWN           : 0       STATIC LSP(S)
Name       FEC               I/O Label  I/O If                 Stat
lsp1       3.3.3.9/32        NULL/100   -/GE1/0/0              Up
表1 display mpls static-lsp命令输出信息描述

项目

描述

TOTAL

静态LSP的总数

UP

处于UP状态的静态LSP数量

DOWN

处于DOWN状态的静态LSP数量

Name

LSP名

FEC

转发等价类，在本显示例中是此LSP的目的地址及掩码长度

I/O Label

入标签和出标签

I/O If

入接口和出接口

Stat

此LSP当前的状态
