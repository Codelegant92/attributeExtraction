send-router-alert（IGMP视图）

命令功能

send-router-alert命令用来在全局性上指定该路由器发送的IGMP报文的报文头中包含Router-Alert预警参数。

undo send-router-alert命令用来在全局性上指定该路由器发送的IGMP报文的报文头中不包含Router-Alert预警参数。

缺省情况下，该路由器发送的IGMP报文头中包含Router-Alert预警参数。

命令格式

send-router-alert

undo send-router-alert

参数说明

无

视图

公网实例IGMP视图、VPN实例IGMP视图

缺省级别

2：配置级

使用指南

该命令和接口视图下的igmp send-router-alert 命令功能完全相同，差别只是作用范围不同。IGMP视图下的配置具有全局性，对所有接口有效；接口视图下的配置只对当前接口有效。系统优先采用接口视图下的配置。当接口视图下没有进行配置，才采用IGMP视图下的配置。

使用实例

# 在IGMP视图下，配置路由器发送的IGMP报文的报文头中不包含Router-Alert预警参数。

<Quidway> system-view
[Quidway] igmp
[Quidway-igmp] undo send-router-alert
