ssh server authentication-retries

命令功能

ssh server authentication-retries命令用来设置SSH连接验证重试次数，在下次登录时生效。

undo ssh server authentication-retries命令用来恢复SSH连接验证重试次数的缺省值。

缺省情况下，SSH连接的验证重试次数为3。

命令格式

ssh server authentication-retries times

undo ssh server authentication-retries

参数说明

times：指定SSH连接的验证重试次数，取值范围是1～5。
