nqa-server icmp-server

命令功能

nqa-server icmp-server命令用来配置NQA测试的ICMP服务器。

undo step命令用来取消当前配置的ICMP服务器。

缺省情况下，不配置ICMP服务器。

命令格式

nqa-server icmp-server [ vpn-instance vpn-instance-name ] ip-address

undo nqa-server icmp-server { [ vpn-instance vpn-instance-name ] ip-address | all }

参数说明

vpn-instance vpn-instance-name：VPN实例的名称。字符串形式，长度范围是1～31。

ip-address：ICMP服务器的地址。点分十进制格式。

all：取消所有的ICMP服务器。

使用指南

只有配置测试类型为ICMP Jitter测试并且已经使能了接口板发包后，才需要配置ICMP服务器。

使用实例

# 配置NQA测试的ICMP服务器。

<Quidway> system-view
[Quidway] nqa-server icmp-server 10.1.1.1
