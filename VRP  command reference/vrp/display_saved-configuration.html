display saved-configuration

命令功能

display saved-configuration命令用来显示路由器下次上电启动时所用的配置文件，即，通过startup saved-configuration命令指定的配置文件。

display saved-configuration last命令用来显示上次保存的系统配置信息，即，本次启动时使用的配置文件。

display saved-configuration time命令用来显示上次保存的系统配置的时间。

命令格式

display saved-configuration [ last | time ]

参数说明

last：显示上次保存的系统配置信息。

time：最近的一次手工或者系统自动保存配置的时间。

使用指南

如果路由器上电之后工作不正常，可以执行display saved-configuration命令查看路由器的启动配置。

使用实例
