display mld interface

命令功能

display mld interface命令用来查看接口上MLD的参数信息。

命令格式

display mld interface [ interface-type interface-number | up | down ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-type interface-number：指定接口类型和编号。

up：MLD接口的协议状态是Up，MLD处于Active状态。

down：MLD接口的协议状态是Down，MLD处于Inactive状态。

verbose：显示所有运行MLD的接口或指定接口的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

使用display mld interface命令时：

  • 指定interface-type interface-number参数，则仅显示该接口上MLD的参数信息，如果该接口未使能MLD，则出现提示信息。

  • 如果指定up，则仅显示所有MLD协议状态为Active的MLD接口参数。

  • 如果指定down，则仅显示所有MLD协议状态为Inactive的MLD接口参数。

  • 当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

表1 display mld interface命令输出信息描述

项目

描述

GigabitEthernet1/0/0(FE80::2E0:B4FF:FE35:FF01)

接口的链路本地地址。

Current MLD version is 2

当前接口上配置的MLD版本号为2。

MLD state

MLD接口状态，up或down。

MLD group policy

MLD组策略的ACL编号。

Value of query interval for MLD (negotiated)

经协商后系统实际采用的MLD查询消息发送间隔，单位是秒。

Value of query interval for MLD (configured):

用户配置的MLD查询消息的发送间隔，单位是秒。

Value of other querier timeout for MLD

MLD查询器超时间隔，单位是秒。

Value of maximum query response time for MLD

MLD查询消息的最大响应时间，单位是秒。

Value of last listener query time

最后侦听者查询时间，单位是秒。最后侦听者查询时间=最后侦听者查询时间间隔×健壮系数。

Value of last listener query interval

查询器发送最后侦听者查询消息（即指定组查询消息）的时间间隔，单位是秒。

Value of startup query interval

查询器启动时发送查询消息的时间间隔，单位是秒。

Value of startup query count

启动时查询消息的发送次数。

General query timer expiry

普遍查询定时器的超时间隔。

Querier for MLD

MLD查询器的链路本地地址。

MLD activity

MLD的活动统计信息（加入或离开）。

Robustness

查询器的健壮系数。

Require-router-alert

是否丢弃不包含Router-Alert选项的MLD报文。disabled表示可接收处理不包含Router-Alert选项的MLD报文；enabled表示忽略不包含Router-Alert选项的MLD报文。

Send-router-alert

发送的MLD报文是否包含Router-Alert选项。

Prompt-leave

是否使能快速离开机制。

SSM-Mapping

是否使能了SSM映射。

Startup-query-timer-expiry
启动时该接口作为查询器的查询间隔定时器状态：
  • off：查询消息发送完毕。
  • on：查询消息未发送完毕。

Other-querier-present-timer-expiry
其他查询器的存在时间定时器状态：
  • off：接口认为自己是查询器，不存在其他查询器。
  • on：接口不是查询器，存在其他查询器。
