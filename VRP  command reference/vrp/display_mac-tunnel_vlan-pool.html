display mac-tunnel vlan-pool

命令功能

display mac-tunnel vlan-pool命令用来查看MAC Tunnel骨干VLAN资源池的信息。

命令格式

display mac-tunnel vlan-pool

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

查看结果会显示骨干VLAN资源池中的具体VLAN ID和骨干VLAN的总数量。

当使用命令display vlan查看VLAN信息时，如果某VLAN已经加入MAC Tunnel骨干VLAN资源池，则显示信息中会显示该VLAN是骨干VLAN（Backbone VLAN）。

使用实例

# 查看设备上骨干VLAN资源池的信息。

<Quidway> display mac-tunnel vlan-pool
 Mac-tunnel VLAN:
 ------------------------------------------------------------------------------
 101  102  104 
 ------------------------------------------------------------------------------
 Total matching items displayed = 3
