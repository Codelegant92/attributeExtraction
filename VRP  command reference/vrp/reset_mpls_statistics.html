reset mpls statistics

命令功能

reset mpls statistics命令用来清除MPLS统计信息。

命令格式

reset mpls statistics interface { interface-type interface-number | all }

reset mpls statistics lsp { lsp-name | all }

参数说明

interface-type interface-number：接口类型和接口编号。

lsp-name：LSP的名称，字符串形式，区分大小写，不支持空格，长度范围是1～19。

all：所有接口或所有LSP。

视图

用户视图

缺省级别

2：配置级

使用指南

可以选择清除所有或单个接口的统计信息，以及所有LSP或单条LSP的统计信息。

使用实例

# 清除名为“Beijing”的LSP统计信息。

<Quidway> reset mpls statistics lsp beijing
