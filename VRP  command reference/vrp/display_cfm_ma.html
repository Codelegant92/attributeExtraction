display cfm ma

命令功能

display cfm ma命令用来查看MA的详细信息，包括：

  • 所在MD的名称

  • MD的级别

  • MD名称输入类型

  • MA的名称

  • CC检测时间间隔

  • 报文优先级

  • 关联的VLAN

  • 关联的VSI

  • MEP的数量

  • RMEP的数量

  • MA继承MD内的MIP生成规则

  • Sender ID TLV类型

命令格式

display cfm ma [ md md-name [ ma ma-name ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

md md-name：显示指定MD内的MA的详细信息。md-name表示MD的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。如不输入本参数，则显示所有MD内所有MA的详细信息。

ma ma-name：显示指定MA的详细信息。ma-name 表示MA的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。如输入参数md-name但不输入本参数，则显示参数md-name指定MD内的所有MA的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

include：只输出包含匹配字符串的行。

exclude：只输出不包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当配置完MA后或CFM功能出现故障时，可使用本命令验证或了解MA的配置和MA内MEP、RMEP的数量。

使用实例

# 查看指定MD下指定MA的详细信息，该MD的名称为md1，MA的名称为ma1。

<Quidway> display cfm ma md md1 ma ma1
The total number of MAs is 1
  MD Name            : md1
  MD Name Format     : md-name
  Level              : 3
  MIP Create-type    : none
  SenderID TLV-type  : SendIdDefer
  MA Name            : ma1
  Interval           : 10ms
  Priority           : 4
  Vlan ID            : 7
  VSI Name           : --
  MEP Number         : 2
  RMEP Number        : 3

项目

描述

MD Name

MD的名称。

MD Name Format
MD名称输入类型：
  • no-md-name
  • dnsname-and-mdname
  • mac-addrss
  • md-name

Level

MD的级别。取值范围是0～7。

MIP Create-type
MIP生成规则：
  • default
  • explicit
  • none

SenderID TLV-type
Sender ID TLV类型，用于CCM报文的填充。显示值可能是：
  • SendIdNone
  • SendIdChassis
  • SendIdManage
  • SendIdChassisManage
  • SendIdDefer

MA Name

MA的名称。

Interval

MA内CC检测的时间间隔。

Priority

MA内CFM报文的802.1p优先级。取值范围是0～7，值越大优先级越高，缺省值是6。

Vlan ID

MA关联的VLAN的ID。

VSI Name

MA关联的VSI名字。

MEP-Number

MA内MEP的数量。

RMEP-Number

MA内RMEP的数量。
