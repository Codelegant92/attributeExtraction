atm cell transfer

命令功能

atm cell transfer命令用来配置在ATM信元透b传输模式。

undo atm cell transfer用来配置ATM AAL5 SDU帧透b传输模式。

缺省情况下，ATM主接口、ATM子接口工作在ATM AAL5 SDU帧透b传输模式下。

命令格式

atm cell transfer

undo atm cell transfer

参数说明

无

视图

ATM接口视图、ATM子接口视图

缺省级别

2：配置级

使用指南

本命令用来配置ATM主接口或ATM子接口的ATM信元透传模式。

对于NE40，NE80，S8016产品，一旦配置某ATM接口工作在端口透b传输模式下，就不能在该ATM接口及其任意子接口下配置IPoA转发、或其它模式的信元透b传输，除非先取消端口透b传输。

对于NE40，NE80，S8016产品，取消端口透b传输ATM信元的步骤应遵循先删除L2VPN连接，后取消端口透b传输ATM信元模式的顺序。

使用实例

# 配置ATM主接口工作在ATM信元透b传输模式下。

<Quidway> system-view
[Quidway] interface atm 1/0/0
[Quidway-atm1/0/0] atm cell transfer

# 配置ATM子接口工作在ATM AAL5 SDU帧透b传输模式下。

<Quidway> system-view
[Quidway] interface atm 1/0/0.1
[Quidway-atm1/0/0.1] undo atm cell transfer
