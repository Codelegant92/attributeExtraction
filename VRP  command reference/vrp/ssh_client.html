ssh client

命令功能

ssh client server-name assign rsa-key key-name命令用来给SSH服务器分配RSA公钥的，key-name必须是SSH客户端已经配置好的SSH服务器的RSA公钥。

undo ssh client server-name assign rsa-key命令用来取消SSH服务器与RSA公钥的绑定关系。

命令格式

ssh client server-name assign rsa-key key-name

undo ssh client server-name assign rsa-key

参数说明

server-name：SSH服务器的主机名或IP地址，字符串形式，长度范围为1～46。

key-name：RSA公钥名，字符串形式，长度范围为1～30。

使用指南

如果SSH客户端是第一次连接SSH服务器且SSH客户端没有使能首次认证，则必须为SSH服务器分配对端RSA公钥。只有这样，SSH客户端对SSH服务器的RSA公钥验证才会通过，SSH客户端第一次连接服务器才会成功。

使用实例

# 为SSH服务器“sshserver1”分配RSA公钥。

<Quidway> system-view
[Quidway] ssh client sshserver1 assign rsa-key 222
说明：

SSH客户端的RSA公钥222，应该是根据SSH服务器sshserver1的本地RSA公钥配置的peer-public-key,否则SSH客户端对SSH服务器的RSA公钥验证失败。
