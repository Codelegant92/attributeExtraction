display virtual-mac

命令功能

display virtual-mac命令用来查看设备的虚拟MAC地址。

命令格式

display virtual-mac

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

因为在每台VRP设备上只能够创建一个虚拟MAC地址，所以查看设备的虚拟MAC地址的结果，只有一个虚拟MAC地址。

当没有创建虚拟MAC地址时，系统显示为0000-0000-0000。

使用实例

# 查看设备虚拟MAC地址配置情况。

<Quidway> display virtual-mac
--------------------
The virtual mac is
--------------------
 0001-0002-0003
