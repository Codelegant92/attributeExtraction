rrpp snooping vsi

命令功能

rrpp snooping vsi命令用于在已使能RRPP Snooping的子接口或VLANIF接口上关联本设备其他的VSI，即与当前子接口或VLANIF接口无绑定关系的VSI。

undo rrpp snooping vsi命令用于删除与RRPP Snooping关联的VSI。

命令格式

rrpp snooping vsi vsi-name

undo rrpp snooping vsi vsi-name

参数说明

vsi-name：VSI实例的名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。一个子接口或VLANIF接口最多可以关联4K个VSI。

视图

Ethernet子接口视图、GE子接口视图、Eth-Trunk子接口视图、VLANIF接口视图

缺省级别

2：配置级

使用指南

执行本命令前，必须已经执行了rrpp snooping enable命令使能了RRPP Snooping功能。

使用本命令配置的接口上关联的VSI是本设备上其他子接口或VLANIF接口绑定的VSI，而不是使能了RRPP Snooping的接口绑定的VSI。在接口上绑定VSI通过接口视图下的l2 binding vsi vsi-name命令配置。

执行本命令后，接口会将RRPP环状态的变化通知与之关联的VSI，使VSI可以同步更新MAC地址表。

使用实例

# 配置与RRPP Snooping关联的VSI，VSI名为abc。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0.1
[Quidway-GigabitEthernet1/0/0.1] rrpp snooping vsi abc
