vlan-mapping modulo

命令功能

vlan-mapping modulo命令用来将所有VLAN映射到指定的生成树实例上。

undo vlan-mapping modulo命令用来将所有VLAN和生成树实例的映射关系恢复为缺省状态。

缺省情况下，所有VLAN均映射到CIST，即实例0上。

命令格式

vlan-mapping modulo modulo

undo vlan-mapping modulo

参数说明

modulo：模值，取值范围是1～48。

视图

MST域视图、MST进程域视图

缺省级别

2：配置级

使用指南

MSTP采用VLAN映射表来描述VLAN与生成树实例的映射关系。使用vlan-mapping modulo命令可以快速配置VLAN映射表，使每个VLAN按照配置被映射到不同的生成树实例上。

vlan-mapping modulo命令将VLAN映射到编号为(VLAN ID-1)％modulo+1的生成树实例上。其中(VLAN ID-1)％modulo为对(VLAN ID-1)进行求模运算，如模值为16，则VLAN1映射到MSTI1、VLAN2映射到MSTI2……VLAN16映射到MSTI16、VLAN17映射到MSTI1，依次类推。

不能将同一个VLAN映射到多个不同的生成树实例上。如果将一个已经和生成树实例建立映射关系的VLAN又映射到另一个生成树实例上，原来的映射关系将被取消。

使用实例

# 将所有VLAN按照模16映射到生成树实例上。

<Quidway> system-view
[Quidway] stp region-configuration
[Quidway-mst-region] vlan-mapping modulo 16

# 在MSPT进程10中将所有VLAN按照模16映射到生成树实例上。

<Quidway> system-view
[Quidway] stp process 10
[Quidway-mst-process-10] stp region-configuration
[Quidway-mst-process-10-region] revision-level 5
[Quidway-mst-process-10-region] vlan-mapping modulo 16
