mac-address blackhole

命令功能

mac-address blackhole命令用来配置黑洞MAC地址表项。

undo mac-address blackhole命令用来删除黑洞MAC地址表项。

缺省情况下，系统没有配置任何黑洞MAC地址表项。

命令格式

mac-address blackhole mac-address { vlan vlan-id [ ce-vlan ce-vlan ] | vsi vsi-name [ pe-vid pe-vid [ ce-vid ce-vid ] ] }

undo mac-address blackhole [ vlan vlan-id | vsi vsi-name ]

参数说明

blackhole：黑洞表项，不被老化。当收到指定地址的帧后，直接丢弃。配置并保存后，系统复位或接口板热插拔表项不丢失。

mac-address：单播MAC地址，格式为H-H-H。其中H为4位的十六进制数，可以输入1～4位，如00e0、fc01。当输入不足4位时，表示前面的几位为0，如：输入e0，等同于00e0。MAC地址不可设置为FFFF-FFFF-FFFF。

vlan vlan-id：VLAN ID。十进制整数形式，取值范围是1～4094。

ce-vlan ce-vlan：配置VLAN Mapping时，映射后的VLAN ID。十进制整数形式，取值范围是1～4094。

vsi vsi-name：指定的VSI实例的名称。字符串形式，不支持空格，区分大小写，取值范围是1～31。

pe-vid pe-vid：用户外层VLAN的Tag值，十进制整数形式，取值范围是1～4094。

ce-vid ce-vid：用户内层VLAN的Tag值，十进制整数形式，取值范围是1～4094。

使用指南

与配置静态MAC表项不同，配置黑洞MAC表项时，不需要指定出接口。

使用实例

# 在VLAN10上添加一个MAC地址为0011-2233-4455的黑洞表项。当收到目的MAC为0011-2233-4455且属于VLAN10的帧时，直接丢弃。

<Quidway> system-view
[Quidway] vlan 10
[Quidway-vlan10] quit
[Quidway] mac-address blackhole 0011-2233-4455 vlan 10
