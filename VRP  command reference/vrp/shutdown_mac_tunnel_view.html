shutdown（MAC Tunnel视图）

命令功能

shutdown命令用来配置MAC Tunnel的管理状态为Down。

undo shutdown命令用来激活MAC Tunnel的管理状态为Up。

缺省情况下，隧道管理状态为Down。

命令格式

shutdown

undo shutdown

参数说明

无

视图

MAC Tunnel视图

缺省级别

2：配置级

使用指南

如果MAC Tunnel的管理状态为Down，则该隧道不能收发报文。如果MAC Tunnel的管理状态为Up，则隧道根据当前的实际情况来处理报文。

当使用undo shutdown命令配置MAC Tunnel的管理状态为Up时，要求该隧道的远端设备的虚拟MAC地址、骨干VLAN、接口三个属性都已经配置完整。如果任何属性没有配置完整，则MAC Tunnel的管理状态不能被配置为Up。

使用实例

# 配置隧道的管理状态为Down。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] shutdown

# 配置当前隧道的状态，配成管理状态Up。

<Quidway> system-view
[Quidway] mac-tunnel tunnel-name mytunnel
[Quidway-mac-tunnel-mytunnel] undo shutdown
