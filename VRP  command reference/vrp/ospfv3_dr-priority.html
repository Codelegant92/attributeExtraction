ospfv3 dr-priority

命令功能

ospfv3 dr-priority命令用来设置接口所在实例在选举DR/BDR时的优先级。

undo ospfv3 dr-priority命令用来将接口所在实例恢复其缺省值。

命令格式

ospfv3 dr-priority priority [ instance instance-id ]

undo ospfv3 dr-priority [ instance instance-id ]

参数说明

priority：接口在选举DR/BDR时的优先级。整数形式，取值范围是0～255，缺省值是1。

instance instance-id：指定接口所属的实例ID。整数形式，取值范围是0～255，缺省值是0。

使用指南

接口的优先级决定了该接口在选举DR/BDR时所具有的资格，优先级高的在选举时被首先考虑。

使用实例

# 设置接口GigabitEthernet1/0/0实例1在选举DR时的优先级为8。

<Quidway> system-view
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ospfv3 dr-priority 8 instance 1
