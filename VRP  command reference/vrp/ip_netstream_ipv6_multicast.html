ip netstream ipv6 multicast

命令功能

ip netstream ipv6 multicast { inbound | outbound } [interface { interface-name | interface-type interface-number } ]命令用来使能接口的IPv4或IPv6的组播出入统计功能。

undo ip netstream ipv6 multicast { inbound | outbound } [ interface { interface-name | interface-type interface-number } ]命令用来删除接口的IPv4或IPv6的组播出入统计功能使能。

缺省情况下，不使能接口的IPv6出入统计功能。

命令格式

ip netstream ipv6 multicast { inbound | outbound } [ interface { interface-name | interface-type interface-number } ]

undo ip netstream ipv6 multicast { inbound | outbound } [ interface { interface-name | interface-type interface-number } ]

参数说明

inbound：入统计。

outbound：出统计。

interface interface-name：指定VLAN下的二层物理接口名。

interface interface-type：指定VLAN下的二层物理接口类型。

interface interface-number：指定VLAN下的二层物理接口号。

使用指南

interface-type和interface-number参数，仅在VLAN-IF接口下可配置。
