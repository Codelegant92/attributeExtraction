admin-vrrp vrid

命令功能

admin-vrrp vrid命令用来指定管理VRRP备份组。

undo admin-vrrp vrid命令用来删除管理VRRP备份组。

缺省情况下，没有配置管理VRRP备份组。

命令格式

admin-vrrp vrid virtual-router-id

undo admin-vrrp vrid virtual-router-id

参数说明

vrid virtual-router-id：被指定的VRRP备份组号。十进制整数形式，取值范围是1～255。

视图

Ethernet接口视图、Ethernet子接口视图、GE接口视图、GE子接口视图、Eth-Trunk接口视图、Eth-Trunk子接口视图、VE接口视图和VE子接口视图

缺省级别

2：配置级

使用指南

配置了管理VRRP备份组后，设备上的其他管理VRRP备份组、业务PW、业务接口才能和管理VRRP备份组绑定。管理VRRP备份组不可以和其他管理VRRP备份组绑定。

使用实例

# 在接口GE1/0/0上，创建VRRP备份组1，该备份组为管理VRRP。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ip address 10.1.1.1 24
[Quidway-GigabitEthernet1/0/0] vrrp vrid 1 virtual-ip 10.1.1.111
[Quidway-GigabitEthernet1/0/0] admin-vrrp vrid 1
[Quidway-GigabitEthernet1/0/0] quit
