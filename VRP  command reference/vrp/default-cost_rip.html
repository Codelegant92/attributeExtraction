default-cost (RIP)

命令功能

default cost命令用来设置引入路由的缺省路由权值。

undo default cost命令用来将其恢复为缺省值0。

命令格式

default-cost cost

undo default-cost

参数说明

cost：所要设定的缺省路由权值。整数形式，取值范围是0～15。

视图

RIP视图

缺省级别

2：配置级

使用指南

该命令用于RIP从其它路由协议引入路由或在引入路由的命令中没有指定路由度量值的情况。

使用实例

# 设置引入路由的缺省路由权为2。

<Quidway> system-view
[Quidway] rip 100
[Quidway-rip-100] default-cost 2

# 恢复缺省路由权的缺省值。

[Quidway-rip-100] undo default-cost
