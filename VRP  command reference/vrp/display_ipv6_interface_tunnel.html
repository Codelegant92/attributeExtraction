display ipv6 interface tunnel

命令功能

display ipv6 interface tunnel命令用来查看Tunnel接口的IPv6属性。

命令格式

display ipv6 interface tunnel interface-number [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

interface-number：指定接口编号。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

Tunnel3/0/0 current state:

  • UP表示打开

  • DOWN表示关闭

对于IPv6 over IPv4隧道，正常工作时此处值应该为UP

  • UP表示打开

  • DOWN表示关闭
