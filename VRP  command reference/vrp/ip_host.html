ip host

命令功能

ip host命令用来设置主机名和对应IP地址。

undo ip host命令用来取消主机名和对应IP地址。

缺省情况下，无主机名与IP地址的对应关系。

命令格式

ip host host-name ip-address

undo ip host host-name [ ip-address ]

参数说明

host-name：指定主机名。字符串形式，长度是1～20。

ip-address：指定与主机名对应的IP地址。点分十进制形式。

使用指南

使用ip host命令最多可以配置50个静态DNS表项。每个主机名只能对应一个IP地址，当对同一主机名进行多次配置时，后配置的IP地址生效。

使用实例

# 设置主机名huawei对应的IP地址为10.110.0.1。

<Quidway> system-view
[Quidway] ip host huawei 10.110.0.1
