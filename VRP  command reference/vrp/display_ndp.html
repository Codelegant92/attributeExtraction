display ndp

命令功能

display ndp命令用来查看本设备的NDP信息。

命令格式

display ndp [ interface { interface-type interface-number [ to interface-type interface-number ] } &<1-10> ]

参数说明

interface interface-type interface-number：查看指定接口的NDP信息。interface-type表示接口类型。interface-number表示接口编号。

视图

所有视图

缺省级别

1：监控级

使用指南

如果不指定可选参数，本命令显示设备的全局NDP和所有接口的NDP信息。

在未使能全局NDP时，如果不指定可选参数，执行本命令将不会显示接口的NDP信息，只显示设备的全局NDP信息。

使用实例

# 显示全局NDP和所有接口的NDP信息。

<Quidway> display ndp
Info: Neighbor discovery protocol is enabled.
Neighbor Discovery Protocol Ver: 1, Hello Timer: 60(s), Aging Timer: 180(s)
Interface: GigabitEthernet1/0/1
    Status: Disabled, Packets Sent: 0, Packets Received: 0, Packets Error: 0
Interface: GigabitEthernet1/0/2
    Status: Enabled, Packets Sent: 114, Packets Received: 108, Packets Error: 0
    Neighbor 1:  Aging Time: 174(s)
       MAC Address : 0018-8203-39d8
       Port Name   : GigabitEthernet1/0/1
       Software Version: NE40E Version V300R006C01B100
       Device Name : NE40E
       Port Duplex : FULL
       Product Ver : NE40E

该显示信息主要内容解释见表1。

表1 display ndp命令输出信息描述

项目

描述

Neighbor discovery protocol is status.
本设备全局NDP功能处于status状态。status包括两个状态：

  • disabled表示禁止全局NDP功能。

  • enabled表示使能全局NDP功能。

Neighbor Discovery Protocol Ver

本设备支持的NDP版本。目前，设备都支持版本1。

Hello Timer

NDP报文的发送时间间隔。以秒为单位。

Aging Timer

NDP信息的老化时间。以秒为单位。

Interface

指定的接口号。

Status
接口的NDP状态。

  • Disabled：表示本接口禁止了NDP功能。

  • Enabled：表示本接口使能了NDP功能。

Packets Send:

接口发送的NDP报文数。

Packets Received:

接口接收到的NDP报文数。

Packets Error:

接口接收到的错误的NDP报文数。

Aging Time

接口连接的邻居节点的NDP信息老化时间。

MAC Address

邻居节点的MAC地址。

Port Name

邻居节点的与本接口相连的接口名称。

Software Version

邻居节点的系统软件的版本信息。

Device Name

邻居节点的主机名。

Port Duplex
邻居节点与本接口相连的接口的双工模式。

  • FULL：表示全双工。

  • HALF：表示半双工。

Product Ver

邻居节点的型号。
