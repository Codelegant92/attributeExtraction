display mpls lsp statistics

命令功能

display mpls lsp statistics命令用来查看当前处于Up状态的LSP数目，并显示在Ingress节点、Transit节点和Egress节点的当前激活的LSP数目。

命令格式

display mpls lsp statistics [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

| ：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当需要统计当前处于UP状态的LSP和CR-LSP的数目时，可以执行此命令。

如果需要进一步查看这些LSP的相关信息，可以执行display mpls lsp命令。

<Quidway> display mpls lsp statistics
