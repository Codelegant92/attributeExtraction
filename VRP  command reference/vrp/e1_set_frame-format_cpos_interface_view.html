e1 set frame-format（CPOS接口视图）

命令功能

e1 set frame-format命令用来设置E1通道的帧格式。

undo e1 set frame-format命令用来恢复缺省设置。

缺省情况下，E1通道的帧格式为no-CRC4。

命令格式

e1 e1-number set frame-format { crc4 | no-crc4 }

undo e1 e1-number set frame-format

参数说明

e1-number：CPOS的E1通道号。整数形式，取值范围是1～63。

crc4：帧格式为CRC4。

no-crc4：帧格式为no-CRC4。

缺省级别

2：配置级

使用指南

配置时需与对端保持一致。

使用实例

# 设置E1通道1使用带CRC4检验的帧格式。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] e1 1 set frame-format crc4
