ripng metricin

命令功能

ripng metricin命令用来给RIPng接口设置接收附加权值。

undo ripng metricin命令用来恢复接收附加权值的缺省值。

缺省情况下，接收附加权值为0。

命令格式

ripng metricin value

undo ripng metricin

参数说明

value：附加权值。整数形式，取值范围是0～15。缺省值是0。

使用指南

当收到一条合法的RIPng路由，在其加入到路由表之前，接口接收权值会附加到该路由上，再加入路由表中，因此路由表中的度量值发生变化。即是说，增加一个接口的接收权值，该接口收到的RIPng路由权值也会相应增加。

使用实例

# 设置RIPng接收附加权值为12。

<Quidway> system-view
[Quidway] interface GigabitEthernet 1/0/0
[Quidway-GigabitEthernet1/0/0] ripng metricin 12
