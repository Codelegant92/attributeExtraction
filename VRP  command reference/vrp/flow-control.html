flow-control

命令功能

flow-control命令用来对流控方式进行配置。

undo flow-control命令用来恢复流控方式的缺省值。

缺省情况下，流控方式为none，即不进行流控。

命令格式

flow-control { hardware | none | software }

undo flow-control

参数说明

hardware：硬件流控，只对AUX口有效。

none：无流控。

software：软件流控。

缺省级别

2：配置级

使用指南

只有相应的串口工作在异步交互模式下，配置有效。

在输出过程中，按<Ctrl+S>停止屏幕输出；按<Ctrl+Q>继续输出。

使用实例

# 在用户界面视图下，配置软件流控：

<Quidway> system-view
[Quidway] user-interface console 0
[Quidway-ui-console0] flow-control software
