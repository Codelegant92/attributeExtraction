display dialer

命令功能

display dialer interface命令用来显示DCC接口信息。

缺省情况下，显示全部DCC接口的信息。

命令格式

display dialer [ interface interface-type interface-number ]

参数说明

interface-type：接口类型。

interface-number：接口序号。

视图

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 显示DCC接口Dialer1的信息。

<Quidway> display dialer interface dialer1
Dial Interface: Dialer1
   Dialer route:
     NextHop_address           Dialer_Number
     7.1.1.2                      002
   Dialer timers(secs):
     Auto-dial: 300      Compete: 20       Enable: 20
     Idle: 120           Wait-for-Carrier: 60
表1 display dialer命令输出信息描述

项目

描述

NextHop_address

dialer route对应的下一条协议地址。

Dialer_Numbers

dialer route对应的号码。

Auto-dial

由dialer timer autodial命令设定的自动拨号时间的间隔。

Compete

由dialer timer compete命令设定的时间。

Enable

由dialer timer enable命令设定的时间。

Idle

由dialer timer idle命令设定的时间。

Wait-for-Carrier

由dialer timer wait-carrier命令设定的时间。
