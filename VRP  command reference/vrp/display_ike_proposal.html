display ike proposal

命令功能

display ike proposal命令用来查看每个IKE提议配置的参数。

命令格式

display ike proposal

参数说明

无

视图

所有视图

缺省级别

1：监控级

使用指南

display ike proposal命令可以观察到的信息包括：

  • IKE提议的优先级

  • IKE提议使用的认证方法

  • IKE提议使用的认证算法

  • IKE提议使用的加密算法

  • Diffie-Hellman（DH）组标识

  • IKE提议使用的ISAKMP SA存活时间

IKE提议按照优先级的先后顺序显示。

使用实例

# 下面是一个配置了两条IKE提议后显示的例子。

<Quidway> display ike proposal
priority authentication authentication encryption Diffie-Hellman duration
              method       algorithm    algorithm     group       (seconds)
--------------------------------------------------------------------
 23       PRE_SHARED     SHA            AES_CBC    MODP_768       86400
 default  PRE_SHARED     SHA            DES_CBC    MODP_768       86400

主要字段

描述

Protection suite priority

IKE提议的优先级，可以是1～100之间的任意一个整数。priority值越大，优先级越低。

encryption algorithm

IKE提议使用的加密算法

authentication algorithm

IKE提议使用的认证算法

authentication method

IKE提议使用的认证方式

Diffie-Hellman group

Diffie-Hellman（DH）组标识

sa duration

IKE提议使用的ISAKMP SA存活时间

Default protection suite

缺省的IKE提议，缺省情况下或配置的IKE提议都不匹配的时候，采用此IKE提议，该提议优先级最低。
