filter-policy import (OSPFv3)

命令功能

filter-policy import命令用来对接收的路由信息进行过滤。

undo filter-policy import命令用来取消此设置。

缺省情况下，不对接收的路由信息过滤。

命令格式

filter-policy { acl6-number | ipv6-prefix ipv6-prefix-name } import

undo filter-policy import

参数说明

acl6-number：ACL6的序号。整数形式，取值范围是2000～2999。

ipv6-prefix ipv6-prefix-name：指定IPv6地址前缀列表名称。字符串形式，长度范围是1～169个字符。

缺省级别

2：配置级

使用指南

filter-policy import命令只对OSPFv3计算出来的路由进行过滤，没有通过过滤的路由将不被加入路由表中，从而不能指导转发。

使用实例

# 根据IPv6地址前缀列表abc对接收的路由信息进行过滤。

<Quidway> system-view
[Quidway] ospfv3 1
[Quidway-ospfv3-1] filter-policy ipv6-prefix abc import
