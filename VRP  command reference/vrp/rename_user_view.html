rename（用户视图）

命令功能

rename命令用来重命名文件或路径。

命令格式

rename source-filename destination-filename

参数说明

source-filename：源文件名或路径名，字符串形式，长度范围是1～64。

destination-filename：目标文件名或路径名，字符串形式，长度范围是1～64。

视图

用户视图

缺省级别

3：管理级

使用指南

若目标文件名与已经存在的目录名重名，或者目标文件名与已经存在的文件名重名，都将出现错误提示信息。

该命令不支持跨路径的文件重命名。

使用实例

# 重命名文件sample.txt为sample.bak。

<Quidway> rename sample.txt sample.bak
Rename flash:/sample.txt to flash:/sample.bak ?[Y/N]: y
Info:Rename file flash:/sample.txt to flash:/sample.bak ......Done.
