vlan（默认MD视图）

命令功能

vlan 命令用来创建默认MD内指定的VLAN。

undo vlan 命令用来删除默认MD内指定的VLAN。

命令格式

vlan { vlan-id1 [ to vlan-id2 ] }&<1-10>

undo vlan

参数说明

vlan-id1：起始VLAN ID。取值范围是1～4094。

vlan-id2：结束VLAN ID。取值范围是1～4094，该值必须大于vlan-id1。

视图

默认MD视图

缺省级别

2：配置级

使用指南

现如今，802.1ag协议分为draft7草案版本和standard2007标准版本，本命令只支持standard2007标准版本使用。

在默认MD视图下，执行本命令，指定VLAN内的所有接口将按照默认MD配置的MIP生成规则生成MIP节点。

说明：
指定的VLAN必须是没有关联MA。

使用实例

# 将VLAN100～200关联默认MD。

<Quidway> system-view
[Quidway] cfm default md
[Quidway-default-md] vlan 100 to 200
