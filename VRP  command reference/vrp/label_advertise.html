label advertise

命令功能

label advertise命令用来配置倒数第二跳弹出的标签分配方式。

undo label advertise命令用来恢复缺省设置。

缺省情况下，出节点向倒数第二跳分配隐式空标签（implicit-null）。

命令格式

label advertise { explicit-null | implicit-null | non-null }

undo label advertise

参数说明

explicit-null：支持PHP特性，出节点向倒数第二跳分配显式空标签。显式空标签的值为0。

implicit-null：支持PHP特性，出节点向倒数第二跳分配隐式空标签。隐式空标签的值为3。

non-null：不支持PHP特性，出节点向倒数第二跳正常分配标签。分配的标签值不小于16。

2：配置级

使用指南

选择哪种标签分配方式需要根据倒数第二跳节点对PHP的支持情况而定。

说明：

修改本节点的标签分配方式后：

  • 如果是MPLS TE隧道，且本节点是出节点，请在入节点用户视图下执行reset mpls te tunnel-interface tunnel tunnel-number，使配置生效；

  • 如果是LDP LSP，请在本节点用户视图下执行reset mpls ldp，使本节点向倒数第二跳的标签分配方式生效。

使用实例

# 配置出节点向倒数第二跳分配显式空标签。

<Quidway> system-view
[Quidway] mpls
[Quidway-mpls] label advertise explicit-null
