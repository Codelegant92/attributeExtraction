apply backup-nexthop

命令功能

apply backup-nexthop命令用来配置备份下一跳。

undo apply backup-nexthop命令用来删除已配置的备份下一跳。

命令格式

apply backup-nexthop { ipv4-address | auto }

undo apply backup-nexthop

参数说明

ipv4-address：备份下一跳的IP地址。

auto：设置为自动寻找备份下一跳模式。在IP FRR和VPN FRR中都适用。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

在配置备份下一跳之前可以配置匹配一些条件，如匹配接口等。

该命令是专门用于IP FRR和VPN FRR的。IP FRR的情况下需要和命令apply backup-interface配合使用。

使用实例

# 在名为ip_frr_rp的Route-Policy中配置备份下一跳192.168.20.2。

<Quidway> system-view
[Quidway] route-policy ip_frr_rp permit node 10
[Quidway-route-policy] apply backup-interface GigabitEthernet1/0/0
[Quidway-route-policy] apply backup-nexthop 192.168.20.2

# 在名为ip_frr_rp的Route-Policy中删除配置的备份下一跳。

<Quidway> system-view
[Quidway] route-policy ip_frr_rp permit node 10
[Quidway-route-policy] undo apply backup-nexthop
