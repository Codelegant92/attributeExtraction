display ipv6 host

命令功能

display ipv6 host命令用来查看静态域名解析表。

命令格式

display ipv6 host [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

项目

描述

Host

主机名称。

Age

老化时间。0表示是静态域名，不老化。

Flags

标识。Static表示是静态域名。

IPv6Address (es)

域名对应的IPv6地址。
