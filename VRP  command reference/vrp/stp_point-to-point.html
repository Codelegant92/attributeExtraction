stp point-to-point

命令功能

stp point-to-point命令用来配置指定端口的链路类型。

undo stp point-to-point命令用来恢复指定端口的链路类型为缺省值。

缺省情况下，端口的链路类型为auto，即由MSTP来检测与端口相连的链路是否是点到点链路。

命令格式

stp point-to-point { auto | force-false | force-true }

undo stp point-to-point

参数说明

force-true：与当前端口相连的链路是点到点链路。

force-false：与当前端口相连的链路不是点到点链路。

auto：自动检测与该端口相连的链路是否是点到点链路。

使用实例

# 配置以太网接口GE 1/0/0的链路类型为点对点链路。

<Quidway> system-view
[Quidway] inerface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] portswitch
[Quidway-GigabitEthernet1/0/0] stp point-to-point force-true
