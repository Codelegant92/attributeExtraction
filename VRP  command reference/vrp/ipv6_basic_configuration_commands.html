IPv6基础配置命令

功能描述

主要介绍使能转发IPv6单播报文、手工配置接口的全球单播地址、为接口配置及删除自动生成的链路本地地址、设置路由设备发布消息的间隔时间、配置邻居发现缓存中的静态邻居信息、配置动态PMTU项的老化时间和设置TCP的FIN-Wait定时器值等配置命令。
