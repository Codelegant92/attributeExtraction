display cpu-usage

命令功能

display cpu-usage命令用来查看CPU占用率的统计信息和配置信息。

命令格式

display cpu-usage [ slave | slot slot-id ]

display cpu-usage entry-number [ offset ] [ verbose ] [ slave | slot slot-id ] [ history ]

display cpu-usage configuration [ slave | slot slot-id ]

参数说明

slave：显示备用主控板CPU占用率的相关信息。

slot slot-id：指定接口板号。

entry-number：指定每次要显示的条目数。整数形式，取值范围是1～60。

offset：从最近一条记录之前的某个条目开始显示。整数形式，取值范围是0～59。

verbose：显示每条记录的详细信息。

history：显示CPU占用率的历史记录。

configuration：显示配置信息所消耗的CPU占用率。
