timer query（IGMP视图）

命令功能

timer query命令用来在全局性上配置IGMP普遍组查询消息的发送间隔。也称为普遍查询间隔。

undo timer query命令用来恢复该配置参数的缺省值。

缺省情况下，IGMP普遍组查询消息的发送间隔为60秒。

命令格式

timer query interval

undo timer query

参数说明

interval：表示IGMP普遍组查询消息的发送间隔。整数形式，取值范围是1～18000，单位是秒。

使用指南

该命令和接口视图下的igmp timer query命令功能完全相同，差别仅仅是作用范围不同。IGMP视图下的配置具有全局性，对所有接口有效；接口视图下的配置只对当前接口有效。系统优先采用接口视图下的配置。当接口视图下没有进行配置，才采用IGMP视图下的配置。

使用实例

# 在IGMP视图下，配置IGMP普遍组查询报文的发送间隔为125秒。

<Quidway> system-view
[Quidway] igmp
[Quidway-igmp] timer query 125
