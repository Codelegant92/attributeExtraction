natpt v4bound dynamic acl

命令功能

natpt v4bound dynamic acl命令用来为IPv4侧到IPv6侧的报文配置动态映射。

undo natpt v4bound dynamic acl命令用来取消动态映射配置。

命令格式

natpt v4bound dynamic acl acl-number prefix natpt-prefix

undo natpt v4bound dynamic acl acl-number

参数说明

acl-number：指定IPv4访问控制列表编号。整数形式，取值范围是2000～2999，即基本ACL。

natpt-prefix：指定NAT-PT前缀，长度只能为96位。

使用指南

将命令中指定的前缀（前缀长度是96位）加在被ACL标识的IPv4报文的源地址前面，就可将其转换为IPv6地址。

说明：

使用命令undo natpt v4bound dynamic acl可能会影响某些动态映射的转换。

使用实例

# 使用基本ACL 2000来对IPv4报文的源地址进行识别，将NAT-PT前缀2001::/96加在IPv4源地址的前面转换为IPv6地址。

<Quidway> system-view
[Quidway] natpt v4bound dynamic acl 2000 prefix 2001::
