tunnel password

命令功能

tunnel password命令用来指定隧道验证时的密码。

undo tunnel password命令用来取消隧道验证的密码。

缺省情况下，启用隧道验证，隧道验证的密码为空。

命令格式

tunnel password { simple | cipher } password

undo tunnel password

参数说明

simple：以明文形式显示隧道验证的密码。

cipher：以密文形式显示隧道验证的密码。

password：隧道验证的密码，字符串形式。区分大小写，不能输入空格和问号等命令行专用字符。如果密码形式是simple，则password必须是明文密码。如果密码形式是cipher，则password是密文形式，传送过程中可以加密也可以不加密，结果视输入而定。不加密时密码是长度小于等于16的连续字符串，如：1234567。加密时密码长度必须是24，并且必须是密文形式如：_(TT8F ] Y\5SQ=^Q`MAF4<1!!。

使用指南

用户可根据实际需要决定是否在创建隧道连接之前启用隧道验证。为保证通道安全，建议不要禁用隧道验证。

如果在LAC侧或LNS侧启用了隧道验证，则对端也需要进行同样的配置。

使用实例

# 设置隧道验证的密码为yougotit，并且以明文方式显示。

<Quidway> system-view
[Quidway] l2tp-group 1
[Quidway-l2tp-1] tunnel password simple yougotit
