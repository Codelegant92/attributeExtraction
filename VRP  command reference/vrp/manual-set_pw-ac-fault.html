manual-set pw-ac-fault

命令功能

manual-set pw-ac-fault命令用来手工设置主用或者备用PW故障。

undo manual-set pw-ac-fault命令用来撤销手工设置的主用或者备用PW故障。

命令格式

manual-set pw-ac-fault [ secondary ]

undo manual-set pw-ac-fault [ secondary ]

参数说明

secondary：设置备份PW故障。

视图

接口视图

缺省级别

2：配置级

使用指南

通过manual-set pw-ac-fault命令设定PW的故障，可以检测主备PW能否正常切换。检测结果将被保存到日志信息中。

使用实例

# 手工设置PW故障。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] manual-set pw-ac-fault
