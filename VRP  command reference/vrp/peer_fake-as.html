peer fake-as

命令功能

peer fake-as命令用来为本端对等体指定一个伪AS号。

undo peer fake-as命令用来取消已有的配置。

缺省情况下，对等体使用本端实际的AS号。

命令格式

peer { group-name | ipv4-address | ipv6-address } fake-as fake-as-number

undo peer { group-name | ipv4-address | ipv6-address } fake-as

参数说明

group-name：对等体组的名称。

ipv4-address：对等体的IPv4地址。

ipv6-address：对等体的IPv6地址。

说明：

  • ipv4-address参数仅在BGP视图和BGP-VPN实例视图下生效。

  • ipv6-address参数仅在BGP视图和BGP-VPN6实例视图下生效。

fake-as-number：伪自治系统号。

视图

BGP视图、BGP-VPN实例视图、BGP-VPN6实例视图

缺省级别

2：配置级

使用指南

使用此命令可以将本地真实的AS编号隐藏，位于其他AS内的EBGP对等体只能看到这个伪AS编号，即其他AS内的对等体在指定本端对等体所在的AS编号时，应该设置成这个伪AS编号。

使用实例

# 为对等体配置伪AS号。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 1.1.1.2 as-number 200 
[Quidway-bgp] peer 1.1.1.2 fake-as 99
