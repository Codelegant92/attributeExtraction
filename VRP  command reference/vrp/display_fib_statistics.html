display fib statistics

命令功能

display fib statistics命令用来查看FIB表项的总数目。

命令格式

display fib [ slot-id ] [ vpn-instance vpn-instance-name ] statistics

参数说明

slot-id：槽位号，根据路由设备硬件的不同而具有不同的取值范围。

vpn-instance vpn-instance-name：VPN实例的名称。字符串形式，区分大小写，长度范围是1～31。
