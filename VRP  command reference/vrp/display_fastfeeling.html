display fastfeeling

命令功能

display fastfeeling命令用来查看接口的快速感知特性的当前运行状态和统计信息。如果不指定接口，将显示所有接口的快速感知特性运行状态和统计信息。

命令格式

display fastfeeling [ interface interface-type interface-number ]

参数说明

interface interface-type interface-number：接口类型和接口编号。

所有视图

缺省级别

1：监控级

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

使用实例

# 查看接口POS1/0/0的快速感知特性运行状态。

<Quidway> display fastfeeling interface pos 1/0/0
 interface Pos1/0/0:
        fastfeeling status : feeling, feeling
        route interface status: up
        ISIS interface status: up
表1 display fastfeeling命令输出信息描述

项目

描述

fastfeeling status
接口的快速感知状态。

  • feeling：当前fastfeeling处于物理层感知状态

  • notifie：当前fastfeeling处于三层感知状态，即路由模块可能已经知道接口处于Down状态，而不需要fastfeeling模块再做处理

route interface status
路由器的接口状态。

  • up：表示当前配置fastfeeling的接口状态正常

  • down：表示当前配置fastfeeling的接口状态不正常

ISIS interface status
接口的IS-IS协议状态。

  • up：表示配置IS-IS的接口状态正常

  • down：表示配置IS-IS的接口状态不正常
