mpls rsvp-te authentication handshake

命令功能

mpls rsvp-te authentication handshake命令用来配置RSVP-TE握手机制并设置本地密码。

undo mpls rsvp-te authentication handshake命令用来删除RSVP-TE握手机制的配置。

命令格式

mpls rsvp-te authentication handshake local-secret

undo mpls rsvp-te authentication handshake

参数说明

local-secret：本地密码，字符串形式，不支持空格，长度范围是8～40，无缺省值。

接口视图

缺省级别

使用指南

执行该命令前，应先在接口启动验证功能。

local-secret只是本地有效，不需要与邻居配置的local-secret一致。

在本端配置handshake功能后，如果本端收到一个没有与自己建立RSVP认证关系的邻居所发送的RSVP消息，则本端会发送携带local-secret参数的Challenge消息给该邻居，邻居收到Challenge消息后回应一个Response消息。Response消息中携带了收到的Challenge消息的local-secret。本端收到邻居发来的Response消息，如果其中携带的local-secret与本端配置的一致，就确定可以与该邻居建立RSVP认证关系。

使用实例

#配置RSVP握手机制并设置本地密码为12345678。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mpls rsvp-te authentication cipher quidway100
[Quidway-GigabitEthernet1/0/0] mpls rsvp-te authentication handshake 12345678
