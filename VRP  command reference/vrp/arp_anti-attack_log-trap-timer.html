arp anti-attack log-trap-timer

命令功能

arp anti-attack log-trap-timer命令用来配置写ARP日志和发送告警的时间间隔。

undo arp anti-attack log-trap-timer命令用来恢复写ARP日志和发送告警的缺省时间间隔。

缺省情况下，写ARP日志和发送告警时间间隔为0，即未使能时间戳抑制的写ARP日志和发送告警功能。

命令格式

arp anti-attack log-trap-timer time

undo arp anti-attack log-trap-timer

参数说明

time：每次告警的时间间隔，范围0～1200，单位为秒。

使用指南

可以根据设备的情况配置写ARP日志和发送告警的时间间隔。

使用实例

# 设置写ARP日志和发送告警的时间间隔为20秒。
