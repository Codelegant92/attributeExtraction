link-protocol hdlc

命令功能

link-protocol hdlc命令用来配置接口封装HDLC协议。

缺省情况下，接口封装PPP协议。

命令格式

link-protocol hdlc

参数说明

无

视图

接口视图

缺省级别

2：配置级

使用指南

加入IP-Trunk的接口，链路层协议必须为HDLC。

改变接口的链路层协议时，必须先执行命令shutdown将接口关闭。配置完成后，执行命令undo shutdown启用接口。缺省情况下，接口为关闭状态。

接口封装HDLC协议时，IP地址必须与对端接口的IP地址在同一网段。

使用实例

# 配置接口POS1/0/0封装HDLC协议。

<Quidway> system-view
[Quidway] interface pos 1/0/0
[Quidway-Pos1/0/0] link-protocol hdlc
