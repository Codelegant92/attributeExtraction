display interface loopback

命令功能

display interface looback命令用来查看Loopback接口的状态信息。

命令格式

display interface loopback [ number ] [ | { begin | exclude | include } regular-expression ]

参数说明

number：Loopback接口的编号，取值范围是0～1023。如果不指定接口编号，将显示所有已创建的Loopback接口的状态。

|：使用正则表达式过滤输出的显示信息。

begin：显示以包含指定正则表达式的行开始的配置信息。

exclude：显示不包含指定正则表达式的行的配置信息。

include：显示包含指定正则表达式的行的配置信息。

regular-expression：正则表达式。字符串形式，支持空格，区分大小写。

使用指南

在监控接口的状态或检查接口的故障原因时，可执行此命令获取接口的状态信息和统计信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

项目

描述

LoopBack6 current state :

Loopback接口当前的物理状态。Loopback接口创建成功后物理状态一直是Up。

Line protocol current state :

显示该接口的链路协议状态。Loopback接口创建成功后链路协议状态一直是Up。

Description :

对接口的说明，支持空格，区分大小写，最多可输入242个字符。该说明信息便于用户了解该接口的作用。

The Maximum Transmit Unit is

接口的最大传输单元（MTU），缺省值是1500字节。长度大于MTU的报文，将会被分片后再发送。如果设置了不准分片，会被丢弃。

Internet Address is

接口的IP地址。

Physical is

物理层链路信息。

Statistics last cleared:

上次清除本接口统计信息的时间。

Last 300 seconds input rate

Last 300 seconds output rate

这一字段显示了在前5分钟内通过接口发送和接收的字节速率和报文速率。
