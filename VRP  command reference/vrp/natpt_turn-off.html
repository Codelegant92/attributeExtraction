natpt turn-off

命令功能

natpt turn-off tos命令用来设置IPv4报文的ToS字段为0。当IPv6报文转换为IPv4报文时，ToS字段设为0。

natpt turn-off traffic-class命令用来设置IPv6报文的流量级别字段为0。当IPv4报文转换为IPv6报文时，流量级别字段设为0。

undo natpt turn-off tos命令用来设置在IPv4报文中的ToS字段与原来的流量级别字段一致。

undo natpt turn-off traffic-class命令用来设置在IPv6报文中的流量级别字段与原来的ToS字段一致。

命令格式

natpt turn-off { tos | traffic-class }

undo natpt turn-off { tos | traffic-class }

参数说明

tos：设置IPv4报文的ToS字段为0。

traffic-class：设置IPv6报文的流量级别字段为0。
