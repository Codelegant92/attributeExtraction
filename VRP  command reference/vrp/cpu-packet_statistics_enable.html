cpu-packet statistics enable

命令功能

cpu-packet statistics enable命令用来使能所有上送到CPU的报文的统计功能。

undo cpu-packet statistics enable命令用来去使能所有上送到CPU的报文的统计功能。

缺省情况下，去使能所有上送到CPU的报文的统计功能。

命令格式

cpu-packet statistics enable

undo cpu-packet statistics enable

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

配置该命令后，所有上送到CPU的报文都会被统计。

如果想只根据PVC或VLAN进行统计，请使用命令cpu-packet statistics pvc enable或cpu-packet statistics vlan enable。

注意：

执行命令undo cpu-packet statistics enable后，配置在接口下的命令cpu-packet statistics pvc enable和cpu-packet statistics vlan enable会同时被删除。

使用实例

# 使能所有上送到CPU的报文的统计功能。

<Quidway> system-view
[Quidway] cpu-packet statistics enable
