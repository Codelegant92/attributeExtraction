apply ospf

命令功能

apply ospf命令用来设置引入路由到OSPF的特定区域。

undo apply ospf命令用来取消设置。

命令格式

apply ospf { backbone | stub-area }

undo apply ospf

参数说明

backbone：设置将路由引入到OSPF的骨干区域。

stub-area：设置将路由引入到OSPF的NSSA区域。

视图

Route-Policy视图

缺省级别

2：配置级

使用指南

无

使用实例

# 设置将路由引入到OSPF骨干区域。

<Quidway> system-view
[Quidway] route-policy policy permit node 10
[Quidway-route-policy] apply ospf backbone
