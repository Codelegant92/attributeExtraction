reset x25 counters

命令功能

reset x25 counters命令用来清除接口的X.25统计信息。

命令格式

reset x25 counters interface interface-type interface-number

参数说明

interface interface-type interface-number：需要清除X.25统计信息的接口类型和接口编号。

视图

用户视图

缺省级别

2：配置级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除串口Serial 1/0/0的X.25统计信息。

<Quidway> reset x25 counters interface serial 1/0/0
