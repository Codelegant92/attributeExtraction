display pim ipv6 interface

命令功能

display pim ipv6 interface命令用来查看接口的IPv6 PIM信息。

命令格式

display pim ipv6 interface [ interface-type interface-number | up | down ] [ verbose ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

interface-type interface-number：指定接口类型和编号。

up：显示IPv6协议状态为Up的PIM接口信息。

down：显示IPv6协议状态为Down的PIM接口信息。

verbose：显示接口关于IPv6 PIM的详细信息。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

请参见IPv4 PIM配置命令。

使用display pim ipv6 interface命令时，若指定了interface-type interface-number，则仅显示在该接口下的IPv6 PIM配置和运行信息。如果该接口未使能IPv6 PIM，则出现提示信息。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示接口GE6/0/0的IPv6 PIM信息。

<Quidway> display pim ipv6 interface gigabitethernet6/0/0
 VPN-Instance: public net
 Interface           State NbrCnt HelloInt   DR-Pri     DR-Address
 GE6/0/0             up    0      30         1          FE80::2E0:8AFF:(local)
