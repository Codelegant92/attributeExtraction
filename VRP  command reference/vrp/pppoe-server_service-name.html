pppoe-server service-name

命令功能

pppoe-server service-name命令用来设定当前接口上启动的PPPoE服务的服务名。

undo pppoe-server service-name命令用来恢复当前一接口PPPoE服务名的缺省值。

缺省情况下，pppoe-server服务名为空。

命令格式

pppoe-server service-name service-name

undo pppoe-server service-name

参数说明

service-name：PPPoE服务的名字，字符串形式，长度为1～128。

视图

Ethernet接口视图、GE接口视图

缺省级别

2：配置级

使用指南

设置服务器的名字以方便记忆和查询。

使用实例

# 在GE1/0/0上使能PPPoE server，配置服务名为123asd。

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] pppoe-server bind virtual-template 1
[Quidway-GigabitEthernet1/0/0] pppoe-server service-name 123asd
