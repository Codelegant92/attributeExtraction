c-rp advertisement-interval（IPv4）

命令功能

c-rp advertisement-interval命令用来配置C-RP周期性发送Advertisement消息的时间间隔。

undo c-rp advertisement-interval命令用来恢复发送时间间隔为缺省值。

缺省情况下，C-RP周期性发送Advertisement消息的时间间隔是60s。

命令格式

c-rp advertisement-interval interval

undo c-rp advertisement-interval

参数说明

interval：表示C-RP周期性发送Advertisement消息的时间间隔。整数形式，取值范围是1～65535，单位是秒。

视图

公网实例PIM视图、VPN实例PIM视图

缺省级别

2：配置级

使用指南

PIM-SM域内的所有C-RP向同一个BSR周期性的发送Advertisement消息。从而该BSR能够收集到完整的RP-Set。

该命令和c-rp interface-type interface-number advertisement-interval adv-interval命令功能完全相同，差别只是作用范围不同。c-rp advertisement-interval interval命令具有全局性，对整个路由器有效；c-rp interface-type interface-number advertisement-interval adv-interval命令只对指定接口有效。

系统优先采用c-rp interface-type interface-number advertisement-interval adv-interval命令的配置。当未执行c-rp interface-type interface-number advertisement-interval adv-interval命令时，才采用c-rp advertisement-interval interval命令的配置。

使用实例

# 在公网实例下，配置C-RP周期性发送Advertisement消息的时间间隔为30s。

<Quidway> system-view
[Quidway] pim
[Quidway-pim] c-rp gigabitethernet1/0/0
[Quidway-pim] c-rp advertisement-interval 30
