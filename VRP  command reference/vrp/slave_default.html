slave default

命令功能

slave default命令用来设置系统启动时缺省的备用板槽号。

undo slave default命令用来恢复缺省值。

命令格式

slave default slot-id

undo slave default

参数说明

slot-id：备板槽号。

视图

系统视图

缺省级别

2：配置级

使用指南

该命令设置当主板和备板都在位的时候，如果重新启动系统（如系统重新加电），哪个板作为备用板。

使用实例

# 设置系统重启动时，把6号槽上的板作为备用板。

<Quidway> system-view
[Quidway] slave default 6
