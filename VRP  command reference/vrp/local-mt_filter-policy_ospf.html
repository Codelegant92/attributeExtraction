local-mt filter-policy (OSPF)

命令功能

local-mt filter-policy命令用来配置OSPF本地组播拓扑特性的策略。

undo local-mt filter-policy命令用来取消该策略。

缺省情况下，不配置本地组播拓扑特性的策略。

命令格式

local-mt filter-policy { acl acl-number | ip-prefix ip-prefix-name }

undo local-mt filter-policy

参数说明

acl acl-number：指定基本访问控制列表编号。整数形式。

ip-prefix ip-prefix-name：IP地址前缀列表名称。字符串形式。

视图

OSPF视图

缺省级别

2：配置级

使用指南

通过配置本地组播拓扑的策略，可以只允许组播源的路由加入到MIGP路由表中，控制MIGP路由表的大小，减小对路由器CPU、内存的占用。

使用实例

# 配置OSPF本地组播拓扑特性的策略。

<Quidway> system-view
[Quidway] ospf 1
[Quidway-ospf-1] local-mt filter-policy acl 2000
