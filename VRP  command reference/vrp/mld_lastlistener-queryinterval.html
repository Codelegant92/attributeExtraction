mld lastlistener-queryinterval

命令功能

mld lastlistener-queryinterval命令用来设置MLD查询器接收到主机发来的MLD离开报文后，发送MLD最后侦听者查询报文的时间间隔。

undo mld lastlistener-queryinterval命令用来恢复缺省值。

缺省情况下，发送MLD最后侦听者查询报文的时间间隔为1s。

命令格式

mld lastlistener-queryinterval interval

undo mld lastlistener-queryinterval

参数说明

interval：指定发送MLD最后侦听者查询报文的时间间隔。整数形式，取值范围是1～5，单位是s。

使用实例

# 设置查询器查询接口GE1/0/0上发送最后侦听者查询消息的时间间隔为3s。.

<Quidway> system-view
[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] mld lastlistener-queryinterval 3
