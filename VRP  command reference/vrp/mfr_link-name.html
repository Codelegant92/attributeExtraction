mfr link-name

命令功能

mfr link-name命令用来设置帧中继捆绑链路标识符（bundle link identification，LID）。

undo mfr link-name命令用来恢复缺省设置。

缺省情况下，LID是相应物理接口的名称。

命令格式

mfr link-name [ name ]

undo mfr link-name

参数说明

name：捆绑链路标识名，字符串形式，长度为1～49。

视图

帧中继接口视图

缺省级别

2：配置级

使用指南

对端设备通过LID来识别帧中继捆绑链路，也可以使用LID将该捆绑链路关联到帧中继捆绑。LID在本地有效，链路两端的LID可以相同。

改变接口的捆绑链路标识符时，必须对该接口执行shutdown/undo shutdown命令，新的捆绑链路标识符才会生效。

使用实例

# 设置多链路帧中继捆绑链路Serial1/0/0:0的捆绑链路标识符为bl1。

<Quidway> system-view
[Quidway] interface serial 1/0/0:0
[Quidway-Serial1/0/0:0] link-protocol fr mfr 1/0/123
[Quidway-Serial1/0/0:0] mfr link-name bl1
