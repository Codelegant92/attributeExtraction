display ppp compression iphc tcp

命令功能

display ppp compression iphc tcp命令用来显示IP报文头压缩中IP/TCP头压缩的统计信息。

命令格式

display ppp compression iphc tcp [ interface interface-type interface-number ]

参数说明

interface interface-type interface-number：指定的接口类型和接口编号。如Virtual-Template、POS、Serial等。

使用指南

display ppp compression iphc tcp命令可以观察到的信息包括：

  • 接收到的报文情况统计情况（压缩报文数/错误报文数/丢弃报文数/报文总数）

  • 发送的报文统计情况（压缩报文数/报文总数、发送字节数/保存字节数/字节总数）

  • 接收方可解压缩的数据流的最大数目

  • 发送方可解压缩的数据流的最大数目

  • 查找压缩表项的次数以及查找压缩表项的失败次数

使用实例

# 显示IP报文头压缩中TCP头压缩的统计信息。

主要字段

描述

IPHC

IP报文头压缩

Interface

报文头压缩应用的接口

Received

接收到的报文情况统计

Compress/Error/Discard/Total

压缩报文数/错误报文数/丢弃报文数/报文总数

Sent

发送的报文统计情况

压缩报文数/报文总数

Send/Save/Total

发送字节数/保存字节数/字节总数

Rx

Tx

作为发送方,可压缩的数据流的最大数目

Long-search

查找压缩表项的次数

Miss

查找压缩表项的失败次数
