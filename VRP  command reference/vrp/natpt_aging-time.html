natpt aging-time

命令功能

natpt aging-time命令用来设置NAT-PT连接的有效时间，不同协议类型设置的时间值不同。

命令格式

natpt aging-time default

natpt aging-time { dns { dns-interval | default } | finrst { finrst-interval | default } | frag { frag-interval | default } | icmp { icmp-interval | default } | syn { syn-interval | default } | tcp { tcp-interval | default } | udp { udp-interval | default } }

参数说明

default：设置NAT-PT连接的有效时间为缺省值。

dns dns-interval：指定DNS连接有效时间。整数形式，单位是秒，取值范围是5～600。缺省值是10秒。

finrst dns-interval：指定FINRST连接有效时间。整数形式，单位是秒，取值范围是5～600。缺省值是5秒。

frag frag-interval：指定FRAG连接有效时间。整数形式，单位是秒，取值范围是5～600。缺省值是5秒。

icmp icmp-interval：指定ICMP连接有效时间。整数形式，单位是秒，取值范围是5～600。缺省值是20秒。

syn syn-interval：指定SYN连接有效时间。整数形式，单位是秒，取值范围是5～600。缺省值是240秒。

tcp tcp-interval：指定TCP连接有效时间。整数形式，单位是秒，取值范围是5～86400。缺省值是86400秒。

udp udp-interval：指定UDP连接有效时间。整数形式，单位是秒，取值范围是5～600。缺省值是40秒。
