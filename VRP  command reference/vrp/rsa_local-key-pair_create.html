rsa local-key-pair create

命令功能

rsa local-key-pair create命令用来生成本地RSA主机密钥对和服务器密钥对。

命令格式

rsa local-key-pair create

参数说明

无

视图

系统视图

缺省级别

2：配置级

使用指南

使用该命令进行配置时，如果RSA密钥已经存在，则系统将提示用户确认是否替换原有密钥。产生的密钥对命名方式为“路由器名称_server和路由器名称_host”，如：Quidway_host、Quidway_server。该命令不在配置文件中保存。

输入该命令后，会提示您输入主机密钥的位数。服务器密钥对的位数与主机密钥对的位数至少相差128位。服务器密钥对和主机密钥对的最小长度为512位，最大长度为2048位。如果已有密钥对存在，则需用户确认是否进行修改。

成功完成SSH登录的首要操作是配置并产生本地RSA密钥对。在进行其它SSH配置之前，请先完成rsa local-key-pair create配置，生成本地密钥对。

此命令只需执行一遍，路由器重新启动后不必再次执行。

使用实例

# 配置生成本地主机密钥对和服务器密钥对。
