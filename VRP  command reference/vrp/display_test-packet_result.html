display test-packet result

命令功能

display test-packet result命令用来查看发送测试报文的结果。显示的结果信息包括：

  • 发送测试报文的个数

  • 接收报文的个数

  • 丢弃报文的个数

  • 发送报文的总字节数

  • 接收报文的总字节数

  • 丢弃报文的总字节数

  • 发送开始时间

  • 发送结束时间

命令格式

display test-packet result

参数说明

无

使用指南

使用test-packet start命令发送测试报文后，可使用本命令查看测试结果。
