crc（CT3接口）

命令功能

crc命令用来配置CT3通过各种方式形成的串口的CRC校验模式。

undo crc命令用来恢复缺省设置。

缺省情况下，使用16位CRC校验。

命令格式

crc { 16 | 32 | none }

undo crc

参数说明

16：使用16位CRC校验。

32：使用32位CRC校验。

none：不使用CRC校验。

视图

串行接口视图

缺省级别

2：配置级

使用指南

对于T3形成的串口、T1通道形成的串口、以及T1通道时隙捆绑形成的串口，都可以在相应的串口视图下配置它的CRC校验。

使用实例

# 设置CT3接口1/0/0在非通道化模式下形成的串口使用32位CRC校验。

<Quidway> system-view
[Quidway] controller t3 1/0/0
[Quidway-T3 1/0/0] using t3
[Quidway-T3 1/0/0] quit
[Quidway] interface serial 1/0/0/0:0
[Quidway-Serial1/0/0/0:0] crc 32
