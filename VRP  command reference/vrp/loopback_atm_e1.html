loopback（ATM E1）

命令功能

loopback命令用来设置ATM E1/T1的环回方式。

undo loopback命令用来取消环回。

缺省情况下，禁止环回功能。

命令格式

loopback { cell | local | payload | remote }

undo loopback

参数说明

cell：对内信元环回。用来检测本端物理芯片是否正常。

local：对内自环。用来检测本端业务芯片是否正常。

payload：对外载荷环回。用来检测数据负荷成帧是否正常。

remote：对外线路环回。用来检测对端是否正常。

缺省级别

2：配置级

使用指南

只有在进行某些特殊功能测试的时候，才将接口设置为对内自环或对外环回。正常工作时，不要启用环回功能。

使用实例

# 设置ATM E1/T1接口1/0/2进行载荷环回。

<Quidway> system-view
[Quidway] interface atm 1/0/2
[Quidway-Atm1/0/2] loopback payload
