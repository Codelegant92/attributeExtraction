display cfm remote-mep

命令功能

display cfm remote-mep命令用来查看指定MD和MA内的RMEP的信息，包括：

  • 所在MD的名称和级别

  • 所在MA的名称和MA关联的VLAN

  • RMEP的ID

  • 所在设备的桥MAC地址

  • 当前设备是否接收该RMEP发送的CCM消息

  • 是否配置了接口闪断功能

命令格式

display cfm remote-mep [ md md-name [ ma ma-name [ mep-id mep-id ] ] ] [ | count ] [ | { begin | include | exclude } regular-expression ]

参数说明

md md-name：显示指定MD内RMEP的信息。md-name表示MD的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。不输入本参数时，显示本设备上所有MD和MA内的RMEP的信息。

ma ma-name：。显示指定MA内RMEP的信息。ma-name表示MA的名称，字符串形式，不支持空格、“-”和“?”，长度范围是1～43。输入md-name参数，但不输入本参数时，显示本设备上指定MD内所有MA内的RMEP的信息。

mep-id mep-id：显示指定RMEP的信息。mep-id表示RMEP的ID，整数形式，取值范围是1～8191。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当配置完RMEP后或CFM功能出现故障时，可使用本命令验证或了解RMEP的配置。

表1 display cfm remote-mep命令输出信息描述表

项目

描述

The total number of RMEPs is

RMEP的数量。

MD Name

MD的名称。

Level

MD的级别，范围是0～7。级别越高MD的优先级越高。

MA Name

MA的名称。

RMEP ID

RMEP的标识。

Vlan ID

MA关联的VLAN的ID。

VSI Name

MA关联的VSI。

MAC

RMEP所在设备的桥MAC地址。

CCM- receive
当前设备是否接收该RMEP发送的CCM消息，取值包括：

  • disabled：当前设备不接收该RMEP发送的CCM消息。

  • enabled：当前设备接收该RMEP发送的CCM消息。

Trigger If-down
RMEP是否配置了接口闪断功能，取值包括：

  • disabled：表示RMEP未配置接口闪断功能。

  • enabled：表示RMEP配置了接口闪断功能。即：当该RMEP所在MA的MEP检测到和该RMEP之间有CFM连通性故障时，对该MEP所在接口进行闪断操作。

详细说明请参见cfm trigger if-down。

CFM Stauts
CFM的运行状态：

  • up：CC链路检测正常

  • down：CC链路检测故障

  • disable：没有配置CFM功能（即没有配置本端CC发送使能和远端CC接收使能）
