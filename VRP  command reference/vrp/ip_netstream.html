ip netstream

命令功能

ip netstream命令用来使能接口的NetStream入或出统计功能。

undo ip netstream命令用来禁止接口的NetStream入或出统计功能。

使能接口的NetStream入或出统计功能时，同时将启动对单播和广播报文的入或出统计功能。

缺省情况下，禁止NetStream入或出统计功能。

命令格式

ip netstream { inbound | outbound } [ interface-type interface-number ]

undo ip netstream { inbound | outbound } [ interface-type interface-number ]

参数说明

inbound：统计入方向的IPv4报文。

outbound：统计出方向的IPv4报文。

interface-type：VLAN下的二层接口类型。

interface-number：VLAN下的二层接口编号。
