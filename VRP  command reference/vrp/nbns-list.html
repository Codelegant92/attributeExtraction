nbns-list

命令功能

nbns-list命令用来配置全局DHCP地址池的客户端的NetBIOS服务器地址。

undo nbns-list命令用来删除分配的NetBIOS服务器地址。

缺省情况下，未配置NetBIOS地址。

命令格式

nbns-list ip-address &<1-8>

undo nbns-list { ip-address | all }

参数说明

ip-address：指定NetBIOS服务器的IP地址，一条命令中最多可以配置8个IP地址，用空格分隔。

all：删除所有NetBIOS服务器的IP地址。

视图

DHCP地址池视图

缺省级别

2：配置级

使用指南

在目前的实现中，每个DHCP地址池最多可以配置8个NetBIOS服务器。

使用实例

# 配置地址池0为它的客户端分配的NetBIOS服务器地址为10.12.1.99。

<Quidway> system-view
[Quidway] dhcp server ip-pool 0
[Quidway-dhcp-0] nbns-list 10.12.1.99
