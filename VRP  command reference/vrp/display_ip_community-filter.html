display ip community-filter

命令功能

display ip community-filter命令用来显示已配置的团体属性过滤器信息。

命令格式

display ip community-filter [ basic-comm-filter-num | adv-comm-filter-num | comm-filter-name ] [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

basic-comm-filter-num：基本团体属性过滤器号。整数形式，取值范围是1～99。

adv-comm-filter-num：高级团体属性过滤器号。整数形式，取值范围是100～199。

comm-filter-name：团体属性过滤器名称。字符串类型，取值范围是1～51。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 显示所有团体属性过滤器。

<Quidway> display ip community-filter
Community filter Number 10
         permit  no-export
Community filter Number 110
         permit 110:110
Named Community basic filter: aa (ListID = 200)
         permit  1 internet
Named Community advanced filter: bb (ListID = 700)
         permit ^20
