ipv6 address eui-64

命令功能

ipv6 address eui-64命令用来给接口配置EUI-64格式的全球单播地址。

undo ipv6 address eui-64命令用来删除接口上已配置的EUI-64格式的全球单播地址。

命令格式

ipv6 address { ipv6-address prefix-length | ipv6-address/prefix-length } eui-64

undo ipv6 address { ipv6-address prefix-length | ipv6-address/prefix-length } eui-64

参数说明

ipv6-address：指定IPv6地址。

prefix-length：指定前缀长度。整数形式，在配置EUI-64地址时其取值不能大于64。

使用指南

此命令支持Ethernet接口及其子接口、GigabitEthernet接口及其子接口、POS接口、Serial接口、Loopback接口、Eth-Trunk接口及其子接口、IP-Trunk接口以及Tunnel接口。

使用实例

# 配置GE1/0/0接口的EUI-64格式地址。

<Quidway> system-view
[Quidway] interface gigabitethernet1/0/0
[Quidway-GigabitEthernet1/0/0] ipv6 enable
[Quidway-GigabitEthernet1/0/0] ipv6 address 2001::1/64 eui-64
