shutdown（CPOS接口视图）

命令功能

shutdown命令用来禁止CPOS物理端口。

undo shutdown命令用来使能CPOS物理端口。

缺省情况下，CPOS物理端口是使能的。

命令格式

shutdown

undo shutdown

参数说明

无

视图

CPOS接口视图

缺省级别

2：配置级

使用指南

对CPOS物理端口执行shutdown操作后，该CPOS的所有通道及捆绑形成的串口将全部被禁止，停止收发数据。如果执行undo shutdown操作，则所有通道和捆绑形成的串口将恢复为up。

使用实例

# 关闭CPOS物理端口1/0/0。

<Quidway> system-view
[Quidway] controller cpos 1/0/0
[Quidway-Cpos1/0/0] shutdown
