channel（multicast-channel视图）

命令功能

channel命令用来进入channel-name视图。

channel type命令用来建立ASM或SSM模型的组播节目组。

undo channel命令用来删除指定节目组的所有信息。

命令格式

channel channel-name type { asm | ssm }

channel channel-name

undo channel channel-name

参数说明

channel-name：指定节目组的名称。字符串形式，不支持空格，只包括数字、字母及下划线，区分大小写，长度范围是1～15。

type：指定节目组的类型。

asm：指定该节目组是ASM模型。

ssm：指定该节目组是SSM模型。

视图

multicast-channel视图

缺省级别

2：配置级

使用指南

创建一个节目组时必须指定节目组的类型。

执行undo channel命令删除指定节目组时，必须先清除掉接口视图下所有关于该节目组的配置，否则不允许执行该命令。

说明：

删除节目组时该节目组视图下的所有配置信息都被删除。

一个实例最多可以支持256个节目组。

使用实例

# 创建ASM模型的节目组SDTV。

<Quidway> system-view
[Quidway] multicast routing-enable
[Quidway] multicast-channel
[Quidway-multicast-channel] channel SDTV type asm
[Quidway-multicast-channel-SDTV]
