reset pppoe-server session statistic

命令功能

reset pppoe-server session statistic命令用来清除指定接口上的PPPoE会话统计信息。

命令格式

reset pppoe-server session statistic interface interface-type interface-number

参数说明

interface-type：接口类型，可以是以太网接口、千兆以太网接口或Virtual Ethernet接口。

interface-number：接口编号。

视图

用户视图

缺省级别

2：配置级

使用指南

在某些情况下，需要统计一定时间内某接口的流量信息，这时必须在统计开始前清除该接口原有的统计信息，使接口重新进行统计。

使用实例

# 清除接口GE 1/0/0上PPPoE会话的统计信息。

<Quidway> reset pppoe-server session statistic interface gigabitethernet 1/0/0
