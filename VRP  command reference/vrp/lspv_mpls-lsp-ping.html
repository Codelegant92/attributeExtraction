lspv mpls-lsp-ping

命令功能

lspv mpls-lsp-ping echo enable命令用来使能对MPLS echo request报文的响应功能。

undo lspv mpls-lsp-ping echo enable命令用来关闭对MPLS echo request报文的响应功能。

缺省情况下，对MPLS echo request报文的响应功能是使能的。

命令格式

lspv mpls-lsp-ping echo enable

undo lspv mpls-lsp-ping echo enable

参数说明

无

2：配置级

使用指南

关闭对MPLS echo request报文的响应功能是通过关闭3503端口来实现的。

使用实例

# 打开对MPLS echo request报文的响应功能。

<Quidway> system-view
[Quidway] lspv mpls-lsp-ping echo enable
