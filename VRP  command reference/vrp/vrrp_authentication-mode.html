vrrp authentication-mode

命令功能

vrrp authentication-mode命令用来设置VRRP备份组的认证字。

undo vrrp authentication-mode命令用来取消VRRP备份组的认证。

命令格式

vrrp authentication-mode simple key

undo vrrp authentication-mode

参数说明

simple：进行简单字符认证。

key：认证字。字符串形式，长度范围是1～8，不区分大小写。

接口视图

缺省级别

2：配置级

使用指南

VRRP协议规定一个接口上的备份组应当使用相同的认证字，因此，vrrp authentication-mode命令为一个接口上的所有VRRP备份组设置认证字。

说明：

同一个备份组的接口应使用相同的认证字。

使用实例

# 设置接口GE 1/0/0上所有VRRP备份组的认证字为“huawei”。

[Quidway] interface gigabitethernet 1/0/0
[Quidway-GigabitEthernet1/0/0] vrrp authentication-mode simple huawei
