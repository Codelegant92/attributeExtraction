display ripng interface

命令功能

display ripng interface命令用来显示RIPng的接口信息。

命令格式

display ripng process-id interface [ interface-type interface-number ] [ verbose ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：RIPng进程号。整数形式，取值范围是1～65535。

interface-type：接口类型，如GigabitEthernet、Loopback等。

interface-number：接口号。

verbose：打开接口的详细内容调试信息开关。

|：使用正则表达式对输出信息进行过滤。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

此命令的输出信息有助于查看RIPng配置及运行状态，进行故障诊断并验证配置效果。

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看GigabitEthernet1/0/0接口上RIPng的信息。

<Quidway> display ripng 1 interface GigabitEthernet1/0/0
 GigabitEthernet1/0/0
    FE80::A0A:200:1
    State : UP, Protocol : RIPNG, MTU : 1440
表1 display ripng interface命令输出信息描述

项目

描述

State
接口状态：

  • Up

  • Down

Protocol

接口运行的路由协议

MTU

链路的MTU值

# 查看GigabitEthernet1/0/0接口上RIP的详细信息。

<Quidway> display ripng 1 interface GigabitEthernet1/0/0 verbose
 GigabitEthernet1/0/0
    FE80::A0A:200:1
    State : UP, Protocol : RIPNG, MTU : 1440
    Metricin : 0 , Metricout : 1
    Default Route : Disabled
    Poison Reverse : Disabled
    Split Horizon : Enabled
表2 display ripng interface verbose命令输出信息描述

项目

描述

Metricin

接收路由附加的度量值

Metricout

发送路由附加的度量值

Poison-reverse

接口是否使能了毒性反转特性

Split Horizon

接口是否使能了水平分割特性
