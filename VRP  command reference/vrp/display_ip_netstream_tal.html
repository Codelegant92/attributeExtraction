display ip netstream tal

命令功能

display ip netstream tal命令用来显示TAL信息，可以通过选择过滤条件来显示需要的TAL信息。

命令格式

display ip netstream tal [ label label | l3vpn | kompella | pwe3 | pwe3-switch | vpls | public-lsp | multicast-vpn | vpn-interface ]

参数说明

label label：显示该label对应的TAL信息。整数形式，取值范围是：16～1048575。

l3vpn：显示L3VPN私网标签对应的TAL信息。

kompella：显示Kompella的VLL私网标签对应的TAL信息。

pwe3：显示PWE3对应的TAL信息。

pwe3-switch：显示PWE3交换对应的TAL信息。

vpls：显示VPLS对应的TAL信息。

public-lsp：显示MBGP Egress节点或者MPLS的公网标签对应的TAL信息。

multicast-vpn：显示组播VPN对应的TAL信息。

vpn-interface：VPN和接口的绑定的TAL信息。
