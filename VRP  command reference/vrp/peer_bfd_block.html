peer bfd block

命令功能

peer bfd blcok命令用来阻止对等体从对等体组中继承BFD功能。

undo peer bfd block命令用来恢复对等体从对等体组中继承BFD功能。

命令格式

peer ipv4-address bfd block

undo peer ipv4-address bfd block

参数说明

ipv4-address：对等体的IPv4地址。

视图

BGP视图、BGP-VPN实例视图

缺省级别

2：配置级

使用指南

Group配置了BFD功能，属于这个Group的Peer都将建立BFD会话，使能命令可以阻止Peer从Group中继承BFD功能。

说明：

peer ipv4-address bfd block命令和peer { group-name | ipv4-address } bfd enable命令是两条互斥命令，配置bfd block命令后，会自动删除BFD会话。

使用实例

# 取消对等体2.2.2.9继承group的BFD功能。

<Quidway> system-view
[Quidway] bgp 100
[Quidway-bgp] peer 2.2.2.9 bfd block
