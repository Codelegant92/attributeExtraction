display isis bfd interface

命令功能

display isis bfd interface命令用来查看使能BFD特性的接口信息。

命令格式

display isis [ process-id ] bfd interface [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

process-id：IS-IS进程号。整数类型，取值范围是1～65535。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。

使用实例

# 查看使能BFD的接口信息。

<Quidway> display isis 1 bfd interface

 Total interfaces: 1                          Total bfd enabled interfaces: 1

项目

描述

Interface

使能BFD特性的接口

BFD.State
接口上的BFD状态：

  • enable：接口上使能BFD

  • disable：接口上未使能BFD

Min-Tx

发送BFD报文的最小间隔

Min-Rx

接收BFD报文的最小间隔

Mul

本地检测BFD报文倍数
