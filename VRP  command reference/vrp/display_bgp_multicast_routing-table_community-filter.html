display bgp multicast routing-table community-filter

命令功能

display bgp multicast routing-table community-filter命令用来查看匹配指定的MBGP团体列表的组播路由信息。

命令格式

display bgp multicast routing-table community-filter { { community-filter-name | basic-community-filter-number } [ whole-match ] | advanced-community-filter-number } [ | count ] [ | { begin | exclude | include } regular-expression ]

参数说明

basic-community-filter-number：指定基本团体属性过滤器号。

advanced-community-filter-number：指定高级团体属性过滤器号。

community-filter-name：团体属性过滤器名称。

whole-match：精确匹配。

|：使用正则表达式对输出信息进行过滤。

count：统计符合输出条件的信息行数。

begin：从包含匹配字符串的行开始输出。

exclude：只输出不包含匹配字符串的行。

include：只输出包含匹配字符串的行。

regular-expression：对输出信息进行匹配的正则表达式。

使用指南

当显示信息太多不方便查看时，可以选择begin、exclude或include参数指定匹配条件来过滤显示信息。
