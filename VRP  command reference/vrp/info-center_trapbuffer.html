info-center trapbuffer

命令功能

info-center trapbuffer命令用来打开告警缓冲区，设置输出告警信息的通道号和告警缓冲区大小。

undo info-center trapbuffer命令用来取消当前设置。

缺省情况下，允许向告警缓冲区输出信息，告警缓冲区的大小为256。

命令格式

info-center trapbuffer [ channel { channel-number | channel-name } | size buffersize ] *

undo info-center trapbuffer [ channel | size ]

参数说明

channel：设置向告警缓冲区输出信息的通道。

channel-number：通道号，取值为0～9，即系统有10个通道。

channel-name：通道名。

size：设置告警缓冲区的大小。

buffersize：告警缓冲区的大小（可容纳消息的条数），取值范围0～1024。

使用指南

只有启动了信息中心，该命令才会生效。

通过设置告警缓冲区的大小，可使信息在该方向输出。

使用实例

# 使能路由器向告警缓冲区发送信息，设定告警缓冲区的大小为30。

[Quidway] info-center trapbuffer size 30
