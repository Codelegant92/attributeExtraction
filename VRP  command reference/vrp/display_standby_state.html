display standby state

命令功能

display standby state命令用来查看备份中心的配置和状态信息，包括主接口与备份接口的接口状态、备份状态、备份状态标识、备份接口优先级和备份负载状态。

命令格式

display standby state

参数说明

无

使用指南

当需要查看备份中心的配置和状态信息时可以使用本命令。

使用实例

# 配置负载分担主备备份，POS1/0/0、POS2/0/0和Tunnel2/0/0作为POS3/0/0的备份接口。

<uidway> system-view
[Quidway] interface pos 3/0/0
[Quidway-Pos3/0/0] standby interface pos 1/0/0 10
[Quidway-Pos3/0/0] standby interface pos 2/0/0 30
[Quidway-Pos3/0/0] standby interface tunnel 2/0/0 20
[Quidway-Pos3/0/0] standby threshold 80 50

# 查看备份中心的配置和状态信息。

[Quidway] display standby state
Interface  Interfacestate  Backupstate  Backupflag  Pri  Loadstate
Pos3/0/0       UP            MUP          MUD           TO-HYPNOTIZE
Pos2/0/0       DOWN          DOWN         BU        30
Tunnel2/0/0    UP            UPDELAY      BU        20
Pos1/0/0       STANDBY       STANDBY      BU        10
Backup-flag meaning:
M---MAIN  B---BACKUP     V---MOVED    U---USED
D---LOAD  P---PULLED     G---LOGICCHANNEL

项目

描述

Interface

主接口和各备份接口的接口类型和编号。第一行是主接口，接下来是各备份接口

Interfacestate
接口状态

  • 主接口的接口状态包括：UP、DOWN

  • 备份接口的接口状态包括：UP、DOWN、STANDBY

Backupstate
备份状态

  • 主接口的备份状态包括：MUP、MUPDELAY、MDOWN、MDOWNDELAY、MDESERT

  • 备份接口的备份状态包括：UP、UPDELAY、DOWN、DOWNDELAY、STANDBY、DESERT

Backupflag
备份状态标志

  • M：MAIN，该接口为主接口

  • B：BACKUP，该接口为备份接口

  • V：MOVED，该接口已经被拔走，或者该接口的主接口被拔走，或者该接口的所有备份接口都被拔走

  • U：USED，该接口作为主接口或者备份接口正在被使用

  • D：LOAD，该主接口参与备份负载分担

  • P：PULLED，该接口所在的接口板已经被拔走

  • G：LOGICCHANNEL，该接口是逻辑通道接口，仅为Tunnel

Pri

备份接口的优先级

Loadstate

备份负载状态，包括：WAKE、TO-HYPNOTIZE、TO-WAKE和稳定状态
