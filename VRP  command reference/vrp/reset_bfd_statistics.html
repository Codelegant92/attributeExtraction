reset bfd statistics

命令功能

reset bfd statistics命令用来清除BFD的报文收发统计信息。

命令格式

reset bfd statistics { all [ slot slot-id ] | discriminator discr-value }

参数说明

all：所有BFD的报文收发统计信息。

slot slot-id：指定接口板号。

discriminator discr-value：指定标识符的统计信息。
